'''科特布斯'''（{{lang-de|Cottbus}}，[[中古高地德语|中古高地德语]]：Kottbus，[[索布语|索布语]]：Chośebuz）是[[德国|德国]][[勃兰登堡州|勃兰登堡州]]的第二大城市，次於[[波茨坦|波茨坦]]，位于[[柏林|柏林]]东南大约125公里的[[施普雷河|施普雷河]]上。是一座工業都市。紧邻[[波兰|波兰]]。

== 地理 ==
科特布斯是{{link-de|下劳齐兹|Niederlausitz}}地区最大的城市，位于[[施普雷河|施普雷河]]中游，在南部的{{link-de|劳齐兹城墙|Lausitzer Grenzwall}}和北部的{{link-de|施普雷瓦尔德|Spreewald}}（Spreewald）之间。该市东西方向宽约15.6公里，南北方向长度为19.2公里。 施普雷河在科特布斯的宽度达到36米，流经该市的长度为23公里。

科特布斯总面积为164.2平方公里，其中森林面积为35.2平方公里，另有3平方公里的水域。距离该市最近的大城市有[[德累斯顿|德累斯顿]]，在该市西南方向约90公里，以及波兰的[[绿山城|绿山城]]，在该市东北方向约100公里，和在该市西北方向约100公里的[[柏林|柏林]]。

=== 气候 ===
科特布斯位于[[温带|温带]]地区。1971-2000期间的年平均气温为9.3℃。最热的月份是7月，平均气温为18.6℃；最冷的月份是1月，平均为-0.6℃。平均最高温和最低温之间的差值是19.2℃。

科特布斯气候潮湿，全年均有降水，其中夏季降水更多。1971年至2000年期间的年平均降雨量为559毫米。7月份平均降水量全年最多，为74毫米，2月份平均降水量34毫米全年最少。

== 历史 ==
科特布斯是始建于10世纪，[[索布人|索布人]]在[[施普雷河|施普雷河]]上的沙岛上建立了一个[[城堡|城堡]]。这城镇的名称第一次被提及是在1156年。在13世纪[[德国人|德国人]]定居者来到这里，之后他们与索布人一起住在这里。在[[中世纪|中世纪]]科特布斯以[[羊毛|羊毛]]闻名，而城镇的布料被输出到勃兰登堡全境、[[波希米亚|波希米亚]]和[[萨克森|萨克森]]。在1462年，勃兰登堡[[选帝侯|选帝侯]]（后来1618年成为[[普鲁士公国|普鲁士公国]]统治者）获得了科特布斯。在1815年上[[卢萨蒂亚|卢萨蒂亚]]及下[[卢萨蒂亚|卢萨蒂亚]]邻近地区由[[萨克森王国|萨克森王国]]割让予统治勃兰登堡的[[普鲁士|普鲁士]]。 

由1949年至1990年[[德国统一|德国统一]]前科特布斯属于[[德意志民主共和国|东德]]。

== 宗教 ==
科特布斯绝大多数（82.3％）的居民不信仰宗教。 居民中11.9％信奉[[福音派|新教福音派]]，3.5％信奉[[罗马天主教|罗马天主教]]，0.6％信奉[[东正教|东正教]]，1.1％信奉其他宗教。

== 文化与教育 ==
科特布斯是下[[卢萨蒂亚|卢萨蒂亚]]少数民族的文化中心。市内很多招牌也是双语的，那里还有一间下[[卢萨蒂亚|卢萨蒂亚]]式[[高级中学|高级中学]]，但是其实索布语很少会在街头听到。科特布斯亦是{{link-de|勃兰登堡科技大学|Brandenburgische Technische Universität Cottbus-Senftenberg}}（BTU）的所在地。

科特布斯每年都會舉行電影節。科特布斯市內有各種劇院。進行多種文藝演出。市內也有多家博物館。展示索布人的文化與生活。美術館則展示當地畫家的作品。市內的大劇院是新藝術運動風格的建築。大會堂也是頗為知名的建築。
