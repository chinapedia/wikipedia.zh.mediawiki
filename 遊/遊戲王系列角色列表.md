'''遊戲王系列角色列表'''是[[日本|日本]]漫畫家[[高橋和希|高橋和希]]所著的漫畫作品《[[遊戲王|遊戲王]]》及改編的動畫《遊戲王》、《遊戲王 怪獸之決鬥》所登場的角色一覽。原作漫畫曾經改編成兩部動畫、且都進行了相當多的原創劇情，因此配音員部份、若無特別標明則為動畫版第二作的配音員。

== 主要角色 ==
;[[武藤遊戲|武藤遊戲]]
:本劇男主角。在高中生武藤遊戲的身體裡寄宿著法老王的靈魂，兩人同為故事的主角。

;[[城之內克也|城之內克也]]
:本劇男主角之一，遊戲最好的朋友。

: 本劇主要配角，遊戲的同班同學，海馬娛樂集團的社長，人稱「卡片貴公子」、「天才決鬥者」，高中時就獲得全國怪獸決鬥大賽冠軍，在日本的遊戲中心保持著多項遊戲的最高分紀錄。本身知識淵博，對魔法卡、陷阱卡的效果有些了解。
: 弟弟海馬圭平是他唯一的親人，兩人從小生長在孤兒院，被海馬重機工業社長海馬剛三郎收養為義子，但對他只充滿著憎恨，從小的目標就是打倒他。推翻義父成功並逼使其自殺後、奪取了海馬集團並改組為娛樂業，將重心放在開發新遊戲及影像系統。他的夢想是在全世界建立「海馬樂園」，讓所有的小孩都有可以盡情玩樂的地方。
: 他是極端的科學主義者，完全不認同超自然現象、試圖將千年古物的神力以科學來詮釋，在杜馬篇中遭遇實體化戰鬥怪獸時，倒是闇遊戲以戰鬥怪獸是確實存在的，只是以現在的科技完全無法抵達他們所在的世界的論點說服。即使面對貝卡斯的千年眼神力也試圖以心理學解釋，並將伊西絲的千年首飾神力斥為無聊的幻象，此外，他也不認同遊戲體內有兩個靈魂存在，在動畫版的決鬥城市篇裡斥責遊戲對敵人的所知訊息，卻又震驚於隨後遊戲對他坦承現在在他眼前的人雖然是武藤遊戲，實際上卻也不是武藤遊戲，武藤遊戲是二心一體的存在。
: 在故事初期、他為了勝利不擇手段，以偽造的彩色印刷卡片騙取武藤雙六的「青眼白龍」，但這個行為被表遊戲發現、他也被闇遊戲打敗。他為了復仇，從世界各地搜刮其餘三張青眼白龍卡片，並將武藤雙六手中最後一張青眼白龍撕碎，成為世界上唯一一位擁有青眼白龍的決鬥者。
: 在遇見埃及考古局局長伊西絲、並從他手中得到神之卡「歐貝利斯克的巨神兵」後，為了得到決鬥王稱號、他舉辦了戰鬥城市大賽，將整個童實野市變成決鬥的舞台。並立下了賭卡規則，藉此吸引有著另外兩張神之卡的古魯斯。在王之記憶篇，貘良了道出了他與青眼白龍的因緣時，使得討厭靈異傳說的他決定親自前往埃及徹底釐清，於記憶世界中得知了自己的前世為古埃及時期的神官瑟特，並了解了自己與體內擁有青眼白龍的少女奇莎拉彼此相遇的過去。
:他是武藤遊戲(闇遊戲)的宿敵，在原作中與遊戲進行了四次決鬥，是與遊戲進行決鬥次數最多的一位。四次決鬥中僅勝過一次，但該次的對決是以自殺來威脅，才使遊戲落敗。他牌組裡的王牌是青眼白龍，並對她非常非常鍾愛，稱她是「我的驕傲、我的靈魂」，在與伊西絲一戰中，甚至「把神當成祭品來召喚青眼白龍」<ref>動畫第94集</ref>。此外、他的牌組以高度攻擊力的怪獸和高破壞性的魔法卡、陷阱卡為主題，例如能將對手牌組所有強力怪獸全部送進墓地的「死亡牌組破壞病毒」。
: 劇場版《[[遊戲王：次元的黑暗面|遊戲王：次元的黑暗面]]》裡，執著於讓亞圖姆回到現世，不但製作了AI亞圖姆，甚至主持大規模發掘，並使破碎的千年積木重新出土，與藍神決鬥在危急時召喚歐貝利斯克的巨神兵迎擊藍神，但最終被藍神搶走兩枚碎片，後來借助電腦將千年積木拚好，甚至叫來遊戲希望他能再次成為亞圖姆存於現世中的容器，但被遊戲告知在拿到碎片時，就已經知道亞圖姆根本不在積木裡。與遊戲聯手對決藍神時，因生命值歸零而敗北，但仍將積木交給遊戲希望他能召喚出亞圖姆，後來察覺到亞圖姆已經回來過一次，看似釋懷其實直接開發了仍是試作品的次元穿梭器，最後成功穿越次元抵達冥界後，見到亞圖姆並與其決鬥。
: 在《遊戲王》官方的部分活動中，經常會冠以「主辦單位：海馬娛樂集團」的名堂，此為故事中虛構的大型企業<ref>[http://natalie.mu/comic/news/185373 「遊☆戯☆王」約300点並ぶ原画展が明日から！海馬ボイスの会場案内も]</ref>。而2016年4月1日，集英社宣布了該公司將被海馬娛樂集團併購成為旗下子公司的[[愚人節|愚人節]]玩笑<ref>[http://animeanime.jp/article/2016/04/01/27816.html 「遊☆戯☆王」の海馬コーポレーションが集英社と合併 完全子会社化を予定]</ref>。此外、週刊少年Jump也企劃過該企業的虛構就職考試<ref>[https://www.shonenjump.com/p/sp/1604/kc/ 海馬娛樂集團入社測驗]</ref>。

: 本劇女主角。童實野高中二年級學生，遊戲的青梅竹馬，暗戀暗遊戲，胸圍根據雙六目測大於80公分。
: 她的夢想是在畢業後前往[[紐約|紐約]]學習舞蹈，成為職業舞者。為了存錢而違反校規私自打工，還被城之內發現，曾因此差點被蝶野老師退學。動畫第二作改為遭人威脅，並將夢想的緣由設定為幼年時父親因工作原因在紐約居住時，看了舞台劇「黑魔導女孩 賢者之寶石」所感動。
: 初期因為成為夏迪測試闇遊戲的棋子，察覺到當時的遊戲並非平常的遊戲，但城之內及其他人都認為是自己看走眼。在一次火災中察覺到闇遊戲是棲宿在千年積木裡的人格，而非原本就沉睡在遊戲心中的人格。與闇遊戲約會時，因其行動讓闇遊戲下定決心前往童實野美術館，親眼目睹那塊石板，因此確定闇遊戲是獨立的靈魂。在戰鬥城市中，被馬利克洗腦並淪為人質，但馬利克主要目的只是跟遊戲說話，清醒後仍被馬利克寄宿，目的轉為向闇人格復仇。
: 在戰鬥城市決鬥飛船的初賽結束後，追上伊西絲並詢問若闇遊戲「若集齊三幻神並尋回記憶，他將會如何？」被伊西斯告知闇遊戲遲早都會回到他應去的地方，也就是冥府。於戰鬥之儀篇，對於兩人的決鬥是無法接受卻又只能旁觀，一度想逃離但被城之內勸下。決鬥結束後，在亞圖姆步入冥界之門時哀求他留下，但沒有明確地表現出自己的愛慕，後被城之內勸阻而目送他離去，曾一度想追上前，被城之內阻止。
: 20周年劇場版《[[遊戲王：次元的黑暗面|遊戲王：次元的黑暗面]]》裡，知道遊戲希望成為遊戲設計師，鼓勵遊戲朝著夢想邁進，最後搭機前往紐約。

: 在王之記憶篇中、體內的人格現身並展開終極的黑暗遊戲「'''黑暗RPG'''」，與闇遊戲展開最終對決，期間甚至將其一部分的靈魂寄宿在本田身上，讓本田成為間諜（動畫劇情）。在輸了之後，體內的黑暗人格隨之消失。在最後見證了兩個遊戲的戰鬥儀式。

: 戰鬥城市篇的主要敵人、稀有卡竊盜集團古魯斯首領、千年錫杖持有者。持有神之卡「太陽神的翼神龍」。
: 於10歲時接受了極為痛苦的'''守墓者儀式'''後，誕生出兇殘暴虐的闇人格，通稱'''闇馬利克'''，一心一意只想毀滅掉一切，就連自己的表面人格也不放過。而原本的馬利克則被稱為表馬利克。與武藤遊戲、貘良了不同，並非一個身體中有兩個靈魂，而是一個雙重人格者，闇人格被利希德以私自刻在臉上的刻印所壓抑著，在闇人格出現時，頭髮會向上豎起。
: 他自幼十分嚮往地上的世界，為了慶祝他的生日、伊西絲違背禁令偷偷帶馬利克外出，馬利克因此撿到了描寫著重型[[機車|機車]]的[[雜誌|雜誌]]。但此事被發現後，禍及利希德、使他遭到馬利克的父親殘酷的懲罰而瀕死昏迷，馬利克的闇人格因而覺醒，奪取千年錫杖後殺了父親；並將父親背後所刻的碑文剝下披在利希德身上，不過利希德恢復意識後，再度壓抑了闇馬利克。

====海馬集團大賽篇====

:在海馬剛三郎還是社長時，朱雷達集團與海馬集團同樣為武器製造集團，然而其排名卻一直排在海馬集團之後；擁有多年歷史的朱雷達家族家訓是不管遇到什麼樣的敵人都絕對不會向對方低頭認輸，因此從小被教育成要超越海馬；在對抗海馬集團的意識愈來愈強時朱雷達集團的業績卻也相對的一直惡化，接著其父由於心理壓力加上長期疲勞，退下了社長的位子，雖然社長還是由其父掛名，不過卻已是擁有朱雷達集團實際管理權的人。
:代替其父接管朱雷達集團後，第一步就是撤除軍需產業、全面蛻變為遊戲娛樂業、徹底捨棄掉對海馬集團的對抗意識，但自從得知海馬瀨人接管海馬集團後也跟進的事情時，便日日夜夜投注耗費心血開發虛擬影像系統，然而此系統的開發[[專利權|專利權]]卻被海馬搶先一步，因此對於海馬抱持著很大的敵意。
:在海馬集團大賽時蓄意入侵海馬集團電腦以擾亂大賽，被海馬查獲並和其挑戰而落敗。
:之後將自己所收集到的「史特龍貝爾克的金色城堡」這張魔法卡偷偷交給弟弟雷恩，但其實那張卡有著精心設計的電腦病毒，能用來摧毀海馬集團的主電腦，不過終究還是無法得逞，因為這張卡片有要求對方需支付一半牌組為代價來維持效果，但是闇遊戲反而故意削減自己的牌組，確認牌組剩下的張數後，因只剩一張卡，所以「史特龍貝爾克的金色城堡」無法繼續存在而自滅，同時停止病毒的行動。在他自以為病毒成功侵入海馬集團主電腦時，海馬現身表示所有系統全都已經修復完畢、根本一點影響都沒有，使得他無法接受自己到頭來還是敵不過海馬的事實，在弟弟雷恩安慰幾句之後，下定決心要重頭來過。
