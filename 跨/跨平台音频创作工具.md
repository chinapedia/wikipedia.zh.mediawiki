'''跨平台音频创作工具'''（{{lang-en|Cross-platform Audio Creation Tool}}，简称'''XACT'''，也称'''跨平台音效製作工具'''）是[[微軟|微軟]]提供的音频[[函式庫|函式庫]]和引擎，為[[DirectX|DirectX]] [[SDK|SDK]]的一部分。<ref>[http://msdn2.microsoft.com/en-us/library/bb174772.aspx XACT on MSDN]</ref>XACT的實作可見諸於[[Xbox|Xbox]]上的Xaudio，[[Windows_XP|Windows XP]]上的[[DirectSound|DirectSound]]，以及[[Windows_Vista|Vista]]和[[Windows_7|Windows 7]]上的新音效堆栈（audio stack）。Xaudio是專用於Xbox的[[API|API]]，專門為了最佳化數位訊息處理（optimal digital signal processing）。XACT亦包含有X3DAudio。

目前XACT的功能已經從DirectX移到[[Microsoft_XNA|Microsoft XNA]]。XACT Audio Authoring Tool也整合進{{tsl|en|XNA Game Studio}}。XACT音樂音效編輯器可以產生XAP檔案，能在XNA中編輯與播放音效。<ref name=wheresDX>[http://msdn.microsoft.com/en-us/library/ee663275%28v=VS.85%29.aspx MSDN - Windows Development - Where is the DirectX SDK?]</ref>

随着Windows SDK for Windows 8开发预览版的发布，XACT已不再于Windows上受到支持。<ref name=wheresDX>[http://msdn.microsoft.com/en-us/library/ee663275%28v=VS.85%29.aspx MSDN - Windows Development - Where is the DirectX SDK?]</ref>

==功能==
*支援格式：[[WAV|WAV]]、[[AIFF|AIFF]]、[[XMA|XMA]]
**16和8[[位元|位元]]的[[PCM|PCM]]資料
*支援混音
*支援[[立體聲|立體聲]]（Stereo）
*支援Auditioning

==檔案型別==
*Sound Banks（.xsb） 
*Wave Banks（.xwb） 
*Global Settings（.xgs）
