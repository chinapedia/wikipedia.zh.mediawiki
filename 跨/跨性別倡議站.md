'''跨性別倡議站'''（{{lang-en|Transgender Punk Activist, Taiwan}}）是[[台灣|台灣]]一個以倡議[[跨性別|跨性別]]議題（部份[[女同性戀|女同志]]、女性主義）為定位核心的組織，於2012年9月成立運作。目前以研究與推動台灣{{tsl|en|Legal_aspects_of_transsexualism|性別變更登記}}議題、群體差異深化與合作連結為工作目標，並關注[[臺灣跨性別權益|台灣跨性別權益]]。

== 發起人 ==

陳薇真（Wei-Jhen Chen）自我認同為跨性女（transwoman），來自台灣[[台北|台北]]，學歷是[[國立政治大學|政治大學]][[哲學系|哲學系]][[硕士|碩士]]，曾擔任[[批踢踢|PTT]]跨性別板（transgender）板主（2010年）。<ref>{{Cite web|url=https://transgenderpunk.wordpress.com/about-2/|title=跨性別倡議站─關於這裡|accessdate=2018-7-8|author=|date=|publisher=}}</ref>

她曾於曾於女學會（2010年）與文化研究年會（2012年），發表台灣[[變性|變性]]/[[變裝|變裝]]網路社群歷史的研究（2010年）<ref>[http://www3.nccu.edu.tw/%7E97154007/tscd.pdf]</ref>，與一篇談生理女性[[性別不安|性別不安]]與MTF在台灣女[[同志|同志]]歷史(2012)的研討會發表<ref>{{cite web |url=http://www.csat.org.tw/paper/D-2-1_%E9%99%B3%E8%96%87%E7%9C%9F.pdf |title=存档副本 |accessdate=2013年4月10日 |deadurl=yes |archiveurl=https://web.archive.org/web/20160305000806/http://csat.org.tw/paper/d-2-1_%E9%99%B3%E8%96%87%E7%9C%9F.pdf |archivedate=2016年3月5日 |df= }}</ref>。

== 歷史 ==
2010年，陳薇真於擔任[[PTT|PTT]]跨性別板[[板主|板主]]期間，突顯台灣跨性別「T」在台灣自2004年後在[[社群|社群]]、[[運動|運動]]與[[媒體|媒體]]中已廣泛使用的「[[LGBT|LGBT]]」之間格格不入、被忽視的情況，因為往往較多仍然只是[[同性戀|同性戀]]的<ref>{{cite web|url=http://www.ptt.cc/man/transgender/D581/DD1F/D436/index.html|title=精華區|work=Ptt 批踢踢實業坊}}</ref>。此事件後來鬆動了台灣[[同志運動|同志運動]]的版圖，跨性別被忽略的情形引起正視。

2012年9月，向台灣知名的情慾自主空間BBS站台「花魁藝色館」申請獨立空間的情慾扮裝版（crossdress），獲站方同意，於11月開板。

2013年1月，發佈「台灣TS/CD網路約愛情境的性安全行為、暴力與求助」[[問卷調查|問卷調查]]<ref>{{cite web |url=http://transgenderpunk.wordpress.com/%E7%B4%84%E6%84%9B%E6%83%85%E5%A2%83%EF%BC%9A%E5%AE%89%E5%85%A8%E6%80%A7%E8%A1%8C%E7%82%BA%E3%80%81%E6%9A%B4%E5%8A%9B%E8%88%87%E6%B1%82%E5%8A%A9%E7%AE%A1%E9%81%93/ |title=存档副本 |accessdate=2013-04-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20131206025601/http://transgenderpunk.wordpress.com/%E7%B4%84%E6%84%9B%E6%83%85%E5%A2%83%EF%BC%9A%E5%AE%89%E5%85%A8%E6%80%A7%E8%A1%8C%E7%82%BA%E3%80%81%E6%9A%B4%E5%8A%9B%E8%88%87%E6%B1%82%E5%8A%A9%E7%AE%A1%E9%81%93/ |archivedate=2013-12-06 }}</ref>。結果指出，MTF中仍有近3成處於無收入狀態。約愛性行為中，肛交比例約50%，全程戴套比例：27%總是、12%經常、12%通常、15%偶爾、4%從不。其中高達近三成曾在網路約愛的跨性女曾遭[[性侵|性侵]]未遂，二成三曾遭性侵。跨性別的性安全防治與網路約愛人身暴力值得受到當局、社群與相關民間非政府組織重視。

2013年2月，於兩公約台灣國際審查發言台灣跨性別議題<ref>[http://transgenderpunk.wordpress.com/2013/02/27/2-25%E5%85%A9%E5%85%AC%E7%B4%84%E5%9C%8B%E9%9A%9B%E5%A7%94%E5%93%A1%E5%AF%A9%E6%9F%A5%E6%9C%83%E8%AD%B0%EF%BC%8C%E8%B7%A8%E6%80%A7%E5%88%A5%E5%80%A1%E8%AD%B0%E7%AB%99%E7%99%BC%E8%A8%80%E7%A8%BF/]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>2013年2月27日，台灣立報，〈換性關卡多 跨性者盼放寬規定〉。http://www.lihpao.com/?action-viewnews-itemid-126995</ref>。並嘗試撰寫一份台灣[[變性|變性]]群體情況的一般報告，儘管尚未是正式文件<ref>[http://www3.nccu.edu.tw/~97154007/ICCPR.pdf]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
