'''箕面市'''（{{jpn|j=箕面市|hg=みのおし|rm=Minoh shi|f=y}}）是位于日本大阪府北部的城市。轄區中部及北部為山區，占了全市面積的三分之二，僅南部為平原，主要市區也位於此。

== 歷史 ==

現在的箕面市大部分區域過去在令制國時代屬於[[攝津國|攝津國]][[豐島郡_(大阪府)|豐島郡]]，東南部屬於[[岛下郡|岛下郡]]；8世紀後設立的[[山陽道|山陽道]]通過南部的平原區域，在現在的萱野地區設有「草野站」、箕面地區「豐島牧」。<ref name="History">{{Cite web| url=https://www.city.minoh.lg.jp/shisei/profile/history.html| title=箕面の歴史| publisher=箕面市政府| accessdate=2019-01-15| language=ja}}</ref>

在[[江戶時代|江戶時代]]大部分區域為幕府直屬或旗本的領地，也有少部分區域屬於[[岡田藩|岡田藩]]、{{Tsl|ja|岡部藩}}和{{Tsl|ja|高槻藩}}。

19世紀末[[明治維新|明治維新]]後，這些區域曾先後隸屬大坂裁判所司農局、大阪府司農局、大阪府北司農局、[[攝津縣|攝津縣]]、[[豐崎縣|豐崎縣]]、[[兵庫縣|兵庫縣]]、半原縣、岡田縣、高槻縣，直到1871年的第一府縣整併中才全數隸屬大阪府。

1889年實施[[町村制|町村制]]，現在的箕面市在當時分屬{{Tsl|ja|止々呂美|止止呂美|止止呂美村}}、'''箕面村'''、{{Tsl|ja|萱野村 (大阪府)|萱野村 (大阪府)|萱野村}}、{{Tsl|ja|豊川村 (大阪府)|豐川村 (大阪府)|豐川村}}4個行政區劃；箕面村在1948年改制為'''箕面町'''後，同年又立刻併入了萱野村和止止呂美村，1956年再併入豐川村，同時升格為'''箕面市'''，不過數天後原屬豐川村東部的區域又劃出改併入[[茨木市|茨木市]]，此後箕面市的轄區不再變動。<ref name="History"/>

源於美國之甜甜圈連鎖店[[Mister_Donut|Mister Donut]]在1971年進入日本市場時，在全日本開出的第一家直營店即位於箕面市的箕面車站附近。

1999年，礦物學者在箕面山旁的平尾礦山發現新的礦石並經過[[國際礦物學協會|國際礦物學協會]]的認定，命名為{{tsl|ja|大阪石}}，2013年又繼續發現新礦石{{Tsl|ja|箕面石}}。

== 交通 ==

位於西南部的主要市區有[[阪急電鐵|阪急電鐵]][[箕面線|箕面線]]，有部分列車可直接經[[寶塚本線|寶塚本線]]通往大阪市區。東南部的栗生地區則有[[大阪高速鐵道|大阪高速鐵道]][[國際文化公園都市單軌電車線|國際文化公園都市單軌電車線]]從東側與茨木市的交界處通過，雖然轄內沒有設車站，但{{Tsl|ja|豊川駅 (大阪府)|豐川車站 (大阪府)|豐川車站}}和{{Tsl|ja|彩都西駅|彩都西車站}}都是緊臨著兩市邊界。

此外[[北大阪急行電鐵|北大阪急行電鐵]]的[[南北線_(北大阪急行電鐵)|南北線]]現也正在興建自[[千里中央站|千里中央車站]]往北延伸進箕面市{{Tsl|ja|萱野_(箕面市)|萱野_(箕面市)|萱野}}地區的路段，預計將在2020年通車啟用，屆時箕面市內將會設置兩個新車站。

公路方面在轄區北部山區有[[新名神高速道路|新名神高速道路]]通過，並設有[[箕面止止呂美交流道|箕面止止呂美交流道]]，可再經由{{Tsl|ja|箕面有料道路}}穿過山區抵達南部的主要市區。[[中國自動車道|中國自動車道]]也從南側[[豐中市|豐中市]]轄區內以東西向通過。

路線客運主要由{{Tsl|ja|阪急バス|阪急巴士}}所經營，箕面市的社區巴士也是委託給阪急巴士，不過阪急巴士並未在箕面市內設置營業所，因此轄內的路線是分別由周邊的{{Tsl|ja|阪急バス石橋営業所|阪急巴士石橋營業所|石橋營業所}}、{{Tsl|ja|阪急バス茨木営業所|阪急巴士茨木營業所|茨木營業所}}、{{Tsl|ja|阪急バス豊能営業所|阪急巴士豐能營業所|豐能營業所}}、{{Tsl|ja|阪急バス千里営業所|阪急巴士千里營業所|千里營業所}}所負責。

== 觀光資源 ==

市區北側的{{Tsl|ja|箕面山}}和周邊一帶山區在1967年被劃設為[[明治之森箕面國定公園|明治之森箕面國定公園]]，同時也是一直向東延伸到位於[[東京都|東京都]][[明治之森高尾國定公園|明治之森高尾國定公園]]的[[東海自然步道|東海自然步道]]在關西這端的起點；國定公園內主要景點包括{{Tsl|ja|箕面滝|箕面瀑布}}、超過千年歷史的{{Tsl|ja|瀧安寺}}和[[勝尾寺|勝尾寺]]。

在箕面山周邊山區過去由於大阪市立大學的學者以投餌的方式聚集了大量的[[日本獼猴|日本獼猴]]，因此在1955年一度以野生獼猴為主題設立了「箕面山自然動物園」，一度因為成為緊鄰市區的野生獼猴棲息地，被劃設為「箕面山日本獼猴棲息地」，但隨著猴群的規模由90隻成長到超過650隻以後，猴群逐漸開始騷擾觀光客，破壞農作，甚至進入市區民宅，因此在1977年決定改變政策，關閉「箕面山自然動物園」，讓山區回復自然。

== 教育 ==

[[File:2015_Osaka_Aoyama.jpg|thumb]]

轄內的高等教育學校包括[[大阪大學|大阪大學]]箕面校區和{{Tsl|ja|大阪青山大学|大阪青山大學}}箕面校區，大阪大學箕面校區過去原本是[[大阪外國語大學|大阪外國語大學]]，2007年大阪外國語大學併入大阪大學後，成為大阪大學的外國語學部。
