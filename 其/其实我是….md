《'''其實我是…'''》（{{lang-ja|実は私は}}）是[[增田英二|增田英二]]的[[日本|日本]][[漫畫|漫畫]]作品，自2013年1月23日起至2017年2月16日止連載於[[秋田書店|秋田書店]]《[[週刊少年Champion|週刊少年Champion]]》，單行本全22冊。故事描述人類與吸血鬼、狼人、惡魔及外星人一起於校園之中生活的搞笑戀愛喜劇<ref name=":0">{{Cite web|url = http://comic.qq.com/a/20150127/068848.htm|title = 增田英二《其实我是》宣布动画化|accessdate = 2015年7月28日|author = 狼咩咩|date = 2015年1月27日|publisher = 腾讯动漫}}</ref><ref name=":1">{{Cite web|url = http://acg.17173.com/news/04162015/143912845.shtml|title = TV动画《其实我是》主角将由花江夏树配音 2015年夏季开播|accessdate = 2015年7月28日|author = 喵酱|date = 2015年4月16日|publisher = 17173动漫频道}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 劇情 ==
男主人公黑峰朝阳喜欢班级里面的白神叶子，然而，在决心表白后，意外地发现她竟然是一個[[吸血鬼|吸血鬼]]。爱情的告白变成了「我们一起做朋友」的戏剧性结果，而不擅长保守秘密又爱着白神叶子的黑峰朝阳，竭力地保守着「吸血鬼」的秘密，以便不让白神叶子退学回家<ref>{{Cite web|url = http://jitsuwata.tv/intro/|title = Introduction｜TVアニメ『実は私は』公式サイト|accessdate = 2015年7月28日|author = |date = |publisher = }}</ref>……而在之后，本作男主人公还在学校中遇到了为发表八卦新闻而狂的青梅竹马朱美蜜柑、实际身份是外星人的蓝泽渚、能够变身狼人的色女紫紫戶狮惠，以及种种奇怪的人。不巧的是，作品中的多个角色都对黑峰朝阳有着爱意之情，而害羞的黑峰朝阳却迟迟无法表达出自己对白神叶子的喜爱，导致误会时常发生，并从而由此产生了一系列奇幻的爱情喜剧故事……

== 角色 ==
同時出現兩種版本的演員時，前者為電視動畫的配音員，後者為舞台劇演員。

=== 主要人物 ===
; {{nihongo|黑峰朝陽|黒峰 朝陽（くろみね あさひ）|聲：[[花江夏樹|花江夏樹]]<ref name=":1" />／演：[[大島崚|大島崚]]<ref name="舞台0" />}}
: 本作男主角，性格表裡合一非常不擅長說謊與保守秘密，被同班同學戲稱為「洞篩」。喜欢同班同学葉子，曾经尝试告白卻意外发现了葉子是吸血鬼的事实，决心與她成為朋友保守這個秘密，不過也變成告白沒有算很成功。以前喜欢过班长蓝泽，但去年夏天还未告白就直接被甩，成为心理阴影（本来早忘了）。
: 先後被藍澤渚，朱美蜜柑和紫紫戶獅穗對其表白，但均被他拒絕，因為自己喜歡的是白神葉子。漫画86话中已成功告白。

; {{nihongo|白神葉子|白神 葉子（しらがみ ようこ）|聲：[[芹澤優|芹澤優]]／演：[[飛鳥凛|飛鳥凛]]<ref name="舞台0" />}}
: 本作女主角，真實身分是'''[[吸血鬼|吸血鬼]]'''。因為憧憬著普通人類的校園生活，向父亲保证不被发现的前提下，開始就讀人類學校。为了防止被他人发现自己的真实身份，她不与班级中的他人交往，極少與人交談（說話很會暴露[[犬齒|犬齒]]，情緒高漲時也很容易暴露[[翅膀|翅膀]]），因此也没有朋友，上學與放學分別在極早與極晚等人少又陽光不強的時間（被照到陽光會曬得很黑），被做為「學園七大不可思議」中「放學後的吸血鬼」由來。
: 一次放學後，偷偷躲在教室展開自己背翼的她，被决心告白的朝陽撞到，从而使自己的吸血鬼身份泄露给了朝陽，朝陽也与叶子成为朋友，后逐渐喜欢上他。一直被認為是冷艷型的美少女，朝陽與她漸漸變熟後才發現性格其實很[[天然呆|天然呆]]，說話還不時會有[[近畿方言|關西腔]]。

; {{nihongo|藍澤渚|藍澤 渚（あいざわ なぎさ）|聲：[[水瀨祈|水瀨祈]]／演：[[若林倫香|若林倫香]]<ref name="舞台0" />}}
: 朝陽班上的班长。真實身分是來觀察地球的'''[[外星人|外星人]]'''，被朝陽和葉子偶然间发现。本體非常小跟[[手办|手办]]差不多，頭上長有觸角（求愛時會發光），被做為「學園七大不可思議」中「走廊裡的妖精」由來，隨身背著自家製的[[步枪|步枪]]（但根據對朝陽和葉子的結果好像沒有殺傷力）。
: 平時外人看到等身大的是類似[[机器人|机器人]]的外部組件，質感做得很逼真，頭上有[[螺絲|螺絲]]型的頭飾是操作席入口，需要外接充電運作。性格有點古板，非常重視紀律，曾經發現朝陽關注自己而主動表示要他停止，後來卻反而暗戀朝陽。

; {{nihongo|朱美橘|朱美 みかん（あけみ みかん）|聲：[[上田麗奈|上田麗奈]]／演：[[永吉明日香|永吉明日香]]<ref name="舞台0" />}}
: 朝陽的青梅竹馬，暱稱「橘子」。新聞社社長，因為以他人的八卦為題材大肆宣揚，而得到「邪道女王」的外號，並常以此自傲，有一对双胞胎弟弟。暗戀朝陽，自己的眼鏡是'''福神'''，是'''[[付喪神|付喪神]]'''，依付在橘的眼鏡身上的靈體（聲：[[下釜千昌|下釜千昌]]）。

; {{nihongo|紫紫戶獅穗／獅狼|紫々戸 獅穂（ししど しほ）／獅狼（しろう）|聲：[[内田彩|内田彩]]（獅穂）、[[宮健一|宮健一]]（獅狼）／演：{{link-ja|柴小聖|柴小聖}}<ref name="舞台0" />（獅穂）、[[田中彪|田中彪]]<ref name="舞台0" />（獅狼）}}
: 葉子的青梅竹馬，真實身份為'''[[狼人|狼人]]'''，見到月圓（或滿月的相片）就會變身，男性時稱作獅狼、女性時稱作獅穗。是個'''[[色狼|色女]]'''。而據本人所說，雖然人格是各自獨立，但因為獅穗是本體，因此即使是在獅狼的狀態下，獅穗仍能保留意識，然而獅狼在獅穗的状态下则不能保持意识。

; {{nihongo|黃龍院凜|黄龍院 凜（きりゅういん りん）|聲：無／演：[[高橋果鈴|高橋果鈴]]（[[Prizmmy☆|Prizmmy☆]]）<ref name="舞台0" />}}
: 來自50年後的未來人，朝陽的外孫女，小朝陽一年級。

=== 諸晴高中 ===
; {{nihongo|紅本茜|紅本 茜（こうもと あかね）|聲：[[市道真央|M·A·O]]／演：{{link-ja|高木友梨香|高木友梨香}}<ref name="舞台0" />}}
: 本作學園的校長，真實身分是數千歲的[[惡魔|惡魔]]，頭上有著大角，因此有「角女」的代稱，個子很小跟小孩子一樣，平時會穿著學校制服在校內閒逛，被做為「學園七大不可思議」中「神出鬼沒的帶角女人」由來。擁有千里眼、念力、分身、變身等能力。
: 個性喜好惡作劇和甜食，只要有甜食就能向她許願，而在得不到甜食或失去甜食時，會以念力製造災害。
: 相當重視明里。雖然經常取笑她是單身，但知道櫻田想對其表白時，依然表示自己會判斷他是否配得上她。成功表白後，也會稍微感到寂寞。

; {{nihongo|紅本明里|紅本 明里（こうもと あかり）|聲：[[新田惠海|新田惠海]]／演：{{link-ja|加藤智子|加藤智子}}<ref name="舞台0" />}}
: 主角的級任導師，神經大條和粗魯，與校長'''紅本茜'''為曾曾祖孫女的遠親關係，擁有惡魔的血統但比較微薄，不過仍有著強大的體能，常用以壓制嘲笑她的人和作亂的茜。在茜的散播之下，以「亞華里」的異名及武勇傳風聞於不良少年圈，是獅狼崇拜的對象。
: 自小時候起已經想成為公主，但一直無機會。櫻田的表白達成了她的夢想，同時而一個女人的身份接受了表白。但表明在在畢業前會保持距離。

; {{nihongo|銀華戀|銀 華恋（しろがね かれん）|聲：無／演：[[三浦菜菜子|三浦菜菜子]]<ref name="舞台0" />}}
: 學校的學生會會長，真實身分是[[天使|天使]]。自己的每一根羽毛含有七宗罪的七股力量寄宿，被羽根刺到的人會被該罪影響（被憤怒刺到會易怒、暴食刺到會容易飢餓）。雖然以頭上沒有了天使的光圈而自視為墮落天使，但個性上還是以服務為己任，並且只求道謝作為回報，不自覺釋放出的高潔人格常讓朝陽和葉子感到刺眼。

; {{nihongo|綠苑坂弓|緑苑坂 弓（りょくえんざか ゆみ）}}
: 主角升上三年級後的副級任導師。和過去朝陽對葉子的想像一樣，被葉子視為典型的『冰山美人』。
: 真實身分為葉子的父親同樣是吸血鬼的白神源二郎，因不放心葉子在學校的生活，故變身成一般女性潛入校園。卻被黑峰很快地發現其真實身分。之後拜託校長（紅本茜）聘請她（源二郎）擔任1年的副級任導師。另外，在名字的設定上並非因為頭髮是綠色，也並非穿著綠色色調布料的大正和服<ref>單行本第12冊封面。</ref>，而是取自白神桐子的舊姓，「弓」則是綠苑坂家裡所養，她非常疼愛的小狗的名字。

; {{nihongo|岡田奏|岡田 奏（おかだ かなで）|聲：[[平川大輔|平川大輔]]|演：[[渡邊健太|渡邊健太]]}}
: 被稱為「扭曲的岡田」，朝陽要好的三名友人中唯一在高中前便已認識的。觀察力優秀，很早就注意到葉子會觀察朝陽，因此當三人慫恿朝陽告白時，是較有信心的一個。對朱美有好感，但總是和對方互相冷嘲熱諷。擁有讀心能力。

; {{nihongo|櫻田康介|桜田 康介（さくらだ こうすけ）|聲：[[羽多野涉|羽多野涉]]|演：[[矢島八雲|矢島八雲]]}}
: 朝陽的友人之一，身形高大，被稱為「老好人櫻田」，個性很好，在四人組裡總是屬於勸阻的身份，但對於隨意毀謗和造謠不能忍受，會恢復成的狀態。家裡是肉店。相當珍惜朝陽等朋友，因為他們真正接受自己，而非視他為「中東的猛虎」。
: 升上高中前是不良少年，有著「東中的猛虎」外號。因為體型和眼神較兇狠，故經常被人找茬打架。因為會反過來把對方打敗。事後總是不被人相信自己不是先動手的一方，故不願相信他人，希望能獲得不依賴他人獨自生存的力量。在入學前被紅本明里打敗，並受她開解和介紹現在所就讀的高中。雖然當時不知道她的名字，但為了再見她一面，告訴她自己已改過自身，而就讀該高中。
: 在高三運動節中，在大家的幫助下，到達紅本明里面前。並透過感謝森羅萬象(曾幫助自己的紅本明里和全部幫助他的同學)轉化為自己的力量，獲得對抗狂暴狀態下的紅本明里的力量，被茜稱為「佛之櫻田」。最終成功在騎馬戰(騎著華戀給他的飛馬)中打敗紅本明里，並向她表白。

; {{nihongo|嶋田|嶋田 結太（しまだ ゆうた）|聲：[[下野紘|下野紘]]|演：[[早川航太|早川航太]]（A-Light）}}
: 朝陽的友人之一。個性糟糕卑鄙，做壞事時會有拖朝陽下水的行為。修業旅行時曾因某事而被全班視為人渣。

; {{nihongo|桃地結香|桃地 結香（ももち ゆか）}}
: 小主角兩個年級的學妹，與鳴為同班同學。腰後揹著形似手裡劍的大型物件。與凜一樣，稱呼朝陽為姥爺，實際上是獅穗與嶋田的外孫女，是個花痴。

== 影響 ==
* 2013年7月，[[美國|美國]][[電影演員|電影演員]][[湯姆·克魯斯|湯姆·克魯斯]]拍摄[[電影|電影]]《[[明日邊緣|明日邊界]]》時，明日邊界的原作者日本[[小說家|小說家]][[櫻坂洋|櫻坂洋]]在他本人的[[Twitter|Twitter]]向湯姆·克魯斯介紹這部作品，而湯姆·克魯斯也在他的Twitter上留言，该作品由此成為話題。随后，[[週刊少年Champion|週刊少年Champion編輯部]]藉機炒作，在他們的官方Twitter表示该作品是湯姆·克魯斯的最愛<ref>[http://www.j-cast.com/2013/07/22179908.html?p=all トム・クルーズ、漫画「実は私は」を突然RT　チャンピオン編集部「トムもお気に入り！」（J-CASTニュース）] {{ja icon}}</ref>，由此引發粉絲们的紛紛購買，导致該漫畫在[[亚马逊日本|亚马逊日本]]上一度售罄。
* 2016年2月在《週刊少年Champion》年13號（2016年2月25日發行）發表了和另一部在同一本雜誌連載及同樣以吸血鬼為題材的漫畫《{{link-ja|吸血鬼すぐ死ぬ|吸血鬼すぐ死ぬ}}》（{{lang|ja|盆ノ木至}}）的角色共演的短篇漫畫<ref>{{cite web|url=http://natalie.mu/comic/news/177388|title=「実は私は」×「吸血鬼すぐ死ぬ」、吸血鬼マンガコラボが週チャンに|publisher=[[Natalie_(網站)|Comic Natalie]]|date=2016-02-25|accessdate=2016-03-23|language=ja}}</ref>。後收錄在同年4月8日發行的單行本第16冊。
