'''其秦线轮渡'''，原称'''其威线轮渡'''。为[[中华人民共和国|中国]][[上海市|上海市]][[黄浦江|黄浦江]]上的一条对江轮渡航线，航线北起[[杨浦区|杨浦区]]、[[虹口区|虹口区]]交界的[[秦皇岛路|秦皇岛路]]1号秦皇岛路轮渡站，南至[[浦东新区|浦东新区]][[东方路|东方路]]1号其昌栈轮渡站。<ref>[http://www.shanghaiferry.com/1-%E7%BD%91%E9%A1%B5ppt%E8%BD%AC%E5%9B%BE%E7%89%87%E8%B5%84%E6%96%99/%E8%88%AA%E7%BA%BF%E8%88%AA%E7%8F%AD%EF%BC%88%E5%90%84%E6%B8%A1%E5%8F%A3%E7%AE%80%E4%BB%8B%EF%BC%89/1-%E5%85%B6%E7%A7%A6%E8%88%AA%E7%BA%BF%E7%AE%80%E4%BB%8B.png 上海轮渡其秦线简介]</ref>航线始于民国20年（1931年），为浦西威赛公码头至其昌栈码头的轮渡航线。民国35年（1946年），航线浦西码头移至[[秦皇岛路|秦皇岛路]]，于同年4月20日通航，航线定名'''其秦线轮渡'''。目前，其秦线轮渡实行通宵运营，为上海轮渡仅存的三条通宵航线之一。
== 历史 ==
民国20年（1931年），由私营振兴、振福两家公司在浦西威赛公码头至浦东其昌栈大街（后更名钱仓路、[[东方路|东方路]]）设渡口，轮渡航线使用小渡轮渡客。该渡小渡轮设施破旧，且过渡客流颇高，每日上下班高峰，经该渡过江的乘客数以千计，秩序混乱，多次发生乘客坠江淹死的事故。同年7月29日，[[江海关|江海关]]决定吊销其济渡执照，并请由市轮渡管理处派渡轮代替渡客。同年8月10日，上海市轮渡管理处派渡轮恢复航线行驶，并对其昌栈码头原设施进行翻修；而浦西威赛公码头为工部局新建的公共码头，为运送货物之用，市轮渡其威线通航后，工部局允许渡客在该码头上下，但市轮渡无管理权，只能在渡船上售票，令渡客颇觉不便。
<br>民国26年（1937年）[[抗日战争|抗日战争]]爆发，码头遭日军封锁，航线停航。民国34年（1945年），抗日战争胜利，国民政府接收轮渡航线，因威赛公码头已为美军所用，遂将浦西渡口迁往秦皇岛码头，并申请将该码头留作市轮渡专用。航线改为浦西秦皇岛路码头至浦东其昌栈码头，同时更名为其秦线轮渡，于民国35年（1946年）4月20日正式通航。1949年[[上海战役|上海战役]]前夕，航线停航。
<br>上海战役后，航线复航。1952年，对浦东其昌栈轮渡站进行扩建，将木踏级码头拆除，又利用上港八区码头一角岸线，建造浮桥及浮码头，新建候船室等设施。浦西秦皇岛路码头同时改建为浮桥及浮码头。工程于1952年12月建成启用。此后轮渡站设施又有多次修建。<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43385.html 上海市地方志 专业志 >> 上海内河航运志 >> 第六章渡运 >> 节] {{webarchive|url=https://web.archive.org/web/20140101093158/http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43385.html |date=2014-01-01 }}</ref><ref>[http://www.shtong.gov.cn/newsite/node2/node4/node2249/node4418/node20207/node21371/node62879/userobject1ai8873.html 上海市地方志 区县志 >> 区志 >> 虹口区志 >> 第十二编交通·邮电 >> 第三章市内客运 >> 节]</ref><ref>[http://www.shtong.gov.cn/newsite/node2/node4/node2249/yangpu/node45350/node45352/index.html 上海市地方志 区县志 >> 区志 >> 杨浦区志 >> 第十七编交通 >> 第一章轮渡] {{webarchive|url=https://web.archive.org/web/20120526055853/http://shtong.gov.cn/newsite/node2/node4/node2249/yangpu/node45350/node45352/index.html |date=2012-05-26 }}</ref><ref>[http://www.shanghaiferry.com/1-%E7%BD%91%E9%A1%B5ppt%E8%BD%AC%E5%9B%BE%E7%89%87%E8%B5%84%E6%96%99/%E7%99%BE%E5%B9%B4%E4%BC%A0%E6%89%BF/%E8%BD%AE%E6%B8%A1%E7%AE%80%E5%8F%B2/10-%E5%90%84%E6%B8%A1%E5%8F%A3%E5%8E%86%E5%8F%B2%E6%B2%BF%E9%9D%A9.html 上海轮渡各渡口历史沿革] {{webarchive|url=https://web.archive.org/web/20140202141841/http://www.shanghaiferry.com/1-%E7%BD%91%E9%A1%B5ppt%E8%BD%AC%E5%9B%BE%E7%89%87%E8%B5%84%E6%96%99/%E7%99%BE%E5%B9%B4%E4%BC%A0%E6%89%BF/%E8%BD%AE%E6%B8%A1%E7%AE%80%E5%8F%B2/10-%E5%90%84%E6%B8%A1%E5%8F%A3%E5%8E%86%E5%8F%B2%E6%B2%BF%E9%9D%A9.html |date=2014-02-02 }}</ref>2008年3月20日上午9时起，[[泰公线轮渡|泰公线轮渡]]停航改建，为配合泰公线改建，其秦线轮渡改为临时通宵航线。<ref>[http://www.shanghai.gov.cn/shanghai/node2314/node2315/node4411/userobject21ai263327.html 公平路轮渡站20日起停航改建 乘客可改从其秦线或东东线过江]</ref>目前，其秦线轮渡正式实行通宵运营，为上海轮渡仅剩的三条通宵航线之一。
