"斷腸草古今傳奇">舊雨：[http://www.civilwind.com/big5/hao/jy070812.htm 斷腸草古今傳奇]，北斗卮言</ref>或[[馬錢屬|馬錢屬]]（''Strychnos''）），是一種具劇毒的[[常綠植物|常綠]]、[[一年生植物|一年生]]<ref name = "星島文化植物"/>、纏繞性的[[藤本植物|藤本植物]]（亦有作[[草本植物|草本植物]]）。鉤吻的屬（''Gelsemium''），其意思為「雅致的」<ref name = "樹木谷"/>，名字來自[[意大利語|意大利語]]的[[茉莉|茉莉]]，取名可能認為此屬近似茉莉，此屬的共通點為花多數黃色或白色；詞（''elegans''）的意思為「秀麗的」“雅致的”<ref name = "樹木谷"/>。

==形态==
鉤吻的枝葉茂盛，枝幹光滑無毛；[[葉|葉]]為對生狀，呈卵形、狹卵形或卵狀披針形，葉頂端漸尖，基部漸窄或近圓形，全緣，膜質，長約5－12厘米，寬約2－6厘米；葉側脈每邊4－7條，腹面平坦而背部則隆起；[[葉柄|葉柄]]長約8－12毫米<ref name = "植物通"/>。

鉤吻長漏斗狀小[[花|花]]，略帶芳香<ref name = "葉脈網"/>，為頂生或長於上部葉腋的[[聚繖花序|聚繖花序]]，花期為5月－11月<ref name = "香港植物標本室"/><ref name = "香港有毒植物"/>；其[[花冠|花冠]]呈黃色或橙色，每片[[花瓣|花瓣]]內側呈淡紅色的斑點<ref name = "斷腸草有毒植物的代表"/>，長約10－19毫米，合共5塊，呈卵形裂片，較花冠管為短；花苞小而狹，長約2毫米，邊緣時有綠毛；[[花梗|花梗]]長約3－8毫米；[[花萼|花萼]]裂片有5塊，呈卵狀披針形，長約3－4毫米；[[雄蕊|雄蕊]]生於花冠管基部，內藏或稍伸出於花冠之外，與花瓣互生；花柱呈絲狀，長約8－10毫米，花藥呈狹卵形，有4淺裂，約1.5－2毫米；[[子房|子房]]有2室，呈卵形，光滑無毛，有多顆胚珠<ref name = "efloras"/><ref name = "植物通"/>。

鉤吻的果為[[蒴果|蒴果]]，呈卵形，分裂為2個2裂的果瓣，有宿萼，果期為8月－翌年2月<ref name = "香港植物標本室"/>。果皮膜質，稍膨脹，長約10－14毫米，寬約6－8毫米，在開裂前具2條淺縱槽。種子膜質，具有不規則裂齒狀的翅<ref name = "efloras"/><ref name = "植物通"/><ref name = "香港有毒植物"/><ref>[http://www.cvh.org.cn/difangzhi/hainan/page/3/199.pdf 海南植物誌 第199頁]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==醫藥用途及其毒性==
鉤吻全株具劇[[毒|毒]]，與[[洋金花|洋金花]]、[[馬錢子|馬錢子]]和[[羊角拗|羊角拗]]合稱「香港四大毒草」。當中新生幼葉以及根部的毒性最強<ref name = "中毒解救"/><ref name = "樹木谷"/>。鉤吻的根部含[[鉤吻鹼|鉤吻鹼]]（[[葫蔓籐鹼|葫蔓籐鹼]]<ref name = "星島文化植物"/>）甲、子、-{丑}-、寅、卯等，在離開泥土時略帶香味<ref name="神農斷腸草">[http://www.bjkp.gov.cn/bjkpzc/tszr/zwdg/zwyhj/160982.shtml 神農斷腸草—鉤吻] {{webarchive|url=https://web.archive.org/web/20080618065136/http://www.bjkp.gov.cn/bjkpzc/tszr/zwdg/zwyhj/160982.shtml |date=2008-06-18 }}，北京科普之窗</ref>，但多聞會令人產生暈眩感。鉤吻的葉則含鉤吻鹼甲、辰等。當中又以鉤吻鹼子含量最高，而鉤吻鹼寅則毒性最烈。一說只需服下約10[[克|克]]的[[莖|莖]][[葉|葉]]或2-8克的[[根|根]]或10-38枚的嫩[[芽|芽]]，即可引致中毒<ref>[http://jiangxi.jxnews.com.cn/system/2005/07/28/002117654.shtml 斷腸草是江西最毒植物 生長在贛南]，大江網</ref>，亦有指服食一小葉片已可致命，甚至，鉤吻的花粉亦帶毒性<ref name = "神農斷腸草"/>。[[清朝|清朝]]醫藥學家[[趙學敏|趙學敏]]在《[[本草綱目拾遺|本草綱目拾遺]]》中指：「胡蔓籐合香，焚之，令人昏迷。」<ref name = "神農斷腸草"/>。

鉤吻的毒性主要成份為多種[[生物鹼|生物鹼]]（包括鉤吻鹼），包括極強烈的神經毒性<ref name = "中毒解救"/>，服食過量即導致[[消化系統|消化系統]]、[[循環系統|循環系統]]和[[呼吸系統|呼吸系統]]的強烈反應<ref name = "神農斷腸草"/>，腸會發黑粘連<ref name = "星島文化植物"/>，中毒症狀包括流涎、噁心、口渴、吞嚥困難<ref name = "中毒解救"/>、[[發熱|發熱]]、[[嘔吐|嘔吐]]、口吐白沫、[[抽搐|抽搐]]、四肢麻木、[[肌肉|肌肉]]無力<ref name = "香港有毒植物"/>、肌肉纖維顫動、舌硬、言語不清、共濟失詞、煩躁不安、咽腹有燒灼或疼痛感、心律失常等。此外尚有[[瞳孔|瞳孔]]放大、[[視力|視力]]模糊，嚴重可致[[失明|失明]]，還有限鹼下垂<ref name = "中毒解救"/>。迷走神經時，可使心跳減慢、加速及失常，出現四肢冰冷、面色蒼白、體溫不開及[[血壓|血壓]]下降等症狀<ref name = "中毒解救"/>。中毒晚期可引起[[痙攣|痙攣]]、呼吸肌麻痺、[[窒息|窒息]]、[[昏迷|昏迷]]及[[休克|休克]]，最後甚至可因[[心臟衰竭|心臟衰竭]]或[[呼吸衰竭|呼吸衰竭]]至身亡<ref name = "神農斷腸草"/><ref name = "香港野花網">[http://www.producegreen.org.hk/hkwildflower/main_08.htm 斷腸草]，香港野花網</ref>。其毒理主要為抑制[[延髓|延髓]]的呼吸中樞，當嚴重受抑制時會引發酸中毒，最終可因延髓呼吸中樞及呼吸肌的麻痺而死於呼吸衰竭，同時亦可抑制[[腦部|腦部]]和[[脊髓|脊髓]]的運動中樞而引起肌肉麻痺。另一方面，亦可產生神經迷走，直接刺激[[心肌|心肌]]引起心律失常和心率之改變<ref name = "中毒解救"/>。

《[[全州志|全州志]]》曾有記載有一種名為斷腸草（又名胡蔓藤）的劇毒植物，其中一種解藥方法是「以狗屎調水灌治」。一說洗胃外，服食碳灰，再利用鹼水和[[催吐劑|催吐劑]]輔助，洗胃後將[[綠豆|綠豆]]、[[金銀花|金銀花]]以及[[甘草|甘草]]大火煎煮成藥後服用<ref name = "星島文化植物"/>。另有解毒劑療法，以[[阿托品|阿托品]]或[[士的寧|士的寧]]在皮下注射以對抗鉤吻鹼的迷走神經抑制作用，亦可在肌肉中注射[[新斯的明|新斯的明]]<ref name = "中毒解救"/>。羊血治療則是以新鮮羊血、白鴨血或鵝血約200－300毫升灌服約1至2次<ref name = "中毒解救"/>，亦有個案使用羊血直灌胃內作治療而得以活命<ref>[http://appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20081211&sec_id=4104&subsec_id=11867&art_id=11954634 婦服斷腸草羊血救活 大陸醫生用偏方建功]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}《[[蘋果日報|蘋果日報]]》2008年12月11 A12版</ref>。然而根據[[泰國|泰國]][[清邁大學|清邁大學]]研究，清除毒性並沒有特效藥，一般之處理方法只有洗胃、催嘔和催瀉以減少毒性劑量<ref>{{en}} Rujjanawate C, Kanjanapothi D, Panthong A. (Nov 2003). [http://www.ncbi.nlm.nih.gov/pubmed/14522437?dopt=Abstract Pharmacological effect and toxicity of alkaloids from Gelsemium elegans Benth.]. Chiangmai University.</ref>。

鉤吻雖然毒性猛烈，然而亦被視作[[中草藥|中草藥]]的一種，[[中醫|中醫]]稱其[[性味|性味]]為辛、溫、大毒<ref name = "現代中藥辭典"/>，有祛風、攻毒、消腫、止痛、抗炎、[[催眠|催眠]]<ref name = "斷腸草有毒植物的代表"/>等功效，入藥後外用，可治[[癬|頑癬]]、[[疥癬|疥癬]]、[[瘡|瘡患]]、[[濕疹|濕疹]]、[[麻瘋|麻瘋]]、[[風濕|風濕]]、[[關節炎|關節炎]]等症狀<ref name = "香港野花網"/>，同時亦是驅蟲藥<ref name = "現代中藥辭典"/><ref name = "香港有毒植物"/>。此外，[[豬|豬隻]]適量食用可治[[豬熱病|豬熱病]]及使毛色具光澤<ref name = "葉脈網"/>。[[明朝|明朝]]醫學家[[李時珍|李時珍]]在其本草學大成《[[本草綱目|本草綱目]]》謂：“斷腸草人誤食其葉者致死，而[[羊|羊]]食其則大肥”。

==鉤吻與金銀花==
過往，不少人常將鉤吻誤認作[[五花茶|五花茶]]成份之一的[[金銀花|金銀花]]，從而誤採誤食，引致中毒。然而，金銀花乃黃白相間，其花亦比鉤吻較長得多<ref name = "斷腸草有毒植物的代表"/>。

2005年，[[中國|中國]][[廣東省|廣東省]][[韶關市|韶關市]][[曲江區|曲江區]]有3名登山的學生在山野中誤將鉤吻當成金銀花而採摘回去，而後將「金銀花」放進滾水中沖泡來喝。不久，當時9名曾喝過「金銀花」水的學生呈中毒狀，其中一人在送院後不治。<ref name = "斷腸草有毒植物的代表"/>。

==神農嘗百草==
傳說「[[神農|神農]]嘗百草」就是因誤食鉤吻（斷腸草）而死。

相傳神農氏自小便非常聰穎，常替他人排難解憂。一說神農四肢和頭臚以外全身透明，他長著一個透明可見的腸胃，吃下東西時食物在胃裡每每也仍清晰能見，而他有抱一副為百姓試藥治權之心，於是不斷走遍山林荒野，嘗盡百草，一試療效，而當該種藥草具毒性時，必能在其通透的胃中看到黑色，從而得知百草藥性，更有神農一日而遇七十毒之說。

一次神農在一處向陽之地發現一種葉塊相對而生、開著淡黃色小花的籐，一如以往神農氏摘下小片嫩葉吃下，一下子毒性卻猛然發作，神農馬上感到不適。正當神農打算吃下他常備在身邊的解毒葉子時，此時卻目睹自己的腸子已截成一段段的，過了不久，神農也因此結束了生命。其後，人們便將該種植物叫作斷腸草，一般認為就是鉤吻<ref name = "斷腸草有毒植物的代表"/><ref name = "神農斷腸草"/>。

==小說中的斷腸草==
[[金庸|金庸]]所著的[[武俠小說|武俠小說]]《[[神鵰俠侶|神鵰俠侶]]》中，劇毒的[[斷腸草|斷腸草]]被描繪成虛構有毒植物[[情花|情花]]的解藥。
