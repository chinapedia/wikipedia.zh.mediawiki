'''鸿府大厦'''，位于[[北京市|北京市]][[海淀区|海淀区]][[五棵松_(北京)|五棵松]]路北甲8号，隶属[[中央军委机关事务管理总局|中央军委机关事务管理总局]]。

==简介==
鸿府宾馆在1983年建成并投入使用，是[[中国人民解放军总参谋部管理保障部|中国人民解放军总参谋部管理保障部]]综合局直属院校招待所，以接待全国各军事院校来京办事人员和各种会议为主<ref name=jianzhang>{{Cite web |url = http://www.m929.cn/zsjyxinxi.asp?art2_id=423&cat2_id=7   |title = 总参谋部管理保障部鸿府大厦招聘简章 |author =  |date = 2010-04-15 |publisher = 烟台职教网 |language =  |accessdate =  |quote =  }}</ref>。2008年在鸿府宾馆大楼东侧建成鸿府大厦并开业，鸿府大厦常年承担党政军重要会议的接待任务，并对外出租房屋<ref>{{Cite web |url =http://www.hbpx.net/news/201501/39722.html  |title = 总参鸿府大厦来保定女子职业中专挑选“两会”服务学生 |author =  |date =2015-01-16  |publisher = 河北培训网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.bidcenter.com.cn/newscontent-4751420-4.html |title =机关出差和会议定点场所（第二批）中标结果  |author =  |date = 2009-04-20 |publisher = 中国采招网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.pkulaw.cn/case/pfnl_121240975.html?match=Exact |title = ***与***等案外人执行异议之诉上诉案 |author =  |date =  2014-05-30|publisher = 北大法宝 |language =  |accessdate =  |quote =  }}</ref>。2016年，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]撤销，鸿府大厦转隶[[中央军委机关事务管理总局|中央军委机关事务管理总局]]<ref>{{Cite web |url = http://www.hbkjxy.cn/html/xwpd/xxxw/4370.html |title =  我校16名礼仪志愿者服务全国“两会”再受赞誉|author =  |date =2017-03-17  |publisher =  河北科技学院|language =  |accessdate =  |quote =  }}</ref><ref name=zongju>{{Cite web |url = http://www.81.cn/jfjbmap/content/2017-08/23/content_186216.htm |title = 奋楫中流勇攻坚——深化军委机关服务保障改革释放新体制效能纪实 |author =  |date = 2017-08-23 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

鸿府大厦自开业后，经常承担每年“全国[[两会|两会]]”中的[[全国人大|全国人大]]会议代表团接待任务。例如2010年，来北京参加[[全国人大|全国人大]]会议的湖南代表团入住鸿府大厦<ref>{{Cite web |url = http://news.163.com/10/0311/08/61FU498J000146BB.html |title = 湖南代表团向驻地宾馆赠送礼品 |author =  |date = 2010-03-11 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2012年，因[[京西宾馆|京西宾馆]]装修，新疆代表团入住鸿府大厦。2013年，重庆代表团入住鸿府大厦。2015年，湖南代表团入住鸿府大厦<ref>{{Cite web |url =http://news.163.com/15/0228/20/AJIODKH000014SEH.html  |title = 今年全国人大代表政协委员开会时住哪儿 |author =  |date = 2015-02-28 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
