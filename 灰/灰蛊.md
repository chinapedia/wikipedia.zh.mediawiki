{{vgname|灰蛊|en=Grey Goo}}，是由[[岩石壁画|岩石壁画]]出版，Six Foot开发的[[科幻|科幻]]即时战略游戏。该游戏于2015年1月23日在Grey Box发布，<ref name="releasedate" />在[[Microsoft_Windows|Microsoft Windows]]运行。<ref name="FAQ" />

== 遊戲介紹 ==
玩家可以選擇三個派別的其中一個，分別是the Humans、the Beta及the Goo<ref>{{Cite web|url=https://www.greybox.com/greygoo/en/info/ |title=GAME INFO |publisher=Grey Box |language=en |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20150412165124/https://www.greybox.com/greygoo/en/info/ |archivedate=2015-04-12 }}</ref>來完成單人任務和多人對戰模式。每個派別都有各自的單位、建築和打法。

== 故事 ==
遊戲故事描述在人類首度離開地球的500年後，為了探索未知星球，探索隊伍制造了可以消耗敵人利用敵人資源自我復制的蠱，以此幫助探索名為「九號生態系統」（Ecosystem Nine）的星球。但這些蠱卻有了自我意識，不再為人類所用，它們決定毀滅這個星球。

== 發售 ==
《灰蠱》於2014年3月13日宣佈，最初的發售日期為2014年秋季，<ref>{{Cite web |url = http://www.engadget.com/2014/03/13/petroglyph-introduces-rts-grey-goo-with-brief-teaser-screenshot/ |title = Petroglyph introduces RTS Grey Goo with brief teaser, screenshots |author = Earnest Cavalli |date = 2014-03-13 |publisher = Engadget |language = en}}</ref><ref>{{Cite web |url = http://www.ign.com/articles/2014/03/13/petroglyph-announces-new-rts-grey-goo |title = PETROGLYPH ANNOUNCES NEW RTS GREY GOO |author = Seth G. Macy |date = 2014-03-13 |publisher = IGN |language = en}}</ref>但後來延期。遊戲於2015年1月23日正式發售。<ref>{{Cite web |url = http://store.steampowered.com/app/290790 |title = Grey Goo |publisher = Steam}}</ref>

DLC包《''Emergence''》於2015年6月11日推出，內容包含三個新的單人任務。<ref>{{Cite web |url = https://www.greybox.com/greygoo/en/news/discover-emergence-grey-goos-new-dlc/ |title = DISCOVER "EMERGENCE" GREY GOO'S NEW DLC |publisher = Grey Box |language = en}}</ref>

DLC包《''The Descent of the Shroud''》於2016年2月1日推出，內容包含一個新的單人任務，以及全新派別「The Shroud」。<ref>{{Cite web |url = https://www.greybox.com/greygoo/en/news/descent-shroud-new-faction-revealed/ |title = DESCENT OF THE SHROUD - NEW FACTION REVEALED! |publisher = Grey Box |language = en}}</ref>
