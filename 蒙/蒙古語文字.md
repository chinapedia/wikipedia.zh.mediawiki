'''蒙古文字'''是[[蒙古族人|蒙古人]]用来记录[[蒙古语|蒙古语]]的文字。從蒙古人開始紀錄自己的语言以來，在不同时期和不同地区使用过多种文字体系。

== 历史概述 ==
[[室韦|室韦]]之[[蒙古部|蒙古部]]發跡於[[斡難河|斡難河]]之時本來沒有文字{{efn|此時草原同語族諸部早有採此字體者，不必單自[[蒙古部|蒙古部]]視之。是則以此字體書寫[[蒙古語|蒙古語]]之始當於13世紀初。參見符拉基米爾佐夫（Борис Яковлевич Владимирцов）之《蒙古書面語與喀爾喀方言比較語法》（Сравнительная грамматика монгольского письменного языка и халхаского наречия）。}}。1204年，[[成吉思汗|成吉思汗]]征討[[乃蠻|乃蠻]]之時，乃蠻掌印官[[畏兀儿|畏兀儿]]人[[塔塔統阿|塔塔統阿]]雖然被俘，却依然守著國家的印信。[[成吉思汗|成吉思汗]]非常嘉許其忠於自己國家的行為，遂命他掌管蒙古的文書印信，並教授太子、諸王畏兀字（[[回鹘字母|回鹘字母]]）以書寫蒙古語<ref>《[[元史|元史]]》列傳第十一：《塔塔統阿傳》。</ref>，[[蒙古族人|蒙古人]]至此時便採畏兀字（[[回鹘字母|回鹘字母]]）以记录蒙古語。

1269年，[[元世祖|元世祖]]委託[[國師|國師]][[八思巴|八思巴]]另製[[八思巴字|八思巴字]]作为[[元朝|元朝]]的官方文字。雖然诏令屢下，當時的蒙古人仍然喜歡使用回鹘式蒙古字母，八思巴字遂轉以轉寫他族語音{{fact}}。至明朝取代元朝后，八思巴字便罕見有人使用了。蒙古本土的蒙古人仍以回鹘式蒙古字母為正宗。

16世紀蒙古信奉[[佛教|佛教]]，廣譯遍傳[[佛經|佛經]]，蒙古文字因而普及、定型。因此16、17世紀之交，為蒙古之文藝復興。在這之前書寫蒙古語的部分缺失，皆改進之。此時引入許多[[藏語|藏語]]、[[梵語|梵語]]之[[佛經|內典]]用語，取代以往蒙古人使用的辭彙。1587年，[[喀喇沁|喀喇沁]]翻譯者[[阿尤希固什|阿尤希固什]]為了便於轉譯藏、梵語音，修改舊有字母，创制了[[阿礼嘎礼字母|阿礼嘎礼字母]]。该字母系统可以表達所有的藏、梵語語音，且可以表達諸蒙古語所無之[[复辅音|輔音群]]。此時部分古語遂棄而不用。學界謂此時之書面蒙语為'''古典蒙古语'''。

[[清代|清代]]亦取鑑蒙古字體，創[[满文|滿文]]以書[[滿語|滿語]]。二者字型相似，故能觸類旁通。此後自17世紀末至18世紀，'''古典蒙古语'''得以充分發展。北京、內蒙古之木版印刷業俱盛，諳蒙古語之帝王或者是[[藏族|藏族]]法師都參與編纂辭典、語法書等。

又1648年[[衛拉特|衛拉特蒙古]][[和碩特|和碩特部]]僧侶[[咱雅班第達|咱雅班第達]]亦根据卫拉特蒙古语的发音，在回鹘式蒙古字母的基础上改良创制出「[[托忒字母|托忒字母]]」，但此新字體只通行於[[准噶尔|准噶尔]]人中{{efn|此新字體亦有其正字法等規範，部分基於詞源學，部分基於衛拉特諸方言等。札雅班迪與其弟子亦以之譯出多部藏梵文獻。}}，東蒙古未有採用。

1686年，[[喀爾喀|喀爾喀]]僧侶[[扎納巴扎爾|扎納巴扎爾]]仿造藏文梵字，創造[[索永布字母|索永布字母]]，共有字母90個，左起橫書，一樣能精確表達藏、梵語，以及蒙古語的語音。索永布字母非常漂亮，寺院經常以此加以裝飾，只可惜此字體寫起來比八思巴字還不方便，因此罕用於社會上。同年扎那巴扎爾另造橫書方塊字，通行於喀爾喀各處之佛寺間。

20世紀初，曾有蒙古文拉丁化之議，當時卻未付諸實行。1946年之后，由於[[苏联|苏联]]的影響，[[蒙古人民共和國|蒙古人民共和國]]改以[[基里爾字母|基里爾字母]]拼寫[[喀爾喀|喀爾喀]]蒙古口語。现蒙古国已经恢复传统蒙文，但使用量较少。而[[中華人民共和國|中華人民共和國]][[內蒙古|內蒙古]]則仍採传统蒙古文。

== 汉字 ==
{{Main|汉字}}

===以汉字标音的《[[蒙古秘史|蒙古秘史]]》（1240年）===
[[File:Secret_history.jpg|thumb]]》的一页，左侧大字为用[[汉字|汉字]]标音的“蒙古文”，右侧为中文翻译和注解。]]

最古老的蒙古语文献是13世纪成书的《[[蒙古秘史|蒙古秘史]]》（{{lang|ru|mongɣol-un niɣuca tobciyan}}），当时蒙古文字尚未被发明，因此是用汉字写出来的。这里是该文献的开头和蒙古文拉丁转写、汉文注解对照：
* 成吉思<sup>中</sup>合罕讷忽扎兀尔
:cinggis　qaɣan-u　ijaɣur
:成吉思　可汗-属格　来源
:成吉思汗的根源

* 迭额<sup>舌</sup>列腾格<sup>舌</sup>里额扯扎牙阿秃脱<sup>舌</sup>列<sup>克</sup>先孛儿帖赤那阿主兀
:deger-e　tngri-ece　jayaɣa-tu　törü-gsen　börte cinua　a-juɣu
:来源-与格　天-离格　指定-名物化　出生-过去　孛儿帖赤那　是-过去
:奉天命而生的孛儿帖赤那（苍白狼），

* 格尔该亦讷<sup>中</sup>豁埃马<sup>舌</sup>兰<sup>勒</sup>阿只埃
: gergei　inu　quɣa maral　a-jiɣai
:妻子　他的　豁埃马兰　是-过去
:他的妻子是豁埃马兰。

由此可见，创制这种文字的人使用[[官话|官话]]，在他的方言中，[[中古汉语|中古汉语]]的[[入声|入声]]已经消失了，因此必须用特殊的符号（如上标“克、勒”等等）来代表[[闭音节|闭音节]]的[[韵尾|韵尾]]。为了区分蒙古语的[[颤音_(语音学)|颤音]]r和[[边音|边音]]l，在带有颤音的[[音节|音节]]前面附加一个上标“舌”字（“<sup>舌</sup>列”读作ri）。

== 回鹘字母 ==
{{Main|回鹘字母}}

回鹘字母是[[回鹘|回鹘]]人在[[粟特字母|粟特字母]]基础上创制的拼音文字。

回鹘字母从[[蒙古帝国|蒙古帝国]]时代起，开始被[[蒙古人|蒙古人]]用来拼写[[蒙古语|蒙古语]]，形成了后来的[[传统蒙古文|传统蒙古文]]。16世纪以后，[[满族|满族]]又仿照[[传统蒙古文|传统蒙古文]]创制了[[满文|满文]]。

=== 回鹘式蒙古字母（13世纪中期） ===

=== 蒙古字母（17世纪中期） ===
[[File:Mongol_khel.svg|thumb]]写法]]
{{Main|传统蒙古文}}

从17世纪开始，原本在字体和正字法上与[[回鹘字母|回鹘字母]]并无显著区别的蒙古文开始演化出独立的字体和形式，逐渐定型后即成为了现行的[[传统蒙古文|传统蒙古文]]。是[[中国|中国]]的[[内蒙古自治区|内蒙古自治区]]的官方文字，也是[[中国|中国]]大部分[[蒙古族|蒙古族]]使用的'''蒙古文'''，俗称“旧蒙文”。包括29个字母，其中表示[[元音|元音]]的有5个，表示[[辅音|辅音]]的有24个，拼写时以词为单位连书。行款竖行从左向右（即字序从上到下，行序从左到右）。

===托忒字母 (1648)===
{{Main|托忒字母|卫拉特语}}

在傳統蒙文字母的基礎上，1648年由蒙古[[和碩特|和碩特]]部僧人[[咱雅班第達|咱雅班第達]]改編、較符合[[卫拉特语|瓦剌語]]的文字。托忒意思是清晰易懂的(文字)。

[[File:Todo_Bichig_Latin.png|280px]]

===阿礼嘎礼字母 (1587年)===
{{Main|阿礼嘎礼字母}}
於1587年由蒙古喀喇沁部人阿尤西國師（Аюуш гүүш）創製，主要用於佛經中。

[[File:Das_Buch_der_Schrift_(Faulmann)_132.jpg|400px]]

== 八思巴字母（1269年～1368年） ==
[[File:Wiki-Phagspa.svg|thumb]]

[[八思巴字母|八思巴字母]]是[[元朝|元朝]][[忽必烈|忽必烈]]时的国师[[八思巴|八思巴]]所创。

== 索永布字母（1686年） ==
{{Main|索永布字母}}
[[File:Soyombo_example_zanabazar.png|thumb]]
于1686年由[[第一世哲布尊丹巴呼图克图|札那巴札尔]]（即第一世哲布尊丹巴呼图克图）发明。这套文字同时也可以用来书写[[藏语|藏语]]和[[梵语|梵语]]。

== 札那巴札尔方形字母 ==
{{Main|札那巴札尔方形字母}}
[[File:Monggol_in_kebtege_dorbeljin_bicig.png|thumb]]
在同期，[[第一世哲布尊丹巴呼图克图|第一世哲布尊丹巴呼图克图]]还创作了横排方形字母（Хэвтээ дөрвөлжин），又称[[札那巴札尔方形字母|札那巴札尔方形字母]]，于1801年被重新发现。此方案基于藏文字母，横排书写，元音附标。

== 拉丁字母（1931年） ==
指二十世纪初发明的用拉丁字母转写蒙古语的文字系统。但该方案没有得到推广。

== [[西里尔字母|西里尔字母]]（1941年） ==
