'''蒙罗维亚'''（{{lang-en|'''Monrovia'''}} {{IPAc-en|m|ə|n|ˈ|r|oʊ|v|i|ə}}<ref>{{Cite web |url=http://www.thefreedictionary.com/Monrovia |title=Definition of Monrovia |publisher=The Free Dictionary |accessdate=2014-01-05 }} {{IPAc-en|m|ə|n|ˈ|r|oʊ|v|i|ə|,_|m|ɒ|n|ˈ|r|oʊ|v|i|ə}}</ref><ref>{{Cite web |url=http://dictionary.reference.com/browse/Monrovia |title=Define Monrovia|publisher=Dictionary.com |accessdate=2014-01-05 }} {{IPAc-en|m|ə|n|ˈ|r|oʊ|v|i|ə}}</ref>）或譯'''門羅維亞'''，是[[西部非洲|西部非洲]][[利比里亚共和国|利比里亚共和国]]的[[首都|首都]]，同时也是利比里亚最大的城市。由于连年内战，导致该市市容破败。该市经济主要依赖其港口，而这个港口主要是由[[美军|美军]]于[[第二次世界大战|第二次世界大战]]期间扩建的。主要出口品包括[[乳胶|乳胶]]和[[铁矿|铁矿]]。本地也有对原材料进行加工，例如[[水泥|水泥]]、成品油、[[食品|食品]]、[[砖|砖]]、[[瓦|瓦]]、[[家具|家具]]、[[化工产品|化学工业]]。该港口还有大量的仓库和设施可以提供船只修理服务。

该城滨临[[大西洋|大西洋]]，是賴比瑞亞的主要港口之一，2008年人口普查時有970,824人，佔了全利比里亞人口的29%，是該國人口最多的城市。<ref>{{cite web|url=http://www.geohive.com/cntry/liberia.aspx |title=Global Statistics |publisher=GeoHive |date=2009-07-01 |accessdate=2010-07-04}}</ref>

== 历史 ==

1816年，為了安置被解放的美國黑奴，[[自由城|自由城]]得以建立，[[美國殖民協會|美國殖民協會]]幫助第一批黑人返回非洲。<ref name="WDL">{{cite web |url = http://www.wdl.org/en/item/446/ |title = Map of Liberia, West Africa |work = [[World_Digital_Library|World Digital Library]] |year = 1830 |accessdate = 2013-06-02 }}</ref> 1822年第二批黑人返回，被安置在[[梅蘇拉多角|梅蘇拉多角]]的一個叫做“Christopolis”的聚居地上。1824年为了纪念当时积极支持殖民运动的[[美国总统|美国总统]][[詹姆斯·门罗|詹姆斯·门罗]]而将此地命名门罗维亚。

1845年，美國殖民協會在此舉行制憲會議，會議中起草的憲法成為了後來[[利比里亞共和國|利比里亞共和國]]建立的基礎。<ref>Robin Dunn-Marcos, Konia T. Kollehlon, Bernard Ngovo, and Emily Russ (2005) in Donald A. Ranard (ed.) [http://www.trincoll.edu/~jmartha/liberians%20history%20and%20culture.pdf ''Liberians: An introduction to their history and culture''] {{webarchive|url=https://web.archive.org/web/20060724191221/http://www.trincoll.edu/~jmartha/liberians%20history%20and%20culture.pdf |date=2006-07-24 }} (Washington: Center for Applied Linguistics) </ref>

20世紀初，蒙羅維亞分成建築風格與[[美國南部|美國南部]]相似的[[美裔利比里亚人|美裔利比里亚人]]居住區和主要人口為[[克魯人|克魯人]]的克魯區，該區的居民還包括[[巴薩人_(利比里亞)|巴薩人]]、[[格雷博人|格雷博人]]和其他少數民族。<ref name="google1">{{cite book|url=http://books.google.com/books?id=ZJfJHjiiDyUC&pg=PA367&lpg=PA367&dq=prominent+americo-liberians&source=web&ots=jLDDQd-Dma&sig=y3RqiNeNk9KDFnYd4W5hAeLT8nY&hl=en&sa=X&oi=book_result&resnum=9&ct=result |title=Tiyambe Zeleza, Dickson Eyoh et al., Encyclopedia of Twentieth-Century African History |publisher=Books.google.com |date= |accessdate=2010-07-04}}</ref> 雖然最初美裔利比里亞人佔據了半數以上的人口，但後來本國民族也漸漸移民到蒙羅維亞以尋求工作。<ref name="google1"/>

在[[第一次利比里亞內戰|第一次]]和[[第二次利比里亞內戰|第二次利比里亞內戰]]期間，蒙羅維亞遭到嚴重破壞，幾乎所有的基礎設施都被毀去。2002年[[莱伊曼·古博薇|莱伊曼·古博薇]]組織了[[利比里亞女性大規模和平運動|利比里亞女性大規模和平運動]]，<ref>[http://www.odemagazine.com/blogs/readers_blog/9001/leymah_gbowee_peace_warrior_for_liberia 2009 Peace warrior for Liberia] {{webarchive|url=https://web.archive.org/web/20091227090728/http://www.odemagazine.com/blogs/readers_blog/9001/leymah_gbowee_peace_warrior_for_liberia |date=2009-12-27 }}</ref> 這項運動對第二次內戰在2003年的結束起到了積極作用，並使得該國及非洲第一任女總統[[埃伦·约翰逊·瑟利夫|埃伦·约翰逊·瑟利夫]]當選。<ref>[http://edition.cnn.com/2009/WORLD/africa/10/30/africa.women/ CNN, October 31, 2009]</ref>

2014年6月17日[[2014年西非伊波拉病毒疫情|西非伊波拉病毒]]爆發，導致蒙羅維亞七人喪生。 <ref>{{Cite web

==地理==
蒙羅維亞位於[[梅苏拉多角|梅苏拉多角]]半島，經緯度為{{Coord|6|19|N|10|48|W}}，介於大西洋和[[梅苏拉多河|梅苏拉多河]]之間，此河的河口形成了一個巨大的天然港。[[聖保羅河|聖保羅河]]流經城北，河中有[[布什羅德島|布什羅德島]]，此島與蒙羅維亞有大橋相連。

===氣候===
根據[[柯本氣候分類法|柯本氣候分類法]]，蒙羅維亞屬於[[熱帶季風氣候|熱帶季風氣候]]（Am）。<ref name="Climate-Data.org">{{cite web|title=Climate: Monrovia - Climate graph, Temperature graph, Climate table|url=http://en.climate-data.org/location/506/|publisher=Climate-Data.org|accessdate=2014-01-05}}</ref> 年降雨量極大，平均為{{convert|5100|mm|in}}，是世界上最潮濕的首都之一。全年氣溫都在{{convert|26|C}}左右。

== 经济 ==
[[File:Monrovia01.jpg|thumb]]
[[File:Carter_and_Tolbert.png|thumb]]和利比里亞總統[[威廉·托爾伯特|威廉·托爾伯特]]1978年在蒙羅維亞合影]]
[[File:Downtown_Monrovia_3348917715_67a2002529.jpg|thumb]]
港口为蒙罗维亚的经济命脉所在。蒙罗维亚是全国[[经济|经济]]和[[交通|交通]]中心、金融中心，[[賴比瑞亞中央銀行|賴比瑞亞中央銀行]]亦位于蒙罗维亚。

全国70％以上的工业企业集中在这里，其中主要有[[炼油|炼油]]、[[橡胶|橡胶]]、[[食品|食品]]、[[啤酒|啤酒]]、[[饮料|饮料]]、[[碾米|碾米]]、[[服装|服装]]、[[水泥|水泥]]和[[化工|化工]]等。在[[布什罗德岛|布什罗德岛]]上建立了“[[自由工业区|自由工业区]]”，美国、德国、[[巴西|巴西]]等国的30多家公司已在此建厂。此外，蒙罗维亚的捕捞业与鱼类加工业也相当发达。

== 市區景貌 ==
市中心[[梅苏拉多角|梅苏拉多角]]是繁华的[[商业区|商业区]]，街道两旁商店林立，人们熙来攘往，边观赏那琳琅满目的商品，边选购当地产的精湛新颖的手工艺品。跨过梅苏拉多桥即达市区北部的布什罗德岛，此为新兴工业区和驰名远近的西非唯一自由港——蒙罗维亚自由港。市区东部的首都山是政府机关的所在地，这里有行政大厦、国会大厦和市政大厅等。西非最古老的高等学府之——[[利比里亚大学|利比里亚大学]]（1862年開學）也设在这里。此外，还有博物馆与国立图书馆等。国会大厦前的广场上有一座高大的纪念碑，其顶端屹立着一位肩扛国旗、手持冲锋号的非洲妇女青铜塑像，为利比里亚自由独立的象征。

== 交通 ==
首都有[[干线公路|干线公路]]连接全国各大行政区。东郊的[[罗伯茨国际机场|罗伯茨国际机场]]是非洲最大的机场之一。利比里亚的海运业享誉世界，首都有20多条国际海运定期航线与外界相通，蒙罗维亚港系西非地区最大的现代化港口之一。 蒙罗维亚港是在第二次世界大战时兴建的。当时，德国法西斯者控制着欧洲及其海上航道，原来走美洲——西欧线的船只都改为美洲——西非线，蒙罗维亚成了这条航线上的枢纽。1944年[[第二次世界大战|第二次世界大战]]最紧张的时候，在旧港以北布什罗得岛兴建新港，1984年投入使用，成为当时西非最现代化的港口。1956年命名新港为“蒙罗维亚自由港”，允许外国轮船自由出入，商品可在港内自由转口，允许储藏、加工、再包装，只收1.5%的领事费，而不收[[关税|关税]]。从而吸引了更多的商船进港，增加了港口的劳务收入。
