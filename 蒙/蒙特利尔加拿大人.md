'''蒙特利尔加拿大人队'''（'''[[英语|英语]]：{{lang|en|Montreal Canadiens}}'''，[[法语|法语]]：'''{{lang|fr|Canadiens de Montréal}}'''）是位於[[加拿大|加拿大]][[蒙特利爾|蒙特利爾]]的[[國家冰球聯盟|國家冰球聯盟]]隊伍，隸屬於[[東部聯盟_(NHL)|東部聯盟]][[大西洋分區|大西洋分區]]。

蒙特利爾加拿大人隊成立於1909年，是现存唯一一支队史长于NHL的队伍，是NHL创建时的五支隊伍之一，同时也是1942～1967年间NHL困难时期的“原初六队”（Original Six）之一。蒙特利爾加拿大人隊曾贏得24次[[史丹利盃|史丹利盃]]，不但是[[國家冰球聯盟|國家冰球聯盟]]各隊之冠，也曾是北美各項職業體育隊伍之首（2009年被[[纽约洋基队|纽约洋基队]]的27次[[美國職棒大聯盟|美國職棒大聯盟]][[世界大賽|世界大賽]]冠軍所超越）。球隊最後一次奪魁是在1993年。

== 长期对手 ==
蒙特利尔的长期对手是[[波士顿棕熊队|波士顿棕熊队]]和[[多伦多枫叶队|多伦多枫叶队]]。蒙特利尔和这两只球队的比赛气氛不亚于[[皇家马德里|皇家马德里]]对碰[[巴塞罗那足球俱乐部|巴塞罗那]]，而且三支球队都有很长时间的球队历史，每一次碰撞都是三支球队底蕴的比拼。比赛当中不出现较大的摩擦会是一种意外。

== 百年球队 ==
蒙特利尔加拿大人队在2009年12月4日在贝尔中心主场度过了自己的百岁“生日”，贝尔中心进行了一次规模盛大的庆祝活动，而蒙特利尔加拿大人队也在庆祝活动之后的常规例行赛中以5比1大胜死敌波士顿棕熊，算是献给球队最好的生日礼物。也因为球队百年文化的传承，越来越多蒙特利尔市民开始关注并喜爱这支球队，其中不乏一部分在蒙特利尔生活的华人球迷。

== 2008年的逆袭 ==
关于蒙特利尔加拿大人队，球队最后一次夺冠还要追溯到1993年。但是球队在2008年2月主场和纽约游骑兵的比赛中打出了队史最惊人的一次反败为胜，在第2节打完10分钟后，纽约游骑兵队5比0领先，但是之后加拿大人队竟然顽强将比分扳平拖入加时赛，并在最后的点球决胜中胜出。这是加拿大人队第一次在5球落后的情况下逆转，也是加拿大冰球界一个非常辉煌的成就。
