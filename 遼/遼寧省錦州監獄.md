'''遼寧省錦州監獄'''是[[中華人民共和國|中華人民共和國]][[遼寧省|遼寧省]]的一所省屬監獄，隶属于[[辽宁省监狱管理局|辽宁省监狱管理局]]，是遼寧省最大的監獄之一。該監獄位於[[遼寧省|遼寧省]][[錦州市|錦州市]][[太和区|太和区]][[南山里|南山-{里}-]]86号，主要關押刑期在10年以上的犯人。<ref name=helan>[http://finance.ifeng.com/news/industry/20090423/574500.shtml 最后的荷兰村：中国最大的烂尾楼和多面杨斌，鳳凰網，2009年4月23日]</ref>

== 沿革 ==

[[辽宁省|辽宁省]][[锦州市|锦州市]]原有两所监狱，均为省属监狱，一为原[[辽宁省锦州永安监狱|辽宁省锦州永安监狱]]，一为原辽宁省锦州监狱。在监狱布局调整中，2002年初，[[辽宁省锦州永安监狱|辽宁省锦州永安监狱]]同原辽宁省锦州监狱合并，组建新的辽宁省锦州监狱。<ref>[http://www.moj.gov.cn/jjjcj/2006-01/23/content_40248.htm 锦州监狱实施“阳光工程”布局调整工作取得好效果，中国普法网，2006-01-23]</ref><ref>[http://www.southcn.com/law/fzxw/200211270175.htm 向现代化监狱迈进——辽宁省监狱布局调整纪实，南方网，2002年11月27日] {{Wayback|url=http://www.southcn.com/law/fzxw/200211270175.htm |date=20110721190957 }}</ref>

== 医疗及服务设施 ==
该监狱设有监狱医院。<ref name=minxin>{{Cite web |url=http://www.mxwz.com/meet/view.aspx?ID=297624# |title=【锦州】锦州监狱行风热线反馈，民心网会客厅，2009-10-15 |access-date=2010-10-11 |archive-url=https://web.archive.org/web/20110812044344/http://www.mxwz.com/meet/view.aspx?ID=297624# |archive-date=2011-08-12 |dead-url=yes }}</ref>

该监狱是[[辽宁省|辽宁省]]首家为犯人特辟“心理宣泄室”的监狱，并配有专门的心理医生为有心理问题的犯人提供医疗及康复服务。<ref>[http://news.china.com/zh_cn/domestic/945/20040309/11640170.html 锦州监狱首次为犯人特辟“心理宣泄室”，中华网，2004-03-09]</ref>
