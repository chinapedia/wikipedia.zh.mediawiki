《'''無限償還'''》（{{lang|ja|無罪モラトリアム}}，''Muzai Moratorium／Innocence Moratorium''）是日本音樂家[[椎名林檎|椎名林檎]]的第一張[[專輯|專輯]]，由[[EMI音樂_(日本)|東芝EMI]]／East World於1999年2月24日發行<ref name="JPN" />，簡稱'''M.M'''。在跨業合作部分，歌曲‘幸福論（悅樂篇）’、‘歌舞伎町的女王’、‘雖然夕陽照耀歸途’及‘警告’更先後成為日本[[三得利|三得利]]公司雞尾酒廣告的主題曲。而歌曲「在這裡接吻」則是日本[[NTV|NTV]]深夜劇‘Shin-D’的主題曲<ref name="KokodeKissShiteDRAMASONG1" />及[[NTV|NTV]]節目‘DOWN TOWN DX’的片尾曲<ref name="KokodeKissShiteDRAMASONG2" />。初回生產限定盤為「特殊封面仕樣」。

在專輯銷售成績方面，本張專輯在日本Oricon銷售榜上最高位居單週第2名<ref name="MuzaiMoratoriumOriconWeek" />，名列1999年年度銷售榜第25位<ref name="1999sale" />及2000年年度銷售榜第42位<ref name="2000sale" />。在銷售認證上，專輯《無限償還》銷售突破120萬張，在2000年5月通過日本唱片協會認定為三白金唱片<ref name="RIAJ200005" />。

== 專輯概要 ==
《'''無限償還'''》是椎名林檎首張專輯，收錄了先前發行的單曲《[[幸福論_(椎名林檎單曲)|幸福論]]》、《[[歌舞伎町的女王|歌舞伎町的女王]]》及《[[在這裡接吻|在這裡接吻]]》中的A面歌曲，而B面歌曲則沒有收錄至本張專輯中。同樣的她寫給[[廣末涼子|廣末涼子]]的歌曲「Summer Sunset」和[[友阪理惠|友阪理惠]]的歌曲「卡布奇諾」與「機器少女」也沒有收錄在這張專輯中。另外，收錄在第三張專輯《[[加爾基_精液_栗子花|加爾基 精液 栗子花]]》中的歌曲「葬列」原本要被收錄在本張專輯，但椎名林檎覺得歌曲並不符合這整張專輯的風格。她便拿去問負責專輯編曲的[[龜田誠治|龜田誠治]]，龜田誠治告訴椎名林檎：「如果不確定的話，那等到時機成熟再發表也不遲阿。」因此，就沒有將「葬列」收錄至本張專輯。日後，唱片公司將本張專輯收錄至2008年11月25日發行的套裝專輯《[[MoRA|MoRA]]》中，藉以慶祝椎名林檎出道十週年。

椎名林檎表示專輯《無限償還》為青少年時期創作的集大成，所以她很快地就決定要把這哪些曲目收入專輯中<ref name="rat4">{{Cite web |url=http://www.emimusic.jp/ringo10th/specials/plugin/004/004_2.htm |title= 「無罪モラトリアム」インタビュー |work=電脳RAT/04 |publisher=EMI |language=ja}}</ref>。但專輯收錄時間僅只有41分1秒，讓她直呼不夠，但她仍扎實地利用這些時間<ref name="rat4" />。至於專輯名稱由來，她則表示：「當歌曲開始錄製時，她只希望要以{{nihongo|Moratorium|モラトリアム}}為題<ref name="rat4" />。」後來，椎名林檎認為每個人都是認真地生活在社會上，因此並沒有「無法融入社會」這樣的說法，因為他們是無辜的、無罪的<ref name="rat4" />。但是這樣的標題一點意義也沒有，讓椎名林檎不太想用{{nihongo|無限償還|無罪モラトリアム}}作為專輯名稱<ref name="rat4" />。至於在專輯封面裡，椎名林檎和背景內的物品完全沒有關係，這樣的技巧首次出現在單曲《在這裡接吻》中，她認為這就像在緊張的法庭中間聆聽判決結果，到底是無罪或是勝訴那樣的有趣<ref name="rat4" />。

在音樂性部分，椎名林檎找來數個樂團來幫忙彈奏歌曲，她表示自己是從樂團中發跡的，因此更能了解樂團對於歌曲的重要性<ref name="rat4" />。而自稱為「新宿系」的主因就是希望詞曲能夠完美的結合在一起，因此她希望用不同的樂團彈奏歌曲，用以貼近歌曲的本質<ref name="rat4" />。在歌詞的部分，椎名林檎以自己16歲到18歲的親身經歷，期待能透過音樂創造出普遍的東西<ref name="rat4" />。而專輯中的歌曲「積木遊戲」與「嗎啡」，隱約可見椎名林檎受到[[竹久夢二|竹久夢二]]的[[日本文學|大正文學]]以及[[太宰治|太宰治]]、[[坂口安吾|坂口安吾]]的昭和文學影響<ref>{{cite web |url=http://mypaper.pchome.com.tw/readyou/post/1289923064 |title=椎名林檎簡介 |publisher=中央社 |language=zh-tw}}</ref>。

在專輯發售時程上，本張專輯於1999年2月24日於日本發行，台灣則等到04月16日才發行台壓版及進口版專輯<ref name="TWN" />。而在專輯銷售成績方面，專輯在發行當週以22.7萬張的銷售量在日本Oricon銷售榜上位居第2名<ref name="MuzaiMoratoriumOriconWeek" />，總計在Oriocn銷售榜上榜79週<ref name="MuzaiMoratoriumOriconConsecutiveWeek" />，累積銷售達143.3萬張，名列1999年年度銷售榜25位<ref name="1999sale" />及2000年年度銷售榜第42位<ref name="2000sale" />。同時，專輯銷售突破120萬張，在2000年5月通過日本唱片協會認定為三白金唱片<ref name="RIAJ200005" />。

首張專輯發行後，因為椎名林檎的身體出現微恙，因此休息了一段時間才開始準備下一張單曲《[[本能_(單曲)|本能]]》<ref name="rat7">{{Cite web |url=http://www.emimusic.jp/ringo10th/specials/plugin/007/007interview.htm |title= 椎名林檎ＦＡＸインタビュー |work=電脳RAT/07 |publisher=EMI |language=ja}}</ref>。而椎名林檎首次全國巡迴演唱會《先攻高潮》在仙台的演出也被迫延期<ref name="rat7" />。在這一兩個月的休養期間，椎名林檎跑去學開車，並入手一部新車取名「希特勒」<ref name="rat7" />。

#正確的大街（<small>{{lang-ja|正しい街}}，{{lang-en|Correct City}}</small>）
#:本首歌曲是椎名林檎18歲時的創作<ref name="rat4-1">{{Cite web |url=http://www.emimusic.jp/ringo10th/specials/plugin/004/004_1.htm |title= 椎名林檎「無罪モラトリアム」を語る。 |work=電脳RAT/04 |publisher=EMI |language=ja}}</ref>。歌曲由來是椎名林檎把要去東京的事和自己必須跟他離別的心情的寫進這首歌，同時椎名林檎也決定「如果這首歌要收錄專輯的話一定要放在第一首<ref name="rat4-1" />」。在這首歌的歌詞中，描繪出拋棄愛情中美好的部分，鮮明的刻劃當時的心境<ref name="rat4-1" />。

#[[歌舞伎町的女王|歌舞伎町的女王]]（<small>{{lang-ja|歌舞伎町の女王}}，{{lang-en|Queen of Kabuki-cho}}</small>）
#:椎名林檎第二張單曲。歌曲取材自椎名林檎18歲時的親身經歷<ref name="rat4-1" />。故事是源自於椎名林檎剛從福岡到東京[[澀谷|澀谷]]唱片行打工時，回家的路途上經過突然有ＳＭ俱樂部的皮條客要拉她去當女王，因此椎名林檎有感而發的寫出此首歌曲，描述淪落到歌舞伎町賣身的女人，每晚都讓不同男人享有自己的肉體，但女王逞強地表示不需要同情，也不需要愛情<ref>{{cite web| url=http://www.medianetjapan.com/2/meeting/pakuru/REVIEW/4/reve615.html | title=無罪モラトリアム |publisher=album review |language=ja}}</ref>。
#丸內虐待狂（<small>{{lang-ja|丸の内サディスティック}}，{{lang-en|Marunouchi Sadistic}}</small>）　
#:這首歌曲是椎名林檎旅居[[英國|英國]]時所創作的[[樣本唱片|樣本唱片]]。起初的版本為英文，後來應歌詞語感的關係改成了日文。歌詞內容以「愛上東京」為主題，但一部分歌詞被音譯詞取代，像是歌詞{{nihongo|benji|ベンジー}}，原本是要填入{{nihongo|淺井先生|浅井さん}}，但是{{link-ja|淺井健一|淺井健一}}認為這樣做太指名道姓了，因此改為用他的小名作為歌詞<ref name="rat4-1" />。而在椎名林檎甚至是[[東京事變|東京事變]]的演唱會上，時常表演這首歌曲。
#[[幸福論_(椎名林檎單曲)|幸福論（悅樂編）]]（<small>{{lang-ja|幸福論（悦楽編）}}，{{lang-en|A View of Happiness (Joy Ver.)}}</small>）
#:椎名林檎第一張單曲。歌曲的創作靈感是來自於在福岡男友，而歌詞中不但反應出椎名林檎對未來戀情的擔憂，也展現出她對愛情的忠貞。但和單曲版本不同，專輯版本採用擴音器演出。而至於單曲版本不收錄的原因可能是由於歌曲已經成為著名歌曲，但在椎名林檎出道十周年所發行的套裝合輯《[[MoRA|MoRA]]》中，有將單曲版本收錄<ref group="注">可以從[[iTunes_Store|iTunes Store]]之類的[[數位音樂下載|數位音樂下載]]服務取得單曲版本。</ref>。這首歌曲也時常在演唱會表演。
#雖然夕陽照耀歸途（<small>{{lang-ja|茜さす 帰路照らされど…}}，{{lang-en|When It Begins to Get Dark…}}</small>）
#:這首歌曲在[[樣本唱片|樣本唱片]]中是叫做「Headphone」。歌詞內容主要是描述椎名林檎16歲時在福岡生活的作息<ref name="rat4-1" />。
#席德與白日夢（<small>{{lang-ja|シドと白昼夢}}，{{lang-en|Sid & Daydreams}}</small>）
#:歌詞內容主要是寫對於19歲來臨的不安<ref name="rat4-1" />。收錄於專輯的版本是由龜田誠治編曲，和之後收錄於單曲《[[真夜中的純潔|真夜中的純潔]]》由{{link-ja|服部隆之|服部隆之}}編曲的版本不同，兩者的差異是前者是樂團版，後者是[[管弦樂|管弦樂]]版。
#:椎名林檎表示這首歌曲和「石膏」、「在這裡接吻」並列為「席德三部曲」<ref name="rat12">{{Cite web |url=http://www.emimusic.jp/ringo10th/specials/plugin/012/012_1.htm|title=椎名林檎「ギブス」「罪と罰」インタビュー|work=電脳RAT/012 |publisher=EMI |language=ja}}</ref>。歌曲「石膏」為正在熱戀中的曲目，而歌曲「在這裡接吻」則述說交往的男生腳踏兩條船的曲目<ref name="rat12" />。前者為17歲的創作，後者則為16歲時的創作<ref name="rat12" />。
#積木遊戲（<small>{{lang-ja|積木遊び}}，{{lang-en|Playing With Blocks}}</small>）　
#:本首歌曲是椎名林檎在高中時期樂團所創造的歌曲，收錄專輯前有稍微修改<ref name="rat4-1" />。而在[[音樂錄影帶|音樂錄影帶]]中椎名林檎穿著和服嬌滴滴地拿著扇子揮舞，另外一方面卻輕輕巧巧地隨著旋律撥弄[[古箏|古箏]]。椎名林檎表示這首歌是「{{lang|ja|箸休め的なお遊び曲}}」，因此此首歌的音樂錄影帶有收錄在音樂視頻專輯《[[性治療~第一輯~|性治療~第一輯~]]》中。
#[[在這裡接吻|在這裡接吻]]（<small>{{lang-ja|ここでキスして。}}，{{lang-en|Kiss Me}}</small>）
#:椎名林檎第三張單曲，在當時是椎名林檎最暢銷的單曲。而這首歌是椎名林檎17歲時的作品，描寫的對象為樂團中的男友，椎名林檎以自身經驗作詞作曲可以說是真正的情歌<ref name="rat3">{{Cite web |url=http://emimusic.jp/ringo10th/specials/plugin/003/003_1.htm |title=「ここでキスして。」インタビュー|work=電脳RAT/03 |publisher=EMI |language=ja}}</ref>。
#同一夜（<small>{{lang-ja|同じ夜}}，{{lang-en|Ordinary Night}}</small>）　
#:椎名林檎表示這就是每一天夜晚的感覺<ref name="rat4-1" />。在音樂性部分，本首歌曲以[[小提琴|小提琴]]為配樂，也是椎名林檎第一次和{{link-ja|齋藤毅 (音樂家)|斎藤ネコ|齋藤毅}}合作的歌曲。日後，在2000年9月13日發行的演唱會現場單曲《[[絕頂集|絕頂集]]》也有同名歌曲「Jｪﾁ~ｯじoTMｳ」，但名稱是椎名林檎故意誤植的。
#警告（<small>{{lang-ja|警告}}，{{lang-en|Caution}}</small>）
#:這首歌曲是椎名林檎18歲時寫給前男友的歌曲。椎名林檎在發行幾張單曲後，累積一定的知名度，然而她的男友卻每每打來抱怨她「妳是不好的」，但事後卻說那是氣憤時所講的話，請求她原諒，椎名林檎對這樣的舉動感到生氣，因此這首歌可謂那時心情寫照<ref name="rat4-1" />。
#嗎啡（<small>{{lang-ja|モルヒネ}}，{{lang-en|Morphine}}</small>）
#:歌詞內容主要是描述「椎名林檎在19歲時做事都倒行逆施，但接著就越來越好」<ref name="rat4-1" />。

== 銷售排行 ==
=== Oricon 銷售榜(日本) ===
《無限償還》於02月24日發行專輯。發行首週，在日本公信榜Oricon的專輯榜1999年3月第2週付的榜單中以237,910張銷售量居單週第二名。同週第一名是銷售量突破24萬大關的女歌手[[hitomi|hitomi]]的首張精選專輯「h」<ref name="MuzaiMoratoriumOriconWeek" />。第二週則以銷售量113,700張位居2位，當週冠軍依然是hitomi的首張精選專輯「h」<ref>{{Cite web |url=http://www.oricon.co.jp/search/result.php?kbn=ja&types=rnk&year=1999&month=3&week=3&submit5.x=30&submit5.y=12 |title=1999年03月第3週の邦楽アルバムランキング情報 |publisher=Oricon |language=ja}}</ref>。之後更盤據前十名長達4週，且之後因為《本能》、《石膏》及《罪與罰》單曲的熱賣，讓首張專輯在連續上榜42週後，正式突破百萬大關，而在上榜第50週時，再度以28,550張銷售量居單週第八名<ref>{{Cite web |url=http://www.oricon.co.jp/search/result.php?kbn=ja&types=rnk&year=2000&month=2&week=3&submit5.x=30&submit5.y=12 |title=2000年02月第3週の邦楽アルバムランキング情報 |publisher=Oricon |language=ja}}</ref>。總計在日本公信榜Oricon的專輯榜上榜79週，累積銷售達143.3萬張，為目前椎名林檎最長賣的專輯<ref name="MuzaiMoratoriumOriconConsecutiveWeek"/>。同時1999年年銷售榜中，《無限償還》以91.9萬張銷售量居全年第25名<ref name="1999sale" />。而全年度歌手總銷售金額排行榜中，則以38.9億日圓居全年第20位<ref name="1999sale" />。在2000年年銷售榜中，則以51.3萬張銷售量居全年第42名<ref name="2000sale" />。而當年度歌手總銷售金額排行榜中，則以122.0億日圓居全年第3位<ref name="2000sale" />。
