'''無障消費計劃'''是由[[香港|香港]]《[[社聯－滙豐社會企業商務中心|社聯－滙豐社會企業商務中心]]》在[[工業貿易署|工業貿易署]]「中小企業發展支援基金」撥款資助下，於2012年1月宣布推出這項計劃，向香港中小企業及[[社會企業|社會企業]]推廣「優質服務．無障關懷」理念<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2012/01/10/HK1201100020.htm | title=無障消費調查：84%遇困難 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date=2012-01-10 |accessdate=2012-03-04}}</ref>。

==背景==
[[社會服務聯會|社會服務聯會]]致力建設無障礙社會，為研究殘疾人士、少數族裔及長者對零售及飲食業的消費經驗及評價，以問卷及小組方式訪問500多名上述人士，報告指出84％人消費時遇上困難或障礙，其中「服務態度惡劣」及「難以與服務員溝通」分別佔41%及37%，而「環境設施不足」則佔26%，反映服務態度才是消費障礙的主因<ref>{{cite web |language=zh-hk |url=http://life.mingpao.com/cfm/reports3a.cfm?File=20120110/rptaa02a/ghb1.txt | title=長者少數族裔殘障 84％消費時遇困難  |publisher=[[明報|明報]] |date=2012-01-10 |accessdate=2012-03-04}}</ref>。

==報告內容==
報告列出一些個案︰
* 聾人及弱聽人士到快餐店用手勢點選3號套餐及飲品，被誤會要3杯飲品
* [[視障|視障]]人士因餐廳店員放下食物便離開，結果面對食物呆坐
* 少數族裔人士用膳時，餐廳給予的英語餐牌食物價錢刻意被標高
* 長者光顧藥房時，常被誤導購入過期藥物

關注聾人及弱聽人士的「[[龍耳|龍耳]]」創辦人[[邵日贊|邵日贊]]稱，如例子中店員往往未能諒解聾人及弱聽人士無法以言語溝通，以為他們低智商，沒有消費能力；部份店員說話太快或太細聲，聾人及弱聽人士看不到口形，無法溝通，店員乾脆不做他們生意<ref>{{cite web |language=zh-hk |url=http://www1.hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20120110&sec_id=4104&art_id=15968950 | title=殘疾人受歧視 獨自購物被騙買爛貨 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2012-01-10 |accessdate=2012-03-04}}</ref>。

[[路向四肢傷殘人士協會|路向四肢傷殘人士協會]]副主席羅偉祥指出，不便輪椅人士出入的街舖多達80%，其餘20%即使有斜路亦不時堆滿雜物，他表示締造無障礙消費環境一點都不難，只需花約2,000港元就可添置摺疊式的手提斜板，讓輪椅人士可以「越過」門前一級<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2012/12/08/HK1212080021.htm | title=200商戶助傷殘人士 社聯頒無障消費獎 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date=2012-12-08 |accessdate=2012-12-09}}</ref>。

==計劃目的==
為向中小企及社企推廣「優質服務．無障關懷」理念，促使香港成為真正的無障礙社會，社聯宣布推出「無障消費計劃」，以提升商戶對殘疾人士、長者及少數族裔的服務水準。

==計劃內容==
這計劃由社聯主辦、香港零售管理協會及平等機會委員會協辦，至今已獲9個中小企商會，另有16家機構、大學及社福機構支持。計劃內容包括向參與商戶提供講座、培訓及無障消費指引，又會派出神秘顧客評估商戶的服務水平，以協助商戶提升服務質素。所有參加這計劃的商戶，只要經過審批，即會獲頒「So-Biz獎項」，分為一星、二星及三星三個等級<ref>{{cite web |language=zh-hk |url=http://www.hkcd.com.hk/content/2012-01/10/content_2881266.htm |title=長者殘疾轟服務業態度惡劣 |publisher=[[香港商報|香港商報]] |date=2012-01-10 |accessdate=2012-03-04 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==頒獎禮==
社聯於2012年12月7日舉行「無障消費計劃」頒獎禮，表揚商戶積極提升無障消費服務水平，在400多間參加計劃的商戶中，近200間執行無障消費服務指引而獲得無障消費星級獎項，其中91間商戶更獲頒三星獎項。[[平等機會委員會|平等機會委員會]]主席[[林煥光|林煥光]]認為計劃對商戶、殘疾人士和社會均有利<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/news/art/20121208/18094555 | title=
400商戶推無障消費 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2012-12-08 |accessdate=2012-12-09}}</ref>。

社聯行政總裁方敏生表示，無障消費服務指引有40項指標，能夠執行10項的商戶可以獲得一星級，包括店舖設有渠道反映意見、在緊急情况下服務員懂得安全疏散長者、殘疾人士或少數族裔顧客等，獲得二星級的商戶除了要執行指引，亦需要培訓員工如何照顧殘疾人士或其他有需要人士，至於三星級別的商戶，則要額外獲顧客或弱勢社群團體的提名<ref>{{cite web |language=zh-hk |url=http://www.hkcd.com.hk/content/2012-12/08/content_3101002.htm | title=無障消費獎91商戶摘三星 |publisher=[[香港商報|香港商報]] |date=2012-12-08 |accessdate=2012-12-09}}</ref>。
