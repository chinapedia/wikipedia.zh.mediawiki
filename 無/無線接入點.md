'''無線存取點'''（英語：Wireless Access Point，縮寫'''WAP'''）是[[電腦網絡|電腦網絡]]中一种連接[[無線網絡|無線網絡]]至有線網絡（[[乙太網|乙太網]]）的裝置，又稱為'''無線基地台'''。它通常作為一個單獨裝置，並透過有線網路連接到[[路由器|路由器]]，也能與路由器整合在一起。無線接入點和[[热点_(Wi-Fi)|熱點]]不同，後者是指稱能夠連接[[無線區域網|無線區域網]]的物理地點。[[漫游|漫游]]則是數個無線接入點運行，將傳送的[[數據|數據]]由一個接入點連至到另一接入點使該無線網絡擴大；反之整個無線網路若沒有任何接入點就會成為一個點對點的[[Ad_Hoc网络|ad-hoc網路]]。無線接入點亦具有[[DHCP|DHCP]]之動態配置IP位址功能。

2010年之後，便宜、容易安裝的無線接收盒迅速地流行。這些裝置避免有線的乙太網絡纜線的纏結混亂。在[[商業|商業]]、[[家居|家居]]或是[[學校|學校]]如要鋪設有線網路經常需要纜線通過[[牆壁|牆壁]]和[[天花板|天花板]]，無線網路可以大幅減少這些纜線，甚至可以不用。無線網路給用戶有更大的流動性，不用被網纜縛住。今天的无线接取器通常支持使用无线电波接受发送数据的标准，这些标准和它们使用的频率是由[[IEEE|IEEE]]定义的。大部分无线接取器使用[[IEEE_802.11|IEEE 802.11]]标准。

== 常见应用 ==
一个典型的企业级应用，包括附加几个无线接入点到有线网络，然后提供无线接入办公[[局域网|局域网]]。无线接入点的管理是由[[无线局域网控制器|无线局域网控制器]]负责处理自动调节射频功率，通道，身份验证和安全性。此外，控制器可以组合成一个无线移动集团，允许跨控制器漫游。该控制器可以是流动性域的一部分，能够让客户在整个大的或地区级办公室地点的访问。这样可以节省客户的时间和管理开销，因为它可以自动重新关联或重新验证。此外，多个控制器和所有连接到这些控制器的数百个接入点都可以通过一个叫思科无线控制系统的软件来管理。这个软件可以处理一个和控制器相同的功能还增加了用户的映射或RFID地点的奖金功能上载的地图，提升控制器和接入点的固件，和流氓检测/处理。在这种情况下，无线接入点的功能是作为客户的无线网关来访问有线网络。

[[热点_(Wi-Fi)|熱點]]是一个常见的公用的无线接入点，其中无线客户端可以连接到互联网。这一现象已经很普遍了，在大城市的咖啡馆，图书馆，目前，在台灣已相當普及。
[[File:Planet_WAP-4000.JPG|thumb]]
在[[工商業|工商業]]方面，無線網路對日常運作有重要影響，企業員工經常需操作附有[[條碼掃描器|條碼掃描器]]及無線連接的可攜式資料終端機，以便讓他們即時更新工作進度和存貨。

一個符合[[IEEE_802.11|IEEE 802.11]]协议的无线接入点可與大約30個位於[[半徑|半徑]][[100米|100米]]之內使用者端聯絡。可是，无线接入点受到室內或室外設置、距離地面高度、附近的阻礙物、使用相同頻率電子裝置、[[天線|天線]]、[[天氣|天氣]]、使用[[頻道|頻道]]和裝置的功率等因素影響，致使通訊範圍變化很大。網路設計工程師藉由[[中繼器|中繼器]]和[[反射器|反射器]]去放大和反射無線信號，擴大無線接收盒的接收範圍。在測試中，無線網路操作範圍可達數[[公里|公里]]。

一個典型的企業應用，就是在有線網路上安裝數個无线接入点，提供辦公室區域網路的無線存取。在无线接入点的接收範圍內，無線用戶端既有移动性的好處，又能充分地與網路連接。在這種場合，无线接入点成為使用者端接入有線網路的一個接口。另外一個用途則是不予許使用網纜連接情況：例如，製造商使用無線網路連接辦公室和貨倉之間的網路連線。

== 安全性 ==
[[File:Network-wireless-encrypted.svg|120px]]
无线接入点在信息安全上有特別的顧慮。許多無線接收盒的安全性僅依靠實體的存取控制，並預設區域網路的所有使用者都是可信任的；然而無線接收盒若被連接到網路上，則其涵蓋範圍內的電子裝置均可能連線到此無線接收盒。目前最常見的解決方案為無線傳輸加密。先進的無線接收盒均有內建加密機制。第一代的資料加密機制[[WEP|WEP]]容易被破解；相較之下，第二代和第三代的[[WPA|WPA]]及[[WPA2|WPA2]]通常被認為是較安全的加密機制（具有較安全的[[密碼_(認證)|密码]]或{{tsl|en|passphrase|通行密碼}}（passphrase）的情形下）。<ref>[http://www.wi-fi.org/faqs.php http://www.wi-fi.org/faqs.php Page not found | Wi-Fi Alliance] {{webarchive|url=https://web.archive.org/web/20060925012015/http://www.wi-fi.org/faqs.php |date=2006-09-25 }}{{En}}</ref>

基於每個人都可享有隨時隨地連上網路的好處，一些人主張无线接入点对大众开放其使用权。
