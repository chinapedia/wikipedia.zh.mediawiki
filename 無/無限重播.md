《'''無限重播'''》（{{lang-ja|ヘビーローテーション}}，{{lang-en|''Heavy Rotation''}}）為[[AKB48|AKB48]]第17張<ref group="註">這是包括音樂下載單曲《[[Baby!_Baby!_Baby!|Baby! Baby! Baby!]]》的數目。若包括獨立製作時期的兩張單曲《[[櫻花的花瓣們|櫻花的花瓣們]]》和《[[裙襬飄飄|裙襬飄飄]]》，則為第19張。</ref>單曲，於2010年8月18日由[[King_Records|You, Be Cool!/KING RECORDS]]發行。這張單曲在2010年取得了[[Oricon公信榜|Oricon公信榜]]的單曲年榜第2位<ref name="oricon yearly" />，其A面曲《無限重播》也在2011年取得了同一排行榜的卡啦OK年榜冠軍。<ref name="oricon karaoke yearly" />

歌名「{{lang|ja|ヘビーローテーション}}」（以及其英語Heavy Rotation）有「在短時間不斷重複」的意思，尤指電台等不斷重播某一首歌曲<ref>{{ja icon}}{{cite web|url=http://dictionary.goo.ne.jp/leaf/jn2/199603/m0u/ |script-title=ja:ヘビー‐ローテーション【heavy rotation】 |publisher=goo |date= |accessdate=2012-07-19}}</ref>，在這首歌引申至對戀人的愛意不斷。

== 發行和宣傳 ==
《無限重播》分為Type-A、Type-B和劇場盤，並三種版本。Type-A和Type-B附送[[DVD|DVD]]，以及在發行首輪附送全國[[握手會|握手會]]活動參加劵。<ref name="Type-A">{{ja icon}}{{cite web|url=http://www.jbook.co.jp/p/p.aspx/4988003388379/j/ |script-title=ja:ヘビーローテーション＜Ｔｙｐｅ－Ａ＞【ＣＤ＋ＤＶＤ】 |publisher=JBOOK |date=2010 |accessdate=2012-07-16}}</ref><ref name="Type-B">{{ja icon}}{{cite web|url=http://www.jbook.co.jp/p/p.aspx/4988003388386/j/ |script-title=ja:ヘビーローテーション＜Ｔｙｐｅ－Ｂ＞【ＣＤ＋ＤＶＤ】 |publisher=JBOOK |date=2010 |accessdate=2012-07-16}}</ref>劇場盤則附送劇場盤發行紀念大握手會參加券和[[生寫真|生寫真]]1張。

2010年5月至6月，AKB48舉辦了[[AKB48第17張單曲選拔總選舉「向母親發誓，這是真的」|第17張單曲選拔總選舉]]，決定這張單曲的主要參與成員。<ref>{{ja icon}}{{cite web|url=http://beauty.oricon.co.jp/news/77055/full/ |script-title=ja:「AKB48総選挙」で大波乱　大島優子が初戴冠で前田のV2阻止！ |publisher=[[Oricon|Oricon]] |date=2010-06-09 |accessdate=2012-07-16}}</ref>單曲的廣告標語是「當跟喜歡的人相戀，腦海中就會，一直，無限重播！」（{{lang|ja|好きな人ができると、頭の中は、ずっと、ヘビーローテーション！}}）。<ref name="CD Journal" />AKB48曾於《[[MUSIC_STATION|MUSIC STATION]]》<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/36221 |script-title=ja:AKB、関ジャニ、HY、北乃きい、コブクロ「Mステ」に登場 |publisher=[[Natalie_(網站)|Natalie]] |date=2010-08-16 |accessdate=2012-07-16}}</ref>、《[[MUSIC_JAPAN|MUSIC JAPAN]]》<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/36331 |script-title=ja:「MJ」夏祭りSP後編にAKB48、TETSUYA、中川翔子ら6組 |publisher=[[Natalie_(網站)|Natalie]] |date=2010-08-18 |accessdate=2012-07-16}}</ref>等音樂節目宣傳單曲，表演《無限重播》一曲。

== 唱片封面 ==
這張單曲每個版本各有不同的封面，故封面共有三種版本。Type-A的封面以[[AKB48第17張單曲選拔總選舉「向母親發誓，這是真的」|AKB48第二屆選拔總選舉]]的頭三名——[[大島優子|大島優子]]、[[前田敦子|前田敦子]]和[[篠田麻里子|篠田麻里子]]為主角。Type-B封面則以選拔總選舉的冠軍、《無限重播》的中心——大島優子一個人為主角。封面上成員都穿上性感的內衣，[[Oricon|Oricon]]的報導稱之為「妖艷」。<ref>{{ja icon}}{{cite web|url=http://www.oricon.co.jp/news/confidence/78376/full/ |script-title=ja:AKB48、ロック調の新曲の衣装は妖艶な“ランジェリー姿”！ |publisher=[[Oricon|Oricon]] |date=2010-07-22 |accessdate=2012-07-16}}</ref>這些封面均由日本女攝影師[[蜷川實花|蜷川實花]]製作。另外，蜷川實花亦擔任了[[音樂影片|音樂影片]]的導演。<ref name="CD Journal">{{ja icon}}{{cite web|url=http://artist.cdjournal.com/d/-/4110051091 |script-title=ja:AKB48 ／ ヘビーローテーション(TYPE A) [CD+DVD] |publisher=CD Journal |date=2010 |accessdate=2012-07-16}}</ref>

== 歌曲 ==
《無限重播》是一首強烈地表現愛意的歌曲，歌詞甫開始就是「I want you! I need you! I love you!」。<ref>{{ja icon}}{{cite web|url=http://www.oricon.co.jp/music/interview/2010/akb0818/index.html |script-title=ja:AKB48大島優子『運命を感じた、自分にピッタリな新曲での初センター！！』INTERVIEW 1 |publisher=[[Oricon|Oricon]] |date=2010-08-18 |accessdate=2012-07-17}}</ref>CD Journal指這段[[副歌|副歌]]歌詞能給人深刻印象。<ref name="CD Journal" />這首歌在音樂上則帶有流行[[搖滾|搖滾]]的風格。<ref name="CD Journal" /><ref name="Listen Japan">{{ja icon}}{{cite web |url=http://listen.jp/store/diskreview_30816.htm |script-title=ja:ＡＫＢ４８／ヘビーローテーション（Ｔｙｐｅ－Ａ） |publisher=Listen Japan |date=2010 |accessdate=2012-07-17 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>Listen Japan的評論指「一眾少女的合唱再加上動感的搖滾音樂，讓AKB48充滿難以取替的魅力」。<ref name="Listen Japan" />

== 反應 ==
這張單曲於發行首天的銷量約為34.0萬張，取得當日的單曲日榜首位。<ref name="oricon daily" />首周銷量則約為52.7萬張，同樣於單曲周榜取得首位。<ref name="oricon weekly" />2011年，單曲以累積銷量約71.3萬張，取得當年單曲年榜的第2位。AKB48憑當年發行的5張單曲——《[[Beginner_(AKB48單曲)|Beginner]]》、這張單曲、《[[馬尾與髮圈|馬尾與髮圈]]》、《[[機會的順序|機會的順序]]》和《[[櫻花印記|櫻花印記]]》，分別取得年榜的第1位、第2位、第5位、第8位以及第12位。<ref name="oricon yearly" />

同年8月25日，在[[RecoChoku|RecoChoku]]每周排行榜上，《無限重播》同時於「手機片段」、「手機全曲付費」、「音樂影片」、「短訊影片」和「短訊鈴載」，合共五個部門取得首位。<ref>{{ja icon}}{{cite web|url=http://www.barks.jp/news/?id=1000063674 |script-title=ja:AKB48、「ヘビーローテーション」がレコチョク週間ランキング5部門で1位 |publisher=BARKS |date=2010-08-25 |accessdate=2012-07-16}}</ref>

《無限重播》是[[卡啦OK|卡啦OK]]的熱門歌曲。同年9月13日，這首歌在Oricon公信榜卡啦OK周榜以第12位首次上榜。在8星期後，這首歌於11月1日登上該排行榜的首位，其後連續48星期都獲得冠軍，為該榜的最長時間紀錄。<ref name="oricon karaoke" />2011年，這首歌取得卡啦OK年榜的冠軍。<ref name="oricon karaoke yearly" />2012年6月6日，這首歌取得首位的周數為81星期，超越[[GReeeeN|GReeeeN]]《[[奇蹟_(GReeeeN單曲)|奇蹟]]》的80星期，成為該排行榜取得首位周數最多的歌曲。<ref name="oricon karaoke">{{ja icon}}{{cite web |url=http://onlystar.oricon.co.jp/news/music/1588/ |script-title=ja:AKB48・大島初センター曲『ヘビーローテーション』通算81週目のカラオケ首位 |publisher=[[Oricon|Oricon]] |date=2012-06-08 |accessdate=2012-07-16 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>此後，也順利蟬聯2012年Oricon公信榜卡啦OK年榜的冠軍。

這首歌在AKB48粉絲間亦廣受歡迎。AKB48自2008年起，每年都會舉辦名為「重溫時間 最佳曲目100」（{{lang|ja|リクエストアワー セットリストベスト100}}）的活動，讓粉絲投票選出最喜愛的歌曲。在[[AKB48_重溫時間_最佳曲目100_2011|2011年（第4次）]]和[[AKB48_重溫時間_最佳曲目100_2012|2012年（第5次）]]，這首歌連續兩次取得了冠軍，為首次有歌曲能連續兩年稱霸。<ref>{{ja icon}}{{cite web|url=http://www.oricon.co.jp/news/music/2006061/full/ |script-title=ja:AKBファン投票ライブで「ヘビロテ」初連覇　「会いたかった」は初圏外 |publisher=[[Oricon|Oricon]] |date=2012-01-23 |accessdate=2012-07-17}}</ref>

《無限重播》的音樂影片在[[YouTube|YouTube]]的觀看次數已突破一億次<ref>[http://www.youtube.com/watch?v=lkHlnWFnA0c 【PV】 ヘビーローテーション / AKB48 [公式<nowiki>]</nowiki> - YouTube]</ref>，讓AKB48成為首組達成此創舉的日本歌手<ref>{{cite web | url=http://www.j-cast.com/2014/02/23197243.html?p=all | title=AKB48「ヘビーローテーション」再生回数1億回突破　日本の音楽動画で初 | date=2014-02-23 | accessdate=2014-02-26}}</ref>。

由於《無限重播》在推出後一度風靡日本社會，在曲中擔任中央位置的大島優子也在大眾中留下了深刻的印象，有媒體更將本曲形容為「大島之歌」。隨著大島於2014年6月「畢業」退出組合，繼任中央位置的成員人選也備受注目，因此大島在該年4月6日的演唱會「[[AKB48_重溫時間_最佳曲目200_2014|AKB48 重溫時間 最佳曲目200 2014]]」上指名了年輕成員[[向井地美音|向井地美音]]為其接班人，代替她擔任未來此曲的中央位置成員<ref>{{cite web|title=大島優子卒業で「ヘビロテセンター」はだれ？　「彼女以外考えられない」で封印説も浮上|url=http://topics.jp.msn.com/entertainment/tv/article.aspx?articleid=4690984|publisher=J-CAST|accessdate=2014-06-10}}</ref>。

== 翻唱 ==
