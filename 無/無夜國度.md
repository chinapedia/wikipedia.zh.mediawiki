{{Vgname|無夜國度|ja=よるのないくに|aka=无夜之国}}是[[Gust|Gust]]开发，[[光荣特库摩|光荣特库摩]]于2015年10月发行的[[电子角色扮演游戏|电子角色扮演游戏]]，对应[[PlayStation_3|PlayStation 3]]、[[PlayStation_4|PlayStation 4]]和[[PlayStation_Vita|PlayStation Vita]]平台。

==评价==
日本杂志《[[Fami通|Fami通]]》打出32/40<ref>{{Cite web|url = http://gematsu.com/2015/09/famitsu-review-scores-issue-1399|title = Famitsu Review Scores: Issue 1399|date = September 23, 2015|website = Gematsu}}</ref>，中国大陆《[[游戏机实用技术|游戏机实用技术]]》杂志给出23/30分<ref>《游戏机实用技术》381期，第16页。</ref>。游戏实体版初周总销量7.9万<ref>{{Cite web|url = http://gematsu.com/2015/10/media-create-sales-9u2815-10415|title = Media Create Sales: 9/28/15 – 10/4/15|date = October 7, 2015|website = Gematsu}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 续作 ==
2016年8月，Gust和光荣特库摩公布了本作的续作《[[无夜之国2：新月的花嫁|无夜之国2：新月的花嫁]]》，并计划登陆PlayStation 4和PlayStation Vita双平台。在[[任天堂Switch|任天堂Switch]]公布后，官方确定续作将增加任天堂Switch平台。游戏将于2017年8月31日在日本地区发售，10月在欧美地区发售。
