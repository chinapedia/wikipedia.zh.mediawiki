'''摩爾多瓦21歲以下國家足球隊'''（'''Moldova national under-21 football team'''，簡稱'''摩爾多瓦U21'''）是[[摩爾多瓦|摩爾多瓦]]的21歲以下國家足球代表隊，由[[摩爾多瓦足球協會|摩爾多瓦足球協會]]組織，為1991年[[苏联解体|苏联解体]]後分裂出來的球隊之一，參加每兩年舉辦一次的[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]。

== 競賽歷史 ==
摩爾多瓦U21自1996年參加[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]至今仍未曾晉級決賽週。[[2007年歐洲U-21足球錦標賽|2007年]]需要進行外圍賽預賽，對手為[[哈薩克21歲以下國家足球隊|哈薩克]]，首回合作客賽和0-0<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2007/round=2237/match=83549/index.html |title=Kazakhstan 0-0 Moldova |publisher=[[UEFA|''uefa.com'']] |date=2008-06-09 |accessdate=2009-09-24}}</ref></small>，次回合主場憑完場前射入[[十二碼|十二碼]]淨勝1-0獲得晉級<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2007/round=2237/match=83550/index.html |title=Moldova 1-0 Kazakhstan |publisher=[[UEFA|''uefa.com'']] |date=2008-06-29 |accessdate=2009-09-24}}</ref></small>；分組賽與[[英格蘭21歲以下足球代表隊|英格蘭]]及[[瑞士21歲以下國家足球隊|瑞士]]同組，雖然作客2-2迫和英格蘭<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2007/round=2238/match=84352/index.html |title=Moldova punish lacklustre England |publisher=[[UEFA|''uefa.com'']] |date=2006-08-15 |accessdate=2009-09-24}}</ref></small>，但主場卻1-3負於瑞士<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2007/round=2238/match=84353/index.html |title=Dzemaili leaves Switzerland on brink |publisher=[[UEFA|''uefa.com'']] |date=2006-09-01 |accessdate=2009-09-24}}</ref></small>，1和1負小組墊底無緣出線。[[2009年歐洲U-21足球錦標賽|2009年]]外圍賽摩爾多瓦U21被編入第9組，8戰4勝4負只得小組第4位，但在主場以1-0擊敗最後贏得錦標的[[德國21歲以下國家足球隊|德國隊]]<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/fixturesresults/round=15084/match=300204/report=rp.html |title=Moldova stun Germany |publisher=[[UEFA|''uefa.com'']] |date=2008-08-19 |accessdate=2009-09-24}}</ref></small>。
