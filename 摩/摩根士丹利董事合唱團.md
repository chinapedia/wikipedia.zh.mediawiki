'''摩根士丹利董事合唱團'''（[[英語|英語]]：'''{{lang|en|Morgan Stanley Directors’ Choir}}'''）是[[美國|美國]][[投資銀行|投資銀行]][[摩根士丹利|摩根士丹利]]成立的合唱團<ref>{{cite web |language=en |url=http://www.scmp.com/news/hong-kong/education-community/article/1892413/hong-kong-bankers-sing-their-hearts-out-good | title=Hong Kong bankers sing their hearts out for a good cause |publisher=[[南華早報|南華早報]] |date= 2015-12-17|accessdate=2016-04-26}}</ref>，由超過70名董事總經理組成<ref>{{cite web |language=zh-hk |url=http://orientaldaily.on.cc/cnt/finance/20151217/00271_004.html | title=絲絲講場：大摩高層唱出「龍耳」心聲 |publisher=[[東方日報|東方日報]] |date= 2015-12-17|accessdate=2016-04-26}}</ref>，俗稱「'''大摩合唱團'''」。現時領唱者為亞太區聯席首席執行官[[高浩澧|高浩澧]]<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/financeestate/art/20151217/19414961 | title= 中環在線：大摩員工為籌款 製彭博機蛋糕 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date= 2015-12-17|accessdate=2016-04-26}}</ref><ref>{{cite web |language=zh-hk |url=http://hk.on.cc/hk/bkn/cnt/finance/20151216/bkn-20151216194928494-1216_00842_001.html?eventsection=hk_finance&eventid=402883474be4f218014c0316734e596c | title=m18講場：大摩高層唱聖誕歌　籌款助「龍耳」 |publisher=[[On.tv東網電視|On.tv東網電視]] |date= 2015-12-16|accessdate=2016-04-26}}</ref>。

== 成立背景 ==
摩根士丹利一向支持慈善基金「[[愛心聖誕大行動|愛心聖誕大行動]]（Operation Santa Claus）」的籌款活動，該慈善基金是由[[香港電台|香港電台]]及[[南華早報|南華早報]]合辦。摩根士丹利為了協助該慈善基金籌款，於2002年成立「摩根士丹利董事合唱團」。

摩根士丹利援助服務的宗旨是幫助小朋友<ref>{{cite web |language=zh-hk |url=http://www.the-sun.on.cc/cnt/news/20141212/00410_031.html | title=城中鏡：60高層唱聖誕歌 大摩籌款破紀錄 |publisher=[[太陽報_(香港)|太陽報 (香港)]] |date= 2014-12-12|accessdate=2016-04-26}}</ref><ref>{{cite web |language=zh-hk |url=http://orientaldaily.on.cc/cnt/finance/20141212/00271_001.html | title=絲絲講場 大摩MD籌款報佳音 |publisher=[[東方日報|東方日報]] |date= 2014-12-12|accessdate=2016-04-26}}</ref>。

== 慈善活動 ==
每年的聖誕節，均由摩根士丹利員工捐款，一眾董事總經理及慈善機構的義工負責慈善演出，所得善款用作資助香港慈善機構的運作<ref>{{cite web |language=zh-hk |url=http://orientaldaily.on.cc/cnt/finance/20101217/00271_003.html | title=絲絲講場：大摩合唱團聖誕送暖 |publisher=[[東方日報|東方日報]] |date= 2010-12-17|accessdate=2016-04-26}}</ref><ref>{{cite web |language=zh-hk |url=http://the-sun.on.cc/cnt/news/20121215/00408_007.html | title=政壇：政商定格：「大摩合唱團」行善傳歡樂 |publisher=[[太陽報_(香港)|太陽報 (香港)]] |date= 2012-12-15|accessdate=2016-04-26}}</ref>。

摩根士丹利表示不但鼓勵員工捐款，亦會鼓勵他們多參與社區活動，善用他們的能力服務社區<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/news/art/20041218/4521867 | title=大摩合唱團 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date= 2004-12-18|accessdate=2016-04-26}}</ref>。
