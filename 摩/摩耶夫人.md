'''摩耶夫人'''（{{lang-sa|मायादेवी}}，[[IAST|IAST]]：{{IAST|Māyādevī}}，{{bd|？||前563年|？|}}），又称摩耶王后，被尊稱為Śrī Māyā（巴利语 Sirī Māyā），即是“吉祥伟大的摩耶夫人”，古印度[[净饭王|净饭王]]之妻、[[释迦牟尼佛|释迦牟尼佛]]之母。摩耶在[[梵语|梵语]]和[[巴利语|巴利语]]中意为幻化，在[[尼泊尔語|尼泊尔語]]則是指為「愛」，在佛經中又被尊称为'''摩诃摩耶'''（{{lang|sa|महा माया}}，{{IAST|Mahāmāyā}}，伟大的摩耶）。

摩耶夫人是净饭王的邻邦[[天臂城|天臂城]][[善觉王|善觉王]]的妹妹。

== 上座部佛教說法 ==
[[Image:MayaDream.jpg|thumb]]浮雕]]
南傳上座部的說法，她和[[淨飯王|淨飯王]]婚後遲遲未生育，摩耶夫人很是憂愁。有一天，夢見一頭[[白象|白象]]進入她的腹部<ref>《轉法輪經講記》，馬哈希尊者著</ref>，後來她便懷孕。按照當時印度的習俗，她回娘家待產，在五月的一個月圓夜，走到[[藍毗尼園|藍毗尼園]]，稍事休息，便作動要生，她在無憂樹下生下太子。當時的摩耶夫人已是四十幾歲<ref>《分別論》，巴利七論之一。</ref>，是高齡產妇。但太子依然是順產出生。

據佛典記載，當[[菩薩|菩薩]]投生至其胎藏時，[[四天王|四天王]]抬著她，到[[喜马拉雅山脉|喜马拉雅山脉]]大雪山中，之後，天王的皇后便帶她到阿耨達池沐浴，後讓她到金天宮中休息，不久就夢到白象，在夢時，菩薩投生了。

根據南傳上座部的說法，[[佛陀|佛陀]]在證悟後第七年上升[[忉利天|忉利天]]為天眾說法，前生是其母的天王{{IAST|Māyādevaputta}}來到，聽完佛陀說完法後，即證初果。

== 大乘佛教說法 ==
[[File:Māyādevī@Guimet.jpg|thumb]]19世紀摩耶夫人像。收藏於法國[[巴黎|巴黎]][[吉美國立亞洲藝術博物館|吉美國立亞洲藝術博物館]]]]
大乘的传说，她和净饭王结婚后20年没有生育。然后有一天，摩诃夫人梦见一头[[白象|白象]]进入她的左肋，从而怀孕。她在怀胎十月后，按照当时的习俗回娘家待产，在四月的第八天于[[藍毗尼園|藍毗尼園]]停留。在园中她举手攀一根树枝（[[無憂樹|無憂樹]]，或[[娑羅樹|娑羅樹]]）休息的时候，释迦牟尼即从她的右肋而出。传说园中的圣池（{{lang|en|Puskarini}}）是她在生产之前沐浴，以及之后她给释迦牟尼第一次沐浴的地方。淨飯王在太子出生後第五天，按當時的習俗舉行命名大典，太子被命名為悉達多。摩耶夫人于太子出生七天后去世，升入[[忉利天|忉利天]]。於是摩耶夫人的妹妹[[摩訶波闍波提|摩訶波闍波提]]嫁給淨飯王，悉達多太子就由姨母抚养。

== 在佛教中的影响 ==

《[[大藏经|大藏经]]》中有《摩诃摩耶经》，又名《佛升忉利天为母说法》，讲述佛陀在[[涅槃|涅槃]]前不久到忉利天为摩耶夫人讲授佛法，并向她告别的情景。另外，摩耶夫人也在《[[地藏经|地藏经]]》中被提到。

《[[巴利聖典|巴利聖典]]》中的《[[本生經|本生經]]》、《[[譬喻|譬喻]]》、《[[分別論|分別論]]》、《長老偈註》、《[[長老偈|長老偈]]》、《法句經註》。
