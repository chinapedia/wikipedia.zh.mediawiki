'''劇辛'''（{{bd|？||前242年|catIdx=J剧}}），又称'''剧子'''、<ref>《史记·卷七十四·孟子荀卿列传》：赵亦有公孙龙为坚白同异之辩，剧子之言。</ref>'''处子'''，<ref name="汉书"/>[[戰國|戰國]]時期[[燕國|燕國]]將領，前242年入侵[[赵国|赵国]]时被[[庞煖|庞煖]]擒杀。

== 生平 ==
剧辛原居于趙国，與[[龐煖|龐煖]]私交很好，[[燕昭王|燕昭王]]求贤时投奔燕国{{NoteTag|梁玉绳的《史记质疑》和钱穆的《先秦诸子系年考辨·一四四·邹衍考》都认为剧辛并非此时投奔燕国。}}。前242年，[[燕王喜|燕王喜]]因赵国长期遭受[[秦国|秦国]]攻击，主将[[廉颇|廉颇]]又出奔[[魏国_(战国)|魏国]]，想趁机进攻赵国。出征前燕王喜曾询问剧辛关于庞煖的情况，剧辛说庞煖是很容易对付的。燕王喜于是以剧辛为将攻打赵国，赵国派庞煖迎战。最终赵军大胜，俘虏燕国两万人，剧辛被擒杀。<ref>《史记·卷三十四·燕召公世家》：乐毅自魏往，邹衍自齐往，剧辛自赵往，士争趋燕。</ref><ref>《史記·卷三十四·燕召公世家》：劇辛故居趙，與龐暖善，已而亡走燕。燕見趙數困于秦，而廉頗去，令龐煖將也，欲因趙弊攻之。問劇辛，辛曰：「龐煖易與耳。」燕使劇辛將擊趙，趙使龐煖擊之，取燕軍二萬，殺劇辛。</ref><ref>《史记·卷四十三·赵世家》：（赵悼襄王）三年，庞暖将，攻燕，禽其将剧辛。</ref>

== 著作 ==
《[[汉书|汉书]]·卷三十·艺文志》在[[法家|法家]]论著里记载有《[[处子|处子]]》九篇，<ref name="汉书">《汉书·卷三十·艺文志》：《处子》九篇。师古曰：“《史记》云：赵有处子。”</ref>[[钱穆|钱穆]]根据[[颜师古|颜师古]]的注解考证《处子》为剧辛的著作。<ref>{{cite book|author=钱穆|coauthors= |title=《先秦诸子系年考辨》|year=1992年|publisher=上海书店|location= |isbn=7805694745}}</ref>

== 文学形象 ==
长篇历史小说《[[东周列国志|东周列国志]]》中，剧辛于第一百零二回《华阴道信陵败[[蒙骜|蒙骜]] [[胡卢河|胡卢河]]庞暖斩剧辛》中登场，本篇详细描述了庞暖与剧辛交战的经过：剧辛自赵奔燕后被任命为蓟（今[[北京市|北京市]]西南）郡守。燕王喜时，[[将渠|将渠]]称病辞去国相一职后由剧辛接任。燕王喜想报[[鄗代之战|鄗代之战]]惨败于赵国之仇，剧辛也在燕王喜面前夸口说庞煖是容易对付的，燕王喜于是任命剧辛为主将攻打赵国，庞煖与[[乐乘|乐乘]]、[[乐间|乐闲]]率军迎敌。庞煖先败剧辛一阵，燕将[[武阳靖|武阳靖]]中箭而亡。次日，剧辛邀请庞煖阵前对话，双方叙旧后再次激战，燕军不占优势，双方收兵。庞煖派使者给剧辛送去书信，说[[李牧|李牧]]已率赵军准备截击燕军后路，剧辛下令撤军，自己亲自断后。当夜庞煖率军追击剧辛，双方大战于胡卢河，剧辛战败后自刎，[[栗腹|栗腹]]之子[[栗元|栗元]]被乐闲俘虏后遭处决。庞煖同李牧会师，乘胜进兵，取燕国[[武遂|武遂]]（今[[河北省|河北省]][[徐水县|徐水县]]西北[[遂城|遂城]]）、[[方城|方城]](今河北省[[固安县|固安县]]南)之地，燕王喜被迫派将渠求和后庞煖才撤军。<ref>见 《东周列国志·第一百零二回·华阴道信陵败蒙骜 胡卢河庞暖斩剧辛》。</ref>
