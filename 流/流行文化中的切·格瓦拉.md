'''流行文化中的[[切·格瓦拉|切·格瓦拉]]'''，或'''切·格瓦拉时尚'''<ref name="age">{{Cite web |last = Berg |first = Chris |authorlink = |coauthors = |title = Che chic: you've ignored the horrors, now buy the T-shirt |work=[[The_Age|The Age]] |location= |date = October 14, 2007 |url = http://www.theage.com.au/news/opinion/che-chic-youve-ignored-the-horrors-now-buy-the-tshirt/2007/10/13/1191696232517.html |doi = |accessdate = November 5, 2010}}</ref>，是一种[[时尚潮流|时尚潮流]]，以阿根廷出生的革命家[[切·格瓦拉|切·格瓦拉]]的头像为特征标志。切·格瓦拉的形象在T恤设计中颇为流行，它们代表着切的政治信念；同时，用金钱去购买一件体现[[马克思主义|马克思主义]]标志的T恤本身也颇具讽刺意味，这些现象都吸引着诸多媒体、政治评论员和古巴裔美国活动家们的注意力。诚如政治评论员克里斯·伯格在[[世纪报_(澳大利亚)|世纪报]]中所说：“讽刺的是，切·格瓦拉这么一个文化标志之所以能够如此经久不衰，竟归功于切·格瓦拉一生都致力于摧毁的经济体制系统。<ref name="age" />

==流行性==
切·格瓦拉的形象在服装设计中很流行，以至于这个肖像被描述成“让成千上万件T恤畅销的那张脸”。<ref name="Guardian">{{Cite web |last = Chrisafis |first = Angelique |authorlink = |coauthors = |title = Che, my father: His face adorned the T-shirts and posters of a student generation. Even 35 years after his death Che Guevara remains an icon. But to his daughter Aleida he is the mysterious visitor who gave her sweets |work=[[The_Guardian|The Guardian]] |location= |date = May 3, 2003 |url = http://www.guardian.co.uk/world/2002/may/03/cuba.angeliquechrisafis |format = |doi = |accessdate = November 5, 2010}}</ref><ref name="ABC">{{Cite web |last = Porter |first = Barney |authorlink = |coauthors = |title = Che Guevara remembered |work=[[ABC_News|ABC News]] |location= |date = October 8, 2007 |url = http://www.abc.net.au/pm/content/2007/s2054050.htm|doi = |accessdate = November 5, 2010}}</ref> 评论员们注意到，此T恤之所以在年轻成年人尤其是大学生中如此流行，是因为他的形象与反叛精神有着密切联系。布里斯班时报的理查德·卡索讽刺地观察道：“在[[布里斯班|布里斯班]]的不伦瑞克大街或教堂大街随便走走，会不由自主感叹：也许切·格瓦拉本人是所有40岁以下年轻人唯一一个还没穿过切·格瓦拉头像衫的人。” <ref name="BT">{{Cite web |last = Castles |first = Richard |authorlink = |coauthors = |title = Have a life, willing to buy the T-shirt  |work=[[Brisbane_Times|Brisbane Times]] |location= |date = June 28, 2009 |url = http://www.brisbanetimes.com.au/opinion/have-a-life-willing-to-buy-the-tshirt-20090628-d0sx.html|doi = |accessdate = November 5, 2010}}</ref> {{When|最近}}与切·格瓦拉有关的潮流可归因于宏观经济困难，这让切所发出的信条更具号召力。<ref name="Balt">{{Cite web |last = White |first = Tanika |authorlink = |coauthors = |title = MEAN WHAT YOU WEAR ; This year's T-shirt has something to get off its wearer's chest|work=[[Baltimore_Sun|Baltimore Sun]] |location= |date = April 4, 2005 |url =http://pqasb.pqarchiver.com/baltsun/access/672766151.html?dids=672766151:672766151&FMT=ABS&FMTS=ABS:FT&type=current&date=Aug+03%2C+2004&author=Tanika+White&pub=The+Sun&desc=MEAN+WHAT+YOU+WEAR+%3B+This+year's+T-shirt+has+something+to+get+off+its+wearer's+chest&pqatl=google|format = |doi = |accessdate = August 3, 2004}}</ref> 与此同时，因为切·格瓦拉形象空前流行，也产生了很多恶搞，比如把他的胡子给刮了，并用他的名字开双关语的笑话。

==历史==
流行文化中切·格瓦拉的形象源自摄影作品[[英勇的游擊隊員|英勇的游擊隊員]]。 尚不明确何时开始被时尚设计界所使用，<ref name="Sal">{{Cite web |last = Watkins |first = Valentine |authorlink = |coauthors = |title =Cool t-shirt, bro |work=[[Salient_(magazine)|Salient (magazine)]] |location= |date = May 3, 2010 |url = http://www.salient.org.nz/features/cool-t-shirt-bro|format = |doi = |accessdate = November 5, 2010}}</ref> 只知1967年爱尔兰艺术家Jim Fitzpatrick基于此照片重新创作了一幅海报。<ref>Trisha Ziff, ''Che Guevara: Revolutionary & Icon'', Abrams Image, 2006, pg 21</ref>

== 回应==
切·格瓦拉潮流的大行其道，引起了诸多批评，声称不应忽视他的缺陷，也不应将他的行为浪漫化。 评论认为年轻人像支持偶像一样支持切·格瓦拉，却没有意识到他形象背后的种种争议，比如，他曾被认为支持[[暴力|暴力]]作为实现目标的手段之一，他也曾把[[古巴|古巴]]经济拖入泥潭。<ref name="age" /> 评论者称此潮流为“昙花一现，不足挂齿”。<ref name="SMC">{{Cite web |last = Bowen III |first = DK |authorlink = |coauthors = |title = Che Guevara: once a revolutionary, now just a T-shirt fad|publisher=[[The_Collegian_(Stewart's_Melville_College)|The Collegian]] |location= |date = October 23, 2007 |url =https://www.google.co.in/?gfe_rd=cr&ei=rPDlU67NHYzV8geH3YCYBg&gws_rd=ssl|doi = |accessdate = November 5, 2010}}{{date=December 2012}}</ref>

流亡海外的古巴人社区也以同样理由，反对切的形象被用作流行文化。<ref name="CI">{{Cite web |last = Marquez |first = Myriam |authorlink = |coauthors = |title = Revolutionist Che Guevara T-Shirts Keep Reality Under Wraps|publisher=[[Clarkson_University|Clarkson University]] |location= |date = April 4, 2005 |url =http://media.www.clarksonintegrator.com/media/storage/paper280/news/2005/04/04/Opinion/Revolutionist.Che.Guevara.TShirts.Keep.Reality.Under.Wraps-910465.shtml|doi = |accessdate = November 5, 2010}}</ref>

切·格瓦拉的长女阿蕾达·格瓦拉，则对这个从她父亲头像衍生出的潮流趋势表示坚决的支持态度，她说：“穿切·格瓦拉T恤的人，通常是不轻易妥协的人，他们不满足于社会现状，时刻思索着如何才能造福于人类社会。我想，我父亲应该是会赞成这样的事的。”<ref name="Guardian" />
