《'''流星花園'''》（{{lang-ja|'''花より男子'''}}，{{lang-en|''Meteor Garden''}}），是一部改編自[[日本漫畫|日本漫畫]]《[[花樣男子_(漫畫)|花樣男子]]》的[[校園|校園]][[愛情|愛情]]劇。由[[徐熙媛|徐熙媛]]（[[大S|大S]]）、男子團體[[F4_(男子團體)|F4]]主演，[[可米瑞智|可米瑞智]]製作。2001年於[[華視|華視]]九點檔《可米偶像劇場》播出後，該劇成為劃分台灣電視史上重要分水嶺，往後電視劇發展皆紛紛打造偶像題材的戲劇，因此媒體對該劇在歷史上定位是「偶像劇鼻祖」形容。因劇中主角F4在台灣掀起熱潮，接著自2002年起拍攝續集《'''[[流星花園II|流星花園II]]'''》，並在華視上映播出。

== 故事概要 ==
本故事主要是圍繞在一個出身貧寒的杉菜被愛錢父母強迫就讀一間貴族學校，和她如何因故與個性跋扈的富家子弟道明寺墮入情網。故事背景是在一所由在政經界最具影響力的四大家族為培養優秀後代而創立的貴族學校：英德學院。四大家族的繼承人－道明寺、花澤類、西門、美作號稱F4（Flower 4、花之四人組）在學校有著各種特權。但沒想到他們囂張跋扈的行為在遇到出身貧寒的杉菜後有了轉變。杉菜不畏強權，一個人挺身挑戰F4，而身為F4之首的道明寺竟然因此欣賞上她、不顧家族反對與杉菜交往……。

== 與原著差別 ==
* 原著的人名是日文翻譯，而在台版的改成符合華人人名的規則，比如'''道明寺司'''改為'''道明寺'''、'''牧野杉菜'''改為'''董杉菜'''、'''西門總二郎'''改為'''西門'''等。
* 原著的故事場景是[[高中|高中]]，而在台版的改為[[大學|大學]]。
* 原著的部份劇情也因為演員不足而刪除，比如杉菜在漫畫中本來有一個弟弟等。

== 影響力 ==
《流星花園》是一部非常成功的電視劇，開創[[台灣|台灣]]拍攝偶像劇的風氣，並且打開台灣連續劇往海外銷售的先鋒。《流星花園》的成功更轟動[[亞洲|亞洲]]演藝圈。截至2005年，《流星花園》製作人[[柴智屏|柴智屏]]評估至少有5億人看過F4的演出，全世界有16個國家和地區播放，《流星花園》掀起的旋風波及世界各地。兩部續集《流星花園Ⅱ》和《流星雨》也隨即開拍。《流星花園》掀獲得史無前例的成功，引發了整個亞洲的翻拍熱潮。《流星花園》劇中的男演員，原本默默無聞的四位大男孩（[[吳建豪|吳建豪]]、[[言承旭|言承旭]]、[[周渝民|周渝民]]、[[朱孝天|朱孝天]]）也因此一夜成名，因為劇中角色的名氣而組成了實際的演唱團體「[[F4|F4]]」，也成為炙手可熱的的廣告商寵兒。2002年起，F4舉辦世界巡迴演唱會，在中、港、台、日、韓、東南亞人民及海外華人地區造成話題。繼《流星花園》後，F4每位成員的個人作品在亞洲都有賣埠市場，也廣受海外關注，成為台灣擁有強大海外號召力的藝人。

===中国大陆===
2002年3月8日，[[國家廣電總局|國家廣電總局]]有關部門已下電文到各省廣電廳，要求立即暫停已經播出6集（即三分一內容，全18集）的台灣青春偶像劇《流星花園》。停播的原因是：《流星花園》播出後，在社會上產生了很大的負面影響，易誤導青少年，一些中小学生效法F4，在校园打骂同学、调戏女生、集体逃课、为非作歹，引起社会的强烈反响。<ref>{{cite web | language =zh-cn  | publisher = 新浪網 | title = 廣電總局：《流星花園》暫播 | url = http://ent.sina.com.cn/m/2002-03-11/75404.html | author = 新華網 | date = 2002-03-11 | accessdate = 2002-03-11 }}</ref>早在停播指令下達前，「流」劇已在青少年和成年人中引起了很大的爭議。《流星花園》在中國掀起的熱潮和中國禁播《流星花園》的做法引起美國《華盛頓郵報》、美國《時代週刊》和英國《BBC全球新聞》等國際媒體的關注和報導。F4、《流星花園》分別成為百度搜索引擎的2002年度「十大熱門關鍵詞」「明星類」和「劇集類」排行榜之冠軍，憑絕大比數的搜索量獲勝。

而在2009年湖南衛視热播的《[[一起來看流星雨|一起來看流星雨]]》，但媒體報道該劇並未取得[[集英社|集英社]]與[[神尾葉子|神尾葉子]]的《[[花樣男子|花樣男子]]》以及台湾的《[[流星花園|流星花園]]》的翻拍版權而自行制作，<ref>[http://post.yule.tom.com/3C000912736.html 集英社拒绝再授权 《流星雨》可能变山寨版(图)。] {{webarchive|url=https://web.archive.org/web/20090825153510/http://post.yule.tom.com/3C000912736.html |date=2009-08-25 }}TOM 娛樂  2009年2月16日</ref>但卻冠以「中國版」《[[流星花園|流星花園]]》作宣傳，因而也被媒體及觀眾稱為「山寨版」《[[流星花園|流星花園]]》。<ref>[http://ent.sina.com.cn/v/m/2009-02-18/13012380342.shtml 中國版《流星花园》新人演“杉菜”] 新浪娛樂 2009年2月18日</ref>其中，“F4”在《流星雨》也赫然变成为“H4”。

于2017年，柴智屏宣布《流星花园》将翻拍中国大陆版并担任其监制人。<ref>[http://dsj.sarft.gov.cn/tims/site/views/applications/note/noteTable.shanty?id=01572707e16f4c444028e4a156559172 2017电视剧拍摄制作备案公示表] </ref><ref>[http://ent.ifeng.com/a/20170621/42950054_0.shtml 《流星花园》招募宣传片曝光 寻找F4全球计划启动] 凤凰娱乐 2017年06月21日</ref>并于2018年7月9日登陆[[湖南卫视|湖南卫视]][[青春进行时_(湖南卫视)|青春进行时]]全国首播。而在杀青宣传前，微博话题瞬间火爆。这是继《[[泡沫之夏_(台灣電視劇)|泡沫之夏]]》后又一部经典台剧被翻拍（前者在[[浙江卫视|浙江卫视]]播出，后者在[[湖南卫视|湖南卫视]]播出）。

于2018年4月16日，台湾原版的《流星花园》第一季的高清修复版在[[腾讯视频|腾讯视频]]重新上线，第二季于同年5月7日重新上线，由[[国家广播电视总局|国家广播电视总局]]自行统筹，此次上线，删减了部分不俗内容。然而在劇集重播後，雖然一星期內已獲得1.6億的點擊率，但是不少[[中华人民共和国|中國]]觀眾忽然發現原來當年紅遍[[亞洲|亞洲]]的《流星花園》其實並不好看，因為當中的情節後來被[[東亞|東亞]]各國的劇集不斷重複使用而使觀眾變得厭倦，只是當年觀眾太單純未能察覺。而其中[[徐熙媛|徐熙媛]]所飾演的角色「杉菜」也因在兩個追求者（[[言承旭|言承旭]]飾演的「道明寺」和[[周渝民|周渝民]]飾演的「花澤類」）之間搖擺不定而被批評為「[[綠茶婊|綠茶婊]]」，及後[[徐熙媛|徐熙媛]]在同年4月25日發[[微博|微博]]回應：「是的，我承認，當初演到杉菜舉棋不定的時候，我跟蔡導（[[導演|導演]][[蔡岳勳|蔡岳勳]]）說：杉菜好賤喔！我討厭她，不會演。總之，我演到一半就不喜歡杉菜了！但還是盡量演到不讓觀眾太恨她！我承認，杉菜是個綠茶⋯但我還是很努力的美化了她。」<ref>{{Cite web |url = https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/181898/%E6%B5%81%E6%98%9F%E8%8A%B1%E5%9C%92-%E9%87%8D%E6%92%AD-%E6%9D%89%E8%8F%9C-%E6%83%B9%E7%88%AD%E8%AD%B0-%E5%A4%A7s-%E6%88%91%E6%89%BF%E8%AA%8D%E6%9D%89%E8%8F%9C%E6%98%AF%E5%80%8B%E7%B6%A0%E8%8C%B6|title = 《流星花園》重播「杉菜」惹爭議 　大S：我承認杉菜是個綠茶|date = 2018年4月25日|publisher = 香港01}}</ref>另外，劇集播出後[[中國大陸|大陸]]互聯網瘋傳三段劇集片段，其中「杉菜」共有四次無故地發出高音尖叫聲，被網友戲稱「杉菜是屬鴨子嗎？」<ref>{{Cite web |url = https://www.bilibili.com/video/av22351206?from=search&seid=126800529931545665|title = 台版《流星花园》-杉菜是属鸭子的吗系列 |date = 2018年4月21日|publisher = bilibili}}</ref><ref>{{Cite web |url = https://star.ettoday.net/news/1155443|title = 杉菜是屬鴨子嗎？　台版《流星》大S飆高音…言承旭嚇到倒彈 |date = 2018年4月22日|publisher = ETtoday星光雲}}</ref>。

===其他国家/地区===
*[[台灣|台灣]]－憑著超旺的人氣，《流星花園》電視原聲大碟在逆市創下30萬銷量。
*[[香港|香港]]－2002年年初於[[香港有線電視|香港有線電視]]播放，2002年8月5日，[[亞洲電視|亞洲電視]]也趁F4熱潮也買下版權，並於8月30日起逢星期一至五晚上8:30[[本港台|本港台]]播放，之後不時將該劇重播，包括在暑假的夜深時段重播。2009年4月，亞洲電視突然腰斬一直在黃金時段播放的台灣劇集《[[意難忘|意難忘]]》，並倉促地將《流星花園》於4月27日起逢星期一至五晚上7:00在本港台重播。
*[[菲律賓|菲律賓]]－《流星花園》至今仍是菲律賓最受歡迎的電視劇，並創下最高57.4%、平均42.9%的收視率，2003年主角言承旭赴菲參加廣告記者會，受到菲律賓總統[[阿羅約|阿羅約]]的邀請會面，全程以皇家級禮遇接待。
*[[日本|日本]]－截至2005年，全日本有15家電視台播出「流星花園」。{{Fact|日本掀起一股學中文的熱潮。|time=2019-03-03T23:06:29+00:00}}
*[[韓國|韓國]]－2002年，韓國一些非官方F4歌迷會（咖啡廳）的會員總數在成立一周內突破20多萬名，促使韓國MBC電視台（韓國三陟文化電視台）重播《流星花園》的決定。結果凌晨時段重播的《流星花園》突破收視紀錄。韓國掀起一股學中文的熱潮。
*[[印尼|印尼]]－《流星花園》是印尼自有電視台以來，收視最高的一部劇集，劇中演員的知名度遍布印尼，印尼皇室也曾發函提出表演邀約。印尼政府公開在媒體上呼籲，希望F4能到峇里島開演唱會輓救當地經濟，以F4的號召力吸引更多雅加達歌迷到峇里島一遊，振興低迷不振的觀光業。當時F4已安排去雅加達開唱，檔期密集未能赴邀。新加坡《聯合早報》報導《流星花園》帶動的中文熱潮在過去不鼓勵學中文的印尼發酵，印尼當地學中文的非華人學生愈來愈多。
*[[泰國|泰國]]－2003年，泰國《[[曼谷郵報|曼谷郵報]]》報導《流星花園》播出兩個月後，當地的F4唱片和產品已經賣出1億[[泰銖|泰銖]]，形容F4為亞洲最速銷的明星。媒體報導不少泰國人爭相學習中文，主要的原因是《流星花園》和F4帶起的風潮。
