'''鼬獾'''（學名：''Melogale moschata''），是[[鼬獾屬|鼬獾屬]]四種動物的一種，俗名：撥田豬、小豚貓、田螺狗、鰗鰍貓<ref>[http://gis.ymsnp.gov.tw/nature/sys_guest/Main_AL_frm.cfm?TAB_CLASS=AL&etab=DB_A_Mammal 鼬獾] {{webarchive|url=https://web.archive.org/web/20160304172212/http://gis.ymsnp.gov.tw/nature/sys_guest/main_al_frm.cfm?etab=db_a_mammal&tab_class=al |date=2016-03-04 }}，陽明山國家公園，自然資源資料庫</ref>，[[台語|台語]]稱「臭羶-{猫}-」（[[白話字|白話字]]：tshàu-hiàn<ref>
http://twblg.dict.edu.tw/holodict_new/result_detail.jsp?n_no=6645&curpage=1&sample=%E8%87%AD&radiobutton=1&querytarget=1&limit=20&pagenum=3&rowcount=47</ref>-bâ<ref>http://twblg.dict.edu.tw/holodict_new/result.jsp?radiobutton=1&limit=20&querytarget=1&sample=%E7%8C%AB&submit.x=54&submit.y=22</ref><ref>http://twblg.dict.edu.tw/holodict_new/result_detail.jsp?n_no=6312&curpage=1&sample=%E7%8C%AB&radiobutton=1&querytarget=1&limit=20&pagenum=1&rowcount=3</ref>）。

== 外型特徵 == 
頭體長35～40公分，尾長14～20公分，體重1～1.75公斤。臉部呈黑褐色，由頭頂經後頸至背中央有一白色縱帶，額頭至眼睛周圍有明顯的白毛，神似國劇人物之化妝臉譜，故有「花臉貍」之稱。吻端突出似豬鼻，又有「小豚貓」之稱。全身披深灰褐色粗毛，身體瘦長，四肢細短呈污灰色，爪尖銳而長，尾部具白色長毛，略為蓬鬆。鼠蹊部有味道濃厚之臭腺，故又有「臭貍」之稱。<br />齒式：門齒3/3，犬齒1/1，前臼齒4/4，臼齒1/2；總齒數= 38。<ref name="nrch.culture.tw">[http://nrch.culture.tw/twpedia.aspx?id=7072 「臺灣大百科全書」網站 http://taiwanpedia.culture.tw/]</ref>

== 生態習性 ==
白天躲藏於樹洞、土洞或岩洞內休息，日落黃昏後始外出覓食。有爬樹能力，但不常上樹活動；行動緩慢，不善跳躍。主要以嗅覺找尋食物，但聽力與觸覺亦佳，喜好捕食蜥蜴、鳥類、小型囓齒類，對於蝸牛、蚯蚓、大型昆蟲等無脊椎動物也來者不拒，有時亦會取食植物果實。排遺呈黑色細長螺旋狀，是明顯易辨的跡相。由於臭腺特別發達，受驚嚇或被逼迫時亦會分泌具惡臭之氣味以驅敵。每年5～6月為其生殖高峰期，通常每胎產1～3隻幼獸。<ref name="nrch.culture.tw"/>

== 棲地分布 ==
原生地不詳，主要棲息於[[中國大陸|中國大陸]]長江以南地區、中南半島東側、[[印度|印度]]東北部、[[臺灣|臺灣]]本島、[[爪哇島|爪哇島]]、和[[婆羅洲|婆羅洲]]的綠地。
台灣特有亞種普遍分布於低、中海拔山區，通常棲居於原始闊葉林、灌叢區及開墾地。分布海拔最高可達2,100公尺。<ref name="nrch.culture.tw"/>

== 狂犬病 ==
正如大部分哺乳類，鼬獾也可能成為[[狂犬病|狂犬病]]宿主。

2012年五、六月在[[台灣|台灣]]的[[雲林|雲林]][[古坑鄉|古坑鄉]]、[[南投縣|南投縣]][[魚池鄉_(臺灣)|魚池鄉]]及[[鹿谷鄉|鹿谷鄉]]等地發現的鼬獾屍體經檢驗後，於2013年六月檢驗證實有[[狂犬病|狂犬病]]，經通報[[世界動物衛生組織|世界動物衛生組織]]（OIE）後，台灣自非疫區除名。<ref>[http://www.oie.int/wahis_2/public/wahid.php/Reviewreport/Review?page_refer=MapFullEventReport&reportid=13775&newlang=en OIE通報紀錄]</ref>

2013年七月22日，[[台東縣|台東縣]][[東河鄉_(臺灣)|東河鄉]]出現第一起狂犬病鼬獾咬人案例<ref>[http://udn.com/NEWS/NATIONAL/NATS2/8048287.shtml 聯合新聞網：台灣第一起！ 狂犬病鼬獾咬傷人] {{webarchive|url=https://web.archive.org/web/20130725055050/http://udn.com/NEWS/NATIONAL/NATS2/8048287.shtml |date=2013-07-25 }}</ref>。被咬的民眾將該鼬獾送驗，檢驗已證實該鼬獾帶有狂犬病病毒。因此該民眾隨即接受[[免疫球蛋白|免疫球蛋白]]及[[疫苗|疫苗]]的注射<ref>[http://www.appledaily.com.tw/realtimenews/article/life/ 蘋果日報：【短片】被鼬獾咬傷家犬　猛流口水遭隔離]</ref>

遭感染並發病之鼬獾，會有異於其生態習性的行為產生，需特別注意。
 2013/07/28 [[南投縣|南投縣]][[仁愛鄉|仁愛鄉]]，鼬獾於白日闖進人類住所，並攻擊咬傷人<ref>[http://news.cts.com.tw/cts/general/201307/201307301285888.htmlm 咬童鼬獾染狂犬病 9歲女急投藥]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
 2013/07/29 [[台南市|台南市]][[南化區|南化區]]，鼬獾於白日離開山區，闖入人類住所，並與飼養之家犬搏鬥<ref>[http://news.cts.com.tw/cts/general/201307/201307281284746.html 鼬獾闖南化民宅咬狗 確染狂犬病]</ref>
