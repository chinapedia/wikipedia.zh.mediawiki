'''惠力寺'''俗称'''西寺'''，为位于[[中国|中国]][[浙江省|浙江省]][[嘉兴|嘉兴]][[海宁市|海宁市]][[硖石街道|硖石街道]]紫薇山（俗称[[西山_(海宁)|西山]]）南麓的一处佛寺，为东晋[[宁康|宁康]]年间尚书[[张延光|张延光]]舍宅为寺，初名志愿寺，北宋[[大中祥符|大中祥符]]二年（1009）赐额惠力寺<ref>清·许三礼修，《康熙海宁县志·卷三·建置下》：惠力寺在县东北六十里硖石紫薇山下，东晋宁康中尚书张延光舍宅为寺，初赐额曰志愿，俗名西寺。唐肃宗乾元元年勑支本县税钱重加修饰，唐季兵燬，宋太祖乾德二年甲子营田将吴仁绶等捐资建复，太宗雍熙二年乙酉告竣，大中祥符二年己酉勑赐名惠力寺。……明洪武二十四年立成丛林。</ref>。建筑屡建屡毁，今除经幢和大雄宝殿外均为近年重建。大雄宝殿建于民国十一年（1922），面阔五间，重檐歇山顶，明间梁架为抬梁式，次间为穿斗式，屋脊均堆塑山水<ref>{{cite web |url=http://www.zjww.gov.cn/news/2012-09-03/604881774.shtml|title=《海宁市惠力寺修缮保护工程正式启动》|accessdate=2014-09-20|work=董月明、王伟|publisher=浙江省文物局}}</ref>。

惠力寺经幢位于惠力寺前，共有东西两座，建于唐[[咸通|咸通]]十五年（乾符元年，874）<ref>清·[[李圭|李圭]]修，《民国海宁州志稿·卷八·名迹》：石经幢，邑中唐石幢凡五，皆刻尊胜陀罗尼经，……其在硖石惠力寺者二，俱咸通十五年五月俞宗厚书。</ref>，东幢刻有造幢题记。1956年曾公布为[[浙江省文物保护单位|浙江省文物保护单位]]，1982年2月公布为县级文物保护单位，1997年8月重新列入浙江省文物保护单位，2013年5月列入[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]。经幢高约5米，平面呈八边形，下为[[须弥座|须弥座]]，[[束腰|束腰]]处浮雕盘龙，柱身上刻[[俞宗厚|俞宗厚]]所书《[[佛顶尊胜陀罗尼经|佛顶尊胜陀罗尼经]]》，幢身上覆[[华盖|华盖]]、连珠、盘石、二重连珠、莲花宝顶相叠，盘石底部有[[飞天|飞天]]浮雕<ref>{{cite web |url=http://www.zjww.gov.cn/unit/2006-02-23/53531814.shtml|title=《惠力寺经幢》|accessdate=2014-10-11|work=|publisher=浙江省文物局}}</ref>。
