'''惠施網'''（{{lang-en|'''Wise Giving'''}}）由[[香港社會服務聯會|香港社會服務聯會]]（簡稱社聯）於2007年成立，目的是推動及提高[[慈善機構|慈善機構]]的問責性和透明度，對象除社聯會員機構外，亦包括[[香港|香港]]所有獲[[稅務局_(香港)|稅務局]]免稅地位的慈善機構。參與的慈善機構必須在網頁內提供管治及財務等資訊，讓公眾參考<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2011/09/26/ED1109260035.htm | title=概念鏈接 |publisher=[[文匯報|文匯報]] |date=2011-09-26 |accessdate=2012-02-29}}</ref>。

==運作方式==
在《惠施網》表列的慈善機構必須提供機構簡介、服務實踐及報告、財務及管治等資訊。這網站根據機構提交的文件，透過電話訪問、文件檢查、會面或實地探訪，來核實所填報資料的真確性。如資料準確及齊全，一般處理需時三至四星期。

當完成核實程序後，相關資訊以統一格式上載於這網站，以便捐款人參考<ref>{{cite web |language=zh-hk |url=http://orientaldaily.on.cc/cnt/news/20110108/00176_044.html | title=慈善機構募捐無王管 |publisher=[[東方日報|東方日報]] |date=2011-01-08 |accessdate=2012-02-29}}</ref>。

捐款人可以透過這網站查出大部份慈善機構的資料。例如是否稅務局認可的慈善機構，某機構進行賣旗的日期，有沒有牌照等資訊。這網站還指導捐款人如何監察善款，甚至申請基金，如某慈善機構還未提交財務報表，稅務局有權通過此網站追索。根據該網站的慈善問責準則，捐款人必須了解慈善機構的監察管治，細閱組織周年報告中向公眾公開的財政、服務成果、服務質素檢討、籌款方式、董事會成員、主事人員的值勤表等資料，以檢視其透明度<ref>{{cite web |language=zh-hk |url=http://www1.hk.apple.nextmedia.com/template/apple_sub/art_main.php?iss_id=20100507&sec_id=12187379&art_id=14003839 | title=捐款要問責　愛的工業網上起底 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2010-05-07 |accessdate=2012-02-29}}</ref>。

==明施慎選==
{{main|明施慎選}}
香港另一監察網站《[[明施慎選|明施慎選]]》，這網站從《惠施網》隨機抽出香港註冊的慈善機構，再按其籌款成本、項目支出、效益、職員薪酬及行政費用等，計算它們的營運及善款效益，以監察團體的營運狀況及透明度，供捐款人參考<ref>{{cite web |language=zh-hk |url=http://www1.hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20110108&sec_id=4104&subsec_id=12731&art_id=14847578 | title=民間組織：慈善機構營運參差 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2011-01-08 |accessdate=2012-02-29}}</ref>，已完成審核的慈善機構包括「婦女動力基金」、「[[龍耳|龍耳]]」和「關懷愛滋」等<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20111023&sec_id=4104&subsec_id=11867&art_id=15731432 | title=蘋果汁 監察慈善組織　網上公開核數報告 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2011-10-23 |accessdate=2012-02-29}}</ref>。
