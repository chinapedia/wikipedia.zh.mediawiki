'''內政部警政署國道公路警察局'''，簡稱'''國道公路警察局'''、'''公路警察局'''、'''公警局'''，為[[中華民國內政部|中華民國內政部]][[警政署|警政署]]附屬[[警察|警察]]機關，屬中央政府四級機關，職司維護[[中華民國國道|中華民國國道]]高速公路及經指定之部分[[快速道路|快速道路]]、[[臺灣省道|臺灣省道]]交通秩序和治安維護工作。

==機關沿革==
[[File:ROC-NHPB_BMW_3Series_9791-K8_2014-10-03.jpg|thumb]]
國道公路警察局之前身為公路警察，設立於1945年，當時屬[[駐衛警察|駐衛警察]]形態，隸屬於[[台灣省公路局|台灣省公路局]]。1950年改為駐衛警察隊，1952年改制成'''台灣省公路警察隊'''，1970年再擴編為'''台灣省公路警察大隊'''。

1978年10月31日[[中山高速公路|中山高速公路]]全線完工通車。1979年3月1日'''內政部警政署公路警察局'''成立，同時管理台灣省公路警察大隊業務。

後又修編組織條例，於1993年1月7日經[[立法院|立法院]][[三讀通過|三讀通過]]，同年2月5日奉[[中華民國總統|總統令]]公布，正式定名為'''內政部警政署國道公路警察局'''。

1995年4月內政部警政署與[[臺灣省政府警務處|臺灣省政府警務處]]分開辦公，台灣省公路警察大隊於同年12月1日遷往[[台中市|台中市]]，其後此局即專責辦理國道交通與治安維護任務。

1999年7月1日配合辦理[[精省|台灣省政府功能業務與組織調整案]]，整併台灣省公路警察大隊，負責國道、[[快速道路|快速道路]]及經指定之部分[[省道|省道]]交通與治安維護任務迄今。

==業務執掌==
依《內政部警政署組織法》第6條及《內政部警政署國道公路警察局組織規程》第2條規定，國道公路警察局掌理下列事項：
# 交通秩序及道路設施（含橋樑、隧道）之安全維護。
# 違反公路交通管理事件之稽查取締。
# 行車事故之處理。
# 行車旅客、貨運之安全維護。
# 國道公路路權範圍內違法案件之偵防與處理及違章事件之協助處理。
# 收費站、地磅、服務區與休息站等之交通秩序維護及稽查取締。
# 經指定之快速公路行車安全秩序之維護。
# 其他有關協助國道公路法令推行及警察業務之規劃、督導等事項。
