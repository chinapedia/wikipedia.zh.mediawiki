'''內惟黃正公祠'''，是位於[[高雄市|高雄市]][[鼓山區|鼓山區]]的廟宇，以前的雙頭塭堵<ref>{{Cite book |title=高雄市地名探源(增訂版) 作者:曾玉昆;高雄市文獻委員會 編印 2004年12月再版(平裝)| isbn= 957-00-9912-7 }}</ref><ref>《高雄市地名探源(增訂版)》【第七三頁第七行起】：塭：臺灣傍海圍築以畜魚者曰「塭」，即國字的「堰」，壅水為壕叫堰，內地無魚堰，惟臺始有之。雙頭塭:位於鼓山區的平和、厚生、民族三里一帶，當年引內圍埤之水養殖，亦兼灌溉之用，曾建雙頭之水閘一座名曰「雙頭塭堵」，供調節水量之用，水閘附近地區的埤潭乃以雙頭塭名之。</ref><ref name="高雄市地名探源原文部份書影">{{Cite web|author=作者：曾玉昆|url=https://commons.wikimedia.org/wiki/File%3A%E9%9B%99%E9%A0%AD%E5%A1%AD%E8%88%87%E6%BA%AB%E5%A0%B5%E7%9A%84%E7%94%B1%E4%BE%86.png|title=雙頭塭與塭堵來由出處-原文段落書影|deadurl=yes|archiveurl=https://web.archive.org/web/20140810110205/https://commons.wikimedia.org/wiki/File%3A%E9%9B%99%E9%A0%AD%E5%A1%AD%E8%88%87%E6%BA%AB%E5%A0%B5%E7%9A%84%E7%94%B1%E4%BE%86.png|archivedate=2014-08-10}}</ref> 旁，舊址在現在[[高雄市|高雄市]][[鼓山區|鼓山區]][[迪化街|迪化街]]90號<ref>{{Cite web|url=http://www.templevisit.org/%E5%8F%B0%E7%81%A3%E5%AF%BA%E5%BB%9F%E9%80%9A%E8%A8%8A%E9%8C%84%5C%E9%AB%98%E9%9B%84%E5%B8%82%E9%BC%93%E5%B1%B1%E5%8D%80.htm|title=【高雄市鼓山區】寺廟一覽表|language=zh-tw|publisher=古今寺廟巡禮|author=張慧意|date=|accessdate=2014-07-23}}</ref>西邊街口的路燈下。被地方上認為是塭堵的守護神。屬於個人靈骨遺骸<ref>內含頭蓋骨與全身肢體骸骨</ref>的祭祀祠，古時候的埤塘易於發生意外也同時因為取水灌溉及養殖的作用，往往都會在週遭附近附帶一個屬於埤塘信仰之圖騰記號或神祠。

== 概述 ==
現在正名為玉旨龍聖宮，副名內惟雙頭塭堵黃正公祠，原主祀黃正公旁有奉祀福德正神，相對內惟埤而言，本祠座落在人口稠密區，故持續地發展成具有實用功能性的廟宇，也因此近年來加入常見的民間信仰神尊（例如[[濟公活佛|濟公活佛]]等）在旁奉祀。

黃正公是從前清時代到日治時代雙頭塭堵地方上的守護神，雙頭塭堵在清朝時屬鳳山縣興隆里內惟庄，日治時代屬高雄州高雄市雙頭塭堵，民國時代則在鼓山區平和里行政區內。以前在九如國小旁的農田圳溝旁的鐵條工廠廠區圍牆內一小角落，小祠的面向是向工廠內，後來因為配合都市計畫將九如國小校區規劃完整的行人道，重新購地建廟，所以從現在街口路燈的位置往旁東北遷移約50公尺，位在[[高雄市|高雄市]][[鼓山區|鼓山區]][[迪化街|迪化街]]90號。有一個方正整齊的廟區基地。

== 歷史 ==

根據建廟碑記載，相傳在前清年間，[[壽山|壽山]]山麓，土名龍園頭<ref>參考古今相疊圖上，應該是[[高雄市鼓山區內惟國民小學|內惟國小]]與[[高雄市|高雄市]][[龍泉禪寺|龍泉禪寺]]村莊前前一帶</ref>，妖氣熾盛<ref>以今日的看法是：此地帶當年，濕地密林區多微生物產生的沼氣，容易產生中毒幻覺甚至死亡的症狀</ref><ref>另有一說，此陂注入硫磺泉,故有硫磺氣,古之傳說信有可考。不外乎是瓦斯中毒現象。按：《鳳山縣采訪冊》〈丙部〉 【地輿（三）】硫磺陂（亦名下陂，與內圍陂相連，僅隔一峰，舊志誤以內圍陂合併在內，故有上下之稱），在興隆里，縣西十四里，周長六里許，源受內圍陂穿岸入水，東南行里許，下注龍水港，溉田四十二甲。施仁陂（俗名鹹草堰仔，即雙頭塭古稱），在興隆里，縣西十五里，周里許，源受龍巖池、硫磺水（按此水源出打鼓山麓，味似硫磺，故名）二流，東南行三里許入陂，又東南行過施仁橋，下注龍水港，溉田二十六甲。</ref>，村民無不恐懼戒慎。有一李姓道士，從龍園頭出發，途中步行時感到似乎不適，但仍勉力而行，走到雙頭塭堵一帶時忽見綠竹倒下，甚為驚恐，於是昏迷不醒人事，在昏迷之中，朦朧聽到彷彿有人斥責曰：「吾為黃正，此乃吾境，不可傷之！」爾後道士甦醒，四處尋覓，在一週<ref>第七天，類似人死亡後的頭七天數，更添巧合神秘感而已</ref>之後發現一骷髏頭骨，知其顯靈，解救其性命，於是就近建一小塔以供奉，村民不知神尊之名稱，皆以「頭骨神」尊稱之。嗣後村民經由民間習俗擲筊方式查悉，此頭骨乃威武顯赫，正氣浩然之黃正公生前之靈骨，於是乎闔境競相奉祀，以其靈異無比，有求必應，區區小廟，有靈則名，香火鼎盛，迄茲百餘年<ref>此段文字採自民國八十年第二次廟宇重建的碑文所述</ref>，從未稍衰。至今其靈骨依然奉在廟中<ref>就放在神尊塑像座下方的金甕裡</ref>。從碑文落成時間往回推算，此廟初次奉塔時間應該就在西元1880年代，也就是清朝要割讓台灣前的十餘年間。據地方上的工作人員描述，此廟雖然面積不大，但是算是附近正德里與平和里間最早的奉祀廟宇。也就是舊時雙頭塭堵的首間廟祠。
