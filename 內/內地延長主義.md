'''內地延長主義'''（{{lang-ja|ないちえんちょうしゅぎ}}）為[[台灣日治時期|台灣日治時期]]，[[大日本帝國|大日本帝國]]在台灣所施行的的一種{{fact|[[法國|法國]]型態}}的殖民統治政策。該項殖民統治政策主張將包括台灣在內的殖民地以「雖與[[內地|內地]]有稍許不同，但視為內地的延伸，直接適用本國法律，以之為本國領土來統治<ref name="TWSDUT5U">『圖解臺灣史』（2012年）第114頁</ref>」此一方針做為基本政策。

== 「六三法」實行時期的台灣 ==
1896年3月30日所公佈的「[[六三法|六三法]]」（{{lang-ja|「臺灣ニ施行スヘキ法令ニ關スル法律」明治29年法律第63号}}），授予[[台灣總督|台灣總督]]發布具有法律的效力之命令的特權<ref name="CVSL">『蔣渭水VS林獻堂』（2015年）第118頁</ref>，於是台灣總督擁有了本該由[[帝國議會_(日本)|帝國議會]]才有的「立法權」，以及台灣最高行政長官本就有的「行政權」。同時[[台灣總督府|台灣總督府]]根據「六三法」所發布的「台灣總督府法院條例」中規定，台灣總督擁有「法院」的管理權及人事權。如此一來，台灣總督在台灣等於同時握有行政、立法、司法三權。「六三法」為台灣總督在台灣的「特別統治」提供了法律基礎<ref name="CVSL"></ref>。

== 「民族自決」的潮流 ==
[[第一次世界大戰|第一次世界大戰]]時期，[[美國總統|美國總統]][[伍德羅·威爾遜|伍德羅·威爾遜]]於1918年1月8日提出[[十四點和平原則|十四點和平原則]]<ref>『蔣渭水VS林獻堂』（2015年）第166頁</ref>，其中的第五條「平等對待殖民地人民」裡提出了「民族自決」此一概念，而在一戰後多數殖民地獲得獨立，更進一步推動民族自決、民主主義的潮流，連日本也不例外。

== 原敬與「内地延長主義」 ==
[[原敬|原敬]]在[[甲午戰爭|甲午戰爭]]後以外務次官的資格擔任台灣事務局委員，參與台灣統治制度的規劃<ref name="TWSDUT5U"></ref>，自此便主張「台灣應實施與內地相同的制度、方針，做為統治台灣的基本政策，但應以漸進的方式逐步實行<ref>『臺灣法律現代化歷程：從「內地延長」到「自主繼受」』（2015年）第28頁</ref>」這樣的論點。[[原敬|原敬]]擔任首相的時期，正好是國際上風行民族自決的潮流，而國內「[[大正民主|大正民主]]」的蓬勃發展，又恰好任內發生朝鮮[[三一運動|三一運動]]，軍方聲勢受挫的時期。

因此在1918年8月，在朝鮮[[三一運動|三一運動]]後對[[台灣總督府|台灣總督府]]、[[朝鮮總督府|朝鮮總督府]]進行官制上的改革，去除了總督必須由武官擔任此一限制<ref name="TWSDUT5U"></ref><ref>『日本文明開化史略』（2015年）第299頁</ref>。第一位由文官出身的台灣總督，[[田健治郎|田健治郎]]於1919年10月29日上任<ref name="TWSDUT5U"></ref>。

== 台灣總督田健治郎與「內地延長主義」 ==
[[田健治郎|田健治郎]]總督為反制民族自決的潮流，實行「内地延長主義」，同時提出了漸進的具體方針，如「日台融合」、「一視同仁」<ref name="TWSDUT5U"></ref>。基於這些方針，[[台灣總督府|台灣總督府]]於1920年開始進行地方制度的改革，創設州、市、街、莊等地方官選議會。1921年2月設置[[台灣總督府評議會|台灣總督府評議會]]。1922年1月，取代「[[三一法|三一法]]」（{{lang-ja|明治40年第31号法律}}）的「[[法三號|法三號]]」（{{lang-ja|大正10年法第3号}}）開始實施。1923年，開始在台灣實施「[[治安警察法|治安警察法]]」。除此之外，還公佈了台灣人官吏特別任用令、廢除台灣人入學限制與日本人共同上學（[[日台共學制度|日台共學制度]]）、認可台灣人與日本人的婚姻（日台共婚法）<ref name="TWSDUT5U"></ref>。

== 內地延長主義對台灣的影響 ==
内地延長主義是台灣為[[大日本帝國|大日本帝國]]殖民地時所推行的政策，造成最重要的影響是讓台灣人追求和日本處在相同的憲法體制下，將台灣納入日本的法治體系。從1914年開始，東京的台灣留学生，如[[林獻堂|林獻堂]]、[[蔡培火|蔡培火]]、[[林呈祿|林呈祿]]、[[蔡式穀|蔡式穀]]、[[鄭松筠|鄭松筠]]等人，為享有憲法中明定如代議制、基本保障等權利，而開始進行「[[六三法撤廢運動|六三法撤廢運動]]<ref>{{cite journal |author=余重信 |year=2002 | title=日治時期六三法制對臺灣法治政治的影響 | language=zh-tw | journal=[[國立臺灣師範大學|國立臺灣師範大學]]三民主義研究所碩士論文}}</ref>」，反對「特別立法統治」。

然而這無異於變相承認了由「同化政策」延伸而來的「內地延長主義」，否定台灣的特殊性，乃至於台灣獨特的歴史、文化、思想、傳統。因此參與「[[六三法撤廢運動|六三法撤廢運動]]」的台灣留學生，如[[林呈祿|林呈祿]]等人，開始由原先的單純主張撤去「[[六三法|六三法]]」，轉變為要求憲政與民權，同時保持台灣的特殊性、變更「六三法」的内容及台灣自治<ref>{{cite journal |author=蘇恆欽 |year=2003 |title=治警事件探討 |language=zh-tw |journal=[[國立臺南大學|國立臺南師範學院]]臺灣文化研究所碩士論文}}</ref>。[[林呈祿|林呈祿]]等人的要求，在1921年正式以「[[臺灣議會設置請願運動|臺灣議會設置請願運動]]」為名開始運作。

== 同化政策與內地延長主義 ==
[[同化|同化政策]]是[[殖民主義|殖民主義]]中關於殖民地統治方針的一種理論，該理論相信精神、文化甚至人種可互相同化<ref name="蔡錦堂">{{cite journal |author=蔡錦堂 |year=2002 | title=日本治台時期所謂「同化政策」的實像與虛像初探 | language=zh-tw | journal=[[淡江大學|淡江大學]]《淡江史學》第13期}}</ref>。內地延長主義這項[[法國|法國]]型態的殖民政策則為漸進的[[同化|同化政策]]，該主義在於先讓法規、制度上與殖民母國達到一致，以進一步達到精神及文化方面的同化<ref name="蔡錦堂"></ref>。

在近代歷史中，[[俄罗斯帝国|俄罗斯帝国]]統治的[[西伯利亚|西伯利亚]]、[[波罗的海国家|波罗的海国家]]、[[烏克蘭|烏克蘭]]、[[中亞|中亞]]、[[波蘭|波蘭]]、[[高加索|高加索]]、[[東普魯士|東普魯士]]、[[卡累利阿|卡累利阿]]、[[外東北|外東北]]；[[大日本帝国|大日本帝国]]對[[阿伊努人|阿伊努人]]、[[鄂罗克人|鄂罗克人]]、[[尼夫赫人|尼夫赫人]]少數民族，及統治[[北海道|北海道]]、[[琉球|琉球]]、[[朝鲜|朝鲜]]與[[台灣|台灣]]後展開的[[皇民化運動|皇民化運動]]，這些都為[[同化|同化政策]]的最明顯例子。

藉由[[同化|同化政策]]衍生的內地延長主義，則是將北海道、琉球、朝鲜與台灣視為日本內地的延長，套用本國法規與制度，最終目的則與[[同化|同化政策]]相同，在於「使殖民地眾完全成為日本之臣民，效忠[[日本政府|日本政府]]，加以教化善導，以涵養其對國家之義務觀念<ref>[[吳三連|吳三連]]、[[蔡培火|蔡培火]]（1971年）《台灣民族運動史》</ref>」。
