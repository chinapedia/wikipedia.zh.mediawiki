'''內本鹿越嶺古道'''為一條位在[[臺灣|臺灣]]的山區古道，該條古道橫跨於[[高雄市|高雄市]][[六龜區|六龜區]]經[[中央山脈|中央山脈]]至[[臺東縣|臺東縣]][[延平鄉|延平鄉]][[紅葉村_(台東縣)|紅葉]]，總長124.7公里。<ref name="forest"/>目前古道西段[[六龜區|六龜]]至[[藤枝|藤枝]]現已闢為'''藤枝林道'''，東段[[鹿鳴|鹿鳴]]至[[清水_(花蓮縣卓溪鄉)|清水]]段改為'''東36鄉道'''，其餘路段皆處於荒廢狀態<ref name="forest">{{cite web |url=http://recreation.forest.gov.tw/Rt/RT_1_6_13.aspx |title=中華民國行政院農業委員會林務局-臺灣山林悠遊網 關山─內本鹿越嶺國家步道系統 | accessdate=2015-05-31}}</ref>。

== 歷史 ==
內本鹿越嶺古道最早之起源是19世紀時臺東山區的[[臺灣原住民族|原住民族]][[布農族|布農族]]郡社群（又稱施武郡群）沿鹿野溪上游向東越過[[內本鹿山|內本鹿山]]、[[出雲山|出雲山]]，再沿馬里山溪（[[濁口溪|濁口溪]]上游）向高雄市擴散定居的山徑<ref>{{cite web |url=http://rukai.loxa.edu.tw/menu/line.htm |title=魯凱對外古道 內本鹿越嶺道 |accessdate=2015-05-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20040911015933/http://rukai.loxa.edu.tw/menu/line.htm |archivedate=2004-09-11 }}</ref>。

=== 日治時期 ===
[[甲午戰爭|甲午戰爭]][[清廷|清廷]]戰敗後，臺灣割讓給[[日本|日本]]進入[[臺灣日治時期|臺灣日治時期]]，1914年，時任第5任[[臺灣總督|臺灣總督]]的[[佐久間左馬太|佐久間左馬太]]下令執行「[[五年理蕃計畫|五年計畫理蕃事業]]」來治理山地原住民族，而當時臺東縣內本鹿地區沿線共有mamahav、masuvanu、halimudun、halipusun、masudada、tansiki、asahi等多個部落，直到1940年間，布農族人仍不願遷至山下，也使得內本鹿地區成為日本臺灣理蕃地圖最後一片空白的區域，並有著「北泰雅、南布農」之稱號<ref name="原住民電視台 【尋找布農族 內本鹿越嶺古道】 ">{{cite web |url=http://web.pts.org.tw/titv/mealc/main.php?Channel=titv&XMAENO=1721&XMBENO=4413 |title=原住民電視台 【尋找布農族 內本鹿越嶺古道】 | accessdate=2015-05-31}}</ref><ref>{{cite web |url=http://www.newtaiwan.com.tw/bulletinview.jsp?bulletinid=12267 |title=新臺灣新聞周刊 內本鹿事件：歷史，凝鍊在戲劇詮釋中… |accessdate=2015-05-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20170324093244/http://www.newtaiwan.com.tw/bulletinview.jsp?bulletinid=12267 |archivedate=2017-03-24 }}</ref>。

在1935年（昭和10年）為管理山地布農族，[[臺灣總督府|臺灣總督府]]計畫開闢'''內本鹿越嶺道'''，並沿越嶺道設置有紅葉（umizi）、清水（siusui）、楓（kaili）、嘉嘉代（kakaiyu）、桃林（tavilin）、橘、（kudubuki）、壽（pisbadan）、常盤（tukiuwa）、朝日（asahi）、出雲（sakakivan）等多個[[駐在所|駐在所]]<ref name="原住民電視台 【尋找布農族 內本鹿越嶺古道】 "/>。

=== 發生「內本鹿事件」 ===
1941年3月9日，Halipusun部落的海樹兒（Hisul）因為不滿日本人強制實施[[集團移住|集團性的移住政策]]，造成該部落的族人在移居至延平鄉[[花東縱谷平原|縱谷平原區]]時，出現水土不服的現象而染病。因此海樹兒與其餘三名布農族男性展開報復行動，在清水駐在所前先把吊橋砍斷，阻斷日本警察上山援助的交通要道。海樹兒等三人後來來到清水駐在所並以槍枝殺死兩名日本警察，又在楓駐在所與日本警察進行交戰，所內三名日本警察也被殺死，後又誤殺一名日本孩童<ref name="原住民電視台 【尋找布農族 內本鹿越嶺古道】 "/><ref name="Yahoo新聞 被遺忘的戰役 布農族「內本鹿事件」">{{cite web |url=https://tw.news.yahoo.com/%E8%A2%AB%E9%81%BA%E5%BF%98%E7%9A%84%E6%88%B0%E5%BD%B9-%E5%B8%83%E8%BE%B2%E6%97%8F-%E5%85%A7%E6%9C%AC%E9%B9%BF%E4%BA%8B%E4%BB%B6-140404330.html |title=Yahoo新聞 被遺忘的戰役 布農族「內本鹿事件」 |accessdate=2015-05-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20160305212323/https://tw.news.yahoo.com/%E8%A2%AB%E9%81%BA%E5%BF%98%E7%9A%84%E6%88%B0%E5%BD%B9-%E5%B8%83%E8%BE%B2%E6%97%8F-%E5%85%A7%E6%9C%AC%E9%B9%BF%E4%BA%8B%E4%BB%B6-140404330.html |archivedate=2016-03-05 }}</ref><ref name="台東延平鄉影像誌 內本鹿腳行：跨越時空的呼喚 日警後裔重返內本鹿尋根 ">{{cite web |url=http://bununbiun.blogspot.tw/2009/11/blog-post.html |title=台東延平鄉影像誌 內本鹿腳行：跨越時空的呼喚 日警後裔重返內本鹿尋根 | accessdate=2015-05-31}}</ref>。

當地的日本軍駐軍發現後，立即前往電話線尚未被截斷的紅葉駐在所通報[[關山郡役所|關山郡役所]]增派警力支援，日本人為防止該事件衍生為群體性的戰爭，對海樹兒等三人進行圍捕。海樹兒三人開始逃亡，甚至其三位位在部落中的居所也被日本警察開槍掃射，至今彈痕斑斑的石板牆仍保存在內本鹿山區中。爾後海樹兒三人仍然逃不過大批警力的圍補，一個月後便遭到日本警方逮補而宣判死刑。臺灣總督府方面也為防止類似事件的發生，把居住在內本鹿沿線的布農族人全數遷居到今日臺東縣[[延平鄉|延平鄉]]的[[鸞山村|鸞山村]]與[[桃源村|桃源村]]<ref name="原住民電視台 【尋找布農族 內本鹿越嶺古道】 "/><ref name="Yahoo新聞 被遺忘的戰役 布農族「內本鹿事件」"/><ref name="台東延平鄉影像誌 內本鹿腳行：跨越時空的呼喚 日警後裔重返內本鹿尋根 "/>。

後續因內本鹿山區已沒有原住民之部落存在，因此之後台灣總督府將內本鹿越嶺道沿途駐紮的兵力逐一撤至山下，內本鹿越嶺道便沒有實際用途而逐漸荒廢。後來經過多次地震與風災侵襲讓越嶺道毀損而不能供人行走<ref name="原住民電視台 【尋找布農族 內本鹿越嶺古道】 "/><ref name="Yahoo新聞 被遺忘的戰役 布農族「內本鹿事件」"/><ref name="台東延平鄉影像誌 內本鹿腳行：跨越時空的呼喚 日警後裔重返內本鹿尋根 "/>。

=== 臺灣戰後時期 ===
[[臺灣戰後時期|臺灣戰後]]，[[中華民國|中華民國]][[臺灣省政府|臺灣省政府]]接收臺灣後，並沒有針對該條古道有任何重建或是興建橫貫公路之計畫，僅古道西段六龜至藤枝闢為「藤枝林道」，東段[[鹿鳴|鹿鳴]]至[[清水|清水]]段改為「東36鄉道」，其餘路段皆處於荒廢狀態<ref name="forest"/><ref>{{cite web |url=http://www.keepon.com.tw/thread-88a4b9b7-11d8-e411-93ec-000e04b74954.html |title=登山補給站-專業戶外活動資訊平台 鹿野溪 - 內本鹿古道 | accessdate=2015-05-31}}</ref>。

目前內本鹿越嶺古道雖已納入[[國家步道系統|國家步道系統]]關山─內本鹿越嶺國家步道系統之中，但其路況因多次山崩與無人維護導致不明顯，僅有[[林務局|林務局]]臺東林管處的相關人員為[[台東蘇鐵自然保留區|台東蘇鐵自然保留區]]巡視而行走在內本鹿越嶺古道上<ref name="forest"/><ref>{{cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=59684&ctNode=174&mp=10 |title=中華民國行政院農業委員會林務局-自然保育網 台東紅葉村台東蘇鐵自然保留區 |accessdate=2015-05-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20130508141303/http://conservation.forest.gov.tw/ct.asp?xItem=59684&ctNode=174&mp=10 |archivedate=2013-05-08 }}</ref>。
