'''美洲非殖民化'''指的是[[北美|北美]]和[[南美|南美]]国家脱离[[殖民|殖民]]统治[[独立|独立]]的斗争过程。

== 美国 ==
{{main|美国独立战争|美国扩张历史}}

[[美国|美国]]是美洲大陆第一个获得独立的国家,在1776年的《[[独立宣言|独立宣言]]》中宣布脱离[[英国|英国]]统治，这以宣言在1783年的[[巴黎条约_(1783年)|巴黎条约]]中获得认可。 除去原北美十三州地区以外，美国的其余领土原来分别属于英国、法国、西班牙、墨西哥、丹麦和俄罗斯。

== 海地 ==
{{main|海地革命}}

紧随[[法国革命|法国革命]], 很多自由思想不斷地传播到[[海地|海地]], 引致1791年出現的首次[[奴隸起义|奴隸起义]]。直至[[1804|1804]]年1月1日 , [[让-雅克·德萨林|雅克·德萨林]]宣布[[海地|海地]]为自由[[共和国|共和国]]，与[[美国|美国]]一起成为[[西半球|西半球]]最初的兩个独立国家。[[海地|海地]]更是西半球唯一通过奴隶起义而获得独立的国家，也是第一个脱离欧洲国家殖民的以非洲裔为主体的国家。

== 西班牙殖民地 ==

[[南美洲|南美洲]]眾多的[[西班牙殖民地|西班牙殖民地]]於十九世紀初, 通過[[南美獨立戰爭|南美獨立戰爭]]而贏取獨立。

[[西蒙·玻利瓦尔|玻利瓦尔]]和[[何塞·德·圣马丁|圣马丁]]领导了独立战争。尽管玻利瓦尔试图美洲大陆上说西班牙语的地区政治上保持统一，但是很多殖民地在各自宣布独立。并且在彼此间爆发了更大规模的战斗，如[[巴拉圭战争|巴拉圭战争]]和[[硝石战争|硝石战争]]。

1810年代，相同的情况也发生在西班牙的[[北美洲|北美]]和[[中美洲|中美洲]]殖民地，如[[墨西哥独立战争|墨西哥独立战争]]和相关的一些斗争。

==葡萄牙殖民地==
[[葡萄牙殖民地|葡萄牙殖民地]]于1822年由[[佩德罗一世_(巴西)|佩德罗一世]] （葡萄牙[[国王|国王]] [[若昂六世|若昂六世]]之子），宣布独立，佩得罗一世成为[[巴西帝国|巴西帝国]]第一任皇帝。

不像西班牙，葡萄牙没有分割它在美洲的殖民地，而是由位于[[萨尔瓦多_(巴西)|萨尔多瓦]]的中央集权管理结构直接对[[里斯本|里斯本]]的王国负责。因此不可能见到例如“葡属美洲”这样的称谓（类似[[西属美洲|西属美洲]]，[[荷属美洲|荷属美洲]]等），[[巴西|巴西]]从很早开始就是一整块殖民地。

由于这个原因，1822年巴西独立的时候没有像它说西班牙语的邻国那样分裂成几个国家，在最初60年采用[[君主立宪|君主立宪]]的整体而不是[[联邦共和制|联邦共和制]]也为国家统一作出了贡献。

== 加拿大 ==
{{main|加拿大}}

1867年7月1日成为[[大英帝国|大英帝国]]的管辖区域。这时的“加拿大领土”包括上加拿大和下加拿大(南[[安大略|安大略]]和[[魁北克|魁北克]]), [[新斯科舍省|新斯科舍省]]和[[新不伦瑞克省|新不伦瑞克省]]。 [[英国殖民地|英国殖民地]][[英属哥伦比亚|英属哥伦比亚]] (1871年), [[爱德华王子岛|爱德华王子岛]] (1873年)和[[纽芬兰|纽芬兰]] (1949年,[[第二次世界大战|第二次世界大战]]后) 也相继加入[[联邦|联邦]]。英国也陆续失去[[Rupert's_Land|Rupert's Land]]，[[西北地区_(加拿大)|西北地区]] (1870年)和[[北极圈|北极圈]]附近属于加拿大的岛屿 (1880年)的控制 。

[[纽芬兰|纽芬兰]]在1907年9月26日也获得自治权。

从1867年到1931年,英国控制着加拿大的外交，直到通过[[1931年威斯敏斯特法令|威斯敏斯特条约]]英国才移交了此权给加拿大。

而加拿大政府一直到1982年4月17日才将加拿大宪法从英国国会收回，至此加拿大正式脱离与英国的联系，新宪法也确认了加拿大作为主权独立国家的地位。但加拿大的国家元首依然是伊丽莎白女王，只不过她作为加拿大国家元首的称号是加拿大女王，而不是英国女王。

[[特克斯與凱科斯群島|特克斯與凱科斯群島]]也曾经被提议作为新斯科舍省的一部分加入联邦。

== 西班牙－美国战争 ==
1898年, 美国赢得[[西班牙-美国战争|西班牙-美国战争]]吞并古巴和波多黎各，最终西班牙退出了美洲殖民的历史舞台。 到20世纪早期依然有来自西班牙的穷苦移民和因政治原因流放的难民移居前殖民地, 特别是[[古巴|古巴]]，[[墨西哥|墨西哥]]和[[阿根廷|阿根廷]]。1970年代以后, 很多人从前殖民地移居西班牙。1990年代, [[西班牙公司|西班牙公司]]如[[Repsol_YPF|Repsol]]和[[Telefónica|Telefónica]]增加在南美洲的投资，经常收购私营公司。

== 二十世纪 ==
直到20世纪才有一些国家陆续获得独立，它们是:
*[[巴哈马群岛|巴哈马群岛]]:1964年英国允许该群岛内部[[自治|自治]]，1973年, 巴哈马群岛取得完全的独立，但是仍然保持[[英联邦|英联邦]]成员地位。
*[[圭亚那|圭亚那]]: 1966年脱离英国
*[[苏里南|苏里南]]: 前身是荷属圭亚那，1975年脱离荷兰完全独立。
*[[特立尼达和多巴哥|特立尼达和多巴哥]]:1962年脱离英国。
*[[伯利兹|伯利兹]] (前英属洪都拉斯): 1981年脱离英国

*[[百慕达|百慕达]] (英国)

*[[开曼群岛|开曼群岛]] (英国)
*[[福克兰群岛|福克兰群岛]] (英国)
*[[法属圭亚那|法属圭亚那]] (法国)
*[[瓜德洛普|瓜德洛普]] (法国)
*[[马提尼克|马提尼克]] (法国)
*[[蒙特塞拉特岛|蒙特塞拉特岛]] (英国)
*[[荷属安地列斯群岛|荷属安地列斯群岛]] (荷兰)
*[[圣皮埃尔和密克隆岛|圣皮埃尔和密克隆岛]] (法国)
*[[特克斯和凯科斯群岛|特克斯和凯科斯群岛]] (英国)
*[[格陵蘭岛|格陵蘭岛]] (丹麥)

严格来说,
*1986年1月1日阿鲁巴岛脱离荷属安地列斯群岛成为荷兰王国分离的自治领。就在按计划1996年完成独立的时候，1990年阿鲁巴岛提出请求终止了独立进程。
*法属圭亚那，瓜德洛普和马提尼克岛并不认为是[[法国殖民地|法国殖民地]]，而认为是海外法国领土的“不可分割的一部分”。
