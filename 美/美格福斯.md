'''美格福斯'''（Magnavox）為隸屬美國電子廠商[[飛利浦|飛利浦]]旗下的[[子公司|子公司]]，最初於1917年由[[彼得·L·詹森|彼得·L·詹森]]、[[埃德溫·L·普里漢姆|埃德溫·L·普里漢姆]]所成立，其中在1972年推行發售的「[[美格福斯奧德賽|美格福斯奧德賽]]」為歷史上第一款家庭娛樂用[[電子遊戲機|電子遊戲機]]。<ref name="CM">{{Cite web |url=http://computermuseum.50megs.com/magnavox.htm |title=THE BEGINNING: THE MAGNAVOX ODYSSEY |work=History of Home and Game Computers |accessdate=2014年12月31日 }}</ref>

此公司原名「Magnavox」；在拉丁文裡含意是「響亮的聲音」。<ref name="nl">{{Cite web |url=http://www.vintageradio.nl/Luidsprekers/magnavoxeng.htm |title=Magnavox Radio Speaker |work=vintageradio.nl |date=2014年5月12日 |accessdate=2014年12月31日 }}</ref>

== 歷史 ==
最初在1910年代期間，曾任職在聯邦電報公司（Federal Telegraph Company）的彼得·L·詹森、埃德溫·L·普里漢姆與金融界的[[理查·奧康納|理查·奧康納]]三人在[[加州|加州]][[納帕|納帕]]成立了處理電話聽筒元件的「商用無線開發公司」（Commercial Wireless & Development Company）。之後在1917年；詹森與普里漢姆將此公司重新取名為美格福斯，而最初美格福斯是經營於通訊方面領域。<ref name="CM2">{{Cite web |url=http://www.radiomuseum.org/dsp_hersteller_detail.cfm?company_id=2789 |title=History of the radio manufacturer  Magnavox Co.; Fort Wayne (ex Oakland) |work=Radio Mesuem |accessdate=2014年12月31日 }}</ref>

於[[第一次世界大戰|第一次世界大戰]]裡，美格福斯有開發出用於抗雜訊以及防水的話機供於[[前線|前線]]使用，另也為美國總統[[伍德羅·威爾遜|伍德羅·威爾遜]]在1919年期間的演說場合裡提供擴音設備。<ref name="nl"/>之後因不敵[[AT&T|AT&T]]公司於公共廣播領域上主導的優勢，美格福斯改轉向無線電與留聲機方面的開發。<ref name="nl"/>另外在1920年代期間也開始製造多款型式喇叭，<!-- <ref name="CM"/> CM2? -->其中在1960年代則生產史上第一批[[電漿顯示器|電漿顯示器]]規格的產品。<ref>{{Cite web |url=http://www.calgarysun.com/2013/09/19/from-the-magnavox-odyssey-to-gta-v |title=From the Magnavox Odyssey to 'GTA V' |work=SUN+ |author=Syd Bolton |date=2013年9月23日 |accessdate=2014年12月31日 }}</ref>

在1972年美格福斯推出了歷史上首款的家庭用電子遊戲機「[[美格福斯奧德賽|美格福斯奧德賽]]」，<ref name="CM"/>該主機隨後達到全球銷售約33萬台成績。<ref>{{Cite web |url=http://www.pong-story.com/odyssey.htm |title=Magnavox Odyssey First home video game console |work=Pong-Story |author=David Winter |accessdate=2014年12月31日 }}</ref>不過因1974年[[飛利浦|飛利浦]]收購了美格福斯，讓飛利浦可在往後美格福斯推出的產品掛上飛利浦的名稱，而1978年發售的奧德賽次代主機「[[美格福斯奧德賽²|美格福斯奧德賽²]]」；在一些美國海外國家有被改稱為「飛利浦奧德賽」或「飛利浦VideoPac」。

1980年代中期裡菲利浦與美格福斯生產了以螢幕生成處理畫面的電子打字器產品獲得不錯成績，並成為該時段中文字處理器的規格標準。往後美格福斯則以販售家庭影視娛樂設備為主要產品。
