==劇情大綱==
大富豪劉軒（鄧超 飾）的地產計畫涉及填海工程，威脅靠海為生的居民。單純善良的珊珊（林允 飾）背負美人魚家族秘密並被派遣前往復仇。二人在交手過程中互生情愫，劉軒最終因愛上珊珊而停止填海工作，但他的合夥人及情人若蘭（張雨綺 飾）嫉妒劉軒與珊珊相愛，誓要滅殺人鱼族。而劉軒為救珊珊及人魚族，不惜與若蘭為敵。

== 製作 ==
《美人鱼》是繼《[[少林足球|少林足球]]》之後周星馳第二次在廣東拍攝的電影，大部分外景均於[[深圳|深圳]]取景，亦有在[[廣州|廣州]]拍攝。

本片投資6000萬美元，勉強達到2009年時好萊塢給周星馳導演的預算水平（如果周星馳不辭導[[青蜂俠_(2011年電影)|青蜂俠]]的話，那麼好萊塢給周星馳導演的預算則是1.25億美元）。預算主要用在周星馳的酬勞和水下全3D實景拍攝器材。不過，周星馳一直不喜歡將影片投資作為賣點來宣傳。有傳聞美國哥倫比亞公司亦參與《美人鱼》的投資和發行，但目前還沒有公開。<ref>[http://www.aintitcool.com/node/44067 Stephen Chow's next movie]</ref>

本片的第四款海報中有出自[[毛澤東|毛澤東]]《滿江紅·和郭沫若同志》中的：「四海翻騰雲水怒，五洲震盪風雷激」，这是继《[[西游·降魔篇|西游·降魔篇]]》之后，再次引用毛泽东的词句<ref>{{Cite web|url = http://www.macaodaily.com/html/2015-12/11/content_1051542.htm|title = 星爺向毛主席致敬|accessdate = 2016-02-19|author = |date = 2015-12-11|publisher = [[澳門日報|澳門日報]]|deadurl = yes|archiveurl = https://web.archive.org/web/20160116023627/http://www.macaodaily.com/html/2015-12/11/content_1051542.htm|archivedate = 2016-01-16}}</ref>。

==宣传==
《美人鱼》的实体广告海报以波涛汹涌的大海作为背景，没有煽动性文案，没有演员阵容，仅有片名和“周星驰”三个大字（白底红字）<ref>{{Cite web|url = http://ent.qq.com/a/20151206/021366.htm|title = 周星驰《美人鱼》发首款海报 猴年大年初一上映|accessdate = 2016-02-21|author = |date = 2015-12-06|publisher = 腾讯娱乐}}</ref>。

《美人鱼》的宣传喷绘甚至比户外海报更加简单直接，连片名都没有，只留下“周星驰”三个大字<ref>{{Cite web|url = http://p2.pstatp.com/large/10a00033be6b96edec7|title = 《美人鱼》的宣传喷绘|accessdate = 2016-02-21|author = |date = 2015-12-06}}</ref>。

周星驰及剧组走访超过20个中国大陆城市，举办了21场发布会，参加了54场影城见面会，同影迷见面交流<ref>{{Cite web|url = http://news.sina.com.cn/o/2016-02-21/doc-ifxprqea4867921.shtml|title = 为宣传《美人鱼》跑了20多个城市 最后一站星爷来到东莞|accessdate = 2016-02-21|author = 朱燕霞、李绰芝|date = 2016-02-21|publisher = 新浪}}</ref>。

===海外===
2016年2月19日，《美人鱼》在北美35间戲院小规模上映，首周末取得101万5000美元票房，每院平均29000美元为中国电影最好成绩。<ref>[http://www.boxofficemojo.com/news/?id=4160&p=.htm Stephen Chow's The Mermaid, made its way stateside this weekend. The film currently stands at a record $419.4 million in China and Sony opened it in a mere 35 domestic theaters this weekend where it grossed an estimated $1.015 million. That makes for a weekend-topping $29,000 per theater average. For some added context, this is the biggest opening day and weekend for a limited release Chinese-language film.]</ref>发行方已决定于2月26日增加排片至77间戲院。截止3月，《美人鱼》在美国取得票房$3,232,685美元。目前为2016年非好莱坞电影在美国最佳票房。

2016年2月14日，仅仅在上映的第5天，《美人鱼》的越南票房超过200万美元，成为越南有史以来票房最高的中国电影。而在2016年3月，《美人鱼》的越南票房已超过1000亿[[越南盾|越南盾]]，成为越南有史以来票房最高的非好萊塢电影。

2016年2月18日，《美人鱼》在澳洲18间戲院小规模上映，首周末取得54万[[澳元|澳元]]票房，每院平均30000澳元，为近十年中国电影最佳首周票房。截止3月29日，《美人鱼》在澳大利亞取得票房$902,192美元，为近十年中国电影最佳票房。

2016年2月19日，《美人鱼》在爱尔兰和英国17間戲院小规模上映，本片亦是第5部在欧洲上映的周星驰电影。<ref>[http://news.china.com.cn/rollnews/ent/live/2016-02/22/content_35317025.htm 《美人鱼》排片并不多，但各大外媒基本都给予这部影片较好的评价。]</ref>截止3月22日，《美人鱼》在英国取得票房$332,397[[英镑|英镑]]。

《美人鱼》在馬來西亞的首周票房和次周票房都超过了同时上映的香港；而在2016年3月，《美人魚》以705万美元票房正式成為馬來西亞有史以来票房最高的非好萊塢电影。
