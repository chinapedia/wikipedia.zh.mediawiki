'''美国公理会差会'''（又名'''美部會'''或音译'''亚比丝喜美总会'''）是紧随[[伦敦会|伦敦会]]和[[荷蘭传教会|荷蘭传教会]]（Netherlands Missionary Society）之后，第三个进入[[中国|中国]]的[[更正教|更正教]][[差会|差会]]，也是第一个进入中国的[[美国|美国]]差会。1829年，[[雅裨理|雅裨理]]和[[裨治文|裨治文]]两位[[牧师|牧师]]出发，次年2月在中国受到[[马礼逊|马礼逊]]的接待；他们在[[海峡殖民地|海峡殖民地]]的[[华人|华人]]和[[马来人|马来人]]中间工作。

美部會随后又接连派遣多名[[传教士|传教士]]。[[帝禮仕|帝禮仕]]（Ira Tracy）和[[衛三畏|衛三畏]]（Samuel Wells Williams）在1833年定居新加坡和[[澳门|澳门]]，同年[[杨顺|杨顺]]（Stephen Johnson）和Samuel Munson牧师前往[[曼谷|曼谷]]和[[苏门答腊|苏门答腊]]。

到1890年，公理会已经在中国的[[直隶|直隶]]、[[福建|福建]]、[[山西|山西]]、[[山东|山东]]、[[广东|广东]]5省建立了14个传教士驻扎总堂，有28名传教士、16名单身女传教士、10名医疗传教士、4名本地牧师、105名未受圣职的本地助手，将近1000名领[[圣餐|圣餐]]信徒、450名学生<ref>Townsend (1890), 233-234</ref>，为在华主要西差会之一。但是此后美国公理会差会在中国的传教事业扩展较慢，逐渐落后于其他不少差会。

== 广东 ==
1835年，伯驾医生在[[广州十三行|广州十三行]]创办[[博济医院|博济医院]]（后来转交给长老会）。1847年，[[裨治文|裨治文]]去上海翻译圣经，公理会将广州传教站让给长老会。1883年，[[喜嘉理|喜嘉理]]至[[香港|香港]]，恢复广州香港公理会。香港公理会堂为[[孙中山|孙中山]]受洗处。广州公理会设有西村堂。

1919年，公理会在广东省驻有外国职员11人（均住在[[广州|广州]]），有27处布道区，受圣餐信徒3098人（居该省各差会第7名，次于[[美北长老会|美北长老会]]、[[巴色会|巴色会]]、[[美南浸信会|美南浸信会]]、[[英国长老会|英国长老会]]、[[德国信义会|德国信义会]]和[[美北浸礼会|美北浸礼会]]）.此外，公理会曾计划在新宁（[[台山|台山]]）与[[美北长老会|美北长老会]]、[[循道会|循道会]]联合设立协和总堂<ref>《中华归主》第三章第十二节</ref>。

1920年代，旅美华侨基督徒组织的中华纲纪慎自理传道会（公理会）回国投资建造广州[[光孝堂|光孝堂]]。1919年，美国公理会差会撤出广东，移交给同宗派的华人教会，中华基督教会。

== 福建 ==
[[File:Rev_and_Mrs_Charles_Hartwell.jpg|thumb]]夫妇在福州|right]]
[[File:Puqian_church.JPG|thumb]]|right]]
1842年《[[南京条约|南京条约]]》签字之前，[[雅裨理|雅裨理]]进入[[厦门|厦门]]，直到1846年去世。1847年1月2日，原在暹罗向华侨传教的[[杨顺|杨顺]]（Stephen Johnson）到达[[福州|福州]]，为第一个进入福州的更正教传教士。随后[[弼利民|弼利民]]（Lyman Birt Peet）、[[摩嘉立|摩嘉立]]（Caleb Cook Baldwin）、[[卢公明|卢公明]]（Justus Doolittle）、[[夏察理|夏察理]]（Charles Hartwell）等人陆续前来加入。卢公明创办了[[福州格致中学|格致书院]]（[[福州第五中学|福州第五中学]]的前身）及[[文山女中|文山女中]]（[[福州第八中学|福州第八中学]]的前身）。1858年，杨顺创建了该会在福建的第一座教堂[[铺前顶救主堂|铺前顶救主堂]]。公理会在福州创建的第2座教堂城内观巷[[刘公纪念堂|刘公纪念堂]]是公理会闽北教区以及1927年以后的中华基督教会闽中协会的总堂。公理会在福州城内外先后设立众多的教堂，包括洪山桥洪山堂、塔头桥尚德堂、马尾堂、西门堂、杨桥巷福世堂、澳桥下晨世堂、开元楼开元堂、三保堂、后洋里后洋堂、窗下堂、水部街水部堂、霞浦街霞浦堂、后屿堂、后洲堂、闽侯南通堂等，城区内以今[[鼓楼区|鼓楼区]]及[[台江区|台江区]]颇为密集。

19世纪，公理会在福建省建立了4座总堂：[[福州|福州]]（1847年）、[[长乐|长乐]]（1863）、[[永泰|永泰]]（1864）、[[邵武|邵武]]（1874）。到1919年，公理会在福建省驻有外国职员56人，有82处布道区，受圣餐信徒3195人（居该省各差会第3名，仅次于[[美以美会|美以美会]]和[[英国圣公会|英国圣公会]]）<ref>《中华归主》第三章第四节</ref>。

該省的[[建宁县|建宁县]]是美国公理会中国差会在1890年以后唯一增设的传教总堂。

== 华北 ==
公理会是第一个进入直隶省的更正教差会：1860年进入[[天津|天津]]，1864年[[白汉理|白汉理]]进入[[北京|北京]]，随后进入[[张家口|张家口]]、[[通州|通州]]（1867）、[[保定|保定]]（1873）。1880年以前，公理会总堂比其他差会多一倍有余。公理会的华北总部设在北京[[公理会灯市口教堂|灯市口教堂]]。公理会在通州成立了[[潞河书院|潞河书院]]，讲授[[天文学|天文学]]、[[数学|数学]]、自然[[哲学|哲学]]和[[历史|历史]]，为[[燕京大学|燕京大学]]前身之一。1900年[[庚子事变|庚子事变]]中，该省传教机构几乎完全被摧毁。事变平息以后，信徒迅速增长，到1919年，公理会在直隶省驻有外国职员81人，有48处布道区，受圣餐信徒4432人（居该省各差会第2名，仅次于[[美以美会|美以美会]]）<ref>《中华归主》第三章第三节</ref>。

1902年，[[梅子明|梅子明]]在灯市口大街重建的灯市口教堂，为北京最大的基督教堂。两侧分别开设[[育英中学|育英中学]]、[[贝满女中|贝满女中]]。灯市口教堂毁于1966年的[[文化大革命|文化大革命]]。

== 山东 ==
公理会在山东省的传教工作开始于1878年的赈灾。1880年，公理会的[[明恩溥|明恩溥]]在该省西北部的[[恩县|恩县]]庞庄建立正式的传教工作。1886年，金法兰在[[运河|运河]]边的港口城市[[临清|临清]]建立起公理会在山东的第二处总堂。1900年[[庚子事变|庚子事变]]中，两处总堂均被[[义和团|义和团]]摧毁。此后，临清总堂得到重建，庞庄总堂则迁往[[德州|德州]]。1919年，公理会在山东省驻有外国职员26人，有36处布道区，受圣餐信徒1542人（居该省各差会第6名，次于[[美北长老会|美北长老会]]、[[美南浸信会|美南浸信会]]、[[英国浸礼会|英国浸礼会]]、[[美以美会|美以美会]]和[[圣道公会|圣道公会]]）<ref>《中华归主》第三章第十五节</ref>。

== 山西 ==
公理会在山西省的传教工作开始于1881年，来自美国欧柏林大学得冕路德牧师夫妇来到山西省，原计划在省会太原创办第一座总堂，但是由于各差会协约的限制，于是转往太原以南的[[太谷县|太谷县]]建立总堂。1887年，公理会在[[汾州|汾州]]创建该省第二座总堂。位于一片[[鸦片|鸦片]]种植区的中心；公理会在山西省的早期工作中，以开设戒烟局吸引信徒为主。1900年7月31日，6名传教士在太谷遇害；8月15日，在汾州的公理会传教士亦遭杀害<ref>名单详见[[太原教案|太原教案]]</ref>。[[庚子事变|庚子事变]]后，山西只有125名公理会信徒。1902年，公理会唯一幸存的传教士文阿德返回太谷，欧柏林大学也陆续增派人力。此后，信徒人数迅速增长，到1919年，公理会在山西省驻有外国职员24人，有51处布道区，受圣餐信徒1495人（居该省各差会第2名，仅次于[[内地会|内地会]]）<ref>《中华归主》第三章第十四节</ref>。

公理会的教会学校，在山西省占有突出的地位。1919年，山西有7所教会中学，其中3所为公理会创办，包括[[太谷铭贤中学|太谷铭贤中学]]、[[汾阳铭义中学|汾阳铭义中学]]和崇德女中。

== 陕西 ==
公理会亦从山西渡过黄河进入陕西省北部，布道范围包括[[榆林|榆林]]、[[绥德|绥德]]、[[清涧|清涧]]、[[米脂|米脂]]、[[橫山縣|横山]]、[[延川|延川]]、[[吴堡|吴堡]]等10县。该区没有外国传教士驻扎，教务由中国职员负责。1919年，公理会在陕西省有28处布道区，受圣餐信徒241人（居该省各差会第5名，次于[[北美瑞挪会|北美瑞挪会]]、[[英国浸礼会|英国浸礼会]]、[[瑞华会|瑞华会]]和[[内地会|内地会]]）<ref>《中华归主》第三章第十六节</ref>。
