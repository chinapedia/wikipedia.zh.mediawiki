'''美国电视剧'''（又稱'''美劇'''），广义上涵盖美国所有[[电视|电视]]及[[網路|網路]]播放的[[戲劇|戲劇]]节目。

== 历史 ==
美国电视剧的起步时期约為1920年代至1940年代。在1930年代，[[國家廣播公司|NBC]]和[[CBS_(電視網)|CBS]]两大广播公司尝试性地播出节目，并成立了自己的电视台。这时的节目多改编自经典歌剧或舞台剧。

1950年代，由于技术的进步，电视剧也得以进行发展。录像机的发明使得电视节目可以先进行录制，再进行后期编辑。而随着经济的增长，越来越多的人有能力购买电视，这也促进了美国电视剧的发展。同样在这个年代，美国进入西部片时代。

约1960年代至1970年代。美国电视剧进入全彩阶段。节目类型也大大丰富。1980年代以後美国电视剧极大地丰富了美国人民的生活，几乎成为人们生活中不可缺少的部分。NBC提出“必看电视 (Must See TV)”，使得周四成为收视率制高点。

== 与亚洲电视剧比较 ==
*商业模式：美国电视剧多由季节播出，每集一个星期，其中大部分将不会重新播放。这帮助观众们形成定时观看电视的习惯。然而，在亚洲，若一个电视剧获得了良好的反响，它将被一次又一次地在不同的频道重新播出。

*编剧水平：美国电视剧有更丰富的主题，但亚洲的主题却很匮乏。另外，美国的电视剧更加注重对角色形象的描写，而亚洲的电视剧更加注重故事。此外，美国的电视剧较亞洲更加注重时效性。

== 美国电视剧对亚洲文化的影响 ==
*语言：有一定影响力的美国电视剧中人物的说话方式会影响亚洲观众。例如引入新的词汇，或新的表达方式。

*时尚：因为电视是视听体验，所以对人的影响也会很大。多年来，亞洲青年的转变是显而易见的。尤其是他们的穿着方式。现在的年轻人从不同的电视节目中汲取灵感，从而形成多样的风格。这也导致美国的流行时尚传递到亚洲来。

*宣传：美国的电视节目极大地为企业宣传了自己的产品。它帮助人们展示人们从未接触过的产品和生活方式。而这些也在亚洲的生活中潜移默化。

*独立性：亚洲人是注重团队的集体，他们住在紧密的大家庭中，一切都为共同利益而做。美国的电视剧则引入了自由、个人主义的观念。如今，越来越多的亚洲年轻人实现独立自主，尽管他们仍保持与家人的联系，然而自我的部分也越来越多。
