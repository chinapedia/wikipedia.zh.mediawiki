'''美洲閃鱗蛇'''（[[學名|學名]]：''Loxocemus bicolor''），又稱'''中美蛇'''，是[[蛇亞目|蛇亞目]]下的一個[[單型|單型]]科及單型屬，<ref name="ITIS-F">{{ITIS|ID=209605|taxon=Loxocemidae|year=2007|date=17 August}}</ref>屬下只有一個品種，就是在[[墨西哥|墨西哥]]發現的美洲閃鱗蛇（L. bicolor）。美洲閃鱗蛇科亦是眾多蛇科中最小型的一個科。暫時在美洲閃鱗蛇下未有其它亞種。此類蛇亦被稱為「[[墨西哥|墨西哥]]蟒蛇」、「墨西哥洞蟒」、「墨西哥洞蛇」。<ref name="ITIS-F"/>

== 特徵 ==
美洲閃鱗蛇是同名科屬中的唯一品種，與其它種屬的蛇類比較之下顯得獨立。某程度上牠的習性跟外型都與[[蟒科|蟒科]]有點相似，可是蟒蛇卻不是分佈於美洲土地的蛇種，兩者間相信應沒有直接的關連。

成年的美洲閃鱗蛇約有1.5[[米_(單位)|米]]長，體型強壯健碩，嘴部呈鐵鏟狀，[[頭部|頭部]]扁窄，[[眼睛|眼睛]]細小，幼小的體型讓其便於居住在地洞之內。牠們被認為是陸行性的蛇類，而且善於掘土，並生活於地底，因此對於研究者而言是較為難以觀察及研究的。體色偏[[黑色|黑]]，雜以零碎的[[白色|白色]]鱗片，一般而言經歷蛻皮後牠們身體的顏色會完全褪掉，最終變成一條僅在頭部附近略有暗色鱗片的白蛇。

繁殖方面，美洲閃鱗蛇是[[卵生|卵生]]動物，每次大約能生產2至4枚蛇卵。

== 地理分佈 ==
美洲閃鱗蛇主要分佈於[[墨西哥|墨西哥]]國內[[太平洋|太平洋]]海岸地區一帶的中低段山脈，包括[[納亞里特州|納亞里特州]]、[[哈利斯科州|哈利斯科州]]、[[科利馬州|科利馬州]]、[[米卻肯州|米卻肯州]]、[[莫雷洛斯州|莫雷洛斯州]]、[[格雷羅州|格雷羅州]]、[[瓦哈卡州|瓦哈卡州]]及[[恰帕斯州|恰帕斯州]]等州份；其分佈區域更從這些地區向南伸延至[[危地馬拉|危地馬拉]]、[[洪都拉斯|洪都拉斯]]、[[薩爾瓦多|薩爾瓦多]]、[[尼加拉瓜|尼加拉瓜]]以及[[哥斯達黎加|哥斯達黎加]]等國家境內。美洲閃鱗蛇的標本產地是位於薩爾瓦多國內的[[聖薩爾瓦多|聖薩爾瓦多]]省及拉烏尼翁省。<ref name="McD99"/>

== 習性 ==
美洲閃鱗蛇能生活於多種不同的環境，不管是熱帶地區、潮濕地區、乾燥地區或森林內都能適應。在[[危地馬拉|危地馬拉]]及[[洪都拉斯|洪都拉斯]]一帶，一些通往[[加勒比海|加勒比海]]的乾燥內陸溪谷裡，亦曾有美洲閃鱗蛇生活。<ref name="McD99"/>

另外，美洲閃鱗蛇主要食用[[齧齒目|齧齒目]]動物（如[[鼠|鼠]]類）及[[蜥蜴|蜥蜴]]類。亦有研究指，美洲閃鱗蛇會進食[[美洲鬣蜥科|美洲鬣蜥]]所誕下的卵。
