'''美洲醫科大學''' (Medical University of the Americas) 是在[[加勒比海|加勒比海]]東部尼維斯島上的一所[[醫學院|醫學院]]，醫學係由四年制：前兩年（五學期）在[[尼維斯|尼維斯]]進行基礎教學，後兩年（七十二星期）在美國與愛爾蘭的醫院進行臨床教學<ref>{{cite web |url=http://mua.edu/curri.php |title=存档副本 |accessdate=2007-10-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20071027034309/http://www.mua.edu/curri.php |archivedate=2007-10-27 }}</ref>。 畢業後所獲得的學位為醫學博士。 學生大多數來自北美地區、教師來自美國、加拿大、英國、墨西哥、黎巴嫩、中國、瑞典、德國、俄羅斯、以及印度<ref>{{cite web |url=http://mua.edu/facul.php |title=存档副本 |accessdate=2007-10-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20071011111100/http://mua.edu/facul.php |archivedate=2007-10-11 }}</ref>。  教學語言是[[英語|英語]]。 畢業後可報考美國醫學執照考試（USMLE）以及有資格申請美國實習醫師、住院醫師等職位。

==政府認可==
美洲醫科大學由聯合國世界衛生組織認可、以聖克裏斯多福與尼維斯教育部認可。<ref>{{cite web |url=http://imed.ecfmg.org/results.asp?country=661&school=&currpage=1&cname=SAINT+KITTS+AND+NEVIS&city=&region=CA&rname=Central+America%2FCaribbean&psize=25 |title=存档副本 |accessdate=2007-10-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20070629042756/http://imed.ecfmg.org/results.asp?country=661&school=&currpage=1&cname=SAINT+KITTS+AND+NEVIS&city=&region=CA&rname=Central+America%2FCaribbean&psize=25 |archivedate=2007-06-29 }}</ref>

美洲醫科大學由紐約州教育部認可、准許美洲醫大的學生在紐約州進行多於十二星期的臨床訓練與教學<ref>{{cite web |url=http://www.op.nysed.gov/medforms.htm |title=存档副本 |accessdate=2007-11-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20071017031215/http://www.op.nysed.gov/medforms.htm |archivedate=2007-10-17 }}</ref>。
