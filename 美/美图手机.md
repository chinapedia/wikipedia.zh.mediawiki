'''美图手机'''，原名'''MeituKiss'''，是[[中国大陆|中国大陆]]软件开发商[[美图公司|美图公司]]于2013年5月16日发布的[[智能手机|智能手机]]系列。该系列以[[自拍|自拍]]功能为主打，拥有其自主研发的7级自动美颜技术、高精度[[自动对焦|自动对焦]]及人脸识别技术。全系列均搭载[[Android|Android]][[操作系统|操作系统]]。现时其最新型号为美图手机M4，于2015年4月8日推出。

根据官方网站上的显示，[[两岸三地|两岸三地]]的不少明星皆在使用美图手机，包括[[林志颖|林志颖]]、[[李小璐|李小璐]]、[[吴昕|吴昕]]、[[张亮|张亮]]、[[张馨予|张馨予]]、[[吴宗宪|吴宗宪]]等等，以及《昕薇》、
《瑞丽》杂志的模特<ref>{{cite web |url=http://www.meitu.com/phone/star_photos |title=美图手机有哪些明星用户？ |publisher=美图手机官网 |date= |accessdate=2015年2月1日 |deadurl=yes |archiveurl=https://web.archive.org/web/20150205075303/http://www.meitu.com/phone/star_photos |archivedate=2015年2月5日 }}</ref>。另外该手机曾在台湾综艺节目《[[女人我最大|女人我最大]]》<ref>{{youtube|0ef3jFOLVpU|還在用TR？Out了！快來看看美圖手機2！}}</ref>和《[[大学生了没|大学生了没]]》<ref>{{youtube|9zl7-cnnnhU|20140523 大學生了沒之超好用美圖手機}}</ref>被介绍。

==发展历史==
在MeituKiss发布之前，[[蔡文胜|蔡文胜]]便正式担任美图公司董事长。2013年4月，媒体曝光MeituKiss的规格参数，称该机拥有4.5寸IPS高清视网膜屏幕，分辨率为1280×720，采用OGS屏幕贴合技术，并搭载[[Android|Android]]4.2系统。而在[[用户界面|用户界面]]设计上，还会针对女性做一些精心设计。前后[[摄像头|摄像头]]均为800万像素，并拥有5层特制镜片，且拥有[[富士通|富士通]]独立ISP图像处理芯片，为全球独家<ref>{{cite web|url=http://www.36kr.com/p/202939.html|title=【独家】美图秀秀手机MeituKiss更多配置曝光，发布会当天接受预订 |publisher=36氪 | date= |accessdate=2015年2月1日 }}</ref>。2013年5月16日，MeituKiss正式发布，以自拍功能为主打，并设置了专门的自拍按键<ref>{{cite web|url=http://epaper.jinghua.cn/html/2013-05/17/content_1989959.htm|title=美图秀秀推“自拍手机”|publisher=京华时报|date=2015年5月17日|accessdate=2015年2月1日|deadurl=yes|archiveurl=https://web.archive.org/web/20150205084854/http://epaper.jinghua.cn/html/2013-05/17/content_1989959.htm|archivedate=2015年2月5日}}</ref><ref>{{Cite web |url=http://tech.qq.com/a/20130516/019927.htm|title= 美图秀秀发布自拍手机MeituKiss 售价2199元|accessdate=2014-12-11 |publisher= 腾讯网|date= 2013年5月16日}}</ref>。

2013年10月8日，MeituKiss正式更名为美图手机。同年11月26日又推出了美图手机1S<ref name="1S">{{Cite web |url=http://digi.tech.qq.com/a/20140117/009989.htm|title= 美图手机1S评测：手机界的自拍神器
|accessdate=2015年2月1日|publisher= 腾讯网|date= 2014年1月17日}}</ref>。2014年1月8日，推出了专为[[学生|学生]]定制的美图手机1C<ref>{{Cite web |url=http://www.anqu.com/wenda_560/62398|title=美图手机1C配置怎么样?美图手机1C多少钱?|accessdate=2015年2月1日|publisher= 安趣网|date= 2014年1月22日}}</ref>。

2014年4月27日，美图公司在北京举行了“美图之夜”春夏新品发布会，正式发布了美图手机2，并发布专为该机打造的[[用户界面|用户界面]]MEIOS UI，还与[[中国联通|中国联通]]、[[苏宁云商|苏宁云商]]进行了战略合作，由此中国大陆的消费者可通过[[中国联通|中国联通]]、[[苏宁云商|苏宁云商]]平台购买到美图手机<ref>{{Cite web |url=http://mobile.zol.com.cn/450/4501241_all.html#p4501241|title=林志颖Angelababy现身 美图手机2发布|accessdate=2015年2月1日|publisher= [[中关村在线|中关村在线]]|date= 2014-04-28 }}</ref>。该手机于2014年8月12日在[[台湾|台湾]]市场销售<ref name="台湾">{{Cite web |url=http://www.ettoday.net/news/20140812/388897.htm|title= 售價僅11,800元！八核心自拍機「美圖手機2」 8/12開賣|accessdate=2014-12-11 |work= [[ETtoday_东森新闻云|ETtoday 东森新闻云]]|publisher= |date= 2014年8月12日}}</ref>。

2014年11月28日，美图公司正式发布美图手机M2，为美图手机2的升级版本，运行基于Android 4.2的MEIOS1.5系统，美颜模式也从图片美颜进化到了视频美颜。其主打的镜头模糊检测功能更可以识别镜头镜片表面是否干净<ref name="M2">{{Cite web |url=http://news.mydrivers.com/1/339/339378.htm|title= 2199元美图手机M2发布：支持镜头模糊检测|accessdate=2015年2月1日|work= [[驱动之家|驱动之家]]|publisher= |date= 2014-12-11 }}</ref>。同时美图公司还聘用[[Angelababy|Angelababy]]成为美图手机的代言人<ref>[http://www.he.xinhuanet.com/news/2014-11/25/c_1113399742.htm Angelababy确定代言美颜手机 海量自拍美照曝光] {{webarchive|url=https://web.archive.org/web/20150325001421/http://www.he.xinhuanet.com/news/2014-11/25/c_1113399742.htm |date=2015-03-25 }}</ref>。

2015年4月8日，美图公司在北京举办发布会，发布了美图手机M4<ref name="M4">{{Cite web |url=http://mobile.yesky.com/108/57007108.shtml|title= 批量生产女神 美图发布第三代智能手机M4|accessdate=2015年4月8日|work=天极网|publisher= |date= 2015-04-09 }}</ref>。

2015年8月7日，美图手机M4开始在[[香港|香港]][[百老汇摄影器材|百老汇摄影器材]]各门店销售<ref name="HK">{{Cite web |url=http://tech.huanqiu.com/original/2015-08/7218241.html|title= 美图手机M4登陆香港 8月7百老汇全线开售|accessdate=2015年8月7日|work=环球网|publisher= |date= 2015年9月13日}}</ref>。

==各型号功能特色==
由于该手机为[[美图公司|美图公司]]研发，因此美图手机内置了[[美圖秀秀|美圖秀秀]]、[[美圖貼貼|美圖貼貼]]、[[美圖GIF|美圖GIF]]、[[美拍|美拍]]等自制軟體；而手机亦提供第三方应用程序，例如[[微信|微信]]、[[新浪微博|新浪微博]]（在台湾销售的型号则以[[Facebook|Facebook]]代替）。另外，美图手机没有安装Google应用程序，因此需要自行重新安装才能使用Google应用程序。
===第一代（MeituKiss）===
第一代美图手机于2013年5月16日发布。配置方面，手机搭载[[索尼|索尼]]生产的800万像素前置摄像头，拥有F/2.2大光圈、5层特制镜片、AF自动对焦等特性，还配备了独立ISP芯片，并拥有自动美颜技术。该手机预装Android 4.2系统，并搭载美图专属定制的UI主题<ref>{{Cite web|url=http://www.meitushop.com/meitukiss/|title=MeituKiss产品信息|accessdate=2015年2月4日|work=美图手机官方网站|publisher=|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20130806144048/http://www.meitushop.com/meitukiss/|archivedate=2013年8月6日}}</ref>。有樱花粉、贝壳白两种颜色。

===美图手机1C／1S===
[[File:Meitu_MK150.JPG|thumb]]
美图手机1S于2013年11月26日推出，而美图手机1C则于2014年1月8日推出，此为专为[[学生|学生]]定制的型号。其中美图手机1S采用[[索尼|索尼]]800万前置摄像头及富士通前置ISP芯片，而16GB版使用升级版1.5GHz四核处理器。另外，美图手机1S使用明星跨界设计的包装盒。美图手机1C提供水晶白、浅暖粉、樱桃红三种颜色<ref>{{Cite web|url=http://www.meitu.com/meitu1c/param|title=美图手机1C规格参数|accessdate=2015年2月4日|work=美图手机官方网站|publisher=|date=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；美图手机1S提供月光白、闪晶粉、炫目红三种颜色<ref>{{Cite web|url=http://www.meitu.com/meitu1s/param|title=美图手机1S规格参数|accessdate=2015年2月4日|work=美图手机官方网站|publisher=|date=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

===美图手机2／M2===
美图手机2于2014年4月27日在北京“美图之夜”春夏新品发布会上发布。此手机无论是外观还是配置，都做了一系列的改进。显示屏的玻璃为[[康宁公司|康宁公司]]生产的大猩猩保护玻璃。其中前、后镜头为1300万像素，还使用了全球独家的美颜技术，为[[发明专利|发明专利]]技术。而手机还使用了基于[[Android|Android 4.2]]深度定制的MEIOS，考虑到美图手机2的用户大部分为[[iOS|iOS]]用户，因此美图手机2采用“桌面即应用”设计理念，可让iOS用户也能无障碍适应<ref>{{Cite web|url=http://www.meitu.com/meios|title=MEIOS介绍|accessdate=2015年2月4日|work=美图手机官方网站|publisher=|date=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2014年8月12日，美图手机2在[[台湾|台湾]]市场销售<ref name="台湾"/>。

其升级版美图手机M2于2014年11月28日发布<ref name="M2"/>，该手机使用其最新美颜技术，以及更加智能的人脸识别技术。另外MEIOS系统也进行了更新，新增功能包括镜头模糊检测功能、正方形拍摄框选择、同时新增更多安全防护机制<ref>{{Cite web|url=http://www.meitu.com/m2/meios|title=新版MEIOS介绍|accessdate=2015年2月4日|work=美图手机官方网站|publisher=|date=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

美图手机2提供月光白、炫目红两种颜色，而美图手机M2则新增薄荷绿、樱花粉两种颜色。另外还提供了[[Hello_Kitty|Hello Kitty]]特别版（限量2000台）<ref>{{Cite web|url=http://www.meitu.com/meitu2_hellokitty|title=美图手机2HelloKitty版|accessdate=2015年2月4日|work=美图手机官方网站|publisher=|date=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>及赛车限量版（全球限量1999台）<ref>{{Cite web|url=http://www.meitu.com/meitu2_limited|title=美图手机2赛车限量版|accessdate=2015年2月4日|work=美图手机官方网站|publisher=|date=}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

===美图手机M4===
美图手机M4于2015年4月8日在北京举办的发布会推出，以“夜间自拍神器”作为市场定位。在推出前，媒体报道称[[Angelababy|Angelababy]]参与到该手机的设计工作中，并担任产品顾问<ref>{{cite news|title=新一代美图手机来了：竟是她设计的！ |url=http://news.mydrivers.com/1/397/397591.htm |publisher=驱动之家 |date=2015-03-09  |accessdate=2015-03-09 }}</ref>。主要升级内容包括针对夜间自拍进行了软硬件优化、升级美颜技术。同时配备了新一代的索尼1300万像素前置摄像头以及富士通的新一代Milbeaut图像处理器。另外，该机还支持[[4G|4G]]（[[TD-LTE|TD-LTE]]）网络（仅限[[中国移动|中国移动]]和[[中国联通|中国联通]]）<ref name="M4"/>。该手机在操作系统方面也做了升级，新增了“寻找美图手机”（类似于[[查找iPhone|查找iPhone]]）和美图应用商店应用<ref>{{Cite web |url=http://www.meitu.com/m4/meios2|title=MEIOS2介绍|accessdate=2015年4月9日 |work= 美图手机官方网站|publisher= |date= }}</ref>。美图手机M4提供炫目红、月光白、薄荷绿三种颜色，还设有HelloKitty版。该机于4月10日将在官网、苏宁易购、苏宁门店发售<ref name="M4"/>。

==销售==
美图手机上市之初，只能通过官网[[抢购|抢购]]才能获得<ref name="抢购">{{Cite web |url=http://it.sohu.com/20130606/n378228012.shtml|title=MeituKiss限量抢购会是卖萌吗|accessdate=2015年2月4日 |work= 搜狐|publisher= |date= 2013年6月6日}}</ref>，此前抢购为不定期举行，现时通常为每周五中午12:00（北京时间）举行。2013年6月6日，第一代美图手机在53分钟内抢购一空<ref>{{Cite web |url=http://www.cnbeta.com/articles/240191.htm|title=美图秀秀自产手机 MeituKiss 53分钟抢购完毕|accessdate=2013-06-06 |work= cnbeta.com|publisher= |date= }}</ref>；2014年4月28日，美图手机2首批5万台售罄，由于销量极好，[[苏宁易购|苏宁易购]]临时增加了4月30日12点的第二次抢购<ref>{{Cite web |url=http://mobile.163.com/14/0430/02/9R1US49K001167B4.html|title=美图手机2首批5万台售罄 30日二次开抢|accessdate=2015年2月4日 |work=网易|publisher= |date= 2014-04-30}}</ref>。而在2014年7月15日，美图手机2Hello Kitty特别版在27秒内售罄<ref>{{Cite web |url=http://mobile.yesky.com/218/38130718.shtml|title=美图手机2 Hello Kitty特别版27秒疯抢售罄|accessdate=2015年2月4日 |work=天极网|publisher= |date= 2014-07-15}}</ref>。

除通过官网抢购外，中国大陆消费者可通过[[中国联通|中国联通]]、[[苏宁云商|苏宁云商]]平台购买，另可在[[淘宝网|淘宝网]]及[[天猫|天猫]]授权店铺购买。

在台湾地区，美图手机2则由[[鴻海|鴻海]]子公司康法科技引进，于2014年8月12日在[[台湾|台湾]]市场销售<ref name="台湾"/>，台湾消费者可通过台湾官网及momo 購物網购买。在香港地区，美图M4在[[百老汇摄影器材|百老汇摄影器材]]各门店销售<ref name="HK"/>。

==转让手机业务==
由于美图手机业务持续亏损，2018年11月19日，美图公司宣布与小米集团达成战略合作伙伴关系。美图公司将旗下美图手机的品牌、影像技术和二级域名，在全球范围内独家授权给小米集团。2019年年中前美图公司将完全关闭手机业务，后续将由小米负责研发、生产、销售和推广，美图公司只是按比例享受销售分成<ref>[https://m.mydrivers.com/newsview/620081.html 美图宣布今年年中前关闭手机业务：后续由小米负责研发]</ref>。
