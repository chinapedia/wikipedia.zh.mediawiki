美國[[州際公路系統|州際公路系統]]是一個綿密的公路網，公路本身或公路之間的連結非常完善。然而，目前仍有一些缺口存在。

==未完工所產生的缺口==
有些缺口是由於公路的某些路段尚未完工造成的。如果要接續同一條公路繼續往目的地行，必須繞行其他的聯絡道路。

* [[File:I-69.svg|25px]]：'''[[69號州際公路|69號州際公路]]'''原本是由[[印第安納波利斯|印第安納波利斯]]往東北行至[[密西根州|密西根州]][[休倫港_(密西根州)|休倫港]]，目前計畫往南延伸至[[德克薩斯州|德克薩斯州]]美墨邊界。2006年2月，[[密西西比州|密西西比州]]的部份路段通車。

* [[File:I-74.svg|25px]]：'''[[74號州際公路|74號州際公路]]'''目前有四段，最早完工的路段由[[辛辛那提|辛辛那提]]向西北至[[愛荷華州|愛荷華州]]，另三段則位於[[北卡羅萊納州|北卡羅萊納州]]：
# 由[[維吉尼亞州|維吉尼亞州]]與北卡羅萊納州交界沿[[77號州際公路|77號州際公路]]往南約5哩後向東南分出至[[52號美國國道|52號美國國道]]。
# [[格林斯伯勒_(北卡罗来纳州)|格林斯伯勒]]南方與[[73號州際公路|73號州際公路]]共構路段。
# 升級後的[[74號美國國道|74號美國國道]]：自[[勞林堡_(北卡羅萊納州)|勞林堡]]往[[蘭伯頓_(北卡羅萊納州)|蘭伯頓]]，跨過[[95號州際公路|95號州際公路]]至[[41號北卡州州道|41號北卡州州道]]。

* [[File:I-95.svg|25px]]：'''[[95號州際公路|95號州際公路]]'''在[[紐澤西州|紐澤西州]][[特伦顿_(新泽西州)|特倫頓]]附近的這個缺口可能最有名也最讓人困擾。往[[紐約市|紐約市]]方向車流由南向北繞行特倫頓市區，在與[[美國國道1|美國國道1號]]交會後，公路編號變為[[295号州际公路_(特拉华州至新泽西州)|295号州际公路 (特拉华州至新泽西州)]]向南行，若未注意，會回到[[費城|費城]]，由I-295必須接上[[195號州際公路_(紐澤西州)|I-195]]往東才能接回I-95主線繼續往北。往費城方向車流向南過了[[賓夕法尼亞州|賓夕法尼亞州]]與紐澤西州州界後，公路編號變為[[276號州際公路_(賓夕法尼亞州)|276]]（[[賓州收費公路|賓州收費公路]]），由於I-276和I-95的交流道仍在施工中，車輛必須繞行其他道路才能再回到I-95主線。等到該交流道完工之後（預計2014年），276號州際公路在I-95以東的路段將改編為95號連上原本I-95在紐澤西州的路段，現在的I-95在I-276以北的路段則改編為I-195。

* [[File:I-265.svg|25px]]：[[印第安納州|印第安納州]]和[[肯塔基州|肯塔基州]]分別修築了'''[[265號州際公路_(印第安納州─肯塔基州)|265號州際公路]]'''，原本的計畫是將I-265興建成為[[路易維爾_(肯塔基州)|路易維爾]]的外環線，但到目前二條公路隔[[俄亥俄河|俄亥俄河]]遙遙相望，並未相連。已有[[俄亥俄河橋樑計畫|計畫興建橋樑]]連接這二個路段，預計2009年開工，2020年完成。<ref>[http://www.kyinbridges.com/ Kentucky Indiana Bridges - Home<!-- Bot generated title -->]</ref>

==人為因素產生的缺口==
[[File:To_plate_blue.svg|right]]
[[File:I-90.svg|right]]
長久以來，屬[[芝加哥|芝加哥]]市管理的[[芝加哥天際公路|芝加哥天際公路]]一直被認為是'''[[90號州際公路|90號州際公路]]'''的一部份，州際公路系統也將其納入。然而，在1999年芝加哥市決定不提出將這個路段納編為州際公路系統的申請（芝加哥天際公路亦未百分之百符合州際公路的標準）。芝加哥市將芝加哥天際公路上原本I-90的符號的上方全部加上'''{{lang|en|TO}}'''（往）以資區別。但[[伊利諾州|伊利諾州]]運輸部和[[聯邦公路管理局|聯邦公路管理局]]仍將此路段視為90號州際公路。<ref>{{cite web |url=http://www.fhwa.dot.gov/programadmin/toll_Rds.html |title=存档副本 |accessdate=2009-08-28 |0=2012-02-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20120204041646/http://www.fhwa.dot.gov/programadmin/toll_Rds.html |archivedate=2012-02-04 }}</ref><ref>{{cite web |url=http://tollroadsnews.info/artman/publish/article_969.shtml |title=存档副本 |accessdate=2006-07-18 |0=2008-09-22 |archive-url=https://web.archive.org/web/20080922090915/http://tollroadsnews.info/artman/publish/article_969.shtml |archive-date=2008-09-22 |dead-url=yes }}</ref>

==公路未達標準所產生的缺口==

這類缺口產生的原因是公路上某些路段不符合州際公路的標準，但這些路段仍使用州際公路的編號。這類路段包括[[可開閉的吊橋|可開閉的吊橋]]（當船隻通過時，吊橋打開，車流停止），但不包括[[收費站|收費站]]、[[邊界管制站|邊界管制站]]及[[農產品檢查站|農產品檢查站]]。

===二線道路段===
[[File:i-93.svg|right]]
'''[[93號州際公路|93號州際公路]]'''通過[[新罕布夏州|新罕布夏州]]的[[弗蘭科尼亞山峽|弗蘭科尼亞山峽]]時僅設有二線道。由於對環境衝擊太大（部份原因是高速公路所帶來的震動會導致[[山中老人石|山中老人石]]有崩塌的危險，但山中老人石已於2003年崩落），原本四線道設計被迫放棄。在過去很長的一段時間裡，這個路段標記為[[3號美國國道|3號美國國道]]往93號州際公路，但現在已經全部改為93號州際公路的藍色盾形標誌。聯邦在1973年准許此路段維持原設計但仍給予州際公路的地位。

==輔助線與主線之間的缺口==
州際公路的輔助線和主線有二種可能的方式交會：直接或透過同樣主線的另外一條輔助線相連（例如：[[加州|加州]]的[[I-280_(加利福尼亞州)|I-280]]是利用[[I-680_(加利福尼亞州)|I-680]]接回[[I-80|I-80]]）。但仍有一些連接上的缺口：

* [[File:I-238.svg|25px]]：'''[[238號州際公路|238號州際公路]]'''不符合編碼規則，州際公路系統中亦無38號存在。I-238連接加州[[舊金山灣區|舊金山灣區]]的[[I-580_(加利福尼亞州)|I-580]]和[[I-880_(加利福尼亞州)|I-880]]，依規則應編為I-80的輔助線。但當時在該地區已編有八條I-80輔助線，加州政府又不願使用唯一可用的编號180（理由是在100哩外的[[弗雷斯諾|弗雷斯諾]]有[[180號加州州道|180號加州州道]]，不用相同編號可避免駕駛人混淆），故使用升級前的编號238。目前，興建[[480號州際公路_(加利福尼亞州)|I-480]]的計畫已取消，故這個編號成為另一個可用的選擇。
* [[File:I-278.svg|25px]][[File:I-478.svg|25px]][[File:I-678.svg|25px]][[File:I-878.svg|25px]]：在[[紐約市|紐約市]]裡的州際公路'''[[278號州際公路_(紐約州)|278號]]'''、'''[[478號州際公路_(紐約州)|478號]]'''、'''[[678號州際公路_(紐約州)|678號]]'''和'''[[878號州際公路_(紐約州)|878號]]'''均未和[[78號州際公路|78號州際公路]]相連。[[78號州際公路|78號州際公路]]計畫穿越[[紐約市|紐約市]]分叉為二條公路後結束，但由於環境影響太大，原計畫取消，另外一個可能的做法是由I-278在紐澤西的終點向西北延伸至I-78和[[24號紐澤西州道|24號紐澤西州道]]交會點，如此這些公路都可以透過I-278和主線相連。
* [[File:I-585.svg|25px]]：在[[南卡羅萊納州|南卡羅萊納州]][[斯巴坦堡_(南卡羅來納州)|斯巴坦堡]]的'''[[585號州際公路_(南卡羅來納州)|585號州際公路]]'''原本和[[85號州際公路|85號州際公路]]是相連的，但自I-85改道後，I-585的終點改在[[585號州際公路商業線_(南卡羅來納州)|585號州際公路商業線]]，目前利用[[176號美國國道|176號美國國道]]當做聯絡道連上I-85。

==其他==
*[[File:I-76.svg|25px]][[File:I-84.svg|25px]][[File:I-86.svg|25px]][[File:I-88.svg|25px]]：這四條州際公路分為東西二段且相距遙遠，亦無後續計畫將其連接{{fact|，當初使用相同編號的原因是當地的編號已用完，不得已而為之}}。這些公路看似有缺口存在，但實際上沒有。
* 有許多路段的路肩寬度或限高不符州際公路標準，在此不列入討論。
* [[阿拉斯加州|阿拉斯加州]]和[[波多黎各|波多黎各]]的標準不同，亦不列入討論。
* 有些系統交流道（州際公路交會點）並不符合標準，如在紐約州的[[I-84|I-84]]和[[I-87|I-87]]。這些交流道的改善計畫預計於2010年代中期陸續完工。
* 有些州際公路相交但沒有交流道，例如在芝加哥的[[I-57|I-57]]和[[I-294_(伊利諾州)|I-294]]與在賓州的[[I-76|I-76]]和[[I-81|I-81]]。
* 輔助線經由另一條屬於同樣主線的輔助線與主線相連，這在編號系統裡是被允許的，如在加州的[[I-280_(加利福尼亞州)|I-280]]、[[I-380_(加利福尼亞州)|I-380]]、[[I-680_(加利福尼亞州)|I-680]]、[[I-880_(加利福尼亞州)|I-880]]和以及[[馬里蘭州|馬里蘭州]]的[[I-270_(馬里蘭州)|I-270]]和[[I-370_(馬里蘭州)|I-370]]。
