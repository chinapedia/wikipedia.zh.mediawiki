'''海軍陸戰隊使館警衛隊'''（{{lang-en|'''Marine Corps Embassy Security Group'''}}<ref name="Marinesmag">{{cite web |url = http://marinesmagazine.dodlive.mil/2010/03/23/marine-corps-embassy-security-group/ |title=Marine Corps Embassy Security Group |date=March 23, 2010 |work=Unit Profile: ''Marines'' Magazine |publisher=United States Marine Corps |accessdate = 1 April 2010}}</ref>），前稱'''海軍陸戰隊衛兵營'''（{{lang|en|'''Marine Security Guard Battalion'''}}），是[[美國海軍陸戰隊|美國海軍陸戰隊]]一支[[營|營級]]的部隊，負責[[美國駐外機構列表|美國駐外大使館]]、[[領事館|領事館]]及美國政府在外機構的保安任務，包括[[布魯塞爾|布魯塞爾]]的美國駐[[北約|北約]]代表處等。隊員稱「海軍陸戰隊衛兵」（{{lang|en|Marine Security Guard}}）。

==歷史==
自美國建國早期至今，海軍陸戰隊與國務院已有一段長遠的合作歷史。從[[第一次巴巴里戰爭|第一次巴巴里戰爭]]中陸戰隊員在[[德爾納|德爾納]]、[[的黎波里|的黎波里]]等地升起[[美國國旗|美國國旗]]，到[[美墨戰爭|美墨戰爭]]時{{Link-en|阿契博德·葛利斯皮|Archibald H. Gillespie|阿契博德·葛利斯皮}}在[[上加利福尼亞省|上加利福尼亞]]的秘密任務，及[[義和團運動|義和團運動]]期間在[[北京|北京]]的作戰等等<ref>[http://www.infzm.com/content/85591 南方周末 - 【纸上军事】中美如何保护驻外使馆]</ref>，海軍陸戰隊已多次於海外的特殊任務中，保衛了衝突區內美國派外人員與僑民的安全。

美國首次正式、永久地指定海軍陸戰隊作為使館警衛的歷史則始於1946年的《外交機構法》（{{lang|en|{{lang|en|Foreign Service Act}}}}），法案中授權[[美國海軍部長|美國海軍部長]]可應[[美國國務卿|國務卿]]要求，派出陸戰隊員擔任駐外機構的保安隊，並接受美國駐該處的高級外交官監督、管理。1948年12月15日簽署了第一份有關陸戰隊員派外相關規定的[[協議備忘錄|協議備忘錄]]。首批海軍陸戰隊員在國務院{{Link-en|美國外交人員學院|Foreign Service Institute|外交人員學院}}受訓後，於1949年初派赴[[摩洛哥|摩洛哥]][[丹吉爾|丹吉爾]]和[[泰國|泰國]][[曼谷|曼谷]]。1954年11月起初級訓練項目交由海軍陸戰隊自行執行。之後1946年份的外交機構法由{{UnitedStatesCode|10|5983}}取代，最新的一份備忘錄則於2008年8月出爐。

在導致美國大使[[克里斯多夫·史蒂文斯|克里斯多夫·史蒂文斯]]喪生的[[2012年班加西攻擊事件|2012年班加西攻擊事件]]發生後，[[美國國會|美國國會]]要求增派陸戰隊衛兵，使得其數量在戰後海軍陸戰隊的裁減趨勢中，反增至原本的近兩倍<ref>[http://www.armytimes.com/news/2012/12/marine-embassy-senate-state-call-more-security-guards-121912/ "Expert: USMC embassy guard boost will be tough."]</ref>。美國海軍陸戰隊從{{Link-en|美國海軍陸戰隊第1團第1營|1st Battalion 1st Marines|第1團第1營}}中調派了一個連作為增援隊<ref>{{cite web |url = http://www.militarytimes.com/article/20130920/NEWS/309200005 |title=Infantry Marines deploy; fill gap in demand for embassy security. |last1=Harkins |first1=Gina |last2=Lamothe |first2=Dan |date=20 September 2013 |website=www.militarytimes.com |publisher=Military Times Group |accessdate=20 September 2013}}</ref>。

==職責==

在大部分場合下，使館警衛隊專門負責駐外使館、領事館建築內部的保安。在最極端的緊急情況下才有權限負責防衛建築外部、甚至替離開館區在外的高級外交人員提供特別保護。若因職責之故，陸戰隊衛兵可享有一定程度的[[外交豁免權|外交豁免權]]<ref name="MSGmission">{{cite web |url=http://www.msgbn.usmc.mil/?pg=company/pub/about/abtMission.htm |title=The mission of the Marine Corps Embassy Security Command |work=Marine Corps Embassy Security Group |publisher=United States Marine Corps |accessdate=2009-03-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20081223180835/http://www.msgbn.usmc.mil/?pg=company%2Fpub%2Fabout%2FabtMission.htm |archivedate=2008-12-23 }}</ref>。

海軍陸戰隊衛兵隸屬於使館警衛隊，但在實際作業上則歸地區安全官管理。

==組織==
目前陸戰隊使館警衛隊為數約1,000人，派駐地點約150個，亦稱「分遣處」（{{lang|en|detachments}}），劃分為9個地區警衛司令部，轄區分布可達全球135個國家、18個[[時區|時區]]，包括了其在[[維吉尼亞州|維吉尼亞州]]{{Link-en|匡提科海軍陸戰隊基地|Marine Corps Base Quantico|匡提科基地}}的隊部<ref name="new CO"/>。匡提科的隊部連和海軍陸戰隊衛兵學校共有100人，提供行政、後勤、法律、培訓及教育支援。

每個分遣處由一名[[參謀士官|參謀士官]]負責指揮，出任者的軍階通常介於[[上士|上士]]至[[一等士官長|一等士官長]]之間，職稱為「陸戰隊分遣處司令」（{{lang|en|Marine Detachment Commander}}），是少數帶有「司令」兩字的陸戰隊士官職稱。分遣處司令一般需服滿2個役期，每期18個月，婚姻狀態可為已婚（但下屬隊員需為單身）。每個分遣隊的規模至少需有5名隊員、一名司令，使得駐在館處隨時有人守崗，同時使每個隊員可以進行慣例訓練、內部管理或是獲得放假時間。

一名陸戰隊衛兵通常需服滿3個役期，每期各12個月。隊員階級通常為[[一等兵|一等兵]]至上士。

==服役==
[[File:USMC-17649.jpg|thumb]]
[[File:Marine_Corps_Security_Guard_Ribbon.svg|thumb]]
在一外交代表處服完24個月役期的海軍陸戰隊衛兵可獲得{{Link-en|海軍陸戰隊衛兵勳表|Marine Corps Security Guard Ribbon}}，或是[[服役星章|服役星章]]。

任何{{Link-en|美國軍職代碼|United States military occupation code|軍職專長}}的海軍陸戰隊員均可志願申請加入警衛隊服一次3年的{{Link-en|服役期|tour of duty|役期}}<ref name="MSGFAQ">{{cite web|url=http://www.msgbn.usmc.mil/?pg=company/pub/bmsg/default.htm|title=Frequently Asked Questions (FAQ's)|work=Marine Corps Embassy Security Group|publisher=United States Marine Corps|accessdate=2009-03-17|deadurl=yes|archiveurl=https://web.archive.org/web/20081221214359/http://msgbn.usmc.mil/?pg=company%2Fpub%2Fbmsg%2Fdefault.htm|archivedate=2008-12-21}}</ref>，然而有攜眷的非參謀士官不具申請資格，若是身上有[[刺青|刺青]]者、有違法紀錄者、非[[美國公民|美國公民]]者、具[[雙重國籍|雙重國籍]]者及任何原因致未能通過{{Link-en|安全許可|security clearance|安全調查}}而無權接觸機密者，也不得申請<ref name="MSFPFS">{{cite web |url=http://www.msgbn.usmc.mil/?pg=company/pub/school/Plan%20For%20success.htm |title=Plan For Success |work=Marine Corps Embassy Security Group |publisher=United States Marine Corps |accessdate=2009-03-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20070707165558/https://www.msgbn.usmc.mil/?pg=company%2Fpub%2Fschool%2FPlan%20For%20success.htm |archivedate=2007-07-07 }}</ref>。

在分派到某駐外館處之前，一名接受使館安全訓練的海軍陸戰隊員必須在匡提科基地成功完成訓練項目。戍衛海外館處或重要的美國駐外設施是海軍陸戰隊中的特殊職位之一，有時亦可能帶有危險性，例如1975年[[西貢陷落|西貢陷落]]時，[[查爾斯·麥馬漢與達爾文·賈基|麥馬漢與賈基]]這最後兩位喪命於[[越戰|越戰]]的美國軍人，當時駐守著[[常風行動|美軍撤僑行動]]中經指定為集合地點的設施。1979年{{Link-en|美國駐巴基斯坦大使館縱火案|1979 U.S. embassy burning in Islamabad|駐巴基斯坦使館縱火案}}及1998年[[1998年美國大使館爆炸案|駐肯亞使館爆炸案]]中都有陸戰隊衛兵殉職。
