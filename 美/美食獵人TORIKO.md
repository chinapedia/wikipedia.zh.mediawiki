《'''美食獵人'''》（{{lang-ja|トリコ}}）是日本漫画家[[島袋光年|島袋光年]]的漫畫作品。於2008年6月至2016年11月在《[[週刊少年Jump|週刊少年Jump]]》上連載，台灣則在《[[寶島少年|寶島少年]]》上連載，漫畫全43卷已完結。在2011年推出動畫，人氣也更加上升<ref>[http://www.geocities.jp/wj_log/rank/  日本漫畫累計銷量 ]</ref>。

== 劇情大綱 ==
{{Main|美食獵人TORIKO角色列表}}

本作屬於[[奇幻|奇幻]]冒險類漫畫作品，作品中的[[世界觀|世界觀]]即世界是一個充滿[[幻想|幻想]]的珍奇[[食材|食材]]的'''美食世界'''。這些幻想的食材例如，全身的肉在舌間融化並且浮出一層油的野獸，一整年都結有美味[[蝦|蝦]][[蟹|蟹]]肉的樹、流出源源不絕的香醇[[白蘭地|白蘭地]]湧泉的泉水等等。

但是這些美味的食材不是兇猛無比的怪物，就是生長在環境惡劣的天險之地。因此在這個探求未知美味的'''美食時代'''裏，有許多專門深入絕地，拼上性命捕獲珍稀美味的'''美食獵人'''。

每個美食獵人的人生目標就是用自己親手捕獲的食材完成包含[[前菜|前菜]]、[[湯|湯]]、[[魚|魚]]、[[肉|肉]]、[[主菜|主菜]]、[[沙拉|沙拉]]、[[甜點|甜點]]、[[飲料|飲料]]一共8道料理的'''人生菜單'''。

主要的故事主軸即為美食獵人'''四天王'''之一的'''特瑞科'''為了完成自己的[[全餐|全餐]][[菜單|菜單]]，和[[廚師|廚師]]'''小松'''一同在美食世界尋找珍奇美食的旅程。

== 設定 ==
; 美食獵人
: 是尋求未知的美食而在全世界到處奔走，捕獲、採取各種各樣的食材探險獵人的职业，一些高級餐廳會與美食獵人訂下契約，藉此確保餐廳能擁有一定質與量食材供應。

* '''點穴'''
: 美食獵人在獵捕時都應該具備此技術「敲進」，通常都是用細針等道具，刺激小腦中所掌管的運動神經組織，讓獵物處於暫時麻痺狀態，在活著的狀態下原封不動的賣出，但這是只有經驗豐富的美食獵人才能習得的技術，隨著使用者的習慣不同亦有各種不同類型的點穴鎗，而像特瑞科這樣的高手則可以手指就直接進行點穴，另外，美食獵人也可以運用點穴來強化自己的身體。
* '''威嚇'''
: 為使用者本身的氣魄化身，單純一氣勢的方式驅趕其他較自己弱的生物，能夠達到不戰而屈人之兵的效果，隨著使用者與實力不同，也會有不同的形象與強度，外觀多數為體內美食細胞的怪物形象。
* '''Preshot Routine'''
: 在現實生活中也存在的動作，藉由做出某些特定的動作，來集中自己的注意力，以提高接下來招式的威力與完美程度，而擊前例行動作做得越好，接下來的招式就越能得到發揮，據特瑞科所言誰都有擊前例行動作，但本作目前為止實際展現過的僅止特瑞科與美食騎士的瀧丸。
* '''脫力'''
: 在攻擊前讓全身都進入放鬆的狀態，藉由這樣使接下來的攻擊速度與威力得到提升，目前僅止有美食黑社會的會長瑪奇使用過此招式。
* '''直覺'''
: 潛藏能力的一種，對處理某件事務累積了無數次的經驗後，開始放棄思考而從經驗中選出最佳對策，即為直覺；累積了無數次戰鬥經驗產生的直覺，即便是面對初次見面的對手，亦能在瞬間找出對方的要害發揮作用；但也相當難以掌握這股力量，不但要有龐大的經驗還必須學習停止思考，不論是前者後者都非一朝一夕能夠掌握的，不過自然界中亦有著像鬥狼這樣天生就擁有直覺力的特殊生物存在。
* '''食義與食沒'''
: 藉由內心時常保持對世間萬物的感謝，而讓使用者的注意力與集中力又達到更高境界的能力，藉此捨棄掉多餘的動作減少消耗、提高動作的精準度，甚至能達到身體層面的改變，使熱量的消耗減少減慢、增加能儲存的熱量，亦是本作中要食用某些特殊食材必要的心境。食沒則又是更高的境界，不單是感謝一切亦必須感謝那些曾化作自己血肉的生命、感謝自己的身體，亦有一份對於死亡領悟包含其中。
* '''Ultimate Routine(極致的臆斷)'''
: 由愛丸傳授特瑞科的「終極」秘技，於漫畫234話出現，被愛丸稱為「只有學習過食沒的人」才可以學習的王者的動作。目前已知愛丸、特瑞科、史塔俊等人皆能使用。
* '''美食細胞'''
: 圍繞著本作中戰鬥勝負的關鍵之一，藉由品嘗擁有美食細胞的食物後體內出現的萬能細胞，能使擁有者的身體得到超乎常人的強化，即便是瀕死的生命也有可能因此而復活，但也藏著相對的風險，如果體內的熱量過低會發生細胞反噬的現象"自食"，讓人瞬間死亡；隨著每個人天生的差異細胞會有著所謂"壁"的存在，這象徵每個人細胞強度的界線，但可以藉由吃下與自己細胞相符的美味食材來超越"壁"，使自己的力量再進化。
* '''自噬作用'''
: 在身體熱量過低的狀況，細胞為了取得足夠的熱量開始吞噬身體，在身體被吞食殆盡前，必須盡快補充熱量否則將會致死，但自食期間熱量會短暫回復，因此在戰鬥中亦有可能成為逆轉的契機。
* '''全餐菜單（人生のフルコース）'''
: 美食獵人會把想要的食材分別作為目標和理想的「全餐菜單」懸掛，清單上的食材基本上寫有自己捕獲的經驗，根據清單上寫的食材的捕獲等級能看出那個美食獵人的實力(也有例外，只是為了理想而製作的全套菜單，如：猶達)，根據菜單編列總共分為8種。

:* 生菜沙拉
:* 甜點
:* 飲料

; 廚師（料理人）
: 在整個美食時代中與美食獵人等同重要的存在，獵人負責捕獲食材料理人負責使食材更加美味，對美食獵人來說料理人是相當重要的存在，如同IGO會長一龍所言「對於供應食材的美食獵人來說，烹飪食材的廚師是比什麼都重要的人」、「不管有多少會供應多棒食材的美食獵人，如果沒有能烹飪的人的話，那就沒有任何價值」，這些話也顯示了料理人確確實實是支撐了整個美食時代的重要一環。

; 再生獵人
: 再生獵人在美食時代亦有著舉足輕重的地位，在整個美食時代中大自然的食材不可能是永無止盡的，再生獵人的使命就在於此，他們負責不讓整個美食時代與大自然的平衡崩毀，將瀕臨滅絕的物種保存下來，以他們獨特的技術讓物種能再次在自然界中正常的繁衍下去，而他們還負責治療病痛與將滅絕的生物復活等工作，毫無疑問他們也是整個美食時代能夠發展至今的大功臣。

; 研磨師
: 對於料理人來說研磨師與美食獵人是等同重要的存在，如果說槍是軍人與警察的靈魂所在，那把菜刀說是料理人的靈魂也不為過，而為料理人打造出這一份靈魂的正是研磨師，對於不斷烹飪的料理人來說，提供料理器具與打磨器具的研磨師簡直就可以稱作是料理人的另一位夥伴，他們精細的鍛造出每一把的菜刀陪伴著無數的料理人度過每一天，與他們一同在他們的戰場奮戰著，用每一把灌注了自己靈魂的菜刀陪伴料理人切開無數的食材，製造出無數美味的佳餚。

; IGO
: 「國際美食機構」的略稱， 決定野生的野獸的捕獲等級，再用高價購買，本來只不過是聯合國的一個小機關，隨著美食家食材的需求量增加而從聯合國獨立，現在加盟國數超過聯合國遍及了360個國家，常在IGO直屬飯店舉行美食家各國首腦聚會等等，互相比較財力和政治力。
* '''庭院'''：人工製造的動植物生態空間，在無限接近大自然的狀態下放養許多野生動植物，IGO都會在此處做生態調查。
