[[美国|美国]]拥有世界上最发达的传播媒介系统。美国[[媒体|媒体]]涵盖了所有主要的形式，包括[[电视|电视]]、[[广播|广播]]、[[电影|电影]]、[[报纸|报纸]]、[[杂志|杂志]]和[[互联网|互联网]]。美国的[[音乐|音乐]]产业也很发达。大多数媒体都从属于商业化的媒体公司。这些公司依靠[[广告|广告]]、订阅以及出售[[版权|版权]]盈利。

[[美国|美国]]的媒体往往具有国际性的特征，其经营范围超越国界，在全球范围内盈利。目前，美国媒体集中、垄断的趋势还在持续，造就了大量的国际性的传媒巨头，如美国三大传媒公司[[迪士尼|迪士尼]]、[[时代华纳|时代华纳]]和[[新闻集团|新闻集团]]。

导致美国媒体日益集中和垄断的趋势，主要原因是[[美国政府|美国政府]]政策上的倾斜，以及商業因素的影響。

==电视==
{{seealso|美国电视}}

美国的电视和广播是在[[联邦通讯委员会|联邦通讯委员会]]的管理下运行的。全美的地方电视台有几千家，其中绝大多数分别从属于五大全国电视网。传统的三大电视网包括[[全国广播公司|全国广播公司]]（[[NBC|NBC]]）、[[美国广播公司|美国广播公司]]（[[ABC|ABC]]）和[[哥伦比亚广播公司|哥伦比亚广播公司]]（[[CBS|CBS]]）。近些年崛與相互合併的有兩大电视网，分别是[[福斯广播公司|福斯广播公司]]（[[Fox|Fox]]）與由哥倫比亞廣播公司（CBS）及[[華納兄弟|華納兄弟]]（Warnar Bros.）共同出資的[[CW電視網|CW電視網]]（[[CW|CW]]）。除上述五个私营电视网外，美国还有一个非營利性、靠政府津贴、在美國民間進行籌款活動，與接受私人基金捐贈方式生存的[[公共電視網|公共電視網]]（[[PBS|PBS]]）。

除上述的这些无线电视网之外，美国还存在很多[[有线电视|有线电视]]频道。这些频道不靠广告盈利，其主要收入来源是收视费。其中实力雄厚的有两家，分别是家庭影院频道（[[HBO|HBO]]）和有线新闻网（[[CNN|CNN]]）。

[[联邦通讯委员|联邦通讯委员]]会目前正致力于将传统的模拟电视信号[[制式|制式]]NTSC转换为新的[[数字电视|数字电视]]信号制式ATSC。

尽管美国声称自己拥有全世界范围内最充分的[[新闻自由|新闻自由]]，但是美国的电视媒体在言论上仍然要遵守[[最高法院|最高法院]]的规定。电视媒体需要受到一定程度的管制，一方面是因为电视媒体所利用的电磁波频谱资源属于公共资源，而且非常紧缺；另一方面则是因为比起报纸、杂志等平面媒体来，电视媒体的内容很难预先控制。

==广播==
美国的[[广播|广播]]有两种制式，分别是[[调频|调频]]（[[FM|FM]]）和[[调幅|调幅]]（[[AM|AM]]）。美国最常见的两类电台分别是谈话台和音乐台，其中音乐台所占比重很大，分类也十分细致，例如[[流行音乐|流行音乐]]、[[乡村音乐|乡村音乐]]、[[爵士乐|爵士乐]]等等，都有自己专门的电台。美国的广播公司在近几年也出现了集中整合的趋势。[[全国公共广播电台|全国公共广播电台]]是美国最大的公共广播网络。然而大多数的美国电台都是私营的商业电台。

1987年，[[联邦通讯委员会|联邦通讯委员会]]废除了早先的“[[公平原则|公平原则]]”，这一政策上的变化导致美国的谈话电台在90年代迎来了繁荣发展的黄金时期。所谓“公平原则”，就是广播的内容必须同时陈述各种不同的意见和观点。广播界著名人物[[拉什·林堡|拉什·林堡]]在这一过程中起到了重要作用。

近些年，[[联邦通讯委员会|联邦通讯委员会]]声言支持发展数字化广播技术。FM和AM两个波段下的电台可以在不影响当前模拟信号广播的前提下逐渐引进数字技术。等到数字技术可以彻底取代模拟技术的时候，高清晰度的广播将取代传统的广播。

== 电影 ==
{{seealso|美国电影}}

20世纪的美国[[电影|电影]]是美国所有产业中经营得最成功，也是发展最为壮大的传媒产业。由于[[全球化|全球化]]浪潮的作用，美国的电影业和其他文化产品出口至世界其他国家，对美国经济的发展起到了巨大的促进作用。

美国电影的大量出口导致全世界在一定程度上都在接受美国文化的影响。当然，很多国家对这一现状非常不满，并想尽办法来抵制美国电影的进口并大力发展本国的电影产业，[[法国|法国]]就是在这一问题上态度最强硬的国家之一。

==报纸==
[[File:the_new_york_times_building_in_new_york_city.jpg|thumb]]
{{seealso|美国报纸}}

近些年来，[[报纸|报纸]]对美国社会的影响力正在逐渐衰弱。在美国，并没有真正意义上的全国性报纸，尽管影响力甚大的《[[纽约时报|纽约时报]]》、《[[华尔街日报|华尔街日报]]》和《[[华盛顿邮报|华盛顿邮报]]》在全美多数大城市内发行。综合性的《[[纽约时报|纽约时报]]》的在政治上是左派媒体的旗舰，而以商业、金融报道为主的《[[华尔街日报|华尔街日报]]》和偏向于报道美国国内政治的《[[华盛顿时报|华盛顿时报]]》则偏右。

尽管《纽约时报》的主要读者群集中在以[[纽约|纽约]]为代表的大城市，但该报在历史中逐渐发展成了一张具有全国影响力的主流大报。该报被誉为“美国社会的忠实记录”，其新闻报道的视角遍及全国。美国境内主要的[[图书馆|图书馆]]内都收藏有《纽约时报》过刊的影印版，很多[[历史学|历史学]]家和司法界人士甚至援引该报上刊登的报道，当作学术研究或法庭断案的依据。除《[[纽约时报|纽约时报]]》外，《[[华尔街日报|华尔街日报]]》和《[[华盛顿邮报|华盛顿邮报]]》的影响力也很大。近几年逐渐崛起的《[[今日美国|今日美国]]》曾经试图将自己打造成一张全国性的报纸，但学术界则揶揄其为“快餐报”，因为该报虽有逾百万销量，却只关注娱乐、体育等领域的话题，而缺乏主流媒体应有的严肃性。其影响力远远不及三大报。

除了上述四张报纸之外，几乎每个主要的大都市区都会发行一些本地报纸。通常一个大城市可以同时并存两张综合性日报，以及不计其数的小型专业性报纸（如娱乐报、体育报等）。尽管随着纸价的上涨和劳动力成本的提高，报纸的印刷和发行费用在逐年上涨，但报纸的价格却始终保持在比较低的水平，这就迫使报纸的经营者们更多的依赖刊登[[广告|广告]]来创收，并更多的采用[[通讯社|通讯社]]的供稿。以为报纸供稿为生的大通讯社包括美国的[[美联社|美联社]]（[[美联社|AP]]）和[[英国|英国]]的[[路透社|路透社]]（Reuters）等。这些大通讯社通常都是国际性的。

绝大多数美国的报纸都是私营的，它们要么属于大型的报业集团，如[[甘乃特报团|甘乃特报团]]就拥有几十家美国报社；要么从属于小型的报业连锁机构；要么则是从属于某个家族或个人。最后一种情况目前已经非常罕见了。

美国的多数综合性报纸都是周报或日报。周报通常每周发行一次，在星期四或星期五，日报则要每天发行。周报的发行量通常较小，而且在[[郊区|郊区]]或[[小城镇|小城镇]]比较普遍。在大城市，通常也存在一些被称为“[[另类周刊|另类周刊]]”的周报，作为对综合性日报的补充，比较著名的“另类周刊”包括纽约的《[[乡村之声|乡村之声]]》和洛杉矶的《[[洛杉矶周报|洛杉矶周报]]》。此外，大城市中也往往存在一些本地的经济类报纸，主要报道本城的[[工商|工商]]业、[[金融|金融]]业、就业等方面的新闻。外裔人居住的社区和一些社会团体通常也有自己的报纸。

行业刊物《[[编辑与出版|编辑与出版]]》的调查显示：由于来自其他媒体的竞争，美国日报的数量在过去的半个世纪中骤减。[[晚报|晚报]]的数量几乎降至1970年的一半，晨报和周末版日报的数量却有所增加。

在1950年，全美共有1722张日报，其中70％是晚报。然而到了2000年，日报的数量仅剩1480份，其中晚报的数量为766，仅占日报总数的一半。

美国日报的发行数量也在逐渐的减少。在大城市中，一般只能容纳两张综合性日报的存在。其他实力相对较弱的报纸只能被吞并或破产。下表是美国近半个世纪日报发行总数的比较（单位：百万）：

[[广告|广告]]收入是报纸的主要经济来源。广告的形式包括分类广告，或夹带在报纸中的广告增刊。然而，从90年代晚期起，这一收入来源也受到了来自网上交易平台[[eBay|eBay]]、[[Monster.com|Monster.com]]和[[Craigslist|Craigslist]]等的威胁。

美国发行量最大的报纸是《[[今日美国|今日美国]]》、《[[华尔街日报|华尔街日报]]》、《[[纽约时报|纽约时报]]》和《[[洛杉矶时报|洛杉矶时报]]》

== 杂志 ==
{{seealso|美国杂志}}

得益于北美巨大的英文媒体市场，[[美国|美国]]的杂志业非常发达，[[杂志|杂志]]的类型和内容几乎无所不包，主要杂志在美国任何一座大城市的报摊上都可以买到。绝大多数杂志社都隶属于某个大型媒体公司或这些媒体公司在本地的下属企业。

美国有三大主流新闻类周刊，分别是《[[時代|時代]]》、《[[新闻周刊|新闻周刊]]》以及《[[美国新闻与世界报道|美国新闻与世界报道]]》。前者的政治倾向偏左，而后两者则偏右。尽管如此，三大周刊都宣称致力于为受众提供客观公正的报道，并将带有个人偏见的内容局限在评论版。《时代》以每年评选年度风云人物的活动而著称，而《[[美国新闻与世界报道|美国新闻与世界报道]]》最著名的是对全美学院和大学一年一度的排名。

此外，美国还有十余种主要时政类杂志，这些杂志的政治倾向各不相同。

除这些以满足公众阅读兴趣的大众杂志之外，美国还存在很多专业的[[学术|学术]]类杂志，由特定的组织发行，目标读者群也较有针对性。例如专门给计算机专业人士读的《[[ACM通讯|ACM通讯]]》以及专门给[[法律|法律]]界人士读的《[[美国律师协会期刊|美国律师协会期刊]]》等。

==互联网==
[[互联网|互联网]]的诞生为美国媒体传播信息提供了新的手段。这一媒介的一个优势就是可以方便读者查阅以前的旧新闻。基本所有的主流[[媒体|媒体]]都拥有自己的[[网站|网站]]。一些媒体网站上的信息是完全免费的，而有一些媒体则只免费开放一部分，想阅览全部则需要付费。显然前者的影响力更大一些，因为这更加符合信息自由的宗旨。通常，当一位阅览者打开一个新闻类网站，一个巨大的“付费”标志突然出现在自己面前，这位阅览者往往会对该网站产生一定程度的反感，而实际上能够提供这种信息服务的免费网站不胜枚举。

这些依靠提供即时信息服务为生的网站实际上正在改写美国媒体的历史，其影响力是人们以前没有预测到的。尽管有些敏感信息，如[[政府|政府]]重要文件等，在解密之前仍不太可能在[[互联网|互联网]]上公布，但新闻类网站的这种即时、快捷的信息服务在不久的将来必将成为公众获取新闻的主要来源。

== 立場 ==
