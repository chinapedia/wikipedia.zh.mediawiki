'''美国铸币局'''（{{lang-en|'''United States Mint'''}}）是[[美国|美國]]負責製造流通性[[貨幣|貨幣]]和控管黃金儲備的政府機構，於1792年創立於費城。它並不製造紙幣。目前有四個主要的鑄造據點：費城、丹佛、舊金山和西點。

== 簡介 ==
鑄幣局乃根據[[美國國會|美國國會]]通過的[[1792年鑄幣法案|1792年鑄幣法案]]成立，設於[[美國國務院|美國國務院]]之下。根據法案條文，第一幢鑄幣大樓位於費城，即當時的美國首都。鑄幣局的首任總監是著名科學家[[戴维·里顿豪斯|戴维·里顿豪斯]]，現時則為[[:en:Edmund_C._Moy|Edmund C. Moy]]。1799年，鑄幣局成為獨立組織，而根據[[1873年鑄幣法案|1873年鑄幣法案]]，鑄幣局成為[[美國財政部|美國財政部]]轄下機構。1981年起，鑄幣局向[[美国财政部司库|美国财政部司库]]負責。

== 歷史 ==
鑄幣1局在費城的大樓名為''Ye Olde Mint''（老铸币局），其第一批分局設於[[夏洛特_(北卡羅萊那州)|夏洛特 (北卡羅萊那州)]]（幣上有C標記）（1838–1861年）、[[達洛尼加|達洛尼加]]（幣上有D標記）（[[:en:Dahlonega,_Georgia|Dahlonega]]）（1838–1861年）和[[新奧爾良|新奧爾良]]（1838–1909年）（幣上有O標記）。[[夏洛特鑄幣局|夏洛特鑄幣局]]和[[達洛尼鑄幣局|達洛尼鑄幣局]]附近有金礦，兩地分局主要鑄造金幣。[[美國南北戰爭|美國南北戰爭]]爆發，兩局被關閉，而[[新奧爾良鑄幣局|新奧爾良鑄幣局]]則於南北戰爭（1861）爆發之初關閉，直至[[重建時期|重建時期]]末的1879年才重開。它負責鑄造11種面值的貨幣，但只曾鑄造其中10種（1851年的銀[[3美分硬幣|3美分硬幣]]、[[5美分硬幣|5美分硬幣]]、[[10美分硬幣|10美分硬幣]]、[[25美分硬幣|25美分硬幣]]、[[50美分硬幣|50美分硬幣]]，以及金[[1美元硬幣|1美元硬幣]]、[[四分之一鹰金币|四分之一鹰金币]]（[[:en:Quarter_eagle|quarter eagle]]）、[[半鹰金币|半鹰金币]]（[[:en:Half_Eagle_(U.S._coin)|half eagles]]）、[[鹰金币|鹰金币]]（[[:en:Eagle_(U.S._coin)|eagles]]）和[[双鹰金币|双鹰金币]]（[[:en:Double_Eagle_(U.S._coin)|double eagles]]）。

新的鑄幣設施在1870年於有大量銀礦的[[卡森城_(內華達州)|卡森城 (內華達州)]]設立（CC標記），1893年關閉。1920年，美國於當時的殖民地[[菲律賓|菲律賓]]的[[馬尼拉|馬尼拉]]設立鑄幣分局，是美國歷來唯一設在美國本土境外的鑄幣廠，當時負責鑄造於菲律賓流通的硬幣（面值1、5、10、20和50[[:en:centavo|centavo]]（菲律賓以及拉丁美洲的貨幣單位））。該廠於1920至1922年鑄幣，然後在1925至1941年（直至[[二戰|二戰]]爆發），其鑄造的硬幣有M標記或沒有標記。

2000年，鑄幣廠已鑄造280億個[[美國硬幣|美國硬幣]]，參見[[美国硬币年发行量一览|美国硬币年发行量一览]]。

== 現時設施 ==
[[File:Denver_mint.jpg|left]]

鑄幣局最大規模的設施位於費城，是四個運作中的鑄幣廠的其中之一。現時的[[費城鑄幣局|費城鑄幣局]]開設於1969年，已是當地第四代鑄幣廠。第一代建於1792年，1793年投入運作。直至1980年，除了少數硬幣，在費城鑄造的硬幣沒有鑄上標記。到1980年，該廠鑄造的硬幣加上P標記，除了[[1美分硬幣|1美分硬幣]]。費城負責鑄造紀念套幣（[[:en:proof_coin|proof coin]]）直至1968年。現時費城鑄造廠負責製造印模，雕版和設計部門也設於該廠。

[[丹佛鑄幣局|丹佛鑄幣局]]前身為官方檢驗[[黃金|黃金]]的地方，開設於1863年，即當地發現金礦的5年後。20世紀初，每年處理總值500萬美元的金和銀，於1906年，鑄幣局於該處成立鑄幣分局，標記為D。

[[三藩市鑄幣局|三藩市鑄幣局]]因應[[加利福尼亞淘金潮|加利福尼亞淘金潮]]而在1854年開設，標記為S。它曾在1955年停產，後來於1960年代中的硬幣短缺期間重開。1968年，它取代費城鑄幣廠鑄造紀念套幣，而由1975年起，它幾乎只鑄造紀念套幣。

[[西點鑄幣局|西點鑄幣局]]位於[[紐約|紐約]]西點，是最新的分局。其前身為西點金庫，於1937年成立。1973至1986年，[[美分|美分硬幣]]在該廠鑄造。於1988年3月31日該廠正式成為鑄幣分局，現時是美國唯一鑄造[[黃金|黃金]]、[[银|白銀]]和[[白金|白金]]美國鷹幣（[[:en:American_Eagle_bullion_coins|American Eagle bullion coins]]）的地方。

位於[[諾克斯堡|諾克斯堡]]的美國金庫（United States Bullion Depository），儲存美國國庫黃金儲備。

== 功能 ==

美國鑄幣廠的功能為：

* 製造硬幣；
* 製造和銷售全國性紀念章；
* 設計和製造[[國會金質獎章|國會金章]]；
* 設計、製造和銷售紀念硬幣；
* 保管黃金儲備；
* 為官方用途支付金或銀；
* 向[[聯邦儲備銀行|聯邦儲備銀行]]發行硬幣。

根據財政部新規定，鑄幣局有權向使用其名字、符號和財政部名字和符號作宣傳和其他用途的人罰款最多$25,000美元。<ref>[http://dig4coins.com/News/1-latest_news/97-us-mint-to-fine-any-misuse-of-its-names-and-symbols.html U.S. Mint to Fine any Misuse of Its Names and Symbols] {{webarchive|url=https://web.archive.org/web/20080109100657/http://dig4coins.com/News/1-latest_news/97-us-mint-to-fine-any-misuse-of-its-names-and-symbols.html |date=2008-01-09 }}</ref>

== 刻印 ==
[[File:United_States_penny,_obverse,_2002.png|thumb]]

* '''[[1美分|1美分]]'''：刻印在正面，[[林肯|林肯]]半身像側的年份之下，沿用其1909年推出時的樣式

* '''[[5美分|5美分]]'''：1968至2005年，刻印在硬幣正面近邊緣位置；2006年重新設計，改為在年份的下面

* '''[[10美分|10美分]]'''：刻印在正面，[[富兰克林·德拉诺·罗斯福|富兰克林·德拉诺·罗斯福]]半身像側的年份之上

* '''[[25美分|25美分]]'''：刻印在[[喬治華盛頓|喬治華盛頓]]半身像側 

* '''[[50美分|50美分]]'''：刻印在[[约翰·肯尼迪|约翰·肯尼迪]]半身像下、年份之上

* '''[[萨卡加维亚1美元纪念币|萨卡加维亚1美元纪念币]]'''（2001年至現在）：刻印在年份下

* '''[[美国总统1美元纪念币|美国总统1美元纪念币]]'''：刻印和年份印在硬幣邊緣

[[File:War_Nickle.jpg|thumb]]上]]
由於[[二次大戰|二次大戰]]時[[鎳|鎳]]出現短缺，5美分硬幣的成份加入[[銀|銀]]。為以茲識別，在費城鑄造的5美分加上P標記，而在三藩市鑄造的5美分其刻印也改為在蒙蒂塞洛之上，1943年丹佛鑄造的5美分也如是。直至1946年，5美分成份改回如戰前一樣，刻印的位置復原。
