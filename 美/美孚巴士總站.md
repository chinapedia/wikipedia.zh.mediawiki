'''美孚巴士總站'''（英文：'''Mei Foo Bus Terminus'''）是[[香港|香港]][[深水埗區|深水埗區]]其中一個主要的巴士總站，位於[[荔枝角大橋|荔枝角大橋]]橋底，鄰近[[美孚新邨|美孚新邨]]及[[港鐵|港鐵]][[美孚站|美孚站]]。

巴士總站於1970年代落成，是[[九龍|九龍]]市區巴士路線的主要巴士總站之一。初時名為'''荔枝角（橋底）巴士總站'''，直至1980年代，[[香港地鐵|香港地鐵]]荔灣站更名為美孚站後，本站在1985年5月31日才正名為'''美孚巴士總站'''。現在以美孚為終點站的巴士路線甚多，目的地差不多涵蓋全香港主要地區，亦是通宵巴士主要轉乘站。

==總站路線資料==
[[File:MeiFooPTI_RouteInfo.JPG|right]]
===[[九龍巴士2A線|九龍巴士2A線]]===
*{{HK Bus Route KLNT Show:2A|format=inline}}
1952年12月22日開辦，途經[[九龍城|九龍城]]、[[旺角|旺角]]、[[長沙灣|長沙灣]]。

===[[九龍巴士2X線|九龍巴士2X線]]===
*{{HK Bus Route KLNT Show:2X|format=inline}}
2015年12月19日投入服務，途經[[彩德邨|彩德邨]]、[[彩盈邨|彩盈邨]]、[[牛頭角|牛頭角]]、[[旺角|旺角]]、[[深水埗|深水埗]]及[[長沙灣|長沙灣]]，此線取道九龍城的天橋並有限停站，為四彩一帶提供來往九龍西的服務。

===[[九龍巴士6C線|九龍巴士6C線]]===
*{{HK Bus Route KLNT Show:6C|format=inline}}
1952年8月投入服務，途經[[長沙灣|長沙灣]]、[[深水埗|深水埗]]、[[旺角|旺角]]、[[油麻地|油麻地]]、[[紅磡|紅磡]]、[[土瓜灣|土瓜灣]]。

===[[九龍巴士6D線|九龍巴士6D線]]===
*{{HK Bus Route KLNT Show:6D|format=inline}}
1958年10月15日：路線投入服務，來往[[荔枝角|荔枝角]]及[[牛頭角|牛頭角]]1968年1月13日：縮短至[[長沙灣|長沙灣]]。2013年8月24日：配合[[九龍巴士40線|九龍巴士40線]]更改路線，總站由甘泉街重返至美孚。途經：[[長沙灣道|長沙灣道]]、[[通州西街|通州西街]]、[[青山道|青山道]]、[[大埔道|大埔道]]、[[白田街|白田街]]、[[窩仔街|窩仔街]]、[[大坑東道|大坑東道]]、[[界限街|界限街]]、[[太子道東|太子道東]]、[[彩虹道|彩虹道]]、彩虹通道、[[觀塘道|觀塘道]]、[[牛頭角道|牛頭角道]]及[[振華道|振華道]]。

===[[九龍巴士38A線|九龍巴士38A線]]===
*{{HK Bus Route KLNT Show:38A|format=inline}}
1988年11月1日投入服務，配合[[海濱花園_(住宅)|海濱花園]]入伙，路線在未加入[[空調巴士|空調巴士]]行走前，曾經是[[荃灣區|荃灣區]]四條全以普通巴士行走的對外路線之一。(另外三條為[[九龍巴士32B線|九龍巴士32B線]]、[[九龍巴士33A線|九龍巴士33A線]]、[[九龍巴士39A線|九龍巴士39A線]])。

===[[九龍巴士46X線|九龍巴士46X線]]===
*{{HK Bus Route KLNT Show:46X|format=inline}}
1990年4月20日投入服務，途經[[隆亨|隆亨]]、[[大圍|大圍]]、[[梨木樹|梨木樹]]、[[葵涌|葵涌]]、[[荔景|荔景]]。由於本線是唯一一條全日途經[[城門隧道|城門隧道]]往大圍和唯一一條往[[九龍|九龍]]的城門隧道巴士路線，所以客量相當龐大。

===[[九龍巴士86線|九龍巴士86線]]===
*{{HK Bus Route KLNT Show:86|format=inline}}
1979年12月10日投入服務，途經[[圓洲角|圓洲角]]、[[沙田第一城|沙田第一城]]、[[禾輋邨|禾輋邨]]、[[瀝源邨|瀝源邨]]、[[沙田市中心|沙田市中心]]、[[獅子山隧道|獅子山隧道]]、[[深水埗|深水埗]]、[[長沙灣|長沙灣]]。

===[[九龍巴士98C線|九龍巴士98C線]]===
*{{HK Bus Route KLNT Show:98C|format=inline}}
1993年10月7日投入服務，來往[[坑口（北）巴士總站|坑口（北）]]和[[美孚|美孚]]，提供[[坑口|坑口]]、[[寶林|寶林]]、[[翠林|翠林]]、[[康盛花園|康盛花園]]、[[寶達|寶達]]來往[[亞皆老街|亞皆老街]]、[[旺角|旺角]]、[[太子道|太子道]]、[[深水埗|深水埗]]、[[長沙灣|長沙灣]]、[[荔枝角|荔枝角]]的巴士服務。
 
===[[九龍巴士98S線|九龍巴士98S線]]===
*{{HK Bus Route KLNT Show:98S|format=inline}}
2002年8月11日投入服務，來往[[日出康城|日出康城]]和[[美孚|美孚]]，繞經[[清水灣半島|清水灣半島]]、[[茵怡花園|茵怡花園]]及[[將軍澳隧道|將軍澳隧道]]，只在上午及下午繁忙時間服務。

===[[過海隧道巴士102線|過海隧道巴士102線]]===
*{{HK Bus Route Cross Harbour Show:102|format=inline}}
1972年8月5日隨[[香港|香港]]第一條[[海底隧道|海底隧道]]－[[紅磡海底隧道|紅磡海底隧道]]通車而開辦，屬於首批過海巴士路線，現由[[九巴|九巴]]及[[城巴|城巴]]聯營，來往[[筲箕灣|筲箕灣]]及[[美孚|美孚]]之間，是來往[[港島|港島]][[東區_(香港)|東區]]及[[九龍西|九龍西]]部的重要路線，也是紅隧過海線王。
 
===[[過海隧道巴士102P線|過海隧道巴士102P線]]===
*{{HK Bus Route Cross Harbour Show:102P|format=inline}}
2002年4月29日投入服務，於平日繁忙時間服務，途經[[東區走廊|東區走廊]]、[[西灣河|西灣河]]、[[油麻地|油麻地]]、[[長沙灣|長沙灣]]。本線名義上為一條聯營線，由[[九巴|九巴]]及[[城巴|城巴]]聯營，但現時兩個方向的班次是各自的營運時段派車－－往[[美孚|美孚]]的早上及黃昏往[[筲箕灣|筲箕灣]]的班次都是由九巴負責派車，而往筲箕灣的早上及黃昏往美孚的班次則由城巴負責派車。

===[[過海隧道巴士102R線|過海隧道巴士102R線]]===
*{{HK Bus Route Cross Harbour Show:102R|format=inline}}
1981年9月19日投入服務，由[[九巴|九巴]]及[[新巴|新巴]]聯營的馬場巴士路線，由[[跑馬地馬場|跑馬地馬場]]單向前往[[美孚|美孚]]，途經[[紅磡海底隧道|紅磡海底隧道]]、[[佐敦|佐敦]]、[[旺角|旺角]]及[[深水埗|深水埗]]，只在跑馬地馬場賽馬日提供服務。

===[[過海隧道巴士105R線|過海隧道巴士105R線]]===
*{{HK Bus Route Cross Harbour Show:105R|format=inline}}
2006年7月23日開辦，由[[九巴|九巴]]及[[新巴|新巴]]營運的一條單向由灣仔(會展)往美孚的特別路線，為[[香港書展|香港書展]]夜書市散場服務的兩條路線之一；途經[[紅磡海底隧道|紅磡海底隧道]]、[[油麻地|油麻地]]、[[旺角|旺角]]及[[深水埗|深水埗]]。

[[File:MeiFoo_Midnight.jpg|right]]

===[[過海隧道巴士N122線|過海隧道巴士N122線]]===
*{{HK Bus Route Cross Harbour Show:N122|format=inline}}
1975年12月17日投入服務，現由[[九巴|九巴]]及[[新巴|新巴]]聯營，途經[[筲箕灣道|筲箕灣道]]、[[英皇道|英皇道]]、[[紅磡海底隧道|紅磡海底隧道]]、[[加士居道|加士居道]]、[[彌敦道|彌敦道]]。

===[[九龍巴士N237線|九龍巴士N237線]]===
*{{HK Bus Route KLNT Show:N237|format=inline}}
2002年11月4日開辦，以[[循環線|循環線]]方式行走，由[[美孚|美孚]]出發，途經[[青山公路|青山公路]]、[[石籬邨|石籬邨]]、[[安蔭邨|安蔭邨]]、[[葵涌邨|葵涌邨]]、[[葵盛東邨|葵盛東邨]]、[[葵盛西邨|葵盛西邨]]，最後經[[葵涌道|葵涌道]]返回美孚。

===[[九龍巴士N252線|九龍巴士N252線]]===
*{{HK Bus Route KLNT Show:N252|format=inline}}

===[[九龍巴士N260線|九龍巴士N260線]]===
*{{HK Bus Route KLNT Show:N260|format=inline}}
1997年7月14日投入服務，來往[[屯門碼頭|屯門碼頭]]及[[美孚|美孚]]，提供[[山景邨|山景邨]]、[[兆康苑|兆康苑]]、[[虹橋|虹橋]]、[[屯門市中心|屯門市中心]]及[[友愛邨|友愛邨]]來往[[荃灣|荃灣]]、[[葵涌|葵涌]]及美孚的通宵巴士服務。

===[[九龍巴士N269線|九龍巴士N269線]]===
*{{HK Bus Route KLNT Show:N269|format=inline}}
1999年3月28日開辦，是一條往返[[元朗區|元朗區]]及[[美孚|美孚]]的通宵巴士路線，往來[[天慈巴士總站|天慈]]及美孚，途經[[天水圍|天水圍]]、[[元朗市|元朗市]]、[[荃灣|荃灣]]及[[葵涌|葵涌]]。

==途經路線資料==
===[[九龍巴士6號線|九龍巴士6號線]]===
*{{HK Bus Route KLNT Show:6|format=inline}}
此特別班次於2011年7月17日投入服務，單向由荔枝角經此站開往尖沙咀碼頭，只於平日09:20－16:00及假日07:00－16:00提供服務。

===[[九龍巴士N241線|九龍巴士N241線]]===
*{{HK Bus Route KLNT Show:N241|format=inline}}
1990年11月18日投入服務，來往[[紅磡鐵路站|紅磡鐵路站]]及[[青衣島|青衣]][[長宏邨|長宏邨]]的通宵路線，途經[[長安邨|長安邨]]、[[長康邨|長康邨]]、[[長青邨|長青邨]]、[[葵芳|葵芳]]、[[荔景山路|荔景山路]]、[[青山道|青山道]]、[[旺角|旺角]]、[[尖沙咀|尖沙咀]]等地。此路線主要用途是方便市民於[[九龍巴士42A線|42A]]和[[九龍巴士46線|46線]]收車後仍可乘搭本線回家。此線只於往長宏方向途經本站。
