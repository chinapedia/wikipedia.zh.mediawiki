'''美容'''指為了追求「美」而對於[[身體|身體]]所進行的[[物理性|物理性]]之修繕巧飾之[[行為|行為]]，通常指[[容貌|容貌]]上的。

'''美容'''有三個定義：

# 其一是專指對容貌（面貌）的美化工作。
# 其二泛指所有令容貌及身體更美的行為。
# 其三被泛用到交通、建筑、市政等等方面。如汽車美容。

本條目指第一及第二項。

== 歷史 ==
我們現在每天出門前必做的事情——洗臉，是文字記載的最早的'''美容'''行為。以中國為例，亦有不少文獻透露出古人對審美的價值觀與追求方式，如《[[诗经|詩經]]》云：「素以為絢兮」，素者，白也；然而並不是每個女人都能天生麗質地擁有白皙膚色，於是人們開始用米製作成一種可以增白的粉敷面，在[[夏商周|夏商]]時期，經過進一步改良，敷面的材料換成了鉛，原來含有[[重金属|重金屬]]的[[化妝品|化妝品]]古亦有之。此外，宋玉在《登徒子好色賦》中描繪鄰家女：「著粉則太白，施朱則太赤」，提出了白裡透紅的和諧美的標準<ref>{{Cite web|url=https://fanti.dugushici.com/ancient_proses/71753|title=登徒子好色賦_原文、翻譯及賞析_宋玉詩詞_讀古詩詞網|accessdate=2018-10-16|work=fanti.dugushici.com}}</ref>。白色、紅色完美結合的描述還有《漢孝惠皇后外傳》：「不傅脂粉而顏色如朝霞映雪」<ref>{{Cite web|url=https://zh.wikisource.org/zh-hant/%E6%BC%A2%E5%AE%AE%E6%98%A5%E8%89%B2|title=漢宮春色 - 維基文庫，自由的圖書館|accessdate=2018-10-16|work=zh.wikisource.org|language=zh-Hant}}</ref>，紅色的朝霞映照在白雪上，單是意境，足以美得令人心醉。於是，胭脂就這樣大搖大擺地登上了歷史舞台，到[[东周|春秋戰國時期]]又出現面脂、唇脂、髮蠟。

除了外部的'''美容'''行為，文史資料表明，[[西周|西周]]時對損美性[[皮膚病|皮膚病]]就有了內治法，[[戰國|戰國]]時《[[山海经|山海經]]》記載了一些'''美容'''的藥物可以治療[[痤疮|痤瘡]]、[[腋臭|狐臭]]、疣等。

== 方式 ==
主要分為侵入式與非侵入式。

=== 非侵入式美容 ===
包括但不限於：

* 蒸面
* 深層潔面（Exfoliation），又稱去[[角質|角質]]。
* 保濕修護：敷[[面膜|面膜]]、塗抹具保濕修護成份的保養品，相關成份常見如[[玻尿酸|玻尿酸]]。
*剃毛
*[[有機|有機]]皮膚護理
*[[化妆|化妝]]
*[[理髮|美髮]]、護髮
*[[减肥|塑身]]

廣義的非侵入式方法還包括[[按摩|按摩]]或[[SPA|SPA]]療程，如：

* 香薰按摩
* 面部拉提
* [[推拿|推拿]]
* 足部按摩
* 通[[淋巴|淋巴]]
* [[豐胸|豐胸]]按摩
* [[桑拿|桑拿]]

=== 侵入式美容 ===
包括但不限於：

* [[果酸|果酸]]換膚：利用A酸、杏仁酸、杜鵑花酸...等果酸類成份，達到去除[[粉刺|粉刺]]、[[痘痘|痘痘]]與[[美白|美白]]的效果。
*[[脫毛|脫毛]]：比起剃毛較為持久。
* [[整形|整形]]
* [[微整形|微整形]]

== 醫療美容 ==
隨著[[医学|醫學]]及科技發展，加上侵入式美容往往比非侵入式美容更快速見效，現代產生了醫療美容（簡稱：醫美）行為，並衍生出相關產業。依據[[中華民國衛生福利部|中華民國衛生福利部]]說明，「醫療性美容」是指透過手術、藥物或物理等侵入性手段，改變人體外部型態、色澤、美觀等，以符合當事人主觀上對美的定義。<ref>{{Cite web|url=https://health.gov.taipei/Default.aspx?tabid=726&mid=2440&itemid=30041|title=臺北市政府衛生福利局|accessdate=|author=|date=|publisher=}}</ref><ref>{{Cite news|url=https://tw.news.yahoo.com/%E8%A1%9B%E7%BD%B2%E6%AD%A3%E5%90%8D-%E7%BE%8E%E5%AE%B9%E9%86%AB%E5%AD%B8-%E9%9D%9E-%E9%86%AB%E5%AD%B8%E7%BE%8E%E5%AE%B9-050623049.html|title=衛署正名 「美容醫學」非「醫學美容」|accessdate=2018-10-16|language=zh-Hant-TW}}</ref>
