'''美都餐室'''（[[英文|英文]]：'''Mido Cafe'''），是一間雙層型的舊式街角[[茶餐廳|茶餐廳]]，位於[[香港|香港]][[九龍|九龍]]的[[油麻地|油麻地]][[廟街_(香港)|廟街]]63號的唐樓內。

== 歷史 ==
美都餐室於1950年開業，直至今日仍保留1950年代的裝潢，見證著香港歷史，除地下外更有當年十分普遍的二樓雅座，可以瀏覽[[廟街_(香港)|廟街]]及[[榕樹頭|榕樹頭]]的街景。

1960年代，為應付附近[[油麻地避風塘|油麻地避風塘]]的水上人，需要聘請25個伙計。現在美都餐室已經交予創辦人的大女黃醒芬接手經營<ref>{{Cite web|url=http://www.hkmemory.org/old-shops/text/index.php?p=home&catId=75&photoNo=0|title=消逝中的上海街：美都餐室|accessdate=2018-08-23|author=香港記憶|date=2012|publisher=康樂及文化事務署}}</ref>。

餐室以焗排骨飯、蓮子鴛鴦冰聞名，餐單與平時茶餐廳的不同。不少遊客都會專誠到餐室感受香港傳統風味。

== 傳統特色 ==

* 掛牆的手寫餐牌
* 紙皮石地板及柱位
* 舊式收銀機及櫃台
* 傳統入榫木製家具
* 上世紀燈飾及燈櫃
* 二樓一列綠色鐵窗
* 一樓「美都飱室」美術字招牌

== 流行文化 ==
美都餐室是多部電影如《[[九龍冰室|九龍冰室]]》或電視劇如《[[酒店風雲|酒店風雲]]》和《[[廟街·媽·兄弟|廟街·媽·兄弟]]》（摹倣，劇中店名為「妙都」）的拍攝所在地。

2011年2月23日，[[香港財政預算案|香港財政預算案]]發表在即，[[無線新聞|無線新聞]]特別委派[[方健儀|方健儀]]在美都餐室主持「茶聚」節目，讓[[中產人士|中產人士]]表達自己在當前經濟環境的難處。<ref>http://www.discuss.com.hk/viewthread.php?tid=13958617 香港網民討論，並附互動新聞台直播電視截圖。</ref>
