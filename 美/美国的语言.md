[[美国|美国]]最广泛的语言是[[英语|英语]]，但[[美國聯邦政府|美國聯邦政府]]並沒有設有[[官方語言|官方語言]]。历史上存在过很多语言。其他有美国本土的语言，也有来自欧洲、亚洲等地区的语言。目前美国所使用的语言约有430种，其中有176种土著语言。其余绝大多数是由殖民者带来，还有个别语言是在美国诞生的，如[[克里奥尔语|克里奥尔语]]、[[美国手语|美国手语]]。

== 官方语言现状 ==
美国没有官方语言，但目前作为通用语的是英语（美式英语）。

受到[[唯英语运动|唯英语运动]]的影响，一些州国家已经通过立法，赋予正式地位的成英语。 {{Template:As of|2014|10}}出50个州中，有31个州已将英语作为官方语言，夏威夷将英语和[[夏威夷语|夏威夷语]]作为正式的语言。

== 主要语言 ==
