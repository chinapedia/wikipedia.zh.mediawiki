'''克羅地亞21歲以下國家足球隊'''（Hrvatska nogometna U-21 reprezentacija，簡稱'''克羅地亞U21'''），其前驅是[[南斯拉夫21歲以下國家足球隊|南斯拉夫]]，[[南斯拉夫社会主义联邦共和国|南斯拉夫]]於1992年解體後[[克羅地亞|克羅地亞]][[獨立|獨立]]，於1993年加入[[國際足協|國際足協]]及[[歐洲足協|歐洲足協]]，代表[[克羅地亞|克羅地亞]]參賽所屬年齡組別的[[英式足球|足球]]競賽，主要為兩年一度的[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]，由[[克羅地亞足球總會|克羅地亞足球總會]]負責監控。

== 競賽歷史 ==
2000年[[歐洲U-21足球錦標賽|歐洲U-21足球錦標賽]]克羅地亞U21在外圍賽取得8戰6勝2和不敗佳績，力壓[[塞爾維亞21歲以下國家足球隊|南斯拉夫聯盟共和國]]首名出線，外圍賽附加賽出戰[[葡萄牙21歲以下國家足球隊|葡萄牙]]，首回合作客先輸0-2，次回合主場扳回兩球，[[加時賽|加時]]後再入一球，才兩回合累計3-2險勝晉級在[[斯洛伐克|斯洛伐克]]舉行的決賽週。但在決賽週卻表現不濟，分組賽3戰1和2負墊底提早出局。

四年後克羅地亞U21捲土重來，陣中擁有[[达里奥·斯尔纳|施拿]]、[[Marko_Babić|巴碧]]（Marko Babić）、[[尼高·卡蘭積卡|尼高·卡蘭積卡]]、[[达尼耶尔·普拉尼奇|柏蘭積]]及[[伊度亞度·達施華|伊度亞度·達施華]]等球員，外圍賽分組賽6戰3勝2和1負，僅以1分壓倒[[比利時21歲以下國家足球隊|比利時]]及[[保加利亞21歲以下國家足球隊|保加利亞]]首名出線<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2004/round=1585/group=1471.html |title=Season 2002-2004 Qualifying round Group 8 |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17}}</ref></small>，外圍賽附加賽對手是[[蘇格蘭21歲以下足球代表隊|蘇格蘭]]，兩回合累計2-1順利過關<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2004/round=1586/match=73606/index.html |title=Croatia survive Scottish siege |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17}}</ref></small>。由[[德國|德國]]主辦決賽週，克羅地亞再一次在分組賽失利，首場面對[[塞爾維亞21歲以下國家足球隊|塞爾維亞和黑山]]，先落後兩球，憑卡蘭積卡及達施華的入球扳成平手，在完場前被[[布拉尼斯拉夫·伊雲奴域|伊雲奴域]]射入，以2-3落敗終場<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2004/round=1608/match=74208/index.html |title=Ivanovic culls Croatian comeback |publisher=[[UEFA|''uefa.com'']] |date=2004-05-27 |accessdate=2009-09-17}}</ref></small>，最後分組賽成績同樣3戰1和2負僅得1分包尾出局<small><ref>{{cite web |language=en |url=http://www.uefa.com/competitions/under21/history/season=2004/round=1608/group=1942.html |title=Season 2002-2004 Final tournament - Group stage Group A |publisher=[[UEFA|''uefa.com'']] |accessdate=2009-09-17}}</ref></small>。
