'''克劳福兹维尔怪物'''（{{lang-en|Crawfordsville monster}}）是指1891年[[美国|美国]][[印第安纳州|印第安纳州]]发生的数起[[神秘生物|神秘生物]]目击事件，通常被认为是[[大气层巨兽|大气层巨兽]]的目击。目前克劳福兹维尔怪物被认为是[[双领鸻|双领鸻]]群的误认。<ref>{{Cite web|url=https://en.wikipedia.org/wiki/Crawfordsville_monster|title=Crawfordsville monster|accessdate=2017|author=|date=|publisher=}}</ref>

== 目击 ==
1891年9月5日上午，送冰人Marshall McIntyre和Bill Gray遭遇了克劳福兹维尔怪物。他们说这个诡异的生物长约18英尺（5.5米）、宽8英尺（2.4米），通过几对侧[[鳍|鳍]]推动空气前行。它外表白色的，没有固定的形状或明确的头尾，但可见它那火红的眼睛以及喘息声，似乎遭受巨大的痛苦。Marshall McIntyre和Bill Gray躲进相对安全的[[谷仓|谷仓]]，并观察了这只在空中徘徊的生物近一小时，之后他们利用带来的马迅速的离开了那附近。<ref>{{Cite web|url=http://blog.newspapers.library.in.gov/crawfordsville-monster/|title=The Crawfordsville Monster|accessdate=2017|author=|date=2015|publisher=}}</ref>

当日晚，[[卫理公会|卫理公会]]的牧师Switzer与其妻子（姓名不详）也目击到一个类似的生物。

== 可能的解释 ==
目击者John Hornbeck 和Abe Hernley追踪并确认克劳福兹维尔怪物是一群双领鸻。《Crawfordsville Daily Journal》据此推测这些双领鸻可能是遭到新安装的路灯干扰，迷失了方向，以至于不停地城镇与天空盘旋。<ref>{{Cite web|url=https://unmyst3.blogspot.jp/2011/08/crawfordsville-monster.html|title=Crawfordsville Monster|accessdate=2017年5月25日|author=|date=|publisher=}}</ref>
