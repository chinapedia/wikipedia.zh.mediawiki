'''（长毛）克洛迪奥'''（{{lang-fr|Chlodio le Chevelu}}，{{bd|390年||450年||catIdx=C}}），428年—450年在位，是[[萨利昂法兰克人|萨利昂法兰克人]]的领袖，这一支日耳曼部落缔造了法兰克人的同盟。克洛迪奥是[[墨洛温王朝|墨洛温王朝]]可以肯定真实存在的第一位国王。然而，历史记载仅仅是略微提及他两次，故而为他写传记十分困难。

== 生平 ==

将近5世纪中叶时，克洛迪奥率军侵入[[罗马帝国|罗马帝国]]的疆域[[康布雷|康布雷]]和[[阿图瓦|阿图瓦]]南部。之后他建立了[[法兰克王国|法兰克王国]]，为[[克洛维一世|克洛维一世]]所继承。

== 逝世 ==

450年，克洛迪奥逝世。

== 子女 ==

他的妻子并没有被提到，只知道有一个儿子[[墨洛维|墨洛维]]。
