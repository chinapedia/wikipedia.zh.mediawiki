'''克萊斯汀·艾莉絲·瑞特'''（{{lang-en|'''Krysten Alyce Ritter'''}}，{{bd|1981年|12月16日}}）<ref>{{cite journal |last=Naoreen |first=Nuzhat |url=http://www.ew.com/ew/article/0,,20654366,00.html |title=Monitor: Dec. 14, 2012 |journal=[[Entertainment_Weekly|Entertainment Weekly]] |issue=1237 |page=26 |date=2012-12-07 |accessdate=2014-05-11}}</ref>是一位美國女演員、歌手和前模特兒。她較著名的是在[[AMC_(電視頻道)|AMC]]電視劇《[[絕命毒師|絕命毒師]]》（2009年-2010年）中飾演珍·瑪格麗絲和《[[23號公寓的壞女孩|23號公寓的壞女孩]]》（2012年-2013年）中的克蘿伊一角。2015年開始於[[漫威漫畫|漫威漫畫]]改編的[[Netflix|Netflix]]電視劇《[[潔西卡·瓊斯_(電視劇)|潔西卡·瓊斯]]》中主演[[潔西卡·瓊斯|潔西卡·瓊斯]]。

== 生涯 ==
出生於賓夕法尼亞州的布盧姆斯堡。她與母親、繼父和妹妹在賓夕法尼亞州郊外的一個農場上長大。瑞特於2000年從西北地區高中畢業。 瑞特有著德國、蘇格蘭和英國的血統。

2015年開始於[[漫威漫畫|漫威漫畫]]改編的[[Netflix|Netflix]]電視劇《[[潔西卡·瓊斯_(電視劇)|潔西卡·瓊斯]]》中主演[[潔西卡·瓊斯|潔西卡·瓊斯]]<ref>{{cite web |last=Strom |first=Marc |url=http://marvel.com/news/tv/23762/krysten_ritter_to_star_in_marvels_aka_jessica_jones |title=Krysten Ritter to Star in Marvel's A.K.A. Jessica Jones |publisher=[[Marvel_Comics|Marvel.com]] |date=2014-12-05 |accessdate=2014-12-05}}</ref>。進而演出電視劇《[[捍衛者聯盟_(電視劇)|捍衛者聯盟]]》。2018年首次嘗試執導《[[潔西卡·瓊斯_(電視劇)|潔西卡·瓊斯]]》第三季<REF>[https://www.dramaqueen.com.tw/news/20180706/011.html 《漫威潔西卡瓊斯》第三季曝片場照！克萊斯汀瑞特首執導演筒架式十足] 2018.7.6 DramaQueen電視迷</REF>。

=== 其他活動 ===
瑞特是[[獨立搖滾|獨立搖滾]]搭檔Ex Vivian的另一半成員，另一位是她的童年朋友William Thomas Burnett。他們的首張同名專輯於2012年在Burnett的WT Records上發行。瑞特的首部心理驚悚小說名為《''Bonfire: A Novel''》，由出版商{{link-en|Crown Publishing Group|Crown Publishing Group|Crown Archetype}}於2017年11月7日發售。<ref>{{Cite web |url = http://ew.com/books/2017/02/15/krysten-ritter-debut-novel-bonfire/ |title = Krysten Ritter to write psychological thriller |author = Nivea Serrao |date = 2017-02-15 |publisher = Entertainment Weekly |language = en}}</ref>
