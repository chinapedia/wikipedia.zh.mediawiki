'''克里斯·比爾德'''（{{lang-en|'''Chris Beard'''}}），[[加拿大|加拿大]]商業領袖，現任[[Mozilla公司|Mozilla公司]][[執行長|執行長]]。

1998年1月，他在加拿大渥太華共同創立了Puffin集團，不到兩年之後被Linuxcare收購<ref>{{cite web|url=http://www.linuxtoday.com/developer/1999121602904PS|title=Linux Today - National Post: California firm buys Ottawa Linux developer the Puffin Group|work=linuxtoday.com}}</ref>。此後，比爾德主要從事資深產品和市場行銷工作，包括[[惠普|HP]]、[[Lustre|Lustre]]和[[Mozilla|Mozilla]]。他於2004年加入Mozilla<ref>{{cite web |url=https://blog.lizardwrangler.com/2004/11/05/people-on-the-move/ |title=People on the Move |date=2004-11-05 |publisher=Mitchell's Blog}}</ref>，在[[Firefox|Firefox 1.0]]發表前曾深入參與Mozilla的各項事務，包括負責帶領產品、行銷、創新、溝通、社群和使用者交流等相關專案<ref>{{cite web |url=https://blog.mozilla.org/blog/2014/07/28/chris-beard-named-ceo-of-mozilla/ |title=Chris Beard Named CEO of Mozilla |date=2014-07-38 |publisher=The Mozilla Blog}}</ref>。

2013年7月，比爾德加入Greylock Partners創投公司擔任駐點企業家<ref>[http://lilly.tumblr.com/post/54930217539/announcing-chris-beard-as-greylock John's Tumblr - Announcing Chris Beard as Greylock Executive-in-Residence]. Lilly.tumblr.com (2013-07-08). Retrieved on 2014-02-20.</ref><ref>{{cite web|url=http://www.greylock.com/teams/64-Chris-Beard|title=Team|work=Greylock Partners|deadurl=yes|archiveurl=https://web.archive.org/web/20140415101340/http://www.greylock.com/teams/64-Chris-Beard|archivedate=2014年4月15日|df=}}</ref>，期間仍同時擔任Mozilla董事長[[米切爾·貝克|米切爾·貝克]]的顧問。

比爾德於2014年4月被任命為[[Mozilla公司|Mozilla公司]]臨時執行長<ref>{{cite web |url=https://blog.mozilla.org/blog/2014/04/14/mozilla-moving-forward/ |title=Mozilla Moving Forward |date=2014-04-14 |publisher=The Mozilla Blog}}</ref><ref>{{cite web |url=http://www.siliconbeat.com/2014/04/14/mozilla-names-interim-ceo/?doing_wp_cron=1440855509.2201259136199951171875 |title=Mozilla names interim CEO |date=2014-04-14 |publisher=SiliconBeat}}</ref><ref>{{cite web |url=http://www.cnet.com/news/mozilla-names-insider-chris-beard-interim-ceo-after-eich-flap/ |title=Mozilla names insider Chris Beard as interim CEO |date=2014-04-14 |publisher=CNET}}</ref>。同年7月28日，比爾德正式擔任執行長職位<ref>{{cite web|url=https://blog.mozilla.org/blog/2014/07/28/chris-beard-named-ceo-of-mozilla/|title=Chris Beard Named CEO of Mozilla|work=The Mozilla Blog}}</ref>。
