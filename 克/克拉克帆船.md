'''克拉克帆船'''（'''{{lang|en|Carrack}}'''，又譯'''大帆船'''或者'''卡瑞克帆船'''，[[西班牙語|西班牙語]]稱之為'''{{lang|es|Carraca}}'''）為一款在15世紀盛行於[[地中海|地中海]]的三桅或四桅的帆船。它的特徵就是其巨大的弧形船尾，以及船首的巨大斜桅。而它在前桅及中桅裝配了數張[[橫帆|橫帆]]，後桅則配上一面[[三角帆|三角帆]]。

而克拉克帆船是[[歐洲|歐洲]]史上第一款用作遠洋航行的[[帆船|船艦]]，這是因為它的龐大體積能夠在汪洋大海中保持穩定；此外，它被劃分大量空間，得以擺放足夠遠洋航行的物資。[[西班牙|西班牙]]和[[葡萄牙|葡萄牙]]兩個海上強權就是在15至16世紀期間使用這種帆船來進行遠洋探險，並建立了一個又一個的殖民地。在[[法國|法國]]，這種船被稱為「'''{{lang|fr|Caraque}}'''」或「'''{{lang|fr|Nef}}'''」，而西班牙和葡萄牙則皆稱為「'''{{lang|es|Carraca}}'''」（意思都是「船」）。

== 由來 ==
「克拉克」這一個名稱最早可以追溯至[[中世紀|中世紀]]傳入歐洲的[[阿拉伯語|阿拉伯語]]，以及[[希臘文|希臘文]]'''{{lang|el|κέρκουρος}}'''（'''{{lang|la|kerkouros}}'''）一詞（即「[[駁船|駁船]]」之意），當時的歐洲人以之來把新研發的大型帆船命名。大約在1300年，歐洲人改良了當時往來[[地中海|地中海]]的[[柯克船|柯克帆船]]，由一桅增加至三至四桅。此外，人們也在它身上增加了船首斜桅，以及減低的艉樓的高度，令它更便於在海戰發揮威力。後來在16世紀初，它更被安裝了多層的甲板，以便於安裝更為巨大的炮座，增加威力。而克拉克帆船的改良更為海上戰爭帶來了新元素，海上戰爭不再單是兩船間短兵相接及弓箭互發的模式，而轉變為更具威力的炮戰，亦引伸出後來[[戰列艦|戰列艦]]的形成。而克拉克帆船精良的裝備亦令歐洲殖民者得以進佔多個地區，建立海上霸權，主宰整片海洋。

== 優點 ==
* 它擁有充足的船艙以擺放貨物、物資及容納[[水手|水手]]。
* 它能抵受來自體型較小船隻的攻擊。
* 它能夠實現遠洋航行。
* 船上的四桅上，掛有三角帆及橫帆，這足夠提供動力令船隻航行速度增加，即使逆風時也無礙航行。
* 它具備的多層甲板可以安放重型[[火炮|火炮]]，亦便於發射並排的側舷火炮，增加船隻威力。

可是，當遇上強風時，它那龐大的體積亦會容易出現重心傾斜，令船隻坍塌。

== 著名的克拉克帆船 ==
[[File:NanbanCarrack.jpg|thumb]][[長崎市|長崎市]]海域航行的[[葡萄牙|葡萄牙]]'''克拉克帆船'''。]]

* [[大密歇尔号|大密歇尔号]]''（{{lang|en|Great Michael}}）''：蘇格蘭製的克拉克帆船，當時歐洲最大型的帆船。
*[[玛丽玫瑰号|瑪丽玫瑰号]]''（{{lang|en|Mary Rose}})''及[[上帝的恩宠亨利号|上帝的恩宠亨利号]]''（{{lang|fr|Henri Grâce à Dieu}}）''：[[英國君主列表|英王]][[亨利八世|亨利八世]]下令建造的兩首旗艦，亦是[[英格蘭王國|英格蘭王國]]開始成為海上軍事強權的標誌。
* [[圣安娜号|圣安娜号]]''（{{lang|pt|Santa Anna}}）''：由[[十字軍|十字軍]][[騎士團|騎士團]]之一的[[医院骑士团|聖約翰騎士團]]所建造的克拉克帆船，第一艘具有重裝甲的帆船。
* [[圣卡塔琳娜号|圣卡塔琳娜号]]''（{{lang|pt|Santa Catarina}}）''：原為葡萄牙製造的克拉克帆船，但在西元1603年被[[荷兰东印度公司|荷兰东印度公司]]所搶奪。
*[[圣玛利亚号|圣玛莉亚号]]''（{{lang|es|Santa Maria}}）''：航海家[[哥倫布|哥倫布]]探索美洲新大陸時所使用的旗艦。
* [[圣安东尼奥号|圣安东尼奥号]]''（{{lang|it|San Antonio}}）''：葡萄牙製造的克拉克帆船，為葡萄牙國王[[若昂三世_(葡萄牙)|若昂三世]]的個人艦隻，西元1527年失事沉沒。
*[[維多利亞號|维多利亚号]]''（{{lang|es|Victoria}}）''：史上第一首完成環球航行的船隻。西元1519年8月10日，由[[麥哲倫|麥哲倫]]率領，連同維多利亞號在內的五艘船由[[西班牙|西班牙]]港口[[塞維利亞|塞維利亞]]出發，開始向西航行。但可惜的是在回程中，麥哲倫不幸被殺，只剩下維多利亞號獨自回航西班牙。

== 退出 ==
在[[十六世紀|十六世紀]]末，克拉克帆船已經進一步演變成[[蓋倫帆船|蓋倫帆船]]而漸漸沒落於航海歷史。然而，它卻促成了[[歐洲|歐]][[亞洲|亞]]之間的海上交通，以及震動整個歐洲的[[地理大发现|大航海時代]]，為世界交通發展史寫下光輝的一頁。克拉克帆船持續被使用至十七世紀。
