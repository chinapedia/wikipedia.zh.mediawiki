'''克謝尼娅·阿法纳西耶娃'''（[[英语|英语]]：{{lang|en|'''Ksenia Afanasyeva'''}}；{{lang-ru|'''Ксения Дмитриевна Афанасьева'''}}，{{bd|1991年|9月13日|||catIdx=Afanasyeva, Ksenia}}），[[俄羅斯|俄羅斯]]女子[[競技體操|競技體操]]運動員。2012年倫敦奧運會女子團体亞軍成員之一，在2010年世界体操锦标赛上与队友一起获得俄罗斯第一个女团世界冠军，她也在2011年东京世界体操锦标赛上获得自由操金牌。

== 少年组 ==
阿法纳西耶娃初次给大家留下印象是在2005年的欧洲少年奥林匹克赛上，那次她拿到了全能和自由体操的第三名，她的富有韻律感的自由体操表演给一些有心人留下了印象。

2006年欧洲少年体操锦标赛是阿法纳西耶娃参加的第一项重要赛事。强手云集的俄罗斯队由天才少女叶里扎洛娃领衔，俄罗斯队不仅在团体赛领先以十分的优势领先罗马尼亚夺冠，全能比赛也包揽了冠军和亚军。单项比赛，俄罗斯拿下了除自由体操之外所有的金牌。在叶里扎洛娃和玛亚斯尼科娃各自包揽三金时，阿法纳西耶娃作为团体冠军成员没有能进入一项决赛，她的强项自由体操也没有足够的难度杀出队友的重围。

然而灾难之神将厄运抛给了2006年欧少赛团体冠军的成员们。2006年年底，高低杠亚军，科卡罗娃重伤退役，随后自由体操难度很大的契尔扎罗娃也遭遇伤病侵袭。2007年年初，两位三金王玛亚斯尼科娃和叶里扎洛娃也因为伤病，放缓了进入成年组比赛的步伐。阿法纳西耶娃这只不起眼的丑小鸭获得了机会。

== 2007年 ==
阿法纳西耶娃早期以自由体操和平衡木见长。2007年体操世界杯德国站比赛，她的一套平衡木难度分超过7分，拥有两个三D连接，还有单手臂支撑的小翻180这个亮点动作，最终她在落木后依然获得了15.150分。自由体操尽管也出现失误，但良好的舞蹈素养和标志性的阿提丢转体720也让人印象深刻。稳定性最终让她在2007年错过了世界大赛的机会。当同门姐妹同样来自图拉的美少女塞梅诺娃以惊世连接获得高低杠世界冠军时，阿法纳西耶娃却没有参加世界锦标赛的资格。稳定性是她的最大问题。

== 2008年 ==
2008年，奥运会来临了。阿法纳西耶娃重新整合了动作，在年初的俄罗斯全国锦标赛上表现强眼，获得了欧洲体操锦标赛的参赛资格。然而，她在欧洲体操锦标赛表现却让人相当失望，跳马落地差点撞马，平衡木摇晃不断，自由体操后直两周跪地，俄罗斯在2008年的欧洲体操锦标赛以5分的巨大劣势败给了罗马尼亚。在塞梅诺娃获得欧洲锦标赛高低杠和平衡木冠军时，阿法纳西耶娃又是一无所获。尽管表现糟糕，但阿法纳西耶娃凭借在夏季俄罗斯杯上全能夺冠这一良好成绩拿到了奥运会的参赛资格。
  
在这只队伍里阿法纳西耶娃似乎是不太招人喜爱的，原因之一就是不够漂亮。甚至有人在看了平衡木决赛后说到：“俄罗斯体操队怎么有这样丑的女孩子？”但生活中，阿法纳西耶娃则是一个谦虚，礼貌，温柔的姑娘。05年那套颇具艺术特色的自由体操《卡萨布蓝卡》到2008年奥运会的《秋日私语》，她展现的艺术的竞技体操运动，一个并不漂亮的女孩子也可以追求艺术和难度的美丽融合。

预赛她的发挥还是很不错的，全能第六，平衡木以15.775分进入了单项的决赛，并且自由体操，她居然很好的完成了900接360这个连接，还出色完成了540接540，要知道她曾经的540接360都时不时的会出现失误。自由体操震撼的表演让人觉得她似乎有些脱胎换骨了，2008年全国锦标赛时候都没有完成的连接在奥运会的赛场上出色的演绎了，看来她是把秘密武器留在了奥运会上，15.050分，分数不高，但这个俄罗斯自由体操预赛的第三个上15的分数足以让俄罗斯团体决赛有望将自由体操撑到45分。

团体决赛那天，阿法纳西耶娃上了三个项目。高低杠她代替科留金娜出场，开场的蹬杠转体180换高杠多少有点霍尔金娜的味道，杠上动作完成很干净，由于难度过低14.925分还算正常完成任务。而后两位出场的选手越来越出色，高低杠俄罗斯赢得一个开门红。自由体操她上场时，俄罗斯情况已经相当的危急，平衡木两人重大失误，一人发挥不好，自由体操帕夫诺娃连14分都没有上，作为全国全能冠军阿法纳西耶娃是需要在这个时候显示自己实力的，但非常遗憾，空翻出界了，整个动作完成的紧紧巴巴，最后连14.500分都没有拿到。倘若她发挥的正常点，俄罗斯或许还真有点希望扳回。但失误的人太多，过错自然不能全算上她的头上，虽然随后跳马正常发挥水平，只是有些可惜的是她没有在最关键时刻证明自己的全国冠军的成色。倒是让克拉马连科抓住了证明自我的机会。

平衡木决赛那天，阿法纳西耶娃显然过于紧张了。平衡木应该是她的强项，她曾经还熟练掌握了单臂阿拉伯前手翻，编排换了很多次，A分始终还是不错，决赛那套编排如果顺利完成也有6.9的A分。但现场的表现是，挺身双脚站接小翻拉拉提没有连上，挺身前空翻接挺身成燕式也没有连上，小翻180接侧空更是做成D组动作独立展示，最终损失了0.6的连接加分外还被扣掉0。5的未特定规则分，14.825唯一一个决赛项目也名落孙山。

早期体操生涯令人失望的表现，让阿法纳西耶娃饱受指责。

== 2009年 ==
北京奥运会后，一批选手离开了圆湖，阿法纳西耶娃留了下来。她的教练拉扎罗娃信心满满地说到，这个周期就要看我们的两个克谢尼亚（指阿法纳西耶娃和塞梅诺娃，她们同名）。

2009年在实行新规则后，阿法纳西耶娃的全能难度超过了塞梅诺娃，欧洲锦标赛将是她的一个很好的机会。全能比赛，平衡木的落木让阿法纳西耶娃失去了竞争金牌的可能，自由体操中，她表演了一套优美和动感兼具的动作，最终获得了全能亚军，冠军属于塞梅诺娃。年底的世界锦标赛，提升了难度的阿法纳西耶娃因伤在赛前退出比赛。

== 2010年 ==
[[File:Ksenia_Afanasyeva,_2010.jpg|thumb]]

2010年，随着穆斯塔芬娜和纳比耶娃进入成年组，阿法纳西耶娃位置尴尬。她先是错过了欧洲体操锦标赛，那次比赛俄罗斯女团夺冠。年底，凭借自由体操的难度和相对稳定性，她进入了世界锦标赛参赛阵容。预赛她发挥出色排名前列进入自由体操决赛。团体决赛，在俄罗斯和美国争夺到白热化阶段的时候，阿法纳西耶娃发挥完美，自由体操几乎所有空翻都钉住地上，以高分为俄罗斯夺冠奠定了基础。只是在自由体操决赛中，原本大有机会的她出场动作跪地，顶住了团体比赛的压力，却在自己的单项比赛里自信心不足失误。

2010年的自由体操决赛的失误，很多人都认为阿法纳西耶娃再也没有机会获得单项奖牌了。但团体决赛的出色发挥，让人觉得阿法纳西耶娃成熟了。亚历山大罗夫在提起这位老将时说到，她是一位自由体操实力很强的选手，但自信心不足，解决心理问题是她将会有提升。2010年团体的夺冠，阿法纳西耶娃获得世界锦标赛金牌。当一些团体夺冠的成员逐渐地走向退役之路时，她坚持了下来。尽管她明白在这些年年轻选手层出不穷的俄罗斯，奥运会5—3—3的赛制名额有限，她很難有机会。

== 2011年 ==
2011年东京世界体操锦标赛预赛，阿法纳西耶娃比了四个项目，个人项目只拿到了全能决赛的资格。团体决赛，她的跳马落地稳健，为俄罗斯队取得开门红，自由体操比赛她压阵出场，在新星科莫娃出现失误的情况下，她顶住压力再次完成一套干净的自由体操，俄罗斯最终保住了银牌。全能比赛，由于弱项平衡木的落木，尽管阿法纳西耶娃在最后一个项目自由体操中再次稳定发挥，她最终获得第7名，与奖牌无缘，却也是个不错的成绩。

当大多数人认为全能比赛结束后，阿法纳西耶娃已告别世界锦标赛后，她却意外地出现在自由体操决赛名单中。科莫娃退赛了，将这个宝贵的名额给了大姐。阿法纳西耶娃最后一个出场，出场的后直两周高飘十足，快速快速接1080也稳稳站住，540接360无任何调整后接了个舞蹈小跳，后屈两周后跳一小步，阿提留转体720和高基安跳步也做出几年来最高水准，尽管水平腿立转720在到位后支撑脚动了，但丝毫不影响这套动作的观赏性。阿法那西耶娃为自己的心在跳舞，她将心、动作与音乐结合在一起，这种对音乐和运动的感知力是她多年坚持来最大的收获，她的舞蹈毫不奢华，浮躁，功利，她做到了自己，也战胜了自己的心理！
  
在等分过程中，阿法纳西耶娃有些忐忑不安，她不知道这套动作会得到什么样的分数，也许她觉得一块奖牌就是最大的奢望了。对她D分认定的疑问主要在水平腿立转720的认定上，最终裁判承认了她的动作，6.1的D分加上9。033的完成分，这个坚持不懈的平凡姑娘获得了世界冠军！在分数出来那一刻，阿法纳西耶娃同她的教练拉扎罗娃紧紧抱在一起！
一位参加过奥运会体操报道的记者朋友谈到阿法那西耶娃时说道，她是个朴实的姑娘，当体操爱好者将一份礼物送到她手里时，她受宠若惊，显得很不好意思，连说谢谢，她惊讶居然会有体操迷送她礼物。这样一个平凡的姑娘最终实现了自己的梦想！比赛中的一个插曲。在阿法纳西耶娃出场时，央视体育频道著名女主持人陈滢调侃地说道，阿法纳西耶娃的超越将是IMPOSSIBLE。这句秀英语的玩笑话或许是迎合前面解说嘉宾的玩笑话：形容平衡木最后一个出场的罗马尼亚选手是来打酱油的。然而阿法那西耶娃用表现证明了自己，也捍卫了作为一个普通运动员站在赛场上的尊严。IMPOSSIBLEISNOTHING！这是李宁品牌的口号，也是每一个运动员所追求的，中国选手和外国选手！

== 2012年 ==
由於西多洛娃和德門蒂耶娃的狀態依舊沒恢復好，阿法納西耶娃穩穩鎖住奧運名單中的第四人，也許是任務比較輕，她僅需要比平衡木和自由体操，資格賽儘管降了些難度，不過完美發揮也讓她進入了這兩個項目的決賽。女團決賽，穆斯塔芬娜和科莫娃高水平地發揮保證了俄羅斯一塊銀牌，不過另外三名隊員卻並沒有發揮好，新秀帕塞卡在奧運前臨時跳馬上6.5難度，落地卻沒控制好，格里什娜在高低杠斷連接，自由操更是出現低級失誤，直接葬送沖擊金牌希望。至於作為老將的阿法納西耶娃，儘管平衡木穩住，但在最後壓軸出場的自由操結束串摔了。

單項決賽，阿法納西耶娃並沒有比好，平衡木斷了一個連接而且下法落地後退兩步，自由操又兩次出界，最終在倫敦奧運僅以一塊銀牌結束。

== 2013年 ==
奧運會結束後，阿法納西耶娃想過退役，但是她覺得自己還能練下去，於是打消了退役念頭。她什至發展難度，在跳馬訓練Amanar並增加第二跳難度，自由操也嘗試立轉連接，把自己的難度分提高到很有競爭力。经过多年比赛洗礼，这位老将是目前俄罗斯自由体操的一号人物，稳定的自由体操也讓她擊敗新秀舍尔古诺娃入選歐錦賽陣容。

歐錦賽中，資格賽阿法納西耶娃在高低杠和平衡木都出現重大失誤，失去了挑戰全能的機會，但是自由操還是順利進入決賽並拿到金牌。7月大運會，阿法納西耶娃以3金1銀的成績跟穆斯塔芬娜平分秋色，更重要的是，阿法納西耶娃在跳馬擊敗了隊友帕塞卡，這讓她在世錦賽的入選增添了法碼。
