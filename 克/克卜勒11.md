'''克卜勒11'''（{{lang en|'''Kepler-11'''}}）是一顆[[類太陽恆星|類似太陽的恆星]]，位於[[天鵝座|天鵝座]]內，距離地球約2,000光年。<ref name = "Lissauer">{{Cite journal | last=Lissauer | first=J. J. | authorlink=Jack J. Lissauer | coauthors = ''et al.'' | title=A closely packed system of low-mass, low-density planets transiting Kepler-11 | journal=Nature | volume=470 | issue=  7332| pages=53–58 | date=3 February 2011 | url=http://arxiv.org/ftp/arxiv/papers/1102/1102.0291.pdf | doi=10.1038/nature09760 | accessdate=4 February 2011|bibcode = 2011Natur.470...53L |arxiv = 1102.0291 }}</ref><ref name="space.com kepler-11">[http://www.space.com/10744-alien-planets-solar-system-kepler-mission.html Astronomers Find 6-Pack of Planets in Alien Solar System]</ref>這顆星是由[[克卜勒太空望遠鏡|克卜勒太空望遠鏡]]發現的，並且由[[美國航空航天局|美國航空航天局]]於2011年2月2日公佈這個發現。克卜勒11是首個被確定擁有6顆行星的恆星。所有克卜勒11的行星质量均介于[[地球|地球]]与[[海王星|海王星]]之间。<ref name = "Boyle"/>

==命名和歷史==
於2011年2月2日，美國航空航天局宣布至少有六顆公轉週期很短的[[太陽系外行星|太陽系外行星]]圍繞這顆恆星运转。克卜勒11是以發現這顆恆星的開普勒太空望遠鏡命名的。這顆恆星是開普勒太空望遠鏡發現的第11顆恆星，因此得名「克卜勒11」。<ref name = "Lissauer"/>

這顆恆星位於[[天鵝座|天鵝座]]，距離地球約2000光年。這顆恆星是由[[開普勒太空望遠鏡|開普勒太空望遠鏡]]的小範圍巡天發現的。從地球的方向觀察開普勒11的6顆行星會發現所有行星在[[凌日|凌日]]時會從恆星盤面通過；且這些行星位置和地球上觀測者觀測方向的夾角小於1度。這種狀況使科學家們可以經由觀測行星凌日現象直接量測行星的公轉週期和直徑。<ref>{{cite web | author=Michael Mewhinney; Rachel Hoover; Trent J. Perrotto  | title=NASA's Kepler Spacecraft Discovers Extraordinary New Planetary System | url=http://www.nasa.gov/mission_pages/kepler/news/new_planetary_system.html | publisher=NASA | date=2 February 2011 | accessdate=4 February 2011}}</ref>  電腦模擬顯示這個行星系統內各行星之間軌道夾角大約為1°，這代表該行星系統比我們的[[太陽系|太陽系]]（夾角約 2.3°）更接近[[共面|共面]]狀態<ref name = "Lissauer"/>。开普勒11是第一个被发现的同时有多于3颗行星[[凌日|凌日]]的系外行星系统。这些行星从内到外以字母顺序分别命名为b、c、d、e、f和g。<ref>[http://www.nasa.gov/mission_pages/kepler/news/new_planetary_system.html Nasa Report]</ref>

==性質==
克卜勒11是一顆[[G型主序星|G型主序星]]，而其[[恆星光譜|恆星光譜]]分類則為G6V。其質量約為[[太陽質量|太陽質量]]的95%，半徑約為[[太陽半徑|太陽半徑]]的110%。其表面溫度約為5680 ± 100 K，比太陽的表面溫度（5,778 K）稍低。<ref>{{cite web |url=http://www.universetoday.com/18092/temperature-of-the-sun/ |title=Temperature of the Sun |author=Fraser Cain |date=September 15, 2008 |work= |publisher=Universe Today |accessdate=19 February 2011}}</ref>其年齡約為80 ± 20 億年<ref name=EPE2 />，比太陽的年齡（45.7億年）高近一倍。<ref>{{cite web |url=http://www.universetoday.com/18237/how-old-is-the-sun/ |title=How Old is the Sun? |author=Fraser Cain |date=16 September 2008 |work= |publisher=Universe Today |accessdate=19 February 2011}}</ref>

這顆星的[[視星等|視星等]]為14.2，而地球上人類以肉眼能夠分辨的最暗天體極限的視星等為6.5，因此人類是無法看見克卜勒11的。<ref name=EPE2 />

== 行星系統 ==
[[File:Kepler-11_planets_comparison.jpg|thumb]]
從地球觀看克卜勒11行星系時，所有克卜勒11的行星都會[[凌_(天體)|凌]]克卜勒11，而且克卜勒11各行星的[[軌道傾角|軌道傾角]]在地球視野上的偏差僅為1°，因此天文學家們能夠透過凌日法發現這些行星。天文學家們製作的克卜勒11行星系模型顯示，克卜勒11各行星的平均軌道傾角為1°，代表克卜勒11行星系要比平均軌道傾角為2.3°的太陽系還要接近[[共面|共面]]。<ref name = "Lissauer"/>

行星b到行星f的估算质量介于[[地球|地球]]与[[海王星|海王星]]之间。这些行星的估算密度都较低，说明这五颗行星中都不具有像地球一样的成分组成。<ref name="Boyle">{{Cite web

开普勒11行星系统是目前已知的最紧凑的行星系统；行星b至f的轨道在水星的轨道范围内，而行星g的半长径比则比水星大约20%。尽管轨道结合的十分紧密，其整體動力狀態一般認為已穩定存在數十億年<ref name = "Lissauer"/>。

虽然行星b和c的公转周期接近于5:4的共振<ref name="Lissauer"/>，但这些行星中不存在低比率的[[轨道共振|轨道共振]]。儘管這些行星之間的距離甚近，但仍然沒有影響對方的軌道，而這個情況已持續了數十億年。<ref name = "Naeye"/>
