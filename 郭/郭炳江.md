'''郭炳江'''（{{lang-en|Thomas Kwok}}，{{bd|1951年|10月6日|catIdx=G郭}}），[[香港|香港]][[企業家|企業家]]，籍貫[[廣東省|廣東省]][[中山市|中山]]，[[新鴻基地產|新鴻基地產]]聯合創辦人[[郭得勝|郭得勝]]次子，[[郭炳湘|郭炳湘]]（已故）之二弟及[[郭炳聯|郭炳聯]]之二兄。

== 生平 ==
郭炳江幼時家族經營雜貨批發等，小一至中五均就讀[[慈幼英文學校|慈幼英文學校]]<ref>{{cite web |url=https://www.ssshk.edu.hk/CustomPage/26/SchoolMagazine/Ch02.pdf |title=感恩 |last= |first= |date=2016 |website= |publisher=慈幼英文學校 |access-date= 2019-3-25 |quote=}}</ref>。後到[[英國|英國]]升學，取得英國[[倫敦商学院|倫敦商学院]][[工商管理|工商管理]]碩士學位及[[倫敦大學|倫敦大學]][[帝國學院|帝國學院]][[土木工程|土木工程]]系學士學位。

郭炳江是[[新鴻基地產發展有限公司|新鴻基地產發展有限公司]]前任[[董事長|董事局聯席主席]]兼前任[[董事總經理|董事總經理]]<ref name="hkbea">{{cite web |url=http://www.hkbea.com/FileManager/TC/Content_2496/ar200204.pdf |title=存档副本 |accessdate=2010-09-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20110322135301/http://www.hkbea.com/FileManager/TC/Content_2496/ar200204.pdf |archivedate=2011-03-22 }} 東亞銀行公司資料</ref>，亦是[[三號幹線（郊野公園段）有限公司|三號幹線（郊野公園段）有限公司]]主席<ref name="hkbea"/>、[[IFC_Development_Limited|IFC Development Limited]] 聯席主席<ref name="hkbea"/><ref name="hkbea"/>及[[東亞銀行有限公司|東亞銀行有限公司]]前任獨立非執行董事<ref name="hkbea"/>。

在地產業界，他是[[香港地產建設商會|香港地產建設商會]]第一副會長。後來獲[[香港特區政府|香港特區政府]]委任為[[臨時建造業統籌委員會|臨時建造業統籌委員會]]委員及[[可持續發展委員會|可持續發展委員會]]成員。及後，他擔任[[營商諮詢小組|營商諮詢小組]]成員、[[土地及建設諮詢委員會|土地及建設諮詢委員會]]、[[註冊承建商懲戒處分委員會|註冊承建商懲戒處分委員會]]及[[香港總商會工業事務委員會|香港總商會工業事務委員會]]之委員、[[建造商會物業管理委員會|建造商會物業管理委員會]]主席及[[香港建造商會|香港建造商會]]幹事。

在社區參與方面，他曾出任[[香港公益金|香港公益金]]董事、[[社會福利政策及服務委員會|社會福利政策及服務委員會]]及[[香港公開大學|香港公開進修學院]]校董會之委員。他曾被選為[[廣州市|廣州市]]榮譽市民及第九屆[[中國人民政治協商會議|中國人民政治協商會議]][[上海市|上海市]]委員會的常務委員。同時，他亦為[[慈幼學校|慈幼學校]]([[小學|小學]])及[[慈幼英文學校|慈幼英文學校]]([[中學|中學]])的校友，為母校辦事，校內的409室亦以其命名。

1991年他獲政府委任為[[市政局_(香港)|市政局]]議員，至1993年離任。

他於1994年起信奉[[基督新教|基督新教]]，並於同年12月與太太[[梁潔芹|梁潔芹]]一同[[洗禮|洗禮]]。郭炳江出生時患有「小耳症」，雙耳天生沒耳道，要接受手術及戴助聽器植入耳道內。

政府於2018年3月2日刊憲，宣佈撤除郭炳江的銀紫荊星章，並同時褫奪太平紳士身份。

==涉嫌賄賂被捕入獄==
{{main|新地風暴}}
* 2012年3月29日，[[香港廉政公署|香港廉政公署]]邀請[[新鴻基地產|新鴻基地產]]董事局聯席主席郭炳江、[[郭炳聯|郭炳聯]]，與及前[[政務司司長|政務司司長]][[許仕仁|許仕仁]]到廉署總部協助調查一宗貪污案，同日傍晚，廉政公署正式宣佈拘捕三人。<ref>[http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120329&55&829767 廉署拘捕郭氏兄弟及許仕仁涉嫌貪污行為失當] {{webarchive|url=https://web.archive.org/web/20140508133357/http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120329&55&829767 |date=2014-05-08 }}，[[香港電台|香港電台]]，2012年3月29日</ref><ref>[http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120329&55&829775 消息稱廉署循官商勾結方向調查郭氏兄弟案件] {{webarchive|url=https://web.archive.org/web/20140508133120/http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120329&55&829775 |date=2014-05-08 }}，[[香港電台|香港電台]]，2012年3月29日</ref>7月13日，香港廉政公署正式起诉前政务司司长许仕仁、新鸿基地产董事局聯席主席郭炳江、郭炳联、新鸿基执行董事陈鉅源和港交所前高级副总裁关雄生。
* 2014年12月19日，郭炳江在[[香港高等法院|高等法院]]被陪審團以七比二裁定一條串謀公職人員行為失當罪罪成，於當日傍晚即時收監，還押[[荔枝角收押所|荔枝角收押所]]。
* 2014年12月23日，法官判處郭炳江監禁5年、罰款50萬[[港元|港元]]及下令禁止他出任任何公司董事5年。
* 2014年12月29日，郭炳江代表律師代郭炳江正式向法庭提上訴，指法官引導陪審團出錯，而且陪審團的裁決不合常理。<ref>[http://hk.apple.appledaily.com/realtime/news/20141229/53278946?top=4h 【新地案】郭炳江正式提上訴]，[[蘋果日報_(香港)|蘋果日報]]，2014年12月29日</ref>
* 2016年7月12日，終審法院批准郭炳江就貪賄案的上訴許可申請，並且准許他以一千萬港元保釋外出。<ref>[http://hk.apple.nextmedia.com/realtime/news/20160712/55348782 【新地案】郭炳江步出終院：生活有規律，但最好唔好响裡面]，[[蘋果日報_(香港)|蘋果日報]]，2016年7月12日</ref>
* 2017年6月14日，郭炳江終審上訴失敗，須重返監獄服刑。<ref>[http://hk.apple.nextmedia.com/realtime/news/20170614/56825515 【新地案上訴】郭炳江終極上訴失敗 重新收押荔枝角再X光「通櫃」]，[[蘋果日報_(香港)|蘋果日報]]，2017年6月14日</ref>
* 2018年3月2日，郭炳江於2007年獲頒授之[[銀紫荊星章|銀紫荊星章]]被政府褫奪，其[[太平紳士|太平紳士]]委任亦被撤銷。<ref>{{cite news|title=許仕仁郭炳江慘被「搣柴」 港府刊憲撤回勳銜|url=https://hk.news.appledaily.com/local/realtime/article/20180302/57894221?_ga=2.106059395.1871829226.1519967822-1014019929.1519967821|accessdate=2018-03-02|publisher=[[蘋果日報_(香港)|蘋果日報]]|date=2018-03-02}}</ref>
* 2019年3月21日，郭炳江刑滿，於早上9:30步出[[赤柱監獄|赤柱監獄]]。<ref>[https://hk.news.appledaily.com/breaking/realtime/article/20190321/59391097 郭炳江刑滿出獄 「我冇怨恨」]，[[蘋果日報_(香港)|蘋果日報]]，2019年3月21日</ref>
