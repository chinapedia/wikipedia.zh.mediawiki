'''郭嘉芙'''（{{lang-en|Katherine Kwok}}），別稱嘉芙姐姐，[[香港|香港]]兒歌創作人、特殊兒童教育導師。
<br>

==簡歷==
郭嘉芙擁有教育碩士(教育發展、特殊教育、資優與輔導)<ref>{{cite web |language=zh-hant | title=Ka Foo MusicLand呈獻：「港孩」專屬兒歌 |publisher=兒童尖子 |accessdate=2014-12-19 |url=http://www.ikid.hk/article/detail?contentID=5106&conCatID=77 }}</ref>、心理學學士及幼兒音樂教育證書<ref>{{cite web |language=zh-hant |title=焦點人物畢業生逐個捉 嘉芙姐姐 |publisher=香港浸會大學持續教育學院SCE |accessdate=2014-12-19 |url=http://www.sce.hkbu.edu.hk/scefriends/enews/2013aug/people.php |archive-url=https://web.archive.org/web/20141219142217/http://www.sce.hkbu.edu.hk/scefriends/enews/2013aug/people.php |archive-date=2014-12-19 |dead-url=yes }}</ref>等學歷，為香港作曲家及作詞家協會會員，致力以「寓學習於音樂中」為創作方針<ref>{{cite web |language=zh-hant |title=音樂導師 郭嘉芙 |publisher=學庫網 |accessdate=2014-12-19 |url=http://www.citipro.com.hk/2012-09-22-6158.html |deadurl=yes |archiveurl=https://web.archive.org/web/20141219133935/http://www.citipro.com.hk/2012-09-22-6158.html |archivedate=2014-12-19 }}</ref>，於2009年開始出版命名為「嘉芙姐姐兒歌專輯」的原創粵語兒歌碟，至今已推出合共四輯<ref>{{cite web |language=zh-hant | title=YesAsia |publisher=Yesasia |accessdate=2014-12-19 |url=http://www.yesasia.com/global/%E5%98%89%E8%8A%99%E5%A7%90%E5%A7%90/0-aid2124324-0-bpt.47-zh_TW/list.html#&l=2&s=7&aid=2124324&sb=137&an=嘉芙姐姐&bpt=47&cid=98&bt=47&vm=32&ss=100&aifs=2124324&vn=33&sea=1 }}</ref>，當中以律動作品《兩手真優秀》及《跟我做運動》最為兒童認識<ref>{{cite web |language=zh-hant |title=本土兒歌 唱出正面信息 |publisher=經濟日報 |accessdate=2014-12-19 |url=http://www.hket.com/eti/article/adc690e3-cbac-4f92-a8ba-4ed4691587a0-089624?sectionId=012 |archive-url=https://web.archive.org/web/20141221185350/http://www.hket.com/eti/article/adc690e3-cbac-4f92-a8ba-4ed4691587a0-089624?sectionId=012# |archive-date=2014-12-21 |dead-url=yes }}</ref>，其作品亦不乏教育意義，內容包括環保<ref>{{cite web |language=zh-hant | title=童樂日：唱兒歌　學環保練膽量 |publisher=蘋果日報 |accessdate=2014-12-19 |url=http://hk.apple.nextmedia.com/supplement/health/art/20120907/18009805 }}</ref>及衛生<ref>{{cite web |language=zh-hant | title=歌詞帝國《洗洗手》 |publisher=OIKTV |accessdate=2014-12-19 |url=http://www.oiktv.com/video/youtube/mfKqix88EFk.html }}</ref>等。

===教育局學前教育電視兒歌===
郭嘉芙於2013年開始與香港電台及教育局合作，創作新一輯學前教育電視的兒歌，包括主題曲《健康小食店》等片中的兒歌<ref>{{cite web |language=zh-hant | title=香港教育城 學前教育電視 |publisher=香港教育城 |accessdate=2014-12-19 |url=http://resources.hkedcity.net/etv/?currId=337 }}</ref>。
