'''郭士强'''{{BD|1975年|7月8日||}}出生于中國[[辽宁省|辽宁省]][[辽阳市|辽阳市]]，前中国国家男子篮球队主力控球后卫，主教练。现任[[辽宁衡润飞豹篮球俱乐部|辽宁男篮]]主教练，兼任辽宁篮管中心副主任。

== 球员生涯 ==
郭士强司职[[组织后卫|组织后卫]]，球员时期以准确的中远距离投篮和组织进攻出名，外号“拼命三郎”。他9岁进入辽阳体校学习篮球，1993年进入辽宁队，1994年入选国家青年队，1999年选入国家队，并以主力控卫的身份参加了悉尼，雅典两届奥运会。从1993年到2006年绝大多数时间效力于[[辽宁衡润飞豹篮球俱乐部|辽宁男篮]]（期间租借至四川男篮一年），两获CBA赛季“助攻王”称号。

== 执教生涯 ==
2006年退役后先后出任辽宁女篮助理教练，及[[辽宁衡润飞豹篮球俱乐部|辽宁男篮]]主教练。凭借辽宁队在06-07赛季的出色发挥，郭士强得以进入中国国家队的教练组辅佐[[尤纳斯·卡兹劳斯卡斯|尤纳斯]]。[[北京奥运会|北京奥运会]]后与[[闵鹿蕾|闵鹿蕾]]与[[阿的江|阿的江]]一起成为男篮国家队主教练的人选之一，并以主教练的身份指导2009年的多场热身赛，取得不错的成绩。但是在[[2009年亚洲篮球锦标赛|2009年男篮亚洲锦标赛]]中，未能带领中国队在本土夺冠。

2017年9月8日，由郭士强率领的辽宁全运队以78-61击败新疆，收获了建队58年来的首个全运会冠军。

郭士强作为主教练带领辽宁男篮四度杀入总决赛，前三次都铩羽而归。2018年4月22日晚，[[辽宁衡润飞豹篮球俱乐部|辽宁本钢]]4-0战横扫由冠军教头[[李春江|李春江]]带领的[[浙江广厦猛狮篮球俱乐部|浙江广厦]]，荣登CBA总冠军宝座。至此，辽宁男篮在经历了二十三年的历练，前后七次进入总决赛，终于摘掉了CBA“千年老二”的帽子，实现了联赛首冠的梦想。同时，辽宁男篮也成为中国篮球历史上，继霸主八一和广东之后，第三支包揽全运会冠军和CBA总冠军的“双冠王”。

== 家庭方面 ==
已婚，育有一女。现任[[中国国家男子篮球队|中国国家男子篮球队]]主力控球后卫[[郭艾伦|郭艾伦]]和前东莞队球员[[郭凯伦|郭凯伦]]是其亲侄子，郭导也因此被广大辽宁球迷在网上戏称“老叔”。

== 职业荣誉 ==
*[[CBA|CBA]]联赛：98-99赛季中国男篮甲A联赛[[辽宁盼盼篮球俱乐部|辽宁男篮]]获亚军时的主力队员；2007-2008、2014-2015、2015-2016赛季CBA联赛[[辽宁盼盼篮球俱乐部|辽宁男篮]]获亚军时的主教练；2017-2018赛季辽宁男篮获冠军时的主教练。
*[[奥运会|奥运会]]：2000年，悉尼奥运会中国男子篮球队主力控球后卫，随队获得第十名；2004年雅典奥运会八强。
*[[亚青赛|亚青赛]]：代表中国队夺得1993、1995、1997年三次[[篮球亚青赛|篮球亚青赛]]冠军。
*[[东亚运动会|东亚运动会]]：2001年东亚运动会男子篮球冠军。
*[[世界大学生运动会|世界大学生运动会]]：2001年世界大学生运动会男子篮球亚军。
*[[亚运会|亚运会]]：2002年亚运会男子篮球亚军。
*[[男篮亚洲锦标赛|男篮亚洲锦标赛]]：2003年男篮亚洲锦标赛冠军。
*2007年与[[阿的江|阿的江]]一起获得该年度[[中国男子篮球职业联赛优秀教练|中国男子篮球职业联赛优秀教练]]。
*[[男篮亚洲锦标赛|男篮亚洲锦标赛]]：以教練身份獲得2009年男篮亚洲锦标赛亞军。
*[[全运会|全运会]]：2013年第十二届全运会男子篮球青年组冠军，2017年第十三届全运会男子篮球冠军。
