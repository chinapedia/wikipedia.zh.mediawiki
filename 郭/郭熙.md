'''郭熙'''（{{bd|約1000年||約1087年後||catIdx=郭}}）{{notetag|name=cance9|1=「徐復觀《中國藝術精神》及陳傳席《中國山水畫史》說1000年－1090年前後。王朝聞《中國美術史》說1000年－1087年後。潘運告《宋人畫論》說1020年－1100年。王伯敏《中國繪畫通史》說1023年－1085年後。」<ref name="multiple19">{{cite book |author=包莉秋 |title=《〈林泉高致〉中的美學思想研究》 |date=2004}}</ref>}}，[[表字|字]]'''淳夫'''，世稱'''郭河陽'''，為[[北宋|北宋]]著名[[画家|画家]]、[[繪畫|繪畫]][[理論|理論]]家，河陽溫縣<ref name="multiple7">{{cite book |title=北宋《宣和畫譜·卷十一·山水二·郭熙》}}</ref>（今[[河南|河南]][[温县|温县]]）人，享年80歲以上{{notetag|name=cance10|1=「[[元祐|元祐]]以來郭熙，明昌、泰和間張公佐，皆年過八十，而以山水擅名。」<ref name="multiple20">{{cite book |author=[[金朝|金]]、[[元朝|元朝]] [[元好問|元好問]] |title=《汾亭古意圖》}}</ref>}}，可徵的活動年代為[[宋神宗|宋神宗]][[熙寧|熙寧]]至[[元豐|元豐]]間，曾任翰林待诏直长，神宗死後，遂少有在畫院的活動紀錄，[[宋徽宗|宋徽宗]][[宣和|宣和]]時[[追贈|追贈]]正議大夫{{notetag|name=cance8|1=係北宋後期文階官制度「從三品」。}}。{{notetag|name=cancel5|1=「先大夫在樞府日，有旨賜第於龍津橋側。先君侍郎作提舉官，仍遣中使監修。比背畫壁，皆院人所作翎毛、花、竹及家慶圖之類。一日，先君就視之，見背工以舊絹山水揩拭几案，取觀，乃郭熙筆也。問其所自，則雲不知。又問中使，乃云：『此出內藏庫退材所也』。昔神宗好熙筆，一殿專背熙作，上即位後，易以古圖。退入庫中者，不止此耳。先君云：『幸奏知，若只得此退畫足矣。』明日，有旨盡賜，且命輿至第中，故第中屋壁，無非郭畫。誠千載之會也。」<ref name="multiple5">{{cite book |author=北宋 鄧椿 |title=《畫繼·卷十·論近》}}</ref>}}

郭熙擅長[[山水畫|山水畫]]，與[[李成|李成]]並稱「李郭」，名列「[[李成|李]]、郭、[[范寬|范]]、[[米芾|米]]」{{notetag|name=cance13|1=郭熙年代晚於范寬，實際上應作「李、范、郭、米」。}}的「北宋四大家」之一。

== 生平 ==
郭熙頗具藝術天分，初始自學，[[宋仁宗|宋仁宗]][[至和|至和]]元年（1054年）以前，曾應[[蘇舜元|蘇舜元]]之邀[[臨摹|臨摹]]李成的六幅《驟雨圖》，繪畫造詣因此更為精進，知名度也迅速攀升。{{notetag|name=cancel6|1=「家世無畫學，蓋天性得之。遂游藝於此以成名。」<ref name="multiple6">{{cite book |author=北宋 郭思 |title=《林泉高致集·序》}}</ref>}}{{notetag|name=cancel11|1=「子由嘆息終日，以為郭熙因為蘇才翁家摹六幅李成《驟雨圖》，從此筆墨大進。」<ref name="multiple13">{{cite book |author=北宋 黃庭堅 |title=《跋郭熙山水》}}</ref>}}宋神宗熙寧元年（1068年），郭熙透過[[富弼|富弼]]推薦來到[[汴京|汴京]]，先後為三司使[[吳中復|吳中復]]繪《秋景平遠圖》，以及[[開封府|開封府]]的邵亢作六幅《雪景[[屏風|屏風]]》，後續又為他人作畫，直到被招入[[皇宮|皇宮]]和著名畫家[[崔白|崔白]]、艾宣和葛守昌共同繪製《紫宸屏風》，受到宋神宗青睞，奉詔翰林圖畫院藝學，<ref name="multiple4"></ref>後官至翰林待詔直長，訂定御府藏畫品目及負責畫院運作等，聲望頗高。

== 風格 ==
郭熙山水画不僅繼承、發揚了北方流派的李成畫法，還「兼收並覽」，揉合南方畫派及各家之長，深入大自然「饱游饫看，历历罗列于胸中」，故能「佈置愈造妙處，然後多所自得」<ref name="multiple7"></ref>而備成一家。

郭熙早期畫風嚴謹精巧，晚年落筆雄壯。擅長用「高遠」、「深遠」和「平遠」三遠構圖巧妙布局，<ref name="multiple16">{{cite book |author=北宋 郭思 |title=《林泉高致集·山水訓》}}</ref>「上留天」，「下留地」，「大松大石必畫於大岸大波之上」，<ref name="multiple14">{{cite book |author=北宋 郭思 |title=《林泉高致集·畫訣》}}</ref>雲頭皴{{notetag|name=cance4|1=也作「-{卷}-雲皴」、「亂雲皴」或「鬼面皴」。}}畫山石和蟹爪畫樹枝{{notetag|name=cance12|1=「郭熙，[[唐棣|唐棣]]之樹……皆自草法中得來。」<ref name="multiple28">{{cite book |author=元朝 |title=《藝苑厄音》}}</ref>}}等技法，「毫鋒穎脫，墨法精微」的筆墨，繪出磅礴大景山水的「遠近淺深，四時朝暮，風雨明晦之不同」特徵，也體現寒林平野「枝茂凄然生陰」的孤寂蕭疏意境，讓觀畫者達到「不下堂筵，坐窮泉壑」，可行、可望、可游、可居之目的。<ref name="multiple16"></ref>

北宋《[[宣和畫譜|宣和畫譜]]》收錄有郭熙畫作30件，[[南宋|南宋]][[周密|周密]]《雲煙過眼錄》也多有著錄，[[元朝|元朝]]湯垕《畫鑒》稱見過真跡約50件。

[[臺北|臺北]][[國立故宮博物院|國立故宮博物院]]收藏的《[[早春圖|早春圖]]》軸是郭熙[[代表作|代表作]]。此外，尚有臺北故宮收藏的《關山春雪圖》軸、[[上海博物館|上海博物館]]收藏的《幽谷圖》半軸、[[雲南省博物館|雲南省博物館]]收藏的《溪山访友图》軸、[[紐約|紐約]][[大都會藝術博物館|大都會藝術博物館]]收藏的《樹色平遠圖》卷及[[北京|北京]][[故宮博物院|故宮博物院]]收藏的《窠石平远圖》卷等作品傳世。[[南京大學|南京大學]]圖書館收藏的《山村圖》軸與臺北故宮收藏的《寒林圖》軸相傳也都是郭熙之作。

郭熙另有山水畫論《[[林泉高致集|林泉高致集]]》傳於後世，為古代中國繪畫論述一大成就，對後世的繪畫發展有著巨大影響。

== 影響 ==
郭熙活耀於宋神宗熙寧至元豐年間，是馳名當代的繪畫大師，其畫風格調清新，獲当时文士[[苏轼|苏轼]]、[[蘇轍|蘇轍]]、[[黄庭坚|黄庭坚]]與[[王安石|王安石]]等人高度赞扬，也深得神宗賞識，皇宮、[[朝廷|朝廷]]的[[尚書省|尚書省]]、[[中書省|中書省]]、[[門下省|門下省]]、後省、[[樞密院|樞密院]]、[[翰林院|翰林學士院]]{{notetag|name=cancel7|1=「尚書省及六朝皆書周官，兩省及後省樞密學士院皆郭熙一手畫，中間甚有傑然可觀者。而學士院畫《春江曉景》為尤工。」<ref name="multiple8">{{cite book |author=北宋 葉夢得 |title=《石林燕語·卷四》}}</ref>}}及許多大型寺廟均隨處可見郭熙繪製的[[壁畫|壁畫]]與屏風掛畫，時有「神宗好熙筆」、「受眷被知，評在天下第一」、「禁中官局多熙筆跡」等說法，<ref name="multiple15">{{cite book |author=北宋 郭思 |title=《林泉高致集·畫記》}}</ref>，於是乎畫師們爭相摹仿，「獨步一時」<ref name="multiple7"></ref>，為北宋中期最具影響力的山水畫大師。

郭熙與李成的「李郭畫派」除盛行於北宋初期至中期，在元朝初期至中期也占有重要地位，師承的畫家主要有[[劉貫道|劉貫道]]、曹知白、王士元、[[王振鵬|王振鵬]]、羅稚川、[[唐棣|唐棣]]、趙雍、[[朱德潤|朱德潤]]、[[王淵_(畫家)|王淵]]及李容瑾等人，堪與[[董源|董源]]、[[巨然|巨然]]的「董巨畫派」相抗頡之二大主流。{{notetag|name=cancel1|1=「近代作畫，多宗董源、李成二家。」<ref name="multiple1">{{cite book |author=元朝 黃公望 |title=《寫山水訣》}}</ref>}}{{notetag|name=cancel2|1=「元季諸君子畫惟兩派，一為董源，一為李成。成畫有郭河陽為之佐，亦猶源有僧巨然副之也。」<ref name="multiple2">{{cite book |author=明朝 董其昌 |title=《畫旨》}}</ref>}}{{notetag|name=cancel3|1=「元時畫道最盛，惟董巨獨行，此外皆宗郭熙。」<ref name="multiple3">{{cite book |author=明朝 董其昌 |title=《畫眼》}}</ref>}}[[明朝|明朝]]的[[戴進|戴進]]、馬軾、[[謝時臣|謝時臣]]、[[仇英|仇英]]、王問及[[清朝|清朝]]的袁江、[[袁耀_(清朝)|袁耀]]等人，也都曾受郭熙畫風影響。

== 評價 ==
*後世將郭熙與李成並稱「'''李郭'''」，謂之「李郭畫派」，又將李成、[[范寬|范寬]]、[[米芾|米芾]]與之同列「'''北宋四大家'''」，世稱「'''李'''、'''郭'''、'''范'''、'''米'''」。
*蘇軾：「玉堂晝掩春日閒，中有郭熙畫《春山》。鳴鳩乳燕春睡起，白波青嶂非人間。」<ref name="multiple9">{{cite book |author=北宋 蘇軾 |title=《郭熙畫〈秋山平遠〉·潞公為跋尾》}}</ref>「目盡孤鴻落照邊，遙知風雨不同川；此間有句五人見，送與襄陽孟浩然。木落騷人已怨秋，不堪平遠發詩愁；要看萬壑爭流處，他日終煩顧虎頭。」<ref name="multiple10">{{cite book |author=北宋 蘇軾  |title=《東坡全集·卷十七·郭熙〈秋山平遠〉二首》}}</ref>
*黃庭堅：「黃州逐客賜未還，江南江北飽看山。玉堂對臥郭熙畫，發興已有青林間。郭熙官畫但荒遠，短紙曲折開秋晚。江村煙外雨腳明，歸雁行邊余疊峫。坐思黃甘洞庭霜，恨身不如雁隨陽。熙今頭白有眼力，尚能弄筆映窗光。畫取江山好風日，慰此將老鏡中發。但熙肯畫寬作程，五日十日一水石。」<ref name="multiple11">{{cite book |author=北宋 黃庭堅 |title=《山谷集·卷二·次韻子瞻題郭熙畫〈秋山〉》}}</ref>，「能作山川遠勢，白頭惟有郭熙」<ref name="multiple12">{{cite book |author=北宋 黃庭堅 |title=《山谷集·卷七·題鄭防畫莢》}}</ref>，「郭熙雖老眼猶明，便面江山取意成」<ref name="multiple13">{{cite book |author=北宋 黃庭堅 |title=《山谷集·卷九·題郭熙〈山水〉扇》}}</ref>，「郭熙元豐末年為顯聖寺悟道者作十二幅大屏，高二[[丈|丈]]余，山重水複，不以雲物映帶，筆意不乏。余嘗招子瞻兄弟共觀之，子由嘆息終日，以為郭熙因為蘇才翁家摹六幅李成《驟雨圖》，從此筆墨大進。觀此圖，乃是老年所作，可貴也。」<ref name="multiple13"/>
*蘇轍：「鳳閣鸞臺十二屏，屏上郭熙題姓名。崩崖斷壑人不到，枯松野葛相欹傾。黃散給舍多肉食，食罷起愛飛泉清。皆言古人不復見，不知北門待詔白髮垂冠纓。袖中短軸才半幅，慘淡百裏山川橫。巖頭古寺擁雲木，沙尾漁舟浮晚晴。遙山可見不知處，落霞斷雁俱微明。十年江海興不淺，滿帆風雨通宵行。投篙椓杙便止宿，買魚沽酒相逢迎。歸來朝中亦何有，包裹觀闕圍重城。日高困睡心有適，夢中時作東南征。眼前欲擬要真物，拂拭束絹付與汾陽生。」<ref name="multiple18">{{cite book |author=北宋 蘇轍 |title=《欒城集·卷十五·書郭熙橫卷》}}</ref>「亂山無盡水無邊，田舍漁家共一川；行遍江南識天巧，臨窗開卷兩茫然。斷雲斜日不勝秋，付與騷人滿目愁；父老如今亦才思，一蓑風雨釣槎頭。」<ref name="multiple21">{{cite book |author=北宋 蘇轍  |title=《灤城集·卷十五·次韻子瞻題郭熙〈平遠〉二絕》}}</ref>
*[[晁補之|晁補之]]：「漁村半落楚江邊，林外秋原雨外天；誰倚竹樓邀大艑，天涯暮色已蒼然。洞庭木落萬波秋，說與南人以自愁；欲指吳松何處是，一行征雁海山頭。」<ref name="multiple22">{{cite book |author=北宋 晁補之  |title=《雞肋集·卷二十·題工部文侍郎周翰郭熙〈平遠〉二首》}}</ref>
*[[張耒|張耒]]：「漁村橘市楚江邊，人外秋原雨外川；遣騎竹邊邀短艇，天涯暮色已蒼然。洞庭葉落萬波秋，說與南人亦自愁；指點吳江何處是，一行鴻雁海山頭。」<ref name="multiple23">{{cite book |author=北宋 張耒  |title=《柯山集·卷二十三·題文周翰郭熙〈山水〉二首》}}</ref>
*畢仲游：「窗間咫尺似天邊，不識應言小輞川；聞說平居心目倦，暫開黃卷即醒然。木落山空九月秋，畫時應欲遣人愁；因思夢澤經由處，二十年間若轉頭。」<ref name="multiple24">{{cite book |author=北宋 畢仲游  |title=《西台集·卷二十·和子瞻題文周翰郭熙〈平遠圖〉二首》}}</ref>
*郭思：「《煙生亂山》，生絹六幅，皆作平遠，亦人之所難。一障亂山，幾數百里，煙嵐綿聯，亂山截嶭，矮林小寺，間見掩映，看之令人意興無窮。此圖乃平遠之尤物也。」<ref name="multiple25">{{cite book |author=北宋 郭思 |title=《林泉高致集·畫格拾遺》}}</ref>
*《宣和畫譜》：「善山水寒林，得名于時。初以巧贍致工，既久又益精深，稍稍取李成之法，布置愈造妙，處然後多所自得。至攄發胸意，則於髙堂素壁，放手作長松巨木，回溪斷崖，岩岫巉絶，峰巒秀起，雲煙變滅，靄之間千態萬狀。論者謂熙獨步一時，雖年老落筆益壯，如隨其年貌焉。熙後著《山水畫論》，……言皆有序，可爲畫式。」<ref name="multiple7"></ref>
*湯垕：「善得煙雲出沒，峯巒隱顯之態，嘗論畫山曰：『春山淡冶而如笑，夏山蒼翠而如滴，秋山明净而如粧，冬山慘淡而如睡』，觀其議論可知其畫也，僕平生見真蹟約五十本，然絶佳者不過一二十軸而已。」<ref name="multiple17">{{cite book |author=元朝 湯垕 |title=《畫鑒·宋畫·郭熙》}}</ref>
*曹昭：「郭熙山水其山聳拔盤迴，水源高遠，多鬼面石。亂雲鷹爪樹，松業攢針，雜葉夾筆，單筆相半，人物以尖筆帶點鑿，絕佳。自著山水訓議論，卓絶千古，可規。」<ref name="multiple26">{{cite book |author=明朝 曹昭 |title=《格古要論·郭熙畫》}}</ref>
*吳其貞：「郭河陽《春雪圖》絹畫二大幅，氣色尚佳。用筆文秀，氣韻生動，得其自然，為神品上。」<ref name="multiple27">{{cite book |author=清朝 吳其貞 |title=《書畫記·卷六》}}</ref>
*[[清高宗|清高宗]][[乾隆|乾隆]]：「樹纔發葉溪開凍，樓閣仙居最上層，不藉柳桃閒點綴，春山早見氣如蒸。」
*[[顧復|顧復]]：「《早春圖》雙軿絹大幅，人物生動，室居幽邃，至於峰嵐波流，林木草卉，無非早春二字寫照焉。此大圖中之傑作也，古印鈐角。」<ref name="multiple29">{{cite book |author=清朝 顧復 |title=《平生壯觀·卷七》}}</ref>
