'''郭爱克'''（{{bd|1940年|2月18日||}}），中国辽宁沈阳人，[[中国|中国]]神经科学和生物物理学家。1965年毕业于[[莫斯科国立大学|莫斯科国立大学]]，1979年获[[慕尼黑大学|慕尼黑大学]]自然科学博士学位。现任中国科学院生物物理研究所和中国科学院上海生命科学研究院神经科学研究所研究员。2003年当选为中国科学院院士<ref name=gak>[http://sourcedb.cas.cn/sourcedb_ad_cas/zw2/ysxx/smkxhyxxb/ 中国科学院学部与院士·院士信息·生命科学和医学学部·郭爱克] {{webarchive|url=https://web.archive.org/web/20131216133759/http://sourcedb.cas.cn/sourcedb_ad_cas/zw2/ysxx/smkxhyxxb/ |date=2013-12-16 }}</ref>。目前研究方向为[[果蝇|果蝇]]学习记忆及抉择等认知功能的神经环路机制<ref name=zsw />。

== 生平 ==
1940年2月18日出生在[[中国|中国]][[辽宁省|辽宁省]][[沈阳市|沈阳市]]。根据国家安排，郭爱克于1960年夏天进入[[莫斯科国立大学|莫斯科国立大学]]生物物理学专业进行学习。1965年毕业后，受到[[文化大革命|文化大革命]]影响，郭爱克于1971年被下放到中科院在[[湖北省|湖北省]][[潜江市|潜江市]]农村开设的五七干校，被迫进行劳动<ref name=gk>{{Cite news |url=http://news.sciencenet.cn/sbhtmlnews/2010/7/233858.html |language=zh-hans |title=
郭爱克：“果蝇院士”的生命礼赞 |author1=郑千里 |author2=刘丹 |accessdate=2018-08-16 |date=2010-07-01 |publisher=科学网 }}</ref>。1976年，郭爱克被选派到[[北京语言学院|北京语言学院]]德语76班进行[[德语|德语]]学习。在德国科学技术交流中心奖学金的支持下，郭爱克选择前往[[慕尼黑大学|慕尼黑大学]]就读。1979年9月，郭爱克以特优的成绩获得[[慕尼黑大学|慕尼黑大学]]自然科学博士学位，是[[中华人民共和国|中华人民共和国]]第一位赴[[德国|德]]留学博士<ref name=int>{{Cite web |url=http://www.cebs.ac.cn/duiwu_teacher.php?id=36 |title=郭爱克 - 博士、研究员 |language=zh-hans |accessdate=2018-08-16 |publisher=[[中国科学院|中国科学院]]脑科学与智能技术卓越创新中心 |archive-url=https://web.archive.org/web/20170721045223/http://www.cebs.ac.cn/duiwu_teacher.php?id=36 |archive-date=2017-07-21 |dead-url=no }}</ref>。郭爱克于1982年11月至1984年6月作为访问学者在德国马普学会生物控制论研究所从事家蝇视觉系统图形与背景分辨的生物控制论课题的研究。1993年，郭爱克在[[中国科学院生物物理研究所|中国科学院生物物理研究所]]建立了中国第一个以果蝇为模式生物的学习与记忆实验室<ref name=zsw />。同年末，郭爱克第三次赴往德国，先后在[[乌尔茨堡大学|乌尔茨堡大学]]和马普学会生物控制论研究所进行学术研究<ref name=gk />。

2000年起，任[[973计划|973计划]]前沿领域项目“脑发育和可塑性基础研究”课题的首席科学家<ref>{{Cite web |url=http://sourcedb.sibs.cas.cn/zw/rck/200906/t20090629_1856081.html |title=郭爱克 |publisher=[[中国科学院|中国科学院]]上海生命科学研究院 |place=[[上海|上海]] |language=zh-hans |accessdate=2018-08-16 |date=2009-06-29 |archive-url=https://web.archive.org/web/20171216222241/http://sourcedb.sibs.cas.cn/zw/rck/200906/t20090629_1856081.html |archive-date=2017-12-16 |dead-url=no }}</ref>。2003年当选为[[中国科学院|中国科学院]]院士<ref name=gak />。同年至2008年，出任[[中国科学院|中国科学院]]神经科学研究所副所长<ref>{{Cite web |url=http://people.ucas.ac.cn/~akguo |title=郭爱克 |publisher=[[中国科学院大学|中国科学院大学]] |language=zh-hans |accessdate=2018-08-16 }}</ref>。2006年起任[[973计划|973计划]]前沿领域项目“脑的结构与功能可塑性”课题的首席科学家<ref name=zsw />。

== 荣誉 ==
1990年，郭爱克被[[中国科学院|中国科学院]]评为中国科学院有突出贡献的中青年专家<ref name=zsw>{{Cite web |url=http://www.ibp.cas.cn/ktzz/ktzz_ysktzz/201309/t20130917_3932645.html |title=院士课题组长 - 郭爱克 |language=zh-hans |accessdate=2018-08-16 |date=2013-09-17 |publisher=[[中国科学院生物物理研究所|中国科学院生物物理研究所]] |archive-url=https://web.archive.org/web/20171216224458/http://www.ibp.cas.cn/ktzz/ktzz_ysktzz/201309/t20130917_3932645.html |archive-date=2017-12-16 |dead-url=no }}</ref>。2006年，郭爱克荣获何梁何利基金会科学与技术进步奖和上海市科教党委系统优秀党务工作者荣誉称号。2008年，荣获亚太神经网络协会杰出成就奖<ref>{{Cite news |url=http://www.new1.uestc.edu.cn/?n=UestcNews.Front.Document.ArticlePage&Id=53820 |title=郭爱克院士做客名师讲堂阐述果蝇研究模型 |publisher=[[电子科技大学|电子科技大学]] |language=zh-hans |accessdate=2018-08-16 |date=2016-04-28 |archive-url=https://web.archive.org/web/20160505103559/http://www.new1.uestc.edu.cn/?n=UestcNews.Front.Document.ArticlePage&Id=53820 |archive-date=2016-05-05 |dead-url=no }}</ref><ref>{{Cite news |url=http://news.lzu.edu.cn/c/201805/48755.html |title=郭爱克院士访问兰州大学信息科学与工程学院 |date=2018-05-10 |language=zh-hans |accessdate=2018-08-16 |publisher=[[兰州大学|兰州大学]] |place=[[兰州|兰州]] }}</ref>。
