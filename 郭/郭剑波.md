'''郭剑波'''（{{bd|1960年|3月||}}），湖北省武汉市人，著名电力科学家，[[中国工程院院士|中国工程院院士]]。

==生平==
毕业于[[华中科技大学|华中科技大学]]电力系统及其自动化专业。之后担任[[中国电力科学研究院|中国电力科学研究院]]院长。2012年11月，当选为[[中国共产党第十八届中央委员会候补委员列表|中国共产党第十八届中央委员会候补委员]]。2013年12月，当选为[[中国工程院|中国工程院]]能源与矿业工程学部院士<ref>{{cite news|title=热烈祝贺我院郭剑波校友当选中国工程院院士|url=http://ceee.hust.edu.cn/article.aspx?id=5916|accessdate=2014-02-06|newspaper=华中科技大学|deadurl=yes|archiveurl=https://web.archive.org/web/20140221220659/http://ceee.hust.edu.cn/article.aspx?id=5916|archivedate=2014-02-21}}</ref>。

长期从事电力系统分析与控制研究，在电网规划、提高电网安全稳定水平和输电能力及风电并网安全等方面成绩显著<ref>{{cite web |author1=中国工程院 |title=郭剑波 |url=http://www.cae.cn/cae/html/main/colys/42379693.html |website=中国工程院}}</ref>。先后参加了三峡输电等电网规划研究，主持了上世纪90年代开展的全国互联电网（2020～2050年）规划系列研究；主持研制了具有自主知识产权的可控串补装置和特高压串补装置，带领团队开发了跨大区交直流协调控制系统，提高了电网安全稳定水平和输电能力；组织建成了“国家能源大型风电并网系统研发（实验）中心”，为行业提供了公共研发实验平台，提高了风电并网安全水平。获国家科技进步一等奖1项<ref>{{cite web |author1=中华人民共和国科学技术部 |title=2008年度国家科学技术进步奖获奖项目 |url=http://www.most.gov.cn/ztzl/gjkxjsjldh/jldh2008/jldh08jlgg/200901/t20090109_66659.htm}}</ref>、二等奖1项，何梁何利基金科学与技术进步奖<ref>{{cite web |author1=何梁何利基金会 |title=郭剑波 何梁何利基金会 |url=http://www.hlhl.org.cn/news/findnews/showsub.asp?id=1186}}</ref>。
