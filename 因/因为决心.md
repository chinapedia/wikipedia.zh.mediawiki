《'''因为决心'''》（{{lang-en|We Just Decided To}}）是美国电视剧《[[新闻编辑室|新闻编辑室]]》的第一集，最早于2012年6月24日在[[HBO|HBO]]台首播。《因为决心》由阿伦·索尔金编剧并由格雷格·莫托拉导演。威尔·麦克沃伊（由[[杰夫·丹尼尔|杰夫·丹尼尔]]饰演）因一次政治上的长篇大论而导致节目的大部分员工离开了他，他新的[[执行制片人#电视|执行制片人]]却是他的前女友麦肯兹·麦克海尔（由[[艾蜜莉·摩提默|艾蜜莉·摩提默]]饰演）。此时墨西哥湾的一个油井[[2010年墨西哥湾漏油事故|突然爆炸]]，《晚间新闻》的职员们也要开始面对新的挑战。

《[[娱乐周刊|娱乐周刊]]》曾在2009年4月报道，还在《[[社交网络|社交网络]]》剧组工作的索金准备写一部关于电视台背后的故事的新电视剧。为了解更多有关有线电视的知识，2010年索金作为不携带照相机的客人去参观了[[MSNBC|MSNBC]]的节目《{{link-en|NBC新闻倒数之凯斯点评|Countdown with Keith Olbermann}}》，他仔细研究了这个节目的制作过程，还在做客《帕克·斯皮策》（现在叫《{{link-en|竞技场上|In the Arena}}》）时与工作人员深入交流。索金告诉《[[电视指南|电视指南]]》他对媒体有了不那么讨厌的看法：“他们尽力在报道内容上下功夫，这不容易，因为他们要担心有没有不小心打广告、政治上正不正确，还有企业的问题。”<ref name="huver1">{{cite web |title=Aaron Sorkin Teases Details on His New HBO Pilot |url=http://www.tvguide.com/News/Aaron-Sorkin-Teases-1028756.aspx |work=《[[电视指南|电视指南]]》 |date=2011-02-01 |first=Scott |last=Huver |accessdate=2011-12-22}}</ref>

根据{{link-en|尼尔森媒体调查机构|Nielsen Media Research}}的调查，《因为决心》吸引了214万人观看，而这一集在评论界的评价褒贬不一。


== 剧情 ==
电视剧《新闻编辑室》以新闻主播威尔参与一个辩论开始，他就美国最近的衰败发表了有争议的言论，引起了轩然大波。为了避风头，威尔去度了两个星期的假，但回来时他发现他的执行制片人和大部分员工都跳槽去了十点档节目。接着，老板给他安排了一个新制片人，但是威尔很快知道了他的新执行制片人，麦肯兹·麦克海尔，正是他的前女友。威尔勉强允许了麦肯兹给他打工，但前提是每个周末威尔都能炒了麦肯兹。与此同时，发生了一个突然事件，墨西哥湾的一个油井发生了爆炸，但是编辑室的所有人都觉得不值得播报。多亏了麦肯兹跟班，吉姆，的两个线人，编辑室的员工才知道这个就在路易斯安那旁边发生的石油泄漏很难堵上。威尔做出了决定，路易斯安那的石油泄漏也成了当天《晚间新闻》的头条。最终，他们的节目成了最早播报石油泄漏新闻，而其他的新闻台则输掉了这一场信息大战。


== 制作 ==
=== 进度 ===
《[[娱乐周刊|娱乐周刊]]》于2009年4月放出消息，仍在《社交网络》剧组工作的编剧索金准备写一部新的电视剧，讲的是电视台幕后的故事。<ref>{{cite web |title=Aaron Sorkin: Come back to TV! |url=http://popwatch.ew.com/2009/04/10/sure-well-watch/ |work=《[[娱乐周刊|娱乐周刊]]》 |first=Lynette |last=Rice |date=2009-04-10 |accessdate=2011-12-22}}</ref>索金曾一手打造了《{{link-en|体育之夜|Sports Night}}》和《[[日落大道60号演播室|日落大道60号演播室]]》两部剧，都描述了有关虚构的电视节目的故事。<ref>{{cite web |title= HBO picks up Aaron Sorkin's cable news drama |url=http://www.avclub.com/articles/hbo-picks-up-aaron-sorkins-cable-news-drama,61506/ |work=[[影音俱乐部|影音俱乐部]] |date=2011-09-08 |first=Sean |last=O'Neal |accessdate=2011-12-22}}</ref>有消息称从2010年开始索金将和HBO合作，<ref>{{cite web |title=Aaron Sorkin's Cable News Network Project Awaits Greenlight At HBO |url=http://www.deadline.com/2011/01/aaron-sorkins-cable-news-network-project-awaits-green-light-at-hbo/ |publisher=Deadline网站 |date=2011-01-23 |first=Nellie |last=Andreeva |accessdate=2011-12-23}}</ref>2011年2月，索金在BBC新闻正式宣布了这个消息。<ref>{{cite video |title=The Social Network's Aaron Sorkin reveals future projects |url=http://www.bbc.co.uk/news/entertainment-arts-12250832 |publisher=BBC新闻网 |date=2011-01-21 |accessdate=2011-12-22}}</ref>

2011年9月8日，HBO预定了2012年夏季一整季10集的电视剧。<ref name="ordered">{{cite web |title=Aaron Sorkin's HBO cable news drama ordered to series |url=http://insidetv.ew.com/2011/09/08/aaron-sorkin-cable-news-drama/ |work=《[[娱乐周刊|娱乐周刊]]》 |date=2011-09-08 |first=James |last=Hibberd |accessdate=2011-12-22}}</ref><ref>{{cite web |title=Aaron Sorkin HBO Cable News Pilot Ordered To Series |url=http://www.huffingtonpost.com/2011/09/08/aaron-sorkin-hbo-cable-news-pilot_n_954209.html |work=《[[赫芬顿邮报|赫芬顿邮报]]》 |date=2011-09-08 |first=Jordan |last=Zakarin |accessdate=2011-12-22}}</ref><ref name="ewfonda">{{cite web |title=Jane Fonda joins Aaron Sorkin's HBO drama |url=http://insidetv.ew.com/2011/12/14/jane-fonda-aaron-sorkins-hbo-cable-news-drama/ |work=《[[娱乐周刊|娱乐周刊]]》 |date=2011-12-14 |first=James |last=Hibberd |accessdate=2011-12-27}}</ref>在《新闻编辑室》第二集播出的第二天，HBO将此剧续订至第二季。<ref name="renew">{{cite web |title= Breaking: HBO renews 'Newsroom,' 'True Blood' |url=http://insidetv.ew.com/2012/07/02/breaking-hbo-renews-newsroom-true-blood/ |work=《[[娱乐周刊|娱乐周刊]]》 |date=July 2, 2012 |accessdate=July 2, 2012 |first=Lynette |last=Rice}}</ref>

=== 剧集名 ===
《新闻编辑室》的试播集还在拍摄的时候，曾一度被命名为《越往后越多》。<ref>{{cite web|title=This Just In: Aaron Sorkin's HBO Newsroom Drama Has a New Title |url=http://www.hollywood.com/news/This_Just_In_Aaron_Sorkins_HBO_Newsroom_Drama_Has_a_New_Title/11671197 |publisher=Hollywood.com |first=Michael |last=Arbeiter |date=2011-12-22 |accessdate=2011-12-22 |archiveurl=https://archive.is/20130103134641/http://www.hollywood.com/news/This_Just_In_Aaron_Sorkins_HBO_Newsroom_Drama_Has_a_New_Title/11671197 |archivedate=2013-01-03 |deadurl=yes }}</ref> 之后在2011年11月29日，HBO向[[美国专利及商标局|美国专利及商标局]]注册了“新闻编辑室”的商标。<ref name="tvn2">{{cite web |title=Will HBO and Aaron Sorkin Draw Canada's Ire With The Name Of Their New Show 'Newsroom'? |url=http://www.mediabistro.com/tvnewser/will-hbo-and-aaron-sorkin-draw-canadas-ire-with-the-name-of-their-new-show-newsroom_b101243 |publisher=TV Newser |first=Alex |last=Weprin |date=2011-12-06 |accessdate=2011-12-22}}</ref>这个新名字很快被用来和肯·克里曼创作的加拿大同名电视喜剧相比较，后者在[[CBC|CBC]]和[[PBS|PBS]]上播放。<ref name="tvn2" /><ref>{{cite web |title=Anyone remember 'The Newsroom'? |url=http://jimromenesko.com/2011/12/06/the-newsroom/ |publisher=JimRomenesko.com |date=2011-12-06 |first=Jim |last=Romenesko |authorlink=Jim Romenesko |accessdate=2011-12-22 |archive-url=https://web.archive.org/web/20120503015202/http://jimromenesko.com/2011/12/06/the-newsroom/ |archive-date=2012-05-03 |dead-url=yes }}</ref>《新闻编辑室》的命名最终在2011年12月21日HBO的一个宣传2012年新剧的活动上确定。<ref name="seriesname">{{cite web |title= Aaron Sorkin's New HBO Series Gets Title: 'The Newsroom' |url=http://www.deadline.com/2011/12/aaron-sokins-new-hbo-series-gets-title-the-newsroom/ |publisher=Deadline.com |date=2011-12-21 |first=Nellie |last=Andreeva |accessdate=2011-12-22}}</ref>

海梅·温曼在《[[Maclean's|Maclean's]]》杂志上撰文称这个命名“有一点冷酷而有趣的提醒了人们，美国的电视产业还没完全取代加拿大...加拿大的那个《新闻编辑室》被认为是加拿大史上最伟大的电视节目，但是一个美国电视台并不惧怕如此直接的比较：就算有人已经听说过那个节目，他们可能以为这个国家大多数人没有听过。”<ref>{{cite web |title= Americans don't take us serious |url=http://www2.macleans.ca/2011/12/22/americans-dont-take-us-serious/ |work=Maclean's |date=2011-12-22 |first=Jaime |last=Weinman |accessdate=2011-12-26}}</ref>
