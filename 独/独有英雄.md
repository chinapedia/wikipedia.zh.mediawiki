《'''独有英雄'''》，2013年上映于中国大陆的古装剧，导演杨文军，主演周一围、薛佳凝、李小冉、包贝尔、张晞临、倪大宏。<ref>{{cite web |url= http://ent.sina.com.cn/r/m/2013-11-14/15594042957.shtml|title= 评论：创新是《独有英雄》兴奋点和着力点|accessdate= 2013|publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.163.com/13/1107/10/9D2QO82N00014Q4P.html |title=《独有英雄》(图) |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/v/m/2013-11-14/10484042639.shtml |title=《独有英雄》制片人狠批狗血雷剧误人子弟 |accessdate=2013 |publisher= 新浪网|work= |language=zh-hans}}</ref>2013年10月20日[[郑州电视台|郑州新闻综合频道]]/[[辽宁电视台|辽宁影视频道]]首播，10月31日[[湖北卫视|湖北卫视]]/[[新疆卫视|新疆卫视]]上星。

==剧情==
[[清朝|清朝末年]]，[[京剧|京剧]]演员马玉郎放弃演艺事业，从事[[煤|煤矿]]开采，实业救国，他用智慧战胜了清朝余孽、西方列强，最终闯出自己的一片天地。
