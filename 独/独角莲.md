'''独角莲'''（[[学名|学名]]：{{lang|la|''Typhonium giganteum''}}）是[[天南星科|天南星科]][[犁头尖属|犁头尖属]]的植物，为中国的特有植物。

== 形态 ==
[[多年生草本|多年生草本]]，地下[[块茎|块茎]]卵圆形，有毒；基生[[叶|叶]]1-4片，戟状箭形，全缘；夏季开[[花|花]]，[[肉穗花序|肉穗花序]]，外具紫色佛焰苞。

== 分布 ==
分布在[[中国大陆|中国大陆]]的[[河南|河南]]、[[吉林|吉林]]、[[山东|山东]]、[[广东|广东]]、[[广西|广西]]、[[四川|四川]]至[[西藏|西藏]]南部、[[河北|河北]]、[[陕西|陕西]]、[[甘肃|甘肃]]、[[湖北|湖北]]、[[辽宁|辽宁]]等地，生长于海拔500米至1,500米的地区，多生于山地较湿阴处，目前尚未由人工引种栽培。

== 用途 ==
中医上块茎入药，称“白附子”、“禹白附”，主治[[中风|中风]]痰壅、口眼斜歪、偏正[[头痛|头痛]]、[[破伤风|破伤风]]等症，但有毒；又可作土农药。

== 别名 ==
滴水参、天南星（湖北鹤峰） 野芋（湖北均县） 白附子（四川、湖北） 禹白附、牛奶白附（河北药材名） 麦夫子（河南嵩县） 疔毒豆（东北） 麻芋子（甘肃） 芋叶半夏（广西临桂） “达瓦罗玛吉节涧”（藏族语） 鸡心白附（西藏）
