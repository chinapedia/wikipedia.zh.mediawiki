'''盛岡市'''（{{jpn|j=盛岡市|hg=もりおかし|rm=Morioka shi|f=y}}）是位於[[日本|日本]][[岩手縣|岩手縣]]內陸地區的一個城市，地理位置接近岩手縣的中央，同時是岩手縣的[[縣廳所在地|縣廳所在地]]。盛岡古稱「不来方」、「古志方」，其歷史起源於[[平安時代|平安時代]][[桓武天皇|桓武天皇]]下令在此興建{{link-ja|志波城|志波城}}<ref>{{cite web |url=http://www.city.morioka.iwate.jp/shisei/moriokagaido/rekishi/1009476/index.html　 |script-title=ja:国指定史跡志波城跡・志波城古代公園 |accessdate=2017-04-18 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[安土桃山時代|安土桃山時代]]，盛岡最初被稱為「森岡」，是[[南部氏|南部氏]]的[[城下町|城下町]]。在[[江戶時代|江戶時代]]，[[盛岡藩|南部藩]]藩主南部氏將森岡改為「盛岡」，寓意祈求這裡興盛繁榮<ref>{{cite web |url=http://www.bunka.pref.iwate.jp/seikatsu/chimei/data/shimei.html　 |script-title=ja:市名の由来 |accessdate=2017-04-18 |publisher={{lang|ja|いわての文化情報大事典}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；南部藩也在此之後改名盛岡藩。[[廢藩置縣|廢藩置縣]]之後，盛岡市成為[[岩手縣廳|岩手縣廳]]所在地。1982年[[東北新幹線|東北新幹線]]的通車令盛岡市一躍成為[[北東北|北東北]]地區的交通樞紐<ref>{{cite web |url=http://www.mlit.go.jp/kokudokeikaku/suishinchousa/pdf/h14/10shinkansen(aomori-ken)/026-040.pdf　 |script-title=ja:第２章 北東北三県における交通ネットワーク及び高次都市機能の現状 |accessdate=2017-04-18 |publisher={{lang|ja|国土交通省}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，使得眾多企業和政府機關將其在北東北的分支機構設在盛岡。盛岡市在2000年11月升格成為[[特例市|特例市]]，並在2008年4月進一步升格為[[中核市|中核市]]<ref name="Chukakushi">{{cite web |url=http://www.city.morioka.iwate.jp/shisei/gappei/chukakushi/1010881.html |script-title=ja:中核市移行まで |accessdate=2017-04-18 |publisher={{lang|ja|盛岡市}} |language=ja}}</ref>，確立了北東北據點城市之一的地位。

== 歷史 ==
盛岡市內最早的人類活動遺跡是小石川遺跡，其歷史可追溯至[[舊石器時代|舊石器時代]]<ref>{{cite web |url=http://www.city.morioka.iwate.jp/shisei/moriokagaido/rekishi/1009437/1009456.html　 |script-title=ja:盛岡の歴史と遺跡1(旧石器時代) |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在大館町遺跡和大新町遺跡則發現了[[繩文時代|繩文時代]]的遺跡<ref>{{cite web |url=http://www.city.morioka.iwate.jp/shisei/moriokagaido/rekishi/1009437/1009457.html　 |script-title=ja:盛岡の歴史と遺跡2(縄文時代1) |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。自[[彌生時代|彌生時代]]至[[奈良時代|奈良時代]]，盛岡地處和人勢力和北方勢力的分界地帶<ref>{{cite web |url=http://www.city.morioka.iwate.jp/shisei/moriokagaido/rekishi/1009437/1009459.html　 |script-title=ja:盛岡の歴史と遺跡4(弥生・古墳時代) |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，和[[大和王權|大和王權]]在東北地方的主要據點[[仙台平原|仙台平原]]相比，中央政權對盛岡的影響力較弱。直到[[平安時代|平安時代]]時期，[[桓武天皇|桓武天皇]]開始經營東北，派[[坂上田村麻呂|坂上田村麻呂]]在現在的盛岡市興建{{link-ja|志波城|志波城}}，是當時[[陸奧國|陸奧國]]的北界，盛岡自這一時期開始進入[[律令制|律令制]]的控制區域之內<ref>{{cite web |url=http://www.city.morioka.iwate.jp/shisei/moriokagaido/rekishi/1009437/1009460.html　 |script-title=ja:古代(飛鳥・奈良・平安時代，約1400年～800年前)1 |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。10世紀時，盛岡處於{{link-ja|安倍氏 (奧州)|安倍氏 (奥州)|安倍氏}}的支配之下<ref>{{cite web |url=http://www.city.morioka.iwate.jp/shisei/moriokagaido/rekishi/1009437/1009461.html　 |script-title=ja:古代(飛鳥・奈良・平安時代，約1400年～800年前)2 |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[後三年之役|後三年之役]]之後，[[奧州藤原氏|奧州藤原氏]]成為盛岡的統治者，直到[[鎌倉幕府|鎌倉幕府]]成立<ref name="History1">{{cite web |url=http://www.city.morioka.iwate.jp/shisei/moriokagaido/rekishi/1009437/1009462.html　 |script-title=ja:中世・近世(鎌倉時代～江戸時代，約800年～150年前)1 |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在[[鎌倉時代|鎌倉時代]]，[[源賴朝|源賴朝]]在攻滅奧州藤原氏之後將岩手郡賜給[[工藤氏|工藤氏]]管理。[[南北朝_(日本)|南北朝]]時期，盛岡一帶被親北朝的斯波氏、稗貫氏和親南朝的北畠氏、葛西氏、南部氏等氏族割據，最終形成以岩手郡為根據地的南部氏的對以紫波郡為根據地的斯波氏的局面。這一態勢持續到16世紀。1588年（[[天正_(日本)|天正]]16年），南部氏完全控制了岩手郡和斯波郡（改名自紫波郡），並在1597年（天正20年）開始修築不來方城（現在的[[盛岡城|盛岡城]]），盛岡城下町的歷史自此開始<ref name="History1"></ref>。[[江戶時代|江戶時代]]的盛岡是[[盛岡藩|盛岡藩]]的中心城市和南部氏的統治據點。

|bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1870年，盛岡藩被盛岡縣取而代之，並在1872年和一些其他藩的領地統合，改名為岩手縣<ref>{{cite |fix-attempted=yes }}</ref>。1874年，盛岡城的大部分建築被拆除<ref name="History2">{{cite web |url=http://www.city.morioka.iwate.jp/shisei/moriokagaido/rekishi/1009334.html　 |script-title=ja:盛岡市の歴史 |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1889年，盛岡正式設市，是日本首批「市」之一<ref }}</ref>。戰前的盛岡是一座軍事都市，不僅陸軍工兵第八大隊和陸軍騎兵第三旅團23、24聯隊駐紮於此，盛岡還設有{{link-ja|陸軍預備士官學校 (日本)|陸軍予備士官学校 (日本)|陸軍預備士官學校}}，有「軍都」之稱<ref>{{cite web link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1945年3月10日，盛岡遭到美軍空襲，站前區域受損嚴重<ref name="Station">{{cite web |url=http://www.soumu.go.jp/main_sosiki/daijinkanbou/sensai/situation/state/tohoku_02.html　 |script-title=ja:盛岡市における戦災の状況（岩手県） |accessdate=2017-04-19 |publisher={{lang|ja|総務省}} |language=ja }}{{dead == 地理 ==
[[File:Mt._Iwate_and_Morioka.jpg|thumb]]
盛岡市位於岩手縣內陸地區{{link-ja|北上盆地|北上盆地}}的中部，市中心地處{{link-ja|雫石川|雫石川}}、{{link-ja|中津川 (岩手縣)|中津川 (岩手県)|中津川}}和[[北上川|北上川]]交匯處<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《東北 地図で読む百年》 |year=2000 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-5052-2|language=ja}}</ref>{{rp|31}}。中心市區被[[岩手山|岩手山]]、{{link-ja|秋田駒岳|秋田駒ヶ岳}}、{{link-ja|秋田駒岳|秋田駒ヶ岳}}、[[早池峰山|早池峰山]]、{{link-ja|姬神山 (岩手縣)|姫神山 (岩手県)|姬神山}}、{{link-ja|南昌山|南昌山}}、{{link-ja|東根山|東根山}}等山峰環繞，但市中心南側地區的地形較為開闊<ref name="Chishi">{{cite book |author={{lang|ja|田村俊和·石井英也・日野正輝}}|script-title=ja:《日本の地誌4　東北》 |year=2008 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16764-4|language=ja}}</ref>{{rp|228}}。自盛岡市中心看到的岩手山的景色被認為是最能代表盛岡的景色之一<ref>{{cite web |url=http://www.ccimorioka.or.jp/rekishi/gaido01.html |script-title=ja:北上川と岩手山 |accessdate=2017-04-19 |publisher={{lang|ja|盛岡商工会議所}} |language=ja}}</ref>。盛岡市中心地處河畔的[[河階|河階]]地形，地盤的地質以[[花崗岩|花崗岩]]為主<ref name="Chishi"></ref>{{rp|228}}。盛岡市在2006年和玉山村合併之前的面積是489.15平方公里，合併之後面積大幅增加到886.47平方公里<ref>{{cite web |url=http://www.city.morioka.iwate.jp/faq/guide/joho/1014100.html |script-title=ja:盛岡市の面積が知りたい |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja}}</ref>。而以盛岡市為中心的都市圈更佔岩手縣面積的23.8%，顯示出盛岡在岩手的重要地位<ref>{{cite web |url=http://www.city.morioka.iwate.jp/_res/projects/default_project/_page_/001/017/145/1.1.pdf |script-title=ja:盛岡市の概要 |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja}}</ref>。
=== 氣候 ===
盛岡市因地處{{link-ja|北上盆地|北上盆地}}而呈現出內陸性氣候的特徵。盛岡地方氣象台過去錄得的最高氣溫記錄是37.2℃（1924年7月12日），最低氣溫記錄是-20.6℃（1945年1月26日）<ref>{{cite web |url=http://www.data.jma.go.jp/obd/stats/etrn/view/rank_s.php?prec_no=33&block_no=47584&year=&month=&day=&view= |script-title=ja:観測史上1～10位の値（年間を通じての値） |accessdate=2017-04-18 |publisher={{lang|ja|気象庁}} |language=ja}}</ref>，冬夏溫差十分明顯。盛岡市的年平均氣溫是10.2℃，在日本全國都道府縣廳所在地城市中僅高於[[札幌市|札幌市]]<ref name="Weather">{{cite web |url=http://www.jma-net.go.jp/morioka/html/knowledge__tokusei.html |script-title=ja:岩手県の気候特性 |accessdate=2017-04-18 |publisher={{lang|ja|盛岡地方気象台}} |language=ja}}</ref>。在夏季，由於盛岡有時會發生{{link-ja|山背|やませ}}（鄂霍茨克海氣團的冷風吹襲[[北海道|北海道]]、[[東北地方|東北地方]]和[[關東地方|關東地方]]太平洋沿岸，導致濃霧發生的現象）或[[焚風|焚風]]現象，因此夏季溫度變動明顯<ref name="Weather"></ref>。冬季的盛岡則是本州最為寒冷的城市，並且容易發生[[輻射冷卻|輻射冷卻]]現象，使得有時會出現晴天反而錄的極端低溫的情況。盛岡市內的藪川氣象台在1945年1月26日錄得-35℃的低溫記錄，是[[本州|本州]]有記錄以來的最低氣溫<ref>{{cite web |url=http://www.forest-osaka.org/kisyo/kisyo08.html |script-title=ja:日本一、本州一、大阪一の寒さ |accessdate=2017-04-18 |publisher={{lang|ja|大阪森林インストラクター会}} |language=ja}}</ref>。

== 人口 ==
盛岡市在設市時面積4.47平方公里，人口有29,190人<ref name="History2"></ref>。此後隨著城市化的進展和轄區範圍的擴大，盛岡市的人口也穩健增加。戰後的1947年時，盛岡市人口已經超過10萬人，1970年代時更超過20萬人<ref name="Population1">{{cite web |url=http://www.city.morioka.iwate.jp/shisei/moriokagaido/morioka_profile/1009307/1009308.html　 |script-title=ja:統計資料 人口の推移 |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。然而1980年代之後，盛岡市的人口增長速度趨於平緩。2000年時，盛岡市人口達到302,857人的峰值，此後人口開始減少<ref name="Population1"></ref>。另一方面，由於岩手縣人口減少速度比盛岡市更快，岩手縣內盛岡市佔據的人口比重從1985年的20%增加到2015年的23.3%<ref name="Population2">{{cite web |url=http://www.city.morioka.iwate.jp/_res/projects/default_project/_page_/001/009/759/shiryou2.pdf　 |script-title=ja:盛岡市人口ビジョン策定後の推移 |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。和[[東北地方|東北地方]]其他主要城市相比，盛岡市的人口減少速度也相對較慢<ref name="Population2"></ref>。盛岡市的[[總和生育率|總和生育率]]為1.32，低於日本平均水準<ref name="Population2"></ref>。在[[東日本大震災|東日本大震災]]之後，由於遷入人口的增加，使得盛岡市人口一度轉為增加，但2014年之後已再次減少<ref name="Population2"></ref>。2017年2月28日時，盛岡市有人口292,599人<ref>{{cite web |url=http://www.city.morioka.iwate.jp/_res/projects/default_project/_page_/001/009/329/1.jinkousetai2903.xls　 |script-title=ja:住民基本台帳登録人口，外国人住民人口，推計人口(最新) |accessdate=2017-04-19 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 交通 ==
[[File:盛岡駅東口_2014-05-18_14-10.JPG|thumb]]
盛岡市是[[北東北|北東北]]地方的交通中轉樞紐城市。[[盛岡車站|盛岡車站]]是盛岡市的鐵路交通中心。[[在來線|在來線]]方面，[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）的[[東北本線|東北本線]]、[[山田線|山田線]]、[[花輪線|花輪線]]、[[田澤湖線|田澤湖線]]途徑盛岡車站，並且盛岡車站是東北本線在北側的起點車站<ref>{{cite web |url=https://www.jreast.co.jp/estation/stations/1565.html　 |script-title=ja:駅構内図（盛岡駅） |accessdate=2017-04-20 |publisher={{lang|ja|JR東日本}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。東北新幹線的盛岡至[[八戶車站|八戶車站]]部分開通之後，東北本線的盛岡以北部分改為[[第三部門|第三部門]]鐵路[[岩手銀河鐵道線|岩手銀河鐵道線]]，並以盛岡車站以南側起點<ref>{{cite web |url=http://www.igr.jp/wp/railway/route-map　 |script-title=ja:路線図 |accessdate=2017-04-20 |publisher={{lang|ja|IGRいわて銀河鉄道}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[新幹線|新幹線]]方面，自1982年[[東北新幹線|東北新幹線]]開通至2002年通車區間延長到八戶車站為止，盛岡車站是東北新幹線在北側的起點<ref name="Shinkansen"></ref>。加上盛岡車站也是[[秋田新幹線|秋田新幹線]]的端點車站之一，使得盛岡車站成為北東北地區新幹線中轉的樞紐。途徑盛岡的高速公路有[[東北自動車道|東北自動車道]]<ref>{{cite web |url=http://kosoku.jp/ic.php?ic=%E7%9B%9B%E5%B2%A1　|script-title=ja:盛岡IC（インターチェンジ） |accessdate=2017-04-20 |publisher={{lang|ja|高速.jp}} |language=ja}}</ref>。盛岡市區的巴士交通主要由{{link-ja|岩手縣交通|岩手県交通}}承擔運行<ref>{{cite web |url=http://www.iwatekenkotsu.co.jp/morioka.html |script-title=ja:盛岡地区の時刻表 |accessdate=2017-04-20 |publisher={{lang|ja|岩手県交通}} |language=ja}}</ref>。盛岡市內並沒有機場，距離盛岡最近的機場是[[花卷機場|花卷機場]]，自盛岡車站運行有前往花卷機場的巴士<ref>{{cite web |url=http://www.hna-terminal.co.jp/2009/09/post-bc34.html |script-title=ja:高速バス・路線バス |accessdate=2017-04-20 |publisher={{lang|ja|花巻空港}} |language=ja}}</ref>。

== 文化教育 ==
[[File:Morioka_Reimen.jpg|250px]]
{{link-ja|盛岡方言|盛岡弁}}屬於{{link-ja|東北方言 (日語)|東北方言|東北方言}}的{{link-ja|北奧羽方言|北奥羽方言}}。由於盛岡市在[[江戶時代|江戶時代]]是城下町，武士階級對城市文化影響強烈，使得盛岡方言有著敬語發達；町人用語和武家用語相近；促音、撥音、長音和其之前的音節發音相連等特徵<ref>{{cite web|url=http://hougen.sakura.ne.jp/shuppan/2014/06_033.pdf　|script-title=ja:岩手県盛岡市方言|accessdate=2017-04-18|publisher={{lang|ja|全国方言文法辞典資料集(2)}}|language=ja}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[石川啄木|石川啄木]]和[[宮澤賢治|宮澤賢治]]是最能代表盛岡文化的作家。石川啄木出生在玉山村，是日本著名的短歌作家。雖然他主要在[[東京|東京]]、[[北海道|北海道]]等地活躍，但對故鄉有深厚感情，寫下眾多以盛岡和東北地方為主題的短歌<ref>{{cite web|url=http://www.odette.or.jp/seishunkan/takuboku.html　|script-title=ja:石川啄木ともりおか|accessdate=2017-04-18|publisher={{lang|ja|もりおか啄木・賢治青春館}}|language=ja}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。盛岡市內有多處石川啄木的歌碑，並且盛岡車站東口的「{{lang|ja|もりおか}}」題字也是取自石川啄木的筆跡<ref>{{cite web |url=http://www.odette.or.jp/?page_id=1761　|script-title=ja:石川啄木 　青春のふるさと散歩  |accessdate=2017-04-18 |publisher={{lang|ja|盛岡市観光文化交流センター}} |language=ja}}</ref>。石川啄木紀念館也是盛岡代表的觀光景點<ref>{{cite web|url=http://www.city.morioka.iwate.jp/shisetsu/bunka/kinenkan/1006825.html　|script-title=ja:石川啄木記念館|accessdate=2017-04-18|publisher={{lang|ja|盛岡市}}|language=ja}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。詩人、童話作家宮澤賢治雖然出生在花卷町（現[[花卷市|花卷市]]），但在盛岡度過了學生時代。這一時期對他之後的創作生涯和作品風格都有很大影響<ref>{{cite web|url=http://www.odette.or.jp/seishunkan/kenji.html　|script-title=ja:宮沢賢治ともりおか|accessdate=2017-04-18|publisher={{lang|ja|もりおか啄木・賢治青春館}}|language=ja}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

盛岡的麵食文化十分聞名，有「麵都」之稱<ref>{{cite web|url=https://www.jreast.co.jp/otona/tvcm/morioka.html　|script-title=ja:岩手県「盛岡篇」|accessdate=2017-04-18|publisher={{lang|ja|JR東日本}}|language=ja}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，其中又以椀麵、盛岡炸醬麵和盛岡冷麵最為著名，並稱為「盛岡三大麵」。{{link-ja|椀麵|わんこそば}}是一種用椀（小木碗）重複添麵的蕎麥麵吃法，原本的用意是希望客人能確實吃飽。在盛岡當地，經常會在婚喪喜慶的宴席結束時提供蕎麥麵供客人食用。盛岡市每年舉辦吃椀麵大賽，在2000年曾有人達成15分鐘內吃下451碗椀麵的記錄<ref>{{cite web |url=http://www.wankosoba-azumaya.co.jp/wanko/　 |script-title=ja:わんこそばについて |accessdate=2017-04-18 |publisher={{lang|ja|東家}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。{{link-ja|盛岡炸醬麵|盛岡じゃじゃ麺}}起源於中國北部的[[炸醬麵|炸醬麵]]，當地的特色吃法是吃完大部分麵之後剩下少許麵條和湯，並在剩餘部分加入雞蛋使其成為雞蛋湯（{{lang|ja|チータンタン}}）<ref>{{cite web |url=http://www.iwate-np.co.jp/men/jaja/jaja-top.htm　 |script-title=ja:みそがうまさのミソ |accessdate=2017-04-18 |publisher={{lang|ja|岩手日報社}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。{{link-ja|盛岡冷麵|盛岡冷麺}}則是由在日本居住的[[朝鮮半島|朝鮮半島]]移民發明，融合了[[咸興|咸興]]冷面和[[平壤|平壤]]冷面的特點，常和[[烤肉|烤肉]]搭配食用<ref>{{cite web |url=http://www.pyonpyonsya.co.jp/remen/about/　 |script-title=ja:盛岡冷麺とは？ |accessdate=2017-04-18 |publisher={{lang|ja|ぴょんぴょん舎}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。盛岡是著名的[[日本酒|日本酒]]產地，釀酒產業傳統悠久，是日本酒南部杜氏流派的主要據點<ref>{{cite web |url=http://www.sakejapan.com/index.php?option=com_content&view=article&id=40&Itemid=54　|script-title=ja:杜氏の流派と分布 |accessdate=2017-04-18 |publisher={{lang|ja|日本酒サービス研究会・酒匠研究会連合会}} |language=ja}}</ref>。

盛岡市內現在有[[岩手大學|岩手大學]]和{{link-ja|岩手醫科大學|岩手医科大学}}<ref>{{cite web |url=http://www.iwate-med.ac.jp/ideology/ayumi/　 |script-title=ja:本学の歩み・年史 |accessdate=2017-04-20 |publisher={{lang|ja|岩手医科大学}} == 姊妹城市 ==
盛岡市和[[加拿大|加拿大]][[不列颠哥伦比亚|-{zh-hans:不列颠哥伦比亚省;zh-hk:卑詩省;zh-tw:不列顛哥倫比亞省;}-]][[維多利亞_(卑詩)|維多利亞]]在1985年締結了姊妹城市關係。盛岡市是[[新渡戶稻造|新渡戶稻造]]博士出生之地，而維多利亞是新渡戶稻造的去世之地，兩者因此成為姊妹城市<ref>{{cite web |url=http://www.city.morioka.iwate.jp/faq/katsudo/koryu/1014013.html　 |script-title=ja:盛岡市の姉妹都市について知りたい。 |accessdate=2017-04-18 |publisher={{lang|ja|盛岡市}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
