'''盛丕华'''（{{bd|1882年||1961年}}），中國实业家，[[中華人民共和國|中華人民共和國]]成立后[[上海市|上海市]]首任副市长。

== 生平 ==
盛丕華，[[宁波人|宁波人]]，1882年3月29日出生于[[浙江省_(清)|浙江省]][[镇海县|镇海县]]（现镇海区）骆驼桥一小店员家庭，家境贫寒。

曾历任上海物品证券交易所理事、证券部常务理事，[[中一银行|中一银行]]董事，[[上海总商会|上海总商会]]会董，[[上元企业公司|上元企业公司]]经理，[[红棉酒家|红棉酒家]]董事长等职。[[中國抗日戰爭|抗日战争]]期间，积极支持上海的抗日救亡运动<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node4485/node23931/node23933/node63685/userobject1ai53449.html 上海市地方志办公室 中国民主党派上海市地方组织志 >> 盛丕华 介绍]</ref>。抗日战争胜利后，参与发起成立[[中国民主建国会|中国民主建国会]]（简称“民建”）。[[第二次国共内战|第二次国共内战]]期间，上海各界代表组织赴南京向国民政府请愿，盛丕华是代表之一。1946年始，任民建常务理事。1949年被选举为[[第一届全国政协|第一届全国政协]]常务委员<ref>[http://www.mjshsw.org.cn/new_look.asp?mingjian_id=184 盛丕华为新中国献策] {{webarchive|url=https://web.archive.org/web/20070929002828/http://www.mjshsw.org.cn/new_look.asp?mingjian_id=184 |date=2007-09-29 }}</ref>。

中華人民共和國成立后，盛丕华曾先后担任[[政务院财经委员会|政务院财经委员会]]委员，[[华东军政委员会|华东军政委员会]]委员、副主席，[[上海市人民政府|上海市人民政府]]副市长，[[全国工商联|全国工商联]]副主任委员，[[全国人大代表|全国人大代表]]，第一、二、三届[[全国政协常务委员|全国政协常务委员]]。1952年10月，盛丕华当选为民建总会副主任委员。1955年4月、1960年2月，盛丕华先后分别当选为民建第一、二届中央副主任委员<ref>{{cite web |title = 民建上海市第一、二届委员会主任委员：盛丕华 |url = http://gov.eastday.com/shmj2011/node741/u1a20899.html |accessdate=2013-12-05 |website = 上海民建 }}</ref>。于1961年2月8日病逝，享年79岁。
