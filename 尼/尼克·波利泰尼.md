尼克·波利泰尼（{{lang-en|Nicholas "Nick" James Bollettieri}}，{{bd|1931年|7月31日|}}）生于美国纽约，俗称“网坛教父”，美国著名[[网球|网球]]教练。他是美国最好的“尼克‧波利泰尼网球训练营（Nick Bollettieri Tennis Academy）”的创始人及校长，培养出了许多网坛传奇巨星，如[[安德烈·阿加西|阿加西]]、[[莫妮卡·塞莱斯|塞莱斯]]、[[鲍里斯·贝克尔|贝克尔]]、[[莎拉波娃|莎拉波娃]]、[[维多利亚·阿扎伦卡|阿扎伦卡]]等。

== 早年 ==
生于[[纽约州|纽约州]][[佩尔汉姆|佩尔汉姆]]，父母均是是意大利后裔。1953年毕业于[[阿拉巴马州|阿拉巴马州]]春山学院，主修哲学。然后美国陆军服役，军衔至[[中尉|中尉]]，退伍后进入迈阿密大学法学院，后退学改作网球教练。
== 网坛教父 ==

尼克曾任[[马塞洛·里奥斯|马塞洛·里奥斯]]的教练达两年。1978年尼克在[[佛罗里达州|佛罗里达州]][[布雷登顿|布雷登顿]]建起了尼克‧波利泰尼网球学校,占地40英亩。<ref name="IMG Academy">{{cite web|title=IMG Academy Bollettieri Tennis Program|url=http://www.imgacademy.com/sports/tennis|accessdate=2013-04-22}}</ref>学校培养的世界顶尖的网球手包括[[安德烈·阿加西|安德烈·阿加西]]、[[吉姆·考瑞尔|吉姆·考瑞尔]]、[[莫妮卡·塞莱斯|莫妮卡·塞莱斯]]、[[玛丽·皮尔斯|玛丽·皮尔斯]]。威廉姆斯姐妹也曾在该校培训数年。[[安娜·库尔尼科娃|安娜·库尔尼科娃]]、[[玛丽亚·莎拉波娃|玛丽亚·莎拉波娃]]、[[达妮埃拉·汉图楚娃|达妮埃拉·汉图楚娃]]、[[耶莱娜·扬科维奇|耶莱娜·扬科维奇]]、[[妮科莱·瓦伊迪索娃|妮科莱·瓦伊迪索娃]]、[[萨比娜·利斯基|萨比娜·利斯基]]、[[萨拉·埃拉尼|萨拉·埃拉尼]]、[[托米·哈斯|托米·哈斯]]、[[马克斯·米尔内|马克斯·米尔内]]、[[克萨维埃·马利斯|克萨维埃·马利斯]]等也曾受训于他。<ref>http://sports.espn.go.com/sports/tennis/columns/story?columnist=garber_greg&id=3572727&lpos=spotlight&lid=tab6pos1</ref>

== 著作 ==
尼克是美国网球杂志的主编。曾出版自传''My Aces, My Faults''。也曾出版网球教学著作《波利泰尼网球手册》（''Bollettieri's Tennis Handbook''），相关的网球教学视频在网络上流传甚广。
