'''尼古劳斯·约瑟夫·冯·雅坎''' （Nikolaus Joseph von Jacquin，{{bd|1727年|2月16日|1817年|10月26日}}）是[[荷兰|荷兰]][[科学家|科学家]]，研究[[医学|医学]]、[[化学|化学]]和[[植物学|植物学]]。

== 生平 ==
他出生于[[荷兰|荷兰]]的[[莱顿|莱顿]]，在[[莱顿大学|莱顿大学]]学习[[医学|医学]]，然后先移居[[巴黎|巴黎]]，最后定居[[维也纳|维也纳]]。

在1755年至1759年期间，他被[[神圣罗马帝国|神圣罗马帝国]]的皇帝[[弗朗西斯一世|弗朗西斯一世]]派往[[中美洲|中美洲]]和[[西印度群岛|西印度群岛]]为他的[[美泉宫|美泉宫]]搜集[[花卉|花卉]]，因此雅克恩搜集了大量的[[动物|动物]]、[[植物|植物]]和[[矿物|矿物]]标本。

1762年，他被任命为位于现在[[斯洛伐克|斯洛伐克]]班斯卡-斯蒂沃尼卡的采矿学院的采矿和矿物学教授。1768年被任命为[[维也纳大学|维也纳大学]]的[[化学|化学]]和[[植物学|植物学]]教授兼植物园园长，直到去世，由他儿子[[约瑟夫·弗朗茨·冯·雅坎|约瑟夫]]接替他的教授职务。因为他的工作，他被授以爵位。从1774年至1783年，他被选为[[瑞典皇家科学院|瑞典皇家科学院]]的外籍院士。在1806年，他被任命为男爵。 1809年，他成为了皇家学会会员，皇家学会后来成为[[荷兰皇家艺术与科学学院|荷兰皇家艺术与科学学院]]。

他的著作包括《美洲标本选编》（1763年）<ref>{{cite web |author1=Nikolaus Joseph von Jacquin |title=Selectarum Stirpium Americanarum |url=http://www.botanicus.org/title/b12074652 |accessdate=2018-12-04 |language=la}}</ref>、《奥地利植物》（1773年-1778年）<ref>{{cite web |author1=Nikolaus Joseph von Jacquin |title=Florae Austriacae |url=http://www.botanicus.org/title/b12003402 |accessdate=2018-12-04 |language=la}}</ref>、《稀有植物图谱》（1781年-1793年）<ref>{{cite web |author1=Nikolaus Joseph von Jacquin |title=Icones Plantarum Rariorum |url=http://www.botanicus.org/title/b11927124 |accessdate=2018-12-04 |language=la}}</ref>。
