'''尼古拉·瓦西里耶维奇·果戈里-亚诺夫斯基'''（{{lang-ru|'''Никола́й Васи́льевич Го́голь-Яновский'''}}，[[俄文拉丁化|俄文拉丁化]]：{{transl|俄语|''Nikolai Vasilievich Gogol-Yanovski''}}；{{bd|1809年|4月1日|1852年|3月4日|catIdx=G}}）是[[俄罗斯|俄罗斯]][[作家|作家]]，生于现在的[[乌克兰|乌克兰]][[大索羅欽齊|大索羅欽齊]]的一个[[哥薩克|哥薩克]]家庭。他自幼爱好[[文学|文学]]，深受[[启蒙运动|启蒙运动]]的影响。1831年发表的《[[狄康卡近乡夜话|狄康卡近乡夜话]]》使他受到了[[亚历山大·普希金|亚历山大·普希金]]的赞誉。1836年，他的讽刺喜剧《[[钦差大臣_(喜剧)|钦差大臣]]》上演，在这部作品中，他用幽默的笔调和有力的讽刺手法，使[[俄国|俄国]]喜剧艺术发生了重大转折。1842年，《[[死魂灵|死魂灵]]》一出版，就“震撼了整个俄罗斯”，成为[[俄罗斯文学|俄罗斯文学]]走向独创性和民族性的重要标志。[[别林斯基|别林斯基]]称他为继[[亚历山大·普希金|亚历山大·普希金]]之后的“文坛盟主”、“诗人的魁首”。而整个19世纪40年代也被[[车尔尼雪夫斯基|车尔尼雪夫斯基]]称为“果戈里时期”。1847年，他发表了《与友人书信选》，公开表示对以前所有作品的忏悔。1852年，他在严重的[[东正教|东正教]]狂热中去世。果戈里是[[俄国|俄国]][[现实主义文学|现实主义文学]]的奠基人之一，也是“自然派”的创始人。

== 早年生活 ==
[[File:Василий_Гоголь-Яновский.jpg|left]]（1777年-1825年4月11日）]]
果戈里出生于[[乌克兰|乌克兰]][[波尔塔瓦省|波尔塔瓦]][[密尔格拉得|密尔格拉得]]附近的瓦西里耶卡村，祖先是[[烏克蘭|烏克蘭]]的小贵族，具有[[波兰|波兰]]血统。他的父亲瓦西里··阿法纳西耶维奇·果戈里-亚诺夫斯基（{{lang|ru|Василий Афанасьевич Гоголь-Яновский}}）是当地有名望的乡绅，曾在邮电部门供职，做过八品[[文官|文官]]，后辞去公职，在乡下当[[地主|地主]]，同时开始尝试写作，并成为一名[[诗人|诗人]]和民间喜剧[[作家|作家]]。他的父亲经常在朋友家的家庭舞台上上演自己写的[[喜剧|喜剧]]，还在其中扮演主要角色。这一切给早年间的果戈里留下了深刻的印象，激发了他对[[戏剧|戏剧]]乃至[[文学|文学]]的爱好。

[[File:Косяровская_Мария_Ивановна.jpg|right]]
他的母亲名叫玛丽娅·伊凡诺芙娜·果戈里-亚诺夫斯卡娅（{{lang|ru|Мария Ивановна Гоголь-Яновская}}）（娘家姓氏为：科夏洛夫斯卡娅，俄文原文为{{lang|ru|Косяровская}}），是一名虔诚的东正教徒，这对后来果戈里的[[东正教|东正教]]狂热埋下了一定的基础。

果戈里从小喜爱[[乌克兰|乌克兰]]的民谣、传说和民间[[戏剧|戏剧]]。1821～1828年就读于[[波尔塔瓦省|波尔塔瓦省]]涅仁高级科学中学，在这所中学读书时已经博览群书，并积极参加学校的文艺活动，曾扮演过[[冯维辛|冯维辛]]的[[讽刺|讽刺]][[喜剧|喜剧]]《纨绔少年》中的主角以及其他角色，而且演得很成功（他后来写的也是讽刺喜剧）。他在这所中学受到了[[十二月党人|十二月党人]]中的一些[[诗人|诗人]]、[[亚历山大·普希金|亚历山大·普希金]]的诗歌的影响（这促使他在创作初期想当一名[[诗人|诗人]]），他还受到了法国启蒙作家著作的深刻影响。这一切为他后来的创作打下了基础。1825年4月11日，父亲去世。在农村的生活是他创作的重大素材，农村生活促成他写成了《狄康卡近乡夜话》、《[[马车|马车]]》、《[[死魂灵|死魂灵]]》等与农村有关的佳作。1828年，他中学毕业，前往[[圣彼得堡|彼得堡]]，想在司法界谋得一官半职，他身上还带着写成了的[[田园诗|田园诗]]《汉斯·丘赫尔加坚》（长诗）的手稿，这是他的[[处女作|处女作]]。1829～1831年先后在圣彼得堡国有财产及公共房产局和封地局供职，亲身体验到小职员的贫苦生活。在此期间还到美术学院学习绘画。

== 初入文坛 ==
在[[圣彼得堡|聖彼得堡]]，他没有获得赏识，然后几经周折，成了圣彼得堡国有财产及公共房产局和封地局的一名缮写员，靠此维生，亲身体验到了小职员的贫苦生活（所以他在《[[外套|外套]]》和《[[狂人日记_(果戈里)|狂人日记]]》中写的是缮写员的故事，这里有着对他自己生活的回忆）。在此期间，他在美术学院学习了绘画。他后来在《涅瓦大街》、《肖像》等[[中篇小说|中篇小说]]中写的都是[[画家|画家]]的悲剧故事或传奇故事。

1829年，他发表了《汉斯·丘赫尔加坚》这一长诗，用的是真名。这首长诗是他登上了[[俄国|俄国]]文坛，但并没有获得太多的关注。他很快意识到[[诗歌|诗歌]]创作并非他的强项，于是转向了[[小说|小说]]和[[喜剧|喜剧]]。1830年，他以“果戈里”（{{lang|ru|'''Гоголь'''}}，他的姓氏的一半）为[[笔名|笔名]]发表了[[小说|小说]]《圣约翰节前夜》，这部小说得到了[[诗人|诗人]][[瓦西里·茹科夫斯基|瓦西里·茹科夫斯基]]的赞赏，并与之成了莫逆之交。1831年9月，[[短篇小说|短篇小说]]《[[狄康卡近乡夜话|狄康卡近乡夜话]]》发表，情节迂回曲折，充满幻想，具有[[乌克兰|乌克兰]]民间风格，内容大部分根据[[乌克兰|乌克兰]]民间传说写成，吸取了民间狂欢文化的营养，充满了欢快和幽默的语言，歌颂劳动人民的智慧、勇敢、情爱和热爱自由的性格，嘲弄邪恶势力的愚昧，被认为受了[[浪漫主义|浪漫主义]]的影响。9月，他出版了以这篇作品的题目命名的短篇小说集，受到了[[普希金|普希金]]和[[别林斯基|别林斯基]]的好评，他们称[[俄国|俄国]]文学已进入果戈里时期。。

这一年，他遇到了[[普希金|普希金]]，之后普希金成为他的朋友并给他提供了许多创作素材，此人的[[现实主义|现实主义]]作品对他影响极大，比如《钦差大臣》和《死魂灵》的素材就是[[普希金|普希金]]提供的。1834年，他进入[[圣彼得堡大学|圣彼得堡大学]]，当[[副教授|副教授]]，教授[[历史|历史]]，[[伊万·屠格涅夫|伊万·屠格涅夫]]就是他的学生之一。

== 文坛地位的巩固 ==
[[File:Nikolai_Gogol_-_Revizor_cover_(1836).jpg|left]]》初版扉页]]
1835年春季，他出版了[[喜剧|喜剧]][[剧本|剧本]]《三等弗拉基米尔[[勋章|勋章]]》和《婚事》，并开始迷恋[[喜剧|喜剧]]创作。《婚事》是他早期[[喜剧|喜剧]]的代表作，宣扬了婚恋自由。同年，他出版了两部短篇小说集：《彼得堡故事》（{{lang|ru|''Петербургские Повести''}}）和《密尔格拉得》。《彼得堡故事》中有《涅瓦大街》、《鼻子》、《肖像》、《外套》、《狂人日记》、《马车》、《罗马》构成。《密尔格拉得》里面有《旧式地主》、《塔拉斯·布尔巴》、《两个伊凡吵架故事》等优秀的中短篇小说。相较以前，这些[[小说|小说]]在题材上有了新的开拓，思想上更趋成熟，风格上有重大发展（讽刺的力度增强，不幽默的也更严肃），思想容量上也更为深刻。[[短篇小说|短篇小说]]《罗马》是他所有[[小说|小说]]中诗意最浓的一部。而他本人也被称为及普希金之后的“文坛盟主”、“诗人的魁首”。这一年，他根据[[普希金|普希金]]启发出来的素材，开始构思[[长篇小说|长篇小说]]《[[死魂灵|死魂灵]]》，并从[[圣彼得堡大学|圣彼得堡大学]]离职，专事创作。

== 创作的高峰 ==
=== 《钦差大臣》 ===
[[File:Мёртвые_души.JPG|right]]》的初版扉页，由果戈里亲自设计]]
1836年，根据[[普希金|普希金]]提供的一则荒诞见闻，果戈里在两个月内创作出了五幕[[喜剧|喜剧]]《[[钦差大臣_(喜剧)|钦差大臣]]》。创作期间，他对[[戏剧|戏剧]]的社会使命有了越来越明确、越来越深刻地认识。他要求在舞台上体现当代社会的生活和[[民族|民族]]特点：“请给我们展示俄罗斯性格，展示我们本身，我们的骗子手，我们的怪人！把我们搬上舞台，让大家去笑！”为此，他努力钻研适合舞台表演的讽刺喜剧。《钦差大臣》使他第一次实现了创作真正的、既真实而又尖刻的社会喜剧的心愿。果戈里把《钦差大臣》看作是自己创作中的一个转折点。他认为：“在《钦差大臣》以前，我作品中的幽默都是无目的的、轻率地，而只有在《钦差大臣》以及以后的创作中，我的嘲笑才有了正确的方向。”同年，他的《钦差大臣》出版了单行本，他揭露出了[[俄国|俄国]]官僚阶层中的真实的黑暗场景。这部喜剧具有强有力的[[讽刺|讽刺]]倾向，具有非凡的思想深度，而且具有独特的艺术风格。在他的作品中，细节、环境和人物性格的真实性，辛辣的讽刺手法，逼真的肖像描绘，个性化的语言以及舞台表演的观赏性等方面，都取得了卓越的成就。这使得俄国喜剧艺术发生了重大转折。[[赫尔岑|赫尔岑]]说道：“（《钦差大臣》写的是）当代俄国骇人听闻的自白，这与17世纪的科托希欣揭露的情况是一样的。”出版的同时，这部喜剧进行了公演，由米哈伊尔·谢苗诺维奇·谢普金主演。《钦差大臣》引起了纷纷议论。大多数观众在观看期间都笑了，因为这不再是专为逗乐而写的[[滑稽剧|滑稽剧]]，甚至[[尼古拉一世|尼古拉一世]]在观看期间也笑了，而且“笑得要死”。但《钦差大臣》也引起了许多[[御用文人|御用文人]]的攻讦，称之为“对[[俄罗斯|俄罗斯]]的诽谤”，指责果戈里是“俄罗斯的敌人”，要求给他“带上镣铐送到[[西伯利亚|西伯利亚]]”，等等，等等，不一而足。

果戈里对这些指责表示震惊，但“并不因此而不安”。可是，《钦差大臣》的出版与公演却很快引起了俄国当局的不满。为此，1836年6月，他离开俄国，进行出国游历，开始了长达6年的侨居生活。最开始的一年，他来到了[[罗马|罗马]]。1837年1月29日，他的好友[[普希金|普希金]]死于阴谋。此后果戈里在[[意大利|意大利]]和[[德国|德国]]生活了近5年时间，在此期间他写成了《死魂灵》的大部分。

=== 《死魂灵》 ===
[[File:Gogol_karandash.jpg|left]]称为“果戈里时期”]]
[[File:Gogol_church_register_entry.jpg|right]]
1841年9月，果戈里携带《[[死魂灵|死魂灵]]》的手稿回到[[俄国|俄国]]。当他把改定后的手稿送到[[莫斯科|莫斯科]]书刊审查机构审查时，当即被否决。于是他托[[别林斯基|别林斯基]]走后门关系，使这本书在[[彼得堡|彼得堡]]通过了审查。1842年，他对《钦差大臣》进行了增补，使它的[[讽刺|讽刺]]力量得到了增强。也是这一年，《[[死魂灵|死魂灵]]》的第一卷出版，引起了比《钦差大臣》更大的轰动。这部小说被公认为“自然派”的奠基石，“俄国文学史上无与伦比的作品”。[[赫尔岑|赫尔岑]]曾回忆说：“该小说的出版震动了整个俄国。”
[[别林斯基|别林斯基]]说道：“只有了解作品的思想和艺术处理手法，着重内容而不是‘情节’得人才能充分领略果戈里的[[史诗|史诗]]一样的作品。”这部小说猛烈抨击了[[农奴制|农奴制]]和当时的官场的黑暗，渴望寻找一条用[[东正教|东正教]]来解决国内问题的路子。果戈里认为上帝赋予他写作的天才，要让他向俄国指明在一个罪恶的世界中如何正确地生活，因此后来他不满意自己的作品。接下来的几年，他都是在争论、疾病和贫困中度过的，他逐渐丧失了创作激情。1845年6月，他将已经出版的第一部《死魂灵》书稿烧毁，继续重写。

== 创作的低谷 ==

果戈里开始沉迷于[[东正教|东正教]]狂热，同时深患着[[忧郁症|忧郁症]]。1847年，他发表和出版了《与友人书信选》，里面主要是与达官、贵妇的书信来往。这部作品里，他歌颂官方教会，歌颂曾经被他谴责过和谴责过他的势力，受到许多人的批评。他公开站到了保守阵营的一边，对自己以往发表的揭露官场腐败和社会黑暗的作品表示公开的忏悔，承认自己对以前所写的全部作品都不满意，他公开声明说，《死魂灵》“充满漏洞，时代错误，作者对许多事物显然一窍不通，有些地方甚至故意使用了侮辱性的冒犯言辞”。对于之前的作家的批评，他表示全部接受，称赞他们的许多意见是“公正的”，并声称：“我生到世上来，绝不是为了要在文学领域占一席之地，而是为了拯救自己的灵魂。”[[别林斯基|别林斯基]]在给他的信中称他是“拿着皮鞭的[[牧师|牧师]]”和“蒙昧主义和最黑暗的压迫的辩护者”。<ref>1847年，[[陀思妥耶夫斯基|陀思妥耶夫斯基]]在一群人中宣读了这封名为《给果戈里的一封信》的信，为此于1849年被判[[死刑|死刑]]，行刑前几分钟被改为[[流放|流放]]。</ref>1848年，果戈里前往[[耶路撒冷|耶路撒冷]]朝圣。回来后，[[神甫|神甫]]马修斯·康斯坦丁诺夫斯基认为他的作品在上帝的眼中是一种罪恶，要求他烧掉《[[死魂灵|死魂灵]]》的第二卷的手稿。

== 去世 ==
[[File:Monumento_di_Nikolay_Gogol_(Villa_Borghese,_Roma,_Italia).jpg|left]]的果戈里纪念雕像]]
1852年2月，他预感自己不久于人世，就向朋友Я.托尔斯泰[[伯爵|伯爵]]（当时果戈里寄居在他在[[莫斯科|莫斯科]]的家中）交待了后事，并让他把手稿拿走，等他死后交给费拉列特[[大主教|大主教]]，但Я.托尔斯泰伯爵并没有拿走他的手稿。2月24日，他烧掉了将近完成的《死魂灵》的第二卷的手稿，然后就病倒了，拒绝进食，经过痛苦的好几天，于1852年3月4日在[[莫斯科|莫斯科]]辞世。<ref>这一年，[[伊万·屠格涅夫|伊万·屠格涅夫]]因不顾当局禁令，写了悼念果戈里的文章，而被逮捕，并被[[流放|流放]]回自己的庄园并被[[软禁|软禁]]了一年。</ref> 人们目前看见的第二卷，是他的出版商——舍维廖夫根据他的遗稿整理出来的。《死魂灵》的第三卷没有写出来。[[File:Gogol_by_Repin.jpg|right]]绘于1909年]]

果戈里被埋葬在[[莫斯科|莫斯科]]的顿斯科依修道院，其墓地竖立着一座青铜十字架，其墓志铭为援引自《[[耶利米书|耶利米书]]》中的一句话：“我将嗤笑我的苦笑。”（{{lang-ru|Горьким словом моим посмеюся.}}引自《耶利米书》20,8）

1931年，顿斯科伊修道院拆迁，当时的[[苏联|苏联]]政府决定将他移葬到诺沃德维奇公墓。移葬时，人们发现果戈里是面朝下葬在棺中，因此出现了传说，传说果戈里是被活埋的。而根据1991年披露的文件《俄国档案》（{{lang|ru|Российском архиве}}）显示，苏联政府打开果戈里的坟墓时，墓中没有发现头颅，<ref>[http://feb-web.ru/feb/rosarc/ra1/ra1-243-.htm ФЭБ: Лидин. Перенесение праха Н. В. Гоголя. — 1994.]</ref>且其头颅至今仍旧下落不明。相传他的头颅曾被收藏家数度转手，后来神秘失踪，并引发了[[都市传说|都市传说]]——[[果戈理幽灵列车|果戈理幽灵列车]]。<ref>[http://gazeta.aif.ru/online/superstar/08/12_01 «Тайны Новодевичьего кладбища», «Аргументы и факты»]</ref><ref>[http://gogol.boom.ru/ogogole/stati/zolot/kto.htm И.Золотусский, «Кто показывает Гоголю нос?», «Российская газета»] {{webarchive|url=https://web.archive.org/web/20080130061652/http://gogol.boom.ru/ogogole/stati/zolot/kto.htm |date=2008-01-30 }}</ref><ref>[http://www.litrossia.ru/2009/41/04586.html Пропавший череп Гоголя, «Литературная Россия»] {{webarchive|url=https://web.archive.org/web/20140223011429/http://www.litrossia.ru/2009/41/04586.html |date=2014-02-23 }}</ref><ref>[http://www.rg.ru/2009/12/31/gogolgrave.html Горьким словом посмеюся, «Российская газета»]</ref><ref>{{Cite web |url=http://revizor.net/o_mogile/mogila_1.php |title=Тайна исчезновения черепа Гоголя (Вариант 1) |access-date=2014-02-15 |archive-url=https://web.archive.org/web/20140222045410/http://revizor.net/o_mogile/mogila_1.php |archive-date=2014-02-22 |dead-url=yes }}</ref><ref>{{Cite web |url=http://revizor.net/o_mogile/mogila_2.php |title=Тайна исчезновения черепа Гоголя (Вариант 2) |access-date=2014-02-15 |archive-url=https://web.archive.org/web/20140222045407/http://revizor.net/o_mogile/mogila_2.php |archive-date=2014-02-22 |dead-url=yes }}</ref>

苏联政府在果戈里的新墓上竖立其雕像，并刻有“伟大的俄罗斯语言艺术家尼古拉·瓦西里耶维奇·果戈里，苏联政府敬立”（{{lang|ru|Великому русскому художнику слова Николаю Васильевичу Гоголю от правительства Советского Союза.}}）的字样。2009年，在纪念果戈里诞辰200年之际，俄罗斯政府在其新墓地重新竖立青铜十字架，并刻上其墓志铭。<ref>[http://news.163.com/10/0108/04/5SFS2MLC000120GR.html 果戈理坟墓终于恢复原貌 大作家头颅仍然下落不明]</ref>

== 影响 ==
果戈里没有选择适应当时的社会，但也没有逃避，他希望改革，并揭露了社会的庸俗与罪恶，他是暴露[[俄罗斯|俄罗斯]]自身面目的第一批作家之一，他在《死魂灵》第一卷中成功地揭露出[[农奴制|农奴制]]时代俄国的黑暗和官场的丑行。但他却在第二卷中提出依靠[[东正教|东正教]]的改革方法，这完全是空想。[[File:Coin_of_Ukraine_Gogol_r.jpg|left]]发行的果戈里的纪念币]]果戈里希望改革俄罗斯的精神状态，可是反而越来越倾向东正教教会，他的矛盾心理和出版《与友人书信选》招致的批评，以及想超越纯文学的徒劳的努力，最后导致了他的健康每况愈下。

果戈里的作品具有华丽生动的散文风格，将社会现实的暴露和讽刺幽默结合，充满了怪异和幻想的因素，因此很能吸引读者。他是俄国[[现实主义|现实主义]]文学的开拓者和“自然派”的奠基人。果戈里将迷信搀杂到现实描写中，用幻想来表白这些事“并不是真实的”，以此来缓解当局的不满，后来有些[[苏联|苏联]]作家也采用了这种手法。他的作品在[[苏联|苏联]]时代受到了[[马克西姆·高尔基|高尔基]]的批判，但深深地影响了[[米哈伊尔·布尔加科夫|米哈伊尔·布尔加科夫]]等[[小说家|小说家]]。果戈里对[[俄罗斯文学|俄罗斯文学]]有着很大的影响，正如[[陀思妥耶夫斯基|陀思妥耶夫斯基]]曾经说的：“我们（指后来一代的俄罗斯作家）都是从他的《外套》中走出来的。”

果戈里同时也是[[尼古拉一世|尼古拉一世]]法定正統信條的支持者。

== 国籍争议 ==
果戈里出生并成长于今天的[[乌克兰|乌克兰]]境内，于是[[乌克兰人|乌克兰人]]认为他是乌克兰人。但他出生之前乌克兰就已经并入[[俄国|俄国]]了，而且持的是俄国国籍，于是[[俄罗斯人|俄罗斯人]]认为他是俄国人。两方为此进行了许多争论。
