'''尼古拉斯·勒布朗'''（{{bd|1742年|12月6日|1806年|1月16日}}），[[法国|法国]][[化学家|化学家]]。

== 早年生平 ==
出生于[[法国|法国]][[谢尔省|谢尔省]]的伊瓦莱普雷。父亲是铁厂的一名下级官员，于1751年去世。随后勒布朗就被送往[[布尔日|布尔日]]，原先的家庭密友Bien医生成为其监护人，并与他居住。在其监护人的影响下，逐渐对医学产生兴趣。1759年Bien医生去世后，勒布朗进入[[巴黎|巴黎]]的École de Chirurgie学习医学。
取得硕士学位毕业后，勒布朗开办了一家诊所。于1775年结婚，四年后生下第一个孩子。由于开办诊所赚取的诊费不足以维持家庭的开支，在1780年勒布朗接受了由路易斯·菲利普二世，奥尔良公爵（Louis Philip II, Duke of Orléans）提供的私人医生的职位。

== 勒布朗制碱法 ==
{{Main|勒布朗制碱法}}
1775年，[[法兰西科学院|法兰西科学院]]设奖悬赏能从廉价的[[食盐|食盐]]生产得到急需的[[纯碱|纯碱]]的方法。1791年勒布朗成功的实现了通过两步从[[食盐|食盐]]合成[[纯碱|纯碱]]的方法并取得了[[法兰西科学院|法兰西科学院]]的悬赏奖励。该方法的第一步是将食盐与浓硫酸在800-900°C下混合，释放出氯化氢[[气体|气体]]，同时得到[[硫酸钠|硫酸钠]]（反应方程式：2 NaCl + H<sub>2</sub>SO<sub>4</sub> → Na<sub>2</sub>SO<sub>4</sub> + 2 HCl）。第二步是将得到的[[硫酸钠|硫酸钠]]碾碎后与[[木炭|木炭]]和[[石灰石|石灰石]]充分混合并焙烧得到纯碱（反应方程式：Na<sub>2</sub>SO<sub>4</sub> + CaCO<sub>3</sub> + 2 C → Na<sub>2</sub>CO<sub>3</sub> + CaS + 2 CO<sub>2</sub>）。
一年后，一家年产量320[[吨|吨]][[纯碱|纯碱]]的工厂(他自己拥有)开始投入运转生产。

== 晚年 ==
两年后，工厂被由法国革命政府没收，并拒绝支付他的已经在十年前赢得了奖金。

1802年，[[拿破仑一世|拿破仑]]退还工厂（而不是奖金）给他，但到那时勒布朗已经无法负担来运行工厂。他于1806年向头部开枪自杀。
