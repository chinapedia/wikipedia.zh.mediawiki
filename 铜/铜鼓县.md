'''铜鼓县'''在[[中国|中国]][[江西省|江西省]]西北部，[[修水|修水]]上游，是[[宜春市|宜春市]]所辖的一个[[县|县]]。客家人口占全县人口的70%，也是江西北部唯一客家县。总面积1547.7平方千米。2015年末全县人口139590人。
==行政区划==
1972年以后，设14乡（人民公社）、1镇、4个乡级林场。共计158村（含居民委员会与林场工区）。1993年1月3日（赣民字[1993]2号批复）大塅、乡、棋坪由乡改镇。1995年6月21日（赣民字[1995]109号批复）三都由乡改镇。1995年12月12日（赣民字[1995]248号批复）撤销二源乡、石桥乡、丰田乡。2001年11月15日（赣民字[2001]513号批复），撤销幽居乡、花山乡、西向乡、古桥乡。  

* 永宁镇。县城。辖：永宁、柳林、西门、南门、东风等五居委会，以及永宁村。合并二源乡、丰田乡之后，面积126.4平方公里，人口2.7万。辖江头、上源、兴源、坪田、钓鱼、八亩、西湖、丰田、小水、顺化10个村委会。2004年辖：城南社区、定江社区、城北社区、花园社区、顺化村、小水村、丰田村、西湖村、钓鱼村、坪田村、上源村、兴源村、江头村、八亩村。
* 温泉乡：位于县境西南部，距县城中心2.5公里。原辖14村，以及温泉农场。1995年与石桥乡合并设温泉镇之后，面积139平方公里，人口1.7万。辖温泉、义田、象湾、黄毗、禄田、石扬、新塘、凤山、凤舞、岭下、濯溪、新开、金锡、石桥、中溪、光明、柳树、金星、青塅、三星、赤碷21个村委会。现辖：温泉集镇居委会、上庄村、石桥村、光明村、三星村、金星村、温泉村、凤山村、禄田村、黄毗村、义田村、新塘村、新开村、温泉直属农场。
* 棋坪乡：位于铜鼓县西北，是西河片的政治、经济、文化中心。合并幽居乡后，面积238平方公里，山林面积33万余亩，人口1.36万,下辖棋坪集镇居委会、棋坪村、炉湾村、柏树村、双溪村、九峰村、观田村、罗坊村、黄泥村、游源村、（枫）丰坦村、幽居村、大梅村等12个村民委员会,134个村民小组。 
* 排埠镇：县西南边陲，距县城18km，  全镇面积131.5km2，总人口11063人，其中农业人口9348人。森林总面积145566亩，水田面积11479亩，活力木蓄积量125万m3之多，年可高产木材3000m3，毛竹蓄积量500万根。辖：排埠集镇居委会、梅洞村、华联村、排埠村、永庆村、高陂村、永丰村、曾溪村、青溪村、黄溪村、南溪村、三溪村。
* 三都镇：县东部，总面积250平方公里，辖：三都集镇居委会、东浒村、东山村、小山村、三都村、黄田村、战坑村、枫槎村、大槽村、浆里村、理溪村、坪段村、西六村、巷口村、新安村、交古村、明星村、望湖村、新建林场、仰天湖林场。总人口1.5万人，镇政府所在地距县城10公里。林地28万亩，森林蓄积量180万立方米，毛竹立积量260万根，高岭土储量达700万吨
* 大塅镇：县城东部30公里，面积205.69平方公里，辖大塅集镇居委会、大塅村、东风村、凤竹村、谭坊村、浒村、浒口村、鹤吼村、交山村、古桥村、隘口村、公益村、庙下村、小芬村、红苏村、红新村、太平村、末中村、光相村。，总人口24000余人。是铜鼓县东河片政治、经济、文化中心 。森林面积18469公顷，活立木蓄积量达884392立方米，毛竹蓄积量260万根。耕地面积18275亩
* 高桥乡：西南部山区，距县城32公里，面积180平方公里，辖胆坑村、花山村、高桥村、濠洲村、梁段（塅）村、白石村、英溪村。芳坪、月形、汪新、小源、𥖨下、太平等村撤并。总人口为6240人。林地面积23万多亩。
* 港口乡：县域西北部，总面积为124平方公里，山林面积16。48万亩，辖港口集镇居委会、港口村、水坪村、英朝村、华仙村、赤洲村。正港、小𥖨、𥮷坪村撤并。
* 带溪乡 铜鼓东部，面积68平方公里，人口7500人，辖大群村、新丰村、高岭村、红群村、西村、东源村、柳溪村7个行政村。森林面积52695亩，活立木蓄积量达311000立方米，毛竹蓄积量51万根，耕地面积13369亩。1993年11月17日，带溪乡政府驻地迁至大塘村（赣民字[1993]168号批复）。
* 二源乡：1995年12月12日，撤销二源乡，并入永宁镇

* 幽居乡：位于县域最西北部。辖幽居、大梅、祖庄、毛湾、上沙、樟坑、新屋村。2001年11月15日，撤销幽居乡，划归棋坪镇。

== 历史沿革 ==

商周屬艾國，故後襲稱「艾」。

隋[[開皇|開皇]]九年，屬[[建昌縣|建昌縣]]。

[[武周|武周]][[長安_(年號)|長安]]九年，建昌縣析置[[武寧縣|武寧縣]]，此地屬武寧縣。

[[唐睿宗|唐睿宗]][[景雲|景雲]]元年，改武寧縣為[[豫寧縣|豫寧縣]]。寶應元年豫寧縣復為武寧縣。

[[唐德宗|唐德宗]][[貞元_(唐朝)|貞元]]十六年二月，析武寧西八鄉置分寧縣，隸屬[[洪州_(隋朝)|洪州]]。

北宋，分寧縣屬[[江南西路|江南西路]]洪州[[镇南軍|镇南軍]]，後改為[[寧縣|寧縣]]。開寶八年，寧縣復為分寧縣。

南宋[[建炎|建炎]]四年，升分寧縣為[[義寧軍|義寧軍]]，在今銅鼓縣境內設立定江寨，置[[游擊_(官名)|游擊]]。

元至元二十三年，於武寧置[[寧州|寧州]]，隸[[龍興路|龍興路]]，下轄武寧、分寧縣。大德八年，升分寧縣為寧州，仍隸龍興路。

元末，朱元璋改龍興路為[[洪都府|洪都府]]；1363年又改洪都府為[[南昌府|南昌府]]。

明[[洪武|洪武]]三年十二月，寧州降為[[寧縣|寧縣]]，仍隸南昌府。

洪武八年置定江巡檢司。

[[弘治|弘治]]十六年，寧縣又升為寧州，屬南昌府。

[[萬曆|萬曆]]四年，李大鑾舉兵抗明，被鄱陽守備[[鄧子龍|鄧子龍]]鎮壓。為此，萬曆五年，在[[瀏陽縣|瀏陽縣]]、[[平江縣|平江縣]]、新昌縣（今[[宜豐縣|宜豐縣]]）、[[萬載縣|萬載縣]]、寧州交界的山區擇地，於寧州武鄉二十四都（今銅鼓縣城）設銅鼓石守備司，守備鄧子龍鎮守，興建城垣，以境內有「銅鼓石」而得名。

清[[雍正|雍正]]元年，[[瑞州府|瑞州府]]同知移駐銅鼓，設置瑞州分防府。雍正六年設「銅鼓營把總𪠘」。

[[乾隆|乾隆]]三十二年設銅鼓營都司衙署。

[[嘉慶|嘉慶]]三年七月鎮壓[[白蓮教|白蓮教]]劉聯登、宋懷璞、魏文宗有功，於嘉慶六年六月改寧州為[[義寧州|義寧州]]，以示嘉許。

[[光緒|光緒]]二十九年，裁撤綠營都司，提議「改營設縣」未果。

光緒三十二年，同盟會「萍醴瀏大起義」波及此地。

光緒三十四年，清廷援[[全南縣|虔南]]先例，改銅鼓為[[撫民廳|撫民廳]]。

宣統二年正月初一（1910年2月10日），廢銅鼓營，設[[銅鼓廳|銅鼓撫民廳]]；勘界時義寧州武鄉金雞橋之南的上半鄉（五十三、五十四、五十五都）、崇鄉高坳、港口之南的上半鄉（二十一都至二十八都）劃歸銅廳鼓。當時反對分治的本地人士稱「銅鼓設縣，地少人少，繳納錢糧，負苛奇重，民無以堪」，直至1915年中秋才平復。

1912年全國廢除府、州、廳，铜鼓废厅建县，直隶江西省。<ref>[http://tggl.cn/E_ReadNews.asp?NewsID=215 铜鼓历史沿革]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

民国三年全省划分四道，属[[浔阳道|浔阳道]]。民国十五年，废道，铜鼓直隶省政府。

自大革命以来，铜鼓县有2万多人参加革命牺牲。其中绝大部分，因为档案无考，或因无子嗣，未能留下姓名。1985年第三次全县烈士普查，在册烈士2828人。

== 地理 ==
位于江西省西北部。

境内以山地为主。县域西部的大围（沩）山，为[[罗霄山脉|罗霄山脉]]一部。最高峰[[七星岭|七星岭]]，海拔1608米，位于西南部[[排埠镇|排埠镇]]与湖南省交界处的湖南境内；羊场尖，海拔1541米。其东支脉新开岭，横贯县域腹地，为定江河与金沙河分水岭。县域西北是幕府山南支余脉山枣岭，是铜鼓与平江县分界。县域东南的九岭山，铜鼓与万载县界的称太阳岭；铜鼓与宜丰县界称黄蘖山。

全县森林覆盖率高达87.4%，居江西省之首。<ref>{{Cite web |url=http://www.tonggu.gov.cn/jianjie.asp |title=存档副本 |access-date=2009-09-21 |archive-url=https://web.archive.org/web/20090602030847/http://www.tonggu.gov.cn/jianjie.asp |archive-date=2009-06-02 |dead-url=yes }}</ref>

气候宜人，适合南方各种作物的生长，年均气温为13.1-15.1℃，且雨量充沛，年降雨量为1764.5-1802.8mm。 冷害天气，十年九遇。冷冻与积雪成灾，最大积雪深度24厘米，造成公路中断。

主要河流有金沙河、定江河，均为[[修水|修水]]上游支流。金沙河，又名西水、东津水，发源于紫草坪，在港口乡涂家坝进入修水县。长50.6km，流域面积695.32km<sup>2</sup>，多年平均径流深907mm，平均径流总量59862万m<sup>3</sup>. 定江河，又名东河、武宁乡水，发源于血树坳，在古桥乡金鸡桥流入修水县。长70.9km，流域面积888.37km<sup>2</sup>，多年平均径流深999mm，平均径流总量68154万m<sup>3</sup>

最大湖泊（水库）为[[大塅水库|大塅水库]]。

== 交通 ==
*[[大广高速公路|大广高速公路]]在县东部带西乡过境，为南北走向。另外一条至省会的叫[[昌铜高速|昌铜高速]]，东西走向横穿县境， 向西出省与湖南浏阳至长沙的高速连接。铜鼓至南昌高速170公里左右，铜鼓至长沙高速130余公里，向北至武汉约370公里。
 
* 铜棋公路：为西水流域五乡镇的老区要道。从县城经温泉、花山、胆坑至棋坪，长43.1km。1966年11月至1968年8月，由省林垦拨款170万元支援老区建设，彻底改变了西水老区人民肩挑步行出山。为省养公路。1975年、1976年又建成县养公路棋坪-幽居公路14.14km；棋坪-港口公路19.5km。
* 省道319线/上(高)浏(阳)公路。上高—铜鼓铁树坳湖南边界，长109.28公里。县域长度36.1km。宜春公路分局宜丰养路段负责。
* 省道240线：宜丰芳溪—万载，长39.31公里。另外宜丰芳溪至铜鼓与省道319共线。
* 省道241线：修水—铜鼓，长100.95公里，县域长36.7km，于1962年8月14日竣工。
* 省道314线/[[奉新|奉新]]—铜鼓带溪，长100.62公里。县域长13.5km。1969-1970年建成。1973年6月24日被洪水冲毁。1977-1980年修复。
 
1956-1960年炸礁治理航道，使得定江河流放竹木排方便安全。1973年同样治理金沙河。
== 经济 ==
2015年，全县国民经济保持平稳快速增长，社会各项事业和谐发展，产业结构日趋优化。全年完成地方生产总值（GDP）360544万元，比上年增长9.0%。全县完成农林牧渔及其服务业总产值（现价）104214万元，按可比价计算比上年增长3.94%。居民收入进一步提高。2015年全县在岗职工年平均工资（在岗加劳派）45466元，比上年增加5072元，增长12.55%；城镇居民可支配收入：20293元，比上年增加1590元，增长：8.5;农民人均可支配收入7725元，比上年增加 901元，增长13.2%。

财政收入：2015年全县财政总收入83400万元，比上年增长16%。其中地方财政收入67696万元，增长15.1%。全县财政总支出133875 万元，比上年增长10%。

金融机构支持地方经济发展的力度进一步加大，2015年金融机构各项存款余额达到338624万元，比年初增加41549万元，增长13.98%；其中储蓄存款余额为236358万元，比年初增加25246万元，同比增长11.95 %。各项贷款余额达到165882万元，比年初增加21906万元，增长15.2%。

保险事业：全年人寿保险各项保费收入3214万元，比上年增长11.01%。全年支付各种赔款220万元，比去年下降10.9%。财产保险保费收入2598万元，比上年增长2.04%。 赔款2055万元，比去年增长65%。

== 人口 ==
民国五年，全县人口159225人。民国20年，全县人口128230人。民国22年，全县人口剧降至53792人。民国30年，全县人口仅剩48050人。民国37年，全县人口51135人。

1953年第一次人口普查，全县人口55678人。2000年，根据第五次人口普查数据：全县总人口131616人；其中： 永宁镇 35332 温泉镇 18646 棋坪镇 10003 排埠镇 11165 三都镇 12062 大段镇 11013 花山乡 2319 高桥乡 5251 幽居乡 2859 港口乡 5036 西向乡 3150 带溪乡 8221 古桥乡 6559（人）

2015年，人口总量保持低水平增长。年末全县户籍总人口139590人，其中：男性72294人，女性67296人，出生人口：1748人，出生率12.71‰，死亡812人，死亡率5.90‰，自然增长率6.81‰。

习惯称赣语人群为土籍，康熙雍正年间入境的为“怀远”人（即客籍）。全县人口70%为[[客家|客家]]人<ref>{{Cite book| author =颜森| title =江西方言的分区| location =北京| publisher =《方言》| date =1986-02-24| pages =19-38| ISBN =| accessdate =2014-11-25| url =http://lib.cnki.net/cjfd/FYZA198601002.html| language =zh| quote =铜鼓县百分之六十以上说客家话...客家话还见于铜鼓县毗连的一些地方}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>，是江西除赣南之外唯一一个客家县。通行[[客家话|客家话]]（[[方言岛|方言岛]]，四周为[[赣语|赣语]]），当地称为“[[怀远声|怀远声]]”；今改用[[铜鼓县革命委员会|县革委]][[官话|官腔]]，当地称为“[[半胡鸭|半胡鸭]]”。本地话又分铜鼓声、武乡声、崇乡声。

==教育文化==
文革时期，由于大批知识分子下基层，各人民公社相继创办初中或完全中学，农村部分小学创设“戴帽”初中班。1976年，全县有初中7所，完全中学10所，在校中学生4785人，其中高中1050人。

1979年，撤销各公社高中班，并入红旗、棋坪、大塅三所片上完中。1981年，撤销棋坪、大塅高中部，并入县中和红旗中学，各初高中取消农业基础知识课。1983年，港口、幽居、高桥的初中并入棋坪初中；二源公社中学改为县二中。1984年，红旗中学高中部改为县职业中学。1985年撤销各林场中学，集中办学为县林业中学。至此，全县设县中、县二中、长林子校、长红子校、职业中学等高中。

2015年，科技研究开发及产业化取得较大发展。年末有各类专业技术人1920人，全年争取上级科研项目8个，申请专利127项，当年授权65项。

教育事业稳步发展，教学质量显著提高。年末全县有中小学校9所，其中，普通高中1所，教师进修学校 1所。在校学生24680人，其中普通高中1863人，职业高中1468人，初中4634人，小学11613人，适龄儿童入学率100 %，巩固率100 %，小学升初中率100%，全县高考二本上线168人，三本上线54人，专科上线243人。
