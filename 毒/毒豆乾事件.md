== 背景 ==
早在1987年12月，[[桃園市|桃園縣]][[大溪區|大溪]]豆類製品曾接連爆發[[防腐劑|防腐劑]]、[[漂白劑|漂白劑]]、[[黃麴毒素|黃麴毒素]]及[[二甲基黃|二甲基黃]]等，相關產品下架銷毀，政府修改法規，禁用二甲基黃，後經多方人員研究改以[[食用色素|食用色素]]（4號、5號）替代。<ref>大溪豆干染毒，《[[民生報|民生報]]》，1988-01-03{{zh-tw}}</ref><ref>豆干染毒，大溪觀光生意下跌，《[[大成報|大成報]]》，1988-01-05{{zh-tw}}</ref>

2013年6月7日，臺北市查獲百年老店[[尤協豐烘豆干店|尤協豐烘豆干店]]添加[[油漆|油漆]]染料[[皂黃|皂黃]]{{efn|name=皂黃 |1= 檢調指出，36號食用染料的學名為「皂黃」（原文為{{lang|en|Metanil yellow}}<ref>[http://www.ey.gov.tw/News_Content4.aspx?n=E7E343F6009EC241&sms=E452EBB48FCCFD71&s=748CEF11AC38B8DA  臺北市衛生局主動移送將工業色素賣給食品業的店家 嚇阻不法業者危害食品安全] {{Wayback|url=http://www.ey.gov.tw/News_Content4.aspx?n=E7E343F6009EC241&sms=E452EBB48FCCFD71&s=748CEF11AC38B8DA |date=20150924005856 }}，[[臺北市政府衛生局|臺北市政府衛生局]]，檢驗科，李盈霖，台北，2013-06-07{{zh-hant}}</ref>），工業用色素，主要是供[[紙|紙]]、[[肥皂|肥皂]]、[[油漆|油漆]]、[[皮革|皮革]]染色用途，不得添加於食品，若長期食用，可能引起[[肝臟|肝臟]]細胞的損害。皂黃每公斤只賣300元，與[[食用色素|食用染料]]4號、5號染料每公斤售價分別為600、700元有落差，{{R|udn-7949371}}常被不肖業者使用於豆干及[[鹹魚|鹹魚]]<ref>[http://www.tainan.gov.tw/tainan/dep_news.asp?id={CCF17DB4-3809-4C87-9051-D896F130FDF8}  發放「過氧化氫」、「皂黃」及「二氧化硫」簡易檢驗試劑給市民，共同為飲食安全把關！]，[[臺南市政府|臺南市政府]]，檢驗科，李盈霖，台北，2014-03-13{{zh-hant}}[http://www.tainan.gov.tw/tainan/dep_news.asp%3Fid%3D%257BCCF17DB4-3809-4C87-9051-D896F130FDF8%257D]</ref>。}}，製作為「[[祭祀|拜拜]]用的大黃豆干」。[[西門町|西門町]]的[[益良食品|益良食品]]，涉嫌以[[工業色素|工業色素]]皂黃混充[[食用色素|食用色素]]販賣，除了供貨給尤協豐，也賣給麵條業者。檢調表示，益良食品在2006年、2010年間分別有出貨紀錄，2012年7、8月也大量出貨。<ref>[http://www.epochtimes.com/b5/13/6/7/n3889246.htm  台黑心豆干 驚爆使用油漆染料]，[[大纪元时报|大纪元时报]]，記者徐翠玲，台北，2013-06-07{{zh-hant}}</ref><ref name ="udn-7949371">[http://udn.com/NEWS/NATIONAL/NATS1/7949371.shtml 油漆染料製黑心豆干 已賣了7年]，[[聯合晚報|聯合晚報]]╱記者王聖藜、黃玉芳，臺北，2013-06-07{{zh-tw}}</ref>

林口長庚醫院臨床毒物科主任[[林杰樑|林-{杰}-樑]]表示，皂黃是工業染劑，過量暴露會傷[[肝|肝]]、長期暴露會導致[[肝癌|肝癌]]風險增加。其外表跟合法[[食用色素|人工色素]]黃色4號、5號的黃豆干差異並不會太大，但有刺鼻的異味。<ref>[http://news.ltn.com.tw/news/life/paper/686270  皂黃、己二烯酸鉀小檔案]，[[自由時報|自由時報]]，記者洪素卿，2013-06-07{{zh-tw}}</ref><ref name ="udn-7949371"/>
