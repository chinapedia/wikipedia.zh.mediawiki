'''毒鴨蛋事件'''是2005年位於[[台灣|台灣]][[彰化縣|彰化縣]][[線西鄉|線西鄉]]所爆發的[[鴨蛋|鴨蛋]]受[[污染|污染]]事件，並在該年環境新聞中高居第一名。

== 事件起因 ==
2005年6月，[[台湾|台湾]][[衛生署|衛生署]]檢驗「年度食品中戴奧辛背景值調查計畫」時，發現[[彰化縣|彰化縣]][[線西鄉|線西鄉]]所養殖的[[鴨蛋|鴨蛋]][[戴奧辛|戴奧辛]]含量過高（每[[公克|公克]]鴨蛋的戴奧辛含量32.6[[皮克|皮克]]，遠高於[[歐盟|歐盟]]規範值3皮克）<ref name="周琦淳2013-p108">{{harvnb|周琦淳|2013|pp=108-109}}</ref>，也間接爆發出[[台湾環保署|台湾環保署]]企圖壓下此事件。<ref>{{Cite web |url= http://e-info.org.tw/node/1180 | author=|language=zh-tw ||title= 2005回顧：彰化戴奧辛毒鴨蛋事件|publisher= 台灣環境資訊協會 }}</ref>
== 事件過程 ==
2005年6月，[[台灣|台灣]][[衛生署|衛生署]]發現[[彰化縣|彰化縣]][[線西鄉|線西鄉]]所產鴨蛋含[[戴奧辛|戴奧辛]]，並通報[[台灣環保署|台灣環保署]]處理，牵扯出[[台灣環保署|台灣環保署]]早已在2月就已經知情，當時[[台灣環保署|台灣環保署]]僅提出報告給予[[行政院農業委員會|行政院農業委員會]]，卻沒有第一時間下令銷毀鴨蛋，也讓媒體懷疑部份鴨蛋已流入市面上，甚至已讓消費者吃下肚了。

然而一波未息一波又起，不到半年，同年9月，位於[[伸港鄉|伸港鄉]]的養殖場也爆發戴奧辛污染，伸港鄉緊鄰線西鄉，因此環保署認為受害的養殖場可能不止於此，再度爆發毒鴨蛋之後，當時在位的[[台灣環保署|台灣環保署]]署長[[張國龍|張國龍]]先生自請處份。

爆發後同一個月裡，[[台灣|台灣]][[環保署|環保署]]針對事發鴨場環境評估，然而許多學者將元兇指向位於受害鴨場附近的[[台灣鋼聯|台灣鋼聯]]，但當時環保局推翻此看法，並表示「[[戴奧辛|戴奧辛]]具生物累積性，生物體內戴奧辛含量近九成是經由食物進入體內，由呼吸進入的量小於百分之十；且[[台灣鋼聯|台灣鋼聯]]排放的戴奧辛指紋特徵與遭汙染的鴨肉、鴨蛋、鴨農並不相同，無法認定[[台灣鋼聯|台灣鋼聯]]是唯一汙染源」。

鴨農不滿[[立法委员|立委]]在還未有正式報告出來時，就已經將事情爆發出來，鴨農所盼的報告在同年月底才出爐並推翻環保署之前的看法，因為此報告一出來，並將元兇指向台灣鋼聯，報告指出事發九座鴨場所遭受到的污染，跟台灣鋼聯所排放的戴奧辛紋道一樣，因此鴨農將所有賠償推到台灣鋼聯，但台灣鋼聯提出一份報告指出環保署所調查的報告，與台灣鋼聯所聘專家調查落差甚大，環保署也表示，台灣鋼聯也僅是元兇之一。停工數月的台灣鋼聯，在隔年2006年3月準備復工，整件事在2006年台灣鋼聯復工後，相關調查皆已落幕。

== 事件總計銷毀數量 ==
據2006年6月台湾農委會表示，[[戴奧辛|戴奧辛]]鴨子汙染事件，總計銷毀[[鴨蛋|鴨蛋]]16萬5905斤、加工蛋17萬9147枚、[[種蛋|種蛋]]10萬3750枚，鴨子3萬6340隻。

== 相關賠償 ==
在鴨農多次抗爭下，[[農委會|農委會]]以每隻5元輔助鴨農復養，將發放多時的每隻鴨12元輔助停止，然而台灣鋼聯的賠償始終未定。

== 事件疏失 ==
*[[台湾環保署|台湾環保署]]在第一時間並沒有召集農委會與衛生署啟動危機處理小組，甚至在當時企圖將此事壓下，事後調查花費一億的情況下，所給的報告仍被不少立委砲轟。
*鴨農在個人疏失中，有不少鴨農意圖暗藏疑似遭受污染的鴨蛋，雖然事後鴨農供稱是未受污染的存貨，並不滿政府不收購鴨蛋，但事後仍將所藏的鴨蛋銷毀。
*台灣鋼聯雖然只是環保署所稱元兇之一，但當時也呈現一個問題，那就是塵埃沒有適當處理，導致土質受汙染。
