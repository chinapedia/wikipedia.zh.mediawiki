《'''毒梟'''》（{{lang-ko|'''마약왕'''}}; {{lang-en|'''The Drug King'''}}），[[韓國|韓國]][[電影|電影]]，描繪一個在[[1970年代|1970年代]]家境貧寒的[[韓國|韓國]][[冶金|金匠]]成為「毒梟」的故事。全片於2017年5月5日開拍，於2018年12月19日上映。<ref>{{Cite web |url=http://www.allthatstar.com/tw/view.php?action=5209&postID=334071&rtn_sns=bom |title={title} |access-date=2017-05-11 |archive-url=https://web.archive.org/web/20180509220455/http://www.allthatstar.com/tw/view.php?

== 劇情 ==
1970年代的[[日本|日本]]強烈掃毒，所有毒品工廠幾乎被抄光，毒-{}-梟們將眼光轉向韓國，[[釜山|釜山]]一位從事冶金的金匠李頭森，本來只是協助[[黑道|黑道]]，來往日本與韓國之間，走私[[金條|金條]]、[[金幣|金幣]]、金錶，孰料被黑道出賣，李頭森陷入獄中，夜夜拷打。透過愛妻四處行賄權貴的幫助，李頭森以嚴重[[肺癆|肺癆]]為由，[[保外就醫|保外就醫]]，這次的事件讓他非常憤怒，於是他改名換姓，努力學習製毒、販毒知識，立志成為「毒梟」。他向[[臺灣|臺灣]]、日本訂製[[麻黃素|麻黃素]]，透過他巧妙的提煉後成為[[冰毒|冰毒]]，外銷到日本，並賦予一個顯赫的商標「MADE IN KOREA」，他也搖身一變，晚上製毒、販毒，而白天變成一個堅決[[反共|反共]]的愛國商人兼慈善家.....

== 主要人物 ==
*; [[宋康昊|宋康昊]] 飾演 李頭森
: [[韓國史|韓國史]]上最大規模的製毒、販毒的人物，經歷警方數次掃蕩卻總能安然脫身。
*;[[裴斗娜|裴斗娜]] 飾演 金霆芽 
: [[社交名媛|社交名媛]]，同時是一位兼具知性與魅力的說客，與李頭森交往後，將李頭森帶入真正的上流社會。
*; [[曹政奭|曹政奭]] 飾演 金仁久
: 首都檢察官，為了與釜山地方檢察廳聯手合作杜絕販毒，從首爾來到釜山。
*; [[李聖旻|李聖旻]] 飾演 徐尚勳
: 釜山地方檢察廳毒品管制科科長，因為李頭森的行賄，對其毒品生意暗中照顧。
*; [[金大明|金大明]] 飾演 李頭煥 
: 李頭森的堂弟，一個意志力薄弱而屢出差錯的人。
*; [[李熙俊|李熙俊]] 飾演 崔振弼
: 走私犯，幫助李頭森將毒品銷售到日本。
*; [[趙宇鎮|趙宇鎮]] 飾演 趙成剛
: 黑社會幫派大哥，幫助李頭森擴大了毒品的銷售市場，包括韓國國內乃至日本。

== 軼事 ==
導演表示，李頭森雖然是虛構人物，但是是利用實際存在過的大毒-{}-梟作為原型，經過幾年的資料調查才完成了劇本。
