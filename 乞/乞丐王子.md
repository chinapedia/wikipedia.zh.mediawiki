《'''乞丐王子'''》（{{lang-en|''The Prince and the Pauper''}}），或譯《'''王子與乞丐'''》<ref>譯名取自{{cite web |url=http://www.joyo.com/book/detail.asp?prodid=bkbk601788&ref=writer% |title=王子与乞丐(新): 综合: 外国名著必读: 文学天地: 少儿: 图书: 卓越网 |accessdate=2007-03-22 |language=zh-hans}}</ref>、《'''王子與貧兒'''》<ref>譯名取自{{cite web |url=http://haotushu.com/book/47748/ |title=王子与贫儿/世界少年文学精选 - 好图书 |accessdate=2007-03-22 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20070313224338/http://haotushu.com/book/47748/ |archivedate=2007-03-13 }}</ref>，是[[美國|美國]][[作家|作家]][[馬克·吐溫|馬克·吐溫]]1881年在[[加拿大|加拿大]]發表（1882年發表於美國）的第一本歷史幻想代表作。這本書的故事背景在1547年，主角是兩名長相一模一樣的小男孩：居住在[[倫敦|倫敦]]的叫做垃圾院（Offal Court）的地方，經常受父親虐待的貧民{{link-en|湯姆·康第|Tom Canty}}；以及[[亨利八世|亨利八世]]的兒子[[愛德華六世|愛德華王子]]。

※日本譯者{{link-en|巌谷小波|巌谷小波}}在1899年（明治32年）時在[[文武堂|文武堂]]譯名為《'''乞丐王子'''》，1927年（昭和2年）日本譯者[[村岡花子|村岡花子]]以《'''王子與乞丐'''》為譯名於[[平凡社|平凡社]]連載。

== 故事情節 ==

因為不了解彼此的生活環境，兩人都很羨慕對方的生活，於是相約交換身分。之後，湯姆被帶回宮廷，愛德華則是努力逃離湯姆的父親的魔掌。愛德華在沒落貴族邁爾斯·亨登 （Miles Anthony）的協助之下得以重返王宮，恰好在此時，亨利國王去世，愛德華是規定的繼任國王。

歷經多次冒險，包括牢獄生活之後，愛德華在湯姆即將受冕之時要求與湯姆換回正確的身分。湯姆迫不及待地要放棄王位，但是貴族們不相信形如乞丐的愛德華才是真正的國王——直到愛德華拿出他在離開王宮之前藏起來的玉璽，才向眾人證明了自己是繼任國王。之後，Anthony獲晉升為[[伯爵|伯爵]]，並得到從此以後家族族長在國王面前可以坐下的殊榮。而湯姆則獲得國王為了感激而賜予的「國王護衛」（The King's Ward）的特許終身頭銜。

== 文學上的重要性與批評 ==
本書大部分的幽默在於兩名男孩在對方環境下的——儘管湯姆已經盡可能地運用他的智慧——無能為力。就許多方面而言，這本書是諷刺社會的作品，對英國[[都鐸王朝|都鐸]]時期的階級不平等多有譴責。從這個角度看來，馬克·吐溫沒有使用著名的[[美國中西部|美國中西部]]寫作風格，而用了仿[[查爾斯·狄更斯|查爾斯·狄更斯]]的風格。

雖然這本書不比馬克·吐溫的其他著作熱門，本書為他後來的歷史幻想創作，例如《[[康州美國佬大鬧亞瑟王朝|亞瑟王廷之康乃迪克佬]]》<ref>譯名取自{{cite web |url=http://www.lib.ncku.edu.tw/multimedia/media_online/naxos/slwebmain02(2)main.htm |title=國立成功大學NAXOS書目清單(小說類_青少年文學) |accessdate=2007-03-22 |language=zh-hant |deadurl=yes |archiveurl=https://web.archive.org/web/20080925093159/http://www.lib.ncku.edu.tw/multimedia/media_online/naxos/slwebmain02(2)main.htm |archivedate=2008-09-25 }}</ref>，開出一條道路。

== 改編作品 ==
本書後來改編成舞台劇的一個插曲，使得馬克·吐溫與該名劇作家大打官司。後來又有許多電影以本書為藍圖：例如1937年發行，由名演員[[艾洛·佛林|艾洛·佛林]]飾演Hendon、Billy與Bobby Mauch飾演湯姆與愛德華的同名電影<ref>{{cite web |url=http://www.amazon.ca/exec/obidos/ASIN/B00009M9AG/701-4773094-7430702 |title=Amazon.com: The Prince and the Pauper: DVD |accessdate=2007-03-22 |language=en}}</ref>，本片原先計畫在[[喬治六世|喬治六世]]加冕時推出，卻延誤到隔年才發行。

1976年時[[英國廣播公司|英國廣播公司]]將本故事改編為六部30分鐘的電視劇，由{{link-en|尼可拉斯·林德赫斯特|Nicholas Lyndhurst}}一人分飾兩角。1996年再次改編。

[[華特迪士尼公司|華特迪士尼公司]]亦有將本書改編成24分鐘的短片電影，1990年發行，由[[米老鼠|米老鼠]]主演，與《[[救難小英雄澳洲歷險記|救難小英雄澳洲歷險記]]》一同上映。2004年本書被改編為85分鐘的電腦動畫[[音樂劇|音樂劇]]《{{link-en|真假公主芭比|Barbie as the Princess and the Pauper}}》<ref>{{cite web |url=http://www.kidsdown.com/youxi/yqita/gbaBarbiePauper.htm |title=芭比娃娃之公主与乞丐，很好玩的一个培养和增强与同伴配合和协作能力的游戏 奇奇玩儿童软件 kids |accessdate=2007-03-22 |language=zh-hans}}</ref>，由[[芭比娃娃|芭比娃娃]]飾演金髮公主Anneliese與褐髮乞丐Erika，Martin Short替反派角色Preminger配音。本片直接以[[DVD|DVD]]與[[VHS|VHS]]的形式發售。2006年，[[加菲貓|加菲貓]]的第二部真人拍攝電影，《[[加菲貓2之雙貓記|加菲貓2之雙貓記]]》<ref>{{cite web |url=http://ent.sina.com.cn/m/f/kitties/index.html |title=《加菲猫2之双猫记》_影音娱乐_新浪网 |accessdate=2007-03-22 |language=zh-hans}}</ref>，是這個經典故事的另一次改編。加菲貓2也有[[網路漫畫|網路漫畫]]版本。

在一些改編版本中，愛德華王子在變裝成湯姆時有預備措施；至於後來有沒有派上用場，各個版本皆有不同。

乞丐王子1937年版電影版權由[[透納娛樂公司|透納電影公司]]（Turner Entertainment Inc.）取得。
