{{vgname|押忍！戰鬥！應援團|ja=押忍！闘え！応援団}}，是由[[iNiS|iNiS]]製作，[[任天堂|任天堂]]發行在[[任天堂DS|任天堂DS]]平台上的[[音樂遊戲|音樂遊戲]]軟體。遊戲以充滿強勁節奏的樂曲，搭配上攪笑的漫畫風格，充分利用[[任天堂DS|任天堂DS]]的觸控螢幕給玩家帶來樂趣。

續編《[[燃燒！熱血韻律魂_押忍！戰鬥！應援團2|燃燒！熱血韻律魂 押忍！戰鬥！應援團2]]》在2007年5月17日發行。

== 遊戲內容 ==

== 難度 ==

# 輕柔支援（{{lang|ja|気軽に応援}}） - 以新入團成員'''{{lang|ja|田中 一}}'''為主角，配合的節奏為最簡單的。
# 果斷支援（{{lang|ja|果敢に応援}}） - 以隊長'''{{lang|ja|一本木 龍太}}'''為主角，中等速度的節奏。
# 激烈支援（{{lang|ja|激烈に応援}}） - 以團長'''{{lang|ja|百目鬼 魁}}'''為主角，更快更密集的節奏，對節奏時間控制的要求遠比上述兩者更為嚴謹。
# 華麗支援（{{lang|ja|華麗に応援}}） - 以女子啦啦隊為主角，難度最高的等級。節奏點消失的速度比'''激烈支援'''更快、更小，而且出現的方向也不同(通常是激烈支援的倒轉版本)。
