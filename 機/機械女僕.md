《'''機械女僕'''》（{{lang-ja|こはるびより}}），標題直譯為「'''小春日和'''」，是[[日本漫畫家|日本漫畫家]][[みづきたけひと|みづきたけひと]]所創作的[[日本漫畫|日本漫畫]]作品，於《[[電擊萌王|電擊萌王]]》連載的漫畫作品。衍生改編作品有[[廣播劇|廣播劇]]及三卷[[OVA|OVA]]。

於2001年在MediaWorks的[[漫畫雜誌|漫畫雜誌]]《[[月刊Comic電擊大王|月刊Comic電擊大王]]》以[[讀切|讀切]]方式刊載，之後不定期在該雜誌刊載（收錄於單行本第1卷）。在《[[電擊萌王|電擊萌王]]》於2002年3月創刊時在該雜誌創刊號正式展開連載，也在《[[電擊帝王|電擊帝王]]》展開並行連載直到該雜誌於2006年11月休刊（收錄於單行本第3卷）。隨著OVA的製作，在《月刊Comic電擊大王》2007年12月號至2008年2月號連載3期。[[單行本|單行本]]全7卷。

== 劇情概要 ==
村瀨貴也是个对收集[[手办|手办]]和[[模型|模型]]有特别嗜好的男孩，他对[[机器人|机器人]]专卖店「MaidWorks」（簡稱“MW”）里的一只最新款的[[女仆|女仆]]机器人一见如故，把她买回家中，起名为“由衣”。村瀨貴也本来想借这款跟真人一样逼真的机器人来满足自己“给女孩换衣服”之类的爱好，然而事情并非那么顺利：拥有自主意识的由衣不但对他的要求十分抗拒，还经常以自己的觀点来改变村瀨貴也“颓废”的生活现状。在不断打闹和和好的过程中，两人的关系越走越近，而一场人与机器人的轻喜剧就此上演。

: 由衣的主人，喜歡手办和模型，擅長料理與裁縫，疼愛由衣，喜歡換由衣的衣服及捉弄由衣。
; {{nihongo|住友美野里|住友 みのり（すみとも みのり）||配音：[[明坂聰美|明坂聰美]]}}
: [[女僕咖啡店|女僕咖啡店]]「Café Cowbeya」的短髮[[巨乳|巨乳]][[看板娘|看板娘]]，暗戀貴也。兩次扮成女僕機器人進入貴也家，都沒被貴也識破。
; {{nihongo|住友哲志|住友 哲志（すみとも てつし）||配音：[[三宅健太|三宅健太]]}}
: 美野里的父親，Café Cowbeya店長。
; {{nihongo|MW店長|MW店長||配音：[[三宅華也|三宅華也]]}}
: MaidWorks店長兼女僕機器人製作者，妙齡[[眼鏡娘|眼鏡娘]]，堅持製作「獨特、唯一的外觀與人格」的機器人。協助美野里兩次扮成女僕機器人進入貴也家。笑聲類似[[白鳥麗子|白鳥麗子]]。
; {{nihongo|萩原步美|萩原 歩美（はぎわら あゆみ）||配音：[[井口裕香|井口裕香]]}}
: [[包包頭|包包頭]]髮型的[[蘿莉|蘿莉]]，喜歡貴也。
; {{nihongo|萩原香苗|萩原 香苗（はぎわら かなえ）||配音：[[柚木涼香|柚木涼香]]}}
: 步美的母親，丈夫已逝世。丈夫經營[[錢湯|錢湯]]，但她沒有參與經營。
; {{nihongo|御堂蘭|御堂 蘭（みどう らん）||配音：[[矢作紗友里|矢作紗友里]]}}
: 以成為一流的除靈師為目標的[[貧乳|貧乳]][[巫女|巫女]]，嫉妒美野里的巨乳，會召喚「那個世界的朋友」（巨大章魚「章魚啾」或巨大烏賊「烏賊啾」）玩弄美野里。
; {{nihongo|御堂菫|御堂 菫（みどう すみれ）||配音：[[早見沙織|早見沙織]]}}
: 御堂蘭的姊姊，也是貧乳巫女。喜歡給御堂蘭打分數。
; {{nihongo|久遠|久遠（くおん）||配音：[[小清水亞美|小清水亞美]]}}
: MaidWorks製造的女僕機器人，機能優秀，但長期賣不出去而深感不滿。賣不出去的原因是MW店長不肯賣。
; {{nihongo|咲夜|さくや||配音：[[清水愛|清水愛]]}}
: 世界知名的機器人製造商「MP」製造的女機器人。原本是貴也一手打造的娃娃，經MP擴大改造及製成等身大機器人。
; {{nihongo|鳳智之|鳳 智之（おおとり ともゆき）}}
: 有錢人家的小孩，想要一個像貴也一樣的哥哥。
; {{nihongo|千鶴|千鶴（ちづる）}}
: 鳳家的女僕機器人兼女僕長。
; {{nihongo|紘|紘（ひろ）}}
: 智之的朋友，對機械非常了解的男孩，堅持自己做女僕機器人。
; {{nihongo|桃華|桃華（ももか）}}
: 紘自製的女僕機器人。
; {{nihongo|二宮遙佳|二宮 遥佳（にのみや はるか）}}
: 從作者的讀切漫畫{{nihongo|《交給我吧！My sister》|おまかせ！マイシスター}}來[[客串|客串]]的女性漫畫家，[[筆名|筆名]]「{{nihongo|黑石舞依|黒石 舞依（くろいし まい）}}」，麻花辮髮型的眼鏡娘，是貴也的高中同學。不喜歡出現在人群面前，理由是她認為這會讓她覺得很丟臉。
; {{nihongo|二宮彩佳|二宮 彩佳（にのみや あやか）}}
: 遙佳的[[雙胞胎|雙胞胎]]妹妹，[[馬尾|馬尾]]髮型。喜歡遙佳的漫畫，但完全不會畫圖。代替遙佳出席遙佳的[[cosplay|cosplay]]簽名會，頗受讀者歡迎。
; {{nihongo|國分巧也|国分 巧也（こくぶ たくや）}}
: 遙佳的[[責任編輯|責任編輯]]，喜歡[[萌系漫畫|萌系漫畫]]。曾請求遙佳畫萌系漫畫，被遙佳拒絕。

== 動畫OVA ==
OVA全3卷。
* 第1卷在2007年11月21日發行。同年10月14日在[[AT-X|AT-X]]先行播放。
* 第2卷在2008年2月8日發行。同年1月13日先行播放。
* 第3卷在2008年4月2日發行。同年3月9日先行播放。
