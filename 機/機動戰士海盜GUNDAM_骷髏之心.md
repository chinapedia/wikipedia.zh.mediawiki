《'''機動戰士海盜GUNDAM 骷髏之心'''》（{{lang-ja|機動戦士クロスボーン・ガンダム スカルハート}}、{{lang-en | Mobile Suit Crossbone Gundam: Skull Heart }}）是[[富野由悠季|富野由悠季]]及[[矢立肇|矢立肇]]原作、[[長谷川裕一|長谷川裕一]]作畫的日本[[漫畫|漫畫]]作品，於[[GUNDAM_ACE|高達ACE]] 2002年11月號至2004年10月號及[[月刊少年ACE|月刊少年ACE]]2003年10月號連載。連載時以《機動戰士海盜GUNDAM外傳》（{{lang-ja|機動戦士クロスボーン・ガンダム外伝}}）為名，在單行本時則改稱為《機動戰士海盜GUNDAM 骷髏之心》（{{lang-ja|機動戦士クロスボーン・ガンダム スカルハート}}）日文單行本由[[角川書店|角川書店]]發行，未有中文代理。本作是《[[機動戰士海盜鋼彈|機動戰士海盜鋼彈]]》的外傳故事。

== 故事簡介 ==
UC0133年，新死亡先鋒揭發並介入木星帝國的陰謀，本作描寫老伯的過去、杜比安在戰爭中的遭遇、新死亡先鋒在戰事完結後的生活及地球聯邦軍的哈里遜在戰後的任務等故事。

== 登場人物 ==
* 使用的台灣譯名參考自《機動戰士海盜GUNDAM》台灣角川版，香港譯名參考自《機動戰士海盜GUNDAM》文化傳信版。配音員為[[SDGUNDAMG世代|SDGUNDAMG世代]]出演的聲優。

; {{nihongo|杜比安·亞羅納克斯|トビア・アロナクス|聲：[[山口勝平|山口勝平]]}}
: 主角，15歲，新人類，海盜高達X1改．改的駕駛員。喜歡貓。木星帝國戰爭結束後，與碧拉狄及其他同伴繼續當宇宙海盜，消滅木星軍餘黨。

; {{nihongo|碧拉狄·布里耶／狄妮絲·度加茲|ベルナデット・ブリエット / テテニス・ドゥガチ|聲：[[夏樹莉緒|夏樹莉緒]]}}
: 木星總統古洛斯．度加茲的女兒。木星帝國戰爭結束後，與杜比安繼續當宇宙海盜。與杜比安關係良好，但未有更進一步的發展。

; {{nihongo|哈里遜·馬汀|ハリソン・マディン|聲：[[青羽剛|青羽剛]]（[[超級機器人大戰|超級機器人大戰]]） / [[平川大輔|平川大輔]]（[[SDGUNDAMG世代|SDGUNDAMG世代]]）}}
: 聯邦的軍人，官階上尉，外號「聯邦的藍色閃光」。故事中與杜比安多次交手，但未有服從上級命令逮捕海盜。更在執行調查任務時獲得杜比安的襄助。似乎喜歡年紀較小的女孩{{refTag|name=RefC|1=富野由悠季、矢立肇(2005)，《機動戦士クロスボーン・ガンダム スカルハート》，角川書店}}。

; {{nihongo|金格度·那烏|キンケドゥ・ナウ|聲：[[辻谷耕史|辻谷耕史]]}}
: 新死亡先鋒的皇牌，是海盜高達X1的駕駛員，故事「星之王女」中為尋找失蹤的杜比安，駕駛海盜高達X1來到資源衞星，擊退木星軍。

; {{nihongo|多爾古·史黛拉·娜貝娜多|トゥインク・ステラ・ラベラドゥ|聲：[[釘宮理惠|釘宮理惠]] }}
:在故事「星之王女」中作為主角。獨自一人在資源行星上生活近十年的女孩{{refTag|name=RefC}}，父母雙逝，偶然救了杜比安，誤以為將杜比安交給木星軍，木星軍就會離開資源行星。戰後加入宇宙海盜，在故事「猿之衞星」中乘搭哈里遜的機體執行任務。

; {{nihongo|古雷·史多克|グレイ・ストーク|聲：[[矢尾一樹|矢尾一樹]] }}
:在故事「最終兵士」中登場的角色，外表是一名中年男子，駕駛大型機體根保，拜託杜比安協助阻止木星軍的計劃。在任務過程中，多次向杜比安提及傳說中的高達駕駛員[[阿姆羅·雷|阿姆羅·雷]]的事蹟。一般推測真正身份為[[傑特·亞希達|傑特·亞希達]]，並與在《機動戰士V GUNDAM 外傳》中「木星爺爺」疑為同一人{{noteTag|1=未經官方證實。}}。

== 登場機體 ==
; {{nihongo|海盜高達X1 |クロスボーン・ガンダムX1}}
: 駕駛員為金格度，機體編號XM-X1，S.N.R.I.內部型號為F97，是新死亡先鋒的王牌MS。背部裝有大型的十字型可動推進器，配合接近戰用武裝，機體更曾作調整，是專為近身戰而設的機體{{refTag| 1=富野由悠季、矢立肇(1996)，《機動戰士海盜高達1》，文化傳信}}。

; {{nihongo|海盜高達X1骷髏之心（X1改．改）|クロスボーン・ガンダムX1スカルハート（X1改・改）}}
: 原駕駛員為金格度，與木星帝國的戰爭完結後，交給杜比安使用的機體{{refTag| 1=富野由悠季、矢立肇(1998)，《機動戰士海盜高達6》，文化傳信}}。

; {{nihongo|F91哈里遜．馬汀專用機|F91ハリソン・マディン専用機}}
: 藍色的F91，是哈里遜的專用機。機體曾作大幅度修改，以方便量產，因此性能較原版F91弱，亦不能引發MEPE現象{{noteTag|1=在《機動戰士海盜高達 鋼鐵的7人 3》中表明機體在拆除裝備及改造後方擁有接近原型的戰鬥力。}}。

; {{nihongo|根保|ガンプ}}
: 古雷．史多克駕駛的MS，與當時的機體相比屬大型機體。左臂裝有巨型鉗，右手持有雙筒激光槍。在與アマクサ的對戰中失去右腳。外型與ZZ GUNDAM相像，亦與在《機動戰士V GUNDAM 外傳》中「木星爺爺」駕駛的機體疑為相同機體{{noteTag|1=未經官方證實。}}。

; {{nihongo|アラナ・バタラ|ARANA BATARA}}
: 機體型號為EMS-06-P，是木星軍為開發アラナ的試作機體。

; {{nihongo|アラナ|ARANA }}
: 機體型號為EMS-12，全高約10米，是木星軍後期計劃量產的機體，亦是巴比茲的原型機{{refTag|name=RefC}}。在大戰後期，由於木星帝國資源不足，開始以高機動性為目標，開發超小型MS，背後的武裝可進行全方位發射。

; {{nihongo|天草|アマクサ}}
: 木星帝國的新型MS，以海盜GUNDAM X2為基礎，注入木星帝國的技術製成的機體，又稱為「{{nihongo|木星GUNDAM |ジュピターガンダム}}」{{refTag|name=RefC}}。機體全高約18米，與X2改同樣因為木星帝國技術不足，無法完全重現GUNDAM的設計，因為機體有不少木星軍自行設計的裝備，亦因此無法將機體小型化。在故事中，由搭載了[[阿姆羅·雷|-{zh-hans:阿姆罗·雷;zh-hk:阿寶·尼爾;zh-tw:阿姆羅·雷;}-]]戰鬥數據的生體電腦控制，具有壓倒性的戰鬥力，單機將三台海盜量產機擊毀，並消滅了基地的木星軍MS，最後在杜比安與史多克合力下擊毀。GGENERATION SPIRITS的機體介紹中提及此機體名稱源自於[[天草四郎|天草四郎時貞]]。

; {{nihongo|B GUNDAM（鐵球特裝）|B GUNDAM}}
: 老伯在一年戰爭中為鐵球加上GUNDAM頭部外型的裝甲而成，亦因此令機體平衡性極差，但卻因禍得福，成功打倒六台德姆。
