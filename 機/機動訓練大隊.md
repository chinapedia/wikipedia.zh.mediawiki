'''機動訓練大隊'''（簡稱'''機訓大隊'''），是[[中華人民共和國|中華人民共和國]][[深圳市公安局|深圳市公安局]]以[[勞務合約|勞務合約]]的性質僱用的[[防暴|防暴]][[單位|單位]]，由深圳市公安局指揮，主要責任為[[人群管制|人群管制]]、[[防暴|防暴]]及協助[[中華人民共和國人民警察|人民警察]]及[[中華人民共和國武裝警察|武裝警察]]提供[[武裝力量|武裝力量]][[人力資源|人力資源]]。

== 組織 ==
機動訓練大隊由[[深圳市公安局|深圳市公安局]]擔任指揮，所領導之人員均非[[公務員|公務]]以至警察[[身份|身份]]，不過於執勤時穿著[[中華人民共和國人民警察|人民警察]]制服，其防暴套裝亦標示「警察」。

== 歷史 ==
2010年代，機動訓練大隊比較多因為[[徵地|徵地]]而引起的示威活動而出動，亦被派遣過駐守[[口岸|口岸]]，以阻嚇水貨攜帶者。

== 訓練 ==
人員需要接受[[人群管制|人群管制]]及[[防暴|防暴]]等<ref>[https://hk.news.yahoo.com/%E6%A9%9F%E8%A8%93%E5%A4%A7%E9%9A%8A%E5%8F%97%E9%81%8E%E9%98%B2%E6%9A%B4%E8%A8%93%E7%B7%B4-215514002.html 機訓大隊受過防暴訓練] 《星島日報》 2014年11月20日</ref>。
