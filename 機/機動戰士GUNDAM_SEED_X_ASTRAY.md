'''機動戰士GUNDAM SEED X ASTRAY'''是連載於月刊 Gundam Ace 的漫畫作品。本作是機動戰士GUNDAM SEED ASTRAY 的續篇，故事發生於動畫 [[機動戰士GUNDAM_SEED|機動戰士GUNDAM SEED]] 第47話與第48話之間空白的2個月。主要講述本篇故事新登場的普雷亞·雷菲力與卡納得·帕爾斯的對決，而前作的羅·裘爾及叢雲劾亦以配角登場。
*'''機動戰士GUNDAM SEED X ASTRAY'''（漫畫）[[時田洸一|鴇田洸一]]

:於[[GUNDAM_ACE|GUNDAM ACE]]連載。單行本全2冊。

== 登場人物 ==
=== 民間人士 ===
* 普雷亞·雷腓力（CV：[[小島幸子|小島幸子]]）
: 為了解決地球上的能源問題，接受馬爾其歐導師的指示，前往宇宙奪取[[YMF-X000A_Dreadnought|勇士鋼彈]]（Dreadnought Gundam）。
: 真正身份是地球聯合軍為了得到「過人的空間識別能力」而複製出來的複製人，原型不詳（已知複製對象是過去聯合軍「梅比烏斯零式」部隊的一員）。
: 由於當時複製技術並非成熟，身體擁有缺陷的端粒，所以身體特別虛弱；與[[劳·鲁·克鲁泽|拉烏·魯·克魯澤]]的情況相近。
: 駕駛X Astray擊敗卡納得以後，衝往已被損壞的核反應爐的亥伯龍高達，救下想自盡的卡納得。但最後因本身的缺陷而死亡。

* 馬爾其歐
: 委託普雷亞往宇宙取得反中子干擾器，以解決地球能源問題。
: 致力維護和平的盲人導師，受到自然人和調整者雙方敬重。

=== 廢物回收商 ===
* 羅·裘爾（CV：[[小野坂昌也|小野坂昌也]]）
: 經過普雷亞·雷腓力所乘搭的宇宙船，其後協助普雷亞·雷腓力取回勇士鋼彈（Dreadnought Gundam）頭部的反中子干擾器。
: 根據安德魯·沃特菲德所給予的龍騎兵設計圖，往「天之御柱」製作其系統。
: 為搭載龍騎兵系統後的勇士鋼彈，以其外形及駕駛員偏離既有道路的理念稱之為「X異端鋼彈」。

* 教授（CV：[[進藤尚美|進藤尚美]]）

=== 蛇尾 ===
* 叢雲劾（CV：[[井上和彥|井上和彥]]）
:傭兵隊「蛇尾」隊長，調整者，曾被聯合軍用作獲取調整者數據，之後為擺脫藥物控制逃走，並開展傭兵生涯。重視同伴，雖然實力非凡，更被稱為「最強的調整者」，其超卓駕駛技術是來自實戰經驗。因此他亦會按照戰局需要，改造專用MS藍色Astray。
: 駕駛異端鋼彈藍色機，搶走勇士鋼彈（Dreadnought Gundam）頭部的傭兵。
: 因受西蓋爾·克萊茵的委託作為「監視者」，洞悉勇士鋼彈內的反中子干擾器事態的變化，並決定將其處置的方式。
: 搶走勇士鋼彈（Dreadnought Gundam）頭部，以便不讓地球聯合軍奪走反中子干擾器有關資料。
: 對羅·裘爾接受勇士鋼彈的龍騎兵系統設計圖並沒有異議。



* 風花·亞哲
: 被叢雲 劾委託觀察回收屋等人的動向，作為蛇尾與回收屋之間的溝通橋樑。
: 本身任務結束後，自願繼續停留在回收屋協助普雷亞·雷腓力等人。

* 伊萊傑·基爾（CV：[[鳥海勝美|鳥海勝美]]）
* 羅蕾塔·亞哲
* 李德·威勒

=== 三艦同盟 ===
* [[安德魯·沃特菲德|安德魯·沃特菲德]]（CV：[[置鮎龍太郎|置鮎龍太郎]]）
: 本有「沙漠之虎」之稱。於正傳故事與煌決戰後重傷，被羅·裘爾救出險境，療養後加入[[三舰同盟|三舰同盟]]軍。
: 提供記載勇士鋼彈（Dreadnought Gundam）龍騎兵系統設計圖的磁碟給羅·裘爾。

* 馬秦·達哥斯塔
: 原屬ZAFT的安德魯·沃特菲德隊的副官。亦跟隨安德魯·沃特菲德加入三盟同艦軍。

=== 歐普 ===
* 隆德·蜜納·薩哈克（CV：[[勝生真沙子|勝生真沙子]]）
: 歐普軌道電梯「天之御柱」的支配者。對殺害弟弟的羅帶有輕蔑與仇恨。
: 讓羅使用「天之御柱」內的工廠設施，以製作Dreadnought Gundam龍騎兵系統。

* 索基烏斯（CV：[[齋賀觀月|齋賀觀月]]）
: 被地球連合軍所改造而成的戰鬥用調整者。

=== ZAFT ===
* 米哈爾·寇斯特（CV：[[私市淳|私市淳]]）
: MS駕駛員，亦為醫生。喜歡把戰場比喻為「患處」、敵人比喻為「病」，所以綽號被人稱作「醫生」。
: 原打算奪回被普雷亞·雷腓力等人帶走的中子干擾器，但因特務部隊「X」的介入，其精銳部隊被全滅。

=== 歐亞聯邦軍 ===
* 卡納德·帕爾斯（CV：[[保志總一朗|保志總一朗]]）
: 特務部隊「X」的特務兵。為[[CAT1-X1/3_Hyperion_Gundam_Unit_1|亥伯龍鋼彈1號機]]的駕駛員。
: 不斷追尋[[基拉·大和|煌·大和]]，想要擊倒他成為「完全體」，並在找到他以前絕不能被打敗。
: 真正身份是「超級調整者」——[[基拉·大和|煌·大和]]之前的失敗作。在遭受破棄前被響博士（煌的生父）的一個助手帶走，被地球連合捕獲，進行了大量的實驗。

* 加路西亞
: 本為小行星要塞「阿爾提密斯」司令，但正傳中在[[GAT-X207_Blitz|電擊鋼彈]]突襲下，要塞失守。事後仍被任命指揮特殊部隊「X」。
: 視下屬如棋子，沒有利用價值便予以放棄。
: 因歐亞聯邦與大西洋聯邦重新交好，被長官認為自己及特務部隊「X」經已沒有價值，自己及特務部隊「X」也分別被貶和解散。
: 認為卡納德·帕爾斯怠慢處理反中子干擾器一事而造成如此境況，決定捉拿卡納德·帕爾斯，並將其處罰。

* 梅莉奧·皮司提司
: 特殊部隊「X」的女士官，是部隊內最了解和關心卡納德的人。
: 協助並跟隨卡納德·帕爾斯逃離特務部隊「X」。

* 巴爾薩姆·阿連得
: 亥伯龍鋼彈2號機的駕駛員。綽號為「阿爾提密斯的荒鷲」。
: 原打算追捕逃離部隊的卡納德·帕爾斯等人，但被卡納德·帕爾斯駕駛已被損壞的亥伯龍鋼彈1號機殺死。

* 摩根‧雪佛蘭
: 原為戰車部隊的指揮官，人稱「月下的狂犬」，後來轉配至MS駕駛員。
: 因擁有有線誘導炮的操作能力，所操作的MS則使用原屬[[穆·拉·弗拉卡|穆·拉·弗拉卡]]的有線誘導炮背包。
