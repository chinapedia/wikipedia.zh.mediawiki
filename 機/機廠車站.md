'''機廠車站'''位於[[臺灣|臺灣]][[高雄市|高雄市]][[鳳山區|鳳山區]]<ref>道路另一側為[[苓雅區|苓雅區]]</ref>，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[高雄臨港線|第一臨港線的東臨港線]]的簡易[[鐵路車站|車站]]。此站供[[高雄機廠|高雄機廠]]員工上下班[[通勤|通勤]]使用，可從西側門進入高雄機廠廠區。<ref>鋼軌上的輪機手—[http://danilechang.pixnet.net/blog/post/24016783-%E9%AB%98%E9%9B%84%E6%A9%9F%E5%BB%A0%E7%9A%84%E9%80%9A%E5%8B%A4%E7%81%AB%E8%BB%8A 高雄機廠的通勤火車]</ref>

== 車站構造 ==
[[File:Railway_Workshop_Station_(Remnant).JPG|200px]]
* [[側式月台|側式月台]]一座。

== 目前狀況 ==
* 原設置於東臨港線的凱旋路圍牆邊月台配合[[2014年臺灣高雄氣爆事故|2014年丙烯氣爆事故]]重建已拆除。
* 新建於高雄機廠通勤員工入口旁月台於2015年中旬完工，員工專列於2015年10月15日配合潮州基地啟用大改點中復駛。
* 員工專列約平日早上7點半及下午4點半停靠，沿途停靠[[高雄車站|高雄車站]]、[[新左營車站|新左營車站]]（左營機務分段），假日停駛。
*隨著第一臨港線最後一段(前鎮＝高雄)於2018年9月28日開出最後一班7214次通勤專列後，隨著第一臨港線停用、廢止

== 歷史 ==
* 1976年：原位於縱貫線鼓山─高雄港間的機廠前車站跟隨機廠一併遷至高雄第一臨港線現址。
* 2014年7月31日深夜：發生[[2014年臺灣高雄氣爆事故|丙烯氣爆事故]]，之後在凱旋三路重建過程中拆除。之後員工改用[[統聯客運|統聯客運專車]]通勤。
* 2015年中旬：新建月台完工。
* 2015年10月15日：員工專列復駛。
* 2018年9月28日：發出最後一班員工專列後停用。
