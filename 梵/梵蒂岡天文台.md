'''梵蒂岡天文台'''（''Specola Vaticana''）是[[梵蒂岡|梵蒂岡]]進行天文研究和教育的機構，總部設在[[義大利|義大利]]首都[[羅馬|羅馬]]近郊的[[冈多菲堡|冈多菲堡]]，也是[[教宗|教宗]]在夏天的居所，所屬的 '''梵蒂岡天文台研究小組'''則附設在[[亞利桑那大學|亞利桑那大學]]的[[史都華天文台|史都華天文台]]內，現任的主任是{{Link-en|蓋伊·康索爾馬尼奧|Guy Consolmagno}}（2015年上任<ref>{{Cite web |url=http://zh.radiovaticana.va/news/2015/09/18/%E6%95%99%E5%AE%97%E6%8E%A5%E8%A7%81%E6%A2%B5%E8%92%82%E5%86%88%E5%A4%A9%E6%96%87%E5%8F%B0%E5%AD%A6%E8%80%85%EF%BC%9A%E6%95%99%E4%BC%9A%E9%BC%93%E5%8A%B1%E5%92%8C%E6%8E%A8%E5%8A%A8%E7%9C%9F%E6%AD%A3%E7%9A%84%E7%A7%91%E5%AD%A6/1172911 |title=教宗接见梵蒂冈天文台学者：教会鼓励和推动真正的科学 |date=2015-09-18}}</ref>）。

這個天文台在美國[[亞利桑那州|亞利桑那州]]靠近[[沙佛|沙佛]]的[[葛拉漢山|葛拉漢山]]有一台1.8米的望遠鏡，鏡片是[[史都華天文台鏡片研究所|史都華天文台鏡片研究所]]研製的第一個鏡片，與位在當地的研究所一起被稱為[[梵蒂岡先進技術望遠鏡|梵蒂岡先進技術望遠鏡]]（VAIT）。

== 歷史 ==
正式的梵蒂岡天文台設在義大利的''Osservatorio Pontificio''，[[安吉洛·西奇|安吉洛·西奇]]神父將這個天文台遷建至[[羅馬|羅馬]]的St. Ignatius教堂的頂樓。

在羅馬的時期，梵蒂岡天文台提供了時間的服務，但是在西奇神父的主持下，天文台對19世紀後期的天文學有著巨大的貢獻，特別是西奇神父特別熱衷於其中的[[太陽|太陽]]。

在西奇神父於1878年逝世後，它的繼承者被逐出了天文台，名稱也被改為''Regio Osservatorio al Collegio Roman'' （現為[[宗座额我略大学|宗座額我略大學]]），或者說是在羅馬大學的皇家天文台。這個天文台持續運作至1923年，在1930年代又重新啟用<ref>{{cite web |url=http://clavius.as.arizona.edu/vo/R1024/History_p2.html |title=存档副本 |accessdate=2007-10-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20071008074352/http://clavius.as.arizona.edu/vo/R1024/History_p2.html |archivedate=2007-10-08 }}</ref>。
