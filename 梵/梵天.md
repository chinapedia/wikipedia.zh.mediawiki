'''梵天'''（[[梵文|梵文]]：'''ब्रह्मा'''，'''Brahmā'''），原為古印度的祈禱神，現[[印度教|印度教]]的創造之神，與[[毗濕奴|毗濕奴]]、[[濕婆|濕婆]]並稱三主神。他的坐騎為[[孔雀|孔雀]]（或[[天鵝|天鵝]]），配偶為智慧女神[[辯才天女|辯才天女]]，故梵天也常被認為是智慧之神。全印度幾萬座印度教[[寺廟|寺廟]]中供奉梵天的寺廟卻極為少見，現存最有名的專門供奉梵天的地方位在[[拉賈斯坦邦|拉賈斯坦邦]]城鎮[[普斯赫卡尔|普斯赫卡尔]]，其他各地[[印度教|印度教]]徒大多供奉[[濕婆|濕婆]]和[[毗濕奴|毗濕奴]]。

佛教也將梵天吸納為護法神之一，在[[南传佛教|南传佛教]]的[[東南亞|東南亞]]，尤其[[泰國|泰國]]，得到很大的發揮，[[華人|華人]]稱之為[[四面佛|四面佛]]或[[四面神|四面神]]，據說有保佑人間富貴吉祥的功能，在東南亞有非常多信眾。

== 名稱 ==
在[[梵文|梵文]]語法裡，{{lang|sa|ब्रह्मा}}（''{{IAST|brahmā}}''）是泛稱中性詞''[[婆羅門|brahman]]''的主格單數型。

梵天在[[馬來語|馬來語]]和[[泰語|泰語]]裡分別稱為''Berahma''和''[[四面佛|Phra Phrom]]''。

== 屬性 ==
在宇宙創造之時，梵天創造了十一位[[生主|生主]]，據信是人類的祖先。《[[摩奴法典|摩奴法典]]》說他們分別是Marici、Atri、Angiras、Pulastya、Pulaha、Kratuj、Vashishta、Pracetas、Daksha、Bhrgu以及Narada{{Citation needed|time=2008-01}}。他也創造了七位聖哲（Saptarishi），共同協助宇宙之創造。但由於他所創造的生主和聖哲都是從他的[[精神|精神]]而非肉體生出來的，因此他們被稱為「心靈之子」。

在《[[吠陀經|吠陀經]]》和《[[往世書|往世書]]》裡，梵天並不常常介入三惡道，宣揚真理和祝福諸眾勝過一切法門。另外梵天又被視為[[法_(佛教)|法]]的創造者。古印度相信梵文字母是由梵天创造的<ref>《大唐西域記》卷第二（三國）:詳其文字，梵天所製，原始垂則，四十七言（47个字母）也。寓物合成，隨事轉用。流演枝派，其源浸廣，因地隨人，微有改變，語其大較，未異本源。</ref>。

== 誕生 ==
根據《[[往世書|往世書]]》的說法，梵天是自我誕生的，並沒有[[母親|母親]]。一說在宇宙肇始之際，[[毗濕奴|毗濕奴]][[肚臍|肚臍]]上的[[蓮花|蓮花]]產生了梵天。這也說明了梵天的名字又叫做''Nabhija''（意味「從肚臍生出來的」）。另一個傳說則說梵天是從漂流在水上的一顆金卵中誕生的，因此梵天又叫做''Hiranyagarbha''（金卵之意）。最後，金卵的剩餘部份則擴張成宇宙。由於他是在水中誕生的，因此他又被稱為''Kanja''（意味「在水中誕生的」）。梵天是至高存在「[[梵|梵]]」和陰性能量[[本性_(佛教)|自性]]（Prakrti）或[[摩耶_(印度哲学)|幻象]]（Maya）的兒子。
[[File:Shesh_shaiya_Vishnu.jpg|thumb]]坐臥在巨蛇[[舍沙|舍沙]]的身上，而從毗濕奴肚臍生長出來的蓮花則誕生了梵天。]]

== 外貌傳說由來 ==
[[File:Brahma_1820.jpg|thumb]]
梵天的傳統形象乃四顆頭、四張臉以及四隻手臂，口中不斷地誦讀《[[吠陀經|吠陀經]]》。特別在北印度，他常常有著一綹白鬍鬚，代表他幾近永恆的存在。不像其他印度教神祇，梵天的手中通常沒有任何[[武器|武器]]；但他的一隻手握有一支湯匙型的權[[杖|杖]]，代表著將神聖的[[酥油|酥油]]滴入自我犧牲的火柴堆。他的一隻手拿著一個[[水壺|水壺]]（有時是裝著水的[[椰子|椰子]]殼），壺中的水代表萬物初始，涵蓋一切。梵天也拿著一串[[念珠|念珠]]，是他計算宇宙時間流逝的[[計時器|計時器]]。此外他也拿著象徵神聖的《吠陀經》或與代表純淨的[[蓮花|蓮花]]。

有一個古[[印度神話|印度神話]]故事講到梵天四個頭顱的由來。梵天創造了一個女神，名喚'''[[辯才天女|莎塔如帕]]'''（辯才天女，Shatarupa）后，卻開始迷戀這位擁有無雙美貌的女神，因莎塔如帕觉得不妥，于是经常转到梵天的左面、右面、后面甚至飞到上面，以避開梵天的目光。为了方便凝視莎塔如帕，原本一张脸的梵天遂长出五张脸来，擁有五個首級。後來梵天與濕婆因故爭執，在盛怒之下，濕婆用額頭上的第三隻眼放射出來的火焰，燒向了梵天朝上方看的那一面，並用左姆指的指甲割下了那顆首級，於是梵天只剩下了現在所常見的四張臉。<ref>[http://www.52tangka.net/html/201203311025.html 中、印、泰三国对四面佛的理解]</ref>

另一個讲到梵天四个头颅的神话来历。梵天原来有五个头，事因於毗濕奴問梵天誰是創造宇宙的至高無上的創造者。梵天答：「他即創造宇宙的至高無上創造者，因為他是創造宇宙的至高無上創造者，所以毗濕奴也應該崇拜他」。

梵天的話令濕婆大怒，因為濕婆認為祂才是創造宇宙的至高無上的創造者，憤怒的濕婆化身為Bhairava（[[陪臚|陪臚]]-意為恐怖的殺戮者），在盛怒之下，濕婆用額頭上的第三隻眼放射出來的火焰，燒向了梵天朝上方看的那一面，並用左姆指的指甲割下了那顆首級，於是梵天只剩下了現在所常見的四張臉。雖然發生了梵天跟濕婆不和的事，但最終梵天跟濕婆還是和好如初。

== 形象象徵 ==
*'''四隻手臂'''：象徵東、西、南、北四個方向，也象徵心靈（右後方手臂）、智慧（左後方手臂）、自我（右前方手臂）、自信（左前方手臂）。
*'''吠陀經'''：語言的一切。
*'''黃金'''：象徵活躍在宇宙中，而金臉則代表梵天在創造宇宙時是處於活躍狀態的。
*'''孔雀'''（或'''天鵝'''）：象徵辨別力，是梵天的座騎。
*'''王冠'''：象徵最高地位。
*'''蓮花'''：皇座。
*'''鬍鬚'''：長有黑色或白色的鬚象徵智慧的永恆成長。包括了鬢、髯、髭。　
*'''四張臉'''：四大皆空。

== 坐騎 ==
梵天的坐騎是一隻[[孔雀|孔雀]]（一說[[天鵝|天鵝]]）。在[[印度教|印度教]]裡，凡是孔雀坐騎的天神皆是三相神，代表著天生能分辨善惡，繼而揚善棄惡。

== 印度的幾處供奉地點 ==
[[File:Templo_a_Brahmā_en_Pushkar,_Rajasthan.jpg|thumb]][[普斯赫卡尔|普斯赫卡尔]]的[[梵天寺廟_(普須卡)|梵天寺廟]]]]
[[File:Thai_4_Buddies.jpg|thumb]]）]]
雖然梵天在印度教裡是三位主神的其中一位，但少有印度教徒實際崇拜他。今天的印度有好幾萬座寺廟供奉三位主神之二的[[毗濕奴|毗濕奴]]和[[濕婆|濕婆]]，但供奉梵天的寺廟卻並不多見。

現存最有名的梵天寺廟位在[[拉賈斯坦邦|拉賈斯坦邦]]城鎮[[普斯赫卡尔|普斯赫卡尔]]。到了一年一度的{{link-en|月圓之夜|Kartik Poornima}}（10－11月間），人們便會開始為梵天而歡慶一番。上千名朝拜者會蜂擁至[[梵天寺廟_(普須卡)|梵天寺廟]]旁的聖湖{{link-en|普須卡湖|Pushkar lake}}進行沐浴。

至於普須卡以外的的梵天寺廟，則以逆時針的方式按印度地理位置分述如下：

1.北印度地區：[[喜馬偕爾邦|喜馬偕爾邦]][[庫魯縣|庫魯縣]]的縣治[[庫魯|庫魯]]。

2.西北印地區：[[拉賈斯坦邦|拉賈斯坦邦]][[巴爾梅爾縣|巴爾梅爾縣]][[巴洛特拉|巴洛特拉]][[鄉_(南亞)|鄉]]的{{link-en|阿所特拉|Asotra}}、[[古吉拉特邦|古吉拉特邦]]的城鎮克德布拉哈瑪也有梵天的塑像、古吉拉特邦[[薩巴坎塔縣|薩巴坎塔縣]]的縣治[[凯德布拉赫马|凯德布拉赫马]]。
          
3.西印度地區：[[孟買|孟買]]附近的[[纳拉索帕拉|纳拉索帕拉]]地區也發現了高約1.8公尺的梵天塑像。[[果阿邦|果阿邦]]{{link-en|索托利|Sattari}}鄉的[[布拉哈瑪·卡爾瑪利|布拉哈瑪·卡爾瑪利]]（Brahma-Karmali） 
           
4.西南印地區：[[喀拉拉邦|喀拉拉邦]]的{{link-en|希魯拿伐亞|Thirunavaya}}。在{{link-en|九夜節|Navratri}}期間，人們前往希魯拿伐亞的梵天寺廟祭祀與慶祝。

5.東南印地區：[[坦米爾納德邦|坦米爾納德邦]][[坦加布爾縣|坦加布爾縣]]的[[库姆巴科纳姆|库姆巴科纳姆]]、坦米爾納德邦大城[[蒂魯吉拉帕利|蒂魯吉拉帕利]]附近的希魯帕圖爾（Thirupattur）。這間廟也是聖哲夫亞克拉帕薩（Vyakrapatha）的[[禪定|禪定]]所在。[[安得拉邦|安得拉邦]]的[[斯里卡拉哈斯蒂|斯里卡拉哈斯蒂]]也有一座供奉梵天的寺廟。

== 神話和文學 ==
印度神話中{{citation needed|沒有提及梵天受到詛咒是因為無上法身被人濫用而在地球上引來災難。

# {{link-en|宇宙 (印度教)|Hindu cosmology|宇宙}}肇始之際，毗濕奴和梵天巧遇濕婆巨大的[[林伽|林伽]]（[[陽具|陽具]]），於是雙方分道揚鑣，前往尋找林伽的起點與終點。毗濕奴負責尋找終點，而梵天負責尋找起點。毗濕奴變成一頭野豬，不斷向下挖掘；梵天則變成一隻天鵝，向上飛翔。但找了許久雙方仍然找不到目標。毗濕奴只好放棄，在濕婆面前跪下，表示濕婆才是宇宙真實的[[梵|梵]]。但梵天並不輕言放棄，他不斷地飛翔，直到發現[[露兜樹|露兜樹]]花。由於梵天不甘示弱，他要求露兜樹花替他作假見證，說梵天已經發現林伽的起點。此時，全知的濕婆動怒，因而[[詛咒|詛咒]]梵天。
# 一種說法是，[[眾生|眾生]]有很重的偏執，為了永享安詳，在沒有任何苦行下，剽竊了梵天的法，結果人間失去秩序，眾生生出愚癡，掉進無間地獄。
# 另一種說法，{{link-en|婆利古聖人|Bhrigus}}邀請梵天參加一場祭典，但梵天卻沉浸他的妻子辯才天女的[[音樂|音樂]]之中，完全不理會[[聖人|聖人]]的呼喚，聖人大怒之下，便詛咒梵天，留作世世代代對梵天攻訐的口實。}}

1856－1857年間，美国文学家[[愛默生|愛默生]]創作了一首標題為《梵天》的詩。

== 梵天的住所薩特亞羅卡（Satyaloka） ==
{{link-en|薩特亞羅卡|Satyaloka}}在塔婆羅卡（Tapoloka）的上方120,000,000[[由旬|由旬]]處，因此從[[太陽|太陽]]到薩特亞羅卡的距離是233,800,000由旬，約莫1,870,400,000[[英里|英里]]。{{link-en|法儀昆塔|Vaikunta}}（Vaikunta）的眾行星在薩特亞羅卡上方26,200,000由旬處，約莫209,600,000英里。

「在法儀昆塔世界裡，精神天空的四方是超越我們這個世界的住所，分別由婆蘇提婆（Vasudeva）、散卡爾善（Samkarshan）、普拉德尤那（Pradyumna）及阿那律（Aniruddha）居住。法儀昆塔的天空是精神之水構成的，其中住著婆蘇提婆，在薩特亞羅卡上方是毗濕奴羅卡（Vishnuloka），住著散卡爾善。在{{link-en|乳海|Ocean of Milk}}的中心座落著一座名為斯威塔德威帕（Swetadwipa）的島嶼，上面住著阿那律。此島嶼的地基是巨蟒阿南塔蛇所形成的蛇床。」《{{link-en|永恆的采坦尼亚|Caitanya-caritamrta}}》Adi-lila第五章  [http://prabhupadabooks.com/?g=176590]
