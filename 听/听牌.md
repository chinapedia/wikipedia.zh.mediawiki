'''听牌'''，是指[[麻将|麻将]]中离[[和_(日本麻将)|和了]]还差必要的一张牌的状态，引申為在賽事之中再贏一次即可取得勝利。

== 听牌的形式 ==
听牌，根据所听的牌是面子的哪一部分，进行分类。[[日本麻将计分方法#符的计算|符数]]的计算也与此有关。

=== 四面子一雀头 ===
四面子<ref>包括顺子（例如{{牌画|三筒}}{{牌画|四筒}}{{牌画|五筒}}）和刻子（例如{{牌画|五索}}{{牌画|五索}}{{牌画|五索}}）</ref>一雀头形、听牌必然是以下列举的两面、嵌张、边张、双碰、单骑五种形式的其中之一或组合。

其中，两面、嵌张、边张这三种，是顺子的听牌形，对对和形式的役是不会出现这样的听牌的。反过来，双碰是刻子的听牌形，两杯口这样的四顺子一雀头的牌形是不会出现双碰听的。与此相对，单骑是雀头的听牌形。还有，原理上，双碰和单骑（国士无双除外）是不可能通过抢杠来和了的。

==== 两面听 ====
'''両面听'''，是指听牌时已经完成了三个面子和一个雀头，剩下最后的两面搭子<ref>两面搭子就是不含１９的两张连续的数牌所形成的组合，（例如{{牌画|四萬}}{{牌画|五萬}}，或者{{牌画|五索}}{{牌画|六索}}）</ref>的状态。

（例）{{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|三索}}{{牌画|四索}} {{牌画|牌背}}{{牌画|牌背}} 听{{牌画|二索}}{{牌画|五索}}

听牌的枚数最大是2种8枚，是五种形式中最好的听牌方式。两面听和了时不会增加符数。两面听是平和和了的成立条件之一。

==== 嵌张听 ====
'''嵌张听'''（又称'''坎张听'''），是指听牌时已经完成了三个面子和一个雀头，剩下最后的嵌搭子<ref>嵌塔子就是中间跳过一个数字的两张数牌的组合，（例如{{牌画|四萬}}{{牌画|六萬}}，或者{{牌画|五索}}{{牌画|七索}}）</ref>的状态。嵌张听时和了符数+2。

（例）{{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|四索}}{{牌画|六索}} {{牌画|牌背}}{{牌画|牌背}} 听{{牌画|五索}}

听牌的枚数最大是1种4枚，比起[[#两面听|两面]]来说是不利的牌形，在这个例子中如果摸到三索或者七索，可以变成两面听。

另外，嵌张听牌，容易形成战略性的引挂立直（{{lang-ja|引っかけリーチ}}）。例如自己{{牌画|一筒}}{{牌画|三筒}}{{牌画|五筒}}切五筒[[立直|立直]]时，对手会假定这是两面听，可能会认为二筒是[[安全牌|安全牌]]。见到自己立直的对手，心理上会倾向于选择打安全牌，反过来利用这种认为[[筋牌|筋牌]]是安全牌的心理，是一种战略。

==== 边张听 ====
'''边张听'''，是指听牌时已经完成了三个面子和一个雀头，剩下最后的边搭子<ref>边塔子就是由两张１２或是８９的数牌所形成的组合，（例如{{牌画|一萬}}{{牌画|二萬}}，或者{{牌画|八索}}{{牌画|九索}}）</ref>的状态。边张听时和了符数+2。

（例）{{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|八萬}}{{牌画|九萬}} {{牌画|牌背}}{{牌画|牌背}} 听{{牌画|七萬}}

听牌的枚数最大是1种4枚。不同于嵌张，边张不能向两面听变化，所以比起嵌张是更不利的牌形。12的边搭子摸到4、89的边搭子摸到6时可以变成嵌张，如果要变成两面听还要再摸一手。另外，顺子形的场合，以及断幺九、绿一色等没有幺九牌的役，是不可能出现边张听的。

另一方面，嵌张听也同样，因为边张听而导致[[三色同顺|三色同顺]]、[[一气通贯|一气通贯]]等役确定的场合中，不必担心[[和_(日本麻将)#振听|振听]]、[[低目|低目]]。不同于两面听，因为对手对自己的听牌读牌困难，所以容易引挂立直。

==== 双碰听 ====
'''双碰听'''，即是听牌时完成了三个面子，还有两个对子的状态。

（例）{{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|發}}{{牌画|發}} {{牌画|五萬}}{{牌画|五萬}} 听{{牌画|發}}和{{牌画|五萬}}

听牌的枚数最大是2种4枚。虽然和两面听同样是听两种牌，但因为枚数和嵌张、边张是一样的，所以比起两面并不是有利的牌形。另外，虽然双碰听牌时不加符，但和了的时候必然会形成刻子，把那部分的符加到了这里。

双碰听和了时，两个对子中一个形成雀头、一个形成刻子。刻子在自摸和了时算'''暗刻'''，荣和时算'''明刻'''。[[三暗刻|三暗刻]]和[[四暗刻|四暗刻]]是[[门前自摸|自摸和了]]还是[[荣和|荣和]]，成不成立的判定会发生改变，这点必须注意。

==== 单骑听牌 ====
'''单骑听牌'''是指，在听牌时已经完成了四个面子，将形成雀头的对子还少一枚的状态，单骑听牌和了符数+2。

（例）{{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|三索}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} 听{{牌画|三索}}

听牌枚数最大是1种3枚，在5种形式中是最少的。只是听牌的转换比较容易，并且此时形成[[和_(日本麻将)#振听|振听]]的可能性也低。还有，根据自摸牌的不同，根据和已经完成的面子的组合，可能向两面听甚至多面听变化。

===== 裸单骑 =====
因为四副露，手牌只剩下一枚，这种状态称为'''裸单骑'''。裸单骑的状态下，安全牌最多只能确保1枚，弃和非常困难。

（例）{{牌画|七萬}}{{spaces|3}}{{牌画|六索}}{{牌画|六索|横}}{{牌画|六索}}{{spaces|3}}{{牌画|二筒}}{{牌画|二筒|横}}{{牌画|二筒}}{{spaces|3}}{{牌画|五筒|横}}{{牌画|五筒}}{{牌画|五筒}}{{spaces|3}}{{牌画|九萬}}{{牌画|九萬}}{{牌画|九萬|横}}
: 这个状态下，遭遇听{{牌画|四萬}}{{牌画|七萬}}的立直时，运气好的话摸到七万就好，摸到四万就完了。
: 在雀庄等地方，有时候禁止裸单骑（因为“不成体统”这样的理由）。

===== 地狱单骑 =====
单骑听牌时，所听的牌在舍牌和其他家的副露中可以看到两枚，有效的和了牌只剩下一枚的状态，称为'''地狱单骑'''。地狱单骑主要是听字牌。因为字牌不能构成顺子，以及被舍去了2枚，所以处于使用起来比较困难的状态，这点可以加以利用。

==== 七对子的单骑听牌 ====
[[七对子|七对子]]听牌时，必然是单骑听牌。只是因为七对子的符是固定的（关东25符，关西30符），所以不计单骑听牌那2符。

（例）{{牌画|白}} {{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}} 听{{牌画|白}}


==== 国士无双一面听 ====
'''国士无双一面听'''，是指[[国士无双|国士无双]]听牌时，雀头已经确定，幺九牌<ref>（包括字牌，数牌的1和9）</ref>差一枚的状态。与单骑听牌听1种3枚相对，国士无双一面听是听1种4枚。这种听牌方式有时被称为'''国士单骑'''。本来单骑听牌（包括七对子）原理上是不可能[[抢杠|抢杠]]的，「国士单骑」是可能抢杠的唯一的单骑听牌，和其他的单骑听牌性质不同。也称为'''国士无双・听○○'''（{{lang-ja|国士無双・○○待ち}}）（○○是「發」・「一索」等、所缺的那一张牌的名字）。

（例）{{牌画|一萬}}{{牌画|九萬}}{{牌画|一索}}{{牌画|九索}}{{牌画|一筒}}{{牌画|九筒}}{{牌画|東}}{{牌画|南}}{{牌画|西}}{{牌画|北}}{{牌画|白}}{{牌画|發}}{{牌画|發}} 听{{牌画|中}}

==== 国士无双十三面听 ====
'''国士无双十三面听'''，是指国士无双听牌时，手牌是13种幺九牌各有一枚，还没确定雀头的状态。从形式上看这是单骑听牌，所以有时也被称为'''十三面单骑'''。这种情况下，最大听牌数是13种39枚。只是，如果舍弃过至少一张幺九牌，就会形成[[和_(日本麻将)#振听|振听]]，所以如果做出了这种听牌形就很容易和了，但做想做出这种听牌形非常的困难。不振听的纯粹十三面听的国士无双，在有的规则中计两倍役满。

（例）{{牌画|一萬}}{{牌画|九萬}}{{牌画|一筒}}{{牌画|九筒}}{{牌画|一索}}{{牌画|九索}}{{牌画|東}}{{牌画|南}}{{牌画|西}}{{牌画|北}}{{牌画|白}}{{牌画|發}}{{牌画|中}} 听所有的幺九牌


== 听牌的种类 ==
听牌时，听了m种牌，就称为「m面听」或「m面张」。四面子一雀头，即四个完成的面子和一个对子的役，听牌时，手牌中含有的未完成的面子的形状，决定了所听的牌是什么种类。

听的牌有三种以上时，称为'''多面听'''（{{lang-ja|多面待ち}}）、'''多面张'''（{{lang-ja|多面張（タメンチャン）}}）。多面听之中，很多很难一眼看出听什么牌。对此，进行一定程度的训练是必要的。另外，四面听以上的复杂的牌型，把它的形状死记硬背下来有时还更快。频繁出现的多面听的牌形并不是那么多。

以下就听牌的种类的数量进行详述。还有，{{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}}代表已经完成的面子，{{牌画|牌背}}{{牌画|牌背}}代表雀头，它们是对所听的牌没有影响的牌。

=== 一面听 ===
上述的[[#嵌张听|嵌张听]]、[[#边张听|边张听]]、[[#单骑听牌|单骑听]]，以及[[#国士无双一面听|国士无双一面听]]是代表。但是，因为某种牌已经使用了4枚，使本来的听牌被破坏了，导致只听一面的情况也是有的。

; 双碰的一面听
双碰听不止有两面听。以下这样，其中一种听牌已经被自己使用了四枚，就成为了双碰的一面听。因为这种情况仍然是双碰听，所以不会得到听牌的符。这种情况下，听牌的枚数最大是1种2枚。

: （例）{{牌画|一筒}}{{牌画|一筒}}{{牌画|二筒}}{{牌画|二筒}}{{牌画|三筒}}{{牌画|三筒}}{{牌画|三筒}}{{牌画|三筒}} {{牌画|北}}{{牌画|北}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}}
:: 在这种情况下，只听{{牌画|北}}。{{牌画|三筒}}因为被使用了4枚，就听不了了。

; 两面的一面听
两面中也有一面听的情况。因为形式上来说是两面，所以不能得到听牌的符。这种情况下，听牌的枚数最大是1种4枚。

: （例）{{牌画|一索}}{{牌画|二索}}{{牌画|二索}}{{牌画|三索}}{{牌画|三索}}{{牌画|三索}}{{牌画|三索}}{{牌画|四索}}{{牌画|四索}}{{牌画|五索}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}}
:: 这种情况下，只听{{牌画|六索}}。{{牌画|三索}}已经被使用了4枚，就听不了了。


; 延单骑的变则形
在后述的延单骑中，有一边的数牌还再形成了一个刻子，因为这个数牌已经被使用了4枚，结果和单骑听牌一样只有一面听。这种情况下，听牌的枚数最大为1种3枚。

: （例）{{牌画|五筒}}{{牌画|五筒}}{{牌画|五筒}}{{牌画|五筒}}{{牌画|六筒}}{{牌画|七筒}}{{牌画|八筒}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}}
:: 这种情况下，听{{牌画|八筒}}

=== 两面听 ===
两面听的情况包括上述的两面搭子与双碰听牌，此处不再赘述。
==== 延单骑 ====
'''延单骑'''（{{lang-ja|ノベタン}}），是指四枚连续的数牌组成的牌形。两边的两种数牌都是所听的牌。听牌枚数最大是2种6枚。虽然和两面听一样，听了一对筋牌，但却是单骑听，所以也被称为两面单骑（{{lang-ja|両面単騎}}）。

: （例）{{牌画|四筒}}{{牌画|五筒}}{{牌画|六筒}}{{牌画|七筒}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}}
:: 这种情况下，听{{牌画|四筒}}{{牌画|七筒}}

==== 亚两面 ====
两面听时，所听的其中的一边已经有一个雀头。听牌枚数最大是2种6枚。
: （例）{{牌画|三筒}}{{牌画|四筒}}{{牌画|五筒}}{{牌画|五筒}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}}
::这种情况下，听{{牌画|二筒}}{{牌画|五筒}}。虽然属于两面听的一种，这种牌型又特别地被称为'''亚两面'''（{{lang-ja|亜両面（ありゃんめん）}}）。

==== 变则二面听 ====
; 数牌n的暗刻和数牌n±2的复合
因为同时存在嵌张听、单骑听，所有听牌有两种。听牌枚数最大是2种7枚。

因为同时存在嵌张听、单骑听，所有听牌有两种。听牌枚数最大是2种7枚。

: （例1）{{牌画|二索}}{{牌画|二索}}{{牌画|二索}}{{牌画|三索}}{{牌画|四索}}{{牌画|五索}}{{牌画|七索}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}}
:: 这种情况下，听{{牌画|六索}}和{{牌画|七索}}

: （例2）{{牌画|二索}}{{牌画|四索}}{{牌画|五索}}{{牌画|六索}}{{牌画|七索}}{{牌画|七索}}{{牌画|七索}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}}
:: 这种情况下，听{{牌画|二索}}和{{牌画|三索}}

=== 三面听牌 ===
三面听牌，有听1-4-7、2-5-8、3-6-9等单一的一路筋的三面听（'''狭义的三面听'''）和1-4-3、2-5-4等听几路筋的'''变则三面听'''两种。

==== <span id="狭义的三面张">狭义的三面听</span> ====
两面搭子和顺子形成m,m+1,m+2,m+3,m+4的形状。通常被简单地称为'''三面听'''（{{lang-ja|三面待ち}}）。也有人称为'''钢琴听牌'''（{{lang-ja|ピアノ待ち}}）。只有5枚的牌，最大听牌枚数却达到了3种11枚，被认为是良好的听牌形的一种。

: （例）{{牌画|二萬}}{{牌画|三萬}}{{牌画|四萬}}{{牌画|五萬}}{{牌画|六萬}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}
:: 这种情况下，听{{牌画|一萬}}{{牌画|四萬}}{{牌画|七萬}}

==== 因4枚牌而成立的三面听 ====
数牌n的暗刻和数牌n±1组成的牌型。是以最少枚数的牌构成的三面听。因为n可以视为刻子也可以视为雀头，所有形成了三面听。听牌枚数最大为3种11枚。因为听了不同路的筋，所以有时也被认为是比狭义的三面听更好的听牌。

: （例）{{牌画|二萬}}{{牌画|三萬}}{{牌画|三萬}}{{牌画|三萬}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}} {{牌画|牌背}}{{牌画|牌背}}{{牌画|牌背}}
:: 这种情况下，听{{牌画|一萬}}{{牌画|四萬}} {{牌画|二萬}}

===== 烟囱形 =====
顺子的一边的牌有3枚，其中两枚可以形成双碰的一方。可以被视为双碰，也可以被视为两面听。听牌枚数最大为3种7枚。因为这种牌型频繁出现，所以为了和其他的变则三面听相区别，特别地命名为'''烟囱形'''（{{lang-ja|エントツ形}}）。

==== 二十二面听？ ====
承认无雀头形的十三不塔时（有地方规则承认[[日本麻将古役#十三无靠|十三无靠（十四不搭）]]），在前述的十三不塔的十三面听听牌时，既可以形成十三不塔也可以形成十三无靠。十三无靠从一面听到九面听都有，和十三不搭的情况加起来，最大听牌数是二十二面。

===== 十三无靠和国士无双 =====
在不采用十三不塔而采用十三无靠时，配牌是是国士无双十三面时，第一次自摸前是二十二面听。
: 听牌多了国士无双的13张牌

==== 三十四面听？ ====
自摸[[#海底_(麻将)|海底牌]]之前，直至此时舍牌全部是幺九牌而且没有被鸣，而且手牌中还有幺九牌。不管海底牌是什么，总可以完成[[#荒牌满贯|流局满贯]]，这可以视为三十四面听。只是，流局满算不算是和了役还有争议，这不一定能说是多面听牌，但如果承认的话，这是最高的听牌数。（有的地方承认，把这种听牌方式称为连面金听牌（{{lang-ja|連面金(レンメンキン)待ち}}）。）
