'''飞行优异十字勋章'''（{{lang-en|'''Distinguished Flying Cross'''}}），或譯'''傑出飛行十字勳章'''，是一项[[美军跨军种通用勋奖|美军跨军种通用勋奖]]，授予“于1918年11月11日之后，飞行任务时表现出英勇行为或有特殊功绩者。” 在此之前立功的人员也可被授予此奖，前提是他未因此功而被授予[[荣誉勋章_(美国)|荣誉勋章]]、[[傑出服役十字勳章_(美國)|傑出服役十字勳章]]、[[海军十字勋章|海军十字勋章]]、[[空军十字勋章|空军十字勋章]]和优异服役勋章。

== 历史 ==
飞行优异十字勋章由1926年7月2日通过的[[美国国会|美国国会]]法案及1938年1月8日签署的7786号总统令确立颁行。该勋章的第一位得主是[[美國陸軍航空隊|美國陸軍航空隊]]的赫伯特·达阿格少將（Herbert Dargue），而非一般所认为的[[查尔斯·林德伯格|查尔斯·林德伯格]]。飞行优异十字勋章第一次授予海军是在1926年5月9日授予海军航空隊，用以表彰其飞越[[北极点|北极点]]。他们同时被授予荣誉勋章。

该勋章著名的获得者包括[[亨利·阿諾德|亨利·阿諾德]]、[[吉米·杜立特|吉米·杜立特]]（3次得獎）、[[柯蒂斯·勒邁|柯蒂斯·勒邁]]、[[陳納德|克萊爾·陳納德]]（2次得獎）、[[查尔斯·林德伯格|查尔斯·林德伯格]]、[[詹姆斯·史都華|詹姆斯·史都華]]（2次得獎）、[[查克·葉格|查克·葉格]]（3次得獎）、[[克拉克·蓋博|克拉克·蓋博]]、[[乔治·赫伯特·沃克·布什|乔治·赫伯特·沃克·布什]]、[[约翰·麦凯恩|约翰·麦凯恩]]、[[泰德·史蒂芬|泰德·史蒂芬]]、[[约翰·格伦|约翰·格伦]]（6次得獎）等。获得该勋章的平民分別是：[[萊特兄弟|萊特兄弟]]、[[威利·波斯特|威利·波斯特]]、[[賈桂琳·科克倫|賈桂琳·科克倫]]、[[愛蜜莉亞·厄爾哈特|愛蜜莉亞·厄爾哈特]]、[[尤金·埃利|尤金·埃利]]等。

战时美军的盟军成员也可获得该勋章。它同样可以颁给飞行学校的符合条件的教练和学生。

== 图案 ==
飞行优异十字勋章图案由伊丽莎白·威尔及亚瑟·E·迪布瓦设计。其主体为铜质十字，十字缝隙中有放射线条。勋章正面十字上有突出的十字螺旋桨图案，同时放射线条组成一个方型。背面无图案但刻有获得者的名字及军衔级别。

二次以上获得该勋章的[[美國陸軍|陸軍]]及[[美國空軍|空軍]]人员在佩戴时加銅質{{Link-en|橡葉章|Oak leaf cluster}}装饰，5枚銅橡葉章折合1枚銀橡葉章；二次以上获得该勋章的[[美國海軍|海軍]]、[[美國海軍陸戰隊|海軍陸戰隊]]、[[美國海岸防衛隊|海岸防衛隊]]人员加金星章（{{lang|en|Award star}}）装饰，5枚金星章折合1枚銀星章（非[[銀星勳章|銀星勳章]]）。

== 相关法律 ==
根据美国相关法律，任何私自佩戴该勋章或伪造此勋章者将面临一年以下监禁和5000美元以下罚款。
