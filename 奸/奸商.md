'''奸商'''泛指唯[[利潤|利]]是圖、不擇手段以獲取[[暴利|暴利]]的[[商人|商人]]。

==歷史==
古代中國，商人位列[[四民|四民]]之末，社會地位低下，但能掌握資源；普通老百姓對之看不起，又愛又恨，認為「無商不奸」、「逢商必奸」、「無奸不成商」等。

== 常見行為 ==
; 欺瞞哄騙
* 以虛偽不實[[廣告|廣告]]，誘使[[消費者|消費者]]進行[[消費|消費]]。
* 以[[詐騙|詐騙]]手段，進行不公正交易。例如：商品[[灌水|灌水]]（[[注水肉|注水肉]]），或偷斤減兩（在[[秤|秤]]上動手腳）
* 惡質[[推銷|推銷]]行為。（惡德商法）
* 超收[[代徵|代徵]]費用，涉嫌[[詐欺|詐欺]]與[[報假帳|報假帳]]。
; 以次充好
* 將瑕疵品隱瞞缺陷進行販售。
* 將[[二手|二手]]商品偽裝成新品販售。
* 將低價商品偽裝成高價商品販售。例如：將低價的[[棉籽油|棉籽油]]偽裝成高價的[[橄欖油|橄欖油]]。
; 哄抬[[囤積|囤積]]
* 散佈不實資訊，使消費者產生錯誤的預期[[漲價|漲價]]心理，藉以哄抬物價。
* 藉由囤積[[民生|民生]][[必需品|必需品]]，使自由市場的供需機制失調，藉以哄抬物價。

== 影響 ==
單一奸商的個人行為，對於社會的影響或許是有限的。但是如果是一群奸商對於民生必需品，聯合進行哄抬囤積的話，可能導致[[恶性通货膨胀|恶性通货膨胀]]。

== 應對方式 ==
*明顯違法的奸商行為，可以報警，或經由司法提告相關訴訟。
*或向各地的主管機關申訴，例如：台灣的[[行政院消費者保護委員會|消費者保護委員會]]、[[行政院公平交易委員會|公平交易委員會]]；中国大陆的[[中国消费者协会|消费者协会]]。
