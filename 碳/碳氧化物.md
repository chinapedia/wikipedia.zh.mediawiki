最简单常见的碳氧化物包括[[一氧化碳|一氧化碳]]（CO）和[[二氧化碳|二氧化碳]]（CO<sub>2</sub>）。除了这两种为人熟知的[[无机物|无机物]]，碳与氧其实还能构成许多稳定或不稳定的碳氧化物，但在现实生活中很难接触到其他碳氧化物（例如[[二氧化三碳|二氧化三碳]]（C<sub>3</sub>O<sub>2</sub>）等）。

[[教科书|教科书]]一般只介绍以上列出的前三种碳氧化物，极少有介绍第四种的。<!--中国大陆教科书只介绍前两种。-->但事实上，已有数十种碳氧化物被人们发现，之中大部分是在20世纪60年代人工合成的。这些碳氧化物有的在[[室温|室温]]下是稳定的，有的却即使在[[超低温|超低温]]环境中也会迅速[[分解|分解]]为较简单的其他碳氧化物。一部分[[寿命|寿命]]极短的[[亚稳态|亚稳态]]碳氧化物是作为[[化学反应|化学反应]]的[[中间体|中间体]]出现而被观测到的，这类碳氧化物的[[化学性质|化学性质]]十分活跃，以至于常常只能在[[气相|气相]]中或[[基质隔离|基质隔离]]下短暂存在。

新碳氧化物的合成量至今仍有不断上升的趋势。[[氧化石墨烯|氧化石墨烯]]以及具有可变结构的稳定碳氧化物[[聚合物|聚合物]]<ref name="hummers"/><ref name="snow"/>的发现说明在这个领域还有许多未知等待人们探索。

== 概述 ==
二氧化碳在[[自然界|自然界]]中广泛存在。[[生物|生物]]的[[呼吸作用|呼吸作用]]和[[化石燃料|化石燃料]]的[[燃烧|燃烧]]是[[大气|大气]]中二氧化碳的主要来源。虽然[[人类|人类]]许多生产生活过程中都需要接触二氧化碳，但[[化学家|化学家]]们直至17、18世纪时才逐渐意识到二氧化碳是一种[[化学物质|化学物质]]，并将其称为“''spiritus sylvestre''”，意为“森林的精气”。

一氧化碳是另一种常见的碳氧化物，它也会在燃烧中产生。很早以前，人类已开始利用一氧化碳的[[还原性|还原性]]来[[冶炼|冶炼]][[铁矿石|铁矿石]]从而获得[[金属|金属]][[铁|铁]]。一氧化碳与其[[氧化物|氧化物]]一样，在[[中世纪|中世纪]]开始被西方的[[炼金术士|炼金术士]]与后来的化学家研究。它的[[元素|元素]]组成则是于1800年由[[威廉·克鲁克香克|威廉·克鲁克香克]]了解的。

== 一般结构 ==
虽然[[氧元素|氧元素]]在含氧化合物中一般只呈固定的负二[[价态|价]]， 只能通过[[共价键|共价键]]与最多两个原子相连，且几乎不存在以超过三个相连的氧原子形成的[[链状化合物|链状化合物]]，过但碳原子最多能与四个原子相连，以碳链为[[骨架|骨架]]可以形成众多链状或网状的[[高分子化合物|高分子化合物]]。因此，碳与氧其实是能构组成许多以碳链为骨架的[[电中性|电中性]]分子的。这些分子中的碳原子可能呈线型排列或围成[[脂环族化合物|脂环]]或[[芳香族化合物|苯环]]，而氧原子则以形成两个[[碳氧单键|单键]]（－O－）、一个[[碳氧双键|双键]]（＝O）或相连形成[[过氧基|过氧基]]（－O－O－）的形式参与形成化合物。

碳氧化物中有许多是[[不饱和化合物|不饱和化合物]]，例如[[一氧化二碳|一氧化二碳]]（[[结构式|结构式]]为︰C＝C＝O）等，但这些化合物的化学性质都十分活跃，难以大量合成。<ref name="brinker"/>碳氧化合物分子得失[[电子|电子]]时，其中的氧原子可能会变为一价的－{{chem|O|-}}或三价的≡{{chem|O|+}}，而其中的碳原子则会变为三价的≡{{chem|C|-}}。后二者已在一氧化碳中被发现——<sup>-</sup>C≡O<sup>+</sup>。<ref name=nigg/>带负电荷的氧原子一般出现在[[碳氧阴离子|碳氧阴离子]]中。
