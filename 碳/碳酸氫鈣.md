'''碳酸氫鈣'''（化學式：[[鈣|Ca]]([[氫|H]][[碳|C]][[氧|O]]<sub>3</sub>)<sub>2</sub>），是一種[[無機化合物|無機]][[酸式鹽|酸式鹽]]。

產生碳酸氫鈣最重要的化學反應即是以[[碳酸鈣|碳酸鈣]]溶於[[二氧化碳|二氧化碳]]水溶液而成碳酸氫鈣，反應式如下：
:Ca(OH)<sub>2</sub> + CO<sub>2</sub> → CaCO<sub>3</sub>↓ + H<sub>2</sub>O
:CaCO<sub>3</sub> + H<sub>2</sub>O + CO<sub>2</sub> → Ca(HCO<sub>3</sub>)<sub>2</sub>

碳酸氢钙微溶于水，但碳酸钙难溶。[[钟乳石|钟乳石]]的形状即是由微溶的碳酸氢钙与难溶的碳酸钙在一定[[二氧化碳|二氧化碳]]浓度下的互相转变造成的。

碳酸氢钙只存在溶液中；将碳酸氢钙溶液蒸发则得到碳酸钙固体。

==化学性质==
碳酸氢钙不稳定，稍受热即可分解：
: Ca(HCO<sub>3</sub>)<sub>2</sub> → CaCO<sub>3</sub> + H<sub>2</sub>O + CO<sub>2</sub>↑

它也可以和酸反应，产生钙盐：
: Ca(HCO<sub>3</sub>)<sub>2</sub> + 2 HCl → CaCl<sub>2</sub> + 2 H<sub>2</sub>O + 2 CO<sub>2</sub>↑
