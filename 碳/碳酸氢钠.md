'''碳酸氢钠'''（sodium bicarbonate、[[IUPAC命名法|IUPAC名]]: sodium hydrogen carbonate、baking soda）是一种[[无机化合物|无机化合物]]，化学式为[[钠|Na]][[氢|H]][[碳酸根|CO<sub>3</sub>]]，俗称小苏打、蘇打粉、重曹、焙用鹼等，白色细小晶体，在水中的溶解度小于[[碳酸钠|碳酸钠]]，呈[[弱碱|弱碱]]性。

==化學性质==

50℃以上開始逐漸分解生成[[碳酸钠|碳酸钠]]、[[二氧化碳|二氧化碳]]和[[水|水]]，270℃時完全分解。碳酸氢钠是强[[碱|碱]]與弱[[酸|酸]]經[[中和作用|中和作用]]後生成的[[酸式鹽|酸式鹽]]，溶於水時呈現弱鹼性。

: 2 NaHCO<sub>3</sub> → Na<sub>2</sub>CO<sub>3</sub> + H<sub>2</sub>O + CO<sub>2</sub>↑

常利用此特性作為食品制作过程中的膨松剂。

碳酸氢钠可以和酸反应，生成相应的钠盐：
: NaHCO<sub>3</sub> + [[乙酸|CH<sub>3</sub>COOH]] → [[乙酸钠|CH<sub>3</sub>COONa]] + H<sub>2</sub>O + CO<sub>2</sub>↑
: NaHCO<sub>3</sub> + HCl → NaCl + H<sub>2</sub>O + CO<sub>2</sub>↑

碳酸氢钠溶液可以和镁反应：<ref>姜伟, 何月华. [https://wenku.baidu.com/view/d22d43f3a45177232f60a2a4.html 镁与碳酸氢钠溶液反应的再探究][J]. 中学化学教学参考, 2015(11):56-57.</ref>
: 4 Mg + 8NaHCO<sub>3</sub> + 4 H<sub>2</sub>O → 3MgCO<sub>3</sub>·Mg(OH)<sub>2</sub>·3H<sub>2</sub>O + 4 Na<sub>2</sub>CO<sub>3</sub> + 4 H<sub>2</sub>↑ + CO<sub>2</sub>↑

== 制法 ==

碳酸氢钠最常见的制备方法是将二氧化碳通入碳酸钠溶液中，结晶得到：
: Na<sub>2</sub>CO<sub>3</sub> + CO<sub>2</sub> + H<sub>2</sub>O → 2 NaHCO<sub>3</sub>

: 2 NaCl + 2 H<sub>2</sub>O + 2 CO<sub>2</sub> -<u><sup>通电</sup></u>→ 2 NaHCO<sub>3</sub> + Cl<sub>2</sub>↑ + H<sub>2</sub>↑

在此反应中，饱和食盐水由于通入过量二氧化碳而电解生成碳酸氢钠和氯气、氢气。

== 應用 ==
* 碳酸氫鈉與[[油脂|油脂]]直接混合時會產生[[皂化|皂化]]，肥皂味太强烈會影響[[西點|西點]]的香味和品質。
* 碳酸氫鈉經常被用作中和劑，例如平衡[[巧克力|巧克力]]的酸性。同時也可使[[巧克力|巧克力]]的颜色加深，使它看起来更黑亮。
* 如在[[西點|西點]]中加入過量碳酸氫鈉，除了會破壞品質、導致鹼味過重，也會令食用者出現[[心悸|心悸]]、嘴唇發麻、短暫失去[[味覺|味覺]]等徵状。
*{{fact|小蘇打可分為藥用、食用、工業用等種類，隨著種類不同在購買地點上也不同。其中，工業用的小蘇打粉的純度較低，其雜質較多，相對的食用的小蘇打粉純度較高。}}
*小蘇打粉應用於家庭清潔，可以去污、除臭。因油膩、汙漬或難聞氣味，大多屬於酸性物質（H+，如脂肪酸），當弱鹼性的氫氧根OH- 與酸性物質相遇，便會產生中和作用。塗抹髒污靜待數分鐘，之後再輕輕擦拭即可。
