《'''膽小鬼LINE'''》（{{jpn|j=チキンLINE|rm=Chicken Line}}）是[[日本|日本]][[日本偶像|偶像]]團體[[SKE48|SKE48]]的第19張[[單曲|單曲]]，2016年3月30日由[[avex_trax|avex trax]]發行。

== 概要 ==
單曲分為Type A<ref>[https://shopping.avex.com.tw/avex/Default.aspx?m=ProductDetail&ItemId=9360 愛貝克思官方網站 「膽小鬼LINE」 Type A介紹]</ref>、Type B<ref>[https://shopping.avex.com.tw/avex/Default.aspx?m=ProductDetail&ItemId=9361 愛貝克思官方網站 「膽小鬼LINE」 Type B介紹]</ref>、Type C<ref>[https://shopping.avex.com.tw/avex/Default.aspx?m=ProductDetail&ItemId=9362 愛貝克思官方網站 「膽小鬼LINE」 Type C介紹]</ref>、Type D<ref>[https://shopping.avex.com.tw/avex/Default.aspx?m=ProductDetail&ItemId=9363 愛貝克思官方網站 「膽小鬼LINE」 Type D介紹]</ref>、劇場版，星光樂園版，6個版本，其中Type A、B、C、D四個標準版本還分別推出了只有唱片上市初期限期發行的初回限定版，與長期發行的通常版，作詞為[[秋元康|秋元康]]。其中，《膽小鬼LINE》和《沒有望遠鏡的天文台》收錄於所有版本，歌曲直白地描述「直接向喜歡的人傳達愛意」的重要性，聽了彷彿有就會有勇氣將自己的心情傳達出去！此曲也是動畫電影《[[星光樂園#星光樂園_大家的憧憬♪_去吧☆星光巴黎|星光樂園 大家的憧憬♪ 去吧☆星光巴黎]]》的主題曲。《他有女朋友》收錄於Type A中，《接吻位置》收錄於Type B中，《Is that your secret?》收錄於Type C中，《旅途中》收錄於Type D中，是宫泽佐江的畢業歌曲。全碟填詞為秋元康。Type A和Type B收錄了其音樂錄影帶和特別影片《SKE48 倒數公演2015→2016》；Type C和Type D收錄了其音樂錄影帶和特別影片《SKE48 無法忘懷的大忘年會2015》。

單曲的選拔人數增至17人。菅原茉椰初次成為選拔成員。

在[[Oricon公信榜|Oricon公信榜]]，單曲以25萬銷量取得週榜第一位。

*初回限定版A：松井珠理奈

*通常版B：江籠裕奈・惣田紗莉渚・菅原茉椰
