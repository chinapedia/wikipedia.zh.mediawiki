'''西北旺天主堂'''，全名'''天主教北京教区西北旺圣若瑟堂'''，位于[[北京市|北京市]][[海淀区|海淀区]][[西北旺地区|西北旺地区]][[永丰路_(北京)|永丰路]]，隶属[[天主教北京教区|天主教北京教区]]。

==简介==
西北旺天主堂的前身是西北旺祈祷所（西北旺弥撒点）。2008年5月，西北旺祈祷所正式成立。历经孙徐生、张芝茂两位本堂神父，每主日上午9时举办[[弥撒|弥撒]]。西北旺祈祷所利用[[北京天主教陵园|北京天主教陵园]]原会客室作为弥撒场地，起初可容60人左右。到2013年，主日弥撒参加者已经超过150人。2013年，在北京教区主教[[李山|李山]]和北京教区的关怀下，经张芝茂神父具体指导，全体教友完成了小圣堂的扩建工作。经过扩建的小圣堂可容200人。2013年5月西北旺祈祷所（弥撒点）成立五周年之际，全体西北旺教友与各方神长教友聚集在西北旺圣若瑟堂，举办庆祝活动和感恩弥撒。<ref>{{Cite web|url =http://www.catholic-bj.org/content.php?id=4429&fuid=60&tid=58|title =西北旺圣若瑟堂简介|author =|date =|publisher =天主教北京教区|language =|accessdate =2016-12-25|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20161227055844/http://www.catholic-bj.org/content.php?id=4429&fuid=60&tid=58|archivedate =2016年12月27日|df =}}</ref><ref>{{Cite web |url =http://www.xinde.org/News/index/id/31229.html  |title =  北京：北京神哲学院师生拜访西北旺堂区暨“天主教北京教区陵园”|author =  |date = 2015-03-28 |publisher =  信德|language =  |accessdate =  |quote =  }}</ref>2016年7月2日，西北旺圣若瑟堂举行开堂典礼。2016年11月26日，海淀区天主教第三次代表会议召开，来自[[正福寺天主堂|正福寺教堂]]、[[北京天主教神哲学院|后八家教堂]]、西北旺弥撒点、[[白家疃天主堂|白家疃弥撒点]]的教友代表和教职人员共80人参加。<ref>{{Cite web|url =http://www.beijing.gov.cn/zfzx/qxrd/hdq/201611/t20161129_1324614.htm|title =海淀区天主教第三次代表会议召开|author =|date =2016-11-29|publisher =首都之窗|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20161227055517/http://www.beijing.gov.cn/zfzx/qxrd/hdq/201611/t20161129_1324614.htm|archivedate =2016-12-27}}</ref>
