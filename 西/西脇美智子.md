'''西脇美智子'''{{BD|1957年|11月21日|||CatIdx=Nishiwaki Michiko}}是一位[[日本|日本]]出身的國際動作女星。生於[[千葉縣|千葉縣]]，有健身界[[山口百惠|山口百惠]]之稱，1996年結婚生子，目前住在[[美國|美國]][[加州|加州]]Moorpark。

西脇原本是健身家與舉重冠軍。在進軍演藝界之前已是日本女子健身界的知名人物，而且擁有兩間和哥哥合資的健身中心。她10多岁开始进行[[体操|体操]]及排球训练，后为了练好身材进入[[健美|健美]]界，练成健美等級的一身肌肉后曾连续三次获得健美冠军和[[舉重|舉重]]冠軍。後來在日本演藝界推出一本露點寫真集和《[[香港・東京特捜刑事|香港・東京特捜刑事]]》等兩部露點三級片以求出名。也就凭此健美形象她偶然出演了港片《[[福星高照|福星高照]]》成了[[香港電影|香港影坛]]关注对象，1987年《[[雌雄大盗|雌雄大盗]]》因為票房大紅更是让她一举知名，是动作片中的经典。<ref name=sirens>{{cite web
|url=http://www.cultsirens.com/nishiwaki/nishiwaki.htm
|title=Michiko Nishiwaki
|publisher=Cult Sirens
|year=2004
|accessdate=2009-03-14}}</ref>

之後在香港為主要舞台發展多年以女打仔形象出演一些反派或配角，但是由於不會說中文和[[廣東話|廣東話]]，導致發展極大受限，星路越走越窄逐漸轉入二線片，後來又重回接拍三級片的老路拍攝《淫妖豪情》，但依然沒有突破遂逐漸退出香港影壇，1996年嫁給美國人轉往美擔任一些動作替身和小角色演出至今，其兒子Kaz DeBear亦為美國健身教練。<ref>{{cite web
