'''西宁西站'''位于[[中国|中国]][[青海省|青海省]][[西宁市|西宁市]][[城北区_(西宁市)|城北区]][[大堡子镇|大堡子镇]]，建于1960年。该站为[[青藏铁路|青藏铁路]]的[[三等站_(中国)|三等站]]，由[[中国铁路青藏集团有限公司|中国铁路青藏集团有限公司]]营运。

== 使用情况 ==
西宁西站是[[青藏铁路|青藏铁路]]的[[三等站_(中国)|三等站]]。 站房设计总建筑面积为6871.7[[平方米|平方米]]，其中旅客客运面积为5999.4平方米，候车室使用面积3886平方米，售票厅面积647.9平方米；有22个售票窗口；股道5条，基本站台3座，均为高站台，与火车车门底部平行。<ref name="multiple">{{cite web |url=http://www.qhxxcn.com/Html/jt/12068.html |language=zh-cn |title=昨晚老站送别最后一趟列车 西宁西站今起正式启用 |publisher=www.qhxxcn.com |date=2011-03-10 |accessdate=2012-01-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20130928085142/http://www.qhxxcn.com/Html/jt/12068.html |archivedate=2013-09-28 }}</ref>

== 历史 ==
* 1960年：西宁西站建成启用。
* 2010年10月：西宁西站过渡工程动土开工。
* 2011年3月：由于[[西宁站|西宁站]]封闭改建，西宁西站接管其所有客运业务。<ref name="multiple" />
* 2014年12月24日：配合西宁站扩建工程完成，西宁西站於当天23:00起停止办理所有客运业务。
