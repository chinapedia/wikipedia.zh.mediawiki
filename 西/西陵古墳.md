'''西陵古墳'''（{{jpn|j=西陵古墳|hg=さいりょうこふん|rm=Saryōkofun}}），又稱'''西二山在古墳'''，是位於日本[[大阪府|大阪府]][[泉南郡|泉南郡]][[岬町|岬町]]淡輪的一座{{Link-ja|前方後圓墳|前方後円墳}}，為日本國史跡，屬於淡輪古墳群，亦是古墳群中規模最大的古墳，日本全國來說則排第28位<ref name="堺市">{{cite web|url=https://www.city.sakai.lg.jp/kanko/rekishi/dkofun/ranking/zenkoku.html |title=古墳大きさランキング（日本全国版）|publisher=[[堺市|堺市]] |accessdate=2016-09-24}}</ref>，推測建於[[古墳時代|古墳時代]]中期（約5世紀前半）。

== 概要 ==
古墳是一座位於大阪府最南端，面臨[[大阪灣|大阪灣]]台地上建成的巨大前方後圓墳，其約800米以東是傳{{Link-ja|五十瓊敷入彥命|五十瓊敷入彦命}}墓的[[淡輪御陵古墳|淡輪御陵古墳]]，兩者之間是已不存在的[[圓墳|圓墳]]{{Link-ja|西小山古墳|西小山古墳}}，這批古墳形成了淡輪古墳群。

古墳是一座建於丘陵尾端的前方後圓墳，前方部分朝東北{{Sfn|岬町の歴史|1995年|pp=45-47}}。墳丘由3層組成，長約210米{{Sfn|岬町の歴史|1995年|pp=45-47}}，西側的中間細部建有方形的{{Link-ja|造出|造出}}，墳丘表面鋪上了{{Link-ja|葺石|葺石}}，亦有{{Link-ja|圓筒埴輪|円筒埴輪}}、{{Link-ja|朝顏形埴輪|朝顔形埴輪}}、蓋形、盾形、短甲形、屋形埴輪等{{Link-ja|形象埴輪|形象埴輪}}。墳丘周圍由闊15-35米的周濠所包圍，其原本的形狀則不明{{Sfn|岬町の歴史|1995年|pp=45-47}}，其外圍有3座[[陪葬墓|陪葬墓]]，其中一座已不存在{{Sfn|西陵古墳（平凡社）|1986年}}。

雖然墓室實際形態不明，但是推斷建有內裡為[[凝灰岩|凝灰岩]]製{{Link-ja|長持形石棺|長持形石棺}}的{{Link-ja|豎穴式石室|竪穴式石室}}{{Sfn|西陵古墳（平凡社）|1986年}}。原本後圓部分的墳頂可以看見曝露在外的石棺蓋石，在指定為日本國史跡時被埋掉{{Sfn|西陵古墳（平凡社）|1986年}}。

從出土埴輪推測，古墳建於420年至440年左右{{Sfn|岬町の歴史|1995年|pp=45-47}}，是在淡輪古墳群中最早建成的一座，這批古墳的圓筒埴輪與別不同，而相同的有{{Link-ja|和歌山平野|和歌山平野}}的木之本古墳群（[[和歌山市|和歌山市]]木之本），由此可見淡輪古墳群是在[[紀伊國|紀伊]]勢力{{Sfn|岬町の歴史|1995年|pp=51-52}}。

1922年，古墳域與兩座陪葬墓獲指定為日本國史跡<ref name="国指定"/>。

== 入葬者 ==
雖然入葬者不明，但是單憑岬町當地的財力的話，興建兩座大型前方後圓墳並不容易，因此相信古墳與[[紀伊國|紀伊]]勢力（{{Link-ja|紀氏|紀氏}}）有關{{Sfn|岬町の歴史|1995年|pp=67-68}}，尤其是《[[日本書紀|日本書紀]]》[[雄略天皇|雄略天皇]]9年3月條和5月條記載了「葬於田身輪邑（{{lang|ja|たむわのむら，即淡輪村}}）」，因此也有說法認為此古墳是5世紀中後期武人{{Link-ja|紀小弓|紀小弓}}的墓<ref name="2014年調査">2014年度の発掘調査に関する各ニュースサイト記事。

* [http://www.sankei.com/life/news/141205/lif1412050018-n1.html “宮内庁が大阪・淡輪ニサンザイ古墳を公開”] {{webarchive|url=https://web.archive.org/web/20150924130935/http://www.sankei.com/life/news/141205/lif1412050018-n1.html |date=2015-09-24 }}（産経ニュース、2014年12月5日記事）
* [http://www.47news.jp/CN/201412/CN2014120501001478.html “淡輪ニサンザイ古墳を公開　陵墓、宮内庁が発掘調査”]{{dead link|date=十月 2017 |bot=InternetArchiveBot }}（共同通信、2014年12月5日記事（47NEWS））
</ref>。
另外，《和泉志》也視古墳為紀小弓的墓，《泉州志》則認為古墳與西陵古墓其中一處是紀小弓的墓，另一處則是{{Link-ja|紀船守|紀船守}}的墓，但是紀船守是8世紀的人物，與古墳建造時期並不吻合{{Sfn|岬町の歴史|1995年|pp=69-70}}。

== 陪葬墓 ==
古墳北面建有稱為第一古墳和第二古墳的圓墳陪葬墓{{Sfn|西陵古墳（平凡社）|1986年}}。以前還有另一座陪葬墓，但是在線路鋪設時遭到破壞，當時出土了提瓶2個、高杯1個、壺2個、平瓶1個、刀身斷片和{{Link-ja|鐵鏃|鉄鏃}}2件。
