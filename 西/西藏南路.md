'''西藏南路'''位於[[上海市|上海市]]，是一條南北向城市次干道，北起[[延安东路_(上海)|延安东路]]接[[西藏中路|西藏中路]]，南到[[黄浦江|黄浦江]]，地底設有[[上海轨道交通八号线|轨道交通八号线]]车站，全长3594米。<ref>[http://shtong.gov.cn/newsite/node2/node2245/node68289/node68294/node68312/node68325/userobject1ai65726.html 专业志 >> 上海市政工程志 >> 第一篇市区道路 >> 第二章主要干道 >> 第二节　南北向主要干道]上海地方志办公室</ref>该路通过其南端的[[西藏南路隧道|西藏南路过江隧道]]通往浦东，[[2010年上海世博会|2010年上海世博会]]大门亦设于该路南端。

== 历史 ==
西藏南路的北段原为[[上海县城|上海县城]]西侧的一条小河，名为周泾，南面到[[老西门|老西门]]附近与东西向的[[肇嘉浜|肇嘉浜]]（今[[复兴东路_(上海)|复兴东路]]）相通，北面在今[[大世界|大世界]]处与东西向的[[洋泾浜|洋泾浜]]（今[[延安东路_(上海)|延安东路]]）相通。19世纪后期，为[[上海法租界|上海法租界]]的西部界限。1900年法租界越过周泾向西扩展。20世纪初，上海法租界公董局填埋周泾，修筑成宽阔的马路，以法国首任驻沪领事[[敏体尼|敏体尼]]命名为'''敏体尼荫路'''。
