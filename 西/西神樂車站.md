'''西神樂車站'''（{{jpn|j=西神楽駅|hg=にしかぐらえき|rm=Nishi-kagura eki|f=y}}）是一由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）所經營的[[鐵路車站|鐵路車站]]，位於[[日本|日本]][[北海道|北海道]][[旭川市|旭川市]]境內，是[[地方交通線|地方交通線]][[富良野線|富良野線]]沿線一個無人駐守的車站，由[[北海道旅客鐵道旭川支社|JR北海道旭川支社]]負責管轄。

「[[神樂|神樂]]」來自對[[阿伊努語|阿伊努語]]的「hetce-us-i」（演出祭祀歌舞的地方）的意譯<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/3140P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-16|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 車站構造 ==
西神樂車站內設有一對[[對向式月台|對向式月台]]兩條乘車線，兩月台間以[[人行天橋|跨線天橋]]連結。車站採無人管理方式經營，因此設有[[自動售票機|自動售票機]]提供必要的票務服務。

== 車站周邊 ==
車站周圍有一個小規模的商店街，聚集了許多附近的主要設施。

* {{Link-ja|旭川東警署|旭川東警察署}}西神樂派出所
* 西神樂郵局
* {{Link-ja|旭川信用金庫|旭川信用金庫}}西神樂分店西神樂特別辦事處

* {{Link-ja|道北巴士|道北バス}}「西神樂站前」巴士站

== 歷史 ==
* 1899年（[[明治|明治]]32年）9月1日 - [[北海道官設鐵道|北海道官設鐵道]]十勝線'''邊別車站'''（{{lang|ja|辺別駅}}）開始營運。一般車站。
* 1905年（明治38年）4月1日 - 由[[日本國有鐵道|日本國有鐵道]]接管。
* 1942年（[[昭和|昭和]]17年）10月1日 - 改稱為'''西神樂站'''。
* 1982年（昭和58年）11月15日 - 取消貨物裝卸。
* 1984年（昭和60年）2月1日 - 取消行李處理。
* 時間不詳 - 車站簡易委託化。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道繼承業務。
* 1989年（昭和64年） - 車站改建。
* 1999年（[[平成|平成]]11年） - 車站無人化。
