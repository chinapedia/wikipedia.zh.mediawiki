'''西藏圍棋'''，藏語音譯為'''哲'''，'''[[密芒|密芒]]'''的一種，又名'''藏式圍棋'''，流傳於西藏地區的傳統棋類。特點是仍保留古代的[[座子制|座子制]]，並且因為不可下在剛被提吃的所有棋點。

'''密芒'''是指棋，非專指西藏圍棋，也有其他西藏棋類<ref>{{Cite web |url=http://sc157.com/book/xiburenwen/2011-01-18/1603.html |title=朵康牧区几种常见棋 |accessdate=2011-03-18 |archiveurl=https://web.archive.org/web/20160304093112/http://sc157.com/book/xiburenwen/2011-01-18/1603.html |archivedate=2016-03-04 |deadurl=yes }}</ref>，如[[排棋|排棋]]<ref name="古老的藏式围棋">[http://www.tibetinfor.com.cn/zt/zt2002002101793730.htm 古老的藏式围棋 ] {{webarchive|url=https://web.archive.org/web/20061123114914/http://www.tibetinfor.com.cn/zt/zt2002002101793730.htm |date=2006-11-23 }}</ref>。

据敦煌莫高窟文献中记载，[[松赞干布|松赞干布]]的父亲[[囊日論贊|囊日論贊]]擅长下西藏围棋，而且棋艺非常出色。“每与人弈棋，当一局中其对手行将败北时，即帮对手，令其取胜”<ref name="古老的藏式围棋" />。相傳[[松贊干布|松贊干布]]的大臣[[瓊波·邦色|瓊波·邦色]]也是一位圍棋高手。根據現代學者研究，西藏圍棋起源於中國內地的[[圍棋|圍棋]]，中國內地的圍棋最遲在[[唐朝|唐朝]]初期傳到了西藏，並逐漸發展為西藏圍棋。<ref>[http://www.lw23.com/pdf_360a8d4b-e55e-48d8-8dc4-0012d191c36e/lunwen.pdf 藏棋起源之初步探討]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 

[[五世達賴|五世達賴]]圆寂，[[桑結嘉錯|桑結嘉錯]]与[[拉藏汗|拉藏汗]]兩人矛盾日趋激烈，因双方認為武力征战将会两败俱伤，在权衡利弊後，双方决定以比赛西藏圍棋论定输赢，经过三盤的棋艺较量后，桑杰嘉措輸棋，且于1705年被拉藏汗杀死。此后藏族认为西藏围棋是不祥之物，西藏围棋因此渐渐失传<ref>{{Cite web |url=http://www.sc157.com/bookzs/cangdishihua/2011-01-18/1669.html |title=藏族棋文化  |access-date=2011-03-17 |archive-url=https://web.archive.org/web/20110713165040/http://www.sc157.com/bookzs/cangdishihua/2011-01-18/1669.html |archive-date=2011-07-13 |dead-url=yes }}</ref>。

十九世紀中葉，藏族天文历算家[[丹巴加措|丹巴加措]]曾有一本关于围棋棋艺的专著《密芒吉单居》（《藏棋之理论》），共三十二頁<ref name="古老的藏式围棋" />。

1958年11月，當時為[[锡金|锡金]]王子的[[巴登顿珠郎加|巴登土登郎加]]访日时和日本棋手{{link-ja|伊予本桃市|伊予本桃市}}对弈西藏圍棋<ref>[http://www.xz.xinhuanet.com/old/xinwenss/lueying/lueying20030103001.htm 密芒围棋]{{dead link}}</ref>。

==規則==
<ref>{{cite web |title=Tibetan Go at Sensei's Library |url=http://senseis.xmp.net/?TibetanGo |website=senseis.xmp.net |accessdate=2019-04-10 |language=en}}</ref><ref>{{cite web |author1=扎拉·达娃桑布 |author2=扎雅·洛桑普赤 |author3=当增扎西 |title=藏族棋文化的见证——关于在强巴敏久林宫遗址发现的藏族石刻棋盘的学术报告 |url=http://www.tibet.cn/periodical/xzyj/2007/02/200805/t20080523_395856.html |website=www.tibet.cn |publisher=西藏研究 |accessdate=2019-04-10 |date=2007-04-21}}</ref>
*雙方先放各六枚座子，方開始行棋。
**白方座子座標：c3、c11、g15、l3、p7、p15。
**黑方座子座標：c7、c15、g3、l15、p3、p11。
**白方的一一角為右下、左上；黑方的為右上、左下。

*雙方輪流下一子，白色先。

*不可下在剛被提吃的所有棋點。

*如果以棋子佔領或圍繞對方兩個一一角，則另得二十目。

*如果以棋子佔領或圍繞天元點，則另得五目。

*其他規則大致與中國古圍棋相同。

==術語==
<ref>{{Cite web |url=http://www.sc157.com/bookzs/cangdishihua/2011-01-18/1669.html |title=藏族棋文化  |access-date=2011-03-17 |archive-url=https://web.archive.org/web/20110713165040/http://www.sc157.com/bookzs/cangdishihua/2011-01-18/1669.html |archive-date=2011-07-13 |dead-url=yes }}</ref>
*贡：天元。
*波：座子。
*具：棋子。
*姜具：駿馬棋，指下好棋。
*旁具：毛驢棋，指下壞棋。
*甲恰：讓子。
*则杰贡杰：天元、兩個一一角都被奪走，而輸掉共二十五目。
*达拉夏瓦杰恰：八隻鹿，表示用八枚棋子在边线处组成兩个眼。
*则拉年莫珠恰：六條魚，表示用六枚棋子在角处组成兩个眼。
*密芒迪吾尼须拉，朗钦达密扫噶乃：彪悍大象，表示兩枚棋子並連。
