'''西安城墙'''是位于[[陕西省|陕西省]][[西安市|西安市]]城区的[[城墙|城墙]]。西安城墙自[[明朝|明朝]]初年在[[唐长安城|唐长安城]]皇城的基础上建造起来，后来又经过多次修补，形成严密的防御体系，是中国现存最完整的古[[城墙|城墙]]之一，与平遥城墙、[[湖北|湖北]][[荆州|荆州]][[荆州城墙|城墙]]、[[辽宁|辽宁]][[兴城|兴城]][[兴城城墙|城墙]]等并列为中国现存最完好的四座古城墙，也是规模最大的古[[城墙|城墙]]之一（最大为[[南京城墙|南京城墙]]），周长13.74公里，城墙周围有[[西安护城河|护城河]]环绕，是中世纪后期最著名的城垣建筑之一。

1961年，西安城墙被[[中华人民共和国国务院|中华人民共和国国务院]]公布为第一批[[全国重点文物保护单位|全国重点文物保护单位]]之一。

== 历史 ==
===唐代===
[[File:Hanguang_Gate_Relics_Mesuem.JPG|thumb]]
今天西安城垣修筑始于明初，西、南两面以唐长安皇城城墙为基础修建。唐末，904年，佑国军节度使[[韩建|韩建]]对长安城进行改建。为便于防守，放弃外城郭与宫城，将皇城加以整修。<ref>{{cite journal |author = 吴永江|date =1980-08 |title = 西安城墙|journal =文物 |volume =08 |issue = |pages =  |issn =  |language =zh-Hans |format = |accessdate = 2014年5月7日|laysource = |laydate = |quote = }}</ref>

===宋元===
唐代以后，宋元两朝中城市的名称城和建制经历数次变更，但城墙规模却无变化。<ref>{{Cite web |url= http://www.xacitywall.com/index.php?m=content&c=index&a=show&catid=73&id=58 |title= 明前的城墙 |author=  |date=  |publisher= 西安城墙景区 |language= zh-Hans |accessdate= 2014年5月3日 |quote=  }}</ref>

===明代===
[[File:Xian_city_wall.jpg|thumb]]
早在明王朝建立前，当朱元璋攻克[[徽州|徽州]]后，一个名叫朱升的隐士便告诉他应该“高筑墙，广积粮，缓称王”。朱元璋采纳了这些建议。当全国统一后，他便命令各府县普遍筑城。[[明朝|明朝]]初年，[[朱元璋|朱元璋]]认为“天下山川，惟秦中号为险固”，因此将次子[[朱樉|朱樉]]封为秦王，就藩西安府。并诏令西安府长兴侯[[耿炳文|耿炳文]]和都指挥[[朴英|朴英]]修整西安城，[[洪武|洪武]]七年（1374年）起，开始兴建西安城墙。<ref name=xian-city-wall>曹静. 西安城墙. 文博:1986,05:86.</ref>

城墙最底一层用石灰、土与糯米汁混合夯打而成，上层使用黄土分层夯打。城墙修建时，西、南两面在原[[唐长安城|唐长安城]]皇城的基础上增修加长而成，东墙和北墙进行了新建。<ref name=xian-city-wall/>整个建设历时八年，到洪武十一年（1378年）基本完工。

此后[[隆庆|隆庆]]二年（1568年），陕西巡抚[[张社|张社]]在城墙外壁和顶部筑垒青砖以改变黄土结构。<ref name=xian-city-wall/>明末[[崇祯|崇祯]]九年（1636年），[[陕西巡抚|陕西巡抚]][[孙传庭|孙传庭]]为防备李自成进攻，修建了四门的“瓮城”。<ref name=xian-city-wall/>崇祯十六年（1643年），[[李自成|李自成]]攻克西安，从[[长乐门|长乐门]]进入西安，后将长乐门焚毁。<ref>{{Cite web |url= http://www.xacitywall.com/index.php?m=content&c=index&a=show&catid=75&id=143 |title= 长乐门 |author=  |date=  |publisher= 西安城墙景区 |language= zh-Hans |accessdate= 2014年5月4日 |quote=  }}</ref>次年（1644年）在此称帝，国号[[大顺_(政权)|大顺]]。

===清代===
[[File:Xi'an_walls.jpg|230px]]
[[清|清]]代基本沿袭明制，对西安城墙进行过十二次补修，其中规模最大的是[[乾隆|乾隆]]四十六年（1781年）的工程，铺设海墁，并增修排水道，统一城墙宽度：将墙根向外延伸，城顶向内收缩城墙剖面呈现梯形，增强了西安城的防御能力。<ref name=xian-city-wall/>

===民国时期===
1911年，[[辛亥革命|辛亥革命]]爆发，新军从城内攻占西安，[[安远门|安远门]]是当时清军的弹药库，城楼中炮燃烧使清兵全线溃败，也使安远门毁于战火。<ref>{{Cite web |url= http://www.sxdaily.com.cn/n/2014/0430/c335-5421081-2.html |title= 千年城百年情 西安城墙的传奇故事【2】 |author=  |date=  2014年4月30日 |publisher= 陕西传媒网 |language= zh-Hans |accessdate= 2014年5月3日 |quote=  }}</ref>民国初年，城墙的四座闸楼及其墙段被拆除。永宁门箭楼毁于1926年[[二虎守长安|二虎守长安]]时期，[[刘镇华|刘镇华]]的[[镇嵩军|镇嵩军]]的攻击。<ref name=ROC>{{Cite web |url= http://www.xacitywall.com/index.php?m=content&c=index&a=show&catid=73&id=102 |title= 民国时期的城墙 |author=  |date=  |publisher= 西安城墙景区 |language= zh-Hans |accessdate= 2014年5月3日 |quote=  }}</ref>

[[File:XiAn_CityWall_SouthGate3.jpg|thumb]]
1936年[[陇海铁路|陇海铁路]]通车，修建[[西安站|西安站]]，修建中正门（后改称解放门），后在1952年扩建火车站广场拆除城墙<ref>{{Cite web |url= http://www.xacitywall.com/index.php?m=content&c=index&a=show&catid=75&id=138 |title= 解放门 |author=  |date=  |publisher= 西安城墙景区 |language= zh-Hans |accessdate= 2014年5月5日 |quote=  }}</ref>直到2004年12月26日，火车站前城墙才完成合拢。<ref>{{Cite web |url= http://www.lifeweek.com.cn/2005/0119/10736.shtml |title= 西安,只有城墙的古城 |author= 马戎戎 |date= 2005-01-19  |publisher= 三联生活 |language= zh-Hans |accessdate= 2014年5月4日 |quote=  }}</ref>

在[[抗日战争|抗日战争]]时期，西安城墙也被作为防空洞使用。<ref>杨邵武. 西安城墙. 陕西档案,2009(2):48.</ref>

===中华人民共和国时期===
[[File:Xi'an_-_City_wall_-_004.jpg|thumb]]
20世纪50年代初，苏联援华专家因为多家大型军工企业建设的需要，主张“拆掉城墙，发展更多的道路，解决当时的交通问题”。而规划专家主张保留西安老城格局，工业区放在东西郊区。而最终老干部主张城墙内挖防空洞可以减少空袭带来的损失，同时“城墙有利于防原子弹、防地面冲击波”，因此西安城墙得以保留。<ref name=chinaweekly>{{Cite web |url= http://www.chinaweekly.cn/bencandy.php?fid=63&id=6052 |title= 幸运儿西安城墙 |author= 刘畅 |date= 2012-05-01 |publisher= 中国周刊 |language= zh-Hans |accessdate= 2014年5月3日 |quote=  }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>1954年，北京[[地安门|地安门]]已经被拆除，外城的拆除工作也在进展中。在此风潮下，部分陕西当地领导提出“拆除封建王朝的陈墙旧砖，矗起一个新社会新城市”，西安市民也开始拔除城墙砖。<ref name=chinaweekly/>时任[[陕西省省长|陕西省省长]][[赵寿山|赵寿山]]因随杨虎城参加过“[[二虎守长安|二虎守长安]]”时期守卫西安的战斗，反对拆掉城墙，他的意见受到[[周恩来|周恩来]]、[[陈毅|陈毅]]的重视，从而赵寿山召开会议强调“西安是我国历代名城，也是我们陕西人的骄傲，古城墙是文物古迹，我们一定保护好城墙，把西安建设成为一座现代化的文明城市。”<ref name=chinaweekly/>

1958年1月，[[毛泽东|毛泽东]]在第十四次最高国务会议上表示“南京、济南、长沙的城墙拆了很好，北京、开封的旧房子最好全部变成新房子”。两个月后在[[成都|成都]]会议上再次指出“拆除城墙，北京应当向天津和上海看齐。”<ref name=chinaweekly/>1958年6月17日，西安市根据毛泽东的讲话精神召开了关于城墙存废的座谈会，会议上主张拆除的意见压倒了以时任陕西省文化局副局长[[武伯纶|武伯纶]]为代表的“保留派”，后武伯纶与文物界的前辈越级向国务院上书，要求保留西安城墙并得到[[习仲勋|习仲勋]]等人的重视。<ref name=chinaweekly/>在1961年，西安城墙被列入首批[[全国重点文物保护单位|全国重点文物保护单位]]。而后，习仲勋因下令“不许拆西安城墙”等原因被革除职务，下放[[河南|河南]]。<ref name=chinaweekly/>武伯纶也因此被列为“反动派”、“学术权威”和“叛徒”。<ref name=chinaweekly/>

[[文化大革命|文化大革命]]期间，因为城墙过于宏大，[[红卫兵|红卫兵]]又缺乏大型工程设备，因此城墙没有被推倒。<ref name=chinaweekly/>70年代初，因与苏联的战争风险，在城墙上挖筑了41千米长的坑洞，并在城墙内修建了仓库。<ref name=chinaweekly/>后西安市城防工程指挥部要求拆掉城墙，效仿北京修建[[西安地铁|地铁]]，陕西省委同意了该计划，但因所需资金浩大而搁置。<ref name=chinaweekly/>

文化大革命结束后，房地局与交通局仍主张拆掉城墙，以解决住房用地与交通问题，但在1981年，时任[[中共中央书记处书记|中共中央书记处书记]]的习仲勋指示要求对西安城墙认真保护。<ref name=chinaweekly/>

1979年，西安城墙管理所成立，开始有计划地对城墙进行修缮，对安定门进行了油漆彩绘，并整修了安远门、永宁门与长乐门。<ref name=xian-city-wall/>1983年2月，成立了西安市城墙建设委员会。环绕城墙设立了[[环城公园|环城公园]]，而当时西安城墙仅存城楼和箭楼六座，以及十四处豁口。<ref name=xian-city-wall/><ref name=ROC/>

1992年西安市政府主要领导在访问日本奈良市时提出了在古城墙上举行国际马拉松赛的构想<ref>{{cite news|author1=黄群|title=我见证西安城墙国际马拉松赛|url=http://tech.gmw.cn/newspaper/2015-11/26/content_109980453.htm|accessdate=2015-12-24|agency=西安晚报|publisher=光明网|date=2015-11-26|language= zh-Hans}}</ref>。1993年11月7日自第一届马拉松赛举办后起，西安每年都举行西安城墙国际马拉松赛，比赛在城墙上进行。<ref>{{Cite web |url= http://sports.sohu.com/20131103/n389455629.shtml |title= 西安城墙国际马拉松赛昨开赛 2600余名选手角逐 |author=  |date= 2013年11月3日 |publisher= 搜狐体育 |language= zh-Hans |accessdate= 2014年5月4日 |quote=  }}</ref>

2006年7月25日，西安城墙永宁门至文昌门段近30米长的内侧城墙发生坍塌。经调查，城墙墙顶的海墁出现沉降裂缝，而城墙墙体夯土土质不密实，连日雨水灌进墙体导致墙体坍塌<ref>{{Cite web |url= http://news.hsw.cn/gb/news/2006-07/26/content_5606654.htm |title= 30米城墙昨日突然坍塌 事发南门至文昌门段(图) |author=  |date= 2006-07-26 |publisher= 华商网 |language= zh-Hans |accessdate= 2014-03-31 |quote=  }}</ref>。城墙管委会表示：“出现墙体塌陷是城墙加固过程中，工程方面对土层进行的有控制的塌陷，城墙的整体是安全的。”<ref>{{Cite web |url= http://tech.sina.com.cn/d/2006-07-27/10511058048.shtml |title= 西安古城墙倒塌管委会解释是维修措施 |author=  |date= 2006-07-27 |publisher= 新浪网 |language= zh-Hans |accessdate= 2014-03-31 |quote=  }}</ref>

2014年7月，西安城墙建起电梯，引发质疑。后来西安市文物局对西安城墙文化投资发展有限公司作出“责令改正违法行为并罚款30万元”的行政处罚决定。当日，该公司连夜拆除违建电梯，修复城垛，恢复了永宁门原貌。2014年11月罚款到位<ref>{{Cite web |url =http://www.guancha.cn/society/2017_04_09_402686.shtml  |title = 开国上将刘亚楼故居被拆后，谁栽了？ |author =  |date =2017-04-09  |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。

== 城墙结构 ==
{{wide image|1 xian city wall 2011.jpg|800px|<center>明西安府城墙一角</center>}}

===周长===
对西安城墙的长度说法不一，目前测量结果是周长13.74公里。东墙长2683米，西墙长2626米，南墙长4194米，北墙长4134米。<ref>{{Cite book | author = 俞茂宏,王源,俞涛,Daniel Giebener | title = 西安古城墙和钟鼓楼：历史、艺术和科学 | location =西安 | publisher = 西安交通大学出版社 | date = 2009年12月 | pages =  |ISBN = 9787560530604 | accessdate = 2014年5月4日 | url =  | language= zh-Hans| quote =  }}</ref><sup>:4</sup>城内面积约12平方公里，为唐长安城的七分之一。

在雍正年间的《陕西通志》中记载西安城墙周长四十里，但对此有不同理解。<ref name=numbers>吴永江.关于西安城墙某些数据的考释. 文博,1986(12):88.</ref>1963年3月，陕西省与西安市文管会合编的《西安城墙》中提到光绪19年测得西安城墙周长4390丈，折合为14088米。<ref name=numbers/>《续陕西省通志稿》中测得城墙周长为4302丈，折合13766米。<ref name=numbers/>

===体系===
[[File:Xian_wall_2006.JPG|thumb]]
城垣高12米，底宽15-18米，顶宽12-14米。墙体最底层用石灰、土和糯米混合（三合土）夯打而成，其上用黄土分层夯筑，每层厚8-12厘米，外壁包砌多层城砖。墙顶使用三层青砖铺设海墁，每隔40-60米，就有一组砖砌的排水口。城墙外沿雉堞有垛口5894个，供射击和瞭望，内沿女墙无垛口，防止跌落。墙的外壁筑98座[[敌台_(城墙)|敌台]]，延伸出墙12米，宽20米，高与城齐，各自相距120米，恰在弓箭和火铳的射程范围内。敌台上筑敌楼，可驻士兵和储存物资。城四隅各有一座角楼，除西南隅沿袭了[[元|元]]代建筑为圆形外，其余三座均为方形。城墙的内侧有十条登城马道，其中四座城门处各有一条。城墙的外侧有[[西安护城河|护城河]]，宽20米，深10米，终年积水，和城墙共同构成完整的防御体系。

西安城有东西南北四座城门，分别名长乐门、安定门、永宁门和安远门，门外均筑[[瓮城|瓮城]]。每座门分为三重三楼，由外向内，分别是闸楼、箭楼和正楼。除南门箭楼外，其余各楼下都设拱形门洞，门洞高、宽各6米，深19.5米。正楼为重楼，面阔七间，进深二间，高36米，三层檐歇山顶，周围有回廊。箭楼面阔十一间，进深二间，高33.4米，单檐歇山顶。箭楼的正面设四层箭窗，每层12孔，左右两面各三层，每层3孔，利于射击。

西安城墙的防御能力直到近代仍有体现，如在[[二虎守长安|二虎守长安]]之时，城墙就发挥很大的防守作用。

=== 城门 ===
{{主条目|西安城门}}
[[File:西安城墙各城门平面图示.png|thumb]]

目前西安城墙共18座门，其筑门历史大体可分为四个阶段：

* 明代始建，沿袭至今的4座城门：[[永宁门|永宁门]]、[[安远门|安远门]]、[[长乐门|长乐门]]、[[安定门|安定门]]。

* 民国初年，为适应交通需要，新辟诸城门：[[中山门|中山门]]、[[玉祥门|玉祥门]]、[[解放门|解放门]]（原名中正门）。

* 抗日战争时期，临时开辟的诸防空便门，后经改建成为正式城门：[[勿幕门|勿幕门]]（原四府街防空便门）、[[建国门|建国门]]（原小差市防空便门）、[[朝阳门|朝阳门]]（原崇礼路防空便门）、[[尚武门|尚武门]]（原西北三路防空便门）。

* 中华人民共和国共和国成立后，新辟、修葺的诸城门：[[和平门|和平门]]、[[文昌门|文昌门]]、[[朱雀门|朱雀门]]、[[尚德门|尚德门]]、[[尚勤门|尚勤门]]、[[尚俭门|尚俭门]]、[[含光门|含光门]]。2004年12月26日，随着解放门建成，西安城墙实现了全面贯通，18座城门的格局至此确定。

18座城门分布，'''南7门'''、'''北6门'''、'''西2门'''、'''东3门''<small>'''（明代始建：明；民国始建：民；共和国始建：共）''</small>：

* '''南城墙7门'''<small>（东→西）</small>：[[建国门_(西安)|建国门]]<small>（民）</small>、[[和平门_(西安)|和平门]]<small>（共）</small>、[[文昌门|文昌门]]<small>（共）</small>、'''[[永宁门|永宁门]]'''<small>（明）</small>、[[朱雀门|朱雀门]]<small>（共）</small>、[[勿幕门|勿幕门]]<small>（民）</small>、[[含光门|含光门]]<small>（共）</small>。

* '''北城墙6门'''<small>（东→西）</small>：[[尚勤门|尚勤门]]<small>（共）</small>、[[尚俭门|尚俭门]]<small>（共）</small>、[[解放门_(西安)|解放门]]<small>（民）</small>、[[尚德门|尚德门]]<small>（共）</small>、'''[[安远门|安远门]]'''<small>（明）</small>、[[尚武门|尚武门]]<small>（民）</small>。

* '''西城墙2门'''<small>（南→北）</small>： '''[[安定门_(西安)|安定门]]'''<small>（明）</small>、[[玉祥门|玉祥门]]<small>（民）</small>。

* '''东城墙3门'''<small>（南→北）</small>：'''[[长乐门|长乐门]]'''<small>（明）</small>、[[中山门_(西安)|中山门]]<small>（民）</small>、[[朝阳门_(西安)|朝阳门]]<small>（民）</small>。

===含光门遗址博物馆===
