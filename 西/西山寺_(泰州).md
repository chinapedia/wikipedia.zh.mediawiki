'''泰州西山寺'''又名'''西山白云寺'''，位于今[[中国|中国]][[江苏省|江苏省]][[泰州市|泰州市]][[海陵区|海陵区]]迎春西路36号、[[江苏省泰州中学|泰州中学]]老校区院内东南侧。该寺始建于南宋，传最初名报国庵，明崇祯五年（1632）重建后改名西山报国禅寺，清咸丰时扩建并改为今名<ref>{{cite web |url=http://sz.tznews.cn/tzwb/html/2012-01/08/content_13126.htm|title=《泰州名刹-晨钟暮鼓韵犹识》|accessdate=2013-08-08|work=李金明|publisher=泰州海陵区政府网}}</ref>，其主体为西路的前殿（天王殿）和[[大雄宝殿|大雄宝殿]]，均为面阔五间的[[硬山顶|硬山顶]]建筑，东路为附属建筑两进，均为面阔三间的硬山顶建筑。

该寺于1939年前后被用作中国国民党鲁苏皖边区游击总指挥部驻地，由[[李明扬|李明扬]]任总指挥<ref>{{cite web |url=http://sz.tznews.cn/tzwb/html/2012-01/08/content_13126.htm|title=《李明扬与明扬路》|accessdate=2013-08-08|work=李金明|publisher=泰州晚报}}</ref>。1939年8月、12月和1940年春，时任新四军军分会书记、江南指挥部指挥员的[[陈毅|陈毅]]曾三进泰州，与李明扬谈判，并成功说服其让道使新四军东进黄桥，进入苏北地区发展。建筑群在1949年后曾作为泰州中学图书馆，今为新四军东进泰州谈判纪念馆，现状为2000年修缮，其中立有吴为山所塑的陈毅塑像<ref>{{cite web |url=http://www.taizhou.gov.cn/gtb/index.jsp?url=http%3A%2F%2Fwww.taizhou.gov.cn%2Fcol%2Fcol645%2Findex.html|title=《新四军东进泰州谈判纪念馆》|accessdate=2013-08-08|work=|publisher=泰州政府网}}</ref>。
