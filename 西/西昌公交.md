'''西昌公交'''是[[四川省|四川省]][[凉山彝族自治州|涼山彝族自治州]][[西昌市|西昌市]]的已被合併的[[公共汽車|公共汽車]]系統，合併前覆蓋西昌市城區以及[[小庙乡|小廟鄉]]，[[四合乡_(西昌市)|四合鄉]]，[[高枧乡_(西昌市)|高梘鄉]]，[[川兴镇|川興鎮]]，[[西郊乡_(西昌市)|西郊鄉]]等周邊鄉鎮。擁有三个客运总站、6个机关科室、1个全资子公司（正清公交出租车公司），拥有三个综合（袁家山车场、川兴车场和胜利北路车场)，场站总面积达19300平方米;目前公司拥有员工969人，66辆出租车，311辆营运公交车，23条固定客运线路，年客运量5750万人次，年客运里程1203万公里。<ref name=":0">{{Cite web|url=http://mp.weixin.qq.com/s/8kJ3F9rpb3yyXDptzNmDSQ|title=西昌市公共交通公司简介|accessdate=2017-06-19|author=|date=|publisher=}}</ref>

== 運營公司 ==
[[File:西昌公交22路車輛.jpg|缩略图]]
西昌公交由西昌市公共交通公司運營。

西昌市公共交通公司创建于1966年10月，是涼山州唯一一家國有城市公交企業。<ref name=":0" />

== 歷史 ==
* 1966年西昌公共汽車站開闢西昌-小廟，西昌-新村客運路線。

* 1976-1977年西昌地區公共交通公司曾在[[德昌县|德昌縣]]境內經營德昌-錦川，德昌-六所，德昌-黃水條客運路線。
* 1988年前，客運路線為十條，計有西昌至大橋，月華，黃水，大箐，馬道，火車站（西昌站），河西，果園橋，經久，川興。1990年，經營的客運路線為9條，單程132.5公里，每日公共汽車197次（冬季為209次）<ref>{{Cite book|title=西昌市志|last=|first=|publisher=四川人民出版社|year=1990|isbn=9787514401127|location=|pages=}}</ref>

== 合併 ==
2017年7月1日，西昌公交與匯通公交合併，成立新公司[[月城公交|月城公交]]。至此，西昌公交走入歷史。<ref>{{Cite web|url=http://www.cnxichang.cn/Item/81170.aspx|title=7月1日起，西昌月城公共交通公司正式挂牌营运-中国·西昌门户网站|accessdate=2018-02-01|work=www.cnxichang.cn}}</ref>
