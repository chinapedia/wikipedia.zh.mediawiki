《'''西厢记'''》是[[中国|中国]]《[[六才子書|六才子書]]》之一。最早取材于[[唐代|唐代]][[诗人|诗人]][[元稹|元稹]]所写的传奇《[[鶯鶯傳|会真记]]》（又名《莺莺传》），后被元代[[王实甫|王实甫]]改编为[[杂剧|杂剧]]（此前有[[宋朝|宋]]人[[趙令畤|趙令畤]]以此題材作《商調[[蝶戀花|蝶戀花]][[鼓子詞|鼓子詞]]》，[[金朝|金]]人[[董解元|董解元]]作《[[西廂記諸宮調|西廂記諸宮調]]》，然影響力均遠不如雜劇《西廂記》），被稱為「[[元杂剧|元杂剧]]的壓卷之作」，對中国的[[汉语|语言]]、[[中国文化|文化]]等各个方面皆頗有影响。

== 历史沿革 ==
西厢记的故事题材最早来自[[唐代|唐代]][[唐诗|诗人]][[元稹|元稹]]所写的传奇《[[鶯鶯傳|会真记]]》（又名《莺莺传》），描写他自己“以张生自寓，述其亲历之境”。讲述他在“有僧舍曰普救寺”中，和一美丽女子“天之所命尤物”名“莺莺”邂逅，认为自己“善补过”，“智者不为，为之者不惑”。

唐代以后，这个[[爱情故事|爱情故事]]的结局，令许多人感到遗憾和不满，斥责张生为“薄情年少如飞絮”。逐渐在民间流传并将结局改变，宋代有趙令畤所作商調蝶戀花鼓子詞，其情節基本同于《鶯鶯傳》，但對張生的態度變爲譴責。宋代以后，由于北方游牧民族不断入侵和[[汉族|汉族]]同化，封建礼法观念在普通人民中间逐渐淡化，[[金朝|金代]]出现了[[董解元|董解元]](良)（一说为[[董解元|董琅]]）所写的《[[西厢记諸宮調|西厢记諸宮調]]》，[[諸宮調|諸宮調]]是当时的一种[[说唱|说唱]][[艺术|艺术]]，类似现代的[[评弹|评弹]]，用[[琵琶|琵琶]]和[[筝|筝]]伴奏，边说边唱。这本《西厢记》将内容大为增加，加入许多人物和场景，最后结局改为张生和莺莺不顾老夫人之命，求助於白马将军，由其做主[[完婚|完婚]]。

[[元朝|元代]]时[[王实甫|王实甫]]基本根据这部诸宫调将《西厢记》改编成多人演出的[[戏剧|戏剧]]剧本，使故事情节更加紧凑，融合了古典诗词，[[文学|文学]]性大大提高，但将结尾改成老夫人妥协，答应其婚事，大团圆结局。这部剧本作者说法不一，有人说是[[关汉卿|关汉卿]]所作<ref>《乐府群珠》卷四有关汉卿作的总题为《崔张十六事》的《普天乐》小令十六支</ref>，也有人说是关作王续，或王作关续<ref>[[王世貞|王世貞]]在《藝苑卮言》中說：「《西廂》久傳為關漢卿撰，邇來乃有以為王實夫者，謂郵亭夢而止；又云：至『碧雲天，黃花地』而止，此後乃關漢卿所補也。」[[梁廷枏|梁廷枏]]《曲話》稱：「世傳實甫作《西廂》至『碧雲天，黃花地，西風緊，北雁南飛』，構想甚苦，思竭，撲地遂死」。清初[[金圣叹|金圣叹]]批本《第六才子书》流行，“王作关续”说幾成定论。</ref>，王作無名氏續，但认为是王实甫所作的说法比较公认<ref>[[钟嗣成|钟嗣成]]的《录鬼薄》认为是王实甫，明初[[朱权|朱权]]以及[[王世贞|王世贞]]亦持同样看法。</ref>。

王實甫原作使用[[北曲|北曲]]，也稱為《北西廂》。[[李景雲|李景雲]]有传奇戏曲《崔莺莺西厢记》，已失传。[[崔時佩|崔時佩]]、[[李日华|李日华]]著有《[[南西厢记|南西厢记]]》，人物情节与《北西厢》相同，曲文亦多沿襲王實甫《西廂記》。[[陸采|陸采]]對李日華《南西廂記》不滿意，重編《北西廂》為《南西廂曲》。

== 故事梗概 ==

在[[山西|山西]][[普救寺|普救寺]]借宿的书生张珙（字君瑞），偶遇扶柩回乡在寺中西厢借住的原崔[[相国|相国]]的女儿崔莺莺，由于互相吟诗而产生爱慕。叛將孫飛虎帶手下慕名围寺，要强抢崔莺莺，三日之内若不交出莺莺，“[[伽蓝|伽蓝]]尽皆焚烧，僧俗[[寸斩|寸斩]]，不留一个”。莺莺的母亲老夫人鄭氏宣称谁能救他女儿就将女儿许配他，张生向他一位故旧“白马将军”蒲州杜太守写了一封求救信，由一位僧人（惠明）突出包围送出，杜太守发兵解围。过后老夫人因门第不当悔婚，只是贈金並让莺莺拜张生为义兄以谢搭救。张生在悲慟之下患病，莺莺也大為傷痛，後來在莺莺的丫鬟[[红娘|红娘]]的帮助下，两人暗通书信，並最終成功幽會。最后私情被老夫人发现，欲責罰二人，但由于红娘据理力争，无可奈何之下，老夫人命令张生上京赶考，如能蟾宮折桂便真的把莺莺许配與他，於是张生進京赴試，考中并回来迎娶莺莺，有情人終成眷屬。

==藝術成就==
《西廂記》有人舉之為元代最佳[[雜劇|雜劇]]，劇情曲折，善於塑造人物，能因應不同人物的身分寫出不同的曲詞或[[賓白|賓白]]，突出地展示人物的心靈活動。《西廂記》又善於描摹景物，醞釀氣氛，襯托人物的內心活動。

《西廂記》打破元[[雜劇|雜劇]]一人獨唱的成規，改為可由多人對唱。元劇慣例一本四折，間或一本五、六折，《西廂記》則全劇共五本廿一折，鋪敘組織之功，突過前人。《西廂記》融化古詩詞中優美的詞句，提煉民間生動活潑的[[口語|口語]]，熔鑄成自然華美的曲詞，文辭優美清麗。

== 文化影响 ==
《西厢记》剧本完成后，迅速流行，几乎[[中国|中国]]所有几百个剧种都以其为原本上演过这部戏，以后的许多著作都提过这部剧本，《[[红楼梦|红楼梦]]》中的主人公贾宝玉和林黛玉都引用过这部剧本中的原词“银样鑞枪头”，“若共你多情小姐同鸳帐，怎舍得叠被铺床”等，但都表现出是市间流传，大家公子不允许读的态度。《西厢记》中许多人物都是民间耳熟能详的，“[[红娘|红娘]]”更成为[[汉语|汉语]]语言中“媒人”的代名词，甚至成功的中介都被称为“红娘”。以后有许多故事和剧本受其影响，开始表现基于爱情，冲破“门当户对”的封建禮教观念的美满婚姻或悲剧，如《[[梁山伯与祝英台|梁山伯与祝英台]]》《[[牛郎织女|牛郎织女]]》《[[天仙配|天仙配]]》《[[宝莲灯|宝莲灯]]》等。《西厢记》可以说是首开先河，在世界上是第一部表现纯爱情的长篇作品，藝術成就甚高。
