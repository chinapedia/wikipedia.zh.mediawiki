[[File:Mapaespaña4-1fr.png|360px]]
'''西班牙省份'''（{{lang-es|Provincia de España}}）是[[西班牙|西班牙]]的二级行政区划。西班牙领土共划分为50个省，分属于17个[[自治区|自治区]]<ref>{{Cite web |url=https://web.archive.org/web/20181016124736/http://es.mofcom.gov.cn/article/ddgk/zhengt/201407/20140700677267.shtml |title=西班牙行政区划 |accessdate=2018-10-16 |author=驻西班牙经商参处 |publisher=中华人民共和国商务部|language=zh}}</ref>，从[[1833年西班牙行政区划分|1833年开始划分省]]以来，基本没有变动，只有[[加那利群岛|加那利群岛]]后来被重新划分为两个省。<ref>{{Cite web |url=http://www.congreso.es/consti/constitucion/indice/sinopsis/sinopsis.jsp?art=141&tipo=2 |title=Sinopsis artículo 141 |accessdate=2018-10-16 |publisher=Constitución española|language=es}}</ref>

省的作用是作为独立的选区、邮政和[[电话|电话]]编码的基础，一般情况下，省都是和它最大的城市同名。其中有7个自治区只包含有唯一个省，社会福利、教育、文化、发展等权利基本归属自治区，政府支出中央政府只占18%，省级占13%，自治区占38%，基本除了外交、军事以外的政府事宜都归自治区一级独立负责。
