'''西安邮电大学'''是位于中国[[陕西|陕西]]省[[西安|西安]]市的一所公立大学，简称'''西邮'''。西安邮电大学肇始于陕西省邮电人员训练班与甘肃省邮电人员训练班。<ref name=history> {{Cite web|url=http://www.xiyou.edu.cn/lminfo.jsp?urltype=tree.TreeTempUrl&wbtreeid=20|title=西安邮电学院历史沿革|author=|date=|publisher=|language=中文|accessdate=2012-07-15|quote=……并于 1986 年正式参加全国普通高校统一招生。|deadurl=yes|archiveurl=https://web.archive.org/web/20120603161622/http://www.xiyou.edu.cn/lminfo.jsp?urltype=tree.TreeTempUrl&wbtreeid=20|archivedate=2012年6月3日|df=}} </ref>西安邮电大学于2012年由'''西安邮电学院'''更名为西安邮电大学。<ref>{{Cite web |url= http://www.moe.gov.cn/publicfiles/business/htmlfiles/moe/s181/201204/xxgk_133896.html |title= 教育部关于同意西安邮电学院更名为西安邮电大学的通知 |author=  |date= 2012-04-12 |publisher= [[中华人民共和国教育部|教育部]] |language= zh-hans |accessdate= 2012-08-16 |quote=  }}</ref><ref>{{Cite web |url= http://www.shaanxi.gov.cn/0/103/9190.htm |title= 陕西省人民政府关于西安邮电学院更名为西安邮电大学的通知 |author=  |date= 2012-04-22 |publisher= 陕西省人民政府 |language= zh-hans |accessdate= 2012-08-02 |quote=  }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref name=university>{{Cite web |url= http://www.snedu.gov.cn/jynews/jyyw/201205/06/26618.html |title= 陕西举行西安邮电大学等四所高校揭牌仪式 赵正永讲话 |author=  |date= 2012-05-06 |publisher=  |language= 中文 |accessdate= 2012-07-15 |quote=  |archive-url= https://web.archive.org/web/20120605023949/http://www.snedu.gov.cn/jynews/jyyw/201205/06/26618.html |archive-date= 2012-06-05 |dead-url= yes }}</ref>学校目前有雁塔、长安两个校区，总占地一千五百余亩，在校学生一万八千余人。<ref name="intro"/>

西安邮电大学一所以信息科技为特色、工学门类为主体、工管理经文法艺相结合的多科性大学。2013年5月，陕西省政府与[[中华人民共和国工业和信息化部|工信部]]签署协议共建西安邮电大学。<ref name=work>{{Cite web |url= http://www.sxdaily.com.cn/n/2013/0521/c266-5136130.html |title= 陕西省政府与工信部签署协议共建西安邮电大学 |author=  |date= 2013年5月21日 |publisher= 陕西传媒网-陕西日报 |language= zh-hans |accessdate= 2013年6月6日 |quote=  |archive-url= https://web.archive.org/web/20140714133752/http://www.sxdaily.com.cn/n/2013/0521/c266-5136130.html# |archive-date= 2014-07-14 |dead-url= yes }}</ref>

==历史沿革==
===成立===
1950年初，西北邮电管理局创办了陕西省邮电人员训练班和甘肃省邮电人员训练班；<ref name=history2/>1952年4月1日，西北邮电管理局决定陕西省邮电人员训练班与甘肃省邮电人员训练班合并成为西北邮电学校；<ref name=history2/>1953年6月，[[邮电部|邮电部]]将西北邮电学校更名为西安邮电学校；<ref name=history/>1955年5月1日，邮电部将学校更名为西安邮政学校，为中专学校。<ref name=history/>

1958年5月1日，邮电部决定撤销西安报务学校，并入西安邮政学校，增设电信专业，将校名改为西安邮电学校；<ref name=history2/>1959年3月16日，西安邮电学院正式成立，开始招收本科生。<ref name=history/>

===撤销===
1969年——[[文化大革命|文革]]期间，学院被撤销，校产划归邮电部第527厂。<ref name=history/>

===复建===
1980年，学校恢复筹建；<ref name=history/>1986年，学校恢复本科招生，划归邮电部管理。<ref name=history/>2000年，2000年学院由信息产业部划转陕西省管理。<ref name=work/>

2003年7月，学校获得硕士学位授权资格。<ref name=history2>{{Cite web|url=http://www.xiyou.edu.cn/new/lminfo.jsp?urltype=tree.TreeTempUrl&wbtreeid=420|title=历史沿革|author=|date=|publisher=|language=zh-hans|accessdate=2012年8月1日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20120903091945/http://www.xiyou.edu.cn/new/lminfo.jsp?urltype=tree.TreeTempUrl&wbtreeid=420|archivedate=2012年9月3日|df=}}</ref>2012年，西安邮电学院更名为西安邮电大学。<ref name=university/>2013年，学校转为陕西省政府与工信部共建。

===雁塔校区===
西安邮电大学雁塔校区位于西安市[[雁塔区|雁塔区]][[长安南路|长安南路]]，通常也被称为老校区。雁塔校区由原管理工程学院(现与原经济管理学院合并为经济与管理学院)与职业技术学院及研究生院共同使用。自2014年9月起，西安邮电大学所有本科新生均就读于长安校区(也称新校区)，即2013级原管理工程学院本科生为就读于雁塔校区的最后一届本科生。<ref>
{{Cite web |url= http://xupt.edu.cn/new/lminfo.jsp?urltype=tree.TreeTempUrl&wbtreeid=556 |title= 管理工程学院 |author=  |date=  |publisher= 西安邮电大学 |language= zh-hans |accessdate= 2012-08-10 |quote= 办公地点：雁塔校区行政楼3层 |deadurl= yes |archiveurl= https://web.archive.org/web/20120920150942/http://xupt.edu.cn/new/lminfo.jsp?urltype=tree.TreeTempUrl&wbtreeid=556 |archivedate= 2012年9月20日 |df=  }}</ref><ref>西安邮电大学2012级新生入学手册</ref>校区由18层教学楼、图书馆、体育馆、体育场、学生宿舍、食堂等单位组成。

雁塔校区体育场在1996年作为[[陕西国力足球俱乐部|陕西国力队]]的训练场地使用；体育馆曾作为1999年第四届[[全国城市运动会|全国城市运动会]][[羽毛球|羽毛球]]比赛的主赛场及训练场。<ref>{{Cite web|url=http://www.sxsdq.cn/dqzlk/sxsnj/nj1997/200912/t20091223_63181.htm|title=第四届全国城市运动会筹备工作|author=|date=|publisher=陕西地情网|language=zh-hans|accessdate=2012-08-06|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20160304113840/http://www.sxsdq.cn/dqzlk/sxsnj/nj1997/200912/t20091223_63181.htm|archivedate=2016年3月4日|df=}}</ref>雁塔校区的图书馆于1996年建成并投入使用，面积4986平方米。<ref name=library>{{Cite web |url= http://lib.xupt.edu.cn/introduce/introduce.asp |title= 图书馆简介 |author=  |date=  |publisher= 西安邮电大学图书馆 |language= zh-hans |accessdate= 2012-08-07 |quote=  }}</ref>

===长安校区===
[[File:西安邮电大学图书馆.jpg|thumb]]
西安邮电大学长安校区也被称为新校区或者是南校区。位于西安市[[长安区|长安区]]，在[[西安郭杜教育科技产业开发区|西安郭杜教育科技产业开发区]]内，地处[[子午大道|子午大道]]与韦郭路交界处。目前是学校主要办公、教学的场所，除管理工程学院和职业技术学院的部分在雁塔校区，其余院系均在长安校区。长安校区分为东西两个区域，中间由子午大道分开。

西区西邻[[陕西师范大学|陕西师范大学]]，已于2003年开始投入使用，<ref name=history2/>目前西区已经基本建成。其中办公教学区包括两座教学楼、三座实验楼、以及图书馆、行政楼、大学生活动中心；学生生活区包括体育场、训练场、网球场、体育馆、食堂、美食广场、宿舍楼、开水房与澡堂等设施。

长安校区图书馆于2004年正式启用，面积达到23441平方米，两校区图书馆共藏书纸质图书146万余册。<ref name=library/>

==其他==
西安邮电大学是“[[国立七中|国立七中]]校友会”总部所在地，该获洋县民政局批准于1993年。
