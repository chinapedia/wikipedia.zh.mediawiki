'''西灣湖廣場'''（[[葡文|葡文]]：'''Praca do Lago Sai Van'''），前稱為'''南灣圓形地'''（葡文：'''Rotunda da Baía da Praia Grande'''）是[[澳門|澳門]]第二大廣場，也是澳門四大廣場之一，是一個位於[[澳門旅遊塔|澳門旅遊塔]]前方的鵝蛋形[[迴旋處|迴旋處]]。廣場面積為17,600平方米。

== 歷史 ==
1998年，[[澳門旅遊娛樂有限公司|澳門旅遊娛樂有限公司]]在該地段興建[[澳門旅遊塔|澳門旅遊塔]]，於是[[澳葡政府|澳葡政府]]在塔的前地興建以圓形為主的廣場，以作大型活動之用。廣場興建屬於“三大廣場計劃”之一。

2001年，隨著旅遊塔的落成啟用，“南灣圓形地”開放啟用。同年年底，[[建設發展辦公室|建設發展辦公室]]為廣場建造進行公開招標，當時工程名稱為“南灣湖廣場建造工程”。

2002年1月25日，“南灣湖廣場建造工程”正式動工，工程包括將現時正圓的“南灣圓形地”大幅擴展成鵝蛋形，也需要靠近西灣湖部分進行塡土，以便興建親水平台及船隻泊位，工程還需要把現時南灣圓形地下方的行車隧道向西灣湖景大馬路方向延長，工程總佔地面積為30,600平方米<ref>{{cite news |language = zh-hant|author =  |coauthors = |url = |title = 耗資三千萬南灣湖廣場動工|publisher = 澳門日報|pages =  |date = 2002年1月26日 |accessdate = }}</ref>。

2002年11月25日，廣場下方行車隧道完成延長工程，重新恢復行車<ref>{{cite news |language = zh-hant|author =  |coauthors = |url = |title = 南灣湖廣場隧道擴建重開|publisher = 澳門日報|pages =  |date = 2002年11月26日 |accessdate = }}</ref>。

2002年除夕，除夕倒數活動首次移師到廣場舉行<ref>{{cite news |language = zh-hant|author =  |coauthors = |url = |title = 南灣湖廣場今除夕倒數|publisher = 澳門日報|pages =  |date = 2002年12月31日 |accessdate = }}</ref>，自此成為每年取代了[[議事亭前地|議事亭前地]]，成為[[澳門|澳門]]區的[[跨年|跨年]]倒數晚會的舉辦地。此外，也成為多個音樂會、同樂日、澳門學界升旗儀式等場所。當時廣場仍有部分工程未完成，因此是為活動舉行而局部開放。

2003年5月，廣場工程完成並正式對外開放。同年9月26日，廣場正式命名為“'''西灣湖廣場'''”（[[葡文|葡文]]：'''Praca do Lago Sai Van'''）<ref>{{cite news |language = zh-hant|author =  |coauthors = |url = |title = 旅遊塔對開圓形地，西灣湖廣場命名
|publisher = 澳門日報|pages =  |date = 2003年9月28日 |accessdate = }}</ref>。

==四大廣場==
澳門四大廣場指西灣湖廣場與另外三個廣場，其餘三個分別是：
* [[議事亭前地|議事亭前地]]
* [[澳門文化中心廣場|澳門文化中心廣場]]
* [[塔石廣場|塔石廣場]]

==誤解==
由於廣場位於[[南灣湖|南灣湖]]以及[[西灣湖|西灣湖]]之間，因此坊間誤稱作為「南灣湖廣場」，但其實並沒有「南灣湖廣場」，而他們所指的是西灣湖廣場。亦有人稱之為「觀光塔前地」，但其實「觀光塔前地」位於[[旅遊塔|旅遊塔]]東側，連接觀光塔街，並非他們所指的西灣湖廣場。
