'''西京医院'''（'''中国人民解放军空军军医大学第一附属医院'''），位于[[陕西省|陕西省]][[西安市|西安市]][[新城区_(西安市)|新城区]]长乐西路127号，隶属[[中国人民解放军空军军医大学|中国人民解放军空军军医大学]]，是一所融医疗、教学、科研、预防、保健于一体的大型综合性医院，为[[三级甲等医院|三级甲等医院]]<ref >{{cite web|url=http://xjwww.fmmu.edu.cn/ABOUT/sort01/27.html | language= | title=西京医院简介 |publisher=西京医院 |date=2015-12-10 |accessdate= }}</ref>。

== 简介 ==
前身是1939年11月在[[延安|延安]]成立的中央医院（1947年2月改编成陕甘宁晋绥联防军第一后方医院）以及1947年在[[山西|山西]]成立的陕甘宁晋绥联防军卫生部直属医院。1949年6月进驻西安后，西北军区人民医学院卫生所合并进直属医院（1950年1月改编成西北人民医学院附属医院）。1950年8月，第一后方医院与西北人民医学院附属医院合并，组成'''中国人民解放军西北军区第二陆军医院'''，由西北人民医学院代管。1954年，第二陆军医院命名为'''中国人民解放军第四军医大学附属医院'''<ref>{{Cite web |url=http://www.huanqiuyixun.com/hospital/323.html |title=环球医讯网－第四军医大学西京医院 |access-date=2017-10-21 |archive-url=https://web.archive.org/web/20171021111827/http://www.huanqiuyixun.com/hospital/323.html |archive-date=2017-10-21 |dead-url=yes }}</ref>。

1954年4月7日，根据中央军委《关于军医大学整编的决定》，[[西安|西安]]的第四军医大学、[[南京|南京]]的第五军医大学合并为中国人民解放军第四军医大学，校址设西安，两校附属医院随之合并。1954年9月，第五军医大学附属医院150人及部分图书资料、仪器设备、药品器材到达西安，正式与第四军医大学附属医院合并，仍称'''中国人民解放军第四军医大学附属医院'''<ref name=yange/><ref>{{Cite web |url = http://xjwww.fmmu.edu.cn/ABOUT/sort07/1998.html |title = 一、1954年两院合并 |author =  |date =2011-06-23  |publisher = 西京医院 |language =  |accessdate =  |quote =  }}</ref>。中国人民解放军第五军医大学附属医院原为[[中华民国|中华民国]]时期1935年成立的[[国立中央大学|国立中央大学]]医学院附属医院，后来先后更名[[南京大学|南京大学]]医学院附属医院、第三军医学院附属医院、第五军医大学附属医院<ref name=yange/>。

1969年8月29日，中央军委批准[[中国人民解放军总后勤部|中国人民解放军总后勤部]]关于医科院校调换校址的报告。第四军医大学附属医院随第四军医大学迁往[[重庆市|重庆市]][[沙坪坝区|沙坪坝区]]高滩岩，接防第七军医大学第一附属医院（现[[西南医院|西南医院]]）。1969年9月20日，第四军医大学附属医院传达了换防命令，1969年10月15日至11月6日该院医护人员及家属1400多人分批到达重庆，完成换防。1975年5月30日，国务院、中央军委批复总后勤部《关于第二、第四、第七军医大学迁回原校址的请示》。1975年7月至8月，第四军医大学附属医院从重庆迁回西安。1984年10月2日起，对外称'''西京医院'''<ref name=yange>{{Cite web |url = http://xjwww.fmmu.edu.cn/ABOUT/sort03/28.html |title = 西京医院历史沿革 |author =  |date = 2011-06-23 |publisher = 西京医院 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://xjwww.fmmu.edu.cn/ABOUT/sort07/2011.html |title =五、1969年换防重庆  |author =  |date = 2011-06-23 |publisher =西京医院  |language =  |accessdate =  |quote =  }}</ref>。2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，改称'''中国人民解放军空军军医大学第一附属医院'''。

西京医院在器官移植、烧伤等领域处于世界领先水平。先后获全国百姓放心示范医院、全国百佳医院、全国拥政爱民模范单位、全军医院建设先进单位、全国抗震救灾英雄集体。“西京医院”院名被[[中华人民共和国国家工商行政管理总局|中华人民共和国国家工商行政管理总局]]认定为[[中国驰名商标|中国驰名商标]]<ref>{{Cite web |url = http://xjwww.fmmu.edu.cn/ABOUT/sort01/27.html |title = 西京医院简介 |author =  |date =2015-12-10  |publisher = 西京医院 |language =  |accessdate =  |quote =  }}</ref>。
