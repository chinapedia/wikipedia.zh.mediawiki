'''西布曲明'''（Sibutramine），化學名(±)''N''-(1-(1-(4-氯苯基)环丁基)-3-甲基丁基)-''N'',''N''-二甲胺，[[化學式|化學式]]C<sub>17</sub>H<sub>26</sub>ClN，是一種作用於[[中樞神經系統|中樞神經系統]]的[[食慾抑制劑|食慾抑制劑]]，其化學結構與[[安非他命|安非他命]]有點關連，但作用機理不同。它在醫學上可作為一種口服的[[減肥藥|減肥藥]]，通常使用'''盐酸西布曲明'''（Sibutramine Hydrochloride，C<sub>17</sub>H<sub>26</sub>ClN·HCl·H<sub>2</sub>O）生產藥品。

該藥由[[美國雅培|美國雅培]]製藥研發，商品名Reductil（諾美婷）、Meridia。

== 藥理學 ==
西布曲明抑制中樞[[神經末梢|神經末梢]]對[[正腎上腺素|正腎上腺素]]、[[5-羥色胺|5-羥色胺]]和[[多巴胺|多巴胺]]的再攝取，從而增加這些[[神經傳導物質|神經傳導物質]]的數量，使服用者的[[食慾|食慾]]減低。

== 适应症 ==
西布曲明在1997年獲[[美国食品药物管理局|美国食品药物管理局]]批准推出市場，与[[低卡食品|低卡食品]]结合使用来[[减肥|减肥]]和维持體重，用于初始[[体重指数|体重指数]]至少30的人，或体重指数大于等于27但有其它风险因素（如[[高血压|高血压]]、[[糖尿病|糖尿病]]）的人。（另見下面[[#停售|停售]]一節）

== 用法及用量 ==
該藥在大多數地區（如不是所有的話）需經醫生處方供應。在醫生處方下，配合飲食控制及運動，該藥可作為控制過胖的輔助治療，但不宜服用超過一年，停藥後體重可能再增加。口服給藥時，初時每日早上服10毫克。如服用4星期後體重減輕不理想，劑量可加至每日15毫克，如再過4星期體重減輕仍不理想，應停止服用該藥。

18歲以下或65歲以上人士不建議服用該藥。

== 不良反應 ==
常見的不良反應有口乾、加強食慾、失眠、[[便秘|便秘]]。其它不良反應有心跳加快、血壓上升等。

有部分人在使用西布曲明後可以引致高血壓及加快脈搏跳動，因此，所有服用者必須定期接受血壓及心跳檢查。另外，使用西布曲明可能會出現以下罕見的副作用：心律不整、皮膚感覺異常、情緒化（例如：過度興奮、迷失、傷心，激動極少數甚至會出現自殺念頭）。

== 停售 ==
由於有增加心臟病的風險，[[美國|美國]]、[[澳洲|澳洲]]、[[歐盟|歐盟]]及[[台湾|台湾]]等國家或地區已陸續廢止該藥品的許可證，並令製藥廠回收。<ref>{{cite web|url=http://www.fda.gov.tw/news.aspx?newssn=6485&classifysn=4|title=食品藥物管理局評估sibutramine成分藥品，自10月11日起廢止所有該成分藥品許可證，許可證持有藥商應於1個月內收回市售品|publisher=中華民國行政院衛生署食品藥物管理局|date=2010-10-09}}</ref><ref>{{cite news|url=http://news.msn.com.tw/news1890067.aspx|title=諾美婷下市 減重班摩拳擦掌|publisher=中央社|date=2010-10-10}}</ref>

2010年10月，雅培製藥同意在美國停售含西布曲明的藥品Meridia。<ref>[http://www.fda.gov/NewsEvents/Newsroom/PressAnnouncements/ucm228812.htm Abbott Laboratories agrees to withdraw its obesity drug Meridia], FDA NEWS RELEASE, 2010年10月8日。</ref>

2010年10月30日，中華人民共和國[[國家食品藥品監督管理局|國家食品藥品監督管理局]]宣佈國內停止生產、銷售和使用西布曲明製劑和原料藥，撤銷其批准證明文件，已上市銷售的藥品由生產企業負責召回銷毀。<ref>[http://www.sfda.gov.cn/WS01/CL0055/55314.html 《關於停止生產銷售使用西布曲明製劑及原料藥的通知》]（國食藥監辦[2010]432號），2010年10月30日。</ref>

而在香港，藥劑業及毒藥管理局已取消所有含西布曲明的藥劑產品的註冊。
