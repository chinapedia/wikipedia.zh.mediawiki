'''西早來號誌站'''（{{jpn|j=西早来信号場|hg=にしはやきたしんごうじょう|rm=Nishi-hayakita shingōjō|f=y}}）位於[[日本|日本]][[北海道|北海道]][[勇拂郡|勇拂郡]][[安平町|安平町]]早來富岡，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[石勝線|石勝線]]沿線的[[號誌站|號誌站]]。

== 車站構造 ==
號誌站兩端設有[[防雪廊|防雪廊]]（snow shelter），防止大雪凍結[[轉轍器|轉轍器]]。設計之初原預定設置車站，並辦理客運業務，但後來此構想沒有實現。

開站以來就一直是沒有設置站員的無人號誌站。

== 車站周邊 ==
位於深山之中，附近沒有居民。號制站夾在[[陸上自衛隊|陸上自衛隊]]東千歳駐屯地與安平駐屯地之間。

== 历史 ==
*1981年（[[昭和|昭和]]56年）10月1日 - [[日本國有鐵道|日本國有鐵道]]石勝線西早來號誌站設立。
*1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道接收管理。
