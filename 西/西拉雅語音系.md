'''西拉雅語音系'''（{{lang|en|Siraya phonology}}）為[[西拉雅語|西拉雅語]]之[[音位學|音系]]。而其音系之[[音位|音位]]大都使用適當的[[Unicode|Unicode]]符號來標示。在[[台灣南島語|台灣南島語]]的[[書寫系統|書寫系統]]的訂定上，元輔音之音系表原則上都是先「[[發音部位|發音部位]]」(橫列)、後「[[發音方法|發音方法]]」(縱列)、再考量「[[清濁音|清濁音]]」，來訂定其音系之音位架構。<ref name=shaolatin>行政院原住民族委員會（族語書寫系統）,"原住民族語言書寫系統[http://ws.moe.edu.tw/001/Upload/6/RelFile/6508/7828/aboriginal.pdf]"{{cite web |url=http://ilrdc.tw/research/rwview/rwsystem.php |title=族語書寫系統-原住民族語言書寫系統 - 語發中心 - 原住民族語言研究發展中心（存档副本） |accessdate=2018-03-03 |date=2005-12-15 }},台語字第0940163297號,原民教字第09400355912號公告,中華民國94年12月15日.</ref>

==語言差異==
根據前中研院史語所副所長李壬癸博士與日本語言學家土田茲研究，西拉雅語、[[大滿語|大滿語]]及[[馬卡道語|馬卡道語]]等南部平埔語言至少有四種音韻演變的差異及一種構詞上的差異<ref name=":0">{{Cite book|title=珍惜台灣南島語言|last=李壬癸|first=|publisher=台灣本鋪：前衛出版社|year=2010|isbn=978-957-801-635-4|location=|pages=139-182}}</ref><ref>{{Cite book|title=新港文書研究|last=李壬癸|first=|publisher=中央研究院語言學研究所|year=2010|isbn=978-986-02-3342-1|location=|pages=1-12}}</ref>：
# 西拉雅語和馬卡道語的 r 對應大滿語的 h 或消失，如西拉雅語的 ''turu''（三）對應大滿語的 ''tuhu''，或西拉雅語的 ''rima''（五）對應大滿語的 ''hima''。
# 西拉雅語和大滿語的 l 對應馬卡道語的 n、l 或 r，如西拉雅語的 ''wali''（牙齒）對應馬卡道語的 ''wari''。
# 西拉雅語的 s 對應大滿語的 r 或 d，如西拉雅語的 ''sa''（和）對應大滿語的 ''ra'' 或 ''da''，或西拉雅語的 ''raos''（西邊）對應大滿語的 ''raur''。
# 西拉雅語在[[元音|元音]]之間的[[舌根音|舌根音]] k 和 g 在大滿語已脫落，如西拉雅語的 ''ako-saij''（沒有）對應大滿語的 ''au-saij''，或西拉雅語的 ''dagogh''（價錢）對應大滿語的 ''daoh''。
構詞上，西拉雅語表示「未來」的動詞後綴為 -ali、-ili（新港社）或 -ati、-ili（卓猴社），而大滿語是 -ah（灣裡社、麻豆社），馬卡道語則是與西拉雅語同源的 -ani（下淡水社）。

由各地語料上的差異規律，李壬癸發現麻豆社與灣裡社有相同的音韻與構詞變化，應同屬大滿語，而非西拉雅語。

李壬癸推論南部平原西拉雅語、大滿語及馬卡道語約在 3 千多年前就開始分化，只是17 世紀荷蘭人據臺後，採用臺南新港社西拉雅語作為通用語，造成各族群間語言逐漸同化。
