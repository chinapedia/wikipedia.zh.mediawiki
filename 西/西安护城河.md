'''西安护城河'''是环绕[[西安城墙|西安城墙]]的一条[[护城河|护城河]]，始建于[[明代|明代]][[洪武|洪武]]年间，距今超过六百年历史。<ref name=renmin>{{Cite web |url= http://sn.people.com.cn/n/2013/0802/c226647-19221020.html |title= 西安护城河：汛期的古城守护者 |author=  |date= 2013年8月2日 |publisher= 人民网-陕西频道 |language= zh-hans |accessdate= 2014年4月1日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20140407093322/http://sn.people.com.cn/n/2013/0802/c226647-19221020.html |archivedate= 2014年4月7日 |df=  }}</ref>护城河是[[西安|西安]]最著名的地标之一。<ref>{{Cite web |url= http://epaper.xiancn.com/xawb/html/2013-08/04/content_228988.htm |title= 护城河的记忆 |author=  |date= 2013年8月4日 |publisher= 西安晚报 |language= zh-hans |accessdate= 2014年4月1日 |quote=  }}</ref>护城河现在是西安市民散步、纳凉常去之地，也起着西安市内防汛的任务。<ref name=renmin/>

==历史==
西安护城河最早建于唐末天佑元年（公元904年），至明洪武七年（1374年），宋国公[[冯胜|冯胜]]拓掘了护城河。<ref name=hsw>{{Cite web |url= http://news.hsw.cn/system/2012/08/12/051426124.shtml |title= 每天1.7万吨污水流入护城河 生态供水时有堵塞 |author=  |date= 2012-08-12  |publisher= 华商网-华商报  |language= zh-hans |accessdate= 2014年4月1日 |quote=  }}</ref>[[清代|清代]]在明城墙的基础上分别于同治二年（1863年）、光绪二十一年（1895年）对西安城墙与护城河进行修缮、加固。<ref name=article>孙静,唐登红. 西安明城护城河的职能衍变与城市发展关系初探. 四川建筑,2009(29)09:46.</ref>民国时期，护城河一直处于无人管理的状态。

中华人民共和国建立以后，护城河被纳入城市排水系统，成为市区唯一的雨水泄洪河道和调节库。<ref name=article/>1961年，西安城墙被纳入[[全国重点文物保护单位|全国重点文物保护单位]]，护城河也被纳入保护体系中。<ref name=article/>20世纪60年代以后，因为缺乏维护，众多单位在雨水管道排放污水，导致护城河污染严重，河道拥挤，泄流不畅，外溢倒流。<ref name=article/>

==污染与治理==
西安护城河在20世纪80年代曾经充满淤泥、脏水堆积，并且散发异味。<ref>{{Cite web |url= http://www.chinanews.com/df/2013/02-07/4557363.shtml |title= (美丽中国行)慢慢变清的西安护城河 |author= 田进、冽玮 |date= 2013年2月7日 |publisher= 中国新闻网 |language= zh-hans |accessdate= 2014年4月1日 |quote=  }}</ref>在2012年，每天约1.7万吨污水流入护城河。<ref name=hsw/>

西安护城河历经多次治理。1998年，驻陕某部队工兵团进驻西安[[环城公园|环城公园]]清淤，历时近一年清淤二十万立方米左右。<ref name=hsw/>2004年12月6日起，采取高压管道输送方式，历时一年花费一千万元，铺设输送管道二十多公里，清理淤泥十三万立方米。<ref name=hsw/>2009年，花费四百多万元对文昌门、西门、东门、东北角进行清淤。<ref name=hsw/>2010年，城墙管委会使用细菌进行清污，投放微生物菌种，针对不同的有机物和腐殖质进行消除，淤泥被就地分解成为水和二氧化碳。
<ref>{{Cite web |url= http://epaper.xiancn.com/xawb/html/2010-07/22/content_258098.htm |title= “生物战”治理护城河未来清水绕城鱼虾游动 |author=  |date= 2010年7月22日  |publisher= 西安晚报 |language= zh-hans |accessdate= 2014年4月1日 |quote=  }}</ref>

==结构==
西安护城河全长14700米，东南角高，西北角低，有11.6米的落差。护城河的正常蓄水量为100万立方，最大蓄水量则为140万立方。<ref name=renmin/>同时，护城河有7座水坝、11个闸门、5处市政箱涵、33处雨水管道、17处污水管道。<ref name=renmin/>

==发展与规划==
到2014年4月15日，护城河[[建国门_(西安)|建国门]]至[[朱雀门|朱雀门]]段，注满再生水和来自[[秦岭|秦岭]][[大峪|大峪]]的河水，形成3至6米深的水面。同年5月对游人开放，可以于水面泛舟。水上游览示范区全长2.6公里，设置4个码头，东起建国门，西至朱雀门，包含[[和平门_(西安)|和平门]]、[[文昌门|文昌门]]、[[永宁门|永宁门]]。<ref>{{Cite web|url=http://sx.sina.com.cn/news/m/2014-04-28/0857104778.html景|title=西安南门箭楼恢复古貌 城河清波泛舟赏景|author=|date=2014年4月28日|publisher=新浪陕西|language=zh-hans|accessdate=2014年4月30日|quote=}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

水位提升后，城墙管委会还将通过灯光、音乐喷泉等现代技术装点河面。<ref>{{Cite web |url= http://news.hsw.cn/system/2014/04/01/051893434.shtml |title= 护城河示范段昨起注水 五一有望开园市民可泛舟 |author=  |date= 2014-04-01 |publisher=  华商网-华商报 |language= zh-hans |accessdate= 2014年4月1日 |quote=  }}</ref>
