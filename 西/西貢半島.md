'''西貢半島'''是[[香港|香港]][[新界|新界]]東部的[[半島|半島]]。從狹義[[香港行政區劃|行政區劃]]方面，西貢半島只包括[[西沙路|西沙路]]以東地區，南部屬於[[西貢區|西貢區]]，北部屬於[[大埔區|大埔區]]，是香港面積最大的[[半島|半島]]；廣義更包括[[沙田區|沙田區]]的東北部（[[馬鞍山_(香港山峰)|馬鞍山]]）。西貢半島風光優美，有'''香港後花園'''之美譽。

== 概況 ==
西貢半島的大部分土地在[[城市化|城市化]]中沒有被開發，都是屬於[[郊野公園|郊野公園]]的範圍。位於西貢半島的[[海下灣|海下灣]]是[[香港海岸公園|香港海岸公園]]之一，受到法律限制，不可以進行海上作業活動等。西貢半島也是[[遠足|遠足]]的熱門地點，長達100[[公里|公里]]的[[麥理浩徑|麥理浩徑]]的起點就是位於半島內的[[北潭涌|北潭涌]]。

西貢半島的也是一個十分適合水上活動的地點，例如-{划}-[[橡皮艇|橡皮艇]]、[[潛水|潛水]]、[[游泳|游泳]]等。此外，海下灣是香港最容易前往的潛水地點之一，適合進行[[潛水|潛水]]訓練及潛水初學者。

=== 歷史 ===
西貢區境內滘西洲曾出土新石器時代文物及漢代文物，[[大廟灣|大廟灣]]有[[南宋|南宋]][[刻石|刻石]]，（已被列為香港法定古蹟），[[清朝|清朝]]初期遷界時，西貢這個地域範圍曾短暫歸併入[[東莞|東莞]]縣，但不久重設[[寶安縣|新安縣]]，此地亦重歸新安縣[[官富|官富司]]的管轄。

1898年[[英國|英國]]租借[[新界|新界]]，並設南約、北約兩個[[理民府|理民府]]管理新界。1942年[[日本|日本]]人佔領香港，實行分區管治，才有西貢區的設立，當時的西貢區包括今日[[西貢北約鄉事委員會|西貢北約鄉事委員會]]、[[西貢區鄉事委員會|西貢區鄉事委員會]]及[[坑口鄉事委員會|坑口鄉事委員會]]所屬的範圍。[[第二次世界大戰|第二次世界大戰]]後，[[理民府|理民府]]制度恢復。在1960年至1963年間，政府曾將新界民政署屬內的南約區，分為[[離島區|離島]]與西貢兩區，但不久又重新合併。到1969年確定在西貢區興建[[萬宜水庫|萬宜水庫]]後，才再次劃分西貢區，範圍包括由[[新九龍|新九龍]]界起，至東面的[[清水灣半島|清水灣半島]]及西貢半島南；再加上[[牛尾海|牛尾海]]及[[糧船灣|糧船灣]]海內及附近各島嶼。初步確立了今日西貢區的範圍。

=== 命名 ===
在清朝時期，所有由西方進入中國的商船、貢船必須在西貢的[[佛堂洲|佛堂洲]]完稅方可進入中國海域。因此，西貢這地方的命名就是指從西方來的貢船。

== 飲食 ==
[[File:SaiKung-SeafoodStreet.png|thumb]]，海鮮食肆林立]]
[[File:HK_Wong_Shek_Country_Park.jpg|thumb]]]]

西貢半島在沒有開埠前是一條[[漁村|漁村]]，[[西貢市|西貢市]]是[[海鮮|海鮮]]愛好者的勝地，本地人及部分海外遊客慕名而來。品嚐海鮮的同時，來臨這裡的人可以到[[街市|街市]]購買海鮮、逛逛商店林立的後巷、參觀[[天后廟|天后廟]]、與朋友吃海鮮宴、光顧不同特色的西式[[酒吧|酒吧]]或[[酒樓|酒樓]]。此外，位於西貢市的著名食店[[滿記甜品|滿記甜品]]吸引了很多本地人及海外遊客。

== 旅遊 ==
位於西貢半島北部的[[黃麻地|黃石]]也是半島內的著名旅遊地點，黃石有很多郊遊設施，有美麗海景。但政府為了保護這區美麗的環境，政府控制這區的汽車流量，在北潭涌設立關卡，只有公共交通工具和有許可證的車輛可以進入。區內設有黃石碼頭，它只有一個街渡碼頭，所以設備上亦較為簡陋，而黃石碼頭的街渡一般是可以到達西貢一些偏僻島嶼如塔門。

在西貢半島的附近，也有很多偏僻[[島嶼|島嶼]]。在[[夏天|夏天]]晚上，有部分人會租[[橡皮艇|橡皮艇]]或[[舢舨|舢舨]]到一些島嶼留宿一夜。遊客普遍到訪的島嶼有：
*[[滘西洲|滘西洲]]
*[[橋咀洲|橋咀洲]]
*[[糧船灣洲|糧船灣洲]]
*[[白沙洲_(西貢區)|白沙洲]]
*[[羊洲|羊洲]]
*[[鹽田仔_(西貢區)|鹽田仔]]

== 行政管理 ==
西貢半島在[[香港行政區劃|行政管理]]上分為兩區，是由於從前[[西貢市|西貢市]]並未開通[[西沙路|西沙路]]往[[十四鄉|十四鄉]]和未有標準道路開往[[北潭坳|北潭坳]]、[[黃石碼頭|黃石碼頭]]等地區，而那些地區的鄉村（如[[深涌|深涌]]、[[荔枝莊|荔枝莊]]及[[榕樹澳|榕樹澳]]）和島嶼都依賴[[街渡|街渡]]或渡輪往來[[大埔滘|大埔滘]]（包括[[塔門|塔門]]及[[東平洲|東平洲]]等，後來街渡改為連接沙田區的[[馬料水|馬料水]]）或者[[西貢市|西貢市]]（包括[[滘西洲|滘西洲]]、[[鹽田梓|鹽田梓]] 、[[吊鐘洲|吊鐘洲]]及已建成[[萬宜水庫|萬宜水庫]]的[[糧船灣|糧船灣]]島等），故此西貢半島北部劃入大埔[[西貢北_(選區)|西貢北]]選區，而南部劃入西貢[[西貢離島_(選區)|西貢離島]]選區。

== 交通 ==
[[File:Sai_Kung_New_Public_Pier.jpg|thumb]]及[[西貢公眾碼頭|西貢公眾碼頭]]（圖中後方）]]

開埠初年來往西貢半島一般只能利用海路，直到1940年代[[日軍|日軍]]興建了[[西貢公路|西貢公路]]，最初只有[[單車|單車]]徑般闊度，1955年才擴闊至現有闊度。

[[西貢市|西貢市]]亦有巴士和小巴路線通往西貢半島內的[[北潭涌|北潭涌]]、[[黃石碼頭|黃石碼頭]]和[[海下灣|海下灣]]等鄉郊地方，西貢市亦有兩個主要公眾碼頭，有定期和租賃的船隻，前往西貢海的多個島嶼。

*[[專線小巴|專線小巴]]：
#[[新界|新界]]小巴路線7號，往返[[西貢市|西貢市]]及[[海下灣|海下灣]]。

*[[巴士|巴士]]：
#[[九巴|九巴]]郊區路線94號，往返[[西貢市|西貢市]]及黃石，班次為30-35分鐘一班；
#[[九巴|九巴]]假日路線96R號，往返[[港鐵|港鐵]][[鑽石山站|鑽石山站]]及黃石，班次為12-20分鐘一班；
