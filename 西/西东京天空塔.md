'''西东京天空塔'''（{{jpn|j='''スカイタワー西東京'''|hg=スカイタワーにしとうきょう|rm=Sukaitawā ni shitō kyō|f=h}}）是位于[[东京都|东京都]][[西东京市|西东京市]]芝久保町五丁目的一座多功能[[電波塔|電波塔]]，1989年6月建成，是一座高195米的自立式[[鐵塔|鐵塔]]。西东京天空塔在建成时位于[[田無市|田無市]]（后与[[保谷市|保谷市]]合并为西东京市），故塔在正式命名前也被称为'''田无塔'''（田無タワー），其运营公司名为田无塔株式会社。<ref name="nikkeibp">{{cite web |url=http://www.nikkeibp.co.jp/article/sj/20110209/259764/ |author=細野透 |date=2011-02-10 |title=「東京スカイツリー」を巡る各種の「高さランキング」 |accessdate=2014-04-28 |publisher=nikkeibp}}</ref>

== 历史 ==
1980年代后期开始，东京首都圈的通信需求日益增长，而建设用地不足，出于对未来通信状况的关注，1987年，田无家族地产集团出资在田无市成立了第三部门田无塔株式会社。<ref name="st195" />

1989年2月，接受了[[郵政省|郵政省]]（现[[總務省|總務省]]）的首批多媒体塔认证。

== 概要 ==
西东京天空塔不设瞭望台，没有游览设施，也不向市民开放。

夜间（18点到0点）会使用颜色灯光预报第二天早上的天气<ref name="nishitokyo">{{cite web |url=http://www.city.nishitokyo.lg.jp/enjoy/spot/skytower.html |author= |date=2011-06-01 |title=スカイタワー西東京 |accessdate=2014-04-28 |publisher=Nishitokyo City |deadurl=yes |archiveurl=https://web.archive.org/web/20140116103029/http://www.city.nishitokyo.lg.jp/enjoy/spot/skytower.html |archivedate=2014年1月16日 |df= }}</ref>：
*晴：紫色
*多云：黄绿色
*雨：蓝色

在[[圣诞节|圣诞节]]的时候会有特殊的颜色点亮，如红色和绿色的条纹。

与塔相邻的是田无家族地产的土地，经营有高尔夫击球场练习中心和柏青哥店。西东京天空塔发送站和社区FM的FM西东京工作室也在其中，另外，也与[[多摩六都科学馆|多摩六都科学馆]]相邻。

== 广播设施配置 ==
现在主要的经营对象是[[FM西东京|FM西东京]]和[[第三者无线|第三者无线]]天线。1993年举行的TAMA Life 21活动中设置的临时FM广播台“Egg STATION”（频率：76.3MHz，功率：300W）曾设置广播天线。

== 出现西东京天空塔的文艺作品 ==
=== 动画和漫画 ===
在附近有[[新荣动画|新荣动画]]等不少大型动画制作工作室，因此西东京天空塔经常出现在动画作品中。
*动画《[[我們這一家|我們這一家]]》中，公寓的背景中的塔的原型。
*动画《[[Keroro軍曹|Keroro軍曹]]》中，出现的“西泽塔”的原型。
*动画《[[蜡笔小新|蜡笔小新]]》剧场版《[[三分鐘百變大進擊|三分鐘百變大進擊]]》中，在怪物于[[東京都道5號新宿青梅線|東京都道5號新宿青梅線]]（新青梅街道）多摩六都科学馆交叉点活动时，出现在背景上。
*动画《[[河童之夏|河童之夏]]》的场景设定在临近的[[東久留米市|東久留米市]]，因此西东京天空塔数次出现于背景中。
*在カラスヤサトシ的漫画《[[东京漂物语|东京漂物语]]》中登场。
*在芦奈野仁的漫画《[[横滨购物纪行|横滨购物纪行]]》中鹰津菜心出去旅游时“印象深刻的塔”。
*出现在特摄电视剧《[[大魔神華音|大魔神華音]]》的片头中。
*动画《[[潜行吧！奈亚子|潜行吧！奈亚子]]》中，从公园看到的塔的原型。
*动画《[[女子落|女子落]]》中，与[[东京塔|东京塔]]和[[東京晴空塔|東京晴空塔]]进行了对比。

*[[朝日电视台|朝日电视台]]《[[田森俱乐部|田森俱乐部]]》2012年5月11日播放《不像东京晴空塔那么挤！“西东京天空塔”观光指南》。
