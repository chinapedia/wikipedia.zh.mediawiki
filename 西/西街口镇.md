'''西街口镇'''是[[中华人民共和国|中国]][[云南省|云南省]][[昆明市|昆明市]][[石林彝族自治县|石林彝族自治县]]下辖的一个[[镇|镇]]。

== 历史沿革 ==
位于石林县城的东北部，东北与陆良县的[[召夸镇|召夸镇]]和[[大莫古乡|大莫古乡]]接壤，南与[[亩竹箐|亩竹箐]]、[[维则乡|维则乡]]毗邻，西接[[北大村乡|北大村乡]]，面积为193．2平方千米。乡政府所在地西街口村，是全乡政治、经济、文化中心，距县城38千米<ref>{{Cite book|title=昆明地名博览辞典|last=吴光范著|first=|publisher=昆明：云南人民出版社|year=2005年1月|isbn=7-222-04154-0|location=|pages=122}}</ref>。

1950年，属[[圭山区|圭山区]]西街口乡，1958年属圭山公社，1963年属石林区，1970年设西街口公社，1984年设区，1988年设乡<ref name=as />。

== 行政区划 ==
西街口镇共辖10个行政村，分别是：西街口村、雨布宜村、绿水塘村、紫处村、威黑村、宜奈村、新木凹村、芭茅村、糯衣村、格渣村<ref name=as>{{Cite book|title=路南彝族自治县志|last=昆明市路南彝族自治县志编纂委员会编|first=|publisher=昆明：云南民族出版社|year=1996年11月|isbn=7-5367-1294-4|location=|pages=77}}</ref>。

== 产业 ==
全镇地形属于[[喀斯特地貌|喀斯特地貌]]，丘陵山路为[[石灰岩|石灰岩]]覆盖，有丰富的石材、矿产和生物资源<ref>{{Cite book|title=新编昆明市情|last=林爽爽主编；新编昆明市情编委会编|first=|publisher=昆明：云南科学技术出版社|year=2000|isbn=7-5416-1374-6|location=|pages=163}}</ref>。粮食主要以玉米、水稻、[[小麦|小麦]]等为主；经济支柱主要是烤烟、石材加工业为主<ref>{{Cite book|title=石林彝族自治县志 1989-2000|last=刘世生主编；石林彝族自治县志编纂委员会编|first=|publisher=昆明：云南民族出版社|year=2006年10月|isbn=7-5367-3448-4|location=|pages=33}}</ref>。森林覆盖率达到35.2%，其中生物主要有野兔、蛇、猫头鹰、穿山甲等；矿产资源主要有青石、白海棠、虎纹石、墨石、秋木石、石板、红黏土、铁矿、石英砂等<ref>{{Cite book|title=石林年鉴 2006|last=《石林年鉴》编辑部编|first=|publisher=昆明：云南美术出版社|year=2006年11月|isbn=7-80695-453-8|location=|pages=82}}</ref>。2010年后，开始种植大量水果作物，包括人参果等<ref>{{Cite news|url=http://travel.sina.com.cn/china/2015-09-26/1325315257.shtml|title=石林首届人参果旅游文化节在西街口镇隆重举行|accessdate=2017-07-28}}</ref><ref>{{Cite news|url=http://travel.sina.com.cn/china/2015-10-22/1338316405.shtml|title=石林西街口人参果远销全国|accessdate=2017-07-28}}</ref>。
