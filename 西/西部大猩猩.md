'''西部大猩猩'''（''Gorilla gorilla''<ref name=MSW>{{MSW3 Groves|pages=181-182}}</ref>）是為人熟悉的[[大猩猩|大猩猩]]。

== 分類 ==
[[File:Gorilla_Male_perspective_5.jpg|thumb]]
西部大猩猩可分成两个[[亚种|亚种]]，即[[西部低地大猩猩|西部低地大猩猩]]（''G. g. gorilla''）及[[克羅斯河大猩猩|克羅斯河大猩猩]]（''G. g. diehli''）。

大部份西部大猩猩都是屬於[[西部低地大猩猩|西部低地大猩猩]]，數量約有94000頭。而[[克羅斯河大猩猩|克羅斯河大猩猩]]的數量則少於300頭。<ref name=WWF>{{cite web | url = http://www.animalinfo.org/species/primate/gorigori.htm | title = Animal Info - Gorilla | publisher = AnimalInfo.org | accessdate = 2007-09-12}}</ref>

== 外表 ==
西部大猩猩的毛色較[[東部大猩猩|東部大猩猩]]淺色。[[西部低地大猩猩|西部低地大猩猩]]是褐色及灰色的，前額呈紅色。牠的鼻端起鈎，但東部大猩猩的則沒有。雄性高1.7-1.8米及重140-180公斤；雌性高1.4-1.5米及重60-100公斤。西部大猩猩較東部大猩猩纖細。[[克羅斯河大猩猩|克羅斯河大猩猩]]在[[頭顱骨|頭顱骨]]及[[牙齒|牙齒]]尺寸都與西部低地大猩猩。

== 簡介 ==
西部大猩猩可以靈活的攀樹，且較[[東部大猩猩|東部大猩猩]]多生活在樹上。牠喜歡吃[[生果|生果]]，差不多100種不同樹木的果實牠都會吃。西部大猩猩難以追蹤及研究。

[[西部低地大猩猩|西部低地大猩猩]]的族群很細小，只有4-8個成員。野生的西部大猩猩是會使用工具的。<ref>[http://biology.plosjournals.org/perlserv/?request=get-document&doi=10.1371/journal.pbio.0030380 PLOS Journal "First Observation of Tool Use in Wild Gorillas"]</ref>

== 保育 ==
[[世界自然保護聯盟|世界自然保護聯盟]]將西部大猩猩列為[[極危物種|極危物種]]，差不多接近[[滅絕|滅絕]]階段。[[埃博拉病毒|埃博拉病毒]]在西部大猩猩的族群中蔓延，令牠們的數量根本不可能增長。<ref name=iucn/>
