'''西雅圖港'''（{{lang-en|Port of Seattle}}）是位於[[美國|美國]][[華盛頓州|華盛頓州]][[西雅圖|西雅圖]]的港区，包括了[[港口|港口]]和[[机场|机场]]。西雅圖港成立於1911年。2011年，有3280萬乘客利用[[西雅图-塔科马国际机场|西雅图-塔科马国际机场]]<ref>{{Cite web |url=https://www.portseattle.org/Newsroom/News-Releases/Pages/default.aspx?bid=256 |title=存档副本 |access-date=2015-10-18 |archive-url=https://web.archive.org/web/20150924080303/http://www.portseattle.org/Newsroom/News-Releases/Pages/default.aspx?bid=256# |archive-date=2015-09-24 |dead-url=yes }}</ref>。西雅圖海港則處理超過200萬TEU的貨櫃，是北美第14大、世界第57大港口<ref>{{Cite web |url=https://www.portseattle.org/About/Publications/Statistics/Seaport/Pages/default.aspx |title=存档副本 |access-date=2015-10-18 |archive-url=https://web.archive.org/web/20150924080233/http://www.portseattle.org/About/Publications/Statistics/Seaport/Pages/default.aspx# |archive-date=2015-09-24 |dead-url=yes }}</ref>。2011年，有超過885,000人郵輪乘客到訪西雅圖港<ref>{{Cite web |url=https://www.portseattle.org/About/Publications/Statistics/Seaport/Pages/Cruise.aspx |title=存档副本 |access-date=2015-10-18 |archive-url=https://web.archive.org/web/20150924080232/http://www.portseattle.org/About/Publications/Statistics/Seaport/Pages/Cruise.aspx# |archive-date=2015-09-24 |dead-url=yes }}</ref>。西雅圖港也為華盛頓州帶來194,000個工作崗位<ref>{{Cite web |url=https://www.portseattle.org/Supporting-Our-Community/Economic-Development/Documents/EconomicImpact_2009Brochurev2.pdf |title=存档副本 |access-date=2015-10-18 |archive-url=https://web.archive.org/web/20150924080322/http://www.portseattle.org/Supporting-Our-Community/Economic-Development/Documents/EconomicImpact_2009Brochurev2.pdf# |archive-date=2015-09-24 |dead-url=yes }}</ref>。
