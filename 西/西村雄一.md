'''西村雄一'''（{{lang|ja|にしむら ゆういち}}，{{bd|1972年|4月17日|catIdx=Nishimura西村}}），[[日本|日本]]籍[[足球裁判|足球裁判]]，也是[[2010年世界盃足球賽|2010年世界盃足球賽]]及[[2014年世界盃足球賽|2014年世界盃足球賽]]的主裁判之一。

== 生涯 ==
西村雄一於2003年进入日本J联赛担任主裁判，2004年正式成為[[國際足協|國際足協]]的國際裁判，翌年他主吹的首个国际大赛是為[[2005年東亞足球錦標賽|2005年東亞國家杯]]判法，在[[中國國家足球隊|中國]]對[[南韓國家足球隊|南韓]]的賽事中，他先後向前者發出了4面[[紅牌|紅牌]]<ref>[http://sports.ifeng.com/gjzq/detail_2010_06/02/1579487_0.shtml 日本主裁判西村雄一眼神不好不妨礙執法世界盃]</ref>，将中国队的助理教练及3名球员[[郜林|郜林]]、[[曹阳|曹阳]]和[[李玮峰|李玮峰]]驱逐出场。同一場比賽中，由於边裁告诉了西村错误情況，故在比赛中誤把郜林当做李玮峰罚下，他其後向郜林道歉<ref>[http://2010worldcup.163.com/10/0630/06/6ADJ89EA00052E97.html 日本裁判：失误不可能避免 我要向郜林道歉]</ref>。

他是[[2007年亚洲杯足球赛|2007年亚洲杯]]及[[2007年世界青年足球錦標賽|2007年世青盃]]的主裁判。

2008年，他是唯一一位為[[2008年非洲國家盃|非洲國家盃]]執法的[[亞洲|亞洲]]球證。另外，[[2010年世界盃足球賽|2010年世界盃足球賽]]，他為[[烏拉圭國家足球隊|烏拉圭]]對[[法國國家足球隊|法國]]的世界盃分組賽判法，在這場賽事中，西村發出了該屆世界盃的首面[[紅牌|紅牌]]。这屇世界杯西村雄一执法了乌拉圭VS法国、西班牙VS洪都拉斯、巴拉圭VS新西兰和巴西VS荷兰的四场比赛。

2014年，他在[[2014年世界盃足球賽|2014年世界盃足球賽]]揭幕戰巴西對克羅地亞中擔任執法球證，但其將點球判給巴西致巴西領先的判決受到眾多專家和民眾質疑<ref>{{cite web |title=名记吐槽世界杯：西村雄一成MVP 执法学中国同行 |url =http://2014.ifeng.com/hq/b/bvc/detail_2014_06/13/36807156_0.shtml |publisher=凤凰体育}}</ref>，也有大量日本網民表示不滿<ref>[http://www.news-us.jp/article/399340705.html 【W杯】ブラジル×クロアチア戦で審判の西村雄一がミスジャッジ！！！？　⇒　正しい判定だった証拠が出る！！！【画像・動画あり】]</ref>。

2014年，西村在[[2014年亞足聯冠軍聯賽|2014亞冠盃]]決賽執法，該場賽事在西村的執法上有很多爭議，其中[[西雪梨流浪者足球俱樂部|西悉尼流浪足球會]]球員辛達利比兩次在禁區內手球，以及高域在禁區明顯撲跌對方球員都沒有判罰12碼。最後兩隊0:0賽和，兩回合計[[沙特希拉爾足球俱樂部|希拉爾]]輸0:1飲恨。賽後[[沙特希拉爾足球俱樂部|希拉爾]]要求對西村作出調查和給予適當的懲罰<ref>[http://hk.apple.nextmedia.com/realtime/sports/20141105/53095163 【亞冠盃】不滿西村執法希拉爾促徹查]</ref>。

2014年11月，由於在亞冠決賽第二回合中表現拙劣，亞足聯決定對日本主裁判西村雄一處以禁賽半年的處罰，並從亞洲精英裁判中除名。<ref>{{cite web |title=亞足聯重罰西村雄一禁吹半年|date=2014年11月17日|url=http://football.on.cc/cnt/news/newc/20141117/fbnewc0105x0.html|newspaper=''[[東方日報_(香港)|東方日報 (香港)]]''}}</ref>但是在亞冠決賽前，本來西村雄一已經向日本足協提出，退出國際比賽執法工作。但亞足聯卻很詭異地任命日本人執法亞冠終極決賽。
