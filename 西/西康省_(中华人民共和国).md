'''西康省'''，簡稱'''康'''，是[[中華人民共和國|中華人民共和國]]成立后不久，在[[中国人民解放军|中国人民解放军]]1950年攻占[[中华民国西康省|中华民国西康省]]、灭[[西康省政府|西康省政府]]后沿袭[[中华民国行政区划|中华民国行政区划]]的省份。於1955年9月底被撤銷，辖境划归[[西藏自治区|西藏自治区]]、[[四川省|四川省]]。

== 人口 ==
[[中華人民共和國國家統計局|中華人民共和國國家統計局]]於1954年11月1日公報的「第一次全國人口調查登記」，統計至1953年6月30日為止，西康省人口為338萬1064人，其中藏族人口占近70%。

== 行政沿革 ==
[[中華人民共和國|中華人民共和國]]建國後，於1950年5月成立[[西康省人民政府|西康省人民政府]]，省府驻[[雅安县|雅安县]]，省主席[[廖志高|廖志高]]。由于自1917年9月类乌齐事件（第一次康藏纠纷）发生后，藏军与川军达成停战协议，双方划金沙江而治，西康省只是名义上管辖金沙江以西的昌都地区，而昌都地区的实际控制权在西藏地方政府手里，因此1950年10月24日[[昌都战役|昌都战役]]结束后，成立了昌都地区人民解放委员会，直属中央政府。1956年[[西藏自治区筹备委员会|西藏自治区筹备委员会]]成立后，昌都地区便划归西藏自治区<ref>[http://www.iccs.cn/contents/401/12262.html {{dead link}} 中央人民政府直辖昌都地区人民解放委员会问题研究] {{Wayback|url=http://www.iccs.cn/contents/401/12262.html |date=20160304095159 }}</ref>。

* 1950年：全省设[[雅安专区|雅安专区]]、[[西昌专区|西昌专区]]等2个[[专区|专区]]和[[西康省藏族自治区|西康省藏族自治区]]。
* 1950年，析昌都地區、康藏交界處置[[盧渡縣|盧渡縣]]。
* 1951年：撤[[金汤设治局|金汤设治局]]，并入康定县；设县级雅安市，由省直辖；
* 1952年：设地级凉山彝族自治区；设地级藏族自治区甘孜办事处、理塘办事处；撤普格、宁东设治局，设普格县、喜德县；撤泸宁设治局，并入冕宁县；设普雄、美姑、金阳、布拖、会东、金矿、石棉、米易8县；设县级[[木里藏族自治区|木里藏族自治区]]；巴安、理化、定乡、瞻化4县更名为巴塘、理塘、乡城、新龙；
* 1953年：撤木里藏族自治区，设[[木里藏族自治县|木里藏族自治县]]。

{{Wikisource|中华人民共和国第一届全国人民代表大会第二次会议关于撤销热河省、西康省并修改中华人民共和国地方各级人民代表大会和地方各级人民委员会组织法第二十五条第二款第一项规定的决议}}
1955年7月30日，[[第一屆全國人民代表大會第二次會議|第一屆全國人民代表大會第二次會議]]通過關於撤銷西康省建制的決議，[[西康省人民委員會|西康省人民委員會]]於1955年9月底撤銷，其政務由[[四川省人民委員會|四川省人民委員會]]接管。

== 政府 ==
{{Main|西康省人民政府}}

1950年4月26日，[[西康省人民政府|西康省人民政府]]成立，省會[[雅安|雅安]]。1954年，《[[中華人民共和國憲法|中華人民共和國憲法]]》改變政府形式，次年西康省依《憲法》規定成立[[人民委員會|人民委員會]]。1955年9月底，西康省撤銷。
