'''西方文化'''指的是最初形成於[[南欧|南欧]]、[[北歐|北歐]]；盛行於西歐、北歐、[[北美|北美]]，[[拉丁美洲|拉丁美洲]]、澳洲，纽西蘭等國家的[[文化|文化]]。与西方这个词相对应的是非西方国家。

== 描述 ==
[[File:Clash_of_Civilizations_mapn2.png|thumb]]在《[[文明衝突論|文明衝突論]]》中的九個世界文明]]
西方文化的概念通常与[[西方世界|西方世界]]传统定义相关联。根据此定义，西方文化是其[[文学|文学]]、[[科学|科学]]、[[音乐|音乐]]以及[[哲学|哲学]]原理的合集，并且有别于其它主要文明。适用于受西欧移民或殖民影响强烈的国家，并不仅限于西欧。这些传统大都在被列入[[西方经典|西方经典]]。

西方文化被用于许多方面，不管实际使用正确与否，包括[[西方文化|西方文化]]、[[唯物主义|唯物主义]]、[[工业主义|工业主义]]、[[资本主义|资本主义]]、[[重商主义|商业主义]]、性的[[享乐主义|享乐主义]]、[[帝国主义|帝国主义]]、[[现代主义|现代主义]]，乃至解说西方文明。

== 基石 ==
西方文化常被认为起源于[[古希腊|古希腊]]和[[罗马帝国|罗马帝国]]的[[印歐人|印歐人]]。西方文化也根源于[[斯拉夫人|斯拉夫人]]還有[[北歐|北歐]]的[[日尔曼人|日尔曼人]]及[[凯尔特人|凯尔特人]]的文化，这些文化是形成欧洲的重要推动力量。[[基督教|基督教]]对[[中世紀|中世紀]]有很大的影响，許多人會将基督教和西方文化相提並論。另一個值得注意的點是：在穆罕默德之前，整個環地中海算同一個文化圈，基督教及伊斯蘭相繼興起後，才被切割為中東及歐洲兩大文化圈，雖然兩地的文化思維在三千年前便已分道揚鑣。

西方文化在文学、音乐、哲学、宗教方面，已经有了极为丰富的发展。重要的传统包括：
* [[经院哲学|经院哲学]]
* [[人文主义|人文主义]]
* [[文艺复兴|文艺复兴]]
* [[啟蒙時代|啟蒙時代]]
* [[世俗化|世俗化]]
* [[科学方法|科学方法]]

== 历史 ==
[[File:Hw-shakespeare.jpg|thumb]]，对现代西方文学影响重大。]]
前1世纪，[[罗马帝国|罗马帝国]]攻佔希腊后，吸收了古希腊的[[科学|科学]]、[[哲学|哲学]]、[[民主|民主]]、[[古典建筑|建筑]]、[[文学|文学]]以及[[艺术|艺术]]，並在此基础上作进一步的发展。

在接下来的500年中，尽管罗马帝国拒绝古代[[雅典|雅典]]很前卫的民主理念，但却将[[希腊语|希腊语]]、[[拉丁语|拉丁语]]随着[[罗马法|罗马法]]传遍了欧洲。罗马文化还融合了[[德国文化|日耳曼]]、[[马其顿文化|斯拉夫]]以及[[塞尔特文化|塞尔特]]文化，但随着[[罗马的衰落|罗马的衰落]]，希腊与罗马（{{lang|en|Greco-Roman}}）的许多艺术、文学以及科学都消失或被取代了。

随着4世紀[[基督教|基督教]]成為[[羅馬帝國|羅馬帝國]]的[[國教|國教]]，[[圣经|圣经]]成为西方文艺中的核心部分，几乎影响到了西方文化的所有领域，包括艺术、法律、哲学、教育以及政治。罗马[[基督教|基督教]]成立了许多神学院，现在的许多大学、学院即起源于此，总体而言这些学院促进了早期西方文化的传播。[[阿拉伯人|阿拉伯文化]]保存了一些古希腊和古罗马的知识，随着[[十字軍東征|十字軍東征]]，[[西班牙|西班牙]]、[[黎凡特|黎凡特]]的[[摩尔人|摩尔人]]所具有的阿拉伯文化，对西欧产生影响，终于在14世纪，希腊的文化遗产重又被西欧发现，于是[[文艺复兴|文艺复兴]]诞生了。

[[File:Whitehouse_north.jpg|right]]建筑仿照了古希腊圣殿的样式]]
16世纪之后，文艺复兴的西方文化被[[地理大发现|探险者]]、[[殖民主义|殖民者]]、[[贸易|商人]]以及[[传教士|传教士]]传播到[[新大陆|新大陆]]。随后的[[启蒙时代|启蒙时代]]，在[[美国革命|美国革命]]、[[法国大革命|法国大革命]]时达到顶峰。一些理念，比如[[公民權利|公民權利]]、[[法律之前人人平等|法律之前人人平等]]、[[司法公正|司法公正]]以及[[民主|民主]]，这些[[社会|社会]]构成的理想，如今这些准则已经成为现代西方文化的基石。

19世纪後，[[美国|美国]]开始发展自成一体的西方文化，1950年代后，以[[美國文化|美國文化]]為主的西方文化在全球占据了主导地位，连同美国时尚、娱乐、技术以及政治，泛滥于其他[[西方世界|西方世界]]。

== 霸权 ==
[[File:Shibuya_tokyo.jpg|thumb]]在保留传统的同时，深受西方文化的影响。]]
在世界范围内，西方文化中的元素对其他文化具有很强的影响力。许多文化背景的人，包括西方与非西方的，都将“现代化”与“西化”等量齐观，当然也有许多非西方人士，反对将采纳西方理念与价值，作为所有社会的必然。一些非西方世界的成员，他们将科技进步跟西方的不同价值观相联系，由于西方价值观与他们当地舊有社会的价值理念大相径庭，以此作为理由，来反对大部分的“[[现代化|现代化]]”。

== 多元文化 ==
由于在文化、艺术、文学、历史方面的基石作用，西方文化在欧美学校课程中占据了支配地位。1970年代开始，因[[新左派|新左派]]影響和[[移民|移民]]的湧入，加上社會的[[世俗化|世俗化]]，開始奉行[[多元文化主義|多元文化政策]]，开始鼓励在西方学习非洲、亚洲的文化、历史与艺术，这项政策在进入1990年代后隨著[[移民|移民]]的進入进展加快。
