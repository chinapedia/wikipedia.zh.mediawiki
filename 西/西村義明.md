'''西村義明'''（{{lang-ja|西村義明}}，{{bd|1977年|9月25日|||Nishimura Yoshiaki}}）是曾隸屬[[日本|日本]][[吉卜力工作室|吉卜力工作室]]的影片製作人、以及動畫工作室[[Studio_Ponoc|Studio Ponoc]]的創建者。

== 經歷 ==
自美國留學返回日本後，西村義明加入吉卜力時最初是在影片《[[霍爾的移動城堡|霍爾的移動城堡]]》中擔任業務方面的工作。<ref name="Page90">{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=90頁 |accessdate=2014年5月16日 }}</ref>

在2006年期間吉卜力代理由工作室內的導演[[高畑勳|高畑勳]]所翻譯之法國動畫《[[國王與小鳥|國王與小鳥]]》時，西村義明在處理此發行作品的宣傳事務中、與高畑勳有較多接觸的機會。隨後在吉卜力多部作品擔任影片製作人的[[鈴木敏夫|鈴木敏夫]]，進一步指派西村義明成為高畑勳構思已久的作品《[[輝耀姬物語|輝耀姬物語]]》之影片製作人。而西村義明則與高畑勳合作數年後、於2013年將此作品完成上映。<ref name="Page90"/>

2014年3月間鈴木敏夫在舉辦記者會上，宣佈今起將不參與吉卜力的影片製作人方面事項，而將此職位往後交接給西村義明擔任。<ref>{{cite web |url=http://www.cinematoday.jp/page/N0061288 |script-title=ja:ジブリ鈴木敏夫、“勇退”の真相語る「落合監督がうらやましかった」 |publisher=Cinema Today |date=2014年3月10日 |accessdate=2014年5年16日}}</ref><ref>{{cite web |url=http://www.cinemacafe.net/article/2014/03/22/22429.html |script-title=ja:ジブリ新時代を担う“西村義明”ってどんな人？　『かぐや姫の物語』＆新作をプロデュース |publisher=Cinema Cafe |author=Ryo Uchida |date=2014年3月22日 |accessdate=2014年5年16日}}</ref>

後續西村義明與[[米林宏昌|米林宏昌]]在共同製作《[[回憶中的瑪妮|回憶中的瑪妮]]》完畢後；倆人雖有意再繼續拍攝動畫電影，但當時吉卜力已面臨計劃要解散動畫製作部門情況，隨後便自行成立個人動畫工作室[[Studio_Ponoc|Studio Ponoc]]來與米林宏昌合作繼續拍攝動畫。<ref>{{cite web |url=http://www.cinematoday.jp/page/N0088351 |script-title=ja:ジブリの血を受け継ぐ新作アニメに宮崎駿、高畑勲、鈴木敏夫の反応は？ |work=Cinema Today |date=2016年12月15日 |accessdate=2016年12月31日}}</ref>
