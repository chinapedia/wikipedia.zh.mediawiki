'''西山 宏太朗'''（1991年<ref name="auditon">{{Cite web|date=|url=http://www.81produce.co.jp/audition/past/2009.shtml|title=2009年 過去のオーディションレポート：81produce オーディション|work=|author=|publisher=[[81_Produce|81 Produce]]|accessdate=2014-03-19}}</ref>10月11日<ref name="pro">{{Cite web |url=http://www.81produce.co.jp/list.cgi?man+4222716115030 |title=西山宏太朗：所属俳優：81produce - 81プロデュース |publisher=[[81_Produce|81 Produce]] |date= |accessdate=2014-03-16 |archiveurl=https://web.archive.org/web/20140314171929/http://www.81produce.co.jp/list.cgi?man+4222716115030 |archivedate=2014-03-14 |deadurl=yes }}</ref> - ）是[[日本|日本]]男性[[声優|声優]]。出身於[[神奈川|神奈川]]。隸屬於[[81_Produce|81 Produce]]。

== 人物介紹 ==
參加了第3屆81試演會，並獲得特別賞<ref name="auditon"/>。2011年5月開始加入81 Produce<ref name="ニュース">{{Cite web |url=http://www.81produce.co.jp/news/ |title=ニュースバックナンバー：81produce - 81プロデュース |date= |accessdate=2014-03-16}}</ref>。

視力為1.5。

2018年，獲得第12回聲優獎的新人男優賞。
