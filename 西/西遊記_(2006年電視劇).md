《'''西遊記'''》（{{lang|ja|さいゆうき}}）是一部於2006年1月9日至3月20日起，由[[日本|日本]][[富士電視台|富士電視台]]於逢星期一21:00-21:54（[[日本時間|日本時間]]）播出的[[日本電視劇|電視連續劇]]。首集、大結局加長至22:24。

另外，本作的電影版公開播映於2007年7月14日（參看：[[西遊記_(2007年電影)|西遊記 (2007年電影)]]）。

== 故事背景 ==
{{main|西遊記}}
* 以[[中國|中國]]古典小說[[西遊記|西遊記]]為藍本，但改變了部分原有登場角色的設定。
* [[孫悟空|孫悟空]]的[[觔斗雲|觔斗雲]]改為翼的形態。當需要使用的時候，從口袋拿出。而三藏法師的愛駒玉龍不用登場，變成了4人徒步旅行。
* 中華影視作品中的[[唐三藏|唐三藏]]一直是由男性演出。但日本影視作品中的[[唐三藏|唐三藏]]，從1978年的版本開始就一直是由女性演出。（1978年版本的唐三藏由[[夏目雅子|夏目雅子]]飾演）。

== 登場人物及演員 ==
=== 主要登場人物 ===
; [[孫悟空|孫悟空]]（[[香取慎吾|香取慎吾]]）（[[陳卓智|陳卓智]]）配音
: 從石頭裡誕生，具高戰鬥能力的石猴，但在500年間，被釋迦<ref>因為到了大結局才知道釋迦的樣貌，故此存在矛盾。</ref>封印。當三藏幫他解除封印後，便陪三藏一起直至到達天竺。具有像年幼調皮的小孩那樣的粗暴性格。於本作中，是三藏的第三弟子<ref>原作為第一弟子，第三弟子為[[沙悟淨|沙悟淨]]。</ref>

: 本作中三藏的第一弟子。<ref>原作為第三弟子，第一弟子為[[孫悟空|孫悟空]]。</ref>樣子很酷，頭腦很好，愛裝模作樣，但是十分喜歡女人<ref>原作設定[[豬八戒|豬八戒]]十分喜歡女人。</ref>

; [[豬八戒|豬八戒]]（[[伊藤淳史|伊藤淳史]]）（[[李錦綸_(配音員)|李錦綸]]）配音

; [[玄奘|三藏法師]]（[[深津繪里|深津繪里]]）（[[鄭麗麗|鄭麗麗]]）配音
: 為了救殺氣騰騰的世界，和三個弟子一起到天竺取經。
