'''西南交通大学希望学院'''是一所由[[西南交通大学|西南交通大学]]与[[华西希望集团|华西希望集团]]校企合办的[[独立学院|独立学院]]。学校建校于2009年。
[[File:西南交大希望学院校门.jpg|right]]

== 历史沿革 ==
* 2009年6月16日，学校成立。同年8月，学校首次招生<ref>{{cite news|author=夏丽莎;方野|title=西南交通大学希望学院成立 今年拟首批在川招生一千八百名|url=http://sichuandaily.scol.com.cn/2009/06/17/20090617741554178173.htm|accessdate=2014-12-16|newspaper=四川日报|date=2009-06-17}}</ref>。
* 2015年，学校成为四川省首批转型应用技术型的本科院校。<ref>{{cite news|title=西昌学院等四川三所高校向应用型本科转型发展|url=http://news.163.com/15/0318/06/AKVGON4700014Q4P.html|accessdate=2016-09-27|agency=新华社|date=2015-03-18}}</ref>

== 校园环境 ==
目前学院有成都、南充两个校区。成都校区位于四川省[[成都市|成都市]][[金堂县|金堂县]]、南充校区位于四川省[[南充市|南充市]]<ref name="2013年">{{cite web|title=西南交通大学希望学院|url=http://news.xinhuanet.com/edu/2013-10/23/c_125585814.htm|publisher=[[新华网|新华网]]教育频道|accessdate=2014-12-16|deadurl=yes|archiveurl=https://web.archive.org/web/20141218221655/http://news.xinhuanet.com/edu/2013-10/23/c_125585814.htm|archivedate=2014年12月18日|df=}}</ref>。

== 院系设置 ==
学院下设土木工程系、经济管理系、交通运输系、机械与电气工程系、外语系、联合系等系部<ref name="2013年"/>。
