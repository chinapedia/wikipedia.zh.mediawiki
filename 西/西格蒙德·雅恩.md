'''西格蒙德·维尔纳·保罗·雅恩'''（[[德语|德语]]:'''Sigmund Werner Paul Jähn'''，{{bd|1937年|2月13日|catidx=Jähn}}），[[德意志民主共和国|德意志民主共和国]][[空军|空军]]少将。因德意志民主共和国参加了由苏联主导的「國際宇宙計劃（[[俄語|俄語]]：ИнтерКосмос）」，由此成為德國第一位[[航天員|航天員]]。

== 生平 ==
出生于[[萨克森州|萨克森州]]的福格特兰。1955年加入德意志民主共和国人民军空军。1966-1970年在[[苏联|苏联]]加加林空军学院学习，之后在德意志民主共和国人民军空军任教。
于1978年9月3日被授予[[苏联英雄|苏联英雄]]荣誉称号。1983年，于[[波茨坦|波茨坦]]获得德意志民主共和国中央学院物理学博士学位。
== 航天 ==
[[File:Sigmund_Jähn_Portrait_TU_Berlin.jpg|thumb]]
1976年，西格蒙德·雅恩和他的替补埃伯哈德·科尔勒被选定为成为“国际宇宙计划”的航天员。經過了两年的专业训练，于1978年8月26日乘坐“联盟-31号”宇宙飞船升空，成功与苏联“礼炮6号”空间站对接进入空间站，期间他和空间站的航天员一起进行了多项科学实验工作，于1978年9月3日乘“联盟-29号”飞船返回地球。由此在德意志民主共和国婦孺皆知。

雅恩分別於1990年與1993年起擔任[[德國航空太空中心|德國航空太空中心]]（DLR）與[[歐洲太空總署|歐洲太空總署]]的顧問，為{{link-en|Euromir|Euromir}}任務提供建議，2002年退休。

== 軼事 ==
在2003年的德國電影《[[再見，列寧！|再見，列寧！]]》中，由Stefan Walz飾演西格蒙德·雅恩。
