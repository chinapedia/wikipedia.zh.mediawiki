'''西安－延安高速公路'''，简称：'''西延高速'''，是[[陕西省|陕西省]]“2367”高速公路网三条南北纵向线中的重要组成部分，也是[[中国国家高速公路网|中国国家高速公路网]][[包茂高速公路|包茂高速公路]]（'''G65'''）陕西境内的重要一段。

西延高速公路是横跨中国[[黄土高原|黄土高原]]地区的第一条高速公路，南起[[西安绕城高速公路|西安绕城高速公路]]吕小寨立交，途经[[西安|西安]]、[[咸阳|咸阳]]、[[铜川|铜川]]、[[延安|延安]]、[[三原|三原]]、[[宜君|宜君]]、[[黄陵|黄陵]]、[[洛川|洛川]]、[[富县|富县]]、[[甘泉|甘泉]]，止于[[延安市|延安市]]西北的[[河庄坪|河庄坪]]，全长299.85公里。全线共有22座隧道（单洞），隧道总长27公里，是世界罕见的黄土隧道群；各式桥梁369座，其中[[洛河特大桥|洛河特大桥]]高达152.9米，被称为“亚洲第一高墩大桥”；沿线设有收费站9处、服务区5处，配备有完善的交通、通讯及收费系统等设施。

== 路段 ==
* 西安－三原：1989年9月通车，设计时速100km/h
* 三原－铜川：1992年10月通车，设计时速100km/h
* [[铜黄高速公路|铜川－黄陵]]：2001年5月通车，设计时速60km/h
* 黄陵－延安：143.205公里，2006年9月30日通车，设计行车速度80公里/小时<ref>{{cite web|url=http://news.sina.com.cn/c/2006-10-01/004810149273s.shtml|title=陕西黄陵至延安高速通车|date=2006年10月1日|publisher=新浪网}}</ref>。

== 改扩建 ==
由于西延高速公路建成时间较早，公路线型、建设标准均相对较低。其中，西安至铜川段为一级公路改建，设计车流量仅为1万辆/日，而随着沿线地区经济发展及高速公路网的形成，该路段的实际车流量已达日均3.8万辆<ref>[http://news.163.com/10/0514/04/66KA6NC8000146BD.html 网易：西铜高速大规模“整容”]</ref>；铜川－黄陵段设计车流量为1.9万辆/日，而实际车流量为日均3.5万辆，高峰期超过5万辆<ref>[http://jt.china.com.cn/news/NewsImgInfo.aspx?id=48631 中国交通：西延高速公路道路保畅纪实]</ref>，严重影响正常通行。因此，陕西省决定另辟新线建设西安－铜川－黄陵高速公路，当时又称“西黄高速第二通道”，即“'''延西高速'''”（'''G65<sub>W</sub>'''）<ref>[http://news.sina.com.cn/o/2008-11-27/041614791244s.shtml 新浪网：西黄高速第二通道昨开建]</ref>。

同时，对既有的西延高速黄陵至延安段进行扩容，沿既有高速公路西侧，新建双向六车道高速公路为主，总投资201亿元人民币，起于铜川市宜君县崖头庄，止于延安市[[安塞县|安塞县]][[沿河湾镇|沿河湾镇]]南侧，全长153.909公里。计划2016年底前建成通车。<ref>{{cite web|url=http://news.hsw.cn/system/2014/02/28/051868257.shtml|title=黄陵至延安高速将增双向六车道 2016年前建成通车|author=杜鹃|date=2014-02-28|publisher=华商网}}</ref><ref>{{cite web|url=http://www.sn.xinhuanet.com/2014-03/11/c_119701894.htm|title=黄陵至延安高速公路扩能项目计划2015年建成通车|author=张黎娜 徐成英|date=2014-03-11|publisher=西安新闻网|deadurl=yes|archiveurl=https://web.archive.org/web/20141025191047/http://www.sn.xinhuanet.com/2014-03/11/c_119701894.htm|archivedate=2014年10月25日|df=}}</ref>

== 运营管理 ==
2001年6月27日，西延高速公路西安－三原－铜川段20年经营权转让给陕西西铜高速公路有限公司，由其负责管理；其余路段由陕西省高速公路建设集团公司西延分公司负责运营管理。
