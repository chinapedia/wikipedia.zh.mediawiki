'''西楠''' ({{bd|1985年|12月15日}})是旅英女[[作家|作家]]、[[诗人|诗人]]和[[翻譯家|翻譯家]]。

== 生平 ==
西楠，原名黄南希，1985年12月15日出生于中国湖南[[湘西|湘西]]，畢業於[[伦敦政治经济学院|伦敦政治经济学院]]，获[[比较政治学|比较政治学]]硕士学位，曾任媒体记者、编辑、翻译、[[BBC|BBC]]中文部网站专栏作家。
现旅居[[伦敦|伦敦]]，是[[歐洲華文作家協會|歐洲華文作家協會]]會員。<ref> [http://search.dangdang.com/author/%ce%f7%e9%aa_1 西楠作品_西楠简介] </ref>


== 寫作 ==
西楠认为，写作是生命过程中的发问与探索，是在生命过程中构筑的一个世界。西楠推崇自由、独立的精神。作品中呈现真实与自我剖析，记录生活本质，不择美丑，直面破碎。她的长篇小说《纽卡斯尔，幻灭之前》获首届“紫金 •[[人民文学|人民文学]]之星”长篇小说类提名奖。

== 獲獎 ==
“紫金•[[人民文學|人民文學]]之星”長篇小說提名獎。<ref>[http://xi-nan.wixsite.com/xi-nan/single-post/2013/10/21/%E3%80%8A%E7%B4%90%E5%8D%A1%E6%96%AF%E7%88%BE%EF%BC%8C%E5%B9%BB%E6%BB%85%E4%B9%8B%E5%89%8D%E3%80%8B%E7%8D%B2%E2%80%9C%E7%B4%AB%E9%87%91%C2%B7%E4%BA%BA%E6%B0%91%E6%96%87%E5%AD%B8%E4%B9%8B%E6%98%9F%E2%80%9D%E9%95%B7%E7%AF%87%E5%B0%8F%E8%AA%AA%E6%8F%90%E5%90%8D%E7%8D%8E- 紫金•人民文學之星提名獎]</ref>
