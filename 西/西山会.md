'''西山会'''，是一个由[[中國|中國]][[山西|山西]]籍在[[北京|北京]]的高官及同籍商人组成的一个神秘组织，由[[令计划|令计划]]于2007年建立<ref name="真话">{{cite news |title = 《打铁记》第15章：山西“西山会” |url = http://zhenhua.163.com/13/1011/11/9ATEKTCR000464E2.html |publisher = 网易真话 |date =  |accessdate = 2015-01-24 |archiveurl = https://web.archive.org/web/20150113223156/http://zhenhua.163.com/13/1011/11/9ATEKTCR000464E2.html |archivedate = 2015-01-13 |deadurl = yes }}</ref><ref>{{cite news|title =揭秘令计划与“西山会”：广纳山西籍官员和商人|url =http://www.mnw.cn/news/china/834737.html|publisher =闽南网|date =2014-12-23|accessdate =2015-01-24|deadurl =yes|archiveurl =https://web.archive.org/web/20150128133231/http://www.mnw.cn/news/china/834737.html|archivedate =2015年1月28日|df =}}</ref>。境外中文媒体报道称这是一个松散的有组织者的政商联盟<ref name="多维">{{cite news |title = 西山会引爆舆论 令计划的政治图谋 |url = http://china.dwnews.com/news/2014-12-24/59626002.html |publisher = 多维新闻 |date = 2014-12-24 |accessdate = 2015-01-24}}</ref>。

== 简介 ==
西山会由的山西籍[[中共中央委员|中共中央委员]]和[[中共中央候补委员|中共中央候补委员]]组成，除官员外，只有个别获得身份认可的同籍商人，才能拥有[[埋单|埋单]]的资格<ref name="真话" />。西山会内部没有固定章程，没有组织程序，也没有固定地点，甚至不会有特殊的秩序编排。西山会的会议地点位于北京西郊，以不低于三个月一次的聚会频率保持联络。聚会期间有豪车负责接送，手机、秘书、情人必须隔离。如果官员手握通往西山[[饭局|饭局]]的门票，则可以有升职机会<ref name="真话" />。
