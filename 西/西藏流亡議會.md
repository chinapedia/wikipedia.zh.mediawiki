'''西藏人民議會'''（{{lang-bo|'''གྲོས་ཚོགས་ཐོག་གསུང་བཤད།'''}}），俗稱為'''西藏流亡議會'''，是[[西藏流亡政府|西藏流亡政府]]的[[立法機構|立法機構]]，實行[[一院制|一院制]]。第一屆流亡議會成立于1960年。

== 歷史 ==

== 議會概況 ==
議會現時一般由43—46名議員組成，包括<ref name=legisl>{{zh-tw}}[http://www.tibet.org.tw/tibet_legisl.php 人民議會] 達賴喇嘛西藏宗教基金會</ref>：

# 居住在歐洲的流亡藏人代表（2名）；
# 居住在北美地區（加拿大和美國）的流亡藏人代表（1名）；
# 西藏三個区域的流亡居民代表（各省10名，且各省女性議員不得少于2名）；
# [[藏傳佛教|藏傳佛教]]四大宗派（[[寧瑪派|寧瑪派]]、[[噶舉派|噶舉派]]、[[薩迦派|薩迦派]]、[[格魯派|格魯派]]）以及[[苯教|苯教]]代表（每個教派各2名）
# 還有1—3名由達賴喇嘛委任的在教育、衛生、科技、公共服務領域的杰出人士。

議會在每年6個月的休會期間會設立常務會議，由12名議員組成，分別為西藏三个区域各2名，五大教派各1名，達賴喇嘛委派1名。每年常務會議都會進行一次改選。

== 選舉 ==
=== 參選資格 ===
根據《西藏流亡憲章》的規定，符合以下資格的25周歲以上的「西藏公民」，不論性別、宗教、家庭情況即可參選議員：
# 未經法院為合法醫生宣布為心理不正常；
# 無債務；
# 無被法院定罪；
# 不任職于任何政府機構及其所屬機構；
# 無接受過任何危害西藏人民利益的國家的利益輸送；
# 不違背選舉法和其他法理、憲章。 

=== 選民資格 ===
凡年滿18周歲的「西藏公民」不論性別、宗教、家庭情況都擁有[[投票權|投票權]]。
