'''西名阪自動車道'''（{{jpn|j=西名阪自動車道|hg=にしめいはんじどうしゃどう|rm=Nishimeihan jidōshadō|f=y}}）是從[[大阪府|大阪府]][[松原市_(日本)|松原市]]道[[奈良縣|奈良縣]][[天理市|天理市]]的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]）。略稱為'''西名阪道'''（{{lang|ja|西名阪道／にしめいはんどう}}、NISHI-MEIHAN EXPWY）、'''西名阪'''（{{lang|ja|西名阪／にしめいはん}}）。

高速公路編號與[[名阪國道|名阪國道]]被共同編為 {{Ja Exp Route Sign|E25}}。

== 概要 ==
1969年（昭和44年）3月全線通車。[[國土開發幹線自動車道|國土開發幹線自動車道]]的預定路線名為'''[[近畿自動車道#國土開發幹線自動車道|近畿自動車道名古屋大阪線]]'''，高速自動車國道的路線名诶'''[[近畿自動車道#高速自動車國道|近畿自動車道天理吹田線]]'''，相當於[[名古屋市|名古屋]] - [[大阪市|大阪]]間的部分路段。

[[奈良縣|奈良縣]]內唯一的[[高速自動車國道|高速自動車國道]]。通車以來皆為收費，當初的4年間為[[一般國道|一般國道]][[國道25號|25號]]的繞道。

與連接的[[名阪國道|名阪國道]]和[[東名阪自動車道|東名阪自動車道]]共同具有[[名神高速公路|名神高速公路]]的[[繞道|繞道]]功能。與名神高速公路相比，本道路是通行名古屋 - 大阪間（特別是[[國道308號|中央大通]]以南的府内一帶）的最短經路，通行費也較便宜。儘管名阪國道的線形不佳，仍然是大貨車常常行經的路線。

另外，東名阪自動車道、名阪國道與[[伊勢灣岸自動車道|伊勢灣岸自動車道]]、[[新名神高速公路|新名神高速公路]]可共同在名神高速公路因大雪而暫停通行之際，發揮前往[[京阪神|京阪神]]地區替代路線的機能。

== 歷史 ==
* 1969年（[[昭和|昭和]]44年）3月21日 : [[日本道路公團|日本道路公團]]管理的[[收費道路|一般收費道路]]「'''西名阪道路'''」（[[一般國道|一般國道]][[國道25號|25號]]）舊松原交流道（松原系統交流道） - 天理交流道間通車。
* 1973年（昭和48年）4月1日 : 以'''西名阪自動車道'''名義昇格編入高速自動車國道。
* 1980年（昭和55年）3月1日 : 與阪神高速14號松原線於松原交流道連接（但通車當時，松原交流道僅與阪神高速和西名阪道直接連接）。
* 1981年（昭和56年）：因應通行費收取的簡單化，從距離制收費改成區間別均一收費（松原 - 香芝交流道間・香芝 - 天理交流道間）。
* 1988年（昭和63年）3月17日 : 近畿自動車道 全線通車並與本道路連接，同時舊松原交流道改為松原系統交流道。
* 1999年（[[平成|平成]]11年）4月 : 為緩和[[松原收費站|松原收費站]]附近經常性的交通堵塞，往大阪方向行車線的通行費徵收業務移轉至新設置的[[柏原收費站|柏原收費站]]（松原收費站改為往名古屋方向行車線專用）。
* 2012年（平成24年）7月4日 : 大和真秀場智慧型交流道（名古屋方向入口、大阪方向出口） 啟用。
* 2014年（平成26年）3月23日 : 大和真秀場智慧型交流道（名古屋方向出口、大阪方向入口） 啟用<ref>[http://corp.w-nexco.co.jp/corporate/release/kansai/h26/0219/ 西名阪自動車道 大和真秀場智慧型交流道(大阪側出入口)於平成26年3月23日(日曜)通車　― 大阪側出入口通車後，自此全面通車 ―] 西日本高速道路株式会社・奈良縣　2014年2月19日付</ref>。
* 2015年（平成27年）3月22日 : 郡山下道系統交流道通車，與京奈和自動車道連接。

=== 通行費 ===
* 天理交流道 - 香芝交流道 : 均一制
* 香芝交流道 - 近畿自動車道 [[松原交流道|松原交流道]]・[[長原交流道|長原交流道]] : 均一制

兩區間皆採用均一收費制。各區間通行費相同如下所列。若行駛超過香芝交流道需支付兩次通行費。

* 輕自動車等 : 310圓
* 普通車 : 410圓
* 中型車 : 410圓
* 大型車 : 570圓
* 特大車 : 930圓

僅在下列路段使用[[ETC|ETC]]無線通行時，可適用折扣通行費。<ref>[http://corp.w-nexco.co.jp/corporate/release/kansai/h27/0202/pdfs/06.pdf 西名阪自動車道通行費] - NEXCO西日本</ref>
* 天理交流道 - 郡山下道系統交流道 (普通車 260圓)
* 郡山下道系統交流道 - 大和真秀場智慧型交流道 (普通車 250圓)
郡山交流道與郡山下道系統交流道緊鄰，但是自郡山交流道出入無法獲得折扣。郡山下道系統僅與京奈和道連接，若要從一般道路出入必須利用[[三宅交流道_(奈良縣)|三宅交流道]]或[[橿原北交流道|橿原北交流道]]。
