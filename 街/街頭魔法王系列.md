'''街頭魔法王系列'''，是[[無綫電視|無綫電視]]一個[[綜藝節目|綜藝節目]]系列，由[[街頭魔法王|街頭魔法王]]、[[街頭魔法王2|街頭魔法王2]]、[[至尊街头魔法王|至尊街头魔法王]]、[[街頭魔法王之魔王之王|街頭魔法王之魔王之王]]以及[[終極街頭魔法王|終極街頭魔法王]]組成。以[[街頭魔術|街頭魔術]]表演爲主。

*第一輯《[[街頭魔法王|街頭魔法王]]》節目於2013年11月9日至2013年11月30日播放。
*第二輯《[[街頭魔法王2|街頭魔法王2]]》節目於2014年7月19日至2014年9月13日播放。
*第三輯《[[至尊街頭魔法王|至尊街頭魔法王]]》節目於2015年3月30日至2015年4月10日播放。
*第四輯《[[街頭魔法王之魔王之王|街頭魔法王之魔王之王]]》節目於2016年7月18日至2016年8月5日播放。
*第五輯《[[終極街頭魔法王|終極街頭魔法王]]》節目於2017年10月16日至2017年10月27日播放。

===第一輯（街頭魔法王）===
{{Main|街頭魔法王}}
於2013年11月9日首播，為街頭魔法王系列的首輯節目。節目於星期六晚上八時三十分至九時三十分播出。每集由阿Bob[[林盛斌|林盛斌]]及[[翟凱泰|泰臣]]主持並由魔術師Louis[[甄澤權|甄澤權]]主演，一共四集。

四集節目分別請來一位美女助手，在不同的地方與Louis會合，並與觀眾一起變出不同的魔術。五位美女助手分別為[[萬綺雯|萬綺雯]]、[[馬賽_(藝人)|馬賽]]、[[黃山怡|黃山怡]]（糖妹）、[[陳煒_(香港藝人)|陳煒]]及[[鄧麗欣|鄧麗欣]]。

===第二輯（街頭魔法王2）===
{{Main|街頭魔法王2}}
於2014年7月19日首播，為街頭魔法王系列的第二輯節目。本輯節目於星期六晚上八時三十分至九時三十分播出。每集由阿Bob[[林盛斌|林盛斌]]、[[許亦妮|許亦妮]]、[[陳嘉寶|陳嘉寶]]及[[梁烈唯|梁烈唯]]主持並由魔術師Louis[[甄澤權|甄澤權]]主演，一共九集。

九集節目加入了全新的環節，讓一眾男、女藝員參加節目，進行淘汰賽，以角逐「魔法新人王」的美譽。

===第三輯（至尊街头魔法王）===
{{Main|至尊街头魔法王}}
於2015年3月30日首播，為街頭魔法王系列的第三輯節目。有別於頭兩輯節目，本輯節目於星期一至五晚上十時三十分至十一時播出。每集由阿Bob[[林盛斌|林盛斌]]及[[吳若希|吳若希]]主持並由魔術師Louis[[甄澤權|甄澤權]]主演，一共十集。

本輯節目迎增《Louis決戰甄澤權》的環節，此環節是讓魔術師甄澤權挑戰其曾在上兩輯表演的魔術並把魔術升級。例如第一輯《街頭魔法王》把叉燒飯變為餐蛋麵，而在這輯的第四集中，甄澤權把叉燒飯變為半叉燒飯半餐蛋麵。

===第四輯（街頭魔法王之魔王之王）===
{{Main|街頭魔法王之魔王之王}}
於2016年7月18日首播，為街頭魔法王系列的第四輯節目<ref>{{cite news |language = 中文  |url = http://hk.on.cc/hk/bkn/cnt/entertainment/20160722/bkn-20160722044516211-0722_00862_001.html |title = 甄澤權大玩銀紙變煙冧死女！ |publisher = 東網  |date = 2016年7月22日 |accessdate = 2016年7月22日}}</ref>。本輯節目於星期一至五晚上十時三十分至十一時播出。每集由阿Bob[[林盛斌|林盛斌]]、[[吳若希|吳若希]]、[[陳凱琳|陳凱琳]]及[[翟威廉|翟威廉]]主持並由魔術師Louis[[甄澤權|甄澤權]]主演，一共十五集。

本輯節目在第六至第十集分別有五位台灣魔術師登場，而在第十一集至第十五集則是他們的韓國篇，Louis還會在韓國街頭表演魔術， 還有五位韓國的帥哥，美女魔術師登場。

===第五輯（終極街頭魔法王）===
{{Main|終極街頭魔法王}}
於2017年10月16日首播，為街頭魔法王系列的第五輯節目。本輯節目於星期一至五晚上十時三十分至十一時播出。每集由阿Bob[[林盛斌|林盛斌]]、[[翟威廉|翟威廉]]、[[陸浩明|陸浩明]]、[[何雁詩|何雁詩]]、[[林穎彤|林穎彤]]主持並由魔術師Louis[[甄澤權|甄澤權]]主演，一共十集。

本輯節目迎增《魔法大整蠱》的環節，Louis於第一至五集、第七集及第十集用魔術整蠱嘉賓，而第八集用魔術整蠱現場街坊。

==軼事==
===撤換主持===
《[[街頭魔法王2|街頭魔法王2]]》中的主持[[許亦妮|許亦妮]]未能在新一輯《[[至尊街頭魔法王|至尊街頭魔法王]]》中繼續擔任主持，有傳是由於她工作態度欠佳和向公司提出不合理的待遇而被雪藏。取而代之的是無綫近日力捧的新人[[吳若希|吳若希]]，唯[[吳若希|吳若希]]否認自己「執二攤」，並指自己擔任主持的決定是公司因應每輯節目都換主持的習慣而定。<ref>{{cite news |language = 中文  |url = http://orientaldaily.on.cc/cnt/entertainment/20150305/00282_011.html |title = 吳若希嫌新歌唔夠慘 |publisher = 東方日報  |date = 2015年3月5日 |accessdate = 2015年5月5日}}</ref><ref>{{cite news |language = 中文  |url = http://www.the-sun.on.cc/cnt/entertainment/20150305/00470_042.html |title = 吳若希否認執二攤 |publisher = 太陽報  |date = 2015年3月5日 |accessdate = 2015年5月5日}}</ref><ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/enews/realtime/20150303/53491589 |title = 【無處不在】許亦妮被無綫掟入雪櫃 吳若希「攝位」做主持 |publisher = 蘋果日報  |date = 2015年3月3日 |accessdate = 2015年5月5日}}</ref>

===杯麵魔術爭議===
在第三輯節目《[[至尊街頭魔法王|至尊街頭魔法王]]》中，甄澤權所表演的杯麵魔術被觀眾拍下穿崩片段<ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/enews/realtime/20150411/53624372 |title = 【掩眼法都冇】網民直擊 踢爆《街頭魔法王》換杯麵「魔術」 |publisher = 蘋果日報  |date = 2015年4月11日 |accessdate = 2015年5月5日}}</ref>，同時被一名[[澳門|澳門]]魔術師[[劉寶明|劉寶明]]指出為抄襲著名魔術師[[劉謙|劉謙]]為其本人創作的魔術<ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/entertainment/first/20150413/19110105 |title = 穿崩片引發行家爆大鑊 Louis杯麵魔術被轟抄橋 |publisher = 蘋果日報  |date = 2015年4月13日 |accessdate = 2015年5月5日}}</ref>。而甄澤權方面則指指控完全失實，並會影響其個人聲譽，所以會向採取劉寶明法律行動<ref>{{cite news |language = 中文  |url = http://www.orientaldaily.on.cc/cnt/entertainment/20150414/00282_047.html |title = 甄澤權控告魔術師失實|publisher = 東方日報  |date = 2015年4月14日 |accessdate = 2015年5月5日}}</ref>。同時，他承認魔術的保密工夫不足，並有所疏忽，並希望日後可繼續為觀眾帶來致多歡樂<ref>{{cite news |language = 中文  |url =  http://hk.apple.nextmedia.com/realtime/magazine/20150412/53625344|title = 【FACE即時】《街頭魔法王》魔術穿崩 Louis認錯：係我自己忽略咗 |publisher = FACE  |date = 2015年4月12日 |accessdate = 2015年5月5日}}</ref>。詳見[[至尊街頭魔法王_#爭議|至尊街頭魔法王]]。
