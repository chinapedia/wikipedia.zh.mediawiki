'''義神星'''（5 Astraea），平均直徑119千米，由[[卡尔·路德维希·亨克|亨克]]（K. L. Hencke）於1845年12月8日發現，是第五顆被發現的[[小行星|小行星]]。

== 命名 ==
早期發現的小行星大多以[[希臘神話|希臘神話]]中的女神來命名，[[阿斯特莉亞|阿斯特莉亞]]（Astraea）為[[宙斯|宙斯]]與[[忒弥斯|忒弥斯]](Themis)所生的正義女神。中文譯為義神星。

== 發現 ==
亨克是個任職於[[郵局|郵局]]的業餘天文愛好者，他在尋找[[灶神星|灶神星]]時發現了義神星，這發現更令他獲[[普魯士|普魯士]]國王[[腓特烈·威廉四世|腓特烈·威廉四世]]頒贈獎金。他後來再發現[[小行星6|小行星6]]。

== 特徵 ==
義神星是顆頗大的小行星，它的[[反照率|反照率]]甚高，其成分可能是鎳－鐵與[[硅酸鎂|硅酸鎂]]及[[硅酸鐵|硅酸鐵]]的混合物。測光發現它是逆向自轉。義神星沒有太大的特色，然而它是自發現首四顆小行星38年後再發現的小行星，打破了當時認為太陽系只有四顆小行星的說法。
