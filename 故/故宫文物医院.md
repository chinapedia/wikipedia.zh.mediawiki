'''故宫文物医院'''，位于[[北京故宫|北京故宫]]内的西河沿，是[[故宫博物院|故宫博物院]]文保科技部展示区<ref>{{Cite web |url =  http://news.ifeng.com/a/20161230/50498523_0.shtml|title = 故宫文物医院专给国宝看病 |author =  |date = 2016-12-30 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

==简介==
故宫文物医院位于北京故宫内的最西部，[[金水河_(北京)|内金水河]]西侧，此地在故宫内传统上称西河沿。西河沿区域在[[清朝|清朝]]时有一排连房，为[[酒醋房|酒醋房]]等机构的办公地点。建筑基址大体在清朝[[道光|道光]]年间被毁并彻底废弃，原址成为垃圾坑。西河沿区域的全部残留建筑自1950年代拆除后，该区域一直作为故宫博物院的料厂、木工厂、库房，场地堆放着许多木料及加工边角料，安全隐患严重。<ref name=bjrb>[http://bjrb.bjd.com.cn/html/2013-12/25/content_137169.htm 故宫宝蕴楼启动百年大修，北京日报2013年12月25日] {{webarchive|url=https://web.archive.org/web/20140404194818/http://bjrb.bjd.com.cn/html/2013-12/25/content_137169.htm |date=2014年04月4日 }}</ref>

2012年[[故宫博物院|故宫博物院]]院长[[单霁翔|单霁翔]]上任之初，便把将故宫博物院办公单位搬出红墙定为目标，2013年1月，[[故宫博物院|故宫博物院]]院长[[单霁翔|单霁翔]]介绍，故宫[[城隍庙_(北京故宫)|城隍庙]]和[[中国第一历史档案馆|中国第一历史档案馆]]所在的屏风楼之间的西河沿区域空地，目前被用作存放建筑材料及加工古建筑构件，现已经过审批将被恢复为约13000平方米的复古的文保科技用房，目前故宫博物院大部分办公及科研机构将被迁至此处，从而使故宫在夜间成为封闭的管理区。<ref>[http://news.artron.net/20130129/n305606.html 故宫博物院发展新格局 今年或将重提高峰限流，雅昌艺术网，2013-01-29]</ref>2013年12月24日，“西河沿文物保护综合业务用房”工程与[[宝蕴楼|宝蕴楼]]修缮工程同日启动。西河沿文物保护综合业务用房工程位于西河沿区域，西河沿区域北邻城隍庙，南邻中国第一历史档案馆，东邻[[寿康宫|寿康宫]]、[[寿安宫|寿安宫]]区域，西邻城墙，平面呈长方形，占地面积19675平方米。该工程经[[国家文物局|国家文物局]]、[[北京市文物局|北京市文物局]]、[[北京市规划和国土资源管理委员会|北京市规划委员会]]等部门批准，建成后将成为集合文物研究、实验、办公、科技交流等功能的综合性设施。该工程根据此处原有连房的历史资料，并结合专家意见及[[神武门|神武门]]内东西两侧长连房的样式进行设计，在外观、材料、建筑高度、檐口高度、建筑进深等方面均与神武门内东西两侧长连房基本相同，总建筑面积13025平方米。该工程完工以后，故宫博物院红墙内现有的9处办公单位都将迁出故宫红墙内，从而使红墙内作为完整文物体受到保护，消除安全隐患。<ref name=bjrb/>2015年，故宫博物院对故宫博物院地库至西河沿文物保护综合业务用房地下连接通道设计进行了招标。<ref>[http://www.zycg.gov.cn/article/show/327519 故宫博物院地库至西河沿文物保护综合业务用房地下连接通道设计中标公告，中央政府采购网，2015-04-29]</ref>

2016年，“西河沿文物保护综合业务用房”竣工，并定名为“故宫文物医院”。2016年12月29日，故宫文物医院正式挂牌开张。故宫文物医院面向故宫博物院内及院外承接文物修复工作，书画、木器、漆器、金属等多种文物的损伤均可修复。故宫文物医院分为科技保护区、分析检测区、书画修护区、综合工艺修护区、综合艺术品修护区、金属钟表修护区这6大区域。观众可通过预约，到故宫文物医院近距离参观文物修复过程<ref>{{Cite web |url =  http://news.xinhuanet.com/2016-12/30/c_1120222437.htm|title =故宫"文物医院"挂牌 观众可预约参观文物修复全过程  |author =  |date =  2016-12-30|publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

2017年11月8日下午，[[中华人民共和国主席|国家主席]][[习近平|习近平]]和夫人[[彭丽媛|彭丽媛]]陪同来华进行国事访问的[[美国总统|美国总统]][[唐纳德·特朗普|特朗普]]和夫人[[梅拉尼娅·特朗普|梅拉尼娅]]参观故宫文物医院，观赏了文物修复技艺展示，现场参与书画修复体验，并共同观看了珍品文物展<ref>{{Cite web |url = http://news.ifeng.com/a/20171108/53086158_0.shtml |title =全程视频：习近平夫妇陪同特朗普夫妇参观故宫博物院  |author =  |date =2017-11-08  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.ifeng.com/a/20171108/53087242_0.shtml |title = 特朗普在故宫体验书画装裱核心工序 专家：很内行  |author =  |date =2017-11-08 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
