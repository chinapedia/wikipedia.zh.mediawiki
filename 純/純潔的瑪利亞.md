{{日本作品|純潔的瑪利亞|純潔のマリア|Maria the Virgin Witch}}是[[石川雅之|石川雅之]]的[[日本|日本]][[漫畫|漫畫]]作品，於2008年11月開始，在[[講談社|講談社]]旗下雜誌《{{link-ja|good!Afternoon|good!アフタヌーン}}》之創刊號上連載，於2013年7月之第33號完結。外傳《純潔的瑪利亞 exhibition》於2014年7月在《good!Afternoon》的第45號上開始連載，於2015年1月號完結。2014年9月1日發表動畫化決定並開啟了官網，電視動畫在2015年1月開始播放<ref>{{cite web|title=《純潔的瑪利亞》動畫官網更新 公開新視覺圖及首支宣傳影片|url=http://gnn.gamer.com.tw/0/107440.html|publisher=巴哈姆特|accessdate=2015年1月19日|date=2014年12月4日}}</ref>。

== 故事簡介 ==
在戰爭不斷的[[中世紀|中世紀]]時代，在[[法蘭西王國|法蘭西王國]]與[[英格蘭王國|英格蘭王國]]交戰之[[百年戰爭|百年戰爭]]中，各國的[[魔女|魔女]]以此戰爭作為賺錢的途徑。而法國之魔女瑪利亞則厭惡戰爭，多次插手中止多次戰爭。

== 登場角色 ==
; {{nihongo|瑪利亞|マリア||聲：[[金元壽子|金元壽子]]}}
: 本作主角。[[魔女|魔女]]。因厭惡戰爭，多次使用魔法干涉並中止多場戰爭，也因此引起天界注意。
:睡覺時習慣[[裸睡|裸睡]]。

; {{nihongo|阿爾忒彌斯|アルテミス||聲：[[日笠陽子|日笠陽子]]}}
: 瑪利亞創造出來的[[魅魔|魅魔]]，原型為白色貓頭鷹。透過誘惑指揮官造成軍隊混亂而中止戰爭。

; {{nihongo|普里阿普斯|プリアポス||聲：[[小松未可子|小松未可子]]}}
: 瑪利亞創造出來的夢魔，原型為白色貓頭鷹。因為瑪利亞不懂男性身體結構，所以沒有重點部份。

; {{nihongo|以西結|エゼキエル||聲：[[花澤香菜|花澤香菜]]}}
: 由米迦勒派遣監視瑪利亞的人。平常會變為白鴿。

; {{nihongo|安|アン||聲：[[加隈亞衣|加隈亞衣]]}}
: 經常看望瑪利亞的小女孩，並不害怕作為魔女的瑪利亞。
: 對瑪利亞阻止戰爭的行為懷有感激之情，堅信瑪利亞絕不會害人。

; {{nihongo|瑪莎|マーサ||聲：[[一城美由希|一城美由希]]}}
: 安的奶奶。很久以前便知道瑪利亞的事情，和她是朋友，對她寄予了信賴。

; {{nihongo|約瑟夫|ジョセフ||聲：[[小野賢章|小野賢章]]}}
: 法國軍通信兵，經常拜訪瑪利亞。對瑪利亞抱持著好感。在大天使米迦勒準備制裁瑪利亞時，舉起弓箭對準了米迦勒。

; {{nihongo|[[米迦勒|米迦勒]]|ミカエル||聲：[[井上喜久子|井上喜久子]]}}
: 大天使。因瑪利亞以魔法介入人界的行動違背了天界的方針，從而對她下了「一旦失去處女便將喪失魔女力量」的法術。

; {{nihongo|維弗|ビブ||聲：[[能登麻美子|能登麻美子]]}}
: 來自[[英國|英國]][[溫徹斯特|溫徹斯特]]的魔女。髮型為長髮燙成{{link-ja|直捲|縦ロール}}。後來與瑪利亞成為朋友。

; {{nihongo|艾德溫娜|エドウィナ|聲：[[內山夕實|內山夕實]]}}
: 法國女巫，使魔為一隻貓。默默製作著各種草藥供教會轉而給人民使用，原本只想默默生存在人類社會當中，在劇情中後段挺身而出協助拯救瑪利亞。

=== 動畫原創 ===
; {{nihongo|迦爾法|ガルファ||聲：[[小野友樹|小野友樹]]}}
: 法國軍的傭兵。

; {{nihongo|伊萬|イーヴァン||聲：[[三宅健太|三宅健太]]}}
: 傭兵團「赤腕之兄弟團」的團長。

; {{nihongo|紀堯姆|ギヨーム||聲：[[島田敏|島田敏]]}}
: 領主。

; {{nihongo|伯納德|ベルナール||聲：[[櫻井孝宏|櫻井孝宏]]}}
: 修道院院長。

; {{nihongo|吉爾伯特|ジルベール||聲：[[花江夏樹|花江夏樹]]}}
: 侍奉伯納德的修道士。

; {{nihongo|蘿洛特|ロロット||聲：[[小島幸子|小島幸子]]}}
: 娼婦。

; {{nihongo|[[科爾努諾斯|科爾努諾斯]]|ケルヌンノス||聲：[[土師孝也|土師孝也]]}}
: 德魯伊信仰、凱爾特神話當中的神祇之一，富饒之神。在動畫版作品中沒有實體形象，並且是被人類遺忘的遠古存在之一，在森林裡作為整個故事的觀察者。
