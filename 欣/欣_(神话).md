'''欣'''(Sin)<ref>《世界神话辞典》，“南纳<苏-阿>”，第670页，辽宁人民出版社，1989年</ref>{{IPAc-en|ˈ|s|iː|n}}，又作“南纳”、“纳兹纳尔”([[苏美尔语|苏美尔语]]：{{lang|sux-Latn|<sup>[[dingir|D]]</sup>ŠEŠ.KI, <sup>D</sup>NANNA}})、“祖恩”(古[[阿卡德语|阿卡德语]]：Su'en)、“苏因”(古亚述语)。是[[阿卡德|阿卡德]]、[[亚述|亚述]]和[[巴比伦|巴比伦]]等[[美索不达米亚|美索不达米亚]]神话中月神。“南纳”(Nanna)是[[苏美尔|苏美尔]]之神，[[恩利勒|恩利勒]]和宁利勒(Ninlil)的长子，并演变为[[闪米特人|闪族]]神话中的“欣”。南纳/欣的崇拜中心主要有二个，分别是位于美索不达米亚南部的[[乌尔|乌尔]]城(Ur)和北部的[[哈兰_(土耳其)|哈兰]]城(Harran)。

==名字==
“南纳”之名的原意并不清楚。从乌尔和乌鲁克城发现的最早拼写是<sup>[[dingir|D]]</sup>LAK-32.NA”(其中“NA”可理解成声旁)。乌尔城的名称拼写是：“</sup>LAK-32.UNUG<sup>KI</sup>=URIM<sub>2</sub><sup>KI</sup>”，它是从[[神名|神名]]导出的，意指“南纳(LAK-32)的住所(UNUG)，其中“UNUG”是“乌努克”，乌鲁克城的别称”。

早期苏美尔时期的标记“LAK-32”后缩写为“ŠEŠ”(“兄弟”的表意文字)，古典[[苏美尔语|苏美尔语]]的拼写是”<sup>[[dingir|D]]</sup>ŠEŠ.KI“，读音为“na-an-na”(纳-安-纳)。“新月”的术词<sup>[[dingir|D]]</sup>U<sub>4</sub>.SAKAR”也可能指该神。后来，该名字按读音拼写为“<sup>[[dingir|D]]</sup>NANNA”(南纳)。

闪族月神“苏因/欣”是起源于[[苏美尔|苏美尔]]“南纳”一位的神，但在[[阿卡德帝国|阿卡德帝国]]时期二者已产生融合和混同。 偶尔，[[亚述|亚述]]语：“<sup>[[dingir|D]]</sup>NANNA-''ar'' <sup>D</sup>''Su'en-e''”的写法是由于与阿卡德语“na-an-na-ru”(照明、灯)的关联，反映了月神的特质。亚述月神“苏因/欣”常被拼写为“DEN.ZU”或简单地用数字30，<sup>[[dingir|D]]</sup>XXX表示。<ref>''[[Reallexikon_der_Assyriologie|Reallexikon der Assyriologie]]'', 1997, ISBN 978-3-11-014809-1, [http://books.google.com/books?id=O1yFrzi-MgYC&pg=RA2-PA360&lpg=RA2-PA360&dq=%22LAK32.NA%22&source=bl&ots=EdyMpkD6IK&sig=Wa3_AX4ZHkYZTyohXRIUeN4SY24&hl=en&ei=5mR5SrusDcOg_AaomoiABg&sa=X&oi=book_result&ct=result&resnum=1#v=onepage&q=%22LAK32.NA%22&f=false p.360]</ref>

== 背景 ==
他通常被看作是“智慧之主”恩祖(En-zu)，在公元前2600年-公元前2400年期间，乌尔城很大程度较[[幼发拉底河|幼发拉底河]]谷地区的其它城邦更为兴盛，欣很自然被认为是神殿中最重要的主神。正是这一时期，发现了他大量的称号：“众神之父”、“众神之主”、“万物创造者”等等。 月神作为“智慧”的化身，同样体现在[[天文学|天文学]]知识或[[占星|占星]]活动中，其中一个重要的活动就观察月相。

他的妻子是“大女主”[[宁伽勒|宁伽勒]](Ningal)，给他生下了[[太阳神|太阳神]][[乌图|乌图]]/沙玛什(Utu/Shamash)和[[伊南娜|伊南娜]]([[金星|金星]]女神)。欣/南纳和他的子女构成的三合一教义使得宇宙之权趋于集中。

欣长着青天色[[胡须|胡须]]，骑在一头长了翅膀的公[[牛|牛]]背上。公牛是他的标志之一(他的父亲[[恩利勒|恩利勒]]被称为“天堂公牛”)，并伴着一新月和三脚支架(可能是一盏灯台)标志。在[[滚筒印章|滚筒印章]]中，代表他的符号是一长有下垂胡须的老头及一新月符号。在约公元前26世纪的乌尔第一王朝王陵中，所发现的[[竖琴|竖琴]]就是用长着天青胡子的金牛头来装饰共鸣匣的。“欣”这一名称的[[楔形文字|楔形文字]]符也表示数字“三十”，大概是指一个[[太阴月|太阴月]](二个[[新月|新月]]之间)的平均天数(准确数字是29.53天左右)。

一份重要的[[苏美尔|苏美尔]]文献(《恩利勒和宁利勒》)<ref>{{cite web |url=http://etcsl.orinst.ox.ac.uk/section1/tr121.htm |title=恩利勒和宁利勒: 译文}}</ref> 讲述了恩利勒和宁利勒下[[冥界|冥界]]的故事，在那里，宁利勒生下了南纳/欣。恩利勒按照冥界之主的要求，创造三个冥神来替换南纳，此后南纳才获准飞升天空。该故事显示了一些与《因南娜下冥界》的相似之处。

据另一则故事说，南纳作为[[乌尔|乌尔]]城的保护神，把一船礼物运往[[尼普尔|尼普尔]]，献给恩利勒，祈求父亲保佑乌尔的河流、山林和田野富饶丰产。恩利勒满足了儿子的要求。大约在很早时期就形成了“南纳船”的观念，即南纳乘船于夜间在天空漫游，白天躲进冥界。

== 崇拜圣地 ==
南纳的主神庙在乌尔城的“埃基什努伽尔”(E-gish-shir-gal)-乌尔城女[[祭司|祭司]]所创角色“恩”(En)的名字，这一角色掌有极大权力，通常由乌尔城国王的长女担任，最著名的是[[阿卡德|阿卡德]]王[[萨尔贡|萨尔贡]]的女儿'''恩西杜安娜'''（Enheduanna），负责对南纳/欣等神的祭祀。<ref>哈尔, M. D. 《苏美尔月神南纳/苏因的研究》. 博士论文, 宾夕法尼亚大学, 1985年, 第227页</ref>

欣也在[[亚述|亚述]][[哈兰|哈兰]]城也有一座圣殿，名为“E-khul-khul”（快乐之所）。月神的崇拜蔓延至其他中心，因此，在[[巴比伦|巴比伦]]和[[亚述|亚述]]所有大城市都发现了他的神庙。在特克特(Tektek)山中的苏玛塔·哈拉贝西(Sumatar Harabesi)，离哈兰和埃德萨(Edessa)不远，发现了一座刻有[[叙利亚语|叙利亚语]]铭文的欣的神庙，其年代可追溯至公元前二到三世纪。
