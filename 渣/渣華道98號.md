'''渣華道98號'''（{{lang|en|'''The Java'''}}），為一位於[[香港|香港]][[北角|北角]][[渣華道|渣華道]]的單幢住宅，樓高共31層，由[[英皇國際|英皇國際]]發展（以[[英皇駿景酒店|英皇駿景酒店]]有限公司為名），物業由[[第一太平戴維斯|第一太平戴維斯]]物業管理有限公司管理，於[[2012年7月|2012年7月]]入伙。

== 歷史 ==
渣華道98號前身為多幢[[唐樓|唐樓]]，由[[渣華道|渣華道]]96號至104號，而於2006年起，[[英皇國際|英皇國際]]已以[[英皇駿景酒店|英皇駿景酒店]]有限公司的名義陸續進行[[收購|收購]]，終在2009年以1.4億元完成收購整個項目。原項目由[[英皇國際|英皇國際]]及[[新世界發展|新世界發展]]共同發展（英皇佔75%，新世界佔25%），但最後則由[[英皇國際|英皇國際]]獨自發展。<ref>[http://hk.news.yahoo.com/article/081009/4/8muc.html 英皇4500萬購 渣華道舊樓]</ref>

在初規劃時，發展商原本打算興建一幢33層高的[[酒店|酒店]]，惟不獲[[城規會|城規會]]放寬高度限制，最後發展商改為興建住宅項目，即「渣華道98號」。<ref>[http://hk.realestate.yahoo.com/091021/300/3k1g7.html 報城規會不支持英皇國際 北角渣華道重建項目放寬高限]</ref>


2017年10月13日英皇國際宣佈，今日售出售出頂層36樓連天台色戶，實用1,608方呎、另連1,421方呎天台，成交價達7,300萬元，折合呎價45,398元，成交價和呎價均屬項目新高，同時標誌The Java 全部75個單位全數沽清。

== 簡介 ==
由於物業部份可望單位可望[[維多利亞港|維多利亞港]]及鄰近[[港鐵|港鐵]][[北角站|北角站]]，故發展商開售呎價達$15,000起。而物業共31層，但住宅單位樓高則只有26層（不設4、13、14、24、34樓），其中地下為3個舖位，一樓為11個車位的[[停車場|停車場]]，而三樓為[[空中花園|空中花園]]，五樓則是[[會所|會所]]，其設施包括健身室、燒烤場等；另外十樓為[[隔火層|隔火層]]。

物業住宅（標準戶）於6樓至33樓，一梯為約兩至四伙；唯35-36樓為特色戶，僅一梯一伙；共75個單位，單位約介乎400-2000[[平方呎|平方呎]]。
