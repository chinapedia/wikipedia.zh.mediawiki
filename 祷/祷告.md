'''禱告'''，或称'''祈祷'''，是主動和[[上帝|上帝]]（[[神|天主]]）、[[神靈|神靈]]或[[靈性|靈性]]的個體[[溝通|溝通]]來讚美、祈求、懺悔或者表達自己的思想或[[願景|願景]]的行為。

== 禱告形式 ==
不同信仰傳統有不同禱告形式，例如早禱晚禱、謝飯禱和相關的身體姿勢。華人民間傳統會叩拜、念誦、敬香。基督徒會低頭合手。[[印第安人|印第安人]]跳舞。蘇菲派穆斯林旋轉。[[印度教|印度教]]徒誦唱。正統[[猶太人|猶太人]]擺動身體。貴格會會員緘默。這些不同方式反映對禱告理解。[[人類學家|人類學家]]相信早期智人從事我們會稱為禱告的活動。

* 有限物能與無限者實質接觸。
* 無限者有興趣和和限物接溝通
* 禱告是會被聆聽雖然不一定有反應。
* 禱告是去培養人某些態度而不是影響接收者。
* 禱告是透過讓人放下哲學和知識執著來訓練人集中於接收者。
* 禱告能使人直接經驗無限者
* 禱告對現存世界有影響。
* 接收者期望或欣賞禱告。

== 基督徒的信仰中的祷告 ==
{{基督教}}
{{ Religious text primary }}
在[[基督徒|基督徒]]的信仰中，祷告是與[[上帝|上帝]]溝通（部分基督徒称之为“交通”）, 可以是[[赞美|讚美]], 祈求, 懺悔或者僅僅是表達自己的思想或願望。

=== 原因 ===
*圣经显示上帝鼓勵信徒祷告。聖經指出：“應當一無掛慮，只要凡事藉著禱告，祈求和感謝，將你們所要的告訴神，神所賜出人意外的平安，必在基督耶穌裡保守你們的心懷意念。”<ref>参看腓立比書4：6，7</ref>
*經常禱告可以增進和上帝的友誼。基督徒认为，上帝应当是基督徒的好朋友，不會只在需要幫忙時才交談，倒會互相關心，無拘無束地向對方説出自己的想法、憂慮和感受，這樣友誼就更牢固。上帝是个實在的个体，是有思想和感情的，可以通過禱告向天父訴説自己的想法和感受，信徒和上帝的关系更親近。<ref>参看雅各書4：8</ref>

=== 目的 ===
基督徒相信经常祷告是与神保持良好关系的一个重要方式。值得注意的是，祷告与得永生并没有直接关系。在圣经中记载，喜欢在大街上公开祷告的法利赛人有大部分最后并没有得永生。部分基督徒相信可以借着祷告来获得上帝的力量。

祷告具体的目的有：颂赞上帝和向上帝谢恩；向上帝祈求事物(可以为自己求，也可以为别人代求)和求赦；与上帝沟通。
=== 要求 ===
基督徒认为禱告是信徒與上帝之間的直接接触，所以需要虔诚而严肃。他们认为基督徒祷告必须遵守奉耶穌的名以及用心诚实，同时要有信心，毅力和爱心，在遇到重要及困難的事，感覺禱告無力，可以加上[[禁食|禁食]]，但不是必需的。
==== 对象 ====
耶穌教導門徒，要向“我們在天上的父親”禱告。<ref>参看记录在馬太福音6：9的耶稣所做的模范祷告/主祷文</ref>所以我們應該只向耶稣所指代的天父耶和華上帝禱告。但是，上帝也要求承認耶穌基督的地位。耶穌被差派到地上來，犧牲生命作為贖價，救贖人类脱離罪和死亡。<ref>参看約翰福音3：16；羅馬書5：12</ref>他是上帝委任的大祭司和審判者。<ref>参看約翰福音5：22；希伯來書6：20</ref>因此，聖經显示，要通過耶穌向上帝禱告。耶穌自己也曾説過：“我就是道路、真理、生命。若不籍著我沒有人能到父那裡去。”<ref>参看約翰福音14：6</ref>因此祷告的对象应当是上帝耶和华，並通過他兒子耶穌禱告。

==== 姿势 ====
在旧约圣经裡，耶和華沒有規定，禱告時手要怎樣擺，身體要有怎樣的姿勢。聖經的記載顯示，人禱告時無論坐着、站着，還是面伏於地、雙膝跪下，各種姿勢都是可以的。<ref>参看歷代志上17：16；尼希米記8：6；但以理書6：10；馬可福音11：25</ref>現今雙手合掌的祈禱姿勢是模仿[[歐洲封建制度|歐洲封建制度]]領主宣誓順從國王的動作<ref>John Hirst. 你一定愛讀的極簡歐洲史</ref>。真正要緊的，不是外在的特定姿勢，而是內在的正確心態。事實上，不管平時還是緊急關頭，都可以就地向上帝默禱。即使周圍的人完全看不出我們在禱告，耶和華也會聽見我們的默禱。<ref>参看尼希米記2：1-6，尼希米在要求返回以色列时做了一个十分简短的祷告，时间可能只有几秒钟。</ref>

==== 内容 ====
聖經説：“我們按照上帝的旨意，不管求甚麽，他都聽我們。”<ref>参看約翰一書5：14</ref>因此，凡是合乎上帝旨意的事都可以求，包括：
*个人的憂慮或難題。大卫王在诗篇里说，向耶和華禱告就像跟密友説話，大可以向上帝傾心吐意，暢所欲言。<ref>参看詩篇62：8</ref>
*祈求聖靈，因為聖靈能幫助人做對的事。<ref>参看路加福音11：13</ref>
*指引作明智的決定，賜給力量去忍受逆境。（雅各書1：5）
*假如犯了罪，就應該懇求上帝本着基督所獻的贖價寬恕。（以弗所書1：3，7）
*為其他人禱告，例如家人和其他崇拜耶和華的人。<ref>参看使徒行傳12：5；歌羅西書4：12</ref>
==== 时长 ====
无论是个人还是集體禱告，聖經都沒有規定禱告應該多長。禱告可長可短，短的如吃飯前的禱告，長的如向上帝傾心吐意的禱告。<ref>参看撒母耳記上1：12，15，撒母耳的母亲不生育，在圣殿里做了很长的祷告</ref>不過，耶穌曾譴責一些自以為義的人，他們故意在别人面前作冗長的禱告，好引人注目。<ref>参看路加福音20：46，47</ref>重要的是，必須衷心地禱告。因此，正确的禱告可長可短，視乎情況和需要而定。
==== 频度 ====
聖經鼓勵基督徒“不斷禱告”，又説“禱告要堅持不懈”。<ref>参看馬太福音26：41；羅馬書12：12；帖撒羅尼迦前書5：17</ref>聖經不是説要每時每刻不停地禱告，而是要經常禱告，常常感謝，向他尋求指引、安慰和力量。

=== 聖經中的禱告 ===
*[[撒母耳記|撒母耳記]]上1：12，15，[[撒母耳|撒母耳]]的母亲在圣殿里做了很长的祷告，以致祭司认为她睡着或者酗酒。
*[[但以理書|但以理書]]6：10,但以理对抗[[尼布甲尼撒二世|尼布甲尼撒]]的命令坚持每天三次面向[[耶路撒冷|耶路撒冷]]祷告。
*[[尼希米記|尼希米記]]2：1-6，[[尼希米|尼希米]]在要求返回[[以色列|以色列]]时做了一个十分简短的祷告。
*[[马太福音|马太福音]]6：9-13，門徒請求[[耶穌|耶穌]]指導禱告，耶穌便教導他們一個模範禱告。被稱為[[主禱文|主禱文]]。
*[[以弗所書|以弗所書]]6:18 時時在[[靈|靈]]裡禱告，並盡力堅持，在這事上儆醒，且為眾聖徒祈求， 
*[[歌羅西書|歌羅西書]]4:2 你們要堅定持續的禱告，在此儆醒感恩， 
*[[腓立比書|腓立比書]]4:6 應當一無挂慮、只要凡事藉著禱告、祈求、和感謝、將你們所要的告訴神； 
*[[羅馬書|羅馬書]]8:26-27 況且，[[那靈|那靈]]也照樣幫同擔負我們的軟弱；我們本不曉得當怎樣禱告，只是那靈親自用說不出來的歎息，替我們代求。那鑒察人心的，曉得那靈的意思，因為祂是照著神的旨意替聖徒代求。

== 伊斯蘭教的禱告 ==
{{ Religious text primary }}
在伊斯蘭教祈禱的名字是'''主回賜你們'''。[[伊斯蘭教|伊斯蘭教]]對禱告（[[拜功|拜功]]）非常重視，並列入為[[五功|五功]]之一。伊斯蘭教中的禱告可算是獨特的；穆斯林必須在固定的時間進行拜功，每天五次。这五次祷告分别称为晨拜、晌拜、晡拜、昏拜和宵拜。除此之外穆斯林自由和直接地向神做的禱告不记在拜功之列。

=== 伊斯兰教的规定 ===
*每天遵行五次祷告；强调敬拜的仪式，须公开，发声。
*每日都把经文背诵多次。
*当谨守祷告([[古兰经|古兰经]]2：43)
*祷告能防止丑事和罪恶。(古兰经29：45)
==== 祷告的时间 ====
* 祷告是定时的义务(古兰经4：103)
* 在白昼的两端和初更的时候祷告(古兰经11：114)
* 在晚夕和早晨，赞颂真主超绝万物；天地间的赞颂，以及傍晚的和中午的赞颂都只归于他 (古兰经30：17，18)
* 从晨时到黑夜的祷告，早晨的祷告，在夜里的一段时间附加的祷告(古兰经17：78，79)；
* 在太阳出没之前，夜间和白昼赞颂神(真主)(古兰经20：130)
* 聚礼日(即星期五)(古兰经62：9)
==== 祷告前的洁凈 ====
*在去祷告之前，需要洗脸和手，两肘，头，脚，至两踝；
*如果不洁，就洗全身；
*在醉酒的时候不要祷告；
*如果有病、旅行、如厕、性交，不方便得到水，就趋向清洁的地面，而摩脸和手(古兰经4：43；5：6)

==== 祷告的方向 ====
在當初，穆斯林是向[[耶路撒冷|耶路撒冷]]禱告的；但公元624年穆罕默德把禱告方向改向[[麥加|麥加]]的[[卡巴聖堂|卡巴聖堂]]。
* 麦加的禁寺(古兰经2：149，150)
* 每个方向都属于神真主(古兰经2：115)
* 把脸转向东方和西方，都不义；正义是信真主，末日，天神，天经，先知，并将财产施济亲戚，孤儿，贫民，旅客，[[乞丐|乞丐]]和赎取[[奴隶|奴隶]]，诚实守信，忍受[[贫穷|贫穷]]、患难和[[战争|战争]](古兰经2：177)

==== 祷告的的姿势 ====
* 环行，立正，[[鞠躬|鞠躬]]，叩头(古兰经22：26)
* 鞠躬，叩头(古兰经2：43；22：77)

==== 注意事项 ====
* 若有[[狗|狗]]，[[驴|驴]]，妇人在祷告的男人面前经过，会令祷告无效。<ref>参看[[布哈里圣训|布哈里圣训]]1：9：490</ref>
* 在祷告的人面前经过是罪(布哈里圣训1：9：489)，令祷告无效。<ref>参看布哈里圣训1：9：488</ref>
* 不要先于[[伊玛目|伊玛目]]在祷告时站起。<ref>参看布哈里圣训1：11：660</ref>
* 祷告时要排列要成直线(布哈里圣训1：11：685，690)，
* 眼不要向上望(布哈里圣训1：12：717)，
* 叩头要完美(布哈里圣训1：12：759)，否则祷告无效，真主会惩罚。
* 集体祷告(古兰经14：40)

== 猶太教的禱告 ==

犹太教徒每天至少祷告三次，在一些特别的节日，例如安息日，会增加祷告的次数。

全世界的犹太教徒祷告都使用统一的[[祈祷书|祈祷书]]（siddur），祈祷书包含了一系列日常祷告内容。

犹太教最重要的祈祷仪式是Shema Yisrael("Hear O Israel")和Amidah ("the standing prayer")。
