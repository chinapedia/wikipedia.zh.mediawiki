'''灣仔運動場'''（{{lang|en|'''Wan Chai Sports Ground'''}}）位於[[香港|香港]][[香港島|香港島]][[灣仔北|灣仔北]]海傍，毗鄰[[新鴻基中心|新鴻基中心]]，佔地2.7公頃，設有3,000座席位，於1979年年底啟用，由[[康樂及文化事務署|康樂及文化事務署]][[管理|管理]]。惟場地設施只供團體、學校及體育總會租用，不設開放給公眾的「[[緩步跑|緩步跑]]時間」。

==設施==
灣仔運動場有一條8線400[[米|米]]的全天候綜合性跑道，符合[[国际田径联合会|国际田径联合会]]的標準，使用13毫米厚的合成纖維物，是香港首條[[花呢格紋|花呢格紋]]跑道。運動埸設有一個真草球場，供[[田賽|田賽]]及[[足球|足球]][[運動員|運動員]]使用。另外，大燈是450[[勒克斯|勒克斯]]光，設有國際標準的電子計時器及終點拍攝儀器等相關設備供予[[競賽|競賽]]用途。於2007年至2012年，灣仔運動場的使用率達到100%，希望租借此處舉辦運動會的必須是田徑成績最優秀的首25間[[學校|學校]]，提早半年申請及需要抽籤候選<ref>[http://hk.news.yahoo.com/啟德體育城定灣仔兩體育場命運-013534099.html 啟德體育城定灣仔兩體育場命運]</ref>。

==活動==
===體育活動===
灣仔運動場是香港最主要的[[田徑|田徑]]比賽場地之一，主要比賽包括由[[香港業餘田徑總會|香港業餘田徑總會]]主辦的[[香港田徑聯賽總決賽|香港田徑聯賽總決賽]]<ref>{{Cite web |url=http://www.hkaaa.com/hkaaa/event/event/Year2008/HKAL/chi_hkal_08.htm |title=香港田徑聯賽 2008 |access-date=2009-01-08 |archive-url=https://web.archive.org/web/20081117200227/http://www.hkaaa.com/hkaaa/event/event/Year2008/HKAL/chi_hkal_08.htm |archive-date=2008-11-17 |dead-url=yes }}</ref>、香港田徑公開賽<ref>{{Cite web |url=http://www.hkaaa.com/hkaaa/event/event/Year2008/OM/OM-2008.htm |title=2008 年度 香港田徑公開賽 |access-date=2009-01-08 |archive-url=https://web.archive.org/web/20081217123513/http://www.hkaaa.com/hkaaa/event/event/Year2008/OM/OM-2008.htm |archive-date=2008-12-17 |dead-url=yes }}</ref> 及香港國際田徑邀請賽<ref>{{Cite web |url=http://www.hkaaa.com/hkaaa/event/event/Year2008/IIM08/IIM_News.htm |title=香港國際田徑邀請賽 |access-date=2009-01-08 |archive-url=https://web.archive.org/web/20081218135017/http://www.hkaaa.com/hkaaa/event/event/Year2008/IIM08/IIM_News.htm |archive-date=2008-12-18 |dead-url=yes }}</ref>。

===其他活動===
1996年，距離香港主權移交僅一年，大批市民爭相到[[灣仔|灣仔]][[人民入境事務處|人民入境事務處]]總部排隊申領[[英國國民(海外)護照|英國國民(海外)護照]]，入境處為疏導人潮，特別向[[市政局_(香港)|市政局]]借用灣仔運動場，供市民作等候用途，期間疑有市民因為排隊秩序問題爭執，繼而大打出手。

[[世界貿易組織香港部長級會議|世界貿易組織香港部長級會議]]於2005年12月13日至18日在香港舉辦期間，[[民間監察世貿聯盟|民間監察世貿聯盟]]表示不接受[[香港政府|香港政府]]建議以[[修頓球場|修頓球場]]為示威場地，因為修頓球場完全看不見舉辦會議的[[香港會議展覽中心|香港會議展覽中心]]。及後，灣仔運動場及附近的[[灣仔貨物裝卸區|灣仔貨物裝卸區]]被初步選擇為逾10,000名反對[[世界贸易组织|世貿]]的示威者的示威範圍。灣仔運動場職員表示恐怕示威者會破壞運動場內的設施，此可能令到很多學校不能夠使用運動場舉行陸上運動會。最後灣仔貨物裝卸區成為了最終的示威範圍。

== 搬遷爭議 ==
2008年，業界支持將灣仔運動場搬走，用作擴建會議展覽中心第3期，但遭到部分居民、港島區學校、以至體育界反對。<ref>[http://orientaldaily.on.cc/cnt/news/20130110/00176_009.html 灣仔政府大樓 建會展3期] 《東方日報》 2013年1月10日</ref>其後[[灣仔區議會|灣仔區議會]]極力否決，認為灣仔交通將無法負荷。<ref>[http://std.stheadline.com/daily/news-content.php?id=653355&target=2 會展擴建鎖定灣仔運動場] 《星島日報》 2009年8月4日</ref>

2012年11月9日，[[民政事務局局長|民政事務局局長]][[曾德成|曾德成]]表示，初步考慮當[[啟德體育城|啟德體育城]]於2019年啟用後，[[香港大球場|香港大球場]]將會改變用途，以取代灣仔運動場，主力成為香港島居民及學校的陸上運動會主要舉辦場地<ref>[http://hk.news.yahoo.com/灣仔運動場或改商業用途-220322005.html 灣仔運動場或改商業用途] 《星島日報》 2012年11月10日</ref><ref>[http://hk.news.yahoo.com/啟德體育城落成無礙紅館-042804531.html 啟德體育城落成無礙紅館] 《明報》 2012年11月9日</ref>。

2017年1月18日，[[香港行政長官|行政長官]][[梁振英|梁振英]]在立法會發表任內最後一份[[香港施政報告|施政報告]]，公布灣仔運動場最快會在2019年進行綜合發展，用作會展用途，以及其他「潮玩和新穎的康樂及體育運動設施」和當區所需社區設施，務求地盡其用。包括室內3人足球、泡泡足球、室內滑冰等。大部分焦點賽事將移去[[啟德體育園|啟德體育園]]舉辦。<ref>[https://www.hkcnews.com/article/1009/%E5%80%A1%E7%81%A3%E4%BB%94%E9%81%8B%E5%8B%95%E5%A0%B4%E4%BD%9C%E6%9C%83%E5%B1%95%E7%94%A8%E9%80%94-%E5%B0%87%E7%A0%94%E9%87%8D%E5%BB%BA%E9%A6%99%E6%B8%AF%E5%A4%A7%E7%90%83%E5%A0%B4%E5%8F%AF%E8%A1%8C%E6%80%A7] 《眾新聞》 灣仔運動場改作會展用途　研究重建香港大球場 2017年1月18日</ref>

不過有運動界人士表示對此感愕然。曾出征2016年[[里約熱內盧奧運|里約熱內盧奧運]]、保持香港多項長跑賽事紀錄的[[姚潔貞|姚潔貞]]認為該處地理、交通位置方便完成訓練，個人成長回憶及比賽記錄都在該場地發生，如學界田徑、[[渣打馬拉松|渣打馬拉松]]等傳統賽事。不明白為何會展需要這麼大的土地。<ref>{{cite news|author1=鄭秋玲|title=【施政報告】灣仔運動場改建新會展　姚潔貞︰唔明點解要霸埋個場|url=https://www.hk01.com/%E6%B8%AF%E8%81%9E/66414/-%E6%96%BD%E6%94%BF%E5%A0%B1%E5%91%8A-%E7%81%A3%E4%BB%94%E9%81%8B%E5%8B%95%E5%A0%B4%E6%94%B9%E5%BB%BA%E6%96%B0%E6%9C%83%E5%B1%95-%E5%A7%9A%E6%BD%94%E8%B2%9E-%E5%94%94%E6%98%8E%E9%BB%9E%E8%A7%A3%E8%A6%81%E9%9C%B8%E5%9F%8B%E5%80%8B%E5%A0%B4|accessdate=2017-01-18|publisher=香港01|date=2017-01-18}}</ref>

在灣仔運動場參加學校運動會的中四學生認為，運動場多年來一直為學界的田徑比賽提供場地，加上政府工程需時多時，會對運動員帶來很大影響。相比香港大球場，他認為自己較熟悉及較喜愛灣仔運動場，因交通更為便利。<ref>{{cite news|title=政府擬將灣仔運動場綜合發展 學生稱對運動員影響大|url=https://hk.news.yahoo.com/%E6%94%BF%E5%BA%9C%E6%93%AC%E5%B0%87%E7%81%A3%E4%BB%94%E9%81%8B%E5%8B%95%E5%A0%B4%E7%B6%9C%E5%90%88%E7%99%BC%E5%B1%95-%E5%AD%B8%E7%94%9F%E7%A8%B1%E5%B0%8D%E9%81%8B%E5%8B%95%E5%93%A1%E5%BD%B1%E9%9F%BF%E5%A4%A7-053635536.html|accessdate=2017-01-18|publisher=香港電台|date=2017-01-18}}</ref>

==未來發展==
