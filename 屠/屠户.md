'''屠户'''，也叫'''屠夫'''或'''屠宰工作者'''，是宰杀动物、分切肉和卖肉的人<ref name="mw">{{cite web |url=http://www.merriam-webster.com/dictionary/butcher |title=Merriam-Webster's Dictionary's definition of "butcher"}}</ref>。屠户可以是自己开店，也可受雇于[[超市|超市]]、肉店和水产市场。<ref name="gov">{{cite web |url=http://www5.hrsdc.gc.ca/NOC/English/NOC/2006/QuickSearch.aspx?val65=6251 |title=Employment information for butchers |publisher= }}</ref>
==历史==
屠户这个行业古已有之，[[水浒传|水浒传]]里就有[[鲁智深|鲁智深]]拳打郑屠户的故事。1272年英格兰就出现了屠户[[行业协会|行业协会]]<ref>{{cite web|url=http://www.yorkbutchersgild.com/pages/history.html |title=York Butchers' Guild |publisher=Yorkbutchersgild.com |date= |accessdate=2012-04-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20120308184834/http://www.yorkbutchersgild.com/pages/history.html |archivedate=2012-03-08 }}</ref>。如今，很多地方的屠户都需要资质许可，有些地方需要三年的学徒见习期<ref name="gov2">{{cite web |url=http://www.servicecanada.gc.ca/eng/qc/job_futures/statistics/6251.shtml |title=Job futures statistics |publisher=Servicecanada.gc.ca |deadurl=yes |archiveurl=https://web.archive.org/web/20090807182133/http://www.servicecanada.gc.ca/eng/qc/job_futures/statistics/6251.shtml |archivedate=2009-08-07 }}</ref><ref name="shaw">{{cite web|url=http://members.shaw.ca/masterbutcher/butcher.html |title=Master Butcher's Guide |publisher=Members.shaw.ca |deadurl=yes |archiveurl=https://archive.is/20060923043724/http://members.shaw.ca/masterbutcher/butcher.html |archivedate=2006-09-23 }}</ref>。
==职责==
屠宰是一项传统的工作。工业化后，屠户就成了屠宰厂流水线的一部分，步骤包括击晕、放血、去毛、剥皮、去内脏、对劈等步骤。之后屠夫要根据需要在损失最小的情况下将肉和骨头分解开来。在工业化前和工业化低的国家，屠户要独立完成从宰杀到分解的一系列过程。没有冷藏的地方屠夫往往还要负责把肉卖掉。
