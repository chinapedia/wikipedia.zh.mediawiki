''' 屠洪刚 '''{{BD|1967年|3月15日|||catIdx=T屠}}别名'''屠洪纲'''，[[中国大陆|中国大陆]]著名歌手，以阳刚霸气洋溢着[[爱国|爱国]]精神的歌曲而闻名音乐界。[[中国民主同盟|中国民主同盟]]成员<ref>[http://ent.163.com/16/0509/11/BMKBOTT800031H2L.html 相声演员于谦已低调加入中国国民党革命委员会]</ref>。

==生平==
1967年3月15日，屠洪刚出生于[[北京|北京]]。<ref name="《酷我音乐：歌手屠洪刚》">{{cite web |url=http://www.kuwo.cn/mingxing/%E5%B1%A0%E6%B4%AA%E5%88%9A/#@|title=《酷我音乐：歌手屠洪刚》|accessdate=|work=|publisher=酷我音乐盒官方网|language=zh-hans}}</ref><br>

最初，屠洪刚学习的是京剧艺术，在[[中国戏曲学院|中国戏曲学院]]获得学士学位。后来参加[[全国青年歌手电视大奖赛|全国青年歌手电视大奖赛]]获优秀歌手奖。<ref name="《酷我音乐：歌手屠洪刚》">{{cite web |url=http://www.kuwo.cn/mingxing/%E5%B1%A0%E6%B4%AA%E5%88%9A/#@|title=《酷我音乐：歌手屠洪刚》|accessdate=|work=|publisher=酷我音乐盒官方网|language=zh-hans}}</ref><br>

1990年，屠洪刚第一次参加[[中国|中国]][[中央电视台|中央电视台]][[春节联欢晚会|春节联欢晚会]]，演唱了春晚开场歌曲《万紫千红》。<ref name="《酷我音乐：歌手屠洪刚》">{{cite web |url=http://www.kuwo.cn/mingxing/%E5%B1%A0%E6%B4%AA%E5%88%9A/#@|title=《酷我音乐：歌手屠洪刚》|accessdate=|work=|publisher=酷我音乐盒官方网|language=zh-hans}}</ref>

== 个人生活 ==
屠洪纲结过三次婚，现在第三任妻子小岳，非娱乐圈人。<ref name=" opss" />第一任妻子[[美籍华人|美籍混血华人]]黛西，屠洪纲23岁时和她结婚，3年后离婚，两人育有1个儿子屠芮葆，屠芮葆现居[[美国|美国]]；<ref name=" opss" />第二任妻子[[方舒|方舒]]，两人在1994年结婚，方舒前任丈夫是导演[[陈国星|陈国星]]，方舒和陈国星有1个女儿，和屠洪纲育有1个女儿。<ref name=" opss" />屠洪纲还有1个继女和1个继子，女儿现和方舒一起生活，小岳的儿子张梓羲跟着屠洪纲和小岳生活，现居[[加拿大|加拿大]]留学。<ref name=" opss" >{{cite web |url=http://ent.qq.com/a/20121122/000419.htm#p=1 |title=独家：屠洪刚夫妇离京 第三任太太貌似方舒 |accessdate= 2012年11月23日|publisher=腾讯网 |archiveurl=http://ent.qq.com/a/20121122/000419.htm#p=1 |archivedate=2012年11月23日 |work= |language =zh-cn}}</ref>
