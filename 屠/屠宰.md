'''屠宰'''指杀死[[家畜|家畜]]、[[家禽|家禽]]，产出供人食用的[[肉類|肉類]]的过程。专门用来屠宰的地方称为屠宰场或屠宰厂，专业人员旧时也称作[[屠夫|屠夫]]或[[屠户|屠户]]。

因为肉类食品中毒是很严重的问题，各国屠宰均有严格的商业屠宰规定。对于宗教屠宰则在遵守宗教习惯的前提下另设规定。不出售而是自用的屠宰比商业屠宰限制少，但也有一定的检疫要求。

有些地方对大型动物的商业屠宰征收[[屠宰税|屠宰税]]。

==现代商业屠宰过程==
[[File:Hong_Kong_Market_(4790169634).jpg|缩略图]]
# 从[[农场|农场]]、[[养殖场|养殖场]]等地收进动物。放在暂时的收容圈里。
# 对动物进行健康检测，排除有不宜食用的疾病、已死亡或健康很不好的动物，但實際執行並非如此嚴格。 
# 用水冲洗动物，减少细菌和寄生虫，另外使得下一步电击致晕更加有效。
# 使动物致晕，有减少动物的痛苦和方便屠宰过程的两个动机。致晕的方法有机械敲击头部、电击、或[[二氧化碳|二氧化碳]]麻醉，以电击最为常用。偶尔也有用子弹瞬间射杀，但因为安全等原因一般不采用。
# 刀刺[[颈动脉|颈动脉]]（cartoid artery）和[[颈静脉|颈静脉]](jugular veins)放血，导致死亡。小屠宰场和私人屠宰则常使用刺[[心脏|心脏]]的方法致死。亦有掛上輸送帶集體刀割或刀刺的方法。
# 对猪、家禽用热水烫洗煺去毛或主要羽毛。猪使用机器刮毛，家禽多用[[石蜡|石蜡]]拔毛。
# 进一步检测后去头、蹄、尾等不适合食用或另行包装的部位。
# 剥皮，一般用机器辅以手工进行。
# 开膛并去除[[内脏|内脏]]。屠宰过程中的污染主要来自皮毛上和消化道内部的细菌、寄生虫。所以剥皮和开膛这两步应防止皮表面接触肉类和胃肠破裂。
# 解体。大的家畜先用电锯锯成左右两片。
# 成熟阶段。动物死亡后几个小时之内肌肉进入[[尸僵|尸僵]]状态，此期间食用口感粗糙僵硬。需要放置一段时间使之成熟。商业上一般先快速冷藏以防止变质，然后冷藏直到尸僵消失。
# 进一步切割成适合销售的大小，或者转入肉食品厂进一步加工。

=== 家禽屠宰 ===

* 家禽屠宰主要包括浸烫和脱毛两个步骤。以中国生产的肉鸡屠宰机械为例，主要的标准流程包含上挂，电麻刺杀，沥血，浸烫，打毛，小毛处理，分类，除内脏，遇冷，切割，包装和冷藏（运输）。

==宗教屠宰过程==
[[犹太教|犹太教]]和[[伊斯兰教|伊斯兰教]]要求教徒只能食用按宗教程序屠宰的肉类。[[符合教規的食物_(猶太教)|符合犹太教規的]]称为{{lang|en|Kashrut}}或{{lang|en|Kosher}}；[[符合教規的食物_(伊斯蘭教)|符合伊斯兰教規的]]称为{{lang|en|Halal}}或清真。二者的主要区别是念诵神的名号不同。很多教徒认为可以接受按对方宗教程序屠宰的肉类。

#由熟悉屠宰程序的教徒进行。
#仅限于宗教戒律允许的动物（例如不能是[[猪|猪]]），必须健康而且会自行走动。
#把準備動物的動物個別入屠宰區，獨立屠宰。屠宰區並須與其他動物有足夠分隔，並確保潔淨。
#餵動物喝水並安撫動物。
#不允许致晕，而是由屠宰者念诵[[耶和華|耶和華]]/[[真主|真主]]之名，用一把锋利的刀具一刀快速切断颈部前方的颈静脉、颈动脉、[[食道|食道]]和[[气管|气管]]，但不切断[[脊髓|脊髓]]；屠宰全程不可讓動物遇見刀具。伊斯兰教中，屠宰者及動物应该面向[[麦加|麦加]]方向；猶太教中在此步驟會加挑動物腳筋，減少動物反抗掙扎。
#放血比商业屠宰彻底。犹太教还将肉浸于[[盐|盐]]水中一定时间以除去残存[[血液|血液]]。[[肝脏|肝脏]]因为含血最多，采用在火上烘烤的方法除血。
#確保動物放血徹底，動物[[腦部|腦部]]沒有血液流動逾3秒，即可斷定該動物死亡，隨後把動物送入脫皮去皮區域，繼而按指定刀法屠宰動物。
#禁止吃一些动物部位，例如[[血管|血管]]、[[淋巴|淋巴结]]和腿部的[[坐骨神经|坐骨神经]]。其中对坐骨神经的禁忌出自《[[圣经|圣经]]·[[创世纪|创世纪]]》第32章中[[以色列_(人名)|雅各]]与[[天使|天使]]摔跤的故事。在猶太教中[[內臟|內臟]]傳統上則留於[[祭祀|祭祀]]作為動物香燃燒而不予以食用。

==道德与人道方面的争议==

===本身的道德争议===
[[孟子|孟子]]在《[[孟子_(書)|孟子]]·梁惠王上》中说：“君子之于禽兽也，见其生，不忍见其死；闻其声，不忍食其肉。是以君子远庖厨也。”这个态度后来成为中国传统文化中对屠宰的常见[[道德|道德]]观点。一些受[[儒家|儒家]]思想影響的國家如在古代[[日本|日本]]與[[朝鮮王朝|朝鮮王朝]]就把屠夫列为低等人民甚至[[贱民|贱民]]。

[[佛教|佛教]]因为根本上反对[[杀生|杀生]]（屠宰與[[殺人|殺人]]同為殺生），对屠宰和屠宰业者持谴责态度。[[部派佛教|部派佛教]]要求在家居士應當盡量持[[五戒|五戒]]，首戒就是不殺生，一般允许出家僧人食用[[三净肉|三净肉]]，三淨肉是出家僧人可以食用之肉需要滿足的三個條件，要求此肉來自沒有看見為自己屠殺的動物，沒有聽說為自己屠殺的動物，不懷疑他人為自己宰殺的動物。

历史上很多[[哲学家|哲学家]]和道德学家，从[[杰里米·边沁|杰里米·边沁]]到[[彼得·辛格|彼得·辛格]]，基于[[動物權利|動物權利]]或人类道德的考虑而反对食用动物。这个立场导致反对屠宰和提倡[[素食主义|素食主义]]。

===过程中的人道问题===

另一些動物權利主义者在不反对屠宰，或者暂时不提倡废止屠宰的前提下，要求屠宰人道化，反对对动物造成无必要的肉体和精神痛苦。具体诉求包括：

*运送环境尽量人道，并保证动物的食水供应。
*屠宰前不惊吓动物，不让之观看同类被宰。
*要求宰杀在动物无知觉的情况下进行。提倡电击而不是棍棒敲击。
*保证各类屠宰器械正常运转，以避免有知觉的动物被屠杀、褪毛等。

在宗教屠宰过程是否比商业屠宰过程更人道的问题上，目前还有争议。
