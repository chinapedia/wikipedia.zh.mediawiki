'''屠光绍'''（{{bd|1959年|1月|catIdx=Tu,屠}}），男，[[汉族|汉族]]，[[湖北|湖北]][[鄂州|鄂州]]人。1977年8月参加工作，1982年7月加入[[中国共产党|中国共产党]]。[[北京大学|北京大学]][[经济学|经济学]]专业毕业，[[硕士研究生|硕士研究生]]学历，高级经济师。曾任[[中共上海市委|中共上海市委]]常委、[[上海市人民政府|上海市人民政府]]常务副市长、市政府党组副书记，上海自貿區金融工作協調推進小組組長，[[中投公司|中投公司]]总经理等职务。

== 生平 ==
屠光绍于1978年考入[[北京大学|北京大学]]历史系学习，本科毕业后转入经济系攻读研究生。1984年，获经济学硕士学位，并进入[[中共北京市委|中共北京市委]]机关工作，任政策研究室主任科员；1987年，任中共北京市委商贸部宣传处副处长。1989年，调[[中国人民银行|中国人民银行]]综合计划司工作，1991年升任中国人民银行全国金融市场报价交易信息系统中心副主任。

1993年，任改制后的中国证券交易系统有限公司副总经理。1995年，任[[中国证券监督管理委员会|中国证券监督管理委员会]]交易部主任；后历任[[中国证监会|中国证监会]]秘书长，[[上海证券交易所|上海证券交易所]]总经理兼党委书记等职；2002年升任中国证监会副主席。

2007年12月27日，任[[上海市人民政府|上海市人民政府]]副市长<ref>{{cite web |url = http://news.sina.com.cn/c/2007-12-27/161414615852.shtml|title=屠光绍艾宝俊出任上海市副市长|publisher=新浪网 |accessdate=2007-12-27}}</ref>。2013年1月，升任[[中共上海市委|中共上海市委]]常委、常务副市长<ref>{{cite web |url = http://news.takungpao.com/mainland/zgzq/2013-02/1429040.html |title=接棒杨雄 上海市委常委屠光绍兼任常务副市长|publisher=大公报|accessdate=2013-02-05}}</ref>。

2016年6月28日，任[[中投公司|中投公司]]总经理，接替此前总经理[[李克平|李克平]]<ref>{{cite news|title=中投总经理变动：屠光绍接任 李克平退休 2016年06月28日 |url = http://finance.caixin.com/2016-06-28/100959590.html |accessdate=2016-06-28|agency=财新网}}</ref>。2016年7月29日，[[上海市人大常委会|上海市人大常委会]]决定免去屠光绍的上海市人民政府副市长职务。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
