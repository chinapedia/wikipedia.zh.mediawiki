'''欽察汗國'''（英語：Golden Horde；{{lang-mn|Алтан Орд}}，''{{lang|mn|Altan Ord}}''；西里尔蒙古文：{{lang|mn-Cyrl|Алтан Ордын улс}}，拉丁轉寫：{{Unicode|Altan Ordyn Uls}}；[[土耳其语|土耳其语]]：{{lang|en|'''Altın Orda'''}}；[[鞑靼语|鞑靼语]]：{{lang|en|'''Altın Urda'''}}）（1242年－1502年）又稱'''金帳汗國'''，是[[蒙古四大汗国|蒙古四大汗国]]之一，建立于[[蒙古帝國|蒙古帝國]]西北部、后来[[突厥化|突厥化]]的[[汗国|汗国]]，位于今天[[哈薩克|哈薩克]][[咸海|咸海]]和[[里海|里海]]北部，佔有[[東歐|東歐]]和[[中歐|中歐]]地區（至[[多瑙河|多瑙河]]），由[[拔都|拔都]]及其後裔管理。至1359年前算是统一。

== 欽察立國 ==
[[成吉思汗|成吉思汗]]生前，分封西方的土地给长子[[朮赤|朮赤]]，即今日[[俄羅斯|俄羅斯]]的[[咸海|咸海]]、[[頓河_(俄羅斯)|頓河]]、[[伏爾加河|伏爾加河]]一帶的[[欽察草原|欽察草原]]，所以世稱“欽察汗國”（因為當時那一帶叫做欽察）。朮赤的血缘受到怀疑，因為在鐵木真未壯大以前，他的夫人[[孛兒帖|孛兒帖]]曾被[[蔑兒乞部|蔑兒乞部]]抓去。鐵木真後來聯合了[[王罕|王罕]]及[[扎木合|扎木合]]三路攻打[[蔑兒乞|蔑兒乞]]，才將孛兒帖救回。孛兒帖在歸途中生了個兒子，但鐵木真不介意，並把他待若自己的親子，給他取名[[朮赤|朮赤]]（意为“客人”）；那時的蒙古尚處在部落氏族社會。朮赤成年後，以果敢坚毅著稱，死在成吉思汗之前，享年49歲。

朮赤有14個兒子，有長子[[斡儿答|鄂爾達]]、次子[[拔都|拔都]]（巴图）。鄂爾達自知才能不及弟弟拔都，所以將繼承汗位的權利讓給了拔都。拔都控制了[[黑海|黑海]]北岸的海滨土地後，许多突厥系人民歸順於他，並被編入他的军队；在1230年代后期和1240年代初，他不断攻击[[伏尔加保加利亚|伏尔加保加利亚]]和[[基辅罗斯|基辅罗斯]]的后继国家，统一了中亚草原，结束了俄罗斯封建割据局面。

拔都率领他的军队持续向西方开拓，在[[列格尼卡戰役|列格尼卡戰役]]和[[蒂薩河之战|蒂薩河之战]]之后，入侵了[[波蘭王國_(1025年–1385年)|波兰王國]]和[[匈牙利王國|匈牙利王國]]。1241年，[[窝阔台|窝阔台]]在蒙古本土去世，拔都终止為追杀匈牙利國王[[貝拉四世|貝拉四世]]而發动的[[维也纳|维也纳]]围城战，东归。此后，蒙古军队再也没有西进至如此之远。1260年，金帳汗[[別儿哥|別儿哥]]及[[那海|那海]]{{le|蒙古第二次入侵波蘭|Second Mongol invasion of Poland|第二次入侵波蘭}}，大勝波兰，攻其首都[[克拉科夫|克拉科夫]],並在波兰[[桑多梅日|桑多梅日]]屠殺48名[[道明会|道明会]]成員,後撤兵。1287年至1288年金帳汗[[秃剌不花|秃剌不花]]再次{{le|蒙古第三次入侵波蘭|Third Mongol invasion of Poland|第三次入侵波兰}}，被擊退從此再未入侵波兰。1285年至1286年秃剌不花和那海{{le|蒙古第二次入侵匈牙利|Second Mongol invasion of Hungary|入侵匈牙利}}，但被匈牙利在[[特蘭西瓦尼亞|特蘭西瓦尼亞]]擊敗。

1242年，拔都在[[萨莱|萨莱]]（今[[伏尔加河|伏尔加河]]下游[[阿斯特拉罕|阿斯特拉罕]]附近）定都，正式建立欽察汗國，又稱金帐汗國。拔都的弟弟[[昔班|昔班]]（朮赤的第五個兒子）西征立了大功，拔都分給了他一片領地，昔班便在[[乌拉尔山|乌拉尔山]]以东的[[鄂毕河|鄂毕河]]与[[额尔齐斯河|额尔齐斯河]]之间，建立了他自己的营帐；版圖最遠至哈薩克的[[阿克托貝|阿克托貝]]，稱[[藍帳汗國|藍帳汗國]]。拔都的兄長鄂爾達讓位給拔都，所以拔都將東方錫爾河一帶分給哥哥，鄂爾達一系建立了[[白帳汗國|白帳汗國]]。[[別兒迪別|別兒迪別]]死後，欽察汗國絕後，汗位由大臣[[馬麥|馬麥]]控制，[[脫脫迷失|脫脫迷失]]後，汗位基本上由白帳汗出任。

== 黄金时代 ==
欽察汗國治下的人民是[[蒙古人|蒙古人]]和[[突厥人|突厥人]]的混合。汗国逐渐突厥化，而丧失了蒙古文化的特性。不过，拔都率领的蒙古战士後裔，始终是社会的上层阶级。這些蒙古人包括[[山只斤|山只斤]]、[[許兀慎|許兀慎]]（[[博爾忽|博爾忽]]出身氏族）、[[弘吉剌|弘吉剌]]、[[乞顏|乞顏]]。汗国的人口主要是[[钦察|钦察]]人、保加尔人、[[花剌子模|花剌子模]]人，以及其他一些[[突厥|突厥]]系族群。尤其以欽察人與土庫曼人居多。

===内部组织===
政治上，欽察汗國原为[[蒙古帝國|蒙古帝國]]之一脈（法律基礎是[[成吉思汗法典|成吉思汗法典]]）。自[[蒙哥|蒙哥]]大汗去世後，基本上完全独立，并曾与位于[[波斯|波斯]]的[[伊儿汗国|伊儿汗国]]，为阿塞拜疆主權多次交战。種族上，有[[欽察|欽察]]、[[康里|康里]]、[[保加爾人|保加爾人]]（[[匈人|匈人]]後人）、[[可薩人|可薩人]]餘部和[[烏克蘭人|烏克蘭人]]、[[希臘人|希臘人]]、[[俄羅斯人|俄羅斯人]]、[[摩爾多瓦人|摩爾多瓦人]]。汗国的统治者是由[[庫里尔台|庫里尔台]]自拔都的後裔中，选举出来。可汗以下，有宗王。宗王以下，有大臣以及一種名為[[答剌罕|答剌罕]]的人，意思是自由的人，享有免稅的地位。答剌罕是行政和軍事長官。行政文書使用[[回鶻文|回鶻文]]、[[察合台文|察合台文]]和[[阿拉伯文|阿拉伯文]]，外交文書用[[蒙古文|蒙古文]]，語言是使用欽察語為語言。政治地位最高的是蒙古人和突厥人，地位較低是[[奄蔡|阿蘭人]]和俄羅斯人、[[北高加索|北高加索]]人，他們在薩萊城有一定地區居住，烏克蘭人居住在[[奧卡河|奧卡河]]下游一帶。希臘人與意大利人住在[[克里米亞|克里米亞]]，大臣下有市民與農民，一種名為撒班赤（sabanchi），用犁者。佃農稱為urtakchi，最低微的是奴隸。

軍隊分東、中、西三路，左右兩翼，中軍由可汗指揮。兵制是以十進制為單位，最小的單位是十戶。每大軍團置統帥一人，每一個軍事單位，屯兵在某地區。行政上可分為萬人團、千人團、百人團。各單位的長官負責管理政務與鎮壓叛逆。

宗教上，[[伊斯蘭教|伊斯蘭教]]受到蒙古人和突厥人信仰，被定为[[国教|国教]]，伊斯蘭教制度和蒙古制度並駕齊驅。烏茲別克時代，完成突厥化與伊斯蘭化，與伊斯蘭國家實無分別。

金帐汗国可分为九个区域：[[花剌子模|花剌子模]]、[[克里米亞|克里米亞]]、[[欽察|欽察]]、[[阿速夫|阿速夫]]、[[切尔卡西亚|切尔卡西亚]]、[[伏爾加保加利亞|伏爾加保加利亞]]、[[瓦拉齐亚|瓦拉齐亚]]、[[阿蘭|阿蘭]]、[[基輔羅斯|基輔羅斯]]。

== 在俄羅斯的行政統治 ==
[[File:Taidula.jpg|thumb]]（Alexius, Metropolitan of Moscow）正在治療[[札尼別|札尼別]]的失明妻子。]]
蒙古對俄羅斯的行政統治，有兩個目的，徵兵與徵稅。手段因地不同。有些地方，由蒙古人直接管理，大部分容許自治。自治的地方大公在可汗的[[欽差大臣|欽差大臣]]宣布下即位。他們的權力受轄於蒙古人。在地方行政上，以十戶為基本行政單位，每人數單位要提供該單位人數的兵力和稅款，十戶供出十人，如此類推，每地數字不同。十戶是最基本的行政單位。西俄地區的第一次人口普查在1345年。東俄地區有兩次人口普查；一次在1258－1259年；一次在1274－1275年。1275年后，再無普查。普查結果：東俄有27個萬戶，西俄有16個萬戶，全俄即43萬戶。在1275年人口有850萬人。不包括在军事官职萬户长统辖下的属民士兵，大約一千萬人。每萬户，蒙古人派官員行政，不受大公指揮，只向可汗負責。這些千户與萬户的軍官，兼任行政長官，並有一位同級的徵稅員。日後再以[[達魯花赤|達魯花赤]]意为“镇守之官”為新的職稱，擔任全區的負責人。達魯花赤分為三個級數：萬户、城、村。每一個行政單位，皆有達魯花赤，有少量軍隊接受直接指揮，維持秩序。

在蒙古人直轄區，十人置一十夫長，百人置一百夫長。第一次在俄羅斯徵兵是男子人數的十分之一，以後是二十分之一。司法上，有最高法庭與地區法庭，大公也會在此受審。設達魯花赤和[[哈的|哈的]]（伊斯蘭教法官），有法官八人，按案件性質決定。

在徵稅上，在[[別兒哥|別兒哥]]時代，最初使用伊斯蘭商人與[[猶太人|猶太人]]及[[亞美尼亞|亞美尼亞]]人[[包稅|包稅]]，第一位包稅長是一位改宗伊斯蘭教的[[俄羅斯人|俄羅斯人]]，名稱叫[[伊佐希馬|伊佐希馬]]。後來改為使用[[八思哈|八思哈]]，再後來由一位弗拉基米爾大公徵稅。稅收分農村和城市兩種，有實物稅和貨幣稅，可汗有權徵收臨時稅。稅率是十分之一。有三種稅吏：書記、農村徵稅員、城市徵稅員，也是八思哈。不同的是羅斯公國，稅收大相逕庭，[[弗拉基米爾|弗拉基米爾]]每年交85000盧布，莫斯科只有4000盧布。

欽察汗國幅員遼闊，社會發展水平不一。[[烏爾根奇|烏爾根奇]]、[[薩萊|薩萊]]、[[別兒哥薩萊|別兒哥薩萊]]、[[阿速|阿速]]、[[喀法|喀法]]、[[速答黑|速答黑]]是貿易中心。經濟上有牧民和城市居民，農人。南俄和北[[高加索|高加索]]大草原是[[土庫曼人|土庫曼人]]、康-{里}-人、欽察人、蒙古人放牧的地方；[[伏爾加河|伏爾加河]]和[[卡馬河|卡馬河]]地區與[[梁贊州|梁贊州]]，是生產糧食的地區。

汗國的兵員多為突厥人。在宗教上，信仰自由，[[忙哥帖木兒|忙哥帖木兒]]時代，[[東正教|東正教]]受到優待；忙哥帖木兒發了一道[[詔書|詔書]]，它和屬民一律免稅，豁免他們的戶口普查，誹謗[[東正教|東正教]]的人一律處死。教會成為一特權團體。他們的工作是為俄羅斯人提供精神生活與道德上的指教；這時代也是東正教最獨立的時期。到[[月即別|月即別]]汗時期，侮辱[[東正教|東正教]]信仰與破壞教會財物的人要處死，[[帖木儿·忽格鲁特|帖木儿·忽格鲁特]]詔書也言明，不得干預教會運作。蒙古統治的第一個世紀，教會繁榮，對精神生活上的活動有甚大幫助。另一方面，東正教有自己的法庭，宗教案件只能由教會法庭審判，不在汗庭審判。

另一方面，欽察汗國與[[元帝国|元帝国]]也有不少的交往。在1330年代，有三萬俄羅斯人在元帝国境内服役，以前也有[[奄蔡|阿蘭]]人與欽察人軍團，証明兩者交往密切。除此以外的對外關係，主要是與埃及[[馬木留克王朝|馬木留克王朝]]交往。

對俄羅斯司法上，所有公國的大公都在可汗的權威下。大公若反抗，可汗會立刻處置。東正教會已立法保護。蒙古人不-{干}-涉俄羅斯貴族與平民之間的訴訟，大公可保留司法權。如俄羅斯人與蒙古人爭執，由蒙古達魯花赤處置。受到蒙古人統治的影響，俄羅斯帝國後來保留了死刑與體罰。

欽察汗國對俄羅斯的政策是不讓任何一個公國太強大，但如果沒大問題，一般會照章做事，發給敕令。在[[烏茲別克汗|烏茲別克汗]]時代後，全俄羅斯[[弗拉基米爾大公|弗拉基米爾大公]]由[[莫斯科大公|莫斯科大公]]世襲。

== 臣下與盟邦 ==
它的主体民族－[[俄罗斯人|俄罗斯人]]、[[亚美尼亚人|亚美尼亚人]]、[[格鲁吉亚人|格鲁吉亚人]]、[[切尔克斯人|切尔克斯人]]、[[哥特人|哥特人]]、[[克里米亚|克里米亚]][[希腊人|希腊人]]、克里米亚[[哥特人|哥特人]]和其他人，只要他们继续順服[[鞑靼|鞑靼]][[蒙古人|蒙古人]]，这些[[附庸|附庸]]永远不会并入汗國。在俄羅斯方面，[[莫斯科|莫斯科]]统治者获得徵稅特权。为了保持控制权，蒙古人经常對其他公國進行惩罚性打击。在另一方面看，古米廖夫说，正是俄羅斯與欽察汗國結盟，才阻止[[日耳曼人|日耳曼人]]入侵東方。

== 覆亡 ==
在[[札尼別|札尼別]]死後，[[別兒迪別|別兒迪別]]即位時期極短；死後，汗國大亂，他的兩兄弟不服他，改宗[[基督教|基督教]]。[[斡兒答|鄂爾達]]與[[昔班|昔班]]的後裔曾立為汗。換了多位可汗。汗位由[[馬麥|馬麥]]掌握。后期，金帳汗分裂。[[帖木儿|帖木儿]]1396年入侵后，日渐衰弱，[[大帐汗国|大帐汗国]]继承对莫斯科公国的统治，但是統治區域只剩下薩萊。1472年，[[莫斯科大公国|莫斯科大公国]]抗税，大汗[[阿黑麻|阿合马]]率兵征讨，结果失败，失去控制。

1480年阿合马再次征讨，再次失败，并死于内乱。至此[[莫斯科大公国|莫斯科大公国]]彻底摆脱蒙古人的统治。此時，距朮赤建立汗國已有二百六十年的歷史。1547年，莫斯科大公国正式改名[[俄罗斯沙皇国|俄罗斯沙皇国]]，[[伊凡四世|伊凡四世]]加冕为[[俄罗斯沙皇|俄罗斯沙皇]]。1550年代，沙皇俄国先後攻打了[[喀山汗國|喀山汗國]]、[[阿斯特拉罕汗國|阿斯特拉罕汗國]]、[[克里米亞汗國|克里米亞汗國]]三个汗国。其中，克里米亞汗國壽命最長，因為它是當代強大的[[奥斯曼土耳其帝国|奥斯曼土耳其帝国]]的封臣。1783年，[[克里米亞|克里米亞]]在[[法國大革命|法國大革命]]前，中衰的[[奥斯曼土耳其帝国|奥斯曼土耳其帝国]]與[[奥地利帝國|奥地利帝國]]有戰端，無暇東顧，汗國遂被歸併於沙俄領內。
