'''饒國樑'''（{{bd|1888年||1911年|catIdx=R}}），字'''作霖'''，號'''紹鋒'''，[[四川|四川]][[大足縣|大足縣]]雲露鄉(今[[重庆市|重庆市]][[大足区|大足区]][[國樑鎮|國樑鎮]]) 人，中國[[同盟會|同盟會]]會員，[[黃花崗七十二烈士|黃花崗七十二烈士]]之一。

1906年秋，饶国梁进入四川陆军速成学校学习，1909年毕业，任[[四川新軍|四川新軍]]65標見習官。期间曾与[[秦炳|秦炳]]、[[陈得藩|陈得藩]]等随[[熊克武|熊克武]]参加[[宜宾起义|宜宾起义]]。1911年，受[[黄兴|黄兴]]派遣来到[[广州|广州]]筹备起义。同年4月27日，[[黃花崗起義|黃花崗起義]]爆发，饶国梁负伤被捕，在法堂上，坚强不屈，正义凛然，怒斥敌人：“吾辈不死，国民不生，牛马奴隶，生何荣焉。求仁得仁，死何憾焉。”索取纸笔书写《绝笔书》，备述革命始末，洋洋千言，宣传主义。4月30日牺牲，遗体安葬在广州黄花岗，时年23岁。<ref>蔣力/陳先學：饒國樑寧死不做清朝官，重慶晚報 2009-05-10

== 纪念 ==
1929年，[[國民政府|国民政府]]出资于饶国梁故乡修建了纪念馆，2010年[[重庆市人民政府|重庆市政府]]拨专款进行修复，目前为重庆市青少年教育基地。<ref name=":0" />

1942年，国民政府将饶国梁故乡大足县云路乡更名为国梁乡（今国梁镇）<ref>{{cite book |author1=四川省大足县地名领导小组 |title=四川省大足县地名录 |date=1987年12月 |accessdate=2018-07-21}}</ref>。

1986年7月，大足县人民政府在大足北山修建了饶国梁烈士纪念碑，临近[[北山摩崖造像|北山石刻]]<ref>{{cite web |author1=何明生 |title=追忆重庆籍黄花岗烈士饶国梁 |url=http://news.ifeng.com/history/gundong/detail_2012_04/04/13643735_0.shtml |publisher=重庆晚报 |accessdate=2018-07-21}}</ref>。

== 家庭 ==
父亲饶树奇，满腹经纶，却屡试不第<ref name=":1">{{Cite web|url=http://www.sczx.gov.cn/system/20141216/000058847.htm|title=红色母亲饶国模的婚姻与家庭|accessdate=2018-07-22|author=赖晨|date=2014-12-16|publisher=四川政协报}}</ref>。

胞兄饶国栋为中国同盟会会员，曾任[[国民党|国民党]][[大竹县|大竹县]]党部书记<ref name=":1" />。

胞弟饶国材、胞妹[[饶国模|饶国模]]为[[中国共产党|中国共产党]]党员<ref name=":0" />。其中饶国模为[[八路军重庆办事处旧址|红岩村]]大有农场主人，在[[抗日战争|抗日战争]]期间将住房借与[[中共中央南方局|中共中央南方局]]、[[八路军|八路军]]重庆办事处，在[[第二次国共内战|解放战争]]期间保护了多名中共地下党员，中华人民共和国成立后被选为[[西南军政委员会|西南军政委员会]]监委委员、[[中国人民政治协商会议第二届全国委员会委员名单|第二届全国政协]]委员。
