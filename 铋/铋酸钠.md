'''铋酸钠'''（分子式： NaBiO<sub>3</sub>·2H<sub>2</sub>O ，分子量：NaBiO<sub>3</sub>·2H<sub>2</sub>O=316.00）是一种[[铋|铋]]酸盐，浅黄色或褐色的无定形粉末，用作[[氧化剂|氧化剂]]<math>E^{\ominus}(Bi^V/Bi^{III}) = 2.03V</math><ref>{{Cite web |title=Sodium bismuthate |url=http://hazard.com/msds/mf/baker/baker/files/s3026.htm |publisher=Mallinckrodt Baker |date=06/19/07 }}</ref>。微有吸湿性。不溶于冷水；加沸水或遇酸迅速分解。在空气中逐渐失去氧。密封保存。

在强碱性的[[氢氧化铋|氢氧化铋]]溶液中用[[氯气|氯气]]等氧化，可以制得铋酸钠。

铋酸钠能将锰离子氧化成[[高锰酸根|高锰酸根]]，因此可用于Mn<sup>2+</sup>的鉴定<ref>{{cite web |url=http://www.alfa.com/en/GP100w.pgm?DSSTK=033219 |title=存档副本 |accessdate=2011-10-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20110930101258/http://www.alfa.com/en/GP100w.pgm?DSSTK=033219 |archivedate=2011-09-30 }}</ref>。
