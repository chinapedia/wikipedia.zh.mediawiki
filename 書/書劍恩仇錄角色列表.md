[[金庸|金庸]]'''《[[書劍恩仇錄|書劍恩仇錄]]》'''武俠小說的劇情角色列表。

== [[紅花會|紅花會]] ==
=== 頭目 ===
* [[于萬亭|于萬亭]]──[[红花会|红花会]]前总舵主，武功深湛。
* 总舵主──[[陳家洛_(書劍恩仇錄)|陈家洛]]，愛作士宦家貴公子打扮。師从天池怪侠[[袁士霄|袁士霄]]，武器為九條珠索（打穴）和劍盾。
* 二[[當家|當家]]──「追魂奪命劍」[[無塵|無塵]]道長，劍法高強，擅使快劍，於常人刺一劍的時間，能刺出四、五劍之多。其「七十二路追魂奪命劍」配上「連環迷蹤腿」，武功為眾當家中數一數二，性格暴躁，本來是一堂香主 ，曾只用腳大破[[鐵旗幫|鐵旗幫]]。曾為受了『一位千金小姐的欺騙』，而自斷左臂，被香堂兄弟救獲，出了家，後來加入紅花會。
* 三當家──「千臂如來」[[趙半山|趙半山]]，[[太極門|太極門]]高手，以暗器縱橫江湖，並自製「飛燕銀梭」等獨門暗器。為人隨和，樂於助人而有「如來」之名，與[[武當|武當]]高手[[陸菲青|陸菲青]]為知交好友。
* 四當家──「奔雷手」[[文泰來|文泰來]]，平常空手不使兵刃，武功與無塵、趙半山相約，為紅花會中頂尖，于萬亭夜闖禁宮時亦只與文泰來同往，顯見其武功之高。他也是十一當家駱冰的丈夫。
* 五、六當家──「黑無常」[[常赫志|常赫志]]、「白無常」[[常伯志|常伯志]]，二人為孿生兄弟，合稱西川雙俠，外貌幾乎一模一樣，唯一人面有黑痣而另一人無。二人師承四川[[青城派|青城派]]掌門慧侶道人，武功高強，擅使[[黑沙掌|黑沙掌]]，輕功超卓，兵刃為一對飛抓。
* 七當家──「武諸葛」[[徐天宏|徐天宏]]，周綺的丈夫，智謀百出，是紅花會的軍師。
* 八當家──「鐵塔」[[楊成協|楊成協]]，原為[[青旗幫|青旗幫]]幫主，紅花會江北一帶會眾歸其統管。
* 九當家──「九命錦豹子」[[衛春華|衛春華]]，平常使雙鈎，勇敢好鬥如有九命而得名。
* 十當家──「石敢當」[[章進|章進]]，是個駝子，力氣奇大，由於駱冰對其甚為友善，毫不歧視其殘疾，故與文氏夫婦特別交好。在京師禁宮大戰中戰死，是紅花會唯一死去的頭領。
* 十一當家──「鴛鴦刀」[[駱冰|駱冰]]，擅使飛刀，個性豪邁，是四當家文泰來的妻子。
* 十二當家──「鬼見愁」[[石雙英|石雙英]]，師承太極門的分支[[無極拳門|無極拳門]]，算得上是三當家趙半山的子侄，臉上傷疤多，長相恐怖，掌管紅花會刑堂。
* 十三當家──「銅頭鱷魚」[[蔣四根|蔣四根]]，兵刃為鐵槳，廣東人，常會不自覺以廣府話對答。
* 十四當家──「金笛秀才」[[余魚同|余魚同]]，武當派[[馬真|馬真]]之徒，擅長吹笛，武器是一把金鑄笛子，曾經考上秀才。起初暗戀十一當家駱冰，後與[[李沅芷|李沅芷]]相戀，結為夫婦。
* 十五當家──[[心硯|心硯]]，原為陳家洛之[[書僮|書僮]]，在回部一戰中與眾人同生共死，破格立為十五當家。
*分舵舵主──[[馬善鈞|馬善鈞]]

=== 其他 ===
*[[駱元通|駱元通]]──外號「神刀」、駱冰之父。

== 鐵膽莊 ==
*周仲英：铁胆莊莊主，師承嵩山少林，武功高絕，为人仗义。
*周大奶奶：周仲英的續弦，周綺和周英傑之母，本是拳師之女，武功平平。
*[[周綺|周綺]]——铁胆莊莊主[[周仲英|周仲英]]之女，徐天宏之妻，外號「俏[[李逵|李逵]]」。
*[[周英傑|周英傑]]——周仲英的兒子，因為洩漏了地道的事情而被父親大怒之下以鐵膽誤殺。
*孟健雄：周仲英的大弟子。
*安健剛：周仲英的弟子。
*宋善朋：鐵膽莊管家。

== 武當派 ==
*[[馬真|馬真]]──武當派掌門人，被張召重毁去雙目、斬去左腿後撞牆而死。
*[[陸菲青|陸菲青]]──武當派掌門人，外號「綿裏針」，武功高絕，早年反清志士，被朝廷通緝，化名「陸高止」於李可秀家中任夫子，後收李沅芷為徒。為趙半山的結義兄弟，出家後道號「無青子」。
*[[張召重|張召重]]──武當派，陸菲青師弟，武功冠絕眾師兄弟，外號『火手判官』，驍騎營佐領。生平作惡多端，最後在沙城被惡狼咬死。

==回部==
*[[霍青桐|霍青桐]]——回部女子，外號「翠羽黃衫」，天山派。
*[[喀絲麗|喀絲麗]]——霍青桐的妹妺，人稱「香香公主」。
*[[木卓倫|木卓倫]]——回族領袖，霍青桐與喀絲麗的父親。最後與兒子霍阿伊戰死在沙場。
*[[袁士霄|袁士霄]]──外號「天池怪侠」，陈家洛师父。自幼與關明梅一起長大，青梅竹馬，後來因為一些小事不合，那時候袁士霄性格古怪，一氣之下遠走漠北。失意之下，發願要做前人未做之事，打前人未打之拳。因而到處尋訪名家武學，利用一切手段，包括學師、偷拳、挑鬥踢場觀其招、明搶暗奪取其譜，幾乎把各門派之拳術學全。中年之後回去，卻發現關明梅已嫁人，黯然神傷。關明梅的丈夫陳正德對袁士霄的歸來不感到高興，便帶著關明梅到回疆躲藏。袁士霄卻一路追隨而來，隱居在天池。由於他通曉百家拳術，自創出一套“百花錯拳”，將各路拳法包容其中，又全然與各拳法原本之路數不同。後來收了[[陳家洛|陳家洛]]為徒，將此在江湖所學之拳法，與自創的“百花錯拳”教授於他。小說結尾他將摧殘許多回部部落的狼群殲滅。武功之淵博高強，可算乃當世第一人，只好友于萬亭方可與其匹敵。
*[[陳正德|陳正德]]──天山雙鷹之一「禿鷲」。在皇宮中被刺身亡。
*[[關明梅|關明梅]]──天山雙鷹之一「雪鵰」，陳正德之妻、霍青桐之師父。最後因陳正德被刺身亡，因而自盡。
*[[阿凡提|阿凡提]]——使鐵鍋的一個怪異老人，面容黝黑，一叢大鬍子遮住了半邊臉，眼睛很小，又喜歡笑，和藹可親。騎了一頭一跛一拐的瘦毛驢，背上負了一隻大鐵鍋，頭戴花帽，有點裝模作樣，還有點瘋瘋癲癲。武功輕功極高，尢勝火手判官張召重。
*[[霍阿伊|霍阿伊]]——木卓倫的長子，與父親同時戰死在沙場。

== 清廷 ==
*[[乾隆帝|乾隆帝]]——陳家洛之兄，化名東方耳。
*[[福康安|福康安]]——乾隆帝的私生子，相貌與[[陳家洛|陳家洛]]極為相似。
*[[白振|白振]]──嵩陽派高手，外號『金爪鐵鉤』。皇帝侍衛高手之一。在舊版中自刎而死，在新版中則離開皇宮。
*[[褚圓|褚圓]]──法名智圓，后來犯了清規，被追繳度牒，逐出廟門外號『一葦渡江』。皇帝侍衛高手之一。
*[[龍駿|龍駿]]——御前侍衛之一。
*[[范中恩|范中恩]]——御前侍衛之一，因曾於西湖落水，在皇帝面前出丑受辱，怀恨在心，見文泰來即將獲救，不管李可秀死活點著藥線，最後被陳家洛丟入火圈內炸死。
*馬敬俠——御前侍衛之一。揹著乾隆逃命後被陳正德一劍穿心。
*瑞大林——御前侍衛之一。被文泰來一拳打死。
*成璜——御前侍衛之一。被天鏡禪師一掌打死。
*[[胡國棟|胡國棟]]──北京名捕頭。
*[[忽倫四虎|忽倫四虎]]──[[忽倫大虎|忽倫大虎]]、[[忽倫二虎|忽倫二虎]]、[[忽倫三虎|忽倫三虎]]、[[忽倫四虎|忽倫四虎]]最後在清軍攻打回族失敗後被捕，經陳家洛勸說後回到遼東牧羊。

=== 李家 ===
*[[李可秀|李可秀]]：先任陝西扶風延綏鎮總兵，後任浙江水陸提督，李沅芷父親。
*[[李沅芷|李沅芷]]：浙江水陸提督[[李可秀|李可秀]]之女，性格精靈古怪，聰慧活潑，敢愛敢恨，武當派陸菲青之徒，後成余魚同之妻。

==關東六魔==
*[[滕一雷|滕一雷]]：被袁士霄舉起來丟在半山腰，腦漿迸裂而死。
*[[顧金標|顧金標]]：死於霍青桐手中。
*[[焦文期|焦文期]]：死於陸菲青之手。
*[[哈合台|哈合台]]：最後回到蒙古，六魔唯一沒死的。
*[[閻世魁|閻世魁]]：被回族人以大鐵錐打死。
*[[閻世章|閻世章]]：死於霍青桐手中。

==其他人物==
*[[貝人龍|貝人龍]]──外號「玉判官」，兩湖豪傑，武器是雙鈎[[吳鈎劍|吳鈎劍]]，死於陸菲青之手。
*[[羅信|羅信]]──外號「鐵臂羅漢」，擅[[五行拳|五行拳]]，武器是長鞭，死於陸菲青之手。
*[[王維揚|王維揚]]──外號「威震河朔」，鎮遠鏢局總鑣頭。
*[[唐六爺|唐六爺]]──外號「糖裡砒霜」，因作惡多端被周綺一刀殺死。
*[[童兆和|童兆和]]——鎮遠鏢局一位鏢頭和朝廷一起捉拿文泰來，一生作惡多端，先是害死了周綺的弟弟並且放火燒了鐵膽莊，後又綁走了周綺的母親，在客棧中被周綺和徐天宏下藥後一刀殺死。
*[[方有德|方有德]]——因在少林寺搶奪周琦兒子又放火燒了少林，最後被趙半山的毒蒺藜殺死。
