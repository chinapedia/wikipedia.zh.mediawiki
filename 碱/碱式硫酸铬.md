'''碱式硫酸铬'''（{{lang-en|Basic chromic sulfate}}），又稱'''硫酸羟铬'''、'''铬粉'''，是一種[[無機化合物|無機化合物]]，其[[化學式|化學式]]為Cr(OH)SO<sub>4</sub>，亦可以記為CrHO<sub>5</sub>S<ref>[http://pubchem.ncbi.nlm.nih.gov/summary/summary.cgi?cid=61561 chromium sulfate - Compound Summary] pubchem.ncbi.nlm.nih.gov [2013-11-16]</ref>，由三价[[铬|铬]]組成，主要用于[[鞣革|鞣革]]工业。

== 性質 ==
常溫下為[[無定形體|無定形體]]，外觀為[[墨綠色|墨綠色]][[粉末|粉末]]偶爾呈現片狀物，其顆粒在顯微鏡下能觀察到有其具有[[玻璃|玻璃]]的結晶狀態。鹼式硫酸鉻易溶於[[水|水]]，不溶於[[醇|醇]]。容易吸水而[[潮解|潮解]]。鹼式硫酸鉻有毒，食用有害<ref>[http://www.chemicalbook.com/ChemicalProductProperty_EN_CB6874896.htm Chromium sulfate,basic,solid] chemicalbook.com [2013-11-16]</ref>。

== 製備 ==
製備鹼式硫酸鉻有很多種方法，甚至可以用[[硫酸鉻|硫酸鉻]]加[[強鹼|強鹼]]，如[[氫氧化鈉|氫氧化鈉]]來獲得，但濃度很低沒有實用價值，一般工業上常以兩種方法製備鹼式硫酸鉻，[[有機還原法|有機還原法]]和[[二氧化硫|二氧化硫]][[還原反應|還原]]法。

有機還原法一般使用[[蔗糖|蔗糖]]來還原六價的[[鉻|鉻]][[離子|離子]]，較常使用[[重鉻酸鹽|重鉻酸鹽]]來作為六價鉻的提供者，如[[重鉻酸鉀|重鉻酸鉀]]。反應的過程會因加入的是[[硫酸氫鹽|硫酸氫鹽]]或[[硫酸|硫酸]]有所差別，但兩者皆有相同的[[生成物|生成物]]，鹼式硫酸鉻，[[副產物|副產物]]也相同，有[[硫酸鈉|硫酸鈉]]，[[水|水]]和[[二氧化碳|二氧化碳]]。

使用硫酸氫鈉的反應機制如下:
:8 Na<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 48 NaHSO<sub>4</sub> + C<sub>12</sub>H<sub>22</sub>O<sub>11</sub> → 16 Cr(OH)SO<sub>4</sub> + 32 Na<sub>2</sub>SO<sub>4</sub> + 43 H<sub>2</sub>O + 12 CO<sub>2</sub>
使用硫酸的反應機制如下:
:8 Na<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub> + 24 H<sub>2</sub>SO<sub>4</sub> + C<sub>12</sub>H<sub>22</sub>O<sub>11</sub> → 16 Cr(OH)SO<sub>4</sub> + 8 Na<sub>2</sub>SO<sub>4</sub> + 27 H<sub>2</sub>O + 12 CO<sub>2</sub>
兩者皆會生成硫酸鈉，可利用降低溫度造成溶解度差使硫酸鈉優先結晶來分離出鹼式硫酸鉻。

== 用途 ==
鹼式硫酸鉻主要用于[[鞣革|鞣革]]工业，也可用在[[陶瓷|陶瓷]][[釉彩|釉彩]]上。另外在[[印刷|印刷]]技術方面，鹼式硫酸鉻可以最為[[活性|活性]][[黑|黑]][[染料|染料]]、[[绿色|绿色]][[油墨|油墨]]。此望有些工業[[無機|無機]][[鉻化合物|鉻化合物]]合成也會使用鹼式硫酸鉻作為原料之一，例如:生產[[氢氧化铬|氢氧化铬]]可以使用鹼式硫酸鉻作為原料。
