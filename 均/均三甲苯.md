'''均三甲苯'''（mesitylene），又名'''1,3,5-三甲苯'''、'''艹+米'''<ref>{{Cite web |url = http://terms.naer.edu.tw/terms/manager_admin/new_file_download.php?Pact=FileDownLoad&button_num=g5&source_id=160&Pval=68 |title = 化學命名原則第四版較第三版新增部分 |language = zh-tw |accessdate = 2018-09-09 }}</ref>，分子式C<sub>9</sub>H<sub>12</sub>，是[[苯|苯]]环上三个[[氢|氢]]对称地被三个[[甲基|甲基]]取代而得到的[[芳香烃|芳香烃]]。

== 制取 ==
用[[硫酸|硫酸]]处理[[丙酮|丙酮]]后[[蒸馏|蒸馏]]（发生[[羟醛缩合|羟醛缩合]]反应），或者将[[丙炔|丙炔]]在硫酸作用下三聚，都可以制得均三甲苯。在这两个反应中，硫酸既作为[[催化剂|催化剂]]，又作为[[脱水反应|脱水]]剂。

== 应用 ==
在[[科学研究|科学研究]]和[[化工|化工]]生产中，均三甲苯广泛地用作溶剂。它[[可燃|可燃]]，有[[刺激性|刺激性]]，而且凝固点很低。

在电子工业中，由于均三甲苯是一种良好的溶剂，它还被用作硅酮感光片的[[显影剂|显影剂]]。

均三甲苯也是一种城市中常见的[[挥发性有机化合物|挥发性有机化合物]]（VOC），主要由[[燃烧|燃烧]]产生。它在很多[[大气化学|大气化学]]的反应中发挥重要作用（包括气雾和[[对流层臭氧|对流层臭氧]]的生成）。

均三甲苯基（Mes）是一个在多种有机化合物中存在的[[官能团|官能团]]。

芳环上的三个氢有着相同的化学环境，它们对应均三甲苯的[[核磁共振氢谱|核磁共振氢谱]]中6.8ppm处的3个氢的单峰。因此，均三甲苯有时在包含芳香质子的有机样品的核磁共振测定中用作[[内标物|内标物]]。<ref>{{cite web|url=http://chemicalland21.com/industrialchem/organic/MESITYLENE.htm|title=MESITYLENE (1,3,5-TRIMETHYL BENZENE)|website=chemicalland21.com}}</ref>
