{{nihongo|'''桐谷和人'''|{{ruby|桐ヶ谷|きりがや}} {{ruby|和人|かずと}}|Kirigaya Kazuto}}，虛擬世界的用戶名為「{{nihongo|'''桐人'''|キリト|Kirito}}」，是日本小說家[[川原礫|川原礫]]筆下的輕小說作品《[[刀劍神域|刀劍神域]]》及其改編動、漫畫作品的主角。在以《刀劍神域》為主的諸多系列作裡作為主角或重要角色登場。

== 人物 ==
原名為{{nihongo|'''鳴坂和人'''|{{ruby|鳴坂|なるさか}}{{ruby|和人|かずと}}|Narusaka Kazuto}}，出生後不久父母便因車禍雙亡，大難不死的和人被阿姨桐谷翠和姨丈桐谷峰嵩收養，並改姓「桐谷」。

生性冷靜聰明，外觀如同柔弱的少女般，態度冷淡，帶給不認識的人一種「捉摸不定」、「年齡不詳」的印象。如果穿著和妹妹同樣的衣服出門的話，甚至有被當成姊妹的可能<ref>小說第一集</ref>。

在電子機械工程方面非常有天賦，在六歲時第一次用廢棄的零件組出了一台電腦，10歲時發現自己的電子戶口有被修改過的痕跡，更進一步發覺自己作為養子的身世，令養父母十分驚訝。

自從發現自己作為養子的身世後，和人就開始對現實世界的人際關係產生疏離感，並開始以本名縮寫來的暱稱「桐人」沉浸在網路遊戲的世界中，成為名副其實的「重度玩家」，在遊戲環境內累積了諸多相關的知識及反應力，而這些才能通通因為SAO嚴酷的淬鍊而開花結果，論遊戲的才能甚至被SAO的開發者[[刀劍神域角色列表#茅場晶彥|茅場晶彥]]評為最強等級。

由於SAO時代長期在最前線作戰的緣故，使得桐人經歷了許多在VRMMORPG的環境下真正危及性命的「實戰」經驗，也讓現實中的和人保有著相當強勁的劍術實力，配合原本就過人、甚至在SAO裡進一步得到淬鍊的洞察力及反射神經，甚至能夠在技巧上壓制住全國中學劍道大賽前八強的妹妹直葉。而這些經歷生死關頭累積下來的實力也讓桐人在某些非等級制的VRMMORPG中得以和頂尖玩家平起平坐。

後來在SAO跟西莉卡透露桐谷直葉不是自己的親妹妹，而是自己的表妹。

== 劇情表現 ==
; {{nihongo|艾恩葛朗特篇|アインクラッド}}
: 第1部。收錄卷數：第1・2卷、第8卷部分內容
:「這雖然是遊戲，但可不是鬧著玩的（これは、ゲームであっても、遊びではない）。」—— 茅場晶彥
:2022年，大型電子機械製造商「ARGUS」發表了新世代的虛擬實境連結裝置「NERvGear」使得真正意義上「完全」的虛擬實境得以實現，並於同年推出史上第一個運用NERvGear作為人機介面的[[虛擬實境|VR]][[MMORPG|MMORPG]]「刀劍神域Online（Sword Art Online）」，有幸抽中僅限1000名封測玩家資格（附帶正式版的優先購買權）的桐人，就此一頭栽進這個比以往都更真實地扮演「另一個自己」的遊戲世界中。在正式營運當天更是馬上「[[刀劍神域世界觀與設定#潛行|完全潛行]]」，和同樣買到正式版的另外9999名玩家一起，享受著正式版的SAO世界。

:然而，四小時多後，包含桐人在內的所有玩家發現登出指令竟然離奇地消失無蹤，隨後所有玩家一起被傳送到開始地點廣場。而玩家所等到的並非是故障的系統公告，而是自稱是SAO遊戲設計者「茅場晶彥」的人淡淡開始的死亡遊戲說明：除了打倒位於「[[刀劍神域世界觀與設定#艾恩|艾恩葛朗特]]頂樓第100層的最終 [[頭目_(角色類型)|頭目]]」，達成「完全攻略」以外，沒有其他能讓玩家登出的方法。並且，在遊戲內生命值歸零，或是嘗試脫下NERvGear，玩家會立刻被NERvGear發出的高頻率微波破壞腦部而死亡。自此，名為SAO的死亡遊戲就這樣揭開了序幕。

:因為恐懼的影響，最初的一個月內死亡人數就突破了2000多人，但在那之後，已經有一部分玩家開始了攻略，一個多月後第一層被通關，而第二層也在十天後被擺平。然而第一層時卻發生了樓層頭目的動作模式與封測玩家提供的情報不同，導致攻略團隊領頭陣亡的事故，為了避免其他提供情報的封測玩家遭人刁難，桐人當場高聲自稱為「封弊者（ビーター）」，承擔了所有對於封測玩家的仇恨，自此，這個充滿貶意的稱號就一直跟著他。

:約在2023年，桐人因故受邀加入期望自我提升、加入攻略集團的公會「月夜黑貓團」並開始指導團員們戰鬥和升級的方法。然而，「月夜黑貓團」的成員卻在同年六月於一次迷宮探險時中了陷阱，導致除了恰好出外辦事的會長以外，僅有桐人一個人因懸殊的等級差距生還的慘劇。這件事造成桐人極為嚴重的[[心理創傷|心理創傷]]，更因此害怕擁有夥伴或與人扯上關係，深怕自己會害死自己重視的人<ref name=月夜黑貓團事件>小說版（文庫版）第2卷第四章《紅鼻子麋鹿》；動畫版第1期第3集《紅鼻子馴鹿》</ref>。直到他遇到生命中的伴侶亞絲娜，才逐漸克服這個陰影。

:在SAO的兩年期間是以攻略艾恩葛朗特最上層為目標，作為頂尖玩家集團「[[刀劍神域世界觀與設定#攻略|攻略組]]」中菁英的獨行玩家，擅長使用單手劍，並因為披著黑色大衣這一渾身漆黑的形象而獲得「黑色劍士」這一別名。而因為緊急情況下以獨有技能「二刀流」獨力斬殺第74層頭目「閃耀魔眼（The Gleam Eyes）」、拯救其他玩家的事蹟，其知名度一炮而紅，被許多玩家視為和另一個獨有技能的持有者——最強公會「血盟騎士團」的團長「神聖劍」希茲克利夫其名的最強玩家。

:由於在決鬥中因故敗給希茲克利夫，使得桐人必須遵守約定加入「血盟騎士團」，但就在加入不久後，其中一名團員克拉帝爾（Kuradeel）便在共同訓練中意圖製造「桐人和其他人遇難，僅有自己活下來」的意外，藉此對桐人報復，但最後被副團長亞絲娜所阻止，雖然最後無法阻止其他同行團員犧牲，但最後整起事件以克拉帝爾被桐人就地正法落幕。而原本就互相抱有好感的亞絲娜及桐人在這次生死交關的危機之後終於真正地心意相通，並且在系統上「結婚」，並買下了艾恩葛朗特22層的玩家小屋開始了新婚生活，在這期間則因故收養了AI結衣作為兩人的「女兒」。<ref>小說第二集</ref>

:在搬到22層約兩週後，新婚中的兩人收到了希茲克利夫的緊急聯絡：第75層頭目的房間一旦進入後就必須將之打倒，否則門下一次打開會是所有人全滅的時候。有鑑於事態的危急，桐人和亞絲娜不得不重返前線支援。然而，該層頭目兇猛的程度，依然導致了32名玩家中有14名犧牲者的重大傷亡。在打倒頭目之後的片刻休息時間中，桐人意外發現了希茲克利夫「生命值從未降至一半以下」這一傳說的真相——系統上被視為「不可破壞物體」，更進一步識破其真正身分就是作為SAO這一遊戲的開發者茅場晶彥。

:做為桐人識破其身分的獎勵，茅場晶彥——也就是希茲克利夫給予了桐人一個機會：在現場一對一決鬥擊敗他的話，就算遊戲通關。然而，桐人因為太過在意對方作為系統管理者的身分及權限而導致心理上出現破綻，導致被對方擊碎了左手的佩劍，但最後致命的一擊卻被突然介入戰局的亞絲娜用身體擋了下來，也將亞絲娜的生命值一分不留的完全削去。亞絲娜的死，讓桐人放棄了活下去的念頭，但亞絲娜蘊藏在佩劍裡的思念成了最後的助力，讓桐人在被希茲克利夫刺穿胸膛的狀態下將亞絲娜的佩劍送入了對方的胸口，成功與對方同歸於盡，為時近兩年的艾恩葛朗特攻略行動亦就此畫下句點。

== 裝備和技能 ==
=== 技能 ===
; 單手劍術
:桐人早年曾受過爺爺些許的劍道指導，因此本身就對用劍有一點心得，而在SAO裡長期累積的戰鬥經驗並未隨著SAO的通關而就此消失，反而是深刻的留在他的腦海，使得桐人本身除了在假想遊戲世界幾乎打遍天下無敵手之外，在現實世界裡也能維持一定的劍術水準。

; 反射神經與洞察力
:桐人先天的反射神經及洞察力就相當優異，而在SAO裡的諸多實戰經驗讓他培養出了異於常人的反應速度及優異戰鬥直覺，甚至能夠感受到對手散發的「殺氣」，這樣的能力與經驗使得桐人能夠經歷諸多的實戰而平安生還，讓他擁有超乎常人想像的強勁實力。

;{{nihongo|[[雙持|二刀流]]|二刀流}}
: 在SAO中，除了一般任何玩家都能取得的技能以外，尚存在有「僅限一人能獲得」的特殊「{{nihongo|[[刀劍神域世界觀與設定#獨技|獨特技能]]|Unique Skill}}」，其中由桐人持有的「二刀流」技能的取得條件為「擁有遊戲內最快的反應速度」。
:在二刀流技能的效果下，桐人能夠不受到一般玩家一次只能發動一招[[刀劍神域世界觀與設定#技能|劍技]]的限制，能夠兩手同時發動劍技，在戰鬥中獲得更為靈活的應用，同時也能夠使用二刀流技能的專屬劍技，其中存在使用紀錄的有「[[星爆氣流斬|星爆氣流斬]]（Starburst Stream）{{noteTag|最初小說版中文翻譯是翻成「星爆氣流斬」，但之後官方正式定名為「星光連流擊」。}}」（16連擊）、「日蝕（The Eclipse）」（27連擊）和「雙重扇形斬（Duble Circular）」（2連擊突進技）。然而，若不是具備良好平衡性的使用者，會造成兩把劍互相干涉而難以施展，也因此要一次使用兩把劍並非易事，從中亦可看出桐人背後的強勁實力。
