'''迎泽公园'''，位于[[中国|中国]][[山西省|山西省]][[太原市|太原市]][[迎泽大街|迎泽大街]]南侧，是[[太原市|太原市]]最大的综合性文化游览公园<ref name="About">{{cite book|author=山西省太原市文史委|title=太原市城建志|pages=57|url=http://c.g.wanfangdata.com.cn/LocalChronicle-FZ001597.aspx}}</ref>。

==简介==
迎泽公园面积为1000余亩，其中湖面240亩<ref name="About"/>。公园原址是一片积水坑。1954年开始，太原市拆迁原有居民，依势挖湖堆山，大规模进行绿化，筹建公园。1956年5月1日正式开放。

1991年9月24日晚8点30分，公园内发生[[1991年太原踩踏事件|特大伤亡事故]]。当时，观赏“煤海之光”灯展的人群在公园七孔桥处发生踩踏，至少105人被挤死，108人被挤伤<ref name="9-24">{{cite web|title=迎泽公园“煤海之光”灯展特大伤亡事故|url=http://www.tyfzw.cn/?sw=774&b=177|work=太原方志网|publisher=山西省太原市政府地方志办公室|accessdate=2014-12-07}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。山西省组成以常务副省长[[郭裕怀|郭裕怀]]为组长的调查组，中央和国务院有关部委也派员到山西省指导调查。1992年3月5日，[[山西省人民代表大会|山西省人民代表大会]]常务委员会第27次会议认为，[[山西省人民政府|山西省人民政府]]副省长[[李振华_(中华人民共和国)|李振华]]在事故中所犯错误严重，会议审议通过了省长[[王森浩|王森浩]]提请的关于撤销李振华副省长职务的议案，决定撤销其副省长职务<ref>{{Cite web |url = http://dangshi.people.com.cn/GB/146570/198300/200214/200219/12464982.html |title = 1992年3月5日 撤销李振华山西省副省长职务 |author =  |date =  |publisher = 中国共产党新闻网 |language =  2017-05-22|accessdate =  |quote =  }}</ref>。

2002年前，迎泽公园曾实行收费入园制度，门票每人2元。2002年5月，太原市人民政府对迎泽公园进行了大规模改造<ref>{{cite news|author=王炤坤;叶健;刘云伶|title=太原市迎泽公园“还园于民”:拆除围墙 皆大欢喜|url=http://www.gov.cn/jrzg/2007-05/21/content_620762.htm|accessdate=2014-07-27|newspaper=新华社|date=2007-05-21}}</ref>，2002年7月改造后的迎泽公园免费向市民开放。2016年7月16日，迎泽公园再一次进行改造。是次改造投入资金最多、规模最大。2017年10月1日，迎泽公园重新开放。<ref>{{Cite web|url=http://www.tyxc.gov.cn/content/2017-09/29/content_83622.htm|title=公园新貌美如画 迎泽公园确定10月1日开门迎客|accessdate=2018-12-12|author=太原宣传网|date=|work=www.tyxc.gov.cn|publisher=}}</ref>
