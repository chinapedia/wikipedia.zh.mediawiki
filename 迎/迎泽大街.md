'''迎泽大街'''是中華人民共和國[[山西省|山西省]][[太原市|太原市]]的一条主要道路，始建于1956年，当时的设计宽度为70米，在那时的中国仅次于北京[[长安街|长安街]]，以故赢得了“中国第二街”、“太原的长安街”、“三晋第一街”<ref name="zxsx">{{cite news|title="中国第二街"迎泽大街51年来首次改造 实行半封闭|url=http://www.sx.chinanews.com/2007-05-09/1/46844.html|accessdate=2012-04-02|newspaper=中新山西网|date=2007-05-09}}</ref> 的称号。迎泽大街历经四次增建，目前有双向14条机动车道（含2条公交专用道），以及非机动车道、人行道、绿化带各两条。迎泽大街东起[[太原火车站|太原火车站]]，向西延伸跨越[[汾河|汾河]]直抵南寒广场，全长10公里，汾河以西的部分被称为迎泽西大街。道路沿线有[[太原火车站|太原火车站]]、[[五一广场_(太原市)|五一广场]]、[[迎泽公园|迎泽公园]]、[[太原工人文化宫|太原工人文化宫]]、[[迎泽宾馆|迎泽宾馆]]、[[并州饭店|并州饭店]]、[[中国煤炭博物馆|中国煤炭博物馆]]、[[太原理工大学|太原理工大学]]等地标建筑。时至今日，迎泽大街虽不再是太原市最宽的大街，但它已经成为太原市的象征之一<ref>{{cite journal|author=高正|title=迎泽大街展雄姿|journal=今日山西|year=1997年|issue=第1期|pages=31}}</ref>。

== 得名 ==
关于迎泽大街的得名，有一个广为流传的说法：20世纪50年代，为迎接[[毛泽东|毛泽东]]到太原视察，太原市政府修建了这条大街，为此命名为“迎泽大街”，取“迎”接毛“泽”东之意。事实上毛泽东终其一生并没有来过太原。更加可信的说法是，老太原城的大南门“[[迎泽门|迎泽门]]”（已被拆除）正好处在迎泽大街上，大街以迎泽门而得名。而迎泽门的名字，一说来源于门外的沼泽地，一说与《[[南风歌|南风歌]]》有关：“南风之薰兮，可以解吾民之愠兮；南风之时兮，可以阜吾民之财兮。”迎泽门在太原城的南面，定名“迎泽”，取迎接南风的恩泽，解民忧阜民财之意。除迎泽大街之外，太原市的迎泽公园、迎泽大桥、[[迎泽区|迎泽区]]皆以“迎泽”为名<ref>{{cite news|title=太原：迎泽大街的来历|url=http://www.sx.chinanews.com/news/2011/0426/36667.html|accessdate=2012-04-02|newspaper=发展导报|date=2011-04-26}}</ref>。

== 初建 ==
20世纪50年代初，时任太原市市长[[岳维藩|岳维藩]]主持制定了太原市1949年后的第一部城市规划，迎泽大街是整个规划中最引人注目的部分。规划中的迎泽大街以太原南城外的一条土路为基础，设计宽度为70米，这在当时引起了相当大的争议，华北局会议上这一方案被批评为“贪大求洋”。50年代初，太原市的大多数街道只有六七米宽<ref>{{cite news|title=迎泽大街：见证太原几十年发展变迁|url=http://www.sx.xinhuanet.com/newscenter/2009-10/04/content_17871310.htm|accessdate=2012-04-02|newspaper=山西电视台|date=2009-10-04|deadurl=yes|archiveurl=https://web.archive.org/web/20131109042838/http://www.sx.xinhuanet.com/newscenter/2009-10/04/content_17871310.htm|archivedate=2013-11-09}}</ref>，70米的宽度足以同时容许四千辆车通行，而当时全太原市一共只有八百辆汽车。面对争议，岳维藩主张城建应当本着“五十年不落后、一百年不后悔”的思想，着眼于太原市的长远发展。最终岳维藩力排众议，迎泽大街的建设得以在1956年拉开大幕。一年后，东至五一广场、西至迎泽大桥的一期工程完工。由于道旁建筑的影响，最终一期工程的实际路宽定为66米<ref>{{cite journal|author=张捷夫 杨富山|title=岳维藩与新太原建设|journal=沧桑|year=1999年|issue=第3期|pages=16}}</ref>。

== 增建 ==
1976年，迎泽大街第二期工程动工，经过两年的建设，迎泽大街的东端由五一广场延至太原火车站（广场到火车站的一段曾一度称“迎泽东大街”）。1985年实施了“迎泽西大街取直工程”，一年后迎泽大街西延至下元（这一段称为“迎泽西大街”）。1996年的“迎泽西大街打通工程”使得迎泽西大街在同年向西通到南寒广场。<ref name="zxsx"/>2007年，迎泽大街迎来了修建以来的第一次拓宽改造<ref>{{cite news|title=好漂亮 迎泽大街展露新姿容|url=http://www.daynews.com.cn/news/tyxw/351292.html|accessdate=2012-04-02|newspaper=山西新闻网 山西晚报|date=2007-09-16|deadurl=yes|archiveurl=https://web.archive.org/web/20131109042619/http://www.daynews.com.cn/news/tyxw/351292.html|archivedate=2013-11-09}}</ref>，并配套修建了“微循环道路”以缓解周围道路的交通压力<ref>{{cite news|title=迎泽大街微循环道路将于近日竣工|url=http://www.daynews.com.cn/news/tyxw/343227.html|accessdate=2012-04-02|newspaper=山西新闻网 三晋都市报|date=2007-09-06|deadurl=yes|archiveurl=https://web.archive.org/web/20131109042556/http://www.daynews.com.cn/news/tyxw/343227.html|archivedate=2013-11-09}}</ref>。
