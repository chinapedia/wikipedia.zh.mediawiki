《'''迎頭重擊'''》（{{lang-en|'''''Headshot'''''}}）是一部2016年[[印尼|印尼]]{{le|武術片|Martial arts film|武術}}[[動作片|動作片]]，由齊莫·史坦波和提摩·塔堅德共同執導，而劇本則為塔堅德所撰寫。電影主演包括[[伊科·烏艾斯|伊科·烏艾斯]]、{{le|茱莉·艾絲特爾|Julie Estelle}}、{{le|雀兒喜·伊斯蘭|Chelsea Islan}}與{{le|查克·李|Zack Lee}}。

電影最先於2016年9月9日的[[2016年多倫多國際影展|多倫多國際影展]]上首映，並於同年12月8日在印尼上映，而美國則於2017年3月3日上映。

== 劇情 ==
故事敘述一名受重傷的失憶男子受到了美麗醫師艾琳的照顧，並給他起了美國著名文學《[[白鯨記|白鯨記]]》中主角「以實瑪利」一名字。以實瑪利的情況漸漸好轉，並打算展開全新的生活；但同時，他的過去找上了自己，使得一場腥風血雨的黑暗即將襲來<ref>[http://deadline.com/2016/09/headshot-toronto-film-festival-xyz-films-iko-uwais-1201823244/ Iko Uwais Starrer ‘Headshot’ Scores Global Deals – Toronto]</ref>。

== 發行 ==
《迎頭重擊》最先於2016年9月9日的[[2016年多倫多國際影展|多倫多國際影展]]上作全球首映<ref name="kompasPangerang">{{cite web |url=http://entertainment.kompas.com/read/2016/09/20/200419110/belum.tayang.di.indonesia.headshot.dapat.penghargaan.di.perancis |title=Belum Tayang di Indonesia, "Headshot" Dapat Penghargaan di Perancis |first=Andi Muttya Keteng |last=Pangerang |editor=Kistyarini |website=Kompas.com |date=20 September 2016 |access-date=2 December 2016}}</ref>。後於2016年9月24日及29日在[[奇幻電影節|奇幻電影節]]上作兩次放映<ref>{{cite web |url=http://entertainment.kompas.com/read/2016/09/28/125024710/.headshot.diputar.di.amerika.serikat |title="Headshot" Diputar di Amerika Serikat |first=Andi Muttya Keteng |last=Pangerang |editor=Kistyarini |website=Kompas.com |date=28 September 2016 |access-date=2 December 2016}}</ref>。10月4日，電影在美國[[洛杉磯|洛杉磯]]的Beyond Fest上放映。電影也於10月13日在Mayhem影展放映。

而印尼本地於2016年12月8日上映<ref>{{cite web |url=http://entertainment.kompas.com/read/2016/12/01/222125410/bertarung.di.pantai.iko.uwais.kelilipan.pasir |title=Bertarung di Pantai, Iko Uwais Kelilipan Pasir |first=Andi Muttya Ketteng |last=Pangerang |editor-first=Irfan |editor-last=Maullana |website=Kompas.com |date=1 December 2016 |access-date=2 December 2016}}</ref>。
