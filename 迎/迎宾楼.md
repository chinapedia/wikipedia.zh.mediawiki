'''迎宾楼'''，是位于[[中国|中国]][[江苏|江苏]][[无锡市|无锡市]]的一家老字号饭店，以经营京苏大菜为特色，其由1930年开业至今，已有80余年历史。美食家[[陆文夫|陆文夫]]曾将迎宾楼与苏州[[松鹤楼|松鹤楼]]、杭州[[楼外楼|楼外楼]]并称为“中国三大楼”。<ref name="历史">[http://www.wxrb.com/node/kandian/2012-3-16/59K1E6634272300.html 〈老字号迎宾楼再现崇安寺 品味老无锡的味道〉，无锡新传媒（2012年3月16日）。]</ref>

== 简介 ==

无锡迎宾楼创办于1930年3月3日，由李文毓、沈运清、顾字彦和尤锦山父子等集股创设，位于城中心寺巷内，[[崇安寺_(无锡)|崇安寺]]山门口右侧。当时特聘在[[先施公司|上海先施公司]]东亚酒家的无锡名厨刘俊英回锡掌厨，<ref name="简介">徐桥猛、王力行编，《无锡菜典》（中国轻工业出版社，2007年），页241。</ref>挂以“京都”金字招牌。当时的迎宾楼菜馆是无锡社会名菜馆之魁，曾接待过不少名流。1935年2月下旬，影星[[阮玲玉|阮玲玉]]来到无锡，由中南大戏院在迎宾楼设宴接风，菜肴尽是地方风味，她盛赞无锡的鲜鱼活虾，鲜美可口。1946年10月31日，[[蒋中正|蒋中正]]偕[[宋美龄|宋美龄]]暨[[马歇尔|马歇尔]]夫妇，来锡避寿，乘坐苹香号画舫游览，举办寿筵六桌（实际吃五桌）。由迎宾楼菜馆主厨丁子坤师傅操作，办在画舫中。食后，宋美龄对马歇尔说：“吃鱼蟹，要到无锡来”。<ref name="简介"></ref>

1954年，迎宾楼曾中断营业，到1959年得以恢复时，仍以经营京苏风味为特色。上世纪80年代，迎宾楼被无锡市饮食服务公司接手，由一层改造成上下四层，在顶楼还建造了当年风靡一时的露天酒吧，成为无锡唯一的夜生活场所。2004年，迎宾楼随着崇安寺一期改造被拆除。2009年，崇安寺改造动工建设，迎宾楼被规划在内。随着崇安寺二期开街，迎宾楼第三次在崇安寺开业，一晃已是近八年。<ref name="历史"></ref>

== 知名菜品 ==

该店有名的菜肴有：蟹粉鱼翅、荷花牡丹、鸡茸蛋、罗汉全鸭、风尾鸽蛋、白雪鸳鸯、广州鲫鱼、金玉满堂等，擅长炸、溜、爆、炒、熏、蒸、焖等烹饪技艺。<ref name="简介"></ref>

== 名厨 ==

* '''刘俊英'''（1901-1991年）：无锡迎宾楼菜馆名厨，20世纪20年代在大新楼菜馆推出“锅塌青鱼片”，一举成名。1928年在[[上海|上海]][[东亚饭店|东亚饭店]]和大喜利菜馆主厨。1930年3月3日无锡迎宾楼菜馆开业，特聘其回锡掌厨。拿手菜有：蟹粉鱼翅、荷花牡丹、鸡茸蛋、罗汉全鸭、凤尾鸽蛋、杏仁葛粉包、[[镜箱豆腐|镜箱豆腐]]等，以原汁原味、色泽和谐为特色，尤擅长烹制陆、海珍馐和应时花点。刘俊英是锡菜的一个重要流派，多年来影响深远，是当代锡菜奠基人之一。<ref name="名厨">徐桥猛、王力行编，《无锡菜典》（中国轻工业出版社，2007年），页260。</ref>

* '''龚裕金'''（1912-1984年)：1925年去上海大同里学艺，1940年回锡，被迎宾楼菜馆聘为主抓师傅。其参与主理过蒋中正、荣德生等名流之用餐，1949年后又被派往太湖饭店、国际旅行社、无锡饭店、梁溪饭店等处掌厨，先后主理过林彪、徐向前、荣毅仁、[[胡志明|胡志明]]、[[西哈努克|西哈努克]]等国内外政治人物的接待宴会。在太湖饭店时，创制出“锅贴鲫鱼”、“荷叶焗鸡”等名菜，有高徒金志德、王世忠等。<ref name="名厨"></ref>

== 逸事 ==

在当时，“京都”为当时全国餐饮行业内的金字招牌，但凡敢挂此招牌者，必是有底气能办“[[满汉全席|满汉全席]]”的，全国各地菜系亦无所不会烹饪。客人点单，即便是菜单上没有，饭店厨师不但要做得出来而且要做得到位，绝不能以“没有”两字搪塞。当时锡邑挂“京都”招牌的共有三家饭店，迎宾楼是头一家，其他两家分别是1942年开在迎宾楼对面的“迎元春”和1947年在工运路开业的“[[中国饭店|中国饭店]]”。<ref>[http://news.thmz.com/col89/folder664/2014/04/2014-04-191417730_2.html 〈独家曝光老照片：迎宾楼与锡邑第一场西式婚礼〉，《无锡广电・无锡新周刊》（2014年4月19日）。] {{webarchive|url=https://web.archive.org/web/20150203100321/http://news.thmz.com/col89/folder664/2014/04/2014-04-191417730_2.html |date=2015年2月3日 }}</ref>
