'''苑裡鎮'''（[[臺灣客家語|臺灣客家語]][[四縣腔|四縣腔]]：地名本作「彎里」，vanˊ liˊ ziinˋ），是[[中華民國|中華民國]][[苗栗縣|苗栗縣]]內西南隅的鎮，南鄰[[大安溪|大安溪]]北岸與[[臺中市|臺中市]][[大甲區|大甲區]]相鄰，東依[[火炎山_(苗栗縣)|火炎山]]與[[三義鄉_(台灣)|三義鄉]]為鄰，北靠[[通霄鎮|通霄鎮]]，西濱[[臺灣海峽|臺灣海峽]]，是苗栗縣開發最早的鄉鎮之一，在苑裡鎮內水坡里有出產砂糖橘。有「苗栗穀倉」、「苗栗米倉」美譽。人口組成以閩南人略多。<ref>{{cite web | url=http://ebooks.lib.ntu.edu.tw/1_file/hakka/78/51272219271.pdf | title=各鄉鎮客家人口統計 |}}</ref>

==地名緣由==
本鎮地名源自於[[平埔族|平埔族]][[臺灣原住民族|原住民]][[道卡斯族|道卡斯族]]苑裡社（宛里社，Wanrie）譯音而來，依安倍明義台灣地名研究就番語語言譯為「里灣里」，在簡化番語之下，由「彎里」、「灣麗」，仍至古文獻所稱「宛里」、「苑裏」，與今日的「苑裡」均為同音異字，指的都是同一個地方。<br>

== 崩山（蓬山）八社 ==
*清領康熙、雍正年間，稱「崩山八社」或「崩山」，乾隆以後改稱「蓬山八社」或「蓬山」。
*八社：大甲東、大甲西、宛裡、房裏、南日、雙寮、貓盂、吞霄。
*大甲東社（今台中市大甲區、外埔區）、大甲西社（德化社，今台中市大甲區、大安區）、宛裡社（苑裡社，今苗栗縣苑裡鎮）、房裏社（房裡社，今苗栗縣苑裡鎮）、南日社（日南社，今台中市大甲區）、雙寮社（擺陳社，今台中市大甲區）、貓盂社（今苗栗縣苑裡鎮）以及吞霄社（今苗栗縣通霄鎮）。
*另有日北社涵蓋苑裡鎮內區及相鄰鄉鎮。
※資料來源出處<ref>資料來源：《諸羅縣志》。</ref>。

== 建制沿革 ==
*[[明鄭時期|明鄭]][[永曆_(南明)|永曆]]十五年（1661年），本鎮屬[[天興縣|天興縣]]。
*[[臺灣清治時期|清領]][[康熙|康熙]]廿二年（1683年），置一府三縣，本鎮屬[[諸羅縣|諸羅縣]]。
*清領[[雍正|雍正]]元年（1723年），原諸羅縣內增設彰化縣以及大甲溪以北增設[[淡水廳|淡水廳]]。
*清領[[乾隆|乾隆]]廿九年（1764年），設苑裡堡，轄苑裡、吞霄（今通宵）兩街，以及山柑、日北、貓盂、房裡十三莊社。
*清領[[光緒|光緒]]四年（1876年），廢廳改縣，劃淡水廳治為淡水與新竹兩縣，本鎮屬新竹縣。
*清領光緒十五年（1889年），新竹苗栗分治，成立苗栗縣。
*清領光緒廿一年（1895年），日本統治台灣，設台北、台灣、台南三縣，本鎮屬[[台灣縣|台灣縣]]。
*[[台灣日治時期|日治]][[明治|明治]]廿九年（1896年），台灣縣改稱台中縣，本鎮屬之。
*日治明治三十年（1897年），設新竹縣，苑裡設辦務署。
*日治明治三十一年（1898年），歸苗栗辦務署管轄。
*日治明治三十四年（1901年），改苗栗辦務署為[[苗栗廳|苗栗廳]]。
*日治明治四十二年（1909年），設新竹廳通霄支廳，置苑裡區役場及山腳區役場。
*日治[[大正|大正]]九年（1920年），設[[新竹州|新竹州]][[苗栗郡|苗栗郡]]苑裡-{庄}-。
*日治[[昭和|昭和]]十八年（1943年），苑裡-{庄}-改制為[[苑裡街|苑裡街]]。
*[[臺灣#中華民國時期|民國]]三十五年（1946年），設新竹縣苗栗區苑裡鎮。
*民國三十九年（1950年），設苗栗縣苑裡鎮。

※資料來源出處<ref>資料來源：洪敏麟《台灣舊地名之沿革》、連橫《台灣通史》、《諸羅縣志》、劉定國《苗栗縣志》、林坤山《苑裡鎮地名與故事》。</ref>。

== 土地使用 ==
===工業區===
第一階段工業土地面積合計約49.41公頃。

====苑裡北區工業區====
位於苑港里，都市計畫5-1號道路。

====苑裡南區工業區====
本區位於房裡里，南區工業區占地約27公頃

===農業區===
現階段農業灌溉面積合計約2938公頃，灌溉水系有臺中農田水利會苑裡工作站和山腳工作站綜理，約43%苑裡土地面積。

===都市計畫區===
*苑裡市場：2018年9月14日凌晨5點左右火災，原消費市場建築燒毀。
*第五次通盤檢討，首度整體設計加入苑裡鐵路高架化與苑裡新政園區。現階段第四次通盤檢討都市計畫面積約455公頃。約佔苑裡土地積百分之6.9。

== 出身名人 == 
*[[郭芝苑|郭芝苑]](已故)：國寶級台灣音樂、台語音樂[[作曲家|作曲家]]。
*阮文池：男高音聲樂家。致力於推動台語藝術歌曲運動，尤其是郭芝苑的作品。理念上強調無論藝術或其他領域(作品)都應該是反映土地情感，以及生活的感受。
*阮雪芬: 台灣大學生命科學系教授, 中華民國十大傑出青年。致力於將生物資訊和系統生物學運用於生物醫學上的研究。
*[[陳筱涓|陳筱涓（陳曉娟）]]：台灣女[[藝人|藝人]]及[[詞曲|詞曲]][[創作|創作]]人，以《愛》和《葉子》二曲榮獲兩屆[[金曲獎|金曲獎]]「流行音樂作品類最佳作曲人獎」。
*[[柯以柔|柯以柔]]：台灣女藝人，演藝圈有「水果妹」之稱，原鄉是住在苑裡，後來搬移至[[汐止區|汐止]]。<ref>名人玩故鄉 柯以柔玩苗栗苑裡 中天電視 2007年12月3日播出</ref>
*[[岳虹|岳虹]]：台灣女藝人及演員。
*[[潘政琮|潘政琮]]：職業[[高爾夫球|高爾夫球]]運動員，2014年[[仁川亞運|仁川亞運]]金牌，2006年[[杜哈亞運|杜哈亞運]]銀牌得主。
*[[蔣清吉|蔣清吉]]：[[馬拉松|馬拉松]]選手，勇奪2008年[[渣打|渣打]]國際馬拉松東北亞區第一名、2008年金石馬拉松賽冠軍、2004年日月潭馬拉松冠軍、2003年阿猴[[奧林匹克|奧林匹克]]國際馬拉松冠軍等。
*[[陳炯輝|陳炯輝]](已故)：[[心雕居|心雕居]]工作坊，被譽為「台灣原始木雕第一人」，曾獲薪傳獎。
*[[張金芳|張金芳]](已故)：藝術傑出人員，擅於[[國畫|國畫]]、[[書法|書法]]、[[銅雕|銅雕]]......等，其中以畫松著稱，為[[蓬山美術會|蓬山美術會]]歲寒三友松之代表人物。
*[[林銘超|林銘超]]：藝術傑出人員，擅於國畫......等，其中以畫竹著稱，為蓬山美術會歲寒三友竹之代表人物。
*[[李久良|李久良]]：藝術傑出人員，擅於國畫、[[攝影|攝影]]、[[刻印|刻印]]、書法......等，其中以畫梅著稱，為蓬山美術會歲寒三友梅之代表人物。
*[[王春風|王春風]]：蓬山文教協會負責人，其中對地方上指導鄉民學子[[國樂|國樂]]樂器演奏為人著稱。
*[[盧彥賢|盧彥賢]]：[[好樂迪KTV|好樂迪KTV]]創辦者，其事業包括榮迪興(前身榮利興)。
*[[葉寅夫|葉寅夫]]：億光電子董事長，被称为[[發光二極體|發光二極體]]（LED）教父。
*[[賴孝義|賴孝義]]：曼都髮廊董事長，曾受聘於[[總統府|總統府]]理髮廳。
*[[呂東興|呂東興]]：藝術家，雕塑。
*[[梁至青|梁至青]]：藝術家，平面繪畫。
*[[黃瑞田|黃瑞田]]：文藝作家，兒童語文教育家。曾獲中國時報第五屆時報文學獎小說推薦獎、教育部小說獎、全國青年文藝創作最佳小說獎，著有《跳進畫裡》、《中國戲劇故事新寫》、《勇敢的黑羊》、《科學詮釋與幻想》、《小學生快快樂樂學作文》等書；在一九八O年，創辦國語日報高雄語文中心，掀起國內兒童才藝班的風潮，其中拼音寫作班、注音先修班、安親班都是他首創，堪稱國內兒童才藝班的前輩。
*[[鄭明典|鄭明典]]：[[中央氣象局|中央氣象局]]副局長。
*[[鄭麗君|鄭麗君]]：曾任[[臺灣|臺灣]][[行政院青年輔導委員會|行政院青年輔導委員會]]主任委員，[[立法委員|立法委員]]，現任[[文化部|文化部]]長。
*[[謝葦蓁|謝葦蓁]]：於特殊奧運會取得女子鉛球、一百公尺、四百公尺接力三面金牌。
*[[林建廷|林建廷]]：雕塑作品「失重」，獲得第27屆奇美藝術獎，作品還被奇美博物館典藏、第20屆桃城美展添生獎(雕塑類)首獎。
*[[謝文苑|謝文苑]]：台灣鎖神封號，受邀至阿拉伯教導當地警察教開鎖，在2014年在法國巴黎「雷平發明展」以及波蘭國際發明展，3項商品總共奪得4金1銅，為國爭光，並榮獲波蘭頒發人類貢獻獎之殊榮。
*[[楊尚融|楊尚融]]：台灣男[[歌手|歌手]]，流行樂團[[noovy|noovy]]之主唱，2016年9月23日在台灣出道、2017年9月27日在日本出道。
