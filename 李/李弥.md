'''李彌'''（{{bd|1902年11月4日||1973年|3月10日|catIdx=L李}}），字'''炳仁'''，號'''文卿'''，[[雲南|雲南]][[騰衝|騰衝]]人，[[中華民國陸軍|中華民國陸軍]]中將，為[[泰北孤軍|泰北孤軍]]最早的領導者。

== 早年 ==
李彌1902年生于[[云南省|云南省]]盈江县（祖籍腾冲县）。弟兄八人，李弥排行第六。小学毕业后，转去[[腾冲县|腾冲县]]升学，就读于清末举人王承漠的私馆。1924年，李弥到[[广州|广州]]，经人介绍在云南驻粤滇军第七师师部当差，任勤务兵。该师[[师长|师长]]李根云不久提拔他为[[副官|副官]]。不久李弥被保送上了[[黄埔军校|黄埔军校]]第四期。编入步科第一团第五连学习。1926年10月，[[黃埔軍校|黃埔軍校]]第四期畢業。分往驻[[南昌|南昌]]的[[朱培德|朱培德]]部任职。

== 第一次國共內戰 ==
1927年[[南昌暴动|南昌暴动]]前，[[朱德|朱德]]以军官教导团团长的身份，对李弥做过不少工作，但李弥拒不参加，由江西到[[上海|上海]]，后去[[南京|南京]]在南京总司令部警卫团任连长。这时，一些地方部队纷纷要求[[蒋介石|蒋介石]]派黄埔军校的毕业学生去。李弥被派往[[四川|四川]]，在国民革命军第二十二军[[赖心辉|赖心辉]]部先后任营的特派员，营长和副团长等职。1930年，李弥随部开赴鄂西，江西，听从[[陈诚|陈诚]]调谴，参与剿共。1936年，任[[江西|江西]][[瑞昌縣|瑞昌縣]]縣長。

== 中國抗日戰爭 ==
1939年，參加[[昆侖關戰役|崑崙關戰役]]。1940年，[[棗宜會戰|棗宜會戰]]後，調升為第八軍[[榮譽第一師|榮譽第一師]]師長。1942年，升为第八军副军长兼[[湖南|湖南]]芷(江)绥(宁)师管区司令。不久，该军开往[[云南|云南]]的[[河口|河口]]、[[马关|马关]]一带，以阻止日军经由[[越南|越南]]向[[云南省|云南]]的进犯。1943年底，在[[马关|马关]]的[[古木|古木]]，主持该军的干部训练班，曾在石壁上刻下「还我河山」四字。1944年，奔赴滇西[[滇西緬北戰役|松山戰場]]，加入[[中國遠征軍|中國遠征軍]]，打響[[松山戰役|松山戰役]]，因功升任第八軍軍長。

== 第二次國共內戰 ==
1947年，在[[雲南省|雲南省]][[蓮山設治局|蓮山設治局]]當選為[[第一屆國民大會代表|第一屆國民大會代表]]。1948年，所部改編為第十三兵團，任十三兵團司令官。[[淮海戰役|徐蚌會戰]]中，奉命支援[[黃百韜|黃百韜]]部未果。

1949年，第十三兵團在[[陳官莊|陳官莊]]全軍覆沒，李彌化裝逃到[[山東|山東]][[濰縣|濰縣]]、[[青島|青島]]、乘海輪赴[[南京|南京]]、[[上海|上海]]。2月7日，[[蔣介石|蔣介石]]約李彌餐敍。「李報告陳官莊突圍經過及其歸途情形」<ref name="寧靜">蔣經國：〈危急存亡之秋〉，刊《風雨中的寧靜》，台北：正中書局，1988年</ref>{{rp|152}}。李彌經蔣發表為第六編練司令部總司令，以曹天戈（浙江鎮海人，黃埔四期畢業，曾任[[王鐵漢|王鐵漢]]第四十九軍第二十六師師長、總統府參軍）、傅克精、邱開基為副司令官，兼第八軍軍長，副軍長為[[柳元麟|柳元麟]]（黃埔四期畢業，浙江慈谿人，曾任侍從室副主任）；未幾率部赴江西鷹潭，繼至湖南衡陽整訓<ref name="胡">{{cite book|author=胡士芳|chapter=〈「異域」孤軍的原型〉|title=《大對抗》|location=台北|publisher=風雲時代出版|year=1992}}</ref>。後往閩西徵兵。6月初[[雲南|雲南]]有不穩現象，7月[[徐永昌|徐永昌]]、[[蕭毅肅|蕭毅肅]]、[[王叔銘|王叔銘]]飛昆明<ref name="胡"/>。8月，蔣命令李彌返回雲南部署<ref name="年表"/>{{rp|61}}。8月27日，蔣派李彌回雲南<ref name="寧靜"/>{{rp|239}}。

== 退出雲南 ==
1949年12月9日，李彌和-{余}-程萬參加[[雲南省政府|雲南省政府]]主席[[盧漢|盧漢]]在省政府召開之軍政聯席會議<ref name="異域">{{cite book|author=[[柏楊|柏楊]]|title=《異域》|location=台北|publisher=躍昇文化事業有限公司|date=1988年11月}}</ref>{{rp|13}}。盧漢以「長官[[張群|張群]]由蓉蒞昆，定今晚九時召開會議」，請李彌、[[余程萬|余程萬]]、空軍第五區副司令沈延世、[[憲兵|憲兵]]副總司令李楚藩、憲兵司令部參謀長章鶴、第二十六軍第一九三師[[師長|師長]]石補天及[[軍統|軍統]]負責人[[沈醉|沈醉]]等到盧漢新公館出席<ref name="胡" />。盧漢宣佈[[卢汉云南起义|云南起义]]，[[雲南省_(中華民國)|云南省]]归降中共。

宣布起义后，盧漢把第八軍各師改編為解放軍第28、29、30等師。時第六編練司令部參謀長卓立（[[國防大學_(中華民國)|陸軍大學]]畢業）從昆明坐吉普車帶關防派給第八軍啟用，曹天戈出頭制止大家憤怒，便放卓立回昆明<ref name="胡" />。12月10日拂曉，盧漢派兩個保安團，進攻第八軍；第八軍教導師師長李楨幹（湖南寶慶人）以不足一團兵力在半天內追到昆明城郊，打死一團長<ref name="胡" />。陸軍副總司令湯堯、第六編練司令部副司令官曹天戈、第一七〇師師長孫進賢、第四十二師師長石建中、第二三七師師長李彬甫、第三師師長田仲達與李楨幹即舉行緊急會議，決定與第二十六軍副軍長彭佐熙（廣東羅定人，黃埔二期畢業）聯絡，向昆明進攻，活捉盧漢<ref name="胡" />。

12月12日，李彌答應「反正」，回到第八軍<ref name="異域" />{{rp|22-23}}。12月16日李彌被盧漢釋出，各師長均主張即攻昆明，活捉盧漢；李彌退居幕後，部隊由曹天戈指揮<ref name="胡" />。李彌馬上進攻昆明<ref>{{cite book|author=柏楊|title=《異域》|location=台北|publisher=躍昇文化事業有限公司|quote=就在李彌將軍脫險之後，政府明令發表他為雲南省政府主席和雲南綏靖公署主任，受他指揮的，還有二十六軍，共六萬餘人|year=1988}}</ref>{{rp|24}}。余程萬仍被扣押，第四十四師師長石建中所部進擊到昆明以北，包圍昆明，兩度猛攻後，盧漢於12月14日把余程萬送出昆明<ref name="異域" />{{rp|26}}。余程萬率領第二十六軍向雲南南部撤退<ref name="異域" />{{rp|27}}。12月19日拂曉，第八軍開始總攻昆明，12月20日即攻佔市郊[[伏馬山|伏馬山]]制高點<ref name="胡" />。盧漢-{于}-介興一軍包圍第八軍，第八軍在[[蒙自市|蒙自]]、建水、石屏一帶佈防，並重修蒙自機場，聯絡國府<ref name="異域" />{{rp|27}}。12月21日，任命李彌為雲南省政府委員兼主席<ref>{{cite news|title=《總統府公報》第貳肆伍號|editors=總統府第五局|date=1949-12-31}}</ref>。第二十六軍前鋒到達昆明城外，佔領[[昆明巫家壩國際機場|巫家壩機場]]，一部已入市區。

12月22日，第六編練司令部改為第八兵團，司令官為湯堯，副司令官為曹天戈，孫進賢升為第九軍軍長，田仲達升為第九軍副軍長，石建中升為第八軍副軍長<ref name="胡" />。12月29日，[[毛澤東|毛澤東]]給[[劉少奇|劉少奇]]電報，請其告知[[第二野戰軍|第二野戰軍]]司令員劉伯承、政治委員[[鄧小平|鄧小平]]，轉知盧漢及雲南解放軍，只可在李彌、-{余}-程萬之先頭阻止其向[[越南|越南]]、[[緬甸|緬甸]]前進，不可向其後尾威脅或追擊，以免其過早退入越南；盧漢及解放軍均應向其進行政治工作，策動起事<ref name="稿1">{{cite book|title=《建国以来毛泽东文稿》第一卷|location=北京|year=1987}}</ref>{{rp|198}}。

== 往來台灣及緬甸 ==
1950年1月14日，李彌和-{余}-程萬飛往[[台灣|台灣]]<ref name="異域"/>{{rp|28}}。國防部第三廳副廳長周芸繁及空軍第三軍區副司令易國瑞飛到蒙自<ref name="胡"/>。李彌參加3次最高軍事會議，報告最高長官願意留在雲南，建立基地<ref name="異域"/>{{rp|28}}。顧祝同宣佈將第二十六軍空運[[台灣|台灣]]，防務全交第八軍；顧祝同並帶700箱[[袁大頭|袁大頭]]，分發第八軍、第二十六軍做軍費，每箱內分裝兩布袋，每袋一千元<ref name="胡"/>。1月17日，李彌、-{余}-程萬和陸軍總司令[[顧祝同|顧祝同]]及[[張群|張群]]同機飛返第二十六軍防地雲南蒙自，在[[海南島|海南島]]途中，第二十六軍已有1個團撤到[[海口|海口]]，余程萬留下來整頓<ref name="異域"/>{{rp|29}}。當晚，李彌通宵召開軍事會議，第二天隨顧祝同和張群飛往[[西昌|西昌]]<ref name="異域"/>{{rp|29}}。孫進賢炸毀[[元江|元江]]鐵索橋，1月25日湯堯、曹天戈等在失散中被俘；時李彌隨顧祝同飛西昌，拜會[[胡宗南|胡宗南]]、[[賀國光|賀國光]]，共商大計後，又折回蒙自，蒙自失守轉飛台灣，以後李彌到[[香港|香港]]居住<ref name="胡"/>。李彌率部撤往緬甸、[[寮國|寮國]]、[[泰國|泰國]]交界地，任雲南省政府主席兼雲南綏靖公署主任，繼續於雲南[[江心坡|江心坡]]地區帶領[[雲南反共救國軍|雲南反共救國軍]]與[[中華人民共和國|中華人民共和國]]對抗<ref>[[胡士方|胡士方]]：〈我所知道的李彌〉，刊《[[傳記文學|傳記文學]]》總第336號與337號，1990年</ref>。雲南人丁中江與李彌、余程萬在[[虎豹別墅_(香港)|胡文虎花園]]商議，與駐泰國武官陳振熙聯絡；李彌毅然到[[曼谷|曼谷]]，通過泰國警察總監{{Tsl|en|Phao Sriyanond|乃炮}}之關係進入[[云南省|雲南]]邊境<ref name="胡" />。1951年，[[蔣中正|蔣中正]]批示，將留[[越南|越]]的[[富台部隊|富台部隊]]接回台灣，但留緬的李彌部隊不准接回。在[[韓戰|韓戰]]爆發之後，[[中華民國政府|中華民國政府]]將美製武器運往緬甸，支援李彌部隊。李彌因爭取「[[美援|美援]]」，中共及[[聯合國|聯合國]]向泰國、緬甸施壓干預，於是武力部隊回到台灣，留下少數由[[柳元麟|柳元麟]]指揮，後柳元麟回台灣<ref name="胡" />。1954年，李彌率部撤往台灣，其下的[[段希文|段希文]]等部隊接獲[[蔣中正|蔣中正]]密令，自願留下，成為[[泰北孤軍|泰北孤軍]]。

== 晚年 ==
李彌回台灣後，居[[台北|台北]][[新店區|新店鎮]]北新路，成為[[天主教|天主教]]徒<ref name="胡"/>。1963年11月在[[中國國民黨|中國國民黨]]第9次全國代表大會上被選為[[中國國民黨中央委員會|中央評議委員]]<ref>{{cite web|url=http://mirokosunnex.miroko.tw/document/bio-a/lime.html |title=陸軍上將李彌將軍墓表 |author=張維翰 |deadurl=yes |archiveurl=https://web.archive.org/web/20140414073252/http://mirokosunnex.miroko.tw/document/bio-a/lime.html |archivedate=2014-04-14 }}</ref><ref>{{cite web|url=http://www.kmt.org.tw/page.aspx?id=146 |title=【歷屆全代會】 |publisher=中國國民黨 |deadurl=yes |archiveurl=https://web.archive.org/web/20140806053554/http://www.kmt.org.tw/page.aspx?id=146 |archivedate=2014-08-06 }}</ref>。長子李雲川、幼子李獲之均學成居[[美國|美國]]<ref name="胡"/>，女兒李心娥嫁至國外。

1973年12月8日，李彌因[[心臟病|心臟病]]復發逝世，死後治喪委員會，由[[何應欽|何應欽]]、[[于斌|于斌]]、[[薛岳|薛岳]]、[[李宗黃|李宗黃]]、[[張寶樹|張寶樹]]主持<ref name="胡"/>。
