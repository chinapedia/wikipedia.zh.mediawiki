'''李仁罕'''（{{bd|873年||934年|catIdx=L}}<ref name=h9k>《[[九国志|九国志]]》</ref>），字'''德美'''，[[五代十国|五代十国]]时期[[后蜀|后蜀]]大将，在后蜀开国皇帝[[孟知祥|孟知祥]]为控制政权而进行的征伐中立有大功，惟在仕途晚期变得骄傲而且贪婪，使得孟知祥之子和继承人[[孟昶|孟昶]]与其他高官视他为孟昶统治的威胁，于是遭到捕杀。

== 家世 ==
李仁罕是[[陈留|陈留]]人。<ref name=SGCQ51>[https://archive.org/stream/06072244.cn#page/n22/mode/2up 《十国春秋》卷五十一]</ref>年轻时在[[宣武军|宣武军]]为小校，容貌瑰伟。[[朱全忠|朱全忠]]为四镇节度使，署李仁罕为衙内。[[唐昭宗|唐昭宗]]被[[宦官|宦官]]劫持到[[凤翔|凤翔]]时，朱全忠派李仁罕去[[行在|行在]]纳贡修礼。朱全忠代唐建立[[后梁|后梁]]后，补李仁罕为宫苑仪鸾等使，历任为[[许州|许州]]、[[蔡州|蔡州]]、[[华州|华州]]马步军都指挥使。<ref name=h9k/>

后梁灭亡后，李仁罕效力[[后唐|后唐]]。[[同光|同光]]三年（925年）后唐灭[[前蜀|前蜀]]一战，李仁罕参战，[[后唐庄宗|后唐庄宗]][[李存勖|李存勖]]命他监先锋军。次年正月，后唐灭蜀大军统帅皇子魏王[[李继岌|李继岌]]准备离开原前蜀都城[[成都|成都]]班师京城[[洛阳|洛阳]]，留下一支由时任马步都指挥使的李仁罕、马军都指挥使[[潘仁嗣|潘仁嗣]]、左厢都指挥使[[赵廷隐|赵廷隐]]、右厢都指挥使[[张业_(后蜀)|张知业]]（李仁罕外甥）、牙内指挥使[[武漳|武漳]]、骁锐指挥使[[李廷厚|李廷厚]]统领的军队留守成都，等候新任[[西川节度使|西川节度使]]孟知祥到任。显然后来孟知祥到任后，李仁罕等人归其统领。孟知祥奏授李仁罕马步军都指挥使。二月，蕃汉马步军都排陈斩斫使兼马步军都指挥使[[李绍琛|李绍琛]]作乱，孟知祥命李仁罕率四万人参与平乱<ref name=ZZTJ274>[[:s:資治通鑑/卷274|《资治通鉴》卷二百七十四]]</ref>、李廷厚率兵二千会合李仁罕，李仁罕会合[[工部尚书|工部尚书]][[任圜|任圜]]、[[东川节度使|东川节度使]][[董璋|董璋]]等击破康延孝，<ref name=NHFD64>[[:s:新五代史/卷64|《新五代史》卷六十四]]</ref>遂得勇战之名。<ref name=SGCQ51/>

== 效力孟知祥 ==
李仁罕积功授左厢马步都指挥使，进[[光禄大夫|光禄大夫]]、[[检校|检校]][[太保|太保]]、守[[彭州|彭州]]刺史、[[上柱国|上柱国]]，充诸军马步军都指挥使，<ref name=SGCQ51/>[[天成|天成]]三年（928年）正月官居[[金紫光禄大夫|金紫光禄大夫]]，<ref>[[:s:创筑羊马城记|李昊《创筑羊马城记》]]</ref>后迁[[蜀州|蜀州]]刺史。<ref name=h9k/>同光四年（926年）李存勖在洛阳的一起兵变中被杀，养兄[[李嗣源|李嗣源]]为帝，即[[后唐明宗|后唐明宗]]。孟知祥训练兵甲，秘密有在蜀地称王之志，增置义胜、定远、骁锐、义宁、飞棹等军七万余人，命李仁罕、赵廷隐、张知业等分别统领。<ref name=NHFD64/>[[长兴_(后唐)|长兴]]元年（930年）时，孟知祥和董璋两镇与朝廷的关系很紧张，必有一战。在此关键时刻，二月，李仁罕、张知业邀孟知祥赴宴。有尼姑告诉孟知祥，他们是要在宴会上害死他；孟知祥诘问之下，发现此事并无证据，最终查出谣言来源为军校都延昌、王行本，将此二人腰斩。随后孟知祥不带左右，单身直接去李仁罕府上赴宴，李仁罕感其信任，叩头哭道：“老兵只能尽死以报德。”诸将因此都亲附服从孟知祥。<ref name=ZZTJ277>[[:s:資治通鑑/卷277|《资治通鉴》卷二百七十七]]</ref>

九月，孟知祥、董璋正式起兵对抗朝廷。先前，[[武信军节度使|武信军节度使]][[夏鲁奇|夏鲁奇]]奉朝廷令，修兵甲以图蜀。孟知祥与董璋合计，先讨夏鲁奇。孟知祥以李仁罕为行营都部署，时任[[汉州|汉州]]刺史的赵廷隐为副，时任[[简州|简州]]刺史的张知业为先锋指挥使，率兵三万（《九国志·赵廷隐传》作步骑十万）会合董璋攻武信军军部[[遂州|遂州]]。十月，李仁罕即围遂州，夏鲁奇守城，派马军都指挥使[[康文通|康文通]]出战。康文通得知原为朝廷治下的[[保宁军|保宁军]]已被两川联军攻陷，就率众投降了李仁罕。孟知祥都押牙[[高敬柔|高敬柔]]率[[资州|资州]]义军二万人筑长城环围遂州。次年正月，李仁罕克遂州，夏鲁奇自杀。<ref name=NHFD64/><ref name=ZZTJ277/><ref>[[:s:旧五代史/卷136|《旧五代史》卷一百三十六]]</ref><ref>《十国春秋·高敬柔传》</ref>

孟知祥以李仁罕为武信军留后，二月又以为峡路行营招讨使，率水军东进从朝廷手中攻取[[三峡|三峡]]地区。当月即攻陷[[忠州|忠州]]，三月攻陷[[万州|万州]]、云安监，兵临[[宁江军|宁江军]]军部[[夔州|夔州]]。[[宁江军节度使|宁江军节度使]][[安崇阮|安崇阮]]弃镇内逃，李仁罕夺取宁江军。<ref name=NHFD64/>十一月，李仁罕引兵回成都。<ref name=ZZTJ277/>

当时李嗣源已撤回由其女婿东川行营都招讨使[[石敬瑭|石敬瑭]]率领的讨伐两川的主力大军，且已诛杀主倡此战的[[枢密使|枢密使]][[安重诲|安重诲]]，朝廷实际已经屈服。李嗣源寻求和两川和解，孟知祥也想和朝廷和解，但董璋不肯，三年（932年）五月反而攻打西川。因赵廷隐出大力，孟知祥反攻败董璋，董璋部将杀董璋，以东川投降孟知祥。明宗下诏褒奖李仁罕等，称可正授节度使。<ref>[[:s:赐孟知祥诏|:s:赐孟知祥诏]]</ref>李仁罕与赵廷隐在东川军部[[梓州|梓州]]相会，赵廷隐亲自欢迎李仁罕到来。但李仁罕不言赵廷隐的大功，反而侮辱他，赵廷隐大怒。孟知祥随后亲临梓州，召见李仁罕、赵廷隐，问谁可为东川节度使，以为他们中有一人会推让，这样即可任命另一人。但李仁罕却说：“如果令公再次让我当蜀州刺史，我也会接受。”而赵廷隐则沉默不语。因二人陷入僵局，孟知祥认为赵廷隐功劳多，怒李仁罕贪愎，最终决定亲自领东川。六月，任赵廷隐为[[保宁军节度使|保宁军节度使]]，八月令[[李昊|李昊]]为武泰留后[[赵季良|赵季良]]、李仁罕、赵廷隐、宁江留后张知业、昭武留后[[李肇|李肇]]等草表，请以自己为蜀王，行墨制，及为自己求旌节，但在李昊劝说下改为请求行墨制自行补授两川刺史以下官职及表请以赵季良等五留后为节度使。<ref name=SGCQ48>《十国春秋·前蜀高祖本纪》</ref>次年三月以墨制署李仁罕为检校[[太傅|太傅]]、武信军节度使，李嗣源也都照准。<ref name=SGCQ51/><ref name=ZZTJ277/><ref>[[:s:旧五代史/卷44|《旧五代史》卷四十四]]</ref><ref name=ZZTJ278>[[:s:資治通鑑/卷278|《资治通鉴》卷二百七十八]]</ref><ref>[[:s:许孟知祥奏赵季良等五人乞正授节旄诏|:s:许孟知祥奏赵季良等五人乞正授节旄诏]]</ref>

次年，孟知祥称帝，建立后蜀。<ref name=ZZTJ278/>二月，以李仁罕为卫圣诸军马步军指挥使，仍领武信军。当时后蜀众多高级将领骄纵，剽掠百姓，常违法夺取百姓的良田，甚至毁坏坟墓，李仁罕和[[张业_(后蜀)|张业]]（即张知业，因避孟知祥讳而改名）是最严重的。<ref name=NHFD64/>李仁罕因此而奢富，还曾想娶有国色的前蜀宫嫔[[花蕊夫人_(前蜀)|花蕊夫人]]，害怕被孟知祥所责，没敢实施。<ref name=SGCQ51/><ref>《十国春秋》仅载李仁罕想娶前蜀主宫嫔，而《九国志》则载张公铎、韩继勋、韩保贞、安思谦等诬陷李仁罕欲娶花蕊夫人。</ref>

== 身亡 ==
七月，孟知祥崩，遗命儿子[[孟仁赞|孟仁赞]]继位，改名孟昶，托孤于[[司空|司空]][[同平章事|同平章事]][[赵季良|赵季良]]、李仁罕、赵廷隐、枢密使[[王处回|王处回]]、捧圣控鹤都指挥使[[张公铎|张公铎]]、奉銮肃卫指挥副使[[侯弘实|侯弘实]]。<ref name=SGCQ48/>孟知祥驾崩时，王处回告知赵季良，赵季良让他回去见李仁罕，但李仁罕严整军备以待，于是王处回没有以孟知祥死讯相告，直到孟昶继位。<ref name=ZZTJ279>[[:s:資治通鑑/卷279|《资治通鉴》卷二百七十九]]</ref>

孟昶继位后，九月，李仁罕坚请判六军。孟昶起初不情愿地答应了，加仁罕兼[[中书令|中书令]]，判六军事，以赵廷隐为副。孟昶自置殿直四番，选将门之子及死王事者的孤儿充当，以李仁罕子李继宏、赵季良子赵元振、张业子张继昭、侯弘实子侯令钦、赵廷隐子[[赵崇韬|赵崇韬]]分为都知领之。委政事于赵季良、张业、李仁罕等。<ref>[[:s:宋史/卷479|《宋史》卷四百七十九]]</ref>李仁罕遇事必谏。<ref name=h9k/>但张公铎和医官使韩继勋、丰德库使韩保贞、茶酒库使安思谦等孟昶近臣随后弹劾李仁罕有异志。十月，孟昶与赵季良、赵廷隐合谋，趁李仁罕入朝，执而击杀之，下诏声言其罪，杀李继宏及李仁罕亲信进奏吏宋从会等数人，灭李仁罕族。奉銮肃卫都指挥使、[[昭武节度使|昭武节度使]]兼[[侍中|侍中]][[李肇|李肇]]本来见孟昶时拄着拐杖不下拜，闻讯震惊，改变态度，变得十分恭顺。<ref name=SGCQ51/><ref name=NHFD64/>孟昶左右请求也诛杀李肇，但孟昶虽也为李肇而心不平，却没有这么做，而是以李肇为[[太子少傅|太子少傅]][[致仕|致仕]]，徙[[邛州|邛州]]。<ref name=ZZTJ279/>安思谦因参与谋杀李仁罕，稍得进用。<ref>《十国春秋·安思谦传》</ref>

李仁罕被诛时，张业正掌禁兵，孟昶惧其反叛，于是用张业为相，兼判度支。[[广政|广政]]十一年（948年），孟昶与时任匡圣指挥使安思谦合谋擒杀张业。<ref name=NHFD64/>此后赵廷隐也致仕废于家。<ref name=SGCQ51/>

== 评价 ==
*《[[十国春秋|十国春秋]]》论曰：（李）仁罕伏辜，（张）业不良死，（李）肇老[[临邛|临邛]]，功高志满，召祸取尤，盖无不自己作之者也。<ref name=SGCQ51/>
