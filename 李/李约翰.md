'''李约翰'''（{{bd|1911年|3月9日|1988年|8月27日|catIdx=}}，{{lang-en|John Lie}}），又名'''雅雅·达尼尔·达尔玛'''（{{lang|en|Jahja Daniel Dharma}}），[[印度尼西亚|印度尼西亚]]华裔海军少将，首位华裔[[印度尼西亚民族英雄|印度尼西亚民族英雄]]，[[基督教徒|基督教徒]]<ref name=yinhuazuoxie>{{Cite web |url = http://www.yinhuazuoxie.com/jishi/johnliedaijunde.html |title = 印尼民族英雄——华裔海军少将李约翰 |author = 戴俊德 |date =  |publisher = [[印华写作者协会|印华写作者协会]] |language = 中文 |accessdate = 2016年1月29日 |quote = }}</ref>。

==生平==
1911年3月9日出生于[[万鸦老|万鸦老]]，原名'''李正泉'''（{{lang|nan|Lie Tjeng Tjoan}}），父亲李开泰（Lie Kae Tae）开办运输存储公司，母亲名为黄贞女娘（Oei Tjeng Nie Nio）。<ref name=yinhuazuoxie />

早年当船员，1945年，投身印尼独立运动并加入海军，多次在[[马六甲海峡|马六甲海峡]]攻破荷兰殖民者防线运送丹药。1957年，平定[[苏门答腊|苏门答腊]]、[[苏拉威西叛乱|苏拉威西叛乱]]，此后担任[[印度尼西亚海军陆战队|印度尼西亚海军陆战队]]第17部队和第25部队首长。1960年，升任海军司令、授海军少将军衔<ref>{{cite news|title=印尼华社纪念首位华人海军少将李约翰百年诞辰|url=http://www.chinanews.com/hr/2011/03-10/2896855.shtml|accessdate=2012-03-16|newspaper=中国新闻网}}</ref>。李约翰在1988年8月27日逝世，安葬在雅加达{{le|卡利巴达烈士公墓|Kalibata Heroes Cemetery}}，对于他的功勋和奉献，在1995年11月10日[[苏哈托|苏哈托]]总统授予一等伟大儿女星形勋章（Bintang Mahaputera Utama）。<ref name=yinhuazuoxie />
