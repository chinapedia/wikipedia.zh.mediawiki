'''李景林'''（{{bd|1885年|3月28日|1931年|11月13日|catIdx=Li李}}）<ref>生卒年月据《棗強县志》。此外，其卒年还有1931年12月3日説、1932年説。</ref>，字'''芳宸'''（又作'''芳岑'''），[[直隶省|直隶省]][[冀州市|冀州]][[棗強縣|棗強縣]][[恩察鎮|恩察鎮]]人，[[清朝|清朝]]及[[中华民國|中华民國]]军事將領，武術家。<ref name=sky>竇希彦「李景林」{{Cite book<!--|和書-->|author = [[中国社会科学院|中国社会科学院]]近代史研究所|title = 民国人物传 第10卷|year = 2000|publisher = [[中華書局|中華書局]]|isbn = 7-101-02114-X}}</ref><ref name=xyc>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=lsl>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

==生平==
=== 在奉系的兴起 ===
1907年（[[光緒|光緒]]33年），李景林毕业于[[保定|保定]][[北洋陸軍速成武備学堂|北洋陸軍速成武備学堂]]，此后他被任命为禁衛軍的下級軍官。1911年（[[宣統|宣統]]3年）10月[[武昌起義|武昌起義]]爆发，他随清軍同革命派作战。<ref name=sky/><ref name=xyc/>

[[中華民国|中華民国]]成立後的1912年（[[民国紀元|民国]]元年），他被[[鮑貴卿|鮑貴卿]]招为[[黑龙江省_(中華民国)|黑龙江]]巡防隊的軍官。1914年（民国3年），他升任黒龙江暫編陸軍第1師参謀長。1917年（民国6年）秋，他随第1師師長[[許蘭州|許蘭州]]加入[[奉系|奉系]]。1918年（民国7年），他任援陝奉軍司令部参議。<ref name=sky/><ref name=xyc/>


1919年（民国8年）1月，李景林任[[曲同豊|曲同豊]]所率師的第1旅第1团团長，参加[[直皖战争|直皖战争]]。当時，李景林所在的曲同豊师属于[[皖系|皖系]]，后来李景林因同皖系要人[[徐樹錚|徐樹錚]]对立，转而投靠[[直系|直系]]。李景林击败徐樹錚部有功。回到[[奉天省|奉天]]后，李景林练成了号称“疾行軍”的精锐部隊，后因战功而获得[[張作霖|張作霖]]重用。1921年（民国10年），李景林就任奉天陸軍第7混成旅旅長。<ref name=sky/><ref name=xyc/>

1922年（民国11年）4月的[[第一次直奉战争|第一次直奉战争]]中，李景林作为奉軍東路軍第3梯隊司令参战。奉系敗北后，李景林因軍功而升任奉天陸軍第1師師長。1924年（民国13年）9月[[第二次直奉战争|第二次直奉战争]]中，李景林被任命为奉軍第2軍軍長，[[張宗昌|張宗昌]]任该军副軍長。李景林率第2軍转战各地，击败了直系的軍队，攻下[[天津市|天津]]，为奉系的勝利作出貢献。直奉战争後的1925年（民国14年）1月，他被張作霖推薦，就任直隷軍務督办。6月兼任直隷省長。<ref name=lsl/><ref name=sky/><ref name=xyc/>

=== 叛服無常===
此後，李景林谋划从張作霖处自立。同年11月，奉系将领[[郭松齡|郭松齡]]拒绝攻击[[馮玉祥|馮玉祥]]的[[国民軍_(中華民国)|国民軍]]，并且和冯玉祥联络，企图叛离张作霖。李景林极度担心自己的军队被北方国民軍改編，馮玉祥和郭松龄夺取自己的地盘。結果，他受到張作霖的笼絡，转而讨伐馮玉祥和郭松齢。<ref name=sky/><ref name=xyc/>

12月，李景林同馮玉祥之間的战斗開始，馮玉祥的国民軍占据優勢。同月末，李景林放弃[[天津|天津]]，同[[山東省_(中華民国)|山東省]]的[[張宗昌|張宗昌]]结成[[直魯聯軍|直魯聯軍]]。1926年（民国15年）2月，李景林开始反击国民軍，但再被国民軍打击，逐渐喪失大部分军力。其后在日本軍队及[[張学良|張学良]]所率奉軍的支援下，3月下旬李景林成功地使国民軍从天津撤退。<ref name=sky/><ref name=xyc/>

天津之战后，李景林不满足久居张作霖之下。他和[[孫传芳|孫传芳]]、馮玉祥、[[靳雲鶚|靳雲鶚]]（直系）联络，准备策划叛变。由于事前发觉，6月奉軍張学良同[[褚玉璞|褚玉璞]]压制了李景林，李景林被迫下野。<ref name=sky/><ref name=xyc/>

同年9月，为了对抗[[中国国民党|中国国民党]]的[[北伐|北伐]]，李景林会见[[孫传芳|孫传芳]]，建议[[北洋政府|北洋政府]]方面的各派实行大联合，遭到孫的拒绝。1927年（民国16年）3月，北伐軍優勢毕見，李景林改投[[中国国民党|中国国民党]]方面，但在投奔途中于4月遭到[[褚玉璞|褚玉璞]]逮捕。经过日本方面调解，李景林获得释放，经日本到[[金陵|金陵]]，被[[蒋介石|蒋介石]]任命为直魯軍招撫使。1928年（民国17年）4月，李景林被任命为[[国民政府軍事委員会|国民政府軍事委員会]]委員。<ref name=sky/><ref name=xyc/>

===武术生涯===
[[File:Li_Jinglin2.jpg|thumb]]
李景林早年曾随[[陳世鈞|陳世鈞]]学习武术。<ref name=wlkz>[http://kungfu.chinese.cn/article/2010-04/09/content_122477.htm 李景林，网络孔子学院，2010年04月09日]</ref>1922年，任奉天陸軍第1師師長的李景林拜武当剑术名家[[宋唯一|宋唯一]]为师。1925年，宋唯一随李景林到天津。李景林遭到撤职后，宋唯一回到[[北镇|北镇]]，并于同年病逝。<ref>[http://www.bzs.gov.cn/bz/mr/2009-12-29/1966.html 宋唯一，辽宁省北镇市人民政府，2009-12-29]</ref>1925年，[[冯玉祥|冯玉祥]]占领天津，李景林避入租界，在此期间，李景林曾與八卦拳[[贾凤呜|贾凤呜]]、武术名家[[孙禄堂|孙禄堂]]等人相聚切磋剑术武技。直至次年3月复出。<ref name=hwk>[http://huawenku.egreenapple.com/html/baokandieshi/baokandieshi-20120305831.html 武林故事：武当剑仙李景林传奇，华文库，于2012-4-4查阅]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

1927年，李景林退去軍務，與[[張之江|張之江]]籌組南京國術研究館。该馆于1928年成立，李景林任副馆长。同年，「國術研究館」改名「[[中央國術館|中央國術館]]」，李景林仍任副館長。<ref name=hwk/>任職後，他邀请諸多名家研討劍術，还创编对剑一套，稱為“武当对剑”，约六十余势分為五趟，故今日有不同的武当剑套。<ref name=wlkz/>

1929年七月八日，位於杭州的[[浙江國術館|浙江國術館]]成立，浙江省主席[[張靜江|張靜江]]兼任館長，聘任李景林籌備「[[浙江國術遊藝大會|浙江國術遊藝大會]]」。同年11月，李景林任[[浙江國術遊藝大會|浙江國術遊藝大會]]打擂比賽的評判委員長。他还曾一度执教于浙江国术馆，[[黄元秀|黄元秀]]从其學得其劍術。<ref name=hwk/><ref name=wlkz/>

1930年底，在[[中原大战|中原大战]]爆发后，他奉国民政府之命，到山东济南策动反击。1931年（民国20年）1月，他受山東省政府主席[[韓復榘|韓復榘]]邀请，在[[济南市|济南]]逗留。他还出任[[山東國術館|山東國術館]]館長，邀請[[李書文|李書文]]任教習，他自己也在该馆任教。<ref name=hwk/><ref name=wlkz/><ref>吴志泉，八千里路云和月，武当2005年第6期</ref>

1931年11月13日，李景林在济南逝世。享年47岁（满46歳）。[[陳微明|陳微明]]为他作《祭李芳宸將軍文》。
