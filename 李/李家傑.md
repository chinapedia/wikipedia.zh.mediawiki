'''李家傑'''{{GBSHK}}{{JP}}（{{lang-en|Peter Lee}}；{{bd|1963年|5月20日|catIdx=L李}}），[[恆基地產|恆基地產]][[副董事長|董事局副主席]]、[[香港煤氣公司|香港煤氣公司]]董事<ref>[http://www.hld.com/chi/corp/director.htm 恆基兆業董事及高層管理人員]</ref>，《[[福布斯|福布斯]]》全球富豪榜第24位[[李兆基|李兆基]]<REF>截正2013年3月。</REF>的長子。現任第十三屆[[中國人民政治協商會議全國委員會常務委員會|全國政協常務委員]]<ref>[http://paper.wenweipo.com/2013/03/12/CN1303120030.htm 第十二屆全國政協常委名單]</ref>。

== 生平 ==
1963年5月20日生於[[香港|香港]]，籍貫[[廣東|廣東]][[順德區|順德]]。

1969年：入讀[[聖保羅男女中學附屬小學|聖保羅男女小學]]。

1977年：到英國求學，於[[基爾大學_(英國)|英國基爾大學]]電腦系畢業。

1985年：加入[[恒基兆業地產|恆基兆業地產集團]]，出任執行董事，主力負責內地業務，并长期致力于慈善公益事业。

2007年：創辦[http://www.ftdhz.org/page142 李家傑珍惜生命基金]。

2008年：創辦[http://www.charity-online.org/about/ 百仁基金會]。

2009年：獲[[香港大學|香港大學]]頒發名譽大學院士。
<ref>[http://the-sun.on.cc/cnt/news/20101027/00407_004.html 鑽石王老五信佛唔結婚]</ref>

2010年：創辦[[新家園協會|新家園協會]]。

== 事業 ==
1985年回港加入恆基兆業，現為恒基地產[[副董事長|董事局副主席]]、恒基中國[[董事長|董事長]]及[[行政總裁|行政總裁]]、中華煤氣董事。<ref>[http://the-sun.on.cc/cnt/news/20101027/00407_004.html 鑽石王老五信佛唔結婚]</ref>

在恆基兆業企業事業之外，李家傑投入大量時間、精力、財富在慈善事業上。李家傑的慈善工作覆蓋中港兩地，重點包括救危扶貧、協助及改善弱勢社群和低收入家庭的生活、推動防止自殺工作以及關注內地貧病兒童。于2008年以创会主席身份成立“[http://www.charity-online.org/about/ 百仁基金]” ，带动香港企业家的第二、第三代参与慈善工作，借着会员在工商界的强大网络，号召旗下的企业参与公益，撮成工商界与社福界之合作，项目涵盖四川地震赈灾、重建学校、为本港基层青年提供职前培训、探访及援助弱势社群等。

2007年創辦[http://www.ftdhz.org/page142 “李家傑”珍惜生命基金]，2009年加入“[http://www.ayfoundation.org/cn/about/22 爱佑慈善基金会]”，现任该会之副理事长。 

至2018年中，先後為超過2萬名貧困先天性心臟病兒童進行了手術。通過“李家傑珍惜生命基金”，針對那些經其救助而獲新生的孩子們展開農業扶持，幫助其家庭提高收入，為孩子們創造更好的成長環境。曾先後舉辦“[http://www.ftdhz.org/page121 北京夏令營]”、“[http://www.ftdhz.org/page121 成都夏令營]”、“[http://www.ftdhz.org/page121 珠海夏令營]”等系列活動，通過與孩子們見面，講學勵優，讓他們成長為棟樑之才。

2012年創立扶貧計畫-開始不牟利、連鎖式的“[http://www.poverty.org.hk/sites/default/files/Home_130904_0.pdf 家園便利店]”，向低收入家庭、長者、新來港人士和少數族裔等弱勢社群，以貼近來貨價售賣食品和日常用品，減輕其生活負擔，減少貧富懸殊。至今會員受惠人數超過數十萬人。

== 婚姻 ==
李家傑至今未婚，於47歲為達成父親李兆基的心願，成為“未婚爸爸”，於2010年在[[美國|美國]]找[[代孕|代母]]誕下三胞胎兒子<ref>[http://news.on.cc/ent/index.html?file=ncbrke00_20101026193212 李兆基長子李家傑美國請代母誕三名兒子]</ref>。此事轟動全港。3名兒子在2010年7月出生，取名智信、智仁、智勇<ref>[http://the-sun.on.cc/cnt/news/20101027/00407_003.html 李家傑三寶貝 智信 智仁 智勇]</ref>。

== 天匯托市案 ==
2010年，恆基旗下的[[西半山|西半山]]樓高34層之豪宅樓盤[[天匯|天匯]]其中20個單位的遭買家撻訂，恆基只對撻訂單位沒收5%訂金而沒有追收差價，被質疑營造高價承接的假象造市。警方[[商業罪案調查科|商業罪案調查科]]親赴恆地總部搜集證據，李家傑常代表恆地回應傳媒之質詢<ref>{{zh-hant}}{{cite web|url=http://881903.com/page/zh-tw/newsdetail.aspx?ItemId=257673&csid=261_341 |title=鄧竟成說警方檢走文件及邀請有關人士助查天匯事件 |publisher=[[商業電台|商業電台]]|date=2010-07-13 |accessdate=2010-07-13}}</ref><ref>{{zh-hant}}{{cite web|url=http://hk.apple.nextmedia.com/financeestate/art/20131112/18503907|title=中環在線：梁粉蔡涯棉　幫四叔打工|publisher=[[蘋果日報_(香港)|蘋果日報]]|date=2013年11月12日|accessdate=2014-03-05}}</ref><ref name="近年趨高調">{{zh-hant}}{{cite web|url=http://hk.apple.nextmedia.com/news/art/20140305/18646204|title=近年趨高調　任政協常委|publisher=[[蘋果日報_(香港)|蘋果日報]]|date=2014-03-05|accessdate=2014-03-05}}</ref>。

== 言論 ==
2013年，李家傑在[[北京|北京]]出席[[全國政協|全國政協]]會議時，指責香港部份青年只懂得「搞對抗」、不夠錢用即向政府「攞綜援」，揮舞[[港英旗|港英旗]]之青年更是「豈有此理」，又質疑[[讓愛與和平佔領中環|佔中]]行動及[[香港立法會議員|立法會議員]]之[[拉布|拉布]]舉動是「想攬住香港一齊死」，認為心態不好。<ref name="近年趨高調" /><ref>{{zh-hant}}{{cite web|url=http://life.mingpao.com/cfm/basicref3b.cfm?File=20130306/braa02d/gga1_er.txt|title=李家傑斥揮英旗豈有此理|publisher=[[商業電台|商業電台]]|date=2010-07-13|accessdate=2014-03-05}}</ref>
