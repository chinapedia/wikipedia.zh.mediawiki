'''李继韬'''（{{bd|9世纪||924年|1月20日|catIdx=L}}<ref name=ZZTJ272>[[:s:資治通鑑/卷272|《资治通鉴》卷二百七十二]]</ref>），小字'''留得'''，[[五代十国|五代十国]]时期[[后唐|晋国]]、[[后梁|后梁]]和晋国的后身[[后唐|后唐]]将领。父[[李嗣昭|李嗣昭]]为晋王[[李存勖|李存勖]]（后来的[[后唐庄宗|后唐庄宗]]）的养堂兄，也是晋国一员受尊敬的大将，但李嗣昭死后，李继韬接管了他的辖区，转投晋国的主要竞争对手后梁。后唐灭后梁后，庄宗起初想赦免李继韬，但后来发现他仍在图谋反对朝廷统治，于是将其诛杀。

== 家世 ==
李继韬生年不详，仅知其弟李继忠生于896年。父李嗣昭是晚唐大军阀河东节度使[[李克用|李克用]]的养侄，母杨氏是李嗣昭的妻子，很善于理财，故李嗣昭家变得极富。据《李继忠墓志》，李嗣昭有八子，李继韬有兄长[[李继俦|李继俦]]及弟[[李继忠|李继忠]]、[[李继达|李继达]]、[[李继能|李继能]]、[[李继袭|李继袭]]、[[李继远|李继远]]、[[李继镕|李继镕]]，其中李继达、李继镕生母不详，其余六人都是杨氏所生。<ref name=ZZTJ271>[[:s:資治通鑑/卷271|《资治通鉴》卷二百七十一]]</ref>杨夫人储存的大量钱财对李嗣昭行军有帮助，如[[天祐_(唐朝)|天祐]]四年（907年）—五年（908年），时任晋王李克用（李克用死后由其子继承人李存勖袭爵）麾下昭义节度使的李嗣昭被李克用的主要竞争对手[[后梁太祖|后梁太祖]][[朱全忠|朱全忠]]围困于昭义军部[[潞州|潞州]]，杨夫人的财物使他得以在被围的情况下供养军队。<ref name=NHFD36>[[:s:新五代史/卷36|《新五代史》卷三十六]]</ref>李继韬少年时就狡狯无赖。<ref name=HFD52>[[:s:舊五代史/卷52|《旧五代史》卷五十二]]</ref>

== 控制安义（匡义） ==
十九年（922年），在李存勖命令下，李嗣昭率晋军对抗[[张处瑾|张处瑾]]。张处瑾的父亲[[张文礼|张文礼]]先前领导针对李存勖盟友赵王[[王镕|王镕]]的兵变，杀死王镕，接管赵地，死后将地盘留给张处瑾。四月，李嗣昭在与赵国叛军的一场战斗中丧生，李存勖命李嗣昭诸子护其丧车回晋都[[太原|太原]]安葬。李继能却无视李存勖命令，聚集其父的数千昭义牙兵，拥丧车归潞州。李存勖派弟弟[[李存渥|李存渥]]追赶他们，重申李存勖之命。李继能及其兄弟不但不遵李存勖之命，还想杀李存渥，李存渥逃回晋都。<ref name=ZZTJ271/><ref name=HFD52/>

当时，李继韬兄长李继俦任[[泽州|泽州]]刺史，应在李嗣昭死后继承军镇，也是李存勖中意的继承人。但李继俦为人懦弱，在苫庐守丧时，李继韬趁机[[软禁|软禁]]他，上表李存勖，称被士兵逼迫，只好自为[[留后|留后]]。李存勖面临赵国叛军、后梁和[[辽国|契丹帝国]]多线作战，只得退让，改昭义为安义以避李嗣昭讳，以李继韬为留后。<ref name=ZZTJ271/><ref name=HFD52/><ref name=NHFD36/><ref name=HFD10>[[:s:旧五代史/卷10|《旧五代史》卷十]]</ref>

李继韬虽然被任为留后，却不满现状。晋军军前粮饷不足，租庸计请[[潞州|潞州]]转米五万贮于[[相州|相州]]，李继韬称经费不足，请求只转三万。被李继韬委以政务的幕僚[[魏琢|魏琢]]、牙将[[申蒙|申蒙]]也试图说服他晋国无人，终将被后梁所并。李继韬起初未决。

二十年（923年）三月，李存勖正意图作为唐朝正统继承人称帝，召昭义[[监军|监军]][[宦官|宦官]][[张居翰|张居翰]]和节度[[判官|判官]][[任圜|任圜]]去他所在的[[魏州|魏州]]，意在给他们在新朝廷之中安排职务，但李继韬却对李存勖的企图愈发担忧。魏琢、申蒙说服李继韬，这是李存勖要向张居翰等询问李继韬的事，意在诛杀他，李继韬的弟弟李继远也附和，称有家财百万、仓储十年，应该自己打算，不要受制于人。李继韬派李继远去后梁都城[[大梁|大梁]]，以安义降梁。<ref>[[:s:新五代史/卷3|《新五代史》卷三]]</ref><ref name=NHFD5>[[:s:新五代史/卷5|《新五代史》卷五]]</ref>后梁皇帝[[朱瑱|朱瑱]]大喜，改安义为匡义，以李继韬为节度使，授[[宰相|宰相]]荣衔[[同中书门下平章事|同中书门下平章事]]。李继韬也派年幼的二爱子去大梁为质。<ref name=NHFD36/>同时，他害怕李存勖来攻，散财招募士兵加入他的军队，[[尧山|尧山]]人[[郭威|郭威]]也在其中，好斗有力气，又好酒，李继韬奇之，郭威多次犯法犯禁，李继韬也都纵容。郭威因杀人而下狱，李继韬惜他有才有勇，放走了他，<ref name=HFD110>[[:s:旧五代史/卷110|《旧五代史》卷一百一十]]</ref>后来又召到麾下。<ref name=NHFD11>[[:s:新五代史/卷11|《新五代史》卷一十一]]</ref>

他麾下的泽州刺史[[裴约|裴约]]为李嗣昭生前亲信，拒绝随李继韬归梁，召集本州人说：“我事故节度使，已二纪（二十四年）有余，每见他分财给士兵，志在平定仇人，不幸薨殁。如今郎君父丧未葬，就背弃君亲，我可以在這裏[[自刃|自刃]]而死，不能随之归梁。”众人都感泣。朱瑱派骁将[[董璋|董璋]]攻之。<ref name=HFD52/><ref name=HFD10/><ref name=HFD29>[[:s:舊五代史/卷29|《旧五代史》卷二十九]]</ref><ref name=HFD62>[[:s:舊五代史/卷62|《旧五代史》卷六十二]]</ref><ref name=NHFD32>[[:s:新五代史/卷32|《新五代史》卷三十二]]</ref><ref name=NHFD51>[[:s:新五代史/卷51|《新五代史》卷五十一]]</ref>这时契丹犯燕、蓟之地，[[枢密使|枢密使]][[郭崇韬|郭崇韬]]奏称：“汴寇（即后梁）未平，继韬背叛，北边捍御，非[[李存审|李存审]]不可。”于是在病中的时任内外蕃汉马步总管李存审被充[[卢龙节度使|卢龙节度使]]。<ref name=ZZTJ272/><ref>[[:s:舊五代史/卷56|《旧五代史》卷五十六]]</ref>

李继韬倒戈正值梁、唐在黄河相持，庄宗有忧色，闰四月，召养兄[[横海军节度使|横海军节度使]][[李嗣源|李嗣源]]于帐中，说：“继韬以上党降梁，而梁正急攻泽州，我出不意袭[[郓州|郓州]]，以断梁右臂，可以吗？”李嗣源应命，袭破郓州。<ref name=ZZTJ272/><ref>[[:s:新五代史/卷6|《新五代史》卷六]]</ref>

裴约拒战很久，告急于已建立后唐皇朝称帝即庄宗皇帝的李存勖。庄宗仍在与梁军相持，得知昭义军只有裴约不背叛，知道他是忠臣，很高兴，对诸将说：“朕对继韬薄吗，对裴约厚吗？裴约能分逆顺，不附贼党，先兄何等不幸，生此鸱枭！”虽然派兵相救，但救兵未到，董璋已于八月攻破泽州，<ref name=HFD62/><ref name=NHFD51/>杀裴约。<ref name=ZZTJ272/><ref name=HFD29/><ref name=NHFD32/>

唐失去德胜城后，梁军每日掠[[澶州|澶州]]、相州，取[[黎阳|黎阳]]、[[卫州|卫州]]，契丹数次进犯[[幽州|幽州]]、[[涿州|涿州]]，李继韬的背叛使唐局势依旧严峻，唐诸将都忧惑以为成败未可知。<ref>[[:s:新五代史/卷24|《新五代史》卷二十四]]</ref>

== 身亡 ==
同年十月，庄宗在一次奇袭中攻陷大梁，城陷之际，朱瑱自杀，后梁灭亡。后梁领地为后唐所控制。<ref name=ZZTJ272/>庄宗入大梁，抓到李继韬二子，对他们说：“你们如此年幼，已经能帮你们的父亲谋反了。你们长大了能做什么呢？”<ref name=NHFD36/><ref name=HFD52/>

十二月，得知大梁陷落，李继韬震恐，不知所措。他想放弃职位，逃到契丹。不久，他收到庄宗召他的命令。李继远称他应该至少守住城池，去面见庄宗意味着马上就要死；但其他人称李继韬将因李嗣昭的大功而蒙赦，且庄宗论关系也是李继韬的堂叔，又有母亲保全。李继韬不顾李继远反对，决意遵旨。杨夫人收集财物，带了四十万两银子，和李继韬一同去被庄宗定为都城的[[洛阳|洛阳]]，一到达，她就贿赂庄宗最宠爱的宦官和伶人，他们因而为李继韬说情，称他为奸人所惑，应看在李嗣昭面子上予以赦免，不可令李嗣昭无后。杨夫人还见了庄宗及其宠妾[[刘玉娘|刘夫人]]，哭泣请死，乞求饶李继韬一命，并陈说李嗣昭功勋。庄宗被感动，因而下诏赦免李继韬，将他留在宫中，予以善待，带他一同打猎，<ref name=NHFD36/>达一月有余。<ref name=ZZTJ272/><ref name=HFD52/>事实上，这时他还被认为合法的安义（名字已改回）节度使，<ref name=HFD30>[[:s:舊五代史/卷30|《旧五代史》卷三十]]</ref>并加检校[[太保|太保]]、同中书门下平章事。

但李存渥仍为李继韬兄弟谋杀他一事所怒，反复指责李继韬，李继韬因而害怕。他贿赂庄宗左右，让他们请求将他遣回安义。但庄宗不同意。李继韬因而秘密送信给李继远，要他让士兵放火以示不满，希望一旦此事发生，庄宗就将遣他回镇安抚士兵。但此信被拦截下来。庄宗发现后，贬李继韬为[[登州|登州]]长史，随后以其有再度谋叛之罪下令将他斩杀于于天津桥南，<ref name=NHFD5/>其二子一并处决。庄宗又遣使去[[上党|上党]]斩李继远。<ref name=ZZTJ272/><ref name=HFD52/><ref name=NHFD36/><ref name=HFD30/>李继韬麾下牙兵被分配隶属从马直，其中就有郭威，<ref name=HFD110/>得补为军吏。<ref name=NHFD11/>

李继韬死后，家室及婢女、仆人、珍玩等为奉命权知潞州的李继俦所据，李继俦纳妓妾，搜刮钱财享乐，被召还京师也不立即出发。李继达见兄弟、侄子四人被杀，而兄长却无骨肉之情如此贪淫，又率麾下攻杀李继俦，<ref name=NHFD5/>随即也败死。此时距李继韬伏诛仅六天。<ref name=ZZTJ272/><ref name=HFD52/><ref name=NHFD36/><ref name=HFD30/>

李继韬伏诛后，其生前所宠牙将[[杨立|杨立]]常想作乱，最终于[[同光|同光]]二年（924年）四月发动兵变，五月被平定。<ref>[[:s:舊五代史/卷74|《旧五代史》卷七十四]]</ref><ref>[[:s:舊五代史/卷94|《旧五代史》卷九十四]]</ref><ref>[[:s:新五代史/卷47|《新五代史》卷四十七]]误作李继韬之叛。</ref><ref>[[:s:資治通鑑/卷273|《资治通鉴》卷二百七十三]]</ref>
