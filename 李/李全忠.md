'''李全忠'''（{{bd|9世纪||886年|catIdx=L}}），[[中国|中国]][[唐朝|唐朝]]末年[[军阀|军阀]]，885年推翻前任[[卢龙节度使|卢龙节度使]][[李可举|李可举]]后统治[[卢龙军|卢龙军]]直至886年去世。儿子[[李匡威|李匡威]]、[[李匡筹|李匡筹]]在他死后相继为节度使。

== 背景 ==
李全忠生年不详，卢龙军部[[范阳|范阳]]人。父李慕勳，守[[平州|平州]][[刺史|刺史]]，累迁检校左散骑常侍。<ref name=pw>[[庞纬|庞纬]]《唐故西都右厢马步使、银青光禄大夫、检校司空、守左武卫将军同正、兼御史大夫、柱国陇西李公墓志铭》</ref>[[唐僖宗|唐僖宗]][[广明|广明]]年间（880年—881年），李全忠为[[棣州|棣州]]司马。他家中长出一尺三节的芦，他心里很讨厌它，对别驾张建说：“我家生芦，不是怪事吗？”张建说：“芦是茅类，得泽而滋，公家有茅土之庆，这不是天意吗！它生有三节，必意味着公家有三个传节钺者。公要勉树功名，别忘了我的话。”李全忠任满（三年）回乡，事节度使李可举为牙将。<ref name=BT180>[[:s:舊唐書/卷180|《旧唐书》卷一百八十]]</ref><ref name=NBT212>[[:s:新唐書/卷212|《新唐书》卷二百一十二]]</ref>

== 夺取卢龙 ==
[[光启|光启]]元年（885年）二月，李可举和盟友[[成德军节度使|成德军节度使]][[王镕|王镕]]害怕[[河东节度使|河东节度使]][[李克用|李克用]]有意于他们的领地，为[[义武军节度使|义武军节度使]][[王处存|王处存]]与李克用结盟而担忧，于是两镇相约合攻夹在卢龙与成德之间的义武，灭之，分其地。<ref name=ZZTJ256>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref>

为达目的，李可举派李全忠率军六万攻[[易州|易州]]。五月，李全忠裨将[[刘仁恭|刘仁恭]]挖地道进入易州城墙内，卢龙军因而克易州。<ref name=HFD135>[[:s:旧五代史/卷135|《旧五代史》卷一百三十五]]</ref>但卢龙军之后就变得骄傲懈怠，不备义武军反攻。王处存趁夜派三千士兵蒙羊皮到易州城下，在其他道路埋伏精骑。卢龙军以为是羊，争着出城掠之，被义武军伏兵突袭，义武军攻易州，复夺之。李全忠大败，刍粮仗铠尽失，逃往[[幽州|幽州]]（即范阳）。<ref name=NBT212/><ref name=ZZTJ256/><ref name=BT19-2>[[:s:舊唐書/卷019下|《旧唐书》卷一十九下]]</ref>

因害怕李可举会因他战败而惩罚他，李全忠反而决定率余众掩袭幽州。李可举猝不及防，六月，绝望举族登楼自焚。李全忠接管军镇，在三军推戴下自称[[留后|留后]]，<ref name=BT19-2/><ref name=NBT9>[[:s:新唐書/卷009|《新唐书》卷九]]</ref>七月得到朝廷认可。<ref name=ZZTJ256/>

== 短暂统治 ==
[[成德军节度使|成德军节度使]][[王镕|王镕]]曾上表推荐李全忠。<ref>[[:s:荐幽州权兵马留後李全忠表|:s:荐幽州权兵马留後李全忠表]]</ref>次年（886年）春，僖宗正式任李全忠为卢龙节度使。<ref name=BT180/>后又加检校[[太尉|太尉]]。<ref name=pw/>八月，李全忠卒，三军立其子李匡威为留后，<ref name=BT19-2/>朝廷认可，后以为全权节度使。<ref name=BT180/><ref name=NBT212/><ref name=ZZTJ256/>
