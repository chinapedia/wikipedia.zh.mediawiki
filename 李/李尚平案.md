'''李尚平案'''是[[湖南省|湖南省]][[益阳市|益阳市]][[龙光桥镇|龙光桥镇]]南塘中学的教师李尚平被枪击致死的案件，此案至今未侦破。

2002年4月26日下午5点半，李尚平在距家门外300米处被枪击身亡。起初益阳市[[赫山区|赫山]]交警大队认为其死于“交通事故”，经过益阳市公安局法医尸检后认定李尚平死于“枪击”。随后警方在草丛中找到凶器：一把自制火枪，法医鉴定认为凶手在2米内开枪射击，可能是职业杀手。<ref>{{cite news|author=曹勇|title=湖南一教师因敢于直言被称“刺头” 惨遭谋杀|url=http://news.sina.com.cn/c/2002-07-18/1746641252.html|accessdate=2013-10-19|newspaper=南方周末|date=2002-07-18}}</ref>

== “刺头”老师 ==
李尚平以“敢于直言”在当地教师中闻名，被个别部门称作“刺头”。2001年12月下旬，当地教师工资被停发，李尚平开始向上级部门交涉。2002年1月15日，李尚平在新闻网站披露了此事，并向[[新华网|新华网]]写信反映当地教师上调的工资两年多未发放、联校领导克扣教师误餐费等现象。3月18日上午，[[湖南电视台|湖南电视台]]记者到当地采访，两天后在电视中播出。3月27日，当地政府在多方“摊派”筹集后发放了42万元工资。

== 后续 ==
案件发生后，当地工作人员告诉记者，因案件“牵涉”了很多人，公安局不好下手<ref>{{cite news|author=曹勇|title=李尚平：被枪杀以后|url=http://www.southcn.com/weekend/tempdir/200212260033.htm|accessdate=2013-10-19|newspaper=南方周末|date=2002-12-26|deadurl=yes|archiveurl=https://web.archive.org/web/20131020012940/http://www.southcn.com/weekend/tempdir/200212260033.htm|archivedate=2013-10-20}}</ref> 。李尚平死后，龙光桥镇教师的工资被克扣了140元。

2003年4月19日，中国[[中央电视台|中央电视台]]以《“刺头”教师李尚平之死》为专题报道了李尚平案。<ref>{{cite news|title=湖南一年轻教师 遭官员买凶枪杀|url=http://www.peacehall.com/news/gb/yuanqing/2003/04/200304252253.shtml|accessdate=2013-10-19|newspaper=博讯新闻网|date=2003-04-25}}</ref> <ref>{{cite news|title=谁是杀害“剌头”教师的凶手|url=http://www.unn.com.cn/GB/channel27/156/634/200304/25/257053.html|accessdate=2013-10-21|newspaper=新华网湖南频道|date=2003-04-25}}</ref> 

2011年1月19日，时任中国国家主席胡锦涛访问美国期间，人权团体在白宫前打出条幅要求当局关注李尚平案。<ref>{{cite news|title=沉冤九年 白宫前飘起声援李尚平的横幅|url=http://m.kanzhongguo.com/node/388602|accessdate=2013-10-19|newspaper=看中国|date=2011-01-19}}</ref>
