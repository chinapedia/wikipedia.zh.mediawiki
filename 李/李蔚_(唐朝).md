'''李蔚'''（{{bd|9世纪||879年|11月20日|catIdx=L}}<ref name=ZZTJ253>[[:s:資治通鑑/卷253|《资治通鉴》卷二百五十三]]</ref>），字'''茂休'''，[[中国|中国]][[唐朝|唐朝]]官员，[[唐僖宗|唐僖宗]]年间为[[宰相|宰相]]。

== 家世和早期仕途 ==
李蔚出自[[陇西李氏|陇西李氏]]姑臧房，李蔚祖父李上公官至司农卿，[[元和_(唐朝)|元和]]初年为[[陕虢观察使|陕虢观察使]]。父李景素官至太子庶子。<ref name=NBT72>《新唐书》卷七十二{{cite web |url=http://www.sidneyluo.net/a/a17/072.htm |title=存档副本 |accessdate=2008-10-03 |0=2008-11-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20081120085821/http://www.sidneyluo.net/a/a17/072.htm |archivedate=2008-11-20 }}{{cite web |url=http://www.sidneyluo.net/a/a17/table/form48.htm |title=存档副本 |accessdate=2009-03-29 |0=2010-06-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100620180318/http://sidneyluo.net/a/a17/table/form48.htm |archivedate=2010-06-20 }}</ref>

[[唐文宗|唐文宗]][[开成|开成]]五年（840年）为礼部侍郎[[李景让|李景让]]所选，中[[进士|进士]]，<ref>[[:s:旧唐书/卷187下|《旧唐书》卷一百八十七下]]</ref><ref>[[:s:新唐书/卷177|《新唐书》卷一百七十七]]</ref>被任为[[山南东道节度使|山南东道节度使]]从事。[[唐武宗|唐武宗]][[会昌_(唐朝)|会昌]]（840年－846年）末年，因通过书判拔萃考试，拜[[监察御史|监察御史]]，转殿中监。[[唐宣宗|唐宣宗]][[大中_(唐朝)|大中]]年间曾在将仕郎守殿中侍御史任上被任为侍御史，<ref>[[:s:李蔚除侍御史卢潘除殿中侍御史等制|杜牧《李蔚除侍御史卢潘除殿中侍御史等制》]]</ref>七年（853年），以员外郎知[[御史台|御史台]]杂事，不久知制诰，转郎中，正拜中书舍人。<ref name=BT178>[[:s:舊唐書/卷178|《旧唐书》卷一百七十八]]</ref>

== 唐懿宗年间 ==
[[唐懿宗|唐懿宗]][[咸通|咸通]]五年（864年）十月，权知礼部贡举。六年（865年），拜礼部侍郎，累次升官转尚书右丞。不久拜[[京兆尹|京兆尹]]、太常卿。<ref>《旧唐书》本传有此时以本官同平章事，加中书侍郎，与卢携、郑畋同辅政，及罢相出为[[襄州|襄州]]刺史、[[山南东道节度使|山南东道节度使]]事。但卢携、郑畋拜相在乾符年间，不取。</ref>七年（866年）尚在京兆尹任。<ref>《唐阙史》卷下《御楼前一日雨》</ref>八年（867年）十月，懿宗以中书舍人[[刘允章|刘允章]]权知礼部贡举，以时任吏部侍郎李蔚、[[卢匡|卢匡]]、兵部员外郎[[薛崇|薛崇]]、司勋员外郎[[崔殷梦|崔殷梦]]考吏部宏词选人。唐懿宗常在宫中设大宴招待大群[[比丘|僧人]]，以旃檀为二高座，赐安国寺给僧人，每逢八日就给一万僧人赐饭。李蔚认为他过度热衷[[佛教|佛教]]，上了一篇长疏，引[[狄仁杰|狄仁杰]]、[[姚元崇|姚元崇]]、[[辛替否|辛替否]]所言，请求懿宗改变方针。<ref>[[:s:谏禁中饭僧疏|:s:谏禁中饭僧疏]]</ref><ref>《[[册府元龟|册府元龟]]》误作咸通十二年事，且作李蔚官职为翰林学士承旨、礼部侍郎。</ref>懿宗下诏嘉奖他，但没有遵从。九年（868年）正月，出为检校刑部尚书、[[汴州|汴州]][[刺史|刺史]]、[[御史大夫|御史大夫]]，充[[宣武节度使|宣武节度]]、汴宋亳观察处置等使。曾兼[[亳州|亳州]]太清宫使，十年（869年）九月，传变军首领[[庞勋|庞勋]]及其余党因太上老君显灵而覆灭，李蔚上奏此事，诏李蔚谢老君并公告天下。<ref>[[:s:答亳州太清宫使李蔚表进老君灵应诏|:s:答亳州太清宫使李蔚表进老君灵应诏]]</ref><ref>[[:s:历代崇道记|杜光庭《历代崇道记》]]</ref>任内潜心学道，修建道观。<ref>《[[云笈七签|云笈七签]]》卷一百一十七《李蔚相国应梦天尊修观验》</ref>十一年（870年）十一月，迁检校吏部尚书、扬州大都督府长史，兼[[淮南节度使|淮南节度副大使]]、知节度事。<ref>[[:s:旧唐书/卷19上|《旧唐书》卷一十九上]]</ref><ref>《旧唐书》本传误作十四年事。</ref>十二年（871年），在军部[[扬州|广陵]]建大旆。<ref>《唐阙史》卷下《泗州风狂尼》</ref>任期满受代时，百姓去宫门请求让他留任，于是懿宗同意让他多留任一年。<ref name=BT178/><ref name=NBT181>[[:s:新唐書/卷181|《新唐书》卷一百八十一]]</ref><ref>《旧唐书》误作乾符三年事。</ref>辟殿中侍御史[[柳玭|柳玭]]为掌书记。<ref>[[:s:旧唐书/卷165|《旧唐书》卷一百六十五]]作李蔚当时镇[[襄阳|襄阳]]，误。当时李蔚尚在淮南任上。</ref>任上，他还建造了水池和亭子，命名亭子为“赏心亭”。<ref>《[[桂苑丛谈|桂苑丛谈]]》</ref>

== 唐僖宗年间 ==
后李蔚被召回[[长安|长安]]。[[乾符|乾符]]元年（874年）四月，为[[吏部尚书|吏部尚书]]。<ref>《旧唐书》本传作四年（877年）事。</ref>二年（875年）六月，在御史大夫、吏部尚书任上被拜为[[中书侍郎|中书侍郎]]，授[[同中书门下平章事|同中书门下平章事]]，为[[实质宰相|实质宰相]]，<ref name=NBT181/><ref name=NBT9>[[:s:新唐书/卷9|《新唐书》卷九]]</ref><ref>[[:s:資治通鑑/卷252|《资治通鉴》卷二百五十二]]</ref><ref>《旧唐书·僖宗纪》作三月为太常卿，六月以太常卿拜相，误。</ref>与[[崔彦昭|崔彦昭]]、郑畋同列。<ref name=BT178/>九月，由[[银青光禄大夫|银青光禄大夫]]进[[金紫光禄大夫|金紫光禄大夫]]。<ref name=BT19-2>[[:s:旧唐书/卷19下|《旧唐书》卷一十九下]]</ref>

李蔚在宰相任内的作为几无记载，仅知赏识有诗名的[[罗隐|罗隐]]。<ref>[[:s:旧五代史/卷24|《旧五代史》卷二十四]]</ref>乾符五年（878年）九月以[[门下侍郎|门下侍郎]]、吏部尚书罢为检校尚书左仆射、检校[[司空|司空]]、充东都留守、东畿汝都防御使。<ref name=NBT9/>乾符六年（879年），[[河东节度使|河东节度使]][[崔季康|崔季康]]在兵变中被杀，继任者[[李侃|李侃]]不为士兵所喜，八月，朝廷因李蔚曾任[[太原|太原]]从事，在太原府有惠政，任时任[[特进|特进]]、检校司空、东都留守的李蔚为检校[[司徒|司徒]]、同平章事，兼[[太原|太原]]尹、北都留守、充河东节度观察，兼代北行营招讨供军等使。<ref name=NBT9/>闰十月，李蔚到任，但次日就病倒，两天后即暴斃而死。<ref name=ZZTJ253/><ref name=BT178/><ref name=NBT181/><ref name=BT19-2/>如前任[[窦澣|窦澣]]、李侃一样，他虽以重臣镇守太原，也未能遏制当时生乱的[[沙陀|沙陀]]势力。<ref>[[:s:旧唐书/卷158|《旧唐书》卷一百五十八]]</ref>

《新唐书·[[陆龟蒙|陆龟蒙]]传》称李蔚、[[卢携|卢携]]素日交好隐士陆龟蒙，拜相后召拜左拾遗，诏书才下，陆龟蒙卒。但李蔚早在乾符五年即罢相，陆龟蒙约卒于[[中和_(唐朝)|中和]]元年（881年），卢携也在之前的[[广明_(唐朝)|广明]]元年（880年）被罢相、自杀，此记载存疑。<ref>[[:s:新唐书/卷196|《新唐书》卷一百九十六]]</ref>
