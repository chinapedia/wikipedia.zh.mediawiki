'''李准'''{{BD|1928年|7月4日|2000年|2月2日|catIdx=L李}}，[[河南省|河南省]][[洛阳市|洛阳市]]人，[[蒙古族|蒙古族]]，中国大陆小说家、电影剧本家，[[中国共产党|中国共产党]]党员。<ref name="ts" >{{cite web |url=http://news.xinhuanet.com/mrdx/2012-05/04/c_131568188.htm |title=1962年河南专业作家的待遇  |accessdate=2012 |publisher=新华每日电讯 |work= |language=zh-hans}}</ref><ref name="po" >{{cite web |url=http://culture.people.com.cn/GB/70806/116544/12841415.html |title=李准：中国作家协会应设影视文学创作中心 |accessdate=2010 |publisher=人民网 |work= |language=zh-hans}}</ref><ref name="ms">{{cite web|url=http://book.sina.com.cn/maodun/2003-11-04/3/22503.shtml |title=李准：《黄河东流去》 |accessdate=2003 |publisher=新浪网 |work= |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20050911233914/http://book.sina.com.cn/maodun/2003-11-04/3/22503.shtml |archivedate=2005-09-11 }}</ref>李准曾出任[[中国现代文学馆|中国现代文学馆]]馆长和[[中国作家协会|中国作家协会]]副主席。

==生平==
===早年===
1928年，李准出生在[[河南省|河南省]][[洛阳市|洛阳县]]麻屯镇的教师家庭。<ref name="fish" >{{cite web |url=http://www.chinawriter.com.cn/xdzj/511.shtml |title=李準 (1928.5.17—) |accessdate= |publisher=中国作家网 |work= |language=zh-hans}}</ref>祖父李祖莲、伯父李明昭、叔父李明善都是教师，父亲李明选是商人，母亲杨氏出身于医生家庭。
1934年，6岁的李准到麻屯镇小学读书，后来考入洛阳县常袋镇达德中学。1942年，河南省[[1942年河南饥荒|大干旱大饥荒]]，李准逃难到[[陕西省|陕西省]][[西安市|西安市]]。15岁时，李准被家里送到洛阳车站恒源盐栈当学徒，业余时间在“聋子书店”借书。<ref name="fish" />17岁时，李准到麻屯镇邮政所当[[邮递员|邮递员]]。<ref name="fish" />19岁时，李准在麻屯镇戏剧团学习[[戏曲|戏曲]]。1948年，李准在豫西中州银行当职员，后来调到洛阳市干部文化学校当[[语文|语文]]教师。<ref name="fish" />

===中年===
[[File:1963-10_1963年_李准_左二.jpg|thumb]]
1953年，李准开始发表作品。1954年，李准调到[[郑州|郑州]]的农村工作。<ref name="fish" />1955年，李准调到河南省文联工作，同年，李准被任命为人大代表。<ref name="fish" />32岁时，李准加入[[中国共产党|中国共产党]]，同时被任命为中国作家协会理事。<ref name="fish" />1964年，李准被任命为[[人大代表|人大代表]]和[[全国青联|全国青联]]委员。1966年，[[毛泽东|毛泽东]]发起[[文化大革命|文化大革命]]，李准被划分为“[[右派|黑帮分子]]”，接受[[中华人民共和国政府|中华人民共和国政府]]的[[批斗|批斗]]，之后[[劳动改造|劳动改造]]，下放到[[周口市|周口市]]的牛棚。<ref name="fish" />1980年，李准被任命为人大代表、河南省文联副主席、河南省作家协会主席、中国电影家协会河南省主席。<ref name="fish" />

===晚年===
1981年，李准调到北京工作。1990年，李准被任命为[[中国现代文学馆|中国现代文学馆]]馆长。1996年，李准被任命为[[中国作家协会|中国作家协会]]副主席。2000年，李准在[[北京|北京]]病逝。

==奖项==
* 1963年，《李双双》荣获中国电影[[百花奖|百花奖]]最佳电影编剧奖。
* 1981年，《王结实》荣获全国优秀短篇小说奖、第五届中国电影[[金鸡奖|金鸡奖]]最佳编剧奖。
* 1985年，《黄河东流去》荣获第二届[[茅盾文学奖|茅盾文学奖]]。
