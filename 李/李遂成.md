'''李遂成'''（生卒年不詳），又名'''李昌'''，有「'''大俠李鬍子'''」、「'''金鉤李鬍子'''」、「'''星龍長老'''」、「'''金鉤禪師'''」等稱號<ref name="名手">{{cite web |url=http://lsw1230795.mysinablog.com/index.php?op=ViewArticle&articleId=4843122 |title=俠家名手金鉤李鬍子 |author=山中人 |publisher=山中札記 |date=2014-01-26 |language= |accessdate= }}</ref><ref>{{cite web |url=http://www.chinesekungfu.com.cn/html/1309/71c6fce1-a2d7-4131-aaba-1a76f03679ce.htm |title=峨眉武术起源与拳种浅析 |author=李聪 |publisher=中国武术在线 |date=2013-09-23 |language= |accessdate= }}</ref>，[[清末|清末]]武術高手。

==生平==

李遂成原籍[[四川|四川]]，是[[廣東十虎|廣東十虎]]中的[[王隱林|王隱林]]和[[梁坤|梁坤]]之師父，把[[俠家拳|俠家拳]]傳給王隱林及傳武術於梁坤。

李遂成原是[[太平天國|太平天國]]的將領，在[[洪秀全|洪秀全]]失敗後隱姓埋名，領著一幫兩湖三湘的子弟，打擊清朝的官吏豪強。<ref name="名手" />

==金鉤李鬍子另一說法==

有另一說法，指金鉤李鬍子名叫'''李春初'''，[[張家界市|張家界市]][[慈利縣|慈利縣]]陽和鄉甄家坪人，早年在太華寺及九龍寺學藝，練成[[無影腳|無影腿]]、口吹神針兩門絕技。藝成後為[[青幫|青幫]][[沅水|沅水]]、[[澧水|澧水]]兩大流域的總龍頭老大。常在溪口老街瑞生堂藥鋪，醫治跌打損傷的病人。

李春初有徒弟[[黃麒英|黃麒英]]和[[王雲清|王雲清]]兩人。黃麒英是廣東十虎之一，把武藝傳給兒子[[黃飛鴻|黃飛鴻]]。王雲清在李春初推薦下，在清將軍[[杜桂珍|杜桂珍]]府中任管家，並把武藝傳了給杜桂珍的兒子[[杜心武|杜心武]]。杜心武、王雲清、李春初三位並稱「民國三俠客」。<ref>{{cite web |url=http://blog.sina.com.cn/s/blog_4b5354b60102vox0.html |title=關於杜心武飛腿功1900年在日本留學時打敗日本相撲的説明 |author=王正鵬 |publisher=新浪博客 |date=2015-07-04 |language= |accessdate= }}</ref>
