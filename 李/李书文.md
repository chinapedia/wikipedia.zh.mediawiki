'''李書文'''（{{bd|1862年||1934年|catIdx=L}}），字'''同臣'''，大清帝國[[河北省|河北省]][[滄州|滄州]]鹽山縣王南良村人。清末武術名家，為[[八極拳|八極拳]]代表人物之一，別號'''神槍'''。

== 生平 ==
[[File:Li_Shuwen.png|thumb]]
出身於貧農家庭，由於生活艱辛，其父將之賣予戲班學戲。李書文身材短小，只能學武丑的角色，學得戲班北派之翻跌表演功夫。由於戲班要求嚴格，經常遭受打罵。一日，李被師父用木刀打傷後腿，傷處潰爛，需切除。後終被戲班解約遣回。因為少年時學戲留下的陰影，所以李書文終生不看戲。{{請求來源}}

李書文幼時從[[孟村|孟村]][[金殿陞|金殿陞]]先生學習八極拳和大槍術。之後至滄州[[羅畽|羅畽]]張家八式房學武，想拜[[黃四海|黃四海]]為師。但黃四海醉心武學，不收徒弟，聞其名來投師的弟子，都請他們至師弟[[張景星|張景星]]門下學習。授藝則是黃四海及張景星共同授藝。

他劄槍出手快，用勁狠，手下不留情，經常有死傷，有「矬爺」、「李狠子」的綽號。他在京津一帶行走，以六合大槍聞名，被稱為「神槍李」。在李書文成名後，曾經具弟子禮，在張景星的推薦下，拜[[黃林彪|黃林彪]]為師習[[劈掛拳|劈掛拳]]，成為黃林彪的過門弟子，在此時被認為是八極門中極具實力的後起新秀。

[[袁世凱|袁世凱]]在天津小站訓練北洋新軍時，因為聽聞黃四海大名，派人禮聘他為武術教練，黃四海以年老推辭，推薦李書文前去，李書文名聲漸漸在京津流傳。1910年，天津[[中華武士會|中華武士會]]成立，八極拳門下張景星率[[王中泉|王中泉]]、李書文至武士會擔任教習，以李書文名聲最盛，曾與[[李瑞東|李瑞東]]、[[李存義_(清朝)|李存義]]結拜，依年紀排行第三。李瑞東曾將他與黃四海換藝所得的金剛八式，又授與李書文，傳為一時佳話。

李書文與許多出身[[北洋新軍|北洋新軍]]的將領交好，其中又以後來加入[[奉軍|奉軍]]的[[許蘭州|許蘭州]]、[[李景林|李景林]]及[[張驤伍|張驤伍]]為著，經常在他們府中來往長住。1907年，許蘭州聘請他為部隊總教習。民國初年，至[[馮國璋|馮國璋]]府教授武術。1925年，李景林聘請他為其部隊總教官，他與弟子[[霍殿閣|霍殿閣]]一同前往。1926年，攜[[李萼堂|李萼堂]]、[[魏鴻恩|魏鴻恩]]至山東[[沈鴻烈|沈鴻烈]]海軍處為武術教練。{{請求來源}}

1927年，張之江在南京創立[[中央國術館|中央國術館]]，李書文受聘為顧問。1929年，許蘭州捐資成立[[河北國術館|河北國術館]]，聘請他為總顧問。同年，赴杭州國術大賽，擔任檢查委員。1931年，李景林邀請他至山東國術館，他攜徒[[劉雲樵|劉雲樵]]同往，同年年底，李景林過世，李書文至黃縣張驤伍處暫居，將劉雲樵交給張驤伍照顧後，自己又四處雲游。1933年春，應邀參加山東省主席[[韓復榘|韓復榘]]主持的全省國術考試，擔任裁判，並表演純陽劍、八極拳、六合大槍。《滄州武術志》與劉雲樵這一系傳說，他1934年在山東濰縣遭人下毒而過世。

李書文喜愛小孩，每次回到滄州鹽山或是天津小站時，都會準備銅板，送給村里中的小孩。他也教導他們拳術，小孩皆暱稱他是把式爺爺。其孫輩李子昆、[[李子芳_(八極拳)|李子芳]]都得到他的親自教導。其孫李子昆與李子芳一系傳說，李書文在1933年回到天津小站，在其養子[[李東堂|李東堂]]家長住。於1934年6月因腦溢血過世於天津小站家中，容顏安詳，享壽72歲，其遺體歸葬於滄州鹽山<ref>[http://www.shizhanbajiquan.com/third.jsp?id=106&resourceId=bjcc 天津小站北闸口流传的八极拳] {{webarchive|url=https://archive.is/20160823092052/http://www.shizhanbajiquan.com/third.jsp?id=106&resourceId=bjcc |date=2016-08-23 }}</ref>。

== 家庭 ==
李書文醉心武術，又常年在外奔走，教拳為生，一生未婚，無子。其家族成員為避免他絕後，將兒子過繼給他。分別為李東堂、[[李萼堂|李萼堂]]、[[李義堂|李義堂]]。

李東堂為其弟李桐生之子，居住在天津小站北閘口，其子[[李子芳_(八極拳)|李子芳]]稱李書文為爺爺。李萼堂早年與李書文到處遊歷，1931年李萼堂受邀至湖南擔任軍隊武術教官，此後便定居湖南。李義堂則居住在滄州鹽山。

== 武術風格 ==
李書文一生最重視大槍，每日起床必練大槍，在大槍之外，只練習八極拳中的六開八招及[[劈掛掌|劈掛掌]]幾個基本拳勢，輕視拳術套路與其他兵器。只有親近弟子能得到他的大槍傳授，可惜因為時代因素，他的大槍並沒有很好的被傳承下來。

其拳法重招勢，輕套路，特別反對花法套子，刪繁就簡。他傳授的八極拳都參以一套劈掛掌。與人比試時，常在數招內就決定勝負，有「李書文無二手」之美稱。

李書文並傳有從李存義處學來的純陽劍法。

== 弟子 ==

李書文一生授徒嚴謹，入室弟子僅近20名，有：王樹德、張德忠 、崔長友、[[霍殿閣|霍殿閣]]、[[李萼堂|李萼堂]]、孟憲忠、張子林、高熙臣、曾寶發、王樹江、張驤伍、許家福、許家祿、魏鴻恩、魏鴻濱、[[劉雲樵|劉雲樵]]等。在弟子中，王樹德入門最早，但早死，後以霍殿閣輩份最長，為頂門弟子，劉雲樵為其所收的最後一名弟子。為避免輩份混亂，其孫輩，如李子昆、李子芳，雖經他指導，但在拳譜輩份上則指定由其他弟子擔任師父，如李子昆為霍殿閣之徒，李子芳為高啟臣之徒。

其徒侄趙樹德、董義清、韓潔泉、李學義、董會亭、張子亭、劉漢周、沈中山、姚春符、董義文等都追隨他闖天下。其軍中高級將領的學生有:[[許蘭洲|許蘭洲]]、任國棟、陳富貴、張驤伍、那玉昆、劉序東、柳虎臣、沈鴻烈、劉之沂、劉之潔、李景林、韓複榘、夏鶴一等。

其再傳弟子高香亭、霍慶雲、邊廷傑、劉子鳴、邊延彬、劉金山、劉琪瑞、李子昆、許禹聲、郝鴻昌、鮑友聲、張世忠等在民國中期亦有聲於國術界。

== 流行文化 ==
*日本漫畫[[拳兒|拳兒]]曾描寫過他與其弟子[[劉雲樵|劉雲樵]]的部分生平。
*李書文在日本電玩《[[Fate/EXTRA|Fate/EXTRA]]》与《[[Fate/EXTELLA|Fate/EXTELLA]]》之中先以「Assassin」的職階登場，寶具為「無二打」；以後以「Berserker」的職階再現，寶具改為「猛虎硬爬山」。
*李書文在日本手機遊戲《[[Fate/Grand_Order|Fate/Grand Order]]》之中以壯年時期的狀態召喚至Lancer級Servant登場，寶具為「神槍無二打」。以及以老年時期的狀態召喚至Assassin級Servant登場，寶具為「無二打」。
