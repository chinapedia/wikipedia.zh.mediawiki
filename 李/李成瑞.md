'''李成瑞'''（{{bd|1922年|1月16日|2017年|2月11日}}），中華人民共和國[[河北|河北]][[唐县|唐县]]人，统计学家、经济学家、左派人物，曾任[[李先念|李先念]]秘书及[[中华人民共和国国家统计局|国家统计局]]局长<ref name=shishi>{{Cite web |url =http://news.ifeng.com/a/20170215/50696462_0.shtml  |title = 95岁国家统计局原局长李成瑞逝世，曾任李先念秘书 |author =  |date = 2017-02-15 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref name=chentong>{{Cite web |url =http://www.wyzxwk.com/Article/shidai/2017/02/376616.html |title = 沉痛悼念李成瑞同志 |author =  |date = 2017-02-14 |publisher =乌有之乡  |language =  |accessdate =  |quote =  }}</ref><ref name=jj>{{Cite web|url =http://cjb.newssc.org/html/2016-01/29/content_2277388.htm|title =冀籍经济学家：思想活跃的知名学者|author =|date =2016-01-29|publisher =企业家日报|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170215215124/http://cjb.newssc.org/html/2016-01/29/content_2277388.htm|archivedate =2017-02-15}}</ref>。

==生平==
李成瑞生于河北省唐县淑闾村，父亲李登魁是抗日烈士。1937年11月，李成瑞参加[[中国共产党|中国共产党]]领导的革命，投身敌后抗日战争，任唐县人民武装抗日自卫会宣传队员。1943年1月加入中国共产党。[[华北联合大学|华北联合大学]]第一期毕业。[[抗日战争|抗日战争]]前期，历任[[晋察冀边区|晋察冀边区]]政府《救国报》编辑、记者，民族革命通讯社晋察冀边区分社记者，晋察冀边区政府机关刊物《边政导报》编辑室主任编辑。1943年至1945年，历任晋察冀边区政府财政处干事、晋察冀边区政府财政处税务科科长，研究拟定农民负担办法、征收救国公粮、支援晋察冀边区部队，曾多次奉派到敌人碉堡林立的敌后游击区从事抗日“两面村”政权的调查工作<ref name=shishi/><ref name=chentong/><ref name=jj/>。

[[第二次国共内战|第二次国共内战]]后期，李成瑞曾任[[华北人民政府|华北人民政府]]财政部政策研究室主任。1949年3月23日，[[毛泽东|毛泽东]]自[[西柏坡|西柏坡]]去[[北平|北平]]途中，曾经在李成瑞父亲李登魁的故居留宿<ref name=shishi/><ref name=chentong/><ref name=jj/>。

[[中华人民共和国|中华人民共和国]]成立后，历任[[中央人民政府财政部|中央人民政府财政部]]农业税司副司长、国务院财贸办公室秘书（[[李先念|李先念]]的秘书）。1974年9月任[[国家统计局|国家统计局]]副局长，1981年10月任国家统计局局长。其间曾兼任国务院[[中华人民共和国第三次全国人口普查|第三次全国人口普查]]（1982年）办公室主任。1984年5月离休，享受部长级医疗待遇<ref name=shishi/><ref name=chentong/><ref name=jj/><ref>{{Cite web |url =http://news.qq.com/a/20160126/057429.htm  |title = 王保安2015年5月接任国家统计局局长一职 |author =  |date = 2016-01-26 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。此后曾任国家统计局顾问、[[第七届全国人民代表大会|第七届全国人民代表大会]]财经委员会顾问。1981年起，先后兼任[[中国人民大学|中国人民大学]]、[[厦门大学|厦门大学]]、[[中央财经大学|中央财经大学]]等高等院校的教授。他曾当选为中国统计学会副会长、会长、名誉会长，国际统计学会副主席（是中国首次进入国际统计学会领导机构的统计学家）。还曾当选为中国人口学会副会长、中国财政学会副会长、[[孙冶方|孙冶方]]经济科学基金会理事。2009年，当选为“影响新中国60年经济建设的100位经济学家”。其主要专著和论文曾获得国家科学技术进步奖、[[孙冶方经济科学奖|孙冶方经济科学奖]]、全国优秀统计图书奖等奖项<ref name=shishi/><ref name=chentong/><ref name=jj/>。

离休之后，李成瑞成为中华人民共和国左派的代表人物之一。他认为中国[[改革开放|改革开放]]时期的社会结构出现了较大变化，认为形成了“新[[资产阶级|资产阶级]]”、分配不合理、对外国的依赖性增强，主张重新确立[[社会主义|社会主义]]经济制度<ref name=jj/><ref name=chentong/><ref>{{Cite web |url = http://jiliuwang.net/archives/48901 |title = 【纪念李成瑞】千人断指叹——当代的“三吏三别” |author =  |date = 2017-02-15 |publisher =激流  |language =  |accessdate =  |quote =  }}</ref>。另外他还提倡以立法手段增加官员财产透明度<ref>{{Cite web |url =http://news.sina.com.cn/c/2009-03-13/054617397986.shtml  |title =原国家统计局长李成瑞:早日为官员财产申报立法  |author =  |date = 2009-03-13 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>，以及通过合法斗争来增加工人的权益<ref>{{Cite web |url = http://news.ifeng.com/history/zhongguoxiandaishi/detail_2010_10/18/2814359_0.shtml  |title = 李成瑞：不支持再来一次文革的“左派老人” |author =  |date =2010-10-18  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。《[[中流|中流]]》和《[[真理的追求|真理的追求]]》停刊后，2003年，一批中共离退休高级干部发起成立了[[毛泽东旗帜网|毛泽东旗帜网]]，这批老干部以顾问团队的形式存在，李成瑞是这个顾问团队的领头人和毛泽东旗帜网的思想主导者<ref>{{Cite web |url = http://www.nfcmag.com/article/2146.html |title = “左派老人”李成瑞 |author =  |date =  |publisher = 南风窗2010年12期  |language =  |accessdate =  |quote =  }}</ref>。生命中的最后几年，他还创办和主持“东方红社科网”（www.dfhsk.org，2014年被关闭）和《东方红文粹》<ref name=chentong/>。

2017年2月11日，李成瑞在[[北京协和医院|北京协和医院]]病逝，享年95岁<ref name=shishi/>。
