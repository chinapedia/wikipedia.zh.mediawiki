'''李玄旭'''（{{bd|1994年|2月5日}}），[[湖南省|湖南省]][[株洲市|株洲市]]人，[[中國|中國]]女子[[游泳|游泳]]运动员。2003年加入[[广东省|广东省]]游泳队师从[[冯真|冯真]]，开始接受专业训练，主攻[[混合泳|混合泳]]项目。2007年在[[中华人民共和国城市运动会|第六届全国城市运动会]]所参加的三个混合游泳项目全部获得金牌，并被评为“城运之星”，从此在中国游泳界脱颖而出<ref>2007年11月8日，[http://sports.tom.com/2007-11-08/070J/05076794.html TOM网]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}《朱启南李玄旭当选城运之星，射击游泳有突破》</ref><ref>[http://www.sports.gov.cn/olympic/aoyun/ao2/2008728115133.htm 湖南体育网]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}，《李玄旭——参加北京奥运会游泳比赛的湖南选手》</ref><ref>[http://data2008.sports.tom.com/athletes-1077.shtml TOM网]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}：奥运数据库</ref>。

== 出生 ==
李玄旭出生于[[株洲市|株洲市]]，父亲李异洲，母亲宋莉。玄旭7岁开始学游泳，2003年被执教[[湖南|湖南]]游泳队的[[辽宁|辽宁]]籍游泳教练[[冯真|冯真]]选中后开始接受专业训练。玄旭平时喜好[[钢琴|钢琴]]和[[书法|书法]]，4岁开始开始练钢琴，10岁时钢琴过8级，书法已过6段，其愿望是退役后去[[香港|香港]]读[[大学|大学]]<ref>（2008年7月25日）[http://jtdb.hnol.net/article/20087/200872581920708743693.html 湖南家庭导报] {{webarchive|url=https://web.archive.org/web/20081122132428/http://jtdb.hnol.net/article/20087/200872581920708743693.html |date=2008-11-22 }}：《李玄旭：父母引导她游进“[[水立方|水立方]]”》（作者：《[[湖南家庭导报|湖南家庭导报]]》记者/陈波，实习生/段燕琪、张玲玲）</ref>。

== 參賽經歷 ==
[[2008年夏季奥林匹克运动会|2008年北京奥运会]][[2008年夏季奧林匹克運動會游泳比賽－女子800米自由泳|女子800米自由泳]]获得第5名（预赛成绩8:24.37／决赛成绩8:26.34），[[2008年夏季奧林匹克運動會游泳比賽－女子400米混合泳|女子400米个人混合泳]]获得第8名（预赛成绩4:36.35／决赛成绩4:42.13）。

[[2012年夏季奥林匹克运动会|2012年倫敦奥运会]][[2012年夏季奧林匹克運動會女子400米混合泳比賽|女子400米混合泳]]中，她在預賽以4分34.28秒，第2名完成比賽，獲得決賽的參加資格。<ref>[http://www.london2012.com/swimming/event/women-400m-individual-medley/phase=sww054900/index.html 2012年倫敦奧運賽程及賽果]</ref>決賽中以4分32.91秒奪得銅牌。<ref>[http://www.london2012.com/swimming/event/women-400m-individual-medley/phase=sww054100/index.html London 2012 Swimming,400m individual medley women Final]</ref>

== 成绩 ==
* 2007年[[全国游泳冠军赛|全国游泳冠军赛]]女子200米混合泳（2:14.90）/400米混合泳（4:39.18）二项冠军；
* 2007年[[全国游泳锦标赛|全国游泳锦标赛]]以获400米混合泳个人冠军（4:43.18）；
* 2007年日本游泳国际邀请赛200米个人混合泳亚军（2:15.04）；
* 2007年[[中华人民共和国城市运动会|第六届全国城市运动会]]女子200米混合泳（2:13.98）、400米混合泳（4:37.56）和800米混合泳（8:27.65）三项冠军；
* 2008年[[全国游泳冠军赛|全国游泳冠军赛]]暨奥运选拔赛女子400米（4:39.07）和800米混合泳（8:31.66）二项冠军，200米混合泳亚军（2:14.54）。
