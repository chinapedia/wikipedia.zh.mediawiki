'''李德水'''（{{bd|1944年|7月|}}）[[江西省|江西省]][[玉山县|玉山县]]人，[[中华人民共和国|中华人民共和国]]政治人物<ref name=jianli>{{Cite web |url = http://news.xinhuanet.com/fortune/2004-05/26/content_1491101.htm |title =李德水简历  |author =  |date =2004-05-26  |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
大学本科毕业。高级工程师。1985年11月加入[[中国共产党|中国共产党]]。1968年7月，自[[江西冶金学院|江西冶金学院]]毕业后，被分配到[[安徽|安徽]][[马鞍山钢铁公司|马鞍山钢铁公司]]工作，曾经担任工段长、调度员。1977年，到[[中华人民共和国冶金部|中华人民共和国冶金部]]规划院工作。1984年，到[[中华人民共和国国家计划委员会|中华人民共和国国家计划委员会]]工作，1988年任长期规划司产业规划一处副处长，1989年后任长期规划司产业规划一处、长期规划和产业政策司处长，1992年任国民经济综合司副司长，1996年5月任国民经济综合司司长<ref name=jianli/>。

1996年11月，出任[[四川省|四川省]][[重庆市人民政府|重庆市人民政府]]副市长。1997年6月任[[重庆市人民政府|重庆市人民政府]]副市长。1999年11月，任[[国务院研究室|国务院研究室]]副主任、党组成员。2002年4月，任[[中国国际工程咨询公司|中国国际工程咨询公司]]党组书记、副总经理。2003年3月，任[[中华人民共和国国家统计局|中华人民共和国国家统计局]]党组书记、局长<ref name=jianli/>。根据[[中国共产党中央委员会|中国共产党中央委员会]]2005年11月7日中委〔2005〕255号通知，中央批准：[[邱晓华|邱晓华]]任中共国家统计局党组书记；免去李德水的国家统计局党组书记职务。根据[[中华人民共和国国务院|中华人民共和国国务院]]2006年3月11日国人字〔2006〕16号通知，任命邱晓华为国家统计局局长；免去李德水的国家统计局局长职务<ref>{{Cite web |url = http://news.163.com/06/0317/08/2CDEFSNQ00011GGF.html |title =  免去李德水国家统计局局长职务 邱晓华接任|author =  |date = 2006-03-17 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2008年，当选第十一届全国政协委员<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日|df=}}</ref>，代表经济界，分入第三十四组。并担任第十一届全国政协经济委员会副主任<ref>{{cite news|title=全国政协信息-李德水|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0012190&wytype=wjwy|accessdate=2013-04-11|newspaper=全国政协委员会}}</ref>。
