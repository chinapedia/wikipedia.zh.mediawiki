'''李扬'''（{{bd|1958年|6月21日|||catIdx=L李}}），生于[[福建省|福建省]][[泉州市|泉州市]]，籍贯福建省[[南安市|南安市]]，[[中华人民共和国|中华人民共和国]]配音演员、商人、官员<ref name=qunlong>{{Cite web |url =http://news.china.com/news100/11038989/20130304/17710006.html  |title =李扬代表：我国动漫产业是一盘散沙 群龙无首  |author =  |date = 2013-03-04 |publisher =中华网  |language =  |accessdate =  |quote =  }}</ref><ref>{{cite web|url=http://hn.rednet.cn/c/2012/12/13/2845787.htm|title=李扬：给真假美猴王那一集配音最累人|publisher=红网|date=2012-12-13|accessdate=2016-02-18}}</ref>。

==生平==
童年时，李扬非常喜欢看动画片。初中毕业后，家人不忍送他“[[上山下乡|上山下乡]]”，便找接兵部队的领导说情，让他加入[[北京军区|北京军区]]工程兵。劳动之余，李扬喜欢给战友讲故事，讲得最多的是当时被视作“禁书”的《[[包公案|包公案]]》和《[[聊斋|聊斋]]》。于是有人向上级反映称李扬用封建思想毒害战友，此后他讲得就少些。当兵两年后，李扬转业回到地方，在发电设备厂当钳工。一年后被调到工厂团委，负责[[中国共青团|共青团]]的组织和宣传工作<ref name=jianzheng>{{Cite web |url =  http://play.163.com/special/jianzheng_54/|title = 动漫好声音之“唐老鸭”李扬_《见证》系列专题第54期 |author =  |date =  |publisher =网易  |language =  |accessdate = 2017-11-28 |quote =  }}</ref>。

1978年，李扬考入[[北京广播学院|北京广播学院]]新闻编采专业。在校期间，1980年李扬将自己模仿[[邱岳峰|邱岳峰]]配音的片段录成磁带寄往[[上海电影译制厂|上海电影译制厂]]邱岳峰收，但因邱岳峰已故，被原物退回。1981年[[中国中央电视台|中央电视台]]成立译制组不久，李扬经中央电视台高级录音师曾文济推荐来试音，模仿了邱岳峰的配音片段后被录取。1982年大学毕业后，家人安排他给某部长当秘书，他却坚持要从事配音，乃进入中央电视台，成为《为您服务》栏目的编导。同年7月，中央电视台筹拍电视连续剧《[[西游记_(1982年电视剧)|西游记]]》，前五集的[[孙悟空|孙悟空]]由[[安徽京剧团|安徽京剧团]]青年演员[[李世宏|李世宏]]配音，李世宏离开剧组后，任《西游记》总录音师的曾文济推荐了李扬，获导演[[杨洁|杨洁]]接受。1982年《西游记》剧组试拍《除妖乌鸡国》，李扬为孙悟空配音，[[董行佶|董行佶]]为唐僧配音，这是两人唯一一次合作。次年6月，董行佶题词“艺术贵在创新，流派应予发展”赠李扬，一周后逝世。《西游记》拍拍四年，李扬配了四年音。1986年该剧在中央电视台正式开播<ref name=qunlong/><ref name=shengya/><ref name=lyjj/><ref name=jianzheng/>。

1984年，[[上海美术电影制片厂|上海美术电影制片厂]]改编“三打白骨精”，创作动画片《[[金猴降妖|金猴降妖]]》，请李扬为孙悟空配音。
《金猴降妖》与该厂此前推出的动画片《丁丁战猴王》、《人参果》均沿用1960年代[[万籁鸣|万籁鸣]]执导的《[[大闹天宫|大闹天宫]]》的画风及人设。配音期间，李扬向[[特伟|特伟]]厂长提出希望拜访万籁鸣，见面后万籁鸣用彩笔在笔记本上画了个孙悟空的头像，题词“赠艺林精英李扬同志”。为《金猴降妖》配音后的二十多年间，李扬陆续为其他影视剧和动画片中的孙悟空配音，包括2012年初上映的3D版《大闹天宫》<ref name=jianzheng/>。

在中央电视台工作期间，他先后主持《文化生活》、《九州方圆》、《祖国各地》、《为您服务》等系列专栏。多次主持中央电视台春节、元旦、国庆、元宵等大型晚会，他表演的电影配音片段和模仿领袖人物讲话经常是最受欢迎的节目。参与制作了近百部专题片，其中《小窍门集锦》（60集）、《沿海明珠》（22集）、《国庆趣话》（3集）、《周总理和党的文艺事业》、《话说香港》（60集）等都在全国专题片评选中获一等奖或特别奖<ref name=qunlong/><ref name=shengya/><ref name=lyjj>{{Cite web |url = http://www.cctv.com/tvguide/tiaozhanzhuchiren/sanji/xuans20020315_05.html |title =李扬简介   |author =  |date = 2002-03-15 |publisher =央视网  |language =  |accessdate =  |quote =  }}</ref>。

1980年代起，参与了数百部外国电影、电视剧的译制配音，其中包括[[苏联|苏联]]大型宽银幕电影《[[莫斯科保卫战_(电影)|莫斯科保卫战]]》中的[[希特勒|希特勒]]，央视版《[[西游记_(1982年电视剧)|西游记]]》及续集中的[[孙悟空|孙悟空]]（[[五行山|五行山]]之后的取经部分），1984年动画片《[[金猴降妖|金猴降妖]]》中的孙悟空，央视版《[[红楼梦_(1987年电视剧)|红楼梦]]》中的贾瑞，1980年代版《[[福尔摩斯探案集|福尔摩斯探案集]]》中的[[福尔摩斯|福尔摩斯]]等<ref name=qunlong/><ref name=shengya/><ref name=lyjj/>。

1980年，中央电视台引进[[美国|美国]][[迪斯尼|迪斯尼]]的《[[米老鼠和唐老鸭|米老鼠和唐老鸭]]》。迪斯尼提出，其他角色可任由中方选择配音演员，但[[唐老鸭|唐老鸭]]的配音演员必须获得他们认可，中央电视台在全中国找了许多配音演员录音，美方听了寄来的所有录音，最终选定李扬。1986年至1988年，104集的《米老鼠和唐老鸭》在中央电视台首播，风靡全中国，李扬也因此成为家喻户晓的明星<ref name=shengya>{{Cite web |url =  http://www.china.com.cn/node_7064072/content_23221247.htm|title = 著名配音演员李扬的配音生涯 |author =  |date =  2011-08-16 |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref>。

1992年底，李扬经[[中共|中共]]党组织安排定居[[香港|香港]]，开办了他的第一家公司“香港李扬集团有限公司”，做中央电视台香港及[[东南亚|东南亚]]节目销售的总代理<ref>李扬，香港回归前从事传媒工作的几点回忆，新闻战线2017(15):61-62</ref>。该集团下设“香港李扬音像有限公司”、“香港李扬广告电视制作公司”、“香港李扬国际贸易有限公司”等5家公司。在香港，李扬集团办公室及电视制作中心面积1400多平方米，资产总值近一亿港元。1990年代，该集团先后将中央电视台的《[[新闻联播|新闻联播]]》、《[[经济半小时|经济半小时]]》等节目重新包装为《神州经贸》在香港[[卫视中文台|卫星电视中文台]]播出。由他本人主持的《中国财经特辑》（中央电视台与香港[[亚洲电视台|亚洲电视台]]合作）、53集专题片《话说香港》（与[[国务院新闻办公室|国务院新闻办公室]]等单位合拍）也分别在中央电视台和香港[[亚洲电视台|亚洲电视台]]播出。他参与包装的大型电视剧《[[三国演义_(电视剧)|三国演义]]》、《[[武则天_(1995年电视剧)|武则天]]》、《[[宰相刘罗锅|宰相刘罗锅]]》、《[[官场现形记_(电视剧)|官场现形记]]》、《[[邓小平|邓小平]]》等获得成功，在电视台播出后创香港五十年最高收视率。该集团还与[[天津电视台|天津电视台]]等省市电视台有节目交流和业务往来<ref name=tianjin>{{Cite web |url = http://news.enorth.com.cn/link/tjhk/tj-hk/human/liyang02161251.htm |title = 第一个在香港成功创办影视业的天津人——李扬 |author =  |date =  |publisher = 北方网 |language =  |accessdate = 2017-11-28 |quote =  }}</ref><ref name=lyjj/>。
