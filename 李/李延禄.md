'''李延禄'''（{{bd|1895年|4月1日|1985年|6月18日}}），[[吉林省|吉林省]][[延吉县|延吉县]]人，[[中华人民共和国|中华人民共和国]]政治人物。

==生平==
李延禄早年入伍[[东北军|东北军]]，历任连长，随后派任中国国民救国军吉林总部参谋长等职，随后组建东北抗日游击总队，并扩编为东北抗日救国游击军。1931年，他整编并创建[[东北抗日联军|东北抗日联军]]第四军，指挥近百次战斗<ref>{{cite news|author=杨兴江|title=李延禄在大罗勒密建立“红地盘”|url=http://dangshi.people.com.cn/GB/85039/13923415.html|accessdate=2011-09-25|newspaper=人民网|date=2011年2月15日}}</ref>。1936年，代表中共到上海、南京做统一战线工作。第二次国共内战期间，担任[[合江省|合江省]]人民政府主席、[[松江省|松江省]]人民政府副主席。中华人民共和国成立后，担任[[黑龙江省|黑龙江省]]副省长、省委委員<ref>{{cite news|title=新中国人物志——李延禄|url=http://www.hljic.gov.cn/sqgl/mrfc/t20110512_549345.htm|accessdate=2011-09-25|newspaper=中国黑龙江信息网|deadurl=yes|archiveurl=https://web.archive.org/web/20110820202810/http://www.hljic.gov.cn/sqgl/mrfc/t20110512_549345.htm|archivedate=2011-08-20}}</ref>。1954年，当选[[第一届全国人民代表大会代表名单|第一届全国人民代表大会代表]]<ref>{{cite book|author=全国人民代表大会常务委员会办公厅|title=中华人民共和国第一届全国人民代表大会代表名单|year=1955年|publisher=全国人民代表大会常务委员会办公厅|url=http://book.ucdrs.superlib.net/views/specific/2929/bookDetail.jsp?dxNumber=000005772167&d=546843D0AE2464CD1C95896344A85614&fenlei=04070402}}</ref>。1955年获一级[[八一勋章_(1955年)|八一勋章]]。此后历任[[第三届全国人民代表大会常务委员会|第三届]]、[[第四届全国人民代表大会常务委员会|第四届]]、[[第五届全国人民代表大会常务委员会|第五届全国人民代表大会常务委员会]]委员<ref>{{cite web | url=http://www.lsjp.org/show.asp?id=851 | title=抗日英雄——李延禄 | accessdate=2011-09-25}}</ref>。

==家庭==
*妻：田佐民，抗日救国军成员，与丈夫儿女一道长期开展抗日救国工作。
*长女：[[李万新|李万新]]，[[中国林业科学研究院|中国林业科学研究院]]副院长。
**长女婿：[[孙西林|孙西林]]，中共合江省工委委员、佳木斯地区专员兼[[佳木斯市|佳木斯市]]副市长。遇刺身亡。
*次女：李万英，抗日儿童团团长，与弟共同建立抗日儿童团。
*子：李万杰，抗日儿童团、抗日救国军成员。
*弟：李延祜，又名[[李常青|李常青]]，[[一二·九运动|一二·九运动]]领导人，中共北平地下市委书记。
*弟：[[李延平|李延平]]，东北抗日联军第四军继任军长，抗战中牺牲。
*弟：李延贵，又名李秉才，[[铁道游击队|铁道游击队]]长，在[[河南|河南]][[焦作|焦作]]战斗中牺牲。
*弟：李延青，抗日救国军第十七团团长，被叛徒暗杀。
*弟：李延田，又名李智，敌军干部训练队区队长。
*弟：李延年，中共[[洛阳|洛阳]]区委书记，曾被逮捕并判五年徒刑。
*妹：李延芹，又名李萍，铁道游击队员，地下工作者。
*侄：李万成，李延贵之子，焦作地区游击队员，为掩护部队撤退独自与敌周旋牺牲，头颅被挂在焦作城头示众。
*侄：李万万，李常青之子，又名范政，是“新旅”（少年儿童革命团体）骨干，与[[郭沫若|郭沫若]]、[[田汉|田汉]]等人在斗争中结下友谊，曾受[[周恩来|周恩来]]等领导多次接见<ref name=huainian>{{Cite web |url =  http://www.caf.ac.cn/news/xwzx/201301/2013-01-23-18-52.html  |title =  深切怀念李万新老院长|author =  |date = 2013-01-23 |publisher = 中国林业科学研究院 |language =  |accessdate =  |quote =  }}</ref>。
