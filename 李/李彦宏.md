'''李彦宏'''（{{lang-en|'''Robin Li'''}}，{{bd|1968年|11月17日}}），[[中华人民共和国|中国]][[山西|山西]][[阳泉|阳泉]]人，[[研究生|研究生]]学历。[[百度|百度公司]]的创建者，现任[[董事长|董事长]]兼[[首席执行官|首席执行官]]以及全国工商联副主席。据2014年胡润百富榜显示，李彦宏身价1050亿元人民币，2015胡润全球富豪榜  位列第47名，身价180亿美元。<ref name="Robin Li">{{cite web| url=http://www.hurun.net/CN/Entrepreneur.aspx?nid=6| title=Robin Li| accessdate=2014-09-24| publisher=胡润百富| 5=| deadurl=yes| archiveurl=https://web.archive.org/web/20140710132323/http://hurun.net/CN/Entrepreneur.aspx?nid=6| archivedate=2014-07-10}}</ref>

李彥宏的資產基本都來自百分之二十點八的百度控股，这些股票直接由李彥宏及其妻子馬東敏所有，並通過註冊地[[英屬維爾京群島|維珍群島]]的 Handsome Rewards Ltd. 控股公司持有。李彥宏還擁有[[京東商城|京東商城]]百分之一的股份。

截止到2014年2月28日，李彦宏本人实益拥有百度公司557万股A类股和B类股，相当于百度公司15.9%的股份，这低于2012年12月31日他持有的558万股股票即相当于百度公司16%的股份。

在政治职务上，李彦宏是[[中国人民政治协商会议第十二届全国委员会|第十二届全国政协]]委员，也是[[中华全国工商业联合会|中华全国工商业联合会]]副主席。<ref>{{cite web | url=http://news.china.com.cn/2013lianghui/2013-03/05/content_28139169.htm| title=政协委员李彦宏:高薪挖著名教授成为不了优秀大学 | accessdate=2016年3月5日 |publisher=中国网}}</ref>

== 简历 ==
1968年，李彦宏出生于[[山西省|山西省]][[阳泉市|阳泉市]]<ref name="2014年福布斯华人富豪榜排名第8名的李彦宏"/>，曾经考入[[山西|山西]][[阳泉晋剧团|阳泉晋剧团]]。<ref>{{cite web |url=http://news.xinhuanet.com/it/2005-09/12/content_3476761.htm |title=李彦宏首谈婚恋故事 |publisher=新华网 |accessdate=2005-09-12 |work= |language =zh}}</ref>

1987年，李彦宏考入[[北京大学|北京大学]]，在北大读书时，李彦宏立志锁定目标留学[[美国|美国]]，精研[[托福|托福]]、[[GRE|GRE]]等英文书籍，过着教室、图书馆和寝室三点一线的生活。

1991年[[北京大学|北京大学]][[信息管理|信息管理]]本科毕业，后在[[美国|美国]][[纽约州立大学布法罗分校|纽约州立大学布法罗分校]]获得[[计算机科学|计算机科学]][[硕士|硕士]]，在留学期间，他白天上课，晚上补习英文和编程，经常学习到凌晨两点。

1994年，李彦宏在[[华尔街|华尔街]]的[[道·琼斯|道·琼斯]]子公司实习，并且工作了三年半时间，曾担任道·琼斯子公司高级顾问、《[[华尔街日报|华尔街日报]]》网络版实时金融信息系统设计人员。

1997年，李彦宏离开华尔街，前往[[硅谷|硅谷]]的[[搜索引擎|搜索引擎]]公司[[Infoseek|Infoseek]]任职工程师。

1999年，Infoseek被[[迪士尼|迪士尼]]收购，再次调整战略。李彦宏放弃了Infoseek的股票期权，和[[徐勇|徐勇]]回到中国创建百度公司，2000年后百度成为全球最大的中文搜索引擎技术公司，百度于2003年第二季度宣布全面盈利。百度公司于美国时间2005年8月5日在[[纳斯达克|纳斯达克]]上市，当日股价升至151美元。李彦宏身为[[CEO|CEO]]，拥有公司的22.9%约9.89亿美元的股东权益。2009年的百度技术创新大会上，李彦宏提出[[框计算|框计算]]（Box Computing）技术概念。

2016年5月，發生[[魏則西事件|魏則西事件]]，大學生[[魏則西|魏則西]]患上罕見癌症，懷疑錯信百度的醫療廣告延誤治療病逝，百度的醫療競價排名和[[莆田系|莆田系]]等問題，引起輿論不滿<ref>{{cite web |url= http://beyondnewsnet.com/20160504-china-medical-shady/|title=「魏則西事件」牽出中國醫療黑幕 |accessdate= 2016年5月4日|publisher=超越新聞網 |work= |language =zh}}</ref>。李彥宏被網信辦約談，同時國家工商總局、廣告監管司、衞生計生委等多個部委組成聯合調查組，進駐百度處理事件。<ref>{{cite web |url= https://hk.news.yahoo.com/新聞點評-百度犯眾怒被清算-200900513.html|title=【新聞點評】百度犯眾怒被清算 |accessdate= 2016年5月4日|publisher=雅虎香港 |work= |language =zh}}</ref>

2017年7月5日，百度召开百度AI开发者大会。李彦宏在会场通过视频直播，展示了自己乘坐百度研发的无人驾驶汽车的情景：其坐在一辆红色的汽车的副驾驶座位上，但驾驶座位没有驾驶员。据《北京青年报》报导称，在视频中，其自称刚刚上五环，正在前往会场的路上，“车处在自动驾驶的状态”。交管部门随后介入对李彦宏的调查。<ref>{{cite web|title=李彦宏乘无人驾驶汽车上北京五环 交管调查|url=http://www.zaobao.com.sg/realtime/china/story20170705-776661|website=联合早报网|language=en|date=2017-07-05}}</ref>
