==生平==
1913年1月21日（农历腊月十五日）生于河北省丰润县[[七树庄镇|七树庄镇]]王豪庄一个农民家庭。1929年秋参加[[中国共产党|中国共产党]]领导的革命<ref name=huaquan>{{Cite web |url = http://news.sina.com.cn/o/2010-01-04/020616872754s.shtml |title = 胡锦涛温家宝送花圈悼念老红军李尔重 |author =  |date = 2010-01-04 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=frdjw/>。6岁入[[私塾|私塾]]学习。14岁升入丰润车轴山中学学习<ref name=guhui/>。在校期间，他喜爱文学和[[书法|书法]]。在[[中国共产党|中国共产党]]党员董志良任校长并在校内讲授[[马列主义|马列主义]]后，李尔重对马列主义产生了浓厚兴趣，积极学习马列主义，并多次到村镇、县城参加演讲，宣传[[俄国十月革命|俄国十月革命]]的胜利，宣传[[孙中山|孙中山]][[新三民主义|新三民主义]]<ref name=frdjw/>。1929年加入[[中国共产主义青年团|中国共产主义青年团]]<ref name=shishi/>。1930年初，参加[[中国共产党|中国共产党]][[薄一波|薄一波]]等领导的[[唐山兵变|唐山兵变]]斗争<ref name=guhui/>，他和另外两位同学主动承担兵变的宣传及联络工作。不久兵变失败，白色恐怖笼罩全校。此后，李尔重为为躲避通缉来到[[北平|北平]]。1931年考入[[南京|南京]][[国立中央大学|国立中央大学]]。1931年末，因与抗日团体联络，遭[[南京国民政府|国民政府]]逮捕，随之被校方开除<ref name=frdjw/>。此后1932年初，他重返北平，考入[[北平大学|北平大学]]农学院<ref name=frdjw/><ref name=zgzjw/>。在南京中央大学期间他写了《唯物的社会观》，宣传[[马克思主义|马克思主义]]的社会观，发表在北平《新社会》杂志第一期上<ref name=frdjw/>。1932年6月加入[[中国共产党|中国共产党]]<ref name=guhui/><ref name=shishi/>。同年他根据坐牢时与一位[[精神病|精神病]]患者同室的经历，写出小小说《飞来的棺材》，发表在《华北日报》，这标志着他的文学创作生涯正式开始。随后，他又在《世界日报》上发表杂文《女人的称呼》。在北平大学农学院，他写了《中国农村经济问题》，发表在陕西省解放日报上。后来有段时间，他在[[山东|山东]]教书，从事过农村调查，写了反映农村斗争生活的小说《鼓》。1935年李尔重又考入[[北京大学|北京大学]]哲学系。同年12月，他参加了[[一二九运动|一二九运动]]<ref name=frdjw/>。在北平期间，因为组织大学生进行抗日活动，他先后两次被[[中国国民党|国民党]]当局投入监狱<ref name=yangsheng/>。后因不满学校对学生的法西斯压迫，他和同学一起留学[[日本|日本]]，入[[仙台|仙台]][[东北大学_(日本)|帝国大学]]法学部社会经济专科学习<ref name=frdjw/><ref name=zgzjw>{{Cite web |url = http://www.chinawriter.com.cn/zxhy/member/2013.shtml|title = 李尔重 (1913～2009) |author =  |date =  |publisher =中国作家网  |language =  |accessdate =2017-11-16  |quote =  }}</ref>。在那里，他学习了《[[资本论|资本论]]》等著作。1936年他从[[英文|英文]]版翻译并出版了{{link-de|弗里德里希·阿尔贝特·朗格|Friedrich Albert Lange|Lange}}著的《[[唯物论|唯物论]]史》。1937年[[七七事变|七七事变]]后，李尔重辍学回国参加[[抗日战争|抗日战争]]<ref name=frdjw/>。

1938年后，历任[[冀南军区|冀南军区]]政治部宣传部长、冀南军区武装部长，五分区地委书记兼政委。曾参加[[百团大战|百团大战]]<ref name=zgzjw/><ref name=shishi/><ref name=huaquan/>。1943年8月30日冀南抗日根据地五分区司令部遭[[日军|日军]]和伪军共2000多人包围，他英勇抗击，率部艰难突围<ref name=zhuisi/>。1945年[[抗日战争|抗日战争]]胜利后，奉派到[[中国东北|中国东北]]，历任东安[[土改|土改]]工作团副团长兼中共[[鸡西|鸡西]]县委书记，中共[[牡丹江省|牡丹江省]]委民运部长，铁道兵团政治部宣传部长等职。曾先后参加[[辽沈战役|辽沈战役]]、[[平津战役|平津战役]]，为部队架桥铺路<ref name=zgzjw/><ref name=shishi/><ref name=huaquan/><ref name=zhuisi/>。在东北曾参加剿匪反霸的斗争<ref name=zhuisi/>。1947年铁道兵团成立，李尔重任党组成员、政治部宣传部长。辽沈战役开始前，铁道兵团冒着国民党飞机的轰炸，将[[中国人民解放军第四野战军|中国人民解放军第四野战军]]主力部队、装备及补给物资运到[[锦州|锦州]]外围，为3天攻下锦州抢得了先机。国民党军撤出[[唐山|唐山]]时，破坏了唐山至[[天津|天津]]的铁路，李尔重率铁道兵团一个营到唐山，与工人们一道在24小时内恢复铁路，为部队争取了时间<ref name=tuwen>{{Cite web |url =   http://news.163.com/09/1231/07/5RRJGCHR000120GR.html |title = 图文：“荷戈老兵”的传奇人生|author =  |date =2009-12-31  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

[[中华人民共和国|中华人民共和国]]成立后，历任中共[[武汉市|武汉市]]委常委、宣传部长、市委书记处书记、第三书记、第二书记，[[中共中央中南局|中共中央中南局]]常委、宣传部长、农委主任，广东省农委副主任、主任，[[中共广东省委|中共广东省委]]常委、[[广东省革命委员会|广东省革命委员会]]副主任，中共[[海南行政区|海南行政区]]党委书记，[[中共陕西省委|中共陕西省委]]常务书记，[[中共河北省委|中共河北省委]]书记兼省长，中共湖北省顾问委员会副主任<ref name=zgzjw/><ref name=shishi>{{Cite web |url =  http://news.sohu.com/20100205/n270092603.shtml|title = 河北省原省委书记、省长李尔重同志逝世 |author =  |date = 2010-02-05 |publisher =搜狐 |language =  |accessdate =  |quote =  }}</ref><ref name=huaquan/>。1954年曾参与指挥武汉抗击[[1954年长江流域特大洪水|特大洪水]]<ref name=zhuisi/>。他是[[中共八大|中共八大]]、[[中共十二大|中共十二大]]候补代表，[[第五届全国人大代表|第五届全国人大代表]]<ref name=shishi/>。

晚年他还曾任[[武汉大学|武汉大学]]中文系教授、[[华中师范大学|华中师范大学]]中文系教授，[[华中理工大学|华中理工大学]]文学院名誉院长，[[湖北经济管理大学|湖北经济管理大学]]名誉校长，[[中国书法家协会|中国书法家协会]]理事、名誉理事，湖北书法家协会名誉主席，中国诗词学会顾问，中国群众文化学会名誉理事，《中国文学》编委会顾问，武汉诗词学会名誉会长，湖北省屈原研究会主任等社会职务<ref name=zgzjw/><ref name=wuyouaidao>{{Cite web |url = http://www.wyzxwk.com/Article/lixiang/2010/12/116988.html |title =沉痛哀悼李尔重同志  |author =  |date = 2009-12-30 |publisher = 乌有之乡网刊 |language =  |accessdate =  |quote =  }}</ref>。

李尔重一生在中国共产党内遇到四次重大政治波折。第一次是[[太行山|太行山]]“[[抢救运动|抢救运动]]”，1943年底至1944年初，李尔重在太行山[[中共中央北方局|中共中央北方局]]党校赶上“抢救运动”，因有三次被国民党当局抓捕的经历，并曾在[[日本|日本]]留学，他差点被以“叛徒”、“特务”、“汉奸”的罪名枪毙。幸而毛泽东“一个不杀，大部不抓”的指示到了太行，他才幸存。后到[[延安|延安]][[中央党校|中央党校]]甄别时，基本查清了[[北平|北平]]两次入狱的情况，[[南京|南京]]一事留下尾巴，因为中共开辟东北根据地急需干部，所以让他带着尾巴去了东北。中华人民共和国成立初期，组织上又将南京入狱一事审查清楚<ref name=bijixu/>。

第二次是武汉市“反右倾宗派”。1959年[[庐山会议|庐山会议]]后，1959年底至1960年初，武汉市开展了一场“反右倾宗派主义”的斗争，受批判的主要有李尔重（时任中共武汉市委书记处书记）、[[杨青_(湖北省纪委书记)|杨青]]（时任中共武汉市委书记处书记）、[[赵长春|赵长春]]（时任中共武汉市委组织部长）、[[纪辉|纪辉]]（时任中共武汉市监察委员会书记）。因各种原因，中共武汉市委未能给这场斗争形成一个比较极端的结论，但斗争发生后不久，李尔重、杨青、赵长春、纪辉先后离开中共武汉市委。两年后在1962年1月至2月间召开的[[七千人大会|七千人大会]]上，有人将此事捅出，要求甄别平反。会后，中共武汉市委将上述四人请回武汉，认错道歉<ref name=bijixu/><ref name=zhuisi/>。

第三次是[[文化大革命|文化大革命]]冲击。1966年11月起，有人又将李尔重的历史问题扯出来<ref name=bijixu/><ref name=zhuisi/>。1966年11月，李尔重家住的大院院墙上贴出很多[[大字报|大字报]]，说李尔重是“叛徒”、“特务”、“汉奸”。此后李尔重曾遭到批斗和殴打<ref name=zhuisi/>。1968年1月20日，李尔重被广州警备司令部带走，辗转搬迁十次，最后关进粤北乐昌监狱<ref name=bijixu/><ref name=zhuisi/>。他在狱中经受磨难，但仍坚持读《[[毛泽东选集|毛泽东选集]]》以及《[[资本论|资本论]]》<ref name=yangsheng/>。李尔重的妻子宋晓非成了“黑婆”，带着最小的孩子接受批斗，三个大些的孩子被下放[[海南岛|海南岛]]劳动<ref name=tuwen/>。直到1972年4月底，李尔重才从监狱放出，到位于[[连山县|连山县]]上草的[[五七干校|五七干校]]劳动，该干校是为原中共中央中南局干部设立。1972年12月重新回到工作岗位<ref name=bijixu/><ref name=zhuisi/>。

第四次是河北争论。[[文革|文革]]结束后，河北省的形势异常复杂，两派斗争严重。文革期间，[[林彪、江青反革命集团|林彪、江青反革命集团]]一些重要成员曾直接插手河北，“左”的影响严重。[[中共十一届三中全会|中共十一届三中全会]]后，中央的重要的方针政策在河北省很难贯彻，河北省在拨乱反正、落实政策等重大问题上进展缓慢，而进京[[上访|上访]]人数却居全国第一位。中央先后多次做工作无效，于是陆续派出多位与河北两派素无瓜葛的老同志接任河北省党政主要领导职务<ref name=yuan/>。1979年12月26日，李尔重奉命与[[金明_(中国政府官员)|金明]]一起到河北任职。到河北仅三四个月，就开始与此前已到河北的中共河北省委第二书记[[江一真|江一真]]在一系列问题上发生分歧，争论不断<ref name=bijixu/>。中央书记处全体成员在不到三年的时间里两次出面召开中共河北省委扩大会议，解决河北问题<ref name=zhuisi/>。其中一次是1981年7月9日至15日，[[中共十一届六中全会|中共十一届六中全会]]新选出的[[中共中央书记处|中共中央书记处]]全体成员压阵，在北京召开中共河北省委扩大会议，短时间平息了纷争。1982年6月16日，中央决定将金明（第一书记）、江一真（第二书记）、李尔重（书记兼省长）全部调离河北<ref name=bijixu> {{Cite book | author = 李尔重著 全胜辑录 | title = 李尔重笔记摘抄（第一、二册） | location =  | publisher = 中国展望出版社  | date = 2014年 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote = 序言 }} </ref>。随后由[[高扬|高扬]]任第一书记，以扭转河北局面<ref name=yuan>袁厚春，省委第一书记，昆仑1984年第4期</ref>。

2003年10月，李尔重出席了母校车轴山中学百年校庆。2000年代他曾先后两次向车轴山中学捐赠图书8700余册。他还曾向[[中国国家图书馆|中国国家图书馆]]赠书<ref name=guhui/><ref>段庚，李尔重向母校图书馆赠书，中小学图书情报世界2001(2)</ref>。

晚年他曾多次上书中共中央，呼吁勿走[[资本主义|资本主义]]和[[修正主义|修正主义]]道路。他坚决反对“[[三个代表|三个代表]]”、反对《[[中华人民共和国物权法|中华人民共和国物权法]]》，他与另外两位中共元老[[马宾|马宾]]、[[魏巍|魏巍]]被人合称“反修三剑客”<ref name=qishou>{{cite web|title=中国「左派旗手」李尔重病逝 曾提倡反腐爱工农|url=http://www.mychinamyhome.com/?viewnews-137180.html|publisher=家中国|accessdate=2011-12-25}}</ref>。2005年7月23日，李尔重专门就清除党内腐败问题向[[中共中央|中共中央]]及总书记[[胡锦涛|胡锦涛]]上书，称“法不能自立，类不能独行，得其人则存，失其人则忘。”“党不敢向自己开刀，不敢承受割癌的切肤之痛，便只有承受灭亡之苦。”“坚持社会主义的共产党正，则天下正。”2006年，李尔重在[[互联网|互联网]]上发表《要[[社会主义|社会主义]]（[[共产主义|共产主义]]），不要[[资本主义|资本主义]]——纪念[[毛泽东|毛主席]]逝世30周年》一文<ref name=fumu/>。2007年春，[[谢韬|谢韬]]在《[[炎黄春秋|炎黄春秋]]》第二期发表了《民主社会主义模式与中国前途》一文称，“只有[[民主社会主义|民主社会主义]]才能救中国”。李尔重随即发表《论社会主义本质——驳民主社会主义谬论》等文章反击<ref>{{Cite web |url =  http://www.wyzxwk.com/Article/cpers/2010/01/117489.html  |title = 奚兆永：深切悼念李尔重同志 |author =  |date = 2010-01-02  |publisher =乌有之乡网刊  |language =  |accessdate =  |quote =  }}</ref>。

2009年12月26日，李尔重在[[武汉|武汉]]病逝，享年96岁<ref name=shishi/>。李尔重逝世后，[[党和国家领导人|党和国家领导人]][[胡锦涛|胡锦涛]]、[[温家宝|温家宝]]、[[李长春|李长春]]、[[习近平|习近平]]、[[李克强|李克强]]、[[贺国强|贺国强]]、[[周永康|周永康]]等分别以不同方式对李尔重逝世表示深切哀悼并送花圈，对李尔重夫人及家属表示慰问。2010年1月3日，李尔重同志遗体送别仪式在[[武昌殡仪馆|武昌殡仪馆]]天元厅举行，[[中共湖北省委|中共湖北省委]]书记[[罗清泉|罗清泉]]致悼词，官方称他是“中国共产党的优秀党员，忠诚的共产主义战士、老红军，我党优秀的党务工作者”。根据李尔重遗嘱，他的骨灰将撒在[[长江|长江]]和家乡[[河北省|河北省]][[唐山市|唐山市]][[丰润区|丰润区]]车轴山中学山地上<ref name=huaquan/><ref>{{cite web|title=罗清泉李鸿忠宋育英出席李尔重遗体送别仪式|url=http://www.hb.xinhuanet.com/ldbd/2010-01/04/content_18670296.htm|publisher=新华网|date=2010-01-04|deadurl=yes|archiveurl=https://web.archive.org/web/20150930070224/http://www.hb.xinhuanet.com/ldbd/2010-01/04/content_18670296.htm|archivedate=2015-09-30}}</ref>。2010年1月6日，在车轴山中学山地上举行李尔重同志骨灰撒播仪式<ref name=guhui>{{Cite web |url = http://tangshan.huanbohainews.com.cn/system/2010/01/07/010571365.shtml |title = 李尔重同志骨灰撒播仪式在丰润区车轴山举行 |author =  |date = 2010-01-07 |publisher = 环渤海新闻网 |language =  |accessdate =  |quote =  }}</ref>。

==著作==
李尔重早在[[九一八事变|九一八事变]]之后的1932年起就发表过若干篇小说及散文、杂文，但大多已散失<ref name=wenxue/><ref name=zgzjw/>。1952年加入[[中国作家协会|中国作家协会]]<ref name=zgzjw/>。1950年代，在担任武汉市领导职务期间，李尔重挤出时间写了不少文学作品，包括小说和剧本，如话剧《扬子江边》（1956年获[[中华人民共和国文化部|文化部]]演出一等奖）、小说《战洪水》等。其中《武汉关的钟声》曾被收录在中学语文课本中<ref name=wenxue/>。年过古稀后，创作了描写[[抗日战争|抗日战争]]的长篇历史小说《新战争与和平》，由原国家主席[[李先念|李先念]]作序，武汉出版社1993年出版，获第五届国家图书奖二等奖（一等奖空缺）、1992年[[上海|上海]]《[[解放日报|解放日报]]》霞飞奖、[[湖北省|湖北省]][[屈原|屈原]]文艺创作奖、中共武汉市委、武汉市人民政府黄鹤文化奖<ref name=wenxue/><ref name=zgzjw/>。1980年代初，本来中央安排他到[[全国人大常委会|全国人大常委会]]做领导工作，但他为创作长篇小说《新战争与和平》，主动申请回到武汉专事文学创作。回到武汉的李尔重积极支持从[[文革|文革]]重创中复苏的人文学科学术研究。此后三十年，他对湖北本地多个人文学科学术组织都给予过切实帮助。2005年，李尔重将多种著作手稿捐赠给[[中国现代文学馆|中国现代文学馆]]<ref name=wenxue/><ref name=yangsheng/>。李尔重的著作有上千万字。另有近300本合计上千万字的笔记，始于1946年7月24日，止于2008年11月22日，其中1946年7月到1982年7月期间，除1966年10月到1972年12月文化大革命受冲击阶段，基本都是工作笔记，1982年7月以后主要是读书笔记<ref name=bijixu/>。1950年代，李尔重曾多次陪同[[毛泽东|毛泽东]]游长江，获毛泽东器重，将他与中共湖北省委第一书记[[王任重|王任重]]并称为“湖北二重”，李尔重被毛泽东誉为“我们的作家和才子”<ref name=huaquan/><ref name=qishou/>。

李尔重自幼喜爱书法，学书曾临写《[[圣教序|圣教序]]》、《[[玄秘塔碑|玄秘塔碑]]》等，旁及[[颜真卿|颜真卿]]、[[欧阳询|欧阳询]]、[[怀素|怀素]]诸家。喜作[[楷书|楷书]]和[[行草|行草]]。作品多次入选全国和国际性展览，被收入《中国新文艺大系书法集》、《中国现代书法选》（第二集）、《全国第一届书法篆刻展览作品集》、《全国第三届书法篆刻展览作品集》、《当代名家书法作品选》、《名人入琼墨迹选》、《牡丹诗词书法选萃》、《河北书法篆刻选》等，为[[陕西省博物馆|陕西省博物馆]]、河北[[邯郸市博物馆|邯郸市博物馆]]以及多处风景名胜区收藏或被刻成碑刻<ref name=wuyouaidao/>。
