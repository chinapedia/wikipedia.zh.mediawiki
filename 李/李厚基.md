'''李厚基'''（{{bd|1869年||1942年|9月}}） ，字'''培之'''，祖籍[[江苏省|江苏省]][[徐州府|徐州府]][[铜山县_(清朝)|铜山县]]大王楼村（今属[[丰县|丰县]][[梁寨镇|梁寨镇]]）。[[清朝|清朝]]及[[中华民国|中华民国]]军事将领、政治人物。<ref name=zhang>张绍祖、张建虹，寓居津门的福建督军李厚基，载 李正中主编，近代天津名人故居，天津人民出版社，2009年</ref><ref name=lai>{{Cite book<!--|和書-->|author = 来新夏等|title = 北洋軍閥史 下冊|year = 2000|publisher = [[南開大学|南開大学]]出版社|isbn = 7-310-01517-7}}</ref><ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>北洋政府时期全威将军、建武将军，是[[李鸿章|李鸿章]]直隶总督署卫队管带、北洋[[陆军第四镇|陆军第四镇]]第七协协统、[[中華民國國民革命軍|中华民国国民革命军]]第四师第七旅旅长、福建镇守使和护军使、福建督军兼巡抚，掌管福建省军政大权。

==生平==
李厚基的父亲叫李忠纯。他家世代军人，其同族世祖是清代雍乾年间的重臣[[李卫|李卫]]。<ref name=zhang/>李厚基自幼随父生活在军营。1889年（[[光緒|光緒]]15年）任[[李鸿章|李鸿章]]的卫队队官。1890年（光緒16年）入天津[[北洋武备学堂|北洋武备学堂]]，同年毕业。1891年（光绪17年）任[[李鸿章|李鸿章]]的[[直隶总督署|直隶总督署]]卫队营管带，再度成为李鸿章的手下。<ref name=lai/><ref name=xu/>1896年3月，在管带任内，作为钦差头等出使大臣李鸿章的随员，参加了俄国[[沙皇|沙皇]][[尼古拉二世|尼古拉二世]]的加冕典礼，其后随李鸿章周游[[德国|德国]]、[[荷兰|荷兰]]、[[比利时|比利时]]、[[法国|法国]]、[[英国|英国]]、[[美国|美国]]。1904年，为[[北洋常备军第二镇|北洋常备军第二镇]]第三协第五标第二营管带。<ref name=zhang/>

1909年（[[宣統|宣統]]元年），升任[[陸軍第四鎮|陸軍第四鎮]]第7協第14標標統，跟随陆军第四鎮統制[[吳鳳嶺|吳鳳嶺]]，赴[[津南区|小站]]练兵。1911年（宣統3年）[[武昌起義|武昌起義]]爆发后，升任[[陸軍第四鎮|陸軍第四鎮]]第7協協統。<ref name=lai/><ref name=xu/>

[[中華民国|中華民国]]成立後，随着軍隊改組，李厚基任陸軍第4師第7旅旅長，驻[[马厂|马厂]]。1913年（[[民国紀元|民国]]2年）7月[[二次革命|二次革命]]期间，率部赴[[上海市|上海]]，在[[吴淞区|吴淞]]地区作战。二次革命被鎮圧後，任吴淞要塞司令。<ref name=lai/><ref name=xu/>

1913年11月，李厚基率部入[[福建省_(中華民国)|福建省]]，12月被任命为福建鎮守使。另外，同一时期福建都督一职被废止，福建鎮守使李厚基承担了都督的職责。1914年（民国3年）7月，改任福建護軍使，事实上和各省将軍同等地位。1915年（民国4年）12月、[[袁世凱称帝|袁世凱称帝]]之際，李厚基获封一等子爵。1916年（民国5年）3月，被任命为福建将軍，4月兼任福建巡按使。袁死後的同年7月，任福建督軍兼省長。<ref name=liu>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref><ref name=lai/><ref name=xu/>

1918年（民国7年）4月，奉[[段祺瑞|段祺瑞]]之命，李厚基被任命为援粤軍正总司令，副总司令为浙軍師長[[童葆暄|童葆暄]]，该军进攻[[广東省_(中華民国)|广東]][[护法軍政府|护法軍政府]]，但敗北。其後，李厚基继续统治福建省，直到1922年（民国11年）11月被南方政府方面的[[許崇智|許崇智]]駆逐。李厚基统治福建9年至此结束，后于1923年（民国12年）1月被正式免职。<ref name=liu/><ref name=lai/><ref name=xu/>

1924年（民国13年）11月，北京政府任命其为[[山西省_(中華民国)|山西]]援軍副司令，但他随即引退。此后，他迁居[[天津日租界|天津日租界]]，经营木材生意。<ref name=lai/><ref name=xu/>

1942年（民国31年）9月，李厚基逝世。享年74岁。<ref name=lai/><ref name=xu/>

==家庭==
李厚基与甘肃督军[[陆洪涛|陆洪涛]]是儿女亲家，陆洪涛之女嫁给李厚基之子李传勋为妻。<ref name=zhang/>
