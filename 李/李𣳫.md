'''昭顯世子'''（{{lang|ko|소현세자}}，{{bd|1612年|正月初四|1645年|4月26日|catidx=L李𣳫}}），是[[朝鮮王朝|朝鮮王朝]]第十六代國王[[朝鮮仁祖|仁祖]]的嫡長子，[[朝鮮孝宗|孝宗]]兄。[[仁烈王后|仁烈王后]]韓氏所生，生于朝鲜汉城会贤坊。名'''𣳫'''（{{lang|ko|왕}}，讀作“汪”）。1625年封为王世子。1627年[[丁卯战争|丁卯战争]]，被派往全州安抚民心，回京后结婚，妻子是西人派姜硕期之女[[愍懷嬪|愍懷嬪]]（{{lang|ko|민회빈}}）姜氏，十二月行嘉礼于太平馆。[[崇祯|崇祯]]七年（1634年，仁祖十二年），明朝派遣了最后一次派往朝鲜的使团——[[太监|太监]][[卢维宁|卢维宁]]一行出使朝鲜，册封他为世子。1637年被掳往清朝。仁祖23年（1645年）回国，当年4月26日病逝於[[昌慶宮|昌慶宮]]歡慶堂，墓於[[昭慶園|昭慶園]]。

== 被扣清朝 ==
[[丙子胡亂|丙子胡亂]]中，1637年初，清朝要求朝鲜世子为人质<ref>朝鲜仁祖实录 卷33 十四年十二月十六日</ref>，仁祖投降，世子“号泣之声，闻于户外”<ref>朝鲜仁祖实录 卷34 十五年正月十八</ref>。很多大臣反对世子为人质，但世子表示：“泰山既垂于鸟卵之上，国步谁措于盘石之坚，事已急矣！予既有弟二人，又有一子，亦可奉宗社。予虽死于贼，尚何憾焉？”<ref>朝鲜仁祖实录 卷34，十五年正月二十二</ref>仁祖起初不愿意世子出城，又萌生了世子代他出城投降的幻想，但被清朝拒绝。仁祖投降後世子与其妻和凤林大君夫妇被清朝抓往[[瀋陽|瀋陽]]抑留。世子在-{zh-hans:沈馆;zh-hant:瀋館}-中受到清人严密监视，朝鲜利用各种机会请求归还世子，但无果而终。皇太极要求世子和凤林大君参与[[松锦大战|松锦大战]]，还借世子之口要求朝鲜军参与对明作战。一度有明军炮弹落在世子营旁。在沈阳世子主动对清进行各种交涉，包括遣返朝鲜境内[[女真人|女真人]]、[[汉人|汉人]]问题，朝鲜人越境採蔘问题，本国被掳人口问题，力保“通明”朝鲜大臣等。每当朝清关系不睦，清朝就责难世子，世子力争抗辩，维护本国利益，维护清朝和朝鲜和平关系。

== 朝鲜上层对世子失望 ==
朝鲜人对世子失望“世子既久处沈阳，一听清人所为，出入于田猎、戎马之间，所亲狎者皆武夫、厮卒。专废讲学，惟事货利，且以土木之役、狗马之玩为事，贻讥敌国，大失人望”<ref>《[[:s:朝鲜王朝实录|朝鲜王朝实录]]·仁祖实录》卷46，二十三年四月二十六日</ref>。大臣[[金荩国|金荩国]]看到“世子颇聚奇玩，罕开书筵”，便说：“今被留异域，厄困极矣，愿数接宫僚，讨论经籍，毋徒为暇豫也。”后来看到世子“营别院”，他又劝谏：“此何如地，而乃兴土木之役，而为苟安之意乎？” 世子回国归省2次都是举国百姓夹道迎候，甚至还有人瞻拜落泪。可见世子赢得了民心。两次回国都由其长子接替他做人质。[[崇德|崇德]]五年（1640年，仁祖十八年）第一次归国时，仁祖“泣而抚之”。<ref>《朝鲜王朝实录·仁祖实录》卷40，十八年三月初七日</ref>[[清世祖|清世祖]]即位后，对昭显世子极为优待，导致昭显世子的反清意识减弱。仁祖询问大臣：“今闻九王年少刚愎，其意何可测也？前则待世子太薄，而今乃太厚云，予不能无疑焉。”<ref>《朝鲜王朝实录·仁祖实录》卷44，二十一年十月十一日</ref>[[顺治|顺治]]元年（1644年，仁祖二十二年）世子第二次回国，仁祖对世子态度转变，沈器远秘谋推戴世子为王。<ref>《朝鲜王朝实录·仁祖实录》卷45，二十二年三月二十一日条</ref>世子返回沈阳时，路经平壤，为当地儒生举行科举，改原题“白马朝周”为“越鸟巢南枝”，“盖自伤去故国而就异域也”。这次考试因未经仁祖许可被拒绝承认。<ref>《朝鲜王朝实录·仁祖实录》卷45，二十二年八月二十九日条</ref>仁祖认为世子在沈阳发展自己的势力：“世子在沈阳时，作室涂以丹雘，又募东人之被俘者，屯田积粟，贸换异物，馆门如市，上闻之不平”。

== 亲清和接触天主教 ==
在朝鲜时，他是坚定的反清派，在沈阳时仍有复仇之志，曾以《思古剑客》为题赋诗，一侍从提醒他：“不密则害，生虏势方盛，岂一剑客所可图也？宜励志以俟时。”世子听到后颇为醒悟，连连称善。<ref>李源益《纪年东史约》卷22，第81页</ref>但他逐渐成长为成熟务实的政治家，要壮大朝鲜的实力。[[清军入关|清军入关]]后他九月到十一月在北京紫禁城[[文渊阁|文渊阁]]居住了70天，与[[耶稣会|耶稣会]][[传教士|传教士]][[汤若望|汤若望]]会面，了解[[西方|西方]][[科学|科学]][[技术|技术]]、[[天主教|天主教]]，汤若望送他[[天文|天文]]、[[数学|数学]]、天主教的书籍及[[地球仪|地球仪]]、天主像等，世子致汤若望的书函原文不存，法文翻译本保存在教会图书馆中，他写道：“曾不知世上还有天球仪及书籍类之物，今获此物，喜悦之情实难言表。敝国虽非无与此物相似之物，然数百年来与天体运行不相符合，而不容疑为伪物。今获此珍品，不胜欣喜。余归故国后不但将使用于宫中，更打算将之付梓并颁行于世。”甚至提出要开放国门。昭显世子回国带到朝鲜的前明[[宦官|宦官]]和[[宫女|宫女]]，均为已[[受洗|受洗]]的天主教徒，是汤若望同昭显世子商量以后决定以清朝皇帝下赐的名义随行的。<ref>山口正之〈昭显世子と汤若望—朝鲜基督教史研究 其四—〉，《青丘学丛》第5号，1931年</ref>。 

世子回到国内后不时在言语中流露出亲近清朝的思想，仁祖对此产生疑心，并对此十分恼怒。昭显世子被扣为人质，既是清朝钳制朝鲜的手段，也是清朝要挟朝鲜攻打明朝等行动的筹码，同时又含有清朝扶植朝鲜亲清势力的意图。

== 突然死亡 ==
[[顺治|顺治]]元年（1644年，仁祖二十二年）十一月十一日，清朝[[摄政王|摄政王]][[多尔衮|多尔衮]]在北京[[紫禁城|紫禁城]][[武英殿|武英殿]]召见昭显世子和凤林大君，宣布：“未得[[北京|北京]]之前，两国不无疑阻，今则大事已定，彼此一以诚信相孚。且世子以东国储君，不可久居于此，今宜永还本国。”<ref>《朝鲜王朝实录·仁祖实录》卷45，二十二年十二月初四日</ref>凤林大君则暂时留在北京，等[[麟坪大君|麟坪大君]]接替他做人质。世子和妻子在清人的护送下回国。仁祖听说后不但不高兴，还说：“清国此举，果出于好意而无别情耶？”<ref>《朝鲜王朝实录·仁祖实录》卷45，二十二年十二月初六日</ref>顺治二年（1645年，仁祖二十三年）正月初九日，世子夫妻到达沈阳，世子常年得病，即将回国时，朝鲜就开始准备他的治疗<ref>《承政院日记》仁祖二十三年正月初十日：“御医臣朴頵以世子症候议药事出去。”</ref>。二月十八日世子回到汉城。随行清使勒令朝鲜仁祖李倧出城迎接。父迎子不合[[儒教|儒教]]礼法，且承认清朝敕使代表天子，所以朝鲜[[两班|两班]][[士大夫|士大夫]]坚决反对。仁祖以生病为由拒不出迎。<ref>《朝鲜王朝实录·仁祖实录》卷46，二十三年二月十八日</ref>农历四月二十三日世子生病，被诊断为[[疟疾|疟疾]]，医官李馨益[[针灸|针灸]]治疗，三天后（公历5月21日）世子暴毙[[昌庆宫|昌庆宫]]欢庆殿，“举体尽黑，七窍皆出鲜血”<ref>《朝鲜王朝实录·仁祖实录》卷46，二十三年六月二十七日</ref>。李馨益受到怀疑，仁祖未下令澄清死因，而是对到场人员加以限制，并下令赶紧将世子入棺。仁祖的宠姬赵昭容（废贵人赵氏）与昭显世子夫妻有隙，经常构陷他们，李馨益就是昭容赵氏的[[外戚|外戚]]，在昭显世子回国前夕被特别叙用为医官。<ref>《承政院日记》，仁祖二十三年正月初四日</ref>大提学李植撰写昭显世子墓志“世子久留异域，数从军旅，东猎朔荒，西穿燕塞，跋履山川，备经危险，虽神气自若，而内受劳伤。还宫以后，连有寒热之感，医方错误，竟至弗禄。”<ref>《朝鲜王朝实录·仁祖实录》卷46，二十三年六月初十日</ref>韩国学者金宗德认为世子有结胸症和阴虚恶热、寒热往来的症状，却被误诊为疟疾，导致死亡。<ref>金宗德〈对昭显世子病症和治疗的研究〉，《奎章阁》第31辑，2007年</ref>
== 死后 ==
根据《昭显世子墓所都监仪轨》，昭显世子的葬礼显然被减杀，规格低于后世思悼世子的葬礼，而且朝鲜李朝安葬世子的墓地一般被称为园所，而昭显世子墓地被称为墓所。在昭显世子死后不久，洪瑞凤、李敬舆、沈悦等大臣曾建议册封昭显世子的长子李栢为世孙，但仁祖以长孙只有10岁的理由加以回绝，并册立[[朝鲜孝宗|凤林大君]]为世子。<ref>[http://sillok.history.go.kr/inspection/insp_king.jsp?id=wpa_12306102_001&tabid=w 《仁祖大王實錄》46卷,仁祖23年(1645年)閏6月2日(壬午)1번째기사]</ref>昭显世子嫔姜氏不久之后即被指控“在沈阳居住时以中宫自居”<ref>[http://sillok.history.go.kr/inspection/insp_king.jsp?id=wpa_12402003_002&tabid=w 《仁祖大王實錄》47卷,仁祖24年(1646年)2月3日(庚辰)2번째기사]</ref>，及“向仁祖食用的鲍鱼中下毒”<ref>[http://sillok.history.go.kr/inspection/insp_king.jsp?id=wpa_12401003_003&tabid=w 《仁祖大王實錄》47卷,仁祖24年(1646年)1月3日(辛亥)3번째기사]</ref>，而被赐死<ref>[http://sillok.history.go.kr/inspection/insp_king.jsp?id=wpa_12403015_002&tabid=w 《仁祖大王實錄》47卷,仁祖24年(1646年)3月15日(壬戌)2번째기사]</ref>，昭显世子的三个儿子也被流放[[济州岛|济州岛]]。<ref>[http://sillok.history.go.kr/inspection/insp_king.jsp?id=wpa_12505013_001&tabid=w 《仁祖大王實錄》48卷,仁祖25年(1647年)5月13日(癸丑)1번째기사]</ref>

世子带来的西洋文物没有在朝鲜发挥影响。从中国所带来的所有物品都在其死后被销毁，而信教前明宦官李邦詔、張三畏、劉仲林、谷豊登、竇文芳和前明宫女也被遣返回中国。<ref>《朝鲜王朝实录·仁祖实录》卷46，二十三年七月二十二日</ref>

===後裔===
[[朝鮮英祖|英祖]]四年三月二十（1728年4月28日），[[少論派|少論派]]人{{link-ko|李麟佐|이인좌}}、[[鄭希亮|鄭希亮]]發動{{link-ko|戊申政變|이인좌의 난}}（又稱李麟佐之亂），擁立昭顯世子的曾孫[[李坦_(朝鲜)|密豐君]]為[[朝鲜王朝君主列表|朝鮮國王]]，但僅四日就以失敗告終，在次年（1729年）三月密豐君被賜死<ref>《[[李朝實錄|英祖實錄]]》16卷·四年三月二十條</ref>。

密豐君次子[[李糼|李晉錫]]被過繼為[[朝鮮肅宗|朝鮮肅宗]]六子延齡君[[李昍|李昍]]的嗣子，後因為早逝沒有後代被取消收養關係；密豐君七世孫{{link-ko|豐善君|풍선군|李漢鎔}}出繼為{{link-ko|清安君|청안군 (1851년)}}嗣子、[[永平君|永平君]]的繼孫。<ref name="Soh">{{cite web |title=璇源續譜七 |url=http://visualjoseon.aks.ac.kr/viewer/view?itemId=jb&gubun=pa&dataId=K24725_007%5E%EC%86%8C%ED%98%84%EC%84%B8%EC%9E%90%ED%8C%8C%5E%EC%86%8C%ED%98%84%EC%84%B8%EC%9E%90%ED%8C%8C}}</ref>
