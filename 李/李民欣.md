'''李民欣'''（{{bd|1890年||1955年|}}）[[广东|广东]][[番禺区|番禺]]人，[[中华民国|中华民国]]及[[中华人民共和国|中华人民共和国]]政治人物<ref name=hxjw/>。

==生平==
1910年，李民欣自陆军速成学校步兵科毕业。历任广东陆军第二师参谋长，[[广东省|广东省]][[四会县|四会县]]县长<ref name=hxjw/>。1926年秋，四会县县长李民欣组织县、区团务委员会，向农民勒收民团费等苛捐杂税，并购置枪支以镇压农会，遭到[[中国国民党|中国国民党]]中央农民部特派员、[[中共|中共]]四会特支书记[[陈伯忠|陈伯忠]]抵制，10月陈伯忠遇害<ref>{{Cite web |url = http://www.xjrb.com/2016/0623/307145.shtml |title = 四会农民运动开拓者——陈伯忠 |author =  |date = 2016-06-23 |publisher =西江网  |language =  |accessdate =  |quote =  }}</ref>。李民欣后历任[[国民革命军|国民革命军]]第八路总指挥部副官长，[[国民政府军事委员会|国民政府军事委员会]]高级参谋、[[桂林|桂林]]办公厅顾问。1946年，参与发起成立[[中国国民党民主促进会|中国国民党民主促进会]]，任中央常务理事。1948年，任[[中国国民党革命委员会|中国国民党革命委员会]]中央常委。1949年，出席[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]。[[中华人民共和国|中华人民共和国]]成立后，历任[[政务院财政经济委员会|政务院财政经济委员会]]委员，[[广东省人民政府|广东省人民政府]]委员，中国国民党革命委员会第二届中央委员、广州市委第一届主任委员，[[广州市|广州市]]人民政府副市长。是[[第二届全国政协委员|第二届全国政协委员]]<ref name=hxjw/>。

1955年，李民欣逝世<ref name=hxjw>{{Cite web |url =  http://www.huaxia.com/zt/sw/07-013/586153.html|title =李民欣(1890？1955)  |author =  |date = 2001-06-15 |publisher = 华夏经纬网 |language =  |accessdate =  |quote =  }}</ref>。
