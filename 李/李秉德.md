'''李秉德'''（1912年7月—2005年5月2日），[[河南|河南]][[洛阳|洛阳]]人，中国[[教育家|教育家]]。曾任[[西北师范学院|西北师范学院]]教授、博士生导师、院长，为[[国务院学位委员会|国务院学位委员会]]授予的首批[[博导|博导]]。1996年获评全国优秀教师，1998年成为政府特殊津贴专家。<ref name="介绍">{{cite web|url=http://henan.sina.com.cn/city/gdyw/2012-07-11/175-19177.html | language= | title=李秉德 |publisher=大河报 |date=2012年7月11日 |accessdate= }}</ref>

== 简介 ==

1934年毕业于[[河南大学|河南大学]][[教育|教育]]专业，1936年攻读[[燕京大学|燕京大学]]乡村教育研究生。曾任湖北教育厅督学、河南大学教育系副教授、河南大学图书馆馆长。1947年赴[[欧洲|欧洲]][[日内瓦大学|日内瓦大学]][[让-雅克·卢梭|卢梭]]学院学习。1950年到西北师范学院任教。 

他长期“致力于课程与教学论的研究，是新中国教学论、教育科学研究方法、小学语文教育等学科领域的开拓者和奠基人之一。”<ref name="介绍"/>

== 头衔 ==
*甘肃教育学会会长
*中国小学语文教学研究会副会长
*中国教育学会、中国高等教育学会第一届理事
*第六、七届[[全国政协委员|全国政协委员]]

== 著作 ==
主要著作有
*《小学语文教学方法》
*《教育科学研究方法》
*《教学论》
*《我国社会经济和科技发展战略问题》
*《李秉德教育文选》
