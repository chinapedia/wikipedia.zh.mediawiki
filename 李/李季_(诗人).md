'''李季'''（{{bd|1922年|8月16日|1980年|3月8日}}）原名'''李振鹏'''，笔名'''里计'''、'''于一帆'''等，[[河南|河南]][[唐河|唐河]]人，[[中华人民共和国|中华人民共和国]]诗人、作家，[[中国作家协会|中国作家协会]]副主席<ref name=zjw>{{Cite web |url = http://www.chinawriter.com.cn/zxhy/member/2117.shtml |title = 李季 (1922～1980) |author =  |date =  |publisher = 中国作家网 |language =  |accessdate =2018-01-04  |quote =  }}</ref><ref name=er>{{Cite web |url =  http://www.hnwriter.com/WriterInfo.aspx?wid=111|title = 李季 |author =  |date =  |publisher = 中国作家网 |language =  |accessdate =2018-01-04  |quote =  }}</ref>。

==生平==
李季生在唐河县一个农民家庭。初中肄业。1938年到[[陕北|陕北]][[抗日军政大学|抗日军政大学]]学习。同年11月加入[[中国共产党|中国共产党]]。历任连指导员，党校支书，小学教师，报社社长，[[延安|延安]]《群众日报》编辑，中南文联编辑出版部长，[[中共中央中南局|中共中央中南局]]宣传部文艺处副处长，《长江文艺》主编，[[玉门油矿|玉门油矿]]党委宣传部长，[[中国作协|中国作协]]创委会副主任，中国作协兰州分会主席，《[[诗刊|诗刊]]》主编，《[[人民文学|人民文学]]》主编。是[[中国文联|中国文联]]第一、四届委员，中国作协副主席、党组副书记，[[第五届全国政协委员|第五届全国政协委员]]<ref name=zjw/><ref name=er/><ref name=zhuidao>著名诗人、中国作家协会副主席李季同志追悼会在京举行，人民日报1980年3月20日</ref>。

1943年开始发表作品。1952年加入[[中国作家协会|中国作家协会]]<ref name=zjw/><ref name=er/>。1946年底，发表长篇叙事诗《王贵与李香香》，具有浓郁的地方色彩，在中国现代诗歌史上占有重要地位<ref name=er/>。[[中华人民共和国|中华人民共和国]]成立后，他创作出了《杨高传》、《石油诗》、《向昆仑》等具有广泛影响的长短诗歌和小说、散文作品。1950年代，李季到玉门油矿深入生活，与“铁人”[[王进喜|王进喜]]结成朋友，后来他又参加了[[大庆油田|大庆石油会战]]。在20多年间，李季几乎走遍了中国所有石油矿区，同石油职工共同劳动，赢得了“石油诗人”的美誉。[[文革|文革]]期间，他敢于抵制“[[四人帮|四人帮]]”及其手下。[[粉碎四人帮|粉碎“四人帮”]]后，他在忙碌的工作中坚持业余写作，发表了赞颂石油工人的长篇叙事诗集《石油六歌》等。他参加过许多中外文学交流活动，并积极培养文学新人。由他主编的刊物推出了大批新的作家和诗人<ref name=zhuidao/>。

1980年3月8日，李季因心脏病突发在北京逝世，享年57岁。3月19日，追悼会在[[八宝山革命公墓|八宝山革命公墓]]礼堂举行，追悼会由[[中国文联|中国文联]]主席[[周扬|周扬]]主持，中国作协副主席[[刘白羽|刘白羽]]致悼词<ref name=zhuidao/>。
