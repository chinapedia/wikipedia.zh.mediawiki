'''李建军'''{{BD|1977年|7月29日|||catIdx=李}}，[[山西|山西]][[平遥|平遥]]人，中国大陆记者，曾就职于《[[山西晚报|山西晚报]]》和《[[成都商报|成都商报]]》。<ref>{{cite web |url=http://stock.caijing.com.cn/2013-08-02/113121575.html |title=《山西晚报》前记者李建军拟下周向香港廉署举报宋林 |accessdate=2013 |publisher=财经网 |work= |language=zh-hans |archive-url=https://web.archive.org/web/20140502013543/http://stock.caijing.com.cn/2013-08-02/113121575.html |archive-date=2014-05-02 |dead-url=yes }}</ref>

==生平==
2012年，李建军实名举报了[[华润集团|华润集团]]董事长[[宋林_(華潤集團)|宋林]]。<ref>{{cite web |url=http://finance.ifeng.com/a/20130802/10335051_0.shtml |title=前记者李建军拟向香港廉署举报宋林 |accessdate=2013 |publisher=凤凰网 |work= |language=zh-hans}}</ref>李建军举报宋林之后，[[山西金业集团|山西金业集团]]总裁张新明、[[山西省公安厅|山西省公安厅]]干部旋即应对，数度对李建军和他的家人使用“威胁恐吓”手段打击报复。<ref>{{cite web |url=http://money.163.com/13/0802/12/9597J1CB002529IR.html |title=前记者李建军拟下周向廉署举报宋林 |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref>
