'''李章洙'''（이장수，Lee Jang-Soo，{{bd|1956年|10月15日|catIdx=L李}}），[[韩国|韩国]]足球教练，以执教严格而著，绰号“铁帅”。李章洙曾经作为主教练分别率领[[青岛贝莱特|青岛贝莱特]]和[[重庆力帆|重庆力帆]]两次夺得[[中国足协杯|中国足协杯]]冠军，是中国足球职业化以来第一位带领不同俱乐部获得足协杯冠军的主教练<ref name="盘点李章洙中国足迹">{{cite web |url=http://sports.sina.com.cn/j/2006-11-23/06532583408.shtml|title=李盘点李章洙中国足迹：封杀高峰 怒向假球痛宰同族|accessdate=2008-11-21 |work= 新浪体育}}</ref>，其任[[北京国安|北京国安]]主教练其间率队获得[[2007年中国足球超级联赛|2007年中超联赛]][[亚军|亚军]]，同时也是在中国足坛执教时间最长的外籍教练<ref>{{cite web |url=http://sports.dzwww.com/zuqiuzixun/gnzq/200704/t20070420_2117405.htm|title=李章洙:屠劲旅靠狡猾 打弱旅倒难进球|accessdate=2008-11-21 |work= 京华时报}}</ref>。2010年开始出任[[广州恒大|广州恒大]]主教练，在率队的第一年就获得[[中甲联赛|中甲联赛]]冠军幚助广州队重返中超联赛。2011年又率领广州恒大以升班马身分获得[[中超联赛|中超联赛]]冠军，这也是他首次率队夺得中国顶级联赛冠军。

==执教生涯==
===韩国===
李章洙的教练生涯开始于1986年退役后<ref name="李章洙做客新浪畅聊07国安争冠幕后的故事">{{cite web |url=http://sports.enorth.com.cn/system/2007/11/20/002359665.shtml|title=李章洙做客新浪畅聊07国安争冠幕后的故事|accessdate=2008-11-21 |work= 北方网}}</ref>，作为主教练执教韩国[[光州|光州]][[湖南大学_(韩国)|湖南大学]]队，但是成绩一般。随后在1988年，李章洙加盟了韩国[[K联赛|K联赛]]劲旅[[城南一和天马|一和天马]]任助理教练。其在助理教练任上连续工作了6年，其间辅佐主教练[[朴钟焕|朴钟焕]]实现了K联赛三连冠<ref name="李章洙做客新浪畅聊07国安争冠幕后的故事"/>。1994年，由于表现出众，李章洙出任一和天马队主教练，在两年中率队再次获得了K联赛冠军，同时还夺得[[亚洲俱乐部锦标赛|亚洲俱乐部杯]]冠军，并继而获得[[亚洲超级杯|亚洲超级杯]]和[[亚非对抗赛|亚非对抗赛]]冠军。

===初来中国===
1998年，随着中国足球职业化改革的深入，李章洙离开了一和天马来到中国，执教[[甲A联赛|甲A联赛]][[重庆力帆队|重庆力帆队]]。当年7月26日，李章洙在中国执教的第一场比赛中带领球队在主场以3-1战胜了[[八一队|八一队]]<ref name="李章洙其人：两获足协杯冠军 两度求辞">{{cite web |url=http://sports.sina.com.cn/c/2006-11-20/13292578549.shtml|title=“荣誉市民”李章洙其人：两获足协杯冠军 两度求辞|accessdate=2008-11-21 |work= 北京晚报}}</ref>。在重庆执教期间，李章洙以“铁腕”管理球队，对球队中不遵守纪律的队员进行了严厉的处罚，其中包括当时的著名球员[[高峰|高峰]]。李章洙一方面认可高峰的实力，另一方面却对高峰的职业态度提出质疑，认为其“经常酗酒，不认真训练”<ref name="盘点李章洙中国足迹"/>。在双方僵持的阶段，李章洙顶住了压力，将高峰放上了替补席，进而将其从比赛名单中除去。赛季末，高峰因为无法上场比赛而离队返回了[[北京|北京]]。李章洙在与违规球员的较量中展现了强硬的个性，这也为他博得了“'''铁帅'''”的称谓。

2000年，李章洙率领的重庆队发挥出色，获得了甲A联赛第四名，同时闯进了中国足协杯的决赛。在两回合比赛中，重庆队以4：2的总比分战胜了[[北京国安队|北京国安队]]，获得了俱乐部历史上第一个全国冠军。为了表彰李章洙的贡献，[[重庆市|重庆市]]政府授予了李章洙“荣誉市民”的称号，同时李章洙也成为了中国足球年度“最佳教练”<ref name="李章洙简介">{{cite web |url=http://www.dzwww.com/tiyu/tiyujingcaitupian/lvyinmingxing/t20040406_184253.htm|title=李章洙 简介|accessdate=2008-11-21 |work= 大众网}}</ref>。2001年，重庆队成绩有所下滑，最终仅位列联赛第11名，年底，李章洙与青岛贝莱特签约成为其主教练。在离开了重庆时，李章洙表示“在长期的工作中，我和这个城市有了很深的感情，这里是我的第二故乡。”<ref name="盘点李章洙中国足迹"/>

2001年12月16日，李章洙与青岛队签约，并于12月19日上任。2002赛季，青岛队取得了球队历史上第二好的联赛成绩，最终位列积分榜第八位。与此同时，李章洙再次在中国足协杯的赛场上取得了成功，2002年11月16日，青岛队主场2：0战胜辽宁队，凭借客场进球的优势获得了足协杯冠军。这也是青岛足球第一个全国冠军。2003年5月26日，鉴于李章洙“对青岛市足球事业的发展做出了杰出的贡献”<ref name="青岛人大常委会通过议案 李章洙加入“青岛籍”">{{cite web |url=http://www.chinanews.com.cn/n/2003-05-26/26/307120.html|title=青岛人大常委会通过议案 李章洙加入“青岛籍”|accessdate=2008-11-21 |work= 青岛新闻网-青岛早报}}</ref>，[[青岛市|青岛市]]授予了李章洙“荣誉市民”的称号。在青岛执教期间，李章洙有意培养了多名有潜质的新秀，这其中就包括了后来成为国脚的[[刘健_(足球运动员)|刘健]]<ref name="李章洙回想带颐中问鼎足协杯时光：那是幸运日">{{cite web |url=http://sports.sohu.com/20081117/n260685625.shtml|title=“李章洙回想带颐中问鼎足协杯时光：那是幸运日|accessdate=2008-11-21 |work= 青岛新闻网-青岛早报}}</ref>。多年后的2008年，李章洙在回忆自己在青岛度过的岁月时也认为“在青岛执教的日子是我最难忘的幸福时光。”<ref name="李章洙回想带颐中问鼎足协杯时光：那是幸运日"/>

2003年11月22日，李章洙在他所执教的最后一场甲A联赛中带领青岛贝莱特队客场以3-1战胜了[[重庆力帆|重庆力帆]]，两支球队合力为李章洙在中国的第一次执教经历画上了句号。

===再来中国===
2005年，李章洙回到韩国，先后执教了[[全南龙|全南龙]]和[[FC首尔|FC首尔]]两支球队，但是成绩一般，另外其与俱乐部高层关系并不融洽，所以李章洙并没有久据其位。2006年底，中超联赛俱乐部[[北京国安|北京国安]]邀请李章洙执掌教鞭，12月11日，李章洙接受邀请，成为北京国安历史上第四位外籍主教练<ref name="国安正式宣布李章洙掌印 两年合同绑定正直铁帅">{{cite web |url=http://news.xinhuanet.com/sports/2006-12/03/content_5426282.htm|title=国安正式宣布李章洙掌印 两年合同绑定正直铁帅|accessdate=2008-11-21 |work= 新华网}}</ref>。

李章洙的到来给国安俱乐部带来了转变。其大运动量的季前训练使得平均年龄偏大的北京国安队反而在体能上得到了加强，其所倡导的[[攻势足球|攻势足球]]也符合北京足球“小、快、灵”的传统特点。另外，李章洙认为每场比赛都要竭尽全力，“哨声没响就全力取胜”<ref name="国安演绎绝杀戏法 李章洙：哨声没响就全力取胜">{{cite web |url=http://sports.sohu.com/20070917/n252190579.shtml|title=国安演绎绝杀戏法 李章洙：哨声没响就全力取胜|accessdate=2008-11-21 |work= 京华时报}}</ref>，这种精神面貌也被绝大多数北京球迷所认可。2007年，李章洙率领球队打破了多项纪录，其中包括首次客场战胜[[上海申花|上海申花]]和[[山东鲁能|山东鲁能]]，并取得了俱乐部历史上最高的客场积分（26分）及最多的客场进球（20球），还第一次在中超联赛客场积分榜上名列榜首<ref name="御林军07创14年最佳战绩 一分之差缔造悲情英雄">{{cite web |url=http://sports.sohu.com/20071114/n253257152.shtml|title=御林军07创14年最佳战绩 一分之差缔造悲情英雄|accessdate=2008-11-21 |work= 搜狐体育}}</ref>。最终李章洙带领北京国安队获得了亚军。尽管李章洙在某些关键比赛中的用人受到了质疑，同时其选定的主力阵容年龄结构也相对老化，但是李章洙的执教能力和处事风格还是得到了北京球迷和媒体较为一致的肯定。2008年12月27日，北京国安足球俱乐部与其正式续约，李章洙继续担任北京国安队主教练，但在2009年9月16日，北京国安正式决定李章洙下课。2009年北京国安最终染指联赛冠军，但却造成李章洙执教球队大半个赛季后被提前下课无缘冠军的尴尬。综合来看李章洙在中国的执教生涯，只是在国安队唯独缺少个冠军。

2010年3月25日，[[中国足球甲级联赛|中国足球甲级联赛]]球队[[广州恒大|广州恒大]]召开新闻发布会，宣布李章洙出任广州恒大主教练。但是由于他并未取得[[中国足协|中国足协]]所承认的职业教练员资格证书，故而在中国足协组织的赛事中是以助理教练的身份执教广州恒大<ref name="李章洙只是恒大助教 足协默许助理教练出席发布会">{{cite web |url=http://sports.people.com.cn/GB/14501568.html|title=李章洙只是恒大助教 足协默许助理教练出席发布会 |accessdate=2011-04-28 |work= 人民网}}</ref>。广州队也成为李章洙执在中国执教的第四支球队，在率队的第一年就以17胜6平1负的成绩获得[[中甲联赛|中甲联赛]]冠军幚助广州队重返[[中超联赛|中超联赛]]。2011赛季，李章洙率领广州恒大在联赛中高歌猛进，一直处于联赛榜首。2011年9月，广州恒大客场败于[[长春亚泰|长春亚泰]]后，球队的44场联赛（21场中甲、23场中超）不败纪录告终，但最终还是提前4轮获得[[中超联赛|中超联赛]]冠军，李章洙率队连续夺得中国两级联赛（中甲联赛及中超联赛）的冠军，李章洙在中国联赛执教的第11个年头，在率领不同球队拿到[[中国足协杯|足协杯]]冠军后，终于首次拿到了中国顶级联赛的冠军。但联赛提前夺冠后广州恒大的两场客场比赛表现低迷，先是2比5不敌[[江苏舜天足球俱乐部|江苏舜天]]，最后一轮又被已经确定降级的[[成都谢菲联|成都谢菲联]]逼平，李章洙在赛后两次向俱乐部和球迷道歉。<ref>{{cite web |url=http://sports.sohu.com/20111028/n323791866.shtml|title=李章洙：向媒体和球迷道歉 队委会就是河水逆流|accessdate=2011-10-28 |work= 搜狐体育}}</ref><ref>{{cite web |url=http://sports.sohu.com/20111103/n324323098.shtml|title=李章洙向许家印道歉 穆里奇调侃：MVP舍我其谁|accessdate=2011-11-03 |work= 搜狐体育}}</ref>

2012赛季，足协放宽教练证书的标准，李章洙得以在中国足协的报名表中开始以主教练的身份执教球队。2012年2月，他带领广州恒大队在[[2012年中国足球协会超级杯|2012年中国足球协会超级杯]]中2比1击败足协杯冠军[[天津泰达足球俱乐部|天津泰达]]，获得超级杯冠军，李章洙在第三次参加该项赛事后，终于获得冠军。虽然广州恒大在[[2012年亚足联冠军联赛|2012年亚足联冠军联赛]]中有客场5比1击败[[K联赛|K联赛]]冠军[[全北现代汽车|全北现代汽车]]，主场3比1击败[[J联赛|J联赛]]冠军[[柏太阳神|柏太阳神]]的辉煌成绩，但主场1比2不敌泰国冠军[[武里喃联足球俱乐部|武里喃联]]和1比3不敌全北现代汽车等比赛却使李章洙陷入争议。很多媒体和球迷都对他的战术和轮换制度提出质疑。在主场不敌全北现代后，球队的头号球星[[达里奥·孔卡|达里奥·孔卡]]甚至在[[新浪微博|微博]]中表达自己对李章洙的不满。<ref>{{cite web |url=http://sports.sina.com.cn/j/2012-05-03/01536046544.shtml|title=孔卡急了！3句话质问李章洙:为什么轮休的总是我?|accessdate=2012-05-03 |work= 新浪体育}}</ref>5月15日，李章洙率领广州恒大在亚足联冠军联赛最后一轮中凭借孔卡在90分钟的进球，2比1险胜武里喃联，积分超过全北现代以小组第一晋级淘汰赛。但在5月16日，有报道称广州恒大俱乐部官员当天在泰国和李章洙进行会谈，并在俱乐部内部宣布李章洙离任。<ref>{{cite web |url=http://sports.sina.com.cn/j/2012-05-16/14036063745.shtml|title=恒大将正式宣布李章洙下课 球员教练微博深情告别|accessdate=2012-05-15 |work= 新浪体育}}</ref>5月17日，广州恒大官方正式宣布由[[意大利|意大利]]名帅[[马尔切洛·里皮|马尔切洛·里皮]]上任接替李章洙出任主教练。<ref>[http://www.gzevergrandefc.com/show_content.php?id=4053 关于李章洙指导不再担任广州恒大足球队主教练的公告] {{webarchive|url=https://web.archive.org/web/20120523054329/http://www.gzevergrandefc.com/show_content.php?id=4053 |date=2012-05-23 }}</ref><ref>[http://www.gzevergrandefc.com/show_content.php?id=4054 关于聘任马塞洛·里皮担任广州恒大足球队主教练的公告] {{webarchive|url=https://archive.is/20121209001848/http://www.gzevergrandefc.com/show_content.php?id=4054 |date=2012-12-09 }}</ref>

2014年7月6日下午，[[成都天诚足球俱乐部|成都天诚足球俱乐部]]官方正式对外宣布：韩国著名教练李章洙与天诚正式签约，李章洙将带领成都天诚队征战接下来的中甲联赛。据了解，李章洙于5日晚抵达成都，并于今天下午观看了成都天诚和四川隆发的热身赛。据悉，李章洙签约天诚的首要任务是帮助俱乐部保级，然后再率领球队冲超并在中超站稳脚跟。

2016年5月6日，[[中超|中超]]球队[[长春亚泰|长春亚泰]]官方正式宣布，李章洙接任球队主教练，他与俱乐部签下了1份3+2的五年长约。

2017年5月4日，由于前5轮联赛成绩不佳，长春亚泰官方宣布李章洙正式下课，陈金刚接任主教练。李章洙也成为2017年中超第一位下课的主教练。

==个人轶事==
李章洙的夫人并没有随其居住在中国，他们育有一子一女；女儿在韩国[[汉阳大学|汉阳大学]]读音乐方向的研究生，儿子在服兵役<ref name="李章洙：找媳妇“有一套”管孩子“简单粗暴”">{{cite web |url=http://sports.sohu.com/20070917/n252190579.shtml|title=李章洙：找媳妇“有一套”管孩子“简单粗暴”|accessdate=2008-11-21 |work= 京华时报}}</ref>，母子三人现均居住在韩国。

李章洙在中国已经执教多年，用[[汉语|汉语]]进行日常交流已经基本没有困难<ref>[http://sports.sina.com.cn/j/2011-01-06/10365396649.shtml 韩国两铁帅比拼中文功底 李章洙自豪出身名牌大学]</ref>，但是其认为自己很难“用汉语准确地表达自己的情感”<ref name="双面李章洙：固执已见不听人言 处事圆滑老油条">{{cite web |url=http://sports.cnwest.com/content/2007-11/05/content_1065696_4.htm|title=双面李章洙：固执已见不听人言 处事圆滑老油条|accessdate=2008-11-21 |work= 搜狐体育}}</ref>，所以每当记者提出采访要求时，李章洙都要首先确保[[翻译|翻译]]在场，然后再用[[韩语|韩语]]回答记者提出的问题。
