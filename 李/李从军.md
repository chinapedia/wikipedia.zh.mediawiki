'''李从军'''（{{bd|1949年|10月||}}），[[安徽|安徽]][[六安市|六安]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[新华通讯社|新华通讯社]]社长、党组书记。

== 生平 ==
1968年10月参加工作。1977年，成为[[安徽师范大学|安徽师范大学]]六安教学点中文系学员。1982年，成为[[吉林大学|吉林大学]]文学系研究生。1983年5月加入[[中国共产党|中国共产党]]。1984年，成为[[山东大学|山东大学]]中文系中国古代文学专业博士研究生。1985年，山东大学中文系中国古代文学专业博士研究生毕业并获文学博士学位<ref name=zgjjw>{{Cite web |url =http://www.ce.cn/xwzx/gnsz/gdxw/201503/02/t20150302_4700926.shtml  |title =  李从军任全国政协教科文卫体委员会副主任|author =  |date =2015-03-02  |publisher =  中国经济网|language =  |accessdate =  |quote =  }}</ref><ref name=hexun>{{Cite web |url = http://renwu.hexun.com/figure_162.shtml |title =李从军 |author =  |date =  |publisher = 和讯 |language =  |accessdate = 2018-02-03 |quote =  }}</ref>。

此后，历任[[中共中央宣传部|中共中央宣传部]]研究室副主任，中共[[邯郸市|邯郸市]]委副书记（挂职），中共中央宣传部政策法规研究室主任，中共[[宁波市|宁波市]]委副书记、[[宁波大学|宁波大学]]党委书记，[[中共浙江省委|中共浙江省委]]宣传部副部长。1998年12月，任中共浙江省委常委、宣传部副部长。1999年1月，任中共浙江省委常委、宣传部部长。2001年2月，任中共中央宣传部副部长<ref name=zgjjw/><ref name=hexun/>。

2007年8月，任[[新华社|新华社]][[党组|党组]]书记、副社长。2008年3月至2014年12月，任新华社社长、党组书记，2014年12月到龄卸任<ref name=zgjjw/><ref name=hexun/><ref>{{cite news|url=http://news.163.com/14/1231/18/AEQHD3UF0001124J.html|title=蔡名照担任新华社社长、党组书记|publisher=网易|date=2014-12-31}}</ref>。2009年10月10日，李从军在世界媒体峰会开幕式上通过讲故事向世界各知名媒体代表介绍了中国媒体近年来在履行社会责任和公益使命上的成就，[[香港|香港]]《[[大公报|大公报]]》评论该演讲颠覆了以往中共官方媒体“报道手段程式化，内容干巴巴的现状”<ref name=hexun/><ref>{{Cite web|url =http://takungpao.com/news/09/10/10/ZM-1154093.htm|title =李從軍挑戰「新華體」|author =|date =2009-10-10|publisher =大公报|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20100102060320/http://takungpao.com/news/09/10/10/ZM-1154093.htm|archivedate =2010-01-02}}</ref>。李从军在任期间推动了[[中国新华新闻电视网|中国新华新闻电视网]]开播，这被视为中国主流媒体增强国际传播能力建设的重大举措。李从军撰写的文章《构建世界传媒新秩序》倡导遵循FAIR观念（公平、共赢、包容、责任），被《[[华尔街日报|华尔街日报]]》刊登<ref name=hexun/>。

李从军是中共第十七届、十八届中央委员，[[第十届全国人大常委会|第十届全国人大常委会]]委员、[[全国人大教科文卫委员会|全国人大教科文卫委员会]]委员。2015年2月28日，政协十二届全国委员会常务委员会第九次会议闭幕，会议决定增补李从军为政协第十二届全国委员会委员、教科文卫体委员会副主任<ref name=zgjjw/><ref name=hexun/>。
