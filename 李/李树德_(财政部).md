'''李树德'''（{{bd|1917年||1980年|9月}}）曾用名'''李容善'''，今[[山东省|山东省]][[新泰市|新泰市]][[翟镇_(新泰市)|翟镇]]后羊村人。[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员<ref name=sdsqw>{{Cite web |url = http://www.sdsqw.cn/town/taian/200707/article_9966.html |title = 翟镇 |author =  |date = 2007-07-12  |publisher = 山东省情网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
[[抗日战争|抗日战争]]全面爆发后，李树德投身抗日救亡活动。1937年11月到[[沂水县|沂水县]]参加了[[中国共产党|中国共产党]]领导的抗日游击队。1938年5月回到家乡新泰县。同年7月，李树德在本村经李仲久介绍，加入中国共产党。此后曾任中共基层组织负责人，并被中共党组织派到[[国民革命军|国民革命军]]第六十九军从事兵运工作。1938年12月，参加[[八路军|八路军]]山东抗日游击第四支队，任连政治指导员。1940年3月，转到地方工作。曾先后在[[新蒙县|新蒙县]]、新泰县任科长等职<ref name=sdsqw/>。

1945年抗日战争胜利后，历任[[山东省人民政府|山东省人民政府]]粮食局副科长、科长，华东前线支前办事处粮食部副部长。1947年9月，调到[[中国东北|中国东北]]工作，历任辽南公署粮食局局长、[[辽宁省|辽宁省]]粮食局局长、[[东北行政委员会_(1953–1954)|东北行政委员会]]粮食总局副局长<ref name=sdsqw/>。

[[中华人民共和国|中华人民共和国]]成立后，历任[[中共中央华南分局|中共中央华南分局]][[珠江三角洲|珠江三角洲]]工委委员、广东省粮食局局长、[[中南行政委员会|中南行政委员会]]财政局处长、副局长。1955年1月，调到[[中华人民共和国财政部|中华人民共和国财政部]]农税司，历任副司长、司长<ref name=sdsqw/>。1960年11月17日，国务院全体会议第105次会议通过，任命李树德、[[曾直|曾直]]、[[陈如龙|陈如龙]]为中华人民共和国财政部副部长<ref name=sdsqw/><ref>周恩来，国务院命令，中华人民共和国国务院公报1960年33期</ref>。1962年，改任[[中共中央华北局|中共中央华北局]]财经办公室副主任。1966年后，调任[[中共内蒙古自治区党委|中共内蒙古自治区党委]]常委、秘书长，[[内蒙古自治区革命委员会|内蒙古自治区革命委员会]]副主任等职<ref name=sdsqw/>。

1980年9月，李树德在北京病逝<ref name=sdsqw/>。
