'''李菁聾健共融基金'''是[[香港|香港]]一個[[慈善基金|慈善基金]]，由[[香港中文大學手語及聾人研究中心|香港中文大學手語及聾人研究中心]]於2008年成立。

2008年3月，[[失聰|失聰]]女教師[[李菁_(聽障教師)|李菁]][[死亡|逝世]]。她曾任職研究助理的香港中文大學手語及聾人研究中心成立「李菁聾健共融基金」，延續她樂於助人美意<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2008/03/07/ED0803070005.htm | title=中大成立「李菁聾健共融基金」 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date= 2008-03-07|accessdate=2012-02-19}}</ref><ref>{{cite web |language=zh-hk |url=https://topick.hket.com/article/2093241/%E6%99%82%E9%96%93%E6%B4%97%E4%B8%8D%E8%B5%B0%E5%B0%8D%E8%BC%95%E7%94%9F%E6%84%9B%E5%A5%B3%E7%9A%84%E7%89%BD%E6%8E%9B%E3%80%80%E5%9A%B4%E7%88%B6%E6%88%B0%E5%8B%9D%E6%8A%91%E9%AC%B1%E5%8A%A9%E4%BA%BA%EF%BC%9A%E8%AE%93%E5%A5%B3%E5%85%92%E3%80%8C%E6%B4%BB%E3%80%8D%E4%B8%8B%E5%8E%BB | title=時間洗不走對輕生愛女的牽掛　嚴父戰勝抑鬱助人：讓女兒「活」下去 |publisher=[[經濟日報|經濟日報]] |date= 2018-06-13|accessdate=2018-06-19}}</ref>。

==曾贊助的活動==
<!-- 注释出：[[File:細聽我心聲的封面.png|thumb]]》書刊]] -->
{{main|細聽我心聲}}
李菁聾健共融基金於2010年贊助《[[細聽我心聲|細聽我心聲]]》的徵文比賽<ref>{{cite web |language=zh-hk |url=http://silence.org.hk/site/modules/news/article.php?storyid=147 |title=失聰英中生 冀進大學報親恩 |publisher=[[香港經濟日報|香港經濟日報]] |date=2011-01-31 |accessdate=2012-02-19 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。該徵文比賽是由[[慈善機構|慈善機構]]「[[龍耳|龍耳]]」主辦，得到「[[国际狮子会#香港青年獅子會|國際獅子總會中國港澳 303 區青年獅子區會]]」的支持。比賽的目的是讓聾人及弱聽學生及在職人士藉著自身的經歷，表達聾人及弱聽人士的心聲<ref> {{cite web |language=zh-hk |url=http://blog.mingpao.com/cfm/content.cfm?OwnerID=1&CategoryID=6&TopicID=10112 |title=無言的愛 |publisher=[[明報|明報]] |date=2011-03-24 |accessdate=2012-02-19 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，並藉此推廣[[融合教育|融合教育]]的理念。《細聽我心聲》為香港第一本由聾人及弱聽人士寫下他們過往求學及家長陪伴聾童的心路歷程之書刊<ref>{{cite web |language=zh-hk |url=http://www.silence.org.hk/file/listen.pdf | title=《細聽我心聲》徵文比賽得獎作品 |publisher=[[龍耳|龍耳]] |date= |accessdate=2011-02-19}}</ref>，一眾得獎聾童及家長期望各界可透過他們的作品多了解聾人。

自從《[[細聽我心聲|細聽我心聲]]》的書刊推出後，引起社會上很大的迴響，此書刊獲[[教育局_(香港)|教育局]][[常任秘書長|常任秘書長]]委任的一個委員會 － 《[[家庭與學校合作事宜委員會|家庭與學校合作事宜委員會]]》上載其網頁<ref>{{cite web|language=zh-hk|url=http://chsc.hk/section4_3_4.php?lang_id=2|title=網上家長錦囊|publisher=[[家庭與學校合作事宜委員會|家庭與學校合作事宜委員會]]|date=|accessdate=2012-02-19|deadurl=yes|archiveurl=https://web.archive.org/web/20111130214800/http://chsc.hk/section4_3_4.php?lang_id=2|archivedate=2011-11-30}}</ref>。

由於反應熱烈，「國際獅子總會中國港澳 303 區青年獅子區會」和「龍耳」於2011年再度合作，再次舉辦《聲空奇遇》徵文比賽<ref>{{cite web |language=zh-hk |url=http://www.leod303.org/projects/-sheng-kong-qi-yu-zheng-wen-bi-sai-1 |title=《聲空奇遇》徵文比賽得獎作品 |publisher=[[国际狮子会|國際獅子總會中國港澳 303 區青年獅子區會]] |date= |accessdate=2012-02-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20141129060705/http://www.leod303.org/projects/-sheng-kong-qi-yu-zheng-wen-bi-sai-1 |archivedate=2014-11-29 }}</ref>，希望藉著一系列的活動宣揚共融訊息，透過文字讓社會大眾了解聾人及弱聽人士的心聲，鼓勵社會大眾以積極正面的文字，勉勵聾人及弱聽人士活出積極及精彩的人生，為社會大眾與聾人及弱聽人士創造彼此溝通和了解的機會。
