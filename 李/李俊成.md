'''李俊成'''（{{bd|1914年||1960年|9月}}），曾用名'''万戈'''、'''李凌'''、'''鲁克'''，[[广东省|广东省]][[大埔县|大埔县]]高陂镇陶溪村龙山人。[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员<ref name=li>{{Cite web |url = http://www.gd-info.gov.cn/books/dtree/showSJBookContent.jsp?bookId=19104&partId=1865&artId=113204 |title = 大埔县志（1979～2000）：李俊成 |author =  |date =  |publisher =广东省情网  |language =  |accessdate =2017-04-09  |quote =  }}</ref>。

==生平==
民国10年（1921年）起，李俊成先后在家乡的育才学校、高陂小学等校学习。民国22年（1933年）7月，在高陂公立学校任教。当时，教育家[[陶行知|陶行知]]在百侯中学开展生活教育，设艺友师范班，李俊成弃教报考艺友师范班获得录取，学习1年。民国23年（1934年）11月去[[广西|广西]][[南宁|南宁]]，在国民基础教育研究院研究班学习。民国24年（1935年）5月，加入[[中国共产主义青年团|中国共产主义青年团]]，负责中国共产主义共青团南宁市委员会的工作。民国25年（1936年）春，转为[[中国共产党|中国共产党]]党员。在广西，李俊成组织学生救国会从事抗日救亡活动，后来去[[绥远省_(中华民国)|绥远]]，在[[八路军|八路军]]办事处从事宣传工作，不久调到八路军[[120师|120师]]教导团任政治教员。民国27年（1938年）8月，来到[[延安|延安]]进入[[中国人民抗日军政大学|中国人民抗日军政大学]]第四期一大队学习。毕业之后，奉命到延安中国人民抗日军政大学总政治部，在敌后工作训练班学习，后来被派到总政治部华北考察团敌军工作组任职。[[皖南事变|皖南事变]]后，李俊成在八路军军政学院学习。中央军委设立[[中央军委办公厅|中央军委办公厅]]时，李俊成任办公厅干部处干事、机要秘书兼管人事工作。[[抗日战争|抗日战争]]胜利后，李俊成调到[[晋冀鲁豫军区|晋冀鲁豫军区]]情报处任副科长<ref name=li/>。
 
民国36年（1947年）11月24日，李俊成奉命到[[大别山|大别山]]，受到[[刘伯承|刘伯承]]、[[邓小平|邓小平]]、[[陈毅|陈毅]]等人接见，并且参与策动[[黄维|黄维]]兵团起义。民国38年（1949年）4月，奉命率接管军政干部400多人到[[南京|南京]]投入接管工作，后来被安排担任[[南京市公安局|南京市公安局]]办公室副主任。同年8月，调任[[重庆市公安局|重庆市公安局]]五处经济保卫处处长。同年10月，调任西南公安部第二处处长<ref name=li/>。1953年8月，调任[[中央人民政府最高人民法院|中央人民政府最高人民法院]]西南分院副院长<ref name=li/><ref name=daqufenyuan>{{Cite web |url = http://www.hybsl.cn/beijingcankao/beijingfenxi/2015-01-27/50584.html |title =何帆：最高人民法院大区分院往事 |author =  |date =2015-01-27 |publisher = 胡耀邦史料信息网 |language =  |accessdate =  |quote =  }}</ref>。1955年7月，调任[[四川省人民委员会|四川省人民委员会]]政治办公室副主任、四川省政法党组副书记。1956年，当选[[四川省政协|四川省政协]]委员。1958年3月，被划为[[右派分子|右派分子]]，开除中国共产党党籍、撤销职务，下放[[西昌|西昌]]大营农场[[劳动改造|劳动改造]]<ref name=li/>。

1960年9月，李俊成在四川病逝，享年46岁。1979年，[[中共四川省委|中共四川省委]]、四川省政法党组为李俊成平反<ref name=li/>。
