'''李菁紀念基金'''是[[香港|香港]]一個[[慈善基金|慈善基金]]，由[[香港嶺南大學|香港嶺南大學]]資源拓展處於2008年成立<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2008/09/10/ED0809100007.htm | title=嶺南大學 成立「李菁紀念基金」 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date= 2008-09-10|accessdate=2012-02-19}}</ref>。

2008年3月，[[失聰|失聰]]女教師[[李菁_(聽障教師)|李菁]][[死亡|逝世]]。嶺南大學資源拓展處得到管理層支持及李菁家人同意，成立嶺南大學「李菁紀念基金」<ref>{{cite web |language=zh-hk |url=https://topick.hket.com/article/2093241/%E6%99%82%E9%96%93%E6%B4%97%E4%B8%8D%E8%B5%B0%E5%B0%8D%E8%BC%95%E7%94%9F%E6%84%9B%E5%A5%B3%E7%9A%84%E7%89%BD%E6%8E%9B%E3%80%80%E5%9A%B4%E7%88%B6%E6%88%B0%E5%8B%9D%E6%8A%91%E9%AC%B1%E5%8A%A9%E4%BA%BA%EF%BC%9A%E8%AE%93%E5%A5%B3%E5%85%92%E3%80%8C%E6%B4%BB%E3%80%8D%E4%B8%8B%E5%8E%BB | title=時間洗不走對輕生愛女的牽掛　嚴父戰勝抑鬱助人：讓女兒「活」下去 |publisher=[[經濟日報|經濟日報]] |date= 2018-06-13|accessdate=2018-06-19}}</ref>。

==成立的目的==
目的是為幫助嶺南大學殘障學生，資助他們在學習上或購置輔助器材上財政需要，讓他們與其他同學一同享受大學校園生活，達致「傷健共融」的理念<ref>{{cite web |language=zh-hk |url=http://www.takungpao.com.hk/news/08/03/07/JX-874103.htm |title=中大嶺大設李菁基金助殘障 |publisher=[[大公報|大公報]] |date=2008-03-07 |accessdate=2012-02-19 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。此外，亦為殘疾人士成立獎學金<ref>{{cite web |language=zh-hk |url=http://www.takungpao.com.hk/news/12/06/10/GW-1488159.htm |title=成立「李菁紀念基金」扶助聽障者 |publisher=[[大公報|大公報]] |date=2012-06-10 |accessdate=2012-06-10 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==曾贊助的項目==
嶺南大學將李菁過往在刊物及網上日誌講述有關「聽障」的文章和生活照結集成《[[凝望—君看雙眼色，不語似無愁|凝望—君看雙眼色，不語似無愁]]》一書<ref>{{cite web |language=zh-hk |url=http://news.sina.com/oth/mingpao/301-101-101-105/2008-09-04/16003274518.html | title=嶺大為自殺聽障教師出文集 |publisher=[[明報|明報]] |date= 2008-09-04|accessdate=2012-02-19}}</ref>，目的為喚起社會對聾人及弱聽人士的關注<ref>{{cite web |language=zh-hk |url=http://www.takungpao.com.hk/news/08/09/05/JX-956767.htm |title=嶺大文集籲關注聽障者 |publisher=[[大公報|大公報]] |date=2008-09-05 |accessdate=2012-02-19 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。這本書集不會公開發售，只供有興趣的讀者聯絡嶺南大學索取。此外，這本書亦推出了網上版本，並上載於[[慈善機構|慈善機構]]「[[龍耳|龍耳]]」的網頁<ref>{{cite web |language=zh-hk |url=http://www.silence.org.hk/file/Memorial.pdf | title=《凝望—君看雙眼色，不語似無愁》|publisher=[[龍耳|龍耳]] |date= |accessdate=2012-02-19}}</ref>，供有興趣的讀者下載。
