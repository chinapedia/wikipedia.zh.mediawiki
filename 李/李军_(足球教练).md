'''李军'''{{BD|1967年|7月31日}}，[[湖北省|湖北省]]人，[[中国|中国]][[足球教练|足球教练]]，退役[[足球运动员|足球运动员]]，现为[[新疆天山雪豹足球俱乐部|新疆天山雪豹足球俱乐部]]主教练。球员时代，李军曾代表湖北省参加[[六运会|六运会]]，夺得第四名。1994年中国足球职业化后，他先后在[[湖北武汉职业足球俱乐部|湖北武钢]]、[[前卫寰岛|前卫寰岛]]效力，1997年退役。退役之后，李军起初在湖北足协负责年轻球员培养。2009年，李军带领新成立的[[湖北绿茵|湖北绿茵]]从[[中乙联赛|中乙联赛]]冲入[[中甲联赛|中甲联赛]]。2012年，李军又带领新成立的[[湖北华凯尔|湖北华凯尔]]自乙级升入甲级，四年内带领两支球队冲甲成功。

== 球员生涯 ==
李军的父亲[[李发清|李发清]]是湖北省自行车队的运动员，曾在1958年[[全国自行车锦标赛|全国自行车锦标赛]]中夺得铜牌，打破全国纪录。李军小时候就喜爱足球，1979年小学毕业后报考[[湖北省体校|湖北省体校]]，走上足球道路。1987年，李军入选[[中华人民共和国第六届运动会|中华人民共和国第六届运动会]]湖北男足代表队，身着18号球衣，帮助球队获得第四名<ref name=cfadata/>，创下湖北足球全运会最佳战绩。这时，李军只有20岁，是队中年纪最小的。1994年中国足球职业化，李军代表[[湖北武汉职业足球俱乐部|湖北武钢]]出战[[1994年中国足球甲B联赛|1994年中国足球甲B联赛]]，球队排名联赛第五。1995年，李军加盟[[前卫寰岛|前卫寰岛]]。1997年，李军退役<ref name=cjz>{{cite news|title=李军冲甲专业户称号名符其实 4年两度带队冲甲成功|url=http://sports.sina.com.cn/b/2012-10-25/02476269617.shtml|accessdate=2015-02-28|newspaper=长江商报|date=2012-10-25}}</ref>。

妻子曾是跳远运动员，女儿[[李茵晖|李茵晖]]是一名羽毛球运动员<ref>{{Cite web |url=http://sports.cnhubei.com/2017/0425/355350.shtml |title=湖北小将家门口盼拿冠军 李茵晖：最想吃豆皮 |accessdate=2018-03-21 |author=记者：马明玉 |date=2017-04-25 |work= |publisher=荆楚网|language=Zh-hans }}</ref>。

== 教练生涯 ==
退役后，李军一直在[[湖北足协|湖北足协]]工作，培养年轻球员，湖北省1989年龄段的球员就由他负责。2008年，[[湖北武汉职业足球俱乐部|武汉光谷]]退出[[中超联赛|中超联赛]]，湖北没有了职业足球俱乐部。[[2009年中国足球乙级联赛|2009年]]，[[湖北绿茵|湖北绿茵]]成立，俱乐部留住了武汉光谷队U19梯队的球员以及[[王文华|王文华]]、[[胡卓伟|胡卓伟]]、[[蔡曦|蔡曦]]、[[周燎|周燎]]、[[吴龑|吴龑]]等原来武汉光谷一线队的球员。这年5月，李军被任命为湖北绿茵队的主教练，带队出征[[中国足球乙级联赛|中国足球乙级联赛]]。球队以预赛不败、南区第一名的战绩挺进全国总决赛的半决赛。半决赛中，球队击败[[杭州三超|杭州三超]]，升入[[中甲联赛|中甲联赛]]，夺得中乙联赛亚军。这一年，李军还率领湖北省U20男足参加了[[中华人民共和国第十一届运动会|中华人民共和国第十一届运动会]]，获得预赛小组第一名、决赛第九名。[[2010年中国足球甲级联赛|2010年]]，李军带领以[[梅方|梅方]]、[[吴龑|吴龑]]、[[柯钊|柯钊]]、[[李行|李行]]等年轻球员为班底的湖北绿茵获得中甲联赛第5名。赛季末，俱乐部资金紧张。[[2011年中国足球甲级联赛|2011年]]，湖北武汉中博置业入主湖北绿茵，球队管理层换人，李军不再担任主教练。<ref>{{cite news|title=他们“回归”不谈德比谈感情|url=http://news.ifeng.com/a/20140425/40035610_0.shtml|accessdate=2015-02-28|newspaper=武汉晚报|date=2014-04-25}}</ref><ref name=cjz/>

离开湖北绿茵后，李军在2011年担任了业余球队[[武汉宏兴|武汉宏兴]]的主教练。<ref>{{cite news|title=两人成湖北新帅热门候选 王文华李军PK巴西洋帅|url=http://sports.sohu.com/20110529/n308816068.shtml|accessdate=2017-08-03|newspaper=汉网-武汉晨报|date=2011年4月29日}}</ref>[[2012年中国足球乙级联赛|2012年]]，[[湖北华凯尔足球俱乐部|湖北华凯尔足球俱乐部]]组建，李军出任主教练，还参与了球队的组建。李军负责为俱乐部联系、挑选球员。媒体报道称，队里的每个球员都是他联系来的，包括湖北本地球员[[周熠|周熠]]、[[王康_(足球运动员)|王康]]、[[胡浩_(足球运动员)|胡浩]]、[[童晓星|童晓星]]等人，还有外地球员[[侯哲|侯哲]]、[[董志远|董志远]]、[[朱子林|朱子林]]等人。俱乐部其他教练也是他联系来的，有[[王文华|王文华]]、[[徐健|徐健]]、[[刘俊峰|刘俊峰]]和[[余捷|余捷]]等人。甚至连球员的球衣都是李军联系[[香港|香港]]的赞助商提供的。这一年，球队在预赛中排在南区第四，打入全国总决赛。总决赛中，虽然不被看好，但湖北华凯尔连续击败北区冠军[[河北中基|河北中基]]和南区亚军[[深圳风鹏|深圳风鹏]]，冲入中甲，夺得中乙亚军。李军被评为该年的中乙最佳主教练。<ref>{{cite news|title=李军 我现在是地道的新疆人(图)|url=http://news.163.com/14/1104/00/AA5SI2SL00014AED.html|accessdate=2015-02-28|newspaper=乌鲁木齐晚报|date=2014-11-04}}</ref>这也是李军四年内第二次带队升入中甲联赛。<ref name=cjz/>2013年，李军率领湖北华凯尔完成保级。2014年，由于资金问题，湖北华凯尔西迁新疆，改名[[新疆天山雪豹|新疆天山雪豹]]<ref>{{cite news|title=主教练介绍|url=http://epaper.ts.cn/ftp/site1/xjdsb/html/2014-04/30/content_97226.htm|accessdate=2015-02-28|newspaper=新疆都市报|date=2014-04-30|deadurl=yes|archiveurl=https://web.archive.org/web/20150402102651/http://epaper.ts.cn/ftp/site1/xjdsb/html/2014-04/30/content_97226.htm|archivedate=2015-04-02}}</ref>，李军带领球队排在联赛第10名。
