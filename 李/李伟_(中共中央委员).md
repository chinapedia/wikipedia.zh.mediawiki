'''李伟'''（{{bd|1953年|8月|||catIdx=}}），[[江苏|江苏]][[丰县|丰县]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中国共产党第十八届中央委员会委员|中国共产党第十八届中央委员会委员]]。

==生平==
1969年3月参加工作，1973年4月加入[[中国共产党|中国共产党]]，大专文化程度。1969年3月，成为[[上海市|上海市]][[川沙县|川沙县]]施湾公社[[知青|知青]]。1970年11月，成为[[中国人民解放军|中国人民解放军]]战士。1976年4月起，历任上海市化工进出口公司办事员、组织科副科长兼公司团委书记、党委委员（1982年9月至1985年10月在上海电视大学中文专业学习），[[中共上海市委|中共上海市委]]组织部经济干部处联络员。1988年3月起，历任[[上海市人民政府|上海市人民政府]]、中共上海市委秘书，[[国务院办公厅|国务院办公厅]]秘书，[[国务院总理办公室|国务院总理办公室]]主任（1998年3月至2003年3月在[[朱镕基|朱镕基]]担任[[国务院总理|国务院总理]]时）。1998年6月，任[[国务院研究室|国务院研究室]]副主任、党组成员兼国务院总理办公室主任。2003年1月，任[[中央金融工委|中央金融工委]]副书记。2003年3月，任[[中国银行业监督管理委员会|中国银行业监督管理委员会]]副主席、党委委员；2005年10月，任[[国务院国有资产监督管理委员会|国务院国有资产监督管理委员会]]副主任、党委副书记。2010年7月，任[[国务院发展研究中心|国务院发展研究中心]]党组书记、副主任。他还担任[[中央文明委|中央文明委]]委员、[[中央保密委员会|中央保密委]]委员<ref>{{Cite web |url = http://news.163.com/10/0916/00/6GLN1JKN0001124J.html |title = 国务院任免工作人员 工信部国土部二把手调整 |author =  |date = 2010-09-16 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。2011年4月，升任国务院发展研究中心主任<ref>{{cite web|url=http://news.ifeng.com/mainland/detail_2011_04/28/6040572_0.shtml|title=李伟任国务院发展研究中心主任 刘鹤任党组书记|publisher=凤凰网|accessdate=2011-04-28}}</ref><ref>{{cite news|title=总书记总理办公室主任揭晓 盘点历任主任|url=http://news.xinhuanet.com/politics/2013-07/25/c_116678525_2.htm|date=2013年7月25日|work=新华网}}</ref>。2012年11月在[[中共十八大|中共十八大]]上当选[[中国共产党|中国共产党]][[中国共产党第十八届中央委员会委员列表|第十八届中央委员会委员]]<ref>{{cite news|title=中央委员李伟：中国产能过剩可能产生经济危机|url=http://money.163.com/12/1229/09/8JSOPU7400252G50.html|accessdate=2013-03-25|newspaper=网易}}</ref>。2018年成为[[第十三届全国政协委员|第十三届全国政协委员]]<ref>{{Cite web |url =  http://mil.chinanews.com/shipin/cns-d/2018/03-07/news759627.shtml|title = 国务院发展研究中心主任李伟谈智库建设 |author =  |date = 2018-03-07 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>、常委<ref>{{Cite web |url =http://hn.ifeng.com/a/20180315/6433796_0.shtml  |title = 政协第十三届全国委员会 主席、副主席、秘书长、常务委员名单 |author =  |date = 2018-03-15 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>，[[全国政协人口资源环境委员会|全国政协人口资源环境委员会]]主任。
