'''李虎臣'''（{{bd|1889年||1954年|}}），<ref>生年1889年据《陕西省志 人物志（中册）》。《中国近代軍閥史詞典》作生于1890年。</ref>原名'''秉信'''，字'''实生'''，后改名'''云龙'''，字'''虎臣'''，[[以字行|以字行]]。[[陕西|陕西]][[临潼|临潼]]房村（今属[[西安市|西安市]][[阎良区|阎良区]]） 人。原为[[刀客|刀客]]，后来为[[中华民国|中华民国]]军事将领。<ref name=sxsz>[http://lib.sxsdq.cn/bin/mse.exe?seachword=%20&K=a&A=1&rec=313&run=13 李虎臣，陕西省志 人物志（中册），于2012-1-25查阅]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==生平==
===刀客和军官===
1889年，李虎臣生于一个贫穷的家庭，13岁起给财主家扛活。1911年[[辛亥革命|辛亥革命]]爆发，李虎臣闻讯后痛打财主，然后跑到[[西安|西安]]投[[秦陇复汉军|秦陇复汉军]]，在兵马都督[[张云山|张云山]]的卫队充任亲兵，随张云山参加了陕西辛亥革命期间的西路作战。1912年春，他离开部队，回到家乡，和扛长工时的伙伴[[张凤鸣|张凤鸣]]、[[南成祥|南成祥]]、[[丁增华|丁增华]]等组织[[刀客|刀客]]队伍，后成为“[[渭北十八娃|渭北十八娃]]造反”中实力最强的农民武装。<ref name=sxsz/>

1916年，[[陈树藩|陈树藩]]任陕西督军，将[[郭坚|郭坚]]的陕西游击军改编为全省警备军。“[[渭北十八娃|渭北十八娃]]”中的[[林茂森|林茂森]]参加警备军，任第六营营长。李虎臣遂率部加入林茂森营，任营副，驻[[彬县|彬县]]。同年秋，李虎臣回[[临潼|临潼]]探亲期间，[[林茂森|林茂森]]因部队发生内讧而被部下杀害。李虎臣随即回到部队，被推为第六营营长。不久，[[陈树藩|陈树藩]]派[[郭金榜|郭金榜]]部攻[[彬县|彬县]]试图消灭李虎臣营，结果郭金榜失败。此时有人向[[郭坚|郭坚]]称，是李虎臣策划害死了林茂森。李虎臣因惧怕被暗算，遂率全营离开彬县，向东投[[胡景翼|胡景翼]]，任胡景翼部第四营营长，驻[[临潼县|临潼县]]关山、武屯、康桥（今均属[[西安市|西安市]][[阎良区|阎良区]]）一带。<ref name=sxsz/>

===三原起义之后===
1918年初， [[陕西靖国军|陕西靖国军]]兴起。李虎臣率部参加了1918年1月25日[[张义安|张义安]]、[[董振五|董振五]]、[[邓宝珊|邓宝珊]]等领导的[[三原起义|三原起义]]， 任靖国军右翼[[胡景翼|胡景翼]]部步兵第二团团长。2月，他作为右翼先锋，随第二游击司令[[张义安|张义安]]讨伐[[陈树藩|陈树藩]]，后和张义安一同说降了陈树藩部团长[[樊钟秀|樊钟秀]]。此后陈树藩因处于劣势，以陕西省长之位诱得[[刘镇华|刘镇华]]率[[镇嵩军|镇嵩军]]前来支援。李虎臣退兵防守[[临潼县|临潼县]]渭北地区。4月下旬，陈树藩在镇嵩军的配合下，向靖国军发起进攻。李虎臣在关山西南迎击，还增援介坊和青寺的靖国军[[杨虎城|杨虎城]]、[[王祥生|王祥生]]部。陈树藩部被迫撤退。 4月25日，李虎臣部攻下陈树藩军的阵地[[固市镇|固市镇]]，生擒了陈树藩军的营长[[刘登科|刘登科]]，在关山战役中取得胜利。 8月，[[于右任|于右任]]就任陕西靖国军总司令，靖国军整编为六路，李虎臣任[[胡景翼|胡景翼]]所率第四路的第四支队司令。 1921年8月，[[冯玉祥|冯玉祥]]督陕西，胡景翼部被收编为陕西陆军第一师，李虎臣部成为该师第一团。<ref name=sxsz/>

===直奉战争===
1922年[[第一次直奉战争|第一次直奉战争]]爆发，冯玉祥通电讨奉。李虎臣率部随冯玉祥、胡景翼东出[[潼关|潼关]]。冯玉祥军击败[[赵倜|赵倜]]军后，胡景翼师驻[[彰德|彰德]]（今[[安阳|安阳]]）。1924年，[[第二次直奉战争|第二次直奉战争]]爆发，李虎臣和胡景翼密议联合[[冯玉祥|冯玉祥]]、[[孙岳|孙岳]]举行起义，推翻[[直系|直系]]的统治。10月22日，[[冯玉祥|冯玉祥]]、[[胡景翼|胡景翼]]、[[孙岳|孙岳]]等发动[[北京政变|北京政变]]。李虎臣率部在[[丰台|丰台]]、[[通州|通州]]一线布防。[[国民军|国民军]]成立后，李虎臣任胡景翼所率的国民军第二军的第十师师长，并被[[孙中山|孙中山]]授予[[少将|少将]]军衔。他随即随胡景翼到[[河南|河南]]。1925年2月，[[胡憨之战|胡憨之战]]爆发。李虎臣率部在[[河南|河南]][[巩县|巩县]][[黑石关|黑石关]]击败[[镇嵩军|镇嵩军]]，[[憨玉琨|憨玉琨]]兵败自杀。此后，李虎臣率部驻防豫西[[陕州|陕州]]、[[灵宝|灵宝]]一带。<ref name=sxsz/><ref name=lai>{{Cite book<!--|和書-->|author = 来新夏等|title = 北洋軍閥史 下冊|year = 2000|publisher =  [[南開大学|南開大学]]出版社|isbn = 7-310-01517-7}}</ref><ref name=tian>{{Cite book<!--|和書-->|author = 田子渝・劉徳軍|title = 中国近代軍閥史詞典|year = 1989|publisher =  档案出版社|isbn = 7-5626-0244-1}}</ref><ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

1925年5月， 陕西掀起驱逐陕西军务督办[[吴新田|吴新田]]的运动，并派代表到河南邀[[国民军|国民军]]入[[潼关|潼关]]。7月，李虎臣部和国民军第三军[[孙岳|孙岳]]部入陕西威胁[[西安|西安]]， 吴新田逃走。8月，北京政府任命孙岳为陕西军务督办，李虎臣为陕西军务帮办。12月，孙岳离开陕西。<ref name=sxsz/><ref>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>1926年1月9日，临时执政[[段祺瑞|段祺瑞]]调[[孙岳|孙岳]]为直隶军务督办，任命李虎臣为陕西军务督办。<ref>西安通览，陕西人民出版社，1993年，第449页</ref>

与此同时，[[吴佩孚|吴佩孚]]、[[张作霖|张作霖]]全面进攻[[国民军|国民军]]。[[胡景翼|胡景翼]]于1925年病逝后，接任的国民军第二军军长[[岳维峻|岳维峻]]在鲁西南作战失利，后来在豫西接近全军覆没。李虎臣奉命驰援，于同年2月任国民軍第二軍第10師師長，但也在豫西遭到失败，于1926年春只身自山西回到陕西。<ref name=sxsz/>

===二虎守长安===
[[镇嵩军|镇嵩军]]的[[刘镇华|刘镇华]]被吴佩孚任为讨贼联军陕甘军总司令，率部在[[吴佩孚|吴佩孚]]、[[阎锡山|阎锡山]]的支援和陕军叛变将领[[麻振武|麻振武]]等接应下入陕西，企图夺取[[西安|西安]]。3月27日李虎臣回到西安，3月29日他赴[[三原|三原]]召集[[杨虎城|杨虎城]]、[[田玉洁|田玉洁]]、[[卫定一|卫定一]]、[[邓宝珊|邓宝珊]]等开军事会议，决定李虎臣辞去陕西军务督办职务并通电下野，推陕西军务帮办兼陕北镇守使[[井岳秀|井岳秀]]“维持全省治安”，并守“中立”，企图通过让步换取刘镇华放弃进入陕西。但刘镇华并未理会，而是限令李虎臣部于4月13日撤出西安。4月12日，刘镇华军推进到西安城外，4月17日对西安形成南北东三面包围，5月15日将西安四面包围。西安守军最初只有李虎臣部以及国民军第二军第十二混成旅[[卫定一|卫定一]]部两个团，人数不到五千人。后来李虎臣派人到三原和国民军第三军第三师师长[[杨虎城|杨虎城]]商量。在[[中国共产党|中国共产党]]党员[[魏野畴|魏野畴]]的参与下，他们决定坚守[[西安|西安]]、 [[三原|三原]]、[[咸阳|咸阳]]等地。4月18日，杨虎城率所部五千人开入西安。“[[二虎守长安|二虎守长安]]”开始。<ref name=sxsz/>

守卫西安的军事将领于5月19日举行会议，决定守军改称陕军，取消国民军第二、第三军番号。会议推李虎臣为陕军总司令兼第一师师长；[[杨虎城|杨虎城]]、[[田玉洁|田玉洁]]为副司令，并分别兼任第三、 第二师师长；[[邓宝珊|邓宝珊]]为指挥；[[卫定一|卫定一]]为副指挥兼第四师师长。5月20日，他们宣布就职，并任命李虎臣部卫队团团长[[史铸|史铸]]兼西安城防司令；李虎臣部旅长[[丁增华|丁增华]]为军警联合督察处处长，杨虎城部旅长[[姬汇伯|姬汇伯]]为副处长。守城期间，李虎臣和杨虎城一度同住[[广仁寺|广仁寺]]。由于被围困八个月，城内物资奇缺，粮食不足，饥荒严重，死伤无数。11月，国民联军抵达西安城外，李虎臣和杨虎城率部出城反击，[[镇嵩军|镇嵩军]]溃逃，11月28日西安解围。12月初，李虎臣和杨虎城等人率部将刘镇华部驱逐出陕西。<ref name=sxsz/>

===从反冯到反蒋===
1927年春，国民联军改编为[[国民革命军第二集团军|国民革命军第二集团军]]，参加[[北伐|北伐]]，[[冯玉祥|冯玉祥]]任总司令。1927年（民国16年）6月，李虎臣任国民革命軍西北軍（第二集团軍南路軍）第八路总司令，第八路辖四个师，驻[[周至|周至]]、[[户县|户县]]、[[商洛|商洛]]一带休整。7月，冯玉祥促成[[宁汉合流|宁汉合流]]后，解散了国民革命军第二集团军驻陕总司令部，和原陕军的矛盾加深。10月，陕军将领在三原开会，推举[[岳维峻|岳维峻]]为总司令，[[冯子明|冯子明]]为渭河以北总指挥，李虎臣为渭河以南总指挥，以反对冯玉祥。冯玉祥部[[宋哲元|宋哲元]]进攻陕军并占领[[三原|三原]]，渭北反冯战争以失败告终。1928年5月上旬，李虎臣和[[胡景通|胡景通]]、[[耿庄|耿庄]]、[[段懋功|段懋功]]等部再次发动反冯战争，后来又告失败，便于1928年12月将余部交第三师师长[[姜宏谟|姜宏谟]]指挥，自己举家迁居[[上海|上海]]。<ref name=sxsz/>

1929年陕西发生严重旱灾，在[[上海|上海]]的陕西人组成以[[焦子静|焦子静]]、李虎臣等为委员的赈灾会。1930年，他任国民革命军新编第三师师长，驻[[湖北|湖北]][[沙市|沙市]]。此後，随着裁军，1930年（民国19年）他改任第34師副師長。1931年（民国20年），他重任新編第3師師長。因为不愿进攻[[湘鄂西苏区|湘鄂西苏区]]和[[中国工农红军|中国工农红军]]作战，他借故[[王俊杰|王俊杰]]旅兵变而通电下野。[[蒋介石|蒋介石]]将李虎臣旧部缩编为一个旅，后又密令[[汤恩伯|汤恩伯]]收编了李虎臣最后的部队。后来李虎臣任国民政府军事参议院参议。他曾经在上海、[[天津|天津]]、[[汉口|汉口]]等地居住。1935年，李虎臣回到家乡。<ref name=sxsz/><ref name=lai/><ref name=tian/><ref name=xu/>

1936年[[西安事变|西安事变]]时，适逢[[二虎守长安|二虎守长安]]十周年，李虎臣受杨虎城邀请到西安，参与商议重要决策。1937年，杨虎城被迫辞职出国考察，李虎臣亲自将杨虎城送到上海。杨虎城送给李虎臣两万元，为李虎臣在西安[[员家巷|员家巷]]购买了一所住宅。[[抗日战争|抗日战争]]期间，李虎臣曾任陕西省第一、二届临时参议会参议员、国民政府顾问等职务。1940年，李虎臣回家乡临潼居住，在家乡捐资兴建了[[新兴堡|新兴堡]][[云兴高级小学|云兴高级小学]]和[[房村高级小学|房村高级小学]]。 1946年5月，国民政府授予他陆军中将銜，并授[[抗战胜利勋章|胜利勋章]]。<ref name=sxsz/>

[[第二次国共内战|第二次国共内战]]时期，他和[[杜斌丞|杜斌丞]]、[[成柏仁|成柏仁]]等进步人士交往密切，支持创办《秦风·工商日报联合版》。1948年，他被[[胡宗南|胡宗南]]任命为陕西省自卫总团司令，但他拒不就任。[[胡宗南|胡宗南]]溃逃时，曾派人到李虎臣在西安的住宅，图谋将李虎臣用飞机劫走，李虎臣闻讯躲避到临潼。<ref name=sxsz/>
 
[[中华人民共和国|中华人民共和国]]成立后，李虎臣任西安市第二、第三届各界人民代表会议协商委员会委员。<ref name=sxsz/>

1954年，李虎臣病逝于西安。享年65岁。<ref name=sxsz/>
