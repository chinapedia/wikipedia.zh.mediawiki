'''李殷衡'''（{{bd|9世纪||10世纪|catIdx=L}}），[[中国|中国]][[五代十国|五代十国]]时期政权[[南汉|南汉]]官员，拜为[[宰相|宰相]]。

== 家世 ==
李殷衡生年不详。他来自显赫的官僚家族[[赵郡李氏|赵郡李氏]]，曾祖父[[李吉甫|李吉甫]]和祖父[[李德裕|李德裕]]都曾任宰相。<ref name=NHS9>[http://archive.org/stream/02079314.cn#page/n40/mode/2up 《南汉书》卷九]</ref>李德裕在[[唐武宗|唐武宗]]年间尤其有权，掌控国政。<ref>[[:s:舊唐書/卷174|《旧唐书》卷一百七十四]]</ref>武宗崩，[[唐宣宗|唐宣宗]]继位后，李德裕失权，最终被贬，儿子[[李烨_(唐朝)|李烨]]即李殷衡的父亲一同被贬。因此李殷衡不能在早年得到重要官职。<ref name=NHS9/>

唐朝灭亡后，李殷衡效力[[后梁|后梁]]开国皇帝[[梁太祖|梁太祖]]，为[[右补阙|右补阙]]。<ref name=SGCQ62>[http://archive.org/stream/06080956.cn#page/n130/mode/2up 《十国春秋》卷六十二]</ref>[[开平_(后梁)|开平]]二年（908年），太祖派膳部郎中[[赵光裔|赵光裔]]充官告使，李殷衡为副，出使控制[[清海军|清海军]]且名义附庸于梁的军阀[[刘隐|刘隐]]，授其清海军和[[静海军节度使|静海军节度使]]。赵光裔和李殷衡到清海军后，刘隐将他们留在自己的幕府，没有送他们回梁。<ref name=ZZTJ267>[[:s:資治通鑑/卷267|《资治通鉴》卷二百六十七]]</ref>李殷衡因[[中原|中原]]多兵戈，于是乐于安居[[岭南|岭南]]。刘隐对他敬重礼待，多次让他参与军事规划，署为节度判官。<ref name=NHS9/>

== 效力刘䶮 ==
[[贞明_(后梁)|贞明]]三年（917年），刘隐的弟弟和继任者[[刘䶮|刘䶮]]称大越皇帝（不久改国号汉，故史称南汉）。他任李殷衡、赵光裔和[[杨洞潜|杨洞潜]]为宰相，授[[同中书门下平章事|同中书门下平章事]]，以李殷衡为礼部侍郎。<ref name=ZZTJ270>[[:s:資治通鑑/卷270|《资治通鉴》卷二百七十]]</ref>不久，他卒于任上，但卒年不详。<ref name=SGCQ62/>

== 轶闻 ==
李殷衡的姐夫唐朝宰相[[刘瞻|刘瞻]]有子[[刘赞|刘赞]]，少孤，也不聪明，李殷衡督促他读书，常打他，也没有长进。一次，刘赞逃到[[嵩山|嵩山]]，遇到一白衣老人，老人说：“给你开心，聪明必定过人十倍。”从此刘赞每天背诵一卷书，又有文才，后来中[[进士|进士]]，后梁年间充崇政院学士，还对李殷衡念念不忘，也是一桩异事。

== 评价 ==
*《[[十国春秋|十国春秋]]》论曰：（李）殷衡为卫公之后，左右霸王，无咎无誉。
