'''李剛'''（{{bd|1955年|7月|||catIdx=Li,李}}），[[山东|山东]][[莱阳|莱阳]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，曾任[[中央人民政府驻澳门特别行政区联络办公室|中央人民政府驻澳门特别行政区联络办公室]]主任、[[国务院侨务办公室|国务院侨务办公室]]副主任（正部长级）。中共第十七、十八届中央纪委委员，第十二届全国人大代表。

== 生平 ==
1970年至1974年，为[[大连外国语大学|辽宁外语专科学校]]英语系学生，大学普通班学历。1974年1月参加工作。1974年至1978年，任辽宁外语专科学校英语教师、团委书记。1978年至1984年，任[[中国共产主义青年团中央委员会|共青团中央]]国际联络部翻译、副处长、处长，[[全国青联|全国青联]]国际部副部长。1984年至1993年，任共青团中央国际联络部副部长、部长，全国青联常委、副秘书长，共青团中央常委<ref name=ren>{{Cite web |url = http://news.ifeng.com/a/20160719/49417657_0.shtml |title = 李刚任国侨办副主任(正部长级) 任启亮不再担任 |author =  |date =2016-07-19  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

1993年至1994年，任[[中华人民共和国文化部|文化部]]对外文化联络局副局长（正局级）。1994年至1998年，任[[中国驻美国大使馆|中国驻美国大使馆]]正局级[[公使衔|公使衔]][[文化参赞|文化参赞]]。1998年至2001年，任文化部对外文化联络局局长、港澳台文化事务司司长，[[中国对外文化交流协会|中国对外文化交流协会]]秘书长。2001年至2003年，任[[中央对外宣传办公室|中央对外宣传办公室]]副主任、[[国务院新闻办公室|国务院新闻办公室]]副主任，[[中国人权研究会|中国人权研究会]]副会长<ref name=ren/>。

2003年至2012年，任[[中央人民政府驻香港特别行政区联络办公室|中央人民政府驻香港特别行政区联络办公室]]副主任<ref name=ren/>。2012年12月19日，国务院任命李刚为[[中央人民政府驻澳门特别行政区联络办公室|中央人民政府驻澳门特别行政区联络办公室]]副主任<ref>{{Cite web |url = http://news.ifeng.com/mainland/special/renshitiaozheng/content-3/detail_2012_12/19/20321526_0.shtml |title = 国务院任命李刚为中央驻澳门联络办公室副主任 |author =  |date = 2012-12-19 |publisher =  凤凰网|language =  |accessdate =  |quote =  }}</ref>。2014年1月9日，国务院任命李刚为中央人民政府驻澳门特别行政区联络办公室主任<ref>{{Cite web |url =  http://www.chinanews.com/ga/2014/01-09/5716363.shtml|title =国务院任命李刚为中央驻澳门联络办公室主任  |author =  |date = 2014-01-09 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。2016年7月，任[[国务院侨务办公室|国务院侨务办公室]]副主任（正部长级）、党组成员<ref name=ren/><ref>{{cite news||title=李刚任国侨办副主任(正部长级) 任启亮不再担任|url=http://district.ce.cn/newarea/sddy/201607/19/t20160719_13956532.shtml|date=2016-07-19|newspaper=中国经济网}}</ref>。

2007年，在[[中国共产党第十七次全国代表大会|中国共产党第十七次全国代表大会]]上当选[[中国共产党第十七届中央纪律检查委员会|第十七届中央纪律检查委员会]]委员。2012年11月14日，在[[中国共产党第十八次全国代表大会|中国共产党第十八次全国代表大会]]上当选为[[中国共产党第十八届中央纪律检查委员会|第十八届中央纪律检查委员会]]委员<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2012-11/14/c_113690850.htm  |title =中国共产党第十八届中央纪律检查委员会委员名单  |author =  |date = 2012-11-14 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。他是[[第十二届全国人大代表|第十二届全国人大代表]]<ref name=ren/>。

2017年8月22日，国务院免去李刚的国务院侨务办公室副主任职务<ref>{{Cite web |url = http://news.xinhuanet.com/renshi/2017-08/22/c_1121522382.htm |title = 国务院任免国家工作人员（2017年8月22日） |author =  |date =2017-08-22  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。2017年9月1日，[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]]第二十九次会议通过《第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告》，报告称“由[[广东省|广东省]]选出的[[第十二届全国人民代表大会代表名单|第十二届全国人民代表大会代表]]、国务院侨务办公室原副主任李刚，因涉嫌严重违纪，被责令辞去第十二届全国人民代表大会代表职务。2017年8月23日，广东省第十二届人大常委会第三十五次会议决定接受其辞职。”<ref>{{Cite web|url=http://www.npc.gov.cn/npc/xinwen/2017-09/01/content_2027933.htm|title=第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告(2017年9月1日第十二届全国人民代表大会常务委员会第二十九次会议通过)|date=2017-09-01|work=中国人大网}}</ref>。

2017年10月9日在[[北京|北京]]举行[[中国共产党第十八届中央纪律检查委员会|中国共产党第十八届中央纪律检查委员会]]第八次全体会议，全会审议并通过中央纪律检查委员会常务委员会关于李刚、[[曲淑辉|曲淑辉]]同志严重违纪问题的审查报告，确认中央纪律检查委员会常务委员会之前作出的给予李刚同志留党察看一年处分的决定<ref>{{Cite web |url =http://www.guancha.cn/politics/2017_10_09_430279.shtml  |title =军委后勤保障部副部长刘生杰被撤销中纪委委员职务  |author =  |date =2017-10-09  |publisher =观察者网  |language =  |accessdate =  |quote =  }}</ref>。

== 家庭 ==
