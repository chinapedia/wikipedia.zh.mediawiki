'''李泉'''（{{bd|1969年|10月12日||}}）出生於上海，是[[中國大陸|中國大陸]]的創作型男歌手，曾為[[林憶蓮|林憶蓮]]、[[范曉萱|范曉萱]]、[[劉德華|劉德華]]、[[梁詠琪|梁詠琪]]等歌手創作歌曲。初二就已經創作出了第一首歌曲《停留》。創作歌曲時他通常用'''李燃'''作為名字，原因是希望給別人燃燒和心如止水的感覺。<ref>{{Cite web |url=http://music.ent.tom.com/1026/1367/20031015-25578.html |title=存档副本 |access-date=2009-07-12 |archive-url=https://web.archive.org/web/20060626185704/http://music.ent.tom.com/1026/1367/20031015-25578.html |archive-date=2006-06-26 |dead-url=yes }}</ref>

2015年7月以灵魂战警面具参加[[蒙面歌王_(中国)|蒙面歌王]]，在8月9日成為第二位被淘汰歌手，但他卻是第一位接受“勇敢揭面”的歌手。<br/>
2018年參加[[歌手2018|歌手2018]]節目，他精湛的琴藝及高超的演繹歌曲能力，獲得廣大迴響，並有中國的[[約翰·傳奇|約翰·傳奇]]之稱號。

== 燃工作室 ==
燃工作室由李泉于2001年在上海成立，目的是希望推動華語流行音樂的發展，至今已經創作出許多音樂作品。

== 情感 ==
1995年李泉剛出道時初識當時還是主持人的[[柯藍|柯藍]]。直到2000年雙方開始談戀愛<ref>http://www.ynet.com/net/view.jsp?oid=2696402{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，2007年和平分手。之間曾有結婚傳言，但遭兩人否認。外傳分手是因為柯藍和[[耿樂|耿樂]]同居，[[柯藍|柯藍]]則說因為李泉希望她放棄事業，是對方“一廂情願”。<ref>http://news.163.com/07/0511/14/3E7IJEP2000120GU.html</ref>
