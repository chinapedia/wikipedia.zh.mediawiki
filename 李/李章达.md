'''李章达'''（{{bd|1890年||1953年|12月9日}}）字'''南溟'''，[[广东省|广东省]][[东莞县|东莞县]]人。[[中华民国|中华民国]]及[[中华人民共和国|中华人民共和国]]政治人物<ref name=dg>{{Cite web |url = http://www.gzsdfz.org.cn/gzrw/lsmr/201601/t20160113_36523.html |title = 李章达 |author =  |date = 2016-01-13 |publisher =广州市情网 |language =  |accessdate =  |quote =  }}</ref><ref name=huanqiu>{{Cite web |url = http://opinion.huanqiu.com/plrd/2016-02/8516105.html |title = 李章达的抗战 |author =  |date = 2016-02-06 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
李章达幼年父母双亡，靠叔父抚养。少年时进入东莞县城的“子曰馆”，后来进[[私塾|私塾]]读[[四书五经|四书五经]]。[[清朝|清朝]][[光绪|光绪]]三十二年（1906年），进入广东黄埔陆军小学。[[宣统|宣统]]元年（1909年）毕业后，升入[[南京|南京]]陆军第四中学。1911年加入[[中国同盟会|中国同盟会]]，并参加了[[武昌起义|武昌起义]]。1911年11月，李章达等南京陆军第四中学学生，与[[武昌区|武昌]]陆军第三中学学生合编，充当[[黄兴|黄兴]]的卫队，驻[[汉阳区|汉阳]]昭忠祠总司令部<ref name=dg/>。

民国元年（1912年）初，李章达从[[武汉|武汉]]回到广州，随[[朱执信|朱执信]]编练民军，并被派到民军中担任教练员。1912年6月，李章达考入[[保定陆军军官学校|保定陆军军官学校]]炮兵科。1913年参加广东[[二次革命|讨袁（世凯）运动]]。讨袁失败后逃到[[上海|上海]]。1917年随[[孙中山|孙中山]]返回广东，参加[[护法运动|护法运动]]。1918年，出任援闽粤军总参谋部少校参谋，随粤军开入[[福建省|福建省]]与[[北洋军阀|北洋军阀]]作战。1918年底[[1919年南北和谈|南北议和]]后，赴上
海协助[[朱执信|朱执信]]开展《建设》杂志发行工作。1920年底，被[[廖仲恺|廖仲恺]]派往[[苏联|苏联]]学习，途中在[[赤塔|赤塔]]受战乱阻碍，乃返[[天津|天津]]，辗转返回广州。1921年春，出任粤军第一师工兵营长，后来任[[孫中山大元帥府紀念館|大元帅府]]警卫团团长、讨逆军支队长、大元帅府参军。1922年6月[[陈炯明|陈炯明]]发动[[六一六事变|六一六事变]]，孙中山迁避[[永丰舰|永丰舰]]。次日，李章达赶到永丰舰，参加了讨伐陈炯明的军事活动<ref name=dg/>。

1924年[[中国国民党第一次全国代表大会|中国国民党第一次全国代表大会]]召开后，李章达被任命为[[中国国民党中央党部|中国国民党中央党部]]秘书，协助廖仲恺办理党务。1925年，任[[广州国民政府|广州国民政府]]惩吏院委员。1926年4月，任[[广州市公安局|广州市公安局]]局长，任内整顿治安。广州国民政府发动[[北伐战争|北伐战争]]后，李章达留守广州，为[[北伐军|北伐军]]招募夫役送往北伐前线。1927年4月，[[蒋介石|蒋介石]]发动[[四一二政变|四一二政变]]后，李章达辞去一切职务，举家迁居[[英属香港|香港]]。其间，[[汪精卫|汪精卫]]曾派[[陈公博|陈公博]]请李章达参与政事，被李章达拒绝<ref name=dg/>。

1931年6月，李章达赴[[赣州|赣州]]会见[[国民革命军第十九路军|国民革命军第十九路军]]将领[[蒋光鼐|蒋光鼐]]、[[蔡延锴|蔡延锴]]，劝他们联合[[广东|广东]]、[[广西|广西]]地方势力共同反对[[蒋介石|蒋介石]]。后来，李章达随十九路军赴上海，在十九路军[[一·二八事变|一·二八抗战]]中，协助蒋光鼐筹划抗击日军。十九路军调往福建后，李章达从上海返回香港。1933年1月，李从香港赴福建，任[[福建省政府|福建省政府]]委员、秘书长。1933年11月间，[[中华共和国人民革命政府|中华共和国人民革命政府]]在[[福州|福州]]成立。李章达被推选为政府中央委员兼政治保卫局局长。[[闽变|闽变]]失败后，李章达继续避居香港<ref name=dg/>。

1936年5月，[[沈钧儒|沈钧儒]]等人在上海成立“[[全国各界救国联合会|全国各界救国会]]”。随后李章达等人在香港成立“全国各界救国会华南区总部”（简称“南总”），李章达任主任委员。其间，广州市及各地县先后成立“南总广州救亡运动协进会”<ref name=dg/>。

1937年[[抗日战争|抗日战争]]全面爆发后，[[中国国民党|中国国民党]]当局迫于形势下令恢复因政治原因而被开除出党的所有党员的党籍，并宣布赦免过去的政治反对派。[[李济深|李济深]]、[[陈铭枢|陈铭枢]]、蒋光鼐、蔡廷锴、李章达等人回到中国国民党内参加抗战。1938年10月，[[日军|日军]]攻占广州。此后适[[叶挺|叶挺]]辞去“广东第四游击司令”职务，[[抗日战争第四战区|第四战区]]命李章达继任，李章达到[[深圳|深圳]]设立指挥机构<ref name=huanqiu/>。

1939年春，李章达被被调回[[韶关|韶关]]，任[[抗日战争第四战区|第四战区]]军法执行中将总监，任内曾经秘密协助和支持[[中共|中共]]领导下的活动<ref name=dg/>。1939年9月，李章达因“不肯填表重新加入国民党”而被迫辞去军法执行总监的职务。当时蒋介石命所有军事单位转令一切军职人员入党编组，李章达告诉[[张发奎|张发奎]]（张发奎也是李章达的学生）：“我既在福州时声明脱离国民党党籍，此时无理由填表加入”。张发奎乃令李章达速递辞呈<ref name=huanqiu/>。

1941年，李章达参与筹组[[中国民主政团同盟|中国民主政团同盟]]（后更名[[中国民主同盟|中国民主同盟]]，简称“民盟”）及创办《[[光明日报|光明日报]]》。1945年[[春节|春节]]期间，李章达与[[李伯球|李伯球]]等人在[[梅县|梅县]]举行民盟东南干部会议，成立了“民盟东南干部会议常设机关”，李章达获推选为常设机关主任。1946年1月，民盟南方总部在香港成立，李章达任主任委员。同年6月23日，李章达与[[何香凝|何香凝]]等名人分别致电[[毛泽东|毛泽东]]及[[美国总统|美国总统]][[哈里·S·杜鲁门|杜鲁门]]，呼吁和平，制止内战。1948年，李章达参与组建[[中国国民党革命委员会|中国国民党革命委员会]]（简称“民革”），任民革第一届中央常务委员兼秘书长。1949年初，应[[中共中央|中共中央]]邀请，李章达代表[[中国人民救国会|中国人民救国会]]到[[北平|北平]]参加[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]<ref name=dg/>。

[[中华人民共和国|中华人民共和国]]成立后，李章达历任[[中华人民共和国中央人民政府_(1949-1954)|中央人民政府]]委员、[[广东省人民政府|广东省人民政府]]副主席兼[[广州市|广州市]]人民政府副市长、[[中南军政委员会|中南军政委员会]]行政委员会委员、广东各界人民代表会议协商委员会副主席。1951年12月30日，[[中共中央华南分局|中共中央华南分局]]决定成立政治、文教、民族事务委员会，李章达被任命为文教、民族事务委员会主任<ref name=dg/>。

1953年12月9日，李章达在广州病逝<ref name=dg/>。
