'''李齐物'''（{{bd|7世纪||761年|7月5日|catIdx=L}}），[[表字|字]]'''道用'''，[[唐朝|唐朝]][[宗室|宗室]]淮安王[[李神通|李神通]]曾孙，[[盐州|盐州]][[刺史|刺史]][[李孝锐|李孝锐]]孙（《新唐书》误作李孝锐子），<ref name=BT112>[[:s:旧唐书/卷112|《旧唐书》卷一百一十二]]</ref><ref name=NBT78>[[:s:新唐书/卷78|《新唐书》卷七十八]]</ref>[[陇州|陇州]]司仓赠[[弘农郡|弘农郡]]太守李璟子。<ref name=YZQ>[[:s:金紫光禄大夫守太子太傅兼宗正卿赠司空上柱国陇西郡开国公李公神道碑铭|颜真卿《金紫光禄大夫守太子太傅兼宗正卿赠司空上柱国陇西郡开国公李公神道碑铭》]]</ref>

== 早期仕途 ==
李齐物没有学术，为官严整。唐[[神龍_(年號)|神龙]]初年，左右[[千牛备身|千牛备身]]出身，历任尚辇直长、[[许州|许州]]司马、[[华州|华州]]司兵。其母过世丁忧，因为丧期哀伤数月导致身体虚弱，在家都扶着拐杖，守墓时在草庐衣不解带直至丧期满。皇帝特地降下玺书，给予在草庐守孝的他慰勉。丧期满后，授鸿胪丞，除尚辇奉御，迁升北都军器监事，太原少尹[[严挺之|严挺之]]连奏李齐物政绩第一，于是擢拜长安令。李齐物在任上以威禁暴，以恕用刑。迁将作少匠、殿中少监、太府少卿。<ref name=YZQ/>

== 唐玄宗年间 ==
[[开元|开元]]二十四年（736年）任[[怀州|怀州]]刺史。举荐[[高尚_(唐朝)|高尚]]，但高尚不得仕官，于是送高尚入京师，助钱三万，并写信给中官将军吴怀实以高尚相托，<ref>[[:s:旧唐书/卷200上|《旧唐书》卷二百上]]</ref>使其得见当权[[宦官|宦官]][[高力士|高力士]]。<ref>[[:s:新唐书/卷215上|《新唐书》卷二百一十五上]]</ref>约二十九年（741年），李齐物累次升官至[[陕州|陕州]]刺史。他曾不满陕尉[[裴冕|裴冕]]，将他上枷锁折辱。[[天宝_(唐朝)|天宝]]元年（742年）正月，李齐物开砥柱，通漕路，凿三门，成渠。<ref>[[:s:旧唐书/卷9|《旧唐书》卷九]]</ref>李齐物在挖开重石时挖出像铧一样的古铁戟，铭文篆字“平陆”，献上，诏命名河北县为平陆县。<ref>[[:s:旧唐书/卷38|《旧唐书》卷三十八]]误作三年（744年）事。</ref><ref>[[:s:新唐书/卷38|《新唐书》卷三十八]]</ref>李齐物将石头弃入河中，水越发湍急，舟船不能入新门，等候水涨了由人輓舟而上。[[唐玄宗|唐玄宗]]怀疑其事，遣宦者视察，李齐物厚赂使者，使者回去后说李齐物兴修水利是惠政。玄宗于是赐李齐物貂裘一领、绢三百匹，特加李齐物[[银青光禄大夫|银青光禄大夫]]，为[[鸿胪卿|鸿胪卿]]，赐玉尺一把，诏称因李齐物能干，故有此赐。三年（744年）十一月，迁河南尹，<ref>[[:s:新唐书/卷53|《新唐书》卷五十三]]</ref><ref>《[[唐会要|唐会要]]》卷八十七</ref>仍任水陆运使。五年（746年）七月，李齐物因与被[[宰相|宰相]][[李林甫|李林甫]]陷害而被贬的右相[[李适之|李适之]]交好，被贬[[竟陵郡|竟陵郡]]太守。<ref>[[:s:资治通鉴/卷215|《资治通鉴》卷二百一十五]]</ref>任内见到伶师[[陆羽|陆羽]]，欣赏其才能，当即赠与诗书，推荐他到隐居于火门山的邹夫子处学习。<ref>[[:s:新唐书/卷196|《新唐书》卷一百九十六]]</ref><ref>[[:s:陆文学自传|:s:陆文学自传]]</ref>天宝末年，李齐物被征拜将作监，又为司农、鸿胪卿。<ref name=BT112/><ref name=NBT78/><ref name=YZQ/>

== 唐肃宗年间 ==
[[至德_(唐朝)|至德]]初年，裴冕为相，除授李齐物[[太子宾客|太子宾客]]，世人称赞裴冕能不以怨报怨。[[乾元_(唐朝)|乾元]]元年（758年），李齐物迁[[凤翔|凤翔]]尹，十月为[[刑部尚书|刑部尚书]]，历任[[太常卿|太常卿]]。褒扬[[晋州|晋州]]长史[[崔衍|崔衍]]之忠，崔衍得授长安令，封武邑县男。<ref>[[:s:新唐书/卷164|《新唐书》卷一百六十四]]</ref>[[上元_(唐肃宗)|上元]]元年（760年）不早于六月，李齐物迁[[京兆尹|京兆尹]]。<ref>[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》：李齐物前任[[郑叔清|郑叔清]]在六月被罢免，继任[[刘晏|刘晏]]七月上任。</ref>他性格苛察寡恩，喜欢揭发他人私事，但廉洁自爱，官吏不敢欺瞒他。不迟于七月，因残暴被罢免。<ref>[[:s:新唐书/卷162|《新唐书》卷一百六十二]]</ref>晚年除授[[太子太傅|太子太傅]]，兼[[宗正卿|宗正卿]]。二年（761年）五月卒，<ref>[[:s:旧唐书/卷10|《旧唐书》卷十]]</ref>时为[[金紫光禄大夫|金紫光禄大夫]]，辍朝一日，诏赠[[太子太师|太子太师]]。<ref name=BT112/><ref name=NBT78/><ref>[[:s:赠李齐物太子太师诏|:s:赠李齐物太子太师诏]]</ref>又赠[[司空|司空]]、[[上柱国|上柱国]]、'''陇西郡开国公'''。<ref name=YZQ/>

== 评价 ==
*《旧唐书》史臣曰：齐物贞廉整肃，复节制权谋；……皆神通之曾玄，宗室之翘楚。……然……齐物积财兴议，……皆尺之短也。<ref name=BT112/>
