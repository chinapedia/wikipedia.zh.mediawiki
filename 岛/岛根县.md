'''島根縣'''（{{jpn|j=島根県|hg=しまねけん|rm=Shimane ken|f=y}}）是日本西部[[日本海|日本海]]沿岸[[山陰地方|山陰地方]]的一個縣，縣廳所在地是[[松江市|松江市]]。多山的地形使得島根縣的人口密度和工業化程度均較低。但島根縣有著悠久的歷史，是日本古文化的發源地之一。島根縣的轄區範圍包括[[出雲國|出雲國]]、[[石見國|石見國]]、[[隱岐國|隱岐國]]三個舊令制國。現在島根縣也大多被分為出雲地方、石見地方、隱岐地方三個地區。在日本的行政區劃中，日本和韓國之間有爭議的領土[[獨島|竹島]]（韓方稱獨島）被劃入島根縣。

島根縣這一縣名來自於縣廳所在地松江市的松江城附近地區舊屬島根郡（嶋根郡）。在《[[出雲國風土記|出雲國風土記]]》中，嶋根一名由[[八束水臣津野命|八束水臣津野命]]命名<ref name="Name">{{cite book |author={{lang|ja|竹内理三}}|script-title=ja:《角川日本地名大辞典 32 島根県》 |year=1979 |publisher={{lang|ja|角川書店}} |location=東京都 |isbn=978-4-04-622945-8|language=ja}}</ref>{{rp|19}}。另外，松江市的轄區範圍橫跨島根郡（大橋川以北）和意宇郡（大橋川以南）。

== 歷史 ==
=== 遠古至江戶時代 ===
[[File:Iwami_Ginzan_Silver_Mine,_Shimizudani_Refinery_Ruins_001.JPG|thumb]]
島根縣內的[[中國山地|中國山地]]地區發現了舊石器時代的遺跡。[[繩文時代|繩文時代]]之後，島根縣內的遺跡無論數量還是規模都大幅增加。隱岐群島雖然和本州之間隔有大海，但發展了日本本土相同的文化。自但馬至石見的山陰海岸都發現了隱岐特產的黑曜石生產的石器跡<ref name="Name"></ref>{{rp|23}}。島根縣內的彌生文化系由[[九州_(日本)|九州]]北部經日本海傳來，縣內的[[彌生時代|彌生時代]]遺跡超過100處<ref name="Name"></ref>{{rp|23}}。彌生時代中期之後，島根的文化開始受到畿內和瀨戶內地區強烈的影響。在[[古墳時代|古墳時代]]，島根縣內發現了山代二子塚<ref>{{cite web |url=https://www.kankou-matsue.jp/about_matsue/rekishi/history/re4-09.html |script-title=ja:山代二子塚 |accessdate=2017-05-17 |publisher={{lang|ja|松江観光協会}} |language=ja}}</ref>、今市大念寺<ref>{{cite web |url=https://www.izumo-kankou.gr.jp/382 |script-title=ja:今市大念寺古墳 |accessdate=2017-05-17 |publisher={{lang|ja|今市大念寺古墳}} |language=ja}}</ref>等大型古墳。出雲地方的古墳以方形墳為主，在日本國內頗為特殊，顯示出雲地方在這一時期文化有很強的獨自特色<ref name="Name"></ref>{{rp|24}}。在[[奈良時代|奈良時代]]的《[[古事記|古事記]]》、《[[日本書紀|日本書紀]]》等神話書籍中，出雲也多次出場。[[出雲大社|出雲大社]]在這一時期也已經出現<ref name="Name"></ref>{{rp|25}}。

[[大化改新|大化改新]]之後，日本開始實施[[律令制|律令制]]，出雲、石見、隱岐三國也正式出現<ref name="Name"></ref>{{rp|26}}。《[[萬葉集|萬葉集]]》中有50首關於山陰地方的詩歌，其中大半都是石見國歌人[[柿本人麻呂|柿本人麻呂]]所作<ref name="Name"></ref>{{rp|27}}。進入[[鎌倉時代|鎌倉時代]]之後，幕府在全國設置[[守護|守護]]主管當地軍政。出雲、石見、隱岐三國的守護是近江的佐佐木氏<ref name="Name"></ref>{{rp|31}}。但在承久之亂過後，益田氏成為石見國實質的掌控者<ref name="Name"></ref>{{rp|31}}。隱岐群島在平安及鎌倉時代是流放地，[[後鳥羽上皇|後鳥羽上皇]]、[[後醍醐天皇|後醍醐天皇]]都曾被流放隱岐<ref name="Name"></ref>{{rp|32}}。

[[室町時代|室町時代]]時，京極氏曾擔任出雲、隱岐守護，大內氏曾出任石見守護。但在經歷戰亂之後，以富田城為據點的[[尼子氏|尼子氏]]最終控制了島根的大部分地區，成為中國地方的一大勢力。然而尼子氏在達到巔峰後很快就被[[毛利氏|毛利氏]]攻滅<ref name="Name"></ref>{{rp|33}}。[[關原之戰|關原之戰]]後，[[堀尾吉晴|堀尾吉晴]]被封為出雲、隱岐兩國的太守。他興建[[松江城|松江城]]並以此為統治據點，此後松江成為島根的中心城市<ref name="Name"></ref>{{rp|33}}。1638年（[[寬永|寬永]]15年），德川家康之孫{{link-ja|松平直政|松平直政}}被封為松江藩主。此後直到幕末，松平家都是松江藩的藩主，松江藩也因此成為[[親藩|親藩]]<ref name="Name"></ref>{{rp|34}}。隱岐雖然是[[天領|天領]]（幕府直轄地），但實際上在[[江戶時代|江戶時代]]時，大部分時間由松江藩管理<ref name="Name"></ref>{{rp|34}}。石見國則分為幕府直接管理的[[石見銀山|銀山]]領、濱田藩、津和野藩三個部分。石見銀山自14世紀就開始得到開發，是江戶時代日本最重要的銀礦之一<ref name="Name"></ref>{{rp|33}}。濱田藩主曾經多次變更，在幕末時由親藩松平家出任<ref name="Name"></ref>{{rp|34}}。津和野藩則在大多數時期由龜井氏出任藩主<ref name="Name"></ref>{{rp|34}}。

=== 近現代島根 ===
[[File:Shimane_prefectural_office(1909-1945).jpg|thumb]]
1868年，隱岐群島爆發當地民眾要求自治的{{link-ja|隱岐騒動|隠岐騒動}}。翌年2月，中央政府宣布成立隠岐縣。但僅半年之後隠岐縣就和濱田藩合併，成立大森縣<ref name="Name"></ref>{{rp|40}}。1871年，大森縣合併[[津和野藩|津和野藩]]並改名[[濱田縣|濱田縣]]<ref name="Name"></ref>{{rp|40}}。在正式[[廢藩置縣|廢藩置縣]]之後，出雲國分為松江、廣瀨、母里三縣。三縣體制只維持了四個月並合併為島根縣。新成立的島根縣的轄區還包括了自濱田縣劃入的隱岐<ref name="Name"></ref>{{rp|41}}。島根縣在成立一個月之後，隱岐被劃入[[鳥取縣|鳥取縣]]。1876年，島根、濱田、鳥取三縣合併，這一時期被稱為大島根縣。但由於鳥取縣士族的強烈反彈，1881年，鳥取縣再次獨立。島根縣轄區遂確定現在範圍<ref>{{cite web |url=http://www.pref.shimane.lg.jp/admin/seisaku/koho/profile/aramasi.html |script-title=ja:島根県誕生までのあらまし |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。

島根縣在確立了管轄範圍之後對縣內的町村和郡進行了大規模合併。由於島根縣交通不便且人口密度較低，近代化進程也相對較慢。1908年，島根縣才開通了縣內的首條鐵路<ref name="Name"></ref>{{rp|49}}。島根縣的產業近代化開始於明治後期和[[大正|大正]]時期，重點發展產業包括絲綢、漁業等產業<ref name="Name"></ref>{{rp|49}}。在[[第二次世界大戰|第二次世界大戰]]時期，島根雖然遭到美軍空襲的次數較少，但松江市為防範空襲而拆除了舊市區四分之一的住宅，導致戰後出現住宅緊缺<ref name="Name"></ref>{{rp|51}}。[[日本投降|日本投降]]之後，松江發生[[松江騷擾事件|松江騷擾事件]]，縣廳等重要建築被燒毀<ref name="Name"></ref>{{rp|51}}。1945年11月，盟軍開始佔領島根縣。1950年代初期，島根縣進行了大規模市町村合併，縣內的自治體數在1961年減少到8市34町19村<ref name="Name"></ref>{{rp|52}}。為擺脫後進性，島根縣在1950年代提出了[[中海|中海]]、[[宍道湖|宍道湖]]淡水化、石見臨海地帶等大型工業開發計劃，但島根縣的縣民所得和日本平均值的差距在1950年代卻反而擴大<ref name="Name"></ref>{{rp|53}}。1960年代，島根縣和鳥取縣合作，使中海地區被指定為新產業都市，促進了兩縣的工業發展<ref>{{cite web |url=http://www1.g-reiki.net/tottori/reiki_honbun/k500RG00000310.html |script-title=ja:中海地区新産業都市建設協議会共同設置規約 |accessdate=2017-05-17 |publisher={{lang|ja|鳥取県}} |language=ja}}</ref>。但另一方面，島根縣現在仍然面臨嚴重的過疏和人口減少等問題。

== 地理 ==
[[File:Matengai_of_Kuniga_Coast_in_Oki_Island_Shimane_pref600.jpg|thumb]]
島根縣面積6708.24平方公里，相當於日本總面積的約1.8%，是日本面積第19大縣<ref>{{cite web |url=http://www.gsi.go.jp/KOKUJYOHO/MENCHO/201610/area_todofuken.pdf |script-title=ja:第Ⅱ章　都道府県別面積 |accessdate=2017-05-16 |publisher={{lang|ja|国土地理院}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170608055310/http://www.gsi.go.jp/KOKUJYOHO/MENCHO/201610/area_todofuken.pdf |archivedate=2017年6月8日 |df= }}</ref>。島根縣位於[[本州|本州]]西部[[日本海|日本海]]東南岸的[[山陰地方|山陰地方]]，東面和[[鳥取縣|鳥取縣]]、南面和[[廣島縣|廣島縣]]、西面和[[山口縣|山口縣]]接壤，北面則是日本海。島根縣地形多山，因此森林覆蓋率達78.8%，與之相對的是耕地面積率則只有5.8%<ref>{{cite web |url=http://www.pref.shimane.lg.jp/industry/norin/kikan/toubu_norin/kyotu/toubunourinn.data/tobu_nourin_gaiyo.pdf |script-title=ja:島根県東部の農林業 |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。島根縣內的大部分山區都屬於[[中國山地|中國山地]]，縣內最高峰是位於島根、廣島兩縣交界處的{{link-ja|恐羅漢山|恐羅漢山}}，高1,346米<ref name="Name"></ref>{{rp|21}}。島根縣內缺乏大型平原。縣內面積最大的平原{{link-ja|出雲平原|出雲平野}}東西寬20公里，南北長5公里，是島根縣人口最密集的地區<ref name="Chishi">{{cite book |author={{lang|ja|森川洋·篠原重則·奥野隆史}}|script-title=ja:《日本の地誌9　中国・四国》 |year=2005 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16769-5|language=ja}}</ref>{{rp|168}}。島根縣東部的[[宍道湖|宍道湖]]和[[中海|中海]]是[[潟湖|潟湖]]，也是日本少有的連結汽水湖<ref>{{cite web |url=http://www.cgr.mlit.go.jp/izumokasen/jimusho/suikei-seibi/files/keikaku_00.pdf |script-title=ja:斐伊川水系河川整備計画 |accessdate=2017-05-17 |publisher={{lang|ja|国土交通省}} |language=ja}}</ref>。宍道湖和中海北側的[[島根半島|島根半島]]上有高400米至500米的山地，起到了保護出雲平原免受海蝕的作用<ref name="Chishi"></ref>{{rp|168}}。

島根縣的主要水系有發源自廣島縣，在島根縣入海的中國地方最大河川[[江之川|江之川]]<ref>{{cite web |url=https://www.cgr.mlit.go.jp/cginfo/syokai/busyo/kasen/suikei/gouno.htm |script-title=ja:江の川 |accessdate=2017-05-17 |publisher={{lang|ja|中国地方整備局}} |language=ja}}</ref>；島根縣流域面積最大的河川{{link-ja|斐伊川|斐伊川}}<ref>{{cite web |url=https://www.cgr.mlit.go.jp/cginfo/syokai/busyo/kasen/suikei/gouno.htm |script-title=ja:斐伊川 |accessdate=2017-05-17 |publisher={{lang|ja|中国地方整備局}} |language=ja}}</ref>。此外島根縣還有{{link-ja|高津川|高津川}}、{{link-ja|神戶川 (島根縣)|神戸川 (島根県)|神戶川}}等水系，但大多短小湍急。

島根縣的海岸線全長1,016公里<ref>{{cite web |url=http://www.pref.shimane.lg.jp/infra/river/sabo/sabou_rekishi/tisei.html |script-title=ja:島根県の地勢 |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。隱岐群島位於島根縣北部，距島根縣本土有40至80公里，由[[知夫里島|知夫里島]]、[[中之島_(島根縣)|中之島]]、[[西之島_(隱岐群島)|西之島]]、[[島後|島後]]四個大島和約180個小島組成<ref name="Name"></ref>{{rp|21}}。隱岐群島是由火山活動形成的島嶼，島上幾乎都是山地，擁有國賀海岸等極具特徵的海蝕崖地形，被劃入[[大山隱岐國立公園|大山隱岐國立公園]]加以保護<ref>{{cite web |url=https://www.env.go.jp/park/daisen/point/index.html |script-title=ja:公園の特長 |accessdate=2017-05-17 |publisher={{lang|ja|環境省}} |language=ja}}</ref>，並且還是世界地質公園<ref>{{cite web |url=http://www.oki-geopark.jp/about/ |script-title=ja:隠岐ユネスコ世界ジオパークの魅力 |accessdate=2017-05-17 |publisher={{lang|ja|隠岐ユネスコ世界ジオパーク}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20150116223028/http://www.oki-geopark.jp/about/ |archivedate=2015年1月16日 |df= }}</ref>。

島根縣可以按照人文歷史及地理特徵分為出雲、石見、隱岐三個地區。就面積來看，出雲佔島根縣總面積的40.5%、石見佔54.3%、隱岐佔5.2%。但就農地、宅地、人口比例來看，出雲地方則遠多於石見地方<ref name="Name"></ref>{{rp|22}}。島根縣人口最多的兩個城市[[松江市|松江市]]和[[出雲市|出雲市]]均位於出雲地方。石見地方人口最多城市[[濱田市|濱田市]]的人口只有出雲市的約三分之一。隱岐群島的人口則只有約2萬人<ref name="population1">{{cite web |url=http://pref.shimane-toukei.jp/upload/user/00019840-FSYbM6.pdf |script-title=ja:島根県推計人口月報 |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。
=== 氣候 ===
島根縣全境屬於[[日本海側氣候|日本海側氣候]]，但因地理位置偏西南，氣候較為溫和，只有部分山麓地區是豪雪地帶<ref>{{cite web |url=http://www.mlit.go.jp/crd/chisei/shimane.html |script-title=ja:島根県 |accessdate=2017-05-17 |publisher={{lang|ja|国土交通省}} |language=ja}}</ref>，呈現北陸型氣候和九州型氣候的折衷特徵。島根縣在冬季受西北季風的影響，自11月開始就多雨雪和陰天。但和[[北陸地方|北陸地方]]及鳥取縣相比，島根縣的降雪量要少很多。石見地方則受對馬暖流的影響，即使降雪也很少積雪<ref name="Name"></ref>{{rp|22}}。島根縣冬季氣溫也較日本海沿岸地區溫暖，以松江為例，一年少有出現全日氣溫在冰點以下的情況<ref>{{cite web |url=http://www.jma-net.go.jp/matsue/chisiki/column/cloud/winter.html |script-title=ja:冬の天気 |accessdate=2017-05-17 |publisher={{lang|ja|松江地方気象台}} |language=ja}}</ref>。島根縣在夏季受高氣壓控制時間較多，時有酷暑，但比山陽地方涼爽<ref>{{cite web |url=http://www.jma-net.go.jp/matsue/chisiki/column/front/summer.html |script-title=ja:夏の天気 |accessdate=2017-05-17 |publisher={{lang|ja|松江地方気象台}} |language=ja}}</ref>。島根縣的降水以梅雨時節的6、7月最多，其次是秋雨時節的9月<ref>{{cite web |url=http://www.jma-net.go.jp/matsue/chisiki/column/other/autumn.html |script-title=ja:秋の天気 |accessdate=2017-05-17 |publisher={{lang|ja|松江地方気象台}} |language=ja}}</ref>。
=== 行政區劃 ===
島根縣東部的出雲地區包括[[松江市|松江市]]、[[安來市|安來市]]、[[出雲市|出雲市]]、[[雲南市|雲南市]]、[[奥出雲町|奥出雲町]]、[[飯南町|飯南町]]。西部的石見地區包括[[濱田市|濱田市]]、[[江津市|江津市]]、[[益田市|益田市]]、[[津和野町|津和野町]]、[[吉賀町|吉賀町]]、[[大田市|大田市]]、[[川本町|川本町]]、[[美鄉町_(島根縣)|美鄉町]]、[[邑南町|邑南町]]，佔島根縣53.3%的面積，但人口只佔不到三成<ref>{{cite web |url=http://www.pref.shimane.lg.jp/shoko/sangyou/index.data/H21tokusyu_2.pdf |script-title=ja:石見地域の産業振興 |accessdate=2017-06-02 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。隱岐地區包括隱岐群島的[[海士町|海士町]]、[[西之島町|西之島町]]、[[知夫村|知夫村]]、[[隱岐之島町|隱岐之島町]]。

== 人口 ==
1920年日本首次人口普查時，島根縣有人口714,712人，排名日本第36位。在當時島根縣就是日本65歲以上人口比例最高的縣<ref>{{cite web |url=http://www.stat.go.jp/info/anotoki/pdf/census.pdf |script-title=ja:第１回国勢調査時（大正９年）といま  |accessdate=2017-05-17 |publisher={{lang|ja|統計局}} |language=ja}}</ref>。此後由於人口外流，島根縣人口增長緩慢。1955年時，島根縣達到人口峰值92.9萬人，此後在[[日本戰後經濟奇蹟|經濟高速增長時期]]大幅下降，1975年時只有76.9萬人<ref name="population2">{{cite web |url=http://www.pref.shimane.lg.jp/admin/seisaku/keikaku/sousei/index.data/shimane_vision.pdf |script-title=ja:島根県人口ビジョン  |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。1970年代後期和1980年代前期，島根縣人口略有回升至79.5萬人<ref name="population2"></ref>。然而1980年代中期之後，島根縣人口再次開始減少。島根縣本來就是人口流出地區，且自1992年開始人口自然增長率也是負數<ref name="population2"></ref>。2017年4月1日時，島根縣有人口684,888人，是日本唯一一個現在人口少於1920年首次人口普查時人口的縣<ref name="population1"></ref>。島根縣的人口減少幅度有顯著的地區差異，出雲地方的人口減少幅度較少，而石見地方和隱岐地方的人口數和峰值時相比幾乎減半<ref name="population2"></ref>。2013年，島根縣的[[總和生育率|總和生育率]]有1.65，遠高於日本平均值1.43，排名日本第三位<ref name="population2"></ref>。

[[島根縣知事列表|島根縣歷任知事]]也都是保守派人士。現任島根縣知事是{{link-ja|丸山達也|丸山達也}}。他在2019年首次當選，並在2011年、2015年順利連任<ref>{{cite web|url=https://go2senkyo.com/local/senkyo/18500　|script-title=ja:島根県知事選挙 |accessdate=2019-04-11|publisher={{lang|ja|選挙ドットコム}}|language=ja}}</ref>。{{link-ja|島根縣議會|島根県議会}}共有37名議員，其中自民黨是最大黨派，有22名議員<ref>{{cite web |url=http://www.pref.shimane.lg.jp/gikai/gaido/meibo/　 |script-title=ja:議員名簿 |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2013年度，島根縣財政收入5427億日元，支出5236億日元。島根縣的財政力指數在2013年度是0.22，遠低於日本平均水準<ref>{{cite web |url=https://www.smbcnikko.co.jp/seminar/nikko_online/IR_public/movie/pdf/disclaimer05.pdf　 |script-title=ja:島根県の財政 |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 第一產業 ===
第一產業佔島根縣經濟總量的1.6%<ref name="economy1"></ref>。2015年時，島根縣有農家33,513戶，位居日本第31位。農業就業人口有24,801人<ref>{{cite web |url=http://pref.shimane-toukei.jp/upload/user/00019331-PEdyer.pdf |script-title=ja:2015年農林業センサス結果報告書 |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>，農業生產額則有570億日元<ref>{{cite web |url=http://www.sanin-chuo.co.jp/news/modules/news/article.php?storyid=563748004 |script-title=ja:農業産出額 島根7.3%増 15年まとめ |accessdate=2017-05-17 |publisher={{lang|ja|山陰中央新報}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170217145752/http://www.sanin-chuo.co.jp/news/modules/news/article.php?storyid=563748004 |archivedate=2017-02-17 }}</ref>。[[大米|大米]]和畜產品在島根縣的農業生產額中佔有最大比重<ref>{{cite web |url=http://www.pref.shimane.lg.jp/industry/norin/seibi/shishin/index.data/shishin_dai2hyou.pdf |script-title=ja:現状と課題 |accessdate=2017-05-17 |publisher={{lang|ja|山陰中央新報}} |language=ja}}</ref>。島根縣的[[葡萄|葡萄]]和[[山葵|山葵]]產量高居日本第四位<ref>{{cite web |url=http://www.pref.shimane.lg.jp/admin/seisaku/zaisei/torikumi/kaikakukaigi/sesaku_tennkenn/sangyou_shinkou.data/06-2-hanro.pdf |script-title=ja:全国に自慢できる島根の農産物 |accessdate=2017-05-17 |publisher={{lang|ja|山陰中央新報}} |language=ja}}</ref>。島根縣面積約八成是森林，是日本第三位的森林縣，也是日本主要木材產地之一<ref>{{cite web |url=http://shinrin-shimane.jp/shinrin.html |script-title=ja:島根県の森林・林業 |accessdate=2017-05-17 |publisher={{lang|ja|島根県森林協会}} |language=ja}}</ref>。2016年，島根縣的總漁獲量有10.8萬噸，總生產額有192億日元<ref>{{cite web |url=http://www.pref.shimane.lg.jp/industry/suisan/shinkou/umi_sakana/tobiuo/ |script-title=ja:トビウオ通信（漁海況の解説） |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。島根縣的[[鰤魚|鰤魚]]捕撈量是日本第一，[[日本竹筴魚|日本竹筴魚]]、[[雪蟹|雪蟹]]、[[蜆科|河蜆]]的捕撈量也是日本前列<ref>{{cite web |url=http://www.stat.go.jp/naruhodo/c3loc32.htm |script-title=ja:島根県 |accessdate=2017-05-17 |publisher={{lang|ja|なるほど統計学園}} |language=ja}}</ref>。

=== 第二產業 ===
第二產業佔島根縣經濟總量的23.5%<ref name="economy1"></ref>。鳥取縣缺乏大型臨海重工業地帶，重化學工業比重較低，製造業規模較小<ref name="Chishi"></ref>{{rp|170}}。2014年，島根縣的工業品生產金額有10,567億日元，其中以鋼鐵、電子零件、信息通信機械的比重最大。在地區分佈上則是以出雲市的4,101億日元最多（佔縣總量的38.8%），其次是安來市的1,789億日元（佔16.9%）和松江市的1,173億日元（佔11.1%）<ref>{{cite web |url=http://pref.shimane-toukei.jp/upload/user/00018758-Ib4O6I.pdf |script-title=ja:平成26年工業統計調査島根県分結果確報 |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。中海地區在被指定為新產業都市之後成為山陰地方最大的工業地區<ref name="Chishi"></ref>{{rp|172}}。島根縣在[[江戶時代|江戶時代]][[吹踏鞴|吹踏鞴]]製鐵產業發達，有著悠久的鋼鐵產業傳統<ref>{{cite web |url=http://www.pref.shimane.lg.jp/admin/seisaku/koho/esque/2013/87/01.html |script-title=ja:島根のたたら |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。{{link-ja|日立金屬|日立金属}}在安來市有鋼鐵工廠<ref>{{cite web |url=http://www.hitachi-metals-yasugi-seisakusyo.co.jp/company/index.html |script-title=ja:会社案内 |accessdate=2017-05-17 |publisher={{lang|ja|日立金属}} |language=ja}}</ref>。東洋製鐵、NTN鋳造等鋼鐵行業企業在島根縣也有設廠。[[富士通|富士通]]在出雲市設有工廠，是日本規模最大的[[筆記本電腦|筆記本電腦]]和[[平板電腦|平板電腦]]生產據點<ref>{{cite web |url=http://www.fujitsu.com/jp/group/sfj/about/history/ |script-title=ja:沿革 |accessdate=2017-05-17 |publisher={{lang|ja|島根富士通}} |language=ja}}</ref>。[[島津公司|島津公司]]<ref>{{cite web |url=http://www.shimane-shimadzu.co.jp/aboutus/ |script-title=ja:企業情報 |accessdate=2017-05-17 |publisher={{lang|ja|島根島津}} |language=ja}}</ref>、[[村田製作所|村田製作所]]<ref>{{cite web |url=http://www.murata.com/ja-jp/group/izumomurata/corporate/overview?intcid5=com_xxx_xxx_cmn_nv_xxx |script-title=ja:会社概要 |accessdate=2017-05-17 |publisher={{lang|ja|出雲村田製作所}} |language=ja}}</ref>等機械企業也在出雲市設有工廠。近年島根縣積極發展IT產業，在松江市設立島根軟體商務園區，通過產學合作促進IT產業發展<ref>{{cite web |url=http://www.pref.shimane.lg.jp/industry/enterprise/richi/danchi/kougyodanchiichiran/kogyo01.html |script-title=ja:ソフトビジネスパーク島根 |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。
=== 第三產業 ===
第三產業佔島根縣經濟總量的74%<ref name="economy1"></ref>。政府服務在島根縣經濟總量中佔比較高，顯示島根縣經濟存在依賴行政的問題<ref name="economy2"></ref>。總部位於島根縣的銀行有[[山陰合同銀行|山陰合同銀行]]和[[島根銀行|島根銀行]]<ref>{{cite web |url=http://www.shimagin.co.jp/company/gaiyo.html |script-title=ja:しまぎんの概要 |accessdate=2017-05-17 |publisher={{lang|ja|島根銀行}} |language=ja}}</ref>。其中山陰合同銀行的業務範圍不僅橫跨山陰地方的鳥取、島根兩縣，還在廣島縣、岡山縣、兵庫縣等地設有分行，是山陰地方最大的地方銀行<ref>{{cite web |url=http://www.gogin.co.jp/www/contents/1000000033000/index.html |script-title=ja:会社概要 |accessdate=2017-05-17 |publisher={{lang|ja|山陰合同銀行}} |language=ja}}</ref>。2014年度，島根縣的商品銷售額超過13,816億日元，其中批發業有7,126億日元，零售業有6,690億日元<ref>{{cite web |url=http://pref.shimane-toukei.jp/upload/user/00018715-cjnL5i.pdf |script-title=ja:平成26年商業統計調査島根県分結果確報 |accessdate=2017-05-17 |publisher={{lang|ja|山陰合同銀行}} |language=ja}}</ref>。島根縣內商業活動主要集中在松江、出雲兩市<ref>{{cite web |url=http://www.meti.go.jp/statistics/tyo/syougyo/dms/2009/pdf/ken32.pdf |script-title=ja:島根県の年間商品販売額は卸売業の6割強、小売業の6割を松江市と出雲市が占める |accessdate=2017-05-17 |publisher={{lang|ja|経済産業省}} |language=ja}}</ref>。縣內主要商業企業有{{link-ja|一畑百貨店|一畑百貨店}}、{{link-ja|三島屋|みしまや}}、{{link-ja|FOODS MARKET HOK|フーズマーケット ホック}}等。2015年，到訪島根縣的觀光客人數有3315.8萬人，為島根縣帶來1,367億日元的消費。出雲市和松江市的年觀光客人數均超過1,000萬人。出雲大社是島根縣觀光客數最多的景點<ref>{{cite web |url=http://www.pref.shimane.lg.jp/tourism/tourist/kankou/chosa/kanko_dotai_chosa/report_h27.data/2_chosa_kekka_gaiyo.pdf |script-title=ja:調査結果の概要 |accessdate=2017-05-17 |publisher={{lang|ja|島根県}} |language=ja}}</ref>。

== 文化 ==
=== 方言 ===
島根縣內的方言可以分為出雲地方的{{link-ja|出雲方言|出雲方言}}、隱岐群島的{{link-ja|隱岐方言|隠岐方言}}、石見地方的{{link-ja|石見方言|石見方言}}，其中出雲方言和隱岐方言同屬{{link-ja|雲伯方言|雲伯方言}}。石見方言屬於{{link-ja|山陽方言|山陽方言}}<ref name="Dialect">{{cite book |author={{lang|ja|友定賢治}}|script-title=ja:《島根県のことば》 |year=2008 |publisher={{lang|ja|明治書院}} |location=東京都 |isbn=978-4-625-62406-3|language=ja}}</ref>{{rp|6}}。雲伯方言是中國地方的語言島，屬於{{link-ja|ズーズー弁|ズーズー弁}}，「{{lang|ja|じ}}」「{{lang|ja|ず}}」「{{lang|ja|ぢ}}」「{{lang|ja|づ}}」發音相同，有較多東日本方言的特徵<ref name="Dialect"></ref>{{rp|15}}。出雲方言的重音也是{{link-ja|東京式重音|東京式アクセント}}。隱岐方言和出雲方言較為接近，但音調和出雲方言有很大不同，比出雲方言更加複雜<ref name="Dialect"></ref>{{rp|7}}。石見方言在發音上和[[山陽地方|山陽地方]]的方言有較多共通之處，和出雲方言反而有較大不同<ref name="Dialect"></ref>{{rp|38}}。
=== 文學 ===
[[File:Birthplace_of_Mori_Ogai_20170503.jpg|左]]
在島根縣出身和與島根縣有密切關係的作家中，以[[小泉八雲|小泉八雲]]和[[森鷗外|森鷗外]]最為著名。小泉八雲是一位英裔日本人，原名派屈克·拉夫卡迪奧·赫恩。他在1890年首次來到日本，最初到訪的日本城市是松江並在此擔任英語教師。小泉八雲在松江生活的時間雖然只有一年三個月，卻在這裡結識他之後的妻子。松江的生活也對他之後的創作有重大影響。小泉八雲有眾多作品都是以島根縣為題材<ref>{{cite web |url=http://www.hearn-museum-matsue.jp/hearn.html |script-title=ja:小泉八雲の生涯 |accessdate=2017-05-17|publisher={{lang|ja|小泉八雲記念館}} |language=ja}}</ref>。現在松江市內有小泉八雲紀念館、小泉八雲故居等紀念設施<ref>{{cite web |url=http://www.pref.shimane.lg.jp/life/bunka/shinkou/internet_kikakuten/bungaku/bungakusha/koizumi.html |script-title=ja:小泉八雲 |accessdate=2017-05-17|publisher={{lang|ja|島根県}} |language=ja}}</ref>。森鷗外出生在[[津和野町|津和野町]]，通曉[[德語|德語]]，原本是一位軍醫。軍旅生涯造就了森鷗外獨特的文學世界，他的主要代表作有《舞姬》、《青年》等作品<ref>{{cite web |url=http://www.pref.shimane.lg.jp/life/bunka/shinkou/internet_kikakuten/bungaku/bungakusha/mori.html |script-title=ja:森鴎外 |accessdate=2017-05-17|publisher={{lang|ja|島根県}} |language=ja}}</ref>。津和野町的森鷗外故居和森鷗外紀念館現在是觀光景點<ref>{{cite web |url=http://www.tsuwano.net/www/contents/1000000440000/index.html |script-title=ja:森鴎外旧宅・森鴎外記念館 |accessdate=2017-05-17|publisher={{lang|ja|津和野町}} |language=ja}}</ref>。島根縣出身的其他著名作家還有子規派俳人大谷繞石、詩人千家元麿、小說家田畑修一郎、兒童文學家天野雉彥等人<ref>{{cite web |url=http://www.pref.shimane.lg.jp/life/bunka/shinkou/internet_kikakuten/bungaku/bungakusha/ |script-title=ja:島根ゆかりの文学者 |accessdate=2017-05-17|publisher={{lang|ja|島根県}} |language=ja}}</ref>。
=== 表演藝術 ===
石見地方的{{link-ja|石見神樂|石見神楽}}是島根縣最具代表的傳統藝術形式之一，以日本神話為題材，帶有[[劇場_(藝術)|戲劇]]色彩。現在石見神樂被指定為{{link-ja|重要無形民俗文化財|重要無形民俗文化財}}，仍然經常在秋祭等時節演出<ref>{{cite web |url=http://gotsu-kanko.jp/iwamikagura/aboutkagura/|script-title=ja:石見神楽とは |accessdate=2017-05-17|publisher={{lang|ja|江津市観光協会}} |language=ja}}</ref>。{{link-ja|安来節|安来節}}是起源於[[安來市|安來市]]的民謠，自[[大正|大正]]時期開始風行日本全國，以泥鰍舞而著稱<ref>{{cite web |url=http://www.yasugi-kankou.com/index.php?view=5676|script-title=ja:安来節の起源 |accessdate=2017-05-17|publisher={{lang|ja|安来市観光協会}} |language=ja}}</ref>。
=== 飲食 ===
[[File:Izumo_soba_by_boreal_eye.jpg|thumb]]
島根縣最著名的鄉土料理是{{link-ja|出雲蕎麥麵|出雲そば}}。出雲蕎麥麵是日本三大蕎麥麵之一。由於在磨粉時蕎麥殼的部分亦一併研磨，因此使得出雲蕎麥麵的顏色和普通的蕎麥麵相比偏黑，食感和風味亦有獨自特徵<ref>{{cite web |url=http://www.izumo-kankou.gr.jp/3188 |script-title=ja:出雲そば特集 |accessdate=2017-05-17|publisher={{lang|ja|出雲観光協会}} |language=ja}}</ref>。島根縣的甜點文化亦頗為發達。江戶時代的松江藩第七代藩主{{link-ja|松平治郷|松平治郷}}是著名的茶道家，他在治理松江期間發明了眾多茶點，使得松江和[[京都市|京都]]、[[金澤市|金澤]]並稱為日本三大菓子處<ref>{{cite web |url=http://www.gojapan.jp/toku3/zsub01/index303.html|script-title=ja:【菓子】 |accessdate=2017-05-17|publisher={{lang|ja|日本の名物特産品・特産物}} |language=ja}}</ref>。
=== 媒體 ===
鳥取縣內主要的地方報紙是{{link-ja|山陰中央新報|山陰中央新報}}。山陰中央新報的發行範圍包括島根、鳥取兩縣，日發行量超過18萬份，是山陰兩縣發行量最多的報紙<ref>{{cite web |url=http://www.sanin-chuo.co.jp/hanbai/koudoku.html |script-title=ja:ご購読、試読のご案内 |accessdate=2017-05-17|publisher={{lang|ja|山陰中央新報社}} |language=ja}}</ref>。{{link-ja|島根日日新聞|島根日日新聞}}總部位於出雲市，是只在島根縣內發行的地方報紙<ref>{{cite web |url=http://www.shimanenichinichi.co.jp/kaisha/ |script-title=ja:会社概要 |accessdate=2017-05-17|publisher={{lang|ja|島根日日新聞社}} |language=ja}}</ref>。島根縣內的電視台有[[NHK松江放送局|NHK松江放送局]]和[[山陰中央電視台|山陰中央電視台]]<ref>{{cite web |url=http://www.tsk-tv.com/outline/ |script-title=ja:会社概要 |accessdate=2017-05-17|publisher={{lang|ja|山陰中央テレビ}} |language=ja}}</ref>。由於島根縣和鳥取縣共享一個電視播出區域，因此島根縣也可以收看總部位於鳥取縣的[[山陰放送|山陰放送]]和[[日本海電視台|日本海電視台]]的節目。
=== 體育 ===
日本籃球[[B聯賽|B聯賽]]球隊{{link-ja|島根魔術|島根スサノオマジック}}是唯一一家以島根縣為主場的職業體育俱樂部<ref>{{cite web |url=https://www.susanoo-m.com/news/3805.html |script-title=ja:球団概要について |accessdate=2017-05-17|publisher={{lang|ja|島根スサノオマジック}} |language=ja}}</ref>。此外島根縣內還有多家社會人足球及棒球俱樂部。每年在出雲市舉辦的{{link-ja|出雲全日本大學驛站接力賽|出雲全日本大学選抜駅伝競走}}是日本三大大學驛站接力賽之一<ref>{{cite web |url=http://www.izumo-ekiden.jp/about/index.html |script-title=ja:出雲駅伝とは |accessdate=2017-05-17|publisher={{lang|ja|出雲駅伝}} |language=ja}}</ref>。{{link-ja|出雲巨蛋|出雲ドーム}}是日本最大的木造巨蛋球場<ref>{{cite web |url=https://www.izumo-kankou.gr.jp/381 |script-title=ja:出雲ドーム |accessdate=2017-05-17|publisher={{lang|ja|出雲観光協会}} |language=ja}}</ref>。
=== 教育 ===
島根縣內有兩家大學。[[島根大學|島根大學]]設立於1949年，由松江高等學校、島根師範學校、島根青年師範學校、島根縣立島根農科大學、島根醫科大學五家學校合併而成，是島根縣唯一的國立大學<ref>{{cite web |url=http://www.shimane-u.ac.jp/introduction/university_profile/history/ |script-title=ja:沿革 |accessdate=2017-05-17|publisher={{lang|ja|島根大学}} |language=ja}}</ref>。島根大學現在總部位於松江市，設有法文學院、教育學院、醫學院、綜合理工學院、生物資源科學院五個學院<ref>{{cite web |url=http://www.shimane-u.ac.jp/introduction/university_profile/profile/ |script-title=ja:概要 |accessdate=2017-05-17|publisher={{lang|ja|島根大学}} |language=ja}}</ref>。[[島根縣立大學|島根縣立大學]]位於[[濱田市|濱田市]]，設立於2000年，是島根縣唯一的公立大學，設有綜合政策學院和看護學院兩個學院<ref>{{cite web |url=http://www.u-shimane.ac.jp/university/history/ |script-title=ja:沿革 |accessdate=2017-05-17|publisher={{lang|ja|島根県立大学}} |language=ja}}</ref>。

== 觀光 ==
[[File:Matsue_castle01bs4592.jpg|thumb]]
島根縣的人文和自然觀光資源均十分豐富。[[出雲大社|出雲大社]]位於出雲市，祭神是[[大國主神|大國主神]]。出雲大社自[[奈良時代|奈良時代]]文獻《[[古事記|古事記]]》、《[[日本書紀|日本書紀]]》就有記載<ref>{{cite web |url=http://www.izumooyashiro.or.jp/sengu.rekishi.html }}</ref>。出雲大社在[[神道教|神道教]]中有很高地位，相傳每年舊曆十月，全日本的諸神都會聚集於出雲大社。因此與日本其他地方把舊曆十月稱為「神無月」相對，出雲地方將舊曆十月稱為「神在月」<ref>{{cite web |url=https://www.izumo-kankou.gr.jp/1275 |script-title=ja:神在月 |accessdate=2017-05-17
