'''鄒語'''（[[原住民族語羅馬字|鄒文]]：{{lang|tsu|Cou}}），前稱「'''曹語'''」，為[[台灣|臺灣]][[鄒族|鄒族]]人所使用的語言，屬於[[南島語系|南島語系]]的[[鄒語群|鄒語群]]，以[[拉丁文字|拉丁文字]]書寫。<ref>國立台灣師範大學進修推廣學院,"98年度原住民族語言能力認證考試"{{cite web |url=http://140.122.109.96/ipt/viewpage.php?page_id=11 |title=存档副本 |accessdate=2017-03-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20091217013326/http://140.122.109.96/ipt/viewpage.php?page_id=11 |archivedate=2009-12-17 }},臺北市,2009.</ref>

==鄒語變遷沿革==
鄒語一般分為北鄒語群與南鄒語群。「北鄒語群」分佈於南投縣信義鄉望美部落；以及嘉義縣阿里山鄉的部落：特富野部落、達邦部落、里佳部落、茶山部落、新美部落、樂野部落、來吉部落、山美部落。而在高雄市[[那瑪夏區|那瑪夏區]]的[[卡那卡那富語|卡那卡那富語]]與桃源鄉的[[拉阿魯哇語|拉阿魯哇語]]則叫「南鄒語群」。各自的語言歷經長年的變遷，自成一個系統。<ref name=tsoulatin>行政院原住民族委員會,"原住民族語言書寫系統"[http://www.nter.tw/thao/attachments/003_%E5%8E%9F%E4%BD%8F%E6%B0%91%E8%AA%9E%E8%A8%80%E6%9B%B8%E5%AF%AB%E7%B3%BB%E7%B5%B1.pdf]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }},台語字第0940163297號,原民教字第09400355912號公告,中華民國94年12月15日.</ref>

===北鄒語群的變遷===
北鄒語群又稱「鄒語」，有特富野社(Tfuya,Tfuea)、達邦社(Tapangu)、伊姆諸社(Imutsu,Limutsu)、與魯富都社(Luhtsu,Luhtu)四個方言語群，而四社中的伊姆諸社與魯富都社均於20世紀初廢社，人口分散到各社。根據[[民族語|民族語]]學院(Ethnologue)記錄1981年伊姆諸社方言語群消失。目前北鄒語方言群分為三支：久美(望美)方言語群、特富野方言語群、達邦方言語群，不過各方言語群差異很小。

特富野方言和達邦方言之間文法差異不大。這兩方言語音上有些微差異，但變化規律，例如許多特富野方言讀z的音，在達邦方言則讀為i。
久美方言保留較多古音。

===臺灣鄒族語典===
1927年6月[[俄國|俄國]][[語言學家|語言學家]][[聶甫斯基|聶甫斯基]]經由[[日本|日本]]來臺灣，並與正就讀[[國立臺南大學|台南師範]]之特富野鄒族青年[[高一生|吾雍·雅達烏猶卡那]]（Uyongu Yata'uyungana；高一生/矢多一生/矢多一夫），於1927年7月至8月初，從事1個多月的田野鄒語調查。聶甫斯基精通日文，兩人以[[日語|日語]]溝通。期間記錄了15個故事，使用[[拉丁字母|拉丁字母]]與[[IPA|IPA]]記寫，並收集鄒語發音、語法、及詞彙等，以及鄒族的民族文物30件。聶甫斯基1929年從日本回[[蘇聯|蘇聯]]後、於1935年在[[聖彼德堡|聖彼德堡]]由[[俄羅斯科學院|蘇聯科學院]]的出版社出版了一部《[[臺灣鄒族語典|臺灣鄒族語典]]》（''N.A. Nevskij:Materaly po govoram jazyka cou, Trudy Instituta Vostokovedenija. X1. Moskva-Leningrad,1935''/''{{lang|ru|Невский, Николай Александрович, Материалы по говорам языка цоу. М.-Л., Изд-во АН. 1935. 134 стр. 1000 экз.}}''），可謂最早的一部專門研究鄒語[[語言學|語言學]]的專書。兩人先後於10年之後（1937年10月4日），聶甫斯基與日本籍妻子{{lang|ja|萬谷イソ/萬谷磯子}}在[[史達林|史達林]][[大清洗|大清洗]]期間均被[[蘇聯|蘇聯]]政府以[[日本|日本]][[間諜|間諜]]及叛國罪逮捕，於11月24日在[[聖彼得堡|聖彼得堡]]夫妻均遭[[槍斃|槍決]]；及27年之後、1954年4月17日高一生被[[中華民國政府|中華民國政府]]以[[二二八事件|228事件]]為由，定以[[叛亂罪|叛亂罪]]槍決。<ref>[[聶甫斯基|聶甫斯基]]([[俄國人|俄國人]])，"臺灣鄒族語典"，臺原藝術文化基金會，台北，7月，1993。</ref>

== 語音系統 ==
{{main|鄒語音系}}

鄒語的音系之[[音位|音位]]大都使用適當的[[Unicode|Unicode]]符號來標示。在[[台灣南島語|臺灣南島語]]的[[書寫系統|書寫系統]]的訂定上，元輔音之音系表原則上都是先「[[發音部位|發音部位]]」(橫列)、後「[[發音方法|發音方法]]」(縱列)、再考量「[[清濁音|清濁音]]」，來訂定其音系之音位架構。<ref name=shaolatin>行政院原住民族委員會,"原住民族語言書寫系統"[http://www.edu.tw/files/list/M0001/aboriginal.pdf],台語字第0940163297號,原民教字第09400355912號公告,中華民國94年12月15日.</ref>

== 鄒語語法 ==
{{main|鄒語語法}}
 
在語法的分類上[[台灣南島語|臺灣南島語]]並不同於一般的[[分析語|分析語]]或其它[[綜合語|綜合語]]裡的動詞、名詞、形容詞、介詞和副詞等之基本[[詞類|詞類]]分類。比如臺灣南島語裡普遍沒有副詞，而副詞的概念一般以動詞方式呈現、可稱之為「副動詞」，類之於[[俄語|俄語]]裡的副動詞。<ref name=diunnbbunhua>張永利,"台灣南島語言語法：語言類型與理論的啟示(Kavalan)"[http://vietnam.nsc.gov.tw/public/Data/012714292271.pdf]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }},語言學門熱門前瞻研究,2010年12月/12卷1期,pp.112-127.</ref> 鄒語語法分類是將基礎的語法之詞類、詞綴、字詞結構及分類法，對比[[分析語|分析語]]等之詞類分類法加以條析判別。
