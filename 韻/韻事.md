《'''韻事'''》（{{lang|ja|浮き名}}，''Ukina／Scandal''）是日本音樂家[[椎名林檎|椎名林檎]]的第二張[[精選輯|精選輯]]，同時也是唱片公司為紀念椎名林檎出道十五週年所發行的作品，和現場錄音專輯《[[蜜月抄|蜜月抄]]》、影音MV合輯《[[LiVE|LiVE]]》及音樂MV合輯《[[The_Sexual_Healing_Total_Care_Course_120min.|The Sexual Healing Total Care Course 120min.]]》一同於2013年11月13日由[[EMI音樂_(日本)|EMI Music Japan]]／[[環球音樂_(日本)|Universal Music]]發行<ref name="JPN" />。初回限定版為「特殊紙盒包裝、未發表寫真照及專輯《韻事》訪談」。

在專輯銷售成績方面，本張專輯在日本Oricon銷售榜上最高位居單週第5名<ref name="UkinaOriconWeek" />，名列2013年年度銷售榜第195位<ref name="2013saleAlbum" />。

== 專輯概要 ==
《'''韻事'''》是椎名林檎繼2008年發行《[[我·放電|我．放電]]》後，睽違五年的精選輯，也是繼2009年發行原創專輯《[[三文八卦|三文八卦]]》以來，睽違四年的作品。同時，這張專輯也是唱片公司為慶祝椎名林檎出道十五週年所發行的作品。本張精選輯收錄椎名林檎自出道以來以客座歌手的身分和其他歌手合作的歌曲<ref name="SONGRESOURCES">{{cite web |url=http://natalie.mu/music/news/103440 |title= 椎名林檎「蜜月抄」＆「浮き名」で紐解く15年の軌跡 |publisher=natalie |language=ja}}</ref>。此外，本張精選輯還收錄了最早曾在《{{link-ja|RISING SUN ROCK FESTIVAL 2008 in EZO|ライジング・サン・ロックフェスティバル}}》演出，由美國音樂製作人[[伯特·巴卡拉克|伯特·巴卡拉克]]提供的歌曲「IT WAS YOU」<ref group="注">首次演出的時候，歌曲標題是「YOU」。</ref><ref name="SONGRESOURCES"/>、日本音樂製作人[[中田康貴|中田康貴]]所製作的新曲「熱愛發覺中」以及先前發售的作品「[[Between_Today_and_Tomorrow|Between Today and Tomorrow]]」和「[[殺手千鈞一髮|殺手千鈞一髮]]」<ref group="注">單曲《殺手千鈞一髮》是以「SOIL&"PIMP"SESSIONS×椎名林檎」為名義發行。</ref>。專輯封面的[[插畫|插畫]]是由{{link-ja|宇野亞喜良|宇野亜喜良}}所製作<ref>{{cite web |url=http://sp.universal-music.co.jp/ringo/15th/discography/index.php?did=notes01 |title= LINER NOTES『浮き名』/『蜜月抄』 |publisher=日本環球音樂 |language=ja}}</ref>。

在專輯發售時程上，新專輯發行的消息最早於2013年9月11日發佈<ref name="RINGO15LIVE1" />，同時也表示將舉行《椎名林檎十五周年 黨大會 平成二十五年神山町大會》演唱會<ref name="RINGO15LIVE1" />。10月2日更表示將舉辦林檎班會員限定的演唱會《椎名林檎十五周年 班大會 平成二十五年濱離宮大會》<ref name="RINGO15LIVE2" />。11月1日則開始接受「IT WAS YOU」的音樂錄影帶視聽服務<ref>{{cite web |url=http://natalie.mu/music/news/102666 |title=椎名林檎、バカラック提供「IT WAS YOU」PV先行公開 |publisher=natalie |language=ja}}</ref>，一個禮拜後11月8日則於影音網站[[Youtube|Youtube]]提供歌曲「熱愛發覺中」音樂錄影帶視聽服務<ref name="NetsuaiHakkakuchuuMV1">{{cite web |url=http://natalie.mu/music/news/103118 |title=椎名林檎「熱愛発覚中」PVで痛快アクション披露 |publisher=natalie |language=ja}}</ref>。而因為椎名林檎在歌曲「熱愛發覺中」音樂錄影帶中性感的打扮，讓影片在三天內撥放次數超過一百萬次，因此11月13日則開始提供歌曲「熱愛發覺中」完整版音樂錄影帶視聽服務<ref name="NetsuaiHakkakuchuuMV2">{{cite web |url=http://natalie.mu/music/news/103434 |title=椎名林檎アクション＆衣装に注目「熱愛発覚中」PV完全版 |publisher=natalie |language=ja}}</ref>。日本於11月13日發行專輯<ref name="JPN" />，香港也於當日同步發行作品<ref name="HKG"/>，台灣則等到一個月後的12月13日同步發行進口版與台壓版<ref name="TWN" />。

最後，在專輯銷售成績方面，專輯在發行當週以2.1萬張的銷售量在日本Oricon銷售榜上位居第5名<ref name="UkinaOriconWeek" />，總計在Oriocn銷售榜上榜8週<ref name="UkinaOriconConsecutiveWeek" />，累積銷售達3.5萬張，名列2013年年銷售榜第195位<ref name="2013saleAlbum" />。

=== 音樂錄影帶 ===
[[Image:Shiina02-09-mv.jpg|thumb]]
在歌曲「熱愛發覺中」的音樂錄影帶中，主角椎名林檎飾演一位為了替被狗仔隊殺死的夥伴報仇，而孤軍深入敵營戰鬥的女人。椎名林檎在音樂錄影帶中不但與舞團Idevian Crew合作，還首次挑戰了以舞蹈為首的動作場面，還一人分飾二角，展現精湛的演技<ref name="NetsuaiHakkakuchuuMV1" />。而身穿馬甲打敗敵人的椎名林檎，其性感的打扮讓影片在三天內撥放次數超過一百萬次<ref name="NetsuaiHakkakuchuuMV2" />。

== 批評及評價 ==
不管是在音樂網站''CD Journal''的評論家或是''Skream!''上的音樂評論家{{nihongo|沖清子|沖 さやこ}}都表示這張精選輯內的歌曲強烈地展現出椎名林檎的風格。音樂網站''CD Journal''特別推薦聽者聆聽歌曲「熱愛發覺中」，在這首歌曲中聽者很容易與歌曲產生共鳴<ref>{{cite web |url=http://artist.cdjournal.com/d/-/4113090706 |title=ミニ・レビュー |publisher=CDJournal |language=ja}}</ref>。而音樂評論家{{nihongo|沖清子|沖 さやこ}}則表示雖然這張精選輯是收錄各個不同類型的歌曲，但椎名林檎總是能夠把它們融合成一張專輯<ref>{{cite web |url=http://skream.jp/diskreview/2013/11/ukina.php |title=Disc Review 椎名林檎『浮き名』 |author=Sayako Oki |publisher=Skream |language=ja}}</ref> 。

== 銷售排行 ==
=== Oricon 銷售榜(日本) ===
《韻事》於11月13日發行專輯。發行首週，在日本公信榜Oricon的專輯榜2013年11月第2週付的榜單中以21,365張銷售量居單週第五名，同禮拜銷售冠軍為銷售突破4.8萬張的歌舞團體[[放浪新世代_from_放浪一族|放浪新世代 from 放浪一族]]的首張專輯《[[GENERATIONS_(GENERATIONS專輯)|GENERATIONS (GENERATIONS專輯)]]》，而同天發行的現場錄音專輯《蜜月抄》則以14,946張銷售量居單週第六名<ref name="UkinaOriconWeek" />。第二週則以銷售量5,710張居單周第十八位，同禮拜銷售冠軍為樂團[[色情塗鴉|色情塗鴉]]的單曲精選合輯《{{link-ja|PORNOGRAFFITTI 15th Anniversary "ALL TIME SINGLES"|PORNOGRAFFITTI 15th Anniversary "ALL TIME SINGLES"}}》<ref>{{Cite web |url=http://www.oricon.co.jp/rank/ja/w/2013-12-02/ |title=2013年11月第3週の邦楽アルバムランキング情報 |publisher=Oricon |language=ja}}</ref>。總計在日本公信榜Oricon的專輯榜上榜8週<ref name="UkinaOriconConsecutiveWeek" />，累積銷售達3.5萬張，名列2013年年銷售榜第195位<ref name="2013saleAlbum" />。
