'''珍妮特·贝德曼'''({{lang-de|'''Jeanette Biedermann'''}},1980年2月22-)是一名德国女[[歌手|歌手]]和[[演员|演员]]。她在1999年的“‘Bild杯’流行歌曲比赛”中胜出后进入娱乐圈，以[[艺名|艺名]]“珍妮特”闻名。其代表作品包括《No More Tears》，《Go Back》等。

珍妮特除参演过《Tatort》(2006年)、《安娜与爱情》("Anna und die Liebe")等影视作品外，也在一些如《Star Search》(2003年)和《[[花样冰舞|花样冰舞]]》(2006年)等电视[[真人秀|真人秀]]节目中担任过评委。

== 生平 ==

=== 早年生活 ===
1980年2月22日，贝德曼出生于柏林，是柏恩德(Bernd)与马里恩·贝德曼(Marion Biedermann)唯一的孩子。“Jeanette”一名的含义为“上帝是慈爱的”。父母在她出生前，曾失去过3个孩子。其中的前2个胎死腹中，第3个孩子邓尼斯(Dennis)也在出生3周后死去。珍妮特在柏林长大。6岁时，珍妮特加入马戏团表演杂技，还在Circus Lilliput登台表演过。中学毕业后，她报考了美艺学校(beauty school)。她学习过美容美发技术。1999年，她中断了技术学习。不过，她参加了“‘Bild杯’歌曲大赛”，并从27万参赛者中胜出。不久后就发售了第一首单曲《Das tut unheimlich weh》。

夺冠一个月后，她参演了德国[[肥皂剧|肥皂剧]]《Gute Zeiten, Schlechte Zeiten》。她在剧中饰演的角色叫Marie Balzer，从1999年演到2004年。<ref>{{cite web|url=http://www.shortnews.de/start.cfm?id=496842 |title=Jeanette Biedermann verlässt "GZSZ"}}</ref>

=== 2008-2010: 新作的曲风变化与个人引退 ===

2008年5月，她与德国演员Roy Peter Link一同主演了电视剧《安娜与爱情》。<ref>{{cite web |url=http://www.monstersandcritics.de/artikel/200821/article_83445.php/Aus-Plenske-wird-Polauke |title=Aus Plenske wird Polauke |deadurl=yes |archiveurl=https://web.archive.org/web/20091106053911/http://www.monstersandcritics.de/artikel/200821/article_83445.php/Aus-Plenske-wird-Polauke |archivedate=2009-11-06 }}</ref>该剧以超过250万的收视率打破了数个纪录，并为贝德曼扩大了在德国的知名度。<ref>{{cite web|url=http://www.annaunddieliebe.de|title=Anna und die Liebe |work=Quotenmeter.de}}</ref><ref>{{cite web|url=http://www.quotenmeter.de/cms/?p1=n&p2=38161&p3=|title="Anna und die Liebe": Antrag bringt Rekord}}</ref>2009年10月，珍妮特表示要在2010年1月离开该剧，其参演的最好一集于2010年4月份播出。她在随后的几个月中将精力集中放在音乐发展和歌唱巡演上。<ref>{{cite web|url=http://www.bild.de/BILD/unterhaltung/TV/2009/10/23/anna-und-die-liebe/jeanette-biedermann-spricht-ueber-zukunft.html|title=Jetzt spricht Jeanette Biedermann |work=Bild.de  |date=November 3, 2009}}</ref>

2008年，珍妮特着手制作的新专辑的曲风由之前的摇滚流行转向时兴的电子流行。加盟制作人包括Remee、Carl Falk、Thomas Troelsen及Johan Bobäck。<ref>{{cite web|url=http://www.antenne.de/nachrichten/boulevard/artikel/24301/Jeanette-Biedermann-gibt-sich-elektrolastig-.html|title=Jeanette Biedermann gibt sich elektrolastig|date=November 4, 2009|deadurl=yes|archiveurl=https://web.archive.org/web/20110718200533/http://www.antenne.de/nachrichten/boulevard/artikel/24301/Jeanette-Biedermann-gibt-sich-elektrolastig-.html|archivedate=2011年7月18日}}</ref> 专辑于2009年初发售，其中的第一首单曲《Undress to the Beat》在德国音乐榜上排名第6。<ref>{{cite web|url=http://www.monstersandcritics.de/artikel/200911/article_128513.php/Jeanette-%7C-%E2%80%9EUndress-to-the-beat-erobert-die-Top-Ten-der-Singlecharts|title=Jeanette "Undress to the beat" erobert die Top Ten der Singlecharts|deadurl=yes|archiveurl=https://web.archive.org/web/20110719063358/http://www.monstersandcritics.de/artikel/200911/article_128513.php/Jeanette-%7C-%E2%80%9EUndress-to-the-beat-erobert-die-Top-Ten-der-Singlecharts|archivedate=2011-07-19}}</ref>专辑中她很喜欢的一首单曲《Solitary Rose》被放在电视剧《安娜与爱情》中首播。她说这首歌是专门写给她在该剧中的角色安娜·珀劳克(Anna Polauke)的。这首歌在德国音乐榜排名第8位。<ref>{{cite web|url=http://www.bunte.de/stars/jeanette-biedermann-auf-rosen-gebettet_aid_12904.html|title=Auf Rosen gebettet |date=November 3, 2009}}</ref>2010年4月起，她进行了一次巡演，但中途由于家里人生病而中断。当时他父亲患上了癌症，但后来又恢复了健康。<ref>http://www.jeanettebiedermann.de/news/details/article/87345/0/eilmeldung--solitary-rose-tour-kann-nicht-statt-finden-/{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2010年末，贝德曼宣布引退。

== 个人生活 ==
2005年初，珍妮特开始与吉他手Jörg Weisselberg约会。<ref>{{cite web |url=http://www.fan-lexikon.de/musik/news/jeanette-biedermann-hat-einen-neuen-freund.10565.html |title=Jeanette Biedermann hat einen Neuen Freund |accessdate=2009-11-04}}</ref>2008年7月，他们因在一起的时间不多且是[[异地恋|异地恋]]而决定分手。珍妮特说他们在分手以后仍保持着朋友关系，以后还会有机会一同合作。2年后，他们再次相恋。2012年夏，他们正式结婚。珍妮特养了2只猫咪。<ref>{{cite web |url=http://www.bunte.de/stars/jeanette-biedermann-kuesse-fuer-ihren-ex-freund_aid_12227.html |title=Küsse für den Ex-Freund |accessdate=2009-11-04}}</ref><ref>{{cite web |url=http://www.tikonline.de/top-stories/detail.php?nr=33618&rubric=Top-Stories |title=Trennung von Freund Jörg! |accessdate=2009-11-04}}</ref>
