'''穿青人'''又称'''穿青族'''、'''川青人'''<ref name="a1">{{cite web |url=http://www.seac.gov.cn/gjmw/zt/2010-09-20/1284970621835222.htm |archive-url=https://web.archive.org/web/20160306190607/http://www.seac.gov.cn/gjmw/zt/2010-09-20/1284970621835222.htm |title=穿青人服饰 |publisher=中华人民共和国国家民族事务委员会网站 来源：贵州民族文化宫 |author= |date=2010-09-20 |archive-date=2016-03-06 |accessdate=2018-03-01 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160306190607/http://www.seac.gov.cn/gjmw/zt/2010-09-20/1284970621835222.htm |archivedate=2016-03-06 }}</ref>、'''川青族'''、'''青族'''<ref>{{cite web|url=http://news.163.com/14/0508/15/9RNVGTCH0001124J.html |title=56个民族之外"穿青人"民族获认可 被写上身份证|publisher=[[网易|网易]]|date=2014年5月8日|accessdate=2014年5月8日}}</ref>，是[[中华人民共和国|中华人民共和国]]境内一个民族归属有争议的族群。中国政府把穿青人归类为[[汉族|汉族]]的分支<ref>{{cite web|url=http://politics1.people.com.cn/n/2014/0508/c1001-24993941.html|title=国家民委:近期未确认新的民族 现有民族仍是56个}}</ref>，定义为[[中国未识别民族|未识别民族]]，在某些情况下享受少数民族的优惠待遇<ref>中国的穿青人，不属于56个民族，到底有多神秘？
http://www.sohu.com/a/257098953_99997293</ref>，有民族標示為穿青人的[[身份证|身份证]]<ref>{{cite web|url=http://news.qq.com/a/20140508/016661.htm|title=56个民族之外“穿青人”获认可 首次写进身份证}}</ref>。但在民族学和历史学上，穿青人是否属于汉族仍争议。主要分布在[[贵州|贵州]]的偏远山区里，也有一些住在县城。穿青人有67万左右，是中国人数最多的未识别民族。大部分穿青人自认为是一个独立于[[汉族|汉族]]的民族<ref >{{cite web|url=http://finance.gucheng.com/201405/2715249.shtml|title=穿青人是什么族？ 一个宁愿死也不肯变成他族的民族}}</ref>。

== 历史 ==
由於穿青人的來源有土著說和分支說，造成穿青人的歷史有所不同。

===土著說===
据明朝史志记载，穿青人的族称早期叫“土人”，又叫“里民子”。称为“土人”，是因为他们“居土日久”，是贵州的土著民族，这是以居住的历史状况来称呼的；称为“里民子”，是因为他们先民与早期“里人”（亦泛称“僚”）有关，这些就是其历史源流来称呼的。而穿青人这个名称在官方記載最早见于[[乾隆|乾隆]]《威宁州志》，继后是[[光绪|光绪]]《平选州续志)和民国《大定县志》、《镇宁县志》、《平坝县志》。《平远州续志》和《[[大定县|大定县]]志》是在记述同治初年农民起义时提到的，是因其“衣尚青”，所以“又曰穿青”<ref>{{citeweb|url=http://www.zh5000.com/yc/xiqu/zgnx/2006/zgnx-0015.htm|title=穿青人与傩戏|publisher=中国非物质文化遗产网 资料来源：中共织金县委 织金县人民政府|author=|date=|accessdate=2010-12-01 |language=Zh-hans}}</ref>。在《镇宁县志》则把“穿青”作为该县的獨立民族载入史册；在民間記載方面，清康熙三年[[吴三桂|吴三桂]]平水西之后流传于民间的木刻唱本《水西传》记有了“穿青”這個民族，並与水西境内的[[彝族|彝族]]、[[苗族|苗族]]、[[仡佬族|仡佬族]]、[[龙家人|龙家人]]、[[蔡家人|蔡家人]]、[[羿人|羿人]]、[[白族|白族]]、[[仲家|仲家]]([[布依族|布依族]]的一支)等八个民族并列，统称“九种夷蛮”。由歷史文獻可知“穿青”这个名称，至迟在清初就有了。

===分支說===
民族专家[[宋蜀华|宋蜀华]]先生为代表的部分学者认为：穿青人是汉族的一支，是[[明朝|明朝]][[朱元璋|朱元璋]]派[[傅友德|傅友德]]、[[沐英|沐英]]征讨云南时从江西，江苏等省随军迁移到贵州的。战争结束后，大军就地垦戍，这部分随军汉人（包括军人和他们的家属）留居当地，和当地土著通婚融合，天长日久就成了今天的穿青人。

=== 识别过程 ===
1953年，中国政府开展了[[中华人民共和国第一次全国人口普查|第一次人口普查]]。两年后，[[全国人民代表大会民族委员会|全国人民代表大会民族委员会]]在贵州组建工作组，由[[费孝通|费孝通]]带头进入穿青人聚居区实地调查。四个多月后工作组完成《贵州省穿青人民族成分调查报告》，判断穿青人是[[明朝|明朝]]初年及以后一段时间内进入黔西少数民族地区的汉族移民，最终认定穿青人是汉族而不是少数民族。

当时费孝通认为：“迁居到少数民族地区的汉人，前后有若干批。早去的汉人曾经长期和内地隔绝，和后去的汉人，在语言、风俗习惯上有一定的区别，并且受到后去汉人的歧视，因而自认和汉人有区别，在新中国成立后，有人要求承认是少数民族。例如贵州的穿青、广西的六甲等。”同时政府出于[[民族团结|民族团结]]的目的，允许穿青人将民族成分填写为“青”。

1978年9月，[[费孝通|费孝通]]在[[中国人民政治协商会议|政协]]全国委员民族组会议中，重申“穿青是汉人，是汉族中的一部分，并不是少数民族”。同时他强调“必须在政治、经济上对穿青人适当照顾，帮助他们更快地发展起来”。

1981年，贵州省政府开展全省未定民族的识别工作。[[织金县|织金县]]、纳雍县、[[大方县|大方县]]民族事务委员会组成工作组，对穿青人民族成分进行[[民族成份恢復|第二次调查]]。1985年2月，工作组在《贵州省穿青人民族成分调查报告》中，提出穿青人是以贵州土著民族为主，与迁入人口融合而生的少数民族。结论“穿青是一个单一的少数民族，不是汉族”。该报告未获政府批准。同年，中国大陆实施[[中华人民共和国居民身份证|身份证制度]]。次年4月，贵州省政府下发《中共贵州省委常委办公会议纪要》(1986)29号文件，指示穿青、蔡家等人群的民族成分问题“没有明确解决之前，先维持现状。即：凡已经按少数民族对待的仍按少数民族对待，填写民族成分时原来怎么填写仍然怎么填写”。

1990年[[中华人民共和国第四次全国人口普查|第四次人口普查]]后，中国政府正式确认[[中华民族|56个民族]]。穿青人被列为“其他未识别民族”<ref name="a2"/>。由於穿青人擁有獨特文化，至今穿青人仍想成為一支獨立民族。同时中国实行的民族政策，使少数民族在[[民族区域自治制度|政治]]、[[计划生育政策|生育]]、教育、就业等方面享有优侍，甚至是[[两少一宽|司法特权]]。将穿青人归于汉族，则会丧失这些权利，{{来源请求|但一方面优秀人才可以避免天花板效应}}。

2014年5月上旬，从贵州[[纳雍县|纳雍县]]嫁到四川[[眉山|眉山]][[丹棱县|丹棱县]]的穿青人、23岁的郭丹，拿到的[[身份证|身份证]]上，“穿青人”三个字写在了身份证的民族一栏里<ref>{{cite web |url=http://sc.ce.cn/picnews/201405/08/t20140508_1501248.shtml |language= |title=56个民族之外“穿青人”写上身份证 |publisher=中国广播网 |date=2014年5月8日 |accessdate= |deadurl=yes |archiveurl=https://archive.is/20140508133331/http://sc.ce.cn/picnews/201405/08/t20140508_1501248.shtml |archivedate=2014年5月8日 }}</ref>。早前準備加入丈夫的戶口时，在民族欄當中，公安在「常住人口管理信息系統」中找不到穿青這個民族，無法辦理身份證。公安層層上報，最後決定為她在民族一欄選擇「其他」，之後再人手填入「穿青人」，以完成登記、製作身份證。

== 分佈和人口 ==
1954年普选登记，穿青人有二十四万八千多人。[[中华人民共和国第五次全国人口普查|2000年中国第五次人口普查]]中，自报穿青人有67万左右。主要分布在贵州省西部的[[毕节地区|毕节]]、[[安顺市|安顺]]、[[六盘水市|六盘水]]、[[黔南布依族苗族自治州|黔南]]、[[黔西南布依族苗族自治州|黔西南]]五个地州市所属的二十多个县内<ref name="a2">{{cite web|url=http://news.sina.com.cn/c/sd/2010-11-24/141521526235.shtml|title=贵州穿青族未被承认少数民族地位 人数近70万 |publisher=新浪网 来源：南都周刊|author=周鹏、吴桂霞|date=2010年11月24日|accessdate=2010-12-01 |language=Zh-hans}}</ref>。[[毕节地区|毕节地区]]的[[织金县|织金县]]、[[纳雍县|纳雍县]]各有穿青人20多万，占半数以上<ref name="a1"/><ref name="a3">{{cite web|url=http://www.gz.xinhuanet.com/zfpd/2008-08/21/content_14184513.htm|title=纳雍穿青人民俗风情|publisher=新华网 来源：中共纳雍县委宣传部|author=王景才|date=2008-08-21|accessdate=2010-12-01|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120505191455/http://www.gz.xinhuanet.com/zfpd/2008-08/21/content_14184513.htm|archivedate=2012-05-05}}</ref>。

== 文化 ==
穿青人在节日、婚姻、丧葬等方面的习俗多数与周围民族不同<ref name="a1"/>。
大多数集中于织金县和纳雍县，另一些生活在大方县、水城县、关岭县、清镇市、普定县、六枝（特区）县。 彝族称穿青族为Sher tu或Sher feizu，意思是“白皮肤的汉人”或是“吃蛇的汉人”，另有一些当地人称穿青族人为“大脚板”、“大袖子”。 穿青族将自己看作是特别的部族，尽管他们说汉语，从历史上来看他们属于汉族，但是政府将他们列入“未识别的民族”。

===信仰===
穿青人以猴为[[圖騰|图腾崇拜]]。祭祀“[[五显|五显]]”兵马是猴类山魈<ref name="a3" />。至今还保持[[五显坛|五显坛]]的信仰<ref name="a1"/>。

===服饰===
妇女没有[[缠足|缠脚]]的习俗<ref name="a3" />{{NoteTag|汉族中的[[客家人|客家人]]亦没有[[缠足|缠足]]的习俗。}}。当代，穿青服饰在大部分地区已近消失，只在偏远地区明显可见<ref name="a1" />。

== 语言 ==

穿青人原先使用一种汉语方言，稱“老辈子话”（今仍流行于黔东南及黔南地区），今多普遍使用贵州官话，但多数地区仍有这种方言的痕迹，如“f”变“h”，“ian”变“ie”等，可能与江西、湖北、湖南通行的汉语方言有渊源关系，關於“老辈子話”是否為独立民族语言，目前尚未有明确的证据。
