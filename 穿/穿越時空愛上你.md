{{作品名稱|name=''Kate & Leopold''|hk=隔世俏佳人|tw=穿越時空愛上你|equiv-cn=tw}}是一部2001年的[[浪漫喜剧|浪漫喜剧]][[奇幻电影|奇幻电影]]，敘述一位從 1876 年[[時間旅行|穿越時空]]到現代的[[公爵|公爵]]，與一位[[紐約|紐約]]女性墜入情海的故事。該片由[[詹姆斯·曼高德|詹姆斯·曼高德]]執導，[[梅格·萊恩|梅格·萊恩]]、[[休·傑克曼|休·傑克曼]]與[[李佛·薛伯|李佛·薛伯]]領銜主演。

== 其他版本 ==
DVD的總共包含兩種版本：原本118分鐘的劇院版，以及122分鐘的[[加长版|導演加長版]]。

== 電影配樂 ==
電影原聲帶於2001年12月25日發行。

== 評價 ==
在[[爛番茄|爛番茄]]上，根據126個評論，該片獲得了50%的指數，並提到：「雖然休‧傑克曼非常的迷人，梅格‧萊恩的表現卻枯燥無味、完全沒有驚喜；而穿越時空的情境也缺少邏輯性。」

== 獎項與提名 ==
[[休·傑克曼|休·傑克曼]]獲得2001年[[金球獎最佳音樂及喜劇類電影男主角|金球獎最佳音樂及喜劇類電影男主角]]提名；而由[[史汀|史汀]]創作與演出的主題曲《Until...》也贏得[[金球獎|金球獎]]最佳原創歌曲獎，並獲得[[奥斯卡最佳原创歌曲奖|奥斯卡最佳原创歌曲奖]]提名。<ref>[http://www.imdb.com/title/tt0035423/awards Awards for Kate & Leopold]</ref>
