'''鯖街道'''（さばかいどう）是[[小濱藩|小濱藩]]領內（相當於現在的[[嶺南_(日本)|嶺南]]）連結[[京都|京都]]的街道之總稱。主要是運送[[魚介類|魚介類]]往京都的[[物流|物流]]路線，其中大多是[[鯖魚|鯖魚]]，所以，近年來被稱作鯖街道<ref name="Wakasa Obama FC">{{cite web|url=http://www1.city.obama.fukui.jp/obm/kankou/blog/2008/02/post-90.html|title=鯖街道のお話（小浜広峰：いづみ町～京都） - 若狭おばまフィルムコミッション|publisher=小浜市観光交流課|accessdate=2014-04-01|deadurl=yes|archiveurl=https://web.archive.org/web/20131004230336/http://www1.city.obama.fukui.jp/obm/kankou/blog/2008/02/post-90.html|archivedate=2013-10-04}}</ref>。

== 概要 ==
狹義指現在的小濱市經由[[若狹町|若狹町]]三宅至京都市左京區的「若狹街道」，大概相當於[[國道27號|國道27號]]、[[國道367號|國道367號]]<ref>向笠千恵子『食の街道を行く』平凡社新書　2010年 p.13</ref>。但是，區間未必一致。廣義指現在嶺南連結京都的全部街道為鯖街道。

[[鐵道|鐵道]]、[[汽車|汽車]]普及以前的時代，在[[若狹灣|若狹灣]]捕獲的鯖魚以徒步的方式運至京都。除鯖魚以外，許多種類的海産也透過鯖街道運送。從[[平城宮|平城宮]]跡、奈良縣[[明日香村|明日香村]]發掘的[[木簡|木簡]]，推定有[[鯛魚|鯛魚]]的[[壽司|壽司]]等10種海産物來自[[若狹|若狹]]，可知鯖街道的起源在1200年以上或約1300年前。現代，在小濱、國道367號沿線等地仍有許多鯖魚壽司店。

2007年（平成19年），[[熊川宿|熊川宿]]作為「若狹熊川・鯖街道」登錄為[[日本風景街道|日本風景街道]]<ref>{{Cite web | url =http://kumagawa-juku.com/index02c11.html| title =若狭鯖街道熊川宿 |publisher= 鯖街道熊川宿まちづくり協議会 |accessdate = 2013-02-07 }}</ref>。

== 其他鯖街道 ==
以下列舉主要的街道、路線。
; 經由琵琶湖的路線<ref>鯖街道 「鯖街道 湖のルート」（p.104-105,p.114,p.119）</ref>
: 從小濱經由[[琵琶湖|琵琶湖]]的水運至京都的路線。
; [[西近江路|西近江路]]
: 從今津經琵琶湖西側陸路的路線。大概相當於現在的[[國道161號|國道161號]]。
; 小濱街道（根來・針畑路線）<ref>鯖街道 「根来・針畑越えルート」（p.152-155）</ref>
: 從小濱沿[[遠敷川|遠敷川]]而上，在根來坂峠・小入谷（現在的高島市朽木小入谷）經現在的[[滋賀縣道783號・京都府道781號麻生古屋梅之木線|滋賀縣道783號]]往[[久多|久多]]・[[花脊峠|花脊峠]]，之後經[[京都府道38號京都廣河原美山線|鞍馬街道]]至京都的路線。是小濱到京都的最短距離，過去的利用率高。
; [[周山街道|周山街道]]
: 從小濱經由名田庄的路線。大概相當於現在的[[國道162號|國道162號]]。
