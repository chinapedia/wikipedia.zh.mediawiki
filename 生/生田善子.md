'''生田善子'''（{{bd|1988年|11月26日|catIdx=Ikuta Yoshiko}}<ref name="Blog profile">{{Cite web |url=http://profile.ameba.jp/ikutayoshiko/ |title=生田善子のプロフィール |accessdate=2012-10-31}}</ref>）是[[日本|日本]]的女性[[聲優|聲優]]，隸屬於[[aptepro|aptepro]]。[[岡山縣|岡山縣]][[津山市|津山市]]出身。血型為[[A型|A型]]<ref name="Blog profile" />。[[桐朋學園藝術短期大學|桐朋學園藝術短期大學]]畢業。

== 經歷、人物 ==
* 電視、寫真雜誌、舞台為活動中心。2010年《[[閃光的夜襲|閃光的夜襲]]》為聲優出道作。
* 喜愛畫插畫，也非常擅長。
* 喜愛吃大的蘿蔓捲，時常攜帶到現場。
* 討厭的食物是納豆與粘糊狀臭氣強的之類的東西。
* 到2010年為止是Ain't的成員<ref>{{Cite web |url=http://ameblo.jp/ikutayoshiko/entry-10775095126.html |title=お知らせ。｜生田善子Blog |accessdate=2012-10-31}}</ref>。
