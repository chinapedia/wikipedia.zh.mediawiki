《'''惡靈古堡：啟示'''》（日版名：{{lang-ja|バイオハザード リベレーションズ}}，Biohazard Revelations；英文版名：{{lang-en|''Resident Evil: Revelations''}}）是[[卡普空|卡普空]]于2012年1月26日發行的[[動作遊戲|动作]][[驚悚遊戲|驚悚遊戲]]，其平台為[[任天堂3DS|任天堂3DS]]。

由於本系列一直以來多往家用主機開發，因此本作乃GBC《[[惡靈古堡_Gaiden|惡靈古堡 Gaiden]]》（未被官方列為正史）以來，系列史上第二部開發在掌機上的作品。《生化危机 啟示》还是首批公布的繁体中文版3DS游戏之一，并作为唯一的一款第三方游戏参与首发。<ref>[http://3ds.tgbus.com/news/201207/20120725130756_4.shtml  港台版3DS售价和游戏售价确定！生化危机首发]电玩巴士</ref>

2013年1月23日，[[卡普空|卡普空]]正式宣布将本作移植到高清平台：[[Xbox_360|Xbox 360]]、[[PlayStation_3|PlayStation 3]]、[[Wii_U|Wii U]]和[[Windows|Windows]]上；并将于5月21日在北美地区率先发售。<ref>[http://www.capcom-unity.com/brelston/blog/2013/01/22/resident-evil-revelations-coming-coming-to-360-ps3-wii-u-and-pc Resident Evil Revelations coming coming to 360, PS3, Wii U and PC]capcom-unity</ref>高清版的《啟示》不仅提高了画面质量，也加入了新的可操控角色，新武器以及新型敌人等新载游戏要素。<ref>[http://www.residentevil.com/revelations/ RESIDENT EVIL: REVELATIONS OFFICIAL SITE(HD)]Capcom</ref>高清版游戏的英文名称依旧为《{{lang|en|Resident Evil Revelations}}》，而日版名称则更改为《{{lang|ja|バイオハザードリベレーションズアンベールドエディション}}》（Biohazard Revelations - Unveiled Edition）。2017年8月，卡普空再次宣布将把游戏移植到[[Xbox_One|Xbox One]]、[[PlayStation_4|PlayStation 4]]和[[任天堂Switch|任天堂Switch]]上；Xbox One和PlayStation 4版本将于8月29日发售，任天堂Switch版将在2017年内发售。<ref>[http://www.ign.com/articles/2017/08/01/resident-evil-revelations-1-and-2-coming-to-nintendo-switch RESIDENT EVIL: REVELATIONS 1 AND 2 COMING TO NINTENDO SWITCH].IGN.2017-08-01.[2017-08-01].</ref>2019年3月28日，卡普空为PS4、Xbox One、STEAM平台的《生化危机 启示录》游戏增加了繁简中文的支持。<ref>{{Cite web |url = https://www.weibo.com/2485196582/HkGSnucUV |title = 【中文化消息】Resident Evil 2和Devil May Cry 5已经相继发售，新旧玩家如果想玩回系列作品，现在就是时候了！ |author = CapcomAsia |date = 2019-03-13 |publisher = 新浪微博 |language = zh-cn |accessdate = 2019-03-30 }}</ref>

== 故事概要 ==
2004年，恐怖主義集團灰獵犬（Veltro）因對地中海海上城市「浮島」（Terragrigia，又稱Grey Earth）的太陽能發展不滿而派出生化武器進行襲擊，聯邦生化恐怖委員會（Federation Bioterroism Commission，簡稱FBC）特工（包括潔西卡·席拉瓦德（Jessica Sherawat）和帕克·路奇亞尼（Parker Luciani））前往對應，城市最終被摧毀廢棄，事件被稱為浮島恐慌（Terragrigia Panic）。

2005年，灰獵犬據報有復活跡象，生化恐怖主義安全評估聯盟（Bioterrorism Security Assessment Alliance，簡稱BSAA）成員克里斯·雷德菲爾（Chris Redfield）與潔西卡因此前往調查，但二人在郵輪潔諾比亞女王號（Queen Zenobia）發出最後訊號後失去聯絡。聯盟主席克萊夫·R·奧布萊恩（Clive R. O'Brian）遂派遣吉兒·范倫廷（Jill Valentine）與帕克尋找二人下落。因此BSAA與恐怖組織的對決正式拉開序幕。

== 登場角色 ==
按照日本版[[生化危機系列|生化危機系列]]的慣例都是採用英文配音日文字幕，本作除了英文配音和日、英文字幕之外還首次加入日文、法文、意大利文、德文及西班牙文的配音及法、意、德及西文的字幕，在遊戲中的設定模式（Option Mode）裡可切換配音和字幕。

而作为系列首次官方中文化的本作，《启示录》只是加入了繁体中文的界面以及台词字幕。

== 游戏开发 ==
《生化危机：启示录》的游戏开发团队之前开发过《[[生化危机：佣兵_3D|生化危机：佣兵 3D]]》，<ref name="andriaiwataasks">{{cite web |url=http://www.andriasang.com/e/blog/2011/05/19/iwata_asks_re/ |title=Iwata Asks About Resident Evil |author=Gantayat, Anoop |date=May 19, 2011 |work=Andriasang |accessdate=July 14, 2013 |archiveurl=https://archive.is/20130121001716/http://andriasang.com/comvpf/iwata_asks_re/ |archivedate=2013年1月21日 |deadurl=yes }}</ref>并且《启示录》的立项时间要早于《佣兵 3D》，虽然是后者先发售的。这样做的原因是因为[[CAPCOM|CAPCOM]]希望通过制作《佣兵 3D》以磨合开发团队对掌上游戏机平台的技术要求以及积累经验，并测试公司对[[任天堂3DS|任天堂3DS]]平台的开发游戏软件可行性。<ref name="andriaiwataasks" />《启示录》的游戏引擎采用了专门为任天堂3DS调整匹配过的CAPCOM自家开发的[[MT_Framework|MT Framework]]引擎，该引擎并被命名MT Framework Mobile。<ref name="engine">{{cite web |url=http://www.andriasang.com/e/blog/2010/09/13/3ds_mt_framework_detailed/ |title=Capcom on 3DS MT Framework and Resident Evil: Revelations |author=Gantayat, Anoop |date=September 13, 2010 |work=Andriasang |accessdate=July 14, 2013 |archiveurl=https://archive.is/20130120235054/http://andriasang.com/comp7w/3ds_mt_framework_detailed/ |archivedate=2013年1月20日 |deadurl=yes }}</ref>MT Framework Mobile是基于使用MT Framework引擎制作的第三人称射击游戏《[[失落的星球2|失落的星球2]]》的一个早期版本引擎。<ref name="engine"/>

== 游戏音乐 ==
《生化危机：启示录》中的游戏音乐主要由鈴木幸太、三浦健与講本一朗完成编曲和演奏，三人此前都有为《生化危机》系列制作音乐的经验。面对《启示录》回归系列原始的惊悚恐怖风格，本作的音乐为游戏气氛的营造起到了很大的作用。在2011年的[[东京电玩展|东京电玩展]]上，CAPCOM为参与《生化危机：启示录》试玩的玩家提供了一张免费的游戏原声音乐CD，内含四首游戏原声。<ref>[http://gnn.gamer.com.tw/8/57668.html 【TGS 11】CAPCOM 七大主題遊戲攤位 大手筆打造祭典熱鬧氛圍].2011-09-16.巴哈姆特.[2013-07-14].</ref>
