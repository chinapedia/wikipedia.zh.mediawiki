《'''生死格斗'''》（{{lang-en|'''DOA: Dead or Alive'''}}）2007年美国、英国、德国联合制作的[[动作片|动作片]]，演员阵容来自英国、日本、德国、中国，导演元奎，主演戴文·青木、莎拉·卡特、杰米·普莱斯利。<ref>{{cite web |url= http://ent.sina.com.cn/m/u/2007-08-20/10171681586.shtml|title=游戏《生死格斗》上银幕 元奎调教众美少女(图) |accessdate= 2007|publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/h/2007-08-14/01411675345.shtml |title= 元奎巨资打造动作大片 《生死格斗》本月来袭|accessdate=2007 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://ent.sina.com.cn/x/2007-08-25/13101689100.shtml|title=“多国部队”《生死格斗》 |accessdate=2007 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==剧情==
日本的一名公主“霞”不听劝阻执意参加“生死格斗”，她并不是为了丰厚的奖金，而是为了寻找哥哥“疾风”，当她过了重重关卡，最终在生死格斗里见到了哥哥，并且打败魔头组织，炸毁比赛场地。

==花絮==
该片曾在[[曼谷|曼谷]]、[[桂林市|桂林市]]、[[浙江省|浙江省]][[横店影视城|横店影视城]]、[[香港|香港]]等地取景，一共花费预算$21,000,000。<ref>{{cite web | url=http://www.imdb.com/title/tt0398913/business |title=Business Data for DOA: Dead or Alive | publisher=Internet Movie Database | accessdate=2006-08-28 }}</ref><!-- "the first Western production shot largely in China's Hengdian World Studios" -->

该片上映之后，因性感火辣美女、动作招式华丽而吸引大批观众，票房大丰收，一共$7,516,532。<ref>[http://boxofficemojo.com/movies/?id=doadeadoralive.htm DOA: Dead or Alive (2007)<!-- Bot generated title -->]</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/2007-08-29/02201693744.shtml |title=元奎带美女杀入暑期档 《生死格斗》票房不俗 |accessdate=2007 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://yule.sohu.com/20070606/n250418191.shtml|title=导演元奎拍《生死格斗》 华丽武打招式吸引观众 |accessdate=2007 |publisher=搜狐网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://yule.sohu.com/20070817/n251645895.shtml |title=《生死格斗》将性感出击 最具魅惑电影风靡全球 |accessdate=2007 |publisher= 搜狐网|work= |language=zh-hans}}</ref>
