'''生命熱線'''（{{lang-en|'''Suicide Prevention Services'''}}，簡稱{{lang|en|'''SPS'''}}），是[[香港|香港]]一個[[志願服務機構|志願服務機構]]，為有自殺傾向、感到絕望及有情緒困擾的人士提供輔導服務。會址位於[[九龍|九龍]][[東頭邨|東頭邨]]逸東樓，於2011年1月10日喬遷[[坪石邨|坪石邨]]新址。

==歷史==
[[File:HK_SPS_TungTauEstate.JPG|thumb]]的舊會址]]
生命熱線的前身「{{lang|en|The Samaritans (Cantonese Speaking Service)}}」（[[撒瑪利亞會|撒瑪利亞會]][[粵語|粵語]]服務）於1995年以社團條例成立，提供每日12[[小時|小時]]的預防[[自殺|自殺]][[熱線輔導|熱線輔導]]服務，於1997年延長熱線服務至18小時，直到1998年提供24小時運作服務。2000年將[[英語|英語]]名稱正式改為現時的「{{lang|en|Suicide Prevention Services}}」。2011年申請註冊成為「生命熱線有限公司」（{{lang|en|Suicide Prevention Services Limited}}）。

1998年10月21日[[溫拿樂隊|溫拿樂隊]]擔任「生命熱線宣傳大使」，並義務拍攝「千祈唔好放棄」宣傳短片宣揚防止自殺的信息，主題曲名為《OLE OLE 永不放棄》。[[陳志雲|陳志雲]]於2009年4月27日至29日一連三場在[[香港文化中心|香港文化中心]]舉行《[[志雲飯局|志雲飯局]]{{lang|en|Live on Stage}}舞台版》為生命熱線籌款<small><ref>{{cite web |language=zh-hk |url=http://www.hkheadline.com/ent/ent_content.asp?contid=75966&srctype=n |title=為「生命熱線」籌款 甘做丑角論盡緋聞 陳志雲：我擔心問題唔夠尖銳！ |publisher=《[[頭條日報|頭條網]]》 |date=2009-04-14 |accessdate=2009-04-29}}</ref></small>。

陳志雲的《志雲情人節首選》於2010年2月7日在[[奧海城|奧海城]]舉行慈善義賣，部份收益撥捐生命熱線，更透露2009年年底[[無綫電視|無綫電視]]藝人[[崔建邦|崔建邦]]因涉嫌襲擊緋聞女友[[嫩模|嫩模]][[鄺敏慧|鄺敏慧]]而情緒受困擾，曾尋求生命熱線輔導<small><ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20100208&sec_id=462&subsec_id=830&art_id=13707856 |title=王喜頂替主持　唔敢收有骨利是 志雲爆崔建邦求助生命熱線 |publisher=《[[蘋果日報|''hk.apple.nextmedia.com'']]》 |date=2010-02-08 |accessdate=2010-02-08}}</ref></small>。

==服務==
[[File:HK_SPS_PingShekEstate.JPG|thumb]]的新會址]]

===熱線服務===
「24小時預防自殺熱線」（2382 0000）自1995年開始服務，由訓練有素的[[義工|義工]]為有自殺傾向的人士提供保密的電話服務。於2001年新開設「珍愛生命長者熱線」（2382 0881），專為患有[[抑鬱症|抑鬱症]]及有自殺念頭的長者提供服務，按需要作出跟進和轉介服務。

===外展服務===
「生命共行—外展長者服務」於2006年10月開展，獲「[[香港賽馬會|香港賽馬會]][[慈善|慈善]][[信託|信託]][[基金|基金]]」撥款，推行為期3年的服務，以回應長者[[自殺|自殺]]問題的嚴重情況。服務透過外展形式及與地區機構合作，發掘高危長者，並招募及訓練[[義工|義工]]探訪有需要的個案。服務成效交由「[[香港大學|香港大學]]香港賽馬會防止自殺研究中心」進行研究和檢討。

「長者護心大使推廣計劃」於2008年7月開展，獲「''Vera Ruttonjee Desai Charitable Fund''」資助，透過招募及訓練「長者護心大使」，為情緒受困或缺乏社區支援的長者提供支援，促進聯繫及強化社區支援網絡。

===支援服務===
「釋心同行－自殺者親友支援服務」於2008年4月開展，獲「[[李家傑_(商人)|李家傑]]珍惜生命基金」贊助375萬港元，為有可能步[[自殺|自殺]]者後塵的親友提供服務，協助他們過渡哀傷，重建生命的意義。

===社區教育===
生命熱線定期舉行社區教育活動，如巡迴展覽、講座及嘉年華等；並印有多款小冊子及單張宣揚有關訊息。
