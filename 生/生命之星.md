'''生命之星'''（{{lang-en|'''Star of Life'''}}）是[[藍色|藍色]]，六邊突出的星，並在中間加上[[阿斯克勒庇奧斯之杖|阿斯克勒庇奧斯之杖]]（另一種說法是[[摩西|摩西]]所鑄之柱），其原本的設計及決定是來自[[美國交通部|美國交通部]]屬下的[[美国国家公路交通安全管理局|美国国家公路交通安全管理局]]。國際上生命之星是表示[[急救學|急救]]單位或者人員。而類似的[[橙色|橙]]星（橙色生命之星）則是給[[搜索及救援|搜索及救援]]人員使用。

== 由來 ==
第一個由來，是由古希臘神話中的蛇與權杖，是為了紀念[[阿斯克勒庇奧斯|阿斯克勒庇奧斯]]這位偉大的神醫。

第二個由來，是依據[[聖經|聖經]]中的[[摩西|摩西]]以青銅鑄造一條蛇的形狀並將他鑲在一根柱子上，若有人被毒蛇咬到，只要到柱子下注視著青銅鑄的蛇，就會馬上獲得痊癒。（[[民數記|民數記]]第21章第4至9節）。

第三個由來，是[[聖杯|聖杯]][[所羅門|所羅門]][[封印|封印]]之形狀，出自[[聖經|聖經]][[以賽亞書|以賽亞書]]第6章第1至3節及[[民數記|民數記]]第21章第4至9節。

== 歷史 ==
生命之星於1967年的專利為[[美國醫學組織|美國醫學組織]]（AMA）擁有，其後「轉讓」予[[NHTSA|NHTSA]]，成為屬下[[急救服務|急救服務]]的標誌。當初，[[美國交通部|美國交通部]]的[[道森·米爾斯|道森·米爾斯]]（Dawson Mills）曾經請求[[國際紅十字與紅新月聯合會|美國紅十字會]]提供紅十字標誌給急救服務的[[救護車|救護車]]使用，惟被拒絕。其後，米爾斯向美國醫學組織的德克法林頓查詢可否提供生命之星，後來被批准。而這六邊突出的星由[[里奧_R‧喬納森|里奧 R‧喬納森]]（Leo R. Schwartz）命名。生命之星在1977年2月1日註冊為[[保證記號|保證記號]]。

[[香港消防處|香港消防處]]在[[2006年9月|2006年9月]]為旗下的[[救護車|救護車]]車身加上生命之星；而[[台灣|台灣]]的消防隊[[緊急醫療救護技術員|緊急醫療救護技術員]]則會穿上印有生命之星且在下方寫上[[EMT|EMT]]的[[背心|背心]]，而消防隊救護車後側車窗上也貼有生命之星和119字樣。Emergency Medical Technician的衣服在中國及各地的救護車也印有相同圖案。

== 符號的象徵 ==
這六分支的星是代表6種主要通過救援者執行的連串緊急任務：
# 第一位救援者觀察及了解現場，明白當前問題，確認病人/傷者(們)的危險，作出適當的估計，保證現場的安全（傳染、電流、化學、輻射等）。
# 第一位救援者報告及要求專業的救援。
# 第一位救援者依權限進行[[急救|急救]]程序並提供立即的照顧。
# EMS人員到達並依權限提供立即的照顧。
# EMS人員進行運送病人/傷者到[[醫院|醫院]]接受治療。並在運送途中提供醫學上的照顧。
# 以適當及專業的照顧送到[[醫院|醫院]]。

更通常地，以下6個能代表EMS觀點:
#發現（Detection）
#報告（Reporting）
#出勤（Response）
#現場處理（On scene care）
#運送處理 （Care in transit）
#運送到醫療機構（Transfer to definitive care）
