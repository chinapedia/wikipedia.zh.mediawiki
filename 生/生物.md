'''生物 '''（[[拉丁语|拉丁语]]，[[德语|德语]]: '''Organismus'''， {{Lang-en|'''Organism'''}}，又称'''有機體'''）是指稱類[[生命|生命]]的[[个体|个体]]。在[[生物学|生物学]]和[[生态学|生态学]]中， 地球上约有870萬種[[物種|物種]]（±130萬），其中650萬種物種在陆地上，220万种生活在水中<ref>{{cite news|title=地球上有多少物種？870萬種|url=http://chinese.eurekalert.org/zh/pub_releases/2011-08/aaft-hms081811.php|accessdate=2011-08-25|newspaper=eurekalert|date=2011年8月24日}}</ref>。

生物最重要和基本的特徵在生物會進行[[新陳代謝|新陳代謝]]及[[遺傳|遺傳]]兩點，前者說明所有生物一定會具備[[合成代谢|合成代谢]]以及[[分解代谢|分解代谢]]（兩個是完全相反的兩個生理反應過程），並且可以將遺傳物質複製，通过自我分裂生殖(無性生殖)或有性生殖，交由下一代[[繁殖|繁殖]]下去以避免滅絕，这是類生命现象的基础。

[[生命的起源|生命的起源]]和生命各个分支之间的关系一直存在争议，古早的生命分類已經過時，近代古典生物學的分類又受到分子生物學的挑戰。一般而言，我們將生物分為兩大類：[[原核生物|原核生物]]和[[真核生物|真核生物]]。原核生物分为兩大域：[[细菌|细菌]]（Bacteria）和[[古菌|古菌]]（Archaea），这两个域相互之间的关系并不比他们和真核生物的关系更为接近。在[[演化|演化]]史的研究上，原核生物和真核生物之间一直缺乏联系。類似麻煩的還有病毒與內共生細菌等的分類，隨著現代生物化學的研究逐漸深入，出現了有如物理學中存在量子現象一般，在特定微觀世界下許多傳統認知出現錯誤，導致以往常理被顛覆的情況。

真核生物的特徵是有[[細胞核|細胞核]]以及其他[[膜狀細胞器|膜狀細胞器]]（例如動物和植物體內的[[粒線體|粒線體]]粒線體也可以說是植物動物體的發電廠，因為他可以釋放出很多的能量，以及植物及藻類中的[[葉綠素|葉綠素]]），一種假說是[[叶绿体|叶绿体]]和[[线粒体|线粒体]]是由[[内共生|内共生]][[细菌|细菌]]（endosymbiotic bacteria）演化而来<ref name=cavaliersmith1987>T.Cavalier-Smith (1987) The origineukaryote and archaebacterial cells, Annals of the New York Academy of Sciences 503, 17–54</ref> 。[[多细胞生物|多细胞生物]] 則指包含多于一个细胞的生物，在地質學上直到五億年前才出現大爆發。

== 共同特征 ==
代謝和遺傳是最大兩點，但在複雜生命中需要更多屬性來定義，只是这些属性并非普遍存在，详见下。例如[[病毒|病毒]]就是相當特殊的生命表現形式。
生物的共有属性主要有四個，一般我們會包含：
#[[新陳代謝|新陳代謝]]（化學物質的生成與分解）
#[[发育生物学|生長發育]]（收集、儲存資源）
#[[繁殖|繁殖]]（遺傳物質的傳遞，但很多生物的个体无法进行[[繁殖|繁殖]]——尽管它们所属的物种可以繁殖）
#[[適應|適應]]（外部物理环境变化不至於影響體內的化學變化）

較不明顯的有其他三個
#[[应激性|应激性]]（但有些生物无法直接对刺激产生反应，如植物沒有神經）
#[[運動|運動]]（许多生物无法独立移动）
#[[組織性|組織性]]（生物間如雌雄的互動、護幼、共生、社會組織等等）

==化學==
生物體是複雜的化學系統，其作用在維持生物體的生存及發展，以及繁衍後代。[[生物化學|生物化學]]主要研究生物體內的化學現象<ref>{{cite web|url=htt學|publisher=博客來|accessdate=2014-02-10}}</ref>。整個生物體的現象可以決定生物是否可以適合其環境，也決定了其中[[DNA|DNA]]內的基因是否可以繼續存續。

生物體的代謝及其他許多內部機能都和化學反應有關，特別那些有關大型有機分子的化學。生物體是[[化學物質|化學物質]]形成的複雜系統，藉由和環境的互動，有各式各樣的角色。

生物體是半封閉的化學系統，雖然生物體是生命的單位，但生物體和環境不是完全封閉。生物體會吸收及釋放能量，[[自养生物|自养生物]]利用陽光或其他無機物質來產生可用的能量（一般會以有機物質的形式出現），[[異營生物|異營生物]]則是利用環境中的有機物質中的能量。

有機物質中主要的化學元素是[[碳|碳]]，碳原子的特點是有很強的親合力可以和小原子鍵結，也可以和其他的碳原子鍵結<ref>{{cite web|url=http://class.ibucm.com/yjhx/1/right2.htm|language=zh |title=碳原子的特性及有机化合物的特点　★★ |accessdate=2014-02-10 }}</ref>，而且因為其體積小，可以同時和多個原子鍵結，因此是有機生物體的基礎。碳可以形成三個原子的簡單分子（[[二氧化碳|二氧化碳]]），也可以形成有數千個原子，可以儲存資料的長鏈（[[核酸|核酸]]）。

===高分子===
組成生物的分子可以分為[[高分子|高分子]]（也称为生物大分子）及其他較小的分子。這些高分子包括[[核酸|核酸]]、[[蛋白質|蛋白質]]、[[醣|醣]]和[[脂質|脂質]]等<ref>[http://www2.nsysu.edu.tw/Bio/images/commen/biomacro98.pdf 生物大分子簡介]</ref>。核酸（特別是去氧核糖核酸，DNA）用[[核苷酸|核苷酸]]的序列來儲存資訊。四種核苷酸（[[腺嘌呤|腺嘌呤]]、[[鳥嘌呤|鳥嘌呤]]、[[胸腺嘧啶|胸腺嘧啶]]和[[胞嘧啶|胞嘧啶]]組成的特殊序列決定了生物體的許多特徵。核苷酸的序列可以拆成許多由三個核苷酸組成的[[遗传密码|遗传密码]]，對應一種特殊的[[胺基酸|胺基酸]]<ref>{{cite web|url=http://www.hightech.tw/index.php/2012-06-06-14-12-38/30-molecular-biology/232-heredity-code|language=zh|title=15-14遺傳密碼(Heredity code)|publisher=科技台灣|accessdate=2014-02-10|deadurl=yes|archiveurl=https://web.archive.org/web/20140222003555/http://www.hightech.tw/index.php/2012-06-06-14-12-38/30-molecular-biology/232-heredity-code|archivedate=2014-02-22}}</ref>。因此DNA的序列對應某一種特殊的蛋白質，而且由於其成份胺基酸的化學性質，蛋白質會依特殊的方式[[蛋白质折叠|折叠]]，而且會執行一特定的機能。

以下是已知蛋白質的機能：
# [[酵素|酵素]]，作為代謝反應的催化劑。
# 結構蛋白質，像[[微管蛋白|微管蛋白]]或[[膠原蛋白|膠原蛋白]]。
# 調節蛋白質，[[轉錄因子|轉錄因子]]或是調節細胞週期的[[週期素|週期素]]。
# 作為信息分子或信息分子的受體，像某些[[激素|激素]]及其受體。
# 防禦蛋白質，包括[[免疫系統|免疫系統]]中的[[抗体|抗体]]，或是毒素（例如{{link-en|曼巴蛇素|dendrotoxin}}），或是包括像{{link-en|刀豆氨酸|canavanine}}等特殊胺基酸的蛋白質。

雙層的[[磷脂|磷脂]]組成了[[磷脂雙分子層|磷脂雙分子層]]，是細胞膜的主要結構，包覆在細胞的外面，防止化學物質自由的進出細胞。由於磷脂雙分子層的[[選擇性滲透|選擇性滲透]]，只有特定分子才能通過細胞膜。在一些多細胞的生物中，這可以是儲存能量及在細胞間傳播資料的方法。相較於脂質及蛋白質，[[醣|醣]]更容易分解，也更容易轉換為能量，在所有生物體中，醣是最常使用的有機分子能量來源。

例如Homo sapiens是現代人類（[[智人|智人]]）的拉丁[[二名法|二名法]]學名，所有[[人屬|人屬]]的生物理論上可以繁殖產生後代，不過若是不同物种的生物，所生下的生物沒有繁殖能力。人屬的生物現在只剩下人類，其他像[[直立人|直立人]]、[[尼安德特人|尼安德特人]]等人屬的生物已經在上千年前就絕種了。最後，會依生物的基因及結構特徵，放在三個域中適合的界中（以此例，為[[動物界|動物界]]）。

所有科學上已知的生物都可依此系統分類，因此同一科的生物在基因上會比同一綱的生物更加接近。

因為病毒不屬於生物，病毒的分類相當有挑戰性。最早病毒會依其宿主分類：動物病毒、植物病毒及[[噬菌體|噬菌體]]，後來會依產生的疾病分類，例如呼吸道病毒。現在病毒是依其核酸、衣壳的对称性及是否有包膜來分類。

== 病毒 ==
{{main|病毒}}
病毒由于不能独立进行[[繁殖|繁殖]]和[[新陈代谢|新陈代谢]]而通常不被认为是生物<REF>{{cite journal lua||author=比雅瑞爾 |coauthors= 涂可欣（翻譯）|year= 2004 |title=病毒不是活的嗎？ |journal=科學人雜誌 |url=http://sa.ylib.com/MagCont.aspx?PageIdx=1&Unit=featurearticles&Cate=&id=607&year= |accessdate= 2014-02-07 }}</REF>。然而，依据[[美国法典|美国法典]]（United States Code）的[[生物武器|生物武器]]和非法使用相关内容中[[病毒|病毒]]被归为[[微生物|微生物]]范畴。由于许多[[寄生|寄生]]动物和[[内共生体|内共生体]]（endosymbionts）也缺乏独立生存能力，所以病毒是否算作生物仍然存在争议。尽管病毒有[[酶|酶]]和其他生物特有的分子，它们在[[寄主细胞|寄主细胞]]外却无法生存，并且病毒[[新陈代谢|新陈代谢]]的过程需要寄主遗传机制的参与。这种[[寄生|寄生]]现象的起源还不清楚，但有可能产生于寄主。

== 壽命 ==
寿命是生物的基本参数之一。有的生物只能生存一天，有的生物例如一些[[植物|植物]]能生存几千年。[[燈塔水母|燈塔水母]]是已知唯一一種目前沒有壽命限制的生物。

[[细胞衰老|细胞衰老]]在决定生物体、[[细菌|细菌]]、[[病毒|病毒]]甚至是[[朊毒体|朊毒体]]的寿命时很重要。

目前，科学界普遍认为存在于细胞染色体末端的一段特殊的[[DNA|DNA]]序列——[[端粒|端粒]]与细胞的寿命有着很大的关系<REF>{{cite journal lua|author= 龐中培|year= 2011年 |title=端粒研究剛開端 |journal=科學人雜誌 |url=http://sa.ylib.com/MagCont.aspx?Unit=featurearticles&id=1864 |accessdate= 2013-12-29}}</REF>。通常情况下，[[细胞|细胞]]每分裂一次，端粒就会变短一些。随着端粒逐渐缩短，最后造成了位于[[染色体|染色体]][[DNA|DNA]]中间段的对[[细胞|细胞]]生命活动有意义的[[DNA|DNA]]序列的缺失。由于此时无法继续进行正常的生理活动，细胞便会进行一种由自身控制的程序性死亡——[[细胞凋亡|细胞凋亡]]。

此外，[[肿瘤|肿瘤]]细胞中的端粒结构通常没有缩短，这也是[[肿瘤|肿瘤]]细胞能够进行无限制分裂的原因之一。

== [[演化|演化]] ==
{{see also|自然發生|共同起源}}
在生物學上，[[共同起源|共同起源]]的理論提出地球上所有的生物都起源於一個共同的祖先或祖先的基因庫，可以在所有生物體之間共同的特徵找到共同的祖先的證據。在達爾文的時代，證據完全是基於共同的特性可見的形態相似，例如所有的鳥有翅膀。
