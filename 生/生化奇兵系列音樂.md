'''[[生化奇兵系列|生化奇兵系列]]的音樂'''由1930年代、1940年代和1950年代的授權音樂，以及由{{link-en|加里·席曼|Garry Schyman}}創作的原創管弦樂配樂所組成。

== 生化奇兵 ==

=== 原創配樂 ===

《[[生化奇兵|生化奇兵]]》的原創配樂由{{link-en|加里·席曼|Garry Schyman}}作曲。 一系列曲目曾透過官方[[2K_Games|2K Games]]網站以[[MP3|MP3]]格式於2007年8月24日免費發給粉絲。 但目前的鏈接是失效。<ref>{{Cite web |url = http://www.2kgames.com/cultofrapture/article/orchestralscore |title = INTRODUCING THE BIOSHOCK ORCHESTRAL SCORE |author = Elizabeth Tobey |date = 2007-08-24 |publisher = 2K Games |language = en}}</ref>
