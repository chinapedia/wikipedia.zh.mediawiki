'''发宝象龙塔'''位于中国[[浙江省|浙江省]][[武义县|武义县]]城区北部金鞍山上，1997年被列为[[浙江省文物保护单位|浙江省文物保护单位]]。

发宝象龙塔建于[[明朝|明]][[万历|万历]]三十年（1602年）九月至三十三年五月，为六角七层砖塔，高42.46米，底层每面3.40米，壁厚1.36米。<ref>{{Cite web |title=发宝象龙塔的意味 |publisher=武义县图书馆 |url=http://www.wylib.com/info/News_View.asp?NewsID=346 }}</ref><ref>{{Cite web |title=发宝象龙塔 |publisher=浙江省文物局 |url=http://www.zjww.gov.cn/subject/2006-02-13/855834.shtml }}</ref>
