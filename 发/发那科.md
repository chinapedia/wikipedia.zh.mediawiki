'''发那科'''（{{lang-en|FANUC}}，也有译成'''法兰克'''），是一間提供如[[机器人|机器人]]和电脑[[数控机床|数控机床]]等自动化产品服务的公司，主要由[[日本|日本]]“发那科公司”（{{jpn|j=ファナック株式会社|rm=Fanakku Kabushikigaisha}}）、[[美国|美国]][[密歇根州|密歇根州]]罗切斯特山的“美国发那科公司”以及[[卢森堡|卢森堡]]的“欧洲发那科公司”組成。发那科是世界上最大的工业机器人制造商之一，公司名称是富士自动化数控（'''F'''uji '''A'''utomatic '''NU'''merical '''C'''ontrol）的英文缩写<ref>{{Cite web|url=http://www.fanuc.co.jp/ja/profile/news/pdf/fanucnews2015iii.pdf |title=FANUC News 2015 |date= |accessdate=2016-06-24}}</ref>。最早期的发那科公司是[[富士通|富士通]]开发的早期数字控制（NC）和[[伺服機構|伺服系统]]的一部分。

1972年，电脑控制分部独立和发那科有限公司成立。<ref>{{cite web |url=http://www.fanuc.co.jp/en/profile/index.htm |title=FANUC's Management - Profile |accessdate=2008-06-06  |publisher=FANUC Corporation}}</ref>

发那科公司被列为[[东京证券交易所|东京证券交易所]]的第一部，是TOPIX100指数<ref>{{cite web |url=http://www.tse.or.jp/english/market/topix/data/b7gje60000003v9e-att/Core30-201310-e.pdf |title=TOPIX Core30 Components |publisher=[[Japan_Exchange_Group|Japan Exchange Group]] |accessdate=March 19, 2014 |deadurl=yes |archiveurl=https://web.archive.org/web/20140513220730/http://www.tse.or.jp/english/market/topix/data/b7gje60000003v9e-att/Core30-201310-e.pdf |archivedate=2014年5月13日 }}</ref>和[[日经平均指数|日经平均指数]]<ref>{{cite web|url=http://indexes.nikkei.co.jp/en/nkave/index/component?idx=nk225#23 |title=Components:Nikkei Stock Average |publisher=[[Nikkei_Inc.|Nikkei Inc.]] |accessdate=March 19, 2014}}</ref>[[股市指数|股市指数]]的成分股。它的总部设在[[山梨县|山梨县]]。

该公司的客户包括美国和日本的汽车和电子产品制造商。工业机器人的使用使得[[尼崎市|尼崎市]]的[[松下電器|松下電器]]一个仅仅25人的工厂在一个月内就能生产2百万台电视机，产品主要是高端等离子液晶屏幕，包括一个103英寸的型号。<ref>[http://www.trustedreviews.com/tvs/review/2007/10/24/Panasonic-Press-Tour-Japan-2007/p3 TrustedReviews.com Panasonic Press Tour Japan 2007 p2-3]</ref>

FANUC在46个国家拥有超过240个[[合资企业|合资企业]]，子公司和办事处<ref>{{cite web|title=Annual Report 2015|url=http://www.fanuc.co.jp/en/ir/annualreport/pdf/annualreport2015_e.pdf|publisher=FANUC Corporation|access date=January 23, 2016}}</ref>。它是数控系统最大市场份额的生产商，具有65％的全球市场份额。<ref>{{cite web |url=http://www.engineering.com/Products.aspx?page=comp/FAN011 |title=FANUC company profile |publisher=ENGINEERING.com, Inc. |accessdate=September 17, 2014}}</ref>它是工厂自动化系统的全球领先制造商。<ref>{{cite web |url=http://www.fanuc.eu/en/about-us/history |title=FANUC History |publisher=FANUC Europe Corporation |accessdate=March 19, 2014}}</ref>

== 历史 ==
* 1972年5月 - 富士通的计算控制单元，变独立作为富士通的子公司。公司名称“富士通发那科公司”。
* 1976年11月 - 列在[[东京证券交易所|东京证券交易所]]第二部。
* 1982年7月 - 并更名为“发那科公司”。
* 1983年9月 - 列在东京证券交易所第一部。
* 1986年 - 与美国[[通用电气|通用电气]]公司的合资企业，成立了GE Fanuc自动化公司。
* 1987年 - 开发了二氧化碳激光器。
* 1991年 - 在德国成立了当地法人FANUC 欧洲公司。
* 1992年 - 成立了合资FANUC INDIA PRIVATE LIMITED，和成立了合资公司北京发那科有限公司。
* 2000年 - 设立了FANUC员工的养老基金。
* 2003年 - 成立了FANUC越南有限公司。
* 2012年 - 成立了FANUC健康保险工会。
* 2015年 - 合作机器人CR-35iA完成。光纤激光器已经研制成功。
* 2016年 - 6月1日，在[[枥木县|枥木县]][[壬生町|壬生町]]的工厂竣工<ref>{{cite web|url=https://www.facebook.com/mibutown.smile/posts/819643734835453|title=壬生町長のすまいるとーく|publisher=[[facebook|facebook]]|format=|date=|accessdate=2016-08-28}}</ref>。
