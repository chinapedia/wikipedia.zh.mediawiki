{{日本作品|妄想學生會|生徒会役員共}}是[[日本|日本]][[漫畫家|漫畫家]][[氏家ト全|氏家ト全]]的一部以高中[[學生會|學生會]]為題材的[[四格漫畫|四格漫畫]]作品。先在《[[Magazine_SPECIAL|Magazine SPECIAL]]》的2007年6月號至2008年7月號上連載，後來改為在《[[週刊少年Magazine|週刊少年Magazine]]》（2008年3月4號開始）上連載。至今已發行16本單行本，[[台灣|台灣]]中文版由[[東立出版社|東立出版社]]代理發行。2010年4月公佈了其[[電視動畫|電視動畫]]化的決定<ref>{{cite news|url=http://www.animenewsnetwork.com/news/2010-04-24/seitokai-yakuindomo-manga-gets-tv-anime-green-lit|title=''Seitokai Yakuindomo'' Manga Gets TV Anime Green-Lit|publisher=[[動畫新聞網|動畫新聞網]]|date=2010年4月24日|accessdate=2013年10月11日}}</ref>，2010年7月3日至9月25日播放電視動畫第1季<ref>{{cite news|url=http://www.tvdrama-db.com/drama_info/p/id-43469|script-title=ja:生徒会役員共|publisher=Furusaki Yasunari|language=ja|accessdate=2013年10月11日}}</ref>，2014年1月4日至3月29日播放電視動畫第2季。另外，官方在《周刊少年Magazine》2017年第2、3月號合刊中發布了劇場版動畫製作決定的消息，沿用第二季製作班底，同時也是二期OAD的20、21話，2017年7月21日上映<ref>{{cite web|url=http://comic.qq.com/a/20161210/015555.htm|title=詳情公布！《妄想學生會》劇場版將於2017年7月21日上映|publisher=騰訊|author=茶茶|accessdate=2016-12-11|date=2016-12-10}}</ref>。

== 作品摘要 ==
私立櫻才學園，原本是傳統的女子高中，但因[[少子化|少子化]]問題影響，不得不改制為[[男女合校|男女合校]]。作為改制後第一年入學的28名男性新生之一的'''津田隆利'''，偶然之下被拉入了學生會。在外人看來，學生會是美女如雲的天堂。但是津田身邊的三位學生會女幹部：會長'''天草篠'''、書記'''七条天空'''和會計'''萩村鈴'''性格都非常獨特，尤其是會長和書記常常在正經事上突然[[開黃腔|開黃腔]]，甚至還無意間教壞了其他清純的女學生。於是津田開始了他每天在學生會的[[吐槽|吐槽]]生活……

== 舞台背景 ==
; {{nihongo|櫻才學園|桜才学園（おうさいがくえん）}}
: 本作主要舞台。傳統私立高中。於1958年成立。原本是傳統的女子高中學校，但是近年來由於少子化問題的影響，而決定在創校50周年（2008年，即男主角津田隆利的入學年）之際改為男女合校。改制首年的學生人數為男28人，女524人。男女制服均為西裝夾克。
: 校規非常嚴格，許多行為都被禁止（如校內[[戀愛|戀愛]]、[[染髮|染髮]]、在校外買東西進校吃、在走廊奔跑、放學後穿[[學生運動服|學生運動服]]回家、帶[[手機|手機]]到校等），還會張貼考試成績。但在津田隆利加入學生會後，許多校規逐漸放寬，如逐漸開放攜帶手機、在適當的場合可以談戀愛等。
: 學生之間流傳著各種校園傳說，例如「在庭院樹下告白的戀愛會修成正果」，但此傳説被津田吐槽「去年爲止這里還是女子高校」。

: 各年級考試成績在該年級前20名者，才有資格加入學生會。在櫻才學園實行男女合校之後，津田隆利是第一位和目前唯一男性學生會成員。
