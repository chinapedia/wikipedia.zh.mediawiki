《'''反经'''》由[[唐|唐]]人[[赵蕤|赵蕤]]著。

== 内容 ==
该书共有六十三篇，上自「君德」、「臣行」、「王霸」，下至「是非」、「通變」、「相術」，旁及「出軍」、「練士」、「教戰」，博采[[諸子百家|諸子百家]]，結合歷代史實，針對近世弊政而發，具[[文学|文学]]、史料、镜鉴三重价值。原名《长短经》，有是非、得失、长短、优劣的意思。[[纪晓岚|纪晓岚]]编撰的《[[四库全书|四库全书]]·〈长短经〉提要》说：“此书辨析事势，其言盖出于[[纵横家|纵横家]]，故以‘长短’为名。”原書十卷，末卷《陰謀》已散佚。

== 思想 ==
该书杂糅[[儒家|儒家]]、[[道家|道家]]、[[兵家|兵家]]、[[法家|法家]]、[[杂家|杂家]]、[[阴阳家|阴阳家]]等诸家思想，论述[[霸王|霸王]]之术和[[官场|官场]]秘诀，深刻而犀利。该书历史事件从[[上古|上古]][[黄帝|黄帝]]时期到[[唐朝|唐朝]]前期，总结了中国历朝历代的政治权谋得失。<ref name="mks " >{{cite book |author= 赵蕤|title=《反经》|year=2009 |publisher= 内蒙古人民出版社|location=内蒙古 |pages=导读 |isbn=9787204036936 |language = zh-hans}}</ref>

== 评价 ==
[[唐朝|唐朝]]人称：“[[赵蕤|赵蕤]]术数，[[李白|李白]]文章”。<ref name="mks " />

[[清高宗|清高宗]][[乾隆帝|乾隆帝]]<ref name="mks " />：{{cquote|题《长短经》<br />郪县创为救弊论，爱憎殴业匠和函。<br />向时虽类纵横说，忧来原归理道谈。<br />宋刊弃自教忠堂，通变称经曰短长。<br />比及乱时思治乱，不如平日慎行王。}}

[[毛泽东|毛泽东]]：《[[资治通鉴|资治通鉴]]》是[[权谋|权谋]]，是[[阳谋|阳谋]]，《长短经》是[[阴谋|阴谋]]，是[[诡谋|诡谋]]。<ref name="mks " />

[[南懷瑾|南懷瑾]]：「泛觀[[秦漢|秦漢]]以後歷經魏、晉而[[南北朝|南北朝]]之歷史人物，慧黠者口說春秋大義而陰用《[[左傳|左傳]]》、《[[戰國策|國策]]》之權謀者，代不乏人，尤其以魏蜀吳之[[三國|三國]]局勢，最為顯著。於是初[[唐|唐]]之際，而有趙蕤著《長短經》之作，評議古今，昭示正反之旨，其於三國權謀，尤所議論。自此以後，[[宋|宋]]、[[元|元]]則誤於[[理學|理學]]之清談，以積弱為能事而已。」<ref>{{cite book|author1=[[羅貫中|羅貫中]]著、[[毛宗崗|毛宗崗]]批、[[金聖歎|金聖歎]]鑑定|title=《[[三國演義|三國演義]]的政治與謀略觀》|author2=南懷瑾|chapter=〈前介〉|publisher=老古文化事業|year=1985}}</ref>
