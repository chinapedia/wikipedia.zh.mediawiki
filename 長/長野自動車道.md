'''長野自動車道'''（{{jpn|j=長野自動車道|hg=ながのじどうしゃどう|rm=Nagano jidōshadō|f=y}}）是[[日本|日本]][[長野縣|長野縣]][[岡谷市|岡谷市]][[岡谷系統交流道|岡谷系統交流道]]至同縣[[千曲市|千曲市]][[更埴系統交流道|更埴系統交流道]]的[[日本高速公路|高速公路]]，略稱'''長野道'''（{{lang|ja|長野道／ながのどう}}，{{lang|en|NAGANO EXPWY}}）。

高速公路路線編號與中央自動車道岡谷系統交流道 - [[小牧系統交流道|小牧系統交流道]]間的路段共同編為「'''E19'''」<ref>[http://www.mlit.go.jp/road/sign/numbering/files/numbering_tsuchi002.pdf 「高速道路ナンバリングの導入について」（平成29年２月14日付け国道企第55号）]、国土交通省道路局長通知、2017年2月14日</ref>。

== 概要 ==
為縱貫長野縣中央的[[高速道路|高速道路]]，連絡長野縣南北，與[[東海地方|東海地方]][[中央自動車道|中央自動車道]]沿線和[[北陸地方|北陸地方]]連結。

正式法定路線名稱為[[中央自動車道|中央自動車道長野線]]（中央自動車道長野線為[[東京都|東京都]][[杉並區|杉並區]]至[[長野市|長野市]]間的路線，東京都杉並區-[[大月市|大月市]]間為[[中央自動車道|中央自動車道富士吉田線]]，東京都杉並區-岡谷市間為[[中央自動車道|中央自動車道西宮線]]，千曲市-長野市間與[[關越自動車道|關越自動車道上越線]]重複。重複區間：中央自動車道；[[上信越自動車道|上信越自動車道]]開通後，長野自動車道為中央自動車道長野線殘餘的區間）。

該自動車道由[[日本道路公團|日本道路公團]]建設、管理，自2005年10月1日公團民營化起[[安曇野IC|安曇野交流道]]以北由[[東日本高速道路|東日本高速道路]]，安曇野交流道以南（含安曇野交流道）由[[中日本高速道路|中日本高速道路]]管理。

== 歷史 ==
* 1986年3月25日 : 岡谷JCT-岡谷IC間的中央自動車道通車。
* 1988年3月5日 : 岡谷IC-松本IC間通車。岡谷JCT-岡谷IC間編入長野自動車道。
* 1988年8月3日 : 松本IC-豐科IC間通車。
* 1993年3月25日 : 豐科IC-更埴JCT間通車，'''全線通車'''，連接上信越自動車道。
* 2006年10月1日 : 姨捨SMART IC開始啟用。
