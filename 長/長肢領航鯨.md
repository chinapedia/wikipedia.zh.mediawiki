'''長肢領航鯨'''（[[學名|學名]]：''Globicephala melas''），又名'''長鰭領航鯨'''、'''大西洋領航鯨'''、'''黑圓頭鯨'''或'''巨頭鯨'''，<ref name=MSW3>{{MSW3 Cetacea|id=14300052}}</ref>是兩種[[圓頭鯨屬|圓頭鯨屬]]的其中一種。牠們屬於[[海豚科|海豚科]]，但其行為較為接近[[鯨魚|鯨魚]]。

== 特徵 ==
長肢領航鯨像[[殺人鯨|殺人鯨]]般其實是[[海豚|海豚]]的一種。牠們的前額圓，呈黑色或深灰色，喉嚨及[[腹部|腹部]]呈灰色或白色，有時在背鰭及[[眼睛|眼睛]]後也有灰色或白色斑紋。背鰭呈鐮刀狀。腳蹼很長，佔了體長的15-20%。雌鯨長約3.7米時及6-7歲大就達至性成熟；雄鯨要長約4.6米及12歲大才達至性成熟。成年重1.8-3.5噸。

== 行為 ==
[[File:LF_Pilot_Whale_Goban_Spur.jpg|thumb]]
長肢領航鯨社交性甚高，可以組成達100頭的群落。牠們有時會與[[寬吻海豚|寬吻海豚]]及[[灰海豚|灰海豚]]社交。成年一天需要約50公斤食物，包括[[頭足類|頭足類]]及[[魚類|魚類]]。牠們一般可以潛水達數分鐘，覓食時則可以潛超過10分鐘。牠們能夠潛入水深600米的地方，而一般則只會到30-60米的地方。

[[妊娠期|妊娠期]]約12-15個月，要3-5年才會生一胎。幼生出生時長1.8米及重102公斤。幼生需要長達22個月的照顧。雖然全年也可以生育，但大部份幼生都是在[[夏天|夏天]]出生。雄性會為了與雌性[[交配|交配]]而互相競爭打鬥。雌性老達35歲仍然可以生育，51歲仍可授乳。由此可見雌性可以照顧幼生一直到10歲的青春期。

長肢領航鯨可以用3-18000[[赫茲|赫茲]]的[[聲音|聲音]]來溝通弓進行[[迴聲定位|迴聲定位]]。牠們可以在1分鐘內發出14-40次這些聲音。

長肢領航鯨非常活躍，經常會用尾葉拍打水面或跳出水面窺探。幼生的也會跳出水面，但成年的則很少。牠們也會走到海灘上，加上其強烈家庭觀念，故有時會一起走上海灘。

== 屠殺 ==
{{seealso|法羅群島捕鯨活動}}
[[File:Killed_pilot_wales_in_hvalba,_faroe_islands.JPG|thumb]]的狩獵行為，已成為法羅群島居民生活文化的一部分。]]
長久以來，[[北歐|北歐]][[法羅群島|法羅群島]]的居民每年都會進行一次大規模的獵殺長肢領航鯨，並視之為他們的傳統「慶典」<ref name=sinabook>{{cite web |url=http://forum.book.sina.com.cn/thread-1944209-1-1.html |title=法罗人与领航鲸的“世仇” |date=2009-04-22 |accessdate=2009-08-03 |language=zh-hans }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。法羅島民會駕駛汽艇，將鯨豚驅趕至海灣處，過程冗長，目地就是將體力耗竭，長肢領航鯨群驅至淺灘處，血腥屠殺就此展開。長肢領航鯨遭開腸剖肚，頭遭重擊，不多久，血染整個海灣，到處散佈著鯨豚支離破碎屍身與奄奄一息的垂死長肢領航鯨<ref name=sinabook />。由於每次行動都會屠殺大量鯨魚，血水把海灣都染紅，他們的行動早已惹來環保團體及愛護動物團體的關注<ref name=sinabook />。2010年7月19日，一名事前混入當地人之中生活的[[海洋守護者協會|海洋守護者協會]]卧底拍攝了一場在Klaksvik城鎮進行的屠鯨行動，並把過程上載上網，立時引起全世界的關注以及人道問題的爭議<ref>{{cite web |url=http://www.hkwildlife.net/viewthread.php?tid=59749 |title=丹麥鯨豚大屠殺現場 |publisher=香港自然生態論壇 |date=2010-08-02 |accessdate=2010-08-03 |language=zh-hant}}</ref>。多個團體在網上發行簽名運動，意圖向丹麥政府施壓，令這種屠鯨的活動被禁止。
