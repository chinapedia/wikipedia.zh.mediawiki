'''長門市'''（{{jpn|j=長門市|hg=ながとし|rm=Nagato shi|f=y}}）是位於日本[[山口縣|山口縣]]北部臨靠[[日本海|日本海]]的[[城市|城市]]，為[[山陰地方|山陰地方]]的最西端，也因為JR[[美禰線|美禰線]]在此與[[山陰本線|山陰本線]]交會，成為[[山口县|山口县]]北部的主要交通要所。

轄區南部位於[[中國山地|中國山地]]，北部沿海多為沿岸，有深川灣、仙崎灣、{{Tsl|ja|油谷湾|油谷灣}}多個適合作為港口的港灣，也因此漁業是本地的主要產業之一，位於{{Tsl|ja|仙崎}}地區的仙崎港也是山口縣北岸規模較大的漁港。

== 历史 ==

現在的長門市轄區在過去的令制國時代屬於[[長門國|長門國]][[大津郡|大津郡]]，其範圍也大致相同；15世紀[[大內氏|大內氏]]統治此地時曾在此建有{{Tsl|ja|亀山城_(長門国)|龜山城_(長門國)|龜山城}}，江戶時代則屬於[[長州藩|長州藩]]，並且曾經因做為[[捕鯨|捕鯨]]基地而繁榮。

1889年實施[[町村制|町村制]]後，現在的長門市轄區分屬{{Tsl|ja|三隅町_(山口県)|三隅町_(山口縣)|三隅村}}、{{Tsl|ja|仙崎|仙崎|仙崎通村}}、{{Tsl|ja|深川町 (山口県)|深川町 (山口縣)|深川村}}、{{Tsl|ja|俵山村}}、{{tsl|ja|菱海村}}、{{Tsl|ja|日置町 (山口県)|日置町 (山口縣)|日置村}}、{{Tsl|ja|宇津賀村}}、{{Tsl|ja|向津具村}}，共8個行政區劃。

位於{{Tsl|ja|仙崎}}的港口在[[第二次世界大戰|第二次世界大戰]]結束後，曾作為海外國民[[引揚歸國|引揚歸國]]的主要港口，當年在此自海外返回的日本人多達41萬，同時原先滯留在日本的外籍人士，從返回自己國家的人也多達34萬。<ref>{{Cite web|url=http://honkawa2.sakura.ne.jp/5226.html| title=図録▽アジア太平洋戦争における海外からの引き揚げ| work=社会実情データ図録| date=2016-09-24| language=ja| accessdate=2019-01-29}}</ref>

1954年，原屬仙崎通村、深川村、俵山村的區域[[市町村合併|合併]]為'''長門市'''，其他區域則整併為三隅町、{{Tsl|ja|油谷町}}和{{tsl|ja|日置町 (山口県)|日置町 (山口縣)|日置町}}3町；直到2005年，這四個行政區劃再次整併為現在的'''長門市'''。<ref>{{cite web| language =ja| publisher =長門市政府| title =市的概要| url =http://www.city.nagato.yamaguchi.jp/plofile/plofile.html| author =| date =| accessdate =2010-08-17| deadurl =yes| archiveurl =https://web.archive.org/web/20091215043447/http://www.city.nagato.yamaguchi.jp/plofile/plofile.html| archivedate =2009-12-15}}</ref>

也因為本地是日本第90、96-98任[[內閣總理大臣|內閣總理大臣]][[安倍晉三|安倍晉三]]的祖籍地點，2016年安倍晉三曾在此地的[[長門湯本溫泉|長門湯本溫泉]]招待[[俄羅斯|俄羅斯]]總統[[普京|普京]]。<ref>{{cite news|url=http://hk.on.cc/int/bkn/cnt/news/20161215/bknint-20161215161701942-1215_17011_001.html|title=普京專機抵達日本　今晚與安倍浸溫泉|date=2016年12月15日|newspaper=''[[東方日報_(香港)|東方日報 (香港)]]''}}</ref>

== 交通 ==

[[西日本旅客鐵道|西日本旅客鐵道]][[山陰本線|山陰本線]]以東西向大致沿著海岸像通過轄內，在主要市區[[長門市車站|長門市車站]]有山陰本縣的支線仙崎線往北進入{{Tsl|ja|仙崎}}地區，同時往南也有[[美禰線|美禰線]]可越過中國山地，通往山口縣南部的[[山陽小野田市|山陽小野田市]]。

高速公路則有[[山陰自動車道|山陰自動車道]]預計以東西向穿過轄內，但目前只完成境內東部三隅地區往東至[[萩市|萩市]]的路段，此段路線目前被稱作{{tsl|ja|萩・三隅道路}}。

轄內的客運業者則有屬於{{Tsl|ja|近鉄グループ|近鐵集團}}以山口縣東部和北部為主要營業範圍的{{Tsl|ja|防長交通}}、前身為{{Tsl|ja|山陽電気軌道|山陽電氣軌道}}以下關市為主要據點的{{Tsl|ja|サンデン交通|山電交通}}、山電交通的子公司以山口縣西部和北部為主要營業範圍的{{Tsl|ja|ブルーライン交通|藍線交通}}三家業者。

== 觀光資源 ==

[[File:元乃隅稲成神社_Motonosumi-Inari-jinja_20090504.jpg|thumb]]
[[File:元乃隅稲荷神社_(33131861841).jpg|thumb]]
[[File:長門千畳敷.jpg|thumb]]

位於主要市區北側的[[青海島|青海島]]及西部向津具半島的沿海區域被劃為[[北長門海岸國定公園|北長門海岸國定公園]]的範圍，這段海岸為沿岸地形，這片區域近年發展為觀光景點，知名度較高的景點包括[[千疊敷_(山口縣)|千疊敷]]和[[元乃隅神社|元乃隅神社]]，在仙崎港有業者經營海上觀光航線可遊覽青海島沿岸的景色；而青海島東部的通地區過去是以捕鯨而發展形成的聚落，其中建於1692年的{{Tsl|ja|鯨墓}}已被列為國家史蹟。

轄內有[[長門湯本溫泉|湯本溫泉]]、{{Tsl|ja|俵山温泉|俵山溫泉}}、{{Tsl|ja|湯免温泉|湯免溫泉}}、{{Tsl|ja|黄波戸温泉|黃波戶溫泉}}、{{Tsl|ja|油谷湾温泉|油谷灣溫泉}}多個溫泉區，其中湯本溫泉是山口縣北部規模最大的溫泉區。

此外當地傳說[[中國|中國]][[唐朝|唐朝]]的[[楊貴妃|楊貴妃]]在[[安史之亂|安史之亂]]後逃到了日本此地，因此在向津具半島南側有個「楊貴妃之村」，內有傳說中的楊貴妃之墓。

20世紀初的童謠詩人[[金子美鈴|金子美鈴]]出生於轄內的仙崎地區，其過去出生住所現在也成為「金子美鈴紀念館」。

== 教育 ==

過去原本有{{Tsl|ja|山口県立大津高等学校|山口縣立大津高等學校}}、{{Tsl|ja|山口県立水産高等学校|山口縣立水產高等學校}}、{{Tsl|ja|山口県立日置農業高等学校|山口縣立日置農業高等學校}}三所高中，在2011年被整併為{{Tsl|ja|山口県立大津緑洋高等学校|山口縣立大津綠洋高等學校}}，同時也保留了過去三所高中各自的課程，設有普通科、屬於農業的生物生產科、生活科學科、屬於水產的海洋技術科、海洋科學科。
