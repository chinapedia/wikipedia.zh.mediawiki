== 概要 ==
描述私立高中「碧陽學園」的[[學生會|學生會]]成員們在學生會辦公室閒聊的短篇小說系列。每章的標題基本上都是「{{nihongo|○○的學生會|○○する生徒会}}」。

本作以[[後設小說|後設小說]]的形式寫成，有本系列是「為了讓全校學生知道學生會有多優秀，會長『櫻野玖璃夢』命令副會長『杉崎鍵』動筆將學生會的作息寫成小說，再和富士見書房合作出版」的設定。

以作者自比為「四格式小說」的登場人物的喜劇性聊天內容（裝傻和吐槽）和隨處可見對其他作品的[[戲仿|戲仿]]，為本系列最大的特徵。擔任責任編輯的中村昭子形容本作是「許多的要素湊在一起，能用各種方式享受的作品」；作者則說是逆轉「被捲進大事情後，想取回日常生活」這種常見的橋段，而去書寫有讓人想取回的價值的日常生活。

== 故事簡介 ==
私立碧陽學園的學生會成員，是用和其他學校不同的特殊方法選出，就是很單純的人氣投票。結果選出櫻野玖璃夢為首的眾多美少女們，可是學生會裡有一位是靠著用功讀書，在期末考中拿到第一名、以優秀學生名額加入學生會的男學生杉崎鍵。故事內容主要是描寫學生會日常運作的情形。

=== 碧陽學園學生會 ===
; {{nihongo|杉崎鍵|杉崎 鍵（すぎさき けん）||聲優：[[近藤隆|近藤隆]]／同左}}
: 學生會副會長，2年B班。血型A型。本作的男主角，也是在故事裡寫這本小說的人。在期末考中拿到第一名，靠著如果成績優秀的人願意就能加入學生會的「優良規定」而加入。知弦因其名而稱他為「Key」。
: 是個[[Galgame|Galgame]]、[[H-game|H-game]]迷，雖然表面上很好色，但在玩H-Game的時候[[18禁|18禁]]部份都是快速跳過，是個比表面純情的少年，能不忌諱地公然說自己之外全都是美少女的學生會是「我的[[後宮|後宮]]」（當然其他成員是沒人會承認的），但也因此而發誓要讓所有人都幸福。另一方面，為了能讓學生會成員聊天的時間多一點而抱著學生會的公務在放學後獨自完成。有時會說些[[性騷擾|性騷擾]]的話而被粗魯的對待（主要是被深夏），但玖璃夢說：「在這所學校裡沒有一個人真的討厭他。」
: 雖然時常喊後宮，不過實際上是個對愛情十分遲鈍的人。
: 小時候父母離婚後父親再婚、因此多了沒有血緣關係的妹妹林檎，與飛鳥交往時因無法兼顧親情與愛情而爆出腳踏兩條船的傳言。
: 於本傳小說十代中被學生會四人告白。
: 在動畫第2作中，講述杉崎鍵於一年級時期的未加入學生會前的經歷。
: 喜歡看韓劇「[[冬季戀歌|冬季戀歌]]」，其手提包裡甚至還有此連續劇的光碟，也曾說過「把項鍊塞進雪球裡丟給要告白的人（於冬季戀歌中知名的劇情）」之類的話。
: 放學後至上學前都在打工，不過似乎是做[[臨時演員|臨時演員]]兼跑龍套的角色。其｢'''打雜用[[手提包|手提包]]'''｣被[[學生會|學生會]]成員叫做｢'''神秘的[[公事包|公事包]]'''｣，是被列為｢'''碧陽學苑第27個神祕之物'''｣，內容包羅萬象：[[棕刷|棕刷]]（下面黏著與搞笑藝人的合照）、舊式美軍制式[[手槍|手槍]]（不具殺傷力，是'''手槍型[[拉砲|拉砲]]'''，擊發後為[[彩帶|彩帶]]與[[紙碎|紙碎]]）、驅魔鹽（附有產地證明）、[[十字架|十字架]]、[[銀子彈|銀子彈]]（以上兩者用途不明）、掌上型彩色電玩（遊戲為校園後宫H-Game）、[[狗牌|狗牌]]、染血的狗牌、[[發信機|發信機]]（用途不明）、女中學生運動褲（試作品非偷來的）、木釘（附掛一串[[蒜頭|蒜頭]]）、[[鞭子|鞭子]]、[[防毒面具|防毒面具]]、{{tsl|ja|乾パン|罐裝乾麵包}}、粉餅化妝盒、[[口紅|口紅]]（以上兩者存在不明）、[[煙斗|煙斗]]和三本書（分別為'''成為受歡迎的男人十大秘訣'''、'''不會被周圍的人無視的方法Best5'''和'''與任性孩子相處的正確方法'''）等等。被學生會成員誤認為鍵的打工是當[[傭兵|傭兵]]……。

; {{nihongo|櫻野玖璃夢|桜野 くりむ（さくらの くりむ）||聲優：[[本多真梨子|本多真梨子]]／同左}}
: 學生會長，3年B班，4月2日生，身高小说版140公分、动画版142公分、漫画版145公分，血型A型，患有[[花粉症|花粉症]]。
: 知弦因其名「{{lang|ja|くりむ}}」聯想到「{{nihongo|真紅|クリムゾン}}」以及其粉紅色的頭髮而稱她為「{{nihongo|小紅|アカちゃん}}{{notetag|「{{nihongo|小紅|アカちゃん}}」跟「{{nihongo|嬰兒|あかちゃん}}」的日文發音是一樣的。}}」。
: 兩眼視力均為1.2，一般水準。

: 其父親向鍵表明自己黑道的身份，雖然對鍵表示欣賞，但嚴禁鍵對自己的女兒出手。被鍵暗稱為「黑道父女」。
: 是新一屆的學生會成員中第二個到任的成員。

; {{nihongo|日守東子|日守 東子（ひのもり とうこ）}}
: 於一年級時期甚少與人交流，身份本貌一切都是謎的原校生，現為碧陽學園二年級學生。
: 在結業二次會時，碰巧被少數人親眼見過本貌，證實是位猶如夢幻的美少女，繼而成為校內的人氣少女。
: 憑著校的人氣及知名度，人氣投票得到第二名而成為新一屆學生會書記。
: 喜歡遺傳自外婆的銀髮，但因為總會吸引男性的目光而引起各種事端，所以一直戴著大口罩和黑色假髮上學。後來被鍵解開心結，正式就任學生會書記一職。
: 學識與玖璃夢相近甚至更讓人遺憾，喜好開論壇、遊戲和BL，不知為何與轉學後真冬有聯絡，主要話題是BL(例如「鍵x守」)。
: 是新一屆的學生會成員中第三個到任的成員。

; {{nihongo|火神北斗|火神 北斗（かがみ ほくと）}}
: 擁有棕色的長髮、身材嬌小的女孩，新入讀的碧陽學園一年級生。
: 為人活潑好動帶點率直，臉上時常是天真的笑容。話語上總是有著無形的主導權，有點小惡魔的性格。
: 其率直活潑可愛的樣子，在人氣投票得到第六名而成為新一屆學生會會計。
: 作為情婦的女兒，眼見寂寞的母親而對父親非常憎惡，為免母親擔心而扮得活潑好動。
: 在如同親姐姐一樣的飛鳥口中得知鍵的事情，決心要破壞鍵的後宮，甚至讓鍵見其花心的父親以此來打擊鍵。後來鍵在新學生會成員支持下，重新了解自己的想法。
: 最後了解到鍵那即使被用裁紙刀所指流出鮮血，也毫不動搖的想法，再加上大家的心意，終於被打動了，正式就任學生會會計一職。
: 其後覺醒了病嬌屬性，決心成為鍵「唯一」的戀人，監視並消滅所有接近鍵的害蟲（武器為圓珠筆和死亡點數筆記...等）。從結果而言，作為破壞鍵後宮的最終Boss的身份仍然無變。
: 是新一屆的學生會成員中最後一個到任的成員。

== 動畫 ==
改編電視動畫《學生會的一存 碧陽學園學生會議事錄》（{{lang|ja|生徒会の一存 碧陽学園生徒会議事録}}）於2009年10月至12月間播放。

2011年宣佈新動畫化計畫開始，第2作動畫轉由AIC擔任製作，暫定名稱{{nihongo|「學生會的一存 新動畫」|生徒会の一存 新アニメ（仮）}}，直到開播當天才公布正式名稱{{nihongo|《學生會的一存 Lv.2》|生徒会の一存 Lv.2}}{{noteTag|1={{cite web|url=https://twitter.com/seitokai_anime/status/257132656096059393 |script-title=ja:エンディング突入と同時に正式タイトル発表！「生徒会の ... |publisher=[[Twitter|Twitter]] |author=@seitokai_anime||accessdate=2012-10-14|date=2012-10-13 |language=ja }}}}，與前作無接續關係；並比照[[蝦掰天文社|蝦掰天文社]]模式先行網路播出。
