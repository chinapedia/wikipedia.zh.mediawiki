'''碧翠丝·安妮卡·"碧"·米勒'''（{{lang-en|Beatrice Annika "Bea" Miller}}，{{bd|1999年|2月7日}}）是一名美国[[唱作人|唱作人]]和女演员。她参加了[[:en:The_X_Factor_USA_(season_2)|美版《X音素》第2季]]，成绩排名第9位。

== 经历 ==
碧翠丝在美国[[新泽西州|新泽西州]][[槭木镇|槭木镇]]出生并长大。她出自同性结合家庭，有2个妈妈，而其父亲是某姓名不详的精子捐献者。她有一对比她年龄小一点的双胞胎越南裔妹妹艾斯瑟（Esther）和乔治亚（Georgia），是由双亲收养的。她的家庭不算和睦，双亲经常争吵，最终以离异收场。双亲在离异后的2年内一直没有分居，而且还不时争吵。在这段难熬的家庭时光里，她承担起了呵护2个妹妹的责任。她的各科学习成绩都是A等，但由于家庭特殊，小学时受过同学的欺负。双亲都对她的事业发展非常支持。<ref>{{Cite web |url = http://nypost.com/2012/11/08/x-factor-maplewood-teen-grieves-separation-of-moms/ |title = "X Factor" Maplewood teen grieves separation of moms |author = Sean Daly |date = 2012年11月8日 |publisher = [[纽约邮报|纽约邮报]] |language = en |accessdate = 2014年6月26日 }}</ref><ref>{{Cite web |url = http://hollywoodlife.com/2012/11/09/x-factor-beatrice-miller-parents-divorced-two-mothers-split/ |title = "X Factor" Teen Beatrice Miller Still Grieving Parents' Separation |author = Russ Weakland  |date = 2012年11月9日 |publisher = hollywoodlife.com |language = en |accessdate = 2014年6月26日 }}</ref>

“碧翠丝”（'''[[:en:Beatrice_(given_name)|Beatrice]]''',{{IPAc-en|ˈ|b|iː|.|ə|t|r|ɪ|s}},{{IPA-it|be.aˈtriːtʃe|lang}}）这个名字也译作“碧雅翠丝”，且其对应法语名"Béatrice"和西班牙语名"Beatriz"也都是把"Bea"拆开成"Be-a"念的。这个名字可能源于拉丁文"[[:en:Beatrix|Beatrix]]"，为"Viator"所对应的阴性名字，有“旅行者”之意；也可能源于拉丁文"beatus"，意为“神圣的/受祝福的”。

=== 2012：参加美版《X音素》 ===

在节目中担任米勒导师的是[[布兰妮·斯皮尔斯|布兰妮·斯皮尔斯]]。由于节目安排不合理，从得知该周主题到导师为选手选出歌曲，时间往往只有1天，有时甚至只有短短几个小时。碧·米勒和[[:en:Jennel_Garcia|Jennel Garcia]]等其他一些歌手曾抱怨说自己并不喜欢导师为其所选的歌曲。碧·米勒在演唱完"[[:en:Time_After_Time_(Cyndi_Lauper_song)|Time After Time]]"后，曾表示虽然自己也喜欢听一些80代的歌曲，但如果真要让自己在比赛中演唱这些慢节奏的老歌又是另一回事了，这些歌曲未必适合拿来参赛。<ref>{{Cite web |url = http://www.hollywoodreporter.com/live-feed/x-factor-beatrice-miller-hated-song-391156 |title = 'X Factor' Finalist Beatrice Miller 'Hated' Britney Spears' Song Choice |author =  Sophie Schillaci, Michele Amabile Angermiller |date = 2012年11月15日 |publisher = hollywoodreporter.com |language =en |accessdate = 2014年6月26日 }}</ref>

=== 2013-现今：签约唱片并筹办专辑 ===
2013年4月11日，她与[[:en:Syco_Music|合成音乐]]和[[:en:Hollywood_Records|好莱坞唱片]]签约，这也是2家唱片公司首度合作的范例。<ref>{{cite web|author=1 years |url=http://www.hollywoodreporter.com/earshot/x-factor-alum-beatrice-miller-437933 |title='X Factor' Alum Beatrice Miller Inks Record Deal |publisher=Hollywood Reporter |date=2013-11-04 |accessdate=2014-04-30}}</ref>米勒目前忙于制作将于2014年发售的首张专辑。她正与Autumn Rowe密切合作中。<ref>{{cite web|url=http://twicsy.com/i/U58MBe |title=BeatriceMBR : Bea e Autumn Rowe |publisher=Twicsy |date=2013年12月10日 |accessdate=2014年6月26日 }}</ref>在美版《X音素》刚落幕后不久，米勒将自己的名字缩为碧·米勒(Bea Miller)。<ref>{{cite web|author=Facebook Twitter Instagram YouTube Bea Miller BeaMiller.com |url=http://www.hollywoodrecords.com/artists/bea-miller/ |title=Bea Miller |publisher=Hollywood Records |date= |accessdate=2014-04-30}}</ref>她的首张专辑连同主打歌曲原先预计将于春季发售，约在3-5月间的样子。在专辑制作的同时，她也在[[Instagram|Instagram]]上分享了制作过程中的一些照片，展示了她所用的设备。<ref>{{cite web|url=http://instagram.com/p/jMtxQ7MaYj/ |title=Instagram |publisher=Instagram |date= |accessdate=2014-04-30}}</ref>她曾在[[YouTube|YouTube]]提前上传了尚未正式发售的新歌"[[:en:Rich_Kids_(Bea_Miller_song)|Rich Kids]]"。仅在1周内，就达到了1万5千的观看量。米勒打算在今年夏季推出迷你专辑，并在晚些时候推出完整版专辑。<ref>{{cite web |author=Username * |url=https://www.fanlala.com/news/bea-miller-kicks-month-surprises-rich-kids-teaser |title=Bea Miller Kicks Off a Month of Surprises with "Rich Kids" Teaser |publisher=Fanlala.com |date= |accessdate=2014-04-30 |deadurl=yes Fire]]"的新歌片段<ref>{{cite web|url=http://instagram.com/p/mGeZpOMaf8/ |title=Instagram |publisher=Instagram |date= |accessdate=2014-04-30}}</ref>，这首"Enemy Fire"于4月在其官方[[Vevo|Vevo]]频道首播。[[:en:busbee|busbee]]、[[:en:Jarrad_Rogers|Jarrad Rogers]]及[[:en:Mike_Del_Rio|Mike Del Rio]]等艺人参与了她的专辑的制作。2014年4月22日，她发售了首张迷你专辑《年轻的血》("[[:en:Young_Blood_(EP)|Young |accessdate=2014-04-30}}</ref><ref>{{cite web|url=https://www.facebook.com/BeaMiller/photos/a.359659354050554.108511.249358218414002/840456569304161/?type=1&theater |title=Bea Miller - Tijdlijnfoto's
