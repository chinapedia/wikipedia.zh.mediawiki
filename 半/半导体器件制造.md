{{半導體製程}}

[[File:Clean_room.jpg|thumb]]]]

'''半導體製程'''是被用于制造[[芯片|芯片]]，一种日常使用的[[电气|电气]]和[[电子|电子]]器件中[[集成电路|集成电路]]的处理工艺。它是一系列照相和化学处理步骤，在其中电子电路逐渐形成在使用纯[[半导体|半导体]]材料制作的[[晶片|晶片]]上。

[[硅|硅]]是今天最常用的半导体材料，其他还有各种[[化合物半导体|复合半导体]]材料。

从一开始晶圓加工，到芯片封装測試，直到发货，通常需要6到8周，并且是在[[晶圓廠|晶圓廠]]內完成。

== 晶圓 ==
{{Main|晶圓}}

典型的[[晶片|晶片]]是用极度纯净的[[硅|硅]]以[[柴可拉斯基法|柴可拉斯基法]]、[[泡生法|泡生法]]等方式长成直径{{convert|12|inch}}的[[单晶|单晶]]圆柱{{tsl|en|Ingot|锭}}（[[梨形人造宝石|梨形人造宝石]]）。这些硅碇被切成晶片大约0.75毫米厚并[[抛光|抛光]]為非常平整的表面。

一旦晶圓准备好之后，很多工艺步骤对于生产需要的半导体集成电路是必要的。总之，这些步骤可分成四组：
* 前端工艺
* 后端工艺
* 测试
* 封装

== 工艺 ==
在半導體製程中，不同的生产工序可归为如下四类：沉积、清除、制作布线图案、以及电学属性的调整。

== 有害材料标志 ==
许多有毒材料在制造过程中被使用。这些包括：

* 有毒元素[[掺杂物|掺杂物]]比如[[砷|砷]]、[[硼|硼]]、[[锑|锑]]和[[磷|磷]]
* 有毒化合物比如[[砷化三氢|砷化三氢]]、[[磷化氢|磷化氢]]和[[硅烷|硅烷]]
* 易反应液体、例如[[过氧化氢|过氧化氢]]、发烟[[硝酸|硝酸]]、[[硫酸|硫酸]]以及[[氢氟酸|氢氟酸]]

工人直接暴露在这些有毒物质下是致命的。通常[[IC|IC]]制造业高度自动化能帮助降低暴露于这一类物品的风险。

== 历史 ==
当线宽远高于10[[微米|微米]]时，纯净度还不像今天的器件生产中那样至关紧要。旦随着器件变得越来越集成，[[無塵室|無塵室]]也变得越来越干净。今天，工廠內是加壓过滤空气，来去除哪怕那些可能留在晶片上并形成[[缺陷|缺陷]]的最小的粒子。半导体制造產線裡的工人被要求穿著[[無塵衣|無塵衣]]来保护器件不被人类污染。

在利润增长的推动下，在1960年代半导体器件生产遍及[[德克萨斯州|德克萨斯州]]和[[加州|加州]]乃至全世界，比如[[爱尔兰共和国|爱尔兰]]、[[以色列|以色列]]、[[日本|日本]]、[[台灣|台灣]]、[[韩国|韩国]]和[[新加坡|新加坡]]，且在今天已是一个全球商业。

半导体生产商的领袖大都在全世界拥有產線。[[英特尔|英特尔]]，世界最大的生产商，以及在美其他顶级生产商包括[[台积电|台积电]]（台湾）、[[三星電子|三星]]（韩国）、[[德州仪器|德州仪器]]（美国）、[[AMD|超微半导体]]（美国）、[[聯电|聯电]]（台灣）、[[东芝|东芝]]（日本）、[[NEC电子|NEC电子]]（日本）、[[意法半导体|意法半导体]]（欧洲）、[[英飞凌|英飞凌]]（欧洲）、[[瑞萨|瑞萨]]（日本）、[[索尼|索尼]]（日本），以及[[恩智浦半导体|恩智浦半导体]] （欧洲）在欧洲和亚洲都有自己的设备。

在2006年，在美国有大约5000家半导体和电子零件生产商，营业额达1650亿美元<ref>[http://www.barnesreports.com Barnes报告]“2006美国工业和市场展望”</ref>。
