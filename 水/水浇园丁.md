《'''水浇园丁'''》（{{lang-fr|''L'Arroseur Arrosé''}}），又名《'''会喷水的洒水车'''》（{{lang-en|''The Sprinkler Sprinkled''}}）、《'''淋成落汤鸡'''》（{{lang-en|''The Waterer Watered''}}），是法国[[卢米埃尔兄弟|路易斯·卢米埃尔]]于[[1895年电影列表|1895年]]拍摄的一部[[黑白|黑白]][[无声电影|无声]][[喜剧片|喜剧片]]，由弗朗索瓦·克莱尔（{{lang|fr|François Clerc}}）和伯努瓦·杜瓦尔（{{lang|fr|Benoît Duval}}）主演。影片于1895年6月10日首映。

影片有着成为最早喜剧片范例的区别，并第一次使用电影描绘虚构故事。电影最初被称为Le Jardinier（意为“园丁”）或“Le Jardinier et le petit espiègle”。

== 剧情 ==
摄于1895年春的[[里昂|里昂]]，电影描绘了一个简单的恶作剧：一名男孩踩住园丁用来浇花的水管，阻止水流动。当园丁向上倾斜水管作检查时，男孩放开水管，让水喷到他脸上。园丁惊呆了，他的帽子被水冲掉，但他很快就明白了。伴随随之而来的追逐，画面不断开和关（摄影机从未从原本的位置上移动），直至园丁揪住男孩揍了他一顿。整部影片只持续49秒，但这个简单的打闹可能是后来所有喜剧电影的开山鼻祖<ref name=autogenerated1>{{cite web|title=L'Arroseur Arrose - Film (Movie) Plot and Review|url=http://www.filmreference.com/Films-Am-Aw/L-Arroseur-Arrose.html|website=Film Reference|accessdate=2015-08-30}}</ref>。

== 演员 ==
路易·卢米埃尔用他自己的园丁弗朗索瓦·克莱尔来描绘园丁。至于那位调皮的孩子，卢米埃尔则使用卢米埃尔家工厂的一名年轻的木匠学徒，多方面相信他叫丹尼尔·杜瓦尔或伯努瓦·杜瓦尔<ref>{{cite web|title=Chronomedia: 1895|url=http://www.terramedia.co.uk/Chronomedia/years/1895.htm|website=Chronomedia|publisher=Terra Media|accessdate=2015-08-30}}</ref>。但莱昂·特罗托巴斯（{{lang|fr|Léon Trotobas}}）似乎是第一位在[[拉西奥塔|拉西奥塔]]扮演角色的男孩<ref>{{cite web|title=Le Cinematographe|url=http://www.museeciotaden.org/cinema/cinematographe.htm|website=LE CINEMATOGRAPHE LUMIERE|accessdate=2015-08-30|language=fr}}</ref>。

== 制作 ==
在[[电影史|电影史]]的早期，电影院供[[托马斯·爱迪生|托马斯·爱迪生]]等先驱享用，卢米埃尔兄弟对纯粹的新颖发明感兴趣，而当时的电影大多数是{{le|福莱德·奥特的喷嚏|Fred Ott's Sneeze|打喷嚏}}或[[火车进站|火车进站]]之类的平凡事件的短录像片。总是追求创新的卢米埃尔兄弟凭借《水浇园丁》迈出了前往叙事电影的第一步。鉴于现有电影直到现在的[[纪录片|纪录片]]性质，一部脚本化的喜剧电影将向观众表示出始料未及的场景，提高戏剧的惊喜价值<ref name=autogenerated1 />。

电影由集电影放映机和开发机所有功能于一身的{{le|电影机|Cinématographe}}拍摄，和卢米埃尔早起的所有电影一样，影片以[[35毫米膠片|35毫米]]格式制作，[[宽高比|宽高比]]1.33:1<ref>{{cite web |url=http://www.imdb.com/title/tt0000014/technical |title=Technical Specifications |work=Internet Movie Database |accessdate=2007-03-28}}</ref>。

== 海报 ==
电影海报有着作为史上第一张设计成宣传单部电影的海报。尽管海报自1890年以来用作宣传影院的投影，但这些早期的海报一般描述录影的质量，吹捧技术的新颖性。电影海报由马尔彻林·奥佐勒（{{lang|fr|Marcellin Auzolle}}）设计，描绘了屏幕前的观众被投影到背景屏幕的电影逗笑的场景。海报描绘出园丁被水射中的那一刻，因此它也是首张描绘电影实际场景的电影海报<ref>{{cite book|author1=Jean Jacques Meusy|title=Cinémas de France, 1894–1918: Une histoire en images|date=2009|publisher=Arcadia Éditions|location=Paris|isbn=978-2-913019-61-4|pages=201}}</ref>。

== 复制与模仿 ==
由于当时的版权法既没有施行，也没有明确定义影院这一新兴技术，影片通常是被相互竞争的制作人们重拍一部受欢迎的电影或是被分销商复制胶片拷贝自行发行的对象。透过这些做法，电影已被复制过多次，在法国和美国以不同的名字上映，卢米埃尔兄弟自己也至少做了一次重拍<ref>{{cite web |url = http://www.let.uu.nl/imr/domitor/2004/more.php?id=25_0_5_0_M|title=Early Cinema: Heyday of Copying|work=Domitor 2004 Utrecht Conference|accessdate=2007-07-25|author = Jane Gaines }}</ref>。大多数复制版鲜为人知，但其中最出名当属[[乔治·梅里爱|乔治·梅里爱]]于1896年重拍的《L'Arroseur》<ref name=autogenerated1 />。[[法国新浪潮|法国新浪潮]]导演[[弗朗索瓦·特吕弗|弗朗索瓦·特吕弗]]后来在他的1958年电影《{{le|扭计祖宗|Les Mistons}}》中也向这一桥段[[致敬|致敬]]<ref name=autogenerated1 />。

== 目前情况 ==
鉴于其年代，这部短片目前已进入[[公有领域|公有领域]]，可在互联网上自由下载。影片还被列入《电影早期地标：第一卷》（{{lang|en|''Landmarks of Early Film Volume 1''}}）等电影年鉴中<ref>{{cite web |url=http://www.imdb.com/title/tt0000014/dvd |title=DVD |work=Internet Movie Database |accessdate=2007-03-28 }}</ref>。
