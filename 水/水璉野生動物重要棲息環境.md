'''水璉野生動物重要棲息環境'''位於[[中華民國|中華民國]][[花蓮縣|花蓮縣]][[壽豐鄉|壽豐鄉]]，是依《野生動物保育法》公告之野生動物重要棲息環境<ref name="林務局">{{cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=3334&ctNode=176&mp=10 |title=水璉野生動物重要棲息環境 |accessdate=2014-10-26 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20141026103501/http://conservation.forest.gov.tw/ct.asp?xItem=3334&ctNode=176&mp=10 |archivedate=2014-10-26 }}</ref><ref>{{cite web |url=https://www.youtube.com/watch?v=Ase9Qlol7ZY |title=演化天堂 水璉野生動物重要棲息環境深度之旅 |date=2014-05-08 |accessdate=2014-10-26 |publisher=農委會林務局}}</ref>。範圍從海拔高度531公尺的水璉尾山，往東延伸到[[鹽寮村|鹽寮]]至[[水璉村|水璉]]之間的花蓮海岸，面積339.86公頃，是東部[[海岸山脈|海岸山脈]]唯一位於[[北迴歸線|北迴歸線]]以北的保護區。[[台灣海棗|台灣海棗]]於此處保有台灣最大族群。交通方面有[[台11線|台11線]]海岸公路穿越，由北而南依序經過12號到15號橋。
