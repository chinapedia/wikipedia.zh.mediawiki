《'''水手服與機關槍'''》是[[日本|日本]]作家[[赤川次郎|赤川次郎]]的一部長篇[[小說|小說]]，在1978年首次出版。小說還發行了續集《畢業 - 水手服與機關槍・從今以後》（{{lang|ja|卒業－セーラー服と機関銃・その後}}）。

== 情節概要 ==
故事女主角「星泉」原本是一個普通的女高中生，但某一天父親發生事故死亡，而星泉因為是黑道組織「目高組」組長的遠親，而被要求擔任組織的第四代組長。

== 衍生作品 ==
該小說在1981年改編為同名電影，由[[藥師丸博子|藥師丸博子]]主演、[[相米慎二|相米慎二]]執導、[[角川映畫|角川映畫]]製作，以少女和大人之間的角色變換為主，以及原作中誇張的故事劇情，在當時成為大受歡迎的另類青春電影；由藥師丸演唱的同名主題曲也成為流行一時的歌曲。1982年[[富士電視台|富士電視台]]推出[[電視劇|電視劇]]版本，由[[原田知世|原田知世]]主演。2006年[[TBS電視台|TBS電視台]]推出重拍電視劇版本，由[[長澤雅美|長澤雅美]]、[[堤真一|堤真一]]主演。2016年，角川映畫推出由小說續集《畢業 - 水手服與機關槍・從今以後》改編的電影《水手服與機關槍 -畢業-》，由[[橋本環奈|橋本環奈]]主演。

從2006年10月13日開始，由[[TBS|TBS]]電視台製作拍攝，在[[日本|日本]]每週五晚間10點播出，一共有7集，主演為[[長澤雅美|長澤雅美]]。主題曲為星泉（[[長澤雅美|長澤雅美]]）演唱的同名歌曲。該劇其後於2007年間先後在[[香港|香港]][[美亞電視劇台|美亞電視劇台]]和[[台灣|台灣]][[緯來日本台|緯來日本台]]播出，其中台灣首播日期是當年6月25日。

長澤在劇中使用的槍枝是[[MP40冲锋枪|MP40]]短機關槍。在播出的同時，長澤主演的[[樂天|樂天]]（LOTTE）的糖果電視廣告，只限定在TBS系列電視台播出。此外，原本贊助星期五連續劇時段播出的[[豐田汽車|豐田汽車]]（TOYOTA）也因為形象的關係取消了贊助。

=== 與舊作相異處及特色 ===
* 黑道組織「目高組」的成員數量，除了組長星泉之外，在原作、電影版，以及舊電視版中都是4個人。本作中則增加為5人。
* 原作中泉為第四代組長，本作中則為第八代。
* 泉居住的公寓房號為「8940」，在日文中的發音近似「藥師丸」，是向電影版致敬的設計。

=== 人物 ===

'''目高組'''
*'''星泉''' - [[長澤雅美|長澤雅美]]
:: 本作的女主角，是一名女高中生。雙親皆已過世，為了找出殺害父親的兇手，因此答應繼承黑道組織「目高組」，成為第八代的組長。雖然沒有什麼特別的專長，但實際上非常有責任感。讀書方面十分用功，聽到黑道用語也會記下筆記。有很深的[[近視|近視]]，但眼鏡卻經常被弄壞，最後佩戴了隱形眼鏡。一看到血就會昏倒。
*'''佐久間真''' - [[堤真一|堤真一]]
:: 目高組的少主(若頭)。原本有意繼承組織，但前組長辰夫在遺言中指示另有接班人，並且要求他絕對支持新的組長。個性溫和，很能通融，非常重視仁義。
*'''酒井金造''' - [[山本龍二|山本龍二]]
:: 目高組的老二，擔任目高組的輔佐工作。在[[東京|東京]][[淺草|淺草]]被稱為「{{lang|ja|仏の金さん}}」（佛面金）。身材很瘦，原本是擔任警察。
*'''西野武''' - [[田口浩正|田口浩正]]
:: 目高組的老三。原本是[[電腦工程師|電腦工程師]]，對於電腦的使用十分拿手。利用網路進行[[股票|股票]]交易，支撐著目高組的財務。是一個容易流汗的胖子，而且很膽小。擅長照顧他人。
*'''剛田英樹''' - [[福井博章|福井博章]]
:: 目高組的組員。之前曾是[[暴走族|暴走族]]的頭目。個性易怒，對於身為女高中生的泉擔任組長感到不悅，但後來因為星泉的的決心而改觀。
*'''酒井健次''' - [[中尾明慶|中尾明慶]]
:: 目高組的組員。金造的外甥。在高中時常被欺負，自己要求加入組織。
*'''目高辰雄''' - [[桂小金治|桂小金治]]
:: 是小型的俠義集團目高組的第七代組長。因為肺炎在70歲時過世。原本指定外甥星貴志成為組織的接班人。離奇的是，兩個人的死亡日都是同一天。

'''其他'''
*'''星貴志''' - [[橋爪淳|橋爪淳]]
:: 泉的父親。自從妻子死後，和女兒星泉兩人相依為命。在第一集的開頭就發生[[車禍|車禍]]死亡，但之後被認為似乎是因為大量[[海洛因|海洛因]]引起的謀殺事件。
*'''真由美''' - [[小泉今日子|小泉今日子]]
:: 在星貴志葬禮上現身的神祕女子。自稱是貴志的戀人，和議員三大寺是父女關係。
*'''黑木幸平''' - [[小市慢太郎|小市慢太郎]]
:: 日本警視廳的刑警、也是組織犯罪應對部門的候補警部。與稻葉是調查上的搭檔。
*'''稻葉通男''' - [[井澤健|井澤健]]
:: 有日本東北口音的刑警，是黑木的部下，後來因為聽說到目高組消息，而被黑木所殺。
*'''濱口昇''' - [[本田博太郎|本田博太郎]]
:: 與目高組敵對的大型黑道組織組長。表面上則是一家企業的社長。
*'''森蘭丸''' - [[森廉|森廉]]
:: 濱口的親信。刀子的使用非常熟練，能夠削出令人驚訝的蘋果皮。
*'''岩倉智男''' - [[岡山一|岡山一]]
*'''金田麗華''' - [[森本夕子|森本夕子]] 
*'''常盤和子''' - [[井端珠里|井端珠里]] 
*'''小林朱美''' - [[谷亞里開|谷亞里開]]
*'''一樓住戶''' - [[杉浦双亮|杉浦双亮]]
*'''柴田光明''' - [[中野英雄|中野英雄]]
*'''三大寺一''' - [[緒形拳|緒形拳]]（特別演出）
:: 現任國會議員。與星貴志之死有關，正在找尋貴志留下來的海洛因。
