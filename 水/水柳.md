'''水柳'''（[[学名|学名]]：{{lang|la|''Salix warburgii''}}）是[[杨柳科|杨柳科]][[柳属|柳属]]的植物，为台灣特有植物。柳屬植物是維管束植物裡最耐寒的植物，最北可以生活在極區環境，而能在熱帶邊緣的台灣形成特有種的原因是演化迅速的物種在特殊環境的時空背景下所演變的結果，除了水柳外還有中國東南部的其他多雄蕊柳樹以及東南亞地區的四子柳都有相似情況，而這些特殊的柳屬植物又被稱為熱帶柳樹，而水柳在[[台湾|台湾]]的分布為低海拔的灌丛中以及河岸边，因扦插法容易生存因此目前尚未由人工引种栽培。

==特徵==
水柳生長於水邊，因種皮薄保水力不佳則需短時間內掉落於水中發芽生長，而裸露出的呼吸根及體內的大量酒精去氫酶能使水柳適應水中環境。花期為 2~4 月，果期則約在 4~6 月。

===葉===
葉柄基部有一對會脫落的耳狀托葉，葉片基部的葉柄上有腺點，特大葉的腺點會長成小葉狀。葉片的形狀有一段幾乎一樣的條片狀，兩端在縮小變窄，先端逐漸變細呈尖尾批針型，葉緣有明顯的鋸齒，葉背沒有毛不過常有灰白色的白粉，通常為單葉互生。表面呈有光澤綠色，背面粉白色。

水柳為雌雄異株植物，早春開花，雌株開綠花雄株開黃花，花無花瓣、花萼，只有雄蕊或雌蕊長於花萼狀的苞片中，花會聚集成穗狀的葇荑花序。<ref>{{Cite web|url=http://www.efloras.org/object_page.aspx?object_id=2142&flora_id=2|title=Illustration: Salix warburgii|accessdate=2017-03-25|work=www.efloras.org}}</ref>雄花穗長約4-8公分，包含了雄蕊4-7枚，花藥為顯著的黃色，花絲的基部有毛，被黃色腺體所圍繞。而雌花穗長約3-6公分，具比苞片長的子房柄，紡錘形的子房裡面有6-10枚胚珠，短短的花柱上裂長4瓣成了十字型，子房底部只有一個腺體。

=== 果實 ===
蒴果是紡錘型，果序長7-12公分，種子被有絨毛，即為常見的的「柳絮」。

== 栽培 ==
（1） 扦插法：所謂的「無心插柳柳成蔭」，恰恰說明了水柳的繁殖方式。若選在春、夏的季節施行，剪強壯2~3年生枝條，長15~25公分，進行扦插的話，便能輕易得到新的植株。 

（2）播種法：水柳的種子壽命短暫，應在採取後兩周內播種為宜。 

【注意事項】

<nowiki>*</nowiki> 喜歡陽光，日照必須充足。 

<nowiki>*</nowiki> 幼株在春、夏季生長迅速，為生長旺季，1~2個月應施肥1次，以及適時補充水分。

<nowiki>*</nowiki> 喜好潮濕環境，栽培土壤以濕潤的壤土或砂質壤土最佳。生長適當溫度在攝氏14~30度。  

<nowiki>*</nowiki> 在冬季落葉時期，要整枝、修剪，美化樹形。對於已經老化的植株施行強剪，可促進萌發新的枝葉。如果想讓樹幹長高，在主幹下方的側枝剪下，則可達成長高的期待。 

== 相關資訊 ==
“柳絮”，是楊柳類的蒴果開裂後，夾帶種絮隨風飛揚的種子，而古人常誤把絮當作花，俗稱「楊花」。所謂「春城無處不飛花」，其實是有原因的，而「楊花落水為萍」，竟使飛花之遺跡成了「一池萍碎」，更使得「楊花水性」成為千古奇冤。

水柳的枝條常上舉開展，風緻與垂柳有別。栽植極為容易，不但耐水濕，還可作為低海拔水邊的綠化樹種，或良好的防風植物。

== 藥用價值 ==
根皮、鬚根、樹皮、枝及葉性苦寒。

而根及莖則利氣、活血、解熱、治跌打、疲勞；

枝葉可以利氣、行血、解熱、治跌打損傷、皮膚病。 

可說是其為具貢獻的一種植物。
