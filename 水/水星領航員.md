《'''水星領航員'''》（[[日文|日文]]原名：'''ARIA'''）是[[日本|日本]]漫畫家[[天野梢|天野梢]]<ref>日文原名為{{lang|ja|[[:ja:天野こずえ|天野こずえ]]}}，目前並無固定中文譯名，暫譯之（此為大陸譯名）。</ref>所創作的[[漫畫|漫畫]]，以及依此改編的[[電視動畫|電視動畫]]、[[小說|小說]]、[[電視遊戲|PS2遊戲]]等。

本書原先以《AQUA》的名稱在[[月刊Stencil|月刊Stencil]]上連載。雜誌休刊後，又自2002年4月起以《ARIA》為名於[[月刊Comic_Blade|月刊Comic Blade]]上開始連載，而第一季的電視動畫自2005年10月5日起至12月28日以'''ARIA The ANIMATION'''之名播映，共十三集。第二季的電視動畫自2006年4月2日至9月26日以'''ARIA The NATURAL'''之名每週播映，共二十六集。第三季的電視動畫'''ARIA The ORIGINATION'''自2008年1月7日至3月31日播映，共十三集。

2015年2月5日，在原作者天野梢的另一部作品《[[藍海少女！|藍海少女！]]》中宣布進行第一季動畫播放10周年紀念企劃《「ARIA」蒼のカーテンコール》，計畫推出藍光光碟BOX，以及製作新作'''ARIA The AVVENIRE '''，計劃於2015年9月26日於全日本11家劇場上映（後在7月擴大上映規模到23家）<ref>{{Cite web|url = https://twitter.com/ARIATheAVVENIRE/status/619089078542163968|title = 上映劇場擴大的宣布|accessdate = 2015-07-24|author = |date = |publisher = }}</ref>，並預定於每月5日（即《[[藍海少女！|藍海少女！]]》被刊載之雜誌《[[Comic_Blade|Comic Blade]]》出刊日）發表紀念企劃的新情報。另外在同日，紀念企劃及新作動畫的官方網頁，網站，推特帳戶及Facebook專頁開設。其中日本時間2月5日晚上11點開設，Mag-Garden內的紀念企劃網頁由於瀏覽量過大，在翌日暫時封閉。截至2月11日，紀念企劃網頁已經恢復<ref>{{Cite web|url = http://www.mag-garden.co.jp/aria/|title = 『「ARIA」蒼のカーテンコール』特設ページ|accessdate = 2015-02-11|language = ja}}</ref><ref>{{Cite web|url=http://www.ariacompany.net/ |title=官方網站 |accessdate=2015-02-08 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20071229191353/http://www.tokyopop.com/product/2123/Aria/1 |archivedate=2007-12-29 }}</ref><ref>{{Cite web|url = https://twitter.com/ARIATheAVVENIRE|title = 官方推特帳戶|accessdate = 2015-02-08|language = ja}}</ref><ref>{{Cite web|url = https://www.facebook.com/ARIATheAVVENIRE?pnref=story|title = 官方Facebook專頁|accessdate = 2015-02-08|language = ja}}</ref>。

2015年4月4日，新作之特報影像在相同製作公司製作的《[[幸福光暈|幸福光暈]] ～畢業寫真～ 第1部　芽～GIZASHI～》首次放映時首次公開，並於4月6日在網上公開<ref>{{Cite web|url = https://www.youtube.com/watch?v=cTnTUsKHsPI&feature=youtu.be|title = ARIA The AVVENIRE 特報|accessdate = 2015-04-06|language = ja}}</ref>。另外官方推特及網頁在同日公布聲優名單，將保持所有在以前電視版出演的聲優不變<ref>{{Cite web|url = http://www.ariacompany.net/character.php|title = 角色介紹頁面|accessdate = 2015-04-06|language = ja}}</ref>。對於擔當雅典娜・葛絡俐的聲優，在2011年去世的川上倫子的問題，經過各方同意，她的名字將會作為製作委員會一員在職員表列出<ref name=":0">{{Cite web|url = https://twitter.com/ARIATheAVVENIRE/status/584923186413703168|title = 發行商製作人在官方推特的發言|accessdate = 2015-04-06|language = ja}}</ref>，而角色介紹上仍會列出她的名字<ref>{{Cite web|url = https://twitter.com/ARIATheAVVENIRE/status/584922660557033473|title = 發行上製作人在官方推特的發言|accessdate = 2015-04-06|language = ja}}</ref>。由於《[[藍海少女！|藍海少女！]]》在5月號《[[Comic_Blade|Comic Blade]]》沒有出刊，5月並無本作的情報公開。

2015年6月5日，官方公布有關於新作之構成，新登場角色的設定圖等。本作將由3集故事構成，並且在本作使用3D技術，協助描繪鳳尾舟等有難度的場景<ref>{{Cite web|url = http://www.ariacompany.net/character.php|title = 人物，性格設定介紹頁面|accessdate = 2015-06-05|author = |date = |publisher = |language = ja}}</ref><ref>{{Cite web|url = http://www.ariacompany.net/comment5.php|title = 原作者，動畫相關人士訪談|accessdate = 2015-06-05|author = |date = |publisher = |language = ja}}</ref>。另外，過往播放之三季動畫的藍光光碟BOX將由原作者繪畫封面圖片，並附送映像特典，音聲特典，以及動畫新作等特典；劇場上映方面將於日本全國11所影院上映，並在6月20日開始發售前售券<ref>{{Cite web|url = https://twitter.com/ARIATheAVVENIRE/status/640706705051914240|title = 有關入場者特典的頁面(第二周)|accessdate = 2015-09-24|author = |date = |publisher = |language = ja}}</ref>。9月7日公布將發行入場者特典，為本作內三家不同公司內角色色紙，並進行每周輪替。

2015年6月18日，官方公布新登場角色的資料，姬屋所屬的梓・B・麥拉倫將由[[中原麻衣|中原麻衣]]飾演，橙色行星所屬的阿尼亞將由[[茅野愛衣|茅野愛衣]]飾演。<ref>{{Cite web|url = https://twitter.com/ARIATheAVVENIRE/status/611368982650556417|title = 有關梓的角色資料|accessdate = 2015-07-24|author = |date = |publisher = }}</ref><ref>{{Cite web|url = https://twitter.com/ARIATheAVVENIRE/status/611369210699120640|title = 有關安妮雅的角色資料|accessdate = 2015-07-24|author = |date = |publisher = }}</ref>官方並在其後的6月25日宣布在一個月後的7月25日進行動畫第一季的一舉放送（一次播放第一季所有話數）。<ref>{{Cite web|url = https://twitter.com/ARIATheAVVENIRE/status/614006184538836992|title = 一舉放送的宣布|accessdate = 2015-07-24|author = |date = |publisher = }}</ref>另外官方也在分別在8月2日，8月9日及9月6日進行第二及第三季的一舉放送。

作者在連載初期曾表示這是一部「水與貓的故事」；漫畫上的介紹則為「[[未來|未來]]式的治癒系漫畫」。
另，幾乎所有出場人物的名字都統一以「A（{{lang|ja|あ}}）」為開頭，而作品中還有作者以前作品的登場人物、作者身邊的貓們會出現。

「'''ARIA'''」名稱的由來，為作者過去飼養的一隻[[貓|貓]]名。

==劇情簡介==
故事背景時代設定在24世紀。彼時人類在[[火星|火星]]上進行行星[[地球化|地球化]]改造工程，由於兩極冰帽融解的水量超乎預期，全星球有90%被海洋覆蓋，因而被人稱呼為「'''水星'''」（Aqua）。在此星球上有個以過去被稱為水都的[[威尼斯|威尼斯]]為範本所建造的城市'''新威尼斯'''（Neo-Venezia），因其仿古的建築與承襲傳統的生活方式而廣受觀光客喜愛，而其中有一種駕駛小舟穿梭於[[運河|運河]]之間，帶領觀光客遊覽整座城市的職業，被稱為'''領航員'''（Undine）。

出身於[[地球|地球]]（故事中稱為「行星Manhome」）的'''水無 燈里'''，以成為能獨當一面的領航員為目標而來到新威尼斯，目前在 艾莉西亞 和 亞利亞社長 的 Aria Company 中見習。身為半熟手的她，每天都持續練習操控貢多拉及觀光導覽。《水星領航員》一書便是在描寫燈里以及她身旁的朋友在一年[[四季|四季]]中的日常生活點滴。

在本作品裡出現的'''Undine'''（本意為水之[[精靈|精靈]]溫蒂妮）指的是專為觀光導覽的貢多拉操控者。能從事的只有女性，是Aqua的明星職業。

領航員手上是否有戴手套為能力的指標。見習生的手為雙手套（Pair）；半熟手為單手套（Single）；能獨當一面的老手則無[[手套|手套]]（Prima）。這是因為有本事的領航員不必使用過多的力量即可操控貢多拉、手也不大會起水泡，所以也就沒有戴手套保護的必要了。

Pair 升級 Single 的測驗是操控貢多拉到達稱為「希望之丘」的地點，而此為測驗內容、其位置與路程皆禁止洩漏；要升格為 Prima 雖須通過貢多拉協會特定的考驗，但升級並無明確標準，一般是交由各家公司或負責指導的前輩，關於測驗的方法或標準完全由主考官決定。Prima 的平均升格年齡為18~20歲，成為 Prima 後便可允許為自己取個通稱。

可以自由營業載客的只有無手套 ，單手套需有負責指導的無手套陪同。渡船（Traghetto）則允許單手套自行營業的，費用上比起Prima 較為便宜，也是由於 Prima 所操控的貢多拉較為安全之故。作品中燈裡經常使出「不收取費用，而以朋友的名義乘船」的鑽漏洞招數，因為「規則上所禁止的是單手套以下的領航員所控船之營業行為 ，並非禁止讓人一同乘船」，而週遭的人們也會默許。此招的發明者是以消磨時間為目的而想免費乘[[船|船]]的看火人阿曉。動畫裡的發明者則是小愛，只是後來阿曉也免費乘船。此外， 單手套也能以助手的身份搭乘無手套所操控的貢多拉，但動畫中，卻有身為雙手套的艾裡須搭乘雅典娜或晃所操控之貢多拉的場面，[[小說|小說]]裡同樣也有雅典娜的貢多拉讓艾裡須搭乘的情況。在月刊Undine第3集（18月號）裡，艾裡須操縱貢多拉卻在有[[雅典娜|雅典娜]]搭乘的情況下接受採訪，想來也是用這個招數來隨機應變。

領航員的制服為統一，夏天為類似[[水手服|水手服]]的襯衫（亞利亞公司的設計有很大的不同，襯衫的前襟為[[西服|西服]]領，只有後領還保留[[水手服|水手服]]的形式）、貼身衣則為下襬帶有開口的[[連衣裙|連衣裙]]，再加上帽子、鞋子、和半指手套，胸口繫上領結；冬季服飾方面，從長袖[[連衣裙|連衣裙]]的上面、肩膀的地方披著繫有領結的短斗蓬（手套的設計也和夏天不同）。每家公司的服飾[[花樣|花樣]]、[[帽子|帽子]]、[[領結|領結]]、[[鞋子|鞋子]]、[[手套|手套]]、及連衣裙下襬的開口位置不盡相同，成為各家公司的特色。另外在袖子和帽子上則帶有各家公司的標識。

==遊戲==
;《ARIA The NATURAL ～遙遠記憶之幻象～》<br />（{{lang|ja|ARIA The NATURAL ～遠い記憶のミラージュ～}}）
為[[Alchemist|Alchemist]]在2006年9月28日發行的[[PS2|PS2]]遊戲。<br />
遊戲本身是以[[視覺小說|視覺小說]]界面構成，故事從青年主角的觀點來描述，內容添加了遊戲獨創要素、忠實呈現原作漫畫及動畫影像，更增強了動畫的呈現效果。作者除了特別為遊戲新繪[[CG|CG]]外，還親自設計遊戲原創角色。在2006年4月Alchemist所舉辦的活動「{{lang|ja|アルケ祭}}」中發表的遊戲主題曲，其作詞、作曲的為近來在遊戲歌曲圈佔有穩固地位的[[志倉千代丸|志倉千代丸]]。<br />

;故事
主角是在地球的大學裡專攻都市工學的青年，為了做[[實地考察|實地考察]]而造訪水星，因某個契機與燈里相遇後便一起行動。原本和燈里等人在水星過著愉快平穩的生活，但那樣的情景卻逐漸一步一步地改變了。

;主題曲
*片頭曲「'''BLUE BLUE WAVE'''」（歌：[[KAORI|KAORI]]，作詞、作曲：志倉千代丸，編曲：磯江俊道）
*片尾曲「'''{{lang|ja|その小さな小さな微笑で}}'''」（歌：KAORI，作詞、作曲：志倉千代丸，編曲：上野浩司）

;《ARIA The ORIGINATION ～藍色行星的天空～》<br />（{{lang|ja|ARIA The ORIGINATION ～蒼い惑星のエルシエロ～}}）
為Alchemist在2008年6月26日發行的PS2遊戲。背景為冬天的新威尼斯，讓玩家從以Prima為目標的見習領航員阿涅絲·杜馬（安妮）的觀點來體驗遊戲。

;主題曲
*片頭曲「'''graceful way'''」（歌：[[Kicco|Kicco]]，作詞、作曲：志倉千代丸，編曲：上野浩司）
*插入歌「'''Il Cielo'''」（歌：安妮（[[花澤香菜|花澤香菜]]），作詞、作曲、編曲：水野大輔）
*片尾曲「'''{{lang|ja|花ノ咲ク星}}'''」（歌：[[諫山實生|諫山實生]]，作詞、作曲：志倉千代丸，編曲：水野大輔）
