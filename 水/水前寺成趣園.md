'''水前寺成趣園'''（{{Jpn|kn=すいぜんじじょうじゅえん|rm=Suizen-ji Jōju-en}}）是位在[[日本|日本]][[熊本縣|熊本縣]][[熊本市|熊本市]][[中央區_(熊本市)|中央區]]{{link-ja|水前寺|水前寺}}公園（町名）的{{link-ja|大名庭園|大名庭園}}。面積約7萬3千平方公尺。通稱'''水前寺公園'''。該園是由[[阿蘇山|阿蘇山]]伏流水湧出之池為中心的桃山式回遊庭園，以[[築山|築山]]、浮石、草坪（{{lang-ja|芝生}}）和松树等观赏用树木來模仿[[東海道五十三次|東海道五十三次]]之景勝。

園內还有[[出水神社_(熊本市)|出水神社]]和稱為古今傳授之間（{{lang-ja|古今伝授の間}}）的古建築。

[[2016年熊本地震|2016年熊本地震]]後，因不明原因池水流失，池塘約80％的底部露出。<ref>{{Cite web|url=http://www.asahi.com/articles/ASJ4S5SQCJ4STIPE025.html|title=水前寺成趣園の池、地震後に底露出　水源枯れる？　熊本|accessdate=2016-04-24|work=|publisher=|date=|language=ja|archive-url=https://web.archive.org/web/20160424224552/http://www.asahi.com/articles/ASJ4S5SQCJ4STIPE025.html|archive-date=2016-04-24|dead-url=no}}</ref>

== 概要 ==
[[熊本藩|熊本藩]]初代藩主[[細川忠利|細川忠利]]在[[寛永|寛永]]13年（1636年）頃築造「水前寺御茶屋」。歷三代藩主，耗費八十年光陰才於{{link-ja|細川綱利|細川綱利}}之時築造泉水・築山等，完成現在所見的庭園。

1929年（[[昭和|昭和]]4年），以「水前寺成趣園」之名，被指定為國之{{link-ja|名勝|名勝}}和國之[[史跡|史跡]]。

== 園內景點 ==
[[File:Suizenji_Park_2.jpg|300px]]
水前寺成趣園的簡要地形可參看右圖。編號1至7為一些景點。
; 1. [[出水神社_(熊本市)|出水神社]]
: 神社祭祀細川家歷代藩主。[[明治|明治]]10年（1877年）[[西南戰爭|西南戰爭]]之後，熊本周邊幾為焦土。因前熊本藩藩士感念故主，為祈求人心安定、和平繁盛，於翌年（1878年）在與細川家淵源甚深的水前寺成趣園建立。
: [[二戰|二戰]]中神殿毀於戰火，[[昭和|昭和]]45年（1970年）開始重建，三年之後完工。<ref name=Jinja>{{Cite web|url=http://www.suizenji.or.jp/izumi/01.html|title=出水神社について|accessdate=2015-07-28|work=|publisher=|date=|language=ja|archive-url=https://web.archive.org/web/20150628192612/http://www.suizenji.or.jp/izumi/01.html|archive-date=2015-06-28|dead-url=no}}</ref>
; 2. 神水“长寿之水”: 是源于阿苏山系的涌泉。
; 3. 古今傳授之間
: 西側池畔的{{link-ja|茅屋|茅葺}}。400年前最早建於[[京都御苑|京都御苑]]的八条宮家（後來的[[桂宮_(宮家)|桂宮]]），作為少年的八條宮智仁親王的學問所。因[[慶長|慶長]]5年（1600年），初代當主[[細川藤孝|細川藤孝]]為八條宮親王做「{{link-ja|古今傳授|古今伝授}}」，即講授《[[古今和歌集|古今和歌集]]》而得名。此建築一度從京都御苑遷往[[長岡京|長岡京]]，後於明治4年（1871）年，賜予細川家。經過解體保管等重重處理，於[[大正|大正]]元年（1912年）在此地复原。<ref name=Kokin>{{Cite web|url=http://www.kobai.jp/kokin/|title=古今伝授の間|accessdate=2015-07-28|work=|publisher=|date=|language=ja|archive-url=https://web.archive.org/web/20150716053857/http://www.kobai.jp/kokin/|archive-date=2015-07-16|dead-url=no}}</ref>
; 4. “富士山”
: “築山”，即人造土丘。周邊為園林主體。細川家第4代當主{{link-ja|細川綱利|細川綱利}}曾於此命名成趣園十景。
; 5. [[能劇|能劇]]舞台
: 位於成趣園南端。每年春秋的大祭典中會在此奉納神事能。每年8月第一個星期六的日落後有御薪能。
; 6. [[流鏑馬|流鏑馬]]的馬場
: 作為每年春秋的奉納儀式的一部分，在此處會有流鏑馬的表演。<ref name=Baba>{{Cite web|url=http://www.suizenji.or.jp/map/3-5.html|title=園内地図／流鏑馬の馬場|accessdate=2015-07-28|work=|publisher=|date=|language=ja|archive-url=https://web.archive.org/web/20141229145950/http://www.suizenji.or.jp/map/3-5.html|archive-date=2014-12-29|dead-url=no}}</ref>
; 7. 细川忠利像
: 於1979年出水神社100週年時設立。<ref name=Douzou>{{Cite web|url=http://www.suizenji.or.jp/map/3-11.html|title=園内地図／御祭神細川忠利公銅像|accessdate=2015-07-28|work=|publisher=|date=|language=ja|archive-url=https://web.archive.org/web/20160304185620/http://www.suizenji.or.jp/map/3-11.html|archive-date=2016-03-04|dead-url=no}}</ref>

* 開園時間：8時30分 - 17時00分（11月 - 2月），7時30分 - 18時00分（3月 - 10月）<ref name=Jojuen>{{Cite web|url=http://www.suizenji.or.jp/suizenji/01.html|title=水前寺成趣園|accessdate=2015-07-28|work=|publisher=|date=|language=ja|archive-url=https://web.archive.org/web/20150801143442/http://www.suizenji.or.jp/suizenji/01.html|archive-date=2015-08-01|dead-url=no}}</ref>
