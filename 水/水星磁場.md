'''水星磁場'''近似於磁偶極 (意思是這個磁場只有兩個磁極)<ref>{{Cite news |url=http://science.nasa.gov/science-news/science-at-nasa/2008/03jul_mercuryupdate/ |title=New Discoveries at Mercury |work=Science@Nasa |author=Tony Phillips |date=2008-07-03 |accessdate=2011-07-16}}</ref>，這是值得注意，而且是全球性的<ref>{{cite web|url = http://nssdc.gsfc.nasa.gov/planetary/factsheet/|title = Planetary Fact Sheet|publisher = [[NASA|NASA]] Goddard Space Flight Center|last = Williams|first = David R.|accessdate = 2011-07-25}}</ref>，在[[水星|水星]]<ref name="Mercury Magnetic Poles">{{cite web|url=http://www.windows2universe.org/mercury/mercury_magnetic_poles.html&edu=high|title=The Magnetic Poles of Mercury|work=Windows to the Universe|author=Randy Russell|date=2009-05-29|accessdate=2011-07-16}}</ref>。 依據[[水手10號|水手10號]]太空船於1974年發現的資料，水星磁場的強度只有地球的1.1% <ref name="Mercury Magnetic Field">{{cite web|url=http://www.universetoday.com/35873/mercury-magnetic-field/|title=Mercury Magnetic Field|work=Universe Today|author=Jerry Coffey|date=2009-07-24|accessdate=2011-07-16}}</ref>。磁場的起源可以用[[發電機原理|發電機原理]]來解釋<ref name="Molten Core Mystery">{{Cite news |url=http://physicsworld.com/cws/article/news/29905 |title=Molten core solves mystery of Mercury's magnetic field |work=Physics World |author=Jon Cartwright |date=2007-05-04|accessdate=2011-07-16}}</ref>，並且因為磁場是足夠強大，可以在附近形成[[弓形震波|弓形震波]]，減緩[[太陽風|太陽風]]的速度，誘發[[磁層|磁層]]<ref name="Magnetosphere of Mercury">{{cite web|url=http://www.windows2universe.org/mercury/Magnetosphere/magsphere_overview.html|title=Magnetosphere of Mercury|work=Windows to the Universe|author=Randy Russell|date=2009-06-01|accessdate=2011-07-16}}</ref>。

== 起源 ==
{{See also|發電機原理}}
[[發電機原理|發電機理論]]可以解釋磁場的起源<ref name="Molten Core Mystery"/>；也就是，通過在[[行星核心|核心]]外圍熔融的導電鐵水的對流<ref name="Magnetic Field Measurement">{{Cite web|url=http://www.spacedaily.com/reports/Measuring_the_Magnetic_Field_of_Mercury_999.html|title=Measuring the Magnetic Field of Mercury|work=SpaceDaily|author=Staff Writers|date=2011-05-20|accessdate=2011-07-16}}</ref>。行星的核心是以沉降至[[質量中心|質量中心]]的一個巨大鐵核，多年來尚未完全冷卻，外核也還沒有完全凝固，並繞著內部流轉著。在1974年發現之前，因為它的體積嬌小，認為在[[太陽系|太陽系]]的演化過程中，水星的核心已經冷卻了。但仍有一些發電機理論無法解釋的，包括長達59天的自轉週期，使它不太可能生成[[磁場|磁場]]的困難。

因為它是由在內核內部熱的部分對流來驅動，所以這個發電機很可能比地球的微弱。在核與地函交界處的溫度梯度是近似絕熱的，因此液化的核心區域外圍是穩定的分出層次，發電機只在一定的深度中運作，產生強大的磁場<ref>{{Cite journal|publisher=Max-Planck Institute|location=Katlenberg-Lindau|title=Nature; 2006|volume= 444|issue=7122|pages= 1056–1058|doi=10.1038/nature05342|year=2006 Max-Planck Institute of Germany|contribution=A deep dynamo generating Mercury's magnetic field|first=Ulrich R.|last=Christensen|journal=Max-Planck Institute, Germany, Nature |url=http://www.nature.com/nature/journal/v444/n7122/abs/nature05342.html|ref=harv|postscript=<!--None-->|bibcode = 2006Natur.444.1056C|pmid=17183319}}</ref>。 由於行星緩慢的自轉，結果磁場主要是由快速波動與時俱進的小型元件產生。

== 磁極和磁性測量 ==
[[File:Mercury_in_color_-_Prockter07_centered.jpg|thumb]]
像地球一樣，水星的磁場是傾斜的<ref name="Mercury Magnetic Poles"/><ref>{{Cite web |url = http://www.windows2universe.org/mercury/mercury_polar_regions.html |title = Mercury's Poles |work = Windows to the Universe |author = Randy Russell |date = 2009-05-29 |accessdate=2011-07-18}}</ref>，意味著水星的磁極與地理的極點不在同一個位置或區域上。水星磁場在內部的南北不對稱的結果是，磁場線的幾何形狀在水星的南極和北極是不同的<ref>{{Cite news|url = http://www.nasa.gov/mission_pages/messenger/multimedia/messenger_news20110616_image3.html|title = Magnetic field lines differ at Mercury's north and south poles|author = Lynn Jenner; Brian Dunbar|publisher = [[NASA|NASA]]|date = 2011-06-16|accessdate = 2011-07-18}}</ref>。特別是，在南極附近磁場的"極冠"場線開放至行星際空間的量大得多。這在幾何上意味著南極地區比北極更曝露在帶電粒子的加熱和太陽風-磁層的交互作用。[[磁矩|四極矩]]和偶極矩力的傾斜完全不受到約束<ref name="Magnetic Field and Magnetosphere">{{Cite web |url = http://www-ssc.igpp.ucla.edu/personnel/russell/papers/merc_mag/ |title = Mercury: Magnetic Field and Magnetosphere |author = C. T. Russell; J. G. Luhmann |publisher = University of California, Los Angeles |accessdate = 2011-07-18}}</ref>。

已經有各種不同的方法量度過水星的磁場。一般情況下，當依據磁場的大小和形狀 (~150–200 nT ''R''<sup>3</sup>) 估計，推斷等效的內部偶極矩是較小的<ref>{{Cite web |url=http://citeseerx.ist.psu.edu/viewdoc/summary?doi=10.1.1.25.5685&rep.pdf |title=Modelling of magnetic field measurements at Mercury |publisher=Imperial College, London |author=Giacomo Giampieri; André Balogh |format=PDF |accessdate=2011-07-18}}</ref>。最近，地基的雷達測量出水星的自轉有著輕微的搖擺運動，解釋水星的核心至少有一部分還是熔融的。言下之意，鐵"雪"有助於維持磁場<ref>{{Cite web |url= http://www.sciencedaily.com/releases/2008/05/080507110712.htm |title= Iron 'snow' helps maintain Mercury's magnetic field, scientists say |work= ScienceDaily |date= 2008-05-08 |accessdate= 2011-07-18}}</ref>。信使號太空船將使用靈敏的磁力儀收集超過5億筆對水星磁場測量的資料<ref name="Magnetic Field Measurement"/>。

== 發現 ==
[[File:Mariner_10.jpg|thumb]]
在1974年以前，水星由於相對較小的直徑和缺乏[[大氣層|大氣層]]，被認為不生成磁場。然而，當''水手10號''飛掠過水星時 (大約在1974年4月)，它檢測到了總規模約為[[地磁場|地磁場]]1/100的磁場。但是這些經過在磁場本質的大小、它的方向和它的調和結構只提供了微弱的拘束力，部分是因為缺乏對太陽風密度和速度的觀察<ref name="Magnetic Field and Magnetosphere"/>。由此一發現，水星的磁場受到大量的關注<ref>{{cite web
 |url = http://www.foxnews.com/story/0,2933,326867,00.html
 |title = NASA Spots Mysterious 'Spider' on Mercury
 |work = [[FoxNews|FoxNews]]
 |author = Clara Moskowitz
 |date = January 30, 2008
 |accessdate = July 20, 2011
}}</ref>，主要是因為水星的體積小和長達59天的自轉週期。

關於磁場的起源，目前認為是來自[[發電機原理|發電機機制]]<ref name="Molten Core Mystery"/><ref>{{cite web|url = http://www.time.com/time/magazine/article/0,9171,879479,00.html|title = Science: Mercury's Magnetism|work = [[Time_(magazine)|Time]]|date = 1975-03-31|accessdate = 2011-07-23}}</ref>，雖然這是否負責磁場尚未得到很好的說明。

== 場特徵 ==
[[File:MESSENGER.jpg|thumb]]的磁場糾纏在一起－－ 有些的寬度達到800公里，或是行星總直徑的1/3。]] 

科學家指出水星的磁常有著極度的"滲漏"<ref>{{cite web|url = http://www.nasa.gov/mission_pages/messenger/multimedia/magnetic_tornadoes_prt.htm|title = Magnetic Tornadoes Could Liberate Mercury’s Tenuous Atmosphere|publisher = [[NASA|NASA]]/Goddard Space Flight Center|last = Steigerwald|first = Bill|date = 2009-06-02|accessdate = 2009-07-18}}</ref><ref>{{cite web|url = http://www.sciencedaily.com/releases/2009/06/090602112255.htm|title = Magnetic Tornadoes Could Liberate Mercury's Tenuous Atmosphere|work = ScienceDaily|author = NASA/Goddard Space Flight Center|date = 2009-06-02|accessdate = 2011-07-25}}</ref><ref>{{cite web|url = http://www.universetoday.com/31953/how-magnetic-tornadoes-might-regenerate-mercurys-atmosphere/|title = How Magnetic Tornadoes Might Regenerate Mercury's Atmosphere|work = Universe Today|author = Brian Ventrudo|date = 2009-06-03|accessdate = 2011-07-25}}</ref>，因為信使號在2008年10月6日第二次飛越水星時遇到磁龍捲風，它們可能在充填[[水星大氣層|大氣層]] (或是天文學家所謂的"散逸層")。當''水手10號''在1974年再度飛越水星時，在進入和離開磁層頂時，測量到弓形激波的訊號，並且磁氣層的空腔大約比地球小了20倍，當信使號接近時，所有都被推測為衰減<ref>{{cite web|url = http://www.lpl.arizona.edu/~shane/PTYS_395_MERCURY/presentations/Mercury_magnetic_field.pdf|title = Mercury's Magnetic Field|publisher = [[University_of_Arizona|University of Arizona]] – [[Lunar_and_Planetary_Laboratory|Lunar and Planetary Laboratory]]|format = PDF|author = Kerri Donaldson Hanna|accessdate = 2011-07-25}}</ref>。即使場的強度剛超越地球的1%，''水手10號''檢測到的這個值讓一些科學家認為水星的核心仍是或至少部分仍是[[行星核|液體]]，且可能具有[[鐵|鐵]]和其它[[金屬|金屬]]<ref>{{cite web|url=http://www.newscientist.com/article/dn11782-molten-core-may-explain-mercurys-magnetic-field.html|title=Molten core may explain Mercury's magnetic field|work=New Scientist|author=David Shiga|date=2007-05-03|accessdate=2011-07-25}}</ref>。
