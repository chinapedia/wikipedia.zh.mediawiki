'''水晶工具'''（[[英文|英文]]：'''Crystal Tools'''）是由[[史克威尔艾尼克斯|史克威尔艾尼克斯]]开发并内部使用的[[游戏引擎|游戏引擎]]。引擎开发由村田琢领导。引擎最初命名为“白色引擎”（[[英文|英文]]：'''White Engine'''），于2007年易名。在引擎制作时，公司决定制作一款供[[电子游戏机历史_(第七世代)|第七世代]]游戏机使用的内部引擎。公司游戏开发者确信，从[[PlayStation_2|PlayStation 2]]到[[PlayStation_3|PlayStation 3]]转变的障碍比从[[PlayStation|PlayStation]]到PlayStation 2的大，因而需要一款[[中间件|中间件]]来帮助支持开发挑战。引擎开发致力于能创作先进面部动画，并在公司内创作统一的3D格式。

== 开发 ==
在2004年初[[史克威尔艾尼克斯|史克威尔艾尼克斯]]决定，需要创作通用的3D数据格式，并因此于2006年9月成立开发研究部门，由村田琢任经理<ref name="tools" />。在2007年[[游戏开发者大会|游戏开发者大会]]上，村田琢称《[[最终幻想XII|最终幻想XII]]》的开发通过使用专有软件，电视上游戏的“即时显示”将会表现的即时而清晰<ref name="tools" />。2007年11月，史克威尔艾尼克斯透露，开发正处于“呆滞”速度，一些人推测《[[最终幻想XIII|最终幻想XIII]]》或《[[勇者斗恶龙IX|勇者斗恶龙IX]]》这种使用白色引擎的游戏将会延期<ref name="delay">{{cite web |title=White Engine development stalled, Final Fantasy XIII too?  |url=http://www.siliconera.com/2007/11/19/white-engine-development-stalled-final-fantasy-xiii/|author=Spencer |date = 2007-11-19|publisher=Siliconera |accessdate = 2013-04-05|language=en}}</ref>。

在2008年游戏开发者大会上，村田制作了一个“最终幻想之技术”的演示，其中他称，他们已经开始制作一款名为“白色引擎”游戏开发引擎，而且1.0版已经做成<ref name="tools">{{cite web | title=‘Crystal Tools’: Final Fantasy Engine Renamed, Supports Wii |url=http://www.wired.com/gamelife/2008/02/finalfantasy/ |author=Chris Kohler |date=2008-02-22 |work=[[Wired_(magazine)|Wired]] |accessdate=2012-08-17|language=en}}</ref><ref name="ign">{{cite web |title=GDC 2008: Final Fantasy Updates |url=http://ps3.ign.com/articles/854/854309p1.html |author=Ryan Clements |date=2004-02-22 |publisher=[[IGN|IGN]] |accessdate=2012-08-17|language=en}}</ref><ref name="joy">{{cite web| title=GDC08: Square Enix unveils Crystal Tools engine |url=http://www.joystiq.com/2008/02/22/gdc08-square-enix-unveils-crystal-tools-engine/ |author=Andrew Yoon |date=2008-02-22 |publisher=[[Joystiq|Joystiq]] |accessdate=2012-08-17|language=en}}</ref>。他还宣称1.1版，或称“水晶工具”，于2007年7月完成<ref name="joy" />。村田随后在采访中称，白色引擎本为代码名称，他们想取名一个同史克威尔艾尼克斯有联系的官方名称，而水晶既能映出“多彩颜色”，又能映出[[最终幻想系列|最终幻想系列]]<ref name="gamasutra">{{cite web | title= Q&A: Square Enix's Murata Talks Crystal Tools, Unreal Engine Initiatives |url=http://www.gamasutra.com/php-bin/news_index.php?story=18246 |author=Brandon Sheffield |date=2008-04-18|work=[[Gamasutra|Gamasutra]] | accessdate=2012-08-17|language=en}}</ref>。在他2008年的演讲的结语中，他称技术将会用于[[Xbox_360|Xbox 360]]、[[PlayStation_3|PlayStation 3]]、PC和“缩小”版Wii，《最终幻想XIII》、《[[最终幻想Versus_XIII|最终幻想Versus XIII]]》（2013年更名为〈最终幻想XV〉<ref>{{cite news|title=《最终幻想V13》已更名FF15转向PS4|url=http://comic.qq.com/a/20130319/000028.htm|accessdate=2013-12-11|newspaper=腾讯动漫|date=2013年3月19日|language=zh-cn}}</ref>）和一款新[[大型多人在线角色扮演游戏|MMORPG]]——即《[[最终幻想XIV|最终幻想XIV]]》——都将使用该技术<ref name="ign" />。

水晶工具引擎从零开始设计。而非改编自既有引擎。较之于制作一款在任何场景下工作的的基本引擎，该引擎通过制作不同插件，来扩展功能制作特定游戏<ref name="gamasutra" />。游戏开发者通过引擎，按照最终幻想风格，请求许多角色特写与风格<ref name="joy" />。当问及史克威尔可否将引擎授权给其它公司开发游戏时，他称史克威尔的重点在别处，他们不愿花时间建立文档与提供支持，让其它开发者使用该工具<ref name="gamasutra" />。[[北濑佳范|北濑佳范]]随后补充了白色引擎和水晶工具的区别，称白色引擎仅用于《最终幻想XIII》，当引擎扩展支持多个游戏时易名为了水晶工具<ref name="GIff13int">{{cite web |url=http://www.gameinformer.com/b/features/archive/2010/03/10/the-making-of-final-fantasy-xiii.aspx |title=The Making of Final Fantasy XIII |publisher=''[[Game_Informer|Game Informer]]'' |date=2012-03-10 |accessdate=2013-02-11|language=en}}</ref>。

在《最终幻想XIII》发行后的2010年，北濑在一次采访中称，不谈水晶工具引擎对其它游戏开发的可能影响，其自身开发便是游戏创作的重要障碍。但是，他称这让游戏从PlayStation 3移植到Xbox 360变得非常容易，尽管让引擎在两个平台上运行的实际编程工作非常费劲<ref name="GIff13int" />。

== 特性 ==
* 先进的音频处理
* 实时物理计算
* 电影质量的特效渲染
* 逐行扫描加载
* 高品质、CGI实时渲染（包括过场动画和游戏）图形功能
* 过场动画至游戏无缝过渡

=== 引擎程序/功能 ===
水晶工具使用GRAPE2通信服务。

* 功能
** 角色查看器
** 效果编辑器
** 过场动画编辑器
** 声音制作器
** 布局工具
* 插件
** [[Adobe_Photoshop|Adobe Photoshop]]
** [[Autodesk_Softimage|Autodesk Softimage]]
** [[Autodesk_Maya|Autodesk Maya]]

=== 兼容性 ===
目前史克威尔艾尼克斯除子公司外，无意象与其它开发者共享引擎<ref name="joy" />。引擎可用来创作[[Microsoft_Windows|Microsoft Windows]]、[[索尼|索尼]][[PlayStation_3|PlayStation 3]]、[[微软|微软]][[Xbox_360|Xbox 360]]和[[任天堂|任天堂]][[Wii|Wii]]平台的游戏<ref name="ign" />。
