'''水韭屬'''（[[学名|学名]]：''{{lang|la|Isoëtes}}''），多年生[[挺水植物|挺水植物]]或[[沉水植物|沉水植物]]，高從5公分到15公分都有。[[葉子|葉子]]通常呈現叢生狀，暗色且繁衍用的[[孢子|孢子]]密生於葉基內側。一般來說，該種植物都生長在湖沼[[溼地|溼地]]。據記錄，全世界現存水韭約有[[70|70]]餘種。

==分類==
約140~150種，包括：
* ''I. alpina''
* ''I. andicola'' ([[synonymy|syn.]] ''Stylites andicola'')
* ''I. appalachiana''
* ''I. brochonii''
* ''I. durieui''
* ''I. echinospora''
* ''I. flaccida''
* ''I. gemmifera'' (syn. ''Stylites gemmifera'')
* ''I. histrix''
* ''I. lacustris''
* ''I. louisianensis''
* ''I. melanospora''
* ''I. nuttallii''
* ''I. taiwanensis'' （[[台灣水韭|台灣水韭]]）
* ''I. tegetiformans''
* ''I. tenella''
* ''I. tenuissima''
* ''I. sinensis'' （[[中华水韭|中华水韭]]，又名华水韭）
* ''I. yunguiensis'' （[[云贵水韭|云贵水韭]]）
* ''I. hypsophila'' （[[高寒水韭|高寒水韭]]）
