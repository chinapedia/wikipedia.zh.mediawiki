《'''水瓶宮'''》（{{lang-en|'''''Aquarius'''''}}，又名《'''罪惡的靈魂'''》）是一部美國[[時代劇|時代]]犯罪電視劇，由{{link-en|約翰·麥克納馬拉|John McNamara (writer)}}創作。本劇於2015年5月28日播放，2015年6月25日續訂第二季，在2016年6月16日播放。<ref>{{Cite web |url = http://deadline.com/2016/04/aquarius-season-2-commercial-free-premiere-nbc-1201740341/ |title = 'Aquarius' Season 2 Gets Two-Hour Ad-Free Launch On NBC |author = Erik Pedersen |date = 2016-04-19 |publisher = Deadline Hollywood |language = en}}</ref>

2016年10月1日，NBC於兩季後取消本劇。<ref>{{Cite web |url = http://variety.com/2016/tv/news/aquarius-canceled-nbc-david-duchovny-charles-manson-1201875658/ |title = NBC Cancels 'Aquarius' After Two Seasons |author = Seth Kelley |date = 2016-10-01 |publisher = Variety |language = en}}</ref>

本劇開始於1967年的[[洛杉磯|洛杉磯]]，故事靈感來自真實人物和事件，但也加上虛構角色和故事單元。[[大衛·杜楚尼|大衛·杜楚尼]]扮演Sam Hodiak，一名負責調查少女失蹤案的虛構洛杉磯警長。[[查爾斯·曼森|查爾斯·曼森]]（[[格辛·安東尼|格辛·安東尼]] 飾演）原本只是一個不起眼的小罪犯，後來竟演變成震驚全國的邪教組織領袖，無數女性不僅對他投懷送抱，還幫助他犯下累累罪行。這名警長和他年輕的搭檔將以臥底身份進入[[曼森家族|曼森家族]]，發現該名失蹤少女Emma Karn（{{link-en|愛瑪·杜蒙|Emma Dumont}} 飾演）與曼森及他的家族在一起。第一季以家族的興起為中心，第二季則集中在莎朗·蒂謀殺案和Helter Skelter。
