'''吡啶氯铬酸盐'''，分子式C<sub>5</sub>H<sub>4</sub>N·HCrO<sub>3</sub>Cl，简称PCC，即'''p'''yridinium '''c'''hloro'''c'''hromate的缩写。一种温和的[[氧化剂|氧化剂]]，橙黄色[[晶体|晶体]]。用来把[[醇|醇]]有限度地氧化为[[醛|醛]]，使用PCC的缺點為反應時間太過冗長和PCC具有毒性。

由[[E.J.Corey|E.J.Corey]]发明。试剂本身呈弱[[酸|酸]]性，所以常跟缓冲剂比如[[乙酸钠|乙酸钠]]配合使用。

== 製備與結構 ==
最初始的製備方法是個偶然的發現，它由吡啶加入冷的[[三氧化铬|三氧化铬]]浓[[盐酸|盐酸]]溶液制备：<ref>Corey, E. J.; Suggs, J. W. (1975). "Pyridinium Chlorochromate. An Efficient Reagent for Oxidation of Primary and Secondary Alcohols to Carbonyl Compounds".Tetrahedron Lett. 16 (31): 2647–2650.doi:10.1016/S0040-4039(00)75204-X.</ref>
: C<sub>5</sub>H<sub>5</sub>N + HCl + CrO<sub>3</sub> → [C<sub>5</sub>H<sub>5</sub>NH][CrO<sub>3</sub>Cl]
這個化合物由吡啶陽離子([C<sub>5</sub>H<sub>5</sub>NH<sup>+</sup>])和結構為四面體的三氯化鉻陰離子(CrO<sub>3</sub>Cl<sup>-</sup>)所組成。實驗可得之相關鹽類，例如：1-丁基吡啶三氧化鉻([C<sub>5</sub>H<sub>5</sub>N(C<sub>4</sub>H<sub>9</sub>)]<sup>+</sup>[CrO<sub>3</sub>Cl]<sup>−</sup>)。

== 性质及應用 ==
=== 醇的氧化 ===
PCC為一個氧化劑，特別在在氧化一級醇和二級醇成[[醛|醛]]和[[酮|酮]]的反應中具有高效率。不像相關的瓊斯試劑，PCC鮮少有過氧化的發生而形成[[羧酸|羧酸]]。一個典型的PPC氧化反應為加入[[醇|醇]]於[[二氯甲烷|二氯甲烷]]中的PCC懸浮夜，簡單的反應機制：
: C<sub>5</sub>H<sub>5</sub>NHCrO<sub>3</sub>Cl + R2CHOH → C<sub>5</sub>H<sub>5</sub>NHCl + H<sub>2</sub>CrO<sub>3</sub> + R<sub>2</sub>C=O
在實際操作過程中，[[鉻|鉻]]的副產物會與吡啶一起沉澱。可加入惰性吸附劑，如[[分子篩|分子篩]]或[[矽膠|矽膠]]，使得黏稠的副產物吸附至表面並更好作用。
=== 其他反應 ===
除了簡單的[[羥基|羥基]]氧化反應外，[[重排反應|重排反應]]也是可行的。例如：三級醇不能直接被氧化，但在Babler 氧化反應中，鉻酸酯的形成與PCC和[[烯丙基|烯丙基]]的三級醇可在碳基成型的氧化步驟前，透過 [3,3]-[[σ遷移反應|σ遷移反應]]來進行[[異構化|異構化]]。其他常見的[[氧化劑|氧化劑]]通常可促使簡單的[[脫水|脫水]]反應，而不能在三級羥基為中心的狀況下，發生任何氧化反應。

另一個PCC的氧化反應為不飽和的醇或醛轉變成環己烯酮的反應。這個方法為氧化陽離子的環化反應，將(-)-香茅醇轉換為(-)-胡薄荷酮。PCC也會影響[[烯丙基|烯丙基]]的氧化反應，如二氫呋喃(Dihydrofuran)轉變成[[內酯|內酯]]。
== 相關試劑 ==
其他可用於醇氧化反應之更方便且少毒性試劑包含[[二甲基亞碸|二甲基亞碸]]為底的氧化反應(如[[斯文氧化反應|斯文氧化反應]]、[[普菲茨納-莫法特氧化反應|普菲茨納-莫法特氧化反應]])和以高價碘為底的氧化反應(如戴斯－馬丁氧化劑)。
