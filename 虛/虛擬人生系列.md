《'''虛擬人生'''》系列是一個[[大富翁|大富翁]]形式的[[電子遊戲|電子遊戲]]系列。原先為英文版，經[[明日工作室|明日工作室]]洽談後同意合作製作。內容近似[[大宇資訊|大宇資訊]]的《[[大富翁系列|大富翁]]》系列，第一代由[[岡業資訊|岡業資訊]]合作製作。第二、三代由明日工作室獨立製作，岡業資訊協助[[特效|特效]]。第四代，明日工作室因財務問題，將全權交與[[娛樂通|娛樂通]]，並於2009年[[英特衛多媒體|英特衛多媒體]]代理上市。之後又接續出了《新虛擬人生》，但社會評價已每況愈下。

== 內容 ==

為[[棋盤|棋盤]]式骰[[骰子|骰子]]遊戲，可擔任不同角色。過程不僅要與[[怪物|怪物]]對抗，還需要善用[[卡片|卡片]]、[[道具|道具]]。角色的[[體力|體力]]、[[愛情|愛情]]度、[[金錢|金錢]]等，皆是玩家需注意的地方。其實就是真實人生的縮影。過程中[[學歷|學歷]]是很重要的，[[證照|證照]]、[[職業|職業]]、[[技能|技能]]也會影響角色未來命運。

=== 遊戲特色 ===

; 人生轉職系統
: 人們到了一定步數後即可以轉職，而轉職後會有不同技能。<ref>[http://acg.gamer.com.tw/acgDetail.php?s=3064 虛擬人生1]{{zh-tw}}</ref>
; 星座系統
: 為第四代後([[娛樂通|娛樂通]])接續作品所擁有，每個星座都有獨特的戰鬥與地圖技能，已學習的技能玩家隨時能加以裝備。<ref>[http://acg.gamer.com.tw/acgDetail.php?s=30103 虛擬人生4]{{zh-tw}}</ref><ref>[http://www.gamebase.com.tw/news/topic/87040572/ 百萬銷量經典系列 國產名作《虛擬人生4》即日歡樂上市]{{zh-tw}}</ref>
; 卡片戰鬥
: 由魔法，卡片，武器三種卡片組成，並不能單以蠻力取勝。
; 小遊戲
: 此也為虛擬人生最大特色。永遠不知道自己會遇到什麼突發事情。
; 任務系統
: 接手任務，可以獲得報酬。

=== 差異 ===

第三代以前仍是大致保留原始遊戲介面，但後代將其[[畫風|畫風]]改變，在其遊戲上設計為適合小朋友的程度，最重要的職業系統被星座系統取代。
以往的事件格會影響主角未來發展，但在後代並沒特殊影響。連戰鬥打輸後就直接遊戲結束，引發前代玩家不滿。
