'''埃斯内托·佩雷斯·巴利亚达雷斯·冈萨雷斯-雷维利亚'''（'''Ernesto Pérez Balladares González-Revilla'''，{{bd|1946年|6月29日||}}） 外号“公牛”(El Toro)，<ref name=NYT1>{{cite web |url=http://www.nytimes.com/1994/02/21/world/panama-journal-democracy-at-work-under-shadow-of-dictators.html?pagewanted=all&src=pm |title=Panama Journal; Democracy at Work, Under Shadow of Dictators |author=Howard W. French |date=February 21, 1994 |work=The New York Times |archivedate=September 2, 2012|deadurl=no |archiveurl=http://www.webcitation.org/6APOVeUGM |accessdate=September 2, 2012}}</ref>1994年—1999年担任[[巴拿马总统|巴拿马总统]]。

佩雷斯·巴利亚达雷斯在美国接受教育，在美国[[圣母大学|圣母大学]]和[[宾夕法尼亚大学|宾夕法尼亚大学]]沃顿商学院获得硕士学位。<ref name=NYT2>{{cite web |url=http://www.nytimes.com/1994/07/21/world/washington-talk-a-panama-enemy-becomes-an-ally.html |title=Washington Talk; A Panama Enemy Becomes an Ally |author=Eric Schmitt |date=July 21, 1994 |work=The New York Times |archivedate=2012-09-03 |deadurl=no |archiveurl=https://www.webcitation.org/6APPdSa3B?url=http://www.nytimes.com/1994/07/21/world/washington-talk-a-panama-enemy-becomes-an-ally.html |accessdate=September 2, 2012 }}</ref><ref name=WP1>{{cite web |url=http://www.highbeam.com/doc/1P2-889827.html |title=Panamanians Vote in Peace, Picking Ex-Aide of Noriega; Millionaire Perez Balladares Bests Widow of Four-Time President |author=Douglas Farah |date=May 9, 1994 |work=The Washington Post |archivedate=2015-03-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20150329104016/http://www.highbeam.com/doc/1P2-889827.html |accessdate=September 2, 2012 }}</ref>1971年到1975年他在[[花旗银行|城市银行]]巴拿马和中美洲分部任信贷官员。后成为巴拿马军方领袖[[奥马尔·托里霍斯|奥马尔·托里霍斯]]的经济和财政部长；<ref>{{cite web |url=http://www.highbeam.com/doc/1P2-8052246.html |title=Noriega Builds Statute |author=Phillip Bennett |date=March 6, 1988 |work=Boston Globe |publisher={{Subscription required|via=[[HighBeam_Research|HighBeam Research]]}} |accessdate=September 4, 2012 |archive-url=https://web.archive.org/web/20150329104018/http://www.highbeam.com/doc/1P2-8052246.html |archive-date=2015-03-29 |dead-url=yes }}</ref>1979年3月参与创建[[巴拿马民主革命党|民主革命党]]（PRD），1984年与新军方领袖[[曼努埃尔·诺列加|曼努埃尔·诺列加]]发生冲突，在西班牙流亡几个月。<ref>{{cite web |url=http://www.highbeam.com/doc/1G1-15255997.html |title=The price of democracy: with an eye to the north |author=Karl Bermann |date=May 6, 1994 |work=Commonweal |publisher={{Subscription required|via=[[HighBeam_Research|HighBeam Research]]}} |accessdate=September 4, 2012 |archive-url=https://web.archive.org/web/20150329104014/http://www.highbeam.com/doc/1G1-15255997.html |archive-date=2015-03-29 |dead-url=yes }}</ref>1989年，他担任亲[[曼努埃尔·诺列加|曼努埃尔·诺列加]]的总统候选人[[卡洛斯·杜克|卡洛斯·杜克]]的竞选经理，同年[[美国入侵巴拿马|美国入侵巴拿马]]推翻诺列加后，佩雷斯·巴利亚达雷斯一度被美军拘留和审讯，后来释放。<ref name=NYT2/>1994年他作为[[巴拿马民主革命党|民主革命党]]（PRD）候选人与[[阿努尔福党|阿努尔福党]]候选人[[米雷娅·莫斯科索|米雷娅·莫斯科索]]和萨尔萨舞歌手[[鲁本·布拉德斯|鲁本·布拉德斯]]竞选总统，最终以33%的选票赢得了选举，9月1日正式就职。

任内他进行自由市场改革和政府服务私有化，加强与美国的亲密关系。1998年宪法公投失败后，次年他竞选连任失败，总统职务由[[米雷娅·莫斯科索|米雷娅·莫斯科索]]接任。

2009年9月，检察机关对他在担任总统期间的腐败指控展开调查，12月28日他被逮捕，并于2010年10月被指控[[洗钱|洗钱]]。<ref name=NP>{{cite web |url=http://www.newsroompanama.com/panama/1098-a-bank-account-of-ex-president-frozen.html |title=A bank account of ex-president frozen |date=April 18, 2010 |publisher=Newsroom Panama |archivedate=2012-09-06 |deadurl=yes |archiveurl=https://www.webcitation.org/6ATJcNQ3f?url=http://www.newsroompanama.com/panama/1098-a-bank-account-of-ex-president-frozen.html |accessdate=5 September 2012 }}</ref>2011年4月最高法院法官驳回对他的指控。2012年2月，佩雷斯·巴利亚达雷斯因诽谤审计官Alvin Weeden被处以3000美元的罚款或一年监禁。<ref name="abc.com.py">{{cite web |url=http://www.abc.com.py/internacionales/ex-presidente-panameno-perez-balladares-es-condenado-por-calumnia-367554.html |title=Ex presidente panameño Pérez Balladares es condenado por calumnia |language=es |agency=Reuters |date=February 17, 2012 |publisher=ABC |accessdate=27 December 2012 |archivedate=2012-12-28 |archiveurl=https://www.webcitation.org/6DEokxCnh?url=http://www.abc.com.py/internacionales/ex-presidente-panameno-perez-balladares-es-condenado-por-calumnia-367554.html |deadurl=no }}</ref>

巴利亚达雷斯妻子朵拉·博伊德(Dora Boyd de Pérez Balladares).<ref>{{cite web |url=http://www.estudio1panama.com/?p=38365 |title=Dora Boyd de Pérez Balladares acudirá este lunes nuevamente a la Fiscalía |date=December 24, 2010 |publisher=Estudio 1 |archivedate=2012-09-04 |deadurl=yes |archiveurl=https://www.webcitation.org/6APvHKIcS?url=http://www.estudio1panama.com/?p=38365 |accessdate=September 2, 2012 |language=es }}</ref>
