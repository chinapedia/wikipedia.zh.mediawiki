'''埃及眼鏡蛇'''（學名：''Naja haje''，[[英語|英語]]：''Egyptian Cobra''），是[[爬蟲綱|爬蟲綱]][[有鱗目|有鱗目]][[眼鏡蛇科|眼鏡蛇科]][[眼鏡蛇|眼鏡蛇屬]]的生物。具毒性，主要分布於[[北非|北非]]及[[中東|中東]]地帶，活動於[[撒哈拉沙漠|撒哈拉沙漠]]及[[敘利亞|敘利亞]][[沙漠|沙漠]]附近。埃及眼鏡蛇是[[非洲|非洲]]最常見的眼鏡蛇，而且經常襲擊[[人類|人類]]，在當地製造了不少死亡事件，是[[非洲|非洲]]中第二大的眼鏡蛇，僅次於[[森林眼鏡蛇|森林眼鏡蛇]]。

== 形態與習性 ==
成型的埃及眼鏡蛇，身長約有1.5至2[[米_(單位)|米]]，最大長度可達約3米。埃及眼鏡蛇最顯著的特徵是碩大的頭部及頸部，以及寬闊的[[喙|喙]]部。[[眼睛|眼睛]]以蛇類而言偏大，而且有著圓型的[[瞳孔|瞳孔]]。頸部的皮摺膨起時可以寬至15至18厘米。

埃及眼鏡蛇主要棲息在較為乾燥而且備有少量水源及植被的[[熱帶草原|熱帶草原]]或半[[沙漠|沙漠]]地區，實際上鮮少出沒於沙漠地帶上。

有時也會在[[綠洲|綠洲]]、[[耕地|耕地]]、有零落植被的[[山|山群]]，與及一般草地。而最危險的情況是，埃及眼鏡蛇亦經常會進入有人類居住的[[房屋|房屋]]之中，而牠們通常都是被人類所食用的[[雞|雞]]隻，或被人類[[廚餘|廚餘]]招來的[[鼠|鼠]]群所吸引的。另外，亦有不少埃及眼鏡蛇會在[[地中海|地中海]]內潛游，由此可知牠們有一定的好水性。

埃及眼鏡蛇屬[[夜行性|夜行性]]的陸上動物，牠們喜歡陰暗的環境，但偶爾也會在清晨的時候發現牠們在享受[[太陽|陽光]]的沐浴。牠們會居住於其它動物所挖掘的[[洞穴|洞穴]]、[[白蟻|白蟻]]丘或山岩之間等地方，有時甚至會闖進人類的生活環境中潛伏，以便獵食各類[[家禽|家禽]]。跟一般眼鏡蛇一樣，當埃及眼鏡蛇進行獵食或感應到危機時，能瞬即逃遁數米之遙，又或支起前半身，並展開頭頸兩側的皮摺以威嚇對手；但當牠們遇到比自己強的對手，感到力有不逮時，就會作出假死的舉動。

== 飲食及繁殖 ==
埃及眼鏡蛇主要進食各種小型的動物、[[蜥蜴|蜥蜴]]類、[[蟾蜍|蟾蜍]]，甚或其它蛇類，包括眼鏡蛇。雌性的埃及眼鏡蛇能生產8至33枚蛇卵，生產地通常都在[[蟻|蟻]]丘，而經過六十天的孵化期後，幼
蛇便會破殼而出。

== 毒性 ==
埃及眼鏡蛇每次咬擊，平均能釋出175至200[[毫克|毫克]]的毒素，其毒之猛僅次於[[菲律賓眼鏡蛇|菲律賓眼鏡蛇]]（''Philippine Cobra'')。可是，埃及眼鏡蛇的致命性，卻被普遍認為高於上述的眼鏡蛇，原因是埃及眼鏡蛇的體型較大、較具攻擊性與及每次咬擊能釋出較大量的蛇毒。

埃及眼鏡蛇的蛇毒具有[[神經|神經]]毒素，主要影響生物體的[[神經系統|神經系統]]，強制中止神經訊息輸送至[[肌肉|肌肉]]，最終連[[心臟|心臟]]及[[肺部|肺部]]等重要器官都會喪失接收神經訊號的機能，令[[呼吸|呼吸]]作用全盤癱瘓，導致死亡。被注入蛇毒後，被咬部位會即時出現疼痛、腫大、發現硬塊、起泡、肌群壞死等表徵，而隨著中毒深淺之別，其它各樣可能出現的表徵還包括頭痛、嘔吐、腹痛、腹瀉、暈眩、昏厥或痙攣等。

== 相關文化 ==
埃及眼鏡蛇的其中一個典型形象，是作為埃及蛇形女神[[瓦吉特|瓦吉特]]的代表物，亦是古[[法老王|法老王]]支配國的權威標誌，因此亦被稱為「神聖的毒蛇」（''Uraeus serpent''）。

在很多古代的文獻中，都紀錄[[古埃及|古埃及]][[托勒密王朝|托勒密王朝]]的著名女-{后}-[[克利奧帕特拉七世|克利奧帕特拉七世]]（亦稱「-{[[埃及艷后|埃及艷后]]}-」）曾操毒蛇咬傷自己以[[自殺|自殺]]，文獻中提到的那條毒蛇，名為「aspis」。[[希臘|希臘]][[作家|作家]][[普魯塔克|普魯塔克]]指出，埃及艷-{后}-曾經在[[死刑|死刑]]囚犯身上做過多次的蛇毒實驗，最終證實「aspis」的毒素在眾多致命毒物中，能帶給中毒者最小程度的痛苦<ref>[[普魯塔克|普魯塔克]]：〈Parallel Lives〉《Life of Antony》。</ref> 。而現代研究中，普遍認同當時的「aspis」，所指的就是埃及眼鏡蛇。
