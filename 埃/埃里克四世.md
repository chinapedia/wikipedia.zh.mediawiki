'''埃里克四世'''（{{lang-da| '''Erik Plovpenning'''}}，又名"犁地金王埃里克"；{{bd|約1216年||1250年|8月10日|catIdx=E}}），[[丹麥國王|丹麥國王]]（1241年－1250年在位）。丹麥國王勝利王[[瓦爾德馬二世|瓦爾德馬二世]]與第二任王后[[葡萄牙的貝倫加利亞|葡萄牙的貝倫加利亞]]（葡萄牙國王[[桑喬一世_(葡萄牙)|桑喬一世]]之女）的長子、{{le|瓦爾德馬 (年輕的)|Valdemar the Young|年輕的瓦爾德馬}}的同父異母弟、[[艾貝爾|艾貝爾]]及[[克里斯托弗一世|克里斯托弗一世]]的兄長。

== 生平 ==
1218年，當埃里克四世的兄長年輕的瓦爾德馬被封為共治國王時，埃里克四世亦受封為石勒蘇益格公爵。1231年，當年輕的瓦爾德馬突然英年早逝，埃里克四世於1232年5月30日在[[隆德主教座堂|隆德主教座堂]]加冕為共治國王及繼承人，他其後將石勒蘇益格公爵的爵位讓給弟弟艾貝爾。1241年，父親瓦爾德馬二世逝世，由埃里克四世繼承王位。埃里克四世的統治是痛苦及需要進行內戰，石勒蘇益格公爵艾貝爾受到[[荷爾斯泰因|荷爾斯泰因]]伯爵們的支持企圖獨立，斯堪尼的農民亦因為重稅及[[犁|犁]]地金而爆發暴亂，所以埃里克四世的稱號是"犁地金王"。

== 即位 ==
1242年，埃里克四世即位的翌年，埃里克四世便與石勒蘇益格公爵艾貝爾發生衝突，兩人的衝突直至1244年，共同計劃向[[愛沙尼亞|愛沙尼亞]]發動[[十字軍東征|十字軍東征]]為止<ref name="Grave">{{cite web |url=http://www.findagrave.com/cgi-bin/fg.cgi?page=gr&GRid=96942494|title= Erik Plovpenning|website=www.findagrave.com }}</ref>。期間埃里克四世亦受到教會要求豁免稅收而煩惱，1245年，教皇更派遣[[教廷大使|教廷大使]]向國王及[[奧登斯|奧登斯]]的主教們進行調停，更以進行[[絕罰|絕罰]]作為威脅。埃里克四世只好遷怒於[[羅斯基勒主教座堂|羅斯基勒主教]]。羅斯基勒主教尼爾斯出逃，埃里克四世逐沒收包括[[西蘭島|西蘭島]]與及新興城市[[哥本哈根|哥本哈根]]等地的教會財產，以作為補償他與艾貝爾的煩惱。教皇[[英諾森四世|英諾森四世]]主張主教和所屬教區的回復職務，但是爭議未能得到解決。尼爾斯於1249年死在[[克萊爾沃修道院|克萊爾沃修道院]]，而教區的財產並沒有回歸，直到1250埃里克四世去世後。

1246年，埃里克四世企圖恢復父親瓦爾德馬二世在位時失去的土地，他率領丹麥軍隊入侵[[荷爾斯泰因|荷爾斯泰因]]。而弟弟艾貝爾卻是前任荷爾斯泰因伯爵[[阿道夫四世|阿道夫四世]]的女婿，也是現任的兩位荷爾斯泰因伯爵的姊夫及監護人，艾貝爾希望逼使兄長放棄此次入侵。翌年，艾貝爾在兩位弟弟[[洛蘭島|洛蘭島]]及[[法爾斯特島|法爾斯特島]]總督[[克里斯托弗一世|克里斯托弗]]及[[布萊金厄公爵克努特|布萊金厄公爵克努特]]與及[[漢薩同盟|漢薩同盟]]城市[[呂貝克|呂貝克]]支持下入侵[[日德蘭|日德蘭]]及[[菲因島|菲因島]]包括搜掠及焚燒[[蘭訥斯|蘭訥斯]]及[[奧登斯|奧登斯]]。同年，埃里克四世立即報復，重新佔據[[里伯|里伯]]和佔領艾貝爾世襲的城市[[斯文堡|斯文堡]]。1247年，他奪取在[[菲英島|菲英島]]上的阿勒斯考城堡，以及將克里斯托弗和克努特拿下成為囚犯。最後由他們的姊妹[[勃蘭登堡藩侯|勃蘭登堡藩侯]]夫人[[勃蘭登堡的蘇菲亞|勃蘭登堡的蘇菲亞]]（[[約翰一世_(勃蘭登堡)|約翰一世]]的妻子）調停下達成和議，丹麥仍然牢牢地由埃里克四世統治<ref name="Grave"/>。

1249年，斯堪尼的農民因為犁地金稅收而爆發暴亂，雖然最後在西蘭島的軍隊幫助下平定了，但是教會、艾貝爾及南部的德意志諸侯已連成一線，共同對抗國王埃里克四世。

== 弒君 ==
1249年，埃里克四世籌集軍隊東征，前去鞏固在愛沙尼亞的領土。當埃里克四世於1250年回程時，他帶領軍隊到達[[倫茨堡|倫茨堡]]，避免德意志諸侯攻陷及讓他們知道埃里克四世仍是丹麥國王。艾貝爾在[[石勒蘇益格|石勒蘇益格]]的[[戈托爾夫城堡|戈托爾夫城堡]]為埃里克四世提供住宿。當二人坐下來在大廳談話時，艾貝爾提醒埃里克四世在即位初年曾攻打此地，對他說：「你可曾記得在幾年前你的人在蹂躪此城市，我的女兒為了逃命連鞋也沒穿上。」埃里克四世回答說：「足夠了，我會送她一雙鞋子。」傍晚，埃里克四世與其中一名德意志的武士賭博，石勒蘇益格公爵的首相[[萊夫·吉蒙臣|萊夫·吉蒙臣]]帶領着一群人士衝進來並俘虜國王。他們將他捆住並且拖出公爵的房子，下了一條船，撐船出[[施萊灣|施萊灣]]。跟着有第二艘船接應。當埃里克四世聽到他的死敵萊夫·吉蒙臣的聲音，他意識到他將會被殺害。埃里克四世要求讓牧師聽到他最後一次懺悔，那些同謀者同意了他的請求。埃里克四世被送回岸上，牧師被帶到聽取埃里克四世的懺悔，然後他又被送回到了海灣。其中一個綁架者的用斧頭擊殺了埃里克四世。埃里克四世被斬首，他的身體被放入施萊灣。第二天凌晨，兩點漁民的用網將國王無頭的屍體打撈起來。他們將屍體送到在石勒蘇益格的[[道明會|道明會]][[修道院|修道院]]<ref name="Grave"/>。他的屍體於1258年被其弟克里斯托弗一世轉移到聖本特教堂<ref name="Gyldendals Åbne">{{cite web |url=http://denstoredanske.dk/Dansk_Biografisk_Leksikon/Monarki_og_adel/Konge/Erik_4._Plovpenning|title= Erik 4. Plovpenning|website=Gyldendals Åbne }}</ref>。

埃里克四世死後沒有男嗣，其弟艾貝爾繼承王位。艾貝爾在繼承王位前與24名貴族發出著名的誓言（{{lang-da| ''dobbelt tolvter-ed''}}）來澄清與弒殺埃里克四世無關<ref name="America Pink">{{cite web |url=http://america.pink/abel-king-denmark_225462.html|title= Abel, King of Denmark: King|website=america.pink}}</ref>。
