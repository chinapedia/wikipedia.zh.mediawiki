'''埃兹奎尔·拉维齐'''（'''Ezequiel Lavezzi'''，{{bd|1985年|5月3日|catidx=Lavezzi, Ezequiel}}），是一名[[阿根廷|阿根廷]]足球运动员，司职前锋，目前效力于[[中超|中超]]的[[河北華夏幸福|河北華夏幸福]]。

2007年4月，尚未满22岁的拉维齐就代表[[阿根廷国家足球队|阿根廷国家队]]出战。同年7月，拉维齐加盟刚刚升回[[意甲|意甲]]的[[那不勒斯足球俱乐部|那不勒斯]]，转会费大致600万欧元。

2012年7月，[[法甲|法甲]][[巴黎聖日耳門|巴黎聖日耳門]]官方網站宣佈，他們與拉维齐簽約，合約期到2016年夏天屆滿。

2016年2月，中超升班马[[河北华夏幸福足球俱乐部|河北华夏幸福足球俱乐部]]官方宣布，他们将与拉维奇签订为期两年的合约，拉维奇在合同期间的薪水预计将会达到2350万英镑<ref>{{Cite web |url = http://www.skysports.com/football/news/11095/10170748/psg-forward-ezequiel-lazezzi-makes-move-to-chinese-football |title = 巴黎前锋拉维奇转会中超  |date = 17/02/16 |publisher = 天空体育 |language = 英语 |accessdate = 14/05/17 }}</ref>。

==种族歧视风波==

2017年5月中，英国媒体记者告诉中国球迷，中超公司发布的河北华夏官方照片里，拉维奇做出了疑似歧视亚裔的[[种族歧视|种族歧视]]动作——双手往外拉动眼角<ref>{{Cite web |url=http://slide.hebei.sina.com.cn/slide_44_86246_459436.html#p=9 |title = 河北华夏幸福2017年创意写真 | date = 20/03/17 | accessdate = 14/05/17 |language = 中文}}</ref>，在中国球迷之间引起巨大风波，拉维奇和河北华夏俱乐部公开道歉并表示动作没有恶意。
