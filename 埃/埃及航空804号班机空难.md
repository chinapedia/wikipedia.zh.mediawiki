'''埃及航空804號班機'''（MS804/MSR804）是埃及航空的定期國際航線班機，是法國巴黎至埃及開羅的航班代號。2016年5月18日執飛的航班在[[欧洲东部时间|歐洲東部時間]]2016年5月19日2時45分從雷達上消失。當時，該航班已進入埃及空域約130公里處，飛行高度為37,000英呎（約11,000公尺）。隨後埃及民航局官員稱目前已經有理由相信在地中海失去聯絡的飛機已經在[[希腊|希臘]][[喀帕苏斯岛|喀帕蘇斯島]]附近墜毀<ref>{{Cite web |url = http://www.bbc.com/news/world-middle-east-36309492 |title = EgyptAir flight MS804 from Paris to Cairo crashed - Hollande | date = 2016-05-19 |publisher = [[BBC|BBC]] |accessdate = 2016-05-19}}</ref><ref>{{Cite web |url = http://www.independent.co.uk/news/world/europe/egyptair-flight-ms804-missing-passenger-jet-has-crashed-into-the-mediterranean-sea-officials-confirm-a7037146.html |title = EgyptAir flight MS804 crash: Military searching for wreckage in Mediterranean Sea, officials say |author = Lizzie Dearden |date = 2016-05-19 |publisher = The Independent |accessdate = 2016-05-19}}</ref><ref>{{Cite web |url = http://www.mirror.co.uk/news/world-news/egyptair-flight-ms804-missing-live-8003059 |title = Egyptair flight MS804 missing: live updates as plane crash with 66 on board 'almost certainly caused by attack' |author = Natalie Evans |date = 2016-05-19 |publisher = [[每日鏡報|每日鏡報網]] |accessdate = 2016-05-19|language = en-gb}}</ref>。

== 失事班机情况 ==
涉事飛機的型號為[[空中巴士A320系列|空中巴士A320-232]]，[[機身編號|機身編號]]為SU-GCC，使用IAE V2527-A5引擎，於2003年7月25日進行試飛並於同年11月3日交由埃及航空營運<ref>{{cite web|url=http://www.airfleets.net/ficheapp/plane-a320-2088.htm |title=EgyptAir SU-GCC |publisher=Airfleets.net |accessdate = 2016-05-19}}</ref>。截止至事故发生前，该客机机龄12年，飛行時數為6275小时。失事当天之前该飞机已执行五次航班飞行任务，分別飛到三個曾經受到恐怖攻擊或曾受到恐怖攻擊威脅的城市，第一趟是从[[厄立特里亚|厄立特里亚]][[阿斯马拉国际机场|阿斯马拉国际机场]]，飞往开罗；第二趟是从开罗飞往[[突尼西亞-迦太基國際機場|突尼西亞-迦太基國際機場]]，然后执行回程航班任務；第四次則是由开罗飞往巴黎。

== 乘客和机组 ==
804号班机机组人员尚不明确，已知的訊息包括：[[机长|机长]]有6275小时飛行經驗，其中A320为2101小时。[[副驾驶|副驾驶]]也有2766小时的飛行經驗。其他机组成员包括三名安全员与五名其他机组成员。乘客一共有56名，其中包括一名小孩与两名婴儿。<ref>{{Cite web |url = http://www.independent.co.uk/news/world/europe/egyptair-flight-ms804-missing-passenger-jet-has-crashed-into-the-mediterranean-sea-officials-confirm-a7037146.html |title = EgyptAir flight MS804 crash: Military searching for wreckage in Mediterranean Sea, officials say |author = Lizzie Dearden |date = 2016-05-19 |publisher = [[英國獨立報|英國獨立報]] |accessdate = 2016-05-19}}</ref>

===机组人员===
机上有10名机组人员，其中3名是机上安全员，5名是乘务员，另有两名飞行员<ref name="newyorktimes">{{cite news|last1=Walsh|first1=Declan|title=EgyptAir Plane Disappears Over Mediterranean, Airline Says|url=http://www.nytimes.com/2016/05/19/world/middleeast/egyptair-plane-disappears-over-mediterranean-airline-says.html?_r=0|accessdate=2016-05-19|work=[[紐約時報|紐約時報]]|date=2016-05-19|language=en-us}}</ref>。据埃及航空的官方数据，執飛當次航班的機長有6,275个小时的飞行经验，其中驾驶A320机型的有2,101个小时，而副機師也有4,000小時的經驗。

== 经过 ==
这班[[空客A320|空客A320-232R]]客机（注册编号：SU-GCC）原定于2016年5月18日20时45分在[[巴黎夏尔·戴高乐机场|巴黎夏尔·戴高乐机场]]起飞，并于第二日03时05分到达[[开罗国际机场|开罗国际机场]]<ref name ="CNN1">{{Cite web |url = http://edition.cnn.com/2016/05/18/middleeast/egyptair-flight-disappears/ |title = EgyptAir Flight 804 disappears |author = Faith Karimi and Hamdi Alkhshali |date = 2016-05-19 |publisher = [[CNN|CNN]] |accessdate = 2016-05-19}}</ref>。实际在当晚09时11分离港，并于次日02时45分从[[雷达|雷达]]上消失<ref name ="CNN1"/>，飞机最后一次与[[控制塔|控制塔]]联络仅在十分钟之前。但飞机并未按照规定在进入埃及领空时向[[控制塔|控制塔]]汇报<ref>{{Cite web |url=http://edition.cnn.com/2016/05/19/middleeast/egyptair-flight-804-liveblog/ |title = EgyptAir missing plane MS804: Live updates |author1 = Nick Thompson |author2 = James Griffiths |author3 = Tiffany Ap |date = 2016-05-19 |publisher = [[CNN|CNN]] |accessdate = 2016-05-19}}</ref>。该飞机回传的最后一次[[ADS-B|ADS-B]]数据如下：
高度：11277.6公尺，速度：976.004公里/小时，航迹角：119度，经纬度：28.297447967517，33.959762573242，当地时间：2016/05/19 上午08:27:06，更新时间：2016/05/19 上午08:28:42。<ref>{{Cite web |url = https://www.flightradar24.com/data/aircraft/su-gcc |title = Flight history for aircraft - SU-GCC |date = 2016-05-19 |publisher = [[FlightRadar24|FlightRadar24]] |accessdate = 2016-05-19}}</ref>

该机预定在3:05着陆。埃及军方最初说，他们在4:26，即飞机从雷达上消失2小时后，收到了求救訊号。但随后他们的官员又否认了这个说法。<ref>{{cite web|title=EgyptAir flight MS804 disappears from radar between Paris and Cairo – live updates|url=http://www.theguardian.com/world/live/2016/may/19/egyptair-plane-cairo-paris-live-updates?page=with:block-573dbb67e4b04a0378340fec#block-573dbb67e4b04a0378340fec|work=The Guardian|date=2016-05-19|accessdate=2016-05-19}}</ref>

[[希腊国防部|希腊国防部]]部长曾宣称，根据希腊军方的雷达数据，804号班机在进入埃及[[飞行情报区|飞行情报区]]后不久就转弯了。在高度37000英尺处，飞机先后向左转90度并右转360度。在此期间，飞机高度从37000英尺迅速下降至15000英尺。当飞机高度降至10000英尺后，飞机就从雷达上消失了。<ref>{{cite web |title=Π. Καμμένος: Στα 10.000 πόδια χάθηκε η εικόνα του airbus - Συνεχίζονται οι έρευνες |url=https://www.youtube.com/watch?v=nKDQyPT3dL0 |publisher=YouTube |language=el |trans_title=P. Kammenos: At 10,000 feet [was] lost the image of the Airbus - Ongoing investigation |date=2016-05-19 |accessdate=2016-05-19}}</ref><ref>{{cite news|url=http://www.independent.co.uk/news/world/europe/egyptair-flight-ms804-crash-plane-swerved-suddenly-before-dropping-off-radar-over-mediterranean-sea-a7037516.html|title=EgyptAir flight MS804 crash: Plane 'swerved' suddenly before dropping off radar over Mediterranean Sea|date=2016-05-19|work=独立报|accessdate=2016-05-19}}</ref>5月23日，一名来自埃及国家空中导航服务公司（National Air Navigation Services Company）的埃及民航局官员又否认了这一说法。他同时宣称，该机在坠毁前并没有改变高度，也没有做出反常动作。<ref>{{cite news|url=http://www.bbc.com/news/world-middle-east-36365256|title=EgyptAir: Crashed flight MS804 'did not swerve'|publisher=[[英國廣播公司新聞頻道|BBC新聞網]]|date=2016-05-23|accessdate=2016-05-23|language=en-gb}}</ref>不过，埃及方面的雷达数据准确度可能不及希腊。<ref>{{cite news|url=https://www.utoronto.ca/news/egyptair-804-u-t-forensic-expert-explains-what-we-know-so-far|title=EgyptAir 804: U of T forensic expert explains what we know so far|last=Ahmed-Ullah|first=Noreen|date=2016-05-30|publisher=[[多伦多大学|多伦多大学]]|accessdate=2016-05-31}}</ref>

飞机制造商空中客车公司，在其Facebook页面上发表声明：“空中客车公司遗憾地证实，由埃及航空公司运营的A320飞机于02:30左右（埃及当地时间）在地中海失去联系。”

===黑匣子===

6月1日，埃及民航部称拉普拉斯号已经探测到黑匣子坐标定位器发出的信号。<ref name="Reuters-Vessel-Detects">{{cite news |first1=Lin |last1=Noueihed |first2=Chine |last2=Labbé |first3=Alison |last3=Williams |title=French vessel detects signals likely from EgyptAir jet black box |url=http://www.reuters.com/article/us-egyptair-airplane-blackboxes-idUSKCN0YN4A0 |publisher=''Reuters'' |location=Cairo |date=2016-06-01 |accessdate=2016-06-01}}</ref><ref name="BBC-36427053"/>BEA也确认了这一消息，他们的发言人还说，他们已根据推测将一片海域定为优先搜索区域。<ref name="BBC-36427053">{{cite news |title=EgyptAir crash: Black box signals heard by search teams |url=http://www.bbc.com/news/world-middle-east-36427053 |publisher=BBC News |date=2016-06-01 |accessdate=2016-06-01}}</ref>

希腊军方雷达显示，MS804航班在进入埃及[[飛航情報區|飞行情报区]]时很快偏离航道，在高度37000英尺处，飞机先是向左转90度，紧接着又右转360度。在此期间，飞机高度从37000英尺迅速下降至15000英尺。当飞机高度降至10000英尺后，飞机就从雷达上消失了。5月23日，一名来自埃及国家空中导航服务公司（National Air Navigation Services Company）的埃及民航局官员又否认了这一说法。他同时宣称，该机在坠毁前并没有改变高度，也没有做出反常动作。

与此同时，[[希腊国防部|希腊国防部]]对一名声称在[[喀帕苏斯岛|喀帕苏斯岛]]以南240处看到“有一团火焰从空中坠落”的船长开展调查工作。<ref name="Reuters">{{cite web|url=http://www.reuters.com/article/us-egyptair-airplane-idUSKCN0YA08W|title=EgyptAir flight from Paris to Cairo missing with 66 on board|first=Lin|last=Noueihed|first2=Eric|last2=Knecht|date=2016-05-18|accessdate=2016-05-19|agency=[[路透社|路透社]]}}</ref>埃及民航部长称，相较于机件故障，恐怖袭击导致飞机失事的可能性更大<ref name=Metro190516>{{cite web |url=http://metro.co.uk/2016/05/19/debris-found-in-sea-in-search-for-crashed-egyptair-flight-5892705/ |title=Debris found in sea in search for crashed EgyptAir flight |first=Richard |last=Hartley-Parkinson |publisher=Metro |date=2016-05-19 |accessdate=2016-05-19}}</ref>。
