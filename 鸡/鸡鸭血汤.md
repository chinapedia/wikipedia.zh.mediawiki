'''鸡鸭血汤'''（[[吴语拉丁化方案|吴语苏沪嘉小片发音]]：'''ci ah shiuih thaon'''<ref>{{cite web | url=http://wu-chinese.com/minidict/| title=吴音小字典 | publisher=吴语协会| date= | accessdate=}}</ref>；[[汉语拼音|拼音]]：'''jī yā xiě tāng'''）是一种流行于[[江南|江南]]一带的传统[[小吃|小吃]]，由鸡鸭血、鸡内脏、鸡汤熬制而成，一般搭配[[生煎馒头|生煎馒头]]或[[小笼馒头|小笼馒头]]享用。

==历史==
===起源与发展===
鸡鸭血汤是[[上海城隍庙|上海城隍庙]][[中式小吃|特色小吃]]的代表，最初是在20世纪早期由一位叫许福泉的小贩首创的，他使用一个俗称「铁牛」的深腹铸铁锅烧汤，中间用铝皮隔开，一半烫血，另一半以鸡头鸡脚吊汤。有客人光顾，就从盛器里拨少许鸡心、肝、肫、肠和鸡卵，浇上一勺血汤，撒上葱花，淋几滴鸡油。<ref name="jiyaxuetang">{{cite web | url=http://news.163.com/07/0422/13/3CMF4E7300011247_2.html| title=上海美食传奇| publisher=新闻午报| date= 2007-04-22| accessdate=2011-10-04|author=沈嘉禄}}</ref>肝肠的鲜嫩、血块的滑爽和汤汁的味美加上低廉的售价使得鸡鸭血汤广受食客青睐。

与最初的鸡鸭血汤相比，如今的鸡鸭血汤有所改良，一般店家会加入豆腐、冬笋、粉丝等辅料以增加口感和提鲜，盛在碗里的鸡鸭血汤再撒上少许[[胡椒粉|胡椒粉]]及[[香菜|香菜]]叶，味道更佳。

===血汤与西哈努克===
1973年，因政变而流亡到中国的柬埔寨亲王[[诺罗敦·西哈努克|诺罗敦·西哈努克]]到上海访问期间，中方为其准备了鸡鸭血汤等小吃。据说当时处于[[文化大革命|文革时期]]的中国将西哈努克亲王吃血汤当作一项政治任务来完成，要求一碗鸡鸭血汤中的鸡卵达到直径相同、色泽相同、形状相同的“三同”标准，为此上海的厨师专程前往市郊的[[南翔镇|南翔镇]]宰杀了108只上海本地草鸡才备齐所需的鸡卵胚胎，熬制的鸡鸭血汤受到了柬埔寨人的热捧，连吃两碗。<ref name="jiyaxuetang"/>这一故事也为平民化的小吃鸡鸭血汤添加了一丝传奇色彩。

==食疗和养生==
鸡鸭血汤适宜[[贫血|贫血]]患者、[[孕妇|孕妇]]和消化系统不佳的人补养。<ref>{{cite web | url=http://www.51kang.com/new/20060513104449/index.shtml| title=健康资讯-鸡鸭血汤| publisher=无忧健康管理网| date= | accessdate=2011-10-04|author=}}</ref>血中含有丰富的[[蛋白质|蛋白质]]及多种人体不能合成的[[氨基酸|氨基酸]]，所含的[[红细胞|红细胞素]]含量也较高，还含有微量元素铁等[[矿物质|矿物质]]和多种[[维生素|维生素]]，这些都是人体造血过程中不可缺少的物质。鸡血具有桧风、活血、通络作用；鸭血味咸、性寒，能够补血解毒，治疗劳伤吐血、[[痢疾|痢疾]]；<ref>{{cite web | url=| title=从进食鸡鸭血汤谈起| publisher=《中国食品》1983年第08期 | date=| accessdate=|author=翁德立}}</ref>鸡汤则可以调节免疫、预防感冒。
