'''遺傳性心律基金會''' (英語: SADS HK Foundation) 成立於2013年6月<ref>{{cite news  |url = http://paper.wenweipo.com/2013/07/01/AY1307010005.htm |title = 遺傳性心律基金會成立 |publisher = 文匯報 |date = 2013-07-01 |accessdate = 2016-06-30}}</ref>, 是一所香港註冊的非牟利[[慈善機構|慈善機構]]。致力於擴大公眾及醫療界對[[突發心律失常死亡綜合症|突發心律失常死亡綜合症]] (英語: Sudden Arrhythmia Death Syndromes, 簡稱: SADS)的認知<ref>{{cite news  |url = http://hk.apple.nextmedia.com/news/art/20130626/18312513 |title = 心律失常　鬧鐘響可致命 專襲年輕人　遺傳率達50% |publisher = 蘋果日報 |date = 2013-06-26 |accessdate = 2016-06-30}}</ref>。宗旨是“救助突發心臟性猝死基因傾向的兒童或年輕人、支援他們的家屬”<ref>{{cite news  |url = http://news.mingpao.com/pns/dailynews/web_tc/article/20160626/s00002/1466878381549 |title = 遺憾胞弟猝逝 兄長推救心App 助尋最近救心機 「免他人同抱憾」 |publisher = 明報 |date = 2016-06-26 |accessdate = 2016-06-30}}</ref> 。於2014年, 聯同香港瑪嘉烈醫院和衞生署,啟動首個針對SADS的醫療研究。<ref>{{cite news  |url = http://std.stheadline.com/daily/news-content.php?id=173256&target=2 |title = 猝死與遺傳醫學研究啟動 |publisher = 星島日報 |date = 2014-06-23 |accessdate = 2016-06-30}}</ref> 

== 突發心律失常死亡綜合症 (SADS) == 
突發心律失常死亡綜合症 (SADS)<ref>{{cite news  |url = http://www.am730.com.hk/article-212981 |title = 心律不齊應及早醫治 發燒噪音可引發心臟猝死 |publisher = AM730 |date = 2013-06-23 |accessdate = 2016-06-30}}</ref> 是遺傳性心律異常情況，能導致表面上健康的年輕人突然猝死。SADS的癥兆是可以預先察覺,配以合適的治療, 可大大減低猝死風險。
=== SADS有關病症 ===
* 布嘉達症候群（Brugada Syndrome） 
* 致心律失常性右心室心肌病（ARVC） 
* 長QT綜合症（LQTS） <ref>{{cite news  |url = http://hk.apple.nextmedia.com/news/art/20120409/16230677 |title = 鬧鐘突然響可變奪命凶鈴 長QT綜合症患者　受刺激會休克 |publisher = 蘋果日報 |date = 2012-04-09 |accessdate = 2016-06-30}}</ref>
* 短QT綜合症（SQTS） 
* 兒茶酚胺敏感性多形性室速（CPVT）

=== SADS預警訊號 ===
* 家族歷史中，曾有年輕家庭成員在不明原因下突然[[猝死|猝死]] 
* 運動、激動或受驚嚇時曾有昏厥狀況 
* 運動時有恆常或不尋常的胸痛或氣喘 

== SADS醫療研究 ==
這項醫療研究於2014年6月啟動, 為香港本地心臟猝死的年輕患者作臨床和基因解剖及為他們的直系親屬作臨床及基因評估，以鑑定SADS是否患者猝死的原因、釐定SADS的類型、評估患病率及其於家庭當中的[[遺傳學|遺傳]]情況。預計於2017年公佈結果,希望為香港醫療界提供重要的數據分折, 以及為未來的醫學研究奠下基礎。<ref>{{cite news  |url = http://orientaldaily.on.cc/cnt/news/20140623/00176_076.html |title = 突發心律病可致死 港首研SADS基因 |publisher = 東方日報 |date = 2014-06-23 |accessdate = 2016-06-30}}</ref>
