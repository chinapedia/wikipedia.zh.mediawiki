'''麥齊光'''{{GBSHK}}{{JP}}（{{bd|1950年|6月16日|||M麥}}），於2010年6月退休前為[[香港特別行政區政府|香港特別行政區政府]][[發展局|發展局]][[常任秘書長|常任秘書長]]（工務）。退休後，出任發展局支援[[汶川大地震#重建|四川重建]]小組的組長。於2012年7月1日，宣誓成為新一任[[發展局局長|發展局局長]]。同年，[[香港行政長官|香港行政長官]][[梁振英|梁振英]]多次傳出僭建醜聞，麥齊光表示會將[[唐英年|唐英年]]和梁振英的僭建事件一視同仁和公正辦理<ref>{{cite news|url = http://hk.apple.nextmedia.com/news/art/20130803/18362198|title = 蘋論：在可詛咒的地方擊退可詛咒的時代|publisher=蘋果日報|date      = 2012年8月3日}}</ref>。2012年7月12日，麥齊光因為涉嫌二十多年前不誠實申請領取[[房屋津貼|房屋津貼]]而被[[廉政公署_(香港)|廉政公署]][[拘捕|拘捕]]<ref>{{cite news|url = http://www.apdnews.com/news/31388.html|title = 港前高官诈骗租屋津贴罪成立判缓刑|publisher = 亚太日报|date  = 2013年8月8日}}</ref>。他同日中午「提出辭任」<ref>{{cite news|url  = http://hk.news.yahoo.com/%E9%BA%A5%E9%BD%8A%E5%85%89%E8%BE%AD%E4%BB%BB%E7%99%BC%E5%B1%95%E5%B1%80%E9%95%B7-055800155.html|title = 麥齊光辭任發展局長|publisher = 《星島日報》|date = 2012-07-12}}</ref>，由上任到辭職歷時僅12天，是特區政府史上任期最短的局長及首位在任期間被拘捕的[[問責官員|問責官員]]<ref>{{cite news|url= http://hk.apple.nextmedia.com/news/art/20120713/16510415|title = 外國例子 日本政壇　不少官員數天下台|publisher = 蘋果日報|date = 2012年7月13日}}</ref><ref>{{cite news|url = http://www.singtao.com.hk/yesterday/loc/0720ao03.html|title= 麥齊光被控否 三個月後定奪|publisher = 星島日報|date = 2012-07-20}}</ref>。2012年7月30日，[[中华人民共和国国务院|國務院]]根據《[[香港特別行政區基本法|基本法]]》相關規定與特首梁振英的意見和提議，正式免去麥齊光發展局局長職務<ref name = "任免"/>。麥齊光其後一度被判串謀詐騙罪成，經兩度上訴後被[[香港終審法院|終審法院]]推翻定罪。

==簡歷==
早年就讀[[聖保羅書院|聖保羅書院]]，也曾就讀[[聖貞德中學|聖貞德中學]]及[[英華女學校|英華女學校]]（[[預科|中七]]）<ref>{{cite news
|url       = http://www.ywgsaa.org.hk/eng/news/document/0910.pdf
|title     = 英華女學校校友會通訊
|date      = 2009-10}}</ref>。1973年在[[香港大學|香港大學]][[土木工程|土木工程]]系畢業，獲頒[[工程學|工程學]][[理學士|理學士]]學位，及後於1985年再獲香港大學頒授理科碩士（[[城市規劃|城市規劃]]）學位。麥齊光現為[[香港公路學會|香港公路學會]]資深會員、[[運輸物流學會|運輸物流學會]]特許資深會員、[[英國土木工程師學會|英國土木工程師學會]]資深會員、[[香港工程師學會|香港工程師學會]]資深會員、[[香港工程科學院|香港工程科學院]]院士和[[中國鐵路學會|中國鐵路學會]]高級會員。

麥齊光於1976年加入政府任助理工程師，在1994年升為總工程師，1997年升為政府工程師，2000年又晉升為首席政府工程師。2000年至2002年，他在前[[拓展署|拓展署]]（今土木工程拓展署）出任[[新界東|新界東]]拓展處處長，其後出任[[路政署|路政署]]署長。2006年9月接替退休的[[盧耀楨|盧耀楨]]出任[[環境運輸及工務局|環境運輸及工務局]]常任秘書長（工務）。隨著政府部門改組、[[發展局|發展局]]於2007年7月1日成立，轉任發展局[[常任秘書長|常任秘書長]]（工務）。

在發展局常任秘書長（工務）任內，麥齊光遇上了[[梁展文#任職新世界集團事件|梁展文申請任職「新世界中國地產有限公司」事件]]。雖然梁展文未曾在工務科下任職，麥齊光不知道梁過往在政府實際處理過的工作；但麥齊光仍然同意下屬的意見，向負責審批的[[公務員事務局|公務員事務局]]提出，直指梁退休後任職地產公司可能會引起公眾「公眾觀感的問題」。雖然，意見未有被公務員事務局接受；但當事件曝光後，麥齊光當日的決定，使得在日後立法會專責委員會聆訊時，免受議員的責難。<ref>{{cite news

== 麥齊光涉貪案 ==
{{main|麥齊光涉貪案}}
在2012年7月，麥齊光出任發展局局長，但是隨即被[[香港傳媒|香港傳媒]]揭發他在1980年代任職公共職務時，與[[路政署|路政署]]助理署長[[曾景文|曾景文]]互相租住對方擁有位於[[城市花園|城市花園]]的名下單位(9座21樓E室和9座22樓E室)，藉以申請領取政府的房屋津貼<ref>[http://hk.apple.nextmedia.com/news/art/20120707/16492489 麥齊光　曾景文　罪證曝光] 《蘋果日報》2012年7月7日</ref>。麥齊光承認事件，但強調他的做法依足當年政府指引，沒有違規和不屬於個人操守問題。後來《[[蘋果日報|蘋果日報]]》等傳媒進一步查出麥曾兩人均簽署法律文件，授權對方全權處理所租住物業的買賣，包括決定售價，而兩個單位後來賣出時在合約上簽署者都是租用的一方，令人懷疑租客才是該單位的真正業主<ref>{{cite news|url=http://hk.apple.nextmedia.com/news/art/20120707/16492489|title=麥齊光曾景文罪證曝光|publisher=蘋果日報|date=2012-07-07}}</ref>。隨著民主黨成員[[陳樹英|陳樹英]]攜同剪報到廉政公署報案，廉署開始進行刑事調查，麥齊光於7月12日早上被廉署人員拘捕帶返北角總部。同日中午過後，政府宣佈麥齊光提出辭任發展局局長，其工作交由財政司司長處理。<ref>{{cite web|url=http://www.info.gov.hk/gia/general/201207/12/P201207120357.htm|title=財政司司長暫時處理發展局局長工作}}</ref>隨後，廉政公署亦發表聲明，證實該署拘捕了一名政府決策局局長、一名政府部門助理署長及另外兩名人士，懷疑他們在申領政府房屋津貼時涉嫌違反《防止賄賂條例》。<ref>{{cite web|url=http://www.icac.org.hk/tc/news_and_events/pr2/index_uid_1310.html|title=廉署聲明|date=2012年7月12日}}</ref>7月30日，麥正式被免任發展局局長<ref>{{cite news|url=http://www.info.gov.hk/gia/general/201207/30/P201207300200.htm|title=政府公布發展局局長任命}}</ref>。2013年6月24日，2人被裁定6項罪名全部成立<ref>[http://hk.apple.nextmedia.com/news/art/20130625/18311290 麥齊光曾景文罪成　公務員自危] 《蘋果日報》2013年6月25日</ref>，於同年8月8日於區域法院，被判監禁8個月，緩刑2年<ref>[http://hk.apple.nextmedia.com/realtime/news/20130808/51625763 【呃租津案】麥齊光曾景文 同緩刑免坐監]《蘋果日報》2013年8月8日</ref>。2016年1月6日，終審法院裁定2人上訴得直，撤銷串謀詐騙的定罪。<ref>{{cite web|title=麥齊光曾景文串謀詐騙租津上訴得直|url=https://hk.on.cc/hk/bkn/cnt/news/20160106/bkn-20160106112234874-0106_00822_001.html|work=on.cc|date=2016-01-06}}</ref>
