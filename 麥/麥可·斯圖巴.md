'''麥可·斯圖巴'''（{{lang-en|Michael Stuhlbarg}}，{{IPAc-en|ˈ|s|t|uː|l|ˌ|b|ɑr|g}}，{{bd|1968年|7月5日||}}）<ref>{{Cite web |url = http://usatoday30.usatoday.com/life/movies/news/2009-10-08-michael-stuhlbarg_N.htm |title = Michael Stuhlbarg: Coen brothers get their 'Serious Man' |author = Susan Wloszczyna |date = 2009-10-11 |publisher = USA Today |language = en}}</ref>是一名美國演員。他較出名於[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》（2010年–2013年）中飾演犯罪組織首領[[阿諾·羅斯汀|阿諾·羅斯汀]]<ref>{{Cite web |url = https://www.pedestrian.tv/entertainment/five-minutes-with-boardwalk-empires-michael-stuhlbarg/ |title = Five Minutes With Boardwalk Empire's Michael Stuhlbarg |date = 2012-09-06 |publisher = pedestrian |language = en}}</ref><ref>{{Cite web |url = http://jewishjournal.com/uncategorized/83096/ |title = “Boardwalk Empire's” Michael Stuhlbarg |author = Naomi Pfefferman |date = 2010-09-18 |publisher = Jewish Journal |language = en}}</ref>，以及2009年[[黑色幽默|黑色幽默]]電影《[[非常戇男離奇失婚|非常戇男離奇失婚]]》中飾演Larry Gopnik。斯圖巴曾在多部電影中扮演真實人物角色，例如《[[林肯_(2012年電影)|林肯]]》（2012年）的{{link-en|喬治·海爾姆·葉曼|George Helm Yeaman}}、《[[驚慄大師：希治閣|驚慄大師：希治閣]]》（2012年）的{{link-en|李維·瓦瑟曼|Lew Wasserman}}、《[[時代教主：喬布斯|時代教主：喬布斯]]》（2015年）的[[安迪·赫茲菲爾德|安迪·赫茲菲爾德]]、《[[荷里活黑名單_(電影)|荷里活黑名單]]》（2015年）的[[愛德華·羅賓遜|愛德華·羅賓遜]]、《[[戰雲密報|戰雲密報]]》（2017年）的{{link-en|A·M·羅森塔爾|A. M. Rosenthal}}。他也出名在《[[黑超特警組3|黑超特警組3]]》（2012年）、《[[情迷藍茉莉|情迷藍茉莉]]》（2013年）、《[[天煞異降|天煞異降]]》和《[[奇異博士_(電影)|奇異博士]]》（2016年），還有電視劇《[[冰血暴_(電視劇)|冰血暴]]》（2017年）的[[冰血暴_(第三季)|第三季]]中飾演配角。
