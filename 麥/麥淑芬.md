== 工作資歷 ==
麥淑芬是一名[[律師|律師]]，擁有[[新南威爾士州|新南威爾斯]]、[[英国|英國]]與[[威爾士|威爾斯]]及[[香港|香港]]執業律師資格。在她20多歲時,在澳洲已晋升為律師事務所合伙人。於2000年加入TOM後擔任[[法律總監|法律總監]]一職，及後於2001年即被委任為公司[[秘書|秘書]]。於2006年3月，麥更獲委任執行董事。在TOM工作8年後，麥的努力和貢獻獲董事和管理層一致認同，於2008年晋升為TOM集團[[首席財務官|首席財務官]]。

在任[[CFO|CFO]]其間，麥淑芬與集團CEO[[楊國猛|楊國猛]]及業務團隊積極優化公司業務，共同開發創新產品和業務，使公司由經營業務產生的現金淨額大幅上升78%。

加入TOM前，她曾任[[和記黃埔|和記黃埔有限公司]]之集團高級法律顧問。作為一位資深管理人，她不諱言和黃的管理模式也对她影响深远。她認為和黃集團的管理是中西的結合，而她個人對中國文化的熱愛，以及作為一位受西方教育的專業管理人員，這套中西結合管理模式能使她處事更公正又具人情元素，有助與員工的溝通。

== 個人 ==
擁有企管及商業事務方面10年以上經驗的麥淑芬<ref name="CFO4">{{cite web|author1=央视网|title=TOM集团麦淑芬 CFO世界的“女掌门”|url=http://media.people.com.cn/GB/40606/9144809.html|website=人民网|accessdate=2018-04-10|date=2009年04月16日}}</ref><ref name="undefined">{{cite web| language =| publisher =CEO商学院| title =低调CFO世界"掌门人"麦淑芬的强者风度| url =http://www.09edu.com/2009/0422154542009.shtml| work =首席财务官| date =2009-04-22| accessdate =2009-05-26| archivedate =2013年4月24日| archiveurl =https://archive.today/20130424164601/http://www.09edu.com/2009/0422154542009.shtml| deadurl =yes}}</ref> 向來行事作風低調務實。出生於[[新加坡|新加坡]]的她沒有被國外文化所薰陶而忘本,相反，她曾對傳媒表示深深熱愛[[中國文化|中國文化]]，因此積極學習[[漢語|漢語]]。 她小時候的家庭環境和嚴謹的家庭教育，亦使她個性更獨立和果斷，在職場上的成就表現出巾幗不让须眉的风范。

麥淑芬對中国文化的喜愛，也反映到她的生活興趣當中。動靜皆宜的她喜欢耍[[太极|太极]]、練[[气功|气功]]，這些除了可以强身健体之余，也能為她減壓，还能领悟中国文化的精粹。靜態活動方面，她會画[[山水画|山水画]]，造就平稳的心态，使能更客观地看待人和事。另外，做飯和做[[蛋糕|蛋糕]]也是她的個人強項。
