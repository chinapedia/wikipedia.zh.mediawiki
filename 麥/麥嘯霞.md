'''麥嘯霞'''（Mak Siu Ha，{{bd|1904年||1941年|12月5日}}），[[香港|香港]][[畫家|畫家]]、[[粵劇|粵劇]]撰曲人、[[電影|電影]][[導演|導演]]及[[演員|演員]]。麥嘯霞的[[銀幕|銀幕]]處男作局部有聲[[粵語|粵語]]文藝片《良心》（1933年11月28日[[香港|香港]]公映，擔任[[導演|導演]]及[[演員|演員]]，並[[撰寫|撰寫]][[歌曲|歌曲]]）；遺作是[[粵語|粵語]] [[黑色|黑色]][[喜劇|喜劇]]《血濺桃花扇》（1940年9月28日[[香港|香港]]公映，擔任[[導演|導演]]及編劇），一生參予廿多[[電影|電影]]。麥嘯霞在1940年撰寫[[書籍|書籍]]《廣東戲劇史畧》，列舉從[[明朝|明朝]]至[[清朝|清朝]]以來較有影響的[[粵劇|粵劇]]撰曲家。

== 公益義演 ==
麥嘯霞為「香港婦女新運會」在1940年1月主辦「中國閨秀青年戲劇慈善會」，[[義務|義務]]撰寫[[粵劇|粵劇]]《虎膽蓮心》，激勵[[中國|中國]]國民奮勇抗擊[[入侵|入侵]]的[[大日本皇軍|大日本皇軍]]，1940年1月5日（星期五）[[晚上|晚上]]在[[上環|上環]][[太平戲院|太平戲院]]公演，<ref>1940年1月5日（星期五），[[香港|香港]]《[[華字日報|華字日報]]》，第三張；第一頁。</ref>。

== 被日軍炸死 ==
* 1941年12月5日（[[農曆|農曆]]辛巳蛇年十月十七日，星期五）[[上午|上午]]9時，[[大日本皇軍|大日本皇軍]][[空襲|空襲]][[香港|香港]]，麥嘯霞不幸在[[香港島|香港島]][[羅便臣道|羅便臣道]]「安華車房」中彈，屍首無存，遺下[[妻子|妻子]]馮潔華在事發4天後才驚悉噩耗<ref>1947年12月17日（星期三），[[香港|香港]]《[[華僑日報|華僑日報]]》，第一張；第四頁。</ref>。

== 麥嘯霞紀念學校 ==
* 1948年2月25日（星期三）至27日（星期五），在[[香港島|香港島]][[皇后大道中|皇后大道中]]38號三樓「中國近代書畫匯」裡，舉行『時賢書畫義展』，[[高劍父|高劍父]]、[[吳公虎|吳公虎]]、[[葉恭綽|葉恭綽]]、[[趙少昂|趙少昂]]、鄧爾雅、陳荊鴻等[[畫家|畫家]]捐出100多幀作品作[[義賣|義賣]]，所得[[善款|善款]]全部撥作「麥嘯霞紀念學校」[[籌募|籌募]]辦學[[基金|基金]]<ref>1948年2月25日（星期三），[[香港|香港]]《[[華僑日報|華僑日報]]》，第二張；第二頁。</ref>。

== 麥嘯霞撰曲 ==

* 1935年粵語文藝片《殘歌》主題曲〈夢〉；演唱：[[李綺年|李綺年]]。
