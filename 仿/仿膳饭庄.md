'''仿膳饭庄'''，位于[[北京市|北京市]][[西城区|西城区]][[文津街|文津街]]1号[[北海公园|北海公园]]，是北京有名的[[宫廷菜|宫廷菜]]馆。设在漪澜堂的仿膳饭庄由三个庭院构成。到2010年代，共有餐厅12间，餐位300多个，内部装饰以中国传统文化为主题，挂有彩绘宫灯，窗帘、餐巾、台布、椅套都是明黄色，餐具是带“万寿无疆”字样的仿清宫瓷器或金器<ref name=qian/>。

仿膳饭庄首屈一指的是[[满汉全席|满汉全席]]，完整的满汉全席需4到6餐才能用完，为此仿膳饭庄推出“满汉全席精选菜单”，吃一餐便可领略其特色<ref name=qian/>。

==历史==
1925年，北海公园正式开放。原在清宫御膳房的御厨[[赵仁斋|赵仁斋]]、[[孙绍然|孙绍然]]等人在北海公园北部开设仿膳茶社，经营风味菜肴及面点小吃。“仿膳”意为仿照清宫御膳房的制作方法<ref name=qian>{{Cite web |url = http://finance.ifeng.com/a/20160421/14336049_0.shtml |title =为保护古建筑 北海仿膳饭庄迁新址  |author =  |date = 2016-04-21 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
[[File:Imperial_Kitchen_imitation_-_panoramio.jpg|thumb]]
1955年，仿膳茶社从私营改为国营，更名“仿膳饭庄”。1959年，从北海公园北部迁至琼华岛上的漪澜堂。1966年到1977年，北海公园因[[文革|文革]]停止开放，由[[党和国家领导人|党和国家领导人]]使用，仿膳饭庄也停止对外营业。这段时间，仿膳饭庄成为党和国家领导人会见外宾、处理政务的场所。1978年，仿膳饭庄恢复对外营业。仿膳饭庄接待过[[周恩来|周恩来]]、[[邓小平|邓小平]]、[[叶剑英|叶剑英]]、[[彭真|彭真]]、[[万里|万里]]等党和国家领导人，[[理查德·尼克松|尼克松]]、[[基辛格|基辛格]]等外国政要，[[琼瑶|琼瑶]]等名人<ref name=qian/>。

2013年12月，[[中央纪委|中央纪委]]、[[中央教育实践活动领导小组|中央教育实践活动领导小组]]发出《关于在党的群众路线教育实践活动中严肃整治“会所中的歪风”的通知》，为贯彻[[中央八项规定|中央八项规定]]精神，坚决反对“[[四风|四风]]”，遵照中央指示，就严肃整治“会所中的歪风”提出明确要求，严禁地方将历史建筑、公园等公共资源变为私人会所<ref>{{Cite web |url = http://www.gov.cn/jrzg/2013-12/23/content_2553121.htm |title = 中央纪委等发出通知要求严肃整治“会所中的歪风” |author =  |date = 2013-12-23 |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>。2014年10月，[[中共中央办公厅|中共中央办公厅]]、[[国务院办公厅|国务院办公厅]]转发十部门《关于严禁在历史建筑、公园等公共资源中设立私人会所的暂行规定》<ref>{{Cite web |url =http://jx.people.com.cn/n/2014/1029/c186330-22751260.html  |title = 探访那些曾经奢华私密的顶级会所 |author =  |date = 2014-10-29 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

2014年1月，为响应中央的整治要求，北京市开始对公园内的24家高档私人会所进行停业整顿。其中包括[[地坛公园|地坛公园]]内的乙十六总店、北海公园乙十六御膳堂等等。乙十六御膳堂被列入“城市公园内的高档餐饮及企业租用经营场所”而停业整顿。随后“御膳堂”的牌匾被摘下，换上“仿膳饭庄”的匾额，仿膳饭庄于2014年5月中旬在乙十六御膳堂原址（即1925年仿膳茶社原址）开张营业。被仿膳饭庄长期占用的古建筑漪澜堂由此获得腾退，归北海公园对外开放<ref>{{Cite web |url =http://tech.sina.com.cn/roll/2017-06-19/doc-ifyhfpat5266268.shtml  |title =北京剑指舌尖上的腐败 公园内高档会所纷纷转型  |author =  |date =2017-06-19  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref name=qian/>。

2010年8月25日，“仿膳饭庄创建85周年暨市级非物质文化遗产‘清廷御膳'技艺展示”活动在仿膳饭庄举行。北京市文化局当天向仿膳饭庄颁发了“[[北京市非物质文化遗产|北京市市级非物质文化遗产]]——仿膳饭庄清廷御膳制作技艺”标牌<ref name=feiyi/>。2011年，“仿膳（清廷御膳）制作技艺”被列入第三批[[国家级非物质文化遗产|国家级非物质文化遗产]]保护名录<ref>{{Cite web |url = http://www.chinadaily.com.cn/hqcj/zxqxb/2011-09-15/content_3790850.html |title = 仿膳饭庄清廷御膳入选国家级非遗 |author =  |date =2011-09-15  |publisher = 中国日报网 |language =  |accessdate =  |quote =  }}</ref>。

仿膳饭庄由北京市仿膳饭庄有限责任公司经营，该公司由[[全聚德|中国全聚德（集团）股份有限公司]]投资<ref>{{Cite web |url = http://qyxy.baic.gov.cn/xycx/queryCreditAction!qyxq_view.dhtml?reg_bus_ent_id=1E3291E2AC6C4C74825516113D1EF2A2&credit_ticket=23728F15C2E48243372C65617289086F |title = 	北京市仿膳饭庄有限责任公司 |author =  |date =  |publisher =北京市企业信用信息网  |language =  |accessdate =2018-02-08  |quote =  }}</ref>。2005年，首旅集团、新燕莎集团、[[全聚德|全聚德]]集团实行战略重组，仿膳饭庄进入全聚德集团，成为首旅集团旗下餐饮版块中的[[中华老字号|中华老字号]]餐饮品牌<ref name=feiyi>{{Cite web |url = http://news.163.com/10/0826/08/6F0HHAFU000146BC.html |title = 仿膳饭庄"清廷御膳"制作技艺入选北京市级非遗 |author =  |date =2010-08-26  |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。

1993年，北京天安餐饮管理有限公司天安门仿膳饭庄开业。天安门仿膳饭庄位于北京市[[东城区_(北京市)|东城区]][[东交民巷|东交民巷]]37号，[[天安门广场|天安门广场]]东侧，[[中国国家博物馆|中国国家博物馆]]南侧<ref>{{Cite web |url =http://qyxy.baic.gov.cn/xycx/queryCreditAction!qyxq_view.dhtml?reg_bus_ent_id=B9B53594FD8B41C29A39C3F9189524C3&credit_ticket=6A23928B51D8002990AC112213DF055D  |title = 北京天安餐饮管理有限公司天安门仿膳饭庄 |author =  |date =  |publisher =北京市企业信用信息网  |language =  |accessdate =2018-02-08  |quote =  }}</ref>。
