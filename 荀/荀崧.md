'''荀崧'''（{{bd|263年？||329年？|catIdx=X}}）<ref name=A>《晉書·荀崧傳》寫荀崧卒於咸和三年，然而本傳中亦寫到荀崧於咸和四年成帝被迎至溫嶠船上時仍在世，可見此卒年有誤。而本傳亦稱當時荀崧「年老病篤」，故荀崧可能在不久便死去</ref>[[表字|字]]'''景猷'''，[[潁川|潁川]][[潁陰|潁陰]]人。[[東漢|東漢]]末著名謀士[[荀彧|荀彧]]的玄孫。[[晉朝|晉朝]]官員，在[[東晉|東晉]]時官至[[光祿大夫|右光祿大夫]]。

== 生平 ==
荀崧志操清純，喜好文學。[[晉武帝|晉武帝]][[泰始_(西晋)|泰始]]年間繼襲父親[[荀頵|荀頵]]'''安陵鄉侯'''的爵位，任濮陽王[[司馬允|司馬允]]的[[文學|文學]]。後被趙王[[司馬倫|司馬倫]]任命為[[相國|相國]][[參軍|參軍]]。多次升遷後官拜侍中、[[中護軍|中護軍]]。

[[永嘉_(西晋)|永嘉]]五年（311年），[[前趙|漢國]]軍隊攻陷[[洛陽|洛陽]]並擄走晉懷帝，荀崧與百官出奔[[密縣|密縣]]。路上荀崧母親逝世，而追兵將至，荀崧被官員們拋下，最終荀崧遇襲還受了重傷。荀崧及後於[[密山|密山]]安葬母親，並為她守喪。守喪完結後，在[[滎陽|滎陽]]行臺的[[荀藩|荀藩]]承制以荀崧監[[長江|江]]北軍事、南中郎將、後將軍、假節、[[襄城|襄城]]太守。當時皇陵都被盜，荀崧便派[[主簿|主簿]][[石覽|石覽]]領兵到洛陽修葺皇陵，後以功勳進封'''舞陽縣公'''。及後遷任都督[[荊州|荊州]]江北諸軍事、平南將軍，鎮守[[宛|宛]]，改封為'''曲陵公'''<ref>[[洪頤煊|洪頤煊]]《諸史攷異·卷三》：「《（晉書）地理志》[[江夏郡|江夏郡]]有曲陵縣。頤煊按《[[漢書|漢書]]·地理志》、《[[續漢志|續漢志]]》、《[[宋書|宋書]]·州郡志》江夏郡皆作西陵縣，此《傳》及《志》曲陵皆當是西陵之偽。」認為應為西陵公。</ref><ref>《[[四庫全書|四庫全書]]》錄[[虞世南|虞世南]]《[[北堂書鈔|北堂書鈔]]》卷五十七載：「溫嶠表荀崧為秘書監云：『夫國史之典，將以明得失之跡以諷其上，記功書過，謂之實錄，使一代之典，煥然可觀。今之秘書著作是宜得文史之才，以辯經緯。散騎常侍、光祿大夫、'''西陵侯'''崧文質杉杉，思義通博，歷位先朝，莅事以穆，宜掌秘奧，宣明史籍。臣等參之，眾議謂崧可領秘書監。』所稱爵位為西陵侯，與此不同</ref>。

建興三年（315年），作亂的[[杜曾|杜曾]]圍攻宛，荀崧見自己兵少而且食糧已盡，於是派女兒[[荀灌|荀灌]]向襄城太守石覽及南中郎將[[周訪|周訪]]求救。二人派兵合力營救，杜曾見援軍前來就逃散。杜曾不久就向荀崧請降，並請求討伐[[丹水縣|丹水縣]]的盜賊。荀崧寄望杜曾成為自己外援，於是不理陶侃的勸阻而答允。但杜曾隨後便領兵圍攻[[襄陽|襄陽]]。荀崧隨後使派兵突襲[[穰縣|穰縣]]，俘殺杜曾堂兄[[杜保|杜保]]。

[[太興_(晉)|太興]]元年（318年），[[晉元帝|晉元帝]]即位，徵召荀崧為[[尚書僕射|尚書僕射]]，與[[刁協|刁協]]共同制定禮儀。後轉[[太常|太常]]，任內建議設置《[[儀禮|儀禮]]》、《[[公羊傳|公羊傳]]》、《[[穀梁傳|穀梁傳]]》及鄭氏《[[易|易]]》的[[博士|博士]]。

[[永昌_(晉)|永昌]]元年（322年），[[王敦之亂|王敦之亂]]爆發，[[大将军_(东亚)|大將軍]][[王敦|王敦]]舉兵攻陷[[建康|建康]]，執掌朝政，又表荀崧為尚書左僕射。同年晉元帝因王敦之亂憂憤而死，群臣商議元帝[[廟號|廟號]]，但王敦則認為現在國家未統一，當日後再議。然而荀崧則不認同，更建議上廟號為「中宗」。

[[太宁_(晋)|太寧]]元年（323年）加[[散騎常侍|散騎常侍]]，後又領[[太子太傅|太子太傅]]。[[王敦之亂|王敦之亂]]被平定後又任金紫[[光祿大夫|光祿大夫]]、[[錄尚書事|錄尚書事]]，封'''平樂伯'''。後更遷右光祿大夫、[[開府|開府]]儀同三司，錄尚書事。後來又加領[[祕書監|祕書監]]，給親兵一百二十人，頗見親待。

[[咸和_(晋成帝)|咸和]]三年（328年），[[歷陽|歷陽]][[內史|內史]][[蘇峻|蘇峻]]率叛軍攻陷建康，荀崧與王導等人在叛軍入宮時共同守衞[[晉成帝|晉成帝]]。不久蘇峻強遷成帝到[[石頭城|石頭城]]，荀崧亦緊隨在成帝身邊。次年，討伐義軍屢敗叛軍，並攻破石頭城，成帝亦被迎至建立行臺的[[溫嶠|溫嶠]]的船上。當時荀崧已病重，但亦然吃力的步行跟從。後逝世<ref name=A/>，享年六十七歲。朝廷追贈侍中，諡號為'''敬'''。

== 性格特徵 ==
*荀崧有孝行，其母親雖然逝世在[[永嘉之亂|永嘉之亂]]的亂局時，但自己仍堅持守護著母親屍體，最終被重創，並幾乎死去。及後更能堅持為母守喪。
*荀崧善待親族子女。好像堂弟[[荀馗|荀馗]]早死，留下兩個年幼的兒子[[荀序|荀序]]和[[荀廞|荀廞]]。荀崧於是讓二人和自己一同生活，感情和親生兒子一樣。及後朝廷打算讓荀崧的兒子承襲[[荀顗|荀顗]]臨淮公爵位，以讓荀顗繼嗣者[[荀徽|荀徽]]死後一直絕嗣的荀顗一系有人繼嗣。但荀崧就將此讓給荀序。
*荀崧雖然衰老，但仍勤讀典籍，獲當時人欣賞讚許。
