{{伊拉克历史}}
'''伊拉克历史'''悠久，伊拉克所在的地区在历史上曾被称为[[美索不达米亚|美索不达米亚]]，是人类文明的主要发源地之一。

两河流域目前发现的最早的古文明距今已有6000多年。雖然巴比伦現已消失，但其影響（尤其宗教方面）卻很多流存至今。成為[[四大文明古國|四大文明古國]]之一實在當之無愧。

巴比伦文明大致以今天的[[巴格达|巴格达]]城为界，分为南北两部分。北部以古[[亚述|亚述]]城为中心，称为西里西亚，或简称亚述；南部以巴比伦城为中心，称为巴比伦尼亚，意思为“巴比伦的国土”。巴比伦尼亚又分为两个地区，南部靠近[[波斯湾|波斯湾]]口的地区为[[苏美尔|苏美尔]]，苏美尔以北地区为[[阿卡德|阿卡德]]，两地居民分别被称为[[苏美尔人|苏美尔人]]和[[阿卡德人|阿卡德人]]。美索不达米亚文明最初就是由苏美尔人创造出来的。

== 史前 ==
美索不达米亚是[[新石器革命|新石器革命]]最早的地区，在约公元前10000年。

== 古代美索不达米亚 ==
{{Main|美索不达米亚|美索不达米亚历史}}
=== 铜器时代 ===
在早期铜器时代的[[欧贝德时期|欧贝德时期]]之中，下美索不达米亚出现了苏美尔文明。随着公元前24世纪[[阿卡德帝国|阿卡德帝国]]的兴起，古典苏美尔文明结束。在[[古提王朝|古提王朝]]时期，苏美尔人在前21世纪有着短暂[[乌尔第三王朝|复兴]]。

苏美尔几乎和[[古埃及|古埃及]]人同时发明了文字。他们用削尖的[[芦苇|芦苇]]做[[笔|笔]]，把文字刻在泥胚上，然后把泥胚烘干，成为泥板。由于这种文字形状成尖劈形，所以被称为[[楔形文字|楔形文字]]。

公元前2006年，[[阿摩利人|阿摩利人]]入侵两河流域，摧毁了[[阿卡德人|阿卡德人]]和[[苏美尔人|苏美尔人]]建立的[[乌尔第三王朝|乌尔第三王朝]]。公元前1894年左右，[[阿摩利人|阿摩利人]]建立起一个以幼发拉底河河畔的巴比伦城为首都的王朝。从那时起，美索不达米亚就被称为“巴比伦尼亚”，那里所有居民都被称为巴比伦人。

古巴比伦王国时期是两河流域历史上最辉煌的时期之一。这个国家出现了一位著名的君主－[[汉摩拉比|汉摩拉比]]。汉摩拉比用武力统一了两河流域后，建立了一个中央集权的专制国家。他个人集[[宗教|宗教]]、[[军事|军事]]、 [[行政|行政]]、[[司法|司法]]和[[水利|水利]]建设等各种大权于一身。为了更有效的统治自己的国家，他颁布了著名的《[[汉摩拉比法典|汉摩拉比法典]]》。法典共282条，刻在一块高2.25米的黑色[[玄武岩|玄武岩]]石柱上，是迄今发现的最早完备成文的法典。

古巴比伦王国在汉穆拉比统治时期达到极盛，但是汉穆拉比死后，帝国就瓦解了。王国先后受到[[赫梯人|赫梯人]]、[[加喜特人|加喜特人]]的入侵，直到公元前729年终于被[[亚述帝国|亚述帝国]]吞并。

===铁器时代===
在铁器时代，新亚述帝国是本地区的主导力量。

在波斯帝国征服前，新巴比伦王国（前626–前539）是本地区最后一个重要的时期。在新亚述帝国陷入内战之后，迦勒底人创建了新巴比伦王国。新巴比伦王国的统治止于前539年。

== 古典时期 ==
=== 阿契美尼德和塞琉古帝国 ===
公元前539年，[[阿契美尼德王朝|阿契美尼德王朝]]的居鲁士大帝征服了美索不达米亚。之后亚历山大大帝征服了波斯帝国。之后此地由塞琉古帝国统治。

===帕提亚和罗马统治===
2世纪初，罗马帝国皇帝[[圖拉真|圖拉真]]率军侵入了帕提亚，征服了美索不达米亚。之后，又复归帕提亚帝国。

公元1世纪，基督教传入美索不达米亚，尤其是是罗马[[叙利亚_(罗马行省)|叙利亚行省]]成为了东方宗教和叙利亚的文学传统的中心。通常还认为，[[曼達安教|曼達安教]]可能起源于此，或者是作为从巴勒斯坦寻求庇护的曼达人而带入的。

===萨珊帝国===
3世纪起，帕提亚帝国被萨珊王朝取代。

直到602年，波斯帝国的沙漠边境由阿拉伯Al-Hirah的[[拉赫姆王國|拉赫姆]]国王守卫。这一年，[[霍斯劳二世|霍斯劳二世]]废除了拉赫姆王国。将边境向游牧民族敞开。在远处的北部，与拜占庭帝国为界。

== 中世纪 ==
656年该地区被[[阿拉伯人|阿拉伯人]]占领。此前相当长的一段时间，这里都是[[波斯|波斯]]的一部分，但幾乎所有政權其首都皆建於兩河流域如[[波斯帝國|波斯帝國]]、[[亞歷山大帝國|亞歷山大帝國]]的[[巴比倫|巴比倫]]、[[塞琉古王朝|塞琉古王朝]]的[[塞琉西亞|塞琉西亞]]、[[安息帝國|安息帝國]]、[[薩珊王朝|薩珊王朝]]的[[泰西封|泰西封]]。762年，[[哈里发|哈里发]]迁到[[安息帝國|安息帝國]]、[[薩珊王朝|薩珊王朝]]首都[[泰西封|泰西封]]北郊的[[巴格达|巴格达]]。此后，巴格达一直是[[阿拉伯|阿拉伯]]世界的中心，直到1258年它被[[旭烈兀|旭烈兀]]率领的[[蒙古西征|蒙古西征]]军摧毁为止。

== 奥斯曼和马木鲁克统治 ==
在14世纪至15世纪初土库曼人的[[黑羊王朝|黑羊王朝]]统治了今伊拉克地区。1466年，[[白羊王朝|白羊王朝]]击败了黑羊王朝，接管了这一地区。16世纪起，大多数今伊拉克地区开始由奥斯曼帝国统治。是为巴格达[[省_(奥斯曼帝国)|省]]。

1534年，经过土耳其人与[[薩法維王朝|萨菲波斯]]人长期的拉锯争夺，伊拉克最终并入了[[奥斯曼帝国|奥斯曼帝国]]，两河流域与伊朗高原合为一体的历史就此画上了休止符。

== 20世纪 ==
1915年，[[英国|英国]]军队占领伊拉克，以[[国联|国联]]的名义对其进行统治，直至1932年伊拉克独立，成為以[[哈希姆王朝|哈希姆王朝]]為統治者的君王國家。1958年，具有[[社会主义|社会主义]]背景的阿拉伯复兴社会党推翻亲西方的哈希姆王朝，取得执政地位。西方国家对此实行抵制。[[萨达姆|萨达姆·侯赛因]]在1979年成为该党的领袖，并实行严酷的独裁统治。在萨达姆执政其间，大肆打击并杀害持不同政见者在1980年代，萨达姆与其邻国[[伊朗|伊朗]]，由于教派冲突，进行了长达8年的[[两伊战争|两伊战争]]。而此时[[伊朗|伊朗]]经过[[伊斯兰革命|伊斯兰革命]]以后，成为敌视美国的[[政教合一|政教合一]]国家，因此，美国在两伊战争中支持伊拉克。1990年伊拉克军队入侵[[科威特|科威特]]，科威特由于军事抵抗能力太低而迅速沦陷，萨达姆宣布“科威特成为伊拉克的一个省”，不过迅速遭到了联合国的谴责和美英联军的[[沙漠風暴行動|沙漠風暴行動]]，对伊拉克地面部队实施空中打击，次年1月[[海湾战争|海湾战争]]爆发，伊拉克被美国领导的多国部队击败并逐出科威特。这导致了以美国为首的国家对伊拉克实行长期制裁，并称伊拥有大规模杀伤性武器而以[[联合国|联合国]]名义多次对其实施武器核查。

== 近期历史 ==
2003年3月20日，美国以伊将[[国际原子能机构|国际原子能机构]]驱逐出伊拉克不受国际监督与核查为理由，组织联军进攻伊拉克，此并非经过联合国表决通过，[[伊拉克战争|伊拉克战争]]爆发，在几周后复兴党政权就被[[美英联军|美英联军]]擊敗，其间伊拉克军队并未对美军有大的抵抗，大量的伊拉克军人投降，伊拉克[[美軍佔領伊拉克|处于美国及其盟国军队的控制之下]]，但是未能找到伊拉克拥有[[大规模杀伤性武器|大规模杀伤性武器]]的证据。由美国任命的[[行政长官|行政长官]][[保罗·布雷默|保罗·布雷默]]为美国负责伊拉克事务的最高行政长官，担任伊拉克首脑。一个由各派政治力量组成的[[伊拉克临时管理委员会|伊拉克临时管理委员会]]在商讨起草宪法和[[基本建设计划|基本建设计划]]。2004年6月1日，联合国、美、伊三方协商推选产生以[[阿拉维|阿拉维]]为总理的伊临时政府。6月28日，美英联军向伊拉克临时政府移交权力。2005年1月30日，伊拉克举行大选。4月6日，[[贾拉勒·塔拉巴尼|贾拉勒·塔拉巴尼]]当选伊拉克过渡政权总统。2006年5月20日，战后首届正式政府在经议会表决批准后宣誓就职。总理为[[贾瓦德·马利基|贾瓦德·马利基]]。[http://news.xinhuanet.com/newscenter/2006-05/20/content_4576829.htm]

然而[[恐怖分子|恐怖分子]]为对抗美英的军事占领进行了针对伊拉克警察、政府机构以及驻伊美军的汽车[[炸弹|炸弹]]袭击。[[什叶派|什叶派]]和[[逊尼派|逊尼派]]穆斯林在伊拉克境内的暴力冲突升级也成为伊拉克政府目前棘手的问题。
