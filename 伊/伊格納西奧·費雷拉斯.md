'''伊格納西奧·費雷拉斯'''（{{lang-es|Ignacio Ferreras}}，{{bd|1960年||catIdx=Ignacio Ferreras}}）是[[西班牙|西班牙]][[蘇馬拉加_(西班牙)|蘇馬拉加]]出身的男性動畫師、動畫導演。<ref name="IF">{{Cite web |url=https://www.festivalscope.com/director/ferreras-ignacio |title=Ignacio Ferreras Spain |publisher=Festival Scope |accessdate=2014年1月28日}}</ref>

== 經歷 ==
伊格納西奧曾於[[夢工廠|夢工廠]]的2000年動畫作品《[[約瑟傳說：夢境之王|約瑟傳說：夢境之王]]》中擔任動畫師一職，隨後則執導過電視短篇動畫《How to Cope with Death》、<ref name="IF"/>以及在《[[東京奧林匹克2008|東京奧林匹克2008]]》中執導部份章節內容。

2010年伊格納西奧發表改編自西班牙漫畫家[[帕科·羅卡|帕科·羅卡]]、關於描述老年人心境世界的作品《歡樂皺紋》之[[歡樂皺紋|同名動畫]]，<ref>{{Cite web |url=http://criterioncast.com/film-festivals/piff-film-festivals/piff-2013/ignacio-ferreras-wrinkles-piff-2013/ |title=James Reviews Ignacio Ferreras’ Wrinkles |publisher=Criterion Cast |author=James McCormick |date=2013年3月1日 |accessdate=2014年1月28日}}</ref>該片後續為伊格納西奧贏得第26屆[[哥雅獎|哥雅獎]]以及2012年匈牙利凱奇凱梅特國際動畫影展的最佳動畫榮譽。<ref>{{Cite web |url=http://www.zippyframes.com/index.php?option=com_content&view=article&id=1047:wrinkles-birdboy-win-at-the-goya-awards&catid=42&Itemid=100008 |title=Wrinkles, Birdboy win at the Goya Awards |publisher=Zippy Frames |date=2012年2月12日 |accessdate=2014年1月28日 |archive-url=https://web.archive.org/web/20140202164031/http://www.zippyframes.com/index.php?option=com_content&view=article&id=1047:wrinkles-birdboy-win-at-the-goya-awards&catid=42&Itemid=100008 |archive-date=2014-02-02 |dead-url=yes }}</ref><ref>{{Cite web |url=https://www.animationfestival.ca/index.php?option=com_content&task=view&id=442 |title=Ottawa International Animation Festival 2012 |publisher=Animation Festival |accessdate=2014年1月28日}}</ref>
