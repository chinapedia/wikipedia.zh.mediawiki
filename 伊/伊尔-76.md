'''伊留申 IL-76'''（{{lang-ru|Ил-76}}，简称'''伊尔-76'''）是[[蘇聯|蘇聯]][[伊留申設計局|伊留申設計局]]研製的四發動機大型[[運輸機|運輸機]]。在蘇聯、[[東歐|東歐]]地區、[[亞洲|亞洲]]和[[非洲|非洲]]也有廣泛使用。原本的設計為軍用運輸機，但是在蘇聯解體後也有大量用作民航用途。而它的北約稱號為'''Candid'''（耿直）。

== 開發 ==
IL-76於1967年開發，目標是能夠運載超過40[[公噸|公噸]]（88,000磅）的貨物於6小時內飛行超過5,000[[公里|公里]]（2,700[[海浬|海浬]]，3,100[[英里|英里]]）。也可以在蘇聯境內多數機場設備不全或設施簡單的機場起飛，同時要適應蘇聯不同氣候，主要是要克服近極地地區（如：西伯利亞）的嚴寒氣候和高加索地區氣溫多變的環境。在定好了這些條件後，[[伊留申設計局|伊留申設計局]]就為這飛機設下了以[[洛克希德|洛克希德]][[C-141|C-141]]為半假想敵。實際上Il-76對起落架和發動機設計就比C-141還要好。此外軍用版本還加入了後射機槍，貨舱尺寸為20公尺×3.4公尺×3.4公尺。結果在1971年3月25日進行首飛。直到今天，IL-76還在[[烏茲別克|烏茲別克]]的廠房生產。

2012年9月，伊留申76改进型伊留申-476（伊留申-76MD-90A型）首飞成功<ref >{{cite news|url=http://mil.huanqiu.com/observation/2012-09/3145186.html |title=伊留申-476运输机首飞中国或成首个海外客户| publisher=环球网 |accessdate= 2012-12-27|date= 2012-09-25}}</ref>，可以明顯發現其引擎與C-17一樣採用了大直徑的渦輪。12月，首架样机涂装正式完成<ref >{{cite news|url=http://mil.huanqiu.com/observation/2012-12/3420542.html|title=伊留申-476首架样机完工传中国将购40架| publisher=环球网 |accessdate= 2012-12-27|date= 2012-12-26}}</ref>。2015年12月7日交付俄空天军军事运输航空兵部队。<ref >{{cite news|url=http://news.china.com.cn/2015-12/07/content_37249812.htm |title=俄首架最新改型伊尔-76MD-90A运输机交付军队| publisher=中国网|date= 2015-12-07}}</ref>

== 使用 ==
[[File:PLAAF_Ilyushin_Il-76_landing_at_Perth_Airport.jpg|thumb]]的IL-76。|300px]]

在蘇聯解體前軍用時期，蘇聯空軍大量使用IL-76作運輸用途。在1979年至1991年，蘇聯空軍共作了14,700趟飛行到阿富汗。據統計，這些的行動共運載了786,200名人員和315,800噸貨物。IL-76運載了到[[阿富汗|阿富汗]]的89%坦克和74%的貨物。<ref>Ilyushin Il-76 Russia's Versatile Airlifter</ref>也因為這個經過，後期加拿大使用民用的Il-76把貨物運到[[阿富汗|阿富汗]]<ref>[http://cmte.parl.gc.ca/cmte/CommitteePublication.aspx?SourceId=187014 Canadian Parliament Website] {{webarchive|url=https://web.archive.org/web/20071013150828/http://cmte.parl.gc.ca/cmte/CommitteePublication.aspx?SourceId=187014 |date=2007-10-13 }}</ref>，IL-76也改裝作[[预警机|预警机]]（AWACS）和[[空中加油機|空中加油機]]（版本稱為[[伊爾-78|IL-78]]）。

在蘇聯解體後，就有不同的運用。1999年台灣發生[[921大地震|921大地震]]，俄羅斯政府即派遣IL-76搭載救災人員及物資飛航台灣，為首次有俄國軍用飛機降落台灣。於2005年8月29日（即是[[2005年颶風卡特里娜|卡特里娜颶風]]消散前一天），俄羅斯決定派遣兩架IL-76執行運送人道救援物資的計劃。並於9月8日抵達了在剛給颶風洗劫完的[[小岩城|小岩城]]機場。而運送第二批物資的IL-76就在9月14日回程。是俄羅斯第一次有同類行動賑濟美國。

[[印度|印度]]也有一架IL-76在9月13日為同樣目的到了[[佛羅里達州|佛羅里達州]]。

飛赴利比亞接運中國人員的4架中國空軍伊留申-76飛機，於2011年3月1日16時許從[[烏魯木齊|烏魯木齊]]起飛，於北京時間3月2日晚22時30分順利降落利比亞{{link-en|薩巴赫機場|Sabha Airport|-{zh-tw:薩巴赫;zh-hant:薩伯哈;zh-hk:拾哈;zh-mo:拾哈;zh-hans:塞卜哈;}-機場}}。途中飛經[[巴基斯坦|巴基斯坦]]、[[阿曼|阿曼]]、[[沙特阿拉伯|沙特阿拉伯]]、[[蘇丹|蘇丹]]、[[利比亞|利比亞]]等5個國家，飛過[[阿拉伯海|阿拉伯海]]和[[紅海|紅海]]，跨越6個時區、8個空中情報管制區，單程空中飛行時間超過12小時以上。中途在[[卡拉奇|卡拉奇]]和[[喀土穆|喀土穆]]加油。<ref>中評社北京3月2日電,"空軍4架伊留申-76飛抵利比亞創最遠航程記"[http://www.chinareviewnews.com/doc/1016/1/4/9/101614951.html], 中國評論新聞,2011-03-02 08:31:03.</ref>

2013年6月，俄羅斯方面宣布將向中國空軍提供十架二手伊留申-76MD軍用運輸機。根據2011年簽署的合同，2013年至2015年期間俄羅斯將向中國提供俄空軍現有的10架伊留申-76MD，並進行維修服務。<ref>{{cite news |url=http://www.airforceworld.com/pla/il-76-transporter-china.htm |title=俄向中國空軍提供十架二手伊留申-76MD |publisher=AirForceWorld.com |accessdate=2013-06-17 |archive-url=https://web.archive.org/web/20130624074440/http://www.airforceworld.com/pla/il-76-transporter-china.htm |archive-date=2013-06-24 |dead-url=yes }}</ref>

=== 海外版本 ===
==== CFTE發動機測試 ====
中國飛行試驗研究院（China Flight Test Establishment-CFTE）由一架IL-76MD作中國「[[渦扇-10|太行]]」發動機的實際測試床。

==== 空警-2000 ====
[[空警-2000|空警-2000]]（北约代号：“主环”[{{lang|en|Mainring}}]，与俄罗斯A-50的北约代号“主桅”[{{lang|en|Mainstay}}]相似），是[[中国人民解放军空军|中国人民解放军空军]]基于A-50平台自行研发的[[预警机|大型预警机]]（AWACS）

==== Adnan I ====
伊拉克開發（法國技術協助），裝設有監察雷達，能監察189海浬範圍。但是在[[海灣戰爭|海灣戰爭]]期间逃亡[[伊朗|伊朗]]，后在阅兵式上和其它飞机相撞坠毁。

==== 费尔康预警机 ====
为[[以色列航太工业|以色列航太工业]]基于A-50平台安装“费尔康”雷达研发的大型预警机（AWACS），原打算售予[[中华人民共和国|中华人民共和国]]，然而在[[美国|美国]]压力下合同取消，该型预警机被改售给印度，被称为[[A-50|A-50I]]。

== 空難及事件 ==

* 1996年11月12日，哈薩克斯坦航空1907號班機和剛在[[英迪拉·甘地國際機場|英迪拉·甘地國際機場]]起飛的[[沙地阿拉伯航空763號班機|沙地阿拉伯航空763號班機]]在新德里附近的哈里亞納邦查基達里上空相撞，兩航機上合計共349人全部罹難，是航空史上最嚴重的空中相撞空難。

* 2005年11月11日一架屬巴基斯坦的伊尔-76，在[[阿富汗|阿富汗]][[喀布爾|喀布爾]]西北30公里撞山。
