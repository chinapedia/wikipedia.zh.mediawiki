'''伊那福岡站'''（{{jpn|j=伊那福岡駅|hg=いなふくおかえき|rm=Ina-fukuoka eki|f=y}}）是位於[[日本|日本]][[長野縣|長野縣]][[駒根市|駒根市]]赤穗福岡的[[東海旅客鐵道|東海旅客鐵道]]（JR東海）[[飯田線|飯田線]]車站。本站平日有一班列車前往岡谷站。

== 車站構造 ==
*相對式月台，2面2線，可以[[列車交會|列車交會]]。
*[[地上車站|地上車站]]。
*由[[伊那市站|伊那市站]]管理的[[無人站|無人站]]。
*設有[[混凝土|混凝土]]造成的站舍和[[候車室|候車室]]。
*車站西南方有一條專用線，供給住友水泥使用。但在1982年（昭和57年）廢用。

== 历史 ==
* 1914年（[[大正|大正]]3年）12月26日 - 伊那電車軌道（1919年改名為[[伊那電氣鐵道|伊那電氣鐵道]]）赤穗站（現在的駒根站）至伊那福岡站延伸段啟用，新設'''伊那福岡站'''，終點站。
* 1915年（大正4年）7月24日<ref>鐵道院『[http://kindai.da.ndl.go.jp/info:ndljp/pid/974223/90 大正四年度 鐵道院年報]』p.145（国立国会図書館近代デジタルライブラリーより）</ref> - 伊那電車軌道伊那福岡站至伊那福岡臨時停車終點站延伸段啟用。車站成為中途站。
* 1918年（大正7年）2月11日 - 伊那電車軌道的伊那福岡臨時停車終點站至飯島站延伸段啟用。伊那福岡臨時停車終點站停用。
* 1943年（[[昭和|昭和]]18年）8月1日 - 伊那電氣鐵道線國有化，成為[[日本國有鐵道|國鐵]]飯田線車站。
* 1971年（昭和46年）12月1日 - 貨物與[[行李|行李]]（不包括專用線的貨物）起卸服裝停止。
* 1982年（昭和57年）11月1日 - 專用線的貨物起卸服裝停止，只有旅客服務。
* 1983年（昭和58年）2月24日 - 車站成為[[無人站|無人站]]。
* 1987年（昭和62年）4月1日 - 因[[國鐵分割民營化|國鐵分割民營化]]，本站成為東海旅客鐵道的車站。
* 2004年（[[平成|平成]]16年）2月 - 現時的候車室完成。
