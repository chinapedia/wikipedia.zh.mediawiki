'''伊蘭羚羊'''（[[學名|學名]]''Taurotragus oryx''），又名'''巨羚'''、'''大羚羊'''，是[[東非|東非]]及[[南部非洲|南部非洲]][[大草原|大草原]]及[[平原|平原]]的一種[[羚羊|羚羊]]。

== 特徵 ==
[[File:Common_Eland.jpg|thumb]]
伊蘭羚羊與[[德氏大羚羊|德氏大羚羊]]一同被認為是最大的[[羚羊|羚羊]]種，不過牠們其實更像[[牛|牛]]。雌羊重300-600公斤，長200-280厘米，肩高125-153厘米。公羊重400-1000公斤，長240-345厘米，肩高150-183厘米。尾巴長約50-90厘米<ref>{{Cite web |url=http://www.arkive.org/common-eland/tragelaphus-oryx/facts-and-status.html |title=Arkive |access-date=2010-05-01 |archive-url=https://web.archive.org/web/20081201023842/http://www.arkive.org/common-eland/tragelaphus-oryx/facts-and-status.html |archive-date=2008-12-01 |dead-url=yes }}</ref><ref>[http://library.thinkquest.org/16645/wildlife/common_eland.shtml Common Eland: Tragelaphus (Taurotragus) oryx] {{webarchive|url=https://web.archive.org/web/20120217221335/http://library.thinkquest.org/16645/wildlife/common_eland.shtml |date=2012-02-17 }}</ref><ref>[http://www.ultimateungulate.com/Artiodactyla/Taurotragus_oryx.html ultimateungulate.com] {{webarchive|url=https://web.archive.org/web/20110109233047/http://ultimateungulate.com/Artiodactyla/Taurotragus_oryx.html |date=2011-01-09 }}</ref>。雌羊呈黃褐色，公羊較為深色，有一點藍灰色，也有一些兩側有白色斑紋。公羊前額的毛髮很密，[[喉袋|喉袋]]很大。公羊及雌羊都有角，角長65厘米。雌羊兩角較闊。

== 生態及行為 ==
伊蘭羚羊棲息在[[南部非洲|南部非洲]]的遼闊[[平原|平原]]及非洲大平原的山腳。牠們吃[[草|草]]、[[樹枝|樹枝]]及[[葉子|葉子]]。牠們是日間活動的，但在太阳照射强烈的時候則不太活动。
一个伊籣羚羊的群体一般有30-80隻，有些甚至超过400隻。它们会根据自身需要而加入或脱离某个群体。
公羊的體型及力量一般足以對抗[[掠食者|掠食者]]，但雌羊則較易受到攻擊<ref name="honzoo">[http://www.honoluluzoo.org/eland.htm Common Eland at the Honolulu Zoo] {{webarchive|url=https://web.archive.org/web/20101216123828/http://www.honoluluzoo.org/eland.htm |date=2010-12-16 }}</ref>。伊蘭羚羊的掠食者包括[[獅子|獅子]]、[[斑鬣狗|斑鬣狗]]、[[非洲野犬|非洲野犬]]及[[獵豹|獵豹]]。

== 分類 ==
伊蘭羚羊有時被認為屬於[[藪羚屬|藪羚屬]]，但通常會與[[德氏大羚羊|德氏大羚羊]]一起被分到[[巨羚屬|巨羚屬]]中。

== 用途 ==
伊籣羚羊会被人类驯养。它们肉质鲜美，这是牠們被獵殺的主要原因。

== 名字 ==
伊蘭羚羊名字中的「伊蘭」来自荷兰语(Eland)，意思是[[駝鹿|駝鹿]]。当初[[荷蘭|荷蘭]]殖民者來到[[開普省|開普省]]看見如此巨大的野生動物時，就用这个名字称呼它们。
