'''伊夫-約瑟夫·德·凱爾蓋朗-特雷馬克'''（{{lang-br|Yves-Joseph de Kerguelen-Trémarec}}，{{bd|1734年|2月13日|1797年|3月3日}}）是[[布列塔尼|布列塔尼]][[探險家|探險家]]及法國海軍官員。

== 早年 ==
凱爾蓋朗生於法國[[非尼斯泰爾省|非尼斯泰爾省]][[朗迪達|朗迪達]]。在[[七年戰爭|七年戰爭]]期間，凱爾蓋朗以[[私掠|私掠者]]的身份在海上從事活動，但沒有取得多少成功。

== 羅科爾 ==
[[Image:Rockall_Kerguelen_map_1771.jpg|thumb]]地區]]
1767年，他在[[羅科爾|羅科爾]]附近航行。儘管他可能沒有直接目視到該島，仍取得了些有用的資訊。凱爾蓋朗所繪製的該島位置僅偏離其正確位置以北6英里，他描述該島在海倫礁附近。<ref name="Fisher, Rockall, 1956" >{{cite book

}}</ref>1771年，他出版了一本該地區的地圖。

== 發現凱爾蓋朗群島 ==
1772年，他航向[[南極|南極]]尋找預言所描述的[[未知的南方大陸|未知的南方大陸]]，而發現了[[凱爾蓋朗群島|凱爾蓋朗群島]]，並為法國帶來許多海外領土。這趟航程中，凱爾蓋朗與自然學家[[讓·紀堯姆·布魯蓋爾|讓·紀堯姆·布魯蓋爾]]隨同前往。

在寫給國王[[路易十五|路易十五]]的報告中，凱爾蓋朗高估了這座荒島的重要性，所以又指揮了第二次的遠征行動前往該島。當時，該島荒涼又無重要性，也不是未知的南方大陸的一部份。在他回國後，因而受到監禁。

在[[法國大革命|法國大革命]]期間，凱爾蓋朗被視為法國[[舊制度_(法國)|舊制度]]中的一名受害者，而恢復他的地位與名譽，後參與[[格魯瓦戰役|格魯瓦戰役]]。1797年，[[海軍少將|海軍少將]]兼[[布雷斯特_(法國)|布雷斯特]]港指揮官的凱爾蓋朗過世。
