'''伊賀國'''（{{jpn|j=伊賀国|kyu=伊賀國|hg=いがのくに|hgt=|rm=Iganokuni|f=y}}），[[日本|日本]]古代的[[令制國|令制國]]之一，屬[[東海道|東海道]]，又稱'''伊州'''，在《[[延喜式|延喜式]]》中国力属于下国，按距离[[京都|京都]]的距离划分则属于近国，領域大約為現在[[三重縣|三重縣]]西部的上野盆地一帶。[[江户时代|江户时代]]，伊贺国境内有{{link-ja|筒井氏|筒井氏}}的{{link-ja|伊賀上野藩|伊賀上野藩}}以及{{link-ja|藤堂氏|藤堂氏}}的[[津藩|津藩]]<ref name=Iga>{{cite web|url=https://kotobank.jp/word/%E4%BC%8A%E8%B3%80%E5%9B%BD-30043|script-title=ja:伊賀国|publisher=朝日新聞社コトバンク|accessdate=2018-09-08|archive-url=https://web.archive.org/web/20180908202507/https://kotobank.jp/word/%E4%BC%8A%E8%B3%80%E5%9B%BD-30043|archive-date=2018-09-08|dead-url=no}}</ref>。

伊贺国與[[近江国|近江国]][[甲賀|甲賀]]同為日本[[忍術|忍術]]的發源地，伊贺国多山的地形是孕育领内忍者的因素之一<ref>{{cite web |title=忍者発祥の地 伊賀・甲賀 |url=http://shinobinosato.com/heritage/list.html |website=忍びの里伊賀甲賀忍者協議会  |accessdate=2018-12-14}}</ref><ref name=Hu2017Iga>{{cite web |author1=胡煒權 |title=《忍者之國》電影外的歷史，信長父子為什麼要進攻伊賀？ |url=https://www.thenewslens.com/article/84658 |website=關鍵評論 |accessdate=2018-12-14 |archive-url=https://web.archive.org/web/20171211035636/https://www.thenewslens.com/article/84658 |archive-date=2017-12-11 |dead-url=no }}</ref>。

==名称的由来==
伊贺国的国名由来存在多种说法。《[[大日本史|大日本史]]》认为，在[[日本神话|日本神话]]中，伊贺国是由[[伊势津彦|伊势津彦]]的女儿吾娥治理，因此原来称为“吾娥”，后来的名字伊贺是讹称<ref name=Tokugawa1909290>{{cite book |author1=德川光圀 |author2=etal |title=大日本史 第142-143 卷290-292 志 |date=1907 |publisher=徳川总子|doi=10.11501/769984|url=http://dl.ndl.go.jp/info:ndljp/pid/769984}}</ref>。另有说法认为，伊贺二字由当地古代[[阿伊努语|阿伊努语]]名字转写而成<ref>{{cite web |author1=澤田謙三 |script-title=ja:伊賀地方のエゾ語系地名 |url=http://www.iwashinbun.co.jp/data/iwanp/ezogo/index.html |website=伊賀の生活情報誌 |accessdate=2018-09-08 |archive-url=https://web.archive.org/web/20080604213617/http://www.iwashinbun.co.jp/data/iwanp/ezogo/index.html |archive-date=2008-06-04 |dead-url=no }}</ref>。

==历史文化==
伊贺国以及[[近江国|近江国]][[甲賀|甲賀]]同為日本[[忍術|忍術]]的發源地，来自伊贺和甲贺的忍者分别称为“伊贺众”和“甲贺众”。伊贺众是伊贺国内从事忍者工作的[[地侍|地侍]]，据传曾在[[本能寺之变|本能寺之变]]后帮助身陷敌国的[[德川家康|德川家康]]{{Link-ja|越過伊賀|伊賀越え|逃脱}}。在[[江户时代|江户时代]]也十分活跃<ref name=Igamono>{{cite web |script-title=ja:伊賀者 |url=https://kotobank.jp/word/伊賀者-30049 |accessdate=2018-12-14||publisher=朝日新聞社コトバンク}}</ref>。伊贺国领内多山，促进了领内忍者众的形成<ref name=Hu2017Iga/>。观点认为，伊贺国内的忍者起源于[[镰仓幕府|镰仓幕府]]末期在当地活动的“{{link-ja|恶党|悪党}}”（反对镰仓幕府的叛军）<ref name="小泉宜右Koizumi1981">{{cite book|author1=小泉宜右|author2=Yoshiaki Koizumi|title=悪党|url=https://books.google.com/books?id=-volAAAAMAAJ|year=1981|publisher=敎育社|page=194}}</ref>。

在日本[[律令制|律令制]]时代之前，统治伊贺国的是伊贺[[国造|国造]]。过去的[[伊势国|伊势国]]，领土曾包含《[[延喜式|延喜式]]》中伊贺国的领地，伊贺国是在公元680年从伊势国分立的。律令制时代后，伊贺国领内多为[[东大寺|东大寺]]等寺社持有的[[庄园|庄园]]。镰仓时代，伊贺一带的守护先后由[[大内氏|大内氏]]、[[山内氏|山内氏]]、[[千葉氏|千葉氏]]担任。镰仓幕府末期，在伊贺一带活动的恶党演化成了后来的伊贺忍者。[[日本战国时代|日本战国时代]]中，1581年爆发了{{link-ja|天正伊贺之乱|天正伊賀の乱}}，[[织田氏|织田氏]]的军队入侵，吞并了伊贺。伊贺国的忍者不得不逃亡他国，为各地的[[大名_(称谓)|大名]]工作。进入江户时代后，伊贺国的原领主{{link-ja|筒井氏|筒井氏}}因在[[大坂冬之阵|大坂冬之阵]]中涉嫌通敌而被改易，一部分领地划入了{{link-ja|藤堂氏|藤堂氏}}的[[津藩|津藩]]<ref name=Iga/><ref name=Hu2017Iga/><ref name=Igamono/>。

==行政区划==
伊贺国{{link-ja|国府|国衙}}的国厅位于今[[三重县|三重县]][[伊贺市|伊贺市]]坂之下一带，按古代的行政区划，在[[阿拜郡|阿拜郡]]境内，遗迹{{link-ja|伊贺国厅遗迹|伊賀国庁跡}}在2009年获日本政府指定为史迹<ref name=IgaA>{{cite web|url=https://kotobank.jp/word/%E4%BC%8A%E8%B3%80%E5%9B%BD%E5%BA%81%E8%B7%A1-1443180|script-title=ja:伊賀国庁跡|publisher=朝日新聞社コトバンク|accessdate=2018-09-08|title=存档副本|archive-url=https://web.archive.org/web/20171121125844/https://kotobank.jp/word/%E4%BC%8A%E8%B3%80%E5%9B%BD%E5%BA%81%E8%B7%A1-1443180|archive-date=2017-11-21|dead-url=no}}</ref>，国分寺和国分尼寺都位于今三重县伊賀市境内。伊贺国的一宫（境内《延喜式》社格最高的神社）是今属三重县伊賀市的{{link-ja|敢国神社|敢国神社}}<ref name=Aekuni/>。
