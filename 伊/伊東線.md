'''伊東線'''（{{jpn|j=伊東線|hg=いとうせん|rm=Itō sen|f=y}}）是一條連結[[靜岡縣|靜岡縣]][[熱海市|熱海市]][[熱海站|熱海站]]與靜岡縣[[伊東市|伊東市]][[伊東站|伊東站]]，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）的鐵路線（{{link-ja|幹線|幹線}}）。
== 概要 ==
雖然伊東線的終點站[[伊東站|伊東站]]是JR鐵路網的末端車站，但該站同時也是私鐵[[伊豆急行|伊豆急行]]的[[伊豆急行線|伊豆急行線]]之起點車站，因此伊東線實際上並不算是末端鐵路線的一類，兩家公司的列車也都會互相[[直通運行|駛入對方的路線中]]。

根據{{link-ja|旅客營業規則|旅客営業規則}}包括在[[大都市近郊區間_(JR)|大都市近郊區間]]的「東京近郊區間」與[[IC卡|IC]][[乘車卡|乘車卡]]「[[Suica|Suica]]」的首都圈地域內。

此線也是通過靜岡縣內的JR線當中唯一一條完全位於縣內的路線<ref>過去還有獲指定為[[特定地方交通線|特定地方交通線]]的{{link-ja|清水港線|清水港線}}與轉換至[[日本第三部門鐵道|第三部門]]的[[天龍濱名湖線|二俁線]]。</ref>。同時此線也是靜岡縣內JR東日本管轄當中唯一一條位於東海道本線的{{link-ja|丹那隧道|丹那トンネル}}以東的路線。

== 路線資料 ==
* 管轄：東日本旅客鐵道（[[鐵路公司|第一種鐵道事業者]]）、[[日本貨物鐵道|日本貨物鐵道]]（第二種鐵道事業者）
* 路線距離（{{link-ja|營業距離|営業キロ}}）：16.9公里
* [[軌距|軌距]]：1067毫米
* 站數：6個（包括起終點站）
** 若只限屬於伊東線的車站，排除屬於東海道本線的熱海站<ref>『停車場変遷大事典 国鉄・JR編』[[ジェイティービー|JTB]] 1998年 ISBN 978-4533029806</ref>則為5個。
* 複線路段：熱海－來宮（1968年完成）
* 電氣化路段：全線（直流1500V、[[高架電纜|高架電纜]]）
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
* 保安裝置：[[自動列車停止裝置#ATS-P形（デジタル伝送パターン形）|ATS-PN]]
* {{link-ja|運轉指令所|運転指令所}}：熱海[[調度集中系統|CTC]]
* 最高速度：95公里/小時
* 最長隧道：新宇佐美隧道（2941米）

全線由{{link-ja|東日本旅客鐵道横濱支社|東日本旅客鉄道横浜支社|橫濱支社}}管轄。

== 運行形式 ==
大部分「普通」班次會向南駛入伊豆急行線直通運作。另外一部分由伊東站開出的「普通」班次會在熱海站，駛入東海道線及[[上野東京線|上野東京線]]直通運行，最遠可駛到[[宇都宮車站|宇都宮]]、[[籠原站|籠原]]等車站，故伊東站為上野東京線最南端的終點及起點站。
