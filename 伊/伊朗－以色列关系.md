'''伊朗－以色列关系'''，是指[[伊朗|伊朗伊斯兰共和国]]和[[以色列|以色列国]]的双边关系，广义上也包括历史上[[波斯人|波斯民族]]与[[犹太人|犹太民族]]间的关系。波斯人与犹太人同属中东地区古老民族，在历史上有着密切的关系，[[巴列维王朝|伊朗帝国]]曾经是[[穆斯林世界|穆斯林世界]]中第二个承认以色列的国家。然而[[伊朗伊斯兰革命|伊朗伊斯兰革命]]后，两国双边关系急剧恶化，伊朗拒絕承认以色列国的存在，当前伊朗伊斯兰共和国与以色列国之间无正式[[外交|外交关系]]。

== 古代时期 ==
=== 巴比伦囚徒 ===
[[File:Cyrus_II_le_Grand_et_les_Hébreux.jpg|thumb]]
{{see also|巴比伦囚虏}}
以色列的犹太民族和伊朗的波斯民族都是中东地区的古老民族，两个民族的交往可以追溯到2500年前。[[公元前11世纪|公元前11世纪]]犹太人在[[中东|中东]][[黎凡特|黎凡特]]地区建立了[[以色列王国_(前期)|以色列王国]]，王国后来分裂为北部的[[以色列王国_(后期)|以色列王国]]和南部的[[犹太王国|犹太王国]]。公元前772年，以色列王国被[[亚述王国|亚述王国]]所灭，犹大王国亦臣服于亚述王国。公元前586年，[[新巴比伦王国|新巴比伦王国]]国王[[尼布甲尼撒二世|尼布甲尼撒二世]]攻克犹太王国都城[[耶路撒冷|耶路撒冷]]，[[所罗门圣殿|所罗门圣殿]]（即第一圣殿）惨遭焚毁，犹太人被押往巴比伦为奴，史称“[[巴比伦之囚|巴比伦之囚]]”<ref name="huanqiu">{{cite web|author1 = 张健|title = 以色列与伊朗：从恩人盟友到对头|url = http://history.huanqiu.com/world/2015-03/5834789.html|accessdate = 2015-11-26|language = zh|archive-url = https://web.archive.org/web/20151127062025/http://history.huanqiu.com/world/2015-03/5834789.html|archive-date = 2015-11-27|dead-url = yes}}</ref>。

波斯人在[[公元前2000年|公元前2000年]]进入了今天伊朗的境内。公元前550年波斯部族首领[[居鲁士|居鲁士]]建立了[[波斯帝国|波斯帝国]]，公元前538年波斯帝国攻占了[[巴比伦城|巴比伦城]]，新巴比伦王国（迦尔底亚帝国）灭亡。居鲁士颁旨，释放“巴比伦之囚”，允许他们回归故土。得益于居鲁士的决定，犹太人重返圣地耶路撒冷，并重建了圣殿（即[[第二圣殿|第二圣殿]]）。居鲁士对犹太人的“解放”正好与犹太人的[[救世主|救世主]]传说巧合，这一事件也促进了[[犹太教|犹太教]]的形成，居鲁士因而被犹太人尊称为“涂圣油的王”。

虽然，居鲁士在犹太第二圣殿建成前就去世了。但是大流士继位后，依然准许犹太人继续修建第二圣殿。根据圣经的记载，先知[[哈该|哈该]]<ref>《[[以斯拉記|以斯拉記]]》6:15</ref>和[[撒迦利亚|撒迦利亚]]<ref>《以斯拉記》5:1</ref>负责督促圣殿的建设工作。

根据《[[圣经|圣经]]·[[旧约|旧约]]·[[以斯帖记|以斯帖记]]》记载，古代波斯帝国君王[[亚哈随鲁|亚哈随鲁]]（一般认为就是[[薛西斯一世|薛西斯一世]]<ref>《袖珍聖經辭典》第68條：亞哈隨魯(Ahasuerus)</ref>）的大臣[[哈曼|哈曼]]打算杀死波斯国内所有的犹太人，他通过抽签决定在[[亚达月|阿达尔月]]行动。犹太人[[末底改|末底改]]和他的养女皇后[[以斯帖|以斯帖]]阻止了他的计划，说服国王处死了哈曼和他的家人。犹太人为了纪念以斯帖皇后将犹太人从灭族的险境中解救出来，创立了[[普珥节|普珥节]]<ref>{{cite book|last = Johnson|first = Sara Raup|title = Historical Fictions and Hellenistic Jewish Identity: Third Maccabees in Its Cultural Context|year = 2005|publisher = University of California Press|isbn = 978-0-520-23307-2|url = http://books.google.com/?id=mfiJ6foYhMAC&pg=PA17&dq=ahasuerus+xerxes#v=onepage&q=ahasuerus%20xerxes&f=false|pages = 16–17}}</ref>。
=== 犹太人大流散 ===
公元前330年，波斯帝国（[[阿契美尼德王朝|阿契美尼德王朝]]）被[[亚历山大大帝|亚历山大大帝]]所灭。而复国后的犹太人也历经了亚历山大大帝的入侵、[[托勒密王朝|托勒密王朝]]的管辖和[[塞琉古帝国|塞琉古帝国]]的统治。公元前65年，巴勒斯坦地区被罗马帝国占领，犹太人的国家不复存在<ref name="huanqiu">{{cite web|author1 = 张健|title = 以色列与伊朗：从恩人盟友到对头|url = http://history.huanqiu.com/world/2015-03/5834789.html|accessdate = 2015-11-26|language = zh|archive-url = https://web.archive.org/web/20151127062025/http://history.huanqiu.com/world/2015-03/5834789.html|archive-date = 2015-11-27|dead-url = yes}}</ref>。罗马帝国对统治下的犹太人起义进行了[[镇压|镇压]]，导致犹太人开始大规模地离开故土。而拜占庭帝国时期，犹太人的境遇变得更为恶劣，当地政府不允许犹太人建造宗教礼拜的场所，也不批准任何犹太宗教机构的设立<ref>Johnson (1987), pp. 162–165.</ref>。基于对[[弥赛亚|弥赛亚]]再度降临的信仰，犹太人与侵入[[巴勒斯坦地区|巴勒斯坦地区]]的波斯人联盟，一度收复了被拜占庭帝国统治的耶路撒冷，并统治了该城三年时间。但此后波斯帝国与拜占庭皇帝[[希拉克略|希拉克略]]議和，[[基督教|基督教]]再次占据巴勒斯坦的统治地位，而之前幸存下来的犹太人再次被驱逐出耶路撒冷，开始了流亡，直到二十世纪的以色列建国<ref>Goldenberg (2007), pp. 185–186.</ref>。

== 以色列复国和巴列维王朝时期  ==
=== 以巴分治 ===
数个世纪以来，诸多流亡海外的犹太人一直试图返回巴勒斯坦地区。[[二战|二战]]后，[[犹太复国主义运动|犹太复国主义运动]]获得了越来越多的国际支持。1947年，[[联合国巴勒斯坦问题特别调查委员会|联合国巴勒斯坦问题特别调查委员会]]（UNSCOP）成立，伊朗成为其11个成员国之一。委员会中有8个成员国支持巴勒斯坦分治计划，而伊朗、[[印度|印度]]、[[南斯拉夫|南斯拉夫]]则认为巴勒斯坦分治会导致地区暴力冲突升级而反对分治计划。1947年11月[[联合国大会|联合国大会]]以33国赞成，13国反对（包括伊朗），10国弃权，表决通过了[[联合国大会181号决议|巴勒斯坦分割方案]]<ref>{{cite web |url = http://domino.un.org/unispal.nsf/0/7f0af2bd897689b785256c330061d253|title = A/RES/181(II) of 29 November 1947|publisher = United Nations|year = 1947}}</ref>。1948年5月14日，英国停止[[巴勒斯坦托管地|对巴勒斯坦的托管]]，当天以色列宣布建国，但是次日阿拉伯国家即组成联军[[第一次中东战争|入侵以色列]]。出于共同对[[泛阿拉伯民族主义|泛阿拉伯民族主义]]的担忧，在战争中，伊朗对以色列抱有一定程度的同情。伊朗虽然投票反对以色列加入[[联合国|联合国]]，但同时也表示不愿介入[[阿以冲突|阿以冲突]]<ref name="huanqiu">{{cite web|author1 = 张健|title = 以色列与伊朗：从恩人盟友到对头|url = http://history.huanqiu.com/world/2015-03/5834789.html|accessdate = 2015-11-26|language = zh|archive-url = https://web.archive.org/web/20151127062025/http://history.huanqiu.com/world/2015-03/5834789.html|archive-date = 2015-11-27|dead-url = yes}}</ref>。

=== 建立关系 ===
1948年以色列建国后，以色列曾利用伊朗作为一个中转站，把伊拉克的犹太人送往以色列。1950年3月14日，伊朗给予以色列国“事实上的承认”<ref name="wh">{{cite journal|author1 = 何志龙|title = 20 世纪伊朗与以色列关系评析|journal = 世界历史|date = 2007年|issue = 第4期|pages = 92-100|language = zh}}</ref>，成为继[[土耳其|土耳其]]之后，穆斯林世界中第二个承认以色列的国家<ref>[http://www.fouman.com/Y/Get_Iranian_History_Today.php?artid=1132 The Iranian History Article : Shah Censors Israel Relations]</ref>。但两国建立正式外交关系的努力受到了来自[[阿拉伯国家|阿拉伯国家]]和伊朗国内宗教保守势力的强烈反对<ref name="huanqiu">{{cite web|author1 = 张健|title = 以色列与伊朗：从恩人盟友到对头|url = http://history.huanqiu.com/world/2015-03/5834789.html|accessdate = 2015-11-26|language = zh|archive-url = https://web.archive.org/web/20151127062025/http://history.huanqiu.com/world/2015-03/5834789.html|archive-date = 2015-11-27|dead-url = yes}}</ref>，加之[[摩萨台|摩萨台]]政府与[[英国|英国]]关系恶化，1951年7月7日伊朗关闭了驻以使馆，并断绝与以色列外交关系，但是仍然承认以色列国，并继续保持了双边交流。

=== 蜜月时期 ===
[[File:Military_relations_of_Israel_and_Iran.jpg|thumb]]
1953年[[1953年伊朗政變|美英发动政变]]推翻了摩萨台政府。伊朗进一步加强了与以色列的关系，借由以色列的力量遏制[[贾迈勒·阿卜杜-纳赛尔|纳赛尔]]革命后兴起的[[阿拉伯民族主义|阿拉伯民族主义]]势力，维护伊朗的利益。1958年以色列提出了“外围联盟”的策略，希望与包括伊朗、土耳其、埃塞俄比亚在内的周边非阿拉伯国家结成联盟，共同应对来自激进的阿拉伯国家和[[苏联|苏联]]的威胁<ref name="wh">{{cite journal|author1 = 何志龙|title = 20 世纪伊朗与以色列关系评析|journal = 世界历史|date = 2007年|issue = 第4期|pages = 92-100|language = zh}}</ref>。而1963年巴列维开始推行的以“[[土地改革|土地改革]]”为中心的“[[白色革命|白色革命]]”导致了伊朗国内的不稳定，穆斯林宗教势力抬头。与此同时，伊朗与阿拉伯国家的关系也不断恶化。共同面临的安全问题和共同的利益需要成为伊以两国关系繁荣发展的根本推动力，这一时期以色列与伊朗关系密切，政治、经济、军事等领域相互支持。双方人员互访频繁，以色列领导人[[戴维·本-古里安|本-古里安]]、[[梅厄夫人|梅厄夫人]]、[[贝京|贝京]]、[[摩西·达扬|达扬]]、[[希蒙·佩雷斯|佩雷斯]]和[[伊扎克·拉宾|拉宾]]等都曾到访问过伊朗<ref>萨缪尔·塞格沃:伊朗三角:伊朗门事件中的以色列揭秘(SamuelSegev,TheIranianTriangle:TheUntoldStoryofIs-raelRoleintheIran-ContraAffair),自由出版社1988年版,前言第2页。</ref>。[[以色列航空|以色列航空]]开辟了[[特拉维夫|特拉维夫]]和[[德黑兰|德黑兰]]之间的直航业务，也增进了两国人民的交往<ref>{{cite journal|title = World airline directory{{spaced ndash}}El Al Israel Airlines|journal = Flight International|date = 28 April 1979|page = 1370|url = http://www.flightglobal.com/pdfarchive/view/1979/1979%20-%201432.html|archiveurl = https://www.webcitation.org/6BCASkCC8?url=http://www.flightglobal.com/pdfarchive/view/1979/1979%20-%201432.html|archivedate = 2012-10-05|accessdate = 5 October 2012|author = |dead-url = no}}</ref>。

经济上，面对周边阿拉伯产油国的封锁，伊朗成为了以色列最大的[[石油|石油]]来源，[[六日战争|六日战争]]之后直到伊斯兰革命之前，伊朗一直通过[[埃拉特阿什凯隆管道|埃拉特阿什凯隆管道]]借由[[地中海|地中海]]的油船向以色列供应石油<ref>{{cite book |first = Daniel|last = Ammann|title = The King of Oil: The Secret Lives of Marc Rich|publisher = St. Martin‘s Press|location = New York|year = 2009|isbn = 0-312-57074-0}}</ref><ref>Bialer, Uri. ''Fuel Bridge across the Middle East—Israel, Iran, and the Eilat-Ashkelon Oil Pipeline.'' In: Israel Studies, Vol 12, No 3 (Fall 2007)</ref>。1959年到1971年，以色列80%至90%的原油来自伊朗，而1976年伊朗则提供了大约以色列75%的石油需求。[[第三次中东战争|第三次中东战争]]后，以色列从伊朗进口的石油不仅可以满足本国需求，而且还出口[[西欧|西欧]]<ref>{{cite journal|author1 = Mark Tessler|title = Israel arms exports and Iran:Some aspects of Israeli strategic thinking|date = 1989|publisher = Arab Studies Quarterly}}</ref>。同样，以色列也帮助伊朗发展农业技术并向伊朗出口武器装备<ref name="huanqiu">{{cite web|author1 = 张健|title = 以色列与伊朗：从恩人盟友到对头|url = http://history.huanqiu.com/world/2015-03/5834789.html|accessdate = 2015-11-26|language = zh|archive-url = https://web.archive.org/web/20151127062025/http://history.huanqiu.com/world/2015-03/5834789.html|archive-date = 2015-11-27|dead-url = yes}}</ref>。

在军事上，以色列帮助伊朗建立和训练[[情报机构|特务机构]]“[[萨瓦克|萨瓦克]]”，并进行军事和情报合作。甚至在[[20世纪70年代|20世纪70年代]]，以色列曾提议与伊朗联合研究[[核武器|核武器]]计划，但是[[伊朗国王|伊朗国王]][[巴列维|巴列维]]认为伊朗没有必要开发核武器，因此拒绝了以色列的提议。两国虽在核武器问题上观点不一，但却签署了代号为“[[鲜花工程|鲜花工程]]”的联合开发可以携带核弹头导弹的秘密计划，该计划因伊朗发生革命而中止<ref>{{cite web |url = http://www.nti.org/country-profiles/iran/delivery-systems/|title = Iran|date = November 2011|publisher = Nuclear Threat Initiative|accessdate = 26 July 2012}}</ref><ref>For details on Iran's strategic reasoning during the 1970s, see "Israel and the Origins of Iran’s Arab Option: Dissection of a Strategy Misunderstood," Middle East Journal, Volume 60, Number 3, Summer 2006.</ref>。1974年，伊朗在众多阿拉伯国家和部分穆斯林国家反对的情况下，力排众议坚持让以色列参加在德黑兰举办的[[第七届亚洲运动会|第七届亚洲运动会]]。在[[阿以冲突|以色列与阿拉伯国家的战争]]中，伊朗[[穆罕默德-礼萨·巴列维|巴列维]]政府都基本上保持[[中立|中立]]，甚至对以色列军队在“[[第三次中东战争|六·五战争]]”中的成就表示钦佩。在[[贖罪日戰爭|十月战争]]后，[[戴维营协议|埃以和平进程]]中，巴列维也尽了绵薄之力。在1970年代正式派驻大使前，以色列一直在德黑兰设有一个[[实质大使馆|实质性的使馆]]<ref>[http://www.pbs.org/wgbh/pages/frontline/tehranbureau/2010/10/iran-primer-iran-and-israel.html Iran Primer: Iran and Israel by Steven Simon], [[公共电视网|PBS]]</ref><ref>[http://www.timesofisrael.com/?p=728128 Did Israel, under the shah, help start Iran’s nuclear program?], ''The Times of Israel'', 1 November 2013</ref>。

== 伊朗伊斯兰共和国时期 ==
=== 伊斯兰革命初期 ===
{{see also|伊朗伊斯兰革命}}
1963年1月，伊朗当局宣布实行“白色革命”，由于触及了宗教势力的[[金權政治|既得利益]]，遭到了[[霍梅尼|霍梅尼]]为首的宗教保守势力强烈反对。此后，霍梅尼领导伊朗国内宗教势力开展了一系列反国王和反世俗集会，并将矛头同时指向了与伊朗国王当局间关系甚密的美国和以色列。霍梅尼公开宣称，以色列是“伊斯兰教的敌人”，美国是“大撒旦”，而以色列是“小撒旦”<ref>{{cite news|title=Iran to examine Holocaust evidence|url=http://www.aljazeera.com/archive/2006/09/200849141128532972.html|work=Al Jazeera|date=3 September 2006 |accessdate=27 July 2012}}</ref>。

1979年1月，伊朗发生伊斯兰革命，巴列维国王被迫流亡国外，宗教领袖霍梅尼建立了[[政教合一|政教合一]]的[[伊斯兰共和国|伊斯兰共和国]]，切断了与以色列的一切官方联系，并采取[[反犹太主义|反犹太主义]]政策，对以色列实施制裁。2月18日伊以正式断交<ref name=usip>{{cite web|last=Nikou|first=Semira N.|title=Timeline of Iran's Foreign Relations|url=http://iranprimer.usip.org/resource/timeline-irans-foreign-relations|work=United States Institution of Peace|accessdate=6 August 2013}}</ref>，伊朗当局关闭了以色列驻德黑兰大使馆，并将其移交给了[[巴勒斯坦解放组织|巴勒斯坦解放组织]]<ref name=bahram>{{cite journal|last=Alavi|first=Bahram|title=Khomeini's Iran: Israel's Ally|journal=Washington Report|date=April 1988|series=Special reports|url=http://www.wrmea.org/wrmea-archives/324-washington-report-archives-1988-1993/april-1988/9811-khomeinis-iran-israels-ally.html|accessdate=3 August 2013}}</ref>。

=== 两伊战争期间 ===
[[两伊战争|两伊战争]]的爆发，伊朗和阿拉伯国家均无暇顾及以色列，这大大减轻了以色列的压力。相较于伊朗，以色列更担心是的[[伊拉克|伊拉克]]，一旦后者获得了两伊战争的胜利，必将极大地鼓舞阿拉伯民族主义，对以色列构成更大的威胁<ref>Daniel Gavron.An Unendi g War.''Jerusalem Post''.17 April 1987</ref>。1980年9月28日，两伊战争爆发后的第9天，以色列国防部长公开宣称，以色列愿向伊朗提供援助以换取伊朗对以立场的改变。以外长达扬也力促美国摒弃前嫌，对伊朗提供援助<ref>特里塔·帕尔西.《以色列与伊朗关系评估: 从力量循环看其战略竞争》,第255页</ref>。1980年10月，达扬进一步强调，如不及时向伊朗提供武器援助，伊拉克有可能摧毁伊朗大部分军事力量。在以色列的游说下，美国默许以色列向伊朗提供美制武器及部件，甚至秘密开展麦克法兰计划，通过第三方渠道直接向伊朗提供武器<ref> ''Washington Post''.29 November 1986</ref>，导致[[伊朗门事件|伊朗门事件]]的发生。据一位伊朗前官员透露，在以色列情报人员的帮助下，伊朗曾试图轰炸伊拉克的核设施，但未能如愿<ref>加利·西科.《十月惊雷: 伊朗的美国人质与罗纳德·里根的总统竞选》,第114页</ref>。1981年6月7日，以色列通过“[[巴比伦行动|巴比伦行动]]”，摧毁了巴格达附近的奥斯拉克核反应堆。甚至直到1987年，以色列总理拉宾还表示，“伊朗是我们最好的朋友，我们不打算改变我们的立场”<ref>{{cite web|title=When Iran and Israel were friendly|url=http://www.salon.com/2012/02/15/when_iran_and_israel_were_friendly/|accessdate=2015-11-26}}</ref>。

1981年，以色列通过第三方公司，向伊朗出售了价值7500万美元的武器。材料包括150挺M-40无后座炮（每一铤含24000枚炮弹），[[坦克|坦克]]和[[战斗机|飞机]]发动机零部件，106毫米、130毫米、203毫米和175毫米炮弹、[[BGM-71陶式导弹|BGM-71陶式导弹]]<ref>[http://books.google.com/books?hl=en&id=NkxZcHL1xdYC&dq=Ronen+Bergman,+The+Secret+War+with+Iran&printsec=frontcover&source=web&ots=hkVy43U066&sig=Wff_fjthnKb0ZjEFFDogl3PLPAw&sa=X&oi=book_result&resnum=3&ct=result#PPA45,M1 Ronen Bergman, ''The Secret War with Iran''], Free Press, 2008, p.40-48</ref>，甚至包括[[F-4幽靈II戰鬥機|f-4战斗机]]配件<ref name="Scott, Peter Dale 1987, p. 173-174">Scott, Peter Dale, ''The Iran-Contra Connection: Secret Teams and Covert Operations in Reagan Era'', 1987, p. 173-174</ref>。根据[[臺拉維夫大學|特拉维夫大学]]杰夫战略研究所估计，1981年到1983年期间，以色列对伊军售总额大约有5亿美元，大多数是以伊朗石油的方式支付给以色列。据服务于伊朗政府的军火商艾哈迈德·哈达里表示，两伊战争爆发后，伊朗大约有80%的武器购自以色列<ref name="Parsi">Parsi, Trita ''Treacherous Alliance: The secret dealings of Israel, Iran and the United States'', by Trita Parsi, Yale University Press, 2007</ref>。而两伊战争期间，以色列向伊朗派遣了至少一百名顾问和工程师驻扎在德黑兰北部的秘密营区<ref>Bulloch, John, ''The Gulf War : Its Origins, History and Consequences'' by John Bulloch and Harvey Morris, London : Methuen London, 1989, p.17</ref>。

两伊战争期间，伊朗对以色列政策存在两面性，即公开言论指责，秘密武器交易<ref name="wh">{{cite journal|author1 = 何志龙|title = 20 世纪伊朗与以色列关系评析|journal = 世界历史|date = 2007年|issue = 第4期|pages = 92-100|language = zh}}</ref>。对此，霍梅尼宣称，伊朗通过第三方购买以色列制造的武器完全合法，而关于伊以武器交易的说法是“[[錫安主義|犹太复国主义]]的谎言”<ref>马苏尔·法尔汗.《伊朗与以色列关系》.第85-98页</ref>。在此期间，两国军队也发生过摩擦，[[第五次中东战争|1982年以色列入侵黎巴嫩]]后，500名[[伊朗革命卫队|伊朗革命卫队]]成员进入[[黎巴嫩|黎巴嫩]]直接同以色列发生战斗，同时，伊朗也通过武装、培训和资助[[真主党|黎巴嫩真主党]]打击以色列<ref>彭树智主编,王新刚著.《中东国家通史 叙利亚和黎巴嫩卷》.商务印书馆.2003年版.第390、365页</ref>。

=== 两伊战争后至今 ===

两伊战争和[[海湾战争|海湾战争]]相继结束，使得阿拉伯世界发生分裂，伊拉克国力大伤，加之[[苏东剧变|苏东剧变]]，使得伊以利益的共同点减少，两国由秘密合作走上公开对抗。1989年6月3日霍梅尼病逝，总统[[哈梅内伊|哈梅内伊]]接任[[伊朗最高领袖|伊朗最高领袖]]。

而历任伊朗伊斯兰共和国政府依然坚持对以色列不予承认的原则。1993年9月，巴以签订《[[奥斯陆协议|奥斯陆协议]]》，伊朗总统[[拉夫桑贾尼|拉夫桑贾尼]]指责“[[巴解组织|巴解组织]]犯下了卖国罪”，造成了伊斯兰世界的分裂，并呼吁[[吉哈德|吉哈德]]对以色列发动[[聖戰|圣战]]<ref name="wh"/>。1994年11月25日，属于伊朗革命卫队的学生和示威者强行闯入巴解组织驻德黑兰代表萨拉赫·扎瓦维的馆舍，称巴解组织是“美国和以色列的代理人”，伊朗遂与巴解组织断交<ref>{{cite web|title=陈天社：伊朗与哈马斯关系探析|url=http://www.21ccom.net/articles/qqsw/qqgc/article_2013120296282.html|date=2013-12-02|accessdate=2015-11-26|language=zh|publisher=[[共识网|共识网]]|archiveurl=https://web.archive.org/web/20160411203211/http://www.21ccom.net/articles/qqsw/qqgc/article_2013120296282.html|archivedate=2016-04-11|deadurl=yes}}</ref>。

1998年5月3日，哈梅内伊在德黑兰会见[[哈马斯|哈马斯]]领导人[[亚辛|亚辛]]时表示，伊朗将继续支持哈马斯的反以斗争，并称以色列是殖民势力在中东地区留下的恶性肿瘤。1999年4月，伊朗以为以色列和美国充当间谍为由逮捕了13名伊朗犹太人，其中10人被判处徒刑<ref>{{cite web|title=伊朗密审“犹太人间谍案”|url=http://www.people.com.cn/GB/channel2/490/20000415/38897.html|website=人民网|accessdate=2015-11-26|language=zh}}</ref>。2001年1月3日，以色列军队在公海上拦截了一艘伊朗货轮，声称该货轮向巴勒斯坦民族权力机构秘密运送武器，并指出伊朗是巴以和平进程的主要威胁，是以色列真正的敌人<ref>Yossi Olmert:The Real Enemy Is Iran. ''Jerusalem Post''.10 Jan 2001</ref>。

2005年，伊朗强硬派总统[[艾哈迈迪内贾德|艾哈迈迪内贾德]]上台后，推行更为强硬的对以政策。2006年爆发的[[2006年以黎冲突|以黎冲突]]更是被认为是伊朗在黎巴嫩发动的代理人战争，[[伊朗革命卫队|伊朗革命卫队]]被认为直接协助[[真主党|真主党]]武装攻击以色列。据称，冲突中有6到9名革命卫队成员在战争中被以色列军队杀害<ref>{{cite web|title=Iranian soldiers join Hizbullah in fighting|url=http://www.ynetnews.com/Ext/Comp/ArticleLayout/CdaArticlePrintPreview/1,2506,L-3280446,00.html|accessdate=2015-11-26}}</ref>。2009年9月18日，内贾德发表讲话称，质疑[[犹太人大屠杀|犹太人大屠杀]]，声称“[[納粹大屠殺|犹太人大屠杀]]”是一个“谎言”，是以色列为建立一个犹太国家而捏造出来的“托辞”，伊朗人民有宗教上的责任对此进行反抗<ref>{{cite web|title=内贾德：“犹太人大屠杀”是以色列为建国所捏造谎言|url=http://news.xinhuanet.com/world/2012-08/18/c_123598970.htm|website=新华网|accessdate=2015-11-26|language=zh}}</ref>，内贾德于2012年3月19日再度宣称此番言论<ref>{{cite web|title=内贾德再次否认犹太人大屠杀 无惧西方国家制裁伊朗|url=http://gb.cri.cn/27824/2012/03/20/2805s3608070.htm|website=国际在线|accessdate=2015-11-26|language=zh}}</ref>。2012年8月17日，内贾德公开称，以色列是—颗即将被消灭的“毒瘤”<ref>{{cite web|title=内贾德又对以色列放狠话|url=http://news.xinhuanet.com/world/2012-08/18/c_123598970.htm|website=新华网|accessdate=2015-11-26|language=zh}}</ref>。

== 伊朗犹太人 ==
[[File:Kalimi_mashrutiat.JPG|thumb]]两周年。]]
伊朗是中东地区除以色列以外犹太人最多的国家，在1979年伊朗革命前，该国有10万左右的犹太人。在革命后，小部分犹太人逃离到以色列或[[欧洲|欧洲]]，大多数撤离到了美国，其中[[洛杉矶|洛杉矶]]是主要目的地之一，当地的犹太人社群被称为“Tehrangeles”（德黑蘭磯）<ref>Littman (1979), p. 5.</ref>。根据伊朗政府2011年人口普查显示，伊朗国内犹太人数量在9000人左右，但有数据指出犹太人的数量应该在2万到2万5千人。伊朗的犹太人多集中在德黑兰、[[伊斯法罕|伊斯法罕]]等大城市，伊斯法罕有伊朗历史上最古老的犹太人社区。[[伊斯蘭議會|伊朗议会]]为犹太代表保留了一个席位，而犹太代表的政治理念是附和于伊朗政府的，持反以色列和反锡安主义立场<ref>{{cite web|title=伊朗犹太人不为人知的四个方面|url=http://int.gmw.cn/2013-09/14/content_8906215.htm|website=光明网|accessdate=2015-11-26|language=zh|archive-url=https://web.archive.org/web/20151127050236/http://int.gmw.cn/2013-09/14/content_8906215.htm|archive-date=2015-11-27|dead-url=yes}}</ref>。
