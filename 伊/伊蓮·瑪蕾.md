'''伊蓮·瑪蕾'''（{{lang-en|Elaine Marley}}）是由電子遊戲製作人[[羅恩·吉伯特|羅恩·吉伯特]]所創造、[[LucasArts|LucasArts]]公司發行的《[[猴島小英雄|猴島小英雄]]》冒險遊戲系列女主角。

== 作品中的描述 ==

伊蓮·瑪蕾於《猴島小英雄》系列首作《猴島的秘密》登場，她的身份為[[加勒比海|加勒比海]]混亂島（Melee Island）上廣受信賴的領主。而她不時被擁有強大魔力的鬼船長「[[老查克|老查克]]」所騷擾，因老查克迷戀伊蓮的魅力、並期望能作為她的新郎。

隨後某日混亂島上出現一名要立志成為海盜的青年「[[蓋伯拉許·崔普伍德|蓋伯拉許·崔普伍德]]」，之後伊蓮因受到蓋伯拉許的注意；他們之間逐漸開始交往起來，但也引起老查克的妒火。而在《猴島小英雄》系列中，皆不時穿插蓋伯拉許為了防止老查克奪取伊蓮、而互相對抗的戲碼。

== 角色設計 ==

在伊蓮·瑪蕾被正式塑造前，《猴島的秘密》遊戲創造者[[羅恩·吉伯特|羅恩·吉伯特]]想在遊戲中安排一個性格冷酷的小島領主角色，<ref name="GG">{{Cite web |url=http://grumpygamer.com/8280380 |title=Stuff and Things and Monkey Island |work=Grumpy Gamer |author=[[羅恩·吉伯特|羅恩·吉伯特]] |date=2009年6月1日 |accessdate=2014年2月14日}}</ref>之後協助作品創作的[[戴文·葛羅斯曼|戴文·葛羅斯曼]]從電影《[[畢業生_(電影)|畢業生]]》裡關於婚禮一段的情節上獲得靈感，提出將此領主命名為「伊蓮」的女性、而她必須被迫與遊戲中反叛人物老查克船長結婚；之後這場婚禮會被遊戲主角蓋伯拉許·崔普伍德給中途闖入的點子。後續羅恩·吉伯特接受戴文·葛羅斯曼的提議，並將此角色逐步修改成一名女性島上領主；且為蓋伯拉許的心儀對象。<ref name="GG" />

最初在《猴島的秘密》中一些出現關於伊蓮·瑪蕾臉部特寫的畫面，是由羅恩·吉伯特的遊戲美工同事艾薇兒·哈里森（Avril Harrison）作為模特兒。<ref>{{Cite web |url=http://pixel.valdaris.com/post/20-anos-de-monkey-island/ |title=20 años de Monkey Island |work=100 Pixel |date=2010年4月26日 |accessdate=2014年2月14日}}</ref>但事後羅恩·吉伯特對於那些臉部特寫的畫面感到困惑，並表示著「雖然那些畫面都畫得很棒，但我還是不認為這種風格特寫畫面適合放在這款遊戲」。<ref name="GG" />事後2009年《猴島的秘密》重製版本也將當時臉部特寫畫面的畫風給改過。

1997年《猴島小英雄》系列第三作《[[猴島的詛咒|猴島的詛咒]]》遊戲結尾中，伊蓮與蓋伯拉許正式結為夫妻。因當時羅恩·吉伯特已離開LucasArt、未參與《猴島的詛咒》作品開發，而他對於伊蓮與蓋伯拉許結婚作為結局一事不能認同；反應著「對於伊蓮來說，她只會將蓋伯拉許當成小弟弟來看待，她不應該會想與他結婚的」，並表示他個人另有一套關於《猴島小英雄》續作的靈感。<ref>{{Cite web |url=http://scummbar.com/resources/articles/index.php?newssniffer=readarticle&article=1004 |title=Ron Gilbert Chat Transcript |publisher=ScummBar |date=2003年7月21日 |accessdate=2014年2月14日}}</ref>

== 配音 ==

扣除在2000年作品《[[猴島小英雄_逃離猴島|猴島小英雄 逃離猴島]]》由查麗汀·詹姆斯（Charity James）擔任外；<ref>{{Cite web |url=http://www.allgame.com/game.php?id=33433&tab=credits |title=Escape From Monkey Island |publisher=All Game |accessdate=2014年2月14日}}</ref>伊蓮的角色配音全由曾在電影《[[春風化雨1996|春風化雨1996]]》演出的女演員[[亞歷山德拉·博伊德|亞歷山德拉·博伊德]]負責。而亞歷山德拉曾表示她能有機會參與這份配音職務是因先前與《[[猴島的詛咒|猴島的詛咒]]》配音製作人達拉赫·歐法雷拉（Darragh O'Farrell）有合作關係，另外曾開玩笑表示「或許也是因為我與伊蓮的髮色一樣才能接洽這份事務」。<ref name="AMO">{{Cite web |url=http://alternativemagazineonline.co.uk/2009/07/24/interview-in-conversation-with-alexandra-boyd/ |title=INTERVIEW – In Conversation With Alexandra Boyd |publisher=A.M.O UK |author=Marty Mulrooney |date=2009年7月24日 |accessdate=2014年2月14日}}</ref>

在往後《猴島小英雄》作品配音過程中，亞歷山德拉也提及「能在遊戲中對蓋伯拉許與老查克大吼大叫是一件很有趣的事」，<ref name="AMO" />與「伊蓮會願意照料、與比她沒擔當的蓋伯拉許交往，也許是因為伊蓮喜歡是主動的一方」。<ref>{{Cite web |url=http://talesofmi.net/?p=736 |title=Alexandra Boyd Interview |publisher=Tales of monkey Island |author=SilverWolfPet |date=2009年11月25日 |accessdate=2014年2月14日}}</ref>

== 反應 ==

網站[[GameSpot|GameSpot]]將伊蓮列為遊戲史上十大女性角色之一，表示著「伊蓮在作品系列中擔任非典型的『被英雄救美的一方』，很多情況下都是她靠自身脫困」。<ref>{{Cite web |url=http://www.gamespot.com/features/tenspot_females/page7.html |title=The Ten Best Female Characters |publisher=[[GameSpot|GameSpot]] |accessdate=2014年2月14日 |archiveurl=https://web.archive.org/web/20001216041200/http://www.gamespot.com/features/tenspot_females/page7.html| archivedate=2000年12月16日}}</ref>英國電子遊戲網站[[Eurogamer|Eurogamer]]將《猴島小英雄》系列第四作《[[猴島小英雄_逃離猴島|猴島小英雄 逃離猴島]]》裡登場的伊蓮給予2001年最佳女性電子遊戲配色榮譽，並評論「身為島上領主的伊蓮；很樂在享受、應用她的聲勢與威望，從她跟蓋伯拉許身上穿的褲子差別就看得出來」，<ref>{{Cite web |url=http://www.eurogamer.net/articles/a_gg2001_3 |title=Gaming Globes 2001 : The Results |publisher=[[Eurogamer|Eurogamer]] |author=Gestalt |date=2001年3月26日 |accessdate=2014年2月14日}}</ref>後續Eurogamer也對系列第五作《[[猴島傳說|猴島傳說]]》中伊蓮的造型給予肯定。<ref>{{Cite web |url=http://www.eurogamer.net/articles/tales-of-monkey-island-chapter-two-review?page=2 |title=Tales of Monkey Island: The Siege of Spinner Cay • Page 2 |publisher=[[Eurogamer|Eurogamer]] |author=Kristan Reed |date=2009年8月25日 |accessdate=2014年2月14日}}</ref>
