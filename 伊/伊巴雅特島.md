'''伊巴雅特島'''（{{lang-en|Itbayat Island}}），是位於[[巴丹群島|巴丹群島]]的島嶼之一，也是[[菲律賓|菲律賓]]最北的有人島，目前由[[巴丹群島省|巴丹群島省]]管轄，人口2988人（2010年5月1日）。行政單位[[伊巴雅特|伊巴雅特市]]設置於此島。

島上居民主要種植[[蒜|蒜]]、[[鳳梨|鳳梨]]、[[棕櫚|棕櫚]]，並有眾多[[椰子蟹|椰子蟹]]棲息。

== 地理 ==
是[[巴丹群島|巴丹群島]]中最大的島嶼，形狀略呈橢圓狀。海岸線主要由陡峭岩壁和淺灣組成，島上大多是平坦的[[丘陵|丘陵]]地形，在東北區域的最高處的海拔超過200公尺。

植被為低矮植物，不過部分土地也作為農業耕種使用。

== 位置 ==
島嶼距離[[菲律賓|菲律賓]][[呂宋島|呂宋島]]約290公里、[[夏揚島|夏揚島]]約25公里、[[汀姆島|汀姆島]]約17公里，離[[臺灣|臺灣]][[蘭嶼|蘭嶼]]約132公里。

== 交通 ==
島上擁有[[伊巴雅特機場|伊巴雅特機場]]，跑道長度約800公尺。此外，從[[阿帕里|阿帕里]]（Aparri）或[[巴士古|巴士古]]的港口搭船也可抵達。

== 其他 ==
考古學家於2002年至2004年的研究當中，證明3500年前即有人類居住在島上的多羅干洞穴（Torongan），以及北沙羅干洞穴（Northern Sarokan）、東沙羅干洞穴（Eastern Sarokan）、多特伯羅洞穴（Do'tboran）、佩維干洞穴（Pevangan）等地中。
