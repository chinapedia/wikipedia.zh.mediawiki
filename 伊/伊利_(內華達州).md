'''伊利'''（{{lang-en|'''Ely, Nevada'''}}）是[[美國|美國]][[內華達州|內華達州]]東部的城市、[[白松縣_(內華達州)|白松縣]]縣治。{{GR|6}}面積18.5平方公里，根據[[美國2000年人口普查|美國2000年人口普查]]，共有人口4,041。{{GR|2}}這個數字於2010年上升至4,255。<ref>[http://factfinder2.census.gov/ 2010 census factfinder2]</ref>

該城建立於1868年，是由採金熱潮興起的小型城市。1907年礦業達最頂點的該城，今經濟來源為[[觀光業|觀光業]]的收入。另外，該市的[[西魯斯礦場|西魯斯礦場]]為全世界最大的露天開採的[[銅|銅]]礦礦場。<ref name=ely>{{cite web
 |url=http://ely.travelnevada.com/

==地理==
根據[[美國2000年人口普查|2000年人口普查]]，本城面積為{{convert|7.1|sqmi|sqkm|2}}，沒有水域。<ref name ="Gazetteer files"/>

伊利擁有[[半乾旱氣候|半乾旱氣候]]，其日夜溫差極大。伊利的晚上是美國其中一個最寒冷的地方，每年伊利都有約218天的最低溫度低於{{convert|32|°F|0}}。<ref>{{cite web | title = Mean Number of Days Minimum Temperature 32 Deg.
