'''伊恩·希克森'''（{{lang-en|'''Ian "Hixie" Hickson'''}}）是[[Acid2|Acid2]]和[[Acid3|Acid3]]、[[HTML5|HTML5]]規範<ref name="html5spec">[http://www.w3.org/TR/2008/WD-html5-20080122/ "HTML5: A vocabulary and associated APIs for HTML and XHTML"], [[W3C|W3C]] Working Draft 2008-01-22</ref><ref name="internetnews.com">[http://www.internetnews.com/dev-news/article.php/3724041/ "HTML 5 Hits First Public Working Draft"], Sean Michael Kerner, internetnews.com,  2008-01-25</ref>、[[Pingback|Pingback]]規範<ref name="pingbackspec">[http://www.hixie.ch/specs/pingback/pingback Pingback Specification]</ref>以及Web Applications 1.0早期工作草案的作者和維護者<ref>{{cite web|title=Web Applications 1.0|url=https://whatwg.org/specs/web-apps/2005-09-01/|website=whatwg.org|accessdate=2017-04-22}}</ref>。他被稱作是[[網頁標準|網頁標準]]的倡導者，並在[[CSS|CSS]]等規範的發展中發揮了關鍵作用。希克森是CSS 2.1規範的共同編輯<ref name="css21">[http://www.w3.org/TR/2004/CR-CSS21-20040225 Cascading Style Sheets, level 2 revision 1], [[W3C|W3C]] W3C Candidate Recommendation 2004-02-25</ref>。

==生平==
希克森出生於[[瑞士|瑞士]][[日內瓦|日內瓦]]，在那裡住了十年<ref>[http://www.webstandards.org/about/members/hixie/ Hickson's biography] on the website of the Web Standards Project</ref> 。他在英國[[巴斯大學|巴斯大學]]學習物理學。後來他在[[Netscape|Netscape]]和[[Opera軟體公司|Opera軟體公司]]工作；他現在[[舊金山灣區|舊金山灣區]]的[[Google|Google]]工作，是[[網頁超文本應用技術工作小組|網頁超文本應用技術工作小組]]的規格編輯<ref name="resume">[http://ian.hixie.ch/career/resume.html Ian Hickson's resume]</ref>。
