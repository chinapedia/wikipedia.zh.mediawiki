'''永井一郎'''（{{bd|1931年|5月10日|2014年|1月27日|catIdx=Nagai Ichirou}}）是[[日本|日本]]的[[演員|演員]]、[[聲優|聲優]]、[[旁白|旁白]]。[[大阪府|大阪府]][[池田市|池田市]]出身。所屬過的藝能事務所有[[劇團三期會|劇團三期會]]、[[東京俳優生活協同組合|東京俳優生活協同組合]]、[[同人舍製作|同人舍製作]]、[[東京美術職業|東京美術職業]]，以及[[青二Production|青二Production]]。

== 來歷 ==
大阪第二師範學校（男子部）附屬國民學校（現[[大阪教育大學附屬池田小學|大阪教育大學附屬池田小學]]）畢業。[[大阪府立池田高等學校|大阪府立池田高等學校]]畢業（入學時是學制改革前的[[舊制中學校_(日本)|舊制中學校]]「大阪府立池田中學校」）。[[京都大學|京都大學]][[文學部|文學部]]法文科畢業。大學時期就開始演戲、畢業後立志要當演員而到都市。待過演員養成所（演員養成學校），在[[廣告代理店|廣告代理店]]的[[電通|電通]]做過2年勤務。從養成所畢業後，並與[[愛川欽也|愛川欽也]]和[[劇團演員座|演員座]]的第3期生集結並加入劇團三期會。據說這個三期會是對美國的電影[[電視電影|電視電影]]『[[超人|超人]]』的單元演出，永井他也開始了配音的工作。在三期會裡由於是最年長的25歳的緣故，當時在舞台是以中年人和老年人的角色為多。特別是[[海外電視劇|海外電視劇]]『[[生皮的鞭子|生皮的鞭子]]』的老[[牛仔|牛仔]]的威許本，成為了專門配老人角色的事情。在那之後，就開始再配音員的事業裡，結果變的淡出舞台與電影的工作，專心致志在配音員的事業裡<ref>『配音之後人生半世紀』（請看[[#廣播|廣播]]的選項），永井自己的評語。</ref>。

2009年第3回獲頒[[聲優獎|聲優獎]]的功勞獎。

2014年1月27日中午12時30分左右（日本時間），他在[[廣島市|廣島市]]的一間旅館被發現倒臥在浴缸裡，送到醫院時已經死亡。經紀公司隨後也張貼公告，表示他由於罹患[[冠狀動脈性心臟病|冠狀動脈性心臟病]]而急遽病逝，享壽82歲。<ref>[http://ccsx.tw/2014/01/28/ichiro-nagai-passed-away/ 永別了…老牌聲優「永井一郎」昨天中午因『冠狀動脈性心臟病』去世，業界晚輩紛紛獻圖、哀悼。]，2014年1月28日查閱。</ref>

== 特色 ==
以『[[海螺小姐|海螺小姐]]』的磯野波平開始，是在許多作品裡演出的日本配音員界裡最重要的1人。動畫，外語配音，特攝作品等活躍非常的廣。在以前也演過年輕人的角色，但隨著年齡增長演出中年人和老年人的角色變多了。特別是後面的主角的爺爺與師傅，上司，老牌的前輩等，扮演配角的角色演出非常多。

動畫，1962年的『[[一千零一夜_辛巴達的冒險_(電影)|一千零一夜 辛巴達的冒險]]』是第一次演出。1963年的電視動畫『[[原子小金剛|原子小金剛]]』也客串演出，1964年的『[[Big_X|Big X]]』的博士為首次擔任主要角色，在動畫界初期非常的活躍著。在那之後，『[[海螺小姐|海螺小姐]]』的「磯野波平」為首，也在很賣座的作品演出。『[[機動戰士GUNDAM|機動戰士GUNDAM]]』與『[[福星小子|福星小子]]』，『[[七龍珠|七龍珠]]』等等的賣座動畫，也在娛樂節目裡演出旁白的機會也很多。

富士電視台『職業王』的配音員121人中被選為No.1配音員裡的第4位。主要投票理由是「保持不變的聲音的改變角色的演功力」。

== 人物像 ==
=== 對工作的態度 ===
從[[東野英治郎|東野英治郎]]開始「對西洋片的配音，動畫的後期配音等自己就沒在表演了、在演舞台劇等其他地方被叫成[[外道|外道]]的作為」在一開始被受到批評的時候，就開使強烈的反駁，保有配音員界的朋友的立場。但是，對於「配音員前的演員」對於沒有演出的見解就和其他老手一樣，對於動畫配音員的偶像配音員們，對於舞台的專研那樣的提醒大家<ref name="out" />。

1988年的「奧爾讀書」的「磯野波平回來年收164萬元」的標題記事動畫被謠傳演出費很便宜的事情。在那之後、配音員360人中的代表[[野澤雅子|野澤雅子]]、[[內海賢二|內海賢二]]一起在動畫錄影帶・DVD化的聲音製作公司的聲音使用要求使用費支付的訴訟，在2004年勝訴了。

由於是演員出身的角色，而對於舞台演出的也有強烈的關心。在晚輩的劇團公演時也有參加，舞台的組裝和整體，都被照顧的相當的好。

; 螺子小姐
: 著作『巴克蒙！波平說日本』（[[新潮社|新潮社]]刊）中的「『海螺小姐』時的年收入是160萬元」。
: 擔任『螺子小姐』第二代[[磯野勝男|磯野勝男]]的[[高橋和枝|高橋和枝]]的葬禮出席、朗誦祭文。在這之中，以波平對勝男怒吼的口氣說「喂！勝男！父母過世後東西放哪裡去了！！」「勝男，櫻花開了。一起去散步如何」。原本永井打算以冷靜的口氣說祭文，可是不知不覺的最後聲音變成有淚聲感覺。事實上高橋那時比永井大2歲。
; 機動戰士鋼彈
: 在『機動戰士鋼彈』裡除了擔任旁白外還有迪金、多倫等有名的角色以外、還有進入[[白色基地|白色基地]]的難民等等，還有擔任數十人不是[[主角|主角]]的聲音（主要都為老人）。第13話，超越其性別擔任「難民營的老婆婆」演出。還有在鋼彈最鼎盛期時，廣播節目『[[動畫NOW！|動畫NOW！]]』裡的[[鋼普拉|鋼普拉]]的CM的旁白的歌曲擔任。歌詞是「鋼彈走吧，鋼彈作戰，鋼彈忍耐」等的內容。<br/>[[Animax|Animax]]播出的『機動戰士鋼彈30周年記念 大家的鋼彈 完全版』裡永井的旁白到「'''演出最多登場人物'''」的事情。
; [[高橋留美子|高橋留美子]]作品
: 從『福星小子』開始、演出很多高橋作品『[[亂馬½|亂馬½]]』、『[[犬夜叉|犬夜叉]]』、『[[高橋留美子劇場|高橋留美子劇場]]』等演出被當作是配音員裡的一位常客。

=== 性格、趣味 ===
關於在工作場合經常穿西裝，[[古谷徹|古谷徹]]等等的他的同事配音員也會說到。還有，喜歡開他的愛車雪鐵龍去兜風是他的興趣。

興趣是[[佛拉明哥吉他|佛拉明哥吉他]]的演奏對<ref name="out">『[[月刊OUT|月刊OUT]]』</ref>。對其他，油畫、滑雪、撞球等（『VOICE ACTOR DIRECTORY』）。

=== 電通時代 ===
電通勤務的當初是打工，京大畢後被上司知道的話可能成為正式員工，而擔憂的永井，就騙說自己是高中畢業的。但是京大畢業被知道了，永井的擔憂就成為正式員工了<ref>2002年7月19日、[[朝日電視台|朝日電視台]]系『[[徹子的房間|徹子的房間]]』</ref>。
