'''永正'''是[[日本|日本]]的[[年號|年號]]之一。在[[文龜|文龜]]之後，[[大永|大永]]之前。指1504年到1520年的期間。這個時代的[[天皇|天皇]]是[[後柏原天皇|後柏原天皇]]。[[室町幕府|室町幕府]]的將軍是[[足利義澄|足利義澄]]、[[足利義稙|足利義稙]]。

== 改元 ==
*文龜4年[[二月三十|二月三十]]（[[陽曆|陽曆]]1504年3月16日） 因遇到[[甲子革令|甲子革令]]而改元
*永正18年[[八月廿三|八月廿三]]（陽曆1521年9月23日） 改元為大永

== 出典 ==
出自『[[周易緯|周易緯]]』的「'''永正'''其道、咸受吉化」。

== 永正年間要事 ==
* 2年（1505年） - [[野田城_(三河國)|野田城 (三河國)]]築城。
* 8年[[八月廿四|8月24日]]（1511年） - 船岡山之戰。害怕[[細川政賢|細川政賢]]進攻京都而逃到[[丹波|丹波]]的前將軍・[[足利義稙|足利義稙]]靠著[[大内義興|大内義興]]等人的援助，擊破在[[山城|山城]]國船岡山布陣的政賢而入京。
* 9年（1512年） - [[玉繩城|玉繩城]]築城。

=== 出生 ===
* 4年[[十一月十五|11月15日]]（1507年） - [[大内義隆|大内義隆]]、[[大名|大名]]（[[天文_(年號)|天文]]20年逝世）
* 5年（1508年） - [[上泉信綱|上泉信綱]]、兵法家（[[天正|天正]]5年逝世）
* 7年（1510年） - [[織田信秀|織田信秀]]、[[武將|武將]]・[[織田信長|織田信長]]之父（天文20年逝世）
* 7年（1510年） - [[松永久秀|松永久秀]]、武將（天正5年逝世）
* 8年[[三月初五|3月5日]]（1511年4月2日） - [[足利義晴|足利義晴]]、[[室町幕府|室町幕府]]12代[[征夷大將軍|將軍]]（天文19年逝世）
* 8年（1511年） - [[松平清康|松平清康]]、武將（天文4年逝世）
* 10年（1513年） - [[大政所|大政所]]、[[豐臣秀吉|豐臣秀吉]]・[[豐臣秀長|豐臣秀長]]之母（[[文祿|文祿]]元年逝世）
* 11年（1514年） - [[島津貴久|島津貴久]]、[[戰國大名|戰國大名]]（[[元龜|元龜]]2年逝世）
* 12年（1515年） - [[北條氏康|北條氏康]]、武將（元龜2年逝世）
* 14年[[五月廿九|5月29日]]（1517年） - [[正親町天皇|正親町天皇]]、106代[[天皇|天皇]]（[[文祿|文祿]]2年逝世）
* 16年（1519年） - [[伊達晴宗|伊達晴宗]]、[[伊達氏|伊達氏]]第十五代當主（天正5年逝世）

=== 逝世 ===
* 3年[[八月初八|8月8日]]（1506年） - [[雪舟|雪舟]]、畫僧（[[應永|應永]]27年出生）
* 4年[[六月廿三|6月23日]]（1507年8月1日） - [[細川政元|細川政元]]、室町幕府[[管領|管領]]（[[文正|文正]]元年出生）
* 4年[[八月初一|8月1日]]（1507年9月7日） - [[細川澄之|細川澄之]]、細川政元之養子（[[延德|延德]]元年出生）
* 5年[[十一月十六|11月16日]]（1508年） - [[斯波義敏|斯波義敏]]、管領（[[永享|永享]]7年出生）
* 8年[[二月十九|2月19日]]（1511年3月18日） - [[吉田兼俱|吉田兼俱]]、神道家・[[吉田神道|吉田神道]]創始（永享7年出生）
* 8年[[八月十四|8月14日]]（1511年9月6日） - [[足利義澄|足利義澄]]、室町幕府11代將軍（[[文明_(年號)|文明]]12年出生）
* 16年[[八月十五|8月15日]]（1519年9月8日） - [[北條早雲|北條早雲]]、[[武將|武將]]（永享4年出生）
* 17年[[六月初十|6月10日]]（1520年6月24日） - [[細川澄元|細川澄元]]、武將（延德元年出生）
