'''永貴'''（{{lang-mnc|ᠶᡠᠩᡤᡠᡳ|v=yunggui}}，{{bd|1706年||1783年|}}<ref>[[康熙|康熙]]四十五年至[[乾隆|乾隆]]四十八年</ref>），字'''心齋'''，拜都氏，滿洲[[正白旗|正白旗]]人。父[[布蘭泰|布蘭泰]]，自云騎尉世職授理籓院員外郎。[[雍正|雍正]]年間，為[[江西巡撫|江西巡撫]]，在[[江西|江西]]整肅綱紀，不怕得罪上級，絕不貪污。成為當地佳話。

== 溫州賑災 ==
[[乾隆|乾隆]]初年，累遷郎中。管理[[湖南|湖南]]辰沅永靖道。升為[[云南|云南]][[布政使|布政使]]。轉任[[浙江|浙江]]，署[[浙江巡撫|浙江巡撫]]。前總督[[李衛|李衛]]領鹽政，發帑收馀[[鹽|鹽]]，叫做“帑鹽”，以防止私鹽充斥市面。永貴上任後即下令緝私，並完善“帑鹽”制度。永貴上書，令文武互相任其責，下部議行。[[溫州|溫州]]連續三年大旱，永貴令知府[[金洪銓|金洪銓]]賑災，但[[金洪銓|金洪銓]]賑災不力。永貴被彈劾失察，上書向[[乾隆帝|乾隆帝]]辭官。[[總督|總督]][[喀爾吉善|喀爾吉善]]再彈劾永貴，[[乾隆帝|乾隆帝]]故此奪[[金洪銓|金洪銓]]官，仍不許罷免永貴。不夠十日，御史[[范廷楷|范廷楷]]又劾永貴手下貪污。[[乾隆帝|乾隆帝]]稱難以有人代替永貴，他不只是先朝大臣，還是乾隆的愛將。乾隆下令不許永貴辭官。命令永貴留[[溫州|溫州]]及運用八十萬担[[米|米]]，借撥江蘇等省五十五萬担米，親自賑災。半年方賑災完畢。

== 平定回部與西北開荒 ==
乾隆二十一年，永貴加副都統銜，兼[[參贊大臣|參贊大臣]]。當年冬天，[[厄魯特|厄魯特]]和[[宰桑達什策凌|宰桑達什策凌]]一同叛亂，定邊右副將軍[[兆惠|兆惠]]駐[[伊犁|伊犁]]辦賊。永貴前往巴里坤地區負責防守，永貴勤加練兵，乾隆嘉獎永貴勤奮練兵，授予三等輕車都尉世職，跟從兆惠自[[額林沁畢爾罕|額林沁畢爾罕]]進兵。命署任[[西安巡撫|西安巡撫]]，未上任即下令到魯克察克[[屯田|屯田]]。乾隆二十三年，以侍郎銜留軍，因授[[刑部侍郎|刑部侍郎]]，主管屯田。[[烏魯木齊|烏魯木齊]]、[[辟展|辟展]]、[[托克三|托克三]]、[[哈喇沙爾|哈喇沙爾]]、[[昌吉|昌吉]]、[[羅克倫|羅克倫]]皆駐兵開墾農地，秋穫得[[穀|穀]]三萬五千八百馀石。當時[[兆惠|兆惠]]正攻打[[葉爾羌|葉爾羌]]，乾隆命永貴駐[[阿克蘇|阿克蘇]]主管[[餽軍|餽軍]]防守[[阿克蘇|阿克蘇]]。

乾隆二十四年，還至[[庫車|庫車]]，布政使[[德舒|德舒]]為回部[[嗎哈沁|嗎哈沁]]所殺。永貴出兵攻打[[嗎哈沁|嗎哈沁]]，平定回部。擢左都御史。乾隆二十六年，乾隆命永貴赴[[克什噶爾|克什噶爾]]辦事。永貴同時獲授[[禮部尚書|禮部尚書]]、[[鑲紅旗漢軍都統|鑲紅旗漢軍都統]]，仍駐克什噶爾。永貴上疏請治河與開墾農地，提議自[[赫色勒河|赫色勒河]]東南浚渠四十馀里，引水入[[赫色勒布伊|赫色勒布伊]]，支流的[[材托庸河|材托庸河]]湍急，要建築堤壩，鑿開山石，用以減弱水勢。五年之後，[[新疆|新疆]]完成基本開發，乾隆命令永貴回京。

== 失意離世 ==
乾隆四十八年，乾女婿[[和珅|和珅]]手握大權，貪贓枉法。永貴本欲阻止，但兒子[[伊江阿|伊江阿]]為和珅一黨，永貴愛子情深，不想對付自己兒子，以致無法阻止和珅權力擴張，同年八月因[[肺病|肺病]]去世，享壽七十七，謚文勤。

== 轶事 ==
* 永貴在軍機處，與[[阿桂|阿桂]]齊名，時人稱“二桂”。
* [[和珅|和珅]][[嫡妻|嫡妻]]馮氏[[和珅#嫡妻|馮霽雯]]為永貴乾女兒。
