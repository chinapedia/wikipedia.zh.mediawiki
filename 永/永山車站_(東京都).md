'''永山站'''（{{jpn|j=永山駅|hg=ながやまえき|rm=Nagayama eki|f=y}}）是一個位於[[東京都|東京都]][[多摩市|多摩市]]{{link-ja|永山 (多摩市)|永山 (多摩市)|永山}}一丁目，屬於[[京王電鐵|京王電鐵]]、[[小田急電鐵|小田急電鐵]]的[[鐵路車站|鐵路車站]]。

兩公司的正式站名不同，但位置十分接近，是互相轉乘站。
*'''京王永山站'''（{{jpn|j=京王永山駅|hg=けいおうながやまえき|rm=Keiō-nagayama eki|f=y}}） - [[相模原線|相模原線]]車站。車站編號為'''KO40'''。
*'''小田急永山站'''（{{jpn|j=小田急永山駅|hg=おだきゅうながやまえき|rm=Odakyū-nagayama eki|f=y}}） - [[多摩線|多摩線]]車站。車站編號為'''OT 05'''。

== 「永山站」的稱呼 ==
[https://web.archive.org/web/20160213201143/http://www.city.tama.lg.jp/shisetsu/4029/002234.html 多摩市役所「設施介紹」]與[https://web.archive.org/web/20160213130522/http://www.library.tama.tokyo.jp/library_guide0060.html 多摩市立永山圖書館的「交通介紹」]、[https://web.archive.org/web/20160217054800/http://tama-h.nms.ac.jp/map.html 日本醫科大學多摩永山醫院的「MAP」]、[http://www.jp-l.co.jp/pavilion/nagayama/06_map/index.html HUMAX PAVILLION永山的「MAP」]等車站周邊設施的交通介紹多將京王、小田急兩站直接寫作「永山站」，或是將正式名稱與「永山站」同時表列。

另外，多摩市在2003年制定的[https://web.archive.org/web/20081022103601/http://www.city.tama.lg.jp/business/4279/001953.html 多摩市商業活性化計畫] 將本站周邊地區定義為「{{PDFlink|[http://www.city.tama.lg.jp/dbps_data/_material_/localhost/04shiminkeizai/40keizaikanko/m_kassei/p11.pdf 永山站周邊]}}」。

該文件內的「{{PDFlink|[http://www.city.tama.lg.jp/dbps_data/_material_/localhost/04shiminkeizai/40keizaikanko/m_kassei/p10.pdf 多摩中心站周邊]}}」章節中記述「車站可搭乘小田急多摩線與京王相模原線」，顯示多摩市當局「不將京王、小田急兩業者視為不同車站，而是與「多摩中心站」一樣兩者駛入同車站」。

另外關於站前巴士總站與巴士站的正式名稱，[[京王電鐵巴士|京王巴士集團]]與小田急系的{{link-ja|神奈川中央交通|神奈川中央交通}}都稱作「永山站停留所」，多摩市在車站附近營運的自行車停車場名稱也是「永山站駐輪場」。

從眾多案例中可見行政或民間設施、團體等多將京王永山站與小田急永山站合稱為「永山站」。

== 歷史 ==
[[File:多摩ニュータウン初期の地区と、都心のビル群130817.jpg|thumb]]
永山站位於多摩新市鎮第一次入住區「諏訪、永山地區」的「地區中心」。但開始入住時由於鐵道工程落後，最近的車站是位於北方約3公里的京王線[[聖蹟櫻丘站|聖蹟櫻丘站]]，居民有3年時間僅靠巴士通勤、通學，被戲稱「陸之孤島」。在本站開設後，居民可搭乘同以新宿為起點的京王、小田急兩條路線，之後兩線再與都心的地下鐵直通運轉，便利性大幅提升。

=== 站名由來 ===
[[File:マンション高層階より多摩市の谷戸部の市街地130807.jpg|thumb]]
站名來自所在地名多摩市永山。「'''永山'''」是車站所在地北方{{link-ja|乞田川|乞田川}}沿岸的舊大字乞田小字名。該地名源自於南方地名「上'''長山'''」、「下'''長山'''」，「長」演變成「永」的原因不明。另外西南側丘陵地區為舊大字貝取小字名「'''瓜生'''（うりゅう）」。

由於[[北海道|北海道]][[旭川市|旭川市]]有同名的[[日本國有鐵道|日本國有鐵道]]（國鐵）[[宗谷本線|宗谷本線]][[永山站|永山站]]，另外京王與小田急兩線皆以新宿站為起點，為避免乘客誤乘或不當乘車，因此冠上公司名為「'''京王永山站'''」與「'''小田急永山站'''」以利區別。

建設計畫的暫時站名有舊大字地名「'''乞田'''（こった）」與小字名「'''瓜生'''（うりゅう）」。之後選擇新市鎮開發的地名「永山」。

== 車站構造 ==
京王、小田急兩永山站是3層構造的混凝土鋼架建築，兩站不僅相近，站房如同一體化，因此轉乘十分便利。第三層為月台，第二層是站房與商店所在，並連通行人天橋的大廳，一樓是商店與連通巴士總站、計程車乘車處的大廳。如同[[多摩中心站|多摩中心站]]，京王線 - 小田急線相互轉乘的乘客眾多，因此2008年3月15日起發售轉乘定期券。

== 利用狀況 ==
京王與小田急同為前往都心、新宿方向的交通方式，兩者呈現相互競爭的態勢。小田急比京王早四個半月通車，但在1990年代後半以前僅行駛各站停車，使得京王在便利性方面獲得優勢。在2000年以後，小田急與地下鐵千代田線、JR[[常磐緩行線|常磐線]]直通的「多摩急行」與下班尖峰時刻保證能有座位的[[浪漫特快|浪漫特快]]開始運行，小田急乘客逐漸增加。

另一方面，京王過去速度最快但未停靠本站的特急班次停駛後，取而代之的急行停靠本站，加上直通都營新宿線，使用者逐漸回流。2013年2月22日改點起，再次復活的特急增停本站、[[京王稻田堤站|京王稻田堤]]、[[南大澤站|南大澤]]。

== 車站周邊 ==
[[File:永山駅高架下の商店街151006.jpg|thumb]]

此地是{{link-ja|多摩新市鎮|多摩ニュータウン}}第1次入住地區。車站位在丘陵地帶的-{谷}-間位置，因此利用周邊地形打造行人與車輛分離的立體空間。

由於這一帶是周邊居民生活據點所在的「地區中心」，食衣住等相關商店與娛樂、醫療保健等設施林立。另外，此地也聚集[[JTB|JTB]]等多個大型企業的研修中心。

=== 巴士路線（永山站） ===
兩站的站前廣場共通，因此巴士站的正式名稱為'''永山站'''。1～4號乘車處在剪票口（車站西側）正面的廣場，5、6號乘車處東側在車站東側的道路上。

此地的路線巴士有[[京王電鐵巴士|京王電鐵巴士]]、[[京王電鐵巴士#京王巴士南|京王巴士南]]、{{link-ja|神奈川中央交通|神奈川中央交通}}。
