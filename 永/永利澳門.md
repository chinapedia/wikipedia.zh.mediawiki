'''永利澳門'''（'''{{lang|en|Wynn Macau}}'''），前稱'''永利澳門度假村'''（{{lang|en|Wynn Macau resort}}）<ref>[http://www.wynnmacau.com/tc/aboutus/directions.html]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>是一座位於[[澳門|澳門]][[新口岸|新口岸]]仙德麗街的[[賭場|賭場]][[度假村|度假村]]，由[[永利澳門有限公司|永利澳門有限公司]]興建營運。該酒店於2004年6月24日動工，於2006年9月5日揭幕，9月6日對外開放，當晚舉行的煙花匯演，外界評為澳門史上最盛大的一次性煙花匯演，吸引不少居民和遊客駐足欣賞。<ref>[http://travel.sohu.com/20061124/n246600533.shtml 永利度假亞洲旗艦酒店]</ref> 整個發展計劃佔地16英畝，第一期工程佔地11英畝，由塔樓的酒店及裙樓的娛樂場與其他配套設施組成，總投資額達56億元，這是澳門首家[[拉斯維加斯|拉斯維加斯]]綜合度假村模式的賭場酒店。

== 設施 ==
永利澳門內的永利澳門酒店現時共有約600間客房，地下設有「永利澳門娛樂場」，娛樂場內劃有休閒博彩區和禁煙區。

=== 永利大樓 ===
酒店大樓樓高25層，設有600間豪華套房，主樓部份的設計意念源於美國[[拉斯維加斯|拉斯維加斯]][[永利度假村酒店|永利度假村酒店]]，酒店房分為豪華客房、特級豪華客房、一房式套房及兩房式套房四種，全部套房面積均超過56平方米。

=== 宴會及會議中心 ===
永利澳門內的宴會及會議中心達2,200平方米，能夠舉辦各種大型會議以及商務活動。

=== 永利名店街 ===
設施內的零售商場「永利名店街」面積達30,000[[平方呎|平方呎]]，第一期工程於2006年9月6日正式開幕，第二期則於2008年1月份開幕。名店街由[[永利度假村(澳門)股份有限公司|永利澳門]]管理及營運，以高級精品為主。商場內有超過20以上的精品品牌商店，包括了[[路易·威登|LV]]、[[Gucci|Gucci]]、[[迪奧|Dior]]、[[FENDI|FENDI]]、[[勞力士|勞力士]]⋯等。

=== 永利澳門娛樂場 ===
永利澳門娛樂場由205,000平方呎的博彩場地所組成，當中有1000平方米大的貴賓賭場。中場部份設有420張賭檯及1,280台角子機。

== 特式項目 ==
=== 表演湖（水舞） ===
{{seealso|音樂噴泉#澳門永利酒店音樂噴泉}}
永利酒店正門的[[音樂噴泉|音樂噴泉]]是該酒店非常引人注目的一個節目，這是澳門首個採用活動噴咀的音樂噴泉，而且間中還會噴出火焰與噴泉一起表演。每日從早上開始直至晚上零時，每隔15分鐘一場。音樂噴泉是由200個噴咀和火焰發射器，及1,000個獨立燈泡組成。噴咀包括Oarsmen®,Fan Oarsmen®,micro Shooters®和Shooters®。傍晚開始更配合燈光一併表演<ref name="primecomhk"/>。水柱隨著音樂的旋律、節奏而變化、噴射，數不盡的水花在激光的照射下，幻化作七彩的星光，配合間中噴出的火焰，營造出一個個大型的音樂噴泉表演。

其音樂噴泉表演歌曲由電腦隨機控制，表演曲目包括電影主題曲、交響樂、西洋和華語流行歌曲等。

=== 吉祥樹 ===
「吉祥樹」是永利第二期入口處的特色地標，吉祥樹表演每30分鐘一場，每場表演時間約8分鐘；「吉祥樹」位於永利第二期擴建部份的大門入口處，表演配合音樂、燈光等效果，開始時天花上十二生肖的屏障像相機快門一樣散開，一幅[[LED|LED]]燈幕放出不同的光，緊接著燈幕頻往兩邊推開，一座大型的水晶燈往下，接着一棵五彩十色的「吉祥樹」徐徐由地底升起，場內播放着柔揚樂曲，同時頭上的天幕亦顯示出不同的斑紋圖案。最後，「吉祥樹」重回地底，天幕亦由金光閃閃、刻有十二生肖的屏障所收起，具神秘與傳統古典氣派，吸引大批旅客、居民圍觀及拍照留念。「吉祥樹」表演於2008年尾停止，緊接下來的就是新的表演項目——「富貴龍」。但因為「富貴龍」反應不及「吉祥樹」，因此現在兩個表演會交替上演。

=== 富貴龍 ===
「富貴龍」是永利第二期入口處的一項表演項目，表演每30分鐘一場，每場表演時間約5分鐘；位置與之前的「吉祥樹」一樣，表演配合音樂、燈光及煙霧等效果，一開始四周會放出白色煙霧，接着一條金色的「龍」徐徐由地底升起，最後，「富貴龍」重回地底。

== 擴建工程 ==
永利澳門酒店首期擴建部分包括36張賭桌、97台角子機，同時將部分食肆位置作出調整，項目於2007年2月份投入使用。

第二期擴建工程亦於2007年12月24日正式對外開放，第二期項目工程包括加建約12.3萬平方呎的博彩場地、一座大門地標建築，以及飲食、劇院*、零售場地。兩期擴建工程完成後，永利澳門內的賭枱及角子機數目分別增至420張及1,280台。位於永利大樓的名店街擴建部份亦已於2008年1月落成投入運作，零售商號增加11家。

永利澳門原來預留土地發展第二期工程，包括一個85,000平方呎、能容納150張賭檯、500臺角子老虎機以及運動博彩設施的賭場，兩家餐廳、設備完善的劇院以及第二期度假酒店，投資額逾27億元。史提芬·永利表示，雖然此地段確可建兩座建築，並增加400間客房，但此設計會令酒店與[[星際酒店|星際酒店]]過於接近，使酒店的景觀不夠「開揚」，亦有擠迫感，有違他一向堅持酒店內部、外觀及周圍環境都要同樣舒適的原則。

*劇院在建成後只舉行過極小場次的表演，不到半年時間就已經拆卸，改建為貴賓廳供富貴的內地客戶豪賭的地方，現今地方內部編號為Pit 95，Pit96

== 永利皇宮 ==
{{main|永利皇宮}}
[[File:Wynn_Palace_201508.jpg|thumb]]
永利在路氹城已經預留土地，興建一所全新的度假酒店。此度假酒店預計造價超過二百億葡幣。預期將在2016年6月25日落成啟用。史提芬·永利表示，永利路氹項目將是集團歷年來最好的項目。酒店外部有纜車和大型的表演湖提供音樂噴泉表演, 規模比永利度假村更大。

2011年7月7日，永利澳門在拍賣會上以7,993,250英鎊（包括買家溢價金）、即超過9,900萬港元購入一組嘉慶年製（1796-1821年）花瓶，創全球鍍金瓷器拍賣紀錄，並較舊有紀錄高1倍。該套工藝品由4個中國鍍金雙耳瓷製花瓶組成，擁有如此史齡及尺寸的4件花瓶組合，是拍賣會上罕見。花瓶將會放置於2015年開幕的路氹度假村中展覽。<ref>[http://paper.wenweipo.com/2011/07/09/HK1107090031.htm 澳門永利斥近億 投嘉慶鍍金瓷花瓶] 文匯報。2011年7月9日。</ref>

2012年5月2日，澳門政府正式批准永利澳門路氹51英畝土地合約。永利澳門預計將於該地興建一間設有娛樂場、約2000間酒店套房、會議室、零售、娛樂及餐飲服務的大型綜合度假村永利皇宮。不過，上述計劃仍會作出修改，而公司則計劃投資20至30億美元（約234億港元）發展上述項目，預期將在2016年6月25日落成啟用。<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20120503&sec_id=15307&subsec_id=15320&art_id=16302897 永利押注 234億拓路氹 建大型渡假村　最快三年後落成] ''蘋果日報 (香港)''. 2012年05月03日.</ref>

2016年6月27日，永利澳門宣佈，其位於澳門路氹地區的全新度假村勝地永利皇宮，將於今年8月22日開業。

== 獎項 ==
永利澳門度假村在2008年獲「TTG China旅遊大奬」選為「澳門最佳豪華酒店」。TTG China旅遊大奬是TTG 旅遊大奬在中國的延伸，依照年度TTG旅游大獎進行設計。而獲得該獎的標準是「服務高端市場旅客的最佳酒店，該酒店擁有反映高端市場定位的最佳視覺效果和環境」。永利澳門是四家獲奬澳門酒店之一，更是39家來自中國內地、香港及澳門的獲奬酒店之一<ref>[[澳門日報|澳門日報]] 2008年4月12日，《[http://www.macaodaily.com/html/2008-04/12/content_158115.htm 永利獲選澳最佳豪華酒店] {{webarchive|url=https://web.archive.org/web/20080414172541/http://www.macaodaily.com/html/2008-04/12/content_158115.htm |date=2008-04-14 }}》</ref>。

2008年11月得到旅遊指南《美孚旅遊指南》（Mobil Travel Guide）授予兩項五星評級獎項，成為全澳門唯一獲授予五星級殊榮的酒店，並與世界各地69間同獲授予五星評級的物業並駕齊驅<ref>[http://tw.lifestyle.yahoo.com/1e3a84d1/081111/180/3vsp1.html 永利澳門酒店頂級奢華，獲《美孚旅遊指南》五星評級 ]</ref>。

== 永利澳門勞斯萊斯幻影車隊 ==
永利澳門於2009年添置了8部[[勞斯萊斯|勞斯萊斯]]「[[勞斯萊斯幻影|幻影]]」加長版轎車。永利度假村迄今於全球擁有19部勞斯萊斯「幻影」轎車。這8部車輛是專為永利澳門訂製、配以酒店識別主色系銅色車身的[[勞斯萊斯幻影|勞斯萊斯幻影]]加長版轎車，為酒店賓客提供接載服務。在每個合金車輪也刻上永利之標誌。「幻影」加長版的內部裝飾採用天然材料，每部轎車均選用18張Moccasin真皮皮革，配以暗紅色的座椅圍邊及繡上永利標誌的座椅頭枕。

內部裝飾選用了伯爾[[胡桃木|胡桃木]]飾，車身設計採用對開式後車門，並配備收藏在車門內的鐵氟龍塗層雨傘。車廂配置連接12吋顯示屏及15個揚聲器的娛樂系統。車內附設[[冰箱|冰箱]]、林蔭大道照明系統及梳妝鏡。另外，「幻影」加長版選用了同步車輪軸心設計，無論車輪如何轉動，軸心的勞斯萊斯標誌永遠保持垂直向上<ref>[http://www.jornalvakio.com/index.php?tn=viewer&ncid=1&nid=139490&dt=20081218&lang=tw 勞斯萊斯幻影轎車  永利澳門引進八輛]</ref>。
