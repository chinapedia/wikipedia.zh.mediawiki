{{engname|《永不磨灭的番号》|''Forever Designation''}}是一部由徐纪周执导、2011年在中国大陆播出的[[抗战剧|抗战剧]]，由[[黄海波|黄海波]]、[[王雷|王雷]]、[[姚芊羽|姚芊羽]]等人主演。该剧讲述了在[[中国抗日战争|中国抗日战争]]期间，一支“李大本事”带领的没有正规建制[[番号|番号]]的县大队，在[[日军|日军]]占领区英勇抵抗入侵者的故事。2011年8月16日在[[北京卫视|北京卫视]]和[[深圳卫视|深圳卫视]]首播<ref>{{Cite web |url = http://ent.ifeng.com/tv/news/detail_2011_08/17/8466167_0.shtml |title = 永不磨灭的番号》首播满地抖包袱 上演荒诞版抗战戏|author = |date = 2011-08-17 |publisher = 凤凰网 |language = zh-hans |accessdate = 2015-08-12 |quote = }}</ref>。

==剧情==
李赤水，外号“李大本事”，是武义县的一位民兵队长，他是一个打仗时经常不按常理出牌、平时爱耍小花招的人，不过也正因为此，他的部队在抗日战争中越打越壮大。日军在华北进行大规模的扫荡，李大本事的队伍接到命令，要求阻击三万多日军扫荡部队，以保证后方[[八路军|八路军]]总部的安全，而李大本事仅提出了一个要求，那就是给他的队伍一个正规主力部队的番号，因为他的部队一直以来都没有一个正式的番号，总部首长答应了他们的请求。在与日军的战斗中，县大队除了“小三子”外，其余全部阵亡。一个多月后，首长重返战场，在一他们的坟堆面前履行了他的承诺。

==爭議==
在大结局第34集里，营长孙成海向天上扔了一颗[[手榴弹|手榴弹]]，把一架飞行中的日军飞机给打了下来。实际上，飞机以最低的飞行高度和飞行速度飞行，不可能依靠投手榴弹来击毁飞机。导演徐纪周在事后接受采访时表示，他都后悔死了，而且还因此事被点名批评<ref>{{Cite web|url =http://gzdaily.dayoo.com/html/2014-12/12/content_2819505.htm|title =徐纪周：我被一个镜头坑惨了|accessdate =|author =|date =2014-12-12|publisher =广州日报|deadurl =yes|archiveurl =https://web.archive.org/web/20150526022325/http://gzdaily.dayoo.com/html/2014-12/12/content_2819505.htm|archivedate =2015-05-26}}</ref>。
