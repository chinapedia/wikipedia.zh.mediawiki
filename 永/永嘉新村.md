'''永嘉新村'''是位于[[上海|上海]][[徐汇区|徐汇区]][[永嘉路_(上海)|永嘉路]]580弄的一片现代式[[里弄#公寓里弄|公寓里弄]]住宅，由[[交通银行|交通银行]]投资建造<ref name="公寓里弄">[http://www.shtong.gov.cn/newsite/node2/node2245/node75091/node75095/node75114/node75126/userobject1ai90864.html 专业志 >> 上海住宅建设志 >> 第一篇旧有住宅 >> 第二章里弄住宅 >> 第六节　公寓里弄住宅] 上海地方志办公室.</ref>。新村建于1947年，最初作为交通银行的职工宿舍使用，现在是普通民居<ref name="徐汇区">[http://www.shtong.gov.cn/newsite/node2/node4/node2249/xuhui/node38190/node38195/node38197/userobject1ai23318.html 区县志 >> 区志 >> 徐汇区志 >> 第十四篇房屋建筑 >> 第二章里弄住宅 >> 第二节　新式里弄] 上海地方志办公室.</ref>。因地处永嘉路，故而得此名。1994年，永嘉新村入选第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]。

与上海其它里弄住宅一样，永嘉新村也在近年来有违搭建筑的出现<ref name="新民晚报">{{Cite web |title = 优秀历史建筑冒出“老虎窗” 永嘉新村个别住户私搭阁楼 |url = http://shanghai.xinmin.cn/xmjd/2012/04/19/14470501.html |publisher = 新民网 | date = 2012-04-19 | accessdate = 2014-01-31 }}</ref>2014年3月起，上海市[[徐汇区政府|徐汇区政府]]开始对区内的违章建筑进行大力整治，其中新村内的40号2楼的违章建筑就在当事人拒绝配合的情况下被拆除<ref name="人民网">{{Cite web |title = 上海徐汇区拆违动真格 永嘉新村40号2楼违建被夷平 |url = http://house.people.com.cn/n/2014/0305/c164220-24530373.html |publisher = [[人民网|人民网]] | date = 2014-03-05 | accessdate = 2014-03-09 }}</ref>。

== 建筑特色 ==
永嘉新村建筑面积2万平方米，为砖木结构，东西两部各不相同<ref name="徐汇区"/>。东部为4排三层楼房，墙立面采用水泥拉毛，屋面则为四坡红瓦。西部为8排二层楼房，与东部不同的是，有了山墙花，屋面是双坡青瓦，而且入口的雨棚还具有西班牙风格。但它们都为行列式排列，且均有小阳台和小花园，环境幽雅<ref name="公寓里弄"/>。1960年代，新村里曾建起3层混合结构办公楼1幢<ref name="公寓里弄"/>。

新村的单元平面符合里弄的布局，分蝴蝶式拼接公寓型、四开间拼接型、六开间拼接型三种类型<ref name="徐汇区"/>。第一种前后平列4幢，每幢有3个单元组成，每层共有4套两室户。每户内都有卧室、起居室、厨房、浴厕间和保姆小卧室。第二种共有10幢，20个单元，每层有4套二室户。六开间拼接单元共6幢，12个单元，每层则有2套3室户。除上述外，还有三个拐角单元的组合体。居室面积在16到18平方米左右，通风良好<ref name="徐汇区"/>。
