'''敖汉旗博物馆'''，也称'''赤峰市敖汉旗博物馆'''、'''敖汉旗史前博物馆'''<ref>{{cite news|title=敖汉旗史前博物馆简介|url=http://www.aohan.gov.cn/Article/Detail/5401|accessdate=2013-07-18|newspaper=中国敖汉网|date=2009-12-23}}</ref>、'''敖汉史前文化博物馆'''，是一家位于[[中華人民共和国|中華人民共和国]][[内蒙古自治区|内蒙古自治区]][[敖汉旗|敖汉旗]][[新惠镇|新惠镇]]中心区[[惠文广场|惠文广场]]北侧的历史博物馆，为[[国家三级博物馆|国家三级博物馆]]，成立于1984年10月，免费开放。<ref>{{cite news|title=内蒙古免费开放的国有博物馆纪念馆|url=http://www.nmg.xinhuanet.com/zjcy/2008-06/19/content_13585748.htm|accessdate=2013-07-17|newspaper=内蒙古日报|date=2008-06-19|deadurl=yes|archiveurl=https://archive.is/20130717144234/http://www.nmg.xinhuanet.com/zjcy/2008-06/19/content_13585748.htm|archivedate=2013-07-17}}</ref>敖汉旗博物馆馆藏文物逾5000件，其中[[国家一级文物|国家一级文物]]112件。<ref name=tj/>敖汉旗内分布有[[小河西文化|小河西文化]]、[[兴隆洼文化|兴隆洼文化]]、[[赵宝沟文化|赵宝沟文化]]、[[小河沿文化|小河沿文化]]遗址，还有多座辽代壁画墓。这些遗址和墓葬出土的文物是博物馆的特色藏品。

== 历史 ==
1984年10月，敖汉旗博物馆自[[敖汉旗文化馆|敖汉旗文化馆]]分出，正式成立。起初，博物馆的馆址位于[[新惠镇|新惠镇]]北侧[[石羊石虎山公园|石羊石虎山公园]]东侧，临近[[石羊石虎山|石羊石虎山]]、[[孟克河|孟克河]]、公园、烈士塔。2001年秋，敖汉旗博物馆新馆开工。2003年，新馆竣工，博物馆搬迁至新惠街中心区惠文广场北侧，成为广场的标志性建筑。敖汉旗博物馆新馆设计面积4000平方米<ref>{{cite web|title=敖汉旗博物馆 简介|url=http://www.orgcc.com/mhtml/ahqbwg/|publisher=中国文化艺术网|accessdate=2013-07-18}}</ref>，建筑面积在内蒙古自治区县级博物馆中位居前列。<ref name="tj" />2005年4月24日，搬迁新址的敖汉旗博物馆开馆。<ref>{{cite web|title=敖汉旗人民政府2005年大事记|url=http://www.aohan.gov.cn/zfxxgk/gkml/Detail/269|accessdate=2013-07-18|publisher=敖汉旗人民政府|deadurl=yes|archiveurl=https://web.archive.org/web/20130722210248/http://aohan.gov.cn/zfxxgk/gkml/Detail/269|archivedate=2013-07-22}}</ref>

== 馆藏 ==
敖汉旗博物馆馆藏文物在5000件以上，其中[[国家一级文物|国家一级文物]]112件，[[国家二级文物|国家二级文物]]、[[国家三级文物|国家三级文物]]共600件左右。<ref name="tj" />敖汉旗内古遗址众多，敖汉旗博物馆也因此获得了丰富的史前文物藏品。敖汉旗博物馆十年间清理了八座辽代壁画墓，揭取壁画八十余幅。部分壁画在“画笔下的大辽风情”常设展展出。<ref>{{cite news|title=我国首家辽代壁画博物馆在敖汉旗免费开放|url=http://nm.people.com.cn/n/2013/0402/c196689-18395543.html|accessdate=2013-07-18|newspaper=人民网-内蒙古频道|date=2013-04-02|deadurl=yes|archiveurl=https://web.archive.org/web/20130518232802/http://nm.people.com.cn/n/2013/0402/c196689-18395543.html|archivedate=2013-05-18}}</ref>壁画藏品包括《马球图》《西瓜图》《备猎图》《出行图》等，其中《西瓜图》是中国古代最早描绘吃西瓜场景的作品。<ref>{{cite news|title=关于古人“吃西瓜”有哪些考古发现？|url=http://www.chnmuseum.cn/Default.aspx?TabId=138&InfoID=92695&frtid=40|accessdate=2013-07-18|newspaper=北京晚报|date=2013-07-03|deadurl=yes|archiveurl=https://web.archive.org/web/20160924182047/http://www.chnmuseum.cn/(S(to4ksjafljorco553kyybs3v))/Default.aspx?TabId=138&InfoID=92695&frtid=40|archivedate=2016年9月24日|df=}}</ref>敖汉旗博物馆的主要藏品还有兴隆洼遗址玉器、红山文化石雕带冠神像、“中华祖神”红陶磨光人像、胡人乐舞纹玉带垮、北宋瓷钵等。<ref>{{cite news|title=敖汉旗文化旅游产业发展巡礼之一：大资源引领大产业|url=http://www.chifeng.gov.cn/html/2012-08/f36de527-c2ec-42fd-857d-d7b92d454eb3.shtml|accessdate=2013-07-18|newspaper=敖汉旗政府|date=2012-08-24|deadurl=yes|archiveurl=https://archive.is/20130718123819/http://www.chifeng.gov.cn/html/2012-08/f36de527-c2ec-42fd-857d-d7b92d454eb3.shtml|archivedate=2013-07-18}}</ref>

== 展览 ==
敖汉旗博物馆目前设四个常设展览：“[[辽河|辽河]]源的文明曙光”史前文物展、“画笔下的大辽风情”辽墓壁画专题展、“八千年原始聚落”兴隆洼人生活场景复原、“马背上的帝国风云”[[辽代|辽]][[金代|金]][[元朝|元]]文物精品展。展出的文物有600件。<ref name=cf>{{cite journal|title=赤峰博物馆巡礼——敖汉旗博物馆|journal=赤峰学院学报(自然科学版)|year=2011|issue=3}}</ref>
