《'''割绳子'''》（''Cut the Rope''，又被称为'''怪物吃糖果'''，曾被误音译为'''卡特罗布'''），是[[俄罗斯|俄罗斯]][[ZeptoLab|ZeptoLab]]公司开发的一款[[益智游戏|益智游戏]]，于2010年由Chillingo公司发行<ref name="Benedetti">{{cite news|last=Benedetti|first=Winda|title=Superb 'Cut the Rope' game ties up iPhone/iPad charts|url=http://technolog.msnbc.msn.com/_news/2010/10/13/5286278-superb-cut-the-rope-game-ties-up-iphoneipad-charts|accessdate=16 October 2010|newspaper=[[Newsvine|Newsvine]] / [[msnbc.com|msnbc.com]]|date=13 October 2010|deadurl=yes|archiveurl=https://web.archive.org/web/20101016233807/http://technolog.msnbc.msn.com/_news/2010/10/13/5286278-superb-cut-the-rope-game-ties-up-iphoneipad-charts|archivedate=2010年10月16日}}</ref>。在iOS平台使用[[Crystal|Crystal]]和游戏中心进行[[社交网络服务|社交网络服务]]。在Android平台，使用Scoreloop进行社交网络服务。2011年推出续篇。

在[[国际消费电子展|CES 2012]]上，微软总裁[[史蒂夫·鲍尔默|史蒂夫·鲍尔默]]宣布与ZeptoLab合作推出基于HTML5技术的网页版《割绳子》，微软的[[Internet_Explorer_9|IE 9浏览器]]独占第三大关。

== 玩法 ==
[[File:CutRope_iOS-screenshot.png|180px]]
遊戲中，玩家要剪斷繩子，讓糖果可以將周圍的星星擊破又能進入怪物的嘴巴。<ref name="a">{{Cite web |url=http://key.chtouch.com/ContentView.aspx?P=1271 |title=Cut the Rope 超好玩的剪斷繩子遊戲 |accessdate=2013-04-20}}</ref>若糖果不知所蹤（即離開遊戲屏幕範圍），這關便不算完成。若糖果能進入怪物的嘴巴，這關便完成，同時下一關也將解鎖。在完成關卡後，玩家將根據獲得一星到三星的評價，玩家可以任意多次重試，當然也可以重玩之前打過的關卡嘗試獲得更高的評價。

== 版本 ==
=== 《割绳子》 ===
《割绳子》在2010年發佈。
=== 《割绳子：實驗》 ===
及後，《割绳子：實驗》（''Cut the Rope: Experiments''）在2011年推出。此版本以實驗室為主題，比《割绳子》多了一個角色，就是實驗室裡的教授，他會給玩家一些指示。<ref>{{Cite web |url=http://www.iloveapps.hk/2011/08/05/cut-the-rope-experiments/ |title=[iPhone/iPad] 反轉實驗室，全新版本 Cut the Rope: Experiments|accessdate=2013-04-20}}</ref>
===  《割绳子：穿梭時空》 ===
《怪物吃糖果：穿梭時空》（''Cut the Rope: Time Travel''）以中世紀、文藝復興時期、海盜船、古埃及、古希臘、石器時代等作為場景。<ref>{{Cite web |url=http://www.hksilicon.com/kb/articles/131544/Cut-the-RopeTime-Travel-Android |title=《Cut the Rope 2》正式於 Android 平台推出|accessdate=2013-04-20}}</ref>。

===  《割绳子2》 ===
《怪物吃糖果2》(cut the rope 2) 以[[森林|森林]]、[[池塘|池塘]]、[[垃圾場|垃圾場]]、[[公園|公園]]、[[地下室|地下室]]、水果市場、[[麵包店|麵包店]]作為場景。

== 評價 ==
因為其商業上的成功，割绳子深受評論家及玩家的喜愛。

截至2012年9月，[[Metacritic|Metacritic]]評定怪物吃糖果有93分（滿分100分）的分數，評級是“普遍好評”。<ref>{{Cite web |url=http://www.metacritic.com/game/ios/cut-the-rope |title=Cut the Rope |accessdate=2013-04-20}}</ref>[[GameSpot|GameSpot]]認為此遊戲是一個迷人的、具挑戰性的、有趣的益智遊戲，但批評此遊戲有時會出現控制問題。<ref>{{Cite web|url=http://asia.gamespot.com/cut-the-rope/reviews/cut-the-rope-review-6310907/|title=Cut the Rope Review|accessdate=2013-04-20|deadurl=yes|archiveurl=https://web.archive.org/web/20130627175351/http://asia.gamespot.com/cut-the-rope/reviews/cut-the-rope-review-6310907/|archivedate=2013-06-27}}</ref>

另外，有玩家認為遊戲的配樂不錯<ref name="a" /><ref>{{Cite web |url=http://www.macuknow.com/node/7934 |title=用智慧來給可愛怪物糖吃 - Cut the Rope |accessdate=2013-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20130228195519/http://macuknow.com/node/7934 |archivedate=2013-02-28 }}</ref>。
