'''綜合型高級中等學校'''（[[英語|英語]]：Comprehensive High School, CHS），簡稱'''綜合高級中學'''、'''綜合高中'''、'''綜高'''。是[[台灣|台灣]][[中等教育|中等教育]]的其中一種學制，綜合型高級中學會同時開設學術課程（普通科）及若干職業課程（依各校屬性而開設的職業類科），結合了[[普通型高級中等學校|普通型高級中等學校]]與[[技術型高級中等學校|技術型高級中等學校]]之課程，讓學生適性選讀，以因應學生的需要，讓學生能有機會跨科（學程）加廣選修學習，強化通識教育的需求。因為制度問題，以及因應少子化帶來的衝擊，自2010年起，全台已有許多高中職學校選擇停辦。

== 簡介 ==
綜合高中於八十五學年度由全國十八所高中職參與試辦<ref>該十八所學校為[[國立基隆高級商工職業學校|基隆商工]]、[[國立華僑高級中等學校|華僑高中]]、[[新北市私立淡江高級中學|淡江高中]]、[[臺北市私立開平餐飲職業學校|開平高中]]、[[臺北市立成淵高級中學|成淵高中]]、[[桃園市私立光啟高級中學|光啟高中]]、[[國立關西高級中學|關西高農]]、[[國立竹北高級中學|竹北高中]]、[[新竹市私立光復高級中學|光復高中]]、[[臺中市私立明德高級中學|明德女中]]、[[國立彰化師範大學附屬高級工業職業學校|師大附工]]、[[彰化縣私立正德高級中學|正德高中]]、[[嘉義市私立立仁高級中學|立仁工商]]、[[國立新豐高級中學|新豐高中]]、[[高雄市私立樹德高級家事商業職業學校|樹德家商]]、[[高雄市立高雄高級商業職業學校|高雄高商]]、[[國立恆春高級工商職業學校|恆春工商]]、[[國立關山高級工商職業學校|關山工商]]、[[花蓮縣私立海星高級中學|海星高中]]，目前已有87校辦理</ref><ref>[http://www.appledaily.com.tw/appledaily/article/headline/20170506/37641449/?utm_source=Via&utm_medium=Android_Share&utm_campaign=%E4%BB%8A%E6%97%A5%E8%98%8B%E6%9E%9C%2F%E8%A6%81%E8%81%9E%2F%E8%A2%AB%E9%81%BA%E5%BF%98%E7%9A%84%E5%AD%B8%E5%88%B6+%E7%B6%9C%E5%90%88%E9%AB%98%E4%B8%AD%E6%80%8E%E9%BA%BC%E8%BE%A6%EF%BC%88%E9%BB%83%E6%A6%AE%E6%96%87%EF%BC%89 被遺忘的學制 綜合高中怎麼辦（黃榮文）  蘋果日報]</ref>，輔導學生依據能力、興趣、性向，選修彈性多樣的學術課程及職業類課程的高級中等學校。

綜合高中兼具高中與高職雙重特質。學生在進入綜合高中第一年以就讀普通科課程為主，並有職業試探之課程，二年級再依據自己的學習成就、能力、興趣選擇高中升學目標（學術學程）或高職升學目標（專門學程），透過課程選修，實現自己的理想。

綜合高中採學年學分制，學生修滿160學分，必修科目不論學術學程或職業學程學生皆須全部及格即可畢業，不及格的科目如為必修可利用空堂或寒、暑假重修，如為選修，除重修外亦可放棄該學分，另外選讀其他科目。
另為培養專業技術的核心能力，專門學程設核心科目，屬校訂[[選修|選修]]科目範圍，規劃26至30學分做為該學程之核心科目，修畢單一學程核心科目可於[[畢業證書|畢業證書]]加註該生所修讀之學程。
=== 學術學程 ===
學術學程為綜合高中學生自高二起分流後進路選擇之一，其課程與[[普通型高級中等學校|普通型高級中等學校]]相近，分為社會學程即升讀文法商等系組的[[第一類組|第一類組]]，和自然學程即升讀理工醫農等系組的[[理科|第二、三類組]]。係以升讀一般[[綜合大學|綜合大學]]大學為主。

=== 專門（職業）學程 ===
專門學程為綜合高中學生自高二起分流後的另一進路選擇，其課程與[[技術型高級中等學校|技術型高級中等學校]]相近，著重專業知識與技能，並分為各類群，各校可依其特色與專業開設不同的專門學程。係以升讀[[科技大學_(臺灣學制)|科技大學]]、[[技術學院|技術學院]]、[[二年制專科學校|二年制專科學校]]或就業等進路為主。

== 綜合高中特色 ==
*學生畢業後可依個人目標，決定報考[[大學學測|大學學測]]、[[四技二專統測|四技二專統測]]、[[大學指考|大學指考]]並參加大學申請入學、甄選入學或參加四技科大申請入學、四技二專推薦甄選、技優保甄、登記分發或直接就業，修讀所需課程。

*綜合高中學生有百分之五十五以上為選修課程。

*學生可依性向、興趣，自由選修各學程課程，可跨學程（科）、跨年級選課或轉學程。

*經試探課程後，可選修適合興趣的課程學習。

== 入學方式 ==
[[國民中學|國民中學]]畢業生可依興趣、性向，參加[[國中教育會考|國中教育會考]]後，經由多元入學管道進入[[普通型高級中等學校|普通型高級中等學校]]、[[技術型高級中等學校|技術型高級中等學校]]、[[綜合型高級中等學校|綜合型高級中等學校]]等學制就讀。

== 綜合高中制度退場 ==
教育部統計指出，綜合高中曾在民國97年達到巔峰，有逾10萬人就讀，但近年來少子化衝擊加上熱度退燒，105年僅剩5萬人，比其他學制更快萎縮。

近年來逐漸有許多中等學校停止綜合高中招生，在不影響已在學學生權益下，停招後兩年廢止綜合高中學制。

== 反面意見 ==
綜合高中制度名義上升學之路更寬廣，可任選高中與高職升學管道，但因學生需花大量時間做生涯探索，高二起才選擇志願，相對其他體制學生起步晚，但和其他高中職學生一樣得面臨相同份量的課業及考試，尤其是職業學程比高職生少了近一年的時間鑽研，實際升學路上反成弱勢，學生往往立定志向後，多數因準備不足，較難考上理想的學校或科系。

實施幾年下來，學生及家長看不見成效，不再是熱門選項，綜高生也逐漸萎縮，開始出現廢止綜高的聲音。<ref>{{Cite web|url=https://udn.com/news/story/6898/2720453|title=學生數從10萬腰斬剩5萬 綜合高中廢？存？|accessdate=2017-09-25|author=聯合新聞網|date=|publisher=}}</ref>
