'''綜合編成頻道'''（{{Lang-ko|'''종합편성채널'''}}，簡稱'''綜編頻道'''）是[[韓國|韓國]]一種不通過[[地面電視|無線電視]]而採用[[有線電視|有線電視]]、[[衛星電視|衛星電視]]或[[寬頻電視|寬頻電視]]等方式進行全國播放的[[電視頻道|電視頻道]]類型，目前有[[MBN|每日廣播網]]（{{lang|en|Maeil Broadcasting Network}}）、[[JTBC|JTBC]]、{{lang|en|[[Channel_A|Channel A]]}}、[[TV朝鮮|朝鮮電視公司]]（{{lang|en|TV Chosun}}）等四家，均於2011年12月1日開播。<ref>{{cite web|author=搜狐娛樂|title=一周韩剧：综编频道开启韩剧新纪元 “强吻”风潮强劲|url=http://gb.cri.cn/27564/2011/12/07/5752s3465963.htm|publisher=國際在線|accessdate=2013-11-02}}</ref>

== 歷史 ==
韓國的電視媒體長期由國營媒體[[韓國放送公社|韓國放送公社]]和私營媒體[[文化廣播_(韓國)|文化放送株式會社]]所經營的两大[[電視聯播網|電視聯播網]]所[[寡占|寡占]]，雖然在1964年誕生的[[東洋廣播公司|東洋廣播公司]]曾一度打破這個局面，可是隨著[[韓國總統|韓國總統]][[全斗煥|全斗煥]]的命令，東洋廣播公司於1980年關閉。直到1981年，韓國電視媒體改革，才在两大電視網之外誕生了國營的第三家電視網[[韓國教育放送公社|韓國教育放送公社]]，1990年民营的[[SBS_(韓國)|首尔放送]]开播，2007年[[京仁電視台|OBS京仁TV]]开播，形成了韩国五間全国[[无线电视|无线电视]]网络。

隨著[[韓國國會|韓國國會]]在2008年通過了《報紙法》、《電視廣播法》和《IPTV法》之後，韓國其他電視媒體才逐步興旺起來。而綜合編成頻道則是在上述三法通過三年後正式獲准成立。<ref>{{cite web|author=中央日報|title=TBC等四个综编频道今日开播，开启传媒Bigbang时代|url=http://www.icnkr.com/article-74161-1.html|publisher=奮鬥在韓國|accessdate=2013-11-02}}</ref>

== 相似點 ==
為了區分傳統的[[地面電視|無線電視]]，綜合編成頻道大多採用[[有線電視|有線電視]]和[[IPTV|IPTV]]的形式實現全國聯播。

同時目前韓國現有的四大綜合編成頻道大多具有報社作為母公司，例如：每日廣播網母公司是《[[每日經濟新聞|每日經濟新聞]]》，JTBC母公司是《[[中央日報_(韓國)|中央日報]]》，Channel A母公司是《[[東亞日報|東亞日報]]》，朝鮮電視公司母公司是《[[朝鮮日報|朝鮮日報]]》。

== 節目類型 ==
和韓國傳統的無線電視臺一樣，綜合編成頻道的節目基本也是[[綜合頻道|綜合類型]]，除了有新聞、綜藝和音樂節目之外，也製作電視劇。例如：中央東洋廣播公司就有[[JTBC月火連續劇|JTBC月火連續劇]]和[[JTBC水木迷你連續劇|JTBC水木迷你連續劇]]兩大劇集播出時段；而Channel A也有[[Channel_A月火連續劇|Channel A月火連續劇]]的劇集播出時段。
