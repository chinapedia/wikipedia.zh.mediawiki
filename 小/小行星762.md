'''小行星762'''（Pulcova）是一颗[[小行星带|主带小行星]]。它由{{link-en|格里高利·诺伊明|Grigory Neujmin}}于1913年发现<ref name=jpldata/>，并以靠近[[圣彼得堡|圣彼得堡]]的[[普尔科沃天文台|普尔科沃天文台]]命名。小行星762的直径大约为137千米<ref name=jpldata>{{cite web
  |date=2009-09-22 last obs
  |title=JPL Small-Body Database Browser: 762 Pulcova (1913 SQ)
  |url=http://ssd.jpl.nasa.gov/sbdb.cgi?sstr=762
  |accessdate=2009-10-20}}</ref>，它是一颗[[C-型小行星|C-型小行星]]，因此它的深色表明它的表面含有[[碳酸盐|碳酸盐]]。

== 卫星 ==
2000年2月22日<ref name=CFHT2000>{{cite web
  |date=2000-02-22
  |title=762 Pulcova
  |publisher=SwRI
  |url=http://www.boulder.swri.edu/merline/press/fig3.html
  |accessdate=2009-10-20}}（[[自适应光学|AO]]图像）</ref>，在夏威夷[[冒纳凯阿火山|冒纳凯阿火山]]的[[加法夏望远镜|加法夏望远镜]]工作的天文学家们发现了小行星762有一颗卫星。卫星的直径约为15千米，是小行星直径的十分之一<ref name=swri2000>{{cite web
  |date=2000-10-26
  |title=Astronomers Image Double Asteroid
  |publisher=SwRI Press Release
  |author=Dr. William J. Merline and Maria Martinez
  |url=http://www.boulder.swri.edu/merline/press/release.txt
  |accessdate=2009-10-20}}（提到了[[小行星90|小行星90]]和小行星762）</ref>，它以[[半长轴|半长轴]]为800千米的[[椭圆|椭圆]]轨道绕小行星762运转，公转周期为4.0天<ref name="Merline2000"/>。卫星的[[视星等|视星等]]相比小行星要暗4等<ref name="Merline2000"/>。它也是最早被确认的[[小行星卫星|小行星卫星]]之一。

== 密度 ==
2000年，莫琳估计小行星762的[[密度|密度]]为1.8 g/cm<sup>3</sup>，这说明它比已知的其他双小行星（例如[[小行星45|小行星45]]和[[小行星90|小行星90]]）更为致密<ref name="Merline2000">{{cite web
  |date=2000-06-21
  |title=Discovery of Companions to Asteroids 762 Pulcova and 90 Antiope by Direct Imaging
  |publisher=SwRI
  |author=W.J. Merline (SwRI), L.M. Close (ESO, U. Arizona), C. Dumas (JPL), J.C. Shelton (Mt. Wilson Obs.), F. Menard (CFHT), C.R. Chapman, D.C. Slater (SwRI)
  |url=http://www.boulder.swri.edu/merline/press/dps.abstract.pdf
  |accessdate=2009-10-21}}</ref>。但是2008年3月的估计表明它的密度只有0.90 g/cm<sup>3</sup><ref name=Baer>
{{cite web
  |year=2008
  |title=Recent Asteroid Mass Determinations
  |publisher=Personal Website
  |author=Jim Baer
  |url=http://home.earthlink.net/~jimbaer1/astmass.txt
  |accessdate=2008-11-28
  |archive-url=https://www.webcitation.org/6HxBzMsli?url=http://home.earthlink.net/~jimbaer1/astmass.txt
  |archive-date=2013-07-08
  |dead-url=yes
  }}
</ref>，说明它可能是由松散的碎石组成，而不是一整块岩石。因为小行星之间的[[引力|引力]]太小，目前测定它们密度的最佳方法仍然是测定[[宇宙飞船|宇宙飞船]]飞掠时受引力偏转的大小，因此确定哪项估计更正确是非常困难的。而[[双小行星|双小行星]]为测定引力乃至质量提供了绝佳的机会。目前认为，这类小行星密度应该比水小<ref name=swri2000 />。
