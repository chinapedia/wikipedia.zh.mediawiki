'''小田原之戰'''（[[日語|日語]]：'''{{lang|ja|小田原の役、小田原攻め、小田原征伐、小田原合戦}}'''）是[[日本|日本]][[戰國_(日本)|戰國時代]]1590年（[[天正_(日本)|天正]]18年）[[豐臣秀吉|豐臣秀吉]]與[[後北條氏|北條家]]之間的戰爭。戰爭起因於北條家家臣[[猪俣邦憲|猪俣邦憲]]違反秀吉頒發的[[惣無事令|惣無事令]]，攻擊[[真田氏|真田家]]的[[名胡桃城|名胡桃城]]。豐臣軍一方面去包圍[[小田原城|小田原城]]，一方面則攻擊北條家的領地，最後[[北條氏政|北條氏政]]、[[北條氏直|北條氏直]]父子開城投降，後北條家滅亡。

== 起因 ==
{{See also|本能寺之变|天正壬午之亂}}
甲相同盟破裂後，[[武田勝賴|武田勝賴]]與[[常陸國|常陸國]]的[[佐竹家|佐竹家]]和在[[關東地方|關東地區]]反抗北條家侵略的諸勢力結盟共同對抗北條家，而北條家則與[[織田信長|織田信長]]和睦。在1582年(天正10年)織田德川聯軍進軍甲州，北條家亦參戰，[[武田家|武田家]]滅亡。北條家幫助[[織田氏|織田家]]滅了武田家並向信長要求賞賜，信長拒絕北條家的要求，北條家對織田家大為不滿。

同年信長在[[京都市|京都]][[本能寺|本能寺]]被大叛臣[[明智光秀|明智光秀]]發動謀反叛亂並突襲信長所在的本能寺迫使信長縱火焚寺自盡而亡，（史稱[[本能寺之變|本能寺之變]]），在得知信長逝世後，北條家立刻就率軍攻入信長在[[上野國|上野]]和信濃的領地，在[[神流川之戰|神流川之戰]]中擊敗[[瀧川一益|瀧川一益]]，但在[[天正壬午之亂|天正壬午之亂]]與德川軍對峙並達成和議，德川家佔領信濃和甲府地區而北條家佔領上野地區並且北條氏直迎娶家康之女[[督姬|督姬]]兩家達成同盟，北條家因此有餘力對抗關東各勢力繼續在關東擴張領地。

北條家與真田家發生領土糾紛，豐臣秀吉介入仲裁，將[[沼田城|沼田城]]附近有爭議的土地給與北條家，但要求氏政、氏直父子上洛。北條家雖不願意但還是派了[[北條氏規|北條氏規]](北條氏康5男)上洛接受和解，可是北條與真田雙方對此和解都不滿意因而要求秀吉再次仲裁。

1589年(天正17年)2月，小田原評定決定由評定眾[[板部岡江雪齋|板部岡江雪齋]]上洛，請豐臣秀吉調解北條與真田的沼田領地問題，秀吉調解雙方領地問題再次命令氏政、氏直父子上洛，但北條父子拒絕上洛。與此同時北條家沼田城代[[猪俣範直|猪俣範直]]攻入真田家的名胡桃城違反了惣無事令要求北條家說明，北條家不但無視豐臣秀吉的命令並且同時[[北條氏邦|北條氏邦]]侵略宇都宮家的領地。同年，豐臣秀吉對北條家宣戰並且要求全國各大名必須派兵協助。

== 戰前準備 ==
北條家徵招15歲至70歲的男子作戰爭前的準備，下令寺裡的鐘必須交出來熔鑄大砲之用。預測豐臣軍可能的進攻路線上，小田原城、[[八王子城|八王子城]]、[[山中城|山中城]]、[[韮山城|韮山城]]進行擴建的工作，阻擋豐臣軍的攻勢。

另一方面，從屬豐臣家底下的諸大名根據所領的石高數派出相對應的士兵數。命令[[長束正家|長束正家]]徵用20萬石糧食與馬匹，[[長宗我部元親|長宗我部元親]]、[[九鬼嘉隆|九鬼嘉隆]]、[[宇喜多秀家|宇喜多秀家]]派出水軍參戰協助軍糧運輸，[[毛利輝元|毛利輝元]]派兵保護京都在戰場後方待命。豐臣軍主要有二股軍勢構成，由[[東海道|東海道]]進攻的豐臣本隊與[[德川家康|德川家康]]軍隊共20萬，由[[東山道|東山道]]進攻的北方軍以上杉軍為主力共3萬5千名<ref name=":0">{{Cite book|title=小田原合戦と北条氏|last=黒田基树|first=|publisher=吉川弘文馆|year=2013年1月|isbn=|location=|pages=p 150-163}}</ref>，再加上歸順豐臣家的[[佐竹氏|佐竹家]]、[[小田長忠|小田家]]、[[大掾氏|大掾氏]]、[[真壁氏|真壁氏]]、[[結城晴朗|結城家]]、[[宇多宮尚綱|宇都宮家]]、[[那須氏資|那須家]]、[[里見義堯|里見家]]關東諸大名共1萬8千名。<ref name=":0" />

豐臣家的戰略是以北方軍做牽制敵方，主力進攻山中、韮山、足柄三城直擊小田原城，水軍在[[伊豆半島|伊豆半島]]壓迫周圍的北條軍。另一方，兵力劣勢的北條軍將5萬名精兵集中防衛小田原城，並與山中、韮山、足柄支城三城協同反擊敵軍。當得知豐臣軍主力由東海道進攻時，北條軍想佈軍在[[箱根山|箱根山]]上抵擋敵軍，與豐臣軍進行持久戰，但遭到野戰派的北條氏邦反對，自行率軍回到缽形山與豐臣軍對抗，最終北條家採取此戰略。除此之外，[[松井田城|松井田城]]的[[大道寺政繁|大道寺政繁]]率數千兵在北關東地區佈軍防禦來自東山道的豐臣軍。總而言之，北條軍的戰略是以小田原城為主力進行守城戰，並佈兵邊防拖延敵軍進攻，以時間彌補兵力的不足。

==战争经过==

=== 攻略北条支城 ===
1590年（天正18年）春天，豐臣主力集結在[[黃瀨川|黃瀨川]]周圍，3月27日豐臣秀吉到達[[沼津|沼津]]，29日攻擊開始。豐臣秀次與德川家康負責攻擊山中城，[[織田信雄|織田信雄]]負責攻擊韮山城。

山中城守備軍4,000人<ref name=":1" />（城主[[松田康長|松田康長]]、援軍[[北条氏勝|北条氏勝]]、[[他間宮康俊|他間宮康俊]]、[[松田康郷|松田康郷]]等)

開戰後山中城守將[[一條直未|一條直未]]戰死，數小時過後山中城宣告淪陷，松田康長與城同歸於盡而北条氏勝逃走。成功佔領山中城的同一天[[鷹之巢城|鷹之巢城]]也成功佔領，隔日德川將領[[井伊直政|井伊直政]]對足柄城攻城開始，4月1日足柄城淪陷。先鋒部隊在4月3日到達小田原城下。

韮山城守備軍约3640人<ref name=":1" />(城主：[[北條氏規|北條氏規]])

因為韮山城的兵力不到攻方的十分之一，因此丰臣秀吉下令採取包圍戰術，命令除織田信雄的旗本外，其余军队轉往小田原城。最后秀吉派遣德川家康入城与北条氏规交涉。德川家康把北条氏的现状告知氏规后，氏规开城投降。

清水康英在坚守了50日后开城投降，水軍部隊擊敗北條軍支配下的伊豆水軍，從伊豆半島登陸並控制沿岸諸城，在小田原城的海岸邊佈軍。

==== 对玉绳城的进攻 ====
从山中城逃回的北条氏胜，因感到羞耻而想自杀，但在家臣朝仓景澄以及其弟劝说下，北条氏胜放弃了这个念头，并率领700骑兵到玉绳城准备抵抗丰臣军的进攻。最终在德川家康的使者都筑秀纲等人的劝说下，北条氏胜开城投降，并为以后下总地方的北条支城无血开城做出贡献。

==== 在小田原城 ====
豐臣秀吉為了削弱北條軍的戰意，在小田原城西3公里處僅花一夜時間建造[[石垣山城|石垣山城]]打擊守軍士氣。命茶人[[千利休|千利休]]連日舉行茶會，秀吉還將[[茶茶|茶茶]]請至前線在箱根地區泡溫泉、助興娛樂展現財富與權力，此舉讓小田原城內的將領失去戰意。

* [[成田长亲|成田长亲]]
* 甲斐姬

忍城之戰以石田三成為大將，長束正家為副將，[[佐竹義重|佐竹義重]]、[[宇都宮國綱|宇都宮國綱]]、[[結城晴朝|結城晴朝]]等常陸、下野、下總諸位大名以及[[真田信繁|真田信繁]]等上野諸將做前鋒。攻擊軍在靠近忍城的丸墓山古墳一地紮營包圍忍城，秀吉命三成利用附近的利根川進行水攻，德川軍在攻略岩槻城完後支援三成軍，石田三成在利根川上建設堤防(石田堤)但因水量不夠導致水攻效果不彰，後來利根川水流增加自行潰堤反過來淹沒石田三成軍隊，得知此事的豐臣諸將就拿此事嘲笑石田三成沒有當武人的料。水攻失敗後攻方於是發動攻城卻因為水攻周圍土地泥濘變得難以進攻而失敗，並且由指揮守軍[[甲斐姬|甲斐姬]]多次擊退，攻城方雖然之後有淺野長政、真田昌幸、真田信繁的支援卻因為甲斐姬的多次反擊一直無法攻入城內，直到7月小田原城開城投降後，忍城才跟著投降。

==== 八王子之战 ====
八王子之战开始于6月23日

八王子城攻击军編成　合計1萬5千人

*[[前田利家|前田利家]]
*[[真田昌幸|真田昌幸]]
八王子城守備軍（[[北条氏照|北条氏照]]其家臣）　合計3千人

當豐臣軍進攻八王子城時，北條軍由八王子城代[[横地吉信|横地吉信]]領兵抵抗，並動員剩下家臣外，也招集領地的農民、婦女一共3千人。
6月23日在清晨濃霧未散前，豐臣軍分成二軍發動總攻擊，由上杉景勝、前田利家等主力攻擊八王子城東正面大手口，另一對攻擊北方絡手，豐臣軍與北條軍發生激戰雙方共有1千人死傷，豐臣軍一度停止進攻，之後豐臣軍成功奇襲北方絡手造成北條軍出現敗勢，當日八王子城宣告淪陷。負責守衛八王子城的北條家臣及眾士兵紛紛自殺，連城內婦女一同自盡，氏照的正室比佐跳瀑布自盡，傳說瀑布被血染紅持續了三天三夜。北條氏滅亡後由德川家康受領舊北條領地，決定廢除血腥的八王子城。

=== 小田原城開城 ===
5月9日豐臣秀吉命奧州大名[[伊達政宗|伊達政宗]]前來參戰，伊達政宗達得知命令後卻故意拖延出兵時間，幸而家臣[[片倉景綱|片倉景綱]]的提醒才帶兵前往小田原城，<ref>{{Cite book|title=小田原合戦と北条氏|last=黒田基树|first=|publisher=|year=|isbn=|location=|pages=p 207}}</ref>為了避免秀吉憤怒而遭到處罰，政宗於是命全軍穿著白色裝束上陣，以表示忠誠，這才保住伊達家的領地。豐臣軍一路降伏北條支城。5月27日曾擔任織田信長的小姓和側近並有「名人太郎」之美名的[[堀秀政|堀秀政]]逝世於軍陣中。6月豐臣軍包圍小田原城要求北條軍開投降，包圍期間雙方互有攻防。

北條軍被包圍期間部分家臣倒戈豐臣軍，德川家康施壓箕輪城的上野和田家使其從城內撤出，6月16日松田憲秀之子笠原政晴與數位北條家臣內通豐臣，北條家直發現後處罰笠原政晴，數日後氏政母親瑞溪院與繼室鳳翔院同一天死去。6月23日北方軍攻陷八王子城，北方軍將北條士兵的首級送入小田原城，另外將俘虜到的將士們妻子綁在城下，6月26日石垣山一夜城建成，這些舉動大大降低了城內的士氣。一連串的打擊不得不讓北條舉行「小田原評定」討論之後是否投降或是繼續抗戰，會議中北條各家臣遲遲不能有共識造成北條在軍略上舉棋不定，變成「不戰、不降、不攻、不守、不打、不和」的窘境，而後世把[[小田原評定|小田原評定]]形容為「冗長討論卻沒有結論的會議」。6月14日以北條氏邦出家為條件鉢形城開成投降，6月24日韮山城開城，而後津久井城開城，北條氏政、氏直父子不得不開城投降，小田原城包圍戰結束。

== 影响 ==
戰後秀吉對北條氏違反惣無事令進行懲罰，命令前北條家當家氏政、其弟御家眾筆頭氏照、老臣松田憲秀和大道寺政繁切腹<ref>{{Cite book|title=小田原合戦と北条氏|last=黑田基树|first=|publisher=|year=|isbn=|location=|pages=p 213-215}}</ref>，北條家當主北條氏直因為德川家康的求情免於一死並流放置高野山，北條家領地全數沒收，領地轉封給德川家康，後北條氏滅亡。

此戰役後全日本各大名皆臣服於豐臣政權，至[[應仁之亂|應仁之亂]]後第一次全國統一，開啟短暫的[[安土桃山時代|安土桃山時代]]和平時期。
