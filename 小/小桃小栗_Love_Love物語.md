《'''小桃小栗 Love Love物语'''》（{{lang-ja|ももくり}}）是由{{lang|ja|くろせ}}創作並在漫画网站[[comico|comico]]上连载的[[網路漫画|網路漫画]]，[[單行本|單行本]]全8卷。2015年改編為[[網路動畫|網路動畫]]與其他衍生作品。

== 故事內容 ==
描述了高中二年級的栗原雪對後輩桃月心也一見鍾情，下定了從遠處偷拍他的身影了達 100 張的相片就要告白的決心，兩人於是成為了戀人開始交往。雖然有些緩慢，但專心一意朝彼此前進的小桃小栗戀愛物語。

== 登場角色 ==
; {{nihongo|桃月心也|桃月 心也（ももつき しんや）|romaji=Momotsuki Shinya|聲：[[岡本信彥|岡本信彥]]<ref name="character">{{cite web|url=http://momokuri-anime.jp/character.html|script-title=ja:キャラクター紹介|accessdate=2017年2月23日|language=ja}}</ref>}}
: 生日6月20日，雙子座，O型，高中1年級，身高159cm。
: 綽號是小桃，喜歡的食物炸蝦，討厭的食物青椒和胡蘿蔔。
: 成績中下，運動水平一般。雪的男朋友，容易嫉妒。因父母經常工作，大部分時間也是自己一個在家。
: 為了長高，每天會喝草莓牛奶（理人哥哥告訴他喝草莓牛奶比喝牛奶的長高的速度會快點，自己自稱喝牛奶會肚子痛）。打了籃球三年，並在開學時參加了網球部，但一個月後退出了，卻好像還長不高。
: 十分要面子，有次在雪面前喝自己不喜歡的黑咖啡，又說自己不喜歡甜食，但實際上個人討厭苦味並喜歡甜味。
: 討厭被別人叫可愛和女生，認為「可愛」是用於女生上而不是男生。
: 沒有看過他生氣的樣子。
; {{nihongo|栗原雪|栗原 雪（くりはら ゆき）|romaji=Kurihara Yuki|聲：[[加隈亞衣|加隈亞衣]]<ref name="character"/>}}
: 生日6月15日，A型，高中2年級，身高152cm。
: 對小桃一見鍾情，之後告白，成為小桃的女朋友。
: 自己的編織、運動、藝術不太好，但烹飪也不錯。
: 有點變態，經常收集小桃的物品。就讀A班（特別升學班）。意外的是樣子傻傻的但卻考得高名次。
: 手機裡有“小桃（偷拍）珍貴影像全集”，當收集100張時許了願要去向小桃告白，所以就跑去告白了，當收集200張時便要和小桃合照，願望也實現了。
: 與紀香關係很好。
: 讓人感覺軟綿綿的可愛女孩，即使不擅長編織及縫紉，為了小桃還是努力學習。
; {{nihongo|澤口理人|沢口 理人（さわぐち りひと）|romaji=Sawaguchi Rihito|聲：[[山谷祥生|山谷祥生]]<ref name="character"/>}}
: 生日4月26日，AB型，高中1年級，身高170cm。成績屬於有念就好的類型。
: 初中因柚姬被說閒話而不再叫柚姬為小柚，改叫島田。似乎對柚姬有好感，不過並不願意面對。
: 父母都是醫師。家裡是連鎖診所。
; {{nihongo|閑翔太|閑 翔太（しずか しょうた）|romaji=Shizuka Shota|聲：[[永塚拓馬|永塚拓馬]]<ref name="character"/>}}
: 生日8月22日，O型，高中1年級，身高172cm。性格溫柔體貼，善解人意。有點迷糊。
: 在柚姬擦黑板時因為擦不到，理人就先幫忙，但這幕似乎被翔太看到，向理人告知自己喜歡柚姬。
: 柚姬的表哥。
; {{nihongo|早柿莉央|早柿 莉央（さかき りお）|romaji=Sakaki Rio|聲：[[前田玲奈|前田玲奈]]<ref name="character"/>}}
: 生日10月5日，AB型，高中1年級，身高168cm。
: 很受女生歡迎。暗戀小桃。曾經有學妹誤會她是男的而向她告白。
: 很會做料理，且縫紉技術也很好，很羨慕雪可以坦白自己的感情。
: 與雪關係不錯，常常幫助她在手工藝上的不足。
; {{nihongo|島田柚姬|島田 柚姫（しまだ ゆずき）|romaji=Shimada Yuzuki|聲：[[仲谷明香|仲谷明香]]<ref name="character"/>}}
: 生日3月16日，B型，高中1年級，身高147cm。
: 自稱「柚姫」，初中因被說閒話而不再叫理人為小理，改叫小澤。和翔太是表兄妹，關係很好。似乎喜歡理人，並不知道翔太喜歡她。很愛吃，常常吃完一個再接一個。
; {{nihongo|宇佐美育繪|宇佐美 育絵（うさみ いくえ）|romaji=Usami Ikue|聲：[[竹下礼奈|竹下礼奈]]<ref name="character"/>}}
: 生日1月13日，A型，高中1年級，身高165cm。
: 有個妹控哥哥。和柚姬是好朋友。在去溫泉旅行時，被要求一個小時傳一則簡訊給他。
; {{nihongo|水山紀香|水山 のりか（みずやま のりか）|romaji=Mizuyama Norika|聲：[[大空直美|大空直美]]<ref name="character"/>}}
: 生日12月7日，A型，高中2年級，身高161cm。
: 雪的朋友。就讀A班(特別升學班)。態度看似冷淡，事實上非常挺朋友。家裡很有錢。
; {{nihongo|宇佐美誠一郎|宇佐美 誠一郎（うさみ せいいちろう）|romaji=Usami Seiichiro|聲：[[小野賢章|小野賢章]]}}
: 栗原的同級生，育繪的哥哥。常幫助栗原提供小桃的資訊，而且還以妹妹的照片為代價。宇佐美育繪的哥哥，看似讀書人的樣子，但卻是一個妹控。在妹妹去溫泉旅行時，要她一小時傳一則簡訊。
; {{nihongo|遠藤|遠藤さん（えんどう-）|聲：[[井上奈奈|井上奈奈]]}}
: 新聞部部長。早柿狂。
; {{nihongo|澤口慧真|沢口 慧眞（さわぐち えま）|romaji=Sawaguchi Ema|聲：[[内匠靖明|内匠靖明]]}}
: 理人的哥哥（4岁年长）目前就讀醫學大學。家裡的診所為全國連鎖。

== 網路動畫 ==
2015年2月13日，作品動畫化的消息公布。2015年12月24日起，在comico網站開始播出，每周四23时00分更新。需使用comico日本官方APP收看，如需免費收看每集等待一個禮拜，想看全部，則需要花費購買點數觀看，目前已出到26集。
