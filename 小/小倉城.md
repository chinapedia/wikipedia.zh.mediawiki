'''小倉城'''（{{Jpn|j=小倉城|rm=kokura-jō|f=t}}）是位于現今[[福岡縣|福岡縣]][[北九州市|北九州市]][[小倉北區|小倉北區]]的城堡。別稱勝山城、勝野城、指月城、湧金城、鯉之城（{{lang-ja|鯉ノ城}}）。是北九州市的著名觀光景點之一。

== 概要 ==
大約在13世紀，小倉城建築於{{link-ja|紫川|紫川}}河口的西岸的山丘上。現在的小倉城的占地範圍和規劃是到了[[安土桃山時代|安土桃山時代]]的1587年，由[[毛利勝信|毛利勝信]]築城時確定下來的。而小倉城的{{link-ja|南蠻造|唐造}}風格的天守則是由[[細川忠興|細川忠興]]所建造。

小倉城是{{link-ja|梯郭式|梯郭式}}平城。城中心是本丸，其南面為松丸，北面為北之丸（{{lang-ja|北の丸}}），在它們外圍依次設置有二之丸（{{lang-ja|二の丸}}）和三之丸（{{lang-ja|三の丸}}）和外郭。建築物包括以未做形狀加工的石材築成的石垣（{{lang-ja|野面積み}}）和其上的天守與1座小天守、117個平櫓、16個二重櫓、12座{{link-ja|櫓門|櫓門}}及以及3271個{{link-ja|狹間|狹間}}。城東流過的紫川被用為天然的護城河，將一部分城下町包圍在外郭之內，形成所謂「{{lang|ja|{{link-ja|総構え|総構え}}}}」式城郭結構。現存建築物為部份石垣及堀，天守、櫓門、庭園及大名屋敷為重建。若想知曉當時完整的城郭構造，可以通過{{link-ja|北九州市立自然史・歷史博物館|北九州市立いのちのたび博物館}}所藏小倉城全體模型了解。

== 天守 ==
[[File:140721_Kokura_Castle_Kitakyushu_Japan01s3.jpg|thumb]]

小倉城的天守是4重5層的天守以及1重的小天守，是一個連結式塔層形天守。天守最上層為{{link-ja|唐造|唐造}}（又名南蠻造）風格，使用“{{lang|ja|外廻縁}}”，即設置於天守牆壁外侧的地板和欄杆。唐造風格除了美觀，還使得天守從外面看來層數顯得較少，在德川幕府時期一些大名天守為了避免猜忌而採用此風格，從實用角度講，突出的外迴緣可以為下層窗戶防雨。原本的天守除了最上層的[[歇山頂|歇山頂]]（{{lang-ja|入母屋}}）[[博風板|破風]]，其餘各層沒有破風。

1837年（天保8年）火災燒毀天守和御殿，1839年重建時，在天守台建造了稱為“御三層”的建築代替天守<ref>{{cite book|author = 西ヶ谷恭弘 | title = 復原 名城天守 | publisher = 学習研究社 | year = 1996年 | language = ja}}</ref>。現在的天守是在1950年代根據《豐前小倉御天守記》、《小倉城繪卷》、《延享三年巡見上使御答書》等資料，用[[鋼筋混凝土|鋼筋混凝土]]結構建成。設計考證由{{link-ja|藤岡通夫|藤岡通夫}}擔任。為了增加觀光性，加上了大入母屋破風、千鳥破風、唐破風等各式各樣的破風。

== 歷史 ==
* 小倉城確實的築城年份不明，但應建於[[文永|文永]]年間（1264年-1274年），當時為緒方大膳亮惟重的[[居城|居城]]<ref>{{Cite web|url = http://www.asahi-net.or.jp/~qb2t-nkns/kokura.htm |title = 豊前 小倉城／歴史 |author = |date = |publisher = |language = ja |accessdate = 2015年7月28日 |quote = }}</ref><ref name=GeoKokura>{{Cite web|url = http://www.geocities.jp/y_ujoh/kojousi.kokura.htm |title = 小倉城（こくら） |author = |date = |publisher = |language = ja |accessdate = 2015年7月28日 |quote = }}</ref>。
* 1330年（[[元德|元德]]2年）為黑崎土佐守景經的居城，其后為[[大內氏|大內氏]]的支城。
* 1442年（[[嘉吉|嘉吉]]2年），太宰少贰親冬攻克此城。[[文明_(後土御門天皇)|文明]]年間（1469-1486年）為{{link-ja|菊池氏|菊池氏}}居城。
* 1569年（[[永祿|永祿]]12年），[[毛利家|毛利家]]將小倉城授予背叛了[[大友氏|大友家]]的[[高橋鑒種|高橋鑒種]]。
* 1587年（[[天正|天正]]15年），城主{{link-ja|高橋元種|高橋元種}}在[[豐臣秀吉|豐臣秀吉]]的[[九州征伐|九州征伐]]中降服。豐臣秀吉將[[豐前國|豐前]]小倉6萬[[石高|石]]（另說10萬）領地賜封予[[毛利勝信|森勝信]]。對當時的小倉城的情况没有详细的描述。此外勝信之子[[毛利勝永|勝永]]也獲封豐前國領地1萬石。隨後[[豐臣秀吉|豐臣秀吉]]令森勝信改姓與[[中国地方|中國地區]]領主相同的毛利氏。
* 1600年（[[慶長|慶長]]5年），[[關原之戰|關原之戰]]後毛利勝信、勝永父子因參加西軍被沒收領地。德川家康論功行賞，[[細川忠興|細川忠興]]入主豐前國，初時細川氏以中津城為居城。從1602年開始，歷經7年時間的修築，細川家將小倉城改建為與其40萬石大名（準確說來是39萬9千石）地位相稱的居城，並以{{link-ja|紫川|紫川}}將[[城下町|城下町]]劃分為東西兩部，西邊是{{link-ja|武家屋敷|武家屋敷}}，東邊是[[町人|町人]]及下等武士的居所。
* 1632年（[[寬永|寬永]]9年），細川氏移封到[[肥後國|肥後國]]，由[[譜代大名|譜代大名]]、原[[播磨|播磨]][[明石城|明石]]城主[[小笠原忠真|小笠原忠真]]（其妹为当时細川家主忠利的正室）就封小倉城，所領為15萬石。自此以後小倉城為{{link-ja|小笠原氏|小笠原氏}}小倉藩的居城。
* 1837年（[[天保|天保]]8年），本丸火災，天守和御殿被燒毀，災後天守沒有被重建。
* 1866年（[[慶應|慶應]]2年），第二次[[長州征伐|長州征伐]]時，小倉藩成為進攻[[長州藩|長州藩]]的前線。幕府側軍隊作由於戰不利，最終決定放棄小倉城。8月1日小倉城被燒毀，藩主逃到[[熊本城|熊本城]]避難。家老以下的藩內首腦在[[田川郡|田川郡]][[香春町|香春]]指揮戰鬥。
* 1867年（慶應3年），長州藩與小倉藩達成和議，包含小倉城在內的[[企救郡|企救郡]]被長州藩佔領。自此，小倉藩在香春以御茶屋（藩主巡視時停留的處所）為中心建立藩庁。1870年（[[明治|明治]]2年），在[[京都郡|京都郡]]豐町新建藩庁并遷移至彼處。
* 1875年（明治8年），[[大日本帝國陸軍|陸軍]]{{link-ja|步兵第14連隊|歩兵第14連隊}}及管轄步兵第14連隊的步兵第12旅團本部駐於小倉城松之丸舊址。
* 1898年（明治31年），陸軍[[第12師團|第12師團]]司令部庁舍設於小倉城本丸舊址，1925年（[[大正|大正]]14年）遷往[[久留米|久留米]]。
* 1934年（[[昭和|昭和]]9年），將{{link-ja|城外鋳物師町的八坂神社|八坂神社 (北九州市)}}遷至城內。
* 1959年（昭和34年），重建天守，天守內部為鄉土資料館，需要付費進場<ref>[http://www.ajkj.jp/ajkj/fukuoka/kitakyusyu/kanko/kokurajyo/kokurajyo.html 小倉城 北九州觀光 九州] {{Wayback|url=http://www.ajkj.jp/ajkj/fukuoka/kitakyusyu/kanko/kokurajyo/kokurajyo.html |date=20090414110657 }}</ref>。
* 1990年（[[平成|平成]]2年），天守內部全面重新裝修，增设全景·模型剧场等，改造为体验型设施。
* 1998年（平成10年），小倉城庭園及[[松本清張|松本清張]][[紀念館|紀念館]]開幕。
* 2004年（平成16年），春季及秋季調查中，在篠崎口至清水門之间的外堀發現畝堀及堀障子，据推测是忠興時代興建的。
* 2007年（平成19年），整修了约8万枚屋瓦。<ref name=Chronology>{{Cite web|url = http://www.kokura-castle.jp/about/chronology.php |title = 小倉城について／小倉城略年譜（概要） |author = |date = |publisher = |language = ja |accessdate = 2015年7月28日 |quote = }}</ref>

== 相關人物 ==
; [[細川忠興|細川忠興]]
: [[細川藤孝|細川藤孝]]之子。關原之戰後入主小倉藩的第一代藩主。小倉城天守建造者。精通[[能劇|能劇]]與[[茶道|茶道]]，是“[[利休七哲|利休七哲]]”之一。
; [[細川玉|細川玉]]
: [[明智光秀|明智光秀]]之女，細川忠興之妻。由於不願被石田三成擄為人質要挾細川忠興，令家臣殺死自己（由於基督教信仰不允許自殺）。據說細川忠興入主小倉藩後，在小倉城為她舉行了盛大的[[彌撒|彌撒]]。<ref name=Meijin>{{Cite web|url = http://www.kokura-castle.jp/about/person.php |title = 小倉城について／小倉城所縁のひとびと |author = |date = |publisher = |language = ja |accessdate = 2015年7月28日 |quote = }}</ref>
; [[小笠原忠真|小笠原忠真]]
: [[德川家康|德川家康]]的外曾孫，小笠原氏小倉藩第一代藩主。招攬了中國[[黃檗宗|黃檗宗]]僧人[[即非如一|即非如一]]建立{{link-ja|廣壽山福聚寺|福聚寺 (北九州市)}}，令來自中國的黃檗宗美術廣為流傳<ref name=Kitakyushu>{{Cite web|url = http://www.city.kitakyushu.lg.jp/kokurakita/file_0082.html |title = 小倉城と小倉藩 |author = |date = |publisher = |language = ja |accessdate = 2015年7月28日 |quote = }}</ref>。
; [[宮本武藏|宮本武藏]]
: 以與細川氏小倉藩時代擔任劍術指南役的佐佐木小次郎之間的[[巖流島|巖流島]]的決鬥而聞名。曾於小倉藩度過七年，養子伊織在小笠原家任家老。他曾有“吾之家族存於小笠原家，而吾之劍技傳於細川家”之語<ref name=Meijin></ref>。

== 小倉城遊樂園 ==
自1960年起的10年間，在小倉城內（今{{link-ja|北九州市役所|北九州市役所}}附近）曾建有[[遊樂園|遊樂園]]。1962年曾有“小倉城與雲霄飛車”（{{lang-ja|小倉城とジェットコースター}}）的明信片發行。<ref> {{cite book | author = 益田啓一郎 | title = ふくおか絵葉書浪漫 | isbn = 4-87415-491-3 | publisher = 海鳥社 | pages = 91頁 | language = ja}}</ref>
