{{漫畫}}
'''小咪漫畫新人獎'''是由[[臺灣|臺灣]]伊士曼小咪出版社（[[大然文化|大然文化]]之前身）主辦之漫畫比賽，自1980年9月2日起每隔約半年舉辦一次，總共連續五屆。許多本土漫畫家如[[張靜美|張靜美]]、[[任正華|任正華]]、[[高永|高永]]、[[周顯宗|周顯宗]]、[[陳弘耀|陳弘耀]]、甘麗瑛、柯翠芬（筆名納蘭真）等，皆從此獎項嶄露頭角。

== 概要 ==
[[呂墩建|呂墩建]]自1979年盜印未經授權的日本漫畫、出版《[[小咪漫畫周刊|小咪漫畫周刊]]》以來，意識到培養本土漫畫家的重要性。他曾於1980年表示：
{{quote|記得去年（按：1979年）在《[[書評書目|書評書目]]》上看到[[洪建全教育文化基金會|洪建全基金會]]在兒童文學創作獎中增加漫畫獎項，在興奮激動的情緒下，我登時歡呼萬歲；還以為[[中國人|中國人]]終於承認漫畫是包含於[[兒童文學|兒童文學]]的領域，漫畫界抬頭的大好時機來臨了！當然，聽說在最後評審階段，漫畫還是受人排擠，當屆從缺，日後還會有被刪除的厄運。呂大哥聽到這個令人噴飯的消息後，一方面氣惱頑固人士的歧見，一方面感嘆：能扳回中國漫畫界生機的唯一冀望又告無著落！在氣憤之餘，將原本要參加比賽的漫畫稿件撕碎，立下誓願：「將來的某一天，我一定要舉行個有生有色的漫畫比賽！他們不願意辦，我來！」爾後，尋思良久，得到一個結論：只有跟著時代的潮流，造成出版周刊的風氣，才有可能扭轉漫畫界日漸沈淪的命脈，在有份量的周刊按造計畫訓練中國漫畫新生代。……<ref>全文請參看《小咪漫畫周刊》第九期，頁228。</ref>
}}
是故，伊士曼小咪出版社自1980年9月2日舉辦第一屆小咪漫畫新人獎，參賽資格為自創16至32頁的短篇故事漫畫，以[[沾水筆|沾水筆]]墨畫稿參加，而獎金則為：
# 第一名：獎學金[[新台幣|新台幣]]6千元，獎品同高級生獎。
# 第二名：獎學金新台幣3千元，獎品同上。
# 第三名：獎學金新台幣2千元，獎品同上。
# 佳作獎：名額視情況而定。

《小咪漫畫周刊》另外開闢「新人講座」專欄，刊載得獎者的訪談介紹；並將得獎作品集結成冊，出版了五本《漫畫新人獎專輯》。1982年11月6日為第五屆，但隨著該漫畫雜誌的停刊，這個漫畫比賽也在此屆成為絕響。

== 評論 ==
雖然《[[小咪漫畫周刊|小咪漫畫周刊]]》未經原出版社授權，盜印許多日本漫畫；但是該刊為了與[[東立出版社|東立出版社]]的盜版漫畫雜誌《[[東立漫畫周刊|東立漫畫周刊]]》抗衡，選擇[[少女漫畫|少女漫畫]]路線作為市場區隔，為了培養本土人才而主辦小咪漫畫新人獎的用心是有目共睹的。有了小咪漫畫新人獎這個平臺，[[張靜美|張靜美]]、[[任正華|任正華]]、[[高永|高永]]、[[周顯宗|周顯宗]]、[[陳弘耀|陳弘耀]]、納蘭真等這些被挖掘出的少女漫畫新星<ref>不過，任正華後來脫離少女漫畫慣有的甜美華麗畫風，轉向爆笑幽默（如《頑劣家族》）、奇幻警世（如《修羅海》）的題材；而陳弘耀的作品則轉向寫實武俠（如《一刀傳》）。</ref>在1980年代以後才能在臺灣漫壇大放異彩。
