'''小家鼠'''（學名：''Mus musculus''）也稱為'''家鼠'''、'''鼷鼠'''或'''小鼠'''。

== 特征 ==
小家鼠体长约8厘米，尾巴略短或略长于体长；毛色变化大，从黑灰色到灰褐色，腹毛基部为灰色，尖端常为灰黄色，有的呈灰白色，有的腹毛全白，足为暗褐色，也有的白色；尾上下两色。
許多人認為在实验中，都是用白色小家鼠，因白所以容易侦查，其實這是不正確的，實驗用的小鼠顏色多樣，以全身黑色或白色的品系為大宗，另有諸多品系為灰色或褐色。

== 习性 ==
世界各地都有小家鼠。是[[生物學|生物學]]上的[[模式生物|模式生物]]之一，它们和人类的构造很相似，而且鼠繁殖得很快，价钱也便宜。它们也是很受欢迎的宠物。

不过，家鼠也是一种害兽，它们会吃掉农作物和传播疾病，[[國際自然保護聯盟|國際自然保護聯盟]][[物種存續委員會|物種存續委員會]]的入侵物種專家小組（ISSG）列為[[世界百大外來入侵種|世界百大外來入侵種]]。

人类饲养的鼠可以活两年至三年，但在野外的鼠通常只会活三个月至一年間。鼠通常吃素，但是它们其实是杂食的。
