'''东京都立小石川后乐园'''（又称'''小石川后乐园'''，东京当地也常略称为'''后乐园'''），是位于日本[[东京都|东京都]][[文京区|文京区]]后乐一丁目的一座都立庭园，是东京都内赏梅、赏樱和观赏红叶的名所。庭园面积约7万平方米，是东侧毗邻的[[东京巨蛋|东京巨蛋]]的约1.5倍，曾经是[[水戶德川家|水戶德川家]]的庭园。庭园名“后乐”来自于[[战国|战国]][[儒学思想家|儒学思想家]][[孟子|孟子]]所作《[[孟子_(书)|孟子・梁惠王上]]》之中的名句“贤者而后乐此”。小石川后乐园是日本的国家指定[[特别史迹|特别史迹]]、[[特别名胜|特别名胜]]。

== 景觀 ==
[[File:Korakuen_Gartenplan.jpg|thumb]]
[[File:Koishikawa_Korakuen_(2008.09).JPG|thumb]]
后乐园位于[[小石川|小石川]]台地的南端（现在的[[文京区|文京区]]后乐1丁目），从[[神田上水|神田上水]]分出一条支流引入庭园内形成了以池水为中心的[[回游式庭园|回游式]][[假山|假山]]泉水[[日本庭园|庭园]]，在约7万平方米的宽广的庭园内，种植了[[梅花|梅花]]、[[樱花|樱花]]、[[杜鹃花|杜鹃]]等3000多株植物，在四季呈现出不同情趣的景色。

由于[[德川光圀|德川光圀]]的友人[[朱之瑜|朱舜水]]的参与，庭园的设计里融入了很多[[儒家思想|儒家思想]]的元素，和明朗开放的[[六义园|六义园]]形成鲜明的对比。后乐园除了将各地著名的湖、山、河流、田园景观巧妙的模拟体现外，还有随处可见以[[中国|中国]]的名胜命名的景观，如[[西湖|西湖]]、[[庐山|庐山]]等，因此被称为“洋溢着中国趣味的深山幽谷”<ref>{{cite web |url=http://www.kensetsu.metro.tokyo.jp/kouen/kouenannai/park/koishikawa_ko.html |title=都立公園·庭園案内 |publisher=东京都建设局 |language=ja |accessdate=2010-04-01}}</ref>。

== 庭园历史 ==
在[[江户时代|江户时代]]初期的1629年（[[宽永|宽永]]6年），[[水戶德川家|水戶德川家]]的初代[[藩_(日本)|藩]]主[[德川赖房|德川赖房]]从[[江户幕府|江户幕府]]拜领了今天后乐园及其周边的土地作为自己和家族在[[江户城|江户城]]的居住地，并在此设计、兴建房屋和后园。庭园的建造在第二代藩主[[德川光圀|德川光圀]]的主持下完成，并听取友人、[[明朝|明朝]]遗臣[[朱之瑜|朱舜水]]的意见，依据[[战国|战国]]儒学思想家[[孟子|孟子]]所作《孟子梁惠王上》之中的名句“贤者而后乐此”将庭园命名为'''后乐园'''<ref>{{cite web |url=http://teien.tokyo-park.or.jp/contents/outline030.html |title=小石川後楽園 > 概要 |publisher=東京都公園協会 |language=ja |accessdate=2010-04-01}}</ref>。

此后的后乐园长期作为水户藩主的私人庭园，直至1869年（[[明治|明治]]2年），最后一任水户藩主[[德川昭武|德川昭武]]在[[版籍奉还|版籍奉还]]时同时将住宅和庭园寄赠给新[[日本政府|政府]]。[[陆军省|陆军省]]管辖这片土地时，将在原先住宅的地块上建设了[[东京炮兵工厂|东京炮兵工厂]]（后来改建为后乐园球场，即现在[[东京巨蛋|东京巨蛋]]的所在地），但是没有破坏后乐园，将庭园风貌保存了下来，作为[[明治天皇|明治天皇]]外出（{{lang|ja|行幸}}）的休息地点<ref>{{cite web |url=http://www.uraken.net/rail/travel-urabe217.html |title=小石川後楽園〜東京都文京区〜 |publisher=裏辺研究所 |language=ja |accessdate=2010-04-01}}</ref>。

1923年（[[大正|大正]]12年）3月7日，后乐园被日本政府指定为国家史迹和名胜，在指定时为了和[[冈山县|冈山县]]的[[后乐园|后乐园]]进行区别，在名称前冠以所在地[[小石川|小石川]]，称为'''小石川后乐园'''<ref>{{cite web |url=http://www.city.bunkyo.lg.jp/sosiki_busyo_shomu_bunkazai_sitei_bunkazai_kunishitei.html |title=国指定文化財 |publisher=文京区 |language=ja |accessdate=2010-04-01}}</ref><ref>{{cite web|url=http://www.city.bunkyo.lg.jp/visitor_kanko_shiseki_garden_korakuen.html|title=小石川後楽園|publisher=文京区|language=ja|accessdate=2010-04-01}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1938年（[[昭和|昭和]]13 年）4月3日，小石川后乐园在[[东京都|东京都]]进行修整之后作为都立庭园向一般民众收费开放。1952年（[[昭和|昭和]]27年）3月，小石川后乐园根据日本的《[[文化財保護法|文化財保護法]]》被指定为日本的国家[[特别史迹|特别史迹]]和[[特别名胜|特别名胜]]。截止目前为止，日本全国只有9处地点受到这样的双重指定（东京都2处，另一处为[[滨离宫恩赐庭园|旧滨离宫庭园]]）。
