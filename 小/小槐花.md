'''小槐花'''（[[學名|學名]]：'''''Desmodium caudatum''''' or '''''Ohwia caudata'''''，別名：'''銳葉小槐花'''、'''茉草'''、'''抹草'''、'''磨草'''、'''魅草'''、巴人草、扁草子、草鞋板、豆子草、逢人打、旱螞蝗、螞蝗草、拿身草，中藥名：'''清酒缸'''，[[閩南方言拼音方案|閩語]]：bbua caur），為豆科[[落葉植物|落葉]][[固氮作用|固氮]]植物。在中國大陸及台灣等地區被作為中藥用、及[[民俗|民俗]]避邪用的小[[灌木|灌木]]。<ref name=qingjiugang>Trout’s Notes #D-2,"Desmodium caudatum(青酒缸)"{{cite web |url=http://trout.yage.net/sc/D2_2004_Trout.pdf |title=存档副本 |accessdate=2006-01-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20050831103118/http://trout.yage.net/sc/D2_2004_Trout.pdf |archivedate=2005-08-31 }},Trout’s Notes
on the Genus Desmodium,March 2004.</ref>

== 形态特征 ==
*莖：小灌木，高約1－2公尺。<ref>{{cite web|url=http://www.pfaf.org/database/plants.php?Desmodium+caudatum|title=Desmodium caudatum - Plants For A Future database report|publisher=www.pfaf.org|accessdate=2008-05-02|last=|first=}}</ref>
*葉：三出複葉，披針形，小葉長約8公分、寬約3公分。葉表為綠色，葉背為白綠色。含當藥素([[:en:Gnetum_gnemon|swertisin]])。
*花：總狀花序，綠白色，長約0.7公分，蝶形花冠。
*果實：果實為莢果扁平，長約5－8公分，被鉤毛。種子為圓形褐色，以種子繁殖。
*根/莖：含有0.087%[[二甲基色胺|二甲基色胺]](DMT)。<ref>"Desmodium caudatum information from NPGS/GRIN"{{cite web |url=http://www.ars-grin.gov/cgi-bin/npgs/html/taxon.pl?311307 |title=存档副本 |accessdate=2008-05-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20100905174753/http://www.ars-grin.gov/cgi-bin/npgs/html/taxon.pl?311307 |archivedate=2010-09-05 }},Retrieved 2008-05-02.</ref>
*[[自然固氮|自然固氮]]特性：為豆科[[固氮作用|生物固氮]]植物。

== 原產地 ==
原產於[[中國大陸|中國大陸]]中南部、[[臺灣|臺灣]]中、北部山野、[[日本|日本]]、[[琉球|琉球]]。<ref>

== 用途 ==
*藥用：中藥內服或外傷用。如:鎮痛，解熱，殺菌，解毒和殺蟲劑。全株熬煮作為婦女腹部絞痛，癤，癰腫，痢疾，十二指腸球部潰瘍，發燒，腸胃炎，流行性感冒，麻疹和乳腺炎等治療。<ref name=qingjiugang/>
*民俗用：民間用於避邪用。
*庭園園藝觀賞用。
