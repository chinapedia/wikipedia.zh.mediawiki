'''小儿经'''或'''小儿锦'''（{{rtl-lang|ar|شِيَوْ عَر دٍ}}），简称'''小经'''、'''消经'''是运用[[阿拉伯字母|阿拉伯字母]]书写[[汉语|汉语]]（特指[[兰银官话|兰银官话]]、[[中原官话|中原官话]]与[[东北官话|东北官话]]）的一种非正规的书写文字。主要是被[[中华人民共和国|中国]]境内信奉[[伊斯兰教|伊斯兰教]]的[[回族|回族]]、[[东乡族|东乡族]]与[[撒拉族|撒拉族]]，为了教导启蒙儿童学习《[[古兰经|古兰经]]》而所运用的。小儿经是用來拼寫漢語的拼音文字中歷史比較悠久的一套。<ref name="宁夏新闻网">{{Cite web|url=http://www.nxnews.net/382/2004-8-12/20@45280.htm|title=中国最早的中文拼音—小经文字|publisher=宁夏新闻网|date=2004年8月4日|accessdate=2008年5月25日|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20051124141129/http://www.nxnews.net/382/2004-8-12/20%4045280.htm|archivedate=2005年11月24日}}</ref>

小儿经文字与其他阿拉伯字母文字系统一样，从右向左读写。与[[维吾尔语|维吾尔语]]文字的写作方式相同的是在运用[[阿拉伯字母|阿拉伯字母]]时，要求标出所有的[[元音|元音]]。然而多数运用阿拉伯字母来书写的语言（如[[阿拉伯语|阿拉伯语]]、[[波斯语|波斯语]]、[[乌尔都语|乌尔都语]]）仅标出[[元音長度#長元音|长元音]]。这有可能是因为汉语对元音[[音位|音位]]的重视，所以小兒經在为漢字標音时采取《古兰经》中赘余地标示[[元音長度#短元音|短元音]]的方式來為漢字標出準確的讀音，而不是像维吾尔语仅用長元音來代替短元音。小儿经文字却并不标出[[聲調|声调]]。这并没有导致日常使用中出现较大的混淆。

== 名称 ==

小儿经不仅是一套不规则的文字系统，也没有一个统一的名称。[[陕西省|陕]]东、[[山西省|晋]]、[[河北省|冀]]、[[河南省|豫]]、[[山东省|鲁]]、[[北京市|北京]]、[[天津市|天津]]以及[[中国东北地区|东北]]地区普遍称之为“小儿经”，简称“小经”或“消经”（“消”字在當地有“复习”的意思）。然而在[[宁夏回族自治区|宁]]、[[甘肃省|甘]]、[[内蒙古自治区|蒙]]、[[青海省|青]]以及[[中国西北地区|西北]]地区则称之为“小儿锦”，简称“小锦”。<ref name="宁夏新闻网" />[[东乡族|东乡族]]称之为“东乡文”或“-{回回文}-”。[[撒拉族|撒拉族]]称之为“撒拉文”。居住於中亚的[[东干族|东干族]]在使用拉丁文字和西里尔文字前曾经也用过与小儿经相似的阿拉伯文字，称之为“-{回文}-”。<ref name="宁夏旅游网">{{Cite web|url=http://www.nxtour.com.cn/1003/1021/20050812/1021@669.html|title=回族的语言和文字|publisher=宁夏旅游网|date=2005年8月21日|accessdate=2008年5月25日|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20080623201715/http://www.nxtour.com.cn/1003/1021/20050812/1021%40669.html|archivedate=2008年6月23日}}</ref>

关于“消经”这名称的来源，学者们持有两种看法。中国西北地区方言中有“消一消”这种说法，就如“消化”一般，意思就是“复习复习”。一种说法便是学徒们在教堂学完后，用小儿经来“消一消”[[阿訇|阿訇]]所讲的阿拉伯文经文。这样来说，用汉语的“消经”则与古阿拉伯语的“本经”成对比。<ref name="华夏经纬网">{{Cite web|url=http://www.huaxia.com/ssjn/hxfq/00197480.html|title=回族语言文字习俗传承|publisher=华夏经纬网|date=2004年4月22日|accessdate=2008年5月25日|language=zh-cn}}</ref>在[[新疆维吾尔自治区|新疆]]有给《[[古兰经|古兰经]]》手工加注多余元音符号的习惯，写出来的经文便称为“消经”。另一种说法认为小儿经在辅音字母上加注元音符号的复杂特征，使得人们给予了“消经”这称谓。

中国[[华北地区|华北]]一带，人们普遍认为小儿经是在[[汉语拼音|汉语拼音]]、[[注音符號|注音符号]]前，回民为了教育启蒙学童学汉字正音的一种工具，之所以得称“小儿经”。中国西北地区有“小儿锦”这种称呼，学者们普遍认为是“小儿经”的讹读。那里的人们也常把-{{IPA|[ŋ]}}韵尾写成-{{IPA|[n]}}韵尾。

== 起源 ==

[[伊斯兰教|伊斯兰教]]自公元7世纪中叶传入[[中国|中国]][[唐朝|唐朝]]以来，迁进了不少以[[阿拉伯语|阿拉伯语]]和[[波斯语|波斯语]]为母语的[[民族|民族]]。<ref name="宁夏旅游网" />百年后、这些[[民族|民族]]与[[中国|中国]]当地的[[民族|民族]]融合，形成了现在的[[回族|回族]]。[[元朝|元朝]]期间，朝廷还专门在国内各个行省设立“回回国子学”和“回回国子监”这类学习兼研究回回语言文字的正式机构，教授“亦思替非文字”（即波斯文字）。<ref name="宁夏新闻网" />[[伊斯兰教|伊斯兰教]]为了《[[古兰经|古兰经]]》的经义不在[[翻译|翻译]]中变义，要求其教徒以[[阿拉伯语|传统阿拉伯语]]的原文来阅读《古兰经》。信奉伊斯兰教的教徒从小（5、6岁）就需要去经堂学习阿拉伯语与《古兰经》经文。这些学徒的[[汉字|汉字]]水平并不高，并且常常是住在[[陕西省|陕]][[甘肃省|甘]][[寧夏省|宁]][[青海省|青]]教育水平、生活水平较低的村乡。中国的不少教徒为了背记《古兰经》，常常在自己的笔记中用[[阿拉伯语|阿拉伯文]]来拼写[[汉语|汉语]]，也时常用这种方式给阿拉伯语经文中的生单词加上汉语注解。<ref name="华夏经纬网" />由此一套用阿拉伯字母来书写汉语的[[文字|文字]]系统逐渐地形成了。目前发现小儿经文最早的实物印证是在现[[陕西省|陕西]]西安大学习巷清真寺内的[[元朝|元代]]碑刻：碑上用[[阿拉伯语|阿拉伯语]]刻着《古兰经》经文，並在底部用小儿经拼写出刻碑者人名与修建此[[清真寺|清真寺]]人员的人名。碑文撰于[[伊斯兰历|伊斯兰教历]]740年，即公元1339年7月9日至1340年6月26日间。<ref name="宁夏新闻网" />

== 使用 ==
[[Image:Book-in-Xiaoerjing.png|275px]]出版的一部伊斯兰专著。阿拉伯语原文下面补充用小儿经标记的汉语译文。]]

小儿经可分为“经堂式”与“现代式”二类。“经堂式”是清真寺经堂裡的学徒和[[阿訇|阿訇]]用来作简单笔记的小儿经系统。“经堂式”掺加着不少阿拉伯语、波斯语的特殊伊斯兰教单词，从来不掺加[[汉字|汉字]]。这个系统较为标准，可以说是一个完整的[[文字|文字]]系统；“现代式”是回民用在日常书信中的小儿经系统，经过了一番更革，字母的拼法被简化，在用阿拉伯字母拼写的单词间还常常掺加简单的[[汉字|汉字]]，是一种阿、汉混合书写体系。<ref name="宁夏新闻网" />这种文字习惯在书写的人与人之间有着很大的区别。书写者按照自己对阿拉伯语、波斯语字母的认识，配上自己方言的发音，来形成自己独特的书写系统。一般的情况下只有写信的与收信的可以看懂，外人看不懂。

=== 现代 ===
[[File:Jianshui_Xian_-_Qujiang_Zhen_-_Malishu_-_P1360760.JPG|thumb]]
随着[[中华人民共和国|中华人民共和国]]的经济开放与[[汉字|汉字]]教育在乡村的普及，小儿经的使用已渐渐地消失了。<ref name="宁夏旅游网" />自1980年代中期，中国内外有不少学者开始了对小儿经的研究工作。他们常常去[[中国西北地区|中国西北地区]]作实地考察，搜取并收集用小儿经书写的文品（其中[[南京大学|南京大学]]民族与边疆研究所“小经”课题研究小组的收藏最为全面），而且还采访了当地认识小儿经的居民。<ref name="宁夏新闻网" />这些学者发现小儿经失去了已往在经堂教育中发挥的用途，而却多用于亲人的书信中。很多亲人、朋友用小儿经来写密信，不懂小儿经的外人读起来很难认懂。

===東干語===
[[清朝|清代]]末年遷入中亞的回族，即俄國人所稱之[[东干族|东干族]]，在1927年以前亦使用小兒經為書寫文字。1927年后[[苏联|苏联]]創制了以小兒經為基礎的回族文字，1928年改用[[拉丁字母|拉丁字母]]，1953年又改爲[[西里尔字母|西里尔字母]]（参见：[[东干语|东干语]]）。

== [[字母|字母系统]] ==

[[阿拉伯语|阿拉伯语]]和[[波斯语|波斯語]]单词的元音均依照相对语言的拼写规则拼写。表示长[[元音|元音]]，但不表示短[[元音|元音]]。
阿拉伯语的{{rtl-lang|ar|<big> ﹾ </big>}}号虽可以在阿拉伯语、波斯语单词上省略，但用于汉语单词时不能省略。
例外是一些常用的汉语单音节字上可以省略{{rtl-lang|ar|<big> ﹾ </big>}}号。例如：“的”、“和”在强调性下分别写成{{rtl-lang|ar|<big> دِ </big>}}与{{rtl-lang|ar|<big> ﺣَ </big>}}，非强调性时写成{{rtl-lang|ar|<big> دْ </big>}}与{{rtl-lang|ar|<big> ﺣْ </big>}}，但也可以省略地写成{{rtl-lang|ar|<big> د </big>}}与{{rtl-lang|ar|<big> ﺣ </big>}}。
同样，表示-{{IPA|[ŋ]}}韵尾的{{rtl-lang|ar|<big> ﹾ </big>}}号也常会被人们写成{{rtl-lang|ar|<big> ﹰ </big>}}或{{rtl-lang|ar|<big> ﹴ </big>}}或{{rtl-lang|ar|<big> ﹲ </big>}}。
代表长元音“-ā”的{{rtl-lang|ar|<big> ﺎ </big>}}在多音节的汉语单词中可以省略为表示短元音“-ă”的{{rtl-lang|ar|<big> ﹶ </big>}}号。

小儿经略同汉语拼音，一个词写在一起，词与词用一空格分开。

阿拉伯语的{{rtl-lang|ar|<big> ﹼ </big>}}号用于汉语单词时，表示着单字的重复，相当于中文的“[[叠字符号|々]]”号。

标点符号可用阿拉伯语标点符号，可用汉语标点符号，也可混着使用（汉语句号、顿号，阿拉伯语逗号、引号）。

小儿经主要用于伊斯兰教的教导工作，用小儿经出版的宗教类书籍也必然占多数。小儿经的读物大多是《小经古兰》类的三文合一（阿拉伯文，中文，小儿经）《[[古兰经|古兰经]]》。综合来说，出版物可分为宗教问答、宗教经文、阿拉伯语语言学习、与工具书四大类。在[[注音符號|注音符号]]与[[汉语拼音|汉语拼音]]的发创之前，回民的不少字典、词典都使用小儿经为汉字注音。其中由《[[中文消经双解噶兑新字典|中文消经双解噶兑新字典]]》和《[[满俩字典|满俩字典]]》最典范、最流行。以下是一个不完整的列表列出了一些用小儿经“经堂文”书写的文献：
