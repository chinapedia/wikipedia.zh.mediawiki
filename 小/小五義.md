'''小五義'''及'''續小五義'''乃是[[清朝|清朝]][[光緒|光緒]]年间口耳相傳的評書內容，為[[三俠五義|三俠五義]]的續作，原創作者有[[石玉昆|石玉昆]]、佚名兩種說法，「小五義」指的五鼠後人，鑽天鼠盧方之子粉面子都盧珍、徹地鼠韓彰義子霹靂鬼韓天錦、穿山鼠徐慶之子山西雁徐良、錦毛鼠白玉堂的姪兒玉面專諸白芸生和「小俠」艾虎五人，全書主題環繞在群俠於忠協助平定藩王作亂、於義懲治為惡盜匪兩大主題，全書貫徹忠義思想，歌頌行俠仗義精神。
== 内容簡介 ==
'''小五義'''及'''續小五義'''延續[[三俠五義|三俠五義]]背景，故事主線集中在破解勾結[[西夏|西夏]]造反的襄陽王所設種種陰謀（如尋回被盜走的顏查散官印、大破銅網陣等）以及討伐為他所用的各地盜匪進而瓦解其勢力，（如策反[[君山|君山]]寨主鐘雄、剷除郭家營、東方家族、朝天嶺盜匪等），故事中間還穿插了飛賊白菊花竊取皇帝衣冠，群俠奉命追捕的支線。內容包羅萬象，比如各式寶刀寶劍的來歷、暗器短打、機關陣法（銅網陣、藏珍樓），並由討伐盜賊的械鬥衍生至大規模的國家作戰。書中的[[五義|五義]]多已成家，但是[[錦毛鼠|錦毛鼠]][[白玉堂|白玉堂]]為探[[襄陽|襄陽]]王府之機密，中了王府中的銅網陣，遭萬箭創身而亡，由其姪子[[白芸生|白芸生]]代叔為國效命。

==主要人物==
===七俠和五義===
*'''｢北俠｣歐陽春'''
於破銅網陣後出家，但被宋仁宗敕令於開封府附近的護國寺修道，與'''｢黑妖狐｣智化'''是至交。武器為一把七星寶刀，後趕赴至陷空島救助[[盧方|盧方]]，擲棍欲擊殺反賊時，不慎誤擊殺｢小諸葛｣沈仲元。
*'''｢南俠｣｢御貓｣[[展昭|展昭]]'''
人稱南俠，為御前四品帶刀護衛，因其身手靈活，仁宗皇帝欽賜御貓稱號。
*'''「奇俠」｢黑妖狐｣[[智化|智化]]'''
十分有謀略，曾设计盗取九龙珍珠冠栽赃马朝贤，又曾诈降君山的钟雄。
*'''｢雙俠｣丁兆蘭、丁兆惠'''
展昭的內兄，於陷空島發生危機時率先發現，並放出信鴿請蔣平先回來坐鎮。
*'''｢隱俠｣「小諸葛」沈仲元'''
是七俠中年紀次輕的，做事頗為衝動，但是善謀劃。於奪回陷空島時，因太急不慎被歐陽春擲出的禪棍重擊[[頭部|頭部]]而陣亡。
*'''｢小俠｣[[艾虎|艾虎]]'''
七俠中年紀最小，同時亦是小五義的人物，智化之徒弟。
*'''鑽天鼠[[盧方|盧方]]'''
｢粉面子都｣盧珍之父，於本書初期已長住陷空島。雖已過耳順之年，仍能力抗比其小一輪的白菊花等人，支撐到蔣平等人趕回來救援。
*'''｢徹地鼠｣[[韓彰|韓彰]]'''
｢霹靂鬼｣韓錦的義父，善於製作火藥及挖掘地道，徹地鼠之名因此得來。
*'''｢鑽山鼠｣[[徐慶|徐慶]]'''
｢白眉鬼｣徐良之父，善於化妝易容，武器是[[鐵爪|鐵爪]](或[[鐵抓|鐵抓]])。
*'''｢翻江鼠｣[[蔣平|蔣平]]'''
人稱'''｢蔣四爺｣'''、'''｢蔣爺｣'''，外號｢詐病鬼｣，五鼠中的智囊，善水，武器於本書中改為短槍。陷空島危機時接到丁家兄弟緊急傳書，趕回丁宅坐陣。
*'''｢錦毛鼠｣[[白玉堂|白玉堂]]'''
｢玉面專諸｣白芸生之叔，功夫了得，五鼠中唯一不善水的人，武器為刀。於故事第三回探[[襄陽王|襄陽王]]府中了銅網陣，遭[[弩|弩]]箭機括和埋伏的兵士萬箭創身而殉職，死後仍站的立挺挺，讓襄陽王府中的賊人大為驚駭。

===朝廷===
*宋仁宗
史實皇帝，因王叔襄陽王一直蠢蠢欲動讓他不得安眠。
*包拯
史實人物，文曲星下凡，嫉惡如仇。擔任龍圖閣大學士一職，幫助顏查散穩軍心。
*顏查散
七俠五義和包公案出場時是為趕考的書生，被馮君衡陷害未果反讓其自食惡果，為該年科考狀元，娶得柳金嬋為妻(詳見包公案之｢探陰山｣段)。白玉堂之義兄。於本書中放為巡查大人，對抗襄陽王一眾。
*王丞相
包拯好友。
本名為王玄齡，為虛構人物。

===開封府相關===
*公孫策
[[開封府|開封府]]的師爺兼參政，是包拯的左右手，在小五義中與白玉堂一起隨隨顏查散到襄陽調查襄陽王。

*王朝、馬漢、張龍、趙虎
四大知名捕快，雖有年紀但尚能智擒奸賊等。

===小五義===
*'''｢玉面專諸｣白芸生'''
[[白玉堂|白玉堂]]的姪子，故事初期上開封府欲投靠白玉堂，於某茶店被冒失的韓錦潑了一頭茶。得知[[白玉堂|白玉堂]]殉職之後，於開封府落戶。武器是[[短槍|短槍]]與與之後在藏珍樓得到的[[魚腸劍|魚腸劍]]，對決賊人時以槍格擋，以劍突刺。
*'''｢粉面子都｣盧珍'''
[[盧方|盧方]]之子，師承其父，故亦會水。因遺傳其母之長相，所以長相標誌清秀，其幼年時常被[[韓彰|韓彰]]取笑說：｢汝父為一臉橫肉大老粗，怎生得如此標誌的孩兒！｣武器是短[[戟|戟]]，他的妻子是展昭的姪女展小霞（展昭的大哥展輝之女）。
*'''｢霹靂鬼｣韓天錦'''
[[韓彰|韓彰]]的義子，外表魁梧，長相略憨厚老實，其力大是其標誌，武器是[[淬棒|淬棒]]。
*'''｢小俠｣艾虎'''
見｢七俠｣說明
*'''｢山西雁｣徐良'''
[[徐慶|徐慶]]之子，别号白眉大侠  特徵是兩道白色倒八字劍眉和白面，故徐慶每次見到他都暗中搖頭。善於化裝與易容術，於劇情中常裝成｢[[日遊神|日遊神]]｣和｢[[夜遊神|夜遊神]]｣驚嚇賊人女眷。武器是金丝大环刀和青龙剑，但更善於[[暗器|暗器]]。其暗器为墨玉飞蝗石，袖箭，斤镖，颈背低头花桩弩 百发百中。

===展家===
*'''丁月華'''
: 展昭的妻子，丁老夫人的女兒(實為姪女)，丁家兄弟的妹妹(實為表妹)。
: 與展昭是「不打不相識」。《[[七俠五義|七俠五義]]》中，丁兆蘭邀請展昭到丁家作客，丁兆惠見展昭的隨身佩劍，請展昭試劍幾下，問展昭此劍的名字，展昭回答這是其父所遺贈的劍，名字叫做'''「[[巨闕|巨闕]]」'''。稍後丁兆蘭說家中有一劍，請展昭試試。於是叫ㄚ環去月華的閨房中借了'''「[[湛盧|湛盧]]」'''，讓展昭看看。展昭稍揮過之後，覺得不太順手，丁兆惠說此劍似乎不合展昭，展昭便回此劍對他而言算輕的。此時兆蘭偷偷跑去找月華，以[[激將法|激將法]]說月華的劍不好，於是月華便在大怒之下出來見客。展昭一見丁月華雖然面容姣好、身材婀娜多姿但卻是怒髮衝冠且臉上出現青筋，覺得奇怪。兆蘭對著展昭說：「輕視劍即是輕視人，現在我家小妹不依了」丁月華請展昭與之對試，不到數回合，展昭的胸口衣服被月華劃破，月華頭上的珠飾被展昭打落。展昭認輸，不過兆惠卻說其實是月華輸了，月華害羞地跑回去閨房。而後丁老夫人說出丁月華其實是她的兄弟的女兒，在月華年幼時就過世，於是把月華當作親生女兒撫養。之後當場招展昭為女婿，並將名劍「[[巨闕|巨闕]]」和「[[湛盧|湛盧]]」交換，數月後成親。
: 展驥、展駿的母親。
*'''展驥、展駿'''
:展昭與丁月華的雙胞胎兒子。展驥使著「巨闕」劍，展駿使用短槍。

== 作者爭議 ==
早在[[光緒|光緒]]16年（1890年）的善成堂刻印版中，署名文光樓主人的序裡頭便寫明作者為[[石玉昆|石玉昆]]，但[[胡適|胡適]]指出小五義和三俠五義內容前後不一，且小五義出版乃石玉昆去世十多年後的事情，但接續者為誰，具體已不可考。
