'''小港街道'''，是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[北仑区|北仑区]]的一个[[街道_(行政区划)|街道]]。东面与[[戚家山街道|戚家山街道]]相连，南面与[[大碶街道|大碶街道]]、[[新碶街道|新碶街道]]相接，西面与[[鄞州区|鄞州区]]相连，北面隔[[甬江|甬江]]与[[镇海区_(宁波市)|镇海区]]相望。总面积63.9平方公里，人口4.3万，[[街道办事处|街道办事处]]位于红联江南东路788号。

== 沿革 ==
小港街道[[清代|清代]]以前为定海县崇邱乡。[[中华民国|民国]]19年（1930年）改为崇邱区，25年（193年）与灵岩部分乡镇合并为崇岩区，30年（1941年）改长山区。1985年10月属新建[[北仑区|北仑区]]（原名滨海区），2003年设小港街道。

== 下辖 ==
目前，小港街道辖有红联社区、陈山社区、高河塘社区、竺山社区共4个社区。共辖有45个行政村。

== 交通 ==
[[甬江隧道|甬江隧道]]连接小港街道和镇海区。[[江南公路|江南公路]]、[[通途路|通途路]]经过小港街道通往宁波市区。[[泰山路|泰山路]]从小港街道出发进入北仑城区和[[北仑港|北仑港]]。[[宁波绕城高速公路|宁波绕城高速公路]]经过小港街道，设小港出口，连接江南公路。

== 工业 ==
小港街道辖地拥有许多大型[[民营企业|民营企业]]。其中的代表有[[贝发集团|贝发集团]]、[[海天集团|海天集团]]和[[球冠集团|球冠集团]]。

== 人文 ==
小港是宁波的人文重地。[[四明学派|四明学派]]的开创者[[沈焕|沈焕]]是小港人。小港也是[[宁波商帮|宁波商帮]]的重要发祥地。[[李也亭|李也亭]]、[[张逸云|张逸云]]等商界名人出自小港。
