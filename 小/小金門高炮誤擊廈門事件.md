'''小金門高炮誤擊廈門事件'''是一起兩岸軍事事件，發生於1994年11月14日。駐守[[小金門|小金門]]的[[中華民國陸軍|中華民國陸軍]]158師砲兵部隊使用[[波佛斯40公釐高射砲|波佛斯40公釐高射砲]]试射时，射程计算错誤，误击[[厦门市|厦门市]]郊[[思明區|思明]]黄厝村塔头社，造成四名[[民工|民工]]被炮弹炸伤，其中重傷兩人<ref>{{cite web |url = http://big5.huaxia.com/lasd/lagxdsj/2003/06/474981.html |title=華夏經緯網-兩岸關係大事記 |publisher=[[華夏經緯網|華夏經緯網]] |author= |accessdate=2016-07-04 |date= 2003-06-12 |language=zh-tw}}</ref>，一度造成兩岸雙方緊張。但當時兩岸正在開始談判，關係算和緩，[[解放军|解放军]]没有还击<ref>{{cite web |url=http://www.rocme.org/archiver/?tid-1301.html|title=四三期戰工回憶(15)_小金門誤擊廈門事件 |language=zh-tw}}</ref>，經兩岸協商後，化解此事。

==背景==
此事件發生在[[李登輝|李登輝]]總統訪美前半年，此時兩岸關係相對和緩<ref>{{cite web |url =http://www.ettoday.net/news/20160704/728326.htm |title=郭正亮／雄三飛彈誤射的兩岸陰謀論 |publisher=[[ETtoday|ETtoday]] |author= |accessdate=2016-07-04 |date= 2016-07-04 |language=zh-tw}}</ref>。

==經過==
[[小金門|小金門]]火砲部隊原規定試射一律向南方海面射擊，但因考慮[[民航機|民航機]]起降安全，士官長通知射手轉向2度以[[檳榔嶼|檳榔嶼]]方向為射向，在考慮射程後將45度仰角提高到55度，再度檢修後的第2次試射，砲兵營長王道平再將仰角提高到65度，總計射擊61枚砲彈。

1994年11月14日，小金門前埔[[波佛斯40公釐高射砲|40高炮]]陣地一名黃姓士官長，為了鑒定[[火炮|火炮]]是否修復完成，不遵守金門戰備規定向金防部報告，徑行試射61發火炮，原本陳姓營長反對射擊，但黃士官長以「火炮射程打不到廈門，使用空炸信管也不會落到對岸」為由說服長官。結果本應空炸的砲彈沒有引爆<ref>{{cite web |url =http://n.yam.com/yam/international/20160705/20160705780729.html |title=雄三誤射非首例 小金門22年前曾誤轟廈門|publisher=yam蕃薯藤新聞 |author= 朱冠宇|accessdate=2016-07-05 |date= 2016-07-05 |language=zh-tw}}</ref>，再加上當時強勁西風，把[[炮彈|炮彈]]吹到射程之外，導致誤擊廈門郊區，造成部分房屋受損與4名大陸居民受傷<ref>{{cite web |url =http://udn.com/news/story/1/1806304-%E8%AA%A4%E5%B0%84%E6%87%B2%E8%99%95-%E5%B0%88%E5%AE%B6%EF%BC%9A%E5%8F%83%E8%80%83%E9%87%91%E9%96%80%E8%AA%A4%E5%B0%84%E4%BA%8B%E4%BB%B6 |title=誤射懲處 專家：參考金門誤射事件
 |publisher=[[台灣醒報|台灣醒報]] |author= |accessdate=2016-07-04 |date= 2016-07-04 |language=zh-tw}}</ref>。

==處理==
厦门政府给予伤者免费治疗，并发给生活补助，並都康復順利出院。他们很快养好伤，出院了。[[海基會|海基會]]還特別前往善後，賠償了事<ref>{{cite web |url =http://www.appledaily.com.tw/appledaily/article/headline/20060409/2526199/  |title=61砲誤打廈門 12年前險開戰|language=zh-tw}}</ref>。当时，[[金门|金门]]有一个爱心团体打电话到[[厦门市|厦门市]]台办金门事务处，表示要派人到厦门来，给伤者送[[慰问金|慰问金]]。但是厦门方面婉言谢绝了他们的善意。因为中央领导有明确指示：两岸尚未结束敌对状态，从[[政治|政治]]上记他一笔帐即可，不提赔偿<ref>{{cite web |url =http://m.xilu.com/view_302804_all.html |title=鲜为人知：1994年小金门台军“炮击厦门”事件|language=zh-tw}}</ref>。

根據[[監察院|監察院]]的調查以及[[立法委員|立法委員]]質詢官員發現，金门当时还装备着二战时期美制的40毫米高炮，但火炮严重老化，需要不断维护，而導致這場事件發生<ref>{{cite web |url =https://newtalk.tw/news/view/2016-07-01/74763|title=戰力升級，從四○高砲到雄三飛彈（管仁健）|language=zh-tw}}</ref><ref>{{cite web |url =http://news.qq.com/a/20160703/011099.htm|title=台军乌龙事件多如牛毛，还曾误轰厦门，击落拖靶飞机|language=zh-tw}}</ref>。

時任師長[[趙世璋|趙世璋]][[少將|少將]]在[[國防部陸軍司令部|陸軍總司令部]]主持的事後檢討會過程中，當著[[中華民國陸軍司令|陸軍總司令]][[李楨林|李楨林]][[上將|上將]]等人當面提出自請處分，以示負責。

==影響==
[[国台办|国台办]]发言人对此事件表示严重关注，认为这是一起破坏[[海峡两岸|海峡两岸]]和平气氛的恶性事件，並强烈谴责台湾当局的罪恶行径，坚决要求台湾当局迅速查明情况，公布事实真相，并严惩肇事者<ref>{{cite web |url =http://news.ltn.com.tw/news/focus/paper/1006647|title=看問題》軍方誤擊不是第一樁|publisher=自由時報author=施曉光|language=zh-tw}}</ref>。

原本在1990年代，在金門有著名的「高空防護射擊訓練」演習，諸如[[50機槍|50機槍]]、[[迫擊炮|迫擊炮]]、[[榴彈炮|榴彈炮]]、[[照明彈|照明彈]]等火力在夜空中形成交叉火網，成為當時金門著名的夜景。但該演習在此事件後中止至今<ref>{{cite web |url =http://taiwan.dwnews.com/big5/news/2016-07-01/59750405.html|title=時隔二十多年 台灣再发生誤擊事件|publisher=多維新聞|author=曾子衿|language=zh-tw}}</ref>。
