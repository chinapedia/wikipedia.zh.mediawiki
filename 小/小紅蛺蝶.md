'''小紅蛺蝶'''，是一種[[蛺蝶|蛺蝶]]，屬[[鱗翅目|鱗翅目]][[蛺蝶科|蛺蝶科]]。可能是世界上分布最廣的蝴蝶，分布於除了[[南極洲|南極洲]]及南美洲以外的所有大陸<ref name=ButterflyConservation>{{cite web|title=Painted Lady|url=http://www.butterfly-conservation.org/butterfly/32/butterfly.html?butterflyid=38|work=A-Z of Butterflies|publisher=Butterfly Conservation|accessdate=22 October 2012|deadurl=yes|archiveurl=https://web.archive.org/web/20121022154913/http://butterfly-conservation.org/Butterfly/32/Butterfly.html?ButterflyId=38|archivedate=2012年10月22日}}</ref>。

== 形態 ==
展翅長5.1 - 7.3 公分。翅膀背面橘、褐色；翅端黑色，有明顯的白帶和小白點。翅膀腹面顏色暗淡，褐或灰色。

== 分布 ==
分布於所有溫帶地區，包含熱帶地區的山區。這個物種在溫暖地區是定居種，但會在春季遷徙，有時秋季。例如：每年5月和6月，這種蝴蝶會從[[北非|北非]]和[[地中海|地中海]]遷徙到[[北歐|北歐]]；在[[北美|北美]]分布於[[中美洲|中美洲]]和[[美國|美國]]西南部；如果冬季有適當的雨量，可以促使族群往北遷徙，最北可達[[加拿大|加拿大]]和[[冰島|冰島]]。春季發生的大遷徙規模可以非常驚人，數以百萬計的蝴蝶連續7 - 8 週通過同一個地區。但很少往南遷徙，在[[巴拿馬|巴拿馬]]以南很罕見。在[[澳大利亞|澳大利亞]]則僅局限分布於西南部。

是[[香港|香港]]一種不太常見的[[蝴蝶|蝴蝶]]。於[[南風半島|南風半島]]這些地方能看見牠們的蹤影。

== 生態 ==
小紅蛺蝶是[[完全變態|完全變態]]昆蟲，經歷[[卵|卵]]、[[幼蟲|幼蟲]]、[[蛹|蛹]]和[[成蟲|成蟲]]等階段。

卵為薄荷綠色，約產卵後3 - 5日孵化。幼蟲有5齡，約12 - 18日後化蛹。蛹期約10日。成蟲可活兩個星期。在溫帶地區終年發生。

成蝶在多種植物上吸蜜，特別是[[菊科|菊科]]植物。幼蟲以超過100種植物為食草，包含[[菊科|菊科]]、[[紫草科|紫草科]]、[[錦葵科|錦葵科]]、[[豆科|豆科]]、[[馬鞭草科|馬鞭草科]]、[[薔薇科|薔薇科]]、[[蓼科|蓼科]]、[[繖形科|繖形科]]和[[鼠李科|鼠李科]]等植物。

由於飼養容易，在[[幼稚園|幼稚園]]常被當作用來了解蝴蝶生活史的教材。
