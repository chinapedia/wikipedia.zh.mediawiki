《'''小松先生'''》（{{lang-ja|おそ松さん}}）是為了紀念日本[[漫畫家|漫畫家]][[赤塚不二夫|赤塚不二夫]]誕辰80周年，由其創作的搞笑漫畫作品《'''[[阿松_(漫畫)|小松君]]'''》（{{lang-ja|おそ松くん}}）所改編的[[電視動畫|電視動畫]]第三作。第1期自2015年10月起於[[東京電視系|東京電視系]]播放，第2期自2017年10月起播放。故事描述松野家六胞胎的日常生活。

== 登場人物 ==
作品中的角色會因為短篇的需要而有各式各樣的身分。由於並無正式翻譯名稱，斜線後為其他常見譯名。
=== 六胞胎 ===
松野家的六胞胎是本作的主角，身高、長相、外型大致相同。為了讓觀眾易於分辨，製作組使用了代表色作區別，並替角色增添了許多個人特點。六人的年齡皆為20多歲，生日5月24日，血型A型。所有人都喜歡魚魚子。全員無業，皆為[[啃老族|啃老族]]、[[處男|處男]]。

; {{nihongo|松野小松／松野阿松／松野粗松|松野 おそ松|Matsuno Osomatsu|}}
: 聲優：[[櫻井孝宏|櫻井孝宏]]
: 長男，代表色為紅色。
: 有兩根[[呆毛|呆毛]]。喜歡[[賭馬|賭馬]]和打[[小鋼珠|小鋼珠]]，女朋友是[[租書店|租書店]]架上的[[色情書刊|色情書刊]]，偶爾會不經意說出下流的言詞。常被輕松說心思不夠細膩，是個單憑興致行事、沒心沒肺的傢伙。長子意識強，表面上說希望自己是[[獨生子|獨生子]]，但其實很重視兄弟；默默地關注著弟弟們的生活，默默地在背後守護著他們。

; {{nihongo|松野唐松／松野空松|松野 カラ松|Matsuno Karamatsu|}}
: 聲優：[[中村悠一|中村悠一]]
: 次男，代表色為藍色。
: 富有自戀傾向、[[中二病|中二病]]，容易被無視，呆毛兩根。常戴墨鏡、身穿皮夾克和印有自己肖像的背心，偶爾還會戴彩色美瞳。易被周遭的人說「痛」，因而自喻為「刺蝟的憂愁」。對工作的事完全沒有籌劃，[[釣魚|釣魚]]時會以[[情書|情書]]和鮮花作為魚餌。喜歡著溫柔的自己因此才溫柔，是個容易沉浸於自我滿足中的[[自戀|自戀]]狂。雖然總是愛耍帥，但只要一被人凶馬上就會變得軟弱起來。常被椴松吐槽金光閃閃的褲子。

: 聲優：[[神谷浩史|神谷浩史]]
: 三男，代表色為綠色。

; {{nihongo|松野一松|松野 一松|Matsuno Ichimatsu|}}
: 聲優：[[福山潤|福山潤]]
: 四男，代表色為紫色。
: 死魚眼、有兩根呆毛，頭髮蓬鬆，渾身散發著負能量，和貓咪很要好，性格彆扭而不擅表達內心真實的想法；看上去十分陰暗，但其實有能力把握與他人相處的平衡，曾被椴松吐槽就是個普通人。平時與十四松相處較多，會被綁在球棒上讓十四松做揮棒練習。面對唐松出其不意的發言總是特別凶。拙於與人相處，帶著些許的自卑感且害怕受到傷害；堅稱自己並非交不到朋友而是不想交，孤癖的外表下藏著一顆渴望得到友情的心。

; {{nihongo|松野十四松|松野 十四松|Matsuno Jyushimatsu|}}
: 聲優：[[小野大輔|小野大輔]]
: 五男，代表色為黃色。
: 有一根呆毛，總是擺著一張笑臉，非常喜歡棒球，平時習慣做揮棒練習，口頭禪是「謝謝超大再見本壘打」等棒球相關用語。活動量極大，但戀愛後會變得異常沉穩，一松的說法則是「變成普通的人類了」。特別怕冷，一旦持續處於受寒狀態便會性格大變，甚至會發瘋似地想殺人。總是活力充沛精神飽滿，若是突然一副輕描淡寫的樣子反而會讓人覺得很可怕。行事出人意料，時而非常笨拙。与一松的关系最为友好，一松会绑上球棒上让对方练习，其他兄弟一般有对他有疑问时，都会找一松，让一松照顾他。

; {{nihongo|松野椴松／松野多多松／松野末松|松野 トド松（松野 トド松）|Matsuno Todomatsu|}}
