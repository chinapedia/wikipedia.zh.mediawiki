'''小菜'''是對份料較小的[[菜餚|菜餚]]之稱呼，與份量較大的「[[主菜|主菜]]」相對，又可稱為'''副食'''或'''配菜'''。有時可指[[開味小菜|開味小菜]]，或指單人或少數人食用的[[餐點|餐點]]，依各地的民情而有不同的定義。

== 中華料理 ==
[[中華料理|中華料理]]中所稱的小菜，是指份量較少、以方便2至4人享用的正式菜餚。相對於小菜，即是標準分量適合一桌人，例如[[婚宴|婚宴]]、[[慶功宴|慶功宴]]的10至12人一圍享用的大碟菜式。

從前光顧[[餐館|餐館]]的[[顧客|食客]]，必須要多約幾位[[朋友|朋友]]或[[同事|同事]]同桌進餐，才可能多點幾款菜，否則要餐後[[打包|打包]]及浪費[[金錢|金錢]]和[[食物|食物]]。而小菜的設計，就是為了吸引每圍枱人數小，又想多試幾味菜式的食客。中式餐館的小菜可以要求加大，但要在點菜時跟[[侍應|侍應]]聲明，例如要加大到6至8人食用，但[[價錢|價錢]]會較高。

===台灣===
[[台灣|台灣]]所稱的小菜有兩種意思。第一種與傳統中華料理上的意義相近，例如台灣有一類標榜「[[清粥小菜|清粥小菜]]」的餐館，會提供多達數十種樣式的小菜，供顧客搭配清[[粥|粥]]或食用；[[麵食|麵食]]餐館也會提供類似的小菜，供顧客搭配麵類食用。另一種則是指正式菜餚（主菜）上來之前，提供顧客開胃並打發時間的簡易菜餚，與開味小菜同義，這種主要出現在[[宴會|宴會]]場合上。台灣的小菜選料以烹調簡單的食物為主，常見的有[[瓜子|瓜子]]、[[涼拌|涼拌]][[小黃瓜|小黃瓜]]、涼拌[[海蜇皮|海蜇皮]]、[[椒鹽|椒鹽]][[毛豆|毛豆]]等，一般均使用小碟子裝。

== 日本料理 ==
[[日本料理|日本料理]]中的小菜可稱為「惣菜/總菜」（{{jpn|j=惣菜|hg=そうざい、そうさい|rm=Sōzai, Sōsai|f=h}}；「惣」字讀音同「總」）或「[[御數|御數]]」（{{jpn|j=おかず|hg=御数、御菜|rm=Okazu|f=h}}）。惣菜是指已經[[調理|調理]]過的[[食物|食物]]，等同中文所稱的「配菜」；為了因應外食族群的增加，現今在日本的[[市場|市場]]、[[百貨|百貨]]等地的餐飲店均會販售即食之用的惣菜，甚至以[[冷凍食品|冷凍食品]]販售。御數則指搭配[[米飯|米飯]]等[[主食|主食]]食用的餐點，與中文稱的「[[家常菜|家常菜]]」相近，通常以小碗或小碟子盛裝。

== 韓國料理 ==
在[[韓國|韓國]]餐館裡，顧客一點完[[主菜|主菜]]，服務生便會送上6至8樣「免費」的「[[飯饌|飯饌]]」(반찬)，而且吃完還可以再加菜。伴餐的內容主要為各式[[泡菜|泡菜]]、[[醃漬物|醃漬物]]以及涼拌菜，其中半數以上都會是辣的。
