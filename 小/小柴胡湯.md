'''小柴胡湯'''是一種[[方劑|方劑]]，源自《[[傷寒雜病論|傷寒雜病論]]》。

== 組成 ==
[[柴胡|柴胡]]<sub>半斤</sub>、[[黄芩|黄芩]]<sub>三两</sub>、[[人參|人參]]<sub>三两</sub>、[[甘草|甘草]]<sub>三两，炙</sub>、[[半夏|半夏]]<sub>半升，洗</sub>、[[生薑|生薑]]<sub>三两，切</sub>、[[大棗|大棗]]<sub>十二枚，擘</sub> <ref>方剂学，[[段富津|段富津]]主编，[[上海科学技术出版社|上海科学技术出版社]]，1995.6. ISBN 978-7-5323-3708-8</ref>

== 用法 ==
上七味，以水一斗二升，煮取六升，去滓，再煎，取三升，温服一升，日三服。

== 主治 ==
伤寒少阳证、妇人热入血室。

== 方解 ==
柴胡與黄芩之組合是[[柴胡劑|柴胡劑]]的核心，而小柴胡湯是柴胡劑的基本型方劑。柴胡依序到大棗這幾樣藥為多數柴胡劑的共通點。柴胡劑在表、裏證分類上，通常歸為半表半裏證型的治療方劑。

中医学认为柴胡與黄芩可以治療所謂「胸脇苦滿」之症，表現上是[[脇|脇]]至[[胸|胸]]苦重。分析上，有說以為是[[痰病|痰病]]在脇肋一帶。

中医学认为半夏可以治療噁心之症。生薑與大棗出現在許多方劑中，古書裡稱其為「調和[[營|營]][[衛|衛]]」之用，大致上可說是調整氣血。[[甘草|甘草]]則可以調和諸藥，也是在許多方劑中會出現的。人參現在認為是補藥，但一說是仲景時代應是用[[黨參|黨參]]，黨參則較不熱燥，不唯補氣，亦能恢復[[津液|津液]]。

全體而言，中医学认为柴胡湯主治半表半裏的發熱虛症。有說法是[[少陽經|少陽經]]的[[經方|經方]]藥。

== 毒副作用 ==

[[日本|日本]]、[[台湾|台湾]]都有报道患者因服用小柴胡汤而出现[[急性肝炎|急性肝炎]]，可能是其中的黄芩引起的。<ref>[http://www.xys.org/xys/netters/Fang-Zhouzi/zhongyi/xiaochaihukeli.txt 常用中成药的真相——小柴胡颗粒（冲剂、片）][[方舟子|方舟子]]，[[新语丝|新语丝]]，2009-01-19造访</ref>

與[[干擾素|干擾素]]併用，以及[[肝硬化|肝硬化]]、[[肝癌|肝癌]]患者給予小柴胡湯，可能會發生[[間質性肺炎|間質性肺炎]]，嚴重甚至可能致死。其他如[[偽醛固酮症|偽醛固酮症]]、[[肌病變|肌病變]](myopathy)、肝功能障害、[[黃疸|黃疸]]皆可能發生。

中醫最忌諱亂用藥，不講辨證施藥。離開了中醫的辯證，任何神藥都是毒藥，
如果邪在表或在裏，用小柴胡就是把邪氣往肝膽引。以上這些報道並不是因為小柴胡的某個成分造成，而是沒有辯證，比如，表證應該解表，發汗後就沒事的，可用了小柴胡就是把邪氣往肝膽引，造成嚴重後果。

中藥的使用必須考慮個人體質，服用前向合資格中醫師查詢，即可避免不良作用發生。
