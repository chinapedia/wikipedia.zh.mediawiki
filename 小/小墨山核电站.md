'''小墨山核电站'''，是一座规划建设的内陆[[核电站|核电站]]，位于[[中华人民共和国|中华人民共和国]][[湖南省|湖南省]][[华容县|华容县]]，计划总装机500万千瓦，总投资700亿元人民币，预计“十三五”规划末期开工建设<ref>[http://www.yueyang.gov.cn/xqdt/hrx/content_263948.html 华容小墨山核电已完成投资3亿元] {{Wayback|url=http://www.yueyang.gov.cn/xqdt/hrx/content_263948.html |date=20160304070323 }}</ref>。

== 地理位置 ==
核电站选址位于华容县东山镇塔市驿管理区杨家湾村小墨山北坡，为[[丘陵|丘陵]]地形，山前为长江冲积平原，与[[湖北省|湖北省]][[监利县|监利县]]隔江相望，附近人口密度较小。选址北距[[长江|长江]]1.7公里，西南距[[华容|华容]]县城34公里，东南距[[岳阳|岳阳]]市区46公里，距离[[长沙|长沙]]196公里。核电站供水水源为长江，取水提升高度25米，能够满足直流供水、循环冷却用水和核电设备航运条件<ref>[http://hn.rednet.cn/c/2009/01/09/1684525.htm 湖南力争将华容小墨山核电列入国家发展规划]</ref>。

== 建设历程 ==
小墨山核电站的建设计划自20世纪70年代即开始酝酿。1977年，湖南省启动核电厂选址，小墨山在22个预选厂址中被确定为最优厂址。1983年华容县核电厂址保护小组成立。2005年4月，在中国电力规划总院牵头组织的湖南核电项目初可研报告评审会上，小墨山被评定为中国内陆核电站厂址资源最佳的厂址之一，推荐其作为湖南核电项目的优先首选厂址。2006年2月22日，小墨山核电项目在国家发改委组织召开的湖南、湖北、江西三省核电项目专家优选会议上脱颖而出<ref>[http://www.heneng.net.cn/index.php?mod=npp&item=nps_show&nps_id=35 中国核电信息网：小墨山核电站]</ref>。2007年4月5日，中国电力投资集团公司在长沙举行了湖南小墨山核电厂建设协议签字仪式，中电投湖南核电公司正式挂牌成立，负责湖南岳阳小墨山核电的开发建设<ref>[http://www.cpnn.com.cn/dlsj/200704090222.htm  小墨山核电厂举行建设协议签字仪式]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 技术规划 ==
小墨山核电站计划建设4台125万千瓦核电机组，项目采用[[西屋|西屋]][[AP1000|AP1000]][[压水反应堆|压水反应堆]]技术，属第三代反应堆型核电站<ref>[http://hunan.sina.com.cn/news/s/2014-03-31/080594098.html 岳阳小墨山核电完成投资约3亿元 总投资约700亿]</ref>。

== 项目停滞 ==
2008年，中国南方地区遭受[[2008年中国南方水灾|严重冰雪灾害]]，[[国家发改委|国家发改委]]将内陆核电建设提上议程。但由于政策原因，发改委只允许[[中国核工业集团公司|中国核工业集团公司]]、[[中国广核集团有限公司|中国广核集团有限公司]]和[[中国电力投资集团公司|中国电力投资集团公司]]这三家公司各选择一个省开发一个项目。由于小墨山核电站开发方中国电力投资集团公司已经选择建设江西[[彭泽核电站|彭泽核电站]]项目，导致小墨山核电项目没有入选内陆首批核电站建设项目。在其后的一段时间内，小墨山核电项目一直处于停滞状态，因厂址保护需要，周边工农业生产和其他建设也被禁止，厂区居住和出行条件恶化，给当地百姓造成极大的不便而饱受诟病<ref>[http://www.chinastock.com.cn/yhwz_about.do?methodCall=getDetailInfo&docId=4579096 等待37年的小墨山核电还需继续等下去吗]</ref>。

2011年3月11日，日本发生[[2011年日本东北地方太平洋近海地震|大地震]]从而引发[[福岛第一核电站事故|核灾]]。灾后第六天，[[中国国务院|中国国务院]]决定暂停审批所有核电项目，包括开展前期工作的项目<ref>[http://money.163.com/12/1025/07/8EL7D83N002529T0.html 内陆核电梦断“十二五” 百亿投资如何善后存疑]</ref>。2012年10月，国务院通过《核电安全规划（2011—2020年）》和《核电中长期发展规划（2011—2020年）》，确定“‘十二五’时期只在沿海安排少数经过充分论证的核电项目厂址，不安排内陆核电项目。”<ref>[http://money.163.com/12/1024/17/8EJLB43O00253B0H.html 国务院：稳妥重启核电建设 暂不安排内陆核电]</ref>。此举使得小墨山核电站项目在短期内有所进展的希望更加渺茫。

== 近况 ==
2014年，国家发改委批复的《洞庭湖生态经济区规划》将小墨山核电站的建设起止年限明确为2020-2030年，总装机500万千瓦，一期装机250万千瓦。2015年4月21日，[[湖南省发改委|湖南省发改委]]发布能源建设重大项目，据报已将小墨山核电站项目列入厂址保护，力争在“十三五”末核准开工。<ref>《潇湘晨报》2015年4月22日A3版：小墨山核电站力争“十三五”末开工</ref>。
