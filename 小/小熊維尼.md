'''小熊維尼'''（{{lang-en|'''Winnie the Pooh'''}}，簡稱為'''Pooh'''、'''Pooh Pooh'''或是'''Pooh Bear'''），1925年12月24日首次面世，以[[聖誕|聖誕]]故事形式在[[倫敦|倫敦]]《新聞晚報》刊出；第一本《小熊維尼》故事書於1926年10月出版。小熊維尼是[[艾倫·亞歷山大·米恩|艾倫·亞歷山大·米恩]]為他的兒子创作的一隻[[漫畫|漫畫]][[熊|熊]]；而古典維尼是由謝培德（E.H.Shepard）所繪，后由[[華特·迪士尼公司|華特·迪士尼公司]]購入并經過重新繪製，推出後因其可愛的外型與憨厚的個性，迅速成為世界知名的卡通角色之一。此后人们為了區別兩種不同風格的維尼，稱呼米爾恩時期由謝培德（E.H.Shepard）繪製的維尼為「古典維尼」（Classic Pooh）；而後全球粉絲便把米爾恩的生日（[[1月18日|1月18日]]）推為國際小熊維尼日。

小熊维尼在故事中有很多朋友，小豬與跳跳虎是最常出現的两个角色。

== 命名由來 ==
=== Winnie ===
“小熊維尼”是一名[[雄性|雄性]]的小[[熊|熊]]。雖然名称Winnie是取自一名同名的雌性小熊，但原作者米爾恩一直用「他」來稱呼小熊維尼。而最初真實的小熊Winnie是以牠家鄉[[加拿大|加拿大]][[緬尼托巴省|緬尼托巴省]]首府[[溫尼伯|溫尼伯]]（Winnipeg）的簡稱來命名。

=== Pooh ===
在小熊系列故事的第一本《Winnie-the-Pooh》的序中提到，“Pooh”曾是维尼的主人罗宾为一只[[天鵝|天鵝]]取的名字；另一說是小熊維尼在吹走停在他[[鼻子|鼻子]]上的[[蜜蜂|蜜蜂]]時，嘴里会发出Pooh、Pooh的声音。
== 歷史 ==
=== 起源（古典維尼時期） ===
維尼的命名是由米爾恩的兒子克里斯多福·羅賓·米爾恩的生日禮物絨毛玩具熊而來，還有其他角色大部分也是根據克里斯多福的玩具而命名，但是其中瑞比跟貓頭鷹則是根據真實世界中的動物來命名的。而克里斯多福則是根據他跟父親在[[倫敦動物園|倫敦動物園]]裡看到的那隻從[[加拿大|加拿大]][[緬尼托巴|緬尼托巴]]省首府[[溫尼伯|溫尼伯]]來的熊，而命名他的那隻生日禮物絨毛玩具熊。

=== [[迪士尼|迪士尼]]時期 ===
迪士尼公司在1961年獲得了授權，於是經過重新繪製後，於1966年起推出迪士尼版畫風的維尼系列。

== 角色 ==
* 小熊維尼（Winnie the Pooh）
: 主角，雄性，只穿著紅色的上衣，不穿[[褲子|褲子]]，主要食糧是[[蜂蜜|蜂蜜]]，而當[[蜂蜜|蜂蜜]]吃完以後常常立刻跑去瑞比家要[[蜂蜜|蜂蜜]]；雖然常常因為[[腦袋|腦袋]]小而提供錯誤解決辦法，不過對朋友很講[[義氣|義氣]]。
: 另維尼家的門上寫著「MR. SANDERZ」。
* 小豬（Piglet）
: 雄性，他是一隻粉紅色的[[小豬|小豬]]，因為體型小，所以特別膽小。
[[File:Hong_Kong_Disneyland_1398.JPG|200px]]]]
* 跳跳虎（Tigger）
: 雄性，一隻喜歡彈跳的[[老虎|老虎]]，常常會用[[彈簧|彈簧]]尾巴到處跳來跳去，因此時常會有小[[破壞|破壞]]。
* 屹耳（Eeyore）
: 雄性，一隻時常很憂鬱的[[驢子|驢子]]，住在山頂上只用幾根木頭所搭建的「屋子」裡。常常愁眉苦臉，自怨自艾，又因頸部不能挺直，令他看來沒精打采；幸有維尼和好朋友為他帶來歡樂，因此他會默默守護在朋友的身邊。特徵是尾巴經常會[[甩尾|甩尾]]。
* 瑞比（Rabbit）
:雄性，一隻喜歡[[園藝|園藝]]的[[兔子|兔子]]，在家門前面有一大片的菜園。
* 袋鼠媽媽（Kanga）
: 雌性，小荳的媽媽。
* 小荳（Roo）
: 雄性，袋鼠媽媽的兒子，是一隻六歲的小袋鼠，性格活潑開朗，跟跳跳虎最要好，也喜歡和大伙一起玩拋樹枝大賽。
* 貓頭鷹（Owl）
: 雄性，常常會扮演解決問題的智者，不過也是跟維尼一樣常常提供錯誤解答。
* 嘟嘟（Lumpy）
: 雄性，原本屬於維尼與他朋友們所害怕的長鼻怪（Heffalump）之一，不過在《嘟嘟的萬聖節》中最後變成他們的好朋友。
* 谷弗（Gopher）
: 地鼠，爆破專家，這個角色是迪士尼原創。
* 克里斯多福．羅賓（Christopher Robin）
: 男孩子，根據米爾恩的兒子-{克里斯多福·羅賓·米爾恩}-所創造出來的小男孩角色，是維尼的主要人類朋友，常常會親暱的對維尼說「小笨熊（''Silly old bear''）」。
* 黛比．羅賓（Darby Robin）
: 女孩子，克里斯多福的妹妹，在《'''My Friends Tigger & Pooh'''》系列初登場，以成為偵探為目標並在百畝林裏與維尼和跳跳虎成立超級拯救隊。
* 捕快（Buster）
: 雄性，羅賓家的愛犬。
