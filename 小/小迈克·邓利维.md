'''小迈克尔“迈克”·约瑟夫·邓利维'''（{{lang-en|'''Michael "Mike" Joseph Dunleavy, Jr.'''}}，{{bd|1980年|9月15日|}}） 通常称'''小邓利维'''，生于[[德克薩斯州|德克薩斯州]][[沃斯堡|沃斯堡]]，美国职业篮球运动员，现效力于NBA联盟的[[亞特蘭大老鷹隊|亞特蘭大老鷹隊]]。他是前NBA球员和前[[洛杉矶快船队|洛杉矶快船队]]主教练[[老迈克·邓利维|老邓利维]]的儿子。

== 早年生涯 ==
小邓利维在1999年毕业于俄勒冈的Jesuit高中，他带领球队在1999年4A州际锦标赛中65-38击败了北塞伦高中。

== 大学生涯 ==
邓利维在1999-2002年在篮球名校[[杜克大学|杜克大学]]打球。大学三年级时他在比赛中打出场均17.3分和7.2个篮板的数据并且帮助球队取的了31胜4负的佳绩，入选了美国篮球教练员联盟(NABC)评选出的全明星第一阵容。[http://sports.espn.go.com/nbadraft/tracker/player?playerId=18349] 二年级时，在NCAA总决赛中他取得了全队最高的21分，并且在下半场投中三个三分球使得球队打出一个11-2的制胜小高潮，最终杜克大学蓝魔队夺得了2001年的NCAA总冠军。

[[杜克大学|杜克]]主教练[[迈克·沙舍夫斯基|迈克·沙舍夫斯基]]称邓利维:“他是我所见过的自[[格兰特·希尔_(篮球)|格兰特·希尔]]后最为全能的球员。”

== 职业生涯 ==
在[[2002年NBA选秀|2002年NBA选秀]]中第一轮第三顺位被[[金州勇士队|金州勇士队]]选中。2005年11月，勇士和小邓利维签订了一份5年4400万美元的扩展合同。金州勇士队总经理[[克里斯·穆林|克里斯·穆林]]说:“从迈克的球场成绩方面，他的表现方面和我们球队的运作方面来看，我们双方都觉得需要对方”。

这份合同受到了来自球迷的批评，尽管球队还和其他球员包括[[阿多纳尔·福伊尔|阿多纳尔·福伊尔]]和[[德里克·费舍尔|德里克·费舍尔]]签订了大合同。2005-06赛季，小邓利维由于在比赛中投篮手感很差，导致多场比赛中失去了主力[[小前锋|小前锋]]的位置，随后在赛季后期又夺回了先发位置。在2006-07赛季他被置于一个球队中的新位置[[大前锋|大前锋]]，在赛季初期的不适应后，勇士队主教练[[唐·尼尔森|唐·尼尔森]]将他放入板凳席中，以寻求球队很好的化学变化和赢球方法。

2007年1月17日，邓利维在一次球员大交换中被交换到[[印第安那步行者|印第安那步行者]]。
[[File:Mike_Dunleavy,_Jr._with_Indiana_Pacers_in_December_2007.jpg|thumb]]

芝加哥公牛前鋒Mike Dunleavy近日動了後背部手術，預計休養8至10個星期，新球季開幕可能報銷。

2017年1月7日，邓利维與[[莫·威廉斯|莫·威廉斯]]被交易到了[[亞特蘭大老鷹|亞特蘭大老鷹]]隊，騎士換來了[[凱爾·柯佛|凱爾·柯佛]]，和一個受保護的未來第一輪選秀權。
