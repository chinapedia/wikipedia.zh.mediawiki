'''小燕子'''（{{Lang-ko|{{Link-Wikt|꽃|제비}}}}，{{Unicode|''Kot Chebi''}})是[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]对本国[[流浪儿|流浪儿]]的称法。因为[[燕子|燕子]]经常在各处停留找食，故得此名<ref>{{cite web |url = http://books.google.com/books?id=cdQ8QZU6H0MC&pg=PT93&lpg=PT93&dq=kotchebi&source=web&ots=qMNU2MUIpU&sig=FKxJPs3I8B0OkFaCnnEyfpyZtjE&hl=en&sa=X&oi=book_result&resnum=3&ct=result#PPT93,M1 |title = Kim Jong Il's North Korea |website = Google Books |language= |accessdate=2008-08-12 }}</ref> 。朝鲜官方并不承认流浪儿的存在，在国家出版物和文献中该词被禁止使用<ref name=dailynk>{{cite web |url = http://www.dailynk.com/english/read.php?cataId=nk01500&num=3659 |title=The Old Generation Calls North Korea "the Great Poverty Country" |website = Dailynk.com |language=|accessdate=2008-08-11 }}</ref>

== 最初来源 ==
“小燕子”一词最初于20世纪90年代[[朝鲜饥荒|朝鲜大饥荒]]时期出现。此期间，朝鲜的食物供给系统全面崩溃<ref name="dailynk" />。为“确保全国流浪乞讨人员的生活”，朝鲜政府于1995年建立儿童收留营。由于收留营条件恶劣，有儿童死于营养不良<ref>{{cite web |url = http://www.awomansvoice.org/nl1-2004-2.html |title = Intervention Agenda Item 13: Rights of Children |website = Awomansvoice.org |language= |accessdate=2008-08-12 |deadurl=yes |archiveurl = https://web.archive.org/web/20120306003144/http://www.awomansvoice.org/nl1-2004-2.html |archivedate=2012-03-06 }}</ref>。

== 小燕子的生活 == 
“小燕子”不得不四处找食，有时也会成群乞讨、盗窃<ref name="dailynk" />。大部分“小燕子”一天只吃一餐；他们通常的食物为野菜汤、菜粥，和草根。<ref>{{cite web |url = http://www.niew.gov.my/project/DPR_KOREA.pdf |title = DEMOCRATIC PEOPLE'S REPUBLIC OF KOREA |website=Niew.gov.my |language= |accessdate=2008-08-12 |deadurl=yes |archiveurl = https://web.archive.org/web/20110722233201/http://www.niew.gov.my/project/DPR_KOREA.pdf |archivedate=2011-07-22 }}</ref>
