{{日本作品|小公主莎拉|小公女セイラ|A Little Princess}}，是一部於2009年10月17日起至2009年12月19日，由[[日本|日本]][[TBS電視|TBS電視台]]於逢星期六19:56 - 20:54（[[日本時間|日本時間]]）播出的[[日本電視劇|電視連續劇]]，由[[志田未來|志田未來]]、[[林遣都|林遣都]]、[[田邊誠一|田邊誠一]]等主演。

== 故事 ==
與原作[[小公主莎拉|動畫]]不同，本作的舞台是現代日本，而非[[英國|英國]]。而且電視劇內容與原作及動畫差異也相當大。

== 登場人物及演員 ==
=== 主要人物 ===
* 黑田莎拉 - [[志田未来|志田未来]]（幼少期：[[中山凛香|中山凛香]]）
: 主角。15歲→16歳。在印度成長的大小姐。小時候母親薫子過世，父親寶貝似的撫育著她。但是在進入米勒紐斯女學院後，父親突然過世。擁有龐大的財產瞬間化為烏有，而淪落為學院的女傭。
* 三浦海人 - [[林遣都|林遣都]]
: 18歲。米勒紐斯女學院的傭人。
* 亞蘭由起夫 - [[田邊誠一|田邊誠一]]（幼少期：[[相澤瑠星|相澤瑠星]]）
: 法語教師。
* 東海林雅美 - [[岡本杏理|岡本杏理]]
: 16歲。女學院的學生，莎拉最好的朋友。
* 水島馨 - [[忽那汐里|忽那汐里]]
: 16歲。女學院的學生。
* 武田真里亞 - [[小島藤子|小島藤子]]
: 16歲。女學院的學生。本名「寬子」，但她本人討厭而改成「真里亞」。
* 黑田龍之介 - [[谷中敦|谷中敦]]（東京スカパラダイスオーケストラ）
: 莎拉的父親。
* 黑田薰子 - [[黑川智花|黑川智花]]
: 莎拉的母親。在故事一開始就已經過世。
* 小沼誠一郎 - [[大和田伸也|大和田伸也]]
: 米勒紐斯女學院的廚師。
* 小沼日出子 - [[廣岡由里子|廣岡由里子]]
:誠一郎的妻子。
* 三村笑美子 - [[齊藤由貴|齊藤由貴]]
: 千惠子的妹妹。
* 三村千惠子 - [[樋口可南子|樋口可南子]]（少女期：[[溝口萬莉萌|溝口萬莉萌]]）
: 米勒紐斯女學院的院長。
