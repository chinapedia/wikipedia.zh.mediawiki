《'''小長門有希的消失'''》（{{lang-ja|長門有希ちゃんの消失}}）是[[谷川流|谷川流]]的原作小說《[[涼宮春日系列|涼宮春日系列]]》的公式[[二次創作|衍生作品]]，由外傳《小涼宮春日的憂鬱》的作者[[Puyo_(漫畫家)|ぷよ]]繪製的[[日本|日本]][[漫畫|漫畫]]作品。

== 概要 ==
《小長門有希的消失》是基於曾經改編成動畫電影的原作小說第4卷《[[涼宮春日的消失|涼宮春日的消失]]》的設定及世界觀為基礎。由外傳《小涼宮春日的憂鬱》的作者[[Puyo_(漫畫家)|ぷよ]]以長門有希為主角及視野從長門進行的另一種可能性發展的外傳漫畫作品。描寫「感情豐富、文靜害羞少女」的長門有希與阿虛的戀愛喜劇故事。

2009年7月起在《Young ACE》創刊號（[[角川書店|角川書店]]）連載至2016年9月號。2013年12月18日宣佈進行動畫製作企劃<ref>{{Cite web |url=http://news.ameba.jp/20131218-163/ |title=《涼宮春日》外傳作品《小長門有希的消失》動畫化企劃進行中|work=日刊[[Ameba_News|Ameba News]]|accessdate=2013-12-18|language=ja}}</ref>。2015年4月至7月播出電視動畫，全16話。

故事和人物設定維持《[[涼宮春日系列|涼宮春日系列]]》原作小說的基本設定，但與原作小說不同的是，「長門有希和朝倉涼子是外星人」、「朝比奈實玖瑠是未來人」、「古泉一樹是超能力者」、「涼宮春日想許的願望必定會實現的能力」這些特殊能力的設定均刪除，取而代之的是所有人物和阿虛一樣都是普通的人類（即所謂的一般正常人）<ref>例如《涼宮春日的憂鬱》登場的綠色長條氣球狗狗綠先生（{{lang|ja|キミドリさん}}）在本作未出場。不過，在長門家有仍有放著狗狗造型的綠色長條氣球。</ref>，本作故事大體按照原小說的設定推進，但因角色均是正常人，所以故事結果均與原作不同。

而這部作品的時間設定則是快速進行，單行本第3卷時長門等人剛升上2年級，朝比奈等人則升上3年級（值得一提的是原作小說版的阿虛和長門升上2年級時被編到不同班，在這部作品裡則是一同被編在5班）。

英文副標題方面，《小長門有希的消失》在單行本被譯作「vanishing」，電視動畫則是譯作「disappearance」，與《涼宮春日的消失》的英文標題相同，電視動畫的英譯使用意圖及原因則不明。

== 電視動畫 ==
2013年12月18日進行「小長門有希的消失」動畫化企劃期間宣佈<ref>{{Cite web |url=http://news.ameba.jp/20131218-163/ |title=『涼宮ハルヒ』スピンオフ『長門有希ちゃんの消失』アニメ化企画が進行中：日刊アメーバニュース|accessdate=2013-12-18|language=ja}}</ref>。是自從2010年電影《[[涼宮春日的消失|涼宮春日的消失]]》上映以來。在《[[Young_Ace|Young Ace]]》2014年10月號首度發表決定以電視動畫形式播放。

2014年12月18日在公式官網正式宣布製作人員與配音班底。除保留[[京都動畫|京都動畫]]《涼宮春日的憂鬱》（2006年與2009年版）、《涼宮春日的消失》、《小涼宮春日的憂鬱》的部分製作人員（企劃、音響監督）和配音班底外，動畫改由[[SATELIGHT|SATELIGHT]]製作，首播時間為2015年4月3日至7月17日，在[[東京都會電視台|TOKYO MX]]、[[AT-X|AT-X]]等[[全國獨立放送協議會|獨立電視網]]播出<ref>{{Cite web |author=平野綾オフィシャルインフォ |url=https://twitter.com/Hysteric_Barbie/status/545524577372078080 |title=【アニメ出演情報】 |publisher=Twitter |accessdate=2014-12-19|language=ja}}</ref><ref name="official">{{Cite web |url=http://www.yukichan-anime.com/onair/ |title=オンエアー情報 |accessdate=2015-02-24 | work=TVアニメ「長門有希ちゃんの消失」公式サイト|language=ja}}</ref>。集數方面，與季度的電視動畫不同，總共有16話。另外，第17話為OVA，收錄於Blu-ray Disc，隨著漫畫第9卷限量版發行時附贈。內容描寫北高文藝部的暑假<ref>{{cite news|url=http://natalie.mu/comic/news/142425 |title= 漫畫「小長門有希的消失」第9巻附贈BD的動畫內容為原創劇情，描寫北高文藝部的暑假。 |work= Comic Natalie |publisher= Natasha |accessdate=2015-07-20|language=ja}}</ref>。另外還收錄了第17話的下集預告。
