'''小樽港'''（{{jpn|j=小樽港|hg=おたるこう|rm=Otaru Ko}}）是一個位於日本[[北海道|北海道]][[小樽市|小樽市]]的港灣。港灣管理單位是小樽市。在日本港灣法上，小樽港被指定為重要港灣<ref name="otaru port">{{cite web|url=http://www.hkd.mlit.go.jp/zigyoka/z_kowan/bayport/profile/otaru.html|script-title=ja:小樽港|accessdate=2014-07-29|publisher={{lang|ja|国土交通省北海道開発局}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140629043120/http://www.hkd.mlit.go.jp/zigyoka/z_kowan/bayport/profile/otaru.html|archivedate=2014-06-29}}</ref>。在港則法上，小樽港則被指定為特定港。

== 概要 ==
[[File:Contrail_Sky_at_Otaru,_-20_March_2009_a.jpg|thumb]]
[[File:Port_of_Otaru_Aerial_photograph.1976.jpg|thumb]]
小樽港位於北海道石狩灣的西海岸，港口地形整體朝東開放。港區範圍北起茅柴岬，南至平磯岬。港口區域面積約571公頃，其中被防波堤保護的地區的面積約285公頃，僅次於[[室蘭港|室蘭港]]、[[苫小牧港|苫小牧港]]、[[釧路港|釧路港]]，是北海道面積第4大港口<ref name="minato">{{cite web|url=http://www.hkd.mlit.go.jp/zigyoka/z_kowan/BayportPage/kowan_kakawari/pdf/05.pdf|script-title=ja:Ⅰ．小樽港の現状|accessdate=2014-07-29|publisher={{lang|ja|小樽市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20150924031140/http://www.hkd.mlit.go.jp/zigyoka/z_kowan/BayportPage/kowan_kakawari/pdf/05.pdf|archivedate=2015-09-24}}</ref>。小樽港的北、西、南三面都被山地包圍，港內波浪起伏較為穩定，是一座天然良港。加上以北海道政治經濟的中心都市札幌市為中心的道央地區是小樽港的腹地，使得小樽港作為日本海沿岸的流通據點港發揮着重要的作用。

隨著石狩煤田的開發，小樽港最初是以一座煤炭輸出港開始其發展歷程的<ref name="kowan_kakawari">{{cite web|url=http://www.hkd.mlit.go.jp/zigyoka/z_kowan/BayportPage/kowan_kakawari/pdf/05.pdf|script-title=ja:小樽港における港湾利用の歴史的変遷|accessdate=2014-07-29|publisher={{lang|ja|国土交通省}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20150924031140/http://www.hkd.mlit.go.jp/zigyoka/z_kowan/BayportPage/kowan_kakawari/pdf/05.pdf|archivedate=2015-09-24}}</ref>。自1899年（明治32年）開港以來，小樽港就作為北海道開發的據點得到發展。1951年，小樽港被指定為重要港口。然而，在戰後隨著能源革命的進展使得主要能源由煤炭變為石油。加上日本對[[庫頁島|庫頁島]]、[[俄羅斯|俄羅斯]]貿易的減少；日本海航路的整理縮小；太平洋沿岸苫小牧港的發展；加上附近[[石狩灣新港|石狩灣新港]]新港的開發，使得小樽港作為道央地區港口的地位大幅下降。

小樽港和[[新潟港|新潟港]]、[[敦賀港|敦賀港]]（[[福井縣|福井縣]]）、[[舞鶴港|舞鶴港]]（[[京都府|京都府]]）之間曾開設有[[新日本海渡輪|新日本海渡輪]]的定期渡輪航路。但在1999年，新潟航路的部分船舶轉往苫小牧港。2002年，敦賀航路的部分船舶也轉往苫小牧港。剩下的航路也陸續削減班次，在最近10年內貨物量已經減少幾乎一半。

2010年，小樽港雖然未能被選入[[重点港灣|重点港灣]]（被選入的是附近的石狩灣新港），但在2011年被選入外航郵輪的日本海沿岸據點港。小樽港現在正在和[[伏木富山港|伏木富山港]]及舞鶴港合作，推動環日本海郵輪實現品牌化<ref>{{cite web |url=http://www.mlit.go.jp/common/000164357.pdf |script-title=ja:日本海側拠点港形成に向けた計画書|accessdate=2014-07-29|publisher={{lang|ja|国土交通省}} |language=ja}}</ref><ref>{{cite web |url=http://www.cruise-mag.com/news.php?obj=20120413_03 |script-title=ja:環日本海クルーズ推進協議会が設立|accessdate=2014-07-29|publisher={{lang|ja|海事プレス社}} |language=ja}}</ref>（2013年時，[[秋田港|秋田]]、[[船川港|船川]]、[[能代港|能代港]]和[[境港|境港]]也參與其中<ref>{{cite web |url=http://www.cruise-mag.com/news.php?obj=20130529_02  |script-title=ja:「環日本海クルーズ推進協議会」総会・港湾説明会開催|accessdate=2014-07-29|publisher={{lang|ja|海事プレス社}} |language=ja}}</ref>）。自2008年開始，小樽港連續5年成為北海道的港口中郵輪停留次數最多的港口。2012年達到20次，創下新高<ref>{{cite web |url=http://www.city.otaru.lg.jp/simin/sumai/minato/kako_jisseki/kyakusen_kikou2012.html  |script-title=ja:2012年　小樽港クルーズ客船寄港実績|accessdate=2014-07-29|publisher={{lang|ja|小樽市}} |language=ja}}</ref>。小樽市為了對應郵輪入港次數增加和郵輪客船大型化的趨勢，正在建設可容納最大15万噸級客船入港的客船用岸壁<ref>{{cite web |url=http://www.cruise-mag.com/news.php?obj=20111116_02 |script-title=ja:小樽港、客船岸壁整備を発表|accessdate=2014-07-29|publisher={{lang|ja|海事プレス社}} |language=ja}}</ref><ref>{{cite web |url=http://www.cruise-mag.com/news.php?obj=20121114_01 |script-title=ja:小樽港、将来的にフリーダムクラス対応可能に|accessdate=2014-07-29|publisher={{lang|ja|海事プレス社}} |language=ja}}</ref><ref>{{cite web |url=http://www.cruise-mag.com/news.php?obj=20130830_02 |script-title=ja:小樽港、ダイヤモンド寄港に備え13万トンまで接岸可能に|accessdate=2014-07-29|publisher={{lang|ja|海事プレス社}} |language=ja}}</ref>。迄今為止，在小樽港停留的最大的客船是[[太陽公主號|太陽公主號]]，噸位為77,441噸<ref>{{cite web |url=http://otaru-journal.com/2013/06/0619-2.php |script-title=ja:過去最大級の豪華客船"サン・プリンセス"　小樽入港|accessdate=2014-07-29|publisher={{lang|ja|小樽ジャーナル}} |language=ja}}</ref>。

== 歷史 ==
[[File:Otaru-overview.JPG|thumb]]富岡的旭展望台看到的小樽港]]
小樽港的歷史開始於1872年，北海道開拓使在當時的色內村修建了石造的碼頭，並在1873年完成<ref name="otaru port"></ref>。1889年，小樽港被指定為特別輸出港。1895年，學者[[廣井勇|廣井勇]]被任命人小樽築港計劃的主導者。廣井勇是當時日本土木工程界的權威人士，對小樽築港貢獻卓著<ref>{{cite web |url=http://www1.hus.ac.jp/~hit_doboku/doboku/1_hiroi.html |script-title=ja:小樽港北防波堤と広井勇博士|accessdate=2014-07-29|publisher={{lang|ja|小樽ジャーナル}} |language=ja}}</ref>。1899年，小樽港被指定為外國貿易港，這也宣告小樽正式開港<ref name="history">{{cite web |url=http://www.otarucci.jp/kankou/rekisi/rekisi.html |script-title=ja:小樽の歴史|accessdate=2014-07-29|publisher={{lang|ja|小樽商工会議所}} |language=ja}}</ref>。1908年，小樽港修築了北防波堤，這也是日本最早的真正的[[混凝土|混凝土]]製[[防波堤|防波堤]]，並宣告小樽港第一期築港工程完成<ref name="otaru port"></ref>。之後小樽港又在1921年修築了完成南防波堤，並在1926年完成了第二期防波堤工程<ref name="history"></ref>。1931年，小樽港舉辦了舉辦小樽海港博覧会，並在六年後完成了第1號碼頭且舉辦了開道70年[[北海道大博覧會|北海道大博覧會]]<ref name="history"></ref>。小樽港進入了戰前的全盛時期。

進入戰後之後，小樽港在1949年舉辦了開港50周年港祭，並在次年完成了完成第2號碼頭，港灣規模進一步擴大。1951年，小樽港被指定為重要港灣。1954年，小樽港完成了第3號碼頭一期工程<ref name="history"></ref>。1970年8月，小樽港開通了至[[舞鶴港|舞鶴]]、[[敦賀港|敦賀]]之間的長距離渡輪，這也是日本首個長距離渡輪航線<ref name="kowan_kakawari"></ref>。在四年之後，又開通了小樽和[[新潟港|新潟]]之間的長距離渡輪。在戰後時期，除了貨運之外，客運和觀光也成為小樽港的主要機能。1990年，小樽港遊艇碼頭宣布開業，加上港口地區多個商業和觀光設施開發計劃的實施，小樽港正試圖開拓新的港口功能。在維持物流機能的同時，也希望能發展觀光和休閒機能<ref>{{cite web |url=http://www.jice.or.jp/jishu/kokudo/200806160/product/010.html |script-title=ja:小樽港|accessdate=2014-07-29|publisher={{lang|ja|一般財団法人　国土技術研究センター}} |language=ja}}</ref>。

== 港口現狀 ==
小樽港現在有兩條日本國內的定期客運渡輪航線，分別是小樽和舞鶴之間的航線（每日運航）<ref name="mina_s3">{{cite web|url=http://www.city.otaru.lg.jp/jigyo/otaruport/mina_s3.html|script-title=ja:小樽港要覧　小樽港の国内・国外航路|accessdate=2014-07-29|publisher={{lang|ja|小樽市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131012150620/http://www.city.otaru.lg.jp/jigyo/otaruport/mina_s3.html|archivedate=2013-10-12}}</ref>，以及小樽和新潟之間的航線（除禮拜一外每日運航）<ref>{{cite web|url=http://www.snf.jp/searoute/niigata_otaru.html|script-title=ja:新潟～小樽|accessdate=2014-07-29|publisher={{lang|ja|新日本海フェリー}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140804035607/http://www.snf.jp/searoute/niigata_otaru.html|archivedate=2014-08-04}}</ref>。此外小樽和敦賀之間還有只在特定日運行的航線<ref>{{cite web|url=http://www.snf.jp/searoute/maiduru_otaru.html|script-title=ja:舞鶴～小樽|accessdate=2014-07-29|publisher={{lang|ja|新日本海フェリー}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140702054452/http://www.snf.jp/searoute/maiduru_otaru.html|archivedate=2014-07-02}}</ref>。在貨運方面，小樽港自2002年9約開始運行小樽與中國之間的定期集裝箱航線<ref name="mina_s3"></ref>。

而在港口設施方面，小樽港現在有六個主要碼頭，分別是港町碼頭、第二號碼頭、第三號碼頭、中央碼頭、勝納碼頭、色內碼頭<ref name="mina_s2">{{cite web |url=http://www.city.otaru.lg.jp/jigyo/otaruport/mina_s2.html |script-title=ja:小樽港要覧　小樽港湾施設|accessdate=2014-07-29|publisher={{lang|ja|小樽市}} |language=ja}}</ref>。2012年，小樽港的貨運進出口量達1124.7萬噸<ref name="otaru port"></ref>。小樽港內的貿易以日本國內貿易為主<ref>{{cite web |url=http://www.city.otaru.lg.jp/jigyo/otaruport/mina_s4.data/2013_otarukotoukeinenpou.pdf |script-title=ja:平成２４年（２０１２年）「小樽港統計年報」|accessdate=2014-07-29|publisher={{lang|ja|小樽市}} |language=ja}}</ref>。除了供大型貨船和客船停留的碼頭之外，小樽港內還有供遊艇等小型船舶停靠的碼頭。小樽港的遊艇碼頭位於港口區域的南部，可停靠300艘遊艇，是北海道第二大遊艇碼頭<ref name="minato"></ref>。小樽港也有漁港的機能。小樽市內的漁港中，高島漁港區即被劃入小樽港的範圍之內<ref>{{cite web|url=http://www.city.otaru.lg.jp/jigyo/sangyo/otarusuisan/gyoko.html|script-title=ja:小樽市の漁港|accessdate=2014-07-29|publisher={{lang|ja|小樽市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20150326182916/http://www.city.otaru.lg.jp/jigyo/sangyo/otarusuisan/gyoko.html|archivedate=2015-03-26}}</ref>。
