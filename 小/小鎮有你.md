《'''小鎮有你'''》（{{lang|ja|君のいる町}}、''A town where you live''）是日本[[漫画家|漫画家]][[瀨尾公治|瀨尾公治]]的作品。在[[講談社|講談社]]的漫画雑誌《[[週刊少年Magazine|週刊少年Magazine]]》于2008年26号至2014年11號連載。<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2008-05-13/suzuka-seo-samurai-deeper-kyo-kamijyo-to-launch-manga|title=Suzuka's Seo, Samurai Deeper Kyo's Kamijyō to Launch Manga|date=2008年5月13日|publisher=[[動畫新聞網|動畫新聞網]]|accessdate=2014年11月2日|language=en}}</ref><ref>{{cite web|url=http://www.animenewsnetwork.com/news/2014-01-17/a-town-where-you-live/kimi-no-iru-machi-manga-to-end|title=''A Town Where You Live'' Manga to End|date=2014年1月17日|publisher=[[動畫新聞網|動畫新聞網]]|accessdate=2014年11月2日|language=en}}</ref>
与前作曾动画化的作品《[[凉风|凉风]]》一样，是一部以[[日本|日本]]高中作为舞台的恋爱故事。前作中《[[凉风|凉风]]》的男主角是从广岛到高中入学来东京，而本作是女主角从[[东京|东京]]的高中入学来[[广岛|广岛]]的。目前台灣部份於週刊《[[新少年快報|新少年快報]]》上連載。<ref>{{cite web|archiveurl=https://web.archive.org/web/20140412040739/http://www.tongli.com.tw/NewBoy_top.aspx |url=http://www.tongli.com.tw/NewBoy_top.aspx |archivedate=2014年4月12日 |title=新少年快報 |publisher=[[東立出版社|東立出版社]] |accessdate=2014年12月15日 |language=Ja |deadurl=yes }}</ref>作品被分為三部份：第1話至109話是第一部（高中），第110話至247話是第二部（大學），第248話至261話是第三部（社會），另外在2011年8月中宣佈決定動畫化。

==故事介绍==
這個春天就要進入高中的廣島少年桐島青大家裡，出現一位來自東京且從沒見過的少女枝葉柚希寄住在他的家裡，並且跟青大上同一所高中。然而事情不如青大所想那麼順利，除了要照顧老是搞砸事情的柚希外，還常常無意中被柚希故意肢體碰觸告白；從那天開始青大每天都一直被自由奔放且笨手笨腳的柚希的行動所困惑着。然而原本從國中開始直到高中就對同班女同学神咲七海有着好感的青大，卻在柚希即將離開廣島回東京前夕開始漸漸地開始喜歡柚希了。知道了青大想法的柚希在離開廣島後會怎麼樣呢？而青大跟神咲七海的關係又會產生什麼變化呢？

; {{nihongo|古葉 遙奈|古葉 遥奈（こば はるな）|聲：[[真野惠里菜|真野惠里菜]]}}
: 神咲七海的朋友，個性輕浮。
; {{nihongo|淺倉 清美（'''由良 清美'''）|浅倉 清美（あさくら きよみ）→由良 清美（ゆら きよみ）|聲：[[遠藤綾|遠藤綾]]}}
: 風間恭輔的青梅竹馬，巨乳，喜歡風間。目前與尊交往並奉子成婚，婚後產下兒子，名為由良 青河。
; {{nihongo|宮永知沙|宮永 知沙（みやなが ちさ）}}
: 柚希在補習社輔導的學生，對青大一見鍾情，現在搬入青大的公寓並成為鄰居。

====大學生====
; {{nihongo|天谷 栞|天谷 栞（あまや しおり）}}
: 青大的大學同學，漫畫家，曾經要脅青大帶她到愛情旅館實地取材結果遭到神咲七海撞見，從此變成青大最頭痛的人物。

; {{nihongo|市原 康生|市原 康生（いちはら こうせい）|聲：[[杉山紀彰|杉山紀彰]]}}
: 大學網球社的社員，個性外向。

; {{nihongo|保科 美友|保科 美友（ほしな みゆ）}}
: 青大的鄰居，巨乳，常常誤會桐島青大的話而做出黃色的動作或開黃腔，飲醉酒後會發酒瘋，通常是除衫，現在去英國留學甚至有未婚夫。
  
; {{nihongo|水本前輩|水本先輩（みずもと）}}
: 水本真菜的哥哥，妹控，家境不錯，父親是一流的廚師。料理天分很高，但不想當廚師。
; {{nihongo|水本 真菜|水本 真菜（みずもと まな）}}
: 水本前輩的妹妹，在天城紫步店裡工作的工讀生。

; {{nihongo|高波奈央|高波 奈央（たかなみ なお）}}
: 是個花心的女同性戀者，起初讓青大誤會高波喜歡藤河但事實上是喜歡神咲七海，而後又試圖接近枝葉祐希，最後纏住枝葉懍讓懍感到困擾。

; {{nihongo|藤河昭人|藤河 昭人（ふじかわ あきひと）}}
: 喜歡神咲七海，208話正式與神咲交往。

; 長門未沙
: 青大在職業介紹所認識的女生，對青大的工作的意見有偏見，242話向青大說出她做了記者的職位。

: 由於動畫改編關係，除了市原康生外，天谷栞及保科美友等大學生人物沒有在動畫中出現。
