'''小金井公園'''（{{jpn|j=小金井公園|hg=こがねいこうえん}}）是橫跨[[東京都|東京都]][[小金井市|小金井市]]、[[小平市|小平市]]、[[西東京市|西東京市]]、[[武藏野市|武藏野市]]的[[東京都立公園|東京都立]]都市公園（廣域公園）。正式名稱為東京都立小金井公園（{{jpn|j=東京都立小金井公園|hg=とうきょうとりつこがねいこうえん}}）。 

== 概要 ==
[[File:Koganei_Country_Club_and_Koganei_Park_Aerial_photograph.1989.jpg|thumb]]
面積約80公頃，為[[日比谷公園|日比谷公園]]的4.8倍、[[上野恩賜公園|上野公園]]的1.4倍，是都立公園中最大規模的公園。另外，東京都最大的[[國營昭和紀念公園|國營昭和紀念公園]]面積是小金井公園的2倍。

公園西側有南北向的{{link-ja|東京都道15號府中清瀨線|東京都道15号府中清瀬線|小金井街道}}（{{link-ja|東京都道15號府中清瀨線|東京都道15号府中清瀬線}}）、南側有東西向的{{link-ja|五日市街道|五日市街道}}（{{link-ja|東京都道7號杉並秋留野線|東京都道7号杉並あきる野線}}）通過。五日市街道旁有[[玉川上水|玉川上水]]流經。公園北側有高爾夫球場{{link-ja|小金井鄉村俱樂部|小金井カントリー倶楽部}}。

以為了{{link-ja|紀元二千六百年紀念行事|紀元二千六百年記念行事|紀元二千六百年紀念事業}}規劃的{{link-ja|東京綠地計畫|東京緑地計画}}「小金井大綠地」為原形，1954年（昭和29年）1月開園。基本主題為「花與綠的寬敞空間」。園內有雜木林與草地，種植的1,800棵櫻花樹更讓本園入選[[日本櫻名所100選|日本櫻名所100選]]。正月時期的公園是放風箏的熱門地點，而園內的「休憩廣場」（いこいの広場）更是定期舉行[[跳蚤市場|跳蚤市場]]。

== 沿革 ==
*1940年（[[昭和|昭和]]15年） - 為了{{link-ja|紀元二千六百年紀念行事|紀元二千六百年記念行事|紀元二千六百年紀念事業}}規劃{{link-ja|東京綠地計畫|東京緑地計画|小金井大綠地}}，至翌年1941年12月為止取得90公頃用地。
*1941年（昭和16年） - 光華殿（現江戶東京建築園遊客中心）移建。借給[[文部省|文部省]]教學局設置國民練成所。
*1946年（昭和21年） -10月，文部省借用地移交給[[宮內廳|宮內廳]]，設置東宮臨時寓所、[[學習院|學習院]]中等科。
*1949年（昭和24年）3月 - 因[[土地改革|農地解放]]，失去用地138,085坪（約45.6公頃，佔整體4成）。
*1951年（昭和26年）3月 - 宮內廳返還用地7萬坪（約23公頃）。
*1954年（昭和29年）1月14日 - 開園（東京都告示第26號，面積8.6公頃），{{link-ja|武藏野鄉土館|江戸東京たてもの園}}開館。
*1957年（昭和32年）12月21日 - 都市計畫決定（建設省告示第1689號，148.1公頃）。
*1975年（昭和50年）4月 - 停放保存車輛（C57形蒸氣機關車、スハフ32形客車）。
*1986年（昭和61年）6月 - 追加開園（東京都告示第642號，69.4公頃），設置收費網球場、棒球場。
*1989年（[[平成|平成]]元年）6月 -  設置小金井市綜合體育館。
*1991年（平成3年）12月 - 武藏野鄉土館閉館。
*1993年（平成5年）3月 - 開設[[江戶東京建築園|江戶東京建築園]]。
*2004年（平成16年）4月22日 - 都市計畫決定（東京都告示第722號，146.9公頃）。

== 小金井的櫻花 ==
過去小金井的玉川上水兩岸櫻花「小金井櫻」十分知名，每年春天吸引許多賞櫻民眾，[[明治天皇|明治天皇]]等[[皇族|皇族]]也曾來此賞花。1889年（明治22年），{{link-ja|甲武鐵道|甲武鉄道}}（之後的[[中央本線|中央本線]]）新宿－立川間通車，但小金井未設站。後來因應龐大的「小金井櫻」賞櫻人潮，1924年（大正13年）在花季增設臨時車站（之後為武藏小金井站）。玉川上水兩岸的櫻花樹是在1737年（[[元文_(日本)|元文]]2年）所種植，1924年（[[大正|大正]]13年）依《史蹟名勝天然紀念物法》指定為名勝。但在1950年代{{link-ja|五日市街道|五日市街道}}拓寬等因素下交通量大增，櫻花樹也逐漸枯萎。在小金井公園開園（1954年）後，逐漸取代過去的玉川上水，成為新的賞櫻名勝。
