== 主要人物 ==
; {{nihongo|[[百鬼夜行系列#關口巽|關口 巽]]|{{ruby|関口|せきぐち}} {{ruby|巽|たつみ}}|Sekiguchi Tatsumi}}
: [[小說家|小說家]]。受'''妹尾'''和'''光保'''所托前往韮山尋找消失了的戶人村。然而在戶人村廢址的佐伯家裡發現了「某樣東西」後失去意識。
: 之前曾和'''加藤麻美子'''和'''宮村香奈男'''見面，并見證京極堂為麻美子「驅除附身之物」的過程。
: 詳情請參閱「[[百鬼夜行系列#關口巽|百鬼夜行系列#主要登場人物]]」。

; {{nihongo|[[百鬼夜行系列#妹尾友典|妹尾 友典]]|{{ruby|妹尾|せのお}} {{ruby|友典|とものり}}|Senoo Tomonori}}
: 赤井書房職員，八卦雜誌《實錄犯罪》的總編。委託關口去尋找戶人村。
: 詳情請參閱「[[百鬼夜行系列#妹尾友典|百鬼夜行系列#主要人物的交友關係]]」。

; {{nihongo|[[百鬼夜行系列#一柳朱美|一柳 朱美]]|{{ruby|一柳|いちやなぎ}} {{ruby|朱美|あけみ}}|Ichiyanagi Akemi}}
: 在「[[狂骨之夢|狂骨之夢]]」後搬到[[東京|東京]]，後因不適應大城市的生活而搬到[[靜岡|靜岡]]。丈夫長期在外工作，因此大部分時間獨居。
: 救了企圖自殺的'''村上兵吉'''，並因此被捲入成仙道的是非之中。
: 詳情請參閱「[[百鬼夜行系列#一柳朱美|百鬼夜行系列#主要人物的交友關係]]」。

; {{nihongo|[[百鬼夜行系列#中禪寺秋彥|中禪寺 秋彥]]|{{ruby|中禅寺|ちゅうぜんじ}} {{ruby|秋彦|あきひこ}}|Chuuzenji Akihiko}}
: 在[[中野區|中野]]經營舊書店「京極堂」，店名也成了他的外號。副業是[[陰陽師|陰陽師]]。
: 在宮村香奈男的引見下，為加藤麻美子驅除她內心的鬱結和陰暗。

: 因在雜誌上刊登了對韓流氣道會的報導，因而得罪對方，被對方的信徒襲擊。期間與'''華仙姑處女'''相遇。
: 詳情請參閱「[[百鬼夜行系列#中禪寺敦子|百鬼夜行系列#主要人物的家人]]」。

: 在敦子和華仙姑處女陷入困境時出手相救。
: 詳情請參閱「[[百鬼夜行系列#榎木津禮二郎|百鬼夜行系列#主要登場人物]]」。

: [[東京|東京]][[警視廳|警視廳]]搜查一課警察。為解開'''三木春子'''被偷窺的事件而奔走，並為此拜訪京極堂。
: 詳情請參閱「[[百鬼夜行系列#木場修太郎|百鬼夜行系列#主要登場人物]]」。

== 各話簡介&登場人物 ==
本作的章節名稱均以《[[畫圖百鬼夜行|畫圖百鬼夜行]]》中的妖怪名字為題。

=== 野篦坊 ===
'''關口巽'''在'''妹尾友典'''的介紹下和前警察官'''光保公平'''見面。光保向關口拜託調查據說過去曾存在於韮山一帶的「戶人村」。根據光保的說法，他過去曾在那裡駐守，戰後回到那裡的時候，那條村子的存在就被抹消掉了。關口和當地的警官'''淵脇'''和途中遇到的流浪作家'''堂島靜軒'''一同造訪韮山。在那裡他們目擊到的真相、以及不可能存在於世的東西究竟是……

: [[靜岡縣|靜岡縣]]警所屬的前警察官。自稱是膽小鬼。周遭的人都說他長得像野篦坊<!--，連他自己也固執於己見-->。如今經營室內裝飾業。

*;{{nihongo|津村 辰藏|{{ruby|津村|つむら}} {{ruby|辰藏|たつぞう}}|Tsumura Tatsuzou}}
: 巡迴磨刀師。發現并通報戶人村居民消失的事情，但之後連同[[憲兵|憲兵]]一起消失不見，一年後回來時變得如同廢人，之後一個月後自殺身亡。

*;{{nihongo|佐伯 癸之介|{{ruby|佐伯|さえき}} {{ruby|癸之介|きのすけ}}|Saeki Kinosuke}}

*;{{nihongo|佐伯 亥之介|{{ruby|佐伯|さえき}} {{ruby|亥之介|いのすけ}}|Saeki Inosuke}}

*;{{nihongo|佐伯 布由|{{ruby|佐伯|さえき}} {{ruby|布由|ふゆ}}|Saeki Fuyu}}

*;{{nihongo|佐伯 乙松|{{ruby|佐伯|さえき}} {{ruby|乙松|おとまつ}}|Saeki Otomatsu}}

*;{{nihongo|佐伯 甚八|{{ruby|佐伯|さえき}} {{ruby|甚八|じんぱち}}|Saeki Jinpachi}}
: 玄藏的兒子。雖在本家居住，但被本家當成是下人般使喚。

: 住在過去戶人村建村的地方的老人。

=== 嗚汪 ===
'''一柳朱美'''離開[[神奈川|神奈川]]移居至[[靜岡|靜岡]]。某天碰巧遇上一個名叫'''村上兵吉'''的企圖自殺的男人。朱美將他救起並給與照顧。村上自過去某個事件開始就懼怕「賣藥郎」。同一時間，鄰居'''松島奈津'''告訴她新興宗教「成仙道」每日都會來勸人入會。在聽到騷動而外出的朱美在和松島談話時，村上再次企圖自殺……

*;{{nihongo|村上 兵吉|{{ruby|村上|むらかみ}} {{ruby|兵吉|へいきち}}|Murakami Heikichi}}
: 企圖上吊自殺的時候被朱美所救。曾經營一間螺絲工廠但破產。「指引康莊大道修身會」的成員。

*;{{nihongo|松島 奈津|{{ruby|松嶋|まつしま}} ナツ|Matsushima Natsu}}
: 朱美的鄰居。外表長得像小女孩但對帶孩子非常熱衷。對「成仙道」感到反感。

*;{{nihongo|尾國 誠一|{{ruby|尾国|おぐに}} {{ruby|誠一|せいいち}}|Oguni Seiichi}}
: 配置藥的老手商人，朱美兩夫婦的恩人。
: 和加藤麻美子相識。

*;{{nihongo|刑部 昭二|{{ruby|刑部|おさかべ}} {{ruby|昭二|しょうじ}}|Osakabe Shouji}}
: 成仙道的人，糾纏著奈津說服其入會。

=== 咻嘶卑 ===
'''關口巽'''和'''京極堂'''的同行前輩宮村香奈男相識。宮村因為女性友人加藤麻美子的祖父的一些煩惱問題而前來京極堂商量。麻美子的祖父最近受到奇怪的新興宗教團體的強烈影響，將所有財產投入其中，因此她勸祖父退出那個團體。然而，麻美子也迷上了名叫「華仙姑處女」的神秘占卜師，而捐献了大額金錢。

*;{{nihongo|宮村 香奈男|{{ruby|宮村|みやむら}} {{ruby|香奈男|かなお}}|Miyamura Kanao}}
: 在[[川崎|川崎]]經營舊書店「薫紫亭」，和京極堂是同行，但在該行當是連京極堂也望塵莫及的達人。
: 周圍的人都叫他「老師」。

*;{{nihongo|加藤 麻美子|{{ruby|加藤|かとう}} {{ruby|麻美子|まみこ}}|Katou Mamiko}}
: 宮村的朋友，直到去年都還是「小說創造」的編輯。因祖父的樣子變得怪異，因此經宮村引薦和京極堂商談。
: 自稱在幼年父親死亡前和最近女兒意外死亡前，都見過「咻嘶卑」。

*;{{nihongo|加藤 只二郎|{{ruby|加藤|かとう}} {{ruby|只二郎|ただじろう}}|Katou Tadajirou}}
: 麻美子的祖父。加入了「指引康莊大道修身會」之後，樣子就變得奇怪了。和'''磐田純陽'''是舊識。

*;{{nihongo|磐田 純陽|{{ruby|磐田|いわた}} {{ruby|純陽|じゅんよう}}|Iwata Junyou}}
: 「指引康莊大道修身會」的會長。

=== 哇伊啦 ===
'''中禪寺敦子'''前往一間名叫韓流氣道會的[[道場|道場]]取材，然後刊登了相關的報道。然而她本人出於好意而編寫的報道，卻引來韓流氣道會的強烈反對，更遭到其門下弟子們的尾隨跟蹤。這時，敦子認識了自稱'''華仙姑處女'''的女人。她自稱也被韓流氣道會的人跟蹤。兩人拚命逃至人跡罕至的小巷，被逼至窮途末路之時，'''榎木津禮二郎'''出現在她們面前。

*;{{nihongo|華仙姑處女|{{ruby|華仙姑処女|かせんこおとめ}}|Kasenko Otome}}
: 世間傳聞百發百中的神秘女占卜師。本人卻不打算靠[[占卜|占卜]]為生。

*;{{nihongo|宮田 耀一|{{ruby|宮田|みやた}} {{ruby|耀一|よういち}}|Miyata Youichi}}
: 在三軒茶屋裡的名為「條山房」的中藥局工作，戴著圓眼鏡的瘦小的男人。照顧被施暴的敦子。

*;{{nihongo|岩井 崇|{{ruby|岩井|いわい}} {{ruby|崇|たかし}}|Iwai Takashi}}
: 韓流氣道會的代理師範，接待前來取材的敦子。為報復報道一事襲擊敦子。其實力如一對一的話連警官也不是對手。

=== 休喀拉 ===
'''木場修太郎'''在經常光顧的酒館「貓目洞」的老闆娘'''竹宮潤子'''的介紹下認識了名叫三木春子的女性。春子表示一直被一個名叫'''工藤信夫'''的跟蹤狂跟蹤而備受困擾。在交談中木場詢問詳細情況，得知工藤每周都會寄信給春子，信中詳細記錄了春子一周內的各項活動，更做出了調查春子的房間甚至偷窺等不可能的行為。

*; {{nihongo|[[百鬼夜行系列#竹宮潤子|竹宮 潤子]]|{{ruby|竹宮|たけみや}} {{ruby|潤子|じゅんこ}}|Takemiya Junko}}
: 酒館「貓目洞」的老闆娘，通稱「阿潤」，介紹有着煩惱的三木春子給木場認識。
: 詳情請參閱「[[百鬼夜行系列#竹宮潤子|百鬼夜行系列#主要人物的交友關係]]」。

*;{{nihongo|張果老|{{ruby|張果老|ちょうかろう}}|Choukarou}}
: 條山房的中藥調劑師。年事已高卻是武藝高手，能同時將韓流氣道會的數個門徒輕易收拾，連代理師範岩井也被輕易地制服。人稱「通玄」。

*;<span id="藍童子">{{nihongo|藍童子|{{ruby|藍童子|らんどうじ}}|Randoushi}}</span>
: 使用照魔之術的神秘少年。利用這種力量去協助警察調查案件。

*;{{nihongo|三木 春子|{{ruby|三木|みつき}} {{ruby|春子|はるこ}}|Mitsuki Haruko}}
: 阿潤的友人。因遭遇跟蹤狂跟蹤而找木場求助。記憶力比常人好一倍。

*;{{nihongo|工藤 信夫|{{ruby|工藤|くどう}} {{ruby|信夫|のぶお}}|Kudou Nobuo}}
: 跟蹤春子的男人。在木場找他的那天因盜竊罪被捕。

*;<span id="岩川真司">{{nihongo|岩川 真司|{{ruby|岩川|いわかわ}} {{ruby|真司|しんじ}}|Iwakawa Shinji}}</span>
: [[警視廳|警視廳]][[目黑|目黑]]署刑事部搜查二課所屬的警察，階級是[[警部補|警部補]]。木場以前的同僚。

=== 歐托羅悉 ===
'''織作茜'''和自稱是織作家遠房親戚的男人'''羽田隆三'''就織作家的土地房產問題展開商談。羽田開出極佳的條件買下茜的土地房產，條件是讓茜成為自己的部下。期間，羽田因被隨從要求買下靜岡縣韮山的偏僻土地而產生懷疑，打算請'''榎木津禮二郎'''進行調查但被爽約，改而派茜和秘書'''津村信吾'''前往韮山。

*; {{nihongo|[[絡新婦之理#織作茜|織作 茜]]|{{ruby|織作|おりさく}} {{ruby|茜|あかね}}|Orisaku Akane}}
: 織作家次女。應允羽田隆三買下織作家相關房產的要求，並與津村一同前往韮山。

*;{{nihongo|羽田 隆三|{{ruby|羽田|はた}} {{ruby|隆三|りゅうぞう}}|Hata Ryuuzou}}
: 羽田制鐵董事顧問。織作茜外祖父的弟弟。認為[[徐福|徐福]]是羽田家的先祖，也是「徐福研究會」的發起人。

*;{{nihongo|羽田 桝太郎|{{ruby|羽田|はた}} {{ruby|桝太郎|ますたろう}}|Hata Masutarou}}
: 隆三的父親，羽田制鐵創立人。

*;{{nihongo|津村 信吾|{{ruby|津村|つむら}} {{ruby|信吾|しんご}}|Tsumura Shingo}}

== 用語 ==
* '''塗佛'''
: 依照'''佐脇嵩之'''<ref>佐脇嵩之（{{lang|ja|さわき すうし}}、1707年-1772年），[[江戶時代|江戶時代]]的畫師。其創作的百怪圖卷由一系列妖怪畫像組成。他用豐富的色彩描畫出最臭名昭著的妖怪，影響了後來以妖怪作為創作主題的藝術家們。</ref>的《百怪圖卷》的說法，塗佛好像全身漆黑的和尚形象，全身毫無毛髮、眼珠突出而下垂，體態臃腫肥胖，肚子彷彿漲滿水一樣，臀部長了條看來像魚尾巴的東西，就好像是突目金魚與溺死的屍體的綜合體。但也有說法認為，「塗」是指髒污了，「佛」是指死者，塗佛其實是不淨的屍體的別稱，從這個說法引伸出去，有論者認為這種妖怪應該是「非自然死亡」的死者的妖怪化結果。<ref name="content">[http://www.books.com.tw/products/0010457529 博客來-塗佛之宴：備宴(上) - 內容簡介]</ref>

* '''野篦坊'''（{{lang|ja|ぬっぺらぼう}}）
: 《一宵話》記載，[[慶長|慶長]]14年（1609）的某個早晨，[[德川家康|德川家康]]住在[[駿河國|駿河國]]的宅第中庭忽然出現了一個小人，看起來像團肉塊上長了四肢，有手但沒有手指，靜悄悄的站著遙指[[京都|京都]]的方向。眾人都很害怕，於是把牠趕到沒有人的地方去。後來有識之士才說那其實是白澤圖<ref>中國的怪物圖鑑。相传[[白澤|白澤]]是中國上古傳說中的一種奇獸，能說人語，[[黃帝|黃帝]]問鬼魅和精怪之事，白澤一一對答，黃帝叫人按白澤所言，將那些鬼魅和精怪的形象畫下來，這就是現在已失傳的神秘古籍《白澤圖》的由來。</ref>上名為「封」的妖怪，割他的肉來吃可以相當勇敢武斷。不過後來的百鬼夜行圖幾乎都把這種妖怪放在荒廢的寺廟中，還有描述牠會發出如腐肉般的味道。<ref name="content" /><ref>{{lang|ja|『一宵話』巻之二より、「肉人」}}</ref><ref>[http://blogblogfun2011.blogspot.hk/2011/06/blog-post_15.html 聚˙Blog: 妖怪大百科（肉人）]</ref><ref>[http://www.youkaiwiki.com/entry/2013/01/20/%E3%81%AC%E3%81%A3%E3%81%BA%E3%81%B5%E3%81%BB%E3%81%B5 {{lang|ja|ぬっぺふほふ - 妖怪うぃき的妖怪図鑑}}] {{webarchive|url=https://web.archive.org/web/20141106210805/http://www.youkaiwiki.com/entry/2013/01/20/%E3%81%AC%E3%81%A3%E3%81%BA%E3%81%B5%E3%81%BB%E3%81%B5 |date=2014-11-06 }}</ref>

* '''嗚汪'''（{{lang|ja|うわん}}）
: 一種用鐵漿染黑牙齒，狀似男性的妖怪，通常會在半空中出現並高舉著雙手揮舞，同時忽然發出「嗚汪」的聲音來嚇唬人，如果被嚇到而沒有立即回應「嗚汪」的話，生氣就會被攫走而死亡（另有一說是會被他拖到墳場裡的棺材中活埋）。有趣的是，百鬼夜行圖裡的「嗚汪」一隻手都只有三根手指頭，有個說法是為了表現他們其實是鬼而不是人的意思。<ref name="content" /><ref>[http://www.youkaiwiki.com/entry/2013/02/02/%E3%81%86%E3%82%8F%E3%82%93 {{lang|ja|うわん - 妖怪うぃき的妖怪図鑑}}] {{webarchive|url=https://web.archive.org/web/20140713061041/http://www.youkaiwiki.com/entry/2013/02/02/%E3%81%86%E3%82%8F%E3%82%93 |date=2014-07-13 }}</ref>

* '''咻嘶卑'''（{{lang|ja|ひょうすべ}}）
: 一般都說是[[河童|河童]]的同伴，造型也與河童類似，都是小童狀，只是並不像河童頭頂有個盛水的盤子，倒是光頭而全身是毛。像一般的河童一樣，「咻嘶卑」也被視為水神、河神的一種，但除此之外也被視為是兵主部神，也就是戰神的意思，整個西日本將牠當作水神跟戰神合併祭祀的神社相當的多。<ref name="content" /><ref>[http://www.youkaiwiki.com/entry/2013/01/29/%E3%81%B2%E3%82%87%E3%81%86%E3%81%99%E3%81%B9 {{lang|ja|ひょうすべ - 妖怪うぃき的妖怪図鑑}}] {{webarchive|url=https://web.archive.org/web/20140713061258/http://www.youkaiwiki.com/entry/2013/01/29/%E3%81%B2%E3%82%87%E3%81%86%E3%81%99%E3%81%B9 |date=2014-07-13 }}</ref>

* '''哇伊啦'''（{{lang|ja|わいら}}）
: 「哇伊啦」有個像牛一樣龐大的軀體，並且有扭曲而蠻橫的臉孔，同時雙手各只有一支銳利的指爪，感覺是個兇殘的怪物。但是相關的記載文字的紀錄全部闕如，雖然有學者從他的名字判斷是從恐怖（{{lang|ja|怖い}}，Kowai）轉來的而猜測這是恐懼具現化的怪物，但並未成為重要的說法。<ref name="content" /><ref>[http://www.youkaiwiki.com/entry/2013/01/23/%E3%82%8F%E3%81%84%E3%82%89 {{lang|ja|わいら - 妖怪うぃき的妖怪図鑑}}] {{webarchive|url=https://web.archive.org/web/20141106210820/http://www.youkaiwiki.com/entry/2013/01/23/%E3%82%8F%E3%81%84%E3%82%89 |date=2014-11-06 }}</ref>

* '''休喀拉'''（{{lang|ja|しょうけら}}）
: 在古代，中國[[道教|道教]]相信人的體內有[[三尸|三尸]]神，「上尸名彭琚，管人[[三焦|上焦]]善惡；中尸名彭瓚，管人[[三焦|中焦]]善惡；下尸名彭橋，管人[[三焦|下焦]]善惡。」他們會記錄人的是非善惡，然後在[[庚申|庚申]]日趁人們睡著的時候上[[天庭|天庭]]打小報告（另有一說是只要人睡著了就會去打小報告），[[天帝|天帝]]就會依據這些報告來剋扣人壽命。於是主張庚申日不睡以逃避三尸的小報告。這個習俗後來傳到日本，在《[[枕草子|枕草子]]》中有提及，不過後來三尸變成妖怪的形象，多半採擬人如蟲大小姿態，而在[[鳥山石燕|鳥山石燕]]的《[[畫圖百鬼夜行|畫圖百鬼夜行]]》，則是將其繪成趴在天窗上偷窺人的野獸模樣。
: 名字又可記為「精螻蛄」。<ref name="content" /><ref>[http://www.youkaiwiki.com/entry/2013/01/29/%E3%81%97%E3%82%87%E3%81%86%E3%81%91%E3%82%89 {{lang|ja|しょうけら - 妖怪うぃき的妖怪図鑑}}] {{webarchive|url=https://web.archive.org/web/20140713061031/http://www.youkaiwiki.com/entry/2013/01/29/%E3%81%97%E3%82%87%E3%81%86%E3%81%91%E3%82%89 |date=2014-07-13 }}</ref>

* '''歐托羅悉'''（{{lang|ja|おとろし}}）
: 跟「哇伊啦」一樣是個完全沒有說明的妖怪，但形象駭人許多，會攀附在屋樑或[[鳥居|鳥居]]之上，繁盛而茂密的頭髮纏繞著依靠著的物事，有說法認為牠會忽然跳到人的面前嚇人，但也有說法認為這只是望圖生意而已。不過「{{lang|ja|おとろし}}」也跟「恐懼」同音，所以也有人認為這是跟恐懼有關的妖怪。<ref name="content" /><ref>[http://www.youkaiwiki.com/entry/2013/01/30/%E3%81%8A%E3%81%A8%E3%82%8D%E3%81%97 {{lang|ja|おとろし - 妖怪うぃき的妖怪図鑑}}] {{webarchive|url=https://web.archive.org/web/20140326110318/http://www.youkaiwiki.com/entry/2013/01/30/%E3%81%8A%E3%81%A8%E3%82%8D%E3%81%97 |date=2014-03-26 }}</ref>
