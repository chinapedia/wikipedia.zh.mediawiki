'''感冒'''是以鼻症状（[[鼻塞|鼻塞]]、[[流鼻水|流鼻水]]）和[[发烧|发烧]]为主要特征的人类常见的[[疾病|疾病]]。发烧是由于[[上呼吸道|上呼吸道]]重度[[发炎|发炎]]。重度发炎引起的并发症还包括[[头晕|头晕]]、[[头痛|头痛]]、[[咽痛|咽痛]]、畏寒、肠胃不适、[[食欲不振|食欲不振]]、全身乏力等等。

==原因 ==
感冒的原因是指引起上呼吸道重度发炎发生的条件。人的肌体表面有一层薄膜，体外部分是[[皮肤|皮肤]]，体内部分是[[粘膜|粘膜]]。体外皮肤与体内粘膜无缝连接，是防止[[微生物|微生物]]侵蚀的屏障。只有在这层薄膜破损后才有可能发炎。

引起上呼吸道发炎的微生物是沉积在呼吸道表面的呼吸道微生物，呼吸道微生物来源于吸入空气中。空气中含有空气颗粒物，其中包含各种微生物，它们随吸入空气进入呼吸道，依粒径的大小依次分布、沉积在鼻腔、咽喉和气管成为呼吸道微生物<ref>朱贵卿 1984 呼吸内科学 人民卫生出版社 北京 p129 统一书号14048.4535</ref>。微生物的种类取决于吸入环境。浓度取决于吸入环境空气微生物的生發状态<ref>车凤翔 2004 空气生物学原理及应用 科学出版社 北京 p28 ISBN 7-03-011536-8</ref>。浓度越大、吸入速度越快，吸入空气微生物在呼吸道沉积的数量就越多，一旦上呼吸道粘膜破损，呼吸道微生物的数量就决定了发炎的严重程度。

[[打噴嚏|打喷嚏]]（[[咳嗽|咳嗽]]）是使上呼吸道粘膜破损的的主要原因。打喷嚏是由外界因素刺激鼻粘膜和肌体发生的。外界因素有颗粒物（浓度和性质）、温度和光照等等。

喷嚏的作用是清除呼吸道异物。打喷嚏时，大量的气体高速冲击造成上呼吸道狭窄和弯转处粘膜破损。同时呼吸道微生物进入破损粘膜之下造成发炎。

== 侵入微生物数量不同导致的不同结果 ==
[[File:一位病毒性感冒患者的血常规.jpg|130px]]数据，病毒性感染可致血液中[[单核细胞|单核细胞]]的比例急剧上升]]
当进入粘膜下的微生物较少时，肌体的自然防御机制能够抵御发炎。

当进入的微生物较多时，粘膜下的发炎会缓慢展开进入[[鼻腔|鼻腔]]和[[口腔|口腔]]（耳、眼）。口鼻生疮被认为是“上火”。

当进入的微生物更多一些时，鼻咽部的发炎会引发鼻塞和流鼻水的鼻症状。不发烧即是“[[傷風|伤风]]”。伤风往往会很快自癒。但发炎也常常会变得严重，体温升高演变成感冒。

当有大量微生物进入时，重症发炎，体温迅速升高。当特别严重，高烧持续数天不退，就会引起恶性继发病甚至导致死亡。

引起打喷嚏因素的不确定性和肌体的特异性使得打喷嚏具有个体不确定性。但深秋、冬季和初春封闭室内（夏天空调间）空气颗粒物浓度达到最高，并且室内外温差较大，气温骤变刺激鼻粘膜和肌体，诱发多人打喷嚏，而后出现鼻症状和发烧。这是造成群体伤害的季节性感冒。

儿童易患感冒是因为儿童对外界刺激更为敏感，上呼吸道粘膜脆弱，呼吸速度较快而且他们的吸入空气环境更恶劣。

== 继发病 ==
感冒发生后而发生的肌体其它部位疾病是感冒的继发病。

鼻塞、流鼻水会使鼻腔过滤功能下降，大量吸入空气颗粒物和上呼吸道发炎的碎屑沉积在下呼吸道会引发咳嗽和哮喘，其中的微生物会引发气管炎和肺炎。

引起头晕、头痛的发炎发生在鼻腔的上部。当发炎持续恶化，微生物会侵蚀、破坏大脑保护层造成[[脑膜炎|脑膜炎]]或[[脑炎|脑炎]]。

从发炎处进入[[血液|血液]]的微生物随血流游遍全身，大量持续进入血液的微生物会造成各种伤害，生成不同的疾病。如体外皮肤的[[疱疹|疱疹]]和肌体内某些器官的发炎。脏器受损严重会衰竭导致死亡。

== 治疗 ==
不论是体外皮肤还是体内粘膜发炎，直接对病灶给药都是最好的消炎方法。消除鼻炎可采用滴鼻法，消除咽炎可采用口腔消炎物保持法。这也是治疗鼻咽炎传统经典的方法。

摩擦鼻两翼，面部热敷，吸入热空气可使鼻症状迅速消失。
