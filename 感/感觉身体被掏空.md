《'''感觉身体被掏空'''》，是由[[上海彩虹室内合唱团|上海彩虹室内合唱团]]于2016年7月演唱的一首[[合唱|合唱]]曲目，以白领日常生活为主题<ref name="Sohu News">{{cite web |url=http://news.sohu.com/20160728/n461415873.shtml|title=彩虹合唱团新歌《感觉身体被掏空》首发|publisher=搜狐新闻|date=2016-07-28| accessdate=2016-07-29 }}</ref>。因为调皮的歌词描述了广大上班族的心声，被网友和部分媒体称作“加班狗必听爆红神曲”。<ref name="Sichuan Online Fan">{{cite web |url=http://sichuan.scol.com.cn/wkd/201607/55600351.html|title=实力洗脑！加班狗必听爆红神曲《感觉身体被掏空》|publisher=四川在线 - Fan 泛而谈|date=2016-07-28| accessdate=2016-07-29 }}</ref>视频发布后仅7个小时播放量就达到了三百多万次。<ref name="lianhexinwen"></ref>

== 歌曲内容 ==
《感觉身体被掏空》由上海彩虹室内合唱团的指挥金承志一人编曲作词<ref name="The Beijing News"/>，有网友指出歌曲前奏是[[王心凌|王心凌]]的《睫毛弯弯》，但據作者金承志說，那來自[[功夫_(遊戲)|功夫]]的音樂。歌曲中的一句「宝贝，加班吧」是由[[黎明|黎明]]本人为这首歌曲所配的真声。<ref name="Sina Taiwan">{{cite web|url=http://news.sina.com.tw/article/20160728/18068105.html|title=《感覺身體被掏空》里的彩蛋和幕後故事你都知道嗎？|publisher=北京[[新浪網|新浪網]]|date=2016-07-28|accessdate=2016-07-29|deadurl=yes|archiveurl=https://web.archive.org/web/20160729163130/http://news.sina.com.tw/article/20160728/18068105.html|archivedate=2016-07-29}}</ref>

== 评价 ==
本歌曲作曲作词人金承志在接受采访时说这首歌是一首触到职业人士痛处的音乐作品。以幽默调侃的方式揭露年轻人的疲惫。创作这首歌的目的是以有趣的方式为他们减压。<ref name="China Daily Language">{{cite web |url=http://language.chinadaily.com.cn/2016-07/29/content_26271826.htm|title=新神曲《感觉身体被掏空》唱哭加班狗 Song strikes a chord with overworked professionals|publisher=中国日报|date=2016-07-29| accessdate=2016-07-29 }}</ref>

《[[京华时报|京华时报]]》在形容这首歌曲的时候认为彩虹室内合唱团在“用严肃音乐一本正经地搞笑”，认为这是一首非常接地气的歌曲，整首歌曲以非常轻松幽默加调侃的笔法，很容易让在大城市奋斗的年轻人，有一种情绪的代入感。其在创作手法上，其实非常接近早期许冠杰和李宗盛这些唱作人的手法，都是将小人物的心声，以自嘲、戏谑等方式，上升到流行文化高度的创作。<ref name="JingHuaShiBao">{{cite web |url=http://opinion.people.com.cn/n1/2016/0729/c1003-28594125.html|title=京华时报：《感觉身体被掏空》用严肃音乐一本正经地搞笑|publisher=[[人民网|人民网]]|date=2016-07-29| accessdate=2016-07-29 }}</ref>在谈到合唱这一演唱类型时，《京华时报》认为彩虹室内合唱团用这么一种看似不正经的方式，让合唱形式，在更大范围内多少得到一些扫盲和普及。<ref name="JingHuaShiBao"/>

有媒体分析其走红原因，主要有三点：其一是因为乐团在生产、改编神曲方面拥有丰富经验；其二是因为歌曲的旋律和歌词做到朗朗上口；其三是加班这个话题，很容易引起城市人群共鸣。<ref name="XinBang@Xaidu">{{cite web |url=http://xudanei.baijia.baidu.com/article/561696|title=加班神曲《感觉身体被掏空》是如何一夜之间爆红的？|publisher=新榜——内容创业服务平台|date=2016-07-28| accessdate=2016-07-29 }}</ref>

[[阿里音乐|阿里音乐]]的CEO[[宋柯|宋柯]]，在听到这首歌曲后，在朋友圈发表了“想想一年来的生活，老泪纵横”这样的感触。<ref name="JingHuaShiBao"/><ref name="CNATaiwan">{{cite web |url=http://www.cna.com.tw/news/firstnews/201607290210-1.aspx|title=陸加班神曲 感覺身體被掏空一夜爆紅[影]|publisher=[[中央通讯社|中央通讯社]]|date=2016-07-29| accessdate=2016-07-29 }}</ref>

有网友表示，歌曲中自嘲的态度触动了他们的心弦，而另一些人则说自己已经被旋律洗脑，突然有一种辞掉工作去放松的冲动。也有微博网友说彩虹室内合唱团的一系列歌曲改变了他对合唱团的印象，高雅艺术也可以与大众文化相结合。一个成功的作品总是能反映普通大众的快乐和悲伤。<ref name="China Daily Language"/>
