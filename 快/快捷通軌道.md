'''快捷通軌道'''（{{lang-en|'''Rapid Rail'''}}），即為'''巴生谷輕快鐵系統'''，是一個服務[[马来西亚|马来西亚]][[巴生谷|巴生谷]]（包括[[雪蘭莪|雪蘭莪]]及[[吉隆坡|吉隆坡]]）的[[城市軌道交通|城市軌道交通系統]]，目前營運中的路線包括三條[[轻快铁|轻快铁]]线路（[[安邦綫|安邦綫]]、[[大城堡线|大城堡线]]、[[格拉那再也綫|格拉那再也綫]]）和一條[[單軌鐵路|單軌鐵路]]（[[吉隆坡單軌列車|吉隆坡單軌列車]]），四条线路加起来总长约{{convert|159.2|km|mi}}。<ref name="LRT">{{cite web |url=http://www.myrapid.com.my/about/our-assets/lrt |title=LRT |publisher=MyRapid (Syarikat Prasarana Negara Berhad) |accessdate=2014-04-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20140202155443/http://www.myrapid.com.my/about/our-assets/lrt |archivedate=2014-02-02 }}</ref>快捷通軌道系統由{{tsl|en|Prasarana Malaysia|马来西亚国家基建公司}}擁有，並由其全資子公司'''快捷通軌道私人有限公司'''（Rapid Rail Sdn Bhd）營運，屬於[[吉隆坡快捷通|吉隆坡快捷通]]（Rapid KL）品牌名稱旗下的公共交通服務之一，而旗下另一項公共交通服務為[[快捷通巴士|快捷通巴士]]（Rapid Bus）。快捷通轨道（以“吉隆坡快捷通”为其名）、[[马来亚铁道|马来亚铁道]]、[[吉隆坡机场快铁|机场快铁（ERL）]]和最新的[[巴生谷捷运系统|巴生谷捷运系统]]组成了巴生谷的公共运输铁路网。轻快铁采用[[标准轨|标准轨]]，而单轨则采用ALWEG型轨道。列车营运时间从早上6时至11时或12时左右，<ref name="hours">{{cite web |url=http://www.myrapid.com.my/rail/operating-hours |title=Operating Hours |publisher=MyRapid (Syarikat Prasarana Negara Berhad) |accessdate=2014-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20140701091236/http://www.myrapid.com.my/rail/operating-hours |archivedate=2014-07-01 }}</ref>尖峰时段时平均每3分钟一趟班车，离峰时段则为最久14分钟一班。<ref name="freq">{{cite web |url=http://www.myrapid.com.my/rail/train-frequency |title=Train Frequency |publisher=MyRapid (Syarikat Prasarana Negara Berhad) |accessdate=12 June 2014 |deadurl=yes |archiveurl=https://web.archive.org/web/20101220041547/http://www.myrapid.com.my/rail/train-frequency |archivedate=2010年12月20日 }}</ref>

== 发展历史 ==
=== 建设背景 ===
[[马来西亚|马来西亚]]自1980年代起突飞猛进的经济发展，使[[首都|首都]][[吉隆坡|吉隆坡]]的[[机动车|机动车]]数量和道路交通流量不断增加；加上[[马来西亚|马来西亚]]政府采取扶持国产汽车工业的政策，使[[马来西亚|马来西亚]]成为国民汽车拥有率最高的[[东南亚|东南亚]]国家。根据[[马来西亚|马来西亚]]政府在1980年代初进行的吉隆坡总体交通规划，预测到2000年首都私人汽车数量将比1980年增加2.8倍，当局有必要改变交通政策和提高[[公共交通|公共交通]]的比重，否则[[吉隆坡|吉隆坡]]未来不得不面对严重的[[交通堵塞|交通堵塞]]，因此该规划报告建议在[[吉隆坡|吉隆坡]]市中心形成以[[城市轨道交通|城市轨道交通]]为主、[[公共汽车|公共汽车]]为辅的交通运输系统。

=== 建设过程 ===
==== 安邦線（Ampang Line） ====
1990年，由[[马来西亚|马来西亚]]和外国公司组成的联合体完成吉隆坡轻轨系统（LRT）的可行性研究，建议借鉴[[香港|香港]]和[[新加坡|新加坡]]兴建地铁的经验，并优先上马从市中心通往东南方向、连接[[安邦|安邦]]和[[冼都|冼都]]的[[轻轨铁路|轻轨铁路]]。1992年，轻轨运输系统私人有限公司（Sistem Transit Aliran Ringan Sdn Bhd）由9家公司合股成立，负责这条轻轨铁路的建设和经营，因此这条路线亦得名为“STAR”（实达轻快铁，现称为[[安邦綫|安邦綫]]）。實達輕快鐵一期工程全长12.1公里，由[[安邦站|安邦站]]至[[蘇丹依斯邁站|蘇丹依斯邁站]]，于1993年8月完成融资协议并正式动工，至1996年12月建成通车。该线是[[马来西亚|马来西亚]]第一条轻轨铁路，轨道采用1,435毫米标准轨距，牵引供电制式为750伏特直流[[第三轨供电|第三轨供电]]；由[[德国|德国]][[AEG|AEG公司]]（后来为[[Adtranz|Adtranz]]）和英国{{tsl|en|Taylor Woodrow|泰勒·伍德罗公司}}分别作为机电和土木工程承建商；投资模式为[[BOO模式|BOO模式]]（建设－拥有－经营）及土地补偿，以配合当时马来西亚政府的[[公用事业|公用事业]]民营化政策。实达线二期工程包括北延线（[[苏丹依斯迈站|苏丹依斯迈站]]至[[冼都东站|冼都东站]]）和南延线（[[陈秀连站|陈秀连站]]至[[大城堡站|大城堡站]]），两条延线均于1998年建成通车。在2016年7月，大城堡线延长线正式通车。现在，安邦线被拆分为两条线路，分别为安邦线和大城堡线。

==== 格拉納再也線（Kelana Jaya Line） ====
1993年，为配合将于[[吉隆坡|吉隆坡]]举行的[[1998年英联邦运动会|1998年英联邦运动会]]，[[马来西亚|马来西亚]]政府决定在吉隆坡兴建第二条轻快铁铁路，于是成立了轻轨运输合作项目私人有限公司（Projek Usahasama Transit Ringan Automotif Sdn Bhd），来负责第二条轻快铁线路的实施，因此这条路线亦得名为布特拉輕快鐵（Putra LRT）（现称为[[格蘭那再也綫|格蘭那再也綫]]）。布特拉輕快鐵全长29公里，在[[吉隆坡|吉隆坡]]市内自东北向西南延伸，连接鵝嘜和格兰那再也，并途径[[雙峰塔|雙峰塔]]和[[吉隆坡塔|吉隆坡塔]]等著名地标。布特拉輕快鐵的投资模式为[[BOT模式|BOT模式]]及土地补偿，于1994年2月正式动工，第一阶段（[[格拉那再也站|格拉那再也站]]至[[中央藝術坊站|中央藝術坊站]]）于1998年9月建成通车，第二阶段（[[中央藝術坊站|中央藝術坊站]]至[[鵝嘜站|鵝嘜站]]）于1999年6月建成通车。这是启用当时是世界上营运里程最长的全自动[[列车自动运行系统|无人驾驶]]轻轨系统，也是全球第四个[[庞巴迪运输|庞巴迪]][[庞巴迪Innovia_Metro|先进快速轨道交通系统]]（ART），采用[[直线电动机|直线电机]]驱动的无人驾驶列车、750伏特直流第三轨供电、“[[SelTrac|SelTrac]]”[[CBTC|CBTC移动闭塞系统]]。在2016年7月，格兰那再也线延长线正式通车。

==== 吉隆坡单轨铁路 （KL Monorail）====
1996年，[[马来西亚|马来西亚]]政府批准兴建[[吉隆坡單軌列車|吉隆坡單軌列車]]（KL Monorail）的计划，该计划最初曾经被称为旅客捷运系统（PRT）。这条跨坐式高架单轨铁路全长8.6公里，串连着[[吉隆坡中央车站|吉隆坡中央车站]]交通枢纽与市中心金三角地带。这条采用BOT模式兴建的单轨铁路于1993年8月正式动工，由吉隆坡基建集团（KLIG）拥有40年的特许经营权，机电和土木工程承建商原本为[[日本|日本]][[日立|日立公司]]，但1997年[[亞洲金融風暴|亞洲金融風暴]]后因融资困难而暂停工程；1998年改由本国企业“MTrans”公司（今{{tsl|en|Scomi Rail|史格米铁路|史格米铁路公司}}）继续完成工程，至2003年8月建成通车，总共耗资11亿8000万令吉。

=== 企业重组 ===
然而，上述三条城市轨道交通路线建成通车之后，三家轻轨铁路和单轨列车营运商都面临严重的财政困难，主要原因是基础建设和车辆购置费用等固定投资的债务偿还压力，而且客流量水平亦远低于建设规划时的估计。由于吉隆坡的私人汽车拥有比率非常之高，加上政府缺乏对泊车设施和接驳交通等配套的规划，使许多吉隆坡市民仍然习惯使用私人汽车出行，阻碍了城市轨道交通系统使用率的提高。以安邦线为例，该线最初预计到2003年的日均客流量将达到50万人次，但实际仅达到11万人次。除此之外，城市轨道交通系统建设规划上的不足亦备受批评，由于三条路线分别由三家公司各自营运且互不隶属，当这几条路线落成时竟然没有统一的换乘车站，让乘客转乘另一条路线时面对极大麻烦。

2001年11月，[[马来西亚|马来西亚]]联邦政府不得不负起最后承担者的角色，接管了两家频临破产危机的轻轨铁路营运公司，并由企业债务重组委员会（CDRC）负责对其进行{{tsl|en|Debt restructuring|债务重组}}，当时这两家公司合共尚有57亿令吉（约合16.1亿美元）的未偿还债务。随后，为了加强整合吉隆坡的公共交通系统、提升公共交通的营运效率和服务质素，马来西亚联邦政府于2004年7月注资成立吉隆坡快捷通公司（RapidKL），作为{{tsl|en|Prasarana Malaysia|马来西亚国家基建公司}}（Prasarana）的子公司，自2005年1月1日起成为吉隆坡的公共交通营运商，负责经营巴生谷的轻快铁铁路和巴士服务。<ref>{{cite web |url=http://www.myrapid.com.my/about/corporate-profile |title=Profile - SYARIKAT PRASARANA NEGARA BERHAD |publisher=[[Syarikat_Prasarana_Negara|Syarikat Prasarana Negara]] |accessdate=2014-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20140202160642/http://www.myrapid.com.my/about/corporate-profile |archivedate=2014-02-02 }}</ref>2007年5月，营运单轨列车的吉隆坡基建集团因无力偿还巨额贷款而宣布破产，并于同年11月由马来西亚国家基建公司收购了吉隆坡单轨列车。自此，安邦线、格拉那再也线、吉隆坡单轨列车构成了巴生谷輕快鐵系統<ref>{{Cite web |url=http://www.myrapid.com.my/corporate/our-business/core-business/rapid-rail |title=Rapid Rail |accessdate=2014-10-20 |publisher=Prasarana Malaysia Berhad |date= |deadurl=yes |archiveurl=https://web.archive.org/web/20141009115513/http://www.myrapid.com.my/corporate/our-business/core-business/rapid-rail |archivedate=2014-10-09 }}</ref>。

=== 近年发展 ===
2011年，吉隆坡快捷通公司推出名为自动售检票系统（AFC）的综合票务系统，使乘客可在自动售票机购买代币式车票，只需一枚代币即可通行所有轻轨和单轨铁路的车站；而持有“[[一触即通卡|一触即通卡]]”（Touch'n Go）或“我的快捷卡”（MyRapid Card）的乘客，可直接在验票闸机上拍卡及进出车站。自动售检票系统于2011年11月起在轻轨车站开始使用，并于2012年2月起扩展至所有单轨列车车站<ref>{{Cite web |url=http://www.nanyang.com/node/438996?tid=510 |title=只收硬币 单轨火车售票机不方便 |accessdate=2014-10-20 |publisher=南洋网 |date=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304113917/http://www.nanyang.com/node/438996?tid=510 |archivedate=2016-03-04 }}</ref><ref>{{Cite web |url=http://mandarin.bernama.com/v3/index.php?sid=news&cat_news=ge&id=70261 |title=轻快铁及单轨火车11月杪起只使用一张票 |accessdate=2014-10-20 |publisher=马新社 |date=2011-06-14 |archive-url=https://web.archive.org/web/20170811055538/http://mandarin.bernama.com/v3/index.php?sid=news&cat_news=ge&id=70261 |archive-date=2017-08-11 |dead-url=yes }}</ref>。2013年1月，马来西亚国家基建公司宣布进行企业架构重组，建立快捷通轨道公司（Rapid Rail Sdn Bhd）经营原本由吉隆坡快捷通公司营运的铁路服务，并建立快捷通巴士公司经营原本由吉隆坡快捷通公司营运的巴生服务<ref>{{Cite web |url=http://www.myrapid.com.my/news-events/media-releases/2013/prasarana-undertakes-new-corporate-restructuring-exercise |title=Prasarana undertakes new corporate restructuring exercise |accessdate=2014-10-20 |publisher=Prasarana Malaysia Berhad |date=2013-01-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20160305005222/http://www.myrapid.com.my/news-events/media-releases/2013/prasarana-undertakes-new-corporate-restructuring-exercise |archivedate=2016-03-05 }}</ref>。
2018年由一触即通公司与支付宝合作研发的Touch n go电子钱包开通了乘车码的服务，现处试运营阶段，适用于部分车站，用户可向Touch n go公司申请，即可体验此项服务。
