== 剧情 ==
故事承接第一部，Mumble与Gloria的儿子艾瑞克出生了。而艾瑞克对歌舞没有半点天赋，因此感到自卑甚至离家出走。后来一股神秘的力量侵入企鹅王国威胁众人的安全，内忧外患之下Mumble和朋友们奋力反击，而艾瑞克也体会到了父亲的勇气。

== 评价 ==
好评有：“片中的一切都是那么美好”，“数量急剧增多的企鹅歌舞场景，在3D技术的帮助下非但没有过载，反而举重若轻”，“通过诸多精妙而细微的渠道，影片传递出相比于前作更加丰富与复杂的情节信息”。

坏评有：“虽然故事堪称丰满，但影片缺少能够让人斟酌品味的创新”，“角色的性格和形象塑造缺乏魅力，再加上笑料的贫瘠，常常给人一种万物冻僵的呆滞感”<ref>{{cite web|author=新浪娱乐|title=本周新片《暮光之城4》《快乐的大脚2》《后人》|url=http://ent.sina.com.cn/m/f/2011-11-21/ba3486348.shtml|date=2011年11月21日}}</ref>。

== 票房 ==
