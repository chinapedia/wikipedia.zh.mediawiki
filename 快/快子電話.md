== 用快子傳訊號到過去 ==

假設甲正坐在以速度β（<1）离开乙。两者都有以超光速α相互傳递訊號的能力。甲向乙發出一個訊號。乙收到訊號後同時向甲發出一個回應。考慮甲和乙分别靜止的两個[[參照系|參照系]]。把乙受到甲訊息此事件設為两參照系的原點。乙在<math>t=0</math>向甲發回應。乙發出的回應的[[世界綫|世界綫]]是：
:<math>(t, x) = \left(t, \alpha t \right)</math>
此世界線在甲的參照系内座標可以用[[勞侖茲變換|勞侖茲變換]]得到：
:<math>(t^{\prime}, x^{\prime}) = \left( \gamma (1 - \alpha \beta)t, \gamma(\alpha - \beta)t \right)</math>
[[File:Tachyon.png|thumb]]。黑座標是甲的靜止系。紅座標是乙的靜止系。時間方向向上。右邊黑線是甲的世界綫。可見快子回到甲發出訊號前。]]在甲的參照系内，甲在<math>x^{\prime} = L^{\prime}</math>靜止。<math>L^{\prime}</math>正是乙發出訊號時甲和原點的距离，所以：
:<math>t = \frac{L^{\prime}}{\gamma(\alpha - \beta)}</math>
因此
:<math>t^{\prime} = \frac{1 - \alpha \beta}{\alpha - \beta} L^{\prime}</math>。
留意這只是相對論速度加法公式的應用。

當初甲發出的那个訊號需時<math>\frac{L^\prime}{\alpha}</math>到達乙。所以甲從發出訊號到收到回應共歷時：
:<math>T^{\prime} = \frac{L^{\prime}}{\alpha} + t^{\prime}
= \left( \frac{1}{\alpha} + \frac{1 - \alpha \beta}{\alpha - \beta}\right) L^{\prime}</math>
當<math>\beta > \frac{2\alpha}{1+\alpha^2}</math>，<math>T^{\prime} < 0</math>。也就是甲在發出訊號之前就收到乙回信。
整個過程的世界線在右圖。

另外甲在:
:<math>(t_0^{\prime}, x_0^{\prime}) = \left(-\frac{L^{\prime}}{\alpha}, L^{\prime}\right)</math>
給乙發訊號。在乙參照系内，此事發生在：
:<math>(t_0, x_0) = \left(\gamma (\beta-\frac{1}{\alpha}) L^{\prime}, \gamma (1 -\frac{\beta}{\alpha}) L^\prime \right)</math>
當<math>\beta > \frac{1}{\alpha}</math>，<math>t_0 > 0</math>。也就是乙收到的訊號來自他的未來。
