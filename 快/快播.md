'''快播'''（{{lang-en|'''QvodPlayer'''}}）是一款基于视频点播的[[播放软件|播放软件]]。与传统播放软件不同的是，快播应用[[點對點技術|P2P]]技术并支持[[RMVB|RMVB]]、[[MPEG|MPEG]]、[[AVI|AVI]]、[[WMV|WMV]]等主流音视频格式，软件本身不收取站长或用户使用费用，用户透過缓冲可观看在线影视节目，操作相对简单，且软件容易订制，是[[中国大陆|中国大陆]]较为流行的一款[[视频|视频]]播放[[下载|下载]][[软件|软件]]，又称为“Q播”、“QVOD”。

在中国大陆，快播拥有着巨大的用户數量。由于采用了专用的[[网络传输协议|传输协议]]<ref name="faq">{{Cite web |url=http://faq.kuaibo.com/file/19/ |title=快播产品介绍 |access-date=2012-10-26 |archive-url=https://web.archive.org/web/20121020140729/http://faq.kuaibo.com/file/19/ |archive-date=2012-10-20 |dead-url=yes }}</ref>，所以快播被部分用户用来下载被禁影片，如有[[暴力|暴力]]或[[色情|色情]]性质的影片、涉及[[政治|政治]]敏感的影片等。另外，在快播中[[盗版|盗版]]影片很泛滥<ref>{{cite web|title=快播：野蛮的代价|url=http://tech.163.com/12/0726/20/87C8U8N000094L5O.html|work=《创业家》|publisher=网易科技|accessdate=2013-04-11}}</ref>。

== 其他版本 ==
快播目前还有以下数个版本供不同用户使用：360安全版、[[Android|Android]]版(最新版3.0.49)、精简版(最新版5.8)、不升级版(最新版3.5)、快播搜狐版、Windows Phone版(最新1.0.0)、iOS版(最新版1.5.32)。

=== iOS/WP/Android客户端 ===
快播[[iOS|iOS]]/[[Windows_Phone|Windows Phone]]/[[Android|Android]]版播放器(Qvod Player)是一款集在线点播、在线直播为一体的媒体浏览器。支持其自主研发的 QVOD 流媒体传输协议，实现高清视频在行動裝置上较为流畅的播放体验。与桌面版一样，支持[[RMVB|RMVB]]、[[MPEG|MPEG]]、[[AVI|AVI]]、[[WMV|WMV]]等主流音视频格式。最新版本为1.1.11版。

== 快播网站 ==
由于快播对于伺服器要求不高，很多站长自己架设快播服务端提供服务。大部分快播网站都提供盗版影视下载服务，并透过在线广告营利。
