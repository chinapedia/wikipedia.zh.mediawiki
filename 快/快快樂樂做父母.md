《'''快快樂樂做父母'''》是香港慈善機構「[[龍耳|龍耳]]」出版的育嬰指南，透過[[手語|手語]]作講解，向聾人及弱聽人士教導正確照顧初生孩子的技巧。

==背景==
手語除了是[[聾|聾]]人社群的文化外，同時亦是聾人的溝通方式及接收社會資訊的渠道，香港已是聯合國《[[殘疾人權利公約|殘疾人權利公約]]》的締約地區，但在社會資訊無障礙，包括手語資訊發放等仍然乏善足陳，例如政府資訊仍未全面提供[[手語翻譯|手語翻譯]]、而新聞節目亦只設字幕而不設即時[[手語傳譯|手語傳譯]]，忽略書面語與手語文法的分別，無疑是大大剝奪聾人社會參與的渠道，亦為家庭生活的質素帶來負面影響。

龍耳提供家庭服務的經驗中，收到不少關於照顧子女技巧的查詢，在得到[[凱瑟克基金|凱瑟克基金]]的贊助以及[[衞生署_(香港)|衞生署]]的支持，耗時兩年製作設有手語翻譯的手語育嬰指南，為聾人家庭提供支援<ref> {{cite web |language=zh-hk |url=http://news.takungpao.com.hk/paper/q/2015/0808/3107440.html | title=手語育嬰指南支援聽障父母 |publisher=[[大公報|大公報]] |date=2015-08-08 |accessdate=2015-08-09}}</ref>。

==個案==
根據報章報導，本身是聾人的趙曉蘭女士，育有同樣有聽覺問題的四歲女兒。她表示照顧女兒時經常遇到困難，雖然嘗試透過坊間的教材學習照顧女兒，但因缺乏手語翻譯，不能完全明白。

趙女士和女兒曾因溝通出現困難，如聽不到女兒叫喊聲而感到失落，她認為有手語翻譯的育嬰指南，能教導聾人育嬰技巧，改善他們和子女相處時遇到的問題，避免重蹈覆轍<ref> {{cite web |language=zh-hk |url=http://hd.stheadline.com/news/daily/hk/361122/ |title=﻿互聯網免費睇 手語育嬰指南支援聽障爸媽 |publisher=[[頭條日報|頭條日報]] |date=2015-08-08 |accessdate=2015-08-09 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

龍耳的社工指出部份聾人父母會將子女交給健聽家人代為照顧，例如祖父、祖母或舅父、舅母，這些方法可能幫助他們照顧子女，但長遠來說會帶來親子關係疏離，甚至有子女長大後會鄙視父母<ref> {{cite web |language=zh-hk |url=http://www.hkatv.com/zh-hk/atvnews/45701/2015-08-08/%E5%9C%98%E9%AB%94%E7%B7%A8%E5%88%B6%E6%89%8B%E8%AA%9E%E8%82%B2%E5%AC%B0%E6%8C%87%E5%8D%97%E5%8D%94%E5%8A%A9%E8%81%BD%E9%9A%9C%E4%BA%BA%E5%A3%AB%E7%85%A7%E9%A1%A7%E5%AD%90%E5%A5%B3 |title=團體編制手語育嬰指南協助聽障人士照顧子女 |publisher=[[亞視新聞|亞視新聞]] |date=2015-08-08 |accessdate=2015-08-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304125614/http://www.hkatv.com/zh-hk/atvnews/45701/2015-08-08/%E5%9C%98%E9%AB%94%E7%B7%A8%E5%88%B6%E6%89%8B%E8%AA%9E%E8%82%B2%E5%AC%B0%E6%8C%87%E5%8D%97%E5%8D%94%E5%8A%A9%E8%81%BD%E9%9A%9C%E4%BA%BA%E5%A3%AB%E7%85%A7%E9%A1%A7%E5%AD%90%E5%A5%B3 |archivedate=2016-03-04 }}</ref>。

==出版目的==
出版目的是希望透過專業的教材，用聾人的母語作講解，教導聾人父母正確照顧初生孩子的技巧，建立溫馨及愉快的親子環境，以解決聾人家庭服務中常見的焦慮、無助或親子關係疏離的情況<ref> {{cite web |language=zh-hk |url=http://std.stheadline.com/breakingnews/20150807a183651.asp | title=全港首套手語育嬰指南支援聾人家庭 (港聞)  |publisher=[[星島日報|星島日報]] |date=2015-08-07 |accessdate=2015-08-09}}</ref><ref> {{cite web |language=zh-hk |url=http://hd.stheadline.com/news/realtime/hk/996788/ |title=全港首套手語育嬰指南支援聾人家庭 |publisher=[[頭條日報|頭條日報]] |date=2015-08-07 |accessdate=2015-08-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20151218205128/http://hd.stheadline.com/news/realtime/hk/996788/ |archivedate=2015-12-18 }}</ref>。

慈善機構「龍耳」表示，聾人理解文字的能力較弱，學習新事物比較困難，於是將衞生署的育嬰指南加設手語翻譯製作成教學光碟，免費派發和上載到網上平台「[[龍耳電視|龍耳電視]]」，教導他們照顧嬰兒的起居飲食和親子溝通的方法，讓彼此的關係更親密<ref> {{cite web |language=zh-hk |url=http://www.mingpaocanada.com/tor/htm/News/20150808/HK-gzf1_r.htm | title=手語育嬰指南 教聽障者父母之道 |publisher=[[明報|明報]] |date=2015-08-08 |accessdate=2015-08-09}}</ref>。
