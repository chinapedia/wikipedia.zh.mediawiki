《'''倚天屠龍記'''》是根据[[金庸|金庸]]同名小说改编的一部古装武侠电视剧，由北京亚环影音制作有限公司、中国电视艺术家协会、北京世纪梦工场影视交流有限公司聯合製作。继[[中央电视台|中央电视台]]拍摄大陆版的《[[笑傲江湖|笑傲江湖]]》后，一股金庸武侠剧热潮在大陆火起来了，在这个大背景下，亚环影音于2001年出资70多万多港币购买了《[[倚天屠龙记|倚天屠龙记]]》的版权，成为继《[[笑傲江湖_(2001年电视剧)|笑傲江湖]]》、《[[射鵰英雄傳_(2003年電視劇)|射雕英雄传]]》、《[[侠客行_(2002年电视剧)|侠客行]]》、《[[连城诀_(中国电视剧)|连城诀]]》后大陆第五部拍摄的金庸作品<ref>[http://ent.sina.com.cn/v/48620.html 70多万元港币购得拍摄权 内地将拍《倚天屠龙记》]</ref>。

本剧于2002年7月开始在[[北京|北京]][[怀柔|怀柔]]拍摄，于12月26日在北京举行了小型记者試片會。制作方投资达2000万元，匯集两岸三地一流的演员阵容，正当红的[[苏有朋|苏有朋]]和[[贾静雯|贾静雯]]擔綱飾演张无忌与赵敏，新人[[高圆圆|高圆圆]]則飾演[[周芷若|周芷若]]，其他像[[张国立|张国立]]、[[张铁林|张铁林]]、[[王刚_(演员)|王刚]]、[[陶虹|陶虹]]、[[郭妃丽|郭妃丽]]、[[释小龙|释小龙]]、[[梁天|梁天]]等演員都是一時之選，外景地则选在北京的各大郊区。<ref>[http://tjtv.enorth.com.cn/system/2003/07/21/000600766.shtml 群星云集写“情义” 苏有朋亮出“屠龙刀”]</ref>

本片由[[亞洲電視|亞洲電視]]在[[香港|香港]]播放。

== 特色 ==
本剧制片人边晓军介绍，这个版本的《倚天屠龙记》比较注重感情戏。尤其是对[[周芷若|周芷若]]前后巨大的轉變将做更細緻的处理<ref>[http://enjoy.eastday.com/epublish/gb/paper235/92/class023500001/hwz1179857.htm 金庸剧明年混战荧屏 《倚天屠龙记》率先亮武器]</ref>。此外本剧美女阵容鼎盛，飾演上一代的有[[郭妃丽|郭妃丽]]與[[阎青妤|阎青妤]]，新一代的有[[贾静雯|贾静雯]]、[[高圆圆|高圆圆]]、[[陈秀丽|陈秀丽]]、[[陈莎莎|陈莎莎]]，這六美各具特色，而後四位演员在气质上走的是青春少女路线，服装造型皆是以丝质、纱质为主。本剧的武打特技也较为出色，由[[马玉成|马玉成]]担任武术指导，使用了大量特技动作。<ref>[http://enjoy.eastday.com/epublish/gb/paper235/92/class023500003/hwz1180022.htm 三大亮点两大靓点-新版《倚天屠龙记》全面解析]</ref>

虽然本剧在主要演员的演绎、武打、化妆等方面都很出色，但由于过于偏重感情戏，小说中各大门派之间的纷争和明教反抗元朝政府的矛盾冲突被转化为张无忌和四女之间的情感纠葛，国-{仇}-家恨让位给了流行的武打動作與愛情對白，这也是本剧被人诟病之处。另外[[张铁林|张铁林]]扮演的[[杨逍|杨逍]]则被認為不符合原著人物的气质，和[[陶虹|陶虹]]的感情对手戏也被讥笑为老牛吃嫩草。<ref>[http://enjoy.eastday.com/epublish/gb/paper235/92/class023500003/hwz1180020.htm 评论：新版《倚天屠龙记》之白璧微瑕]</ref>

== 播出争议 ==
亚环影音在拍摄初期和中華民國[[中華電視公司|中華電視公司]]签订了播出协议，但在本剧还未被中共[[国家广电总局|国家广电总局]]审核，[[中華電視公司|华视]]便已拿到前二十集的样带，并在[[台湾|台湾]]播出了前十集，对于香港剧组未经审批即釋出本剧，相关人士表示可能会受到中共广电部门的处罚<ref>[http://enjoy.eastday.com/epublish/gb/paper235/92/class023500001/hwz1179869.htm 《倚天屠龙记》擅自提前抢播 审核期间亮红灯]</ref>。2003年1月13日，广电部门最终决定本剧发行方亚环影音将不得再向[[中華電視公司|华视]]提供样带，而亚环影音则在审批通过后将在[[中國大陸|中國大陸]]各地陆续发行播出，[[中華電視公司|华视]]也调整了播出时间<ref>[http://enjoy.eastday.com/epublish/gb/paper235/92/class023500001/hwz1179872.htm 《倚天屠龙记》被从轻发落 3月前后将在天津播出]</ref>。

== 收视 ==
[[AC尼尔森|AC尼尔森]]的收视数据显示，本剧在福建的平均收视率为4.6%，在武汉的为5.9%，均胜过《[[射鵰英雄傳_(2003年電視劇)|大陆版射雕英雄传]]》。<ref>[http://ent.163.com/edit/030513/030513_164981.html 《倚天屠龙记》收视力压《射雕》]</ref>在[[江苏|江苏]]、[[深圳|深圳]]等地，最高收视率曾达到26%，成为2003年度最受瞩目的电视剧之一<ref>[http://ent.sina.com.cn/v/2003-03-07/1800136385.html 苏有朋携两美女京城谈“倚天” 新浪网上征问题]</ref>。

[[中華電視公司|华视]]在台湾复播此剧后，据[[AC尼尔森|AC尼尔森]]的调查结果，1月20日收视率一度高达6.34%，跃居全台第一<ref>[http://enjoy.eastday.com/epublish/gb/paper235/92/class023500001/hwz1179876.htm “倚天”收视率大好 苏有朋会否实现裸奔诺言？]</ref>。
