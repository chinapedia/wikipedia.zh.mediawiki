'''麴町警察署'''（{{jpn|j=麹町警察署|hg=こうじまちけいさつしょ|rm=Kōjimachi Keisatsusyo}}）位於[[日本|日本]][[東京都|東京都]][[千代田區|千代田區]]，是[[警視廳|警視廳]]隸下的一座[[警察局|警察局]]，開設於1881年，為日本創立時間最早的警署<ref name="千代田區觀光協會">{{Cite web |url=http://www.kanko-chiyoda.jp/tabid/2076/Default.aspx |script-title=ja:第34回　麹町1丁目その2 |publisher={{lang|ja|千代田区観光協会}} |accessdate=2014-05-07 |archiveurl=http://archive.is/qC3G2 |archivedate=2017-10-21}}</ref>，其轄區緊鄰著[[皇居|皇居]]西側，是日本[[日本政治|政治]]、社會及[[日本文化|文化]]中樞地帶，其內涵蓋了[[日本政府|日本政府]]眾多[[日本行政機關|機關部門]]、[[總理大臣官邸|首相官邸]]和數座[[外國駐日本機構列表|外國駐日使館]]，勤務範圍也包含這些官廳建築周邊的維安<ref name="警察署">{{Cite web|url=http://www.keishicho.metro.tokyo.jp/1/kojimachi/index.htm|script-title=ja:麹町警察署|publisher={{lang|ja|麹町警察署}}|accessdate=2014-05-07|deadurl=yes|archiveurl=http://archive.is/ETPji|archivedate=2014-05-08}}</ref>。由於麴町警察署屬於大規模警署，因此署長的[[警銜|階級]]為[[警視正|警視正]]，高於擔任一般中、小規模警署署長的[[警視|警視]]<ref>{{Cite web |url=http://report.penghu.gov.tw/OpenFront/report/show_file.jsp?sysId=C09501513&fileNo=1 |title=日本觀光警政與旅遊安全考察（第二梯次） |accessdate=2014-05-07 |date=2006-08-23 |format=[[Microsoft_Word|Microsoft Word]] |publisher=[[澎湖縣|澎湖縣]]警察局 |page=p7 |archiveurl=http://archive.is/QoRT1 |archivedate=2019-04-08}}</ref>。

== 歷史 ==
1874年1月15日，警視廳設立，以「大區小區制」將[[東京府|東京府]]分為6個大區，每個大區內又分為16個小區、設置16個「巡查屯所」，負責治安的維持<ref>{{Cite web |url=http://www.bosaijoho.jp/reading/item_6149.html |publisher={{lang|ja|{{Link-ja|防災情報新聞|防災情報新聞}}}} |script-title=ja:2012年1月の周年災害／日本の災害・防災年表(「周年災害」リンク集) |date=2012-01-05 |accessdate=2014-05-07 |archiveurl=http://archive.is/myomr |archivedate=2014-05-07}}</ref>，當中第三大區的廳舍便於同年設置在麴町五丁目，成了該町第一個警務機構。1881年1月14日，警視廳在[[半藏門|半藏門]]外新設了麴町警察署，同時也成立麴町巡查屯所{{ref label|屯所|註1|}}。1883年10月5日，麴町署的辦公地點從原本的第三大區廳舍移動到剛完工的廳所內，不過新署僅是一棟瓦葺式的木造平屋，內部除了警員辦公室外，還有擊劍道場、等候室、電報室<ref name="警察署"></ref>。

1903年6月19日，麴町警察署正式遷至麴町一丁目的現址辦公。1923年9月1日發生了[[關東大地震|關東大地震]]後，整座警署毀於祝融之災，麴町警署被迫將業務移轉到臨時警局處理，1926年3月30日，才在原地建好兩層高的鋼筋混凝土建築<ref name="警察署"></ref>。1936年的[[二二六事件|二二六兵變]]中，[[日本陸軍|日本陸軍]]內部派系「[[皇道派|皇道派]]」的一批官兵佔領了警視廳本部，其機能一度交由麴町警署代行，當時[[昭和天皇|昭和天皇]]還親自聯絡麴町警署的署長以掌握事態發展<ref name="千代田區觀光協會"></ref>。

1945年5月25日，[[美軍|美軍]][[美國陸軍航空軍|航空部隊]]對東京實施[[東京大轟炸|第二次大轟炸]]，大火再度襲捲了警署，把它燒到只剩下外牆。然而這道外牆在建築修復了之後，仍繼續使用了31年，直到1976年7月為止。從那個月起，建築結構日趨老化的警署開始進行重建，預計要改造成一座地上十層、地下一層的新警署。在施工期間，[[最高裁判所|最高裁判所]]（最高法院）出借了其設在麴町西南方、位於[[紀尾井町|紀尾井町]]的司法修習所分室，以供該署署員上班。1978年6月19日，新警署大樓竣工，麴町署才從紀尾井町遷回<ref name="警察署"></ref>。

== 轄區 ==
麴町警察署的轄區為千代田區西部地帶（舊[[麴町區|麴町區]]西部）。轄區界線由東起順時針依序為[[皇居|皇居]][[護城河|護城河]]（半藏濠、櫻田濠）、[[國道1號_(日本)|櫻田通]]、{{Link-ja|東京都道405號外濠環狀線|東京都道405号外濠環状線|外堀通}}、[[JR|JR]][[中央本線|中央本線]]、{{Link-ja|首都高5號池袋線|首都高速5号池袋線}}、[[首都高都心環狀線|首都高都心環狀線]]<ref name="交番地圖">{{Cite web |url=http://www.keishicho.metro.tokyo.jp/1/kojimachi/koban/koban.htm |script-title=ja:交番マップ |publisher={{lang|ja|麹町警察署}} |accessdate=2014-05-07 |archive-url=http://archive.fo/zl1In |archive-date=2014-05-08 |dead-url=yes }}</ref>。

轄區與本署負責範圍接壤的警署，除了同在千代田區內的{{Link-ja|丸之內警察署|丸の内警察署}}、[[神田警察署|神田警察署]]外，還包括南邊[[港區_(東京都)|港區]]內的{{Link-ja|赤坂警察署|赤坂警察署}}、西邊[[新宿區|新宿區]]內的{{Link-ja|牛込警察署|牛込警察署}}和{{Link-ja|四谷警察署|四谷警察署}}，以及北邊[[文京區|文京區]]內的{{Link-ja|富坂警察署|富坂警察署}}<ref>{{Cite web|script-title=ja:警察署検索 - 地図から|url=http://www.keishicho.metro.tokyo.jp/kankatu/psmap/psmap3.htm|publisher={{lang|ja|[[警視庁|警視庁]]}}|accessdate=2014-05-07|deadurl=yes|archiveurl=http://archive.fo/GEENe|archivedate=2014-04-27}}</ref>。
