'''麴'''<ref>又写作-{麯、麹}-</ref>，又称'''麴蘖'''，酿酒中称'''酒母'''，是[[稻|米]]、[[糯米|糯米]]、[[小麦|小麦]]、[[大麦|大麦]]、[[裸麦|黑麦]]、[[燕麦|燕麦]]、[[豆类|豆类]]等[[粮食|粮食]]作物，及其外皮碾磨而成的白色粉末[[米糠|米糠]]或[[麦麸|麦麸]]受到[[麴霉|麴霉]]菌等[[微生物|微生物]]感染，经[[醱酵|醱酵]]使微生物有效繁殖而得到的产品，广泛应用於[[白酒|白酒]]、[[黄酒|黄酒]]、[[清酒|清酒]]、[[醋|醋]]、[[酱油|酱油]]、[[甜面酱|甜面酱]]、湿仓[[普洱茶|普洱茶]]<ref>《[http://www.puertea8.com/html/192.html 什么是普洱茶的干仓和湿仓，如何分辨？] {{webarchive|url=https://web.archive.org/web/20090106054115/http://www.puertea8.com/html/192.html |date=2009-01-06 }}》，普洱茶吧</ref>、[[味噌|味噌]]、[[泡盛|泡盛]]和[[醪糟|醪糟]]等发酵食品中，是[[东亚|东亚]]、[[东南亚|东南亚]]及[[喜马拉雅山脉|喜马拉雅]]地区特有的醱酵技術。

== 历史 ==
目前已知的最早记载麴的文字是《[[尚书_(经)|尚书]]·说命下》中的“若作酒醴，爾惟麴糵；若作和羹，爾惟鹽梅。”其中曲蘖指的是酒曲，“梅”指的是醋。[[北魏|北魏]]时代，[[贾思勰|贾思勰]]的《[[齐民要术|齐民要术]]》中首次全面总结了酿酒和酿醋的技术。到了宋代，制麴技术已经达到相当完善的地步，酿酒业繁荣发展。现代，人们利用[[生物工程|生物工程]]和[[发酵工程|发酵工程]]使麴在酦酵中更广泛的应用，麴酦酵技术更加系统化、科学化。<ref name="jn">[http://www.sytu.edu.cn/zhgjiu/j2-2.htm 酒麴的种类]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}，《中国酒大观目录》，江南大学网</ref>

== 分类 ==
麴有很多种类，主要包括[[大麴|大麴]]、[[小麴|小麴]]（[[根霉|根霉]]麴、饼曲）、[[红麴|红麴]]、米麴（散曲）、[[麦麴|麦麴]]、[[麸麴|麸麴]]、黑麴、黄麴、豆麴等，其中大麴、小麴、红麴、麦麴和麸麴是酿酒使用的5种主要酒麴。<ref name="jn"/>红麴是一种健康食品，广泛用于制酒、食品上色、烹调<ref>[http://keko.eat.51ttyy.com/material/material_1135230330000_1088.html 红麴] {{webarchive|url=https://web.archive.org/web/20081013100437/http://keko.eat.51ttyy.com/material/material_1135230330000_1088.html |date=2008-10-13 }}，天天营养网</ref>和中药<ref>[http://www.shiliao.com.cn/2005/4-8/15500965296.html 红麴]，中国食疗网</ref>；黑麴用于制醋和生产[[柠檬酸|柠檬酸]]；小麴用于制[[醪糟|醪糟]]；米麴用于制[[黄酱|黄酱]]和[[甜面酱|甜面酱]]；大豆曲可制[[酱油|酱油]]。日常生活中谷物发霉而生长的黄麴中含有致[[癌|癌]]物质[[黄麴毒素|黄麴毒素]]，一定要避免食用。不过现在已有专门用于发酵工程的[[黄麴霉菌|黄麴霉菌]]，是经过筛选的无害菌种，可以放心食用。<ref>《[http://www.xugu.net/wst/P97241.htm 酱油、黄酱中有黄麴霉毒素吗？] {{Wayback|url=http://www.xugu.net/wst/P97241.htm |date=20060530004535 }}》，中国幼教网</ref>

传统上，麴按原料、处理方法、形态、添加物和微生物来源分类，各个分类法互有交叉，较为混乱。酿酒时，曲霉和[[酵母|酵母]]通常一起使用。

== 制作方法 ==
麴以不同用途的[[麴菌|麴菌]][[孢子|孢子]]作[[種麴|種麴]]，用米粉、蒸熟的米饭或熟大豆等作原料醱酵製成，以酒麴為例。传统酒麴的制法是天然接种微生物，因此有可能多种麴菌混杂在一起，比较有名的种类包括乌衣红麴和黄衣红麴，其中乌衣红麴是由[[红麴霉菌|红麴霉菌]]、[[黑麴霉菌|黑麴霉菌]]和[[酵母|酵母]][[共生|共生]]而制成。<ref>《[http://www.foodmate.net/topic/165/35058.html 红麴生产技术]》，食品伙伴网</ref>

现代酿酒工业使用的麴一般是纯种酒麴，利用[[生物工程|生物工程]]技术，筛选出所需的高品质微生物进行醱酵，用于批量生产。<ref name="jn"/>

== 使用方法 ==
# 用作发酵原料，可制作[[醪糟|醪糟]]和[[味醂|味醂]]，发酵时要保证微生物活性。
# 制作用于[[接种|接种]]的[[種麴|種麴]]。
# 用于[[发酵工程|发酵工程]]。通常在发酵过程中加入水和盐，微生物会大量繁殖，产生各种[[酶|酶]]，约20天后停止发酵，高温灭菌<ref>《[http://www.39kf.com/cooperate/tu/Food-Gallery/2007-02-08-324325.shtml 酱油]》，医源世界网</ref>，获得微生物的[[代谢|代谢]]产物。黄酒、清酒、烧酒、酱油和黄酱一般采用此法。
# 日本[[味噌|味噌]]制作兼有法1和法3的步骤，[[中部地方|中部地方]]的豆味噌制法同法3，而日本其他地区的味噌用大豆作主料，一般用法1。[[近畿地方|近畿地方]]的白味噌和[[九州_(日本)|九州]]的麦味噌分别含有比例较高的米麴和麦麴，这两种麴是法3的重要原料，可以使味噌颜色更深。

== 所含酶 ==
麴是[[菌丝|菌丝]]密集似绒毯的[[麴霉菌|麴霉菌]]感染谷物而得到的物质，因此在生长过程中会分泌出各种[[酶|酶]]，可以高效率的催化产生[[淀粉|淀粉]]、[[蛋白质|蛋白质]]、[[脂肪|脂肪]]等[[低分子|低分子]]物质。

=== 淀粉酶 ===
{{seealso|淀粉酶}}
淀粉是[[葡萄糖|葡萄糖]]以α-1，4-[[糖苷键|糖苷键]]和α-1，6糖苷键聚合生成的一类多糖，麴霉菌产生的淀粉酶可水解淀粉，使麴有甜味。

[[酵母|酵母]]可将[[多糖|多糖]]转化生成[[酒精|酒精]]，麴霉菌可使淀粉水解生成葡萄糖和[[麦芽糖|麦芽糖]]（麦芽糖是2分子葡萄糖以α-1，4-[[糖苷键|糖苷键]]结合而成的[[二糖|二糖]]），这两种[[真菌|真菌]]是[[酿酒|酿酒]]中主要使用的微生物。

麴霉菌会大量向菌体外分泌[[α-淀粉酶|α-淀粉酶]]、[[β-淀粉酶|β-淀粉酶]]、[[α-葡萄糖苷酶|α-葡萄糖苷酶]]和[[葡萄糖淀粉酶|葡萄糖淀粉酶]]（γ-淀粉酶），可以使[[稻|米]]和麦所含淀粉水解。

* α-淀粉酶：能水解淀粉中任意α-1，4-[[糖苷键|糖苷键]]，终产物是麦芽糖，但不能水解α-1，6糖苷键。
* β-淀粉酶：能作用於淀粉的非还原性末端，依次以麦芽糖为单位切割相隔的α-1，4-[[糖苷键|糖苷键]]，水解生成β-麦芽糖<ref>《[http://scholar.ilib.cn/Abstract.aspx?A=hdyxgdzkxxxb200103038 诺卡氏菌β-淀粉酶的纯化及一般性质]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}》，赵朝贤、刘春燕、赵晓唐、贾丽霞</ref>，但不能水解α-1，6糖苷键。
* 葡萄糖淀粉酶：能作用於淀粉的非还原性末端，以葡萄糖为单位水解淀粉。可同时作用于α-1，4-糖苷键和α-1，6糖苷键，因此从[[底物|底物]]中不能得到分子量相对小些的淀粉分子。
* α-葡萄糖苷酶：水解麦芽糖，生成葡萄糖。

=== 蛋白酶 ===
{{seealso|蛋白酶}}
蛋白质是由多个[[氨基酸|氨基酸]]脱水缩合而成的多[[肽|肽]]链盘麴折叠而成，而麴霉中的蛋白酶可以使粮食中的蛋白质水解成氨基酸和短肽可以增加食物的鲜味。

=== 脂肪酶 ===
{{seealso|脂肪酶}}
脂肪是由[[甘油|甘油]]和[[脂肪酸|脂肪酸]][[脱水|脱水]]生成的[[脂类|脂类]]物质，用于制醋的黑麴霉中含有大量的脂肪酶。
