'''王俊生'''{{BD|1948年||||W}}是一名中国体育界、足球界的知名人物，曾经担任[[中国足协|中国足协]]专职副主席，是中国足球职业化的重要奠基人。

==简介==
王俊生年轻时为足球守门员，不到16岁就被选入[[北京国安|北京队]]，后来成为[[北京国安|北京队]]主力门将，1973年，时任[[中国国家足球队|国家队]]主教练[[年维泗|年维泗]]曾将其选进[[中国国家足球队|国家队]]。1981年退役后，王俊生进入[[北京体育学院|北京体育学院]]学习，获得大专学历，之后进入北京市体委工作。<ref>{{cite web | title = 王阎谢三大掌门“血统论” 聪明的脑袋不长毛 
 | | publisher = 南方网| date = 2005年2月18日 | url = http://www.southcn.com/sports/scroll/200502181006.htm | accessdate = 2009-02-27 }}</ref>1991年进入中国足协工作后，王俊生于1992年至2000年出任[[中国足协|足球运动管理中心]]主任、党委书记和[[中国足协|中国足协]]专职副主席、秘书长<ref>{{cite web | title = 王俊生简介 | | publisher = cctv.com| date = 2008年11月21日 | url = http://sports.cctv.com/special/C22821/20081121/108465.shtml | accessdate = 2009-01-22 }}</ref>。2002年7月，他离开[[中国足协|中国足协]]，出任[[中体产业|中体产业公司]]监事会主席。<ref>{{cite news | title = 图文-王俊生穿上守门员球衣 风采不减当年 | | publisher = 新浪| date = 2002年7月8日 | url = http://sports.sina.com.cn/c/2002-07-08/08283847.shtml| accessdate = 2009-02-27 }}</ref>

==足协工作成绩==
担任[[中国足协|中国足协]]专职副主席、[[国家足球运动管理中心|国家足球运动管理中心]]主任的八年时间是王俊生最受关注的一段时间，也是中国足球发生巨大变化的一段时间，在此期间中国足球进行了一系列全新的尝试。

1992年，[[中国足协|中国足协]]聘请德国人[[克劳斯·施拉普纳|克劳斯·施拉普纳]]担任[[中国国家足球队|国家队]]主教练。[[克劳斯·施拉普纳|施拉普纳]]由此成为[[中国国家足球队|中国国家足球队]]历史上第一个外国籍教练。<ref>{{cite web | title = 施拉普纳冒出水面 | | publisher = 南方网| date = 2002年11月27日 | url = http://www.southcn.com/sports/special/20021108/200211270797.htm | accessdate = 2009-02-27 }}</ref>

1993年，[[中国足协|中国足协]]经过挑选，派遣一批少年球员到[[巴西|巴西]]训练。由于受[[健力寳|健力寳]]贊助，冠名为[[健力寳青年隊|健力寳青年隊]]。後來，在1999年參加[[奧運會|奧運會]]預選賽的那一批國奧隊的很多隊員都來自[[健力寳青年隊|健力寳青年隊]]。

1993年底，[[中国足协|中国足协]]開始推广俱乐部制，1994年全新的[[中國足球甲級A組聯賽|全國足球甲級A組聯賽]]开战，成爲中國足球職業化的開端。
