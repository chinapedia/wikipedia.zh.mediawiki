{{中国伊斯兰教}}

'''王靜齋'''（{{bd|1880年|3月12日|1949年|5月25日|CatIdx=Wang}}）原名'''文清'''，经名赛尔德·伊利亚斯，[[回族|回族]]，[[天津|天津]]人，[[中华民国|中华民国]][[伊斯兰教|伊斯兰教]]大[[阿訇|阿訇]]、经学教育家，与[[马松亭|马松亭]]、[[哈德成|哈德成]]、[[达浦生|达浦生]]并称中国现代“四大阿訇”<ref name=seac>{{Cite web|url =http://www.seac.gov.cn/gjmw/zlk/2004-09-07/1165370090749670.htm|title =王静斋|author =|date =2004-09-07|publisher =中华人民共和国国家民族事务委员会|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20081202073820/http://www.seac.gov.cn/gjmw/zlk/2004-09-07/1165370090749670.htm|archivedate =2008年12月2日|df =}}</ref><ref name=ah>{{Cite web |url = http://www.chinaislam.net.cn/cms/zjjy/jsrw/201205/25-683.html|title = 中国现代伊斯兰教四大阿訇之王静斋 |author =  |date = 2012-05-25 |publisher = 中国伊斯兰教协会 |language =  |accessdate =  |quote =  }}</ref><ref name=guang>{{Cite web |url =http://www.norislam.com/?viewnews-11105  |title = 王静斋 |author =  |date = 2009-10-30 |publisher = 伊斯兰之光 |language =  |accessdate =  |quote =  }}</ref><ref name=nianpu>{{Cite web |url = http://www.norislam.com/?viewnews-11106 |title =  王静斋年谱|author =  |date = 2009-10-30 |publisher = 伊斯兰之光 |language =  |accessdate =  |quote =  }}</ref>。

== 生平 ==
[[清朝|清朝]][[光绪|光绪]]六年（1880年）农历二月初二日，王静斋生于[[天津|天津]]清真北大寺附近的一户[[阿訇|阿訇]]世家。8岁开始随父亲王兰庭阿訇学[[阿拉伯文|阿拉伯文]]。光绪二十年（1894年）起，进入经堂学习阿拉伯语课本及浅近教法学十多种，并涉猎[[波斯语|波斯语]]。不久，先后随[[李长贵|李长贵]]、[[马玉麟|马玉麟]]、[[于勉斋|于勉斋]]、[[金连荣|金连荣]]、[[刘绪魁|刘绪魁]]、[[海思福|海思福]]等阿訇学习<ref name=ah/><ref name=guang/>。1905年，再度投奔已离开天津清真北大寺到安育村清真寺任教长的于勉斋阿訇。1905年冬，于勉斋阿訇在安育村清真寺为王静斋赐幛穿衣。不久，王静斋受聘为北京[[大兴县|大兴县]]白塔村南寺阿訇，在任时间两年。1907年，王静斋改任[[怀来县|怀来县]]新保安清真寺阿訇。1908年重返白塔村南寺任教长。约1908年，北京《正宗爱国报》刊登了该报总经理[[丁宝臣|丁宝臣]]的署名文章《疑问求教》四则，王静斋见报后作答。不久，丁宝臣亲自来到天津，聘王静斋任北京[[花市清真寺|花市清真寺]]教长，在任时间两年。1910年，应聘出任奉天（今[[沈阳|沈阳]]）开源清真寺教长，在任时间三年，1913年辞职返回家乡<ref name=nianpu/>。

20世纪初，中国伊斯兰教和回族发生了许多变革。1906年，[[童琮|童琮]]在[[江苏|江苏]][[镇江|镇江]]创办东亚穆民教育总会。1907年，中国14个省的36位留日回族学生成立留东清真教育会，天津回民[[丁竹园|丁竹园]]出版《竹园白话报》，北京的[[丁宝臣|丁宝臣]]出版《正宗爱国报》。1908年，中国留日回族学生出版《醒回篇》。1907年至1908年，[[王宽|王宽]]阿訇在北京城内创办了回文师范学堂、京师第一两等小学堂。青年阿訇王静斋十分关注这些新变革，他在新保安时便订阅《竹园白话报》、《正宗爱国报》，重返白塔村任职后又订阅《[[大公报|大公报]]》、《民兴报》，在奉天任职时订阅《法政浅说》。此外他还读过《饮冰室》、《中国魂》等图书。在奉天时曾主动随耶稣教徒王某学[[英语|英语]]，但仅学40天便因清真寺部分穆斯林反对而停止。1912年，王静斋加入[[国民党_(宋教仁)|国民党]]。王静斋常为《正宗爱国报》、天津《民兴报》投稿<ref name=ah/><ref name=guang/>。

1914年，王静斋阿訇应[[山东省|山东省]]济南镇守史[[马良_(民国)|马良]]的邀请，自天津到山东省[[潍县|潍县]]谈《古兰经》翻译事宜。马良想请王静斋主译《古兰经》。王静斋则建议马良请[[河北|河北]][[沧州|沧州]]经学家[[海金|海金]]主译。经马良同意，王静斋亲自到沧州请来海金，后又请到[[山东|山东]][[冠县|冠县]]阿訇[[沙锦章|沙锦章]]（人称“傲沙”）加入。经过两年完成经堂语初稿，后又请[[沧县|沧县]][[李正芳|李正芳]]（即李五阿訇）修改。该译本完稿后存在马良处，从未发表。王静斋在帮助马良组织起上述译经人员后，未参加译经，而是在1914年受聘出任[[潍县|潍县]]宋家庄清真寺教长，并开始翻译《回耶辨真》。同年，王静斋离开宋家庄，改任京东[[三河县|三河县]][[大厂回族自治县|大厂镇]]清真寺阿訇。1915年辞职回到家乡<ref name=nianpu/>。1915年，王静斋一度出任《北京新报》编辑，代[[张子文|张子文]]阿訇编辑该报的《雷门鼓》、《回教史》这两个栏目<ref name=nianpu/><ref name=ah/><ref name=guang/>。1916年至1918年，任北京[[什刹海清真寺|什刹海清真寺]]阿訇。1919年，任北京[[崇文门|崇文门]]外[[上唐刀清真寺|唐刀胡同清真寺]]阿訇，未任满40天即自行辞职。1920年，任京南[[安次县|安次县]]安育村清真寺阿訇，在任时间两年<ref name=nianpu/>。

1922年到1923年，王静斋“耻于应具之知识落后，当为之善功有缺”，经[[太原|太原]][[马君图|马君图]]、[[南京|南京]][[马榕轩|马榕轩]]、[[甘肃|甘肃]][[马乾三|马乾三]]、[[天津|天津]][[杨小廷|杨小廷]]等人资助，王静斋偕徒弟[[马宏道|马宏道]]自[[天津|天津]]启程，经[[南京|南京]]、[[上海|上海]]、[[香港|香港]]、[[新加坡|新加坡]]、[[印度|印度]][[马德拉斯|麻达拉司]]、[[孟买|孟买]]、[[苏伊士运河|苏彝士]]到达[[埃及|埃及]][[开罗|开罗]]，进埃及[[爱资哈尔大学|爱资哈尔大学]]进修。1923年秋，和马宏道到[[麦加朝觐|麦加朝觐]]，朝觐期间因为麦加地方政府不准中国哈吉在帐棚的顶上挂[[中华民国国旗|中华民国国旗]]，中国哈吉“反抗无术，乃忍辱撤下”，王静斋受到极大刺激，强烈感觉到“弱国无外交”。之后，王静斋和马宏道到[[沙特阿拉伯|沙特阿拉伯]]各地以及[[土耳其|土耳其]][[君士坦丁堡|君士坦丁堡]]、[[安卡拉|安卡拉]]等地游历<ref name=ah/><ref name=guang/><ref name=nianpu/>。马宏道留在土耳其学习，王静斋独自返回埃及，再度进入爱资哈尔大学学习，并被该校任命为中国学生部部长。王静斋是爱资哈尔大学中国学生部第一任部长（1931年12月后又有[[沙国珍|沙国珍]]（汝诚）、[[庞士谦|庞士谦]]（益吾）先后任中国学生部部长）<ref name=nianpu/>。王静斋在埃及受到伊斯兰教改良主义思潮影响，并搜集抄录600多种阿拉伯文经典<ref name=ah/><ref name=guang/><ref name=nianpu/>。

1924年春，王静斋由苏彝士登[[法国|法国]]轮船经[[西贡|西贡]]至上海回国。回国后，因不愿卷入济南新派和旧派的纷争，拒绝了山东济南北大寺的聘请，而是接受马君图聘请出任[[山西省教育厅|山西省教育厅]]谘议。1925年回天津后，开始在[[时子周|时子周]]帮助下翻译《[[古兰经|古兰经]]》，不久即在1925年初离开天津到[[山东省|山东省]][[阳信县|阳信县]]六营村清真寺任教长。同年，在[[马松亭|马松亭]]等人创办的[[成达师范学校|成达师范学校]]兼任教职。1925年秋，王静斋到北京[[西单清真寺|西单牌楼清真寺]]开始翻译《古兰经》甲种本的工作<ref name=nianpu/>。1927年，王静斋、时子周、[[杨敬修|杨敬修]]等阿訇在天津筹办中阿大学，但因困难而未能办成<ref name=seac/>。

1927年9月，在弟弟王济民、儿子王宝琮的帮助下，王静斋创办《伊光》月报，自任总经理兼编辑、记者，[[尹伯清|尹伯清]]、[[陈鹭洲|陈鹭洲]]、[[王辉庭|王辉庭]]、[[张石麟|张石麟]]等人先后任编辑、会计、发行等职务，社址设在天津清真北大寺。《伊光》自创刊至1939年2月止，出刊达12年，每期印刷一两千份，读者遍及全中国，全部为免费赠阅。[[抗日战争|抗日战争]]期间，《伊光》社址经常改变，王静斋到哪里就将社址设在哪里<ref name=ah/><ref name=guang/>。

1928年，王静斋任奉天省安东县（今[[辽宁|辽宁]][[丹东|丹东]]）东寺教长。1929年，改任[[哈尔滨|哈尔滨]]清真东寺教长，在任时间三年。1932年，任天津三义庄清真寺教长。1933年，王静斋在三义庄清真寺倡导遵经革俗，引起天津部分穆斯林不满。1933年农历闰五月，王静斋的儿子王宝琮病逝<ref name=nianpu/>。

1934年，王静斋任北京[[教子胡同清真寺|教子胡同清真寺]]教长，[[萧德珍|萧德珍]]阿訇接任三义庄清真寺教长。1934年秋，海宏鋆、[[海朝英|海朝英]]父子率天津部分穆斯林发动“天津市驱逐新行运动”，称王静斋、萧德珍等人是“猴都斯”（新行），发表《天津市驱逐新行运动第一次报告书》，采取武力占领了三义庄清真寺，将萧德珍阿訇等提倡遵经革俗者逐出了三义庄清真寺。王静斋得知后，连续发表《新旧与是非辨》、《海氏父子听着》等文章，萧德珍也先后发表第一、二、三号《天津三义庄清真寺教长萧德珍宣言》。王静斋具呈天津检察处，控告海宏鋆、海朝英父子“纠众作乱，侵占寺权罪”。海氏最终被判处有期徒刑两个月<ref name=nianpu/>。

1935年，王静斋辞去教子胡同清真寺教职，应山东第一路总指挥、青州人[[赵明远|赵明远]]的推荐，到青州任教长，管辖青州城内、外两座清真寺教务。1935年，有人向[[中国国民党中央党部|中国国民党中央党部]]报告称王静斋和[[日本|日本]]人[[川村狂堂|川村狂堂]]勾结，煽动华北回民异动。“中央党部疑信参半，乃密令全国大小机关，随时侦我行动。余得此消息后，亲赴南京中央党部自首。结果，得当事者充分谅解，通知全国各机关取消前令。于是，仇我者未得如愿以偿。”同年，王静斋辞职，从青州回到天津。行前，赵明远约请王静斋专任翻译伊斯兰教经典的工作，并嘱王静斋在北京购房作为常住地。1936年，王静斋在北京[[西单牌楼|西单牌楼]][[中国回教俱进会|中国回教俱进会]]总部成立“中国回教典籍编译社”<ref name=nianpu/>。1937年，王静斋在[[河南|河南]]与时子周创办“中国回民抗日救国协会”<ref name=seac/>。

王静斋对《古兰经》经注学、圣训学、教义学、教法学、伊斯兰教历史都有深入研究，通晓阿拉伯语、波斯语、[[英语|英语]]、[[古汉语|古汉语]]。王静斋历任[[河北|河北]]、北京、[[辽宁|辽宁]]、[[黑龙江|黑龙江]]、天津、[[山东|山东]]、宁夏、[[台北|台北]]等地二十多所[[清真寺|清真寺]]的教长<ref name=ah/><ref name=guang/>。

1914年担任山东宋家庄清真寺教长时，王静斋用5个月翻译了[[印度|印度]]伊斯兰教教长赖哈麦图拉的《伊祝哈鲁络汗格》（中文名《回耶辨真》），由北京[[牛街|牛街]]清真书报社出版。1928年，王静斋出版了自己编译的《中亚字典》（俗称“王氏小字典”）。1931年和1935年，王静斋先后出版《选译详解伟嘎业》第1、第2集。1934年，王静斋在北京出版《中阿双解中阿新字典》（俗称“王氏大字典”）。1936年，在北京西单牌楼中国回教俱进会总部，王静斋成立“中国回教典籍编译社”，拓《欧母代序文》（回教法学通论），并且修润[[赵明远|赵明远]]的母亲丁蕴辉的[[波斯文|波斯文]]遗稿《老欧母代》，均出版。同年，王静斋翻译《回教亲属篇》、《回教遗产继承法》。1947年，翻译波斯诗人[[萨迪|萨迪]]的《古洛斯汤》为中文版《真境花园》，1947年4月由北京清真书报社出版<ref name=ah/><ref name=guang/>。

王静斋的三部《古兰经译解》十分有名。1925年，即王静斋归国后的第二年，王静斋便开始翻译《[[古兰经|古兰经]]》，4个月完成全部译稿，但是感到“译词欠妥”，仅节选出一本《赫提》出版。不久，接受北京[[赵文府|赵文府]]的建议及300元资助，在北京[[东四清真寺|东四清真寺]]南讲堂再度试译《古兰经》。经[[侯德山|侯德山]]（松泉）等人帮助下，经过20个月完成全部译稿，存在侯德山处。1932年，该《古兰经译解》（甲种本）由中国回教俱进会在北平出版，译者署名“王文清”，扉页有[[许宝蘅|许宝蘅]][[篆书|篆书]]题签，卷首有[[徐世昌|徐世昌]]、[[许世英|许世英]]分别作的两篇《序》以及侯德山写的《弁言》。1937年12月17日起，王静斋动手翻译经堂语体《古兰经》。1938年12月7日完成全经译稿。1940年该译稿被[[日军|日军]]飞机炸毁。1940年9月，王静斋迁居[[重庆|重庆]][[北碚|北碚]]云龙山庄，重新译经。1941年3月17日，第二次完成全经译稿。随即应邀到[[宁夏|宁夏]]整理该译稿，1942年8月整理完成，石印出60部“非定草”，此即《古兰经译解》（乙种本）。1938年12月1日，在《古兰经译解》（乙种本）译稿即将完成时，王静斋开始汉译丙种本（初名《白话汉译古兰天经》）。1945年夏，王静斋飞抵重庆，向[[时子周|时子周]]请教译稿中一些句子的译法。时子周熟读穆罕默德·阿力的英文译本，并以郁思甫·阿力的译本对照，不到两个月便解决了大部分疑难译句。1946年10月上旬，上海永祥印书馆出版《古兰经译解》（丙种本）5千部，封面由[[白崇禧|白崇禧]]题签<ref name=ah/><ref name=guang/>。

1948年，王静斋到中国西南各省考察伊斯兰教教育及学术。1948年下半年，应[[台湾|台湾]]穆斯林之邀，出任[[台北清真寺|台北清真寺]]阿訇。但因不习惯当地环境，于1949年春返回大陆。不久在路经[[贵阳|贵阳]]时病倒，1949年5月25日（农历四月二十八日）病逝，享年70岁。安葬在贵阳郊外的白桦山回民公墓<ref name=ah/><ref name=guang/>。

==著作==
王静斋撰写的部分重要文章有：<ref name=ah/><ref name=guang/>
*《五十年求学自述》
*《我之译经小史》
*《中国近代回教文化史料》
*《发扬伊斯兰文化之必要》
*《巡礼游记》
*《中国回教经堂教育的检讨》
*《中国回教掌故》
*《喂！果园马老阿衡逝世矣——一代贤豪，生来值得》

王静斋的主要译著有：
*《古兰经译解》
*《选译详解伟嘎业》
*《回耶辨真》
*《回耶雄辩录》
*《中亚字典》（俗称“王氏小字典”）
*《中阿双解中阿新字典》（俗称“王氏大字典”）
*《真境花园》
