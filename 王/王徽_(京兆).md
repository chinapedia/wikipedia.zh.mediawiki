'''王徽'''（{{bd|810年代||891年|catIdx=W}}），字'''昭文'''，封爵'''琅琊侯'''，谥号'''贞'''，[[唐朝|唐朝]][[官员|官员]]，[[唐僖宗|唐僖宗]]年间为[[宰相|宰相]]。

== 家世 ==
王徽生年不详，只知当[[唐宣宗|唐宣宗]]在850年代命宰相们为公主们挑选合适的驸马时，王徽四十多岁。<ref name=BT178>[[:s:舊唐書/卷178|《旧唐书》卷一百七十八]]</ref><ref name=ZZTJ249>[[:s:資治通鑑/卷249|《资治通鉴》卷二百四十九]]</ref>他是[[京兆府|京兆]][[杜陵县|杜陵]]人，[[战国|战国]]时期[[魏国_(战国)|魏国]]著名公子信陵君[[魏无忌|魏无忌]]之后，这一支因是王族之后，故改姓王。[[北周|北周]][[同州|同州]]刺史[[王羆|王羆]]为王徽十代祖，<ref name=NBT185>[[:s:新唐书/卷185|《新唐书》卷一百八十五]]</ref>其后人后来效力[[隋朝|隋朝]]和唐朝。王徽祖父[[王察|王察]]为[[连州_(隋朝)|连州]]刺史，父[[王自立|王自立]]为[[緱氏|緱氏]]令。<ref>《[[新唐书|新唐书]]》卷七十二{{cite web |url=http://www.sidneyluo.net/a/a17/072.htm |title=存档副本 |accessdate=2008-10-03 |0=2008-11-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20081120085821/http://www.sidneyluo.net/a/a17/072.htm |archivedate=2008-11-20 }}{{cite web |url=http://www.sidneyluo.net/a/a17/table/form56.htm |title=存档副本 |accessdate=2010-06-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100620180648/http://sidneyluo.net/a/a17/table/form56.htm |archivedate=2010-06-20 }}</ref>

== 早期仕途 ==
唐宣宗[[大中_(唐朝)|大中]]十一年（857年），王徽中[[进士|进士]]，任[[秘书省|秘书省]]校书郎。<ref name=BT178/>[[户部侍郎|户部侍郎]][[沈询|沈询]]判度支，辟王徽为巡官。<ref>尽管《新唐书》证实沈询曾判度支，但《旧唐书》《新唐书》的沈询传都没给出他判度支的时间，故王徽在他手下任职的时间也不可考。见[[:s:舊唐書/卷149|《旧唐书》卷一百四十九]]和[[:s:新唐書/卷132|《新唐书》卷一百三十二]]。</ref>当时宣宗正为女儿们挑选合适的驸马，要求宰相们观察中进士的官员，王徽被提名。但他不想与皇室结亲，见宰相[[刘瑑|刘瑑]]，以年龄为由辞让，因为他已经四十多岁且多病。刘瑑告诉了宣宗，王徽因而没有被命令娶公主。<ref name=BT178/><ref name=NBT185/><ref>因为刘瑑只在死前于858年短暂拜相，此事肯定发生于858年。见[[:s:舊唐書/卷177|《旧唐书》卷一百七十七]]。</ref>

宣宗子[[唐懿宗|唐懿宗]]年间，前宰相[[令狐绹|令狐绹]]历任宣武、淮南节度使，王徽都作为掌书记跟随他，官至大理评事。<ref>令狐绹于[[咸通|咸通]]二年（861年）被任为宣武军节度使，三年（862年）被任为淮南节度使，王徽为他效力大致在这段时间。见[[:s:舊唐書/卷172|《旧唐书》卷一百七十二]]。</ref>后又被召回拜为右拾遗，前后上疏提了23条建议，人们因而敬重他的洞察力。宰相[[徐商|徐商]]判盐铁事务，邀王徽为参佐。<ref>徐商于咸通六年（865年）至十年（869年）为相，判盐铁事务时更早，尽管他具体任此职及解职的时间不详，故王徽为他效力肯定在此期间。见《新唐书》卷六十三[http://www.sidneyluo.net/a/a17/table/form03.htm 63] {{webarchive|url=https://web.archive.org/web/20120518033947/http://www.sidneyluo.net/a/a17/table/form03.htm |date=2012-05-18 }}、[[:s:新唐書/卷113|《新唐书》卷一百一十三]]。《旧唐书·王徽传》暗示王徽效力徐商在宣宗年间，但徐商的仕途年表意味着这是不可能的。</ref>徐商罢相后为[[荆南节度使|荆南节度使]]，想邀王徽为他效力，但不知他是否愿意再次离开长安而不敢问。王徽闻知，对徐商说：“我刚中进士的时候，是公照顾我，现在公就要佩戴印绶领军了，下官怎能不跟从呢？”徐商很高兴，奏授王徽殿中侍御史，赐绯，署任为荆南节度判官。<ref name=BT178/><ref name=NBT185/>

== 唐僖宗年间 ==
[[高湜|高湜]]监督[[御史台|御史台]]时，奏王徽回长安为侍御史知杂兼职方员外郎，后升考功员外郎。<ref>《旧唐书》《新唐书》的高湜传都没称他曾任御史大夫，故此事不能确定时间，但根据王徽和萧仿之间的互动，可见此事在僖宗年间。见[[:s:舊唐書/卷168|《旧唐书》卷一百六十八]]和[[:s:新唐書/卷177|《新唐书》卷一百七十七]]。</ref>任上于咸通十四年（873年）奏请将考簿上的上中下字由红字改为用不易涂改的黑字书写，制止了考功作弊行为。<ref>《册府元龟》</ref>宰相[[蕭倣_(唐朝)|萧仿]]尤其敬重王徽精通为官之术。[[乾符|-{乾}-符]]初年，迁司封郎中、[[长安|长安]]县令。当时[[翰林学士|翰林学士]]缺人，萧仿任王徽为翰林学士，<ref name=NBT185/>改职方郎中、知制诰，三年（876年）九月正拜中书舍人。<ref name=BT19-2>[[:s:旧唐书/卷19下|《旧唐书》卷一十九下]]</ref>王徽于[[延英殿|延英殿]]中谢恩，被当面赐以金紫，后又迁户部侍郎、学士承旨，后又改兵部侍郎、尚书左丞，仍任学士承旨。<ref name=BT178/><ref>因萧仿于咸通十四年（873年）至乾符二年（875年）拜相，此事肯定在此期间发生。见《旧唐书》卷一百七十二。</ref>曾奉命授[[西川节度使|西川节度使]][[高骈|高骈]]功状，六年（879年）作《创筑罗城记》纪念高骈在成都筑大城事。<ref>[[:s:创筑罗城记|:s:创筑罗城记]]</ref>

[[广明_(唐朝)|广明]]元年（880年）十二月，[[黄巢|黄巢]]农民大军逼近长安，当权宦官[[田令孜|田令孜]]为转移对朝廷军队无力阻止黄巢的指责，归咎首席宰相[[卢携|卢携]]战略错误，贬之。卢携因而自杀。王徽和[[裴澈|裴澈]]被加[[同中书门下平章事|同中书门下平章事]]，代卢携为相，王徽改户部侍郎。<ref name=BT19-2/><ref name=NBT9>[[:s:新唐书/卷9|《新唐书》卷九]]</ref><ref>《册府元龟》作十一月事。</ref>但同日，传来黄巢军将攻长安的消息。田令孜带僖宗乘夜逃往[[成都|成都]]，王徽等官员们上朝时才知道，为了随驾和躲藏而乱窜。王徽想随驾逃跑，落入荆榛中，坠入崖谷受伤，被黄巢的士兵所获，带回长安。当时黄巢已在长安自称大齐皇帝。[[中和_(唐朝)|中和]]元年（881年）二月，黄巢想任王徽大齐官职，但即使周围都是白刃，王徽也假装重伤不能说话。黄巢命他回官邸养病，严密监视，派医生为他治病。过了一个多月，守卫懈怠，王徽装作商贩逃到河中。<ref name=BT178/>他撕裂绢布写上表文，遣人献上已在成都的僖宗，僖宗遥授他[[光禄大夫|光禄大夫]]，任为[[兵部尚书|兵部尚书]]。期间，王徽厚待退居河中的礼部郎中[[司空图|司空图]]。<ref name=BT190-2>[[:s:旧唐书/卷190下|《旧唐书》卷一百九十下]]</ref>王徽想去成都会合僖宗，但僖宗诏令他以本官充东面宣慰催阵使。<ref name=NBT185/><ref name=ZZTJ254>[[:s:資治通鑑/卷254|《资治通鉴》卷二百五十四]]</ref>

王徽有雅望，但拜相才一天就遭遇此乱，故没有可称道的建树。<ref name=NBT185/>

同时，昭义镇历经混乱：当年昭义将[[成麟|成麟]]兵变杀节度使[[高浔|高浔]]，又被天井关戍将[[孟方立|孟方立]]所杀。<ref name=ZZTJ254/>昭义人请监军[[吴全勖|吴全勖]]知留后，但孟方立拒绝，称宦官不能为节度使，软禁吴全勖，称希望朝廷派人来代任。三月，朝廷任时任右仆射、租庸使、京城四面催阵使、守兵部尚书王徽为检校左仆射兼潞州大都督府长史、昭义节度、潞邢洺磁观察等使、同平章事。<ref name=BT19-2/><ref name=NBT9/>王徽表司空图为副使。但王徽知道孟方立已经完全控制了昭义镇全部五州中在[[太行山|太行山]]以东的三州，自认不能实际控制昭义，上表坚辞获准，<ref>[[:s:辞泽州节度表|:s:辞泽州节度表]]</ref>请求任知昭义军事[[郑昌图|郑昌图]]为节度使，而司空图也因此没赴任。<ref name=BT190-2/>郑昌图仅三个月后即离任，使孟方立控制了昭义。<ref>[[:s:新唐书/卷187|《新唐书》卷一百八十七]]</ref><ref>[[:s:旧五代史/卷62|《旧五代史》卷六十二]]</ref><ref name=ZZTJ255>[[:s:資治通鑑/卷255|《资治通鉴》卷二百五十五]]</ref>王徽则被任为诸道租庸供军使、[[大明宫|大明宫]]留守、京畿安抚制置、修奉园陵等使。<ref name=BT178/><ref name=NBT185/>

宰相[[王鐸_(唐朝)|王铎]]都统行营兵马对抗黄巢，不能取胜。二年（882年）十月，行营都监[[杨复光|杨复光]]想请[[沙陀|沙陀]]叛军[[李克用|李克用]]加入唐军，王徽同意了。于是王铎以僖宗名义下敕书赦免李克用，召他加入唐军。唐军在李克用的巨大贡献下于中和三年（883年）初击败黄巢、收复长安，王徽因功加尚书右仆射。<ref name=BT178/><ref name=NBT185/>四月，僖宗诏留忠武军及延州、凤翔、博野军及东西神策军二万人于长安，令王徽护卫诸门，安抚居民，<ref>[[:s:新唐书/卷225下|《新唐书》卷二百二十五下]]</ref>与京畿制置使[[田从异|田从异]]守长安。<ref name=ZZTJ255/>

当时，长安因战乱而毁坏，僖宗尚未返回。王徽受任为[[大明宫|大明宫]]留守、京畿安抚制置、修奉园陵等使、[[御史大夫|御史大夫]]，任上表奏前[[庐州|庐州]]刺史[[郑綮|郑綮]]为兵部郎中、知台杂。<ref>[[:s:旧唐书/卷179|《旧唐书》卷一百七十九]]</ref><ref>[[:s:新唐书/卷183|《新唐书》卷一百八十三]]</ref>九月，权知京兆尹事。王徽安抚百姓，百姓开始返回，又修缮宫室，粗略整顿百司。<ref name=ZZTJ256>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref>时任淮南节度使的高骈命从事[[崔致远|崔致远]]给王徽写信。<ref>[[:s:租庸王徽相公|:s:租庸王徽相公]]</ref>四年（884年）十二月，王徽与留司百官上表请僖宗车驾还宫。<ref name=BT19-2/><ref>[[:s:请车驾还京表|:s:请车驾还京表]]</ref>但宫内外权臣重建官邸，常侵扰百姓，百姓抱怨，王徽试图以法律妥善对抗权臣，因而冒犯了他们，他们以其党羽[[薛杞|薛杞]]为少尹，知府事，希望薛杞剥夺王徽实权，但王徽称薛杞刚丧父，应服丧，拒绝让他就职。这愈发激怒了权臣，王徽因而被免职召去成都。他被授太子少师——当时并无太子，这完全是荣誉职位，并被命令在河中退休养病。大约一百天后，太子少师也被免，[[光启|光启]]元年（885年）僖宗回长安后复授太子少师，召回长安，他称病不就。宰相们责王徽不受皇恩，奏贬[[集州_(北周)|集州]]刺史，<ref name=NBT185/>命他立即赴任。<ref name=BT178/>

不数日，田令孜激怒了[[河中节度使|河中节度使]][[王重荣|王重荣]]和时任[[河东节度使|河东节度使]]李克用，他们攻打长安，僖宗逃到[[兴元|兴元]]。田令孜出逃前，称王重荣反了，下令火烧宫城，将宫室、房屋烧毁十之六七。乱军入长安后也放火，王徽勉强修好的宫阙又化为灰烬。<ref name=BT19-2/><ref>[[:s:新唐书/卷208|《新唐书》卷二百零八]]</ref>[[静难军节度使|静难军节度使]][[朱玫_(唐朝)|朱玫]]占领长安，立襄王[[李煴|李煴]]为帝。田令孜被解职，逃到成都任西川监军使，他的哥哥[[陈敬瑄|陈敬瑄]]在那里当节度使。田令孜被罢后，僖宗念及王徽无罪，诏王徽拜[[吏部尚书|吏部尚书]]，封琅邪郡侯，食邑千户。王徽想去见驾，但道路被李煴所阻。随后李煴的诏书到了河中，也召王徽。王徽称中风不能走路，拒绝归顺李煴。同年，朱玫被部将[[王行瑜|王行瑜]]所杀，李煴逃到河中也被王重荣处决后，<ref name=ZZTJ256/>僖宗回长安，途经凤翔时，任王徽为御史大夫，但王徽坚称脚有病不就。<ref name=BT178/><ref name=NBT185/>

== 唐昭宗年间 ==
[[文德_(唐朝)|文德]]元年（888年），唐僖宗崩，弟[[唐昭宗|唐昭宗]]继位，<ref name=ZZTJ257>[[:s:資治通鑑/卷257|《资治通鉴》卷二百五十七]]</ref>复授王徽太子少师。当他去便殿向昭宗致谢时，昭宗说：“王徽精神还好，怎能仅仅自便？”改授吏部尚书。尽管当时唐朝政府已数遭灾祸，王徽仍能筛选记录，适当处理公务，验看选官、补调官职等。进位检校[[司空|司空]]，守尚书右仆射。[[大顺_(唐朝)|大顺]]元年（890年）三月，[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]还上表称王徽等为缙绅名族，宜用为徐郓青兗等道节度使，昭宗从之，但没有任王徽为节度使。十二月卒，赠司空、太尉，谥'''贞'''。<ref name=BT178/><ref name=NBT185/><ref>[[:s:旧唐书/卷20上|《旧唐书》卷二十上]]</ref>
