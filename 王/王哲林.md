'''王哲林'''（{{bd|1994年|1月20日|catIdx=W王}}），出生於[[中華人民共和國|中國]][[福建省|福建省]][[福州市|福州市]]，現役[[中華人民共和國|中國]]篮球运动员，现效力於[[CBA|CBA]]聯盟的[[福建鲟浔兴篮球俱乐部|福建鲟浔兴]]。[[2016年NBA选秀|2016年NBA选秀]]中被[[孟菲斯灰熊队|孟菲斯灰熊队]]选中，和[[周琦_(篮球运动员)|周琦]]一起成为成为继[[宋濤_(籃球運動員)|宋涛]]、[[王治郅|王治郅]]、[[姚明|姚明]]、[[孟克·巴特爾|巴特尔]]、[[薛玉洋|薛玉洋]]、[[易建联|易建联]]、[[孙悦_(篮球运动员)|孙悦]]之后第八位被[[NBA|NBA]]选中的中国籍篮球运动员。

== 青年队时期 ==
王哲林生于福建福州，原籍[[福建省|福建省]][[三明市|三明市]][[沙县|沙县]]，父母分别是原福建男女篮的队员，因而有独天得厚的基因遗传。小学就读于[[晋安区第三中心小学|晋安区第三中心小学]]，五年级身高就达到了1.70米，14岁的王哲林的身高已经有2米出头，加盟了福建省青年队。<ref>[http://sports.163.com/12/0412/06/7USCC35C00052UUC.html 成长轨迹似姚明大郅阿联 王哲林一鸣惊人非偶然]2012-04-12</ref>

2010年首次入选[[中国国家男子青年篮球队|中国国青队]]，参加了在德国举行的[[2010年U17世界青年籃球錦標賽|2010年U17世界青年籃球錦標賽]]，获得了第七名的好成绩。虽然他是全队年龄最小，经验最浅的球员，但场均贡献5.8分4.9篮板，从替补变成了主力，被认为是一个潜力之星<ref>[http://sports.xinmin.cn/2010/07/11/5713223.html 世青赛中国铸就辉煌历史 超白金一代悄然崛起] {{webarchive|url=https://web.archive.org/web/20140202161749/http://sports.xinmin.cn/2010/07/11/5713223.html |date=2014-02-02 }}2010-07-11</ref>。9月他又跳级参加了在也门举行的[[2010年亚洲U18青年篮球锦标赛|2010年亚洲U18青年篮球锦标赛]]，帮助中国队九场全胜拿到冠军，是球队最大发现，场均得到14.3分8.6篮板，决赛对阵韩国更是拿下27分19篮板，成为队中仅次于[[郭艾伦|郭艾伦]]的二号人物<ref>[http://sports.cn.yahoo.com/ypen/20101002/38622.html 中国队员评分：郭艾伦最高分 王哲林成最大发现]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}2010年10月2日</ref>。

2011年6到7月，参加了在拉脱维亚举行的[[2011年U19世界青年籃球錦標賽|2011年U19世界青年籃球錦標賽]]，因“兵谏门”事件（13名球员集体写请愿信要求撤换主教练范斌）影响，球队表现不佳，最终只获得第13名。

2012年3月，王哲林首次入选了[[中国国家男子篮球队|中国国家队]]的21人集训名单，但未能进入参加伦敦奥运会的12人正式大名单。而在4月举行的2012年耐克篮球峰会，王哲林得到19分8篮板2盖帽，带领国际队84比75战胜美国队，惊艳全场，是耐克全球峰会17年历史上表现最出色的中国球员<ref>[http://sports.sina.com.cn/cba/2012-04-08/12406014399.shtml 17年全球峰会最强中国球员 2米13新星离NBA有多远]2012年04月08日新浪体育</ref>。

8月，作为适龄球员参加了在蒙古国举行的[[2012年亚洲U18青年篮球锦标赛|2012年亚洲U18青年篮球锦标赛]]，这也是王哲林第二次参加亚洲U18比赛，作为队中惟一入选过国家队的球员，场均得到22.3分10.3篮板，与韩国队的决赛中更是在球队落后13分的情况下挺身而出，得到30分13篮板并投中了扳平比分的3分球，球队以9连胜的成绩成功卫冕，他个人也当选为本届比赛MVP<ref>[http://sports.people.com.cn/n/2012/0830/c22176-18873925.html U18男篮亚青赛评奖结果揭晓 王哲林当选MVP]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}2012年08月30日</ref>。

9月，入选[[中国国家男子奥运篮球队|中国国奥队]]，参加了在日本举行的[[2012年亚洲杯篮球锦标赛|2012年亚洲杯篮球锦标赛]]，这是他第一次参加成人级别的比赛，场均得到14分10篮板，但命中率只有40%反映他还需要继续提高自己的对抗能力<ref>[http://sports.sina.com.cn/cba/2012-09-22/14016235134.shtml 国奥12人点评：王哲林需提高一点 后卫线有新发现]2012年09月22日</ref>。

== CBA时期 ==
2012年6月，王哲林与[[福建鲟浔兴篮球俱乐部|福建鲟浔兴]]正式签约<ref>[http://sports.163.com/12/0622/20/84KLEPFT00052UUC.html 福建男篮正式宣布签约王哲林 合同细节尚未透露]2012-06-22</ref>，将征战[[2012-13年中国男子篮球职业联赛|2012-13年中国男子篮球职业联赛]]。11月25日首场联赛对阵战青岛双星，拿下11分6篮板，12月7日客场对阵浙江稠州银行，得到31分15篮板。2013年2月1日，客场对吉林队的比赛中得到24分21篮板“双二十"数据并有6次助攻<ref>[http://sports.sina.com.cn/cba/2013-02-01/21216407151.shtml 王哲林砍24+21赵泰隆30分 福建客场送吉林三连败]2013年02月1日</ref>。

== NBA选秀以及NBA夏季联赛 ==
2016年，官方年紀已滿22歲的王哲林獲得[[NBA|NBA]]選秀資格，並於第57順位被[[孟菲斯灰熊|孟菲斯灰熊]]選中。

2018年年7月6日，由中国篮协对接NBA联盟组织了与2018NBA夏季联赛球队打的练习赛，王哲林跟随中国男篮红队参加了此次练习赛。在首场比赛当中，王哲林所在的红队以36：56不敌步行者。<ref>{{Cite web |url=https://www.zhiboba.com/article/show/2018_5b3f17cd040ea |title=红队36：56不敌步行者 |access-date=2018-07-06 |archive-url=https://web.archive.org/web/20180706163615/https://www.zhiboba.com/article/show/2018_5b3f17cd040ea |archive-date=2018-07-06 |dead-url=yes }}</ref>第二场比赛，王哲林所在的红队以66：78不敌火箭。<ref>[http://www.sohu.com/a/240003893_495962 红队不敌火箭]</ref>第三场比赛，王哲林拿了三分，红队大比分不敌国王。<ref>

== 評價 ==
*優點︰王哲林有著還算不錯的低位腳步和背打，他靠著這些技術能場均得20分左右。

*缺點︰王哲林沒有一個合格中鋒應有的身体條件，臂展只有2米08加上溜肩的先天性問題，導致他的摸高很差（只有2米64）；他的下肢也過於肥大，嚴重影響他的移動速度和彈跳，他甚至無法跟上比他更高的大個子，也沒有封蓋能力（最大摸高只有3米30，CBA場均蓋帽不足1個）。
他的低位單打在國際賽場經常被吹走步，而且頻繁的轉圈跟跳舞一樣，故被稱「舞王」
