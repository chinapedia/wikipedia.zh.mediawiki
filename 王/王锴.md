'''王锴'''（{{bd|？||？|catIdx=W}}），字'''鱣祥'''，[[五代十国|五代十国]]时期[[前蜀|前蜀]][[宰相|宰相]]。後跟隨蜀主[[王衍_(前蜀)|王衍]]投降[[後唐|後唐]]。

== 家世 ==
王锴生年和出生地都不详，[[唐朝|唐朝]]末年[[唐昭宗|唐昭宗]][[天复_(唐朝)|天复]]年间（901年-904年）作为其使节出使[[西川节度使|西川]]，从此留在当时由[[西川节度使|西川节度使]][[王建_(前蜀)|王建]]统治的西川。<ref name=SGCQ41>[http://archive.org/stream/06072241.cn#page/n160/mode/2up 《十国春秋》卷四十一]</ref>

== 前蜀年间 ==
[[天祐_(唐朝)|天祐]]四年（907年）唐朝灭亡、王建在自己控制的地盘上建立[[前蜀|蜀国]]（史学界称[[前蜀|前蜀]]）称帝即高祖皇帝后，<ref>[[:s:資治通鑑/卷266|《资治通鉴》卷二百六十六]]</ref>王锴官至[[翰林学士|翰林学士]]，<ref name=NHFD63>[[:s:新五代史/卷63|《新五代史》卷六十三]]</ref>又迁[[御史中丞|御史中丞]]。<ref name=SGCQ41/>曾任[[司空|司空]]，与[[御史大夫|御史大夫]][[冯涓|冯涓]]等小酌，王锴举行酒令，要求一字三呼、两物相似，说：“乐乐乐，冷淘似馎饦。”冯涓说：“己已巳，驴粪似马矢。”座上大笑，冯涓只是长啸。<ref>《十国春秋》卷四十冯涓传</ref>[[武成_(前蜀)|武成]]二年（909年）八月，王建任他为[[中书侍郎|中书侍郎]]，授[[同中书门下平章事|同中书门下平章事]]，拜为宰相。<ref>[[:s:資治通鑑/卷267|《资治通鉴》卷二百六十七]]</ref>[[永平_(前蜀)|永平]]元年（911年），王建建造新宫，于其中收集四部书，王锴上表称赞此事，历数历代帝王用不同手段收集和鼓励阅读书籍。<ref name=SGCQ41/><ref>[[:s:上蜀主奏记|:s:上蜀主奏记]]</ref>永平二年（912年）五月，王锴罢相任[[兵部尚书|兵部尚书]]，但次年四月复为宰相、中书侍郎。<ref>[[:s:資治通鑑/卷268|《资治通鉴》卷二百六十八]]</ref><ref>《十国春秋》卷三十六前蜀高祖本纪下</ref>

王锴为相时，曾为僧人贯休作像赞。<ref>《十国春秋》卷四十七僧贯休传</ref>

在王建之子[[王衍_(前蜀)|王衍]]统治期间，王锴和[[庾传素|庾传素]]同朝为相。王衍的狎客文思殿大学士[[韩昭|韩昭]]、内皇城使[[潘在迎|潘在迎]]、武勇军使[[顾在珣|顾在珣]]诱导王衍宴游无度，王锴、庾传素和大多数其他官员在改变王衍的行事方式上几乎无所作为。<ref>[[:s:資治通鑑/卷272|《资治通鉴》卷二百七十二]]</ref>[[乾德_(前蜀)|乾德]]元年（919年）八月，王衍东巡[[阆州|阆州]]，花费很大，群臣多直谏，当月王锴刚判六军诸卫事，没有为此说话，也没有为抗议而辞职。<ref name=SGCQ41/>国事日非，王锴、庾传素自保宠禄，无所规正。[[后唐|后唐]]的一场大规模入侵导致[[咸康_(王衍)|咸康]]元年（925年）王衍投降，前蜀灭亡。王衍命翰林学士[[李昊|李昊]]草降表给[[后唐庄宗|后唐庄宗]]，王锴草降书给后唐入蜀军队的名义总指挥、西川四面行营都统庄宗子魏王[[李继岌|李继岌]]。<ref name=ZZTJ274>[[:s:資治通鑑/卷274|《资治通鉴》卷二百七十四]]</ref><ref>[[:s:王衍降表|:s:王衍降表]]</ref>百官王锴等着衰绖，徒步赤脚抬棺号哭待命，李继岌、郭崇韬承制释罪，君臣东北向拜谢。<ref>《十国春秋》卷三十七前蜀后主本纪</ref>

== 前蜀灭亡后 ==
庄宗最初答应封王衍以荣衔，于是李继岌于[[同光|同光]]四年（926年）将王衍及大批前蜀官员及家属数千人送往都城[[洛阳|洛阳]]。<ref name=NHFD63/>但因军队不满庄宗没有给士卒足量奖赏及杀将领[[郭崇韬|郭崇韬]]、[[李继麟|李继麟]]，后唐政权开始被卷入多起叛乱，庄宗起初命王衍暂歇于[[长安|长安]]，后下令诛杀其一家。<ref name=ZZTJ274/>前蜀官员们奉令继续前往洛阳，但当他们到达时，庄宗已在洛阳死于一场兵变，由养兄[[后唐明宗|后唐明宗]]继位。明宗任很多原前蜀官员为各州府刺史、少尹、判官、司马，<ref>[[:s:資治通鑑/卷275|《资治通鉴》卷二百七十五]]</ref>王锴为首被任为州刺史，唯有原前蜀永平军节度使兼侍中[[马全|马全]]慨然说“国亡至此，生不如死！”绝食而死。<ref>《十国春秋》卷四十二马全传</ref>王锴和前蜀诸臣上表乞求安葬王衍骸骨，为人所称道。<ref name=SGCQ41/>明宗宣原蜀官员宰相王锴、[[张格|张格]]、庾传素及御史中丞[[牛希济|牛希济]]，各赐一韵，让他们试作蜀主降唐诗五十六字，王锴等都言及王衍僭号、荒淫、失国，只有牛希济不谤故主。<ref>《十国春秋》卷四十四牛希济传</ref>[[天成_(后唐)|天成]]三年（928年）秋七月，因王建族子陕州行军司马[[王宗寿|王宗寿]]请求，王衍得以以诸侯礼下葬。<ref>[[:s:資治通鑑/卷276|《资治通鉴》卷二百七十六]]</ref>

王锴此后的仕途和去世日期不详。王锴家藏异书数千本，亲自书写释藏书若干卷，也以书法和好学闻名。<ref name=SGCQ41/>

== 评价 ==
*《十国春秋》论曰：[[许寂|许寂]]温和而儒素，王锴淹洽而有文，黼黻太平，宜矣，迺社稷倾危，勿之能救，其罪均也。
