'''王雪纯''' （{{bd|1970年|11月22日|catIdx=W王}}），[[汉族|汉族]]，祖籍[[辽宁|辽宁]][[营口|营口]]，出生于[[江苏|江苏]][[无锡|无锡]]。1993年6月毕业于[[北京广播学院|北京广播学院]]（现[[中国传媒大学|中国传媒大学]]）播音系，[[澳大利亚|澳大利亚]][[墨尔本大学|墨尔本大学]][[传播学|传播学]][[硕士学位|硕士学位]]，后分配至[[中国中央电视台|中国中央电视台]]国际部工作<ref>{{Cite web|url=http://enjoy.eastday.com/e/2010chunwan/u1a5025075.html|title=盘点历届央视春晚美女主持[组图]-春晚 正大综艺 1986年 1983年 青岛39中 美女主持 主持节目 央视 倪萍 王雪纯-东方娱乐-东方网|accessdate=2018-04-22|work=enjoy.eastday.com}}</ref>。主持的主要栏目有《[[正大综艺|正大综艺]]》、《[[环球|环球]]》等<ref>{{Cite book|title=线上故事 讲述中央电视台台前幕后的故事|last=刘连喜主编|first=|publisher=北京：中华书局|year=2003.04|isbn=7-101-03918-9|location=|pages=26-27}}</ref>。

== 简介 ==
1993年，王雪纯从[[中国传媒大学|中国传媒大学]]毕业，被分配到[[中国中央电视台|中国中央电视台]]工作。 1994年起，王雪纯担任《正大综艺》的主持人。当前主持《音乐故事》。

== 家庭 ==
父亲：[[王枫|王枫]]，原[[中央电视台|中央电视台]]台长（1982年11月—1988年5月在任），[[中华人民共和国广播电影电视部|中华人民共和国广播电影电视部]]副部长。

母亲：林如，[[中央人民广播电台|中央人民广播电台]]第一代播音员。

== 奖项 ==
1997年，第十二届全国电视文艺“[[星光奖|星光奖]]”优秀主持人单项奖；第三届全国广播电视节目主持人“[[金话筒|金话筒]]”银奖；广电部“十大杰出青年之一”。1999年，“金话筒”金奖；“全国十佳电视节目主持人”。
