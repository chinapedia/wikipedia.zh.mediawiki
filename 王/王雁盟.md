'''王雁盟'''（{{bd|1970年||||catIdx=W}}）為台灣的男性手風琴家。

== 經歷 ==
大學畢業後王雁盟原在一些科技公司裡擔任職員，之後在26歲前往[[捷克|捷克]]的[[布拉格|布拉格]]旅行中；看見當地一名老者以手風琴演奏時開始對此樂器產生興趣。<ref name="Pccu">{{Cite web |url=http://jou.pccu.edu.tw/weekly/database/personage123/personage1068-2.html |title=金曲獎手風琴家王雁盟 二十六歲開啟手風琴生涯 4年後打敗馬修‧連恩奪金曲 |work=[[文化大學|文化大學]] |author=陳怡君 |date=2006年6月18日 |accessdate=2013年7月16日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140415231957/http://jou.pccu.edu.tw/weekly/database/personage123/personage1068-2.html |archivedate=2014年4月15日 }}</ref>

回到台灣後王雁盟開始以先前學習過鋼琴的經驗、來自行摸索手風琴。<ref name="Pccu"/>在1999年，王雁盟首次獲得邀約為電影《[[三橘之戀|三橘之戀]]》演奏插曲，之後也曾在[[夏光雷|夏光雷]]等人的音樂專輯裡參與演奏。<ref>{{Cite web|url=http://www.wind-records.com.tw/shop/person/monte/TCD-5264/about.htm|title=王雁盟個人經歷|work=[[風潮唱片|風潮唱片]]|accessdate=2013年7月16日|deadurl=yes|archiveurl=https://web.archive.org/web/20070605082525/http://www.wind-records.com.tw/shop/person/monte/TCD-5264/about.htm|archivedate=2007年6月5日}}</ref>

2003年王雁盟考慮到志向問題而決心辭去原有工作、專心擔任手風琴手。<ref>{{Cite web |url=http://www.businessweekly.com.tw/KIndepArticle.aspx?id=19036&pnumber=3 |title=科技新貴轉業 成金獎風琴手 |work=[[商業週刊|商業週刊]] |author=王仕琦 |date=2013年5月22日 |accessdate=2013年7月16日}}</ref>同年王雁盟則推出個人第一張專輯《[[飄浮手風琴|飄浮手風琴]]》；該專輯後續曾入圍[[第15屆金曲獎|第15屆金曲獎]]最佳流行音樂演奏專輯獎的名單，<ref>{{Cite web |url=http://www.bamid.gov.tw/BAMID/Code/Resource/zh_tw/HtmlEditor/Pic/c8f6fd46-0bbe-4440-912c-0d9304b60df1.doc |title=中華民國九十三年（第十五屆）金曲獎入圍名單 |work=[[中華民國文化部|中華民國文化部]] |format=doc |date=2004年3月30日 |accessdate=2013年7月16日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140318180405/http://www.bamid.gov.tw/BAMID/Code/Resource/zh_tw/HtmlEditor/Pic/c8f6fd46-0bbe-4440-912c-0d9304b60df1.doc |archivedate=2014年3月18日 }}</ref>2005年則發行第二張專輯《音樂。偵探。王雁盟》。

除持續從事音樂演出外，王雁盟另有在家鄉[[台中|台中]]裡開設名為「Forro Cafe」的咖啡店。<ref>{{Cite web|url=http://dreamingfest.com/dreamingfestival/about_04_5.php|title=瞇瞇講 & 王雁盟|work=2011台北夢想音樂節|accessdate=2013年7月16日|deadurl=yes|archiveurl=https://web.archive.org/web/20130809061546/http://dreamingfest.com/dreamingfestival/about_04_5.php|archivedate=2013年8月9日}}</ref>
