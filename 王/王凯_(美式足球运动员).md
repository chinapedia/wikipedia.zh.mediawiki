'''王凯'''（{{lang-en|'''Edward Kai Wang'''}}，{{bd|1987年|3月12日||}}），是效力于[[美国|美国]][[国家橄榄球联盟|国家橄榄球联盟]][[奥克兰突袭者|奥克兰突袭者]]的一名球员，身高1.96米，体重140公斤。他在2010年NFL选秀中被水牛城比爾选中，成为NFL历史上第一个父母均为华人的球员。<ref>{{cite news|url=http://sports.sina.com.cn/o/2010-04-25/11404955547.shtml|title=NFL选秀大会王凯被老牌劲旅选中 成史上华裔第1人|publisher=新浪网|date=2010-04-25|accessdate=2011-02-09}}</ref>

== 家庭 ==
王凯的父母都是参加过[[1984年夏季奥林匹克运动会|1984年夏季奥林匹克运动会]]的[[田径|田径]]运动员。王凯的母亲周英曾获得过100米栏中国全国冠军，父亲王泽和则从事跳高运动。洛杉矶奥运会结束后，二人移民美国，在1987年生下了王凯<ref>{{cite news|url=http://news.163.com/10/0430/10/65GTG18900014AED.html|title=王凯 闯进NFL的“中国哥斯拉”|publisher=网易|date=2010-04-30|accessdate=2011-02-09}}</ref>。他的弟弟David Wang也在弗吉尼亚理工学院打橄榄球。王凯是NFL第一个完全具有中国血统的球员<ref>{{cite web|url=http://profootball.scout.com/2/969396.html|title=Next Year's Role Model|last=Dyer|first=Kristian R.|publisher=Scout.com|date=2010-05-11|accessdate=2010-05-13|deadurl=yes|archiveurl=https://web.archive.org/web/20100515030018/http://profootball.scout.com/2/969396.html|archivedate=2010-05-15}}</ref>，并能说[[普通话|普通话]]。

== 进入NFL前 ==
王凯幼年时，他父母就想把他培养出一个出色的运动员。从5岁开始，他就坚持每天300个俯卧撑和300个仰卧起坐的高强度训练<ref name = "wk1">{{cite web|url=http://sports.163.com/special/wangkai/|title=王凯，NFL的新“姚明”？|publisher=网易|accessdate=2011-02-09}}</ref>。

王凯最开始希望成为一名[[篮球|篮球]]运动员。2003年，王凯曾经到八一篮球青年队试训，但是由于骨龄测试不太理想，时任八一体工大队大队长的[[李富胜|李富胜]]称“王凯以后的身高肯定达不到2米”。之后，王凯逐渐把重心转移到橄榄球运动上。<ref name = "wk1"/>

回到美国，王凯就读于弗吉尼亚的石桥高中（{{lang|en|Stone Bridge High School}}）。在这期间，他被评选为佳得乐年度全州最佳球员<ref name = "wk1"/>。

在[[弗吉尼亚理工学院暨州立大学|弗吉尼亚理工学院]]，王凯成为了学校史上最年轻的球员。大二时，他成为球队主力并帮助球队在2008年夺得ACC分区冠军。不久他被选入2009年的全ACC第二阵容<ref name = "wk1"/>。由于他在场上打球风格极具侵略性，王凯被人取了一个绰号，叫做“Godzilla”（即[[哥斯拉|哥斯拉]]，一个电影里的怪兽）。<ref>{{cite web|url=http://www.nwasianweekly.com/2010/04/godzilla-is-coming-ed-wang-seeks-to-become-the-first-chinese-american-to-be-drafted-into-the-nfl/|title=Northwest Asian Weekly | Godzilla is coming: Ed Wang seeks to become the first Chinese American to be drafted into the NFL|publisher=Nwasianweekly.com|date=2010-04-08|accessdate=2011-02-09}}</ref>

== NFL职业生涯 ==
王凯在该年的NFL选秀中于第5轮第9顺位（总第140顺位）被水牛城比爾选中。在2010赛季，王凯出场6次，但都没有首发。

2011年初，王凯一家作为嘉宾受到了[[中国|中国]]国家主席[[胡锦涛|胡锦涛]]的接见。<ref>{{cite news|url=http://news.cntv.cn/20110125/111058.shtml|title=胡锦涛接见NFL首位美籍华人王凯 鼓励"好好打球"|publisher=中国网络电视台|date=2011-01-25|accessdate=2011-02-09}}</ref>
