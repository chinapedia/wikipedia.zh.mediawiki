'''王诤'''（{{bd|1909年|5月16日|1978年|8月13日}}），原名'''吴人鉴'''，字'''凤岗'''，号'''雨峰'''。[[江苏省|江苏省]][[武进县|武进县]]人。1955年被授予[[中国人民解放军中将|中国人民解放军中将]]军衔并获一级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]、一级[[解放勋章|解放勋章]]<ref name="官方">[http://dangshi.people.com.cn/GB/165617/165618/166491/167908/9975154.html 王诤]，中国共产党新闻网，2010年3月2日查阅。</ref>。

== 生平 ==
幼年读私塾，1924年，考入苏州工专高中部。1927年，从[[江苏省立第二高级中学|江苏省立第二高级中学]]毕业。1928年2月，考入[[南京军事交通技术学校|南京军事交通技术学校]]（后改称中央军事政治学校交通大队，不久又并入[[黄埔军校|黄埔军校]]第六期），学习无线电通信技术。1929年春，分配到中华民国国军第九师任无线电台报务员。1930年9月，调第十八师[[张辉瓒|张辉瓒]]部任中尉报务员，参加对中央苏区[[第一次围剿战争|第一次围剿战争]]。12月[[龙岗战斗|龙岗战斗]]中，第十八师全军覆没，王诤被俘。之后经动员，参加[[中国工农红军|中国工农红军]]。1931年1月，负责组建红一方面军总部无线电队，任队长。同年2月至1932年2月，任中革军委总参谋部无线电侦察训练班负责人、队长。1931年5月成立[[红一方面军|红一方面军]]总司令部电台大队，任大队长，12月任无线电总队总队长<ref>{{Cite book|title=中国革命和建设历史时期人物辞典 1|last=郑福林编著|first=|publisher=长春：吉林人民出版社|year=1988年5月|isbn=7-206-00083-5|location=|pages=25}}</ref>。1931年11月，被推选为出席中华苏维埃第一次全国代表大会代表，并荣获二等[[红星奖章|红星奖章]]<ref>{{Cite book|title=毛泽东百科全书|last=廖盖隆等主编|first=|publisher=北京：光明日报出版社|year=1993年10月|isbn=7-80091-484-4|location=|pages=810}}</ref>。

1932年至1934年1月，任红一方面军司令部通信主任。1934年1月至1937年7月，任中央革命军事委员会总司令部通信局局长。1934年9月，加入中国共产党。10月参加长征。同年兼红军通信学校校长。参加创建中央苏区和工农红军无线电通信事业，完成中央革命根据地第二至第五次[[反围剿战争|反围剿战争]]中通讯联络任务，培养红军通讯人才。1935年8月任中革军委通信科科长。9月红军陕甘支队成立任司令部通信科科长<ref>{{Cite book|title=磨砺89 一个老八路的跌宕人生|last=傅克，刘明明著|first=|publisher=北京：团结出版社|year=2012年10月|isbn=978-7-5126-1293-8|location=|pages=119}}</ref>。同年冬到达[[陕北|陕北]]后，任军委第三局局长。负责红军东征、西征及同红二、红四方面军通讯联络。1937年初，入红军大学第二期学习。[[抗日战争|抗日战争]]时期，1937年7月至1945年8月，任[[中共中央军委|中共中央军委]]总参谋部第三局局长；1940年1月至1945年8月，兼第三局政治委员。1938年3月至1941年1月，任中共中央军委军事工业局副局长、兼通信学校政治委员；创建解放区第一个广播电台<ref>{{Cite book|title=开国将帅|last=汤少云主编；中国人民解放军军事科学院军事百科部编|first=|publisher=太原：山西人民出版社|year=2005年4月|isbn=7-203-05207-9|location=|pages=120}}</ref>。

1945年4月至6月，以中直、军直代表团成员出席[[中共七大|中共七大]]。同年9月至1949年9月，任[[中共中央军委|中共中央军委]]总参谋部第三局局长、作战部副部长（1945年11月起）。1947年3月，任中共中央后方委员会委员。1949年5月，起任中华全国民主青年联合总会委员。同月起兼中共中央军委电信总局局长<ref>{{Cite book|title=西柏坡人物|last=赵福山编著|first=|publisher=北京：中国广播电视出版社|year=2003年7月|isbn=7-5043-3953-9|location=|pages=27}}</ref>。

=== 中华人民共和国时期 ===
中华人民共和国成立后，1949年10月至1954年9月，任[[中央人民政府邮电部|中央人民政府邮电部]]副部长。1949年11月至1953年5月，兼邮电部党组书记。1950年5月至1954年3月，任[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]通信部部长，其间，1952年4月至1956年4月，兼中央军委通信部党委书记。1950年10月，起兼[[国家电信工业局|国家电信工业局]]局长。1951年12月至1954年，任中央军委直属队总党委委员。1956年4月至1958年10月，任国防部航空工业委员会委员<ref>{{Cite book |title = 中国近现代史名人辞典 |editors = 马洪武 等 主编 |location = 北京 |publisher = 档案出版社 |date = 1993年11月 |ISBN = 7-80019-381-0  |page = 27 }}</ref>。

1956年4月至1959年4月，任[[中国人民解放军通信兵部|中国人民解放军通信兵部]]主任。1956年10月至1958年7月，兼任通信兵部党委第一书记。1957年3月至11月，兼任解放军电子科学研究院院长，8月至12月，任院临时党委书记。同年11月至1964年12月，任[[国防部第五研究院|国防部第五研究院]]副院长（至1959年2月）。1957年12月至1963年3月，任研究院党委副书记兼第二分院院长（至1959年2月）。1959年4月至1960年8月，任解放军总参谋部通信兵部党委常务委员。1963年4月至1964年9月，兼任第四机械工业部党组书记，1963年5月至1967年5月，任第四机械工业部部长，1964年9月至“文化大革命”初期兼任第四机械工业部党委书记<ref>{{Cite book |title = 中国民主革命时期人物简介 |editor = 张敏孝 主编 |publisher=辽宁大学历史系函授部 |year=1981 |ISBN = |location= |page = 13 }}</ref>。

[[文化大革命|文化大革命]]初期接受[[批斗|批斗]]。1967年5月，被“隔离审查”遭迫害。1972年7月，恢复工作。1972年9月至1978年8月，再任[[第四机械工业部|第四机械工业部]]部长、党的核心小组组长（至1977年9月）。1977年9月至1978年8月，兼任[[第四机械工业部|第四机械工业部]]党组书记。1977年4月至1978年8月，任[[中国人民解放军副总参谋长|中国人民解放军副总参谋长]]兼总参谋部第四部部长。1977年9月至1978年8月，兼任总参谋部党委常务委员。1977年8月至1978年8月，任[[中共中央军事委员会|中共中央军事委员会]]委员<ref>{{Cite book |title = 中国共产党第七次全国代表大会代表名录 上 |editor =
中共中央党史研究室第一研究部 |location = 上海 |publisher=上海人民出版社 |date=2005年7月 |ISBN = 7-208-05733-8 |pages = 43-44 }}</ref>。

1955年被授予[[中国人民解放军中将|中国人民解放军中将]]军衔，获一级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]、一级[[解放勋章|解放勋章]]。[[中华人民共和国|中华人民共和国]]第三届国防委员会委员。第一、第二、第三届全国人大代表。中共第十、第十一届中央委员。1978年8月13日，因病在北京逝世<ref>{{Cite book |title = 中国共产党历届中央委员大辞典 1921-2003 |author1 = [[中共中央组织部|中共中央组织部]] |author2 = [[中共中央党史研究室|中共中央党史研究室]] |location = 北京 |publisher =中共党史出版社 |date = 2004年11月 |ISBN = 7-80136-946-7 |pages = 798-799 }}</ref>。
