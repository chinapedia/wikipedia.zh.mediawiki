'''王志平'''（{{bd|1953年|8月|2008年|2月5日|catIdx=W王}}），[[蒙古族|蒙古族]]，[[吉林省|吉林省]][[镇赉县|镇赉县]]人；[[包头|包头]]高等师范专科学校中文系毕业。1972年5月加入[[中国共产党|中国共产党]]。曾任[[中共|中共]][[呼和浩特|呼和浩特]][[市委书记|市委副书记]]，[[呼和浩特市|呼和浩特市]][[检察院|检察院]]检察长。2008年2月5日下午，王志平在自己的办公室，遭枪杀身亡，终年54岁。当年的2月15日，[[内蒙古自治区|内蒙古自治区]]民政厅正式决定，追认王志平为“革命烈士”。

== 从政经历 ==
王志平早年曾在[[东风汽车公司|中国二汽]]安装公司做工人；18岁时，加入[[中国人民解放军|中国人民解放军]]，并在第24军212团服役三年。复员后，他被安排到[[内蒙古|内蒙古]][[包头市|包头市]]冶金矿山机械厂工作。[[高考|高考]]恢复后，1979年王志平考入[[包头|包头]]高等师范专科学校学习；毕业后，他就被分配到[[中共|中共]][[包头|包头]]市委组织部工作。从而，完成了他个人“工人—士兵—学生—干部”的转变。

此后，王志平一直在[[包头市|包头市]]工作；曾历任[[中共|中共]][[包头市|包头市]]市委组织部青年干部科科长，[[包头市|包头市]][[中华人民共和国国家安全部|国家安全局]]副局长，市纪委常委、兼监察局副局长，市政府秘书长等职。1998年9月，王志平离开[[包头|包头]]，出任[[呼和浩特市|呼和浩特市]][[检察院|检察院]]检察长；任职期间，他还曾在[[最高人民检察院|最高人民检察院]]挂职锻炼七个月，并任[[最高人民检察院|最高人民检察院]]侦查监督厅副厅长。2001年9月，王志平进入[[中共|中共]][[呼和浩特|呼和浩特]]市委常委领导班子；三年后，晋升为市委副书记；并曾历任市委秘书长、常务副市长、市纪委书记等职<ref>[http://www.nmg.xinhuanet.com/zt/2006-10/24/content_8338932.htm 中国共产党呼和浩特市第十届委员会常务委员会委员名单] {{webarchive|url=https://web.archive.org/web/20070829110321/http://www.nmg.xinhuanet.com/zt/2006-10/24/content_8338932.htm |date=2007-08-29 }} 新华网内蒙古频道</ref>。

== 被杀事件 ==
王志平被害前是[[中共|中共]][[呼和浩特|呼和浩特]]市委的专职副书记，并具体分管政法工作；而开枪行凶的是原[[呼和浩特市|呼和浩特市]]公安局经济技术开发区分局局长[[关六如|关六如]]，属于王志平的分管的下级。据报道，[[关六如|关六如]]的杀人动机，可能源于买官不成；又或是职务被撤换，而心怀不满，所导致的；但这些说法尚未得到官方证实<ref>[http://news.cctv.com/china/20080213/102957.shtml 呼和浩特市委副书记王志平被枪杀] 中国中央电视台网站</ref>。当时，在王志平办公室的另外一位工作人员，也遭了杀害。而[[关六如|关六如]]在杀死两人后，亦饮弹自尽<ref>[http://news.sina.com.cn/c/2008-02-14/022514934113.shtml 呼和浩特市委副书记王志平在办公室遇害] 新浪网新闻中心</ref>。而关六如欲杀的对象并不限于王志平，其自杀前曾留下遗书及买官送礼列表，涉及当地多名高层官员，这一批呼和浩特的官员事後遭到调查。关六如想枪杀市委书记[[韩志然|韩志然]]、市委副书记李鹤及王志平。<ref>[http://www.topchinesenews.com/readarticle.aspx?topic_id=1000&msg_id=55353呼和浩特杀人局长留遗嘱 中纪委照单查高官]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 官方生平 ==
由于王志平被害时，正是2008年[[农历|农历]]新年之际；[[呼和浩特市|呼和浩特市]]官方是在假期结束之后的2月15日才在正式对外通告这一事件的。而就在案发前三天，即2月2日，王志平刚刚为儿子举行完婚礼。

在官方告文中，对王志平的评价是，“[[中国共产党|中国共产党]]的优秀党员、党的优秀民族干部”；将他的遇害，形容为是“英勇牺牲”、“我们失去了一位好领导、好同志、好战友，使党失去了一位优秀的民族干部，是呼和浩特经济社会发展事业的一大损失”。2月15日，[[内蒙古自治区|内蒙古自治区]]民政厅依据《革命烈士褒扬条例》第三条第四项“‘因执行革命任务遭敌人杀害’可批准为‘革命烈士’”的规定，追认王志平为“革命烈士”<ref>[http://news.sina.com.cn/c/2008-02-17/040014953015.shtml 呼和浩特市委副书记遇害后被追认烈士] 新浪网</ref>。
但对于这个评价，在网民中引起强烈质疑，并觉得决定过于草率。很多人提王志平是不是在“执行革命任务”，关六如为什么是“敌人”。 外界传闻当地政府在中央调查组刚进驻当地着手调查时，急于给王志平烈士称号，以及正面评价，是在于先声夺人，保护当地的官僚腐败。
