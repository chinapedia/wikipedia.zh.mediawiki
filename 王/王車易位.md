'''王车易位'''是[[國際象棋|國際象棋]]中一種特别著法，可同時移動己方的[[王_(國際象棋)|王]]和其中一隻[[車_(國際象棋)|車]]。走法是把王向一車的方向平移兩步，再把该車越過王，停在王旁一格。此著法於國際象棋記譜法中以0-0或0-0-0表示：0-0表示車向王方向移了兩步，即“王翼易位”（俗稱「短易位」），0-0-0表示表示車向王方向移了三步，即“-{后}-翼易位”（又稱「長易位」）。

==規則==
[[File:ChessCastlingMovie.gif|thumb]]
王车易位须满足以下条件：
# 王与要進行易位的車都未经移動。
# 王與要進行易位的車之間沒有其他棋子阻隔。
# 王所在、经过和到达的格子皆未受到攻击（即易位时与易位后不能被照将，也不能穿越被对方控制的格子）。
# 王與要進行易位的車必須位處同一橫行（即不能与王前兵升变所成之车易位）。<ref>该規則為[[世界國際象棋聯合會|國際棋聯]]於1972年所加。此前Max Pam曾發現王与兵升变所成之车的易位方法，Tim Krabbé更於此法被禁前在一棋謎中使用过。參见{{tsl|de|Pam-Krabbé-Rochade|Pam-Krabbé-Rochade|Pam-Krabbé-Rochade}}中的圖例。</ref>

一常見誤解是王車易位的規則比以上還要嚴格，事實上：
# 即使王之前曾被將軍，但於易位時未被將軍，仍可進行易位。
# 车被攻击不妨碍易位。
# 車可以穿越被攻擊的格子（只適用於長易位）。
