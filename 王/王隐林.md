'''王隱林'''，又名'''王飛龍'''<ref>{{cite web |url=http://www.tyj.gd.gov.cn/html/styj/2006_04_27_21_6747.html |title=广东体育概述 |author= |publisher=廣東省體育局 |date= |language= |accessdate= }}</ref>，[[廣東十虎|廣東十虎]]之一，與[[黃林開|黃林開]]同為[[星龍|星龍]]的學生，中國武術門派[[俠家|俠家]]的祖師，成名武學是「[[白鶴俠家拳|白鶴俠家拳]]」。

他主要的學生有[[朱亦傳|朱亦傳]]、[[王漢榮|王漢榮]]、[[王倫_(俠家)|王倫]]（王敬初）、[[王林開|王林開]]、[[潘鑒|潘鑒]]、[[蔡懿恭|蔡懿恭]]與[[吳肇鍾|吳肇鍾]]等人。

==生平==

[[李遂成|李遂成]]（又名大俠李鬍子、星龍長老、金鉤禪師）<ref>{{cite web |url=http://www.chinesekungfu.com.cn/html/1309/71c6fce1-a2d7-4131-aaba-1a76f03679ce.htm |title=峨眉武术起源与拳种浅析 |author=李聪 |publisher=中国武术在线 |date=2013-09-23 |language= |accessdate= }}</ref><ref>{{cite web |url=http://www.kungfumagazine.com/forum/showthread.php?43791-On-Behalf-of-Grandmaster-Doo-Wai |title=don't know if this will come out? ("On Behalf of Grandmaster Doo Wai" 帖子之內) |author= |publisher=Kung Fu Magazine Forum |date=2006-03-11 |language= |accessdate= }}</ref><ref>{{cite web |url=http://www.angelfire.com/extreme5/fit2fightclub/lamakungfuhistory.html |title=Lama Kung Fu History |author= |publisher=Angelfire |date= |language= |accessdate=2016-06-28 }}</ref>從[[四川|四川]]雲游到[[廣東|廣東]]，在[[肇慶|肇慶]][[鼎湖山|鼎湖山]][[慶雲寺|慶雲寺]]掛單，把俠家拳傳給僧人王隱林（又名王飛龍）。

王隱林到[[廣州|廣州]]後還俗，在[[黃沙_(廣州)|黃沙]]兼善街開設武館授徒，把[[十二支橋|十二支橋]]、[[小羅漢|小羅漢]]、[[大羅漢|大羅漢]]、[[虎鶴相鬥|虎鶴相鬥]]、[[俠家刀|俠家刀]]、槍棍等拳械套路傳給黃漢榮、王倫、蔡懿恭、王林開、潘​​鑑等徒弟，廣州[[洪家拳|洪家拳]][[鄧龍_(洪拳)|鄧龍]]對俠家拳表示欣賞，便讓兒子[[鄧錦濤|鄧錦濤]]拜王倫為師，鄧錦濤傳給他的兒子[[鄧鎮江|鄧鎮江]]。

王隱林在廣州傳授了七個高徒，其中王倫一脈傳給鄧錦濤，王林開一脈傳給吳肇鐘，吳肇鐘發揮了俠拳的鶴形特點，分支[[白鶴派|白鶴派]]，蔡懿恭一脈向海外發展，稱星龍長老是喇嘛，故又分支為[[喇嘛派|喇嘛派]]。<ref name="各派">{{cite web |url=http://cforum1.cari.com.my/forum.php?mod=viewthread&action=printable&tid=437079 |title=中國各派武術簡介 |author= |publisher=佳礼资讯网 |date=2005-12-15 |language= |accessdate= }}</ref>

王隱林之武術，初期稱為鶴陽拳，即惠陽白鶴拳。1928年，王隱林正式把其定名為俠家。
  
王隐林据说是广东省新兴县小稳村人。

==絕技==

俠家拳，俠家拳械。拳套有四套：十二支橋，小羅漢，大羅漢、虎鶴相鬥。器械有單刀，左把槍，九點十二槍響棍等。<ref name="各派" />
