'''王新海'''{{BD|1958年|1月||}}，[[江苏|江苏]][[连云港|连云港]]人。[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
1974年1月参加工作，1976年12月参军，1978年8月加入[[中国共产党|中国共产党]]。[[中共中央党校|中共中央党校]]大学本科学历。曾任舟嵊要塞区炮兵团政治处组织股干事；舟嵊要塞区政治部组织处干事；舟嵊守备区守备五旅政治部组织科科长；[[南京军区|南京军区]]政治部组织部组织处干事；南京军区司令部办公室第一秘书处秘书；南京军区政治部干部部科技干部处副处长、处长，任免处处长；南京军区政治部干部部副部长；73156部队代理政治委员；南京军区政治部干部部部长；装甲二师（73081部队）代理政治委员；[[中国人民解放军陆军第十二集团军|陆军第十二集团军]]（73061部队）政治部主任；[[中国人民解放军陆军第一集团军|陆军第一集团军]]（73011部队）政治部主任；陆军第一集团军（73011部队）副政治委员。2012年6月任[[浙江省军区|浙江省军区]]政治委员、党委书记、[[中共浙江省委|中共浙江省委]]常委<ref >{{cite web|url= http://district.ce.cn/newarea/sddy/201206/11/t20120611_23396438_11.shtml| language= | title=浙江新一届省委书记、副书记、常委简历(图) |publisher=中国经济网 |date=2012年6月11日 |accessdate= }}</ref><ref name=zlk>{{Cite web|url = http://gbzl.people.com.cn/grzy.php?id=121000318|title = 王新海|author = |date = |publisher = 中国领导干部资料库|language = |accessdate = 2017-04-29|quote = |archiveurl = https://web.archive.org/web/20150928123627/http://gbzl.people.com.cn/grzy.php?id=121000318|archivedate = 2015-09-28|deadurl = yes}}</ref>。
