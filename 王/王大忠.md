'''王大忠'''（{{bd|1962年|2月||}}），[[安徽省|安徽省]][[宿州市|宿州市]][[泗县|泗县]]人，[[中国人民解放军海军少将|中国人民解放军海军少将]]。

==生平==
王大忠入伍参军后加入[[北海舰队|北海舰队]][[中国人民解放军北部战区海军航空兵|航空兵]]二师六团（低空霸王团）三大队，为正连级中队长。1987年3月31日，[[中国人民解放军海军|中国人民解放军海军]]司令员[[刘华清|刘华清]]向[[中国人民解放军总参谋部|中国人民解放军总参谋部]]和[[国防科工委|国防科工委]]汇报了海军[[航空母舰|航空母舰]]和[[核潜艇|核潜艇]]装备规划。为了先期培养航空母舰和其编队的指挥人员，1987年5月，经中央军委批准，海军决定在[[中国人民解放军海军广州舰艇学院|中国人民解放军海军广州舰艇学院]]开办水面潜艇指挥专业[[飞行员舰长班|飞行员舰长班]]，让海军航空兵飞行员改学水面舰艇指挥专业。1987年9月，10人被选送到海军广州舰艇学院学习。开学一个月后有一位学员因学习跟不上进度而退回部队，剩下[[飞行员舰长班|飞行员舰长班]]学员9人（即[[李晓岩|李晓岩]]、[[柏耀平|柏耀平]]、[[杨宏|杨宏]]、王大忠、[[王仲才|王仲才]]、[[王玉成|王玉成]]、[[马业隆|马业隆]]、[[何虎|何虎]]、[[彭建林|彭建林]]）。当时，王大忠是以海军航空兵第二师六团中队长的身份入学。历时3年半学习后，1991年1月这9名学员毕业取得舰艇指挥专业本科学历和工学学士学位，被分配到各驱护舰部队任飞行副舰长职务，后来提升为舰长。其中，王大忠被分配到[[北海舰队|北海舰队]]驱逐舰第一支队，任[[哈尔滨号导弹驱逐舰|哈尔滨号导弹驱逐舰]]副舰长，后升任[[青岛号导弹驱逐舰|青岛号导弹驱逐舰]]舰长<ref>{{Cite web |url = http://news.sohu.com/20050413/n225156563.shtml |title =  18年后的回眸与展望—中国航空母舰舰长班揭秘|author =  |date = 2005-04-13 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1653834 |title = 王大忠少将出任海军副参谋长，此前担任海军参谋长助理 |author =  |date = 2017-04-02 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。多次立三等功，2001年被评为海军优秀共产党员<ref name=caixin>{{Cite web |url = http://china.caixin.com/2016-05-11/100942109.html |title = 海军参谋长助理王大忠等11名军官晋升海军少将 |author =  |date =  2016-05-11|publisher =财新网  |language =  |accessdate =  |quote =  }}</ref>。后来王大忠是[[中国人民解放军国防大学|中国人民解放军国防大学]]中青班硕士研究生<ref name=pp/>。

至迟在2015年7月，王大忠已任海军参谋长助理。2016年5月6日下午，海军在北京举行将官军衔晋升仪式，王大忠、[[王东凯|王东凯]]、[[王再杰|王再杰]]、[[翟永远|翟永远]]、[[王建勛_(解放军)|王建勋]]、[[林庆国|林庆国]]、[[陈义浩|陈义浩]]、[[王守信_(解放军)|王守信]]、[[李安_(海军)|李安]]、[[傅耀泉|傅耀泉]]、[[周名贵|周名贵]]等11名军官由海军大校军衔晋升为海军少将军衔<ref name=caixin/>。2017年初，升任海军副参谋长<ref name=pp/>。2018年5月，调任[[军委后勤保障部|军委后勤保障部]]副部长<ref>{{cite web |author1=岳怀让 |title=海军原副参谋长王大忠调任军委后勤保障部副部长 |url=https://www.thepaper.cn/newsDetail_forward_2169860 |website=澎湃新闻 |accessdate=2018-06-02 |date=2018-06-02}}</ref>。
