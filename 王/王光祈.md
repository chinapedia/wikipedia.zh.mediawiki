'''王光祈'''（{{bd|1892年|8月15日|1936年|1月12日}}）字'''潤玙'''，一字'''若愚'''，[[中國近代|中國近代]]著名[[社會活動家|社會活動家]]和[[音樂家|音樂家]]，他參與過當時的[[五四運動|五四運動]]等重要學生運動，也是[[少年中國學會|少年中國學會]]的創始人之一。<ref name="圖書館"/>

不僅是少年中國學會的元老，王光祈也曾主辦「工讀互助團」等學生互助團體，就是在主辦工讀互助團時認識了[[毛澤東|毛澤東]]，兩人關係很好。1920年遠赴德國留學，主張“音樂救國”，成為了最早在西方獲得博士學位的中國音樂學家，對中西音樂理論的融合起到了很大的作用。<ref>文蕾，[http://www.cnki.com.cn/Article/CJFDTotal-SHIJ201431074.htm 王光祈开东方民族音乐之先河]，《兰台世界》，2014年31期</ref><ref>王瑞，[http://mall.cnki.net/magazine/Article/XSPL2013S2056.htm 从《翻译琴谱之研究》透视王光祈对古琴艺术的贡献]，《小说评论》，2013年第S2期</ref>

==生平==
[[File:追悼王光祈先生專刊.jpg|thumb]]
===早年===
王光祈於[[光緒|光緒]]十八年（1892年）出生在[[四川|四川]][[溫江|溫江]]鱼凫镇小河村（現在的[[溫江區|溫江區]]柳城镇德通桥），生後不久父亲病逝，由出生富家母亲罗氏撫養、教育。曾在三官庙私塾、社学巷私塾（今王光祈舊居）等私塾中讀書。<ref name="成都晚報">{{Cite web |url= http://www.cdwb.com.cn/html/2014-06/14/content_2064877.htm|title=王光祈 从革命启蒙者到音乐家|accessdate=2015-01-19|date=2014-06-14|publisher=成都晚報}}</ref> 少年時吹過簫、笛。1908年進入成都高等學堂附屬的中學堂學習，這時愛好[[川劇|川劇]]。當時他的同班同學後來很多都很有名氣，其中包括[[周太玄|周太玄]]、[[郭沫若|郭沫若]]、[[曾琦|曾琦]]、[[李璜|李璜]]等。1911年四川發生[[保路運動|保路運動]]，这一班學生有多人都曾加入其中。<ref name="文史春秋"/>
===學生運動===
1914年，王光祈在[[北京|北京]]進入[[清史館|清史館]]當書記，並且在[[中國大學|中國大學]]攻讀法律，1918年7月以第二名的成績毕业。<ref name="文史春秋">{{Cite web|url=http://img.mg1912.com/news/2014/03/04/4ae6df18437c1b8801448c4b0497002e.html|title=“少年中国学会”回溯|accessdate=2015-01-19|date=2014-03-04|publisher=文史春秋|deadurl=yes|archiveurl=https://web.archive.org/web/20150119224825/http://img.mg1912.com/news/2014/03/04/4ae6df18437c1b8801448c4b0497002e.html|archivedate=2015-01-19}}</ref> 當時王也曾先後擔任《[[四川群報|四川群報]]》駐京記者和《[[京華日報|京華日報]]》的編輯。1918年，王光祈與[[李大釗|李大釗]]、周太玄、曾琦等七人發起了「[[少年中國學會|少年中國學會]]」，以進行中國人的思想啟蒙為要務。按王光祈的話來說就是：<ref name="炎黃世界"/>
{{cquote|聯合同輩，殺出一條道路，把這個古老腐朽、呻吟垂絕的被壓迫被剝削的國家，改變成為一個青春年少、獨立富強的國家。}}

當時擔任《[[晨鐘報|晨鐘報]]》副刊主編的李大釗十分欣賞王光祈，曾說：“光祈是一個能想、能行的青年，極有志氣”。<ref name="炎黃世界">{{Cite web |url= http://news.takungpao.com/history/story/2013-02/1444917_2.html|title=毛泽东为何对王光祈念念不忘？|accessdate=2015-01-19|date=2013-02-19|publisher=炎黃世界}}</ref>

同年7月1日少年中國學會的成立大會上，王光祈被推選為執行部主任，成為該會的實際負責人。同時他又投身於學生運動之中，曾在第一時間把[[五四運動|五四運動]]爆發的消息電報給《[[川報|川報]]》。此外，他也一直從事著寫作工作。[[五卅運動|五卅運動]]之後，左右兩派因為鎮壓工人的問題而鬧翻，王光祈出國不在，最終少年中國學會也不得不解散。<ref name="炎黃世界"/>

在此之前，1918年10月初到北京的[[毛澤東|毛澤東]]經過李大釗的介紹，亦加入了少年中國學會下。據王光祈的好友魏時珍所述，毛澤東當時和王光祈十分友好，常與其長談，並在認為他“頗重實踐”而維護毛免受其他人的攻擊。1919年末[[陳獨秀|陳獨秀]]、[[蔡元培|蔡元培]]和李大釗協助王光祈創建了「工讀互助團」，目的是幫助北京青年進行半工半讀，雖然有不錯的反響，但因為欠缺經費而最終不了了之。<ref name="炎黃世界"/>
===在德國===
1920年5月，他和一些少年中國學會會員在[[上海|上海]]乘法國輪船赴歐洲留學，而在[[黃浦江|黃浦江]]邊與毛澤東等握手告別。王光祈前往德國，一開始在[[法蘭克福大學|法蘭克福大學]]讀政治經濟學，但依舊關心國內事務，曾兼任《[[申報|申報]]》、《[[時事新報|時事新報]]》和《[[晨報_(北京)|晨報]]》的駐德特約記者，勤工儉學完成了在法蘭克福大學的學業。1922年因為沒法參加國內社會運動、生活苦悶而改學音樂，主張「音樂救國」。1927年到[[柏林大學|柏林大學]]師從[[埃里希·馮·霍恩博斯特爾|埃里希·馮·霍恩博斯特爾]]（Erich von Hornbostel）等人進修音樂，最終在[[波恩大學|波恩大學]]獲得博士學位，成為第一個獲得西方博士學位中國音樂學家。<ref name="炎黃世界"/><ref name="鳳凰網">{{Cite web |url= http://book.ifeng.com/special/wusiwenrenpu/list/200904/0430_6351_1134521.shtml|title=王光祈生平|accessdate=2015-01-19|date=2009-04-30|publisher=鳳凰網}}</ref>

在客居德國期間，他翻譯過一些政論著作，同時寫成了《歐洲音樂進化論》、《東方民族之音樂》等十八部音樂專著和四十多篇論文，<ref name="圖書館">{{Cite web|url=http://118.122.113.7:82/hl/yyj_wgq.asp|title=王光祈|accessdate=2015-01-19|publisher=四川音樂學院圖書館|deadurl=yes|archiveurl=https://web.archive.org/web/20150119213834/http://118.122.113.7:82/hl/yyj_wgq.asp|archivedate=2015-01-19}}</ref> 其論題多是主張發揮音樂的社會功能以激發愛國主義精神。後來他在波恩大學擔任中文講師，1936年1月12日，因積勞成疾突發[[腦溢血|腦溢血]]在[[波恩|波恩]]病逝。當年國內有刊文《追悼王光祈先生专刊》表示追悼。<ref name="炎黃世界"/>
===遺產===
少年時，王光祈曾和李劼人、周太玄、魏時珍在成都東郊菱角堰周太玄家結拜。他在德國病逝後，骨灰由[[沈君怡|沈君怡]]帶回國，<ref name="成都晚報"/> 并於1941年冬由李劼人在結拜處安葬，由周太玄題字“王光祈先生之墓”，[[土地改革運動|土地改革運動]]時曾被移到別處。<ref name="炎黃世界"/> 1983年遷至[[四川音乐学院|四川音乐学院]]，建立了王光祈建纪念碑亭，周太玄的题碑也保留在這裡。<ref name="成都晚報"/> 1984年6月，[[中国音乐家协会|中国音乐家协会]]聯合[[四川音樂學院|四川音樂學院]]等其他單位一起舉辦了“王光祈研究学术讨论会”。1985年他被錄入[[人民音乐出版社|人民音乐出版社]]的《[[中国音乐词典|中国音乐词典]]》中。1986年，温江公园内建成王光祈纪念馆，後來又建造了光祈音乐广场。<ref name="文史春秋"/>

==家庭==
1910年奉母親之命，娶罗次珊（後改名王次玙）为妻，先後育有一子一女。但长子因病夭折，他的母親也因不能承受喪孫之痛而去世，不久之後女兒在一岁半時罹患[[天花|天花]]病故。這也是王光祈在1914年春決定留下妻子獨自一人離家的緣故。他的妻子在1946年逝世，因為沒有任何子嗣，其故居也就沒能保留下來。現在當地的王光祈故居是温江区政府劃定的原社学巷私塾。<ref name="成都晚報"/>

==軼事==

王光祈舉辦「工讀互助團」時討論窮學生能做什麼工作，毛澤東說：「不要只是說說，讓我來試行一下！你們諸位把換洗衣服都交與我來洗。一個銅子一件，三天交貨！」但是沒有人響應，只有王光祈事後把自己的衣服交給毛澤東清洗。<ref name="炎黃世界"/>

王光祈和少年中國學對毛澤東的思想形成產生了較大的影響。毛澤東一直不知道王光祈已經去世，1949年中華人民共和國成立後曾想要邀請王加入政府，曾借[[陳毅|陳毅]]之口詢問王光祈的下落，才從當時的成都副市長[[李劼人|李劼人]]那裡得知時其已經逝世的消息。後來再次派陳毅到成都詢問王光祈的家屬下落，但亦無人知曉。<ref name="炎黃世界"/>

==評價==
*日本音乐学家[[岸邊成雄|岸邊成雄]]：“把柏林学派的比较音乐学观点第一个介绍到东方来的，是中国人王光祈。”<ref name="成都晚報"/>
*[[冼星海|冼星海]]：“我们不能忘记这位音乐理论家王光祈，他推动了新音乐的发展，他的刻苦耐劳是我们从事中国新音乐的模范。”<ref name="成都晚報"/>
*[[吕骥|吕骥]]：“王光祈是我国五四运动前后到抗日战争之前的近20年中，文化界一位著名的爱国主义著作家，卓越的音乐学家。”；“将东西方之音律、东方各民族之音律进行比较研究，始创于王光祈，这无疑是我国音乐学上一大贡献。”<ref name="文史春秋"/>
