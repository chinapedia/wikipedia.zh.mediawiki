'''王崑'''（{{bd|1931年|12月|1958年|10月21日}}）[[吉林省|吉林省]][[舒兰县|舒兰县]]人，[[中国人民解放军海军航空兵|中国人民解放军海军航空兵]]某师第十团（[[海空雄鹰团|海空雄鹰团]]）副团长、战斗英雄、烈士<ref name=zgjw/><ref name=xhw/>。

==生平==
1947年5月入伍。1948年加入[[中国共产党|中国共产党]]。[[第二次国共内战|第二次国共内战]]中，立大功一次，获[[中国人民解放军第三野战军|中国人民解放军第三野战军]]颁发艰苦奋斗奖章一枚<ref name=zgjw/><ref name=xhw/>。

后来奉派当学员学习飞行，成为飞行员。在[[抗美援朝战争|抗美援朝战争]]中，1952年8月，王崑在一次空战中击落、击伤[[美军|美军]][[F-86军刀战斗机|F-86飞机]]各1架，首开全团空战记录。在此之前，他仅有80余小时的飞行经验，驾驶[[米格-15战斗机|米格-15战斗机]]仅20余小时。1953年3月27日，王崑和僚机在执行任务返航的途中，发现4架[[英国|英国]]战斗轰炸机正要袭击[[中国人民志愿军|中国人民志愿军]]地面部队。王崑和僚机在燃油所剩不多的情况下击落、击伤敌机各1架。王崑曾获[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]二级战士荣誉勋章1枚<ref name=zgjw/><ref name=xhw/><ref name=yx/>。

在[[中华人民共和国|中华人民共和国]]国土防空作战中，王崑驾机自2000米高度一直打到距海平面70米，这创造了喷气式战斗机超低空作战纪录。在[[福建|福建]]前线发生的一次空战中，王崑指挥一个中队{{Link-en|緊急升空|Scrambling (military)}}，击伤[[中华民国空军|中华民国空军]]F-86飞机2架<ref name=zgjw/><ref name=xhw/><ref>{{Cite web |url =http://mil.huanqiu.com/Forum/2012-05/2708241.html  |title = 福建沿海罕为人知的国共空战内幕 |author =  |date = 2012-05-09 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>。

王崑历任飞行员、中队长、大队长、副团长等职。在抗美援朝战争和国土防空作战中，共击落敌机3架，击伤敌机2架，立一等功2次、二等功多次。1956年，王崑出席全国社会主义建设青年积极分子代表大会，受到[[毛泽东|毛泽东]]等[[党和国家领导人|党和国家领导人]]接见，毛泽东握着王崑的手说：“很好！我记住你，也记住你们海军航空兵十团了，祝你们多打胜仗！”<ref name=zgjw/><ref name=xhw/><ref name=yx>{{Cite web |url =  http://news.xinhuanet.com/mil/2011-05/30/c_121472650.htm|title = 王崑：击落击伤敌机5架的战斗英雄 |author =  |date = 2011-05-30 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>

1958年10月21日，在福建前线的一次空战中，王崑因胃病突发昏迷而殉职，年仅27岁<ref name=zgjw>{{Cite web |url = http://www.81.cn/2015hkxyt/2015-12/22/content_6826509.htm |title =王昆：创造喷气式战斗机超低空作战的记录  |author =  |date = 2015-12-22 |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref><ref name=xhw>{{Cite web |url = http://news.xinhuanet.com/mil/2009-08/19/content_11906743.htm |title =人民英模王崑：空中作战逞英豪  |author =  |date = 2009-08-19 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。
