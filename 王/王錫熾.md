'''王錫熾'''，英文名：Dr Sih Tse Wang，1892年—1946年，原名'''王恭寅'''<ref name="wong family">{{cite book|volume= |title=《東成西就——七個華人基督教家族與中西交流百年》|author=羅元旭  |year=2012 |location=北京 |page=440 |isbn=9787108045096 |publisher=三聯書店(北京) }}</ref>，民国时著名医学家<ref>{{cite web|url=http://news.eastday.com/eastday/13news/auto/news/china/u7ai2815376_K4.html|title=陈竺以“德、命、技”与海归医学界人士共勉|publisher=东方网|author=中国新闻网|date=2014-10-19|deadurl=yes|archiveurl=https://web.archive.org/web/20150725212124/http://news.eastday.com/eastday/13news/auto/news/china/u7ai2815376_K4.html|archivedate=2015-07-25}}</ref>，曾任[[北京协和医院|北京协和医院]]院长<ref>{{cite web |url=http://www.pumch.cn/Category_33/Index.aspx |title=历史沿革 |publisher=北京协和医院 |author=历届院长 |accessdate=2015-07-25 |archiveurl=https://web.archive.org/web/20150725224210/http://www.pumch.cn/Category_33/Index.aspx |archivedate=2015-07-25 |deadurl=yes }}</ref>。字'''錫熾'''，[[以字行|以字行]]。

==生平==

早年赴[[美国|美国]]留学，获[[哈佛医学院|哈佛医学院]][[医学博士|医学博士]]学位。学成回国后于1924年入北平协和医院（今[[北京协和医院|北京协和医院]]）当医生。任副院长并主管医院门诊部。1934年，任协和医院院长。并任[[欧美同学会|欧美同学会]]总干事和北平[[扶轮社|扶轮社]]社长<ref name="wong family"/>。

[[日本侵华|日本侵华]]后，在日占北平力保协和医院院产。其中，将[[北京|北京]][[周口店|周口店]]出土的[[北京猿人|北京猿人]]头骨化石装箱交予盟军美国以防落入[[日军|日军]]之手，是北京猿人头骨化石最后见证人，但化石被交予[[美军|美军]]后就下落不明，成为20世纪一大谜团<ref>{{cite web |url=http://www.19zs.com/shehui/d/4378/6.html |title=第五章 转移“北京人”(4) |publisher=第六章战争状态下的北平(2) |author=《考古中国》 |accessdate=2015-07-25 |archiveurl=https://web.archive.org/web/20160306035121/http://www.19zs.com/shehui/d/4378/6.html |archivedate=2016-03-06 |deadurl=yes }}</ref><ref >{{cite web|url=http://news.sohu.com/20041008/n222370005.shtml |title=最后见到“北京人”头盖骨的中国人 |publisher=搜狐新闻 |author=新华网 |date=2004年10月8日}}</ref>。

1945年二战结束后，任[[国立北京大学|国立北京大学]]医院院长，直至1946年6月以54岁壮年猝逝。是北京协和医院历史上最长任的院长（1934年至1946年）。任上引入美制，是中国医院学习美国医院制度之始，采用美国医院管理制度，建立制定了中国现代在院医生住院医师制度、医学学生住院实习制度和护理制度，后世中国医院深受影响<ref>{{cite web |url=http://www.bddyyy.com.cn/bdysg/hou/xymy/20131027/398266.shtml |title=九一八事变后的抗日历程（1931-1945年） |publisher=北大医院网上院史馆 |author=西医萌芽 |deadurl=yes |archiveurl=https://web.archive.org/web/20150725235236/http://www.bddyyy.com.cn/bdysg/hou/xymy/20131027/398266.shtml |archivedate=2015-07-25 }}</ref>。王在北京协和的制度受益协和医院医学毕业生，故创立了一个学派，著名医学家中如[[吴阶平|吴阶平]]、[[严仁英|严仁英]]、[[邓庆曾|邓庆曾]]等都是他的学生<ref name="wong family"/>。王也是浙江红十字运动早期领导人物<ref>{{cite web|url=http://www.zjda.gov.cn/zjdazz/wzdd/201010/t20101013_193293.htm|title=浙江红十字运动起源探究|publisher=浙江档案网|author=浙江档案杂志社|date=《浙江档案》2010年第9期|deadurl=yes|archiveurl=https://web.archive.org/web/20150726004731/http://www.zjda.gov.cn/zjdazz/wzdd/201010/t20101013_193293.htm|archivedate=2015年7月26日|df=}}</ref>。

==家庭==
浙江[[奉化县|奉化县]]人，生于基督教世家，祖父[[王际唐|王际唐]]（有光）牧师，父亲[[王正庸|王正庸]]。叔父著名外交家[[王正廷|王正廷]]，矿业家[[王正黼|王正黼]]，银行家[[王正序|王正序]]。弟[[王恭宸|王恭宸]]，医生。弟[[王恭守|王恭守]]（Kung-shou Wang），外交官，[[南京政府|南京政府]]驻美国[[波士顿|波士顿]]领事馆[[领事|领事]]<ref name="wong family"/>。王妻美国白人Lillian Bartow Towner，出自美国[[纽约州|纽约州]]首府[[奥尔巴尼|奥尔巴尼]]木材商之家，毕业于[[康奈尔大学|康奈尔大学]]，王錫熾身过后移居[[香港|香港]]，1961年5月在香港逝世。 独女[[王憶德|王憶德]]，在王身过后随母移居香港，母身过后再移居美国[[科罗拉多州|科罗拉多州]][[丹佛市|丹佛市]]行医，是[[美籍华人|美籍华人]]医生<ref name="wong family"/>。
