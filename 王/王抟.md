'''王抟'''（{{bd|9世纪||900年|7月12日|catIdx=W}}<ref name=ZZTJ262>[[:s:資治通鑑/卷262|《资治通鉴》卷二百六十二]]</ref>），[[表字|字]]'''昭逸'''，爵封'''鲁国公'''，[[中国|中国]][[唐朝|唐朝]][[官员|官员]]，[[唐昭宗|唐昭宗]]年间的895—900年间为[[宰相|宰相]]。[[光化|光化]]三年（900年），他有权势的同僚宰相[[崔胤|崔胤]]诬告他勾结[[宦官|宦官]]，他被昭宗賜死。

== 家世 ==
王抟生年不详，祖上有两位宰相，即女主[[武则天|武则天]]年间拜相的十世祖[[王方庆|王方庆]]和[[唐肃宗|唐肃宗]]年间拜相的曾祖父[[王玙|王玙]]。王抟的祖父王及官至[[中书舍人|中书舍人]]，父亲王鐬官至右谏议大夫。<ref name=NBT72>《[[新唐书|新唐书]]》卷七十二</ref>

== 早期仕途 ==
王抟中[[进士|进士]]，直[[弘文馆|弘文馆]]，佐前[[宰相|宰相]]、[[义成军节度使|义成军节度使]]、[[滑州|滑州]][[刺史|刺史]][[王铎_(唐朝)|王铎]]为推官。<ref>[[:s:新唐书/卷185|《新唐书》卷一百八十五]]</ref><ref>[[:s:资治通鉴/卷254|《资治通鉴》卷二百五十四]]</ref><ref>王铎在882—884年间任义成军节度使、滑州刺史，可见王抟效力王铎在此期间。见[[:s:舊唐書/卷164|《旧唐书》卷一百六十四]]。</ref><ref>[[:s:授王搏检校殿中侍御史充义成军节度推官制|刘崇望《授王搏检校殿中侍御史充义成军节度推官制》]]</ref>王抟累迁[[苏州|苏州]]刺史，曾任兵部员外郎兼侍御史知杂事，<ref>[[:s:授王搏兵部员外郎兼侍御史知杂事等制|李磎《授王搏兵部员外郎兼侍御史知杂事等制》]]</ref>后以户部侍郎判户部。<ref name=NBT116>[[:s:新唐書/卷116|《新唐书》卷一百一十六]]</ref>[[景福_(唐朝)|景福]]二年（893年）十一月，加[[同平章事|同平章事]]、<ref name=BT20-1>[[:s:旧唐书/卷20上|《旧唐书》卷二十上]]</ref>[[中书侍郎|中书侍郎]]。<ref>《[[册府元龟|册府元龟]]》</ref>

== 拜相 ==
[[乾宁|乾宁]]元年（894年）十月，王抟被任为[[武安军节度使|武安军节度使]]，但未赴任。二年（895年）三月，昭宗任宰相崔胤为[[护国军节度使|护国军节度使]]意图收回朝廷对护国军的控制，复任王抟为[[中书侍郎|中书侍郎]]，授[[同中书门下平章事|同中书门下平章事]]，<ref>《旧唐书·昭宗纪》作六月事。</ref>以代崔胤，<ref>[[:s:新唐书/卷10|《新唐书》卷十]]</ref><ref name=ZZTJ260>[[:s:資治通鑑/卷260|《资治通鉴》卷二百六十]]</ref>且仍判户部。<ref>[[:s:授王搏中书侍郎同中书门下平章事判户部制|段安节《授王搏中书侍郎同中书门下平章事判户部制》]]</ref>王抟虽明达有度量，并被时人称为良相，<ref name=ZZTJ262/>但作为宰相时的举措却几无记载，仅知曾进和诗<ref>[[:s:史馆王相公进和诗表|钱珝《史馆王相公进和诗表》]]</ref>及请求立[[积善太后|何淑妃]]为[[皇后|皇后]]。<ref>[[:s:中书省请册皇后表|钱珝《中书省请册皇后表》]]</ref><ref>[[:s:史馆王相公请册淑妃何氏为皇后表|钱珝《史馆王相公请册淑妃何氏为皇后表》]]</ref>[[威胜军节度使|威胜军节度使]][[董昌|董昌]]称大越罗平帝，五月，昭宗令[[镇海军节度使|镇海军节度使]][[钱镠|钱镠]]讨之，王抟作诏书。<ref>[[:s:命钱镠讨董昌诏|:s:命钱镠讨董昌诏]]</ref>七月昭宗为避[[军阀|军阀]][[凤翔军节度使|凤翔军节度使]][[李茂贞|李茂贞]]和[[静难军节度使|静难军节度使]][[王行瑜|王行瑜]]劫持，逃离京城[[长安|长安]]而入[[秦岭|秦岭]]时，王抟与其他宰相[[崔昭纬|崔昭纬]]、[[徐彦若|徐彦若]]一同随驾。<ref name=ZZTJ260/><ref name=BT177>[[:s:旧唐书/卷177|《旧唐书》卷一百七十七]]</ref>九月，在正议大夫、中书侍郎、同平章事任上被加[[金紫光禄大夫|金紫光禄大夫]]、户部尚书、门下侍郎、监修国史、判度支。<ref name=BT20-1/>

三年（896年）五月，昭宗杀[[崔昭纬|崔昭纬]]，与崔昭纬同宗的中书侍郎同平章事崔胤因而作为崔昭纬党羽被罢相。<ref name=NBT223-3>[[:s:新唐书/卷223下|《新唐书》卷二百二十三下]]</ref>同月，钱镠败杀后来取消帝号的董昌，从而控制威胜军。七月，昭宗因李茂贞再攻京城避难[[镇国军|镇国军]]军部[[华州|华州]]。王抟等宰相畏惧[[镇国军节度使|镇国军节度使]][[韩建|韩建]]，不敢专断政事。八月，王抟以扶危匡国致理功臣、判度支、诸道盐铁转运等使、金紫光禄大夫、兼户部尚书、门下侍郎、平章事、监修国史、[[上柱国|上柱国]]、'''琅琊郡开国公'''、<ref>《旧唐书·昭宗纪》作太原郡开国公。</ref>食邑二千户的身份被任为检校户部尚书、检校尚书左仆射、同平章事、兼[[越州|越州]]刺史、充威胜军节度、浙江东道管内观察处置宣抚等使、使持节越州诸军事守[[处州|处州]]刺史，<ref name=BT20-1/><ref>[[:s:授王搏威胜军节度平章事制|杨钜《授王搏威胜军节度平章事制》]]</ref>还未出发，又加检校尚书右仆射。但钱镠并不愿让出威胜军，令两浙吏民上表请以钱镠兼领浙东。昭宗不得已，十月，召回王抟，王抟以扶危匡国致理功臣、新授武胜军节度、浙江东道管内观察处置兼宣抚等使、金紫光禄大夫、检校尚书右仆射、同中书门下平章事、使持节越州诸军事、越州刺史、上柱国、'''太原郡开国公'''、食邑二千户的身份被拜为光禄大夫、守吏部尚书、同中书门下平章事，<ref>[[:s:新唐书/卷90|《新唐书》卷九十]]</ref><ref>[[:s:授王抟平章事制|韩仪《授王抟平章事制》]]</ref><ref>《册府元龟》作八月事。</ref>以钱镠为镇海、威胜两军节度使。<ref name=ZZTJ260/>四年（897年），昭宗立皇子[[李𥙿|李𥙿]]为[[皇太子|皇太子]]，<ref name=ZZTJ261>[[:s:資治通鑑/卷261|《资治通鉴》卷二百六十一]]</ref>王抟建议昭宗大赦天下以显隆重。王抟一度以扶危匡国致理功臣、光禄大夫、守吏部尚书、同中书门下平章事、上柱国、琅琊开国公、食邑二千户的身份被任为门下侍郎、兼吏部尚书、同中书门下平章事、监修国史、充诸道盐铁转运等使，<ref>[[:s:授王抟诸道盐铁转运等使制|郑璘《授王抟诸道盐铁转运等使制》]]</ref>后正拜右仆射，光化二年（899年）迁[[司空|司空]]，<ref name=NBT10>[[:s:新唐书/卷10|《新唐书》卷一十]]</ref>封鲁国公。<ref name=NBT116/>王抟表辞司空。<ref>[[:s:为王相公让加司空表|钱珝《为王相公让加司空表》]]</ref>

王抟曾上表谢昭宗加其食邑、实封<ref>[[:s:代王相公谢加门下侍郎食邑表|钱珝《代王相公谢加门下侍郎食邑表》]]</ref><ref>[[:s:代史馆王相公谢加食邑实封表|钱珝《代史馆王相公谢加食邑实封表》]]</ref>及任自己为监修国史、盐铁使等。<ref>[[:s:代史馆王相公谢监修国史盐铁使表|钱珝《代史馆王相公谢监修国史盐铁使表》]]</ref><ref>[[:s:代史馆王相公谢令枢密使宣谕奸邪表|钱珝《代史馆王相公谢令枢密使宣谕奸邪表》]]</ref>当年冬，上表请辞宰相及接任上表请求接任的[[岭南节度使|岭南节度使]]嗣薛王[[李知柔|李知柔]]。<ref>[[:s:代史馆王相公让相位第一表|钱珝《代史馆王相公让相位第一表》]]</ref>次年，再次为此上表。<ref>[[:s:第二表|钱珝《第二表》]]</ref>

== 身亡 ==
此前崔胤和昭宗密谋屠戮宦官时，王抟害怕这样的谋划会引起朝臣和宦官的激烈对抗，建议谨慎。崔胤当时就诬告王抟奸邪，为[[枢密使|枢密使]][[朱道弼|朱道弼]]和[[景务修|景务修]]外应，<ref name=ZZTJ262/>后也怀疑自己先前在光化二年又一次罢相是被王抟所排挤，<ref name=ZZTJ261/>愈发恨王抟。三年（900年）六月，昭宗追究崔昭纬旧恶，崔胤受到牵连被遣出任[[清海军节度使|清海军节度使]]，离京前，他又弹劾王抟。崔胤给盟友[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]写信告知王抟说的话，并要朱全忠上表称崔胤应该留居相位。昭宗召崔胤回长安再拜为宰相，崔胤在朝中威权极盛，因徐彦若、王抟曾揭发崔昭纬而排挤他们，朱全忠也上书称王抟勾结宦官，王抟以[[特进|特进]]、扶危匡国致理功臣、[[开府仪同三司|开府仪同三司]]、守司空、兼门下侍郎、同中书门下平章事、监修国史、判度支、上柱国、鲁国公、食邑三千户、食实封一百户的身份被贬为特进、工部侍郎。<ref>[[s:贬王抟工部侍郎制|s:贬王抟工部侍郎制]]</ref>次日，又贬王抟为[[溪州_(武周)|溪州]]刺史，<ref>[[s:贬王抟溪州刺史制|s:贬王抟溪州刺史制]]</ref>又次日再贬崖州司户参军员外置同正员。<ref>[[s:再贬王抟崖州司户制|s:再贬王抟崖州司户制]]</ref>同日，在蓝田驿赐死，<ref name=NBT10/><ref>[[:s:诛王抟敕|:s:诛王抟敕]]</ref>朱道弼、景务修也皆被赐死。<ref name=ZZTJ262/><ref name=BT20-1/><ref name=BT177/><ref name=NBT223-3/><ref>[[:s:旧唐书/卷184|《旧唐书》卷一百八十四]]</ref><ref>[[:s:新唐书/卷208|《新唐书》卷二百零八]]</ref>王抟推荐的中书舍人[[钱珝|钱珝]]也被贬为[[抚州|抚州]]司马。<ref>[[:s:新唐书/卷177|《新唐书》卷一百七十七]]</ref>
