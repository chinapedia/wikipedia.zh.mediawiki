'''王儒林'''（{{bd|1953年|4月|catIdx=W,王}}），[[河南省|河南]][[濮阳市|濮阳]]人。1973年11月加入[[中国共产党|中国共产党]]。[[吉林农业大学|吉林农业大学]]农业经济专业毕业，[[吉林大学|吉林大学]]经济管理学院经济学硕士。中共第十七届中央候补委员，第十八届中央委员。曾任[[吉林省人民政府|吉林省人民政府]][[省长|省长]]，[[中共吉林省委|中共吉林省委]]书记、省[[人大常委会|人大常委会]]主任，[[中共山西省委|中共山西省委]]书记、省人大常委会主任。现任[[第十二届全国人民代表大会常务委员会|全国人大农业与农村委员会]]副主任委员。

== 早年经历 ==
1969年4月，王儒林响应[[中共中央|中共中央]]号召赴[[吉林省|吉林省]][[抚松县|抚松县]][[插队|插队]]落户，在露水河镇清水河大队当[[知青|知青]]。1970年，进入露水河林业局当工人。1975年，获推荐，进入吉林省直机关第一期青干班学习。学习结束后，进入吉林省农林办公室工作，并开启从政之路；期间，1980年至1985年间，在吉林农业大学农业经济专业学习。

== 仕途 ==
王儒林曾长期在吉林省地方任职。1984年3月，由省委农研室‘下放’[[松原市|松原市]][[前郭县|前郭县]]工作；先后担任八郎乡党委副书记、书记，1985年7月，升任副县长。1986年11月，擢升[[共青团|共青团]]吉林省委副书记、党组副书记；1987年8月，升任共青团吉林省委书记。1991年，王儒林再次赴地方工作，历任中共[[四平|四平]][[市委书记|市委副书记]]，中共[[通化|通化]]市委副书记、市长，[[通化|通化]][[市委书记|市委书记]]、市人大常委会主任。期间，于1990年至1993年，在[[吉林大学|吉林大学]]经济管理学院国民经济计划与管理专业学习，获经济学硕士学位<ref>{{cite book|title=《中国民族年鉴》|year=2001年|publisher=民族出版社|isbn=9787105046645|pages=221-222页}}</ref>。

1996年至1997年，王儒林在[[中共中央党校一年制中青年干部培训班学员列表|中共中央党校一年制中青年干部培训班]]进修。1997年12月，转任中共[[延边朝鲜族自治州|延边朝鲜族自治州]]委书记；并于1998年4月召开的[[中共吉林省委|中共吉林省委]]第七届委员会上，当选省委常委；此后，王儒林先后兼任省委政法委书记、吉林省常务副省长、[[中共|中共]][[长春|长春]][[市委书记|市委书记]]等职。2007年5月，晋升吉林省委副书记、兼省委党校校长，并在同年的[[中国共产党第十七次全国代表大会|十七大]]上当选第十七届候补中央委员。2009年12月，接替[[韩长赋|韩长赋]]，出任[[吉林省|吉林省]][[省长|省长]]。在[[中国共产党第十八次全国代表大会|十八大]]上当选第十八届中央委员。2012年12月，升任[[中共吉林省委|中共吉林省委]]书记，并于2013年1月31日在吉林省第十二届人民代表大会第一次会议上，当选[[吉林省|吉林省]][[人大常委会|人大常委会]]主任<ref>{{cite web|url=http://news.xinhuanet.com/local/2013-01/31/c_114573582.htm|title=王儒林当选吉林省人大常委会主任 巴音朝鲁当选吉林省省长|publisher=新华网}}</ref>。

2014年9月，因[[2014年山西官场地震|山西官场塌方性腐败]]，中共中央宣布，任命王儒林为[[中共山西省委|中共山西省委]]书记<ref>{{cite news|url=http://news.ifeng.com/a/20140901/41816497_0.shtml|title=王儒林任山西省委书记 袁纯清不再担任 |date=2014-09-01|publisher=凤凰网|accessdate=2014-09-01}}</ref>，并在9月29日，当选[[山西省|山西省]][[人大常委会|人大常委会]]主任<ref>[http://news.xinhuanet.com/renshi/2014-09/29/c_1112680670.htm 王儒林当选山西省人大常委会主任]</ref>。自2014年9月王儒林履新至2016年1月，山西省各级纪检监察机关立案28668起，处分31164人，包括结案处理和正在立案查处的厅局级干部129人，移送司法机关34人；结案处理和正在立案查处的县处级干部1565人，移送司法机关157人<ref>{{cite web |author1=曾建中 |title=反腐优等生王儒林“不正常”卸任 带来哪些信号 |url=https://new.qq.com/cmsn/20160706/20160706048813 |website=腾讯 |accessdate=2018-12-28 |date=2016-07-06}}</ref>。2016年6月30日，卸任中共山西省委书记<ref>{{cite web|url=http://news.ifeng.com/a/20160630/49272010_0.shtml|title=骆惠宁任山西省委书记 王儒林不再担任|publisher=凤凰网|accessdate=2016-06-30}}</ref>。

2016年7月2日，十二届全国人大常委会第二十一次会议任命王儒林为[[第十二届全国人民代表大会常务委员会|全国人大农业与农村委员会]]副主任委员<ref>{{cite web|url=http://news.163.com/16/0702/19/BR0BOII40001124J.html|title=全国人大常委会:王儒林罗志军强卫赴全国人大任职|publisher=网易|
accessdate=2016-07-02}}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。

== 两会念稿子事件 ==
2014年3月10日，中央政治局常委、中纪委书记[[王岐山|王岐山]]参加全国两会吉林代表团审议并发言。在王发言后，吉林省委书记王儒林按流程作总结發言时，被王岐山要求“讲短点”。王儒林由于要照本宣科已经打印好的演讲稿，便回答“可能短不了”。王岐山便指着王儒林手中的稿子，批评道：“我刚才又没稿子，你怎么知道并事先打印出来那么多呢？这不是形式主义么？你不用念了！”事后，这一新闻被新京报等媒体报道后引起网络热议。但随后，遭到吉林省方面要求删除网络稿件<ref>{{cite web|url=http://cn.nytimes.com/china/20140313/c13formalism/ |title=王岐山批评吉林省委书记言论被删除 |publisher=纽约时报 }}</ref>。
