'''王正福'''（{{bd|1947年|9月||catIdx=W}}），[[贵州省|贵州省]][[凯里市|凯里市]]人，1984年11月加入中国共产党，1969年9月参加工作，贵州农学院林学系林业专业毕业，大学学历。

早年毕业于[[贵州农学院|贵州农学院]][[林学|林学]]系[[林业|林业]]专业，后供职于[[贵州省|贵州省]][[从江县|从江县]]林业局。1987年，升任贵州省[[从江县|从江县]]县长。1991年，任从江县委书记。1993年，改[[凯里市|凯里市]]市长。1996年，任[[黔东南州|黔东南州]]州长。1998年，升任贵州省副省长。后兼任省纪委书记<ref>{{cite web |title = 贵州省委常委、省纪委书记  王正福 |url= http://www.gz.xinhuanet.com/ztpd/zfzx/contents/ld/ld-wzf.htm |publisher = 新华网 |accessdate = 2012-03-03}}</ref>。2010年，担任贵州省政协主席<ref>{{cite web |title = 王正福当选贵州省政协主席 接替黄瑶职务 |url = http://news.163.com/10/0123/18/5TNV454I000120GU.html |publisher = 网易 |date=2010-01-23 |accessdate = 2012-03-03}}</ref>。2013年1月，王正福不再担任贵州省政协主席<ref>{{cite news|title=王富玉当选省政协主席|url=http://news.gog.com.cn/system/2013/01/30/011951577.shtml|accessdate=2013-07-25|newspaper=贵州商报|date=2013-01-30|deadurl=yes|archiveurl=https://web.archive.org/web/20130629103027/http://news.gog.com.cn/system/2013/01/30/011951577.shtml|archivedate=2013-06-29}}</ref>。后调任[[中纪委|中纪委]]第九巡视组组长。
