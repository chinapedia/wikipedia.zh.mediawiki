'''王業'''（1934年-2016年2月20日），台灣省台南縣人，素人藝術家，畫風大膽，被藝術家[[朱銘|朱銘]]稱作「台灣[[達利|達利]]」<ref name="聯合報2016">{{Cite news|url=http://udn.com/news/story/7314/1523299-創作個展中-「台灣達利」王業83歲辭世|title=創作個展中-「台灣達利」王業83歲辭世|last=范|first=振和|date=2016-02-25|work=聯合報|accessdate=2016-02-26|location=花蓮}}</ref>。

== 生平 ==
王業出生自[[台南市|台南縣]][[安定區_(台灣)|安定鄉]]，年少喪親，藉著繪畫排解痛苦<ref name="羅芙奧2014"/>。年輕時北上到台北打拼，16歲時，因為對[[皮影戲|皮影戲]]感到好奇，開始鑽研寺廟中的雕飾背後代表的意義與表現技巧，但後來因為師傅擔心學徒技巧超過而中途作罷。在街頭到處流浪了兩年、兜售零食，直到家中二哥幫他找到新的老師得以學習繪製玻璃的技術<ref name="羅芙奧2014"/>。軍旅生活時，因為過去的學習成果屢屢獲獎，25歲退役後便開始關注傳統歷史人物之陶器製作<ref name="羅芙奧2014"/>，以及從事民間信仰如寺廟中的繪畫工作與[[浮雕|浮雕]]創作。46歲後開始進行[[水墨画|水墨]]創作<ref>{{cite web|title=台灣樸素藝術家聯展------林淵.王業|url=http://artnews.artlib.net.tw/ArtProperty/artcontents24.html|accessdate=2016-02-26|date=2007-06-01}}</ref>，50歲後開始進行藝術性更大的創作，後常在花蓮舉辦展覽<ref name="公視2013">{{cite news|title=「台灣的達利」王業 花蓮展出畫作|url=http://news.pts.org.tw/article/235810?NEENO=235810|date=2013-03-20|author=溫嘉楷|location=花蓮|accessdate=2016-02-26|publisher=公視|work=台語午間新聞}}</ref><ref>{{cite news|title=台灣原生藝術家 王業＆王甘榮父子聯展|url=http://www.ksnews.com.tw/index.php/news/detail/0000538898|date=2014-02-05|publisher=更生日報|accessdate=2016-02-26|author= 張小菁}}</ref>，去世之時也是在花蓮舉辦個展之時。遺體安葬於台北市[[天母|天母]][[台北市南海普陀山慧濟寺|南海普陀慧濟寺]]<ref name="聯合報2016"/>。

== 藝術成就 ==
作品曾於1988年至美國文化中心展出，1991年至[[金山|金山]]設下工作室進行創作，1992年朱銘文教基金會贊助出版畫集《王業畫夢: 湧動色彩的魔術師》。因為王業喜歡到處尋訪、透過[[酒|飲酒]]助興產生創意，其作品都十分生動跳脫常見的敘述風格，許多神靈鬼怪的繪製<ref name="公視2013"/>，也受到法國傳赫尼樸素藝術館學術顧問羅宏·董向·馬克斯的讚賞，認為是「原生藝術的偉大職業畫家」<ref name="聯合報2016"/>1994年於台北縣立文化中心舉辦「東方綺夢幻影世界」個展，1997-1998年世界巡迴展出「台灣樸素藝術」，2007年於台南文化中心發佈「南瀛樸素之美」聯展 ，2012年在花蓮維納斯藝廊舉行「原生台灣-王業的繪畫世界」巡迴展。<ref name="羅芙奧2014"/>2014年，其作品〈造福〉獲得香港藝術集團看中，在拍賣會上以70萬元的起標價拍賣，並以91萬元台幣價格成交<ref>{{cite news|title=臺灣達利！王業畫作「造福」 進香港秋拍會|url=http://www.artworld.tw/content/meet/act.aspx?id=1817|publisher=中廣新聞網|date=2014-11-12|author=梁國榮|accessdate=2016-02-26}}</ref><ref name="羅芙奧2014">{{cite web|title=藝術家簡歷：王業 |url=http://ravenel.com/artist.php?id=727&lan=tw|year=2014|accessdate=2016-02-26|publisher=羅芙奧藝術集團}}</ref>。2015年在花蓮舉辦《不必盡出於理~王業＆上山的對話》交流展<ref>{{cite news|title=藝術不必盡出於理 收藏家觀點的畫展|url=http://eradio.ner.gov.tw/news/?recordId=21245&_sp=detail|author=趙佳韻|publisher=國立教育廣播電台|date=2015-08-20|accessdate=2016-02-26|archive-url=https://web.archive.org/web/20170305103338/http://eradio.ner.gov.tw/news/?recordId=21245&_sp=detail|archive-date=2017-03-05|dead-url=yes}}</ref>。2016年在花蓮舉辦個展《猴吉花蓮港～王業的神佛慶典與奇幻劇場》<ref name="聯合報2016"/><ref>{{cite news|title=王業病逝 經紀人推崇原生藝術留典範|publisher=中央通訊社|url=http://www.cna.com.tw/news/ahel/201602240416-1.aspx|date=2016-02-24|accessdate=2016-02-26|author=李先鳳|location=花蓮}}</ref>。

== 家庭 ==
