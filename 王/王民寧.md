'''王民寧'''（{{bd|1905年||1988年|12月23日}}），原名'''長裕'''，字'''一鶴'''，[[臺灣|臺灣]][[半唐山|半唐山]]政治人物，[[中華民國陸軍|中華民國陸軍]][[中將|中將]]，[[中國化學製藥|中國化學製藥]]創辦人。

== 早年生平 ==
王民寧為[[臺北|臺北]][[樹林|樹林]]人。早年就讀[[開南商工|私立台灣商工學校]]，與[[王井泉|王井泉]]、[[蘇紹文|蘇紹文]]同為同學。後來赴中國，入[[北京大學|北京大學]]就讀經濟。

== 戰後 ==
1945年[[日本投降|日本戰敗]]投降後，王民寧成為[[國民政府接管臺灣|國民政府接管臺灣]]的第二號人物。亦是當時少數臺籍將領，時任臺灣省警備總司令部副官處處長。1945年10月5日，與[[葛敬恩|葛敬恩]]同赴臺北。當時由於葛敬恩心懷恐懼，王民寧遂成為第一位步下飛機露臉的中華民國接管官員。

1947年3月[[二二八事件|二二八事件]]後，王民寧調任[[臺灣省政府警務處|臺灣省政府警務處]]處長，在二二八事件後的清鄉中，王民寧台灣商工的同學省參議員[[林日高|林日高]]曾遭警方逮捕，王民寧在得知消息後密令所屬不得對外聲張，並親自督促下屬連夜趕作筆錄，天一亮便把林日高連案帶人移送台灣高等法院檢查處，使林日高未落入[[警總|警總]]手中，未受皮肉之苦，當下保存一命。{{r|辛|page=368}}同年底當選[[第一屆國民大會臺灣省代表|第一屆國民大會]]台灣省台北縣代表。

1949年，王民寧調任[[總統府|總統府]][[中將|中將]][[參軍|參軍]]、後歷任[[臺灣省政府|臺灣省政府]]委員及[[光復大陸設計研究委員會|光復大陸設計研究委員會]]委員等職。<ref>{{Cite web|url=http://shulintrc.blogspot.tw/2017/02/blog-post.html|title=王民寧與二二八相關文獻|accessdate=2018-03-30|author=forest|date=2017-02-28|publisher=}}</ref><ref>{{Cite web|url=http://web1.knvs.tp.edu.tw/web/pagech4-04.htm|title=第四節 校友相挺，延續開南於不墜|accessdate=2018-03-30|author=|date=|publisher=}}</ref>

1947年，台灣商工學校與私立開南商業學校等校合併為「私立開南商工職業學校」，由周延壽出任校長。二二八事件後，王民寧出任[[臺北市私立開南高級商工職業學校|臺北市私立開南高級商工職業學校]]首任董事長。

== 創辦中國化學製藥 ==
1952年，王民寧與[[林柏壽|林柏壽]]、[[林宗賢|林宗賢]]等人，與[[陳啟川|陳啟川]]、[[巫永昌|巫永昌]]集資創辦[[中國化學製藥|中國化學製藥]]。

1954年，王民寧獲[[國民黨|國民黨]]提名參選[[臺北市|臺北市]][[1954年中華民國縣市長選舉#臺北市|第二屆民選市長選舉]]，並得到國民黨當局的全力支持。然而最後卻意外敗給[[黨外|黨外]]的[[高玉樹|高玉樹]]。據住在[[雙園區|雙園區]]崛江町的[[康寧祥|康寧祥]]回憶，當時每到中午一到，左鄰右舍就會到王民寧的競選總部報到，吃大魚大肉。飽餐一頓後再抱王民寧的選舉傳單出來，直接就送去高玉樹的競選總部的火爐裡，然後換高玉樹的傳單去發，也就是「吃王民寧的飯，投高玉樹的票」。{{r|康|page=29}}此事後來獲[[陳重光_(1913年出生)|陳重光]]向他證實，陳重光是國民黨員，奉命幫王民寧競選，但私下又跟高玉樹很熟，兩邊的競選總部都會去跑。有次他在高玉樹的競選總部後頭發現一個[[焚化爐|焚化爐]]一直在燒，走近一看發現燒的都是王民寧的傳單。想不到國民黨員再多，特務再多，還是敵不過民心向背。{{r|康|page=29}}

1961年，王民寧等人邀請[[戴德發|戴德發]]出任中國化學製藥總經理，然而之後王民寧與戴德發發生衝突，最後戴德發去職。

王民寧晚年[[中風|中風]]。1986年，王民寧退出開南商工董事長一職，由[[廖欽福|廖欽福]]接任。

1988年12月23日，病逝。<ref>{{Cite book|title = 第一屆國民大會台灣省台北縣代表王民寧於民國77年12月23日病逝，應依法註銷其名籍。(78)華總(一)義字第0241號(78.01.13),總統府公報第5037號|last = |first = |publisher = |year = |isbn = |location = |pages = }}（建立者取自[[:第一屆國民大會臺灣省代表|:第一屆國民大會臺灣省代表]]）</ref>王民寧逝世後，[[中國化學製藥|中國化學製藥]]為紀念其創辦貢獻，成立「財團法人王民寧先生紀念基金會」，該基金會並於1991年設置「王民寧獎」鼓勵醫藥及學術研究。<ref>{{Cite web|url=http://www.ccpg.com.tw/chinese/03_social/01_detail.aspx?MID=3|title=王民寧基金會 - 中化製藥關係公司|accessdate=2018-03-30|author=中化製藥關係公司|date=|publisher=}}</ref>

== 其他 ==
* 1945年日本投降後，王民寧於10月5日隨[[葛敬恩|葛敬恩]]赴台北[[松山機場|松山機場]]接管臺灣。然而當挺著閃亮[[軍刀|軍刀]]的日本兵列隊相迎時，葛敬恩卻因害怕而躲在飛機上，推王民寧先出來，結果使王民寧成為第一個步下飛機的中國官員。<ref>{{Cite web|url=http://magazine.sina.com/bg/newtaiwan/622/2008-02-21/ba48234.html|title=槍手、賣藥、募款 馬以南作風大膽 拆掉馬英九包裝|accessdate=2019-01-28|author=新台灣新聞周刊|date=|publisher=}}</ref>
* 二戰結束後，王民寧出任臺灣省警備總司令部副官處處長。有次他的眼睛患結膜炎，到臺大附屬醫院眼科求診，想要插隊先看。當時林姓醫師不同意，見王穿軍裝，以為他是[[外省人|外省人]]，聽不懂日文，遂當著日本教授面前用[[日語|日語]]罵他。當王民寧向院長[[杜聰明|杜聰明]][[投訴|投訴]]林姓醫師無禮時，林醫師再罵：「我的チンポ（[[陽具|陽具]]）比你大」。<ref>杜聰明，《回憶錄》（下），頁187。臺北：龍文出版社。1998年6月初版。</ref>結果林醫師被王民寧留置在警備總司令部好幾天。日後[[1954年中華民國縣市長選舉#臺北市|臺北市長選舉]]時，王民寧這件事再被舊事重提，成為落選原因之一。
* 二二八事件後的清鄉中，省參議員[[林日高|林日高]]在臺北縣與新竹縣（1949年前新竹縣包含桃園）交界遭警方逮捕。當時任省警務處長的王民寧得知消息後後將林日高連夜逕送警務處，並親自坐鎮警務處。督促下屬連夜趕作筆錄，天一亮便把林日高連案帶人移送台灣高等法院檢查處，使林日高未落入[[警總|警總]]手中。林日高在獄中鄰床的[[鍾逸人|鍾逸人]]評論王民寧此舉使林日高方得保存一命，免受皮肉之苦。否則如果消息外洩，警總得到風聲，必定向警務處要人。王民寧即使身為警務處長，也只有眼睜睜看自己的老同學在警總受拷刑。{{r|辛|page=368}}
* 二二八事件後，坊間曾將王民寧與[[蘇紹文|蘇紹文]]都列入「台奸黑名單」。然而曾參與[[二七部隊|二七部隊]]的[[鍾逸人|鍾逸人]]則認為此說有失公道，他認為「王民寧在警總不過當一名無權無勢的副官處長。拿來騙騙台灣人，『台灣人也有當處長的』。至於他之接掌警務處，那是警察惹禍，造成數萬台灣人死亡。正是名聲狼籍的時候，被派來給[[胡福相|胡福相]]（前任警務處長）擦屁股的。由他的任期之短，便不難想見他也是被拿來利用利用罷了。」{{r|辛|page=369}}
