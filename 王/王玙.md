'''王玙'''（{{bd|？||768年|7月6日|catIdx=W}}<ref name=BT11>[[:s:旧唐书/卷11|《旧唐书》卷一十一]]</ref>），[[中国|中国]][[唐朝|唐朝]][[官员|官员]]，因研究巫术及为[[唐肃宗|唐肃宗]]雇用巫师而受其宠信并拜为[[宰相|宰相]]。

== 家世 ==
王玙生年不详，为[[武则天|武则天]]年间宰相[[王方庆|王方庆]]五世孙，而王玙除了高祖父王曒官至[[潞州|潞州]][[刺史|刺史]]、祖父王仲连官至扬府录事参军外，曾祖父王宠、父亲王绍都没有仕官的记载。<ref>《[[新唐书|新唐书]]》卷七十二</ref>王玙年轻时学礼，<ref name=NBT109>[[:s:新唐书/卷109|《新唐书》卷一百零九]]</ref>研究进行祠祭仪式的合适时间。<ref name=BT130>[[:s:旧唐书/卷130|《旧唐书》卷一百三十]]</ref>

== 唐玄宗年间 ==
[[唐玄宗|唐玄宗]][[开元|开元]]（713年—742年）末年，玄宗推崇[[老子|老子]]道学，喜好神仙之事。二十五年（737年），王玙在太常博士任上上言请求在京城[[长安|长安]]东郊筑春坛祭祀[[青帝|青帝]]，玄宗采纳，擢升他为[[侍御史|侍御史]]，充祠祭使。他因为玄宗作祭祀词而得圣宠。自[[汉朝|汉朝]]以来有办丧事时烧[[纸钱|纸钱]]的[[风俗|风俗]]，王玙将这一风俗带到了[[官场|官场]]。王玙祈祷或焚纸钱类似巫师所为，<ref name=NBT109/><ref name=BT130/>学礼者认为这种行为很羞耻。<ref>[[:s:资治通鉴/卷214|《资治通鉴》卷二百一十四]]</ref>

== 唐肃宗和唐代宗年间 ==
唐肃宗[[至德|至德]]元年（756年），王玙累迁[[太常卿|太常卿]]，也因祭神祈福而得宠。[[乾元|乾元]]元年（758年）五月，肃宗罢宰相[[崔圆|崔圆]]、[[李麟_(唐朝)|李麟]]，而以时任太常少卿、知礼仪事的王玙为[[中书侍郎|中书侍郎]]、[[同中书门下平章事|同中书门下平章事]]，为[[实质宰相|实质宰相]]。<ref name=BT10>[[:s:旧唐书/卷10|《旧唐书》卷十]]</ref><ref name=NBT6>[[:s:新唐书/卷6|《新唐书》卷六]]</ref><ref>[[:s:新唐书/卷139|《新唐书》卷一百三十九]]</ref><ref name=ZZTJ220>[[:s:资治通鉴/卷220|《资治通鉴》卷二百二十]]</ref>王玙本就缺乏名望，不被称誉，<ref name=NBT109/>拜相后，因无能，声誉更差。<ref name=BT130/>从此宫禁中日夜祈祷祭祀，[[宦官|宦官]]用事且受供很多，群臣不敢劝谏。<ref>[[:s:新唐书/卷202|《新唐书》卷二百零二]]</ref>当时正值[[安史之乱|安史之乱]]，归降的叛将们不自安，而三司却连年用刑，将很多归降叛将都处死。王玙对议论有耳闻，请求下诏从此三司讯问无果的，一律释放免罪，以此收揽人心。<ref>[[:s:旧唐书/卷50|《旧唐书》卷五十]]</ref>但叛军还是害怕被诛杀而不肯投降，兵祸仍旧无法缓解。<ref>[[:s:新唐书/卷56|《新唐书》卷五十六]]</ref>

六月，在王玙建议下，肃宗在长安南郊之东置太一神坛，并亲自祭祀，命王玙代行祭祀之事。肃宗病，太卜称是山川作祟，王玙遣女巫在[[宦官|宦官]]保护下穿华服乘驿车分别去天下名山大川祈祷。这些女巫趁机向地方官员索贿。其中一个盛年女巫美丽而淫邪，有数十恶少相随，奸诈不法。她驰入[[黄州|黄州]]，在传舍住下。早上刺史[[左震|左震]]来传舍请事，发现门锁着，大怒，破锁而入，将女巫斩杀于庭下，并杀随从少年，没收其赃物十余万。随后左震上书请求以赃钱代贫民租税，并遣宦官回京。肃宗采纳，却没有处罚王玙。<ref name=ZZTJ220/>次年（759年）正月，肃宗从王玙言，祀九宫贵神。二月，遣王玙与[[侍中|侍中]][[苗晋卿|苗晋卿]]分录囚徒。三月，罢王玙为刑部尚书。<ref name=NBT109/><ref name=BT130/><ref name=BT10/><ref name=NBT6/><ref>[[:s:新唐书/卷140|《新唐书》卷一百四十]]</ref><ref>[[:s:资治通鉴/卷221|《资治通鉴》卷二百二十一]]</ref>

先前彭王府谘议参军[[韩滉|韩滉]]兄知制诰[[韩汯|韩汯]]起草王玙拜官之词的时候不加虚美之辞，王玙很记恨他，拜相后，诸使奏用韩滉兄弟时，王玙都授以冗官。王玙罢相后，群臣们都认为韩滉受屈，韩滉才得以升为[[殿中侍御史|殿中侍御史]]。<ref>[[:s:旧唐书/卷129|《旧唐书》卷一百二十九]]</ref><ref>[[:s:新唐书/卷126|《新唐书》卷一百二十六]]</ref>

同年七月，王玙被任为兼[[蒲州|蒲州]]刺史、充蒲同绛等州节度使。<ref>两唐书本传均作次年七月事，误。此从《旧唐书·肃宗纪》。详见[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》。</ref>三年（760年）四月，在刑部尚书任上改太常卿。<ref name=BT10/>[[上元_(唐肃宗)|上元]]二年（761年），兼[[扬州|扬州]][[长史|长史]]、[[御史大夫|御史大夫]]，充[[淮南节度使|淮南节度使]]。一次肃宗行完南郊礼，以王玙为使持节都督[[越州|越州]]诸军事、越州刺史，充浙江东道节度观察处置使，以本官兼御史大夫，仍任祠祭使。奏[[王栖曜|王栖曜]]为马军兵马使。<ref>[[:s:旧唐书/卷152|《旧唐书》卷一百五十二]]作上元元年事，误。</ref>后又被召回为[[太子少保|太子少保]]，转[[太子少师|太子少师]]。[[唐代宗|唐代宗]][[大历|大历]]三年六月卒。<ref name=BT11/><ref name=BT130/>赠[[开府仪同三司|开府仪同三司]]，谥'''简怀'''。曾孙[[王抟|王抟]]，为唐末[[唐昭宗|唐昭宗]]年间宰相。<ref name=NBT109/>
