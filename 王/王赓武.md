'''王赓武'''{{CBE}}（{{bd|1930年|10月9日|catIdx=Wang王}}），中国[[历史学家|历史学家]]及[[教育家|教育家]]<ref name="海外华人社会科學家傳记">{{cite book|title=海外华人社会科學家傳记|url=http://books.google.com/books?id=K8cNAQAAMAAJ|accessdate=19 January 2013|year=1991|publisher=广东人民出版社出版}}</ref><ref>{{cite web |url= http://renwu.hexun.com.tw/figure_5408.shtml |title=王賡武-人物頻道-和訊網 |author=和訊網 |work=hexun.com.tw |year=2013 [last update] |accessdate=19 January 2013}}</ref><ref>[http://www.worldwhoswho.com/views/entry.html?id=wan-0127 WANG, Gungwu] International Who's Who. accessed 1 September 2006.</ref>，研究現代中國史、[[國族主義|國族主義]]、[[海外華人|海外華人]]、華人移民的權威。

在國際關係理論研究中，王赓武認為中國[[天下|天下]]观念更有包容性，中国版图的问题、主权的问题非常复杂，為避免狭隘民族的概念及政策的問題，主張古代中国向来有的[[天下|天下]]观，與民族国家的概念是矛盾的，但可以和理想的国家之间互相照顾、互相支持秩序結合<ref name ="施雨华2013"/>。澳大利亚首任驻华大使费思棻曾半开玩笑地说，王赓武可以当好中国总理，他也是马来西亚[[民政党|民政党]]创办人之一<ref name="施雨华2013">{{cite magazine|url=http://www.nfpeople.com/News-detail-item-2228.html|title=文章-王赓武 民族国家与天下观念-南方人物周刊|author=施雨华|work=南方人物周刊 nfpeople.com|year=2011-12-02|accessdate=21 January 2013}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==簡歷==
王赓武祖籍河北正定，其祖父为官而南迁[[江苏|江苏]][[泰州|泰州]]，后迁上海，其父则到[[南洋|南洋]]教中文。1930年出生于[[荷属东印度|荷属东印度]]（今[[印尼|印尼]]）[[泗水_(印尼)|泗水]]，旋即随双亲迁居[[英属马来亚|英属马来亚]]（今[[马来西亚半岛|马来西亚半岛]]）<ref>{{cite web |url=http://chinese.people.com.cn/n/2012/1112/c42309-19551598.html |title=82岁华人教授王赓武: “复杂”并“简单”着--华人华侨--人民网 |author=人民网 |work=chinese.people.com.cn |year=2012 [last update] |accessdate=19 January 2013 |deadurl=yes |archiveurl=https://web.archive.org/web/20130122153708/http://chinese.people.com.cn/n/2012/1112/c42309-19551598.html |archivedate=2013年1月22日 }}</ref>。1955年获硕士学位。1957年获[[英国|英国]][[伦敦大学|伦敦大学]]博士学位。1957年起先后任[[马来亚大学|马来亚大学]]历史系讲师、教授兼系主任、文学院院长。1968年任[[澳大利亚|澳大利亚]]首都[[坎培拉|坎培拉]]的[[澳洲國立大學|澳洲國立大學]]远东历史系主任与太平洋研究院院长。曾任[[澳洲人文科學院|澳洲人文科學院]]院长、[[亚洲|亚洲]][[历史学|历史学]]家国际协会主席、澳中理事会主席、[[香港|香港]]演艺发展局主席等职。并任[[新加坡国立大学|新加坡国立大学]]东亚研究所所长、《[[南洋学报|南洋学报]]》主编多年。

王赓武在1986年至1995年期间擔任[[香港大学|香港大学]]校长。其前任者為[[黃麗松|黃麗松]]教授，繼任者為[[鄭耀宗|鄭耀宗]]教授。1994年荣获[[日本国|日本国]][[福冈亚洲文化奖|福冈亚洲文化奖]]。

目前為[[中華民國|中華民國]][[中央研究院|中央研究院]]院士及[[新加坡國立大學|新加坡國立大學]]特級教授。於2007年成為[[新加坡國立大學|新加坡國立大學]]有史以來第三位任命為大學教授的學者。<ref>{{cite web |url=http://www.dso.org.sg/news_details.aspx?news_sid=20090320548643799626 |title=存档副本 |accessdate=2016-08-08 |deadurl=yes |archiveurl=https://archive.is/20120729064517/http://www.dso.org.sg/news_details.aspx?news_sid=20090320548643799626 |archivedate=2012-07-29 }}</ref>

==生平及華人研究的選擇==
成長時家里雖保持着中文语境，因外面是多民族共处环境，自小就接触各种语言：汉语、英语、马来语、印度语<ref name ="施雨华2013"/>。

[[国共内战|国共内战]]前在[[國立中央大学|國立中央大学]]读了一年，因戰爭爆發转入新加坡马来亚大学<ref name ="施雨华2013"/>。

1957年马来亚独立，当地华人面临回到中国还是留下入籍的选择。'''王赓武'''选择留下並留心他人选择，使其研究興趣擴展到东南亚各地、美国、澳大利亚、非洲华人的历史<ref name ="施雨华2013"/>。

同時他認為還有一種在地理、職業及文化個性上不同的國族情緒：

{{quotation|然而也有另外一種中國，一個內陸鄉村的中國。這個中國的文化遺產和精神狀態可以從傳統官員及文人的價值觀找到，現則是普遍地通過農民對社會文化價值變遷的懷疑中表達出來。對這沉默的大多數人來說，過去十年經濟奇蹟仍是遙遠的，若還不到遙不可及的程度的話。經濟發展在他們眼中，被認為是由海外外部力量所產生的，而其帶來的好處，也只有在沿海地區和少數幾個大城市才看的到。一直與他們相伴的是，發達區域與內陸區域間生活水平的尖銳反差，還有，各種道德敗壞的刺耳故事。如果精神腹地的國族主義訴求的被激起的話，與沿海發達省分不同的是，他們會訴諸相當不同的記憶和本能。}}

他進一步指出，在中國近現代史上這兩種國族情緒的差異，鼓勵另外一種型態的復興民族主義來恢復中國一貫奉行的大陸的世界觀：

{{quotation|國家需要拯救，要回歸到有更好的計劃及及對那些不道德及邪惡的人給與更緊更強的控制。這包括糾正內陸及沿海地區間日益不平衡的發展上。無論在時間及空間上，這復興民族主義更為深遠的具體表現，就是要恢復中國一貫奉行的大陸的世界觀。根據這大陸世界觀，海洋利益已經過於狹窄和僅為自我服務，從這些擁有大陸世界觀人的眼光來看，中國沿海在整個歷史的行為證明了這一點。位於中央內陸的官員及士兵長久以來懷疑在周邊地區熱情商人的忠誠度。近代中國史上，農民革命家戰勝沿海國族主義者的領導，點出一個既定模式；而文化大革命的許多特徵文革也指向同一種模式。有一股內在力量推往拉大陸中國的歷史位置，恢復其優勢，那種可以超越較富有且有可疑算盤的海洋中國的優勢，而這心態認知的海洋中國，包含了香港與台灣，他們代表的正是財富的極至化身……
更重要的是中國最近內部的發展。經濟改革愈成功，就更威脅到現行的政治制度。鄧小平的大膽的改革已挑戰共產黨，使其之排除僵化和無效率，使其從“死亡之手”的中央計劃經濟走向以“有形之手”和“無形之手”的新組合治理。放鬆管制已經釋放出比預期中更大的能量。經濟增長一直都不是均衡的，受惠的沿海地區的發展速度，遠遠超過相對受到忽視的內陸。政府的許多部門也未能與時並進。這情況讓積極進取和大膽的人受惠，但不幸的，受惠的也包括貪婪和腐敗的人。因此，大多數百姓的情緒是需要更一個強有力的方向感，需要更多對經濟成長的制衡監督，甚至懷念起當所有事情是比較可預測的，更踏實的往日時光。}}
