'''华阳县'''，[[中国|中国]]旧[[县|县]]名，位今[[四川省|四川省]][[成都市|成都]]市区东部、南部。[[唐朝|唐朝]]时由[[成都县|成都县]]划出，[[两县同城而治|两县同城而治]]逾千年，构成了今成都市市区。今成都市[[成华区|成华区]]、[[锦江区|锦江区]]、[[武侯区|武侯区]]、[[双流区|双流区]]的部分区域曾属华阳县<ref name=":0">{{Cite news|url=http://news.chengdu.cn/2015/0425/1683829.shtml|title=江原、崇宁、华阳……成都历史上消失的古县|author=朱鸿伟|date=2015-04-25|work=华西都市报|accessdate=2018-08-29}}</ref>。

== 历史沿革 ==
[[唐朝|唐]][[貞觀_(唐朝)|贞观]]十七年（643年），分[[成都县|成都县]]东置[[蜀县|蜀县]]，因为[[蜀郡|蜀郡]]郡治[[成都府|成都府]]的附郭县而得名。[[乾元_(唐朝)|乾元]]元年（758年），更名为华阳县，属[[剑南道|剑南道]]成都府。得名华阳，据说是因古人论[[蜀|蜀]]之繁富有“地称天府，原号华阳”之说<ref name=":0" /><ref>《[[舆地广记|舆地广记]]》：“昔人论蜀之繁富曰‘地称天府，原号华阳’。县之得名实本诸此。”</ref><ref>《[[元和郡县志|元和郡县志]]》：“华阳本蜀国之号，因以为名。”</ref>。唐朝时，两县同属[[蜀郡|蜀郡]]、[[成都府|成都府]]，皆为成都府[[治所|治所]]<ref name=":1">《[[新唐书|新唐书]]·志第三十二·地理六》：“成都府蜀郡，赤。至德二載曰南京，為府，上元元年罷京......華陽，次赤。本蜀，貞觀十七年析成都置，乾元元年更名。”</ref>。前后蜀时期，华阳县属益州成都府。宋代属[[成都府路|成都府路]]成都府。[[元朝|元]]属[[成都路|成都路]]录事司，隶[[四川等处行中书省|四川等处行中书省]]。元末属[[明夏|大夏]]。[[明朝|明]]属[[四川布政使司|四川布政使司]]成都府。明末属[[大西|大西]]。<ref name="华阳"/>{{R|双流|page=62}}

清初属[[四川省_(清)|四川省]]成都府。[[清朝|清]][[康熙|康熙]]九年（1670年），华阳县并入成都县，[[雍正|雍正]]六年（1728年）复置，仍同属成都府。[[嘉庆|嘉庆]]时属成都府[[成绵龙茂道|成绵龙茂道]]，[[光绪|光绪]]时改 为[[川西道|川西道]]{{R|双流|page=62}}。 [[新唐书|新唐书]]对华阳县的评价为[[唐朝行政区划#名称及级别|次赤]]<ref name=":1"/>，[[元史|元史]]对华阳县的评价为下<ref>《[[元史|元史]]·志第十二·地理三》：“華陽，下。……”</ref>，[[清史稿|清史稿]]对华阳县的评价为[[衝繁疲難|冲、繁、难]]<ref>《[[清史稿|清史稿]]·志四十四·地理十六》成都府......（康熙）九年，省华阳入成都。雍正六年，复设华阳......华阳冲，繁，难。倚。康熙九年并入成都。雍正五年复置。东：大面山。西：西山，亦名雪岭。南：六对、铁炉。锦江一名汶江，自郫县入，迳城南，折而东，会成都之郫江。又折而西，新开河自双流来会，下流入彭山。浣花溪在城东南，一名百花潭。驿同成都（[[锦官驿|锦官驿]]）......</ref>。

[[辛亥革命|辛亥革命]]后，成都府废，1914年起，华阳县属[[西川道|西川道]]，后又废道复省，属[[四川省_(中華民國)|四川省]]。1921年，华阳县与成都县合并为市，至1928年建立省辖成都市，其部分辖区划入成都市。1949年11月，[[中国人民解放军第二野战军|中国人民解放军第二野战军]]发起[[西南战役|西南战役]]。12月9日，[[云南省政府主席|云南省政府主席]][[盧漢|卢汉]]宣布[[卢汉云南起义|云南起义]]。12日，[[中華民國政府|民国政府]]四川地方派系的[[刘文辉|刘文辉]]、[[邓锡侯|邓锡侯]]、[[潘文华|潘文华]]宣布[[彭县起义|起义]]。使解放军迅速完成对成都的合围。27日，解放军在成都会师。30日，[[中国人民解放军第一野战军|中国人民解放军第一野战军]][[司令员|司令员]][[贺龙|贺龙]]率军进入[[成都城|成都城]]<ref>{{Cite web |url=http://dangshi.people.com.cn/GB/151935/170754/10245140.html |title=1949成都解放记 |accessdate=2015-09-17 |author= 胡跃先|date=2009-10-23 |work= |publisher=中国共产党新闻网，来源：四川日报|language=Zh-hans }}</ref>。其后，华阳县隶属[[川西行政区|川西行政区]]下属的[[温江专区|温江专区]]。1959年，双流县被撤销分别并入[[温江县|温江县]]和[[华阳县|华阳县]]，1962年恢复建置。<ref name="晚报" />1965年，撤销华阳县，并入[[双流县|双流县]]。<ref name="晚报">{{cite news|title=华阳当初是古蜀国“三都”之一 曾是“四川首县” 管辖区域直到暑袜街|url=http://news.163.com/16/1230/05/C9GRV5KE000187VI.html|accessdate=2018-02-20|publisher=成都晚报|date=2016-12-30}}</ref>
== 行政区划 ==
清朝末年华阳县除成都城区外另辖9区、6镇、3乡、36场镇<ref name="华阳">{{Cite book|title=中国地方志集成·四川府县志辑⑨ 民国华阳县志|last=[[曾鑑_(光緒拔貢)|曾鉴]]等|first=|publisher=巴蜀书社|year=1992|isbn=7-80523-478-7|location=成都|pages=}}</ref>。[[民国纪年|民国]]二十四年（1935年）华阳县所辖成都城区划归[[成都市|成都市]]，另辖4区、27乡。民国三十一年（1942年）全县共辖6区、2镇、33乡。民国三十七年（1948年）全县共辖3区、2镇、29乡。{{R|双流|page=67-68}}

1950年，华阳县划出顺河乡至[[彭山县|彭山县]]，划入[[仁寿县|仁寿县]]佛洞乡，全县共辖8区、2镇、29乡。1954年全县共辖6区、28乡、1直属镇、3乡级镇。1956-1957年撤销全部[[县辖区_(中华人民共和国)|县辖区]]。1958年实行[[人民公社化运动|人民公社政社合一]]，全县共辖1个镇、24个公社。1959年双流县撤销并划归华阳县、[[温江县|温江县]]，1962年恢复。1965年华阳县并入[[双流县|双流县]]前，共辖3个镇、18个公社{{r|双流|page=68-69}}：

*[[行政建制镇|镇]]：[[华阳镇街道|中兴镇]]、[[中和街道|中和镇]]、[[太平镇_(成都市)|太平镇]]
*[[人民公社|公社]]：中兴公社、鹤林公社、协和公社、中和公社、新兴公社、白家公社、文星公社、万安公社、正兴公社、太平公社、合江公社、平安公社、永兴公社、兴隆公社、三星公社、永安公社、公兴公社、黄佛公社
1981年，为纪念华阳县，改华阳县原县城中兴镇为双流县华阳镇。2006年9月，经[[四川省人民政府|四川省人民政府]]批准<ref>[http://www.sc.gov.cn/zwgk/gggs/mz/200701/t20070114_167735.shtml 川府民政[2006<nowiki>]</nowiki>47号]：四川省人民政府同意双流县撤销华阳、中和两镇建制，原两镇所属行政区域分别实行街道办事处管理体制。</ref>，撤销华阳镇，设立双流县[[华阳镇街道|华阳街道]]<ref>{{Cite web|url=http://www.xzqh.org/html/show/sc/18219.html|title=双流区历史沿革_行政区划网(区划地名网)|accessdate=2018-08-28|work=www.xzqh.org}}</ref>。2013年，成为[[四川天府新区|四川天府新区]]党工委（管委会）、[[四川天府新区#天府新区成都直管区|天府新区成都]]党工委（管委会）驻地。

=== 县治 ===
华阳县自建置起即与[[成都县|成都县]]同为[[成都府|成都府]][[附郭县|附郭县]]，治所设在成都城内，与成都县分治成都城。1949年11月，华阳县治所由成都市正府街迁至中兴场（今[[华阳镇街道|华阳街道]]），1950年3月又迁至中和场，1951年3月复迁至中兴场，直至撤县。{{R|双流|page=63}}

== 人口 ==
据[[清朝|清]][[嘉庆|嘉庆]]旧志载，华阳县六甲共有48883户，男113446丁，女92215口，共205661人<ref name="华阳" />。民国二十四年（1935年）全县共76569户，383197人，其中男205021人，女178176人。民国三十八年（1949年）全县共91538户，471059人，其中男251968人，女219091人。{{R|双流|page=104-105}}
