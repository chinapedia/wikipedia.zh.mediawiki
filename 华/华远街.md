'''华远街'''，是位于[[北京市|北京市]][[西城区|西城区]]中部的一条道路。

==简介==
华远街为南北走向，南到[[复兴门内大街|复兴门内大街]]，北到[[大木仓胡同|大木仓胡同]]。全长1500米，平均宽度为24米。1990年至2003年间新建，新皮库胡同（位于如今的大木仓胡同至[[皮库胡同|皮库胡同]]之间）拆除并入<ref>[http://www.bjdclib.com/subdb/laneculture/lane/200907/t20090702_3910.html 华远街，东华流韵，2009-07-02]</ref>。1993年8月18日，[[北京市人民政府|北京市人民政府]]地名办公室发布《关于华远街及相关地名命名的通知》，将西单地区的4条道路分别命名为华远街、[[华远北街|华远北街]]、[[华远东街|华远东街]]、[[华远西街|华远西街]]，这4条道路皆因[[华远地产|华远地产]]而得名<ref>{{Cite web |url = http://www.huayuan.com.cn/summaryText.asp?rid=137 |title = 华远集团1983年-1995年大事记 |author =  |date =  |publisher = 北京市华远集团有限公司 |language =  |accessdate = 2017-07-22 |quote =  }}</ref>。

新皮库胡同地处皮库胡同以北，[[中华民国|中华民国]]成立后称新皮库胡同，当时含如今[[大木仓胡同|大木仓胡同]]西段。1965年北京市整顿地名，又将新皮库胡同的东西段并入[[大木仓胡同|大木仓胡同]]，从此新皮库胡同成为一条南北走向的胡同，北到大木仓胡同，南到皮库胡同。新皮库胡同即如今华远街的北段<ref>[http://www.bjdclib.com/subdb/laneculture/lane/200907/t20090702_4092.html 新皮库胡同，东华流韵，2009-07-02]</ref><ref>王彬、徐秀珊，北京地名典，中国文联出版社，2008年，第217页</ref>。
