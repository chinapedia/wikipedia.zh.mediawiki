'''华安大楼'''，后称'''华侨饭店'''及'''金门大酒店'''（{{lang-en|'''Pacific Hotel'''}}），位于[[上海|上海]][[南京西路_(上海)|南京西路]]104号，对面是[[人民公园|人民公园]]，现在是上海市优秀近代建筑保护单位。建筑由哈沙德洋行设计，工程专家康益督工，江裕记营造厂施工，并在1926年5月建成<ref name="dd">[http://www.shtong.gov.cn/newsite/node2/node71994/node81772/node81774/node81784/userobject1ai108970.html 特色志 >> 上海名建筑志 >> 欧美风情的沪滨广厦 >>华商保险业翘楚：华安大楼（金门酒店大楼、华侨饭店大楼）] 上海地方志办公室</ref>。1989年，华安大楼入选第一批[[上海市优秀历史建筑|上海市优秀历史建筑]]。现时，华安大楼为金门大酒店，由[[锦江国际集团|锦江国际酒店集团]]管理。

== 历史 ==
华安大楼是由华安合群人寿保险公司建造的。华安公司主要的创办者是中国人身保险业的创始人、爱国实业家[[南汇区|南汇]]人[[吕岳泉|吕岳泉]]。[[商务印书馆|商务印书馆]]、新闻报馆、内地自来水公司都是公司的最初客户<ref name="dd" />。原先的华安公司的办公地点位于[[上海外滩|上海外滩]]30号的一个教会房屋。之后华安保险的生意逐渐做大，原先的办公地已经显得过于拥挤了。便于1919年搬迁至江西路31号，次年迁今北京路11号<ref name="dd" />。后在1922年，公司在静安寺路（今南京西路）[[上海跑马厅|上海跑马厅]]对面购买了11亩的地，在其西边2亩的土地上投资70万建造了华安合群人寿保险公司大楼。1926年5月，建筑建成。大楼建成后，三至八层是华安饭店，有客房120多间，底层用于出租<ref name="dd" />。

1939年，大楼被一香港华侨租下，并更名为金门大酒店，成为了[[上海滩|上海滩]]的一家知名酒店。1945年[[抗日战争|抗日战争]]胜利后，为了纪念[[日本|日本]]在芷江向中国军队呈投降书，酒店楼下设立了芷江厅。1950年酒店停业并在1958年改名华侨饭店。当时，华侨饭店与[[锦江饭店|锦江饭店]]、[[国际饭店|国际饭店]]等并称为“八大饭店”<ref name="ad">[http://www.shtong.gov.cn/newsite/node2/node2247/node79045/node79394/node79408/userobject1ai104456.html 上海通志>>第十九卷商业服务业>>第十章饮食服务业>>第五节 旅馆行业] 上海地方志办公室</ref>。1992年，恢复原名，后来在1998年进行了大规模的装修<ref name="dd"/>。

== 建筑特色 ==
华安大楼建筑面积共1.4万平方米，高38.16米，共9层<ref name="ad"/>，是新古典主义建筑，同时采用了多种风格装饰。大楼为[[意大利|意大利]]宫殿式建筑，平面呈工字形。底层外墙为花岗石，墙面为水泥錾假石。大门处分为三段，使用了[[多立克柱式|多立克式]]柱子，而大楼一、二层则采用了[[爱奥尼柱式|爱奥尼克式]]的柱廊。中央顶部设有塔楼，上层为塔什干柱，下层为[[科林斯柱式|科林斯式]]环柱，屋顶的钟楼高为9米<ref name="dd"/>。

== 现时情况 ==
现时，酒店为一家三星级酒店，共有171间客房、4间餐厅和一间酒吧<ref>[http://pacific.jinjianghotels.com/ 金门大酒店酒店综述]</ref>。4间餐厅分别为罗马厅、郁金香厅、丁香厅和西西里厅，前三间皆以[[广东菜|广东菜]]与[[福建菜|福建菜]]盛名，最后一间为意大利餐厅。酒店管理方为[[锦江国际集团|锦江国际集团]]旗下的[[锦江国际酒店集团|锦江国际酒店集团]]。
