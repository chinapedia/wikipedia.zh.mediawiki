'''华文独立中学统一考试'''，简称统考，其所颁发的文凭称为统一考试证书（{{lang-en|Unified Examination Certificate|简稱=UEC}}），是属于[[马来西亚|马来西亚]][[独立中学|华文独立中学]]的内部考试，只有华文独立中学的初中三和高中三学生才可以报考。第一届统考于1975 年举办。

统考的负责单位是[[馬來西亞華校董事聯合會總會|马来西亚华校董事联合会总会]]（董总），为了处理统考事务，[[董教总|董教总]]设立了马来西亚董教总华文独立中学工委会，在该工委会下设统一考试委员会。

== 背景 ==
1961年10月21日，马来西亚国会通过了根据《1960年拉曼达立报告书》制定的《[[1961年教育法令|1961年教育法令]]》，除非接受[[马来西亚政府|马来西亚政府]]献议改制成英文中学，否则将不会获得政府的津贴，另外教育部为华文中学举办以华文为考试媒介的三项考试，即初中三年级考试、华文中学升学考试和华文中学离校文凭考试，只是举办以马来文或英文为考试媒介的中学公共考试。当时不接受改制的华文中学自此被排除在国家教育体系之外成为[[独立中学|华文独立中学]]。

从1962年至1974年，[[华文独立中学|华文独立中学]]没有一个统一的公共考试。有鉴于当时独立中学缺乏一项统一鉴定独中毕业生的水平考试，不利于独中毕业生升学。1973年底，《华文独立中学建议书》的发表为华文独立中学确立了正确的办学路线和方针，指明了华文独立中学发展的方向。当时，董教总设立了统一考试委员会，决定着手为华文独立中学毕业生办理一项统一考试。隔年〈统一课程及考试小组报告书〉出炉，于是筹办马来西亚华文独立中学高中、初中统一考试E式从研拟过程进入执行阶段。经过近一年紧锣密鼓的筹备，第一届马来西亚华文独立中学高中、初中统一考试于是在1975年12月11-12日以及15-17日如期顺利举办。

== 目的 ==
统考的举办目的为：

# 统一衡量各华文独立中学的学术水平；
# 为华文独立中学毕业生的升学和就业制造有利的条件；
# 为本地大专院校和外国高等教育机构提供招生的学术依据；
# 为社会人士提供一项征聘人才的依据。

==机制==
独中也采用了有别于国民中学五年制的教学年限，独中生必须完成六年的中学课程方可正式毕业。因此，独中统考可分为初中统考及高中统考，而高中统考亦分为普通考试及技职科考试。初中统考主要是独中学生完成三年的初中教育的评量方法，而高中统考主要是三年的理工科、文商科或技职科学生的毕业考试。参与统考的学生会获得由董总考试局颁发的'''统一考试证书'''。

2011年3月12日，华文独中统考的执行与管理业务终于获得[[國際標準化組織|ISO]]认证，并成功获得马来西亚、中国、英国和日本认证单位颁发的IS09001:2008质量管理体系认证证书，认证范围为"执行和管理华文独中统一考试"。

尽管统一考试证书不被[[马来西亚政府|马来西亚政府]]所承认，因此独中毕业生无法在马来西亚公立大学深造，然而马来西亚国内超过300所私立高校都接受独中毕业生以统一考试证书进行报读。此外，国外也有超过1000所公私立高等学府同样认可统考的学术水平，接受统考生直接申请入学，这些国家和地区包括: 新加坡、香港、台湾、中国（大陆地区）、日本、印度、澳洲、纽西兰、英国、美国等。

=== 董总态度 ===
联邦政府指统考历史及地理科课程内容不符合国情的说法遭到董总的反驳，董总表示，华文独中统一课程是按照马来西亚多元国情的实际状态以及当前国际中等教育标准所进行编撰。课程遵循国家教育课程标准的方针，在拟订课程标准过程中是以教育部颁布的学科课程标准为蓝本，不会接受更改历史科内容。董总还强调，政府应该从学术角度看待统考，而非政治角度。<ref>{{Cite news|url=https://www.malaysiakini.com/bulletin/375701|title=政府未严谨看待统考文凭课题|author=|date=14-03-2017|work=[[当今大马|当今大马]]|accessdate=07-07-2018}}</ref>

董总也曾经表明，要求独中生进入马来西亚大专院校必须在SPM国文科考获优等是合理的条件，但要求修改历史科内容是不合理的。

2012年，董总还曾揭示国阵成员党马华公会在1995年基于团结原则在国阵达致协议，决定维持不承认统考状态。<ref>{{Cite news|url=http://www.sinchew.com.my/node/1275451|title=马华当年允保持现状不承认统考”．董总公开1995年官方文件|author=|date=02-06-2012|work=[[星洲日报|星洲日报]]|accessdate=}}</ref>

=== 其他组织 ===
土著权威组织主席拿督依布拉欣阿里恫言，若政府承认统考文凭，将不惜把事件带上法庭。

== 最近进展 ==
第14届马来西亚选举进行前，[[國民陣線|国民阵线]]与[[希望聯盟|希望联盟]]双方将统考课题加入各自的竞选宣言，对于[[國民陣線|国民阵线]]来说是第一次。

[[國民陣線|国民阵线]]在2018年4月7日发布的竞选宣言中，华文版宣言表示国阵将“承认统考”，但在马来文原版只说“可以考虑（boleh dipertimbangkan）承认统考”，引起议论。<ref>{{Cite news|url=https://www.zaobao.com.sg/realtime/world/story20180408-849188|title=国阵是否承认统考各说各话|author=|date=08-04-2018|work=[[联合早报|联合早报]]|accessdate=}}</ref>

[[希望聯盟|希望联盟]]的大选宣言中，承诺承认统考文凭，并可凭统考文凭申请进入公立大学（IPTA）及申请成为公务员，但条件是统考生需要报考[[馬來西亞教育文憑|马来西亚教育文凭]]单科马来文并获得优等即可。<ref>{{Cite news|url=http://www.sinchew.com.my/node/1734647|title=与2013年宣言有差异．希盟承认统考也附条件|author=|date=09-03-2018|work=[[星洲日报|星洲日报]]|accessdate=}}</ref>
