'''华夏赛车大奖赛'''，简称“华夏杯”（{{lang-en|Chinese Racing Cup}}）<ref>{{Cite web |url = http://www.chinanews.com/ty/2014/06-10/6265706.shtml |title = 两岸三地首次共办“华夏杯赛车大奖赛” |date = 2014年6月10日 |publisher = 中国新闻网 |language = zh-cn  |quote = 　中新网北京6月10日电 (记者 应妮)中国大陆、台湾、香港及澳门将首次共同举办“华夏杯赛车大奖赛”，这是中国赛车运动史上的第一次。 }}</ref>，由[[中国|中国]][[国家体育总局|国家体育总局]]、[[香港特别行政区政府|香港特别行政区政府]]、[[澳门特别行政区政府|澳门特别行政区政府]]、国际汽车运动联合会（FIA）认证支持，中国汽车运动联合会（FASC）、香港汽车会（HKAA）、澳门汽车总会（AAMC）、中华台北赛车会（CTMSA）联合主办，上海力盛赛车文化股份有限公司承办。是首个两岸四地共同举办的赛事。

== 赛事规则 ==
华夏赛车大奖赛由单一品牌车型统一改装、统一维修后勤服务作为参赛车辆，提供一个公平、高水准的竞技平台。华夏赛车大奖赛每年为一个赛季，采用巡回赛制，[[大陆|大陆]]、[[香港|香港]]、[[澳门|澳门]]、[[台湾|台湾]]各有一站主场，分别由四地汽车协会主办。华夏杯四站比赛分别作为由FIA国际汽联认证的CTCC中国房车锦标赛、TTCC中华台北房车锦标赛、MGP澳门格兰披治大赛车的一部分，三大平台联合推出更高级别的跨区域顶尖赛事。在赛制上，华夏杯采用积分大奖赛，四站积分累积，决出年度团队冠军和个人总冠军。

== 车手阵容 ==
华夏赛车大奖赛的参赛选手由四个协会自主选派，每队5名顶级车手（执国际汽联的C级或以上级别的赛车执照）<ref>{{Cite web |url =http://www.chineseracingcup.com/xingguang_csjj.php?id=9 |title =华夏赛车大奖赛车手介绍 |language =zh-cn |4 = |deadurl =yes |archiveurl =https://web.archive.org/web/20160304133045/http://www.chineseracingcup.com/xingguang_csjj.php?id=9 |archivedate =2016年3月4日 |df = }}</ref>。

== 赛事日程 ==
华夏赛车大奖赛共四站比赛，分别是台湾站、香港站、大陆站、澳门站。2014赛季首战将由台湾队主场开启大幕，8月16日-17日在台湾屏东地区的PIC台湾大鹏湾国际赛车场举行。随后9月20日-21日，位于GIC[[广东|广东]]国际赛车场；10月11日-12日，SIC[[上海|上海]]国际赛车场；11月14日- 15日，澳门东望洋赛道。
