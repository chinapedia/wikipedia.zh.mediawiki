'''莊司雅子'''、'''莊雅子'''<ref name = "受困的思想：臺灣重返世界_2016"> {{ cite book | title = 受困的思想：臺灣重返世界 | url = https://books.google.com.tw/books?id=1UsoDwAAQBAJ&pg=PA190&lpg=PA190&dq=%E8%8E%8A%E9%9B%85%E5%AD%90&source=bl&ots=1e_YNoN_W6&sig=ACfU3U13NwVgAnpaC-snQ5Lvh9ycCklnig&hl=zh-TW&sa=X&ved=2ahUKEwj-_M7tr9jhAhUEGqYKHdB5D0YQ6AEwDnoECAgQAQ#v=onepage&q=%E8%8E%8A%E9%9B%85%E5%AD%90&f=false | publisher = 讀書共和國出版集團／衛城出版, 2016 | author = 作者：吳叡人 | isbn = 9789869211369 | page = 190 | language = zh-tw | date = 2016-07-14 }} </ref>（{{jpn|j=莊司雅子|kn=しょうじ まさこ}}，{{bd|1909年|10月6日|1998年|2月22日}}），原名'''莊嫌'''，後父母改名為'''莊無嫌'''<ref name = "南女風華一世紀_2017"> {{ cite web | title = 南女風華一世紀 | url = https://tngs100.blogspot.com/2017/05/blog-post_45.html | author = 楊淑芬 | language = zh-tw | date = 2017-05-18}} </ref>，臺灣[[嘉義|嘉義]][[朴子|朴子]]人，為臺灣出身的日本教育學者。[[日本|日本]]奈良女子高等師範[[奈良女子高等師範學校|奈良女子高等師範學校]]（現奈良女子大學）畢業後，赴[[旧制广岛文理科大学|廣島文理科大學]]（現[[廣島大學|廣島大學]]）教育學科攻讀，師從專攻瑞士教育學者[[约翰·亨里希·裴斯泰洛齐|裴斯泰洛齊]]的教育史學者{{Internal link helper/ja|長田新|長田新}}，專研德國教育學者[[福祿貝爾|福祿貝爾]]及其幼稚園教育學。

== 生平 ==
1909年10月6日出生於嘉義縣布袋庄，為家中排行第四，祖父莊寬為日本時代布袋庄庄長，其父莊玉成為私塾教師，家中有十位兄弟姊妹。1928年完成自臺南州立第二高等女學校（現[[國立臺南女子高級中學|臺南女中]]）學業後，考上總督府公費赴日[[奈良女子高等師範學校|奈良女子高等師範學校]]並於1932年畢業，旋即回臺南州立第二高等女學校任教三年。1935年再赴日[[旧制广岛文理科大学|廣島文理科大學]]攻讀教育學並受洗為基督徒，三年後取得學位入學研究科，1953年則以「[[福祿貝爾|福祿貝爾]]之研究」取得日本第一位女性文學博士學位，曾擔任[[廣島大學|廣島大學]]助理、講師、助教授、乃至教授。歷經二戰廣島核災使得她畢生追求世界和平。1973年退休後任廣島大學終身名譽教授，及聖和女子大學（後改為聖和大學、現為關西學院大學學教育學部）教授，關注看護、福祉、心理等課題，擔任關注兒童教育之日本保育學會、日本[[裴斯泰洛齊|裴斯泰洛齊]]‧[[福祿貝爾|福祿貝爾]]學會會長，1971年以[[福祿貝爾|福祿貝爾]]研究獲德國[[耶拿大學|耶拿大學]]頒發席勒獎。主要著作為《福祿貝爾研究》（フレーベル研究）、《福祿貝爾研究教育學》（フレーベルの教育学），並與玉川大學教育學者[[小原國芳|小原國芳]]教授共編《福祿貝爾全集》（フレーベル全集），並於1950年代（昭和20年代）初翻譯紹介美國教育學家赫威斯特（{{Internal link helper/en|Robert J. Havighurst|Robert J. Havighurst}}）之著作。

莊司雅子一生獨身，晚年不良於行須以輪椅代步，於1998年2月22日逝世，下葬於[[名古屋|名古屋]]教堂。於戰火中收養的義女莊司安子<ref name = "南女風華一世紀_2017" />於2010年逝世。同為研究[[福祿貝爾|福祿貝爾]]教育學的常磐會學園大學國際教育學部教授荘司泰弘為其甥。

== 理論與主張 ==
莊司雅子畢生專研福祿貝爾教育思想，主張兒童自主獨立，非受兩親所有，順其內在教養。且贊同杜威教育精神，重文化教養、社會教育等。就幼兒教育建樹頗豐，畢生推動人本教育。歷經二戰廣島核災更使得她畢生追求世界和平，其墓誌銘以[[福祿貝爾|福祿貝爾]]名言「kommt, Lasst uns unsern Kindern leben」（來吧，讓我們同我們的孩子生處）為誌。
