'''莊家古厝'''位於[[臺灣|臺灣]][[高雄市|高雄市]][[大樹區|大樹區]][[姑山里|姑山里]]，曾被列為[[高雄市歷史建築|高雄縣歷史建築]]，後因家族因素而暫時除名。

== 沿革 ==
[[File:莊家古厝中庭.jpg|thumb]]
莊家古厝位於[[姑婆寮|姑婆寮]]山頂東側，為莊家[[堂號|堂號]]「天水堂」的家族所有{{refTag|name="319旅行事誌"|{{Cite web|url=http://naturallybread.yam.org.tw/2005-oldhouse/oldhouse-da/oldhouse-da-index.htm|accessdate=2014-11-27|author=阿達碼|publisher=319旅行事誌|title=莊家古厝|date=2005-12-29}}}}{{refTag|name="上下游"|{{Cite web|url=http://www.newsmarket.com.tw/blog/2440/|accessdate=2014-11-27|publisher=上下游News&Market|title=地方行腳：大樹姑婆寮莊家古厝|author=陳坤毅|date=2011-10-05}}}}，早期莊氏族人以種植[[鳳梨|鳳梨]]起家，以儉持家而漸漸發達；清代莊家先祖莊清標兄弟開始建造起大厝，聘請了[[唐山|唐山]]師傅，建材除了砂石外，幾乎都從[[福建省|福建]]運送而來，花費數年的時間，完成[[四合院|四合院]]形制{{refTag|name="上下游"}}{{refTag|name="鄉土教學資源網"|{{Cite web|url=http://content.ks.edu.tw/33k/007_dashutownship/004_kno06.html|accessdate=2014-11-27|publisher=高雄縣鄉土教學資源網|work=大樹舊鐵橋教材-高屏溪右岸大樹巡禮|title=莊家古厝}}}}。日治時期莊姓先祖莊財種[[蕃薯|蕃薯]]養[[豬|豬]]而逐漸累積致富，於[[明治|明治]]末年到[[大正|大正]]初年間又將建築增建{{refTag|name="三腳寮"|{{Cite web|url=http://placesearch.moi.gov.tw/portal_m9_page.php?button_num=m9&cnt_id=54351|accessdate=2014-11-27|publisher=中華民國內政部|work=臺灣地區地名查詢系統|title=大樹區三腳寮}}{{dead link|date=十月 2017 |bot=InternetArchiveBot }}}}；而後任當地[[保甲制度|保正]]的莊其中於1921年擴建成今之模樣{{refTag|name="大樹文史協會"|{{Cite web|url=http://kaoh.org.tw/?p=305|accessdate=2014-11-27|publisher=高雄市大樹文史協會|title=莊家古厝山居無歲月|author=柯武村|date=2012-08-09-}}}}{{refTag|name="區公所莊家古厝"|{{Cite web|url=http://dashu.kcg.gov.tw/WebArticle.aspx?UnID=2de681ac-37b7-4c32-80ff-9e6ff28db704|accessdate=2014-11-27|publisher=高雄市大樹區公所|work=古蹟風貌|title=姑婆寮莊家古厝}}{{dead link|date=十月 2017 |bot=InternetArchiveBot }}}}。至今日仍保留完好，莊家人曾對建築作整修使整體格局完整，並向高雄縣文化局申請為[[臺灣歷史建築|歷史建築]]，因其完整建築具文化價值而登錄，並編列預算維護，但兩年後部分莊家人提出撤銷，為高雄縣史上頭例；縣府經討論後，暫時將預算凍結，並將莊家古厝暫列為歷史建築{{refTag|{{Cite web|url=http://archive.is/K3Hsq|accessdate=2018-07-24|format= |title=主動申請 卻又廢止 登錄歷史建築莊家古厝曇花一現
|chapter= |publisher=文化部|date= }}}}{{refTag|{{Cite news|url=http://ntso.gov.tw/MOC/Code/NewListContent.aspx?id=c184d491-90ac-4757-b18e-63fab2ab1baa|accessdate=2014-11-17|publisher=中國時報公司|work=中國時報|title=主動申請 卻又廢止 登錄歷史建築莊家古厝曇花一現|date=2012-04-11|author=黃怡珍|archiveurl=http://webcache.googleusercontent.com/search?q=cache:FumuQjghCNIJ:ntso.gov.tw/MOC/Code/NewListContent.aspx%3Fid%3Dc184d491-90ac-4757-b18e-63fab2ab1baa+&cd=1&hl=zh-TW&ct=clnk&gl=tw|archivedate=2014-11-27}}}}。近年因常為鄉土戲劇拍攝地，以及[[電視|電視]]報導，漸為人所知，成為大樹區有名的景點。{{refTag|name="三腳寮"}}{{refTag|name="319旅行事誌"}}{{refTag|name="大樹文史協會"}}

== 建築特色 ==
[[File:莊家古厝外埕.jpg|thumb]]
莊家古厝合外埕面積，約有八百坪左右，外埕現用作[[停車場|停車場]]，宅院前堂供奉[[關聖帝君|關聖帝君]]，後堂供奉[[觀世音菩薩|觀世音菩薩]]及莊家祖先牌位{{refTag|name="大樹文史協會"}}{{refTag|name="319旅行事誌"}}。整棟建築年代可分清治時期的四合院和日治增建的前落增建左右護龍兩部分{{refTag|name="區公所莊家古厝"}}，古厝呈長方形，為二落二護龍的四合院，有正廳七開間、門房三間，以[[護龍|護龍]]及過水門廊與三開間的前廳相接，稱為「七包三」的格局{{refTag|name="上下游"}}{{refTag|name="大樹文史協會"}}，為當時鼎盛極致表現{{refTag|{{Cite web|url=http://fund.sme.gov.tw/upload_file/plan/p20130122164015.pdf|accessdate=2014-11-27|format=pdf|title=高雄縣大樹鄉特色產業補助計畫|chapter=景點特色介紹|publisher=高雄縣大樹鄉公所|date=2010-09-10|archive-url=https://web.archive.org/web/20141129041434/http://fund.sme.gov.tw/upload_file/plan/p20130122164015.pdf|archive-date=2014-11-29|dead-url=yes}}}}；亦有建築學者[[楊博淵|楊博淵]]以「二進加門樓的合院形制」稱呼之{{refTag|name="鄉土教學資源網"}}。大門進入後，地面紅磚的排列方式呈「入」字形，以作歡迎訪者之意，外牆空隙則用竹、書做塑雕以表注重節操、節儉和勤讀；壁角則留有貓孔及外壁鳥踏{{refTag|name="區公所莊家古厝"}}{{refTag|name="大樹文史協會"}}。古厝內共有八幅書卷軸[[彩繪|彩繪]]，是以泥塑書卷做為裝飾，開展的書卷中間有摺曲，兩側卷成卷筒狀，中央及左右摺成三個小單位，可彩繪或題字，為[[臺灣府城|府城]]彩繪名家[[陳玉峰_(彩繪匠師)|陳玉峰]]的所作。{{refTag|name="大樹文史協會"}}
