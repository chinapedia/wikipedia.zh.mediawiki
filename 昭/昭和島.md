'''昭和島'''（{{jpn|j=昭和島|hg=しょうわじま|rm=Showajima|f=y}}）是[[東京都|東京都]][[大田區|大田區]]的[[人工島|人工島]]及同區町名。現行行政地名為昭和島一丁目與昭和島二丁目。人口0人（2013年8月1日）<ref>[http://www.city.ota.tokyo.jp/kuseijoho/suuji/jinkou/setai_jinkou/oota_suji2508.html 大田区ホームページ：平成25年8月1日現在]</ref>。[[填海|填海地]]名為京濱第3區埋立地。

== 概要 ==
位於大田區東部。北鄰[[平和島|平和島]]，東鄰[[京濱島|京濱島]]，南鄰[[羽田空港_(大田區)|羽田空港]]，西鄰[[大森東|大森東]]、[[大森南|大森南]]。

與東邊的京濱島之間為[[京濱運河|京濱運河]]，與西邊的大森東、大森南之間為[[勝島運河|勝島運河]]。
町內多為[[工業|工業]]用地，密集的鐵工廠形成[[工業園區|工業園區]]。（羽田鐵工團地）

== 年表 ==
* 1939年1月31日 - 京濱第3區埋立地取得許可。事業主體為東京都。
* 1967年7月18日 - 完全竣工
* 1967年9月30日 -　編入大田區
* 1967年10月20日 -　町名定為「昭和島」。
* 1968年4月15日 -　實施住居表示
* 1985年2月7日 -[[東京單軌電車羽田線|東京單軌電車羽田線]]、[[昭和島站|昭和島站]]開業。

竣工面積：616,044m²

== 地名由來 ==
意為昭和時代建造的島。

== 地域內的町名 ==
* 昭和島一丁目…地域西部。[[首都高速1號羽田線|首都高速1號羽田線]]以西的區域。羽田鐵工團地的第一地區，鐵工所聚集。南部有大田區營[[昭和島棒球場|昭和島棒球場]]。
* 昭和島二丁目…地域東部。首都高速1號羽田線以東的區域。羽田鐵工團地的第二地區，鐵工所聚集，也是東京單軌電車昭和島站與車輛基地所在地。北部有昭和島北綠道公園，南部有[[東京都下水道局|東京都下水道局]][[森崎水再生中心|森崎水再生中心]]（[[污水處理|下水處理場]]）。

== 地域內的公園==
===昭和島北綠道公園===
'''昭和島北綠道公園'''（しょうわじまきたりょくどうこうえん）是本地北部、[[首都高速道路|首都高速道路]]下的公園，園內有綠地與[[網球|網球]]練習場，可進行[[滑輪|滑輪]]與傳接球等活動。（全部免費）

*開園年月日：1975年12月1日
*面積：24800.00m<sup>2</sup>
*交通方式：東京單軌電車羽田線、昭和島站步行5分。另可從大森站搭乘巴士（[[京急巴士|京急巴士]]）至「昭和島二丁目」巴士站下車
===昭和島南綠道公園===
'''昭和島南綠道公園'''（しょうわじまみなみりょくどうこうえん）是本地南部與昭和島運動場（大田區管轄）相鄰的公園。公園內有廣場與步道，廣場在昭和島運動場旁有。（全部免費）

*開園年月日：1975年12月1日
*面積：3218.00m<sup>2</sup>
*交通方式：東京單軌電車羽田線、昭和島站步行5分。另可從大森站搭乘巴士（京急巴士）至「昭和島運動場前」巴士站下車步行2分，

== 交通 ==
京和橋可通往東邊的京濱島。南海橋可通往北邊的平和島。避難橋可通往西邊的大森東。

地域內有[[首都高速道路|首都高速道路]][[首都高速1號羽田線|1號羽田線]]與[[首都高速道路灣岸分岐線|灣岸分岐線]]通過，兩者由島內的[[昭和島JCT|昭和島交流道]]連結。

鐵路方面，島內有東京單軌電車昭和島站。另可搭乘[[大森站_(東京都)|大森站]]出發的[[京濱急行巴士|京濱急行巴士]]，但通勤時間外的班次較少，午間時段約一小時一班。
