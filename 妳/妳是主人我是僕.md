《'''妳是主人我是僕'''》（{{lang|ja|君が主で執事が俺で }}），是日本[[Minato_Soft|Minato Soft]]（{{lang|ja|みなとそふと}}）於2007年5月25日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=http://www.getchu.com/soft.phtml?id=324525|title=君が主で執事が俺で 初回限定版|publisher=Getchu.com|accessdate=2014年3月10日|language=Ja}}</ref>，由[[Minato_Station|Minato Station]]（{{lang|ja|みなとすてーしょん}}）發售[[PlayStation_2|PlayStation 2]]版《妳是主人我是僕〜服侍日記〜》（{{lang|ja|君が主で執事が俺で〜お仕え日記〜}}）和[[PlayStation_Portable|PlayStation Portable]]版《妳是主人我是僕〜服侍日記〜 Portable》（{{lang|ja|君が主で執事が俺で〜お仕え日記〜 ぽーたぶる}}）。<ref>{{cite web|title=君が主で執事が俺で ～お仕え日記～ぽーたぶる|url=http://www.4gamer.net/games/175/G017571/|website=4Gamer.net|language=ja}}</ref>後來改編為[[電視動畫|電視動畫]]、[[漫畫|漫畫]]<ref>{{cite web|title=Kimi ga Aruji de Shitsuji ga Ore de (manga)|url=http://www.animenewsnetwork.com/encyclopedia/manga.php?id=16352|website=Anime News Network|language=en}}</ref>、小說，動畫於2008年1月至3月首播<ref>{{cite web|title=Kimi ga Aruji de Shitsuji ga Ore de (TV)|url=http://www.animenewsnetwork.com/encyclopedia/anime.php?id=8151|website=Anime News Network|language=en}}</ref>。

== 故事 ==
上杉鍊和姐姐上杉美鸠是在[[家庭暴力|家庭暴力]]及單親的恶劣家庭环境下长大的。姐姐和鍊为了脫離父親的暴力，离家出走来到了[[橫濱|七濱市]]。因为没有钱，所以在城市里生活非常艰苦。但是，意想不到的有一天被久远寺家的大小姐看中了……于是姐姐成了女佣，「鍊」成了管家，开始在这里工作……

;{{nihongo|久远寺 森罗|久遠寺 森羅（くおんじ しんら）||声优：[[三咲里奈|三咲里奈]] / [[伊藤静|伊藤静]]}}

;{{nihongo|久远寺 未有|久遠寺 未有（くおんじ みゆ）||声优：[[九條信乃|九條信乃]] / [[後藤邑子|後藤邑子]]}}
: 久远寺三姐妹中的二女，个子是三姐妹中最小的，已經大學畢業了，IQ達到240，估計約20幾歲。本人认为自己已经是一个很优秀的淑女了，所以决不可以叫她二小姐。对自己要求严格，是三姐妹中最努力的一个，才能也很广泛。

;{{nihongo|久远寺 梦|久遠寺 夢（くおんじ ゆめ）||声优：[[天乃空|天乃空]] / [[水森志寿香|水森志寿香]]}}
: 久远寺三姐妹中的三女，性格温和喜好安静，爱好是打扮自己和收集漫画。很尊敬南斗星，二人的关系非常好，个性软弱，因为小时候的影响，很不擅长妖怪和鬼魂一类的话题。

: 久远寺森罗的专人女仆，自称女仆长。做这份工作的时间是久远寺家里仅次于上校的人，擅长烹饪，在新人面前会摆出一付前辈的架子，表现得很强势，是个为了掌握主导权而喜欢欺负人的家伙，但是因为性格问题，常常被人欺负。

;{{nihongo|上杉 美鳩|上杉 美鳩（うえすぎ みはと）||声优：[[北都南|北都南]] / [[一美|一美]]}}
: 鍊的姐姐，未有的专人女仆，能滴水不漏的完成家务工作。一直在为弟弟的幸福着想，鍊遇到麻烦就会找她解决，不过唱歌的难听程度也是超一流的。

;{{nihongo|南斗星|南斗星（ナトセ）||声优：[[神代岬|神代岬]] /[[浅川悠|浅川悠]]}}
: 梦的专人女仆兼保镖，平时为了方便行动，身穿管家服，名字的来历是因为自己是看着南十字星长大的，性格诚实认真，但是因为害羞而容易被戏弄。和梦的关系如同好朋友一样，最喜欢和梦一起散步。很想要一個弟弟，所以有時鍊會像弟弟一樣抱住她、或叫一聲姊姊，效果顯著。

;{{nihongo|上校|大佐（たいさ）||声优：[[巖蟬秋|巖蟬秋]] / [[秋元羊介|秋元羊介]]}}
: 本名田尻耕，為久远寺家的管家，因为那艰深的眼神和给人感觉很强的气质而被梦叫做「上校」，之后这个名字就固定了下来，年轻的时候曾做过佣兵，缺点是有时候喜欢自言自语。

;{{nihongo|Deniro|デニーロ||声优：[[勝どき一分|勝どき一分]] / [[たてかべ和也|たてかべ和也]]}}
: 未有做出来的机器人，拥有高度的[[人工智能|人工智能]]，以及足以當作武器的歌聲，面对未有时，随时不忘记绅士的举止，是未有的好朋友。属于吉祥物一般的存在。

;{{nihongo|清原 千春|清原 千春（きよはら ちはる）||声优：[[飯田空|飯田空]] / [[雨宫侑布|雨宫侑布]]}}
: 管家中的打扫专家，称呼鍊为「鍊哥」，是鍊的朋友。雖然綁著馬尾和蝴蝶結，不過卻是男的。

;{{nihongo|九鬼 扬羽|九鬼 揚羽（くき あげは）||声优：[[神代岬|神代岬]] /[[田中理惠_(聲優)|田中理惠]]}}
: 梦的同班同学，世界有名的九鬼财阀的女儿，身为大小姐的同时也是个武斗派，曾在分年龄组的世界格斗技大赛上获胜。说话的方式也是采用古风，对年纪大的老人很是尊敬，喜歡鍊。
: '''《'''認真和我談戀愛！'''》中再次登場'''

;{{nihongo|武田 小十郎|武田 小十郎（たけだ こじゅうろう）||声优：[[龍田虎次|龍田虎次]] / [[桧山修之|桧山修之]]}}
: 九鬼扬羽的管家，总是和扬羽一起行动，为扬羽着想，忠诚心很高，纯情且单纯、满腔热血的好青年，是鍊的好朋友。

;{{nihongo|稻村 圭子|稲村 圭子（いなむら けいこ）||声优：[[田内夏子|田内夏子]]}}
: 与梦同一年级，好友三人组中的一人，公开场合的队长（暗地里是梦），自家在当地的中华街经营中华料理店，称自己为「[[僕娘|俺]]」，说话方式像男孩子一样又会照顾人，虽然被指缺少女人味，不过本人却毫不在意，坚持我行我素。

;{{nihongo|安娜斯塔西娅·米斯蒂娜|アナスタシア・ミスティーナ||声优：[[水捲真魚|水捲真魚]] / [[千晶まひろ|千晶まひろ]]}}
: 来自北方国家的留学生，与梦同一年级，是个成绩优秀的优等生，但是她的性格和性取向和常人极其不同，本人是重度的受虐狂，是个让人头疼的女孩子。

;{{nihongo|上杉 巖|上杉 巌（うえすぎ いわお）||聲優：[[子太明|子太明]] /[[藤原啟治|藤原啟治]]}}
: 上杉鍊和上杉美鳩的父親，經常喝酒因而對子女上杉鍊和上杉美鳩施以暴力，因為父親的因素而兩人離家出走去投靠久遠寺家，之後找上鍊所在的久远寺家，企圖要帶回姊弟倆，但鍊在森罗的話下醒悟，並且克復他心中的恐懼，被鍊打倒後，以及被久远寺家其他人順道教訓了一番，與女兒美鳩約定好，只要改善自己喝酒和施暴的毛病，姊弟倆就會偶爾回家看他，最後履行約定梳妝打理，不再是之前那留了一身鬍子的醉漢樣子了，並與子女們一同去自己的妻子墓前拜拜。
