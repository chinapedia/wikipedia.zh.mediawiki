'''次氯酸钙'''是[[钙|钙]]的[[次氯酸盐|次氯酸盐]]，[[化学式|化学式]]为[[钙|Ca]]([[氯|Cl]][[氧|O]])<sub>2</sub>，是[[漂白粉|漂白粉]]的主要成分之一，有杀菌性及[[氧化性|氧化性]]。与其性质类似的还有[[氯气|氯气]]及[[次氯酸钠|次氯酸钠]]，但是它们都不如次氯酸钙的稳定性高。

漂白粉的组分包含次氯酸钙与[[氯化钙|氯化钙]]、[[氢氧化钙|氢氧化钙]]和[[水|水]]，真正的分子式可以写为Ca(ClO)<sub>2</sub>·CaCl<sub>2</sub>·Ca(OH)<sub>2</sub>·2H<sub>2</sub>O，<ref>《无机化学丛书》第二卷，铍、碱土金属、硼铝镓分族。北京：科学出版社，1984年。</ref>是一种不易吸水的干粉。

== 制备 ==
次氯酸钙的制备方法有：
* [[氯气|氯气]]通入[[氢氧化钙|氢氧化钙]]溶液中。高浓度时可结晶出Ca(ClO)<sub>2</sub>·2CaCl<sub>2</sub>，大量应用于漂白纸张：
:2 Ca(OH)<sub>2</sub> + 2 Cl<sub>2</sub> → Ca(ClO)<sub>2</sub> + CaCl<sub>2</sub> + 2 H<sub>2</sub>O

* 或氯气通入氢氧化钙与[[氢氧化钠|氢氧化钠]]混合溶液中：
:Ca(OH)<sub>2</sub> + 2 Cl<sub>2</sub> + 2 NaOH → Ca(ClO)<sub>2</sub> + 2 H<sub>2</sub>O + 2 NaCl

== 性质 ==
次氯酸钙在水中的[[溶解度|溶解度]]不大。它的稳定性与含水量成反比，分解反应强烈放热，产物是[[氯化钙|氯化钙]]及[[一氧化二氯|一氧化二氯]]。次氯酸钙与[[二氧化碳|二氧化碳]]反应生成[[碳酸钙|碳酸钙]]，放出[[氯气|氯气]]：
:2Ca(ClO)<sub>2</sub> + 2CO<sub>2</sub> → 2CaCO<sub>3</sub> + 2Cl<sub>2</sub> + O<sub>2</sub>

与[[盐酸|盐酸]]反应生成[[氯化钙|氯化钙]]：
:Ca(ClO)<sub>2</sub> + 4 HCl → CaCl<sub>2</sub> + 2 H<sub>2</sub>O + 2 Cl<sub>2</sub>
 
次氯酸钙水解离子方程式:ClO－+H2O→HClO+OH-(可逆)
