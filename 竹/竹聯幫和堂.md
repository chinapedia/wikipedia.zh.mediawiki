'''和堂'''，[[台灣|台灣]]廣域性黑社會組織[[竹聯幫|竹聯幫]]的直系[[堂口|堂口]]（黑社會團體），成立於1982年，是竹聯幫創幫八大堂口之一。其幕後事業金主為商真。活躍於[[台北市|台北市]]林森北路、錦州街、師大路、敦化北路、八德路、忠孝東路知名百貨商圈一帶做為聚眾與金源來源之處，總成員數不明。'''分會''' 保和會，星和會，狼和會，板和會，義和會，嘉和會，宜和會

== 簡歷 ==
1980年代，竹聯幫開始著手將組織企業化。並在 1982年陸陸續續設立完成最初8個分堂（忠、孝、仁、愛、信、義、和、平）<ref>[http://www.setn.com/News.aspx?PageGroupID=6&NewsID=39684 夜店殺警、惹怒黑白兩道 竹聯幫和堂恐「自尋死路」]</ref>，和堂即為其中之一，主要之經濟來源為地下錢莊、暴力討債、酒店圍事。

== 活動區域 ==
和堂的分會大多活動於大台北地區，或許是因為歷任堂主皆活躍於台北縣市。前堂主文龍兄於板橋地區聲望頗高，現任堂主達哥在任期間成立極道會與板和會。板和會由阿舜接任會長，瑋哥接任副會長。除了台北縣市各分會，其他縣市諸如桃園、新竹、苗栗...高雄皆有主要分會。目前北市酒店圍事頭為阿昌，經常揪眾鬧事與於所圍事之酒店內傷人並引起多方注意。<sup class="noprint Template-Fact">[[Wikipedia:列明来源|<span style="white-space: nowrap;" title="{{lan]]</sup>

== 發展歷史 ==
和堂近年來凝聚力極度不足，<sup class="noprint Template-Fact">[[Wikipedia:列明来源|<span style="white-space: nowrap;" title="{{lan]]</sup>首任堂主周士弘，二任阿典接任後與當時鴻源三巨頭互相配合下盛極一時，阿典因車禍亡後由守智接任，再由阿梅接任，文龍則是阿梅後面幾任了，在前堂主「文龍」卸任後，全省各分會較少有互動。文龍卸下職務後陸續由騾子、阿達上任， 騾子上任期間，曾一度整合各分會，並以巡迴方式交代各分會開立香堂儀式，經整合的分會遍部北、中、南，其中包括澎湖的[[澎和會|澎和會]]。<sup class="noprint Template-Fact">[[Wikipedia:列明来源|<span style="white-space: nowrap;" title="{{lan]]</sup>
高雄[[高和會|高和會]]會長小黑於文龍在任期間，經常南北往返參與堂口會議，近年來較少露臉。位於桃園地區之[[狼和會|狼和會]]，精神領袖為遠在海外的竹聯幫份子[[白狼|白狼]]，由於該團體成長速度快，騾子上任後的香堂儀式，聲勢最為壯大成員最多，後因該時期會長阿夫因管訓，槍砲案入監服刑，衍生周邊非核心成員意圖取而代之，遂使該團體漸漸四分五裂。近年堂主阿達重整狼和會。

== 幫內關係 ==
竹聯幫[[竹聯幫忠義堂|忠義堂]]與和堂份子共同於[[臺北市|臺北市]][[大安區_(臺北市)|大安區]]合開pub，兩幫派發生衝突後，和堂份子將忠義堂份子綽號「大林」的盛威霖押走痛毆，忠義堂自己動手砸毀合開的酒吧洩憤。盛威霖2005年12月2日下午赴[[台北市第二殯儀館|台北市第二殯儀館]]參加公祭時，遭7、8名身穿黑衣的幫派份子分持刀、槍追殺。警方於殯儀館附近橋下停車場與人行道，分別起獲10顆散落的彈殼，經初步勘查，發現現場沒有明顯彈著點，研判歹徒可能是對空鳴警告。<ref>[http://news.ltn.com.tw/news/society/paper/46298 黑幫仇殺 殯儀館外槍響]，2003年12月3日</ref>

[[竹聯幫仁堂|竹聯幫仁堂]][[弘仁會|弘仁會]]與和堂不和，雙方為了爭奪[[台北市東區|台北市東區]]地盤，搶奪酒店[[圍事|圍事]]和販賣[[搖頭丸|搖頭丸]]已經火拚數年。媒體報導忠孝東路上幾家知名酒店都由和堂圍事，弘仁會則是於夜店和pub販賣搖頭丸。<ref>[http://www.nownews.com/n/2010/02/23/750837 2010年2月23日  竹聯幫弘仁會副會長被砍死　黑幫火拚一觸即發│Nownews 社會中心]</ref>弘仁會不滿和堂拿混充劣質毒品坑騙，雙方多次談判不成，演變成兩幫人馬血鬥。<br>
2008年4月15日兩方於基隆殯儀館混戰，和堂分子高國祥遭弘仁會分子刺死。<br>
2008年年4月22日，弘仁會長綽號「小川」陳啟鴻酒在復興南路上之酒店富麗皇家前遭和堂分子綽號「冠軍」吳冠勳伏擊，臀部中彈。也使得陳啟鴻中彈送醫後，隨即被警方逮捕，並直接送往監獄服刑。<br>
2008年12月，和堂副堂主「世豪」也被槍擊，弘仁會副會長卓士傑亦被殺死。<ref>[http://www.chinatimes.com/newspapers/20140809000362-260102 2014年8月9日 月月槍響 搶毒搶地盤搶女人│中時電子報]</ref>由於「世豪」外型酷似藝人于冠華，致使該槍擊案件一度躍上各大媒體版面。[[台北市東區|台北市東區]]時尚夜店「2046」發生槍擊案，竹聯幫和堂份子「世豪」被槍擊，槍手連開三槍。<br>
2010年2月22日晚間，弘仁會副會長卓士傑則於在知名設計師[[溫慶珠|溫慶珠]]於北市仁愛路所開設的咖啡店Khaki遭殺害。卓士傑和兩女一男之友人在餐廳用餐，四個黑衣人闖入將卓士傑狂砍致死。頸部、胸部都中刀，送醫後不治身亡。四名嫌犯行凶後，將兇器丟在巷弄內，再搭計程車逃離現場。<ref>[http://www.nownews.com/n/2010/02/23/741505 2010年2月23日  4煞闖溫慶珠餐廳　砍死疑竹聯成員│Nownews 社會中心]</ref>

== 和堂堂主蔡岳甫命案 ==
竹聯幫和堂堂主綽號菜脯的蔡岳甫，於2003年8月遭綽號「結冰」的角頭蔡仲彬以[[鈍器|鈍器]]圍毆致死。<ref>[http://news.tvbs.com.tw/entry/384905  2004/01/30 竹聯堂主蔡岳甫命案　直擊緝捕現場│TVBS]</ref>蔡岳甫8月2日間為了女性友人遭徵信社員工迷姦的糾紛，和徵信社約在一家海產店談判，後遭三名男子強押上車，當日攔車的男子協助將蔡岳甫抬上計程車後，丟了兩百塊錢車資就不見蹤影，半小時後被丟棄路邊, 計程車司機送醫急救，凌晨不治死亡。 <ref>{{Cite web |url=http://www2.tvbstv.com/entry/406440 |title=2003/08/03 竹聯堂主談判惹殺機　昏迷溪畔不治│TVBS |access-date=2014-09-21 |archive-url=https://web.archive.org/web/20160304080845/http://www2.tvbstv.com/entry/406440 |archive-date=2016-03-04 |dead-url=yes }}</ref>

== 2014年臺北夜店殺警案 ==
{{main|2014年臺北夜店殺警案}}
2014年9月14日凌晨發生在[[臺北市信義區|臺北市信義區]]松壽路的ATT 4 Fun百貨公司內夜店Spark發生員警薛貞國遭50人圍毆致死事件。事件後到案者均為「[[中山聯盟|中山聯盟]]」成員。和堂份子萬少丞和受同屬「中山聯盟」的和堂份子蕭叡鴻指使，帶約20名小弟到場助陣，且找到萬少丞，和四海幫份子周譽騰在場，指揮徒眾攻擊薛貞國。<ref>[http://udn.com/NEWS/SOCIETY/SOC8/8948234.shtml  2014/09/20 萬少丞辯：只拿紅龍柱叫囂│聯合新聞網]</ref>[[臺北市政府警察局|臺北市政府警察局]]於9月17日晚派出500名員警連續掃蕩中山、大安、信義、松山地區的酒店及夜店數天，並宣稱不排除「滅堂」。據媒體報導，目前竹聯幫和堂主要幹部已經跑路，走避中國大陸。竹聯幫總堂則找來四大老組織調查會，要求堂主配合警方辦案。<ref>[http://www.setnews.net/News.aspx?PageGroupID=6&NewsID=40213  2014/09/18 被萬少丞「裝笑維」　警怒嗆「滅堂」逼竹聯幫交人│三立新聞網 SETNEWS.NET]</ref>
根據警方調查糾眾於Spark前聚集者為和堂份子萬少丞。事件發生時萬少丞年僅22歲，其父為[[松聯幫|松聯幫]]大老綽號「萬年」的萬年祥。萬年祥不希望兒子加入幫派，<ref>[http://www.chinatimes.com/realtimenews/20140920001505-260401  2014/09/20 萬少丞涉殺警 嬸嬸挺「不是他的錯」│中時電子報] {{webarchive|url=https://web.archive.org/web/20141130023002/http://www.chinatimes.com/realtimenews/20140920001505-260401 |date=2014-11-30 }}</ref>萬年祥續娶妻的事情，影響到父子間的感情，萬少丞不再與父親同行。<ref>[http://www.appledaily.com.tw/realtimenews/article/local/20140921/473523/  2014/09/21 萬少丞內湖躲藏期間　買飲料遇父舊識苦勸│蘋果日報]</ref>警方找上萬年祥時，萬年祥才知道兒子闖下大禍，痛斥殺警是犯了江湖大忌，好不容易透過朋友聯絡上兒子。<ref>[http://www.nownews.com/n/2014/09/18/1417948  2014年9月18日   萬少丞稱未毆警　爆薛貞國滿身酒味還踹人│Nownews 社會中心]</ref>事發後準備存糧躲藏山區，似乎有意長期與警方抗戰，萬年祥得知消息後，趕緊出面欲尋找兒子下落。<ref>[http://www.appledaily.com.tw/realtimenews/article/local/20140921/473523/  2014/09/21 萬少丞內湖躲藏期間　買飲料遇父舊識苦勸│蘋果日報]</ref>
警方表示，萬少丞是綽號「燕子」的竹聯幫和堂副堂主親信，因曾與和堂另一名大哥「冠軍」糾紛，遭對方砍斷手筋。事件發生時萬少丞左手手筋萎縮開刀，纏著繃帶，但右手仍手持兇器攻擊薛貞國。<ref>[http://udn.com/NEWS/SOCIETY/SOC8/%E6%AE%BA%E8%AD%A6%E6%A1%88%E8%90%AC%E5%B0%91%E4%B8%9E%E3%80%81%E9%99%B8%E8%BB%8D%E9%A6%AC%E5%AF%85%E7%B4%98%E5%85%A9%E5%AB%8C%E7%BE%88%E6%8A%BC%E7%A6%81%E8%A6%8B-8949308.shtml  2014/09/21 殺警案萬少丞、陸軍馬寅紘兩嫌羈押禁見│聯合新聞網]</ref>
