《'''竹林中'''》（{{jpn|j=藪の中|hg=やぶのなか|rm= Yabu no Naka}}，又譯作'''竹藪中'''、'''藪之中'''），是日本作家[[芥川龍之介|芥川龍之介]]在1922年發表於「新潮」月刊一月號的短篇[[小說|小說]]作品。

故事講述一名被殺[[武士|武士]]的身亡經過，故事中各人對事件的說法不一，令劇情更加迷離。[[黑澤明|黑澤明]]所執導的電影[[羅生門_(電影)|羅生門]]中取材自本作品。

本作取材於「[[今昔物語集|今昔物語集]]」卷二十九第二十三話「'''具妻行丹波國男 於大江山被縛語'''」裡的說話內容。至今有百篇以上針對「竹林中」真相作探討的論文，但是沒有一個結論。近年來有人開始借用小說名作為形容各說各話真相不明的情況；因為受到[[黑澤明|黑澤明]]執導的[[電影|電影]][[羅生門|羅生門]]的影響，便將撲朔迷離的情形稱為「[[wikt:罗生门|羅生門]]」。

== 劇情概要 ==
一個[[日本武士|日本武士]]陳屍於竹林中，案件披露後，[[檢非違使|檢非違使]]（[[日本|日本]]古代的官名，類似[[警官|警官]]或[[檢察官|檢察官]]）於是召集了七個相關人員加以訊問，本作品將七個人的証言以[[告白|告白]]的形式表現。最早發現屍體的[[樵夫|樵夫]]、路過的[[僧侶|僧侶]]、辦案的[[差人|差人]]、被捕的[[強盜|強盜]]、懺悔的[[妻子|妻子]]、借[[靈媒|靈媒]]之口出現的[[武士|武士]][[亡靈|亡靈]]等，每個人的証辭都具說服力，但又相互矛盾，各執一詞。全部的真相隱於竹林之中

*[[樵夫|樵夫]]：男性屍體的第一發現者。現場遺留物品是[[繩子|繩子]]與女用的[[梳子|梳子]]。沒有看到[[馬|馬]]和[[刀|刀]]。

*旅行[[僧侶|僧侶]]：兇案發生的前一日，在路上遇到男子與乘坐在馬上的女子。

*檢非違使[[衙門|衙門]]的[[差役|差役]]： 強盜多襄丸被捕時穿著粉紅色的衣服、帶著死者的[[太刀|太刀]]和[[弓箭|弓箭]]並騎著[[馬|馬]]。沒有看見女子。

*老婦：死者是[[若狹國|若狹國]]國府之[[武士|武士]]金澤武弘，是其女婿。同行女子是其妻真砂，也就是老婦的女兒。不知女兒的下落。

*[[強盜|強盜]]多襄丸的自白：殺人的正是我本人。我覬覦武士身邊女子的美貌，於是誘騙了武士且將之綁住，以佔有女子的身體。[[強姦|強姦]]了武士的女人後，我也沒有打算要殺害[[武士|武士]]。直到後來女人要求嫁給真正的强者，我就將武士鬆綁了，堂堂正正的[[決鬥|決鬥]]二十三回合後，我把武士殺死。人是我殺的，請把我[[處死|處死]]罷。

*從[[清水寺|清水寺]]前來之女子的[[懺悔|懺悔]]：因為在丈夫面前被盜賊[[強姦|強姦]]，感到非常羞恥，所以用手中的小刀將丈夫殺了。本來想要隨後[[自殺|自殺]]，卻又辦不到。

*[[附身|附身]]於[[靈媒|靈媒]]的[[武士|武士]][[鬼魂|鬼魂]]的告白：事情發生之後，妻子對於我的存在很厭惡，慫恿強盜殺了我。那強盜竟然因此而憤怒，詢問我是否要殺了妻子，而妻子察覺到危險就丟下我逃跑了。在竹林中只剩下我一人，我感到絕望而厭世，所以用妻子遺落的小刀[[自殺|自殺]]了。

== 訊息 ==
透過《竹林中》，芥川希望帶出「軟弱何在，謊言就何在」的訊息。

[[武士|武士]]的說辭，顯示出他想保持自己的[[士族|士族]]形象，與被殺相比，自殺更是一種表現出[[武士道|武士道]]的行為；至於女人，她意在顯示自己[[貞操|貞操]]的形象，她因為不甘受姦淫之辱而把丈夫殺死再打算自殺，表現出自己被汙辱的只是肉體，靈魂仍是貞潔的。強盜多襄丸則希望表現出自己的[[武藝|武藝]]强者形象，武士不過是跟他在汰弱留强的決鬥不敵的被殺者。

小說中每個人各執一詞，得不出一個真相。唯一能肯定的是每人都藉著說謊來展現理想中的自己，而掩飾現實中自己的軟弱。

== 電影 ==
自從黑澤明將「竹林中」拍成電影後，獲得全世界的好評。電影名為「羅生門」的由來，是借用芥川龍之介另一部短篇小說「[[羅生門_(小說)|羅生門]]」為全劇故事背景的緣故。

以下是改編自「竹林中」的電影作品。

*1950年「'''[[羅生門_(電影)|羅生門]]'''」 - [[日本|日本]]。導演：[[黒澤明|黒澤明]]；演員：[[三船敏郎|三船敏郎]]、[[京町子|京町子]]、[[森雅之|森雅之]]、[[志村喬|志村喬]]等。
*1964年「'''The Outrage'''」 - [[美國|美國]]。導演：Martin Ritt；演員：Paul Newman等。
*1991年「'''Iron Maze'''」 - [[美國|美國]]。導演：[[吉田博昭|吉田博昭]]；演員：Jeff Fahey、Bridget Fonda、[[村上弘明|村上弘明]]等。
*1996年「'''{{lang|ja|藪の中}}'''」 - [[日本|日本]]。導演：[[佐藤壽保|佐藤壽保]]；演員：[[松岡俊介|松岡俊介]]、[[坂上香織|坂上香織]]、[[細川茂樹|細川茂樹]]等。
