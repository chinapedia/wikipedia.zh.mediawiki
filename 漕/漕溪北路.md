'''漕溪北路'''是[[中国|中国]][[上海市|上海市]][[徐汇区|徐汇区]]中部的一条街道，南北走向，北起[[徐家汇|徐家汇]][[肇嘉浜路|肇嘉浜路]]、[[虹桥路|虹桥路]][[徐家汇|徐家汇]]接[[衡山路_(上海)|衡山路]]、[[华山路_(上海)|华山路]]，南至[[中山南二路|中山南二路]]、[[中山西路_(上海)|中山西路]]交会处接[[漕溪路|漕溪路]]。长1488米，宽35米到56米。漕溪北路是上海市区三横三纵主干道西纵的一部分，也是[[沪闵高架路|沪闵高架路]]的起点，是从上海南部进入市中心以及徐家汇的重要途径。[[上海轨道交通1号线|上海轨道交通1号线]]沿漕溪北路地下连接徐家汇和上海体育馆。[[徐家汇站|徐家汇站]]一号口旁有[[徐光启|徐光启]]像。

== 历史 ==
此处原为河道，干涸后于1945年填河筑路。1949年以后拓宽形成交通干道。因位于漕溪路之北并与之相接而命名。1950年漕溪北路翻建成弹街路面，即用小石块铺砌的路面。1973年则再次拓宽并建成钢筋水泥混凝土路面，也就是如今的样子。<ref>{{cite web | title = 徐汇区志 >> 第十六篇市政基础建设 >> 第一章道路 >> 节 | publisher = 上海地方志办公室 | url = http://www.shtong.gov.cn/newsite/node2/node4/node2249/xuhui/node38237/node38239/node63493/userobject1ai23363.html | accessdate = 2013-11-17 | language = zh }}</ref>

== 沿路建筑 ==
漕溪北路两侧原是[[天主教|天主教]]机构集中的区域，西侧为男性机构[[徐家汇圣依纳爵主教座堂|徐家汇圣依纳爵主教座堂]]、[[耶稣会|耶稣会]]会院、大修道院（今[[徐汇区|徐汇区]]人民政府）、[[徐家汇藏书楼|徐家汇藏书楼]]（今[[上海图书馆|上海图书馆]]分部）、[[徐汇中学|徐汇中学]]，东侧为女性机构[[徐家汇圣母院|徐家汇圣母院]]（拯亡会、献堂会和孤儿院）、圣衣院（今[[上海电影制片厂|上海电影制片厂]]）。1970年代在马路西侧建起了[[徐汇新村|徐汇新村]]。1980年代以后新建[[华亭宾馆|华亭宾馆]]、[[东方商厦|东方商厦]]、[[上海体育馆|上海体育馆]]等建築物。
