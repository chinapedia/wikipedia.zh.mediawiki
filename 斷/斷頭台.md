'''斷頭台'''（{{lang-fr|Guillotine}}）是一種執行[[死刑|死刑]]的[[刑具|器具]]，用以將犯人的[[頭|頭]][[斬首|斬去]]。斷頭台由一個高的直立架和一塊刀片組成，而刀片則用繩索懸掛在頂部，當執行死刑時，刀片垂直墜落，頭從身體截斷。根據歷史記載，斷頭台起源於法國並且是該國執行死刑的最主要方法，特別是在[[法國大革命|法國大革命]]期間。在法國大革命之後，包括法國在內的多個國家也在使用斷頭台。在1981年法國廢除死刑前，以斷頭台斬首是該國執行死刑的唯一合法方式。

== 执行过程 ==
死刑犯会在清晨6点被[[五花大绑|五花大绑]]，押出囚室，吃早餐。7—9点，死刑犯被[[捆绑|捆绑]]在囚车上游街示众。9点—11点50分，死刑犯被送上断头台，一般会被捆绑在柱子上或者跪在断头台上，等待[[斩首|斩首]]示众。11点50分时，[[牧师|牧师]]为死刑犯做祷告，让死刑犯忏悔。12点时，死刑犯被斩首，[[头颅|头颅]]会被示众，死刑犯的[[遗体|遗体]]不会被[[松绑|松绑]]，与头颅一起被埋葬在[[公墓|公墓]]之中。

== 歷史 ==
=== 早期的斷頭台 ===
最早的斷頭台至少可追溯至14世纪。[[愛爾蘭|愛爾蘭]]曾於1307年以機器砍去一名犯人的頭顱。在此之前，[[英國|英國]]、[[德國|德國]]及[[意大利|意大利]]都有使用類似機器的文字记錄，但其確切性無從稽考。

[[英格蘭|英格蘭]]的哈里法克斯曾長年設立一座斷頭台，直至1648年以後停用。另外[[蘇格蘭|蘇格蘭]]在1564年起亦有類似的器材。

上述的斷頭台都是用來替代[[斧|斧頭]]的，所以它們的切割器呈斧狀。

=== 法國大革命 ===
[[File:Hinrichtung_Ludwig_des_XVI.png|thumb]]的死刑]]
1789年，[[法國大革命|法國大革命]]開始，國會開始研究新的[[刑法|刑法]]。當時其中一位本職是[[醫生|醫生]]的議員[[约瑟夫-伊尼亚斯·吉约丹|约瑟夫-伊尼亚斯·吉约丹]]提出六點有關建議，認為死刑應以快速及[[人道|人道]]的方式執行，而且應對所有人施同一刑罰以提倡[[平等|平等]]。他的選擇是[[斬首|斬首]]，吉約丹在提出這項方法後受到議員們及報社的嘲諷，因此一蹶不振，此後退出政治。<ref>《劊子手世家》一書所提到</ref> 

1791年，[[法國|法國]]國會通過以斬首作為唯一合法的死刑執行方式。

1792年，由吉约丹醫生（另一說法為安東莞·路易）與一位法國劊子手[[夏爾·亨利·桑松|夏爾·亨利·桑松]]（桑松四世，他於法國大革命中執行了路易十六的死刑）一同合作，又透過一位德國工匠托皮亞斯·施密特動手製造了一台原型機，從此新型斷頭台便正式面世，並以他的姓氏命名。這種斷頭台參考的是[[劍|劍]]的設計。它的刀刃本是與地面平行的，但是在測試中發現有實用上的困難，例如因為受到實體橫切面的阻力而無法順利截斷實體，或者因為阻力而產生[[刃部翻捲|刃部翻捲]]或[[缺口|缺口]]，因此設計者又從大鐮刀的形狀與功能還有成效而得到靈感，而在正式使用前改成斜刃。

一個作者不明的故事宣稱提議用斜刃替代平刃的是具有鎖匠技能的[[法國國王|法國國王]][[路易十六|路易十六]]，現在大多認為這是傳聞而已<ref name="MM">{{cite video| people = Executive Producer Don Cambou | title = Modern Marvels: Death Devices | publisher=A&E Television Networks |date =2001}}</ref>。當時有一位法國醫生[[安東莞·路易|安東莞·路易]]也有份參與建造斷頭台，後者對於死亡的方式很有研究，有些人認為他才是提出斜向刀刃設計的人<ref>{{cite web |title=Joseph Ignace Guillotin |publisher=whonamedit.com |url=http://www.whonamedit.com/doctor.cfm/2275.html |accessdate=9 March 2008}}</ref><ref>《劊子手世家》一書提到，他為著名學者，曾任為巴黎外科醫學院生理學教授，本書認為他才是真正開發斷頭台的人。</ref>。同年4月，斷頭台進行了第一次處決。受刑者是一名[[強盜|強盜]]。

1793年1月，法王[[路易十六|路易十六]]被送上斷頭台處死，同年10月他的王后[[玛丽·安东尼特|玛丽·安东尼特]]也被斷頭台處決；至1794年為止三年內，斷頭台斬首「反革命份子」（當中有[[法蘭西第一共和國|法蘭西第一共和國]][[公安委員會_(法國)|公安委員會]]的異見人士）保守估算超過60,000人。

斷頭台在[[拿破崙|拿破崙]]時代傳入德國，到[[納粹|納粹]]時期仍有使用，主要是對付判死刑的罪犯。若對付政敵則多用毒氣、絞殺、槍殺，例外者為白玫瑰為首成員6人，包括[[蕭爾兄妹|蕭爾兄妹]]，1943年2月22日受審後數小時遭受斷頭台處死。

吉约丹醫生在1814年因肩膀生[[疽|疽]]而過世，非如傳說中的死在源於其姓氏的斷頭台。他的後人更改了自己的姓氏，以免和斷頭台扯上關係。

=== 後期發展 ===
斷頭台在1870年再次被改良。主要的改動在刀刃方面，用意是減少刀刃衝力所造成的損耗。

1939年6月，一名犯下六宗[[謀殺|謀殺]]案的犯人[[歐根訥·魏特曼|歐根訥·魏特曼]]（Eugene Weidmann） 被斷頭台處決後，法國政府認為群眾反應過於強烈，於是禁止公開行刑，改為於[[巴黎|巴黎]]一所監獄中進行。魏特曼的受刑情況曾被拍成紀錄片段，直至現在仍於[[互聯網|互聯網]]上流通。

1977年9月，斷頭台最後一次在法國處決犯人。[[密特朗|密特朗]]於1981年當選為[[法國總統|法國總統]]後全面廢除了死刑。此前法國的死刑都以斷頭臺執行，除了某些[[叛國|危害國家安全]]罪的死刑，才以[[槍決|槍決]]執行。

在[[2015年11月巴黎襲擊事件|2015年11月巴黎襲擊事件]]發生後，極右派政治家[[讓-馬里·勒龐|讓-馬里·勒龐]]要求重新啟用斷頭台用來處決[[恐怖份子|恐怖份子]]。<ref>{{Cite web |url= http://www.thedailybeast.com/articles/2015/11/25/monsieur-le-pen-wants-to-guillotine-terrorists-in-france.html|title= Monsieur Le Pen Wants to Guillotine Terrorists in France|accessdate=2016-07-02 |work= |publisher= The Daily Beast|date= 2015-11-25}}</ref><ref>{{Cite web |url= http://www.express.co.uk/news/world/621098/Reinstate-French-death-penalty-after-Paris-attacks-says-former-NF-leader-Jean-Marie-Le-Pen|title= Far-right French politician wants to bring back death penalty and BEHEAD terrorists|accessdate=2016-07-02 |work= |publisher= Daily Express|date= 2015-11-21}}</ref>

== 在法國以外 ==
在[[比利時|比利時]][[安特衛普|安特衛普]]，1798年至1856年間共使用斷頭台19次。

在[[德國|德國]]，斷頭台又叫做 Fallbeil，歷史上有叫做 Fallschwertmaschine 或 Köpfmaschine。德國的斷頭台採用了更改的設計，包括金屬外框和更重的鍘刀。在[[納粹德國|納粹德國]]時期有16,500個罪犯被斷頭台處決，其中10,000個是在最後兩年處決的。

在[[瑞典|瑞典]]，斬首於1866年成為死刑的唯一方式。斷頭台於1903年被採納。瑞典唯一一次使用斷頭台時是在1910年。死刑於1921年被廢除。

斷頭台於[[越南共和國|越南共和國]]都有被採用。其中一個在[[胡志明市|胡志明市]][[戰爭遺跡博物館|戰爭遺跡博物館]]展出。

== 設計 ==
以下資料是根據[[法國|法國]]的標準。

斷頭台的鍘刀呈梯形，刀刃斜向，重約40公斤；其[[木|木]]製支架高4公尺（也有高4.5公尺的），支柱間距37釐米，斷頭台總重約600公斤。斷頭台設有木枷以固定犯人的頭部；刀片與地面底槽的落差約2.25公尺，垂落的速度可達約每秒7公尺，能快速有效地斬斷犯人的頸項。

== 立場 ==
吉約坦醫生最初引入斷頭台的目的是為了提供一個更人道的方法去執行死刑，很多人也認為犯人的[[頭顱|頭顱]]在被切離身體後很快便失去知覺。但這個觀點曾被質疑及引起爭議。

== 日常用語 ==
現在[[中國|中國]]很多人都以'''送上斷頭台'''比喻給予嚴重的懲罰。

斷頭台的英語guillotine也被用來形容「突然截止對某議案或法案的辯論」。
