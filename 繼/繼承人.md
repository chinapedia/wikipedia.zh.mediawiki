'''繼承人'''指的是當[[被繼承人|被繼承人]][[死亡|死亡]]時，擁有[[繼承權|繼承權]]之人，概括地承繼原先被繼承人一身之權利義務及法律關係上之地位例如：占有，但專屬於被繼承人本人者不在此限。但是一般所使用繼承人一詞的意義也可能非指[[法律|法律]]上之繼承人，而屬事實上承接被繼承人家業、財產、地位、權力、爵位或名望之人，此時的繼承人則是'''接班人'''的意思。

各地[[法律|法律]]、[[习俗|习俗]]对继承人的确定、继承顺序不尽相同。亲属是最为常见的继承人。近亲属中，[[配偶|配偶]]、[[子女|子女]]（除[[遺腹子|尚未出世子女]]、[[养子女|养子女]]外，各国对[[继子女|继子女]]继承权规定差异巨大）、[[父母|父母]]、[[兄弟姐妹|兄弟姐妹]]是继承顺序最近的继承人。

== 东亚 ==

东亚社会长期奉行[[长子继承制|长子继承制]]或[[諸子均分制|諸子均分制]]，通常只有儿子才有继承权，女性作为配偶、女儿通常无继承权。进入20世纪后，东亚各地随着现代化进程，在法律上逐渐摈弃长子继承制，但在社会习俗上仍有保留。

=== 中国 ===
中国传统社会，在[[中国爵位|爵位]]、[[世袭|世袭]][[官职|官职]]方面为[[长子继承制|长子继承制]]，女性皆无继承权。財產方面則採[[諸子均分制|諸子均分制]]，无论是妻子或女儿身份，女性仅财产继承权非常有限，并随着历朝法律不同而变化。明清司法实践中，如被继承人没有儿子，则需要从[[宗族|宗族]]中过继[[嗣子|嗣子]]，继承财产。如有未出嫁的女儿则需为其保有一定份额的财产，以作[[嫁妆|嫁妆]]<ref>{{Cite thesis |degree= |chapter=二、中国古代女性继承权的沿革与内容 |title=《中国古代女性继承权问题研究》 |url=http://d.wanfangdata.com.cn/Thesis/Y2351140 |author=高桐 |last= |first= |year= |publisher= |accessdate=2017-12-24 |date=2013|docket= |oclc=|language = Zh-hans }}</ref>。[[明朝政府|明朝政府]]提倡女性守节，除法律规定[[寡妇|寡妇]]改嫁，无法继承丈夫遗产外，甚至要放弃[[嫁妆|嫁妆]]<ref>{{cite journal en|quotes =

==== 中华人民共和国 ====

中华人民共和国境内的继承事务，遵循1985年实行的《[[中华人民共和国继承法|中华人民共和国继承法]]》。

=== 台湾（中华民国） ===

==== 要件 ====
須符合以下要件方為繼承人：
# 須有[[權利能力|權利能力]]
# 須為[[自然人|自然人]]
# 須符合[[同時存在原則|同時存在原則]]
# 須具備法定之親屬關係

==== 種類 ====
* 配偶繼承人

*配偶，必須為被繼承人死亡當時，尚具有合法配偶身分之人。故雖尚在離婚訴訟中，或提出撤銷婚姻請求，若尚未確定喪失配偶身分，仍具繼承人資格。

* 血親繼承人

依據[[中华民国民法|中华民国民法]]第1138條之規定，血親繼承人有下列四種，並依法定順序依次繼承，若有先順序繼承人存在時，後順序繼承人不得繼承。
#一、直系血親卑親屬：指從己身所出者，但以親等近者為先。如，子女、孫子女、曾孫子女、玄孫子女等。
#二、父母:己身所從出之一親等直系血親尊親屬。但非婚生子女，與其未認領或準正之生父，尚未建立親子關係，故生父並非繼承人。
#三、兄弟姊妹：全血緣（同父同母）或半血緣（同父異母或同母異父）均屬之。
#四、祖父母:己身所從出之二親等直系血親尊親屬，包含內祖父母、外祖父母。
