'''穆家善'''（1961年- ），[[江苏省|江苏省]][[连云港市|连云港市]][[赣榆县|赣榆县]]人。旅美著名画家。[[江苏省赣榆高级中学|江苏省赣榆高级中学]]1979届校友。号“[[青口镇_(赣榆县)|青口]]山人”。

==人物履历==
七岁起受祖父影响，入门书画，师从[[中国画|中国画]]大师[[陈大羽|陈大羽]]教授（[[齐白石|齐白石]]弟子），又经范扬引荐拜师著名人物画家[[范曾|范曾]]教授研学。<ref>旅美著名画家穆家善简介{{cite web |url=http://www.lygmedia.com/2008-12/08/cms90508article.shtml |title=存档副本 |accessdate=2011-04-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20120106004944/http://www.lygmedia.com/2008-12/08/cms90508article.shtml |archivedate=2012-01-06 }}</ref>1983年考入[[南京艺术学院|南京艺术学院]]，1991年开始攻读南京艺术学院中国画硕士研究生课程。 
<ref>穆家善：不改的乡音[http://www.gmw.cn/01gmrb/2008-12/05/content_866102.htm] </ref>1995年，赴美讲学。1998年，在[[陈香梅|陈香梅]]和中国驻美使馆资助，联合48位亚洲著名艺术家，共同创办[[美国亚太艺术研究院|美国亚太艺术研究院]]。陈香梅任董事局主席、[[张仃|张仃]]教授（中国国徽设计者、原中央工艺美术学院院长）为名誉院长，穆家善为首任院长。。<ref>旅美著名画家穆家善简介{{cite web |url=http://www.lygmedia.com/2008-12/08/cms90508article.shtml |title=存档副本 |accessdate=2011-04-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20120106004944/http://www.lygmedia.com/2008-12/08/cms90508article.shtml |archivedate=2012-01-06 }}</ref>

==荣誉奖项==
美国亚太艺术研究院院长、[[蒙哥马利学院|蒙哥马利学院]]中国画兼职教授、[[上海戏剧学院|上海戏剧学院]]客座教授、[[上海大学|上海大学]]艺术研究院兼职教授、上海大学数码艺术学院客座教授、[[厦门大学|厦门大学]]艺术学院客座教授、英国剑桥国际名人传记中心和美国传记研究院终身艺术顾问、美国美术考级评审委员会主席、《中国艺术收藏年鉴》、《中国书画三十年史志》、《水墨画名家经典》国际编委。<ref>旅美画家穆家善北大演讲{{cite web |url=http://www.enet.com.cn/student/xj/hd/200801/20080122298641_1.shtml |title=存档副本 |accessdate=2011-04-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20130601152911/http://www.enet.com.cn/student/xj/hd/200801/20080122298641_1.shtml |archivedate=2013-06-01 }} </ref>
