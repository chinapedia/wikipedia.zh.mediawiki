'''營團03系電力動車組'''（{{jpn|j=営団03系電車|hg=えいだん03けいでんしゃ|rm=Eidan 03 kei densha|f=y}}），是供[[東京地下鐵|東京地下鐵]]（前[[帝都高速度交通營團|帝都高速度交通營團]]）[[日比谷線|日比谷線]]使用的[[通勤型電力動車組|通勤型電力動車組]]。於1988年7月1日開始營運。目前已陸續由東京地下鐵13000系所取代，預計2020年全數退役。

== 概要 ==
為了加快[[日比谷線|日比谷線]]的运行车辆空調化、增强[[日比谷線|日比谷線]]的运能以及替换已经老化的[[營團3000系電力動車組|3000系]]，因此[[帝都高速度交通營團|帝都高速度交通營團]]投放此列車在[[日比谷線|日比谷線]]。<br/>這款列車由[[東急車輛製造|東急車輛製造]]、[[日本車輛製造|日本車輛製造]]、[[川崎重工業|川崎重工業]]及[[近畿車輛|近畿車輛]]負責製造，從1988年到1994年間共生產了42组共336輛。列车的机器设备参考了[[银座线|银座线]]的[[营团01系电力动车组|01系]]和[[东西线_(东京地下铁)|东西线]]的[[营团05系电力动车组|05系]]<ref>{{lang|ja|帝都高速度交通営団「60年のあゆみ - 営団地下鉄車両2000両突破記念 - 」}}<!-- これは営団が発行した公式な資料。 -->を参照。</ref>。此外列车在设计方面也具有「高品质」、「成人视角」的特点<ref>{{lang|ja|『[[铁道爱好者_(杂志)|鉄道ファン]]』1991年9月号「6000系から01・02・03・05系に至るデザインプロセス」を参照}}。</ref>。

== 历史 ==
1987年-1994年，共生產了42组共336輛，其中01编成为[[川崎重工业|川崎重工业]]生产，02编成为[[日本车辆制造|日本车辆制造]]生产，其他的列车为[[近畿车辆|近畿车辆]]和[[东急车辆制造|东急车辆制造]]负责生产。其中1次车的使用目的是为了增强[[日比谷线|日比谷线]]的运能，2次车的使用目的则是为了完全替换已经老化的[[營團3000系電力動車組|3000系]]。其中，2000年3月8日发生的[[营团地下铁日比谷线事故|营团地下铁日比谷线事故]]，导致02编成的03-802号车厢报废，在2001年9月，由[[东急车辆制造|东急车辆制造]]重新制造的车厢替代了已经报废的车厢并重新上线运营。

== 车体构造 ==
每节车厢长18m，拥有3对车门（部分车厢为5对车门），材质为[[铝合金|铝合金]]，使用连续[[焊接|焊接]]工艺，车体主颜色为灰色。列车车头的挡风玻璃与[[營團3000系電力動車組|3000系]]类似，呈曲线状。在挡风玻璃上方还拥有[[方向幕|方向幕]]和[[列车编号|列车编号]]两个显示窗，周边以黑色为主。其中列车两端的[[紧急出口|紧急出口]]为[[内崁式门|内崁式门]]。

== 客室内部 ==
列车客室内部的地板颜色为米色和橄榄绿色，行李架采用[[不锈钢|不锈钢]]网格结构。列车的部分车窗可以打开通风，并设置有灰色窗帘。列车在早期的座席颜色全部为茶色，在2001年-2006年的1-3次车的座席颜色开始更改为红紫色；4次车的座席颜色开始也变为红紫色，但是优先席为青色。车厢内设置有6个[[扬声器|扬声器]]，[[车内紧急通报装置|车内紧急通报装置]]也作了较大的修改。<br/>列车空调采取集中性搭载，先头车6台，中间车7台，也是[[帝都高速度交通营团|帝都高速度交通营团]]中首次装备有空调的车辆。空调使用直流600V，单机功率为48.9kW，其温度会通过计算机调节。其中弱冷房車（弱冷氣車）位于3號車，女性專用車在第1號車。

== 列车驾驶室 ==
驾驶室内主色调为淡绿色，驾驶台颜色为藏青色，驾驶室进深为1,900mm。主操纵设备为旋转盘式，为牵引4段+制动7段+紧急制动，这些操纵设备在[[VVVF|VVVF]]控制方式的列车中有较大的改动。仪表盘可显示的最高速度为120km/h，仪表盘右侧是车辆控制情报管理装置。另外，驾驶室与客室之间的门通常会用遮光幕遮盖，这种方式在之后的[[營團06系電力動車組|06系]]和[[營團07系電力動車組|07系]]中均有所应用。

== 行走设备 ==
*1-4次车及5次车的部分车辆（01-25编成）采用的控制设备是与[[银座线|银座线]]的[[营团01系电力动车组|01系]]相同的[[斩波控制|电机子斩波控制]]方式，并采用[[可关断晶闸管|GTO]]作为[[半导体器件|半导体器件]]。其中为确保“高粘着控制”和“加速度一定控制”及提高车辆性能<ref>{{lang|ja|帝都高速度交通営団「60年のあゆみ - 営団地下鉄車両2000両突破記念 - 」ならびに鉄道ピクトリアル1988年9月号「営団地下鉄日比谷線 03系」を参照}}。</ref>，列车的[[动拖比|动拖比]]设定为1:1，[[起动加速度|起动加速度]]设定为3.3km/h/s。在列车的电动机方面，第01编成、3次车、4次车是[[三菱电机|三菱电机]]制造，第02编成、2次车、5次车（部分）是[[日立制作所|日立制作所]]制造。[[转向架|转向架]]主要采用SS-111・SS-011型的SU摇枕转向架。
*5次车的部分车辆及6、7次车采用的控制设备为[[VVVF|VVVF]]，并采用[[IGBT|IGBT]]作为[[半导体器件|半导体器件]]。在列车的电动机方面，5次车（部分）、6次车是[[日立制作所|日立制作所]]制造，7次车是[[三菱电机|三菱电机]]制造。[[转向架|转向架]]主要采用SS-135、SS-035型的无摇枕转向架。在第26编成以后，电动车设置的主电动机检查盖被废止了。其中[[VVVF|VVVF]]车辆识别标志位于前面玻璃下部，并用“V”标注的贴纸贴着。
*制动方式采用相对便于维修的[[踏面制动|踏面制动]]作为基础制动方式，与ACT联动的制动方式为[[再生制动|再生制动]]和[[电气指令式制动|电气指令式制动]]。
*[[压缩机|空气压缩机]]采用了与[[营团01系电力动车组|01系]]相同的C-2000LA型、电动机采用了交流化的C-2500LB型。此设备在2002年[[半藏门线|半藏门线]]新投入使用的[[营团08系电力动车组|08系]]上有所使用。
*[[逆变器|DC-DC转换器]]可将直流1500V逆变为直流600V，用于供给空调设备<ref>{{lang|ja|なお、冷暖房やCPは[[三相交流|三相交流]]440Vを電源とするため、これらには専用のインバータで三相交流に変換される}}。</ref>。

== 保安装置 ==
列车在初期为了能够乘入至[[伊勢崎線|伊勢崎線]]和[[東横線|東横線]]，在[[營團3000系電力動車組|3000系]]搭载了多种保安装置的基础上，对03系列车的搭载保安装置做了进一步更改：安装有东急型和东武型的[[自动列车停止装置|ATS]]，后来又安装了东急型的新[[列车自动控制系统|ATC]]。不过列车在1997年后，对[[日比谷线|日比谷线]]开始逐渐更新[[列车自动控制系统|ATC]]。随着列车在2013年3月26日不再乘入至[[東横線|東横線]]，东急型的[[自动列车停止装置|ATS]]开始陆续拆除。

== 车辆控制情报管理装置 ==
本款列车装备有车辆控制情报管理装置（Train control Information manegement System，简称'''TIS'''），由[[三菱电机|三菱电机]]在1988年制造，在日本国内的铁路车辆中首次装备。搭载方式为先头车TIS中央装置、中间车TIS单元局模式，使得驾驶员能够时刻看到列车的工作情况，这套监控装置在以后的所有车辆中均有配置（[[營團02系電力動車組|02系]]除外）。

== 5门车厢 ==
1990年9月开始在上午的[[繁忙时间|繁忙时间]]进行缩短上下车时间的试验，为此在1、2、7、8号车厢开始使用5门车，首先装备于第09编成。这种车厢外观以3门车为基础，车门开度均为1,300mm，座席大规模减少，增加了大量站位。其中在平时，第2、4车门可执行[[选择性车门操作|选择性车门操作]]<ref>{{lang|ja|現在は中目黒・北千住などの始発駅のみ終日締切り、発車後に車掌より次の駅から5扉すべて開閉する旨のアナウンスがなされることがある}}。</ref>，并有指示灯加以提示。为保证列车的通气效果，在车厢两端加装有2台[[风机|风机]]。<br/>这样的车厢通常会张贴有「5 DOORS」标志，其中早期的「5 DOORS」标志张贴在[[紧急出口|紧急出口]]的下部，目前移动至[[紧急出口|紧急出口]]的上部。此外，[[日比谷线|日比谷线]]内的各站设置的乘车位置标志，在5门车停靠处会有「5 DOORS」的标志。<br/>由于第09编成的5门车反应良好，5门车在第10-28编成中也开始大规模装备，并应用于早高峰于[[北千住站|北千住站]]发车的列车中（在1993年之前因乘入[[东武铁道|东武铁道]]内会导致排队混乱，仅能在[[日比谷线|日比谷线]]内行走，目前已解禁）。此外5门车在[[东武20000系电力动车组|东武20050系]]中也有所推广。

== 列车向导服务 ==
[[File:Tokyometro03_inside_first_information_board.png|thumb]]
=== 车外 ===
列车车外装载有[[扬声器|扬声器]]，可播放广播、[[发车音乐|发车音乐]]及关门提示音。其中斩波控制车辆的扬声器设置在列车外部的侧面，[[VVVF|VVVF]]控制车辆的扬声器则因为空调原因设置在列车的内部。另外，乘入至[[日比谷线|日比谷线]]的[[东武20000系电力动车组|东武20050系·20070系]]及曾经乘入的[[东急1000系电力动车组|东急1000系]]也装备有同样的扬声器。<br/>列车车头设置有[[方向幕|方向幕]]、[[列车编号|列车编号]]显示器，侧面设置有[[方向幕|方向幕]]。不过4次车的前后及侧面向导屏幕均能显示[[日语罗马字|日语罗马字]]字幕。自5次车以后的[[列车编号|列车编号]]显示器改为LED模式，侧面仅显示[[日语罗马字|日语罗马字]]，不过亮度较4次车略低。
=== 车内 ===
列车客室内的车门上部拥有LED式的[[车内向导表示装置|车内向导表示装置]]和[[门铃|门铃]]，此外还有目的地、站名、换乘路线显示，这些内容均采用自动播放方式。其中斩波控制车辆的[[车内向导表示装置|车内向导表示装置]]多数呈静态播放，[[VVVF|VVVF]]控制车辆的扬声器和[[车内向导表示装置|车内向导表示装置]]可以同步播放<ref>{{lang|ja|ただし、自動放送は2013年3月までは東武線内の英語放送には対応していなかった}}。</ref>。另外，5门车的「本班列车」的播放字幕已经被删除。

== 列车运送过程<ref>{{lang|ja|交友社「鉄道ファン」1996年10月号特集「カラフル営団地下鉄2401両」50ページ参照}}。</ref> ==
=== 1-5次車 ===
列车首先从[[东鹫宫站|东鹫宫站]]出发，经JR[[宇都宫线|宇都宫线]]到达[[久喜站|久喜站]]，之后进入[[东武铁道|东武铁道]]的[[伊勢崎線|伊勢崎線]]到达<ref name="kuki">{{lang|ja|当時は久喜駅でJRと東武伊勢崎線が接続され、貨物列車の連絡があった}}。</ref>[[业平桥站|业平桥站]](今[[东京晴空塔站|东京晴空塔站]])，进入[[千住检车区竹之冢分室|千住检车区竹之冢分室]]，其中东武铁道内使用[[东武ED5060型电力机车|东武ED5060型电力机车]]牵引进入<ref>交友社「铁道爱好者」1988年9月号读者記事参照。</ref>。
=== 6次車 ===
列车首先从[[長津田站|長津田站]]出发，经JR[[横滨线|横滨线]]进入[[东京急行电铁|东京急行电铁]]的[[田园都市线|田园都市线]]到达[[鹭沼检车区|鹭沼检车区]]后进行整备<ref>交友社「铁道爱好者」1993年7月号读者記事参照。</ref>。之后，经[[鷺沼站|鷺沼]] - 二子玉川園（現[[二子玉川站|二子玉川]]） - （经[[大井町線|大井町線]]） - [[大岡山站|大岡山]] - （经旧[[目蒲線|目蒲線]]、[[東横線|東横線]]）- [[元住吉站|元住吉]]折返 - （经[[東横線|東横線]]）- [[中目黑站|中目黑]] - [[千住检车区|千住检车区]]进行自身动力回送<ref>交友社「铁道爱好者」1993年8月号读者記事参照。</ref>。
=== 7次車 ===
列车首先从[[东鹫宫站|东鹫宫站]]出发，经JR[[宇都宫线|宇都宫线]]到达[[久喜站|久喜站]]，之后进入[[东武铁道|东武铁道]]的[[伊勢崎線|伊勢崎線]]到达<ref name="kuki" />[[東武動物公園站|東武動物公園]]后折返，进入[[竹之冢检车区|竹之冢检车区]]。

== 列车大修 ==
按照[[东京地下铁|东京地下铁]]规定，列车需要每4年进行一次全方面检查<ref>{{lang|ja|日本鉄道車両機械技術協会「R＆m」2010年9月号「千代田線16000系車両の登場にあたって」記事参照}}。</ref>。第12年进行'''C修工程'''，第24年进行'''B修工程'''，第36年进行C修工程，第48年让列车停用。目前，1、2次车陆续进入到了大修阶段，目前第01-04编成已经在2012年12月末大修完毕<ref name="Rfan2012-8">{{lang|ja|交友社「鉄道ファン」2012年8月号付録「大手私鉄車両ファイル2012」}}。</ref><ref name="DJ2013-1">{{lang|ja|交通新聞社「鉄道ダイヤ情報」2013年1月号「私鉄車両のうごき」125-127頁記事}}。</ref><ref name="DJ2013-4">{{lang|ja|交通新聞社「鉄道ダイヤ情報」2013年4月号「私鉄車両のうごき」126-128頁記事}}。</ref><ref>{{lang|ja|工事に伴って東武伊勢崎線･日光線の自動放送も搭載された。なお、搭載された自動放送は26編成以降に搭載されている男声（東武旧タイプ）のものではなく、東武20050系･20070系に更新搭載された女声（東武新タイプ）のものと同様になっている}}。</ref>，同时也开始对部分列车的[[VVVF|VVVF]]进行维修<ref>[http://www.metosha.co.jp/srvc/srvc01.html 車輛改修工事業務] {{lang|ja|メトロ車両ホームページ、2012年5月12日閲覧}}</ref>。
