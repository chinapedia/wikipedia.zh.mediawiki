'''營養補充品'''，又称'''营养补充剂'''、'''膳食补充剂'''、'''营养剂'''、'''饮食补充剂'''等，是一種成分是從[[食物|食物]]中[[萃取|萃取]]對[[人體|人體]]有益的[[营养素|营养素]]（如[[氨基酸|氨基酸]]、[[微量元素|微量元素]]、[[维生素|维生素]]、[[矿物质|矿物质]]等）的補充品，主要的功能是維持[[健康|健康]]、[[預防|預防]][[疾病|疾病]]。而'''营养补充品'''必須經過[[衛生|衛生]][[機關|機關]]的檢查，證明其[[安全|安全]]性和保建功能才能販售。値得注意的是，它並不是[[藥物|藥物]]，不過可視[[身體|身體]]需要作適量補充。<ref name ="a1"/>。

由於'''营养补充品'''有許多不同的形態，因此應該因應使用者的身體狀況、方便性、[[吸收_(生物)|吸收]][[速度|速度]]、特殊的需要性等方面來選擇。<ref name ="a1"/>

==形態==
目前营养补充品有以下形態：<ref>{{cite book|publisher=《維生素礦物質事典》|title=第25頁|year=2004年10月20日|url=http://postimg.org/image/5a4babyuv/5e18ddc8/|access-date=2016-01-26|archive-url=https://web.archive.org/web/20160202102241/http://postimg.org/image/5a4babyuv/5e18ddc8/|archive-date=2016-02-02|dead-url=yes}}</ref>。
===液状===
優點是身體[[吸收_(生物)|吸收]][[速度|速度]]快，適合小朋友，及對呑嚥藥片和膠囊有障礙的人；缺點是容易[[攝取|攝取]]過量。<ref name ="a1"/>
*[[糖漿|糖漿]]
*[[滴劑|滴劑]]
*[[發泡錠|發泡錠]]
*[[口服液|口服液]]
===含嚼錠劑===
優點是身體吸收僅次於液状营养补充品；缺點是由於好吃，容易不小心就攝取過量。<ref name ="a1"/>
*[[口含錠|口含錠]]
*[[口嚼錠|口嚼錠]]
===膠囊===
優點是身體吸收僅次於液状與含嚼錠劑营养补充品；缺點是通常用於[[脂溶性維生素|脂溶性維生素]]。<ref name ="a1"/>
*[[膠囊劑|膠囊]]
*[[軟膠囊|軟膠囊]]

===膜糖衣錠劑===
優點是收藏容易、可長期保存，且長期服用會較[[便宜|便宜]]；缺點是吸收速度慢、糖衣錠類似[[糖果|糖果]]，容易被小朋友誤食<ref name ="a1"/>、膜衣錠若被咬碎服用，會影響人體吸收效果（膜衣的作用是抗胃酸和膽鹼，使成分能夠在小腸有效被人體吸收，因此若在口中就咬碎吞服，那麼就會影響效用）。<ref>{{cite news|title=膜衣錠咬碎吞服 藥效恐降低|url=http://www.appledaily.com.tw/appledaily/article/supplement/20100302/32330267/|publisher=[[蘋果日報|蘋果日報]]|date=2010-03-02|accessdate=2016-01-26}}</ref>
*[[糖衣錠|糖衣錠]]
*[[膜衣錠|膜衣錠]]
===其他===
*[[粉末状|粉末状]]，優點是身體吸收速度快、由於不含其他添加物與結合劑，因此適合過[[過敏|過敏]]體質使用。<ref name ="a1"/>
*[[軟糖|軟糖]]，優點是較為適合[[小朋友|小朋友]]食用的營養補充品。<ref name ="a1"/>
*[[注射劑|注射劑]]，優點是可對[[營養素|營養素]]嚴重缺乏或具特別體質的病人使用。<ref name ="a1"/>

==相關法律==
===美国===
在[[美国|美国]]，膳食补充剂（dietary supplement）的定义由1994年[[美国国会|美国国会]]通过的《[[膳食补充剂健康教育法|膳食补充剂健康教育法]]》（Dietary Supplement Health and Education Act of 1994）给出：
#它是作为饮食的补充剂并含有下列一种或多种成分： 
#*[[维生素|维生素]]
#*[[矿物质|矿物质]]
#*[[蛋白质|蛋白质]]
#*[[药草|药草]]或其它植物制剂（不包括[[烟草|烟草]]）
#*[[氨基酸|氨基酸]]
#*可以帮助增加每日进食量的补充剂
#*浓缩素
#*代谢物、组成物或提取物
#它以片剂、胶囊、液体等形式供人摄入。
#它不能取代某种传统食物或成为饮食的唯一来源 。
#它需要标记有「dietary supplement」。
#多种调节人体机能的功能性。
[[美国食品及药物管理局|美国食品及药物管理局]]（FDA）规定膳食补充剂是一种食品，而不是一种[[药品|药品]]。美国食品及药物管理局并不保证膳食补充剂的安全性和有效性。美国食品及药物管理局只能在某种膳食补充剂出现问题后才去调查生产商。

===欧盟===
[[欧盟|欧盟]]的《[[食品补充律令|食品补充律令]]》（Food Supplements Directive）要求膳食补充剂必须保证品质和数量的安全可靠。有些维生素在小剂量时是对人体有益的，但是过多的剂量是有害的。有些药草，比如[[金丝桃素|金丝桃素]]（[[貫葉連翹|貫葉連翹]]），和某些药品同时摄入会产生毒副作用。只有那些经证实没有危险的膳食补充剂才可以不需[[处方|处方]]就在市场上销售。

===中华人民共和国===


《保健食品注册管理办法(试行)》将保健食品定义定义：保健食品是指声称具有特定保健功能或者以补充[[维生素|维生素]]、矿物质为目的的食品，即适宜于特定人群食用，具有调节机体功能，不以治疗疾病为目的，并且对人体不产生任何急性、亚急性或者慢性危害的食品。<ref>{{cite web|author1=中华人民共和国卫生部|title=《保健食品注册管理办法（试行）》（局令第19号）|url=http://www.sda.gov.cn/WS01/CL0053/24516.html|publisher=中华人民共和国卫生部|accessdate=2016-05-02}}</ref>

《保健食品管理办法》及[[国家标准|国家标准]]GB16740-2014《食品安全国家标准 保健食品》将保健食品定义为：并具有特定保健功能或者以补充维生素、矿物质为目的的食品。即适用于特定人群食用，具有调节机体功能，不以治疗疾病为目的，并且对人体不产生任何急性、亚急性或慢性危害的食品。<ref>{{cite web|author1=中华人民共和国国家卫生和计划生育委员会|title=食品安全国家标准 保健食品|url=http://www.xjda.gov.cn/directory/web/WS01/images/R0IxNjc0MC0yMDE0ICCxo72hyrPGty5wZGY=.pdf|publisher=GB 16740-2014|accessdate=2018-05-16}}</ref>
《保健食品管理办法》第二十一条规定，保健食品标签和说明书必须符合国家有关标准和要求，并标明下列内容：
* （一）保健功能和适宜人群；
* （二）食用方法和服用量；
* （三）贮藏方法；
* （四）功效成分的名称及含量。
* （五）保健食品批准文号；
* （六）保健食品标志；
* （七）有关标准或要求所规定的其它标签内容。
* 补充内容
:* 保健食品的名称应准确科学，不得使用人名、地名、代号及夸大容易误解的名称。
:* 保健食品的标签、说明书和广告内容必须真实，符合其产品质量要求，不得有暗示可使疾病痊愈的宣传。

===臺灣===
包括目前[[臺灣|臺灣]]的保健食品屬於業者自主管理,屬於一般食品規範;其中衛生署推動的[[健康食品|健康食品]]認證必須通過[[動物實驗|動物實驗]]功效而通過，另有由[https://web.archive.org/web/20110125120226/http://s2.misa.com.tw/ibmi/client/ 生策會]推動的[http://www.snq.org.tw/ SNQ國家品質標章]。
