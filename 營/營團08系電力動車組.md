'''營團08系電聯車'''（{{jpn|j=営団08系電車|hg=えいだん08けいでんしゃ|rm=Eidan 08 kei densha|f=y}}）、乃[[東京地下鐵|東京地下鐵]]（前[[帝都高速度交通營團|帝都高速度交通營團]]）[[半藏門線|半藏門線]]用的[[通勤型電車|通勤型電車]]。

2003年1月7日開始營運。

== 概要 ==
為了應付在2003年3月19日半藏門線[[水天宮前站|水天宮前站]]至[[押上站|押上站]]的啟用，以及與[[東武鐵道|東武鐵道]][[東武伊勢崎線|東武伊勢崎線]]、[[東武日光線|東武日光線]]作直通運轉所帶來的服務需求，6组(每组10辆)由[[日本車輛製造|日本車輛製造]]製造。

== 簡介 ==
* [[鋁合金|鋁合金]]車體配線色彩帶
* 控制裝置：[[三菱電機|三菱電機]]製IPM[[VVVF|VVVF]]
* 制動裝置：TRT-11型[[再生制動|再生制動]]+電子制動+純電气電力制動
* 主電動機：[[三相電動機|三相電動機]]　(165kW×4)
* 輔助電源裝置：東芝製[[IGBT|IGBT]]輔助逆變器
* 齒輪：7.07 
* 起動加速度：3.3km/h/s
* 減速度：3.5km/h/s（緊急4.5km/h/s）
* 最高速度：110km/h
* 安全裝置：東京地下鐵[[CS-ATC|CS-ATC]]／東急CS-ATC／東武[[ATS|ATS]]（東武稱作TSP）

== 車内設備 ==
* [[輪椅|輪椅]]放置位：3、9號車
* 弱冷氣車：2號車

== 編成、運用 ==
* 10卡6列編成編入[[鷺沼檢車區|鷺沼檢車區]]。
