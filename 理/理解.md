'''理解'''（'''{{lang|en|Understanding}}'''），又称为'''领会'''、'''了解'''、'''懂得'''、'''思维作用'''（'''{{lang|en|intellection}}'''），是指一种[[心理学|心理]]过程，与诸如人、情形或[[讯息|讯息]]之类的某种[[抽象|抽象]]的或[[有形|有形]]的[[对象|对象]]相关，藉此一个人能够对其加以思考，并且运用[[概念|概念]]对该对象加以适当的处理。

理解乃是[[概念化|概念表达]]（又称为'''概念化'''）的界线。理解某一事物，也就是已经对该事物实现了一定程度的概念表达或者说概念化。

== 有关理解的例子 ==
# 一个人如果能够做到对天气加以[[预测|预测]]并对天气的一些特点加以[[解释|解释]]等等之类的话，那么，就说这个人理解了天气。
# 如果一位[[精神科医生|精神科医生]]知道某位病人的[[焦虑|焦虑]]及其原因，并且能够针对如何应对焦虑给予有益的[[忠告|忠告]]，那么，就说这位[[精神科医生|精神科医生]]理解了该病人的焦虑。
# 就某条[[命令|命令]]来说，如果某个人知道这究竟是谁下达的命令，下令者的期望究竟是什么以及该项命令是否合法或者说正当等等，那么，就说这个人理解了这条命令。
# 如果一个人能够清醒地某条消息所传达的信息内容，那么，就说这个人理解了其中的[[修辞|修辞]]、[[论据|论据]]或某种[[语言|语言]]。
# 如果一个人能够利用某一[[数学|数学]][[概念|概念]]解决[[问题|问题]]，尤其是那些此前未曾见过的问题，那么，就说这个人理解了这个数学概念。
