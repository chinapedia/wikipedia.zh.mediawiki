'''静安别墅'''，是位于[[上海|上海]][[静安区|静安区]]的一片[[里弄#新式里弄|新式里弄]]住宅，曾被称为“小上海”<ref name="sd">[http://www.shtong.gov.cn/newsite/node2/node4/node2249/node4412/node17431/node17651/node17661/userobject1ai6472.html 区县志 >> 区志 >> 静安区志 >> 第三编街道 >> 第二章街道分述 >> 第四节　威海路街道] 上海地方志办公室</ref>。住宅北临南京西路，南临[[威海路|威海路]]，西临[[陕西北路_(上海)|陕西北路]]，东临茂名北路。静安别墅是上海最大型的新式里弄<ref name="vg">[http://sh.eastday.com/m/20130911/u1a7655722_1.html 静安别墅整治成封闭小区 年底前回归"宁静幽雅"] 东方网</ref>，由[[高观四|高观四]]设计施工<ref name="ff">[http://www.shtong.gov.cn/newsite/node2/node2245/node69543/node69552/node69640/node69644/userobject1ai67960.html 专业志 >> 上海建筑施工志 >> 人物篇 >> 第一章人物传略 >> 高观四] 上海地方志办公室</ref>，于[[民国21年|民国21年]]建成，因地处[[静安寺路_(上海)|静安寺路]](今南京西路)，故而得此名。1994年，静安别墅入选第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]。

演员[[郑小秋|郑小秋]]从1936年便开始在静安别墅182号居住直至其1989年逝世。此外，电影[[色，戒_(电影)|色，戒]]也曾在此取景<ref name="kt">[http://xmwb.xinmin.cn/metro/html/2013-10/25/content_3_1.htm 静安别墅“弄”没了]</ref>。

== 建筑特色 ==
静安别墅占地2.5公顷，有楼房193幢<ref name="dd">[http://www.shsmi.cn/dmb/dmkw/7-3-1-8.htm 第一节　市区(一)：静安区] {{webarchive|url=https://web.archive.org/web/20110111170235/http://www.shsmi.cn/dmb/dmkw/7-3-1-8.htm |date=2011-01-11 }} 上海地方志办公室</ref>，是典型的新式里弄住宅。里弄呈行列式布局，南北向的总弄和多条支弄垂直交叉。每栋别墅为砖木结构，共三层，砖墙瓦顶，装饰简洁。每个单元都有小庭院与阳台。

== 近况 ==
静安别墅因其独特建筑风格，一直吸引着很多游客前来。同时，静安别墅内近些年也曾开有多家餐馆、奶茶铺、饰品店、咖啡馆等商铺，但其有不少都为违章搭建或无证经营。2013年下半年开始，静安区工商局开始要求这些商铺搬迁停业，并且将安装门禁，限制游客等人员出入<ref name="vg"/>。
