'''静安面包房'''，隶属于[[锦江国际集团|锦江国际集团]]旗下的[[香港证券交易所|香港上市公司]]锦江酒店集团（02006.HK），于1985年9月由沪港合资而成，是上海第一家中外合资经营的法式西点房，为上海颇有知名度的老字号。

== 历史 == 
=== 风靡上海 === 
1980年代，随着[[改革开放|改革开放]]步伐的加快，中国人开始以生日蛋糕逐渐代替传统面食作为生日祈福品，面包房应市场需求从各大酒店中独立出来，直接为市民所服务<ref name = "中国食品">{{cite book |url =  |title =  饼屋的机遇与挑战 |date = 2005年3期 |location = 中国 |publisher = 《中国食品》 |author = 香山 |page = 32}}</ref>。在这一背景下，1985年9月，由上海静安宾馆、香港三隆行两方共同出资创建了静安面包房，并开始承接[[上海市人民政府|上海市人民政府]]所承办的各类招待宴会的西点面包供应任务，为其面包的品质奠定了基础。创建之初，静安面包房从[[法国|法国]]请来面包师傅坐镇，其生产的白脱小球面包、白脱别司忌、[[法棍|法式长短棍]]、'''栗子蛋糕'''、[[羊角面包|羊角面包]]、[[拿破仑蛋糕|拿破仑蛋糕]]<ref name="新华网">{{cite book|url =http://www.sh.xinhuanet.com/zhuanti/csj/2010-10/12/content_21105633.htm|title =上海20大蛋糕甜品店|date =2010-10-12|location =中国|publisher =[[新华网|新华网]]长三角频道|author =|page =|deadurl =yes|archiveurl =https://web.archive.org/web/20140221200342/http://www.sh.xinhuanet.com/zhuanti/csj/2010-10/12/content_21105633.htm|archivedate =2014-02-21}}</ref>等招牌西点风靡1980年代至1990年代的上海，经常出现排长龙购买面包的景象<ref name="腾讯网">{{cite book |url = http://2010.qq.com/a/20100406/000132.htm |title = 静安面包房 25年法式长棍飘香 |date = 2010-04-06 |location = 中国 |publisher = [[腾讯网|腾讯网]] |author =  |page = }}</ref>。

=== 现代化运营 === 
[[锦江国际集团|锦江国际集团]]先后于1996年和2004年收购了香港三隆行与静安宾馆，静安面包房也就此成为锦江国际集团旗下的全资公司。1999年，面包房改制成立上海静安面包房有限公司。

2009年，持有静安面包房30%股权的香港上市公司锦江酒店，以1535万元人民币的价格向其母公司锦江国际集团购买了静安面包房余下的70%股权。锦江酒店也借此进一步开拓食品与餐饮业务，形成了锦江国际集团控股的多元投资的股份制结构，静安面包房也改以现场烘焙模式进行运营<ref name="每经">{{cite book |url = http://funds.hexun.com/2009-06-02/118208505.html |title = 锦江酒店再花1535万从母公司完全买下静安面包房 |date = 2009-06-02 |location = 中国 |publisher = 每日经济新闻 |author = 李凌霞 |page = }}</ref><ref name="静安面包房">{{cite book|url =http://www.jinganbakery.com/about/events|title =品牌介绍与历史|date =|location =中国|publisher =静安面包房官网|author =|page =|deadurl =yes|archiveurl =https://web.archive.org/web/20111126113341/http://www.jinganbakery.com/about/events|archivedate =2011-11-26}}</ref>。2010年，静安面包房聘请中国乒乓球明星[[王励勤|王励勤]]担任其形象大使<ref name="新民网">{{cite book|url =http://sports.xinmin.cn/2010/07/19/5853414.html|title =王励勤成面包房代言人 王励勤展现亲和力|date =2010-07-19|location =中国|publisher =[[新民晚报|新民网]]|author =|page =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
