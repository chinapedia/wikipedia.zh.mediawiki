{{Campaignbox 士族叛亂}}

'''士族叛乱'''是[[日本|日本]][[明治|明治]]初期舊[[武士_(日本)|武士]]階級（[[士族|士族]]）對明治政府的一連串反政府活動。

== 概要 ==
明治政府採取[[四民平等|四民平等]]政策，廢止大名・武士階級，創設[[華族|華族]]・士族。[[秩祿處分|秩祿處分]]後，撤廢俸祿（家祿）制度。施行[[廢刀令|廢刀令]]等，廢除身分的特權。此外，[[文明開化|文明開化]]、[[殖產興業|殖產興業]]政策和[[征韓論|征韓論]]所引發的政爭，演變成[[明治六年政變|明治六年政變]]，導致[[西鄉隆盛|西鄉隆盛]]、[[江藤新平|江藤新平]]、[[板垣退助|板垣退助]]等人下野，都對士族階級帶來影響，這些反對明治政府的士族被稱為「不平士族」。

明治7年（1874年）江藤新平、[[島義勇|島義勇]]在故鄉[[佐賀縣|佐賀縣]]領導的[[佐賀之亂|佐賀之亂]]，明治9年（1876年）[[熊本縣|熊本縣]]的[[神風連之亂|神風連之亂]]，與[[神風連|神風連]]相呼應的[[福岡縣|福岡縣]][[秋月藩|秋月藩]]士[[宮崎車之助|宮崎車之助]]為首的[[秋月之亂|秋月之亂]]，同年10月[[山口縣|山口縣]][[前原一誠|前原一誠]]等人的[[萩之亂|萩之亂]]等反亂相繼發生，但皆被政府軍鎮壓。明治10年（1877年）[[鹿兒島縣|鹿兒島縣]][[薩摩|薩摩]]士族擁立西鄉，爆發最大規模的士族反亂[[西南戰爭|西南戰爭]]，也已失敗告終。

西南戰爭之後，不平士族的反對運動轉變為主張國會開設、憲法制定的[[自由民權運動|自由民權運動]]。

== 佐賀之亂 ==
{{main|佐賀之亂}}
由[[明治六年政變|明治六年政變]]下野的征韓黨人[[江藤新平|江藤新平]]和憂國黨的[[島義勇|島義勇]]帶領11000人的不平士族在[[佐賀縣|佐賀縣]]發動的一場士族反亂，發生在1874年2月1日至1874年3月1日。只經歷一個月，就比由[[大久保利通|大久保利通]]帶領下的新政府官軍平定。

== 神風連之亂 ==
{{main|神風連之亂}}
由通稱[[神風連|神風連]]的[[敬神黨|敬神黨]]約170人的不平士族發動的一場士族反亂，發生在1876年10月24日的[[熊本縣|熊本縣]]。只經歷1天就被[[兒玉源太郎|兒玉源太郎]]的官軍完全鎮壓，神風連中124人戰死，約50人被捕；官軍方面約60名士兵陣亡，超過200人受傷。

== 秋月之亂 ==
{{main|秋月之亂}}
由原[[秋月藩|秋月藩]]士[[宮崎車之助|宮崎車之助]]在[[福岡縣|福岡縣]]發動的一場士族反亂，發生在1876年10月27日至同年11月14日，與神風連的士族反亂互相呼應。約有230－240人參加，甚至有說超過400人參加叛亂，被長州名將、人稱「軍神」的[[乃木希典|乃木希典]]平定。

== 萩之亂 ==
{{main|萩之亂}}
由自稱「殉國軍」200多人的不平士族發動的一場士族反亂，領導人物是原[[長州藩|長州藩]]士[[前原一誠|前原一誠]]。發生在1876年10月28日至同年12月8日的[[山口縣|山口縣]]，被同樣也是原長州藩士的[[三浦梧樓|三浦梧樓]]為首的新政府軍鎮壓。

== 西南戰爭 ==
{{main|西南戰爭}}
「維新三傑」之一的[[西鄉隆盛|西鄉隆盛]]，在[[明治六年政變|明治六年政變]]下野後返回鄉下[[鹿兒島縣|鹿兒島縣]]（[[薩摩|薩摩]]）。回鄉後開設專門培訓士族的「私學校」，上至縣[[知事|知事]]、下至一些普通[[警察|警察]]也是該校學生。全縣依舊實行舊制，不理明治新政府的命令，等同獨立國家。1877年明治新政府決定奪回鹿兒島縣的控制權，派官兵搬走鹿兒島縣內的軍火，被士族學生發現並武力攻擊官兵打算奪回軍火，官兵拘捕多個士族學生。西鄉隆盛救回士族學生，由於奪取軍火等同宣戰，西鄉被迫帶領「私學校」的士族學生叛亂，舊[[薩摩藩|薩摩藩]]士和[[九州|九州]]各地的士族亦支持西鄉向明治新政府宣戰。最終西鄉軍團不敵[[有栖川宮熾仁親王|有栖川宮熾仁親王]]為首的官軍而戰敗（主要原因是官軍使用的新式武器比起西鄉軍團的武器先進），西鄉隆盛本人亦[[切腹|切腹]]自殺。西南戰爭中證明士族不一定比農民子弟兵優勝，所以沒有士族再敢叛亂，此戰為最後一次士族反亂。
