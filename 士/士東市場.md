'''士東市場'''是位於[[臺北市|臺北市]][[士林區|士林區]]成立於1992年10月4日的[[傳統市場|傳統市場]]，提供空調、外送服務與免費使用的手推車，是一棟地上兩層樓，地下一樓停車場的市場建築。<ref name = "士東市場特色_2018"> {{ cite web | title = 士東市場特色 | url = http://www.shi-dong.com.tw/_tw/Home.php | publisher =   士東市場官方網站 | accessdate = 2018-09-17 }} </ref><ref name = "2010臺灣樂活市集暨名攤精選_2010"> {{ cite book | title = 2010臺灣樂活市集暨名攤精選 | url = https://books.google.com.tw/books?id=PGwmDwAAQBAJ&pg=PA4&dq=%E5%A3%AB%E6%9D%B1%E5%B8%82%E5%A0%B4&hl=zh-TW&sa=X&ved=0ahUKEwjUgKu87L_dAhUMv7wKHZipD5cQ6AEIOjAE#v=onepage&q=%E5%A3%AB%E6%9D%B1%E5%B8%82%E5%A0%B4&f=false | publisher = 中華民國政府出版品, 2010 | author = 作者：經濟部中部辦公室，中國生產力中心 | isbn = 9789860261486 | page = 4 | language = zh-tw | date = 2010-12-01 }} </ref><ref name = "士東市場「人情味」的社會學初探_2016"> {{ cite web | title = 士東市場「人情味」的社會學初探 | url = http://www.airitilibrary.com/Publication/alDetailedMesh?docid=U0001-2108201600474300 | publisher = 臺灣大學 | author = 姚國偉 | language = zh-tw | date = 2016-01-01 }} </ref><ref name = "臺北市市場處機關入口網_2018"> {{ cite web | title = 臺北市公有士東市場 | url = https://www.tcma.gov.taipei/News_Content.aspx?n=1A4C7120ABFD0F9F&sms=26C5BF03E7963F4E&s=F39FB7349CDFF219 | publisher = 臺北市市場處機關入口網 | author =  | date = 2018-10-16 | language = zh-tw }} </ref><ref>{{ cite web | title = 臺北市公有士東市場 | url = https://www.gov.taipei/News_Content.aspx?n=9F394A83D9DF4A19&sms=70D42B01046FC56C&s=A881EFECA1ACF0C9| publisher =臺北市市場處  | author =  | date = 2018-10-30  | language = zh-tw  }} </ref><ref name = "士東市場_台北畫刊_2017"> {{ cite web | title = 士東市場  友善創新心行銷 | url = https://www.businesstoday.com.tw/article/category/154685/post/201705110054/%E5%A3%AB%E6%9D%B1%E5%B8%82%E5%A0%B4%20%20%E5%8F%8B%E5%96%84%E5%89%B5%E6%96%B0%E5%BF%83%E8%A1%8C%E9%8A%B7 | publisher = 台北畫刊 | author = 文／凃心怡 | language = zh-tw | date = 2017-05-11 }} </ref>

== 市場樓層 ==
* 一樓設有127個攤位販賣蔬菜、花卉、肉類、鮮魚、什貨、素食
* 二樓設有128個攤位販賣百貨、古董、珠寶與美食
* 地下一樓為收費停車場<ref name = "士東市場特色_2018" />

== 食材捐贈 ==
士東市場攤商成立「盛食交流平台」，將收市後剩餘新鮮食材放置專用冰箱，透過社會局捐贈社會福利團體。<ref name = "士東市場_台北畫刊_2017" />

== 環保節能 ==
市場屋頂架設60.2千瓦（KW）[[太陽能板|太陽能板]]善用天然能源，回收雨水設施可以循環使用天然水源，燈具改用省電燈具與[[LED|LED]]電燈節省照明用電。<ref name = "士東市場特色_2018" />

== 榮譽 ==
* 台北市環保局2008年度市場、攤販集中場清潔維護評比，士東市場為年度最整潔市場<ref name = "北市最乾淨公有市場   士東市場奪冠_2009"> {{ cite web | title = 北市最乾淨公有市場   士東市場奪冠 | url = http://www.epochtimes.com/b5/9/6/15/n2558652.htm | publisher = 大紀元時報 | author = 林麗玉 | language = zh-tw | date = 2009-06-15 }} </ref>
