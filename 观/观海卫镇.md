'''观海卫镇'''是[[中国|中国]][[浙江省|浙江省]][[慈溪市|慈溪市]]一个[[建制镇|建制镇]]，东接[[掌起镇|掌起镇]]，西连[[桥头镇_(慈溪市)|桥头镇]]、[[附海镇|附海镇]]，北濒[[杭州湾|杭州湾]]，南接[[江北区_(宁波市)|江北区]][[慈城镇|慈城镇]]。辖区面积150.2平方公里，常住人口20万，镇政府驻振观路6号。<ref name="xzqh">[http://www.xzqh.org/html/zj/0241.html 行政区划网-观海卫镇] {{webarchive|url=https://web.archive.org/web/20100619145532/http://xzqh.org/html/zj/0241.html |date=2010-06-19 }}</ref>2008年，观海卫镇成为[[宁波市|宁波市]]卫星城市试点镇，享有部分县级经济社会管理权限。<ref>{{cite web|url=http://news.cnnb.com.cn/system/2010/01/08/006384776.shtml|title=宁波市７个镇将开展卫星城市试点方案|publisher=中国宁波网|date=2010-01-08|accessdate=2010-09-25}}</ref>

== 沿革 ==
观海卫地区驻军始于[[宋代|宋代]]。<ref name="xzqh" />[[明代|明]][[洪武|洪武]]二十年（1387年）由信国公[[汤和|汤和]]筑卫。不久以后，[[福建省|福建省]]福宁卫旗军在观海卫驻防并定居。[[闽语|闽语]]与本土的[[吴语|吴语]]混合，形成独特的[[方言岛|方言岛]]——[[燕话|燕话]]（又名卫里话）。<ref>{{cite web|url=http://news.cnnb.com.cn/system/2009/12/25/006370727.shtml|title=观海卫老城六百年燕话何去何从？|date=2009-12-25|accessdate=2010-09-25}}</ref>2001年12月，合并原观城镇、师桥镇、[[鸣鹤镇|鸣鹤镇]]设立观海卫镇。<ref name="xzqh" />

== 工业 ==
观海卫工业以轻型加工业为主，低压插座、针织服装、电子元件、家用电器、塑料制品、五金工具和打火机是观海卫主要的工业部门。<ref name="namecard">{{cite web|url=http://www.peoplexz.com/316/562/20080512104911.htm|title=慈溪市观海卫镇名片-浙江省宁波市|publisher=人民乡镇在线|date=2008-05-12|accessdate=2010-09-25}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 旅游 ==
观海卫旅游资源丰富。
* 鸣鹤古镇拥有1200多年历史，目前保存较为完好。[[File:白雪覆盖下的鸣鹤古镇.jpg|thumb]]
* [[五磊山|五磊山]]景区是[[鸣鹤|鸣鹤]]-[[上林湖|上林湖]]省级风景名胜区的重要组成部分。其中，[[五磊寺|五磊寺]]是浙江东部最古老的寺院；[[藏云溪|藏云溪]]是[[东吴|东吴]][[孙权|孙权]]为方便母亲[[吴国|吴国]]太上五磊山烧香拜佛而派臣找到的一条捷径。<ref>{{cite web |url=http://www.nbwls.com/NewsList.aspx?id=21 |title=存档副本 |accessdate=2013-08-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20120621163333/http://www.nbwls.com/NewsList.aspx?id=21 |archivedate=2012-06-21 }}</ref>这也是十分具有旅游价值的。[[File:五磊讲寺.jpg|thumb]]
* 明代卫城和抗倭烽火台也是当地重要的旅游资源。<ref name="namecard" />

== 交通 ==
观海卫交通便捷。[[沈海高速公路|沈海高速公路]]（[[杭州湾跨海大桥|杭州湾跨海大桥]]南岸连接线）经过观海卫，设有观海卫出入口。<ref name="namecard" />[[329国道|329国道]]、中横线横穿观海卫，连接慈溪市其他地区。<ref name="namecard" />
