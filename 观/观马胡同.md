'''观马胡同'''，是[[北京市|北京市]][[东城区_(北京市)|东城区]]南部的胡同。

==简介==
观马胡同原先北起[[广渠门内大街|广渠门内大街]]，南到[[延庆街_(北京)|延庆街]]，全长300米，宽5米。观马胡同[[清朝|清朝]]称为“观马圈”，[[乾隆|乾隆]]十三年（1748年）称“官马圈”，是[[正蓝旗|正蓝旗]]兵养官马的场所。清朝乾隆《京城全图》将这里标记为“[[兵部|兵部]]马圈”。《京师坊巷志稿》记载为“刘家马圈”。此处所养马匹是每年皇帝[[冬至|冬至]]祭天、春季祭神时，专门供随行的[[太监|太监]]乘用。实际上马圈内每年有十个月是空的，只在每年将近冬至时，马圈的马官才派员赴[[塞外|塞外]]购买马匹，经驯养后承应冬至祭天、春季祭神的差事，随后就将马匹出售。但是无论马圈内是否有马，马圈的官员都按月从上级领取马料，并按照额定人数领取马夫的俸银，这些钱就被马圈的官员侵吞了。[[中华民国|中华民国]]时期，此处沿称。1965年北京市整顿地名，改称为“观马胡同”。1990年代，观马胡同北部被拆除，观马胡同变成了仅从延庆街可进入的死胡同<ref name=ma>{{Cite web|url =http://bjrb.bjd.com.cn/html/2014-03/04/content_157347.htm|title =马与胡同不了情|author =朱小平|date =2014-03-04|publisher =北京日报|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170202020255/http://bjrb.bjd.com.cn/html/2014-03/04/content_157347.htm|archivedate =2017-02-02}}</ref><ref>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/lane/200907/t20090717_10584.html|title = 观马胡同 |author =  |date = 2009-07-17 |publisher = 东华流韵 |language =  |accessdate =  |quote =  }}</ref>。

清朝北京有[[内务府|内务府]][[上驷院|上驷院]]，取代明朝的[[御马监|御马监]]管理御马。[[顺治|顺治]]年间，御马监改为“阿敦衙门”（“阿敦”是[[满语|满语]]，意为“马群”），[[康熙|康熙]]年间改为“上驷院”。明朝的御马圈在[[景山东街|景山东街]]，清朝上驷院的养马场所称“马厩”，在[[紫禁城|紫禁城]]内外及[[南苑|南苑]]共设17个马厩，养马700多匹，在[[张家口|张家口]]外以及[[盛京|盛京]]等地还设四个牧厂，养马260多群（每群四百到五百匹）。此外，驻京[[八旗|八旗]]也分别有养马场所。八旗分京营八旗和驻防八旗。驻守北京的京营八旗约有十万人，按照各旗划分驻地：[[正黄旗|正黄旗]]驻扎[[德胜门|德胜门]]内，[[镶黄旗|镶黄旗]]驻扎[[安定门|安定门]]内，[[正白旗|正白旗]]驻扎[[东直门|东直门]]内，[[镶白旗|镶白旗]]驻扎[[朝阳门|朝阳门]]内，[[正蓝旗|正蓝旗]]驻扎[[崇文门|崇文门]]内，[[镶蓝旗|镶蓝旗]]驻扎[[宣武门|宣武门]]内，[[正红旗|正红旗]]驻扎[[西直门|西直门]]内，[[镶红旗|镶红旗]]驻扎[[阜成门|阜成门]]内。各旗在驻地均分别设有养马场，所养的马匹俗称“官马”，养马场所称“马圈”（“圈”字读音为“券”，并加[[儿化音|儿化音]]）<ref name=ma/>。
