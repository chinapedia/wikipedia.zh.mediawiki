'''查尔斯·格贝克'''（'''Charles Gbeke'''，又被译为'''查尔斯·比奇'''，{{bd|1978年|3月13日|||G}}），是一名[[科特迪瓦|科特迪瓦]]裔[[加拿大|加拿大]]职业足球运动员，司职[[前锋_(足球)|前锋]]。现在是一名自由球员。

==职业生涯==
格贝克出生于[[科特迪瓦|科特迪瓦]][[阿比让|阿比让]]，成长于[[加拿大|加拿大]][[蒙特利尔|蒙特利尔]]，并获得加拿大国籍。格贝克的青训生涯在[[温哥华白帽足球俱乐部|温哥华86人]]度过，当时他的位置是[[后卫_(足球)|后卫]]。1999年，格贝克加盟[[法国|法国]][[特魯瓦足球俱樂部|特鲁瓦B]]，随后又短暂效力于[[巴西|巴西]]低级别球队科莫西亚和松原体育。2001年，格贝克回到已经改名为[[温哥华白帽足球俱乐部|温哥华白帽]]的母队，随队征战美国甲级联赛(次级联赛)，但只得到三次替补出场机会。后来他转投至另一支美国甲级联赛球队[[西雅图音速足球俱乐部_(美国甲级联赛)|西雅图音速]]，但也只为球队替补出场一次。

2002年，格贝克转会到[[加拿大职业联赛|加拿大职业联赛]]球队[[渥太华巫师足球俱乐部|渥太华巫师]]，他的位置从后卫改造成[[前锋_(足球)|前锋]]，这成为他职业生涯的转折点。他在2002赛季出场18次打进10球，渥太华巫师也在联赛中连续11场比赛保持不败并获得加拿大联赛杯赛三冠王。2003赛季，格贝克又以出色的表现帮助渥太华巫师整个赛季保持不败并再次获得联赛冠军。他的表现再次引起美国甲级联赛球队的注意，由于渥太华巫师被撤销，格贝克在2004年被租借到甲级联赛球队[[多伦多山猫足球俱乐部|多伦多山猫]]。在多伦多，格贝克占据球队主力前锋的位置，并取得6个联赛进球，其中他的第一个进球正是在对阵老东家温哥华白帽时打进的<ref>{{cite web |url=http://22066.vws.magma.ca/04lynx74.htm |title=TORONTO LYNX GAME REPORT |accessdate=2010-02-04 |work=Toronto Lynx game report }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在赛季结束后，他又被短暂租借到加拿大职业联赛球队[[汉密尔顿雷霆足球俱乐部|汉密尔顿雷霆]]。

2005年，格贝克加盟[[丹麦足球甲级联赛|丹麦足球甲级联赛]]球队[[赫尔弗格足球俱乐部|赫尔弗格]]，但没能取得一个进球。赫尔弗格在赛季结束后降级，格贝克也和球队解除合同。在[[萨格勒布迪纳摩足球俱乐部|萨格勒布迪纳摩]]试训失败后，格贝克回到多伦多山猫。但这次他的表现没能达到此前的水平，出场6次只取得2个进球。8月1日，在转会窗口即将关闭的一刻，格贝克转会到[[蒙特利尔冲击足球俱乐部|蒙特利尔冲击]]<ref>{{cite web |url=http://22066.vws.magma.ca/05lynx1h.htm |title=DiPlacido and Gbeke join Impact! |accessdate=2010-02-04 |work=Toronto Lynx media release }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2006年，格贝克加盟[[罗彻斯特怒犀足球俱乐部|罗彻斯特怒犀]]，但在球队的表现却让人失望，在2006赛季出场21次，只有2个联赛进球。2007年，格贝克回到[[蒙特利尔冲击足球俱乐部|蒙特利尔冲击]]，并在2007赛季以10个进球获得联赛最佳射手称号。2008年6月30日，温哥华白帽以球员加现金的方式买入格贝克。在该赛季甲级联赛总决赛中，格贝克打进2球，帮助球队2比1击败[[波多黎各岛民足球俱乐部|波多黎各岛民]]<ref>[http://www.whitecapsfc.com/archive/feature10130803.aspx]</ref>，历史上第二次获得甲级联赛冠军。2009赛季，格贝克以12个进球再次获得联赛最佳射手称号，但温哥华白帽在总决赛中被蒙特利尔冲击击败屈居亚军。赛季结束后，由于温哥华队将为加入[[美国职业足球大联盟|美国职业足球大联盟]]而对俱乐部重组，格贝克和球队解约<ref>[http://www.theprovince.com/sports/Goodbye+Gbeke+gonna+miss/2135149/story.html# Goodbye, Gbeke, we're gonna miss ya] {{webarchive|url=https://web.archive.org/web/20100606080847/http://www.theprovince.com/sports/Goodbye%2BGbeke%2Bgonna%2Bmiss/2135149/story.html%23 |date=2010-06-06 }}</ref> <ref>[http://www.whitecapsfc.com/archive/feature10230905.aspx Goodbye, Gbeke, we're gonna miss ya - The Province]</ref>。

2010年2月，在美国甲级联赛近十年最佳25人的评选中，格贝克排名第18位<ref>{{cite web|url=http://www.uslsoccer.com/home/398536.html |title=USL-1 Top 25 of the Decade |accessdate=2010-02-03 |work=USLsoccer.com |deadurl=yes |archiveurl=https://web.archive.org/web/20110610062408/http://www.uslsoccer.com/home/398536.html |archivedate=2011-06-10 }}</ref>。不久后，格贝克加盟[[中甲|中甲]]球队[[广州恒大足球俱乐部|广州恒大]]。4月3日，他在和[[北京理工大学足球俱乐部|北京理工大学]]的联赛中首次代表广州队出场，并在第23分钟赢得点球机会，帮助队友[[郜林|郜林]]为球队首开纪录<ref>[http://sports.sohu.com/20100404/n271306499.shtml 韩国铁帅赢得首演 郜林梅开二度广州击败学生军]</ref>。4月10日，他在和[[上海浦东中邦足球俱乐部|浦东中邦]]的联赛中梅开二度。格贝克在[[2010年中国足球甲级联赛|2010赛季中甲联赛]]中出场21次，打进9球，帮助球队以联赛冠军的成绩升入[[中超联赛|中超联赛]]。赛季结束后，格贝克没有和广州队续签合同，成为自由球员。2011年5月，格贝克曾经到巴西乙级联赛球队[[亚美黎加纳足球俱乐部|亚美黎加纳]]试训，但没有获得留队机会。

==国家队生涯==
2008年2月1日，格贝克在[[加拿大国家足球队|加拿大]]1比0击败[[马提尼克足球代表队|马提尼克]]的比赛中首次代表国家队出场。随后他又代表国家队在[[2010年世界杯外围赛_(中北美洲及加勒比海区)|2010年世界杯预选赛]]中出场2次，但都没取得进球。

格贝克还入选加拿大队参加[[2009年美洲金盃|2009年中北美洲及加勒比金杯赛]]的参赛名单，但没有在杯赛中出场。
