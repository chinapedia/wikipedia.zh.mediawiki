'''查爾斯·麥金托什'''<sup>{{le|英國皇家學會院士|Fellow of the Royal Society|FRS}}</sup>（{{lang-en|Charles Macintosh}}，{{bd|1766年|12月29日|1843年|6月25日|catIdx=Charles Macintosh}}；或譯稱「查爾斯·麥金塔」）是[[蘇格蘭|蘇格蘭]]男性化學家及發明家，為用於衣物的防水材質開發改良者。

== 生平 ==
年輕時查爾斯開始對於化學產生濃厚興趣，在20歲期間便投入化學物品製造相關事務，<ref name="Strath">{{cite web |url=http://scienceonstreets.phys.strath.ac.uk/new/Charles_MacIntosh.html |title=Charles Macintosh |work=[[斯特拉斯克萊德大學|斯特拉斯克萊德大學]] |language=en |accessdate=2016年12月29日}}</ref>之後曾於1797年[[倫弗魯郡|倫弗魯郡]]一帶成立蘇格蘭的第一間[[明礬|明礬]]工廠。<ref name="Todayinsci">{{cite web |url=https://todayinsci.com/M/Macintosh_Charles/MacintoshCharlesBio.htm |title=Charles Macintosh |work=Today in Science History |language=en |accessdate=2016年12月29日}}</ref>

在1823年查爾斯申請了一項專利；該專利內容是透過[[天然橡膠|天然橡膠]]溶解在[[煤焦油|煤焦油]]的[[石腦油|石腦油]]裡來產生出一種可防水材質，<ref>{{cite web |url=http://inventors.about.com/od/estartinventions/a/elastic_2.htm |title=Charles Macintosh - Waterproof Garments |work=About Com |author=Mary Bellis |language=en |accessdate=2016年12月29日}}</ref>該年度查爾斯亦因此而被獲選為[[皇家學會|皇家學會]]的院士。<ref name="Page113">{{cite book |title=[[牛津國家人物傳記大辭典|牛津國家人物傳記大辭典]]35期 |publisher=[[牛津大學出版社|牛津大學出版社]] |author=Richard Bissell Prosser |location=英國 |date=1893年 | pages=113頁}} </ref>

之後1824年查爾斯來到[[曼徹斯特|曼徹斯特]]找尋棉廠和織工合作，設法將所弄出的材質合成在布料上做出防水衣物，但因為製作出的成品有氣味方面問題，讓查爾斯的防水衣物起初除了一些軍方和海運單位外未被一般大眾接受。<ref name="bouncing-balls">{{cite web |url=http://www.bouncing-balls.com/serendipity/chasmacintosh.htm |title=Charles Macintosh and Co.  The History of the Company |work=Bouncing-Balls |language=en |accessdate=2016年12月29日}}</ref>而在1825年英國發明家湯瑪斯·漢考克（Thomas Hancock）從查爾斯的專利取得了許可，為查爾斯提供了更易於被溶解的橡膠而促進合作，<ref name="Page113"/>到了1831年查爾斯與湯瑪斯則將彼此個人創立的公司給一同合併。<ref name="bouncing-balls"/>

除在防水材質製造方面外，查爾斯另在1825年構想一種可將鑄鐵轉換成鋼的方式；<ref name="Todayinsci"/>並在1828年有和同鄉出身的工程師詹姆斯·伯特蒙·尼爾森（James Beaumont Neilson）合作，設計有關可節省熱爐燃料消耗的方法。<ref>{{cite web |url=http://www.oxforddnb.com/view/article/19866 |title=Neilson, James Beaumont |work=[[牛津大學|牛津大學]] |author=Francis Espinasse |language=en |accessdate=2016年12月29日}}</ref>

1843年年6月25日查爾斯離世，死後於[[格拉斯哥大教堂|格拉斯哥大教堂]]火化並葬在教堂旁的墓園。<ref name="Page113"/>

== 家庭 ==
[[File:The_grave_of_Charles_Macintosh,_Glasgow_Cathedral.jpg|thumb]]墓園的查爾斯·麥金托什墓碑。]]
查爾斯的父親喬治是來自[[蘇格蘭高地|蘇格蘭高地]]的住民，之後搬至格拉斯哥並在附近一帶成立以[[地衣|地衣]]來製作染色粉末的工廠。<ref name="Strath"/>查爾斯所娶的妻子瑪麗·費雪則是居於格拉斯哥的一名商人女兒，<ref name="Page113"/>並有為他生下一名同樣取名為喬治的兒子。

== 爭議 ==
早在查爾斯申請製作防水材質的方法前，蘇格蘭外科醫生詹姆斯·賽姆（James Syme）便已有想出類似的構思，但因詹姆斯本身並未對商業行為感興趣而沒即時發表出。<ref>{{cite web |url=http://www.whonamedit.com/doctor.cfm/2088.html |title=James Syme |work=Whonamedit |language=en |accessdate=2016年12月29日}}</ref><ref>{{cite web |url=https://pubs.acs.org/cen/whatstuff/84/8424raincoats.html |title=Charles Macintosh's waterproof-fabric concept, patented more than 180 years ago, is still relevant |work=American Chemical Society |author=Bethany Halford |language=en |date=2006年6月12日 |accessdate=2016年12月29日}}</ref>
