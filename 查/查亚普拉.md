'''查亚普拉'''（Jayapura）是[[印尼|印尼]][[巴布亞省|巴布亞省]]的首府，是[[巴布亚省|巴布亚省]]以及[[西新几内亚|西新几内亚]]的最大城市<ref>{{cite web |url=http://www.citypopulation.de/Indonesia-CU.html |title=INDONESIA: Cities & Settlements |last= |first= |date=2017-04-22 |website=citypopulation.de |publisher= |access-date=2017-07-16 |quote=}}</ref>。位於[[新幾內亞|新幾內亞]]島北岸，2002年時約有20萬人口，2010年人口普查中有256,705人口; 最新的官方估计（截至2014年1月）为315,872。行政区划为查亚普拉市（印尼语：Kota Jayapura）。

現在的查亚普拉的居民裡，大約有80%是從印尼其他地方移入的，[[巴布亞人|巴布亞人]]則佔了約20%左右。1962年起，查亚普拉設有{{le|极乐鸟大学|Cenderawasih University}}（印尼語：Universitas Cenderawasih, UNCEN）。該大學是巴布亞省重要的教育機構。[[巴布亚新几内亚独立国|巴布亚新几内亚独立国]]在查亚普拉设有总领事馆<ref>{{Cite web|url = http://www.kundu-jakarta.com/index.php/about-us/embassy-information/embassy-contacts.html|title = PNG Embassy Office|date = |accessdate = |website = |publisher = |last = |first = }}</ref>。

==地名==

'''查亚普拉'''由[[苏哈托|苏哈托]]命名，为[[梵语|梵语]]'''胜利之城'''，用以纪念{{le|特里科拉行动|Operation Trikora|特里科拉行动}}的成功。

查亚普拉在纳入[[荷兰|荷兰]]统治之前被称为'''诺贝'''（Numbay）或'''诺贝港'''（Port Numbay）。1910年至1962年，[[荷兰|荷兰]]开始统治这一地区，该城名為'''荷蘭地亞'''（Hollandia）。1962年，荷蘭將[[荷屬新幾內亞|荷屬新幾內亞]]交給[[聯合國|聯合國]]管理，该城同时受[[印度尼西亚|印度尼西亚]]和[[荷兰|荷兰]]的双重命名，1962年11月[[印度尼西亚|印度尼西亚]]将其命名为'''哥打巴鲁'''（Kota Baru），[[印尼语|印尼语]]意为''新城市''；[[荷兰|荷兰]]则继续称之为[[荷兰地亚|荷兰地亚]]。[[印度尼西亚|印度尼西亚]]控制这个城市之后，于1963年5月1日短暂更名为'''[[苏卡诺|苏卡诺]]城'''（Sukarnopura）。1968年，再改名為查亚普拉。

==行政区划==

查亚普拉市设立于1993年9月21日。全市共分为五个区，区下再设39个村（其中25个为kelurahan，14个为kampung）。

==地理==

全市地形复杂多样，从沿海平原到丘陵和海拔700米之上的小山。约30%的土地有人居住，其余地区由沼泽、受保护的森林和崎岖的地形区构成。查亚普拉为[[热带雨林气候|热带雨林气候]]，平均气温27℃，雨季和旱季间气温变化不规律。城市到郊区的空气湿度在79%-81%之间变化。

==交通==

===公路===

[[跨巴布亚公路|跨巴布亚公路]]经过查亚普拉。查亚普拉与[[巴布亚新几内亚|巴布亚新几内亚]]之间有国际班车，即查亚普拉-[[瓦尼莫|瓦尼莫]]。查亚普拉市区到边境大约需要45分钟车程。

===铁路===

在巴布亚铁路发展计划中，一期工程从[[索龙_(印尼)|索龙]]到[[马诺夸里|马诺夸里]]，预计2019年完工；二期工程连接到[[纳比雷|纳比雷]]，同期修建查亚普拉到[[萨米_(印尼)|萨米]]的铁路；三期工程将[[萨米_(印尼)|萨米]]和[[纳比雷|纳比雷]]的铁路相连。另有支线从[[纳比雷|纳比雷]]连接[[蒂米卡|蒂米卡]]。相关事宜尚在研究中<ref>http://finance.detik.com/berita-ekonomi-bisnis/3110744/ini-rute-jalur-ka-papua-setelah-sorong-manokwari</ref>。

===航空===

[[仙谷国际机场|仙谷国际机场]]位于邻近的[[查亚普拉县|查亚普拉县]]小城[[森塔尼|森塔尼（仙谷）]]。有飞往印尼各大城市和巴布亚新几内亚的国际航班。机场位于查亚普拉市区以西大约30千米。

==宗教==

查亚普拉的天主教会隶属于[[天主教马老奇总教区|马老奇总教区]]的[[天主教查亚普拉教区|查亚普拉教区]]
