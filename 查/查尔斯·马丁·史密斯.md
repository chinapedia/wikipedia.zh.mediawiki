'''查尔斯·马丁·史密斯'''（{{lang-en|Charles Martin Smith}}，{{bd|1953年|10月30日}}）是一名[[美国|美国]]电影男演员、编剧和导演。

==生平==
===早年===
他生于[[加州|加州]][[圣费尔南多谷|圣费尔南多谷]]的Van Nuys。他父亲Frank Smith是一名电影[[漫画家|漫画家]]、动画制作者兼导演<ref>[http://www.filmreference.com/film/18/Charles-Martin-Smith.html Charles Martin Smith Biography (1953–)<!-- Bot generated title -->]</ref>，他的叔叔Paul J. Smith也是Walter Lantz Studios公司的[[动画电影|动画电影]]制作者兼导演<ref>[http://www.imdb.com/name/nm0006761/ imdb.com]</ref>。他青少年时期在[[巴黎|巴黎]]住了三年，当时他父亲在那里负责一家[[法国|法国]]动画工作室的英语部门<ref>[http://www.theofficialjohncarpenter.com/pages/bios/cmsmit.html John Carpenter] {{webarchive|url=https://web.archive.org/web/20010422171422/http://www.theofficialjohncarpenter.com/pages/bios/cmsmit.html |date=2001-04-22 }} web page.</ref>。他在[[洛杉矶|洛杉矶]]的 ''Grover Cleveland High School'' 读中学。后来考入 ''California State University, Northridge'' 学习，获得[[戏剧|戏剧]]专业学士学位<ref>[http://www.nndb.com/people/372/000078138/ NNDB/Soylent Communications] web page.</ref>。
