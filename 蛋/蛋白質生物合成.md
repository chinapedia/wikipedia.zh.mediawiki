[[File:proteinsynthesis_chinese.png|thumb]]內，[[基因|基因]]被轉錄成[[RNA|RNA]]。RNA接著被後轉錄修飾及控制，形成成熟的[[信使RNA|信使RNA]]（mRNA），並運往細胞核外的[[細胞質|細胞質]]進行[[翻譯_(遺傳學)|翻譯]]。mRNA由[[核糖體|核糖體]]翻譯成與mRNA的基本[[密碼子|密碼子]]，通过与[[轉運RNA|轉運RNA]]上的[[反密碼子|反密碼子]]形成配对进行翻譯。新合成的蛋白質會被再行修飾，并可以與[[效應分子|效應分子]]結合，最终成為具有生物学活性的蛋白質。]]

'''蛋白質生物合成'''是指在生物[[細胞|細胞]]內製造新的[[蛋白質|蛋白質]]，它是通過[[蛋白酶解|蛋白酶解]]或{{link-en|蛋白质导向|Protein targeting}}細胞蛋白的損耗被平衡。[[翻译_(遗传学)|翻译]]，是在核糖體組裝蛋白質，是生物合成途徑的一個重要組成部分，隨著生成的[[信使RNA|信使RNA]]（mRNA），[[tRNA#氨酰化|轉移RNA（tRNA的）氨酰化]]，合作翻譯轉運，並[[翻譯後修飾|翻譯後修飾]]。蛋白質的生物合成在多個步驟有嚴格的调控<ref>Kafri M, Metzl-Raz E, Jona G, Barkai N. 2016. The Cost of Protein Production. Cell Rep 14:22–31. https://dx.doi.org/10.1016/j.celrep.2015.12.015</ref>，和已建立錯誤檢查機制。它们主要是转录（从DNA模板合成RNA的现象）和翻译（从RNA中氨基酸组装的现象）。

[[順反子|順反子]]DNA被轉錄成RNA的各種中間體。最後的版本被用作在合成[[肽|多肽]]鏈的模板。蛋白質通常會直接從基因通過[[翻译_(遗传学)|翻譯]]的[[信使RNA|mRNA]]合成。

這個名詞曾經是指蛋白質的[[翻譯_(遺傳學)|翻譯]]，但現時則是指一個多重的步驟，以[[轉錄|轉錄]]開始及翻譯作結。

[[原核生物|原核生物]]的蛋白質生物合成雖然與[[真核生物|真核生物]]的很相似，但是它们有所不同。

== 氨基酸合成 ==
{{Main|氨基酸合成}}

將[[氨基酸|氨基酸]][[聚合物|聚合]]就可以得到[[蛋白質|蛋白質]]。氨基酸合成是一系列的[[生物化學|生物化學]]過程（[[代謝途徑|代謝途徑]]）將從[[葡萄糖|葡萄糖]]等建立氨基酸。並非所有氨基酸都可以[[生物|生物]]合成，例如成人必需的20中氨基酸中，就有8種氨基酸是人体无法自身合成的，需要從食物中攝取。氨基酸會被運送至[[轉運RNA|轉運RNA]]（tRNA）用在[[翻譯|翻譯]]的過程。

== 轉錄 ==
{{Main|轉錄}}

[[File:MRNA.svg|thumb]]

[[轉錄|轉錄]]是由[[基因組|基因組]]產生含有[[蛋白質|蛋白質]]序列的[[信使RNA|信使RNA]]（mRNA）模板，以進行[[翻譯_(遺傳學)|翻譯]]。只需要[[脫氧核糖核酸|脫氧核糖核酸]]（DNA）雙螺旋的其中一条链就能進行轉錄，此链稱為[[模板链|模板链]]。轉錄可以分為3個階段：起始，延伸和終止，每個階段由大量蛋白質調節，例如[[轉錄因子|轉錄因子]]和{{le|共激活因子|Coactivator (genetics)}}，確保正確的基因被轉錄。

# [[核糖核酸聚合酶|核糖核酸聚合酶]]（RNA聚合酶）先與DNA的用作轉錄起始的特別區域結合。這個結合區域稱為[[啟動子|啟動子]]。當RNA聚合酶與啟動子結合後，該DNA链會開始捲開。
# 第二個轉錄步驟是轉錄延伸。RNA聚合酶與未編碼的模板链一起，合成核糖核苷酸聚合物。RNA聚合酶並不會使用已編碼的链為模板，這是因為任何一股的拷貝會生產一組補充的基礎序列，所以只有未編碼链才會作為模板來複製已編碼链。
# 當聚合酶到達最後階段，全新轉錄的[[信使RNA|信使RNA]]（mRNA）需要修飾以能夠到達[[細胞|細胞]]的其他部份，包括[[細胞質|細胞質]]及[[內質網|內質網]]。一個[[5'端帽|5'端帽]]會加入並保護mRNA，免於降解。[[聚A尾|聚A尾]]會被加入[[3'端|3'端]]保護，並成為接下步驟的模板。在[[真核生物|真核生物]]中，最重要的[[基因拼接|基因拼接]]步驟會於此階段出現，移除[[內含子|內含子]]及合併[[外顯子|外顯子]]。

轉錄的第一個產物在原核細胞中與真核細胞不同，因為在原核細胞中第一個產物是[[信使RNA|信使RNA]]（mRNA），其不需要轉錄後修飾，而在真核細胞中，第一個產物被稱為初級轉錄物，需要後 轉錄修飾（用7-甲基鳥苷加帽，用[[聚A尾|聚A尾]]加尾）得到hnRNA（異源核RNA）。 然後hnRNA通過[[剪接體|剪接體]]經歷[[內含子|內含子]]（基因的非編碼部分）的剪接以產生最終[[信使RNA|信使RNA]]（mRNA）。

== 轉譯 ==
{{Main|翻译 (遗传学)}}

[[File:Protein_synthesis.svg|thumb]]
[[File:Ribosome_mRNA_translation_zh.svg|300px]]

在轉譯的過程中，先前從[[DNA|DNA]]被轉錄的[[mRNA|mRNA]]會被特別的細胞結構解碼以製造[[蛋白質|蛋白質]]，這個特別的細胞結構稱為[[核糖體|核糖體]]。蛋白質生物合成會被分為起始、延伸及完成階段。

核糖體可以提供場所，讓另一種特別的[[RNA|RNA]]，稱為[[轉運RNA|轉運RNA]]（tRNA）與mRNA結合。tRNA內有一組「反密碼子」能將相對應的序列與mRNA在核糖體內互相形成[[氫鍵|氫鍵]]。因此，相對應的的tRNA（[[化學|化學]]上與特定的[[胺基酸|胺基酸]]結合）會被導向至核糖體，加入在發展中的[[多肽|多肽]]。以下展示了兩個胺基酸聯合的化學過程：

[[File:Amino-veresterung.jpg|framed]]

核糖體會一個接一個的在mRNA的[[密碼子|密碼子]]運行，另一個tRNA會借核糖體附著mRNA。首個tRNA會被釋放，但附著該tRNA的胺基酸會被運送至第二個tRNA，並與它的胺基酸結合。這種轉移不斷進行，直至生成一條胺基酸的長鏈。 

當整個單位到達mRNA最後的密碼子時，它會離開及新形成的蛋白質會被釋放，這就是完成的階段。在這個步驟中，有很多的[[酶|酶]]會用來協助整個過程。

== 蛋白質翻譯之後的事件 ==
{{Main|蛋白酶解|翻譯後修飾|蛋白质折叠}}

生物合成之後的事件包括[[翻譯後修飾|翻譯後修飾]]和[[蛋白质折叠|蛋白質折疊]]。在合成的期间和之後，多肽鏈往往折疊卷曲成所謂的原生[[二級結構|二級結構]]和[[三級結構|三級結構]]。這就是所謂的[[蛋白質折疊|蛋白質折疊]]。

很多蛋白質會進行翻譯後修飾。這包括[[雙硫鍵|雙硫鍵]]的形成或附在任何生化[[官能團|官能團]]，如[[醋酸鹽|醋酸鹽]]、[[磷酸鹽|磷酸鹽]]、不同的[[脂類|脂類]]及[[糖類|糖類]]。有些[[酶|酶]]亦會將多肽鏈前端的一個或以上的[[胺基酸|胺基酸]]移除，或者促进多肽链形成链内或链间[[二硫鍵|二硫鍵]]。
