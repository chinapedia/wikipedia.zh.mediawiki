'''豫园'''座落于[[中国|中国]][[上海市|上海市]][[黄浦区|黄浦区]]，原[[上海老城厢|上海老城厢]]东北部，北靠[[福佑路|福佑路]]，东临[[安仁街|安仁街]]，西南有[[上海城隍庙|城隍庙]]、[[豫园商城|豫园商城]]，是建于[[明朝|明朝]]时期的古典园林。豫园建于1559年，最初是[[四川|四川]][[布政使|布政使]][[潘允端|潘允端]]的私人园林，他建造此园林的目的是“愉悦老亲”，而“愉”和“豫”意思相通，故而得名豫园<ref name="海上名园">[http://shtong.gov.cn/newsite/node2/node71994/node71995/node71998/node72045/node72069/userobject1ai77479.html 上海名街志——海上名园 名不虚传] 上海地方志办公室.</ref>。之后，豫园辗转于多人之手并经受多次战火的摧残。如今，其是典型的[[江南园林|江南园林]]代表并已经成为上海著名的游览胜地。

豫园于1959年被公布为[[上海市文物保护单位|上海市文物保护单位]]，1961年开始对公众开放，1982年被[[国务院|国务院]]列为[[全国重点文物保护单位|全国重点文物保护单位]]。園內有[[江南三大名石|江南三大名石]]的玉玲珑和原[[小刀会|小刀会起义]]的指挥所——[[点春堂|点春堂]]。园側亦有[[上海城隍庙|城隍廟]]及商店街等遊客景點，附近有多家著名食店，包括以[[小籠饅頭|小籠饅頭]]著名的[[南翔馒头店|南翔馒头店]]、宁波汤团店、[[綠波廊|綠波廊]]及[[上海老飯店|上海老飯店]]。

== 历史 ==
豫园由[[潘允端|潘允端]]在[[嘉靖|嘉靖]]三十八年始建于家宅世春堂西的大片菜畦上，但因在外地做官，无暇兼顾建筑事宜，故园林到[[万历|万历]]末年，潘退休后才得以竣工<ref name="始建"/>。园林由[[张南阳|张南阳]]担任设计和叠山，当时共占地70余亩，被誉为“东南名园冠”<ref name="始建">[http://shtong.gov.cn/newsite/node2/node4/node2249/nanshi/node44902/node44904/node44906/userobject1ai30791.html 南市区志——第一节 始建] 上海地方志办公室.</ref>。然而之后，潘家家道衰落，在明末为潘允端孙婿张肇林所有。清朝时期，豫园被上海一些士绅改为清和书院，供以时[[松江|松江]]知府[[张羽明|张羽明]]的长生禄位<ref name="兴衰">[http://shtong.gov.cn/newsite/node2/node4/node2249/nanshi/node44902/node44904/node44906/userobject1ai30792.html 南市区志——第二节 兴衰] 上海地方志办公室.</ref>。后书院尚未完工，张羽明便遭到罢免，只得停工。停工后原来园内的优美风景也早就荡然无存了。1709年，上海士绅购得城隍庙东部约2亩的土地建造庙园，即灵苑，又称东园，用于公共活动之场所。1760年，原豫园旧地由被一些富商购买后重新修建，园林在24年后再度得以恢复，称为“西园”<ref name="兴衰"/>。

在[[第一次鸦片战争|第一次鸦片战争]]中，园林遭到破坏，湖心亭甚至成为了英军的司令部。自此之后的数百年里，豫园一直经受着战火的摧残。1855年，小刀会起义失败，清军驻扎豫园，豫园再度遭到破坏。1860年，[[太平军|太平军]]东征，豫园成为了西方洋枪队的兵营<ref name="兴衰"/>。清[[嘉庆|嘉庆]]、[[道光|道光]]年间开始，豫园逐渐成为了一些行业的同业公所，茶楼和酒馆等也逐渐开始出现。但在此同时旧有的园林景观也早已不存了。在解放后，各公所逐渐迁出<ref name="海上名园"/>，1956年起，豫园的修复工程开始陆续进行，初始规模得以大半恢复。

== 景观 ==
豫园共有西部、东部、中部和内园四个景区。西部景区有三穗堂、仰山堂、点春堂、大假山和元代铁狮等知名景观。三穗堂为修建西园时所建，在清代是官府庆典和“宣讲圣谕”之处。大假山是[[江南|江南]]地区现存最古老和最大的黄石假山，由张南阳设计<ref>[http://www.yugarden.com.cn/yugarden/shengJing.aspx?code=200100&page=1 三穗堂景区] {{webarchive|url=https://web.archive.org/web/20140729151350/http://www.yugarden.com.cn/yugarden/shengJing.aspx?code=200100&page=1 |date=2014-07-29 }} 豫园.</ref>。山高约14米，被周允端被人称为“峻，颇惬观赏”。东部景区有江南三大名石之一的玉玲珑、积玉水廊、会景楼等景观。中部景区有得月楼、织亭、浣云假山等景观。

内园景区在修复时与豫园连接在了一起，故可谓是“园中之园”，其只有二亩一分八厘六毫大，袖珍却十分精致<ref>[http://www.yugarden.com.cn/yugarden/shengJing.aspx?code=200600&page=3 内园景区] {{webarchive|url=https://web.archive.org/web/20140729153041/http://www.yugarden.com.cn/yugarden/shengJing.aspx?code=200600&page=3 |date=2014-07-29 }} 豫园.</ref>。内园景区包括古戏台、九龙池、观涛楼等景观。其中古戏台原位于闸北上海北钱业公所内，后在修缮中移至此处。九龙池则因东西两壁隙间中藏的4个石雕龙头在水中的倒影与形似龙身的池体而得名。
