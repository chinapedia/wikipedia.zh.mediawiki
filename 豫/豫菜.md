{{中国菜}}

'''豫菜'''，即'''河南菜系'''，是对在带有[[中原文化|中原传统文化]]内涵的烹饪理论指导下，运用具备[[中原|中原地域]]特点的技术和材料所制作的菜肴、面点和筵席的总称<ref>{{cite web |url = http://henan.mofcom.gov.cn/aarticle/sjshangwudt/200708/20070804998189.html |title = 豫菜定义确定重提豫菜振兴 |publisher=[[中华人民共和国商务部|中华人民共和国商务部]] |date=2007-08-16 |language = zh-hans }}</ref>。豫菜是[[中國菜系|中国各大菜系]]的渊源<ref name="ycyy">{{cite web |url = http://newpaper.dahe.cn/hnrbncb/html/2009-03/31/content_163891.htm |title = 豫菜成大器 任重而道远 |publisher=大河网 |date=2009年3月31日| language = zh-hans }}</ref>，被中国烹饪界称为中国八大菜系的“母菜”<ref>{{cite web |url = http://eat.gd.sina.com.cn/news/2006-01-18/2108629.html |title=感受豫菜文化传统底蕴 |publisher=广东新浪 |date=2006-01-18| language =zh-hans }}</ref>，有“烹饪鼻祖”和“中华厨祖”之称的[[伊尹|伊尹]]便出生于河南[[洛阳|洛阳]]，当代豫菜是在原宫廷菜、官府菜、市肆菜和民间菜的基础上，根据中原物质条件，逐步积累演变而发展起来的<ref name="ycyy" />。

因河南地处[[中原地区|中原地区]]，其饮食文化口味因地域影响，河南北部多为面食，南部多为米饭。豫菜中一些菜名比较华丽优美，主要因为一些菜式源自于[[汉|汉]][[宋|宋]][[唐|唐]]宫廷，有些饮食文化历史可以追溯到[[商|商]][[周|周]]时期。

==豫菜特点==
现今的豫菜以[[郑州|郑州]]为中心，由四个部分构成:
*豫东以[[开封|开封]]为代表，恪守传统、扒制类菜肴较为典型，口味居中
*豫西以[[洛阳|洛阳]]为代表，水席为典型风味，口味稍偏酸
*豫南以[[信阳|信阳]]为代表，炖菜类较为典型，口味稍偏辣
*豫北以[[安阳|安阳]]为代表，善用土特产，熏卤、面食甚佳，口味稍重
