== 室女座α流星雨 ==
'''室女座α流星雨'''發生在3月10日至5月6日，峰值在4月7日至18日之間，每小時約5-10顆流星。它們最初是在1895年中檢測到的<ref>{{cite web|url=http://meteorshowersonline.com/showers/alpha_virginids.html|title=Alpha Virginids|publisher=Meteor Showers Online|pages=1|accessdate=2008-08-05|archive-url=https://web.archive.org/web/20080827202359/http://meteorshowersonline.com/showers/alpha_virginids.html|archive-date=2008-08-27|dead-url=yes}}</ref>。

== 室女座γ流星雨 ==
The North and South '''室女座γ流星雨'''是分為南北兩群的慢速的小流星雨<ref name="gamma1"/><ref name="gamma2"/>，然而5月的室女座γ流星雨和白晝的室女座γ流星雨都是快速的移動<ref name="gamma3"/><ref name="gamma4"/>。南北兩群的母體分別是[[2002_FC|2002 FC]]和[[2003_BD44|2003 BD44]]<ref name="gamma1"/><ref name="gamma2"/>。它通常出現在4月5日至4月21日，並且在4月14-15日到達峰直，每小時的流星數量低於5顆。首度發現是在1895年<ref>{{cite web|url=http://meteorshowersonline.com/showers/gamma_virginids.html|title=Gamma Virginids|publisher=Meteor Showers Online|accessdate=2008-08-07|archive-url=https://web.archive.org/web/20080827201148/http://meteorshowersonline.com/showers/gamma_virginids.html|archive-date=2008-08-27|dead-url=yes}}</ref>。

== 室女座η流星雨 ==
'''室女座η流星雨'''發生在2月24日至3月27日，峰值在3月18日左右，每小時只有1-2顆流星。在1961年第一次被檢測到<ref>{{cite web|url=http://meteorshowersonline.com/showers/eta_virginids.html|title=Eta Virginids|publisher=Meteor Showers Online|pages=1|accessdate=2008-08-05|archive-url=https://web.archive.org/web/20080807161455/http://meteorshowersonline.com/showers/eta_virginids.html|archive-date=2008-08-07|dead-url=yes}}</ref>。

== 室女座θ流星雨 ==
'''室女座θ流星雨'''發生在3月10日至4月21日，3月20日左有達到峰直，每小時只有1-3顆流星。最初是在1850年觀測到，至1948年才被確認<ref>{{cite web|url=http://meteorshowersonline.com/showers/theta_virginids.html|title=Theta Virginids|publisher=Meteor Showers Online|pages=1|accessdate=2008-08-05|archive-url=https://web.archive.org/web/20080807160910/http://meteorshowersonline.com/showers/theta_virginids.html|archive-date=2008-08-07|dead-url=yes}}</ref>。

== 室女座ι流星雨 ==
'''室女座ι流星雨'''是一個微弱的白晝流星雨<ref name="iota"/>。

== 室女座λ流星雨 ==
'''室女座λ流星雨'''是一個小流星雨<ref name="lambda"/>。

== 室女座μ流星雨 ==
室女座μ流星雨是在4月和5月初可見的小流星雨<ref>{{cite web|url=http://www.meteorobs.org/maillist/msg21552.html|title=(meteorbs) NAMN Notes: May 2001|last=Davis|first=Mark|date=3 May 2001|publisher=meteorbs|accessdate=2008-08-07|deadurl=yes|archiveurl=https://web.archive.org/web/20080516093352/http://www.meteorobs.org/maillist/msg21552.html|archivedate=2008年5月16日}}</ref>。通常地出現日期是從4月1日至5月12日，峰值在4月24日<ref>{{cite web|url=http://www.imo.net/files/data/vmdb/vmdbrad.txt|title=meteor shower list|publisher=International Meteor Organization|accessdate=2008-08-07}}</ref>至4月25日。它的輻射點靠近[[天秤座|天秤座]]，每小時的流星樹在7-10顆<ref>{{cite web|url=http://www.projectpluto.com/gloss/help_17.htm|title=glossary|publisher=Project Pluto|accessdate=2008-08-07}}</ref>。

== 室女座π流星雨 ==
'''室女座π流星雨'''發生在2月13日至4月18日，峰值在3月3日和3月9日，每小時有2-5顆流星。這個流星雨在1908年首度被觀測到，並且在1948年得到確認<ref>{{cite web|url=http://meteorshowersonline.com/showers/pi_virginids.html|title=Pi Virginids|publisher=Meteor Showers Online|pages=1|accessdate=2008-08-05|archive-url=https://web.archive.org/web/20080807161559/http://meteorshowersonline.com/showers/pi_virginids.html|archive-date=2008-08-07|dead-url=yes}}</ref>。

== 室女座φ流星雨 ==
'''室女座φ流星雨'''是一個小的白晝流星雨<ref name="psi"/>。

== 室女座三月流星雨 ==
'''室女座三月流星雨'''是一個小流星雨；輻射點在室女座的北邊，母體可能是[[1998_SJ70|1998 SJ70]]<ref name="march"/>{{dead link|date=March 2014}}。[[獅子座β流星雨|獅子座β流星雨]]也被稱為'''室女座三月流星雨'''，持續時間從2月14日至4月25日，3月20日左右到達峰值，每小時的流星數量約3-4顆<ref>{{cite web|url=http://meteorshowersonline.com/showers/beta_leonids.html|title=Beta Leonids|publisher=Meteor Showers Online|accessdate=2008-08-07|archive-url=https://web.archive.org/web/20080516071922/http://meteorshowersonline.com/showers/beta_leonids.html|archive-date=2008-05-16|dead-url=yes}}</ref>。
