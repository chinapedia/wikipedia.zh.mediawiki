'''常林'''{{BD|144？||227？|catIdx=C}}，[[表字|字]]'''伯槐'''，[[河內|河內]][[溫縣|溫縣]]（今[[河南|河南]]溫縣）人。[[東漢|東漢]]末及[[三國|三國]]時[[曹魏|曹魏]]官員。

== 生平 ==
常林早年都在家鄉河內郡以耕作為生，生活貧困。[[初平|初平]]元年（190年），河內[[太守|太守]][[王匡_(三国)|王匡]]起兵討伐[[董卓|董卓]]，派了門生到所屬縣內監察人民有沒有犯罪，發現後就立刻將他們收押，命他們要以錢或穀物贖罪，遲了交就要被屠滅宗族，藉以提升威嚴。當時常林叔父在河內郡毆打傷人，被王匡門生發現，因而被王匡捉拿；當時常林的宗族都很害怕罰金有多少，怕負擔不了會連帶一起被誅殺，不去營救。常林於是找來王匡同鄉[[胡母彪|胡母彪]]，向他陳述王匡捉拿常林叔父的最終目的。胡毋彪聽後即寫書譴責王匡，王匡最終都饒恕常林叔父。後來常林又依附在[[上黨郡|上黨郡]]的前[[河間|河間]]太守[[陳延壁|陳延壁]]。當地[[陳|陳]]、[[馮|馮]]二姓都是以前的富貴豪族，上黨太守[[張楊|張楊]]貪圖二家的女子和財寶，出兵圍困他們；常林帶領他的宗族，為他們兩姓人劃策，最終被圍困六十日後解圍，他們兩族都沒有損失。

[[并州|并州]][[刺史|刺史]][[高幹|高幹]]後來[[表|表]]常林為[[騎都尉|騎都尉]]，常林辭讓不接受。高幹叛亂被[[曹操|曹操]]平定後，[[梁習|梁習]]上任并州刺史，推舉州內名士，常林和[[楊俊_(三國)|楊俊]]、[[王象|王象]]、[[荀緯|荀緯]]和[[王淩|王淩]]等都獲舉薦，曹操將常林任命為[[南和|南和]][[縣長|縣長]]。後因為在南和有治績，先後獲提拔為[[博陵|博陵]]太守和[[幽州|幽州]]刺史，亦都有治績。[[建安_(漢獻帝)|建安]]十六年（211年），[[曹丕|曹丕]]獲任命為[[五官中郎將|五官中郎將]]，常和被選為五官將[[功曹|功曹]]。同年曹操西征[[馬超|馬超]]等人，河間人[[田银|田银]]和[[苏伯|苏伯]]在當地叛亂，並煽動鄰近的幽州和[[冀州|冀州]]，曹丕打算親自討平，但常林認為他們只是烏合之眾，難成大器；而且曹操出征，曹丕應安守鄴城，不應輕率出征。曹丕聽從並派將軍[[賈信|賈信]]討平。常林後來又先後出任[[平原|平原]]太守、[[魏郡|魏郡]]東部[[都尉|都尉]]和[[丞相|丞相]][[東曹|東曹]][[屬|屬]]。建安十八年（213年），曹操封魏公並建魏國，常林升任[[尚書|尚書]]。

[[黃初|黃初]]元年（220年），曹丕稱帝，常林遷任[[少府|少府]]，封'''樂陽亭侯'''。後又轉任[[大司農|大司農]]。[[太和_(曹叡)|太和]]元年（227年），[[曹叡|曹叡]]繼位，封'''高陽鄉侯'''，改任[[光祿勳|光祿勳]]、[[太常|太常]]。當時有人認為常林節操廉潔高尚，應該進為三公，常林於是稱病。後轉任[[光祿大夫|光祿大夫]]。常林八十三歲時逝世，獲追贈驃騎將軍，以公禮安葬，[[諡|諡]]'''貞侯'''。

== 性格特徵 ==
*常林年少孤貧，但不是自己力量換取的他都不要。
*常林為人個性好學，東漢末年做門生時更帶著經書去耕作，由妻子送飯給他。而且二人相敬如賓。
*常林和[[司馬懿|司馬懿]]是同鄉，而司馬懿認為常林在故鄉人中年老德高的人，見面都對常林行拜禮。有人認為司馬懿位高權重，常林不應受拜，應當停止。但常林則認為司馬懿是自願遵守長幼的禮序，作後人的榜樣；而他亦不畏懼有權勢人士，同時拜禮亦不是他制定要人作的，不認為有任何不妥。

== 子女 ==
*[[常旹|常旹]]，常林之子，嗣侯，官至[[泰山|泰山]]太守。後因[[诸葛诞|诸葛诞]]作乱时称疾不出而獲罪，與[[李廣（三國）|李廣]]一起被司馬昭所誅。
*[[常靜|常靜]]，常峕之弟，常峕死後獲封為繼嗣。

==評價==
[[陳壽|陳壽]]：和洽清和幹理，'''常林素业纯固'''，杨俊人伦行义，杜袭温粹识统，赵俨刚毅有度，裴潜平恆贞幹，皆一世之美士也。'''至（常）林能不系心於三司，以大夫告老，美矣哉！'''
