'''常州市文物保护单位'''是[[中国|中国]][[江苏省|江苏省]][[常州市|常州市]]的市级[[文物保护单位|文物保护单位]]，目前已公布六批，共170处（另有16处已升级），已升级至[[常州市境内的江苏省文物保护单位列表|江苏省文物保护单位]]和[[全国重点文物保护单位|全国重点文物保护单位]]的不再列入。

*第一批：1964年4月公布，数量不详。
*第二批：1987年12月26日公布，数量不详。
*第三批：2003年12月公布，数量不详。
*第四批：2008年2月26日公布，共68处<ref>{{cite web |url=http://www.changzhou.gov.cn/zhuanti/cznj/2010/social_cause.asp-id=251&listid=33.htm|title=《常州年鉴·文博事业》|accessdate=2015-10-24|work=|publisher=常州市政府}}</ref>。
*第五批：2011年1月6日公布，共56处<ref>{{cite web |url=http://news.cz001.com.cn/2011-01/20/content_1462008.htm|title=《我市公布第五批市级文物保护单位 附名录》|accessdate=2015-10-24|work=|publisher=中国常州网}}</ref>。
*第六批：2013年4月15日公布，共18处<ref>{{cite web |url=http://www.changzhou.gov.cn/gi_news/201136635855229|title=《市政府关于公布第六批市级文物保护单位的通知》（常政发〔2013〕55号）|accessdate=2015-10-24|work=|publisher=常州市政府}}</ref>。
