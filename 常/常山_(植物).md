==分佈==
本種現分佈於[[亞洲|亞洲]]的[[中國|中國]]、[[台灣|台灣]]、[[日本|日本]][[琉球群島|琉球群島]]<ref name="海南">{{cite encyclopedia| title=常山| encyclopedia=海南植物誌| volume=2| pages=190-191| url=http://v2.cvh.org.cn/difangzhi/hainan/page/2/190a.pdf}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、[[越南|越南]]、[[緬甸|緬甸]]、[[泰國|泰國]]、[[柬埔寨|柬埔寨]]、[[老撾|老撾]]、[[馬來西亞|馬來西亞]]、[[印尼|印尼]]<ref name="貴州">{{cite encyclopedia| title=常山| encyclopedia=貴州植物誌| volume=8| pages=53| url=http://v2.cvh.org.cn/difangzhi/guizhou/page/8/053.pdf}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、[[菲律賓|菲律賓]]<ref name="貴州"/>、[[印度|印度]]、[[尼泊爾|尼泊爾]]、[[不丹|不丹]]等地<ref name="ars"/>。[[中國|中國]]國內分於[[廣西|廣西]]、[[廣東|廣東]]、[[香港|香港]]<ref name="香港"/>、[[福建|福建]]、[[四川|四川]]、[[安徽|安徽]]、[[湖南|湖南]]、[[湖北|湖北]]、[[甘肅|甘肅]]、[[江西|江西]]、[[陝西|陝西]]、[[西藏|西藏]]、[[貴州|貴州]]等省區<ref name="ars"/>，本種喜生於山野、林邊、溝邊等陰濕之處<ref name="貴州"/><ref name="浙江">{{cite encyclopedia| title=常山| encyclopedia=浙江植物誌| volume=3| pages=112-113| url=http://v2.cvh.org.cn/difangzhi/zhejiang/page/3/112.pdf}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，亦有人工栽種<ref name = "南方藥用">{{cite book|title=《南方藥用植物 近600種中國南方藥用植物的彩色圖鑒》|author= |series=王玉生、蔡岳文|publisher=南方日報出版社|page=第94頁|isbn= 978-7-5491-0180-1|date=2011年4月}}</ref>。

==形態特徵==
常山是一種落葉[[灌木|灌木]][[植物|植物]]，高約0.4－2[[公尺|公尺]]<ref name="福建">{{cite encyclopedia| title=常山| encyclopedia=福建植物誌| volume=2| pages=190-191| url=http://v2.cvh.org.cn/difangzhi/fujian/page/2/231.pdf}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。主根圓柱形，常為彎曲<ref name="浸大"/>，粗壯，木質化<ref name="河南"/>，表皮灰棕或黃棕色<ref name="浸大"/>，斷面黃色，粗約1.5[[厘米|厘米]]<ref name="河南"/>，長約30[[厘米|厘米]]以上<ref name="浸大"/>；莖枝直立<ref name = "中藥彩色">{{cite book|title=《中華人民共和國藥典中藥彩色圖集(1995年版)》|author= |series=中華人民共和國衛生部藥典委員會|publisher=三聯書店|page=第405-406頁|isbn=962-04-1326-1|date=1996年11月}}</ref>，圓形，綠色<ref name = "中藥彩色"/>，幼時披黃色短毛，具明顯的節<ref name="浸大"/>；小枝圓柱形或稍有4鈍棱<ref name="浙江，無毛或疏披黃色微柔毛<ref name="河南"/>，表面綠色，帶紫色，稍肉質<ref name="浙江"/>。

===葉===
[[葉|葉]]為長圓形、橢圓形、倒卵狀橢圓形、闊披針形<ref name="貴州"/>，少為披針形<ref name="浸大"/>，形狀差異大，單葉對生<ref name = "南方藥用"/>，頂端短漸尖<ref name="西藏">{{cite encyclopedia| title=常山| encyclopedia=西藏植物誌| volume=2| pages=528-529| url=http://v2.cvh.org.cn/difangzhi/xizang/page/2/528a.pdf}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，基部楔形，邊緣疏披或密披鋸齒或小齒<ref name="海南"/>，表面深綠色<ref name="浸大"/>，無毛或披黃色柔毛<ref name="西藏"/>，背面有時稀疏短柔毛<ref name="浙江"/>，葉中脈表面凹陷，網脈表面不明顯<ref name="貴州"/>，側脈每邊8－10對<ref name="福建"/>，稍為彎拱向上<ref name="浙江"/>，長約5<ref name="貴州"/> －25<ref name="浙江"/>[[厘米|厘米]]，寬約2<ref name="海南"/>－10<ref name="浙江"/>[[厘米|厘米]]；[[葉柄|葉柄]]無毛或有時疏披短柔毛<ref name="浙江"/>，長約1<ref name="浸大"/>－5<ref name="浙江"/>[[厘米|厘米]]。

===花果===
[[花|花]]為兩性花<ref name="西藏"/>，藍色或青紫色，密生而排列成近圓錐狀傘房[[花序|花序]]，頂生或長於枝條頂端的葉腋<ref name="浙江"/>，披細柔毛，長可達8[[厘米|厘米]]<ref name="福建"/>；[[花蕾|花蕾]]時近球形<ref name="河南"/>，盛開時花[[直徑|直徑]]約1[[厘米|厘米]]<ref name="浙江"/>；[[花序梗|花序梗]]密披棕黃色短柔毛<ref name="河南"/><ref name="西藏"/>，長約2[[厘米|厘米]]<ref name="浸大"/>；小花梗披短柔毛<ref name="河南"/><ref name="西藏"/>，長約3－5[[毫米|毫米]]<ref name="浸大"/>；[[苞片|苞片]]線狀披針形，早落<ref name="浸大"/>；[[花萼|花萼]]倒圓錐形，淡藍色，長約4[[毫米|毫米]]<ref name="浸大"/>；[[萼片|萼片]]4－7枚，寬三角形，急尖<ref name="海南"/>，密披棕色細柔毛<ref name="浸大"/><ref name="福建"/>；[[花瓣|花瓣]]長圓狀橢圓形或卵形<ref name = "浸大"/>，藍白色<ref name = "有害植物"/>，頂端鈍，基部截形<ref name = "浸大"/>，4－7枚，近肉質，無毛<ref name = "福建"/>，花開時反卷，長約8[[毫米|毫米]]，寬約3[[毫米|毫米]]<ref name = "浸大"/>；[[雄蕊|雄蕊]]10－20枚，長於[[花瓣|花瓣]]基部<ref name = "浸大"/>，半數與[[花瓣|花瓣]]對生；[[花絲|花絲]]扁平，常有斑點<ref name = "西藏"/>，初時與[[花瓣|花瓣]]貼着，其後分離，長約2－6[[毫米|毫米]]<ref name = "浸大"/>；[[花藥|花藥]]近長橢圓形，藍色<ref name = "浸大"/>，無毛，2室縱裂<ref name = "浸大"/>，於[[花蕾|花蕾]]時藥室的背部貼着[[花絲|花絲]]<ref name = "海南"/>；[[雌蕊|雌蕊]]1枚<ref name = "有害植物"/>；[[子房|子房]]半下位<ref name = "有害植物"/>，長圓形，藍色，1室，胚珠多數<ref name = "浸大"/>；[[花柱|花柱]]棒狀<ref name = "浙江"/>，4－6枚，初為基部合生<ref name = "貴州"/>，上半部擴大<ref name = "海南"/>，長約2[[毫米|毫米]]<ref name = "浸大"/>；[[柱頭|柱頭]]長橢圓狀卵形<ref name = "海南"/>。[[果|果]]為[[漿果|漿果]]，卵球形<ref name = "浙江"/>，成熟時深藍色<ref name = "中藥彩色"/><ref name = "有害植物"/>，乾後變為黑色<ref name = "浸大"/>，[[直徑|直徑]]約4<ref name = "貴州"/>－7<ref name = "浸大"/>[[毫米|毫米]]；具宿存的[[花柱|花柱]]及[[萼片|萼片]]<ref name = "河南"/>；[[種子|種子]]多數，長約1[[毫米|毫米]]，[[種皮|種皮]]具網紋<ref name = "浙江"/>。

==醫藥用途==
{{medical}}
===根莖===
本種以乾燥根莖入藥，中藥名為常山，始載於《[[神農本草經|神農本草經]]》，列為下品<ref name = "當代藥用">{{cite book|title=《當代藥用植物典 1》|author= |series=趙中振、蕭培根|publisher=香港賽馬會中藥研究院|page=第309-311頁|isbn=978-988-99226-0-3|date=2006年8月}}</ref>，《[[神農本草經|神農本草經]]》中原文記載為「常山，味苦，寒；主傷寒寒熱；熱發溫瘧；鬼毒；胸中痰結，吐逆。一名互草。生川谷。」<ref name = "神農本草經"/>。[[藥材|藥材]]主產於[[四川|四川]]、[[貴州|貴州]]、[[湖南|湖南]]、[[湖北|湖北]]、[[廣西|廣西]]等省區<ref name = "當代藥用"/>，味苦、辛，性寒，有毒<ref name = "本草綱目"/>，歸肺、心、肝經<ref name = "中藥的故事"/>，為[[中醫|中醫]]傳統治療[[瘧疾|瘧疾]]的藥物，但副作用較大<ref name = "當代藥用"/>。具截瘧、祛[[痰|痰]]、散結、清熱解毒、燥濕利痰、活血止痛等功效<ref name = "浸大"/><ref name = "神農本草經"/><ref name = "當代藥用"/>，主治[[瘧疾|瘧疾]]、[[傷寒|傷寒]]、傷寒引致的發燒、發燒造成的神志異常、腹痛、欲吐不能、胸膈脹滿、宿食難消、老痰積飲、熱毒瘡瘍、[[風濕|風濕]]痹痛、經閉痛經、跌打損傷等<ref name = "浸大"/><ref name = "神農本草經"/>。現代藥理研究表明常山中的活性成份黃常山鹼為抗瘧的有效成份<ref name = "當代藥用"/>，常山具抗瘧、抗[[阿米巴原蟲|阿米巴原蟲]]、抗[[病毒|病毒]]感染、抗炎、抗[[腫瘤|腫瘤]]、降[[血壓|血壓]]<ref name = "神農本草經"/>、殺滅[[滴蟲|滴蟲]]、催吐、解熱等作用<ref name = "浸大"/><ref name = "當代藥用"/>，水浸膏對雞瘧具顯著療效<ref name = "神農本草經"/>，臨床應用於[[結腸炎|結腸炎]]、病毒性[[肝炎|肝炎]]、急性[[闌尾炎|闌尾炎]]、慢性[[盆腔炎|盆腔炎]]、[[高膽紅素血症|高膽紅素血症]]等治療上<ref name = "當代藥用"/>。但因有催吐的副作用，[[孕婦|孕婦]]忌用<ref name = "中藥彩色"/>；老年體弱者慎服<ref name = "南方藥用"/>；因易損傷真氣，故久病體弱及氣虛者不宜服用<ref name = "神農本草經"/>。

===枝葉===
本種以枝葉入藥，中藥名為蜀漆，《[[神農本草經|神農本草經]]》中原文記載為「蜀漆，味辛，平。主瘧及欬逆寒熱；腹中癥堅、痞結積聚；邪氣蛊毒、鬼疰。生川谷。」<ref name = "神農本草經"/>，味辛，性平，有毒<ref name = "本草綱目"/>，具開通氣血、通利水道等功效<ref name = "神農本草經"/>，主治瘧疾、咳嗽氣喘、時冷時熱、欲吐不能、胸膈脹滿、老痰積飲、咳逆寒熱等，現代藥理研究表明蜀漆與常山藥效基本相同<ref name = "神農本草經"/>。

==藥材鑑定==
[[中藥|中藥]]常山以乾燥根莖入藥，於[[秋季|秋季]]採挖，清除鬚根等雜質後洗淨曬乾<ref name="香港中藥材">{{cite book|title=《香港中藥材圖鑑》|author= |series=趙中振|publisher=香港浸會大學中醫藥學院|page=第290頁|isbn= 988-97448-1-3|date=2003年10月}}</ref>；本品呈圓柱形，質地堅硬，枯瘦光滑<ref name = "常用中藥材">{{cite book|title=《常用中藥材鑑別圖典》|author= |series=趙中振、陳虎彪|publisher=萬里機構|page=第168頁|isbn= 978-962-14-4150-8|date=2010年1月}}</ref>，不易折斷，折斷時有粉塵飛揚，呈乾柴狀<ref name="香港中藥材"/>；表皮棕黃色，剝落處露出淡黃色木部，木化程度高，橫切面黃白色，呈放射狀，射線類白色<ref name="香港中藥材"/>；表皮容易剝落，具細縱紋<ref name="香港中藥材"/>；常呈彎曲扭轉，或有分枝，形如雞骨狀<ref name = "常用中藥材"/>，[[直徑|直徑]]約0.5－2[[厘米|厘米]]，長約9－15[[厘米|厘米]]<ref name="香港中藥材"/>；氣微，味微苦，無臭<ref name = "常用中藥材"/>；傳統經驗則認為以質地堅硬及斷面色淡黃色者為佳<ref name = "常用中藥材"/>。本品種載錄於《[[中國藥典|中國藥典]]》2005年版，定為[[中藥|中藥]]常山的法定原植物來源種，主要透過性狀、組織粉末特徵等鑒別資料，以控制[[藥材|藥材]]的質量<ref name = "當代藥用"/>。

==毒性==

本品全株具毒性，服用10克乾燥的根部足以導致中毒，毒性成份為[[常山鹼|常山鹼]]，常山鹼對[[迷走神經|迷走神經]]及[[交感神經|交感神經]]末梢起刺激作用，並會破壞[[黏膜|黏膜]]，引致腸胃道出血等症狀，中毒症狀包括噁心、嘔吐、腹痛、腹瀉、吐血、便血、低血壓、心悸、心律不正、[[循環系統|循環系統]]衰竭，嚴重甚至死亡<ref>{{cite web| url=http://www3.ha.org.hk/toxicplant/hk/dichroa_febrifuga.html| title= Dichroa febrifuga 常山| publisher=香港醫院管理局毒理學參考化驗室香港有毒植物圖鑑| deadurl=yes| archiveurl= https://web.archive.org/web/20180518184512/http://www3.ha.org.hk/toxicplant/hk/dichroa_febrifuga.html | archivedate=2018-05-18}}</ref>。
