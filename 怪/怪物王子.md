《'''怪物小王子'''》（{{lang|ja|怪物くん}}）是[[藤子不二雄A|藤子不二雄A]]的日本[[少年漫畫|少年漫畫]]作品，另外亦有改編的動畫。日文標題則是主角的通稱。2010年4月、[[日本電視台|日本電視台]]預定要加以改編成[[電視劇|電視劇]]。其中'''くん'''，是日本对較年輕男性的暱稱。 <!-- 它也必须出示2011年分拆出去的活动。-->

1980年代在香港[[無線電視|無線電視]][[翡翠台|翡翠台]]播放，香港播出時譯作《'''怪物王子'''》、《'''[[怪物小王子|怪物小王子]]'''》，中國大陆翻译为《'''怪物太郎'''》，台灣在無版權時期的出租帶上也曾譯為《'''怪物王子'''》。大陆由[[人民美术出版社|人民美术出版社]]出版。之後[[大然|大然]]正式代理版權漫畫則翻為《'''怪物小鬼'''》，出版共13集。

==劇情簡介==
本作是部描述從怪物島來到人間的不可思議少年、怪物王子和他的隨從[[吸血鬼|吸血鬼]]、[[狼人|狼人]]、[[科學怪人|科學怪人]]引起各種騷動的[[搞笑|搞笑]][[漫画|漫画]]。除了他們以外，也有許多不同的善良怪物登場。因為時常會有和登場的敵人戰鬥的描寫、所以動作場景也相當多。

從1965年1月 - 1969年4月起、在少年画報社的《少年画報》上連載。1967年6月 - 1969年5月，也在《週刊少年KING》上連載。之後1980年 - 1982年期間，重製版在[[小学館|小学館]]的《[[快樂快樂月刊|快樂快樂月刊]]》、《学習雜誌》、《てれびくん》等雜誌連載。

根據作者所述「雖然將三個隨從加以變形化很簡單，但因為作為主角的怪物太郎的臉直到截稿前都還無法決定」，所以在《少年画報》上的新連載預告就變成「之後姿態的怪物太郎和他三位會讓人嚇得發抖的隨從」。雖然第1回的名字、草稿和主角的臉都由作者本人所畫、但大部分的作畫都由他的助手[[筱田英男|筱田英男]]負責。

== 登場角色 ==
角色介紹包含黑白動畫版和彩色動畫版。

=== 怪物屋的居民 ===
; 怪物君（黑白版：[[白石冬美|白石冬美]]、彩色版：[[野澤雅子|野澤雅子]]；[[林美秀_(配音員)|林美秀]]（台灣）；[[方煥蘭|方煥蘭]]（香港））
: 本名：怪物太郎。是一個住在不知位於何處，並只有怪物們居住，叫作「怪物島」的國家的王子。是現任国王怪物大王的兒子、有三名隨從，並都叫他「小少爺」。外表看來是個普通嬌小的少年、但卻擁有能夠伸縮自如的手腳、可以變身成各種臉孔和身材的許多超能力。最大的武器是當喊出「1、2、3!」然後喊到三時，會使用念力引起爆炸、在電影《[[怪物小王子_前往怪物島的招待|前往怪物島的招待]]》中使用這招時，還有著能讓怪物城半毀的威力。臉上長著三根[[鬍鬚|鬍鬚]]（[[火影忍者|火影忍者]]里的[[漩涡鸣人|漩涡鸣人]]也有类似的设定）。招牌特色的[[棒球帽|棒球帽]]就算睡覺時也不會拿下來、並直到最終話才揭曉帽內的秘密（因为怪物太郎头上有一对跟怪物大王一样的触角）。為了王位繼承的修行而來到人間，並且半強迫式地要求偶然遇到的小宏當他的朋友、還借用小宏姐弟所住的地方的隔壁房間專居。裡面雖然只有一台[[冰箱|冰箱]]、但其中卻有可以連接到怪物屋廚房的通路。有著很高的自尊心和性急任性的性格、對於勸告他的人會大叫「吵死了!」。但一方面是個很重感情的人、和小宏有著深深的友情、當怪物大王把小宏變成石頭時還會反抗父親，對於要傷害小宏的人，也會表現出他憤怒的一面、有時也會看小宏的面子而行動。很怕雷聲，只要打雷就會縮成一團什麼也做不了。

; [[吸血鬼|吸血鬼]](台灣)／杜拉基拉(香港)／德拉克拉(大陸)（黑白版：[[大竹宏|大竹宏]]、彩色版：[[肝付兼太|肝付兼太]]；[[何志威|何志威]]（台灣）；[[張炳強|張炳強]]（香港））
: 怪物太郎的隨從之一。出身於血統純正的伯爵家，主要負責怪物太郎的教育，不太會說出太激烈的話。平常戴著黑色的[[大禮帽|大禮帽]]和[[斗蓬|斗蓬]]以及[[燕尾服|燕尾服]]。有潔癖，自尊心也很高，相當博學，特別是對於怪物和妖怪的事情相當了解。雖然是[[吸血鬼|吸血鬼]]但卻不吸血、取而代之的是愛喝[[蕃茄汁|蕃茄汁]](其实想喝的，只是怪物太郎说不可以)。有著可以應用吸血能力，把進入体内的毒素吸出來等其他吸血鬼所學不來的高級技能。可以利用愛用的披風來飛行，其他人穿上的話亦能夠飛行，但普通人長時間使用的話會變成蝙蝠。可以操控吸血蝙蝠，主要是讓牠們負責偵察工作。害怕[[大蒜|大蒜]]，[[十字架|十字架]]和陽光，但在怪物島中，即使是白天也能行動（因为怪物岛一直是阴天，没有阳光）。在人間中，白天時會睡在地下室的[[棺材|棺材]]裡，夜晚則是出來散步並且反覆進行這種生活。在白天活動時，需要科學怪人把棺材扛出來，所以讓他對於夥伴變成一種障礙。作為父親的他雖然喜歡古老的東西，但自己的兒子卻反而喜歡新的事物，而且還會穿著紅色的燕尾服加上短褲。

; [[狼人|狼人]]（黑白版：[[兼本新吾|兼本新吾]]、彩色版：[[神山卓三|神山卓三]]；[[夏治世|夏治世]]（台灣）；[[石景初|石景初]]（香港））
: 怪物太郎的隨從之一。出身於怪物島的坎斯。平常看來是個留著光頭且身材肥胖的[[中年|中年]]男子、眼睛有一点眯、脸上也有[[胡须|胡须]]、但在看到[[滿月|滿月]]或[[圓|圓]]的東西時會變成狼人，也會有單純看到圓的東西就變身的時候﹝例如第十集中，因為老板娘送错成月间乌龙麵（就是在乌龙麵上打一颗蛋，像满月一样），讓他一看到就现出原形﹞。有著狼特有的敏銳嗅覺和飛快的動作並且以他的牙和爪子當武器。本人曰「平常人樣是他的真面目」、作中有時也會有「在怪物中較接近人類」的發言、但在怪物島中經常以狼人的樣子出現。只要不變成狼人就無法發揮力量、基本上因為平常時的能力就像是一般人一樣，所以這也成為他唯一的弱點。在怪物島中是著名的廚師、所以在怪物屋中擔任煮菜和做家事的工作。是個性格坦率，好親近，愛照顧別人的人。喜歡的東西是[[年輪蛋糕|年輪蛋糕]]。

; [[科學怪人|科學怪人]]（台灣）／弗蘭肯（大陸）（黑白版：[[今西正男|今西正男]]、彩色版：[[相模太郎|相模太郎]]→[[兼本新吾|兼本新吾]]；[[林谷珍|林谷珍]]（台灣）；[[金雷|金雷]]（香港））
: 怪物太郎的隨從之一。由[[弗蘭肯斯坦博士|弗蘭肯斯坦博士]]所製造的人造人。體格巨大結實、但是個一派輕鬆心地善良又愛哭的人。頭腦不好、擅長利用他的怪力和手指、主要擔任修理房屋和搬運買的東西的角色。也兼任怪物太郎的保鑣、其戰鬥力比起只用念力的怪物太郎更強、雖然如此但卻沒有更加活躍。不過當有能夠操控怪物太郎害怕的雷聲的怪物出現時、因為雷＝[[電力|電力]]於是變成他的能源而讓他可以勇敢面對完成護衛的工作。之後會出現個叫作庫拉肯弟的第，而他也是由弗蘭肯斯坦博士（彩色動畫變成貝姆）所作、但和哥哥不一樣的是行格冷酷無情。名字在德語中是「空肚子」意思的「科學怪人」平常相當沉默、但那些怪物同伴不知為何可以和他溝通。因為惡魔組織戴蒙尼斯的刺客阿修蘭的魔力、讓他在一段時間可以正常地講話。科學怪人也在《[[多啦A夢|多啦A夢]]》、《[[吸血鬼小姐|吸血鬼小姐]]》中登場。
: 吸血鬼、狼人、科學怪人在怪物島都有兒子、偶而會來到人間並且引起騒動。吸血鬼的兒子不怕太陽、科學怪人的兒子能夠正常地講話、狼人的儿子能自由控制狼与人类的样子、都已經克服了父親的弱點，可谓青出於蓝。

=== 市川家 ===
; 市川歌子（黑白版：[[向井真理子|向井真理子]]、彩色版：[[川島千代子|川島千代子]]；[[鄭仁君|鄭仁君]]（台灣）；[[周婉侶|周婉侶]]（香港））
: 小宏的姐姐。18歳。因為雙親早逝、於是在公司工作來養育弟弟。比小宏更常遇到怪物，所以很容易被嚇昏。跟《[[Q太郎|Q太郎]]》里的正太的妈妈、'''大原节子'''没关系。

; 市川宏（黑白版：[[松島美野里|松島美野里]]、彩色版：[[三輪勝惠|三輪勝惠]]；[[雷碧文|雷碧文]]（台灣）；[[區瑞華|區瑞華]]（香港））
: 與姐姐一起住在怪物屋附近的公寓“新真莊”（大陆叫“荒间庄”）的小学生。是個膽小，頭腦不太好、但個性卻相當善良的少年。因為意想不到的機會而和這群怪物們成為好友並且因此被捲入各種事件中。一開始看到怪物時被嚇到暈倒、但在彼此認識後而改善這件事情。跟《[[Q太郎|Q太郎]]》里的'''正太'''没关系。

===小宏的同學們===

; 紀夫（彩色版：[[はせさん治|はせさん治]]；[[何志威|何志威]]（台灣）；[[梁緻盈|梁緻盈]]（香港））
:經常和番野在一起，是他的小跟班。性格愛現且驕縱，似乎家裡相當有錢。
; 亞子（彩色版：[[潘惠子|潘惠子]]；[[林美秀_(配音員)|林美秀]]（台灣）；[[樂蔚|樂蔚]]（香港））
:小宏同學，是個和善親切的女孩子。
; 老師：（彩色版：[[津村隆|津村隆]]；[[曹冀魯|曹冀魯]]（台灣））
:是小宏他們所上學校的老師。

=== [[怪物|怪物]]・[[惡魔|惡魔]]們 ===
; 怪子（黑白、彩色版[[増山江威子|増山江威子]]；[[馮嘉德|馮嘉德]]（台灣））
: 本名：怪子公主。怪物島一級貴族古利奇的獨生女。怪物太郎的女友和皇妃候選人。雖然是位留著[[金髮|金髮]]短髮的美少女、但生氣時會操作頭髮自由地揮舞破壞著、即使是沉重的東西也可以自在地抓起來。和怪物太郎一樣任性、性急且愛吃醋。对小宏抱有一些的好感。
:廚藝非常差勁但卻喜歡叫人吃她做的菜，所以讓怪物太郎及三人組聽到是她負責煮菜時，都急著想跑走。

; 格林婆婆（彩色版：[[堀絢子|堀絢子]]；[[鄭仁君|鄭仁君]]（台灣））
: 在克利奇家中工作的女傭。能夠變身成[[烏鴉|烏鴉]]在天上飛。對於怪子的任性總是負責在後面收尾。

; 諾歐醫生（彩色版：[[野本禮三|野本禮三]]；[[曹冀魯|曹冀魯]]（台灣））
: 負責治療從怪物島到人間的怪物的怪物専門醫生。是身材短小且臉長的很奇怪的人、但外表不同的，其實本人是個心地善良的。有從頭部長出第三隻手的描寫。擅長[[整容|整容]]、所以也有對因為事故而毀容的人進行[[手術|手術]]的事情。

; 怪物大王（彩色版：[[金井大|金井大]]；[[林谷珍|林谷珍]]（台灣））
: 太郎的父親、怪物島的最高權力者。有著像是怪物的巨大体格和恐怖的臉孔、結束了怪物和平戰争並且之後守住了自由和和平相當長的時間、因此受到国民深厚的信賴。和兒子一樣性格頑固且性急、也會有容易感到寂寞和為孩子擔心的地方、常用裝病來叫兒子回來。有著能夠從眼睛放出閃光來將物體融化、把生物石化及瞬間移動等能力。石化攻擊則需要用大王的眼淚才能回復。雖然和兒子長的不太一樣、最終回中，能知道其他們身體有一部分一樣。年輕時、以留學生的身分到日本山中的『妖怪島』上留學、當時的同學妖怪大王也和他是好友。笑声是「喀波哈哈哈」。

; 惡魔王子（彩色版：[[山田榮子|山田榮子]]）
: [[惡魔|惡魔]]島王子。本来怪物族和惡魔族是敵人、由於父親惡魔王是對怪物大王施加詛咒的惡魔組織戴蒙尼斯的首領，所以使他對父親加以反抗並且擔任潛入到惡魔島的太郎的嚮導而讓他成功解開咒語、也讓仇恨怪物大王的父親因此改邪歸正。因為這個緣分使他常來到人間遊玩、但由於比太郎更加不知道人間的事情所以常引起騷動。在[[學習研究社|學習研究社]]發行的《トップラーン》中、有著以惡魔王子為主角的漫畫，[[惡魔王子|惡魔王子]]在裡面連載著。

; [[埴輪|埴輪]]君（[[鄭仁君|鄭仁君]]（台灣））
: 小宏所上學校的轉學生。是個長得像是[[埴輪|埴輪]]的少年、真面目是日本怪神的化身。有著性格認真且正義感強、難以通融的一面。生氣時會變成像[[特攝電影|特攝電影]]『[[大魔神|大魔神]]』中登場的阿羅羯磨一樣的巨大怪神。當初和太郎是對立關係、但在單挑輸給太郎後，而和他和解。

; [[隱形人|隱形人]]（[[曹冀魯|曹冀魯]]（台灣））
:怪物島上的探員。負責取締不法鬧事的怪物。平常都裹著繃帶並且穿著一件紅襯衫及帽子行動，等到要潛入時，會全身脫掉來隱匿行蹤。

; 貝姆（[[林谷珍|林谷珍]]（台灣））
: 太郎最大的[[對手|對手]]。為了征服[[地球|地球]]、從貝拉波星搭乘[[幽浮|幽浮]]來地球的宇宙怪物。雖然多次輸給太郎，但總是像沒學到教訓般地一直出現。

; 諾啦利斯（諾比拉斯）[[何志威|何志威]]（台灣）
: 是個愛吃且頭腦不好的善良恐龍型怪物。當初小宏想帶牠給同學看來讓大家大吃一驚、但因為害怕會引起巨大的騷動而放棄。由於自己想要出名、於是跑去演出怪獸電影後而大受好評。從“快樂快樂”版以後以及新作動畫中都改名叫作「諾比拉斯」。

== 兩個最終回 ==
本作分別在《少年畫報》《少年King》上連載、雖然最終回中回到怪物島的概要是相同的、但彼此結局的內容還是有少許的不同。另外少年画報版最終回以黑白頁開始、但在故事的最後兩頁卻破例使用彩色頁。

; 少年King版
: 由於攝影師怪物「攝影機魔」仔細地拍攝在人間的太郎，讓小宏開始感到太郎要回到怪物島。拿到相簿的小宏在觀看時、為太郎一直不拿下帽子的事情感到奇怪、於是想要知道帽子底下的真面目。然後有天他終於拿下了太郎的帽子、但下面卻是光禿禿一片。不過那只是假髮、太郎頭上其實長著兩根和怪物大王一樣大的觸角。然後就這樣回到了怪物島。

; 少年画報版
: 太郎沒有和小宏告別、只在怪物屋上留下紙條就這樣回到怪物島。然後從怪物大王那邊繼承王位、而且還娶了怪子為王妃。太郎受到父親的命令、來到禁忌之地「幻之花園」要摘取婚禮派對上用的花朵。在那裡則是有大王的王妃，太郎的母親的身影出現，而這也是怪物島「在王子出生後、皇后必須馬上與他隔離並且直到他繼承王位前都要分開著」的規定所致。

彩色動畫版的最終集、則是把上開兩者的結局合而為一而成的內容。

==動畫版 ==
=== 黑白版 ===
由[[TBS電視台|TBS電視台]]在1968年4月21日 - 1969年3月23日播放。一話15分，一集由2話構成，全48集96話。OP動畫則切成像是漫画的格子一樣播放。裡面由電影評論家淀川長治對登場的怪物負責解說，並擔任結局的旁白。

根據當時編寫劇本山崎敬之的書中所述、山崎對於自己寫的最終回劇本、因為原作者[[藤子不二雄A|藤子不二雄]]一直不點頭而無法出來、結果直到最後時間的早上、在TBS電視台的大廳再度和他商量，才獲得他的首肯。山崎說「不過往後想的話，這麼難纏的作者大概也沒有了吧」。彩色動畫版則是把這個事情拿來惡搞一番。

== 搞笑版 ==
*彩色版主題曲中所出現的台詞，被用在2007年上映的《[[幸运☆星|幸运☆星]]》片頭曲之中，而且直到第14話時都還在使用。
*彩色版主題曲中所出現的台詞，也被用在2009年上映的《[[碧陽學園學生會議事錄|碧陽學園學生會議事錄]]》第1話。但《[[幸运☆星|幸运☆星]]》似乎有發現到這件事情、於是也在動畫裡說出「請認真開始吧」「這樣重複玩的話，不會很糟糕嗎」等台詞。
*《[[怪物王女|怪物王女]]》登場人物設定來自《怪物王子》。
*《[[鍊金三級魔法少女|鍊金三級魔法少女]]》登場人物設定來自《怪物王子》。
*在[[NHK|NHK]]《大家的歌》節目中播放的《吸血鬼之歌》（作詞：小黒恵子、作曲：クニ河内、初次播放：1975年）、其電視版所用的片頭動畫、有像是狼人、吸血鬼、科學怪人等角色登場。
*[[櫻玉吉|櫻玉吉]]所作的漫畫《[[幸福的樣子|幸福的樣子]]》中把怪物王子變成可怕的乾物小王子(取其諧音)來加以搞笑。
