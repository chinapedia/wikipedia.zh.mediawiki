'''怪談'''是[[香港|香港]][[香港有線電視|有线电视]][[有線電視娛樂台|娛樂台]]的長壽直播[[靈異|靈異]][[節目|節目]]。于1996年首播，曾任節目主持有[[雷宇揚|雷宇揚]]、[[施明|施明]]、[[陳雲海|雲海]]、[[關寶慧|關寶慧]]、[[張學潤|張學潤]]、[[歐綺霞|歐綺霞]]、[[梁嘉琪|梁嘉琪]]及[[梁思浩|梁思浩]]，[[余思敏|余思敏]]和[[林熹瞳|陳曉華]]。現任主持為[[梁思浩|梁思浩]]；節目原本逢星期日深夜十二時半播放（[[有線第一台|有線A台]]翌日同時段重播），後改為逢[[星期六|星期六]]晚上十一時正播出（[[香港有線電視第1台|有線電視第1台]]翌日同時段重播），放映時間為一小時。

2011年8月13日開始，有線電視娛樂台及高清娛樂台（舊稱有線電視娛樂台及hd212台、[[香港有線電視奇妙台/高清奇妙台|香港有線電視奇妙台/高清奇妙台]]）同步現場直播。

2009年，《[[#2008年|亞庇不思議手記‧奪魄墓場]]》（2008年12月20日播出）獲提名角逐[[2008年度電視欣賞指數調查最佳節目頒獎禮得獎名單|2008年度電視欣賞指數調查最佳節目頒獎禮]]評審團大獎<ref>[http://www.rthk.org.hk/special/tvai/2008/ceremony/ceremony.htm  第二十屆 TV·AI 最佳節目頒獎禮]</ref>。該節目為首個獲得該項提名的同類電視節目，惜未能奪魁。

==歷史==

===停播===

據第571期《[[快週刊|快週刊]]》報道，因《怪談》節目突然改變形式，不談鬼神，因此遭到廣告商叫停近200萬的節目贊助費。<ref>http://blog.sina.com.cn/s/blog_603b3e0b0100e6s3.html  華興行官方博客轉載《快週刊》文章︰《怪談》改革不再玩靈異 華興行暫時抽起贊助</ref>而接替徐小明出任有線娛樂執行董事的[[胡容卿|胡容卿]]，本身是虔誠基督徒，她上任後即銳意改革有線娛樂平台，據報因為她覺得電視台「不應導人迷信」，所以《怪談》這個靈異節目即使擁有收視亦首當其衝，而由麥玲玲主持的節目《[[玩轉姻玄路|玩轉姻玄路]]》 ，以及原定由蘇民峰接棒主持的陰宅風水節目也不能倖免。記者向胡容卿求證，是否停播所有玄學節目，甚至賠錢給藝人也在所不惜，她強調︰「任何節目都有完結的一日」。而蘇民峰則表示要從其他人口中才知自己的節目胎死腹中，非常沒面子。<ref>http://bbs.ent.163.com/bbs/tvb/144354005.html 《忽然1週》︰有線斬靈異風水節目　梁思浩蘇民峰過檔無綫</ref>

及至2010年2月26日，怪談官方在怪談討論區內發表公告，稱「怪談討論區將於3月5日凌晨12時關閉」顯示怪談將走向終結。作為前主持之一的司徒法正則在網誌表示節目改革與創新不可因「個人喜好」而盲目進行，「就市場學而言，這是極不合理的」<ref>http://hk.myblog.yahoo.com/masterszetofc/article?mid=30330 司徒法正官方博客︰心痛的決擇</ref>，並在一次訪問中暗示有人罔顧觀眾喜好，「有人就個人喜好說腰斬就腰斬，對有線來說是很大的損失」。<ref>http://www.hkheadline.com/ent/ent_content.asp?contid=104491&srctype=n 《頭條日報》︰批有線高層摺埋《怪談》 司徒法正:罔顧觀眾喜好！</ref>

=== 第四代至第六代 ===

《怪谈》於2006年來了一次大革新，由第四代主持接棒；而第四代至第六代的內容基本上一脈相承，俱以「不思議手記」為骨幹，男主持[[梁思浩|梁思浩]]不变，只更換了女主持，分别是[[歐綺霞|歐綺霞]]（Monique）<ref>leungszeho.net：[http://leungszeho.net/Spencers_news-2006.htm 梁思浩 歐綺霞主持《怪談》，文匯報，2006年1月6日] {{webarchive|url=https://web.archive.org/web/20090828035002/http://leungszeho.net/Spencers_news-2006.htm |date=2009年8月28日 }}</ref>（06年）、[[梁嘉琪|梁嘉琪]]（06年-07年）、[[余思敏|余思敏]]（07年加入）和[[陳曉華_(藝人)|陳曉華]]（07年下旬加入）。每期节目还会邀请一名玄学嘉宾来为观众分析，讲解该集内容，或者为观众来电解答疑问。常任怪谈嘉宾有：[[司徒法正|司徒法正]]師傅、[[林千博|林千博]]師傅、[[簡信回|簡信回]]師傅和[[馬君程|馬君程]]老師，因《怪谈》09年11月的改革，所有师傅均与[[有线电视|有线电视]]解约，在播放最后一辑《不思議手記》期间没有邀请嘉宾参与节目，改由梁思浩电话访问「圈中好友」来充实节目。

==== 不思議手記 ====

此系列是怪谈的主干环节，由2006年1月7日起播出，至2009年12月26日结束，共有22辑。'''不思議手記'''由一隊有線電視拍攝小組冒險到亚洲各国不同的角落探究靈異事件，曾經涉足[[泰國|泰國]]、[[台灣|台灣]]、[[東南亞|東南亞]]、[[日本|日本]]等地。有观众认为这部分内容的拍摄手法误导观众，以似是而非的画面诱导人们相信有鬼神的存在。受09年11月的改革影响，与《怪谈》合作十年的司徒法正師傅与有线电视解约，没有参与最后一辑《不思議手記》的拍摄，该辑的内容亦改变风格，不涉及「靈異探秘」，使得收视下滑。

2007年，怪谈舉辦「[[#不思議自由行|不思議自由行]]」，公開招募對不可思議事件有濃厚興趣的觀眾成為「怪談攝製隊」成員，隨隊出發到海外進行拍攝工作，反應相當熱烈，兩位女參賽者最後脫穎而出<ref name="ghoststory.1">[http://www.i-cable.com/unbelievable/ghoststory.html 奇趣怪異事件簿：怪談攝製隊成員 - 測試]</ref>。

===== 不思議自由行 =====

此活動於[[2007年1月|2007年1月]]首辦，公開招募對不可思議事件有濃厚興趣的觀眾成為「怪談攝製隊」成員，入圍者可隨隊出發到海外進行拍攝工作，參與者要經過嚴格篩選才可獲得入圍資格。

2008年，本活動亦有作出宣傳招募，但後來不了了之，節目中亦無正式公開宣佈腰斬的決定及解釋箇中原因，以致外間對此一無所知。

==== 笑談 ====

鑑於一般人在[[春節|農曆新年]]期間較為避忌[[鬼怪|鬼怪]][[邪靈|邪靈]]之說，《怪談》在农历新年前後至[[元宵節_(華人)|元宵節]]期間易名為《笑談》。节目由三位主持及師傅(包括：司徒法正師傅、[[林千博|林千博]]師傅等)講解新一年的[[風水|風水]]佈局，內容也改以[[生肖|生肖]][[運程|運程]]、[[招財|招財]]、[[開運|開運]]、趨吉避凶、迎[[桃花運|桃花]]等話題為主。

==== 警界靈訊 ====

此環節由余思敏及簡信回師傅主持，自2008年4月5日起至6月28日止共十三集，每集約五分鐘，於節目的第四節播出，以檔案形式講述[[香港警察|警隊]]內部的靈異傳聞及剖析其可信性，當中涉及的案件，如[[嘉利大廈大火|嘉利大廈大火]]、[[Hello_Kitty藏屍案|Hello Kitty 藏屍案]]、[[徐步高槍擊案|徐步高槍擊案]]、[[跑馬地馬場大火|跑馬地馬場大火]]等，皆轟動一時。

==== 屍前‧死後 ====

該盂蘭節特備節目由2008年7月26日起至9月27日止一連十集，主持梁思浩以第一身來演繹一個人死後「七七四十九日」的真實經歷，打破禁忌'''「為自己舉行一個完美的[[喪禮|喪禮]]」'''，堪稱「[[屍體|屍體]][[真人秀|真人秀]]」。據[http://www.i-cable.com/unbelievable/home.html 官網]介紹，內容還會提及殯葬儀式的[[習俗|習俗]]和[[忌諱|忌諱]]、[[化妝師|化粧師]]如何為先人遺容上粧、[[破地獄|破地獄]]儀式是否可以破開[[地獄#中國信仰的地獄|地獄]]之門、[[葬礼#埋葬|土葬]][[墳墓|墳穴]]究竟要掘多深……等話題。

==== 奇人詭異檔案 ====

《奇人诡异档案》由2009年4月11日起至5月23日结束，共有七集，每集片段長約十多分鐘，內容以搜羅世界各地奇人異士的超人力量、特異功能、非常怪行為主，令觀眾不但眼界大開，更可以增廣見聞。

==== “玄”来你要知 ====
《“玄”来你要知》是教授玄学知识的小环节。由2009年8月22日起播出，每集长度约1分钟，通过陈晓华与司徒法正师傅之间的问答对话，让观众了解更多玄学方面的知识。

==== 恐怖密室 ====

==== 2011年死亡預告 ====
《死亡預告》是怪谈2011年8月重播時的全新環節系列节目，由梁思浩、譚凱琪共同演出，每集梁思浩都會以不同的死亡方法作為戲劇內容。

== 怪談 FAN CLUB ==
怪談是本地少有設立[[影迷|影迷會]]（[[非牟利|非牟利]]）的電視節目，旨在凝聚一羣喜歡解構不可思議事件的[[朋友|朋友]]，在活動中互相交流<ref>[http://www.i-cable.com/unbelievable/fanclub.html 怪談 FAN CLUB 網頁]</ref>。

== 網上侵權問題 ==
自《不思議手記》系列推出以來，怪談的知名度增加了不少，有[[網民|網民]]把每集都錄製成[[視頻文件格式|視頻檔案]]上載到[[互聯網|互聯網]]公諸同好（例如 [[YouTube|YouTube]]、[[土豆網|土豆網]]、[[优酷|优酷]]等），或利用[[點對點技術|點對點技術]]（P2P，如 [[BitTorrent|BitTorrent]]、[[Foxy|Foxy]] 軟件）任人下載，造成[[版權|網上侵權]]問題。

於2008年尾至2009年初，有線電視「徇眾要求」，開始在《怪談》官方網頁免費提供最近數集網上重溫及預告片段，並向 YouTube 作出正式投訴，又在第１台[[#重播|重播]]了三個系列的《不思議手記》，以回應這種非法的上傳、下載行為。
