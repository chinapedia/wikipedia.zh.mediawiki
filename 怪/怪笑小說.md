《'''怪笑小說'''》是日本作家[[東野圭吾|東野圭吾]]的[[黑色幽默|黑色幽默]][[小說|小說]]集，也是「'''〇笑小說系列'''」的第一本小說。本書之短篇原本在[[新潮社|新潮社]]發行的月刊小說雜誌{{link-ja|小說新潮|小説新潮}}刊載，自1993年~1995年。1995年10月由[[集英社|集英社]]發行[[單行本|單行本]]，1998年8月集英社發行了[[文庫版|文庫版]]。

本書作者對社會細膩觀察後，以較為搞笑的文筆，檢視人生百態。

* 原題：{{lang|ja|おっかけバアさん}}
: 獨居的老婆婆勝田茂子自從用鄰居送的票去看了藝人杉平健太郎的公演，自此就迷上了杉平。平素節儉的她將所有的錢都用在了門票及美容服裝方面，逐漸變成了一個病態的[[追星族|追星族]]……

=== 一徹老爹 ===
* 原題：{{lang|ja|一徹おやじ}}<ref>「勇馬」（{{lang|ja|Yuuma}}）音似「飛雄馬」（{{lang|ja|Hyuuma}}），有諧音之趣。「飛雄馬」的父親名字為「一徹」（小說裡沒有提及父親姓名），「一徹」在日文裡是“固執”的意思。「飛雄馬」與「一徹」的典故皆出自於《[[巨人之星|巨人之星]]》。</ref>
: 為了圓自己的[[棒球|棒球]]選手夢的父親從兒子勇馬小時候開始就對他進行特訓，希望他能夠成為職業棒球選手。到了勇馬長大後，棒球選秀的時期終於到來，但是……

: 在主人公肇的眼中，家人們都是不同的動物。不僅如此，連班上的同學和老師也是各種各樣的動物。

: '''登場人物'''
:*; {{nihongo|肇|{{ruby|肇|はじめ}}|Hajime}}
:: 中學生。在他眼中周圍的人都被看成是動物。
:*; 狸貓
:: 肇的父親。有外遇。
:*; 狐狸犬
:: 肇的母親。
:*; [[鬣狗|鬣狗]]
:: 肇的哥哥。大學生。
:*; 貓
:: 肇的姐姐。高中生。
:*; 白狐
:: 肇的祖母。
:*; {{nihongo|橋本|ハシモト|Hashimoto}}
:: 肇從小學開始的好友。
:*; [[大鯢|大鯢]]
:: 肇的同學。不良學生的老大。
:*; 白蛇
:: 狸貓（肇的父親）的情人。
