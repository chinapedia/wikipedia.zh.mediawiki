《'''怪物猎人4'''》（{{lang-ja|モンスターハンター 4}}，{{lang-en|Monster Hunter 4}}）是一款由[[卡普空|卡普空]]开发并发行在[[任天堂3DS|任天堂3DS]]平台上的[[动作角色扮演游戏|动作角色扮演游戏]]。本作是《[[怪物猎人系列|怪物猎人]]》系列的第四款正统续作，也是该系列首次在掌机平台推出正统续作。《怪物猎人4》最早公布于2011年的任天堂秋季发布会，并作为最后的压轴游戏作品登场。

== 游戏系統 ==
===主要特点===
《怪物猎人4》與前作的不同之處在於其對3D圖像的高度強調。玩家在此遊戲爬牆時的動畫相對前作更暢順，並且在不同角度觀看爬牆也不會影響動畫的流暢度。此遊戲亦是系列中首個允許玩家爬上怪物背部和在半空中攻擊怪物的遊戲。<ref>{{cite web |url=http://www.youtube.com/watch?v=agJyyPk9tjY |title=Monster Hunter 4 Promotion Movie |date= 2012-09-21|accessdate= 2012-10-06}}(Capcom Summer Jam Trailer)</ref>相對前作而言，此遊戲的狩獵場地亦大於前作。怪物在此遊戲中懂得使用環境的優勢來攻擊玩家，令到怪物攻擊模式更多樣化。<ref name="Official new element">[http://www1.capcom.co.jp/monsterhunter/4/action_01.html CAPCOM：モンスターハンター4 公式サイト - 注目の新元素]</ref>儘管此遊戲新增了不少新元素，但在前作《[[怪物猎人3|怪物猎人3]]》中出現的水底狩獵卻在此遊戲中被取消。遊戲中的[[非玩家角色|非玩家角色]]數量亦是系列中最多的。此遊戲亦支援線上多人遊戲。<ref>{{cite web |url= http://www.siliconera.com/2012/09/19/monster-hunter-4-has-online-multiplayer-out-in-march-2013/ |title= Monster Hunter 4 has Online Multiplayer}}</ref>《怪物猎人4》開發工程的監督藤岡要指出此遊戲的重點將會放在冒險和故事進展上，而非像前作般將重點放在狩獵上。<ref>{{cite web |url=http://www.nintendolife.com/news/2012/07/monster_hunter_4_director_says_its_all_about_adventure |title=Monster Hunter 4 Director Says It's All About 'Adventure'}}</ref>此遊戲亦新增了2種武器：操蟲棍和蓄力斧<ref name="Official new element"/>，以及一種怪物：黑蝕龍。<ref>{{cite web |url=http://www.uniqlo.com/hk/product/men/ut/monsterhunter |title= English name was confirmed on Hong Kong MHxUniqlo collaboration site}}</ref>另外，此遊戲亦加入了一些曾在任天堂著名遊戲中出現的裝備，如[[瑪利歐系列|瑪利歐系列]]和[[塞爾達傳說系列|塞爾達傳說系列]]。<ref>{{cite web|url=http://www.ggsgamer.com/2013/09/09/suit-up-as-link-in-monster-hunter-4/|title=Suit-up as Link in Monster Hunter 4|date=2013-09-09|author=Olly Jones|publisher=ggsgamer.com|accessdate=2013-09-11|deadurl=yes|archiveurl=https://web.archive.org/web/20131031114126/http://www.ggsgamer.com/2013/09/09/suit-up-as-link-in-monster-hunter-4/|archivedate=2013年10月31日|df=}}</ref>游戏更加入其它合作方的漫画作品内容，如[[海贼王|海贼王]]的乔巴的猫服装等。<ref name="tgbus">{{cite web | url=http://3ds.tgbus.com/news/201310/20131030091848.shtml | title=《怪物猎人4》与《海贼王》合作配信乔巴装 | publisher=tgbus | accessdate=2014-01-02}}</ref>

===探索===
探索任務系統是本作的新增要素。探索目标地点是「未知樹海」，分为下位跟上位（村任务通关后才可进入）。通过对小地图的出入口封闭与重新连接，探索任务生成随机地图。任務生成的裝備將會在隨機地點出現，而那些裝備的屬性能力每次都不一樣。<ref>{{cite web |url= http://blog-imgs-60.fc2.com/m/h/3/mh3gf/2013082722312078d.jpg |title= Old armor sets from Capcom's Monster Hunter 4 press conference}}</ref>探索任務亦會包含前作出現過的怪物材料制作而成裝備。<ref>{{cite web |url= http://www.4gamer.net/games/140/G014036/20130827004/SS/001.jpg |title= Qurupeco Slash Axe featuring on Official Weapon Guide Book}}</ref>

从野外探索中获取可以获取探索任务，并可以记录在公会任务之中。玩家可以通过网络與其他玩家交換探索任務。玩家最多能儲存50個探索任務。<ref>{{cite web |url=http://blog-imgs-60.fc2.com/m/h/3/mh3gf/20130827194510d4b.jpg |title= Guild Quests page from Capcom's Monster Hunter 4 press conference}}</ref>

== 游戏音乐 ==

《怪物猎人4》由铃木麻里香、茅根美和子 和 裏谷玲央负责本作的编曲工作，[[东京爱乐乐团|东京爱乐乐团]]担当演奏。

=== 原声大碟 ===

== 發佈 ==
於2011年任天堂遊戲發佈會上，卡普空正式宣佈他們正在開發《怪物猎人4》，並指出其平台為[[任天堂3DS|任天堂3DS]]。<ref>{{cite web|url=http://www.eurogamer.net/articles/2011-09-13-monster-hunter-4-announced|title=Monster Hunter 4 announced|publisher=[[Eurogamer|Eurogamer]]|author=Martin Robinson|date=2011-09-13|accessdate=2013-09-11}}</ref>

於2012年12月19日，卡普空在其官方網站上貼出了一份公告。該公告指出此遊戲在日本發佈的日期將會延遲到2013年夏季，並指出「我們需要多一點時間……來提升遊戲的質量」。<ref>{{cite web |url=http://www.capcom.co.jp/ir/english/news/html/e121219a.html |title= Capcom Announces Revision in the Launch of "Monster Hunter 4"}}</ref>

于2013年2月6日，任天堂香港及任天堂溥天宣布未来将在繁体中文区的[[任天堂3DS|任天堂3DS]]上发售《怪物猎人4》的日文版。<ref>[http://hk.games.yahoo.net/news-%E9%A6%99%E6%B8%AF%E8%A1%8C%E7%89%88Monster%20Hunter%204-%E7%9C%9F-%E5%A5%B3%E7%A5%9E%E8%BD%89%E7%94%9FIV-%E5%B9%B4%E5%85%A7%E6%8E%A8%E5%87%BA-591189288.html 香港行版《Monster Hunter 4》《真・女神轉生Ⅳ》年內推出].2013年2月6日.雅虎香港.[2013-09-19].</ref>

於2013年5月30日，卡普空宣佈此遊戲的兩款特別限定版任天堂3DS LL主机將會與原版在日本同時發佈。這兩個特別版的名稱分別是「黑色黑蝕龍版本」和「白色艾路貓版本」<ref>{{cite web |url=http://www.capcom.co.jp/monsterhunter/4/topic_hard.html |title= Capcom announces 2 special edition 3DS LL for Japan}}</ref>

於2013年6月3日，卡普空宣佈此遊戲的限定版將會與原版在日本同時發佈，但只限於卡普空電子商店上發售。完整版除了包含遊戲本身，亦附加了一條黑蚀龙风格的吊帶、一个手办模型和一個特制的装入3DS的袋。<ref>{{cite web |url=http://www.e-capcom.com/mh4/ |title= Capcom announces different editions for Monster Hunter 4 }}</ref><ref name="duowan">{{cite web | url=http://3ds.duowan.com/1306/234270693183.html | title=《怪物猎人4》限定版公开 超帅雄火套装手办图赏 | publisher=多玩 | accessdate=2014-01-02}}</ref>

于2013年8月8日，卡普空亚洲宣布本作的繁体中文区版本将与日本同时于9月14日发售。并公布了中文说明书与中日跨区联机等细节及售价。<ref>[http://www.nintendo.com.hk/pressrelease/3ds_20130808_mh4.htm 《MONSTER HUNTER 4》亞洲版鐵定於9 月14 日與日本同時發售].2013-08-08.任天堂香港 & 卡普空亚洲.[2013-09-19].</ref>

于2013年12月4日，卡普空宣布本作的出货量突破400万份。<ref>{{cite web | url=http://3ds.pcgames.com.cn/333/3330883.html | title=CAPCOM宣布《怪物猎人4》出货量超过400万 | publisher=太平洋游戏网 |date=2013-12-04 | accessdate=2014-02-02}}</ref>

2014年10月，游戏强化版《魔物獵人4G》推出<ref>http://3ds.duowan.com/1502/288037346580.html</ref>。

== 游戏评价 ==
《怪物猎人4》在2012年[[东京游戏展|东京游戏展]]上获得由电脑娱乐供应商协会（CESA）颁发的『未来部门大赏』。<ref>[http://awards.cesa.or.jp/2012/prize/future/10.html 受赏作品概要《モンスターハンター4》]</ref>

《怪物猎人4》獲日本著名游戏杂志《[[Fami通|Fami通]]》給予38/40的得分，並獲頒白金殿堂獎。<ref>{{cite web|url=http://www.vg247.com/2013/09/05/monster-hunter-4-just-misses-perfect-score-in-famitsu-farming-simulator-rated/|title=Monster Hunter 4 just misses perfect score in Famitsu, Farming Simulator rated|date=2013-09-05|publisher=vg247|accessdate=2013-09-11}}</ref>雖然本作在分數上不及獲得满分的上一款正统续作《[[魔物獵人3|魔物獵人3]]》<ref name=Famitsu>{{cite web|last=Sims|first=Daniel|date=2009-08-05|title=Monster Hunter 3 Gets a Perfect Review From Famitsu|url=http://wii.kombo.com/article.php?artid=14624|publisher=Kombo|accessdate=2009-08-05}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，不过它仍然与同一平台的《[[魔物獵人3G|魔物獵人3G]]》的分数持平。<ref>[http://3ds.pcgames.com.cn/301/3011680.html 3DS《怪物猎人4》获FAMI白金殿堂评价].2013-09-04.太平洋游戏网.[2013-09-04].</ref>

中国大陆的游戏杂志《[[游戏机实用技术|游戏机实用技术]]》给本作打出了28/30的分数评价，并获得“黄金珍藏”。
