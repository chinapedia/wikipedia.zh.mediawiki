== 假面騎士 ==
'''{{Nihongo|桐生戰兔 / 葛城巧|きりゅう・せんと / かつらぎ・たくみ}}'''（[[犬飼貴丈|犬飼貴丈]]飾；香港配音：[[張振熙|張振熙]]）

: 熟知一切困難的公式，擁有優異的頭腦和身體能力，平時喜歡做實驗，每當看到讓他感到興奋的東西時，頭髮一部分會豎起来<ref>第一話中當看到潘朵拉之箱的時侯可見</ref>，而且每一感到不解或兴奋時會抓起頭髪<ref>這也是葛城巧常有的一个习惯</ref>。
: 幾乎每天都在發明新的東西和原理，其中包含創造驅動器內Best Match 的探查機能，淨化滿裝瓶的機器和武器裝備在內所有Build 相關道具均由其發明。
