《'''假面騎士龍騎'''》（{{lang-ja|仮面ライダー龍騎}}）為《[[假面騎士系列|假面騎士系列]]》的第3部[[平成|平成]]系列作品，自2002年2月3日至2003年1月19日期間，於每週日早上8:00-8:30於[[朝日電視台|朝日電視台]]播映，全50集。台灣[[華視|華視]]自2004年7月29日至2005年3月16日期間，於每週四17:30-18:00於[[華視主頻|華視主頻]]播出，全50集。香港由2004年9月25日至2005年10月22日於[[無綫電視|無綫電視]][[翡翠台|翡翠台]]首播（同時也是第一套假面騎士於無綫播映），每[[週六|週六]]12:25～12:50播出，後於2006年重播。本片另有海外版本《[[假面騎士龍騎_(美國電視劇)|Kamen Rider Dragon Knight]]》，全40集。

== 本作特色 ==
*本作的設定、世界觀與重視的異於前兩作，且騎士角色的數量和他作系列相較也意外地多。
*由於本作是關於幪面超人之間的戰鬥，所以本作是[[幪面超人系列|幪面超人系列]]史上最多幪面超人變身者死亡的作品。（簡單來説，任何變身者皆陣亡）
*從第2話開始就已表明本作假面騎士主要是敘述「騎士與騎士之間的戰鬥」，而對付怪人的劇情卻反而較鮮少，有些怪人（契約獸）甚至成爲該幪面超人的拍檔。
*本作是第1套以使用卡片的假面騎士系列作品，亦是第1套使用“幪面超人”這個名詞的[[平成|平成]][[幪面超人系列|幪面超人系列]]作品,首部主要戰鬥場景於異度空間(鏡世界)進行。
*本作騎士數量達13個，若把2位擬似超人包括在內，則為15個。此數量令本作在總共25個騎士登場的《[[假面騎士鎧武|假面騎士鎧武]]》啟播前一直保持住最多原創假面騎士登場的紀錄。
*本作是首部平成時期出現邪惡騎士與主角騎士外觀一模一樣,連變身者樣貌都一模一樣,後作《[[假面騎士Kabuto|假面騎士Kabuto]]》亦有相同設定。
*在日本地區的模型玩具方面，銷售了1390億日元的驚人紀錄。
*本作在日本地區因結合了時下流行的卡片遊戲和戰鬥而受兒童歡迎，但因劇情為「騎士與騎士之間的戰鬥」是各騎士為了自身的欲望而戰鬥、殘殺，而曾受報紙提出評論有些兒童不宜。
*由本作起，主題曲的歌詞不再含有「幪面超人」一詞。

== 劇情概要 ==
故事一開始是一位女性在家中被存在於鏡世界的蜘蛛怪獸拖入鏡中而成失蹤事件。見習記者城戶真司在調查她和榊原耕一的離奇失蹤事件時意外得到'''假面騎士龍騎'''的變身卡盒，且之後還在鏡世界中認識了同為騎士的秋山蓮，在了解假面騎士之間的鬥爭後，決定阻止此戰爭和保護人類不受怪獸侵擾，但也因此捲入其中......

== 名詞 ==
*'''鏡世界'''（Mirror World）
:現實世界相對的另一空間，環境與現實世界相同，字體卻是方向相反，任何能反射的物體皆是假面騎士到鏡世界的入口。並沒有人類，只有捕食人類的鏡怪人存在。而且人間體待在此世界過久就會消失這點，在之後的續篇《[[ridertime假面騎士龍騎|ridertime假面騎士龍騎]]》則去除了此設定。
:騎士想要離開就必須從一開始進來的反射物出去。但在《[[幪面超人帝騎|幪面超人帝騎]]》中的鏡世界可以在任何反射物中自由出入。

*'''鏡世界傳送機'''（Ride Shooter）
:假面騎士進入鏡世界的座騎，皆為一輛全罩式的[[機車|機車]]。
:長度：325cm
:高度：140cm
:寬度：112cm
:時速：930km

*'''騎士卡盒／卡牌組'''
:是神崎士郎所製造出來的一種變身卡盒。假面騎士在未締結契約獸前，卡牌組為空白形式，只有在擁有契約獸後卡盒中央會出現代表該騎士的圖樣。一旦卡牌組在戰鬥中（變身姿態）被破壞，契約獸就會失控而將主人反噬或無法回去現實世界。假面騎士都有專用卡牌組，範疇有武器、召喚、絕技、特別技能等，只有持著卡牌組的人才能感覺到鏡世界和鏡怪人的異象。當要抽卡時會感應騎士的腦波來排列卡牌順序，因此不會有抽不到想要的卡牌的情況（除非該卡片的剩餘張數是0）

*'''召喚機（讀卡機）'''
:假面騎士從卡盒抽出卡組後，會把卡組嵌入召喚機（讀卡機）裡，用以使出各種技能、或召喚出契約獸。每人的召喚機形式皆不同，如鐵兵的槍、猛虎的斧，基本的為像龍騎位於左手的龍頭召喚機。卡片放進召喚機之後到解除變身前都會放在裡面，因此除非牌組中有兩張以上的相同卡片，否則基本上無法重複使用（例如大部分騎士都只有一張契約卡，但王蛇至少有三張）

*'''ORE JOURNAL'''
:城戸真司工作的[[報社|報社]]。規模不大。
:ORE 乃"Open Resources Evolution"的簡稱。
:[[總編輯|總編輯]]為大久保大介。桃井玲子和城戸真司為[[記者|記者]]。島田奈々子為[[技術員|技術員]]。第30話起浅野恵加入成為第3位記者。
:一度被芝浦純操控。

*'''花雞餐廳'''
:神崎優衣和神崎沙奈子經營的[[紅茶|紅茶]]店和住所。目前亦是秋山蓮和城戸真司的住所。

*'''清明院大學'''
:本作一所的大學。
:神崎士郎、小川絵里曾在此就讀。
:大學的401室先後成為江島教授和香川教授的研究室。
:在江島研究室時期曾進行一次實驗，及後發生了事故。當時研究室全員除仲村創因當日並無參與實驗外全員受不同程度創傷。自此401室一直重門深鎖直至香川教授使用。
:後期香川教授與仲村創和東条悟2位研究生在此進行對鏡世界的研究，實際上打算關掉鏡世界，並創造出2個抉擇者卡盒。
:第42話香川教授決定廢掉此研究室。

*'''北岡秀一法律事務所'''
:北岡秀一律師和其助手由良吾郎工作和居住的地方。地方闊落。

== TV版登場之假面騎士 ==
本作的假面騎士為「鏡騎士」，是一種能穿越鏡世界和鏡怪獸戰鬥的騎士，十三位騎士皆為神崎士郎製造。「不戰鬥就無法生存」為本作的假面騎士之座右銘。騎士在未和契約獸簽訂契約前皆為空白的形式，力量不強，必須使用「Contract（契約）」卡來和一隻鏡怪獸締結契約，便能使用其力量。使用者若要變身成騎士時需要在有反射物的地方（如鏡子、積水處、能反射之物品等等），將卡盒對準反射物後腰帶就會自動裝備在身上，喊出「變身（Henshin）」後再把卡盒置入腰帶上即可變身。

===假面騎士龍騎 ===
變身者：城戶真司、榊原耕一（替身演員：、CV：）
:原文：Masked Rider Ryuki

由城戸真司所變身的假面騎士，主體色為紅色，騎士面罩最上方有龍的標記為象徵，召喚機掛於左手前臂。

以近身格鬥戰為主，因城戶的心態影響而實力一般，但隨著劇情發展即使是通常形態也能發揮不俗實力，後來得到生存形態後戰鬥力更是飛升。

在S.I.C. 英雄傳說中甚至還有生存王蛇的存在。
