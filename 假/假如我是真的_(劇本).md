{{Otheruses|假如我是真的|other=其他事物}}
《'''假如我是真的'''》，是[[中國大陸|中國大陸]][[作家|作家]][[沙葉新|沙葉新]]與[[演員|演員]][[李守成|李守成]]、[[姚明德|姚明德]]於1979年8月為[[話劇|話劇]]創作的[[劇本|劇本]]，一齣取材自真人真事、藏着悲劇的[[諷刺|諷刺]][[喜劇|喜劇]]，隱約有[[蘇聯|蘇聯]]作家[[果戈里|果戈里]]筆下《[[钦差大臣_(喜剧)|欽差大臣]]》的味道，惟內部試演至翌年即被[[中共|中共]]禁止公演<ref>[http://www.thebeijingnews.com/news/reform30/2008/08-02/011@124502.htm 沙葉新：討論被禁話劇開民主理性之風] {{webarchive|url=https://web.archive.org/web/20110519173435/http://www.thebeijingnews.com/news/reform30/2008/08-02/011@124502.htm |date=2011-05-19 }}</ref>。後於1981年被改編成[[假如我是真的_(電影)|同名台灣電影]]，由[[台灣|台灣]][[導演|導演]][[王童|王童]]執導，旋即又被中共列為[[禁片|禁片]]；[[香港|香港]]亦曾一度禁映，1989年始獲解禁。劇本先後被[[翻譯|翻譯]]成[[英語|英語]]、[[德語|德語]]，在國外發表演出。

== 話劇大綱 ==
連同序幕及尾聲，劇本共分八場，藉中國大陸青年李小璋為求上調冒充[[高幹子弟|高幹子弟]]張小理，帶出中共官場腐敗的眾生相。最後身份被拆穿，李小璋被遞上[[法庭|法庭]]受審，庭上點題自辯：「我錯就錯在我是個假的。假如我是真的，我真的是張老或者其他首長的兒子，那我所做的一切就將會是完全合法的。」結局有別於[[電影|電影]]，並無交待李小璋的判刑和接續下場。<ref>[http://bianning.blshe.com/post/535/289764 1979年，沙葉新《假如我是真的》（存檔）]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 創作過程 ==
劇本創作於[[文革|文革]]結束與[[粉碎四人幫|粉碎四人幫]]之後、中共[[幹部|幹部]]層面不正之風冒起之時，根據一宗真人真事的[[上海|上海]]騙案加工改編，用了約兩三個星期寫成，主要由沙葉新執筆。劇本起初名為《騙子》，經當時[[上海人民藝術劇院|上海人民藝術劇院]]院長[[黃佐臨|黃佐臨]]畫龍點睛取名《假如我是真的》。

其時上海很多編劇都在採訪收集資料，也想寫是宗真人真事騙案；但因涉及一位尚在位的上海老幹部，遭上面壓下來禁寫。沙葉新念在「人做，我不做」本不想寫，見沒人寫反而想寫起來；只是劇本與話劇之所以能夠面世，一定程度上還是托賴當時黃佐臨和上海人民藝術劇院黨組織的支持。

話劇於上海預演時，[[百姓|百姓]][[觀眾|觀眾]]反應熱烈，[[領袖|領導]]幹部沉默無語，引起極大爭論。為避開中共審查，劇團只好折衷作內部試演，不登報、不宣傳、不公開演出。然而試演還是在[[福建|福建]]、[[廣東|廣東]]、[[浙江|浙江]]、[[江蘇|江蘇]]、[[河南|河南]]、[[河北|河北]]、[[新疆|新疆]]、[[北京|北京]]等全國多地演了數十場，引起強烈迴響。

最終話劇因涉[[體制|體制]]問題，難逃禁演命運，[[中共中央宣傳部|中宣部]]禁止再「給[[中國共產黨|共產黨]]抹黑」，[[中共中央|中共中央]]召開[[全國劇本創作座談會|全國劇本創作座談會]]，沙葉新遭受嚴厲批評。在中宣部部長[[胡耀邦|胡耀邦]]保護下，沙葉新未有被秋後算帳，且獲准加入中共成為黨員，接任退休的黃佐臨出任上海人民藝術劇院院長；沙葉新寫在《假如我是真的》之前、可謂姊妹作的《[[陳毅市長|陳毅市長]]》，後來亦獲批在[[中南海|中南海]]演出。<ref>[http://ndent.oeeee.com/a/20080428/295582.html 《假如我是真的》：一齣未曾「公演」已被禁的話劇] {{webarchive|url=https://web.archive.org/web/20150220151928/http://ndent.oeeee.com/a/20080428/295582.html |date=2015-02-20 }}</ref>

== 評論 ==
2008年8月2日，北京學者[[解璽璋|解璽璋]]评论到：「《假如我是真的》的價值在於，它並不是當時流行的控訴文革的作品，而是直指社會陰暗面……[[改革開放|改革開放]]30年一路走到今天，站在當下回望歷史，我們發現《假如我是真的》所提出的質問並不過時。30年來，中國在各個方面都發生了天翻地覆的變化，體制改革與思想解放也進入深水區；但有一點仍然沒變，那就是社會對於「特權」的崇拜仍然存在，而且也早已不侷限於話劇裡所[[大批判|批判]]的上學、參軍、[[上山下鄉運動|回城]]、住房這些事。{{Better source|reason=此處原本引用360doc為來源，為不可靠來源|date=2019年4月}}」

2009年7月19日，作家[[倪匡|倪匡]]评论到：「劇本通過一個人假冒了高幹子弟，到處招搖，無往不利的過程，不但刻劃出整個統治[[極權主義|極權]]的愚蠢和腐化，而且前瞻性地指出，這個統治集團必然演變為掌握國家財產、權力的極權統治[[社會階級|階級]]，更而且無可避免地不斷腐爛，具有極深刻的社會意義。當時，統治集團如果肯定這劇，有所醒悟，或許事情發展，不致於出現如今那樣滿床錦繡下覆蓋的腐屍已爬滿[[蛆蟲|蛆蟲]]的情形。可是，當時，統治者對付的方式是禁演。<ref>[http://www.boxun.com/news/gb/z_special/2009/07/200907191802.shtml 《江青和她的丈夫們》]</ref>」
