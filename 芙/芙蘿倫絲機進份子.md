'''芙蘿倫絲機進份子''' （{{lang-en|Florence + the Machine}}，亦稱：Florence and the Machine）<ref>{{cite web |url=http://florenceandthemachine.net/ |title=Florence and the Machine |publisher=florenceandthemachine.net |accessdate=26 February 2010}}</ref> 是一個[[英國|英國]][[獨立搖滾|獨立搖滾]]樂團，2007年於[[倫敦|倫敦]]成立。其成員包括[[主唱|主唱]][[弗洛倫斯·韋爾奇|弗洛倫斯·韋爾奇]]，[[鍵盤手|鍵盤手]]{{link-en|伊莎貝拉·薩默斯|Isabella Summers}}和其他音樂家等等。

== 簡介 ==
此樂團的首張錄音室專輯{{link-en|Lungs|Lungs_(album)}}在2009年7月6日發售, 並在英國專輯排行榜(UK Albums Chart)首次保持在第2位，連續5週。

== 專輯 ==
*Lungs (2009)
*Ceremonials (2011)
*How Big, How Blue, How Beautiful (2015)
*High as Hope (2018)

== 巡迴演唱會 ==
*Lungs Tour (2008–11)
*Ceremonials Tour (2011–13)
*How Big Tour (2015-2016)
