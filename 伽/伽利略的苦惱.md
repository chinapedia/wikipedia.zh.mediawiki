《'''伽利略的苦惱'''》是[[日本|日本]]作家[[東野圭吾|東野圭吾]]的[[短篇小說|短篇]][[推理小說|推理小說]]集，也是「[[伽利略系列|伽利略系列]]」的第四本小說。在2008年6月於[[文藝春秋|文藝春秋]]和《[[聖女的救贖|聖女的救贖]]》同時發行單行本，2011年10月發行了[[文藝春秋|文春]][[文庫版|文庫版]]。之後陸續有不同語言翻譯的版本。

本作收錄許多和湯川相關的案件。電視劇原創女主角“內海薰”在《伽利略的苦惱》首章第一次由東野圭吾執筆登場。人物個性由東野圭吾重新描繪，與電視劇劇版的內海予人的觀感非全然相同。

== 故事簡介＆登場人物 ==
採用中國大陸「譯林出版社」的翻譯。<br/>
關於'''[[伽利略系列#湯川學|湯川學]]'''、'''[[伽利略系列#草薙俊平|草薙俊平]]'''、'''[[伽利略系列#內海薰|內海薰]]'''等人物請參看「[[伽利略系列#登場人物|伽利略系列－登場人物]]」。

=== 第一章·墜落 ===
* 原題：{{lang|ja|落下る（おちる）}}
: 一個女人在公寓樓上墜落死亡。死者頭部曾有被鍋毆打的痕跡，於是警察懷疑是他殺，案件交由搜查一課調查。之後死者的前輩岡崎光也主動找到警方，表示自己曾造訪死者家裡，離開死者的公寓後就目擊到死者從公寓上墜落，在現場還和[[披薩|披薩]]店的店員撞上，擁有不在場證明。但薰以死者將郵購的女性內衣放在玄關為由認為岡崎和死者是戀人關係，從而對岡崎產生懷疑。但岡崎是如何從遠處將死者拋落樓就成為了疑點。
: 這時薰請求草薙讓她前去拜訪以前曾經協助調查的湯川教授，請他再次協助。但湯川看過草薙的介紹信後並沒協助的意思，且執意拒絕薰的請求。當灰心的薰將自己對犯罪手法的假設向湯川提出時，湯川對她說「沒有毫無價值的實驗」，鼓勵她親自證實自己的假設。

=== 第二章·操縱 ===
* 原題：{{lang|ja|操縦る（あやつる）}}
: 過去曾被稱為「金屬魔術師」的前帝都大學助理教授友永幸正久違地邀請過去的學生來家中飲酒。當友永教授離開座位，學生們在談話的時候，友永家的别屋突然著火。住在別屋裡的友永教授的兒子邦宏死在了裡面，警察認為邦宏是死於火災，驗屍後發現邦宏是被刺殺後別屋才著火的。但案發當時別屋是一間密室，而且凶器也找不到，也沒有特定嫌疑人。
: 擔任搜查的草薙和薰來到現場，發現湯川也在那裡。湯川原本也是參加友永家聚會的學生之一，遲到的他從其他學生口中得知案發的經過。被偶然發生的案件捲入來的湯川，在追尋案件真相的過程中發現了意外的事實。

:*; {{nihongo|友永 邦宏|{{ruby|友永|ともなが}} {{ruby|邦宏|くにひろ}}|Tomonaga Kunihiro}}
:: 幸正和前妻的兒子，品行極度惡劣。本案的死者。
:*; {{nihongo|紺野 宗介|{{ruby|紺野|こんの}} {{ruby|宗介|そうすけ}}|Konno Sousuke}}
:: 奈美惠的男朋友。因奈美惠家裡的事而一直不能和她結婚。

=== 第三章·密室 ===
* 原題：{{lang|ja|密室る（とじる）}}
: 湯川受經營旅館的朋友藤村所託，請他前去解決與一宗案件相關的謎團，該謎團由藤村旅館的一個客人在旅館外面掉出護欄跌死而開始的。案發前在晚飯時藤村曾去叫喚死者但無人應答，但死者房間的內側掛上了門鏈，窗戶上也上了鎖。第二次去叫死者時感覺到死者在房內，不久後藤村從其他客人口中得知死者房間的窗戶打開了，去到房間後發現死者失踪，後來發現他已經跌死。藤村根據死者房間的前後狀態認為房間在當時是一間「密室」，所以才委託湯川前來調查。

:; 登場人物
:*; {{nihongo|藤村 伸一|{{ruby|藤村|ふじむら}} {{ruby|伸一|しんいち}}|Fujimura Shinichi}}
:: 湯川和草薙的朋友，旅館老闆。原本是商社人士，通過草薙得知湯川協助破案的事蹟，委託湯川前來解開旅館發生的「密室」之謎。
:*; {{nihongo|藤村 久仁子|{{ruby|藤村|ふじむら}} {{ruby|久仁子|くにこ}}|Fujimura Kuniko}}
:: 藤村的妻子。雙親因[[泥石流|泥石流]]意外而過身，之前曾在俱樂部打工，3年前和藤村交往親密繼而結婚。
:*; {{nihongo|祐介|{{ruby|祐介|ゆうすけ}}|Yuusuke}}
:: 久仁子的弟弟，在觀光協會工作。
:*; {{nihongo|原口 清武|{{ruby|原口|はらぐち}} {{ruby|清武|きよたけ}}|Haraguchi Kiyotake}}
:: 藤村的旅館的住客，從自己的房間神秘失踪後，被發現跌死。

=== 第四章·指示 ===
* 原題：{{lang|ja|指標す（しめす）}}
: 一個老婆婆被人所殺，家中收藏的金條也被偷走，而且案發當時死者所養的狗也失踪了。搜查當中發現當日有去過死者家裡的真瀨貴美子有嫌疑，但卻找不到證據，搜查進入迷宮。這時和同僚岸谷一起監視真瀨家的薰看到貴美子的女兒葉月獨自一人出門，於是尾隨她至非法棄置垃圾的地方，在那裡他們在廢棄洗衣機裡面發現了死者失踪的狗的屍體。

: 根據葉月的證詞，她是靠[[尋水術|探礦術]]找出死者的狗的所在地的。但薰覺得難以相信，於是便向湯川詢問。為確認真偽，湯川讓薰把葉月帶到了他面前。

:: 中學生。以一條可以「預知未來」的水晶鍊子指示自己的行動。
:*; {{nihongo|真瀨 貴美子|{{ruby|真瀬|ませ}} {{ruby|貴美子|きみこ}}|Mase Kimiko}}
:: 保險公司的推銷員，葉月的母親。自三年前丈夫過世後，一手將女兒葉月養大。
:*; {{nihongo|碓井 俊和|{{ruby|碓井|うすい}} {{ruby|俊和|としかず}}|Usui Toshikazu}}
:: 貴美子的上司。在工作方面給予貴美子照顧，私下和她也是戀人關係。
:*; {{nihongo|野平 加世子|{{ruby|野平|のひら}} {{ruby|加世子|かよこ}}|Nohira Kayoko}}
:: 案中被害的老婆婆，擁有丈夫留給她的金條。

=== 第五章·擾亂 ===
* 原題：{{lang|ja|攪乱す（みだす）}}
: 某天一封信寄到了警視廳，自稱為「惡魔之手」的寄信人聲稱自己可以隨心所欲葬送任何人的性命。警察一時間也搞不清信的內容是惡作劇還是真的預告信。但信中提到了T大學的Y副教授的名字，草薙由此聯想到湯川，於是給他打電話。然而湯川表示類似的信也寄到了他手中。這是犯人向湯川發出的挑戰書。犯人究竟是誰？真正目的為何？在警察毫無頭緒之際，「惡魔之手」開始出動。

: 之後「惡魔之手」以神奇的方法發出預告並開始殺人，用盡各種手段將警察玩弄於鼓掌之中，警察陷入毫無還手之力的狀態。然而憑著湯川的推理和薰的執著，他們憑藉些微的破綻將犯人的犯罪手法曝光。
