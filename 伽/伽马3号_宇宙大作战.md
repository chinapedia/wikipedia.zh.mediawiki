『'''ガンマー第3号 宇宙大作戦'''』（ガンマーだい3ごう うちゅうだいさくせん、''GAMMA3'' or ''The Green Slime'' ）是1968年的[[日本电影|日本]][[美国电影|美国]]合作电影<ref>{{Cite web |url = http://www.allcinema.net/prog/show_c.php?num_c=5096 |title = 映画 ガンマー第3号 宇宙大作戦 - all cinema |publisher = [[allcinema|allcinema]] |language = ja <!--|archiveurl = http://megalodon.jp/2013-0314-1203-37/www.allcinema.net/prog/show_c.php?num_c=5096 |archivedate = 2013-3-14--> |accessdate= 2013-03-14 }}</ref>。77分（日本公開版）、90分（海外公開版）。[[製作|製作]] ： [[東映|東映]]、ラム・フィルム。

公開当時的書籍『'''ガンマ第3号 宇宙大作戦'''』表記<ref name = "日本特撮・幻想映画">{{Cite book|author = 石井博士ほか |year = 1997 |title = 日本特撮・幻想映画全集 |publisher = 勁文社 |page = 199 |isbn = 4766927060}}</ref>。

==介绍==
有颗小行星用很快的速度向地球撞来被航空署观测到。为拯救地球，以杰克为首的队伍登陆小行星，想将其炸毁。
