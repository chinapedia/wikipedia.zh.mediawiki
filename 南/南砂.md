'''南砂'''（{{jpn|j=南砂|hg=みなみすな|rm=Minamisuna|f=y}}）是[[東京都|東京都]][[江東區_(東京都)|江東區]]的地名，屬[[城東區_(東京都)|城東]][[地域|地域]]。

[[郵遞區號|郵遞區號]]為136-0076。

== 概要 ==
[[江戶時代|江戶時代]]起為繁榮的住宅地。江東區六個[[江東區_(東京都)#都市核|都市核]]（[[行政|行政]]、[[商業|商業]]、[[文化|文化]]等的廣域據點）之一，是江東區東部的中心。

町內展現[[下町|下町]]、江戶情懷的老街與新興街區混雜。10 - 30層的中高公寓大廈眾多，人口約4萬人，多於一樣進行再開發的[[豐洲|豐洲]]、[[東雲_(江東區)|東雲]]地區。南砂過去為寶六島與[[砂町|砂町]]的一部分。

=== 地理、公園 ===
位於江東區東部、聯繫[[龜戶|龜戶]]與[[新木場|新木場]]的[[明治通_(東京都)|明治通]]中間。東邊的[[東京地下鐵|東京地下鐵]][[東京地下鐵東西線|東西線]][[南砂町站|南砂町站]]到[[大手町_(千代田區)|大手町]]約10分鐘車程。

鄰近都心綠地眾多，[[荒川_(關東)|荒川]]河畔的「荒川砂町水邊公園」吸引民眾來此散步與垂釣。南砂町站東口附近是「南砂三丁目公園」。此外，可釣[[蝦虎|蝦虎]]與賞花的[[仙台堀川公園|仙台堀川公園]]，是能展現四季變化的親水綠色休閒步道。

「南砂綠道公園」利用了[[東京都電車|都電]]的軌道遺跡，春天湧入大量賞花客。

=== 區東部的商業中心 ===
南砂吸引了周邊的區內與[[江戶川區|江戶川區]]的購物客來訪。

複合商業設施[[TOPYREC_PLAZA|TOPYREC PLAZA]]有[[佳世客|佳世客]]、[[DOIT|DOIT]]、OSSO（健身中心），[[ROUND_ONE|ROUND ONE]]等約40多間店鋪入駐。

南砂町站南側、進行再開發的[[新砂|新砂]]是'''南砂町購物中心SUNAMO'''所在地。SUNAMO是城東地域最大的複合商業設施，包含7間大型店在內共有100間以上的店鋪入駐。

從車站步行到兩座商業設施約5分鐘。

此外，南砂還有[[宜得利|宜得利]]、[[山田電機|山田電機]]、[[安托華|安托華]]等大型商店，是江東區東部主要的商業地。

=== 地域 ===
; 中心地
: TOPYREC PLAZA、區役所南砂出張所、區民館等為中心的地區。2000年TOPYREC PLAZA完工以後，周邊開始建造高層公寓。主要設施有區役所出張所、區民館、區立[[圖書館|圖書館]]的中央館「江東圖書館」，人潮眾多。
; 南砂町站周邊
: 車站周邊，近年的開發區域之一，發展尚未成熟。2007年為止，站前商店以[[便利商店|便利商店]]與餐飲店為主，店數逐漸增加。
: 隨著車站東部的再開發，[[醫院|醫院]]、高層公寓等開始動工。[[巴士站|巴士站]]、計程車乘車處位在東口。
; 再開發地域
: 車站東部，有大規模高層住宅、部分企業、醫療中心。2008年10月SUNAMO完工。
; 西部與北部
: 住宅街為主，無超大型商業設施。北部有南砂最高的建築物「SKY CITY南砂」，仙台堀川公園等綠地眾多。西部最近的車站是[[東陽町站|東陽町站]]。

== 歷史 ==
現行的南砂是舊[[武藏國|武藏國]][[葛飾郡|葛飾郡]]寶六島一部分，之後為砂村新田全域與永代新田、平井新田大半區域。南砂街區的發展可追溯至江戶初期。

=== 地名由來 ===
南砂在江戶時代稱「寶六島」，之後因砂村一族的開拓而稱「砂村」，町制施行後命名「砂町」。編入東京市城東區，為「南砂町」。

=== 奈良時代 ===
傳聞749年（[[天平勝寶|天平勝寶]]元年），[[藤原鎌足|藤原鎌足]]三世孫[[藤原豐成|藤原豐成]]在當時未開拓之地建立[[富賀岡八幡宮|富賀岡八幡宮]]。但也有說是在[[江戶時代|江戶時代]]初期建立。

=== 江戶時代 ===
[[File:100_views_edo_029.jpg|thumb]]』「砂村元八幡」（現在的富賀岡八幡宮）。]]
[[德川家康|德川家康]]開發江戶初期，[[江東區|江東區]]決大部分是泥灘、沙洲，蘆葦叢等的濕地。南砂稱作「寶六島」。

1657年（[[明曆|明曆]]3年）發生[[明曆大火|明曆大火]]。江戶進行都市改造期間的1659年（[[萬治|萬治]]2年），[[相模國|相模國]][[三浦郡|三浦郡]]（[[神奈川縣|神奈川縣]]）的[[砂村新左衛門|砂村新左衛門]]來此開發新田，為砂村新田。大約是現在的南砂一丁目〜七丁目與東砂八丁目<ref>『江東區のあゆみ』（江東區広報広聴課発行）</ref>。北端為境川（現[[清洲橋通|清洲橋通]]。別名砂村川），也是與其他新田的邊界。此外，砂村新左衛門還開發[[橫須賀市|橫須賀市]][[久里濱|久里濱]]內川新田與[[橫濱市|橫濱市]][[吉田新田|吉田新田]]。

1681年（[[延寶|延寶]]9年），永代島新田與砂町新田被指定為江戶市區的垃圾傾倒場<ref>{{cite web|url =http://www.geog.or.jp/journal/back/pdf113-6/p785-801.pdf|title =東京臨海部における埋立ての歴史|accessdate =2008年7月14日|author =遠藤毅|deadurl =yes|archiveurl =https://web.archive.org/web/20051222084018/http://www.geog.or.jp/journal/back/pdf113-6/p785-801.pdf|archivedate =2005年12月22日}}</ref>，開始填埋低地地區。砂村新田成為[[近郊農業|近郊農業]]地帶，進行[[葱|砂村葱]]、[[茄子|茄子]]、[[黃瓜|黃瓜]]、[[西瓜|西瓜]]等的[[促成栽培|促成栽培]]。

開發時勸請的八幡社是現在[[富賀岡八幡宮|富賀岡八幡宮]]（南砂七丁目）的前身。但因缺乏淡水，後來移往深川（[[富岡八幡宮|富岡八幡宮]]），之後此處稱作「元八幡」<ref>宮尾しげを 『東京 昔と今 I』 保育社〈保育社カラーブックス〉，1963年，51頁。</ref>。[[洲崎_(東京都)|洲崎]]到此地是一片廣大的沿海[[濕地|濕地]]，也是風景名勝。元八幡以[[櫻花|櫻花樹]]、[[松樹|松樹]]聞名。

現在町內的商店街與[[交番|交番]]仍保留「元八幡」的地名。除此之外，仙氣稻荷神社因[[空襲|空襲]]而燒毀，現在三丁目內有神社遺跡的石碑。

南砂曾在『[[四谷怪談|東海道四谷怪談]]』的「砂村隱亡堀之場」登場。隠亡堀是真實存在的溝渠で，位在境川與橫十間川交會的岩井橋附近，因火葬場而得名。岩井橋則是源自於阿岩小姐（四谷怪談女主角）<ref>{{cite web|url = http://www.kcf.or.jp/fukagawa/pdf/03010090000501.pdf|title = 資料館ノート第74号 江東の掘割・川(6) 江戶近郊農漁村の掘割 -生業の場，ハナシの舞台として-|accessdate = 2008年7月16日|author = 江東區深川江戶資料館|deadurl = yes|archiveurl = https://web.archive.org/web/20110703160406/http://www.kcf.or.jp/fukagawa/pdf/03010090000501.pdf|archivedate = 2011年7月3日}}</ref>。

=== 明治時代 ===
[[明治|明治]]時代，南砂的[[海苔|海苔]]、[[牡蠣|牡蠣]]，[[金魚|金魚]]的[[養殖業|養殖]]活動興盛。江東區活躍的金魚養殖家初代[[秋山吉五郎|秋山吉五郎]]在砂町與[[浦安市|浦安市]]<ref>浦安市の金魚池跡の史跡表示</ref>等地進行金魚養殖，培育出[[朱文金|朱文金]]等新品種。1910年（明治43年）的「[[關東大水災|明治43年大水災]]」讓富賀岡八幡宮的松樹、櫻花樹全毀。

=== 大正、昭和時代 ===
1917年（大正6年）的「[[關東大水災|大正6年大海嘯]]」（正確來說是颱風造成的[[暴潮|暴潮]]，不同於[[海嘯|海嘯]]）<ref>{{cite web|url = http://www.saigaidensho.soumu.go.jp/saigai/import.2006-12-27.190811-2/|title = 『災害伝承情報データベース』「大正6年の大津波」|accessdate = 2008年7月16日}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>與1923年（大正12年）的[[關東大地震|關東大地震]]，南砂災情慘烈。1924年（大正13年），「砂町海水浴場」開設。這是由地方青年團主辦的[[海水浴場|海水浴場]]，1日吸引3萬164人入場<ref>藤田清「砂町誌」（中央自治研究会，1924年）</ref><ref>{{cite web|url = http://wf.ocean.cst.nihon-.ac.jp/2005-2006kennkyuugyouseki/gakujutu%20pdf/gakujutu%20takeda.pdf|title = 『平成17年度 日本大学理工学部 学術講演会論文集』「東京のウォーターフロントにおける余暇文化の変遷に関する研究 海水浴場を中心として」|accessdate = 2008年7月14日|author = 武田知大，横内憲久，岡田智秀，寶泉立夫，松本真奈美}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。加上当時，[[洲崎_(東京都)|洲崎]]的遊郭與[[洲崎球場|洲崎球場]]，這一帶成為遊樂地區。城東電氣軌道的路面電車在此地營運到1972年（昭和47年）為止，「砂町海水浴場」營業到1948年（昭和23年）為止。

1930年（昭和5年），記取「明治43年大水災」的教訓所修築的[[荒川放水路|荒川放水路]]<ref>{{cite web|url = http://www.ara.go.jp/category/13_data/history/sanko_11.html|title = 荒川放水路改修平面図|accessdate = 2008年7月14日|deadurl = yes|archiveurl = https://web.archive.org/web/20090430202312/http://www.ara.go.jp/category/13_data/history/sanko_11.html|archivedate = 2009年4月30日}}</ref>完工，[[大正|大正]]時代至昭和時代繼續進行填海工程，為現在的[[新砂|新砂]]。能代表過去海岸線的事物僅剩下成為圍欄地基的堤防與社區通（海岸堤通）。

之後，此地成為工業區。1929年（昭和4年），越中島支線開設。北砂設置[[小名木川站|小名木川站]]。1931年（昭和6年），[[汽車製造|汽車製造]]的[[平岡工場|平岡工場]]（東京製作所）移至此地。此時，[[地層下陷|地層下陷]]的問題逐漸浮出檯面，南砂二丁目從1918年（大正7年）至1980年（昭和55年），共下沉4.57公尺。1945年（昭和20年）3月10日的[[東京大空襲|東京大空襲]]，本地因工廠聚集而遭到轟炸，損害嚴重。戰後，1949年（昭和24年）[[凱蒂颱風|凱蒂颱風]]的暴潮襲擊南砂。

1959年（昭和34年），「首都圏建成區限制工業等的相關法律」（[[工業等制限法|工業等制限法]]）制定，工廠的設立、擴增受到限制。在地層下陷等公害不斷加重的情況下，企業負擔不斷增加。1965年（昭和40年）左右，工廠開始遷離<ref>{{cite web|url =http://www.geog.or.jp/journal/back/pdf115-4/p500-507.pdf|title =東京低地における工場等の分布を主体とした土地利用状況の変遷|accessdate =2008年7月15日|author =遠藤毅}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1969年（昭和44年），[[帝都高速度交通營團|帝都高速度交通營團]]（現東京地下鐵）東西線延伸。「南砂町站」開設。從原來沒有都電與[[都營巴士|巴士]]等公共交通機關營運的「陸上孤島」搖身一變，工廠舊地陸續改建為團地。汽車製造東京製作所舊址改建為南砂2丁目住宅。南砂一丁目至六丁目各町也建造[[公營住宅|都營住宅]]。

=== 平成時代 ===
[[平成|平成]]時代，高層公寓大廈開始興建。[[日本泡沫景氣|泡沫景氣]]期間，SKY CITY南砂湧入大量申請書。1987年（昭和62年），達140倍的申請倍率成為[[日本參議院|國會]]的議題之一<ref>{{cite web|url = http://kokkai.ndl.go.jp/SENTAKU/sangiin/109/1410/10910161410004a.html|title = 参議院会議録情報 第109回国会 決算委員会 第4号 |accessdate=2008年7月20日 }}</ref>。

2000年（平成12年），城東地區最大的商業設施[[TOPYREC_PLAZA|TOPYREC PLAZA]]開幕。2004年（平成16年），[[清砂大橋|清砂大橋]]開通。「新砂土地區劃整理事業」開始進行<ref>{{cite web|url = http://www.metro.tokyo.jp/INET/OSHIRASE/2004/08/20e82300.htm|title = 新砂地區に新たな複合市街地が誕生 土地區画整理事業の完了について|accessdate = 2008年7月18日|deadurl = yes|archiveurl = https://web.archive.org/web/20070824052633/http://www.metro.tokyo.jp/INET/OSHIRASE/2004/08/20e82300.htm|archivedate = 2007年8月24日}}</ref>，南砂町站南側進行再開發。[[永代通|永代通]]北側的大規模公寓大廈，南側的醫院與商業設施陸續完工。

=== 沿革、年表 ===
* 1878年（[[明治|明治]]11年）11月：[[郡區町村編制法|郡區町村編制法]]實施，葛飾郡中屬[[東京府|東京府]]的部分為[[南葛飾郡|南葛飾郡]]。
* 1889年（明治22年）：[[市制_(日本制度)|市制]][[町村制|町村制]]實施，砂村新田、龜高村、又兵衛新田、荻新田、太郎兵衛新田、中田新田、大塚新田、冶兵衛新田、八郎右衛門新田大部分區域與永代新田、平井新田、久左衛門新田、八右衛門新田、南本所出村一部分合併乘自[[砂町|砂村]]。
* 1921年（[[大正|大正]]10年）7月1日：砂村施行町制，為[[砂町|砂町]]。
* 1932年（[[昭和|昭和]]7年）：砂町編入[[東京市|東京市]]，與[[龜戶|龜戶町]]、[[大島町_(東京府)|大島町]]合併為[[城東區_(東京都)|城東區]]。舊大字永代、本砂町、平井、四十町合為南砂町一丁目至九丁目，其餘為北砂町一丁目至十丁目。
* 1967年（昭和42年）7月1日：住居表示施行，南砂町一、二、三、五丁目與四、六丁目一部分、深川平井町、深川東陽町一部分進行區劃整理，成立現行的南砂一丁目至七丁目。南砂町六丁目大半與七、八丁目與北砂町八丁目合併為東砂六丁目至八丁目，南砂町四丁目大半與九丁目合併為新砂一丁目至三丁目。

== 交通 ==

=== 鐵道 ===
最近的車站是東京地下鐵東西線南砂町站與[[東陽町站|東陽町站]]。

東西線執行[[直通運轉|相互直通運轉]]後，東達[[津田沼站|津田沼站]]、[[東葉勝田台站|東葉勝田台站]]，西至[[中野站_(東京都)|中野站]]、[[三鷹站|三鷹站]]，約10分鐘左右可達[[日本橋站_(東京都)|日本橋站]]、大手町站（東京站）。南砂町站在2000年代以後東口附近進行再開發後，利用者逐漸增加。此外，南砂町站不是[[東京地下鐵東西線#快速|快速]]停車站，但有[[東京地下鐵東西線#通勤快速|通勤快速]]停靠が停車し，2008年3月15日起，平日早晨尖峰時刻的7點〜8點半日本橋、大手町方向的快速列車全部統一為通勤快速列車。

東陽町站是全列車停車站。

此外，目前有建造龜戶〜南砂町〜新木場間的[[輕軌|LRT]]新線構想。
