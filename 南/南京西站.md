'''南京西站'''是位于[[中国|中国]][[江苏省|江苏省]][[南京市|南京市]][[鼓楼区_(南京市)|鼓楼区]]的火车站，又称'''下关站'''。1908年建成投用，当时称南京站，是[[沪宁铁路|沪宁铁路]]终点站。1968年[[南京长江大桥|南京长江大桥]]建成后，配套建设了新的[[南京站|南京站]]，原南京站改为南京西站。现在的南京西站是[[宁铜铁路|宁铜铁路]]起点站，有六座[[港湾式月台|港湾式月台]]。

== 历史 ==
===早期历史===
[[File:Nanking_Station.jpg|thumb]]
[[File:Xiaguan_Railway_Station_1930.jpg|thumb]]
[[File:Loaded_train_ferry_crosses_the_Yangtse_at_Nanjing_(1930s).jpg|thumb]]
[[File:Xiaguan_Railway_Station_1947.jpg|thumb]]
南京西站于1905年开始建设，1908月年建成投入使用，命名'''南京车站'''，是南京最早的火车站。根据《[[申报|申报]]》的报道，在南京西站的通车典礼上“中西官绅到者甚多，颇极一时之感”。同年12月15日，[[沪宁铁路|沪宁铁路]]全线通车。当时的车站根据英国铁路车站“一等站屋”规格设计，共设有站房18间，候车室两间，总面积530平方米。站场设有[[月台|月台]]三座，到发线总长469米。此外车站管辖江边货场<ref name="shrail">{{cite book|title=上海铁路局志|publisher=中国铁道出版社|isbn=9787113053802|year=2004}}</ref>{{rp|218}}<ref name="yangtze">{{cite web|author=扬子晚报|title=南京西站服役104载行将退出历史 曾见无数事件|url=http://cq.people.com.cn/news/2012313/201231382285441465.htm|publisher=人民网|accessdate=2012-09-22|deadurl=yes|archiveurl=https://archive.is/20130101112006/http://cq.people.com.cn/news/2012313/201231382285441465.htm|archivedate=2013-01-01}}</ref>。车站建成后，与江北[[南京北站_(1912年通车)|浦口站]]间的人流、物流通过轮渡接驳<ref name="njrail">{{cite web|title=南京市志 交通　邮电|url=http://221.226.86.187:8080/webpic/njdfz/UpLoadFile/html/sz1_4/thd.html|publisher=南京市地方志办公室}}</ref>。

车站在建成后经历了许多历史事件，如1912年元旦，[[孙中山|孙中山]]由此站进入南京并宣誓就任中华民国临时大总统。此外，1927年北伐战争期间，[[军阀|军阀]][[孙传芳|孙传芳]]也由此逃离<ref name="yangtze"/>。

1930年，通往下关的[[热河路_(南京)|热河路]]修筑完成。由于原南京车站过于老旧，已不再适应当时不断增长的客货运量，[[国民政府|国民政府]]决定投资9万余元在惠民河东异地重建该站，定名'''下关车站'''，同时新建铁路轮渡以缓解沪宁-津浦直通人、物流中转不便的状况。新站于1931年建成，新站房面积850平方米，全部采用柳安木料，外墙中部及两翼皆使用花岗石。其中中间主楼3层，设有椭圆形候车室，楼顶中央有园亭，并由两侧山形屋架连接屋顶；主楼两侧设有2层附楼，设有三等、四等候车室<ref name="yangtze"/>。而站台到发线长度则分别延长至400米、430米、450米。该站房于1937年[[南京保卫战|日军占领南京]]时被破坏<ref name="shrail"/>{{rp|218}}。[[南京铁路轮渡|南京铁路轮渡]]于1933年10月22日正式通航，从此直通列车通过岸边栈桥即可上船过江。日军占领南京期间，渡轮由[[华中铁道|华中铁道]]管理并更名“南京航送”<ref name="njrail" />。

1945年车站复名'''南京站'''<ref name="shrail"/>{{rp|218}}，并由[[京沪铁路管理局|京沪铁路管理局]]管辖。1947年国民政府投资90余亿原重建站房。新的站房由[[基泰工程公司|基泰工程公司]]的建筑师[[杨廷宝|杨廷宝]]设计，[[徐顺兴|徐顺兴]]负责建造。新站房于当年完工，并由[[于右任|于右任]]题写站名。1948年，车站新建站台1座<ref name="njrail"/><ref name="yangtze"/>。1949年[[第二次国共内战|第二次国共内战]]后期，[[国民政府|国民政府]]在[[渡江战役|撤出南京]]时炸毁了该站站房的候车室<ref name="shrail"/>{{rp|218}}。

===1949年后===
1950年，[[中华人民共和国|中华人民共和国]]政府拨款对该站进行了修复，同时将站房阁楼用作上海铁路局南京铁路分局机关<ref name="yangtze"/>，货运则集中在江边货场办理<ref name="njrail"/>。1958年9月，车站在宋家埂地区（现工农新村）修建规模19股道的调车场，同时增加轮渡南岸站配线25股。1960年12月，面积2.7万平方米的建宁路货场建成启用，开始分担江边货场的货运业务<ref name="shrail"/>{{rp|218}}。

1968年9月[[南京长江大桥|南京长江大桥]]和新的[[南京站|南京站]]投入使用后，车站更名为'''南京西站'''且下辖的建宁路货场划归新南京站管理。1973年[[南京东站|尧化门编组站]]扩建后建宁路货场复归本站管理<ref name="shrail"/>{{rp|218}}。作为[[南京长江大桥|南京长江大桥]]和新的[[南京站|南京站]]启用的后续影响，南京西站的客流量开始下降，而[[南京铁路轮渡|南京铁路轮渡]]则于同年5月停航<ref name="njrail"/>。

1987年，车站为解决笨重零担货物的装卸问题，在建宁路货场北侧的郭家山地区新建面积3.15万平方米的笨重零担货场；1991年又在建宁路货场内开辟集装箱场。同年，车站对原西运转场和东运转场进行改造，利用原东运转场地块，将原西运转场的客车到发线长度延长958米并合并两个运转场为宁西客场<ref name="njrail"/>。

===客运停办及搬迁===
[[File:Train_7102_in_Nanjing_West_Railway_Station_Platform.jpg|thumb]]停靠在车站站台（2012年）]]
南京西站在铁路运输中的地位在[[南京长江大桥|南京长江大桥]]和[[南京站|南京站]]投用后已经大不如从前，2012年，根据江苏省第十一届人民代表大会第五次会议南京代表团分组讨论会上透露的信息，南京西站的货场搬迁将实行“先建后搬”，新址选定在[[南京东站|南京东站]]附近。<ref>{{Cite web|url=http://news.longhoo.net/2012-02/11/content_8468473.htm|title=南京西站今年正式启动搬迁 宁杭城际年内开通|publisher=龙虎网|date=2012-02-11|deadurl=yes|archiveurl=https://web.archive.org/web/20120415055331/http://news.longhoo.net/2012-02/11/content_8468473.htm|archivedate=2012-04-15}}</ref>[[上海铁路局|上海铁路局]]称，考虑到南京市政府实施市政改造的需要，决定自2012年3月25日起至另有通知时止，临时停办该站客运业务。3月20日起，在南京西站始发、终到的列车陆续改在[[南京站|南京站]]始发、终到。<ref>{{Cite web |url=http://www.12306.cn/mormhweb/zxdt/tlxw_detail_3167.html |title=关于上海铁路局管内南京西站临时停办客运业务的公告 |accessdate=2012-03-20 |work=上海铁路局 |publisher=铁路客户服务中心 |date=2012-03-15 }}</ref>消息一经传开，许多南京市民前往[[中华门站_(铁路)|中华门站]]乘坐[[7101/7102次列车|7102次列车]]前往南京西站为该站“送别”，单次旅客发送量多达1500多人。<ref>{{Cite web |url=http://nanjing.auto.sohu.com/20120321/n338410773.shtml |title=中华门到南京西7102次 "清新"列车受追捧 |accessdate=2012-03-21 |work= |publisher=搜狐汽车 |date=2012-03-21 }}</ref>2012年3月25日晚，开往[[厦门|厦门]]的[[K161/162次列车|K161次列车]]离开南京西站后，南京西站客运业务中止。

2015年末，南京西站搬迁计划浮出水面。根据南京市与[[上海铁路局|上海铁路局]]的协商。原南京西站区域，包括站房，站场，大龙池列车存车场，宋家梗调车场以及南京东机务段南京西运用车间将在近期移交给南京市地方，并利用灯泡线改造成为城市有轨电车，南京西站站房和站台铁轨将被保留并维修建成[[南京铁路博物馆|南京铁路博物馆]]。

现有站场西侧的铁路存车线和机务段，将搬迁至站场东部的郭家山地块，规模缩小一半。保留6条存车线，机务段保留主要整备车间，其余迁移至[[尧化街道|尧化门]]的[[南京东机务段|南京东机务段]]总部。2016年6月，尧化门货场开始建设，占地984.6亩，工期大约18个月<ref>{{cite news|title=上海铁路局尧化门货场工程全面展开|url=http://www.peoplerail.com/rail/show-456-274763-1.html|date=2016-06-18|publisher=人民铁道网}}</ref>，后于2018年3月30日投入使用<ref>{{cite web|url=http://www.sohu.com/a/226796523_391456|title=好消息！尧化门站货场今日开通运营，欢迎入驻！|work=[[上海铁路局|上海铁路局]]|date=2018-03-30|accessdate=2018-12-18}}</ref>，同时南京西站的集装箱运输业务转由尧化门货场办理<ref>{{cite news|title=南京一货场两千多万的检视仪闲置生锈 每年维护费高达百万|url=https://xw.qq.com/cmsid/20181102A10KCY00|publisher=腾讯网|date=2018-11-02}}</ref>。

南京西站客场又被称为宁西客场，位于整个站场的西南侧。除了旅客站台外，客场还包括轮渡封存线和其他线，共设有42条股道，并设有专用线通往上海铁路局南京东机务段南京西整备车间、南京工务段、[[下关电厂|下关电厂]]等8家单位<ref name="njrail"/>。

宁西客场设有4座旅客站台，7条到发线，4座站台被总面积1429平方米的雨棚覆盖<ref name="shrail"/>{{rp|219}}。客场旁为合肥车辆段南京西客车整备所，主要为[[南京站|南京站]]始发终到的普速列车进行整备、检修、调配作业<ref>{{cite news|title=南京西站，最后一批扳道工仍在默默坚守|url=http://www.xinhuanet.com/local/2017-01/21/c_1120356468.htm|publisher=新华网|date=2017-01-21}}</ref>。南京站终到的普速列车到站后会回送至本站进行整备，并在开车前回送至南京站<ref>{{cite news|title=南京西站有一群与高温为伴的调车人 他们用汗水换来乘客顺利出行|url=http://jsnews.jschina.com.cn/shms/201807/t20180721_1776733.shtml|publisher=中国江苏网|date=2018-07-21}}</ref><ref>{{cite news|title=南京-南京西，最牛临客是“零客”|url=http://news.sina.com.cn/c/2011-01-18/031621830996.shtml|publisher=新浪网|date=2011-01-18}}</ref>；而列车的客运乘务员亦在本站出退乘<ref>{{cite news|title=高铁更快更密，南京人作别T66/T65|url=http://js.xhby.net/system/2019/01/06/030913359.shtml|publisher=新华报业|date=2019-01-06}}</ref>。
====站房====
南京西站的站房建于1947年，由[[杨廷宝|杨廷宝]]设计，日后历经多次改造。车站站房呈“U”形将车站客场半包围，面积1.87万平方米。其中主楼位于客场西侧、股道末端，宽75.6米、纵深17米、高15米共三层。西侧外立面设有五孔13米高的大拱门作为进站口，内设有面积2213平方米的候车室、面积180平方米的软席候车室和面积91平方米的母子候车室，最多可同时容纳4000余名旅客候车。主楼南北两侧为高2层的附楼。南附楼设506平方米行李房和售票处，并设有检票口及辅助出入口各1个；北附楼为贵宾室、邮件用房以及出入口<ref name="shrail"/>{{rp|219}}<ref name="yangtze"/>。

南京西站站房目前已不办理客运业务，目前为南京客运段的机关所在地。

===货场===
南京西站共设有建宁路、郭家山、江边三座货场，办理整车、集装箱、集零为整混装货物业务，主要到发货品为钢铁、化工、饮食、其他及集装箱类，共设有20股道<ref name="njrail"/>。[[南京海关|南京海关]]下辖的[[金陵海关|金陵海关]]在建宁路货场设有查验点<ref>{{cite news|title=江苏：南京市工商联赴金陵海关对接工作|url=http://www.acfic.org.cn/gdgsl_362/js/jsgslgz/201803/t20180302_51430.html|publisher=中华全国工商业联合会|date=2018-03-02}}</ref><ref name="12306info">{{cite web|url=http://hyfw.12306.cn/hyinfo/action/FwcszsAction_index?type=1 |title=12306货运站点查询 上海铁路局 |publisher=Hyfw.12306.cn |date= |accessdate=2019-03-09}}</ref>。

===动车所===
南京动车段南京动车运用所于位于大庙村郭家山货场南侧、[[南京站|南京站]]的西侧，建于2010年3月，6月29日投入运行<ref>{{cite journal|author=国敢|title=沪宁城际铁路南京动车运用所动车组管理信息系统正式投入运行|year=2010|journal=铁路计算机应用}}</ref>。南京动车运用所主要负责[[沪宁城际高速铁路|沪宁城际高速铁路]]及[[宁启铁路|宁启铁路]]部分动车组的一二级检修、运用和整备工作<ref>{{cite news|title=“动车医生”：40度高温里夜行万步|url=http://www.xinhuanet.com/politics/2018-07/14/c_1123126008.htm|publisher=新华网|date=2018-07-14}}</ref>。

动车所部分由宋家埂调车场改建而成。宋家埂调车场建于1958年，原设有到发线6股、其他线13股<ref name="njrail"/>。

==事件==
*1946年6月23日，[[马叙伦|马叙伦]]、[[盛丕华|盛丕华]]、[[雷洁琼|雷洁琼]]等11人组成的和平请愿团在抵达南京下关火车站后遭到国民党特务围殴受伤，史称“[[下关惨案|下关惨案]]”<ref name="yangtze"/>。
*1958年9月14日车站轮渡区下船的甲调机在9号道岔与由丙机顶送上船的车辆发生正面冲突，造成机车中破、3节货车大破并中断[[南京铁路轮渡|轮渡]]航行10小时53分<ref name="shrail"/>{{rp|591}}。
