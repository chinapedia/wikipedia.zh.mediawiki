'''南雅镇'''是[[中华人民共和国|中国]][[福建省|福建省]][[南平市|南平市]][[建瓯市|建瓯市]]南部的一个镇，位于[[建溪|建溪]]中游，与[[延平区|延平区]]及[[顺昌县|顺昌县]]相邻。

== 名称由来 ==
《建瓯县志》记载“东汉献帝建安初年，即隶属建安县，因系建瓯南面门户，初名‘南瓦口’。后人因这里的山水相依，四季山青水秀，环境幽雅，因而更其名为南雅。”<ref> {{cite web |url=http://www.agri.com.cn/town/350783103000.htm |title=南雅镇金农网 |accessdate=2008-08-31 |date= |publisher=金农网 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160305093826/http://www.agri.com.cn/town/350783103000.htm |archivedate=2016-03-05 }}</ref>。

== 地理 ==
南雅镇属中低山地，境内高山有云龙岩、龙岗岩、打四门山、莲花山、老保岗、通天阳、九洞岩、高世林岗等。溪河属建溪流域，主要有高阳溪、集北溪、白沙溪、中湾溪等8条。

== 交通 ==
南雅镇距离建瓯市区23公里，距离南平市区45公里。[[205国道|205国道]]、[[横南铁路|横南铁路]]从南雅镇经过，境内设有火车货车站和[[浦南高速公路|浦南高速公路]][[互通口|互通口]]<ref> {{cite web |url=http://www.jianou.gov.cn/DeptDetail.aspx?uuid=63 |title=南雅镇 |accessdate=2008-08-31 |date=2008-05-12 |publisher=建瓯市政务网 |language=zh-hans }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 行政区划 ==
南雅镇共辖1个居委会，21个行政村，2个镇办场，117个自然村<ref> {{cite web|url=http://xzqh.org/quhua/35fj/0783jo.htm |title=建瓯市 |accessdate=2008-08-31 |date= |publisher=行政区划网站 |language=zh-hans }}</ref>，分别是：

四龙居委会、南雅村、新建村、伊村村、新村村、小雅村，十字街村、集瑞村、杉溪村、小康村、叶康村、皇康村、大康村、梅村村、鲁口村、白水源村、房村村、仁墩村、黄园村、山安村、白沙村、太平村<ref> {{cite web |url=http://www.jianou.gov.cn/details.aspx?cid=3&m_id=2 |title=行政区划、人口 |accessdate=2008-08-31 |date= |publisher=建瓯市政务网 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20090322194645/http://www.jianou.gov.cn/details.aspx?cid=3&m_id=2 |archivedate=2009-03-22 }}</ref>。

== 经济 ==

南雅镇的主要产业有[[林业|林业]]、[[茶叶|茶叶]]、[[柑桔|柑桔]]、[[葡萄|葡萄]]及[[锥栗|锥栗]]等，素有“柑桔之乡”、“南国葡萄之乡”、“闽北竹乡”之称。竹制品加工、茶叶加工、食用菌加工、绿色食品加工、奶牛饲养为南雅镇的五大龙头产业。
