'''南昌地铁5号线'''<!--（官方文件正式名称：南昌市轨道交通4号线）-->是南昌一条拟建的轨道交通线路，该线路规划从高新到下罗，成U字形。南昌轨道交通集团相关的微博V认证人员在微博上交流曾表示5号线的环线的一部分<ref>[http://weibo.com/1870478701/yttSV3CNA 2012-7-21 16:20来自新浪微博]</ref>。

== 简介 ==
5号线代表色为南昌轨道橙{{color|#{{南昌地铁识别色|5}}|■}}

== 历史 ==
* 2008年9月15日开始公示5条线路的规划中包括5号线<ref>[http://www.lsnc.cn/gs/gdbgs.html 南昌市城市快速轨道交通线网及建设规划公众征询意见] {{webarchive|url=https://web.archive.org/web/20130720103017/http://www.lsnc.cn/gs/gdbgs.html |date=2013-07-20 }}</ref><ref>[http://www.jxgdw.com/jxgd/xwzxztmb/2008jxx/knc/userobject1ai845634.html 南昌地铁远景规划征询民意 将有5条地铁线路] {{webarchive|url=https://web.archive.org/web/20160303222001/http://www.jxgdw.com/jxgd/xwzxztmb/2008jxx/knc/userobject1ai845634.html |date=2016-03-03 }}</ref><ref>[http://jiangxi.jxnews.com.cn/system/2008/10/16/002860601.shtml 南昌地铁远景规划征询民意(图)]</ref>。
* 2009年7月29日确认5号线作为远期规划将根据实际情况而建设<ref>{{Cite web |url=http://www.ncnews.com.cn/ncxw/bwyc/t20090729_469995.htm |title=南昌地铁线路站点公布 1号线2014年通车 |access-date=2013-08-17 |archive-url=https://web.archive.org/web/20140219125225/http://www.ncnews.com.cn/ncxw/bwyc/t20090729_469995.htm |archive-date=2014-02-19 |dead-url=yes }}</ref>。
* 2013年4月市政府表示将提前建设目前规划的线路<ref>[http://jiangxi.jxnews.com.cn/system/2013/04/11/012366266.shtml 南昌考虑再建五条地铁线 拟规划到十号线形成半小时经济圈]</ref><ref>[http://jiangxi.jxnews.com.cn/system/2013/04/17/012375524_02.shtml 南昌市市长陈俊卿：2015年五条地铁全部开建]</ref>。
* 2013年5月底政府规划第二轮地铁建设的三个方案，三个方案只有最后一套会建设5号线<ref>[http://jndsb.jxnews.com.cn/system/2013/05/31/012446645.shtml 明年进入第二轮地铁建设期]</ref><ref>{{Cite web |url=http://www.ncnews.com.cn/ncxw/jrnc/t20130531_1024778.htm |title=南昌未来7年有望建成3至4条地铁 |access-date=2013-08-17 |archive-url=https://web.archive.org/web/20130909090344/http://www.ncnews.com.cn/ncxw/jrnc/t20130531_1024778.htm |archive-date=2013-09-09 |dead-url=yes }}</ref><ref>[http://jiangxi.jxnews.com.cn/system/2013/05/30/012446400.shtml 专家把脉南昌地铁第二轮建设规划 三套方案备选]</ref>。
* 2013年6月1日确认第二轮建设的线路中无5号线<ref>[http://jiangxi.jxnews.com.cn/system/2013/06/02/012449170.shtml 地铁3号线一期工程明年下半年或开工]</ref>。
* 2013年11月有报道指出5号线计划在2016年开工<ref>[http://jiangxi.jxnews.com.cn/system/2013/11/26/012816940.shtml 南昌2015年起每年通一条地铁 同时通过干线路网改造]</ref>。
