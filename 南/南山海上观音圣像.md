'''南山海上观音圣像'''（{{lang-en|'''Guan Yin of the South Sea of Sanya'''}}）位于[[中华人民共和国|中华人民共和国]][[海南省|海南省]][[三亚市|三亚市]][[南山寺|南山寺]]，由[[中华人民共和国|中华人民共和国]][[国务院宗教事务局|国务院宗教事务局]]和[[海南省人民政府|海南省人民政府]]一同建造的观音像。

== 沿革 ==
农历1999年9月19日，南山海上观音圣像正式开建。<ref name="guanyin " >{{cite web |url=http://news.enorth.com.cn/system/2005/04/24/001011190.shtml |title= 世界最高海上观音—108米南山海上观音建造纪实 |accessdate=2005 |publisher=北方网 |work= |language=zh-cn}}</ref>

农历2000年1月5日，金钢洲工程开工。<ref name="guanyin " />

农历2000年12月，圆通宝殿建成。<ref name="guanyin " />

农历2003年1月，金刚宝座建成。<ref name="guanyin " />

农历2003年3月，观音像主体钢结构开建。<ref name="guanyin " />

农历2005年3月16日，观音像举行开光典礼。<ref>{{cite web |url=http://news.ifeng.com/gundong/detail_2013_04/27/24727989_0.shtml |title=海上观音圣像开光 |accessdate=2013 |publisher=凤凰网 |work= |language=zh-cn}}</ref>

== 雕像 ==
[[观音|观音]]站立在莲花台上，安置在金刚洲（观音岛）上，整个地域被称为“观音净苑”景区，分为观音金身、佛光、千叶宝莲、紫檀木雕须弥底座。<ref name=" bull" >{{cite web |url=http://fo.ifeng.com/special/jiudukunandabeiguanyin/guanyindaochang/detail_2013_03/22/23418197_0.shtml?_from_ralated |title= 海南三亚海上观音|accessdate=2013 |publisher=凤凰网 |work= |language=zh-cn}}</ref>观音像高108米，耗资8亿人民币，[[黄金|黄金]]100多公斤，120多克拉[[南非|南非]][[钻石|钻石]]，数千粒[[红宝石|红]][[蓝宝石|蓝宝石]]、[[祖母绿|祖母绿]]、[[珊瑚|珊瑚]]、[[松石|松石]]、[[珍珠|珍珠]]，100多公斤[[翠玉|翠玉]]等，<ref name=" bull" />有东、西、北三面，一手持箧，寓意智慧，一手持莲，寓意和平，一手持珠，寓意慈悲，由赵朴初题写“南山海上观音”。<ref name="guanyin " />

== 制作方 ==
上海现代设计集团制作了“观音净苑”景区，葛洲坝集团第一工程有限公司和上海远东国际桥梁有限公司制作基础设施，南京晨光集团有限责任公司制作观音像。
