'''南氣仙沼車站'''（{{jpn|j=南気仙沼駅|hg=みなみけせんぬまえき|rm=Minami-kesennuma eki|f=y}}）是一位於[[日本|日本]][[宮城縣|宮城縣]][[氣仙沼市|氣仙沼市]]仲町二丁目、[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[氣仙沼線|氣仙沼線]]沿線的[[鐵路車站|鐵路車站]]。南氣仙沼是由[[氣仙沼車站|氣仙沼車站]]代為管理，屬[[東日本旅客鐵道盛岡支社|JR東日本盛岡支社]]的管轄範圍。

== 車站構造 ==
伴隨[[公車捷運系統|BRT]]開始運行，BRT車站移動至鄰近{{link-ja|氣仙沼市立醫院|気仙沼市立病院}}的道路上，並改名為帶有副站名的「'''南氣仙沼（市立醫院入口）站'''」<ref name="jr-sendai20120718">{{PDFlink|[http://www.jr-sendai.com/wp-content/uploads/2012/08/press20120718-kesennumasen.pdf 気仙沼線における暫定的なサービス提供開始について]}} - 東日本旅客鉄道仙台支社プレスリリース（2012年7月18日）</ref>。

以下為地震發生前的情況。

[[側式月台|側式月台]]1面1線的[[地面車站|地面車站]]。以前曾為1面2線。

南氣仙沼車站內設有[[綠窗口|綠窗口]]（營業時間：7時50分～17時30分）與[[自動售票機|自動售票機]]。除此之外，JR氣仙沼旅遊中心（{{lang|ja|JR気仙沼旅行センター}}）也設在站內，旅行社社員也兼負責售票窗口的業務。

== 歷史 ==
*1957年（[[昭和|昭和]]32年）2月11日 - 配合氣仙沼至[[氣仙沼港車站|氣仙沼港]]之間的貨運支線通車，新設南氣仙沼車站作為分歧點。車站由氣仙沼站長負責管理。
*1975年（昭和50年）4月1日 - 南氣仙沼車站配置站長，成為管理車站。
*1979年（昭和54年）11月1日 - 貨運支線廢線。
*1986年（昭和61年）11月1日 - 南氣仙沼車站的站長職位廢除，又改移回由氣仙沼車站的站長管理之模式，但繼續由氣仙沼方面派遣站員至本站服務。
*1987年（昭和62年）4月1日 - [[日本國鐵分割民營化|日本國鐵分割民營化]]，車站管轄劃歸由東日本旅客鐵道繼承。
*1990年代前半 - 設置JR氣仙沼旅行中心。
*2003年（[[平成|平成]]15年）春 - 站內的會車設備撤除。
*2007年（平成19年）4月1日 - 站內的[[Kiosk_(便利商店)|Kiosk便利商店]]（{{lang|ja|キヨスク}}）停業。
* 2011年（[[平成|平成]]23年）3月11日 - 東北地方太平洋近海發生[[2011年日本東北地方太平洋近海地震|嚴重地震]]，地震引發的[[海嘯|海嘯]]沖毀靠近海岸的許多鐵路設施，松岩也是23個被沖毀的鐵路車站之一<ref>{{cite web |url=http://www.sankeibiz.jp/business/news/110401/bsd1104010801004-n1.htm |language=ja |title=<span lang="ja">【東日本大震災】ＪＲ東日本、太平洋沿岸の２３駅流失</span> |publisher=[[產經新聞社|產經新聞社]] |date=2011-04-01 |accessdate=2011-04-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20110413050421/http://www.sankeibiz.jp/business/news/110401/bsd1104010801004-n1.htm |archivedate=2011-04-13 }}</ref>。
