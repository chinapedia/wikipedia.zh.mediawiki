'''南关厢'''为位于[[中国|中国]][[浙江省|浙江省]][[嘉兴|嘉兴]][[海宁市|海宁市]][[硖石街道|硖石街道]]东南部的一处历史街区，南北分别至塘桥弄和大瑶桥，东西分别至洛塘河和人民路（未来西侧拟规划为河西路），南北长约350米，东西宽约50米，占地面积约1.7公顷，主街南关厢街呈南北走向、与洛塘河平行。

南关厢因其中有明末[[周宗彝|周宗彝]]为抗清而设置的'''关厢'''（多位于水陆要道，早开晚关，以防盗贼）而得名，历史上以此为界，以北称永秀坊（一说保秀坊），以南称锦秀坊（一说锦绣坊）<ref>{{cite web |url=http://hnnews.zjol.com.cn/hnnews/system/2010/07/26/012425706.shtml|title=《走进南关厢》|accessdate=2014-09-12|work=|publisher=海宁新闻网}}</ref>，清中叶和民国初因地处米市中心的南部而一度兴旺，至抗战时米市开始衰败，街区经济地位由强转弱。沿街建筑大多建于清代至民国，以东西朝向、二层为主，南关厢、会源庵及个别民居院落的历史可以追溯到明代。现状整个街区较好地保存了明清时期江南市镇商住合一、前店后河、小桥流水的历史风貌和传统格局，2006年6月被公布为[[浙江省历史文化街区|浙江省历史文化街区]]<ref>{{cite web |url=http://www.zjww.gov.cn/unit/2006-05-11/56810502.shtml|title=《海宁南关厢》|accessdate=2014-09-12|work=|publisher=浙江省文物局}}</ref>。自2008年11月起海宁市政府投资约1.2亿元启动整修工程<ref>{{cite web |url=http://zjnews.zjol.com.cn/05zjnews/system/2008/11/18/014986187.shtml|title=《海宁硖石南关厢面貌将改写》|accessdate=2014-09-13|work=|publisher=浙江在线}}</ref>，至2012年4月已基本完工<ref>{{cite web |url=http://hnnews.zjol.com.cn/hnnews/system/2012/04/20/014955428.shtml|title=《南关厢二期工程基本完工》|accessdate=2014-09-13|work=|publisher=海宁新闻网}}</ref>。
