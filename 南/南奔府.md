'''南奔府'''（{{thai|t=จังหวัดลำพูน|r=Changwat Lamphun}}，{{IPA-th|t͡ɕāŋ.wàt lām.pʰūːn}}），一譯'''喃奔府'''，是[[泰國|泰國]]北部的一個[[泰國的府|府]]，府治為[[南奔市|南奔市]]。

== 历史 ==
该府首府为泰北最古老的城市，建于公元663年，原名为“立甫猜城”，曾一度被缅甸统治，后被[[郑昭|郑王大帝]]夺回，隶属于清迈府，至曼谷王朝时代，该城逐渐强大，于1954年升格为府治。

== 地理 ==
该府面积约为4，505平方公里，距曼谷约为670公里。其疆域西北临[[清邁府|清邁府]]，南临[[達府|達府]]，东濒[[南邦府|南邦府]]。

南奔府大部分为平原地带，有4大河流，东南方则多山林，盛产木材，荧光石，钨，锰，褐煤，磷以及水果[[龙眼|龙眼]]等。

== 行政 ==

南奔府被分成 8縣(''Amphoe'')；再分成 51區(''Tambon'')；並細分成 551村(''Muban'')。
