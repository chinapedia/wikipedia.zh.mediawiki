'''南明货币'''，是指[[明朝|明朝]]灭亡后，由[[明朝皇族|明朝皇族]]和官员在[[中国北方与南方|中国南方]]建立的各个政权（史称“[[南明|南明]]”）所发行的[[货币|货币]]。南明各个政权共有十八年历史，其中有四个政权曾铸行了货币，分别是[[弘光|弘光]]帝[[弘光帝|朱由崧]]、[[隆武|隆武]]帝[[隆武帝|朱聿鍵]]、鲁王[[朱以海|朱以海]]和[[永曆_(南明)|永历]]帝[[永历帝|朱由榔]]。

== 弘光通宝 ==
[[File:Chinese_coin_Hongguang_Tongbao_(flickr_8719045207).jpg|缩略图]]
'''弘光通宝'''是[[南明|南明]][[弘光|弘光]]皇帝[[弘光帝|朱由崧]]在位期间所发行的货币。1644年4月25日，[[李自成|李自成]]攻克[[北京|北京]]，[[崇祯|崇祯]]皇帝[[崇禎帝|朱由检]]自杀殉国，明朝灭亡，明朝宗室及遗留大臣多辗转到了南方。6月7日，福王朱由崧在南京称监国，6月19日即皇帝位，改元弘光，铸行弘光通宝。1645年6月21日，弘光帝被清军所俘，弘光通宝停铸。

虽然弘光通宝的铸行时间短，但铸行数量却相当可观。弘光通宝分小平和折二两种。小平有光背、背上星、背上凤字三大类，“凤”指安徽省凤阳，因马士英曾任凤阳总督。小平的直径在2.4厘米左右，折二型直径在2.7厘米至2.8厘米。<ref>{{Cite web|url=http://news.artxun.com/hongguangtong-1307-6531501.shtml|title=短命的弘光通宝 |publisher=博宝艺术网}}</ref>

== 隆武通宝 ==
[[File:Longwu_Tongbao.jpg|缩略图]]
'''隆武通宝'''是[[南明|南明]][[隆武|隆武]]皇帝[[朱聿鍵|朱聿鍵]]在位期间所发行的货币。1645年6月21日，弘光帝[[朱由崧|朱由崧]]被清军所俘，同年唐王朱聿鍵在[[福州|福州]]称帝，改元[[隆武|隆武]]，铸行隆武通宝。次年，朱聿鍵被俘，隆武通宝停铸，行用一年时间。

隆武通宝只有小平、折二两种。钱文楷隶相间，版别多样；小平有大、小样之分和宽缘窄缘之别，钱文有“生”字隆和“正”字隆两种，通字有双点通和单点通、武字右偏旁，有上点武和下点武及无点武，钱背有铸星纹及“户”、“工”、“南”、“留”等字，“南”字有大小二样，“南”、“留”二字较罕。<ref name=art>{{Cite web|url=http://www.artww.com/evaluate/knowledge/show-1108|title=南明时期的钱币|publisher=艺粹|deadurl=yes|archiveurl=https://web.archive.org/web/20140113190813/http://www.artww.com/evaluate/knowledge/show-1108|archivedate=2014-01-13}}</ref>

== 大明通宝 ==
[[File:Da_Ming_Tong_Bao_(大明通寶)_-_Scott_Semans.jpg|thumb]]

'''大明通宝'''是南明鲁王[[朱以海|朱以海]]被奉为[[监国|监国]]时所铸。1645年，在台州（今浙江[[临海市|临海]]）避难的鲁王朱以海被明朝旧臣[[张国维|张国维]]、[[张名振|张名振]]等在[[绍兴府|绍兴]]拥立为监国。当年12月铸行了大明通宝。1646年6月，清军攻打鲁王，朱以海流亡。

大明通宝材质有红铜、黄铜两种，铸造不精，只见小平钱，背铸“户”、“工”、“帅”等字，也见光背平钱，有的背“帅”在穿上，也有穿右“帅”字。初铸钱径26毫米，后铸钱径为24毫米至25毫米。<ref name=art/>

== 永历通宝 ==
[[File:Yongli_Tongbao._10cash.jpg|缩略图]]
'''永历通宝'''是南明[[永曆_(南明)|永历]]皇帝[[永历帝|朱由榔]]在位期间所发行的货币，是南明发行时间最长的货币。1646年11月20日，桂王朱由榔在[[肇庆府|肇庆]]被明朝两广总督[[丁魁楚|丁魁楚]]推为监国，12月24日即皇帝位，改元永历，以明年为永历元年，铸行“永历通宝”，有小平、折二、及背文五厘、一分四等钱。

郑成功及其后人统治台湾期间，曾请[[日本|日本]]代铸永历通宝，直至康熙年间清朝占领台湾后，被收买销毁改铸为康熙通宝钱。
