'''南昌大学科技学院'''是坐落在[[中华人民共和国|中华人民共和国]][[江西省|江西省]][[南昌市|南昌市]]的一所普通高校，<ref>{{Cite web |url =http://www.ndkj.com.cn/mainShowSingleNews.aspx?sid=80  |title =南昌大学科技学院概况 |author = |date =  |publisher =  |language = zh-hans |accessdate = |quote =  }}</ref>是由[[南昌大学|南昌大学]]和[[江西昌大企业集团|江西昌大企业集团]]合作举办的[[独立学院|独立学院]]，成立于2001年（民国90年）。曾获[[“全国十佳独立学院”|“全国十佳独立学院”]]称号。共有本科专业42个，分为5个学科部，分别是人文学科部、理工学科部、财经学科部、基础学科部和信息学科部。占地面积571.3亩。<ref>{{Cite web |url =http://www.ndkj.com.cn/mainShowSingleNews.aspx?sid=80  |title =南昌大学科技学院概况 |author = |date =  |publisher =  |language = zh-hans |accessdate = |quote =  }}</ref>2014年3月26日省教育厅建议南昌大学：正式行文向省政府报告科技学院是否入驻[[共青城大学城|共青城大学城]]。<ref>{{Cite web |url = http://news.ifeng.com/gundong/detail_2014_03/26/35170186_0.shtml |title = 江西将压缩南昌大学本科规模 改制共青学院入驻大学城 |author =  |date =  |publisher =  |language = zh-hans |accessdate =  |quote =  }}</ref>
