'''南美球會盃'''（{{lang-es|Copa Sudamericana}}，{{IPA-es|ˈkopa suðameɾiˈkana}}；{{lang-pt|Copa Sul-Americana}}，{{IPA-pt|ˈkɔpɐ ˈsuw ɐmeɾiˈkɐnɐ}}），为[[南美足联|南美足联]]举办的年度性国际[[足球|足球]]俱乐部赛事。南美杯也称南美俱乐部杯或南美洲球会杯，是南美足联举办的年度性国际足球俱乐部赛事，是南美洲第二大俱乐部杯赛，类似于[[欧洲足联|欧洲]]的[[欧罗巴联赛|欧罗巴联赛]]（UEFA Europa League，之前称欧洲联盟杯UEFA Cup）性质，与其所不同的是，只要符合赛制规定，同一个俱乐部既可以参加[[南美解放者杯|南美解放者杯]]也可以参加南美俱乐部杯的赛事。赛事比赛时间安排在下半年的八月至十二月间进行，与解放者杯时间完全错开。

== 历史 ==
南美俱乐部杯始于2002年，但前身可追溯到创办于1992年的[[南美足联杯|南美足联杯]]，以及始于1998年的[[南方共同市场杯|南方共同市场杯]]等赛事。2002年，南美俱乐部杯正式创办，但巴西足球俱乐部没有参赛该年度的赛事。2003年，日本汽车制造商[[日產|日產]]开始赞助该项赛事，巴西球队开始参赛。

== 赛制 ==
南美俱乐部杯的比赛分为外围赛和淘汰赛两个阶段，每轮比赛都采用主客场淘汰制，没有小组循环赛，直到决出冠军。
*外围赛：南美足联各个成员国根据该国所分配到的参加淘汰赛的名额，在本国的球队之间争夺这些名额。南美足联每年分配名额的方式都会有微调。
*淘汰赛：由各个国家获得淘汰赛名额的球队根据抽签捉对厮杀，直到决出冠军。

== 参赛球队 ==
南美杯足球赛的参赛资格规定较为复杂。2010年版的参赛名额如下：
* 1队为卫冕冠军
* 8队为[[巴西|巴西]]球队
* 6队为[[阿根廷|阿根廷]]球队
* 南美足联其余8个成员各3个参赛队名额
