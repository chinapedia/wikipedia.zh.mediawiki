'''南美貘'''（[[学名|学名]]：''{{lang|la|Tapirus terrestris}}''），又名'''巴西貘'''或'''低地貘'''，是一種[[貘|貘]]，在[[南美洲|南美洲]]是最大的野生陸地[[動物|動物]]。

== 外觀 ==
南美貘呈深褐色，由頭頂至頸背有一道短而直立的鬃毛。牠們體長達1.8-2.5米，尾巴長5-10厘米，重230公斤。牠們肩高達77-108厘米。

== 分佈 ==
南美貘棲息在[[南美洲|南美洲]][[亞馬遜雨林|亞馬遜雨林]]及[[亞瑪遜盆地|亞瑪遜盆地]]近水的地方。牠們的分佈地北臨[[委內瑞拉|委內瑞拉]]、[[哥倫比亞|哥倫比亞]]及[[圭亞那|圭亞那]]，南至[[巴西|巴西]]、[[阿根廷|阿根廷]]及[[巴拉圭|巴拉圭]]，西至[[玻利維亞|玻利維亞]]、[[秘魯|秘魯]]及[[厄瓜多爾|厄瓜多爾]]。

== 行為 ==
[[File:Tapirus.terrestris.flehmen.jpg|thumb]]。]]
南美貘能游善跑，就算在崎嶇的山地也能奔走自如。牠們的壽命約25-30歲。在野外，牠們的天敵主要是[[鱷魚|鱷魚]]及大型的[[貓科|貓科]]，如[[美洲豹|美洲豹]]及[[美洲獅|美洲獅]]。南美貘當受驚時會走入水中。牠們每個月也會[[交配|交配]]。

== 食性 ==
南美貘是[[草食性|草食性]]的。牠們利用靈活的吻，來吃[[葉子|葉子]]、芽、嫩枝及細小的樹枝。

== 保育狀況 ==
南美貘的數量減少是因被烹吃及失去棲息地所致。牠們一般被認為是[[瀕危物種|瀕危物種]]，但[[滅絕|滅絕]]的可能性則較其他的[[貘|貘]]為低。<ref name=IUCN/>
