'''南極圈'''（{{lang-en|Antarctic Circle}}）是地球上[[南緯|南緯]]66[[角度|°]]34′的一个假想圈，是[[南寒带|南寒带]]与[[南温带|南温带]]的界线，其精确的[[纬度|纬度]]数值是与[[黄赤交角|黄赤交角]][[余角|互餘]]，其以内大部分是[[南極洲|南極洲]]。南极圈附近因無山脉阻攔，盛行[[西風|西風]]，[[溫帶氣旋|溫帶氣旋]]常常出现，又有海冰漂浮，是非常凶險的海域。西方航海界有“[[咆哮40度|咆哮的40°S]]、[[狂暴50度|狂暴的50°S]]、[[尖叫60度|尖叫的60°S]]”的说法。

由于地球围绕太阳公转时的[[章动|章动]]现象，地球的[[转轴倾角|黄赤交角]]一直在轻微地改变着。因此，纬度数值与黄赤交角互餘的南极圈也相应有改变。截至2000年，它位於南緯66°33′39（或66.56083°）上。南极圈南部的区域被稱為南極地区，而北部则是南溫帶。南极圈和[[北半球|北半球]]的[[北极圈|北极圈]]是对称的。

在南极圈以南的地方，一年中会有太阳二十四小时不下山和二十四小时都是夜晚的时候，这两种现象分别称为极昼和极夜。南极圈是南半球的极昼和极夜开始出现的界线，正如北半球的北极圈一般。

== 极昼和极夜 ==
{{main|极昼|极夜}}
[[File:axial_tilt_vs_tropical_and_polar_circles.svg|thumb]]

极昼和极夜（又称永昼、永夜）是[[地球|地球]]两极地区特有的现象，是指一天24小时之内全都是白天（极昼）或黑夜（极夜）。北极圈是[[北半球|北半球]]内仅有的可能发生极昼极夜的地方。每年的[[公历|公历]]3月23日，也就是[[春分|春分]]的时候，太阳直射赤道。从这一天开始，南极圈内开始出现极夜。一开始只有[[南极点|南极点]]，然后逐渐扩大到整个南极圈范围。6月22日，[[夏至|夏至]]日时，太阳直射[[北回归线|北回归线]]（北纬23°26′），这时南半球极夜范围达到最大，为南极圈以南的全部地方。之后，极夜范围慢慢缩小，到了9月23日[[秋分|秋分]]的时候，恢复到只有南极点有极夜。这一天过后南极的极昼开始，最初也是在南极点，然后延伸到整个南极圈的范围。12月22日[[冬至|冬至]]时，太阳直射南回归线，整个南极圈以南的区域都是极昼。接着极夜范围逐渐回缩，到了第二年的3月23日，即春分时极昼结束，而极夜重新开始。

[[File:Antarctic_circle.svg|thumb]]
南极圈内的地方，越靠近南极点，出现极昼极夜的时间就越长。南极点附近的地方一年有近一半的时间是极昼，另一半是极夜。南极圈内靠近边缘的地方则只有几天的极昼极夜。

南极圈内的地方太阳永远无法直射，[[太阳高度角|太阳高度]]也不高，在冬至日的时候达到最大高度。纬度为<math>\alpha</math>的地方，[[太阳高度角|太阳高度角]]最大为113°26′<math>- \alpha</math>，小于46°52′。

== 气候 ==
南极圈内日照短暂，太阳高度不大（阳光穿过的[[大气层|大气层]]厚度增加，导致反射和散失的热量增多），以及地表冰盖对阳光的反射，因此南极地区终年寒冷，属于冬季严寒、夏季凉爽的冰原气候。冷空气在进入南极圈内之后变冷沉降，形成了极圈内的[[极地高压带|极地高压带]]。从极地高气压区流向低纬度的[[空气|空气]]，由于地球[[自转偏向力|自转偏向力]]的作用，一律偏东，在南极圈内的地区形成东南风（从东南吹向西北），叫作[[极地东风带|极地东风带]]。在南极圈附近，极地东风携带的冷空气向下沉降，造成近地面的东南风，而从低纬度南下的暖空气与南面往北来的极地冷空气相遇，形成[[锋面|锋面]]，称为[[极锋|极锋]]。暖空气由于较轻，爬升到较重的北上冷气流之上，形成[[副极地上升气流|副极地上升气流]]。上升气流在高空又分为南向和北向流动的两个分支。向北的一支在南纬30°左右再度下沉，构成了南半球的[[大气环流|中纬度环流圈]]。向北南的一支到了南极点附近沉降，完成[[大气环流|高纬度环流圈]]。由于副极地上升气流的影响，南极圈附近的气压较低，称为[[副极地低压带|副极地低压带]]。<ref>{{Cite book | title = 地理 普通高中课程标准实验教材| publisher = 湖南教育出版社 | pages = 51| language = zh-hans }} </ref>由于南极圈附近没有山脉阻挡，中高纬度的盛行西风畅通无阻，海岸附近的风速可达17-18米每秒。

== 地理和人口 ==
南極圈内大部分由[[南極大陸|南極大陸]]占据，沒有人類在南極圈内永久定居。然而，南极圈内有來自不同國家的南极研究中心。科學家們輪流地按季節前往[[南极洲|南极洲]]作暫時的居住，並進行各種科學研究。在過去的一个世纪裡，南极大陆上已经建立起一些半永久性的捕鯨站，捕鯨的人可以生活在那裡一年或更長時間。至2009年为止，已经至少有三個孩子出生在南極洲，然而，他們都出生在南极圈内最靠北的北站。

== 洋流 ==
南极圈以南，南极洲边缘的海上，在极地东风带的吹拂下形成环绕南极洲大陆流动的南极绕极环流。而南极圈以北为受盛行西风影响的自西向东环绕南极大陆一周的[[西风漂流|西风漂流]]。
