'''南芬兰省'''（{{lang-fi|Etelä-Suomen lääni}}，{{lang-sv|Södra Finlands län}}）是[[芬兰|芬兰]]在1831至1997年间的一个[[省|省]]，毗连[[西芬兰省|西芬兰省]]和[[东芬兰省|东芬兰省]]，并和[[芬兰湾|芬兰湾]]及[[俄罗斯|俄罗斯]]接壤。

== 历史 ==
''历史，地理和文化方面，参见 [[:en:Tavastia|Tavastia]]，[[:en:Karelia|Karelia]]和[[:en:Nylandia|Nylandia]]''

在1997年，芬兰对行政区域进行了重新划分，将原有的12个省减少为6个省。[[新地省|新地省]]、[[屈米省|屈米省]]和[[海梅省|海梅省]]南部地区重新组成[[南芬兰|南芬兰]]省。

== 地区行政 ==

省行政机关由地区内部来自不同的部门的权威人事所组成，以促进中央政府对本区域行政的有效管理。南芬兰省行政机构由380名雇员组成，办公室分别分布在[[海门林纳|海门林纳]]、[[赫尔辛基|赫尔辛基]]和[[科沃拉|科沃拉]]。

== 行政区划 ==
南芬兰省划分为6个行政区:
* [[南卡累利阿区|南卡累利阿区]]
* [[派亚特海梅区|派亚特海梅区]]
* [[坎塔海梅区|坎塔海梅区]]
* [[新地区|新地区]]
* [[东新地区|东新地区]]
* [[屈米河谷区|屈米河谷区]]

南芬兰划分为88个[[芬兰市镇|市镇]]。
