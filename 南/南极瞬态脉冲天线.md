'''南極脈衝瞬態天線'''（Antarctic Impulsive Transient Antenna；ANITA）实验被设计来探究宇宙高能[[中微子|中微子]]，其方式为探测他们与[[南极|南极]]冰层的交互作用发射出的无线电脉冲。这将用到一个由32个无线电天线组成的阵列（布置成圆柱形，半径约为3米，高度约为5米），再以一个[[氦气球|氦气球]]悬挂到大约35000米的高空。 <ref>{{Cite web|url=http://www.ps.uci.edu/~anita/|title=ANITA: Antarctic Impulsive Transient Antenna}}</ref>
[[File:ANITA_3.jpg|缩略图]]
该[[中微子|中微子]]的[[能级|能级]]是 [[数量级_(能量)|10<sup>18</sup> eV]], 是由[[阿斯卡莱恩效应|阿斯卡莱恩效应]]而产生无线电脉冲。这些高能宇宙[[中微子|中微子]]产生于[[超高能宇宙射线|超高能宇宙射线]]和[[宇宙背景辐射|宇宙背景辐射]]的[[光子|光子]]间的交互作用，该观点被广泛接受。该实验希望可以揭示那些[[宇宙射线|宇宙射线]]的起源。<ref>{{Cite journal|title=High flyer listens for cosmic pings|author=|url=|date=23 December 2006|journal=[[New_Scientist|New Scientist]]|issue=|doi=|volume=|pages=}}</ref>

==  实验时间轴 ==
ANITA-I 在2006-07的夏天于[[南极洲|南极洲]][[麥克默多站|麥克默多站]]发射。该天线阵列应该在被[http://www.csbf.nasa.gov/ CSBF]收回前随着极地风飘了一个月。每一个任务（若被提供资金）需要和下一个任务至少间隔两年。ANITA-II经过修改，带有40个天线，于2008-09年的夏天在[[麥克默多站|麥克默多站]]发射。ANITA-III被期望提升5-10倍的敏感度且期望于将于2014年十一月发射。

== 资金 ==
ANITA目前被[[美国国家航空航天局|美国国家航空航天局]]提供资金支持。

== 合作者 ==

伦敦大学学院，[[芝加哥大学|芝加哥大学]]，[[国立台湾大学|国立台湾大学]]的成员。
