'''南紀勝浦溫泉'''（{{jpn|j=南紀勝浦温泉|hg=なんきかつうらおんせん}}）是位於[[日本|日本]][[和歌山縣|和歌山縣]][[東牟婁郡|東牟婁郡]][[那智勝浦町|那智勝浦町]]的一個[[溫泉|溫泉]]。舊稱勝浦温泉，但近年旅遊業界多習慣加上「南紀」。

== 概要 ==
南紀勝浦溫泉和[[南紀白濱溫泉|南紀白濱溫泉]]並為和歌山縣最具代表性的溫泉景區，也是前往[[世界遺產|世界遺產]]{{link-ja|那智山|那智山 (山)}}、[[那智瀑布|那智瀑布]]、[[熊野三山|熊野三山]]、[[吉野熊野國立公園|吉野熊野國立公園]]的據點。作為[[近畿圈|近畿圈]]、[[中京圈|中京圈]]双方的{{link-ja|奧座敷|奥座敷}}，距溫泉最近的{{link-ja|紀伊勝浦站|紀伊勝浦駅}}有自[[京都市|大阪]]、[[大阪市|大阪]]、[[名古屋市|名古屋]]出發的特急列車。

== 溫泉街 ==
[[File:Bokido_of_Hotel_Urashima01o4592.jpg|thumb]]
[[File:Hotel_Nakanoshima_Nachikatsuura_Wakayama02n4592.jpg|thumb]]
位於面對[[太平洋|太平洋]]的[[溺灣|溺灣式海岸]]，{{link-ja|紀之松島|紀の松島}}至{{link-ja|勝浦漁港 (和歌山縣)|勝浦漁港 (和歌山県)|勝浦港}}之間聚集許多飯店旅館。由於有些位於島嶼或[[海岬|海岬]]上，因此需要搭船往返本土。涵蓋整個狼煙半島的浦島飯店，擁有忘歸洞與玄武洞等面[[太平洋|太平洋]]的天然洞窟浴場，與{{link-ja|中之島飯店|ホテル中の島}}的紀州潮聞之湯同為當地的象徵之一。

南紀勝浦與南紀白濱同為著名的遊樂溫泉區，但不同於適合一日行的白濱，沒有大型休閒設施。許多遊客來此是為了風光明媚的自然環境與鄰近的[[世界遺產|世界遺產]][[熊野古道|熊野古道]]、[[那智瀑布|那智瀑布]]等。另外，紀伊勝浦漁港也是全國知名的[[鮪魚|鮪魚]]基地。附近的[[太地町|太地町]]有調査捕鯨活動，因此勝浦也有鯨魚料理。

溫泉街外有{{link-ja|共同浴場|共同浴場}}「はまゆ」，多為當地漁師等使用。

每年11月舉行[[熊野那智大社|熊野那智大社]]的獻湯祭。

== 歷史 ==
古時分為兩個溫泉，一是磯之湯，另一個是赤島溫泉。磯之湯傳聞始於[[弘化|弘化]]以前。赤島溫泉開湯年代不明。現在兩者皆稱為南紀勝浦溫泉。作為溫泉地則是起於[[大正|大正時代]]的開鑿。之後不斷的開鑿，源泉數量已超過100處
各旅館皆有自己的源泉。大正時代，[[紀州德川家|紀州德川家]]15代當主{{link-ja|德川賴倫|徳川頼倫}}到此地的洞窟溫泉時，讚賞「讓人忘了回家」（帰るのを忘れるほどである）。當時的洞窟溫泉就是現在的忘歸洞。

在急速發展下，1950年（昭和25年）指定為[[吉野熊野國立公園|吉野熊野國立公園]]。隨著知名度上升，觀光客不斷增加，昭和30年代成為新婚旅行的聖地。1978年（昭和53年），[[日本國有鐵道|國鐵]][[紀勢本線|紀勢本線]]（[[和歌山站|和歌山站]]〜{{link-ja|新宮站 (和歌山縣)|新宮駅|新宮站}}間）電氣化，交通更為便利。另一方面，隨著[[山陽新幹線|山陽新幹線]]通至博多後，國內旅行人氣逐漸轉移至[[九州_(日本)|九州]]，住宿客從最高的100萬人逐漸下降。

近年，「[[紀伊山地的聖地及朝聖路|紀伊山地的聖地及朝聖路]]」指定為[[世界遺產|世界遺產]]，本地的大自然與文化遺產再次獲得注目。
