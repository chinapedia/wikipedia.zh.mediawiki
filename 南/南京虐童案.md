'''南京虐童案'''是发生于[[2015年3月|2015年3月]]31日，[[中國|中國]][[江蘇省|江蘇省]][[南京市|南京市]][[浦口区|浦口区]]的一位养母李征琴[[虐待|虐待]]养子的[[刑事案件|刑事案件]]。4月3日，被虐待养子的照片在[[新浪微博|微博]]曝光，进而引起舆论关注。随后[[公安机关|公安机关]]介入，事件进入[[司法程序|司法程序]]。事发后，亲生父母“心疼孩子，理解养母”，同时在南京市江宁区人民法院起诉曝光者<ref name="a1">{{Cite web |url=http://www.thepaper.cn/newsDetail_forward_1363181 |title=南京虐童案剧情逆转：男童父母起诉发帖人|accessdate=2015-09-28 |author=记者：蓝天彬 |date=2015-08-12 |work= |publisher=|language=Zh-hans }}</ref>。

== 家庭背景 ==
事发时，男童九岁，报道采用[[化名|化名]]施小宝。他的生父是桂某、生母是张某<ref name="a1"/>，另有一个哥哥和姐姐<ref name="gczw1"/>。养父施某斌，养母李征琴是张某的表姐。施氏夫妇是高级[[知识分子|知识分子]]<ref name="gczw1"/>，同时家境富裕。因家境贫寒，亲生父母在2013年5月，将施小宝[[过继|过继]]给李征琴<ref>{{Cite web |url=http://js.xhby.net/system/2015/04/06/024246021.shtml|title=南京“虐童案”养母被刑拘|accessdate=2015-09-28 |author= |date=2015-04-06 |work= |publisher=新华报业网，来源：南京晨报 |language=Zh-hans }}</ref>。6月，施氏夫妇通过安徽省[[来安县|来安县]]民政局收养了施小宝，带回南京抚养<ref name="a2"/>。当事人李征琴是某媒体驻江苏记者站副站长，丈夫是一名从业20多年的律师施某斌，两人之前都有过一次婚姻，离婚后重组家庭<ref name="后续">{{Cite web |url=http://news.sina.com.cn/c/nd/2016-03-22/doc-ifxqnski7856091.shtml|title=南京虐童案男童：若养母不要自己 就死在南京|accessdate=2016-03-29|author= |date=2016-03-22 |work= |publisher=新浪网，来源：新京报  |language=Zh-hans }}</ref>。施某斌和李征琴各自有一个子女，施氏夫妇并不符合《[[中华人民共和国收养法|中华人民共和国收养法]]<ref>《[http://www.gov.cn/banshi/2005-05/25/content_849.htm 中华人民共和国收养法]》第六条收养人应当同时具备下列条件：（一）无子女</ref>》要求收养人无子女的要求。案发后证实，收养过程中，施氏夫妇递交的[[文件|文件]]上使用了伪造政府[[公章|公章]]。同时，李征琴否认是自己伪造了政府公章<ref>{{Cite web |url=http://china.cnr.cn/ygxw/20150928/t20150928_520000459.shtml |title=南京虐童案庭审：李征琴办理收养手续所用证明被指伪造|accessdate=2015-09-28 |author= |date=2015-09-28 |work= |publisher=央广网|language=Zh-hans }}</ref>。

== 刑事调查 ==
4月5日，[[南京市公安局|南京市公安局]]在官方[[微博|微博]]“平安南京”发布消息称，养母因涉嫌故意伤害被[[刑事拘留|刑事拘留]]<ref name="a2"/>，后变更为[[取保候审|取保候审]]<ref name="gczw1"/>。施小宝则被亲生父母带回安徽抚养<ref name="a2"/>。有专家表示，现行法律并未剥夺施氏夫妇对施小宝的监护权。南京警方无权将施小宝送回亲生父母身边。同时认为，警方与民政部门将孩子送回亲生父亲身边的做法“令人匪夷所思，这等同于孩子刚出虎穴又进狼窝。”<ref>{{Cite web|url=http://news.ycwb.com/2015-04/05/content_20039149.htm|title=南京遭虐打男童被曝因家境贫寒而过继，生母养母系表姐妹|accessdate=2015-09-28|author=澎湃新闻记者：龚菲|date=2015-09-29|work=|publisher=金羊网，来源：澎湃新闻网|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20150712123209/http://news.ycwb.com/2015-04/05/content_20039149.htm|archivedate=2015-07-12}}</ref>

7月20日，南京市浦口区检察院依法向浦口区人民法院提起[[公诉|公诉]]，养母李征琴涉嫌故意伤害犯罪<ref name="a1"/>。

8月12日，亲生父母桂某和张某在南京市江宁区人民法院起诉曝光者徐某某。后法院判决驳回桂氏夫妇的起诉，此判决获得了网民的认可，得到舆论的支持<ref>{{Cite web |url=http://www.icswb.com/h/147/20150928/352063.html |title=发帖曝虐童“不侵权”为何备受认可|accessdate=2015-09-28 |author=记者：汪昌莲，编辑：曾茜 |date=2015-09-29  |work= |publisher=长沙晚报网|language=Zh-hans }}</ref>。张某则指责：“都是网络所造成的，我们两家全部的劫难都是他（徐某某）害的，他就是罪魁祸首。”“这给我们两家和孩子都带来了劫难，在人前人后都抬不起头。”张某表示希望浦口区人民法院的判决，能让施氏夫妇重新获得施小宝的[[抚养权|抚养权]]<ref name="gczw1">{{Cite web |url=http://www.guancha.cn/society/2015_09_27_335745.shtml |title=南京虐童案周一开审 被虐男童生母斥网友发帖毁了两家|accessdate=2015-09-28 |author= |date=2015-09-29  |work= |publisher=观察者网|language=Zh-hans }}</ref>。

9月28日，案件在南京市浦口区人民法院开庭审理<ref name="a2">{{Cite web |url=http://society.huanqiu.com/article/2015-09/7660463.html?bdwz |title=南京虐童养母受审画面曝光 被告人否认犯罪(图)|accessdate=2015-09-28 |author= |date=2015-09-28 |work= |publisher=环球网，来源：中国新闻网|language=Zh-hans }}</ref>，审理期间，被告人李征琴在法庭上情绪失控，喊叫哭闹，并出现企图[[自杀|自杀]]行为，致使庭审中断两次。之后审判长宣布，经院长批准，决定对李征琴[[逮捕|逮捕]]。<ref>{{Cite web |url=http://www.chinanews.com/sh/2015/09-29/7549984.shtml|title=南京虐童案养母休庭期间企图自杀 法院决定对其逮捕)|accessdate=2015-09-28 |author= |date=2015-09-28 |work= |publisher=中国新闻网|language=Zh-hans }}</ref>

9月30日，南京市浦口区人民法院对该案作出[[一审|一审]]宣判，被告人李征琴因犯故意伤害罪，被判处有期徒刑6个月<ref>{{Cite web |url=http://news.ifeng.com/a/20151001/44771756_0.shtml |title=南京虐童案被告人被判六个月 |accessdate=2015年10月1日 |author= |date=2015年10月1日 |work= |publisher=凤凰网（来源：北京青年报）|language=Zh-hans }}</ref>。宣判后，李征琴因不服判决提出上诉，要求法院判她无罪，并表示希望能继续收养案件被害人施小宝，而法院则表示，李征琴不宜继续抚养其养子<ref>{{Cite web |url=http://news.sohu.com/20151001/n422475912.shtml |title=南京“虐童案”一审宣判 养母因故意伤害获刑6个月 |accessdate=2015年10月1日 |author= |date=2015年10月1日 |work= |publisher=搜狐新闻（来源：现代快报）|language=Zh-hans }}</ref>。

11月20日下午，南京市中级人民法院二审宣判，裁定驳回被告人李征琴上诉，维持原判<ref>{{Cite web |url=http://legal.people.com.cn/n/2015/1120/c42510-27838328.html|title=南京虐童案二审宣判：驳回上诉 维持原判 |accessdate=2015年11月20日 |author= |date=2015年11月20日 |work= |publisher=人民网|language=Zh-hans }}</ref>。宣判后，李征琴在[[常州|常州]]女子监狱服刑。2016年3月13日上午7点50分，李征琴刑满释放<ref name="后续"/>。
