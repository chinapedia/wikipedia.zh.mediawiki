'''南宁沛鸿民族中学'''是[[南宁市|南宁市]]完全公办中学、[[广西壮族自治区|广西壮族自治区]]示范性普通高中。她以[[广西|广西]]著名教育家[[雷沛鸿|雷沛鸿]]先生命名。学校办学特色鲜明，师资力量雄厚。学校现有桃源、江南校区。随着桃源校区最后一届高三毕业，学校现在以桃源校区为初中部，江南校区为高中部。学校拥有[[广西|广西]]一流的教育基础设施和教学设备，是一所具有光荣历史和优良传统的学校。

== 校园 ==
'''南宁沛鸿民族中学'''桃源校区（原东区）位于南宁市桃源路86号，为初中部。桃源校区有雷沛鸿纪念花园一座，园中有[[雷沛鸿|雷沛鸿]]先生纪念雕塑。江南校区（原西区）位于[[南宁市|南宁市]]江南区沛鸿路（原槎路）5号，为高中部。江南校区有逸夫图书馆一座。在桃源和江南校区都有3块石头，上面刻着“和谐”、“自强”、“厚德”。<ref name="地址">[http://www.nnph.com.cn/about_63.html 学校地址] {{Wayback|url=http://www.nnph.com.cn/about_63.html# |date=20130324010419 }}，南宁沛鸿民族中学网站学校概况。</ref><ref name="校区图片">[http://www.nnph.com.cn/scenery.html 校园风光] {{Wayback|url=http://www.nnph.com.cn/scenery.html# |date=20120826122103 }}，南宁沛鸿民族中学网站校园风光。</ref>

== 校史<ref name="校史">[http://www.nnph.com.cn/about_62.html 学校校史] {{Wayback|url=http://www.nnph.com.cn/about_62.html# |date=20120728010659 }}，南宁沛鸿民族中学网站学校校史。</ref> ==
1951年，开始筹建广西军区干部子弟学校。

1952年6月1日，成立广西军区小学校。

1954年，更名未南宁市第二十一小学 。

1955年，学校小学部迁往柳州。

1956年8月。学校从柳州迁回南宁。

1957年，学校更名南宁市珠江小学

1958年8月，南宁市珠江小学撤销，学校停办1年。

1959年3月，广西军区决定恢复办学，名字仍用南宁市珠江小学。

1960年8月，学校更名南宁珠江子女学校并增加初中一年级。

1969年9月，学校挂牌更名为南宁市第十六中学。

1970年，学校设立高中部，成为一所完全中学。

1990年12月30日，为纪念著名教育家[[雷沛鸿|雷沛鸿]]先生，[[南宁市|南宁市]]将学校命名为南宁沛鸿学校。

2002年6月25日，南宁沛鸿学校与南宁民族高中合并，成立南宁沛鸿民族中学。

2012年11月16日，为南宁沛鸿民族中学建校60周年。

== VEX机器人项目 ==
开展机器人活动是'''南宁沛鸿民族中学'''的优良传统。从2003年至今获世界级金奖2个，国级级金奖2个，银奖3个，铜奖7个等等。2008年走向辉煌，新开辟[[VEX|VEX]]第二战场和常规战场双双在[[南宁市|南宁市]]和[[广西|广西]]比赛中包揽冠军和亚军，在第八届[[全国青少年机器人竞赛|全国青少年机器人竞赛]]中勇夺初中VEX金奖。2009年走向世界，载誉归来，勇夺2009年美国[[VEX机器人世界锦标赛|VEX机器人世界锦标赛]]“最高评判奖”（又最佳判断奖）金奖；在[[南宁市|南宁市]]和[[广西|广西]]比赛中蝉联冠军和亚军，继续在全国赛中锤炼队伍。美国达拉期早报、世界联合报对'''南宁沛鸿民族中学'''的机器人活动进行报道。2010年继续广西霸主地位，勇夺第十届全国青少年机器人竞赛高中VEX一等奖和2010年美国VEX机器人世界锦标赛“最佳创新(Innovate Award)”金奖两个金奖。在2012年12月举办的第六届亚太机器人锦标赛上，勇夺VEX项目金奖。
