'''南滨路'''，位于[[重庆市|重庆市]][[南岸区|南岸区]]滨江路，与[[渝中区|渝中区]]隔江相望。属城市公园类自然风景旅游景区，南滨路旅游观光区全长25公里，占地16万平方米，是集防洪护岸、城市道路、旧城改造和餐饮、娱乐、休闲为一体的城市观光休闲景观大道。

== 南滨路的来世今生 ==
百年南滨见证了东西方文明的交融；见证了中华民族的荣辱兴衰；见证了重庆从这里走向世界；………… 1891年，中国西部的第一个海关验关处便设立于南岸的狮子山下。抗战时期，立德乐洋行老建筑中国海关总署仍设立于海狮路70号。 重庆开埠以后，这里曾经遍布外国洋行，在此经营出口贸易、航运，使重庆成为上海、天津、汉口后的第四大洋货销售中心。抗战时期，美国、英国、比利时、法国等国曾在此设大使馆或公使馆。达官商贾云集，使海棠溪至弹子石一带形成了著名的“富人聚居区”。 在晚清时期，英、美、法、德、日等列强，在重庆修建兵营、使领馆和租界，也在南滨路留下了历史的痕迹，如法国水兵营、美国水师俱乐部、日本租界等。 　　北临长江，背依南山，可观最美渝中夜景；巴渝文化、宗教文化、开埠文化、大禹文化、码头文化、抗战遗址文化如珍珠般遍布沿线，使南滨路具有了深厚的历史文化背景和极高的文化产业开发价值，也使南滨路获得了“重庆外滩”的美誉。
== 交通 ==
南滨路夜景重庆素有“桥都”之名，目前南滨路上规划、在建和已建的桥梁有七座，即鹅公岩长江大桥、菜园坝长江大桥）、石板坡长江大桥及复线桥、东水门长江大桥、朝天门长江大桥、大佛寺长江大桥。南滨路一共规划了17条配套公路。这17条道路建成后，将形成完善的循环式道路网络，改善南滨路和南岸区的交通条件。 　　目前正在建设的还有轻轨三号线一期工程。随着相关工程的完工，已经繁华的南滨路将再次腾飞。

== 休闲观光 ==
南滨路不仅是一条交通要道。其建设口号是中国的首个游憩商业区，在此地段，尤其是正对渝中区的近8公里路段上，云集了大量餐饮、娱乐、宾馆，办公，高档住宅、旅游与观光项目。这里坐拥闻南滨路看过去名遐迩的重庆夜景，是观赏渝中半岛都会风光的首选。南滨路同时打造黄葛晚渡、海棠烟雨、龙门浩月、字水宵灯、峡江开埠和禹王遗踪六大景区，形成中国最大的滨江公园。使得该地区不论昼夜皆成为当地居民消遣的好去处。全长6.8公里的南滨公园作为重庆名片早已成为了许多外地游客必到之处，加上洋人街的开发，南滨路的旅游景点已经形成了文化与游乐互为补充的“旅游一条龙”。 而在南滨路管委会的计划中，还要通过对南滨路深层次文化内涵的有效挖掘，整合自然资源、人文资源，发展旅游观光业，以此来带动休闲娱乐业。南滨路上将新建金港游艇俱乐部、两江游码头、宗教旅游精品街、慈云寺老街、弹子石龙门浩老街、民间文化博物馆等项目，届时将联合旅行社包装推出一条适合大众口味的体验型旅游线路。
== 文化活动与创意 ==
南滨路除了举办各式各样的美食节、具有重庆特色的火锅节，还举办充满时尚文化气息的大型活动，如2005年亚太市长峰会焰火晚会、中法文化年图卢兹文化节、沙雕节，2007迎春灯会、重庆市第十一届山水都市旅游节、山城啤酒节、英伦艺术节等等。 　　在年轻的服装设计师魏昕眼中，南滨路自建成以来就是文化创意的先锋地。“还记得南滨路刚建成时，举行了首届沙雕艺术展，这对当时的重庆可谓是一大震撼。”魏昕说，每次到南滨路总会有新的感受，滨江路上的艺术雕塑、领先全市的新型展览。如开设画廊和艺术家工作室、游人手创馆、新型广告动漫影视图片生产基地，聚集一批雕塑、油画、版画、中国画等艺术原创工作室，以及相关的展示、传播和销售商店，着重发展南滨路的文化创意产业。

== 餐饮商务与会展经济 ==
南滨路看渝中半岛目前南滨路凭借其68个著名商家、包括陶然居，德庄，顺风123，澳门豆捞，渝信川菜等全国餐饮名店相继入驻，经营总面积10万平方米的经营规模，在去年实现营业额3亿元，并成为重庆打造美食之都的生力军。2007年3月南滨路荣获中国烹饪协会授予的“中华美食街”称号。 　　随着南滨路地产业的发展和CBD南区核心区域定位的确定,南滨路酒店业的发展已具备良好的基础和物质载体,喜来登，丽笙世嘉，皇冠假日等一批超五星酒店项目的建设，不仅契合南滨路商业发展的需要,也将为南滨路集聚更旺盛的人气。长江国际，喜来登金融中心等一批高档甲级写字楼与江南体育中心的落成，南滨路早已不只是美食一条街，带着自己依山傍水的独特地缘优势，它已发展成为融会展商贸、旅游、文化创意多种经济业态的多元化经济中心。 　　在重庆国际会议展览中心的影响下，南滨路的商务发展也愈发多样。在南岸区的规划中，南滨路将建设10万平方米的会展广场和50万平方米的黄葛渡商贸中心，加快酒店业“60颗星”工程和商业步行街建设，共同繁荣会展业。 　　同时，南滨路也将引进国内外知名百货品牌入驻,重点推出珊瑚水岸精品百货、喜来登国际名品商业城、洋世达旺角商业街、沃尔玛超市,南坪另一个商业零售中心呼之欲出。
== 高档宜居社区 ==
北靠南山，傍水长江，渝中半岛的繁华夜景尽收眼底。交通便利，商业繁荣，加上得天独厚的地理条件的使得南滨路上高档住宅小区星落密布，不仅有帝景名苑，海棠晓月，阳光华庭等高档社区，更有新落成的喜来登，长江国际等豪华公寓，和记黄埔地产投资的珊瑚水岸的售价更是高达两万五一平方米，创下了重庆之最。目前南岸是重庆最值得投资地区第一，在环境、价格的驱使下，重庆很多中高产阶级人群已经逐渐向南滨路移居，而南滨路也逐渐成为了重庆市民公认的继[[新牌坊|新牌坊]]之后的重庆第二富人区。
== 水陆风情神似塞纳河 ==
去过巴黎的人都对水情万种的塞纳河流连忘返，而南滨路其实与塞纳河在风景、文化上有许多相似之处，我们的目标是把南滨路打造成东方的塞纳河。”南滨路管委会的工作人员表示，现在他们已经列出了五大目标，来发展南滨路的水岸经济。 　　对于各自为政的零星渔港，南滨路将在海棠溪码头水域附近打造重庆最大的滨江渔港，与陆上的美食街相呼应，共同构成市民在南滨路的美食天堂。同时整合洋世达旅游码头、金港旅游码头、益航休闲码头、玄坛庙旅游船停泊点等资源，建成我市集中的旅游专用码头，形成游客就餐、观光、购物、上下船等一条龙服务，推动岸上、水上经济和谐发展。 　　以后坐船看夜景就并不只有朝天门一个去处了，南滨路上将发展起长江上游最大的游艇舰队，不仅可以提供给市民进行两江游，还将提供包船游等高端旅游项目。 　　重庆以码头文化著称，而现在越来越多的市民却对这一特色文化逐渐疏远，南滨路上将打造游船水上博物馆，游客们可以在水上博物馆里通过文字、画面、电动模型、灯光、音乐等了解过去的历史，感受重庆魅力无穷的码头文化。 　　南滨路将着力打造水岸经济，建滨江大渔港、游艇舰队、水上博物馆等。 　　到南滨路品美食看夜景，不仅是重庆人热衷的休闲方式，也是许多外地游人来重庆的必经之旅。除了吃和看还能到南滨路做什么，那就是玩了。记者从南滨路管委会获悉，南滨路将着力对现在的水域资源进行整合规划，发展旅游休闲码头和水域文化丰富的娱乐项目，打造水岸经济，体现黄金水域的巨大价值。
