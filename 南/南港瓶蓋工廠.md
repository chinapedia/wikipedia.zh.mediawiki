'''南港瓶蓋工廠'''全名為'''[[台灣省菸酒公賣局|台灣省菸酒公賣局]]瓶蓋工廠'''，位於[[台北市|台北市]][[南港區|南港區]]南港路二段13號。工廠原以製造各種高低壓瓶蓋、螺口鋁蓋及塑膠木栓蓋，供應公賣局所屬的14家酒廠包裝各種瓶蓋、軟木栓為主要業務。但2004年起即停止生產。停產後曾配合[[台北市都市再生前進基地推動計畫|台北市都市再生前進基地推動計畫]]作為展覽、音樂季、拍片場地。原預訂於2013年下半年配合[[臺北市南港區第三期市地重劃區|臺北市南港區第三期市地重劃區]]拆除<ref>[http://www.lda.taipei.gov.tw/ct.asp?xItem=38455827&ctNode=56997&mp=111011 南港區第三期市地重劃區]</ref>，但是由於本基地具有歷史與文化藝術的價值，已有部分民眾發起保存活動。

== 沿革 ==
前身為[[台灣日治時期|台灣日治時期]]之民營「國產軟木工業株式會社」所屬工廠，於1943年台灣酒壜統制會社與日本水產株式會社共同出資成立，與[[萬華區|萬華]]台北製糖所、[[華山1914|台北酒廠]]、[[松山菸廠|松山菸廠]]等同為依存[[縱貫線_(鐵路)|縱貫線鐵道]]而存在的產業遺跡之一，並同列為[[模板:台北市首都鐵道沿線產業遺產群|台北市首都鐵道沿線產業遺產群]]。

二戰後，工廠由[[台灣省專賣局|台灣省專賣局]]（今[[臺灣菸酒股份有限公司|臺灣菸酒股份有限公司]]）接管並改名為「木栓工廠」，並提供當地居民不少就業機會。1958年再度更名「台灣省菸酒公賣局瓶蓋工廠」，成為後來俗稱的「南港瓶蓋工廠」。2004年南港瓶蓋工廠結束營運。

2010年[[臺北市都市更新處|臺北市都市更新處]]提出「臺北市都市再生前進基地」推動計畫（Urban Regeneration Station；URS），將南港瓶蓋工廠規劃為台北都市前進再生基地URS13。近年來已在南港瓶蓋工廠規舉辦過大型展覽、[[音樂季|音樂季]]、社團夜遊、平面或動態影像拍攝等活動，並且為熱門拍片場地。曾進駐的單位有[[台北市電影委員會|台北市電影委員會]]、[[台北數位藝術中心|台北數位藝術中心]]及[[強心針多媒體文化|強心針多媒體文化]]等。

== 現況 ==
南港瓶蓋工廠空盪的廠房雖歷經多年棄置，廠區的建築仍保持相當完整。屋頂上的瓦片、挑高完整的連續空間、屋頂比例勻稱的樑柱結構、衍生在建物裡外的植栽與老樹、藝術家在剝落牆面上的畫作等，皆讓南港瓶蓋工廠在時間的積澱下，展現其獨特的韻味。

[[File:No-entry_sign_on_Nangang_Bottle_Cap_Plant_by_Land_Development_Agency_20150815.jpg|thumb]]土地開發總隊懸掛於南港瓶蓋工廠外圍鐵絲網上的禁止進入告示牌]]
南港瓶蓋工廠目前由[[財政部國有財產署|財政部國有財產署]]保管，並委託[[臺北市都市更新處|臺北市都市更新處]]管理，禁止民眾進入參觀，違者以《[[中華民國刑法|中華民國刑法]]》「侵入住宅罪」與「竊佔罪」移送法辦。根據[[台北市政府|台北市政府]]公告，南港瓶蓋工廠的拆遷期限（2013年8月30日）已公告確定。目前已有民眾發起保存南港瓶蓋工廠活動。
馬英九市長任內，因致力於[[北部流行音樂中心|北部流行音樂中心]]的建設案, 預計做土地重劃, 但隨即當選中華民國總統;於是郝龍斌市長扛起重責大任,市府與八十三位相關地權人士協商劃分, 稱為「南港第三期市地重劃」。

南瓶工廠十米道路恐成承包商商場通廊爭議:

2012年，郝龍斌時代核定的都市計畫中，為促進商業發展, 新增10米道路計畫。
同年通過「台北市南港區鐵路地下化沿線土地細部計畫案」，將南港瓶蓋工廠劃為特定商業區, 可以招商承包方式結合觀光與文創. 

[http://imgur.com/eUH8QEc 2012年南港瓶蓋規劃為「特定商業區」規劃圖]

民團提出質疑，南瓶東側已有兩條20米寬的興華路和興中路，可直接從忠孝東路通到環東快速道路，從瓶蓋工廠開一條僅有十米寬破壞南瓶原有建築之道路必要性頗受爭議.
當時民團指出, 原先規劃區中十米道路區通過屬潤泰集團承包之商業建築, 郝市長及其市府團隊需釐清此道路之方便性及經濟價值。

2018年10月24日，北市府、北市府產業局在上午舉行「南港瓶蓋工廠歷史建築及保留建物整修工程」開工典禮 ，南港瓶蓋工廠全區保留後,目前是北市府「[[東區門戶計畫|東區門戶計畫]]」中，為文創及新創產業軸帶計畫的策略基地。北市府產業局委託專業建築師事務所規劃設計並進行整修工程，修復項目包含建物結構補強，如：木構造部份的大跨距桁架撓度補強、蟲蟻防治、牆體裂縫處理及外牆整體劣化修復等，及新設門窗、必要之水電、基本照明、消防、衛生、排水、園區鋪面、景觀綠化、無障礙電梯等設備，工期共計360日曆天，工程預計於2019年10月正式修復完工。<ref>[https://tw.news.yahoo.com/%E5%8D%97%E6%B8%AF%E7%93%B6%E8%93%8B%E5%B7%A5%E5%BB%A0%E6%AD%B7%E5%8F%B2%E5%BB%BA%E7%AF%89%E6%95%B4%E4%BF%AE%E5%B7%A5%E7%A8%8B%E9%96%8B%E5%B7%A5-051347821.html 南港瓶蓋工廠歷史建築整修工程開工 自立晚報	2018年10月24日 下午1:13 yahoo奇摩新聞]</ref><ref>[https://udn.com/news/story/7323/3439477 柯文哲出席 「南港瓶蓋工廠」整修開工典禮 2018-10-24 12:11聯合報系攝影中心 記者林俊良╱即時報導 聯合新聞網]</ref>

==志工發起保留運動==
瓶蓋守衛隊志工到[[萬華|萬華]][[剝皮寮|剝皮寮]]文資審議會場外聲援，表示臺北市政府申辦2016[[世界設計之都|世界設計之都]]，曾在南港瓶蓋工廠取景拍宣傳影片，若無法保留南港瓶蓋工廠，無異是自打嘴巴。<ref>{{cite web |url=http://news.ltn.com.tw/news/local/paper/729791 |title=〈台北都會〉志工搶救 南港瓶蓋工廠部分保留 |publisher= 自由時報 |accessdate= 2013-11-12}}</ref>南港瓶蓋工廠和現為松山文創園區的松山菸廠，同屬台北市鐵道沿線產業文化象徵，極具歷史意義。
