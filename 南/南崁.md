'''南崁'''，是[[台灣|台灣]][[桃園市|桃園市]]地名，昔日為[[桃園臺地群|桃園平原]]的總稱，現常指稱跨越[[蘆竹區|蘆竹區]]及[[龜山區|龜山區]]兩區，是蘆竹區的行政、商業、工業重心。有[[南崁溪|南崁溪]]（青溪）穿過。

== 名稱 ==
地名來源眾說紛紜。荷治時期即有文獻紀錄「Lamcam」一名，語源為閩南人稱的「南崁」 （原住民則自稱部落為''Parricoutsie''），清代後成為桃園平地區域的總稱<ref>{{cite journal |author1=簡宏逸 |title=從Lamcam到南崁：荷治到清初南崁地區村社歷史連續性之重建 |journal=臺灣史研究 |date=2012-3 |volume=19 |issue=1 |page=1-28 |url=http://www.ith.sinica.edu.tw/quarterly_download.php?name=01+%E7%B0%A1%E5%AE%8F%E9%80%B83%E6%A0%A1%E5%AE%9A%E7%A8%BF0321.pdf&filename=13345409962.pdf}}</ref>。一說昔為[[平埔族|平埔族]]原住民[[龜崙族|龜崙族]]南崁社所在地，因而得名。另說[[鄭成功|鄭成功]]軍隊攻台時曾在南崁港停泊，由海上眺望該港地形為一斷崖（崖在[[閩南語|閩南語]]中稱為崁），南崁恰在斷崖之南，因以為名。（然而，正史中鄭成功不曾到過[[北台灣|台灣北部]]）

== 歷史 ==
[[台灣日治時期|日治時期]]1920年之前原有南崁區，屬[[桃澗堡|桃澗堡]]。1920年實施街-{庄}-制後，將原南崁區水汴頭庄併入[[桃園街|桃園街]]，蘆竹厝、南崁下、南崁廟口、南崁內厝等併入[[蘆竹庄|蘆竹庄]]，南崁頂庄併入[[龜山庄|龜山庄]]，延續至今造成南崁橫跨兩鄉之情況。其中蘆竹庄役場設於南崁下278番地，奠定行政中樞地位。（此為蘆竹庄誌說法，戰後之臺灣省通誌則認為初置於新興[[大字_(行政區劃)|大字]]，隨後再遷至南崁下。）

昔日龜山鄉南崁頂即今龜山區南上、南美、大坑3里。今日3里學區為自由學區，水力及電力設施亦與南崁一體等；[[林口線|林口線鐵路]][[南祥車站|南祥車站]]亦位於龜山區南上里，但距離蘆竹區公所不到300公尺。

南崁本以農業為主，近二十年來因為[[中山高速公路|中山高速公路]][[桃園交流道|桃園交流道]]（俗稱南崁交流道）的興建，再加上地近[[臺灣桃園國際機場|桃園國際機場]]，蘆竹鄉的人口從原本的5萬成長到今日的15萬，其中大部分的人口增加歸功於南崁地區的快速發展。經過許多建商於1990年代以來的不斷開發，南崁地區具備的生活機能已相當完整。台灣第一座綜合型[[購物中心|購物中心]][[台茂購物中心|台茂購物中心]]以及[[長榮航空|長榮航空]]總部皆位於南崁。

戰後初期，南崁大致涵蓋蘆竹鄉南崁、錦興、五福及龜山鄉南上、大坑等5村。今日因人口稠密，將南崁村細分為南興、南榮、南崁三村；錦興村細分為錦興、錦中、長興三村；五福村細分為五福、羊稠、營盤、瓦窯三村以及龜山鄉南上村與大坑村細分為南美、南上、大坑三村等共12村。

在2014年蘆竹鄉改制為直轄市一區後，所轄的村皆改成里，且原本的村又再細分成更多里，顯示南崁地區發展的快速。

== 區域範圍 ==
目前「南崁」的範圍大致涵蓋以下里：
*[[蘆竹區|蘆竹區]]：南崁、南興、順興、福昌、福興、蘆興、南榮、興榮、錦興、錦中、長興、五福、中山、福祿、羊稠、長壽、吉祥、營盤、營福、瓦窯等20里。
*[[龜山區|龜山區]]：南上、南美、大坑等3里。
<br />此外當地人也會把南崁區分為[[上南崁|上南崁]]與[[下南崁|下南崁]](亦或稱作[[南崁下|南崁下]])。<br />
<br />上南崁行政區域大致分為:五福、中山、福祿、羊稠、長壽、吉祥、營盤、營福、瓦窯、南上、南美等11里。<br />
[[下南崁|下南崁]]行政區域大致分為:長興、南崁、南興、順興、福昌、福興、蘆興、南榮、興榮、錦興、錦中等11里。
<br />也有一說是[[上南崁|上南崁]]從中正路以北截至中山路與仁愛路二段交界。
[[下南崁|下南崁]]則是中正路以南截至大竹與[[桃園市|桃園市]]交界。<br />

== 國一甲開發爭議 ==
因應桃園機場以及[[桃園航空城|桃園航空城]]的聯外需求，交通部國道新建工程局擬在[[桃園市|桃園市]]興建[[國道一號甲線|國道一號甲線。]]<br />路線從[[桃園市|桃園市]][[竹圍漁港|竹圍漁港]]附近的[[台61線|台61線]]起，行經[[蘆竹區|蘆竹區]]並於[[龜山區|龜山區]]銜接[[國道一號|國道一號]]，續經[[桃園煉油廠|桃園煉油廠]]東側，迄於[[桃園區|桃園區]]，全長約18.5公里，沿線設置6處交流道。<br />這條18.5公里的路除了將耗費549億元，且可能通過26個里、13萬人、3座山與中油煉油廠，恐衝擊生態與居民安全。

=== 改善交通？ 道路必要性受質疑 ===
目前[[南崁交流道|南崁交流道]]到[[五股交流道|五股交流道]]之間路段確實壅塞，甚至回堵到[[台四線|台四線]]，但[[國道一號甲線|國道一號甲線]]銜接到國一的大坑段，正是最易壅塞處。
此舉等於把車流導進「一條昂貴的停車場」；而且與台四線平行的[[桃林鐵路|桃林鐵路]]原址，已規劃配合油管路拓寬，可取代國一甲所規劃的功能。<ref>[http://e-info.org.tw/node/201535 航空城爭議未明 聯外道「國一甲」程序卻不停 居民怒吼快撤案]</ref>
