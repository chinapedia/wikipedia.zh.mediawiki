'''南京长江二桥'''为越[[长江|江]]公路桥，位于[[南京长江大桥|南京长江大桥]]下游11[[公里|公里]]处，全长21.337公里，桥下最大通航净高24米。大桥由南、北汊大桥和南岸、[[八卦洲|八卦洲]]及北岸引线组成（亦简称“二桥一路”），总投资33.5亿元。<ref name=":0">{{Cite web|url = http://www.nj2q.com/about.asp|title = 工程简介|accessdate = 2016-02-28|author =|date =|publisher = 南京长江第二大桥有限责任公司}}</ref>其中，南汊大桥为双塔双索面钢箱梁斜拉桥，桥长2938米，跨径布置为58.5m + 246.5m + 628m + 246.5m + 58.5m ，以其628米主跨在其建成的2001年成为继[[日本|日本]][[多多罗大桥|多多罗大桥]]、[[法国|法国]][[诺曼底大桥|诺曼底大桥]]之后[[世界斜拉桥列表|世界第三大斜拉桥]]。北汊大桥为[[钢筋混凝土|钢筋混凝土]]预应力连续箱梁桥，桥长2172米，主跨为3×165米，桥长2212米，桥面宽32米。全线采用6车道[[高速公路|高速公路]]标准。<ref name=":0" /><ref name=":1">{{Cite news|url = http://news.sohu.com/35/82/news144418235.shtml|title = 南京长江二桥创多项中国桥梁建设“之最”|last = |first = |date = 2001-03-26|work = 新华社|accessdate = 2016-02-28|archive-url = https://web.archive.org/web/20160305123613/http://news.sohu.com/35/82/news144418235.shtml|archive-date = 2016-03-05|dead-url = no}}</ref><ref>{{Cite news|url = http://news.sohu.com/46/79/news144417946.shtml|title = 疑是彩虹天外来——写在南京长江二桥通车时|last = |first = |date = 2001-03-26|work = 扬子晚报|accessdate = 2016-02-28|archive-url = https://web.archive.org/web/20160305130026/http://news.sohu.com/46/79/news144417946.shtml|archive-date = 2016-03-05|dead-url = no}}</ref>

该桥于1997年10月6日正式开工，2001年3月26日建成通车，[[江泽民|江泽民]]题写了桥名。<ref name=":0" /><ref name=":1" /><ref>{{Cite news|url = http://news.sohu.com/98/78/news144417898.shtml|title = 南京长江二桥今天通车 翻开我国桥梁建设史新一页|last = |first = |date = 2001-03-26|work = 扬子晚报|accessdate = 2016-02-28|archive-url = https://web.archive.org/web/20030304123301/http://news.sohu.com/98/78/news144417898.shtml|archive-date = 2003-03-04|dead-url = no}}</ref>。大桥由[[南京市政府|南京市政府]]和[[马来西亚|马来西亚]]成功集团合资修建，总投资32亿元人民币。
