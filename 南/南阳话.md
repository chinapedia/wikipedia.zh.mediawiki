'''南阳话'''是[[河南省|河南省]][[南阳市|南阳地区]]的方言，[[中原官话|中原官话]]的分支，基本上与[[河南话|河南话]]一样，但也有自己的特色。

由于南阳地大人多，各县之间方言也有小小差异。虽然和普通话的差异性不大，但中老年人很难再学会标准的普通话。

== 使用人口 ==
包括现在南阳地区的1100万人口，及周边县市的部分地区，估计总的使用人口在1500万人左右。

== 语法 ==
基本上与[[普通话|普通话]]近似。但有较多的语气助词。如“有没有干什么？”通常答为“木有住哩！”、“冇啊！”。

一般对话以短句为主，长句较少见。存在不少单字句、两三字句，比较典型的如“zhuá li？”（有什么事吗？）。

== 发音 ==
南阳话与普通话相比，发音方面较突出的特点是：三声上声字（如“马”）发音无明显转折现象，趋近一声阴平，但调位比一声字（如“妈”）稍高。另外，一声阴平字（如“妈”）的南阳话发音调位，相较普通话偏低。

和很多北方方言类似，儿化现象也很普遍。如：盆儿、罐儿。

声母 j, q, x 在南阳话中很多时候趋近于 z, c, s，能够区分尖音团音，如：
jing → zing（如“妖精”的“精”）, jin → zin（如“浸湿”的“浸”）；

qing → cing（如“蜻蜓”的“蜻”）, qin → cin（如“亲戚”的“亲”）；

xing → sing（如“星星”的“星”）, xin → sin（如“相信”的“信”）。

另外，
ji → ㄗ丨（如“挤眉弄眼”的“挤”），
xi → ㄙ丨（如“媳妇”的“媳”）
注：汉语拼音方案中的 zi、si 分别代表 ㄗ、ㄙ（无韵母，整体认读音节），而此处表示这两个声母和韵母相拼的音节，因此这里使用注音符号表示，避免误解。
