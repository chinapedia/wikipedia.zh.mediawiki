'''香港南北行公所'''（{{lang-en|'''Nam Pak Hong Union'''}}），現稱[[南北行商會|南北行商會]]，是[[香港|香港]]第一個[[華人|華人]][[商會|商會]][[组织_(社会学)|組織]]，會址現在[[香港島|香港島]][[上環|上環]][[文咸街|文咸西街]]。

==歷史==
南北行公所由[[南北行|南北行]]龍頭商號元發行[[暹羅|暹羅]]潮籍[[高滿華|高滿華]]聯同[[潮州|潮籍]]的[[陳煥榮|陳煥榮]]（乾泰隆行） 、[[陳殿臣|陳殿臣]]（元發行）；[[廣東|粵籍]]的[[招雨田|招雨田]]（廣茂泰行）、[[馮平山|馮平山]]（兆豐行）；[[福建|閩籍]]的[[胡鼎三|胡鼎三]]、[[吳理卿|吳理卿]]等華商合議，於1868年成立，用以排除不同鄉籍華商間的矛盾和糾紛，並維持南北行各業之間的公平競爭<ref>{{citeweb |language =zh-hk |url = http://www.am730.com.hk/column-33048 |title =  香港人香港史 - 劉智鵬 高滿華—南北行的元祖(1) |work =''[[am730|am730.com.hk]]'' |accessdate =2015-02-20}}</ref>。初時稱為「公所」，後改稱「南北行公所」。公所成立後，推選招雨田為第一屆主席。

在1920年，訂出《南北行例》，規定行內守則，聘更練維持[[治安|治安]]，及負責[[滅火|滅火]]消防等。

南北行公所位於文咸東街，正門面向文咸西街，門框的橫匾刻有由[[順德|順德]]陳如柏書寫「公所」二字，旁有「同治八年南北行建」等字，門聯：「利藪南州萃、恩波北闕深」。在1952年，南北行公所由一幢兩層高的紅磚屋改建成一幢樓高四層的大廈。1997年與地產商合作再重建，公所與相鄰4幢樓宇重建成一幢高層大廈，於2000年落成入伙。公所擁有大廈其中四層，五樓用作會址，會址自成立一直沿用至今<ref>{{citeweb |language = zh-hk |url = http://www.amo.gov.hk/b5/trails_sheungwan1.php?tid=33 |title = 南北行公所舊址 |work = ''[[古物古蹟辦事處|amo.gov.hk]]'' |accessdate =2015-02-20}}</ref><ref>{{citeweb |language = zh-hk |url = http://202.55.1.84/news/10/08/12/XY-1294904.htm |title = 南北行公所/ 陳天權 |work = ''[[大公報|takungpao.com]]'' |accessdate = 2015-02-20 |deadurl = yes |archiveurl = https://web.archive.org/web/20150221101900/http://202.55.1.84/news/10/08/12/XY-1294904.htm |archivedate = 2015-02-21 }}</ref>。
