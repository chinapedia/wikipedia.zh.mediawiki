'''南京大校场机场'''，位于[[江苏省|江苏省]][[南京市|南京市]][[光华门|光华门]]外[[七桥瓮|七桥瓮]]南，[[雨花台区|雨花台区]]中和桥路142号，距离南京新街口6.2公里，北、東、南三面為[[秦淮河|秦淮河]]水系围绕，总面积2.72平方公里{{r|南京交通志|南京市誌}}。曾为军民合用机场，后为空军军用机场。2015年7月30日关闭。

==历史==
大校场位于七桥瓮以南，[[大明路_(南京)|大明路]]以东，该区域面积约900亩。1367年[[明故宫|明故宫]]建成后，[[明太祖|明太祖]]朱元璋在宫城以南建起多处兵营，以保卫皇宫及都城，并在[[响水河|响水河]]和[[秦淮河|秦淮河]]之间的平地上建成大教场，设阅兵台，用于练武阅兵。此后“大教场”一名逐渐演变为“大校场”。大校场建成后，不但附近兵营官兵前来操练，而且宫城内的卫兵每日清晨三更出[[正阳门_(南京)|正阳门]]，经过兵桥、七瓮桥来到大校场操练。后来明朝还在此地设立若干军事设施和军事机构。据传说，当时大校场内还设有两个点将台，每个占地面积十三、四亩，高度一丈多，其中一个是供将领操练用，一个专供皇帝阅兵及选拔高级将帅用。[[清朝|清朝]]将明朝的宫城改成“驻防城”，由[[江宁将军|江宁将军]]率“[[八旗兵|八旗兵]]”驻守，大校场成为清军军事基地，[[康熙帝|康熙帝]]和[[乾隆帝|乾隆帝]]六次下江南，每次都在此阅兵。[[太平天国|太平天国]]时期，大校场的各类建筑毁坏殆尽，变成荒地<ref name=xhbyw>{{Cite web |url = http://js.xhby.net/system/2015/08/01/025726230.shtml |title = 86岁的大校场机场前天凌晨正式关闭 |author =  |date =2015-08-01  |publisher = 新华报业网 |language =  |accessdate =  |quote =  }}</ref>。

1929年，[[国民政府|国民政府]]军政部航空署征收大校场700余亩土地，作为军用训练场，并设立当时全国最大的靶场。1931年4月5日，国民政府在大校场建立航空学校，同年12月2日学校迁往[[杭州|杭州]][[笕桥机场|笕桥机场]]。1934年，国民政府将大校场辟为军用飞机场，设一条土质跑道<ref name=xhbyw/>{{r|南京交通志}}。1935年，国民政府航空委员会扩建大校场飞行场，长1300米，宽1200米。1936年，国民政府继续扩建大校场飞行场{{r|南京交通志}}。

1937年[[抗日战争|抗日战争]]全面爆发，1937年12月[[侵华日军|侵华日军]]占领南京，大校场机场被日军占用<ref name=xhbyw/>{{r|南京交通志}}。1939年，侵华日军对大校场机场跑道加以改建{{r|南京交通志}}。

1945年，国民政府光复南京后，航空委员会整修大校场机场，跑道延长400米{{r|南京交通志}}。1947年夏，[[美国|美国]]特使[[乔治·卡特利特·马歇尔|马歇尔]]的座机在大校场机场降落时损坏。国民政府乃决定彻底改造大校场机场。1947年按[[国际民航组织|国际民航组织]]B级标准在旧跑道（土跑道）南侧新建一条2200米长、45米宽、0.3米厚的水泥混凝土跑道，1948年4月29日完工。1948年，在跑道北侧东端300米处建造一座候机楼，面积474平方米<ref name=xhbyw/>{{r|南京交通志}}。

1949年1月21日下午，国民党政权在[[第二次国共内战|第二次国共内战]]中已风雨飘摇，身披黑色斗篷的[[蒋中正|蒋中正]]驱车来到大校场机场，登上[[美龄号|“美龄”号专机]]飞离南京<ref name=xhbyw/>{{R|中央社}}{{R|臺灣蘋果日報}}。

1949年4月23日，[[中国人民解放军|中国人民解放军]]占领南京，[[华东军区|华东军区]]空军接管大校场机场。此后新建混凝土滑行道。1954年，南京空军扩建大校场机场，跑道加宽至60米，两端设安全道，新建远近归航台，增设夜航灯光装置{{r|南京交通志}}。1956年7月，[[明故宫机场|明故宫机场]]关闭，南京民航从明故宫机场迁至大校场机场，与空军合用，大校场机场由此成为军民合用机场<ref name=xhbyw/>{{r|南京交通志}}。1959年，扩建候机楼1750平方米，改建停机坪6300平方米，新建停机坪5585平方米。1971年，因停机坪狭小，建成停机坪、滑行道等4.24万平方米{{r|南京交通志}}。1972年8月9日，开工新建候机楼，包括候机厅、贵宾休息室、餐厅及航行、通信、气象等单位业务用房总面积3826平方米，由[[杨廷宝|杨廷宝]]设计。1973年9月29日竣工{{r|南京市誌}}。

1984年6月25日，[[江苏省人民政府|江苏省人民政府]]提交《关于请求改造南京大校场机场的报告》。1985年2月27日，江苏省人民政府、[[南京军区空军|南京军区空军]]提交《关于报送扩建南京飞机场的会议纪要和请求早日核批的报告》。1985年6月19日，[[中华人民共和国国务院|国务院]]、[[中央军事委员会|中央军委]]《关于扩建大校场机场的批复》（国函〔1985〕91号）下发，批复同意扩建大校场机场，机场按起降[[MD-82|MD-82]]型飞机的技术要求扩建，跑道基础可按起降[[波音747|波音747]]型飞机的要求处理，机场扩建后产权仍属空军<ref>{{Cite web |url = http://www.gov.cn/xxgk/pub/govpublic/mrlm/201308/t20130823_66266.html |title = 国务院、中央军委关于扩建大校场机场的批复 |author =  |date = 1985-06-19 |publisher = 国务院、中央军委 |language =  |accessdate =  |quote =  }}</ref>。

1988年，在国内候机楼西侧新建国际旅游包机候机厅，总面积1717平方米的。1988年8月23日开工，1989年1月启用{{r|南京市誌}}。1991年11月10日，开工扩建候机楼及其配套工程6182.52平方米，1992年8月15日竣工。1991年10月15日，开工新建2个MD-82飞机停机坪及其场道工程4.2万平方米，1992年5月25日竣工{{r|南京交通志}}。

1992年7月31日，从南京大校场机场飞往[[厦门国际机场|厦门国际机场]]的[[中國通用航空|中國通用航空]][[中國通用航空2755號班機空難|2755號班機滑跑起飞时冲出跑道]]，造成机上126人中108人遇难、18人受伤{{R|新华日报2008|aviation-safety|南京交通志}}。

随着[[南京禄口国际机场|南京禄口国际机场]]在1997年6月28日通航，1997年7月1日大校场机场结束了其民航机场的作用，此后仅为军用机场<ref name=xhbyw/>{{R|中安在線}}{{R|南京市誌}}。

自從1934年大校場機場落成後，總共經歷超過十次擴建。1999年，機場已經擁有一條長2,200米、寬60米、厚46厘米的跑道，能夠承載[[波音737|波音737]]、[[波音757|757]]起降。機場總佔地面积{{convert|2720000|m2}}{{R|南京市誌}}。

随着南京市城市建设发展，机场逐渐被城市建成区包围。2003年，军地双方启动了空军南京机场迁建前期工作。2006年1月，确定了新机场场址。2009年1月，[[中华人民共和国国务院|国务院]]、[[中央军事委员会|中央军委]]正式批复同意迁建空军南京机场<ref name=guanbi/>。2012年8月，空军南京新机场建设工程开工<ref name="新浪"/><ref name=guanbi/>。2015年7月，空军南京新机场[[六合马鞍机场|南京马鞍国际机场]]建成运行，根据军地双方协议，新机场建成后，南京大校场机场大部分土地将移交给南京市人民政府<ref name=guanbi/>。2015年7月30日凌晨，南京大校场机场正式关闭<ref name=guanbi>{{Cite web|url = http://www.njdaily.cn/2015/0731/1180839.shtml|title = 空军南京新机场建成投运 原机场昨正式关闭|author = |date = 2015-07-31|publisher = 南报网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20151229090709/http://www.njdaily.cn/2015/0731/1180839.shtml|archive-date = 2015-12-29|dead-url = yes}}</ref><ref>{{Cite web |url = http://news.163.com/15/1123/16/B94BDDUF00014AED.html |title =  空军徐州机场迁建工程奠基 原址或成市中央活动区|author =  |date = 2015-11-23  |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>{{R|南京日報}}{{R|中央社}}。

==未来发展==
2012年，南京市规划局编制了《南京红花-机场跑道两侧地区概念规划》，规划将机场西、中、东段分别建设为新媒体创意区、酒店购物演艺区和电影音乐文化主题区。原机场飞行区将建设成[[神机营|神机营]]广场，同时为保证原机场飞行区的完整，不建设与东西向跑道平交的南北向道路，所有南北向道路或下穿跑道、或在跑道处被截断<ref>{{cite web|title=南京红花-机场跑道两侧地区概念规划|url=http://www.njghj.gov.cn/PortalIPS/Library/DefaultSite/bbd35aea-a92c-4c5e-9ed8-653aeb8b17e7/fcf0b002-0c12-49d0-94fd-2bea2219ad76.pdf|publisher=南京市规划局|date=2012-03|access-date=2017-03-13|archive-url=https://web.archive.org/web/20170313214627/http://www.njghj.gov.cn/PortalIPS/Library/DefaultSite/bbd35aea-a92c-4c5e-9ed8-653aeb8b17e7/fcf0b002-0c12-49d0-94fd-2bea2219ad76.pdf|archive-date=2017-03-13|dead-url=yes}}</ref>。

《南京市秦淮区总体规划（2013-2030）》将大校场机场片区定位为建设生态、科技型项目、高端服务业的“智慧新城”，规划人口10万人。其中，机场跑道已经列入历史文化遗产保护规划，规划建成“跑道记忆轴”；机场跑道南面大部分规划为二类居住用地，限高60米以上，最靠近跑道的建筑，规划高度超过100米；机场跑道北面规划为商贸中心，规划高度超过100米；外秦淮河以南，七桥瓮附近的地块，多规划为居住用地<ref name=xhbyw/>。

2016年[[中华人民共和国住房和城乡建设部|住建部]]发出《关于进一步推进工程总承包发展的若干意见》，南京市南部新城开发建设管委会决定借鉴外地经验，对于红花-机场片区实施{{link-en|设计，采购和施工|Engineering, procurement, and construction|EPC模式}}的工程模式，以加快建设进度并节约成本。预计机场片区基础设施建设将于2017年开始，建设内容包括各类主次干道、支路网、跨河桥，各类综合管廊，机场跑道整体地下空间建设以及[[响水河|响水河]]、机场河整治等，总投资约117.5亿元人民币<ref>{{cite news|title=红花—机场片区总投资超百亿元 三大片区均由央企中标|url=http://nj.house.ifeng.com/detail/2017_02_27/51013930_0.shtml|publisher=凤凰房产|date=2017-02-27}}</ref>。
