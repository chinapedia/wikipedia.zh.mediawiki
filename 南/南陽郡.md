'''南阳郡'''，[[中国|中国]]古[[郡|郡]]名。[[秦国|秦国]]于前272年设置，辖境在今[[河南省|河南省]]、[[湖北省|湖北省]]等地。其后辖境多有变化。[[唐朝|唐朝]][[乾元_(唐朝)|乾元]]时，全国废郡改州，改为[[邓州_(隋朝)|邓州]]。

== 秦代 ==
秦将[[白起|白起]]攻[[楚国|楚]]占宛，[[秦昭王|秦昭王]]三十五年（前272年）设南阳郡，[[治所|治所]]在[[宛县|宛县]]（今[[河南省|河南省]][[南阳市|南阳市]]）。辖境相当今南阳市地域，及鲁山、叶县、舞阳、栾川的一部分和湖北的随州、枣阳一带。

== 三国两晋南北朝 ==

三国时，南阳为曹魏所有，隶属荆州。晋，南阳曾为[[南阳国|南阳国]]，辖十四县，都宛。东晋，南北朝仍为南阳郡。

南梁，[[天监|天监]]十年（511年）析[[武陵郡|武陵郡]][[侨置|侨置]]南阳郡，以安置南阳郡南下的流民。治建昌县（今[[湖南省|湖南省]][[辰溪县|辰溪县]]西北），属卢州。隋开皇九年（589年），废南阳郡置寿州。 

== 隋唐 ==
[[隋朝|隋朝]][[开皇|开皇]]三年（公元583年）废南阳郡为邓州，[[大业_(年号)|大业]]三年（607年）复置南阳郡，领县八，范围大大缩小。

[[唐代|唐代]]，唐初复废南阳郡为[[邓州_(隋朝)|邓州]]。[[天宝_(唐朝)|天宝]]（公元742年）到[[至德_(唐朝)|至德]]二年（757年）置以邓州为中心的南阳郡，郡治邓州穰城（今河南省[[邓州市|邓州市]]）。[[乾元_(唐朝)|乾元]]元年（758年）又改南阳郡称邓州，此后不再设置。
