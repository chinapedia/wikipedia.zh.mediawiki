[[南京|南京]]有2500多年的建城史，[[中国历史|中国历史]]上曾有十二个朝代定都于此。南京一名得于[[明朝|明朝]]，[[永乐迁都|永乐迁都]]后称[[应天府_(明朝)|应天府]]为南京。清初，改应天府为[[江宁府|江宁府]]。民国初年，改江宁府为[[南京府|南京府]]，南京正式成为[[行政区|行政区]]名。1927年，定名为[[南京特别市|南京特别市]]，是为[[中华民国首都|中华民国首都]]。[[中华人民共和国|中华人民共和国]]成立后，[[南京市行政区划|南京市行政区划]]经过多次调整，随着[[溧水县|溧水县]]、[[高淳县|高淳县]]在1983年划入南京市，南京辖区达到目前规模。

== 先秦时代 ==
[[南京猿人|南京猿人]]头盖骨和相关遗迹表明，距今约60～100万年前南京就有了古人类的活动。6000年前南京就出现了原始村落，在1950年代发掘的北阴阳营遗址年代约为公元前4000年至前3000年，为长江下游地区[[新石器时代|新石器时代]]文化类型——[[北阴阳营文化|北阴阳营文化]]，南京地区早期文化遗存还有200多处，著名的有[[点将台文化|点将台文化]]和[[湖熟文化|湖熟文化]]。

南京古为吴地，位于“[[天下|天下]][[九州_(中国)|九州]]”之一的[[扬州_(九州)|古扬州]]地域。公元前570年前后，[[吴国|吴国]]於[[长江|长江]]南北设置棠邑和濑渚邑，在今南京[[六合区|六合区]]和[[高淳区|高淳区]]境内，为南京地区建置之始。前495年，传说[[吴国|吴]]王[[夫差|夫差]]在今[[南京朝天宫|朝天宫]]筑[[冶城_(南京)|冶城]]，聚集了一批铸造铜器的作坊。[[周元王|周元王]]四年（前472年），[[越國|越]]王[[勾践|勾践]]灭吴，[[范蠡|范蠡]]於今[[中华门|中华门]]南的[[长干里|长干里]]一带筑[[越城_(南京)|越城]]。前333年，[[楚威王|楚威王熊商]]灭越，埋金以镇“王气”，并於石头山（今[[清凉山|清凉山]]）筑[[金陵邑|金陵邑]]，故南京别称“金陵”。

== 秦汉时代 ==
[[秦始皇|始皇帝]]廿六年（前221年），秦并天下。棠邑为棠邑县属[[九江郡|九江郡]]。

始皇帝三十七年（前210年），秦始皇东巡[[会稽山|会稽山]]至丹阳，置[[丹阳县|丹阳县]]。於江乘渡大江，因此置[[江乘县|江乘县]]（於今南京[[栖霞区|栖霞区]]内）。同时，改金陵邑为[[秣陵县_(秦朝)|秣陵县]]，属[[会稽郡|会稽郡]]。同时，贯穿境内连接都城咸阳和会稽郡的驰道也开始修筑。

汉六年（前201年），册封[[刘贾|刘贾]]为荆王，以[[淮河|淮河]]以东52县属之。并析会稽郡西境置鄣郡，并辖秣陵、丹阳、江乘等县，属荆国。同年，册封[[陈婴|陈婴]]为棠邑侯，封地为棠邑。五年后，英布谋反，刘贾因不敌叛军，薨於[[洪泽县|富陵]]。

汉十二年（前195年），册封刘濞为吴王，尽领荆国故地。

汉武帝元狩六年（前117年），改棠邑为堂邑。元鼎元年（前116年），武帝姑姑[[窦太主|窦太主]]逝世，其子堂邑侯[[陈须|陈须]]於服丧期内淫乱，兄弟争财，论罪当死，故而自杀，死后国除。

== 五代宋元：金陵的中兴 ==

[[南唐|南唐]]（937年—975年）在金陵建都，称江宁府。南唐[[金陵城|金陵城]]的范围，南到今中华门，西到今水西门和汉中门，北到今珠江路北门桥，东到大中桥。今天的内桥就是南唐皇宫大内正门前的桥梁，[[中華路_(南京)|中华路]]就是当时的御道和城市的中轴线。宋朝以后南京城市就是在南唐金陵城的基础上逐步扩建而成。北宋降江宁府为昇州，后又升为江宁府，南宋改称[[建康府|建康府]]，为江南东路首府。[[元朝|元朝]]时为[[集庆路|集庆路]]。

== 明清：大一统帝国的南方中心 ==
元至正十一年，[[红巾军|红巾军]]起义爆发。1356年，原为濠州红巾军[[郭子兴|郭子兴]]部的[[朱元璋|明太祖朱元璋]]攻占集庆，改集庆为“应天府”，以此为基地。他採納謀士朱升「高築牆，廣積糧，緩稱王」的建議，先后消灭[[陈友谅|陈友谅]]、[[方国珍|方国珍]]、[[张士诚|张士诚]]等势力。1368年正月，朱元璋在應天稱帝，建立[[明朝|明朝]]，定都于此。当年，明军北伐攻入[[大都|大都]]。建都之后，朱元璋动用了巨大人力，建成全长96华里、世界最长，也是中国最高的[[南京城墙|城墙]]，有将近三分之二的城墙保留至今；因而明代南京城面积达到南唐金陵城的三倍（约42平方公里）：西南部市民区即南唐金陵城，东部是新建的皇宫和官衙，西北部则分布有不少军营。永樂十九年，[[明成祖|明成祖]][[遷都|遷都]][[北京|北京]]（北京皇宫及官衙均仿照南京制度），将应天府改为南京（留都），仍设六部等机构。

明代的南京城是当时世界上最大的都城，人口高达120万，其次为巴黎<ref>《南京历史》，第53页——江苏少年儿童出版社。</ref>。当时修建的[[大报恩寺_(南京)|大报恩寺]]为中国古代最高建筑。当时的来华传教士[[利玛窦|利玛窦]]这样评价南京：“在中国人看来，论秀丽和雄伟，南京这座城市超过世上所有其他的城市，而且在这方面，确实或许很少有其他城市可以与它匹敌或胜过它。它真的到处都是殿、庙、塔、桥，欧洲简直没有能超过这些的类似建筑。在某些方面，它超过我们的欧洲城市……有广阔的园林、山和树林，相间着湖泊，然而城中居民区仍然占有它的绝大部分。”<ref>利玛窦：《利玛窦中国札记》</ref>

1644年，[[李自成|李自成]]攻陷北京，[[崇祯帝|崇祯帝]]自杀，福王[[朱由嵩|朱由嵩]]在南京即位，即[[弘光|弘光]]帝。1645年，南京被[[清|清]]兵攻陷，降为江宁。

[[文件:Marble_Boat_in_Nanjing.jpg|thumb]]内的[[石舫|石舫]]，[[乾隆帝|乾隆帝]]题匾额“不系舟”]]

清初，江宁（南京）为[[江南省|江南省]]省府，以后直到清末一直是[[两江总督|两江总督]]（统辖江苏、安徽、江西三省）驻地。康熙六年江苏、安徽分别独立建省后，[[江苏巡抚|江苏巡抚]]和江苏布政使驻扎在[[苏州|苏州]]，[[两江总督|两江总督]]和安徽布政使驻扎在南京，[[安徽巡抚|安徽巡抚]]驻扎在[[安庆|安庆]]。1780年，安徽布政使迁往安庆后，在南京又设立[[江宁布政使|江宁布政使]]，管辖江宁府和江苏省长江以北的三府二州一厅：[[扬州府|扬州府]]、[[淮安府|淮安府]]、[[徐州府|徐州府]]、[[海州直隶州|海州直隶州]]、[[通州直隶州|通州直隶州]]、[[海门厅|海门厅]]。

== 晚清：天国倾覆与近代的黎明 ==

== 中華民國 ==
{{main|南京市 (中華民國)}}
[[File:Presidential_Palace_at_Nanjing_Sun's_Office.JPG|200px]]
[[File:Border_of_Nanjing.jpg|thumb]]
[[File:Nanking_Congress_building.jpg|thumb]]，建成于1936年]]

1911年([[宣統|宣統]]三年)辛亥革命爆發，1912年1月1日（[[民国|民国]]元年）[[孫中山|孫中山]]在[[南京|南京]]宣佈成立[[中华民国|中华民国]]，出任臨時大總統，以南京為中華民國首都。2月溥儀在北京宣佈退位，孫中山在南京辭去臨時大總統一職，由[[袁世凱|袁世凱]]繼任，為防止袁世凱闊權，孫中山頒布[[中華民國臨時約法|中華民國臨時約法]]以為規範。同年，改[[江宁府|江宁府]]为[[南京府|南京府]]。

北洋軍閥混戰期間，南京曾為軍閥[[孫傳芳|孫傳芳]]的大本營，成為號令華東諸省的中心，1926年冬，北伐的國民革命軍擊敗孫傳芳部隊，取回南京。1927年（民国16年），[[中华民国|中华民国]]奠都南京，同年置南京特別市。1929年（民国18年）6月1日，安葬孫中山於南京[[中山陵|中山陵]]，置首都特別市。1930年（民国19年）更改為院轄市（[[直轄市|直轄市]]）。1931年(民国20年)，六月一日公布《[[中华民国|中华民国]]训政时期约法》，其中第五条明定“中华民国国都定于南京”，確定南京在法律上做為中國首都的地位。1929年－1937年间，南京进行了大规模的首都建设，基本形成了近代南京的城市格局。

1929年（民國18年），國民政府為了迎接由北京至南京奉安的[[孫中山|孫中山]]靈柩，趁勢在南京進行建設，包括在長江口修建新碼頭(中山碼頭)，並將移靈經過的道路進行拓寬與鋪設柏油，形成今天的[[中山北路_(南京)|中山北路]]、[[中山路_(南京)|中山路]]、[[中山东路_(南京)|中山东路]]，並大量種植法國梧桐樹綠化市容。此后直到1937年的数年间，南京陆续建成中正路（中山南路）、太平路、白下路、汉中路、中华路、雨花路、山西路、国府路、热河路、珠江路、[[莫愁路|莫愁路]]、[[昇州路|昇州路]]、上海路、广州路、以及[[颐和路|颐和路]]一带住宅区道路，形成了南京的近代干道系统。

1937年（民国26年）7月7日[[抗日战争|抗日战争]]全面爆發，8月13日淞滬會戰展開，日軍開始對南京進行大規模的空襲轟炸，市容與城市建設受到嚴重破壞，11月国民政府遷都至[[重庆|重庆]]，12月8日日軍進犯南京城，由於指揮官[[唐生智|唐生智]]臨陣脫逃，中國軍隊在南京的防禦潰散，12月12日南京沦陷。12月13日開始後的大約六個星期時間，入侵的日本军队在南京进行了[[南京大屠殺|大屠杀]]，至少30万人遇难，形成人類歷史的污點。南京淪陷後，侵華日軍將南京做為扶植[[漢奸|漢奸]][[傀儡政權|傀儡政權]]的根據地，1940年(民國29年)[[汪精衛|汪精衛]]在南京成立「中華民國國民政府」，替日本人統治大前方的淪陷區。

1945年(民國34年)8月15日日本投降，中國抗戰勝利，不久之後國民政府收復南京。1946年（民国35年）5月5日，[[中华民国|中华民国]][[国民政府|国民政府]]还都南京。由於不久之後国共内战全面爆发，國府已無充足的財政在这一时期的南京大规模的建设活动，南京不復抗戰前的盛況榮景。

同年11月，中華民國[[制憲國民大會|制憲國民大會]]在南京的[[國民大會堂|國民大會堂]](今人民大會堂)召開。原本憲法草案定國都為南京，但審查會及第一讀會決定改為[[北平|北平]]。經大會主席團代表蔣中正出面說明，主張國都地點，不必定於憲法，始將第一讀會通過的「國都定於北平」一條予以刪除<ref>荊知仁，《中國立憲史》，聯經出版公司，1984年12月，461頁</ref>。12月25日[[中華民國憲法|中華民國憲法]]三讀通過，1947年(民國36年)1月1日公佈，同年12月25日正式實行。自此，南京失去中華民國法定國都的地位。

1947年11月21日至23日依憲法規定舉行[[1947年國民大會代表選舉|第一屆中華民國國民大會代表選舉]]，1948年1月21日至23日則舉行[[1948年中華民國立法委員選舉|第一屆中華民國立法委員選舉]]。1948年3月29日於南京報到召開行憲後的[[國民大會|國民大會]]，4月在南京選舉出[[蔣中正|蔣中正]]為第一屆[[中華民國總統|中華民國總統]]，[[李宗仁|李宗仁]]為第一屆中華民國副總統，5月20日在[[南京總統府|南京總統府]]舉行就職典禮。1月間選出的立法委員則於5月8日自行集會於南京國民大會堂，並於5月18日正式開議。

1948年(民國37年)底，國共內戰情況對國民黨趨於不利的態勢，1949年1月國府[[行政院|行政院]]及各單位開始遷往廣州，僅[[中華民國總統府|總統府]]仍在南京。1月26日，中共政权攻占长江北岸的[[六合县|六合县]]县城（当时六合县属[[江蘇省_(中華民國)|江苏省]]第五行政督察区，不属[[南京特别市|南京特别市]]）。4月21日，[[中国人民解放军|中国人民解放军]]发起[[渡江战役|渡江战役]]，渡過長江。4月23日，解放军攻陷[[南京城|南京城]]，24日凌晨，[[中国人民解放军第35军|解放军第35军]]攻占[[南京总统府|南京总统府]]。国民政府南迁[[广州|广州]]，後又遷[[重慶|重慶]]、[[成都|成都]]最後迁抵[[台北|台北]]。

== 中華人民共和国 ==

1949年10月1日，[[中华人民共和国|中华人民共和国]]成立，南京成为中央直辖市，由中央人民政府直辖，1950年1月改由新成立的华东军政委员会领导，仍为中央直辖市，同时政务院设有驻宁办事处。1952年江苏建省，南京降为[[江苏省|江苏省]]省会。由于南京民国首都的身份，在新政权统治下，发展受到压抑。

1968年，连接[[津浦铁路|津浦铁路]]与[[沪宁铁路|沪宁铁路]]的南北交通要津[[南京长江大桥|南京长江大桥]]建成。南京的重要性又逐渐被领导层所重视，开始在南京兴建一些大中型企业，如南京炼油厂。

1994年中央政府升格為副省級城市，一直至今。
