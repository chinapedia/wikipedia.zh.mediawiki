'''重庆市南桐礦區'''，[[中國|中國]][[四川省|四川省]][[重慶市|重慶市]]曾经下轄的一個[[市轄區|市轄區]]，1955年1月21日获[[中华人民共和国国务院|国务院]]批复设立{{efn|name=a1}}，1993年4月更名為重庆市[[萬盛區|萬盛區]]。今隶属于[[重庆市|重庆市]][[綦江区|綦江区]]。

== 沿革 ==
建区前，南桐矿区辖地原分属[[四川省|四川省]][[涪陵专区|涪陵专区]][[南川县|南川县]]、[[江津专区|江津专区]][[綦江县|綦江县]]和[[贵州省|贵州省]][[遵义专区|遵义专区]][[桐梓县|桐梓县]]，煤矿储量丰富，煤矿隶属于西南工业部煤业管理局，但各矿毗邻乡、场隶属于四川、贵州的3个县，区划互相交错，不便管理。1954年，经[[四川省人民政府|四川省人民政府]]商得[[贵州省人民政府|贵州省人民政府]]同意，达成划南川县、綦江县、桐梓县部分地区组成南桐矿区并归[[重庆市人民政府|重庆市人民政府]]领导的共识，并于同年12月上报[[中华人民共和国国务院|国务院]]{{efn|1954年12月6日，府民密(54)字第60号文《南桐煤矿区情况调查及建立政权意见的报告》。}}，于1955年1月得到批复{{efn|name=a1}}。1955年7月6日，南桐矿区交接委员会成立；8月16日，[[重庆市人民委员会|重庆市人民委员会]]南桐矿区办事处成立，作为南桐矿区临时政权机构，南桐矿区正式成立；1956年1月25日至27日，重庆市南桐矿区第一届人民代表大会召开，选举产生了第一届人民委员会。{{r|区志|page=44}}

建區後，1956年至1957年在南桐、東林兩個煤礦地區和區委、區人委所在地萬盛地區先後建立3個街道辦事處。在此期間，鄉鎮的建制進行過多次調整。1956年1月撤銷原有兩個縣轄區委，將原27個鄉鎮合併為15個鄉；1958年7月又將15個鄉合併為7個大鄉，9月：全區71個高級農業合作社改建為5個政社合一的人民公社。1961年5月和1962年3月，先後調整為9個和11個公社。

1992年底，四川省人民政府请示将南桐矿区更名为万盛区{{efn|1992年12月5日，《关于重庆市南桐矿区更名为万盛区的请示》。}}，并于次年初获国务院批复同意，由民政部公布{{efn|1993年2月16日，民行批〔1993〕42号文《关于四川省重庆市南桐矿区更名为万盛区的批复》。}}。1997年随重庆市直辖。

2011年10月，经国务院批准，重庆市调整部分行政区划，撤销万盛区和[[綦江县|綦江县]]，设立[[綦江区|綦江区]]<ref>{{cite news |title=重庆调整部分行政区划：4区(县)并为2区 |url=http://news.163.com/11/1027/18/7HD2AKAQ00014JB5.html |accessdate=2019-03-16 |work=中国广播网 |date=2011-10-27}}</ref>。綦江区于同年12月26日正式挂牌成立。

== 行政区划 ==
重庆市南桐矿区在四川、貴州兩省交界處，下辖原[[四川省|四川省]][[南川縣|南川縣]]的萬盛、鄧家、大埡、腰子、叢林、簸箕、松林等7個鄉，[[綦江縣|綦江縣]]的青年、金靈、建設等3個鄉及[[貴州省|貴州省]][[桐梓縣|桐梓縣]]的第十區全區平壩、興文、中朝、青林、上壩、板遼、天橋、茶園、大壩、農林、廟壩、民權、桃子、營寨、王家壩、景星、青山等17個鄉共27個鄉。礦區人民委員會設在萬盛{{efn|name=a1|1955年1月21日，(55)國政常字第6號文。}}。

== 人口 ==
南桐矿区成立前，因位于两省三县交界，人口稀少，至民国二十七年（1938年）国民政府建立南桐煤矿后人口开始增加。1955年南桐矿区成立时，全区共有24924户，121619人。至1985年，全区共有55259户，239225人{{r|区志|page=97-98}}。

== 经济 ==
1949年，全区社会总产值为1256万元（1980年不变价），其中[[第一产业|第一]]、[[第二产业|第二]]、[[第三产业|第三]]产业占比为69.5:22.5:8。经过[[土地改革运动|土地改革]]、[[农业合作化运动|农业合作化]]，1955年南桐矿区成立时，全区社会总产值为3398万元（1980年不变价），其中第一、第二、第三产业占比为39.8:53.8:6.1。[[三年困难时期|三年困难时期]]，区内工农业产值发生下滑。[[文化大革命|文化大革命]]时期，区内农业产值停滞不前，工业产值发展迅速，1976年文革结束时，全区社会总产值为13010万元（1980年不变价），其中第一、第二、第三产业占比为17.8:74.2:7.9。[[改革开放|改革开放]]后，社会经济发展迅速。1985年，全区社会总产值为22872万元（1980年不变价），其中第一、第二、第三产业占比为13.8:80.1:6.1{{r|区志|page=317-318}}。
