'''南延君坟墓盗掘事件'''1868年发生在[[朝鲜王朝|朝鲜王朝]]的一起外国人盗窃[[朝鲜高宗|朝鲜高宗]]国王祖父坟未遂事件。
== 背景 ==
在[[上海|上海]]的[[英国|英国]][[怡和洋行|怡和洋行]]职员，[[德意志|德意志]][[漢堡|漢堡]]商人[[恩斯特‧奧佩特|恩斯特‧奧佩特]]1866年4月和8月兩次造訪[[朝鲜半島|朝鲜半島]]，要求和朝鲜贸易。但是朝鲜采取闭关锁国政策，拒绝了他。这年，朝鲜国王[[李熙|李熙]]之父[[兴宣大院君|兴宣大院君]]下令取缔[[天主教|天主教]]，全国捕杀天主教徒。[[法國|法國]][[傳教士|傳教士]]費龍逃離朝鲜，抵达[[中國|中國]]的[[上海|上海]]。費龍见到奧佩特，提议盗取兴宣大院君之父[[南延君|南延君]]的遗骨，迫其同意開放港口。

== 过程 ==
奧佩特从上海的漢堡公司禪臣洋行租賃一艘船，并承诺向禪臣洋行提供好处。船上挂[[德国国旗|普魯士國旗]]，船上共37人，包括在[[美国|美国]]驻上海领事馆当过翻译的[[美国人|美国人]]詹金斯，法国传教士費龍和朝鲜天主教徒以及中国[[卫兵|卫兵]]。

他们4月19日在[[牙山市|牙山]]湾登陸，到[[德山郡|德山郡]]袭击德山郡官府，夺取[[武器|武器]]，又在民间掠取[[盜墓|盜墓]]工具。登陸6小时后找到位于德山郡的南延君墓。因灌有铁水，盗墓失败。21日船在[[永宗岛|永宗岛]]和[[江华岛|江华岛]]之间的[[东检岛|东检岛]]海面下了锚。他们试图同[[朝鲜王朝|朝鲜王朝]]谈判，兴宣大院君下令开火。船26日逃回上海。奧佩特手下兩個[[菲律賓|菲律賓]]雇傭兵死亡，一名德意志人受傷。

== 事后 ==
奧佩特支援者的對手，向[[普魯士|普魯士]]政府報告說，奧佩特的海盜船濫用了普魯士國旗，破壞了普魯士的聲譽，普魯士駐上海領事因此被撤職。普魯士[[參議院|參議院]]認為奧佩特應該受到審判。[[西班牙政府|西班牙政府]]代表船員中的菲律賓人要求得到賠償。漢堡法院判奧佩特入獄3個月。詹金斯也受到美國駐華領事館的審判，但是被宣佈無罪釋放，法庭認為他有犯罪意圖但是沒有實行。

此事件使朝鮮王朝加深了对西方和天主教的敌视，加强了闭关锁国政策。[[礼山|礼山]]和[[瑞山|瑞山]]地区的1千多名天主教徒被拷打致死。
