'''南京市第十三中学'''是一所位于[[中国|中国]][[江苏省|江苏省]][[南京市|南京市]][[玄武区|玄武区]][[西家大塘|西家大塘]]的国家级示范[[中学|中学]]，占地72亩，毗邻[[玄武湖|玄武湖]]及[[台城|台城]]。

== 历史 ==
十三中创办于1955年，是[[中华人民共和国|中华人民共和国]]成立后在南京新开设的第一所完全中学。自六十年代确定为省重点，2003年被评为首批江苏省四星级学校，连续四次被评为“江苏省文明单位”、“江苏省德育先进学校”，2011年成为江苏省首批课程基地学校。2002年通过国家级示范性普通高中验收后在南京市教育改革中屡出新招<ref>{{cite web | url=http://www.nj13z.cn/News/Article.aspx?MenuId=13&ArticleId=9 | title=南京市第十三中学简介 | accessdate=2012-12-21}}</ref>，既有语文课程基地建设<ref>{{cite web | url=http://www.nj13z.cn/News/Article.aspx?MenuId=66&ArticleId=490 | title=南京十三中搭建“一主六辅”的课程基地建设平台 | accessdate=2012-12-21}}</ref>、“改革教育理念，实现弯道超车”的大课堂学本尝试<ref>{{cite book | title=南京市第十三中学各年级大课堂学本 | publisher=南京市第十三中学 | author=南京十三中各年级学科备课组}}</ref>，也有饱受争议的21:30晚自习。

截至2019年2月，该校有一千多名学生，二百零五名教职工，高中有四十四个班级（含邦德国际高中）。

== 校训 ==
校训是“志远行近”，“志远”是指“志在中华，志向高远，志在一流，志者必胜”，“行近”指的是“行在脚下，行在今天，行在细节，行者必成”。

== 争议 ==
十三中自2004年起开始晚自习到21:30的制度，后改为自愿报名。其官方解释称“提高学生素质，从读书习惯、生活习惯抓起，‘管好时间’，‘一进校就学习’,‘减少隐性浪费，提高课堂即时率’......‘加强精细化管理，坚持个性化教学’,从‘过好有道德的课堂生活’做起，从‘管住嘴、打好球、走好路’做起，把先进思想转化为明明白白的要求，把先进理念转变成实实在在的操作步骤，坚持“素质+效率”的教育模式。”这一举动据称极大提高了教学质量。但也引起诸多争议<ref>{{cite web| url=http://edu.anhuinews.com/system/09/08/2005/001322230.shtml| title=破解大考之后南京教育三大悬疑| accessdate=2012-12-21}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web | url=http://www.xici.net/d112651526.htm | title=北方教师进南京，“县中模式”是否水土不服 | accessdate=2012-12-21}}</ref>。虽官方给出诸多解释<ref>{{cite web | url=http://www.xici.net/d51746355.htm | title=南京十三中 在争议中阔步前行 | accessdate=2012-12-21}}</ref>，主张校长王军更在当时推出多篇文章为之辩护<ref>{{cite web| url=http://news.sina.com.cn/o/03-09-2005/07396849965s.shtml| title=十三中校长沉默之后吐抱怨| accessdate=2012-12-21}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，但传言仍甚嚣尘上<ref>{{cite web | url=http://news.qq.com/a/20050423/000192.htm | title=关注学生升学压力 南京家长激辩“十三中现象” | accessdate=2012-12-21}}</ref>。更在2004年的晚自习学生遇袭<ref>{{cite web | url=http://www.people.com.cn/GB/jiaoyu/1055/2974617.html | title=“女生遭强奸”传言令人震惊——谣言的叩问 | accessdate=2012-12-21}}</ref>和2012年10月23日的学生跳楼事件中<ref>{{cite web | url=http://bbs.eduu.com/thread-1725217-1-1.html | title=追踪13中跳楼事件，为何十三中总摊上这事？ | accessdate=2012-12-21 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web | url=http://nj.zhongkao.com/e/20121023/5086284f1023a.shtml | title=多角度看13中——跳楼阴霾下的新晋名校 | accessdate=2012-12-21}}</ref>达到高潮。
