'''南开大学体育中心'''，即'''南开大学新体育馆'''，位於[[中華人民共和國|中華人民共和國]][[天津市|天津市]]，始建于2005年1月，於2006年10月竣工及投用，占地约30,000平米，建筑面积24,600平米，总投资1.5亿元，落成时是[[教育部直属高校|教育部直属高校]]中规模最大的综合性体育中心，也是[[南开大学|南开大学]]自建校以来单体建筑面积最大、投资最多的建筑物<ref>{{cite web| url=http://www.xinhuanet.com/chinanews/2007-11/15/content_11682035.htm| title=南开大学新体育中心万事俱备迎待“挑战杯”| publisher=新华网| accessdate=2013-08-03| deadurl=yes| archiveurl=https://web.archive.org/web/20140808050326/http://www.xinhuanet.com/chinanews/2007-11/15/content_11682035.htm| archivedate=2014-08-08}}</ref>。南开大学体育中心内部分为体育馆和游泳馆两部分，是[[2012年全国大学生运动会|2012年全国大运会]]和[[2013年东亚运动会|2013年东亚运动会]]的比赛场馆。

== 位置 ==
[[File:南开大学体育中心游泳馆.jpg|thumb]]
南开大学体育中心位于南开大学原网球场、篮球场原址，东眺天津大学-南开大学联合研究大厦和南开大学行政楼，西临南开大学体育馆旧馆，南靠[[南开大学日本研究院|南开大学日本研究院]]，北望天津大学第26教学楼。

== 历史沿革 ==
2003年3月24日，南开大学体育部制定了关于新建体育馆、游泳馆附属教学设施的意见，设计方开始根据业主要求进行设计。2004年，南开大学网球场、篮球场拆除进行土地平整。2005年1月7日，南开大学体育中心正式开工建设，有天津三建公司承建。同年10月，主体结构封顶，2006年10月竣工并投用。<ref>{{cite web | url=http://news.enorth.com.cn/system/2005/01/06/000939270.shtml | title=南开大学体育中心今动工 | publisher=北方网 | accessdate=2013-08-02}}</ref>

== 建筑与功能 ==
=== 建筑设计 ===
根据《南开大学关于新建体育馆设计委托书》和《南开大学体育部关于新建体育馆、游泳馆附属教学设施的意见》，业主[[南开大学|南开大学]]对南开大学体育中心所提出的设计要求是一座集体育教学、群众体育活动、体育比赛、大型集会和文艺演出等等多功能于一体的综合性建筑。体育中心建筑体量较大，长约160米，宽约75米。由于天津地区地下水位较高，地下设施建设较为复杂且风险和成本较高，因而南开大学体育中心尽量减少对地下空间的使用，包括游泳池均建造在地面以上。由于根据体育馆相关设计规范，体育馆比赛场地上空要求不小于15米，但游泳馆由于不设跳水池，游泳馆比赛场地净高比体育馆比赛场地净高低5.4米。<ref>南开大学体育中心设计方案说明</ref>

=== 承办活动和赛事 ===
南开大学体育中心内部分为体育馆和游泳馆两部分，体育馆在体育中心西侧有3000个固定座位和2000个活动座位，游泳馆在体育中心东侧有1000个固定座位、50×25米比赛池和16×25米练习池。除作为南开大学体育部的一般教学场地外，南开大学体育中心承办过南开大学开学典礼、毕业典礼以及第十届“挑战杯”飞利浦全国大学生课外学术科技作品竞赛等活动。<ref>{{cite web| url=http://www.tj.xinhuanet.com/2007-11/15/content_11678455.htm| title=南开大学新体育中心万事俱备迎接“挑战杯”| publisher=新华网| accessdate=2013-08-06| deadurl=yes| archiveurl=https://web.archive.org/web/20160304194432/http://www.tj.xinhuanet.com/2007-11/15/content_11678455.htm| archivedate=2016-03-04}}</ref>同时，是2009年男子篮球亚洲锦标赛<ref>{{cite web | url=http://sports.sina.com.cn/cba/p/2009-08-05/20054519452.shtml | title=男篮世锦赛天津比赛场馆即景 南开大学体育馆 | publisher=新浪体育 | accessdate=2013-08-08}}</ref>、2012年全国大学生运动会和[[2013年东亚运动会|2013年东亚运动会]]的比赛场馆。
