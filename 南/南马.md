'''南马'''是马来西亚的一个地理大区，意即马来西亚的南部。南马狭义指[[柔佛|柔佛]]；广义则指[[马六甲|马六甲]]、[[森美兰|森美兰]]和[[柔佛|柔佛]]三州。本条目介绍的是广义的南马。南马东临[[南中国海|南中国海]]，西临[[马六甲海峡|马六甲海峡]]与[[印尼|印尼]][[苏门答腊|苏门答腊]]相望，南面与[[新加坡|新加坡]]之间隔着一条[[柔佛海峡|柔佛海峡]]。

== 历史 ==
=== 分治时期 ===
==== 马六甲 ====
1400年，[[马六甲王朝|马六甲王朝]]成立<ref name=Coedes>{{Cite book
  | last =   Cœdès
  | first = George
  | authorlink = Georges Coedès
  | title = The Indianized states of Southeast Asia
  | publisher = University of Hawaii Press
  | year = 1968
  | url = https://books.google.com/books?id=iDyJBFTdiwoC
  | isbn =978-0-8248-0368-1 }}</ref>，极盛时期疆域涵盖了整个南马。1511年，马六甲被葡萄牙占领，[[葡属马六甲|葡属马六甲]]成立，马六甲苏丹后裔逃亡到柔佛<ref name="RICKLEFSp23">{{cite book |last=Ricklefs |first=M.C. |title=A History of Modern Indonesia since c. 1300, 2nd Edition |publisher=MacMillan |year=1991 |location=London |pages=23 |isbn=0-333-57689-6 }}</ref>。1641年，荷兰占领马六甲，[[荷属马六甲|荷属马六甲]]成立<ref>{{cite book |last=De Witt |first=Dennis |title=History of the Dutch in Malaysia |location=Malaysia |publisher=Nutmeg Publishing |year=2007 |isbn=978-983-43519-0-8 |url=https://books.google.com/books?id=T92ajjSBi_wC }}</ref>。1824年，[[1824年英荷条约|英国和荷兰签订条约]]，而英国取得马六甲的控制权<ref>{{cite web |title=traktaat van Londen, 1824 |url=http://www.antenna.nl/wvi/nl/dh/geschiedenis/traktaat.html |website=www.antenna.nl}}</ref>。2年后，[[海峡殖民地|海峡殖民地]]成立。

==== 森美兰 ====
15世纪，[[苏门答腊|苏门答腊]]的[[米南佳保人|米南佳保人]]在[[森美兰|森美兰]]地区建立城邦，先后接受[[马六甲王朝|马六甲王朝]]和[[柔佛苏丹王朝|柔佛苏丹王朝]]保护<ref name=travel>{{cite web|url=http://go2travelmalaysia.com/tour_malaysia/ns_historical.htm|title=Negeri Sembilan - History and Culture|website=go2travelmalaysia.com}}</ref><ref>{{cite web|url=https://museumvolunteersjmm.com/2016/04/04/the-minangkabau-of-negeri-sembilan/|title=The Minangkabau of Negeri Sembilan|date=2016-04-04|publisher=}}</ref>。1874年，[[森美兰|森美兰]]接受参政司制度<ref>{{cite web|url=https://pekhabar.com/h-i-d-s-perjanjian-sungai-ujung-1874/|title=Perjanjian Sungai Ujung 1874 - PeKhabar|date=2016-04-21|publisher=}}</ref>，并于1895年成为[[马来联邦|马来联邦]]的一员<ref>{{cite web|url=http://www.utusan.com.my/rencana/menyelusuri-sejarah-negeri-sembilan-1.163009|title=Menyelusuri sejarah Negeri Sembilan|publisher=}}</ref>。

==== 柔佛 ====
1528年，[[柔佛苏丹王朝|柔佛苏丹王朝]]正式成立。1699年，柔佛的马六甲苏丹王朝血统正式断绝，柔佛宰相王朝开始。1824年的英荷条约导致柔佛宰相王朝分裂为柔佛本土以及[[廖内群岛|廖内群岛]]。1866年，柔佛天猛公王朝正式成立，苏丹阿布巴卡迁都至丹绒布蒂里，并改名为柔佛峇鲁（[[新山|新山]]）。1885年，[[柔佛王国|柔佛王国]]成为[[大英帝国|大英帝国]]的[[保护国|保护国]]。1914年，柔佛接受顾问官制度。

=== 合并 ===
1946年，[[马来亚联邦|马来亚联邦]]成立，将原本分别属于不同国家、但皆为[[大英帝国|大英帝国]]殖民地的南马三州：马六甲（[[海峡殖民地|海峡殖民地]]）、柔佛（[[柔佛王国|柔佛王国]]，[[马来属邦|马来属邦]]一员）、森美兰（[[马来联邦|马来联邦]]）统一管理。

1948年，马来亚联邦改组为[[马来亚联合邦|马来亚联合邦]]，并于1957年8月31日取得独立。1963年9月16日马来西亚成立，这段时期的南马也包括新加坡，一直到1965年8月9日新加坡独立为止。

== 人口 ==
截至2015年，南马人口总数为5,524,900人。其中[[马来西亚马来人|马来人]]占55.52%，[[马来西亚华人|华人]]占27.59%，[[马来西亚印度人|印度人]]占7.89%，原住民占1.69%<ref>{{cite web |title=Unjuran Populasi Penduduk 2015 |url=https://web.archive.org/web/20160212125740/http://pmr.penerangan.gov.my/index.php/info-terkini/19463-unjuran-populasi-penduduk-2015.html |date=2016-02-12}}</ref>。华人大多数居住在市区，马来人和土著则多数住在郊区<ref>{{cite web|url=http://ywlock.blogspot.com/2011/08/2010.html|title=Lock's Simple Life 知足常樂: 2010年馬來西亞人口調查報告(分析二)|first=|last=Lock|date=2011-08-02|publisher=}}</ref>。

=== 宗教 ===
截至2010年，南马信奉[[马来西亚伊斯兰教|伊斯兰教]]的占59.86%，信奉[[佛教|佛教]]的占27.06%，信奉[[兴都教|兴都教]]的占7.8%，信奉[[马来西亚基督教|基督教]]的占3.09%，信奉[[中国民间信仰|中国民间信仰]]的人占0.6%<ref>{{cite web |url=https://www.statistics.gov.my/censusatlas/images/ReligionEN.pdf|title=ReligionEN.pdf}}</ref>。

== 华人方言 ==
由于受到[[新加坡|新加坡]]“多讲华语，少说方言”以及[[讲华语运动|讲华语运动]]的影响，南马的方言使用率急速下降，但近年来受到大马华团保护乡音运动的影响，使用人数才逐渐回升<ref>{{cite web |title=潘正联·乡音，寄何处？ |url=http://www.sinchew.com.my/node/1709452 |website=星洲网 Sin Chew Daily |language=zh-hans |date=2017-12-11}}</ref>。其中，[[柔佛|柔佛]]、[[马六甲|马六甲]]和[[森美兰|森美兰]]南部以[[福建话|福建话]]（[[泉州|泉州]][[闽南语|闽南语]]，与[[北马|北马]]的[[漳州|漳州]][[闽南语|闽南语]]有少许差异）、[[潮州话|潮州话]]为主，并且有较少的人士使用[[客家话|客家话]]；[[森美兰|森美兰]]北部受[[中马|中马]]影响则以[[广东话|广东话]]为主。

== 地理 ==
南马的地势与西马其它地区相比之下较为低平，以[[丘陵|丘陵]]及[[平原|平原]]为主。最高山峰是位于[[森美兰|森美兰]]的大鬼山（Gunung Besar Hantu），高达1,462米。

实际上南马并无与任何周边国家接壤。但是由于[[新加坡|新加坡]]与[[柔佛|柔佛]]之间有两条桥梁衔接（一为[[新柔长堤|新柔长堤]]，另一个是[[马新第二通道|马新第二通道]]），因此南马的柔佛州也可以被视为和新加坡接壤。

===主要市镇===
南马共有[[马来西亚县份|20个县]]和28个[[马来西亚地方政府|地方政府]]。其中获得市议会或市政厅等级的地方政府区域共有12处，县议会则有25个，以及一个特别地方行政处（[[边佳兰|边佳兰]]）<ref>{{Cite web |url=http://www.epbt.gov.my/osc/PBT2_index.cfm?Neg=00&Taraf=0 |title=存档副本 |access-date=2018-07-17 |archive-url=https://web.archive.org/web/20180206100710/http://www.epbt.gov.my/osc/PBT2_index.cfm?Neg=00&Taraf=0 |archive-date=2018-02-06 |dead-url=yes }}</ref>。
*[[马六甲|马六甲]]：[[马六甲市|马六甲市]]（首府）和[[爱极乐|爱极乐]]
*[[森美兰|森美兰]]：[[芙蓉市|芙蓉市]]（首府）和[[波德申|波德申]]
*[[柔佛|柔佛]]：[[新山|新山]]（首府）、[[依斯干达公主城|依斯干达公主城]]、[[巴西古当|巴西古当]]、[[麻坡|麻坡]]、[[峇株巴辖|峇株巴辖]]、[[居銮|居銮]]和[[昔加末|昔加末]]

===气候===
南马属于热带雨林气候，终年炎热多雨。[[森美兰|森美兰]]的{{tsl|ms|Kuala Pilah|瓜拉庇劳}}是马来西亚降雨量最少的地区。2006年南马大水灾中，南马多地遭受重创<ref>{{cite web |title=光华日报::旧网站 |url=http://archive.kwongwah.com.my/kwyp_news/news_show_old.asp?n=69673&rlt=138&cls=101&txt=2006/12/22/gn20061222_69673 |website=archive.kwongwah.com.my}}</ref>。

=== 烟霾问题 ===
与[[北马|北马]]及[[中马|中马]]相比，南马一般上受烟霾的影响不大，但有数次例外。1997年，南马首度面对严重的烟霾问题。2013年，与南马相隔接近15年的烟霾问题再次来袭。其中[[麻坡|麻坡]]的空气污染指数更一度突破750点，达到非常危险水平，以致当地政府宣布启动紧急状态<ref>{{cite web |title=指數一度飆至750點 麻坡禮讓緊急狀態 |url=http://www.chinapress.com.my/node/537361 |publisher=中國報 |accessdate=2014-09-21 |archiveurl=https://web.archive.org/web/20140709045435/http://www.chinapress.com.my/node/537361 |archivedate=2014-07-09 |deadurl=yes }}</ref>。2014年<ref>{{cite web |title=【更新】苏岛热点增．烟霾来袭．南马空气污染指数攀升 |url=http://www.sinchew.com.my/node/1412038 |website=星洲网 Sin Chew Daily |language=zh-hans |date=2014-10-08}}</ref>及2015年，烟霾问题再次发生<ref>{{cite web |title=新山空氣污染指數中等 |url=http://www.orientaldaily.com.my/s/101369 |website=東方網 {{!}} 馬來西亞東方日報 |language=zh}}</ref>，多州学校也在当时被迫停课<ref>{{cite web |title=烟霾严重 大马宣布停课2天 - 大纪元 |url=http://www.epochtimes.com/gb/15/10/4/n4542558.htm |website=大纪元 www.epochtimes.com |language=zh-CN |date=2015-10-04}}</ref>。
