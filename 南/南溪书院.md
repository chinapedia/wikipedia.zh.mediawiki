'''南溪书院'''位于[[福建省|福建省]][[三明市|三明市]][[尤溪縣|尤溪縣]]城南的公山上，係[[南宋|南宋]]著名理学家[[朱熹|朱熹]]的诞生地。1996年被列为[[福建省文物保护单位|福建省文物保护单位]]。<ref name="fjsq">{{cite web |title=二、书院、文庙 - 南溪书院 |publisher=福建省情资料库 |url=http://www.fjsq.gov.cn/showtext.asp?ToBook=152&index=30}}</ref>
[[File:Zhu_Xi.jpg|thumb]]十年（1921年）]]

== 历史 ==
南溪書院的前身為[[鄭安道|鄭安道]]的宅邸，[[北宋|北宋]][[宣和|宣和]]五年（1123年），朱熹之父[[朱松|朱松]]任尤溪[[县尉|县尉]]，[[建炎|建炎]]四年（1130年）五月，他辞去官职后為避戰亂而寓居于此，同年朱熹诞生于此。朱熹幼年在此接受父親的教導，儒學就此啟蒙。在朱松過世後，朱熹即將此地改建為尤溪縣学。<ref>{{cite web| title =朱熹在武夷山的教迹| url =http://rwxy.xmu.edu.cn/exp/ShowArticle.asp?ArticleID=1102| author =明小莉| publisher =[[厦門大學|厦門大學]]人文學院| language =zh-cn| accessdate =2011-04-09}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>

朱熹逝世后，[[县令|县令]][[李修|李修]]于[[嘉熙|嘉熙]]元年（1237年）捐资在此修建了文公祠、韦斋祠、半亩方塘和尊道堂等建筑，以祀朱家父子。

南宋[[宋理宗|理宗]]、[[宋度宗|度宗]]年間，因為官民力量的結合，書院快速發展，理宗對於各地書院多給予田地、匾額或御書等賞賜。[[淳祐|淳祐]]元年（1241年），此地設立'''南溪書院'''{{R|鄧洪波|page=74}}；[[宝祐|宝祐]]元年（1253年），理宗亲自题匾「南溪书院」。

元[[至正|至正]]元年（1341年），分建两个[[祠堂|祠堂]]，明清后对该建筑屡有修缮和扩建。<ref>{{cite web| title =南溪書院| url =http://www.fjyx.gov.cn/zjyx/tpyx/10/8505.shtml| publisher =尤溪縣人民政府辦公室| language =zh-cn| accessdate =2011-04-09| deadurl =yes| archiveurl =https://web.archive.org/web/20110223074400/http://www.fjyx.gov.cn/zjyx/tpyx/10/8505.shtml| archivedate =2011-02-23}} </ref>

[[明代|明代]]編修書院之風大盛，[[萬曆|萬曆]]年間有[[葉廷祥|葉廷祥]]、[[郭以隆|郭以隆]]、[[紀延譽|紀延譽]]、[[陳翹卿|陳翹卿]]合編《南溪書院志》四卷{{R|鄧洪波|page=183-185}}。

清[[乾隆|乾隆]]二十九年（1764年），知县[[蔡述谟|蔡述谟]]将旧址改建为'''开山书院'''。[[道光|道光]]三年（1823年）重修。

1928年[[尤溪公立师范学校|尤溪公立师范学校]]创办于此。[[抗日战争|抗日战争]]时期，福建[[马尾勤工学校|马尾勤工学校]]曾迁到此处办学。

1985年辟为尤溪县博物馆。<ref name="fjsq"/>

== 现状 ==
南溪书院古朴庄严，现存建筑包括开山书院、半亩方塘、活水亭、溯源处、观书第、韦斋祠、毓秀坊、毓秀亭等。

开山书院为二进木构建筑，占地1000平方米。<ref name="fjsq"/>正堂当中立有朱熹的[[石膏|石膏]][[塑像|塑像]]，两旁是他手书的板联四幅：“读书起家之本，和顺齐家之本，勤俭治家之本，循理保家之本”。四壁高悬有上溯其八代列祖列宗的画像。

书院正门外的[[池塘|池塘]]称为“半亩方塘”，名称源于其著名诗篇《观书有感》：“半亩方塘一鉴开，天光云影共徘徊；问渠那得清如许，为有源头活水来”。明[[弘治|弘治]]十一年（1498年）期间，在知县[[方溥|方溥]]的主持下，半亩方塘得以扩大、浚深，在此“方池”上建造了“活水亭”，并通以石桥，期间历经破坏，今皆修复。
