'''南勇'''（{{lang|ko|'''남용'''}}，{{bd|1962年|6月16日|}}），[[男|男]]，[[朝鲜族|朝鲜族]]，[[吉林|吉林]][[延边|延边]]人，毕业于[[沈阳体育学院|沈阳体育学院]]，曾担任[[中国足协|中国足协]]专职副主席和[[国家足球运动管理中心|国家足球运动管理中心]]主任。

==个人经历==
出生于一个中国共产党干部家庭。父亲南兴烈曾是[[东北抗日联军|东北抗日联军]]战士，长期担任[[延边县|延边县]]委副书记，1980年代出任过延边农学院书记、延边州人大常委会副主任。<ref>{{cite web |url=http://sports.cn.yahoo.com/10-01-/346/2b32d.html |title=存档副本 |accessdate=2010-01-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20120119101226/http://sports.cn.yahoo.com/10-01-/346/2b32d.html |archivedate=2012-01-19 }}</ref>

在图们市一小上学时，被选入延边速滑队。1975年延边州儿童速滑1000米比赛中打破州纪录。

1980年2月退役，同年考入[[沈阳体育学院|沈阳体育学院]]。入学后加入中国共产党，后成为校学生会主席。毕业时被国家体委选为后备干部。<ref>{{cite web |url=http://sports.cn.yahoo.com/10-01-/346/2b32d.html |title=存档副本 |accessdate=2010-01-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20120119101226/http://sports.cn.yahoo.com/10-01-/346/2b32d.html |archivedate=2012-01-19 }}</ref>

南勇1997年9月从国家体委人事司处长调入[[中国足协|中国足协]]担任专职副主席、司库等职。2001年作为代表团团长率[[中国足球队|中国足球队]]参加[[世界盃足球外圍賽|世界杯亚洲区预选赛]]，冲击[[世界盃足球賽|世界杯]]成功。2002年继续作为代表团团长率[[中国足球队|中国足球队]]参加[[2002年世界盃足球賽|2002韩日世界杯]]。

2005年3月，南勇担任[[国家体育总局|国家体育总局]][[国家足球运动管理中心|足球运动管理中心]]党委书记（正司级）、副主任。2009年1月，南勇升任[[国家足球运动管理中心|中国足球运动管理中心]]主任<ref>[http://sports.sina.com.cn/n/2009-01-14/18314171262.shtml 足协高层调整南勇扶正 谢亚龙接替王俊生任董事长]</ref>，成为中国足球“掌门人”。

== 被捕入狱及减刑 ==
2010年1月21日，南勇和原足管中心副主任[[杨一民|杨一民]]被辽宁公安机关专案组传讯，次日[[国家体育总局|国家体育总局]]召开新闻发布会，免去其一切职务。1月27日，公安部正式宣布南勇与杨一民已经被依法刑事拘留。2010年3月1日，检察机关批准逮捕。2012年4月25日，[[铁岭市中级人民法院|铁岭市中级人民法院]]一审开庭审理。2012年6月13日，因犯受贿罪被铁岭市中级人民法院一审宣判，判处有期徒刑10年零6个月，并处没收个人财产20万元。

2014年12月9日，因在服刑期间7次获表扬，经北京市第二中级人民法院审理裁定，南勇获判减刑一年。<ref>{{cite news |language=zh-cn |publisher=新浪新闻 |url = http://news.sina.com.cn/c/2014-12-10/023031266233.shtml |title=足协原副主席南勇减刑一年 还有四年半刑期|date=2014-12-10 |accessdate=2014-12-10}}</ref>
