'''南疆系列爆炸案'''1993年6月17日至9月5日，[[东突厥斯坦独立运动|东突厥斯坦独立运动]]在[[新疆|新疆]]南部地区制造的一系列爆炸事件。他们在[[商场|商场]]、集贸[[市场|市场]]、[[饭店|饭店]]、[[文化|文化]]场所等地制造10起爆炸案。炸死2人、炸伤36人。

== 过程 ==
6月17日，[[喀什市|喀什市]]地区农机公司办公楼发生爆炸，大楼坍塌，２人[[死亡|死亡]]、７人受伤；8月1日[[喀什地区|喀什地区]][[莎车县|莎车县]]外贸公司录像厅爆炸，15人受伤；8月19日[[和田市|和田市]]文化宫前[[象棋|象棋]]摊爆炸，6人受伤。<ref>[http://www.china.com.cn/chinese/zhuanti/224386.htm “东突”恐怖势力难脱罪责，中国网，2002年1月21日]</ref><ref>[http://news.163.com/09/0707/10/5DK6I5E4000120GR.html 恐怖分子罪行累累，网易，2009-07-07]</ref>

此外，6月30日，叶城县[[公安局|公安局]][[政保|政保]]股长停放在一饭馆门前的三轮[[摩托车|摩托车]]被炸；7月1日，[[英吉沙县|英吉沙县]]2处炸弹因发现及时被排除；7月22日，[[自治区|自治区]]卫生厅工作组停放在莎车县一民族餐厅前的[[丰田|丰田]][[越野车|越野车]]被炸；8月10日晚，喀什市红旗[[照相|照相]]馆陈列橱窗爆炸，16日，喀什市人武部武器库房顶爆炸，均未造成人员伤亡；9月5日，英吉沙县公安局局长家门上发现爆炸装置。
== 破案 ==
[[新疆维吾尔自治区|新疆维吾尔自治区]][[公安厅|公安厅]][[厅长|厅长]]、两名副厅长先后坐镇南疆,采用全疆启动、统一指挥、政刑合一、整体作战的战略部署,经过一年的努力,案件相继告破,连续制造系列爆炸的是与境外有密切勾联的“东突厥斯坦民主伊斯兰党”及“伊斯兰改革者党突击队”等四个组织。

[[乌鲁木齐二·五爆炸案|乌鲁木齐二·五爆炸案]]骨干阿力木江与[[皮山县|皮山县]]“东突厥斯坦民主伊斯兰党”合谋，网罗一批人制造10起爆炸案。他们派专人学习电器知识,研制[[爆炸|爆炸]]装置.并试制成功一批[[钟表|钟表]]、[[洗衣机|洗衣机]]定时器、[[导火索|导火索]]和投掷式爆炸装置。 从1993年2月开始,他们具体谋划进行爆炸、暗杀的系列活动。5月派人到乌鲁木齐打探2·5爆炸案犯的审判过程,决定在开庭之际在南疆实施爆炸,制造国际影响,对政府施加压力,阻止对2·5案犯的审判。同时,他们还拟定了暗杀名单。 当他们再次制订计划,准备前往乌鲁木齐实施爆炸暗杀行动时,其组织被警察破获。<ref>[http://wuxizazhi.cnki.net/Search/XSJC199502014.html 南疆系列爆炸案侦察认识]《中国刑事警察》1995年第02期</ref>     

事后查明，喀什农机公司办公楼爆炸系东突厥斯坦民主伊斯兰党所为，该党受喀什市[[政协|政协]]副主席阿布都力米提大毛拉经济支持，他住在[[疏附县|疏附县]][[克拉克苏乡|克拉克苏乡]]，在喀什市办实业、[[工厂|工厂]]、[[学校|学校]]。学校名义上教[[阿拉伯语|阿拉伯语]]，还讲[[天文|天文]]、[[地理|地理]]、[[沙烏地阿拉伯|沙烏地阿拉伯]]的[[瓦哈比思想|瓦哈比思想]]。该校属于政府“统招统分”。警方随即关闭了其在喀什市的工厂和语言学校<ref>[http://zhongguo.oushinet.com/chinanews/20150226/184273_2.html 新疆暴恐分子用人民币包裹SD卡传播极端思想] {{webarchive|url=https://web.archive.org/web/20150227113000/http://zhongguo.oushinet.com/chinanews/20150226/184273_2.html |date=2015-02-27 }}</ref>。

== 评论 ==
1993年7月[[江泽民|江泽民]]批示喀什6•17系列爆炸案，给自治区党委书记[[宋汉良|宋汉良]]打[[电话|电话]]：“不要光看到几个暴力恐怖犯罪，要看到背后以[[美国|美国]]为首的西方敌对势力。对这股势力要狠狠地打，不能手软，手软是没有退路的。”为此中央对新疆派出一系列调查组。
