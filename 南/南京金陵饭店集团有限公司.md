'''南京金陵饭店集团有限公司'''（{{lang|en|Nanjing Jinling Holdings. Ltd}}），或称'''金陵饭店集团'''，是[[江苏省人民政府|江苏省人民政府]]授权[[江苏省人民政府国有资产监督管理委员会|江苏省人民政府国有资产监督管理委员会]]管理的省属国有独资公司<ref name="a1">{{Cite web |url=http://jssasac.jiangsu.gov.cn/col/col48448/index.html|title=金陵饭店|accessdate=2019-03-10 |author= |date=2017-09-19 |work= |publisher=江苏省人民政府国有资产监督管理委员会网站|language=Zh-hans }}</ref>。

集团由[[金陵饭店|金陵饭店]]发展而来。通常过各分、子公司开展业务，主要经营酒店、健康养老两个产业<ref name="a2">{{Cite web |url=http://www.jinlingholdings.com/about/dashiji/|title=集团概况》大事记|accessdate=2019-03-10 |author= |date= |work= |publisher=南京金陵饭店集团有限公司网站|language=Zh-hans }}</ref>。

1979年，筹建金陵饭店。1982年时，金陵饭店归江苏省人民政府外事办公室归口领导。1983年，在江苏省工商局注册成立南京金陵饭店，为[[全民所有制企业|全民所有制企业]]。3月，划归[[江苏省旅游局|江苏省旅游局]]领导。5月1日，金陵饭店开始试营业。10月4日，正式营业。其后，历经变革。2001年，对下属多种经营部分企业进行改制。2002年10月，江苏省人民政府批准，南京金陵饭店整体改建为南京金陵饭店集团有限公司。12月30日，控股设立金陵饭店股份有限公司。2006年时，集团拥有22家酒店、5247间客房。2007年，集团拥有43家酒店、10318间客房。2008年时，拥有74家成员酒店、18175间客房。2011年时，成员酒店为110家。2012年底，连锁酒店130家<ref name="a2"/>。江苏省人民政府国有资产监督管理委员会网站的简介中，称下属的南京金陵酒店管理有限公司管理连锁酒店126家、34000间客房<ref name="a1"/>。

== 集团组织 ==

集团最高机构为[[董事会|董事会]]、[[党委|党委]]、[[总经理室|总经理室]]，并设立[[监事会|监事会]]。除下属各个行政部门外，拥有十余个分、子公司<ref name="a3">{{Cite web |url=http://www.jinlingholdings.com/about/|title=集团概况》组织架构|accessdate=2019-03-10 |author= |date= |work= |publisher=南京金陵饭店集团有限公司网站|language=Zh-hans }}</ref>。
